## <a name="security-transparency-examples"></a><span data-ttu-id="b5a88-181">Przykłady przezroczystości zabezpieczeń</span><span class="sxs-lookup"><span data-stu-id="b5a88-181">Security Transparency Examples</span></span>  
 <span data-ttu-id="b5a88-182">Aby użyć zasady przejrzystości programu .NET Framework 2.0 (poziom 1 przejrzystości), należy użyć następujących adnotacji zestawu:</span><span class="sxs-lookup"><span data-stu-id="b5a88-182">To use the .NET Framework 2.0 transparency rules (level 1 transparency), use the following assembly annotation:</span></span>

