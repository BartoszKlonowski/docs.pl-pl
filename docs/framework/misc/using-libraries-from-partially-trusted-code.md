- <span data-ttu-id="a1e3f-115">Aby wyłączyć automatyczne **LinkDemand** i zapobiec wyjątek, można umieścić **AllowPartiallyTrustedCallersAttribute** atrybut w zakresie zestawu biblioteki udostępnionej.</span><span class="sxs-lookup"><span data-stu-id="a1e3f-115">In order to disable the automatic **LinkDemand** and prevent the exception from being thrown, you can place the **AllowPartiallyTrustedCallersAttribute** attribute on the assembly scope of a shared library.</span></span> <span data-ttu-id="a1e3f-116">Ten atrybut umożliwia biblioteki, które mają być wywoływane z częściowo zaufanego kodu zarządzanego.</span><span class="sxs-lookup"><span data-stu-id="a1e3f-116">This attribute allows your libraries to be called from partially trusted managed code.</span></span>  
  
- <span data-ttu-id="a1e3f-117">Częściowo zaufany kod, który jest przyznawany dostęp do biblioteki z tym atrybutem nadal podlega dalszym ograniczeniom zdefiniowanym <xref:System.AppDomain>przez .</span><span class="sxs-lookup"><span data-stu-id="a1e3f-117">Partially trusted code that is granted access to a library with this attribute is still subject to further restrictions defined by the <xref:System.AppDomain>.</span></span>  
  
- <span data-ttu-id="a1e3f-118">Nie ma programowego sposobu częściowo zaufanego kodu do wywołania biblioteki, która nie ma **atrybutu AllowPartiallyTrustedCallersAttribute.**</span><span class="sxs-lookup"><span data-stu-id="a1e3f-118">There is no programmatic way for partially trusted code to call a library that does not have the **AllowPartiallyTrustedCallersAttribute** attribute.</span></span>

- <span data-ttu-id="a1e3f-122">Kod, który został pilnie przetestowany pod kątem luk w zabezpieczeniach i jest zgodny z wytycznymi opisanymi w [wskazówki dotyczące bezpiecznego kodowania.](../../standard/security/secure-coding-guidelines.md)</span><span class="sxs-lookup"><span data-stu-id="a1e3f-122">Code that has been diligently tested for security vulnerabilities and is in compliance with the guidelines described in [Secure Coding Guidelines](../../standard/security/secure-coding-guidelines.md).</span></span>
