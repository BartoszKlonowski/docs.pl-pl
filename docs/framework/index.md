# <a name="net-framework-guide"></a>.NET framework — przewodnik

> [!NOTE]
> Ten zestaw zawartości .NET Framework zawiera informacje o wersji systemu .NET Framework 4.5, 4.5.1, 4.5.2, 4.6, 4.6.1, 4.6.2, 4.7, 4.7.1 i 4.7.2. Aby pobrać program .NET Framework, zobacz [Instalowanie programu .NET Framework](../../docs/framework/install/guide-for-developers.md). Aby uzyskać listę nowych funkcji i zmian w .NET Framework 4.5 [!INCLUDE[net_v46](../../includes/net-v46-md.md)], ich wersje punktu i .NET Framework 4.7, 4.7.1 i 4.7.2, zobacz [nowości w programie .NET Framework](../../docs/framework/whats-new/index.md). Aby uzyskać listę obsługiwanych platform, zobacz [wymagania systemowe programu .NET Framework](../../docs/framework/get-started/system-requirements.md). 

.NET Framework jest to platforma programistyczna do tworzenia aplikacji sieci web, systemu Windows, Windows Phone, Windows Server i Microsoft Azure. Zawiera środowisko uruchomieniowe języka wspólnego (CLR) i biblioteki klas .NET Framework, która zawiera wiele funkcji oraz pomocy technicznej dla wielu standardy branżowe.

.NET Framework zapewnia wiele usług, w tym zarządzania pamięcią, typ i pamięci bezpieczeństwa, zabezpieczeń, sieci i wdrożenia aplikacji. Udostępnia struktur danych łatwy w użyciu i interfejsów API niższego poziomu systemu operacyjnego jako abstract. Z programu .NET Framework, w tym C#, F # i Visual Basic, można użyć różnych języków programowania.  

Aby uzyskać ogólne wprowadzenie do programu .NET Framework dla użytkowników i deweloperów, zobacz [wprowadzenie](../../docs/framework/get-started/index.md). Aby obejrzeć wprowadzenie do architektury i najważniejsze funkcje programu .NET Framework, zobacz [omówienie](../../docs/framework/get-started/overview.md).  

.NET Framework może służyć z Docker i [kontenery Windows](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview). Zobacz [aplikacji wdrażanie .NET Framework z rozwiązaniem Docker z](./docker/index.md) więcej informacji na temat do uruchomienia aplikacji w kontenerach Docker.

## <a name="installation"></a>Instalacja

.NET Framework jest dostarczany z systemu Windows, co umożliwia uruchamianie aplikacji .NET Framework. Nowsza wersja programu .NET Framework nie pochodzi z wersją systemu Windows może być konieczne. Aby uzyskać więcej informacji, zobacz [Zainstaluj program .NET Framework w systemie Windows](./install/index.md).

Zobacz [naprawy programu .NET Framework](./install/repair.md) informacje na temat napraw instalację programu .NET Framework, jeśli występują błędy podczas instalowania programu .NET Framework.

Aby uzyskać bardziej szczegółowe informacje dotyczące pobierania programu .NET Framework, zobacz [Zainstaluj program .NET Framework dla deweloperów](../../docs/framework/install/guide-for-developers.md).
