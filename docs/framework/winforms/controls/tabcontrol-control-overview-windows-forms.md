# <a name="tabcontrol-control-overview-windows-forms"></a>TabControl — Informacje o formancie [Formularze systemu Windows]
<xref:System.Windows.Forms.TabControl> Windows Forms wyświetla wiele kart, takich jak rozdzielacze w notesie lub etykietach w zestawie folderów w pliku cabinet. Karty mogą zawierać obrazy i inne kontrolki. Możesz użyć kontrolki karta do tworzenia rodzajowego okna dialogowego z wieloma stronami, które pojawia się w wielu miejscach w systemie operacyjnym Windows, takich jak właściwości wyświetlania panelu sterowania. Ponadto <xref:System.Windows.Forms.TabControl> może służyć do tworzenia stron właściwości, które są używane do ustawiania grupy powiązanych właściwości.  
  
## <a name="working-with-tabcontrol"></a>Praca z elementem TabControl  
 Najważniejszym właściwością <xref:System.Windows.Forms.TabControl> jest <xref:System.Windows.Forms.TabControl.TabPages%2A>, która zawiera poszczególne karty. Każda karta jest obiektem <xref:System.Windows.Forms.TabPage>. Po kliknięciu karty wywołuje zdarzenie <xref:System.Windows.Forms.Control.Click> dla tego obiektu <xref:System.Windows.Forms.TabPage>.
