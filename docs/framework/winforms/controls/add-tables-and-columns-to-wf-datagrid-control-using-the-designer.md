### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="0acfc-116">Aby dodać tabelę z formantem DataGrid w Projektancie</span><span class="sxs-lookup"><span data-stu-id="0acfc-116">To add a table to the DataGrid control in the designer</span></span>  
  
1.  <span data-ttu-id="0acfc-117">Aby wyświetlić dane w tabeli, należy najpierw powiązać <xref:System.Windows.Forms.DataGrid> formantu do zestawu danych.</span><span class="sxs-lookup"><span data-stu-id="0acfc-117">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="0acfc-118">Aby uzyskać więcej informacji, zobacz [porady: powiązywanie formantu DataGrid formularzy systemu Windows do źródła danych przy użyciu narzędzia Projektant](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span><span class="sxs-lookup"><span data-stu-id="0acfc-118">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>  
  
2.  <span data-ttu-id="0acfc-119">Wybierz <xref:System.Windows.Forms.DataGrid> formantu <xref:System.Windows.Forms.DataGrid.TableStyles%2A> właściwości w oknie właściwości, a następnie kliknij przycisk oznaczony wielokropkiem (![VisualStudioEllipsesButton — zrzut ekranu](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) obok pozycji Właściwość do wyświetlenia **edytora kolekcji Element DataGridTableStyle**.</span><span class="sxs-lookup"><span data-stu-id="0acfc-119">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![VisualStudioEllipsesButton screenshot](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>  
  
3.  <span data-ttu-id="0acfc-120">W oknie Edytor kolekcji kliknij przycisk **Dodaj** do wstawienia styl tabeli.</span><span class="sxs-lookup"><span data-stu-id="0acfc-120">In the collection editor, click **Add** to insert a table style.</span></span>  
  
4.  <span data-ttu-id="0acfc-121">Kliknij przycisk **OK** Zamknij okno Edytor kolekcji i otwórz go ponownie, klikając przycisk wielokropka obok <xref:System.Windows.Forms.DataGrid.TableStyles%2A> właściwości.</span><span class="sxs-lookup"><span data-stu-id="0acfc-121">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>  
  
     <span data-ttu-id="0acfc-122">Po ponownym otwarciu edytora kolekcji, wszystkie tabele danych powiązane z formantem pojawi się na liście rozwijanej <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> właściwości stylu tabeli.</span><span class="sxs-lookup"><span data-stu-id="0acfc-122">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>  
  
5.  <span data-ttu-id="0acfc-123">W **członków** okno edytora kolekcji, kliknij odpowiedni styl tabeli.</span><span class="sxs-lookup"><span data-stu-id="0acfc-123">In the **Members** box of the collection editor, click the table style.</span></span>
