# <a name="panel-control-windows-forms"></a>Panel — Formant (Formularze systemu Windows)
Kontrolki `Panel` Windows Forms są używane do zapewniania możliwej do zidentyfikowania grupowania dla innych formantów. Zazwyczaj można używać paneli do podzielenia formularza przez funkcję. Kontrolka `Panel` jest podobna do kontrolki <xref:System.Windows.Forms.GroupBox>; jednak tylko formant `Panel` może mieć paski przewijania i tylko formant <xref:System.Windows.Forms.GroupBox> wyświetla podpis.
