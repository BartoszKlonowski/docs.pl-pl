# <a name="panel-control-windows-forms"></a>Panel — Formant (Formularze systemu Windows)
`Panel`Formanty Windows Forms są używane do zapewniania możliwej do zidentyfikowania grupowania dla innych formantów. Zazwyczaj można używać paneli do podzielenia formularza przez funkcję. `Panel`Kontrolka jest podobna do <xref:System.Windows.Forms.GroupBox> kontrolki, jednak tylko `Panel` formant może mieć paski przewijania i tylko <xref:System.Windows.Forms.GroupBox> formant Wyświetla podpis.
