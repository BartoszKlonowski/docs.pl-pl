# <a name="picturebox-control-overview-windows-forms"></a>PictureBox — Informacje o formancie [Formularze systemu Windows]
Formularze systemu Windows <xref:System.Windows.Forms.PictureBox> formant jest używany do wyświetlania grafiki w formacie mapy bitowej, GIF, JPEG, metaplik lub ikonę.  
  
## <a name="key-properties-and-methods"></a>Właściwości klucza i metody  
 Obraz, który jest wyświetlany jest określany przez <xref:System.Windows.Forms.PictureBox.Image%2A> właściwość, która może być ustawiony w czasie wykonywania, lub w czasie projektowania. Można również określić obrazu przez ustawienie <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> właściwości, a następnie ładowania obrazu synchronicznie przy użyciu <xref:System.Windows.Forms.PictureBox.Load%2A> metody lub asynchronicznie za pomocą <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> metody. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> Właściwość kontroluje sposób dopasowania obrazu i kontroli ze sobą. Aby uzyskać więcej informacji, zobacz [porady: modyfikowanie rozmiaru lub położenia obrazu w czasie wykonywania](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
