# <a name="picturebox-control-overview-windows-forms"></a>PictureBox — Informacje o formancie [Formularze systemu Windows]
Formularze Windows <xref:System.Windows.Forms.PictureBox> formant jest używany do wyświetlania grafiki w formacie mapy bitowej, GIF, JPEG, metaplik lub ikonę.  
  
## <a name="key-properties-and-methods"></a>Kluczowe właściwości i metody  
 Obraz, który jest wyświetlany jest określana przez <xref:System.Windows.Forms.PictureBox.Image%2A> właściwość, którą można ustawić w czasie wykonywania, czy w czasie projektowania. Obraz, który można alternatywnie określić, ustawiając <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> właściwości, a następnie załadować obraz synchronicznie przy użyciu <xref:System.Windows.Forms.PictureBox.Load%2A> metody lub asynchronicznie przy użyciu <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> metody. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> Właściwość kontroluje sposób dopasowania obrazu i formant ze sobą. Aby uzyskać więcej informacji, zobacz [jak: Modyfikowanie rozmiaru lub położenia obrazu w czasie wykonywania](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
