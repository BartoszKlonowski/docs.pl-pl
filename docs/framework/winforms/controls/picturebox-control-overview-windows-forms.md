# <a name="picturebox-control-overview-windows-forms"></a>PictureBox — Informacje o formancie [Formularze systemu Windows]
Formant Windows Forms <xref:System.Windows.Forms.PictureBox> służy do wyświetlania grafiki w formacie mapy bitowej, GIF, JPEG, metapliku lub ikony.  
  
## <a name="key-properties-and-methods"></a>Właściwości i metody klucza  
 Wyświetlany obraz jest określany przez właściwość <xref:System.Windows.Forms.PictureBox.Image%2A>, którą można ustawić w czasie wykonywania lub w czasie projektowania. Możesz również określić obraz, ustawiając właściwość <xref:System.Windows.Forms.PictureBox.ImageLocation%2A>, a następnie załaduj obraz synchronicznie przy użyciu metody <xref:System.Windows.Forms.PictureBox.Load%2A> lub asynchronicznie za pomocą metody <xref:System.Windows.Forms.PictureBox.LoadAsync%2A>. Właściwość <xref:System.Windows.Forms.PictureBox.SizeMode%2A> kontroluje, jak obraz i kontrolki są zgodne ze sobą. Aby uzyskać więcej informacji, zobacz [jak: Modyfikowanie rozmiaru lub położenia obrazu w czasie wykonywania](how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
