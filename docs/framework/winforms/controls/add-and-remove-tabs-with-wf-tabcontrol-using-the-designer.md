# <a name="how-to-add-and-remove-tabs-with-the-windows-forms-tabcontrol-using-the-designer"></a>Instrukcje: dodawanie i usuwanie kart za pomocą formularzy systemu Windows TabControl przy użyciu narzędzia Projektant
Gdy umieścisz <xref:System.Windows.Forms.TabControl> formant w formularzu, domyślnie zawiera on dwie karty. Możesz dodawać lub usuwać karty przy użyciu narzędzia Projektant.

 Poniższa procedura wymaga projektu **aplikacji systemu Windows** z formularzem zawierającym <xref:System.Windows.Forms.TabControl> kontrolkę. Aby uzyskać informacje na temat konfigurowania takiego projektu, zobacz [How to: Utwórz projekt](/visualstudio/ide/step-1-create-a-windows-forms-application-project) aplikacji Windows Forms i [instrukcje: Dodaj formanty do Windows Forms](how-to-add-controls-to-windows-forms.md).

## <a name="to-add-or-remove-a-tab-using-the-designer"></a>Aby dodać lub usunąć kartę przy użyciu narzędzia Projektant

- W tagu inteligentnym kontrolki kliknij przycisk **Dodaj kartę** lub **Usuń kartę**

     —lub—

     W oknie **Właściwości** kliknij przycisk wielokropka ( ![przycisk wielokropka (...) w okno właściwości programu Visual <xref:System.Windows.Forms.TabControl.TabPages%2A> Studio.](./media/visual-studio-ellipsis-button.png)) obok właściwości, aby otworzyć **Edytor kolekcji elementów TabPage**. Kliknij przycisk **Dodaj** lub **Usuń** .
