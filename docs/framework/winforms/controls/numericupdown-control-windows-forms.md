# <a name="numericupdown-control-windows-forms"></a>NumericUpDown — Formant (Formularze systemu Windows)
Formant Windows Forms `NumericUpDown` wygląda jak kombinacja pól tekstowych i parę strzałek, które użytkownik może kliknąć, aby dostosować wartość. Kontrolka wyświetla i ustawia pojedynczą wartość liczbową z listy wyboru. Użytkownik może zwiększyć i zmniejszyć liczbę przez klikanie przycisków w górę i w dół, naciskając klawisze strzałek w górę i w dół lub wpisując liczbę. Kliknięcie klawisza Strzałka w górę przenosi wartość w stronę maksymalną; Kliknięcie klawisza Strzałka w dół przenosi pozycję w kierunku minimum. Przykładem, że ten rodzaj kontrolki może być przydatny dla kontrolki głośności w odtwarzaczu muzycznym. Numeryczne kontrolki w górę są używane w niektórych aplikacjach panelu sterowania systemu Windows.
