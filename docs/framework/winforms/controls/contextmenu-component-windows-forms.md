 Składnik Windows Forms `ContextMenu` służy do zapewniania użytkownikom łatwego dostępnego menu skrótów zawierającego często używane polecenia, które są skojarzone z wybranym obiektem. Elementy w menu skrótów są często podzbiorem elementów z menu głównego, które pojawiają się w innym miejscu aplikacji. Menu skrótów są zwykle dostępne przez kliknięcie prawym przyciskiem myszy. Na Windows Forms są one skojarzone z innymi kontrolkami.
