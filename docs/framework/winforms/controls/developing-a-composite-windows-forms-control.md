# <a name="develop-a-composite-windows-forms-control"></a>Opracowywanie złożonego formantu Windows Forms

Możesz opracować formant Windows Forms złożone przez połączenie innych kontrolek Windows Forms. Kontrolki złożone pochodne od <xref:System.Web.UI.UserControl> są nazywane kontrolkami użytkownika. Klasa bazowa, <xref:System.Windows.Forms.UserControl>, zapewnia Routing klawiatury dla formantów podrzędnych, dzięki czemu można uzyskać fokus formantów podrzędnych. Aby zapoznać się z przykładem kontrolki użytkownika, <xref:System.Windows.Forms.UserControl> Zobacz przykład [w temacie How to: Zastosuj atrybuty w kontrolkach](how-to-apply-attributes-in-windows-forms-controls.md)Windows Forms.
