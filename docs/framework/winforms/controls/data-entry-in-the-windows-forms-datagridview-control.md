# <a name="data-entry-in-the-windows-forms-datagridview-control"></a>Wprowadzanie danych w formancie DataGridView formularzy systemu Windows
`DataGridView` Formant zawiera kilka funkcji umożliwiające zmienić sposób użytkowników, dodawanie lub modyfikowanie danych w formancie. Na przykład możesz wprowadzić wprowadzania danych bardziej wydajne, podając wartości domyślne dla nowych wierszy oraz alerty użytkowników w przypadku wystąpienia błędów.
