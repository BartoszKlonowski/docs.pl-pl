# <a name="radiobutton-control-windows-forms"></a>RadioButton — Formant (Formularze systemu Windows)
Windows Forms `RadioButton` kontrolki zawierają zestaw dwóch lub kilku wzajemnie wykluczających się opcji dla użytkownika. Chociaż przyciski radiowe i pola wyboru mogą wyglądać podobnie do działania, istnieje ważna różnica: gdy użytkownik wybierze przycisk radiowy, inne przyciski radiowe w tej samej grupie nie mogą być również zaznaczone.
