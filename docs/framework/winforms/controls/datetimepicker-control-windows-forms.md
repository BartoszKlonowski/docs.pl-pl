# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker — Formant (Formularze systemu Windows)
Formant Windows Forms `DateTimePicker` umożliwia użytkownikowi wybranie jednego elementu z listy dat lub godzin. Gdy jest używany do reprezentowania daty, pojawia się w dwóch częściach: Lista rozwijana z datą reprezentowaną w tekście i siatką wyświetlaną po kliknięciu strzałki w dół obok listy.
