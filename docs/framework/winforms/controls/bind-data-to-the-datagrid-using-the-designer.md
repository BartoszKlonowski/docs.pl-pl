<span data-ttu-id="a19f8-103">Projektant umożliwia łączenie <xref:System.Windows.Forms.DataGridView> kontrolki źródła danych na kilka różnych odmian, w tym baz danych, obiekty biznesowe lub usług sieci Web.</span><span class="sxs-lookup"><span data-stu-id="a19f8-103">You can use the designer to connect a <xref:System.Windows.Forms.DataGridView> control to data sources of several different varieties, including databases, business objects, or Web services.</span></span> <span data-ttu-id="a19f8-104">Gdy powiąże formant ze źródłem danych przy użyciu narzędzia Projektant kontrolki zostanie automatycznie powiązany <xref:System.Windows.Forms.BindingSource> składnik, który reprezentuje źródło danych.</span><span class="sxs-lookup"><span data-stu-id="a19f8-104">When you bind the control to a data source using the designer, the control is automatically bound to a <xref:System.Windows.Forms.BindingSource> component that represents the data source.</span></span> <span data-ttu-id="a19f8-105">Ponadto kolumny są automatycznie generowane w formancie do dopasowania informacji o schemacie, dostarczone przez źródło danych.</span><span class="sxs-lookup"><span data-stu-id="a19f8-105">Additionally, columns are automatically generated in the control to match the schema information provided by the data source.</span></span>

### <a name="to-bind-the-control-to-a-data-source"></a><span data-ttu-id="a19f8-117">Aby powiązać formant ze źródłem danych</span><span class="sxs-lookup"><span data-stu-id="a19f8-117">To bind the control to a data source</span></span>  
  
1.  <span data-ttu-id="a19f8-118">Kliknij symbol tagu inteligentnego (![symbol tagu inteligentnego](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) w prawym górnym rogu <xref:System.Windows.Forms.DataGridView> kontroli.</span><span class="sxs-lookup"><span data-stu-id="a19f8-118">Click the smart tag glyph (![Smart Tag Glyph](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control.</span></span>  
  
2.  <span data-ttu-id="a19f8-119">Kliknij strzałkę listy rozwijanej dla **wybierz źródło danych** opcji.</span><span class="sxs-lookup"><span data-stu-id="a19f8-119">Click the drop-down arrow for the **Choose Data Source** option.</span></span>  
  
3.  <span data-ttu-id="a19f8-120">Jeśli projekt nie ma źródła danych, kliknij przycisk **Dodaj źródło danych projektu** i postępuj zgodnie z instrukcjami, wskazane przez kreatora.</span><span class="sxs-lookup"><span data-stu-id="a19f8-120">If your project does not already have a data source, click **Add Project Data Source** and follow the steps indicated by the wizard.</span></span>  
  
     <span data-ttu-id="a19f8-121">Aby uzyskać więcej informacji, zobacz [Kreatora konfiguracji źródła danych](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="a19f8-121">For more information, see [Data Source Configuration Wizard](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span></span> <span data-ttu-id="a19f8-122">Nowego źródła danych będą wyświetlane w **wybierz źródło danych** okna listy rozwijanej.</span><span class="sxs-lookup"><span data-stu-id="a19f8-122">Your new data source will appear in the **Choose Data Source** drop-down window.</span></span> <span data-ttu-id="a19f8-123">Jeśli nowe źródło danych zawiera tylko jednego członka, takiego jak tabela pojedynczej bazy danych, formant będzie automatycznie wiązany tego członka.</span><span class="sxs-lookup"><span data-stu-id="a19f8-123">If your new data source contains only one member, such as a single database table, the control will automatically bind to that member.</span></span> <span data-ttu-id="a19f8-124">W przeciwnym razie przejdź do następnego kroku.</span><span class="sxs-lookup"><span data-stu-id="a19f8-124">Otherwise, continue to the next step.</span></span>  
  
4.  <span data-ttu-id="a19f8-125">Rozwiń **innych źródeł danych** i **zdroje dat projektu** węzłów, jeśli nie są jeszcze rozwinięte, a następnie wybierz źródło danych, aby powiązać formant.</span><span class="sxs-lookup"><span data-stu-id="a19f8-125">Expand the **Other Data Sources** and **Project Data Sources** nodes if they are not already expanded, and then select the data source to bind the control to.</span></span>  
  
5.  <span data-ttu-id="a19f8-126">Jeśli źródło danych zawiera więcej niż jeden element członkowski, np. Jeśli utworzono <xref:System.Data.DataSet?displayProperty=nameWithType> zawierający wiele tabel, rozwiń węzeł źródła danych, a następnie wybierz określonego członka, aby powiązać.</span><span class="sxs-lookup"><span data-stu-id="a19f8-126">If your data source contains more than one member, such as if you have created a <xref:System.Data.DataSet?displayProperty=nameWithType> that contains multiple tables, expand the data source, and then select the specific member to bind to.</span></span>  
  
6.  <span data-ttu-id="a19f8-127">Aby tworzenie relacji wzorzec/szczegół w **wybierz źródło danych** okna listy rozwijanej na sekundę <xref:System.Windows.Forms.DataGridView> sterowania, a następnie rozwiń <xref:System.Windows.Forms.BindingSource> utworzone dla tabeli nadrzędnej, a następnie wybierz pokrewnej tabeli podrzędnej na liście wyświetlane.</span><span class="sxs-lookup"><span data-stu-id="a19f8-127">To create a master/detail relationship, in the **Choose Data Source** drop-down window for a second <xref:System.Windows.Forms.DataGridView> control, expand the <xref:System.Windows.Forms.BindingSource> created for the parent table, and then select the related child table from the list shown.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="a19f8-128">Jeśli projekt zawiera już źródło danych, można również użyć **źródeł danych** okna, aby utworzyć formularz danych.</span><span class="sxs-lookup"><span data-stu-id="a19f8-128">If your project already has a data source, you can also use the **Data Sources** window to create a data form.</span></span> <span data-ttu-id="a19f8-129">Aby uzyskać więcej informacji, zobacz [okna źródeł danych](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="a19f8-129">For more information, see [Data Sources Window](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span></span>
