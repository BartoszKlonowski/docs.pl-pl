# <a name="printdocument-component-windows-forms"></a>PrintDocument — Składnik (Formularze systemu Windows)
Składnik `PrintDocument` Windows Forms służy do ustawiania właściwości, które opisują, co należy wydrukować, a następnie do drukowania dokumentu w aplikacjach opartych na systemie Windows. Może być używany w połączeniu ze składnikiem <xref:System.Windows.Forms.PrintDialog>, aby znajdować się w poleceniu wszystkich aspektów drukowania dokumentów.
