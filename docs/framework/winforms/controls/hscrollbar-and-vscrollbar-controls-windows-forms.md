# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>HScrollBar i VScrollBar — Formanty (Formularze systemu Windows)
Windows Forms kontrolki paska przewijania umożliwiają łatwe nawigowanie za pomocą długich list elementów lub dużej ilości informacji przez przewijanie w poziomie lub w pionie w aplikacji lub kontrolce. Paski przewijania są wspólnym elementem interfejsu systemu Windows.
