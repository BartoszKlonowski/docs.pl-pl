# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>HScrollBar i VScrollBar — Formanty (Formularze systemu Windows)
Formanty paska przewijania formularze Windows są używane umożliwia prostą nawigację w za pośrednictwem długą listę elementów lub dużych ilości informacji, przewijając albo w poziomie lub w pionie w obrębie aplikacji lub formantu. Paski przewijania są typowe element interfejsu Windows.
