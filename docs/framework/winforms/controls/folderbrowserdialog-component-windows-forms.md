# <a name="folderbrowserdialog-component-windows-forms"></a>FolderBrowserDialog — Składnik (Formularze systemu Windows)
Składnik `FolderBrowserDialog` Windows Forms zawiera interfejs, za pomocą którego użytkownicy mogą przeglądać i wybierać foldery lub tworzyć nowe. Jest to uzupełnienie składnika [składnika OpenFileDialog](openfiledialog-component-windows-forms.md) , który jest używany do przeglądania i wybierania plików.
