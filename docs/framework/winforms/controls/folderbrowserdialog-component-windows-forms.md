# <a name="folderbrowserdialog-component-windows-forms"></a>FolderBrowserDialog — Składnik (Formularze systemu Windows)
Formularze Windows `FolderBrowserDialog` składnik wyświetla interfejs za pomocą którego użytkownicy mogą przeglądać i wybierz folder lub Utwórz nową. Jest to uzupełnienie [OpenFileDialog, składnik](openfiledialog-component-windows-forms.md) składnik, który służy do przeglądania i zaznaczania plików.
