# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Porady: tworzenie podstawowych formularzy systemu Windows ToolStrip z elementami standardowymi przy użyciu narzędzia Projektant
Poniższa procedura pokazuje, jak utworzyć <xref:System.Windows.Forms.ToolStrip> i dodać siedem <xref:System.Windows.Forms.ToolStripButton> formantów, które reprezentują typowe zadania.

## <a name="to-add-standard-items-in-the-designer"></a>Aby dodać elementy standardowe do projektanta

1. Utwórz formant <xref:System.Windows.Forms.ToolStrip>.

2. W prawym górnym rogu <xref:System.Windows.Forms.ToolStrip>kliknij strzałkę zadania inteligentnego, aby wyświetlić okienko **zadania ToolStrip** .

3. W okienku **zadania ToolStrip** wybierz pozycję **Wstaw elementy standardowe**.
