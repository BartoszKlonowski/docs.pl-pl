 <span data-ttu-id="28f97-109"><xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> Właściwość identyfikuje element menu, w których listy rozwijanej zostanie wypełnione tytuły bieżącego MDI nadrzędne elementy podrzędne MDI formularza.</span><span class="sxs-lookup"><span data-stu-id="28f97-109">The <xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> property identifies the menu item whose drop-down list you will populate with the titles of the current MDI parent form's MDI children.</span></span> <span data-ttu-id="28f97-110">Na przykład, zwykle listę aktualnie otwartych na elementy podrzędne MDI **okna** menu.</span><span class="sxs-lookup"><span data-stu-id="28f97-110">For example, you typically list MDI children that are currently open on the **Window** menu.</span></span>
