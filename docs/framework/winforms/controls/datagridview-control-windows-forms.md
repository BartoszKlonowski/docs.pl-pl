# <a name="datagridview-control-windows-forms"></a>DataGridView — Formant (Formularze systemu Windows)
`DataGridView`Formant zapewnia zaawansowany i elastyczny sposób wyświetlania danych w formacie tabelarycznym. Można użyć `DataGridView` kontrolki do wyświetlania widoków z niewielką ilością danych tylko do odczytu lub można skalować ją w celu wyświetlenia edytowalnych widoków bardzo dużych zestawów danych.  
  
 Formant można rozbudować `DataGridView` na kilka sposobów, aby tworzyć niestandardowe zachowania w aplikacjach. Można na przykład programowo określić własne algorytmy sortowania i można utworzyć własne typy komórek. Wygląd formantu można łatwo dostosować, `DataGridView` wybierając spośród kilku właściwości. Wiele typów magazynów danych może służyć jako źródło danych lub `DataGridView` kontrolka może działać bez powiązanego ze źródłem danych.  
  
 W tematach w tej sekcji opisano koncepcje i techniki, których można użyć do tworzenia `DataGridView` funkcji w aplikacjach.
