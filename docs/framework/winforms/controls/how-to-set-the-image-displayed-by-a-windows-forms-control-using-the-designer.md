# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Instrukcje: Ustawianie obrazu wyświetlanego przez kontrolkę Windows Forms przy użyciu narzędzia Projektant

Kilka kontrolek Windows Forms może wyświetlać obrazy. Obraz może być ikoną, która objaśnia przeznaczenie kontrolki, na przykład ikonę dysku na przycisku oznaczającego polecenie **Zapisz** . Lub ikona może być obrazem tła, aby nadać formantowi odpowiedni wygląd.

## <a name="to-set-the-image-displayed-by-a-control"></a>Aby ustawić obraz wyświetlany przez kontrolkę

1. W oknie **Właściwości** wybierz właściwość **obraz** lub **BackgroundImage** kontrolki, a następnie wybierz wielokropek (![przycisk wielokropka w programie Visual Studio](./media/visual-studio-ellipsis-button.png)), aby wyświetlić **wybrany zasób** . okno dialogowe.

2. Wybierz obraz, który chcesz wyświetlić.
