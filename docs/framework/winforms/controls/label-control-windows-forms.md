# <a name="label-control-windows-forms"></a>Label — Formant (Formularze systemu Windows)
> [!IMPORTANT]
> Formant <xref:System.Windows.Forms.ToolStripLabel> zamienia i dodaje funkcje do kontrolki <xref:System.Windows.Forms.Label>. Można użyć <xref:System.Windows.Forms.ToolStripLabel> z innymi nowymi kontrolkami, takimi jak <xref:System.Windows.Forms.ToolStripDropDown>. Niemniej jednak kontrolka <xref:System.Windows.Forms.Label> jest zachowywana na potrzeby zgodności z poprzednimi wersjami i w przyszłości.  
  
 Kontrolki <xref:System.Windows.Forms.Label> Windows Forms są używane do wyświetlania tekstu lub obrazów, które nie mogą być edytowane przez użytkownika. Służą one do identyfikowania obiektów w formularzu — w celu udostępnienia opisu działania określonej kontrolki, na przykład, lub wyświetlenia informacji w odpowiedzi na zdarzenie lub proces w czasie wykonywania w aplikacji. Ponieważ formant <xref:System.Windows.Forms.Label> nie może odebrać fokusu, można go również użyć do utworzenia kluczy dostępu dla innych kontrolek.
