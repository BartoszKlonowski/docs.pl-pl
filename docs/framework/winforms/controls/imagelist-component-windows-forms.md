# <a name="imagelist-component-windows-forms"></a>ImageList — Składnik (Formularze systemu Windows)
Formularze Windows `ImageList` składnik jest używany do przechowywania obrazów, które następnie mogą być wyświetlane przez formanty. Listy obrazów umożliwia pisanie kodu dla jednego, spójne katalog obrazów.
