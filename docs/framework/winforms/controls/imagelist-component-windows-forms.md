# <a name="imagelist-component-windows-forms"></a>ImageList — Składnik (Formularze systemu Windows)
Składnik `ImageList` Windows Forms jest używany do przechowywania obrazów, które są następnie wyświetlane przez kontrolki. Lista obrazów umożliwia pisanie kodu dla jednego spójnego wykazu obrazów.
