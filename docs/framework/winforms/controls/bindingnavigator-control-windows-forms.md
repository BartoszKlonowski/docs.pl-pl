# <a name="bindingnavigator-control-windows-forms"></a>BindingNavigator — Formant (Formularze systemu Windows)
`BindingNavigator` Formant jest nawigacji i manipulowania interfejsu użytkownika (UI) dla formantów, które są powiązane z danymi. `BindingNavigator` Formant umożliwia nawigowanie po i manipulowanie danymi w formularzu Windows.  
  
 W tematach w tej sekcji przedstawiono omówienie `BindingNavigator` kontroli i oferują instrukcje krok po kroku, jak za pomocą kontrolki nawigowanie po danych oraz przechodzenia między <xref:System.Data.DataSet>.

 Ogólne pojęcia związane z `BindingNavigator` formant, który umożliwia użytkownikom przechodzenie między elementami źródła danych.  
  
 [Instrukcje: Nawigowanie po danych za pomocą kontrolki BindingNavigator formularzy Windows Forms](how-to-navigate-data-with-the-windows-forms-bindingnavigator-control.md)  
 Zawiera kroki, aby powiązać `BindingNavigator` formant ze źródłem danych.  
  
 [Instrukcje: Przenieś za pomocą zestawu danych za pomocą kontrolki BindingNavigator formularzy Windows Forms](move-through-a-dataset-with-wf-bindingnavigator-control.md)  
 Demonstruje użycie `BindingNavigator` formantu do przechodzenia między rekordami w <xref:System.Data.DataSet>.
