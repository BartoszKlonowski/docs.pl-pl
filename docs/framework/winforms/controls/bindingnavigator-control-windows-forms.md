# <a name="bindingnavigator-control-windows-forms"></a>BindingNavigator — Formant (Formularze systemu Windows)
Formant `BindingNavigator` to Nawigacja i manipulowanie interfejsem użytkownika (UI) dla formantów, które są powiązane z danymi. Formant `BindingNavigator` umożliwia użytkownikom nawigowanie po danych w formularzu systemu Windows i manipulowanie nimi.  
  
 Tematy w tej sekcji zawierają omówienie kontrolki `BindingNavigator` i udostępniają instrukcje krok po kroku, jak używać kontrolki do nawigowania po danych i poruszania się przez <xref:System.Data.DataSet>.

 Wprowadza ogólne koncepcje `BindingNavigator` kontroli, co umożliwia użytkownikom przechodzenie przez elementy źródła danych.  
  
 [Instrukcje: nawigowanie po danych za pomocą kontrolki BindingNavigator formularzy Windows Forms](how-to-navigate-data-with-the-windows-forms-bindingnavigator-control.md)  
 Zawiera kroki, które należy wykonać, aby powiązać formant `BindingNavigator` ze źródłem danych.  
  
 [Instrukcje: przechodzenie między elementami DataSet za pomocą kontrolki BindingNavigator formularzy Windows Forms](move-through-a-dataset-with-wf-bindingnavigator-control.md)
