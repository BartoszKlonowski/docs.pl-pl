# <a name="how-to-resize-controls-on-windows-forms"></a>Instrukcje: Zmień rozmiar kontrolek na Windows Forms

Można zmienić rozmiar poszczególnych kontrolek i można zmienić rozmiar wielu kontrolek tego samego lub różnych rodzajów, takich <xref:System.Windows.Forms.Button> jak <xref:System.Windows.Forms.GroupBox> i kontrolek.

## <a name="to-resize-a-control"></a>Aby zmienić rozmiar kontrolki

W programie Visual Studio wybierz formant, który ma zostać zmieniony, i przeciągnij jeden z ośmiu uchwytów zmiany rozmiaru.

> [!NOTE]
> Zaznacz kontrolkę i naciśnij klawisze **strzałek** , przytrzymując klawisz **SHIFT** , aby zmienić rozmiar kontrolki o jeden piksel w danym momencie. Naciśnij klawisz **Strzałka w dół** lub **Strzałka w prawo** , przytrzymując wciśnięty klawisze **SHIFT** i **Ctrl** , aby zmienić rozmiar formantu w dużych przyrostach.

## <a name="to-resize-multiple-controls-on-a-form"></a>Aby zmienić rozmiar wielu kontrolek w formularzu

1. W programie Visual Studio, przytrzymaj wciśnięty klawisz **Ctrl** lub **SHIFT** i wybierz kontrolki, dla których chcesz zmienić rozmiar. Rozmiar pierwszej kontrolki wybieranej jest używany dla innych kontrolek.

2. W menu **Format** wybierz opcję **Wyrównaj rozmiar**i wybierz jedną z czterech opcji. Pierwsze trzy polecenia zmieniają wymiary formantów, aby były zgodne z pierwszą wybraną kontrolką.
