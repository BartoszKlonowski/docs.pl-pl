# <a name="how-to-resize-controls-on-windows-forms"></a>Porady: zmienianie rozmiaru formantów na formularzach systemu Windows
Możesz zmienić rozmiar poszczególnych formantów i można zmienić rozmiar wielu formantów tego samego lub innego rodzaju, takich jak <xref:System.Windows.Forms.Button> i <xref:System.Windows.Forms.GroupBox> kontrolki.  
  
### <a name="to-resize-a-control"></a>Aby zmienić rozmiar formantu  
  
1.  Kliknij przycisk formantu można zmienić rozmiar i przeciągnij uchwyty osiem zmiany rozmiaru.  
  
    > [!NOTE]
    >  Wybierz kontrolkę i klawiszy strzałek, trzymając naciśnięty klawisz SHIFT, aby zmienić rozmiar formantu jeden piksel w czasie. Naciśnij klawisz Strzałka w dół lub w prawo, trzymając wciśnięty klawiszy SHIFT i CTRL, aby zmienić rozmiar formantu w większych przyrostach.  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>Aby zmienić rozmiar wielu formantów w formularzu  
  
1.  Naciśnij i przytrzymaj klawisz SHIFT lub CTRL i zaznacz formanty, które chcesz zmienić. Rozmiar pierwszego formantu, którą wybierzesz jest używany dla innych formantów.  
  
2.  Na **Format** menu, wybierz **Wyrównaj rozmiar**i wybierz jedną z czterech opcji. Pierwsze trzy polecenia zmienić wymiary formantów, aby dopasować wybrany pierwszy kontroli.
