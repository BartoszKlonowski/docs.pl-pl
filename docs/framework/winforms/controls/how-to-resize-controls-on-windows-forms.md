### <a name="to-resize-a-control"></a><span data-ttu-id="49fb2-104">Aby zmienić rozmiar kontrolki</span><span class="sxs-lookup"><span data-stu-id="49fb2-104">To resize a control</span></span>  
  
1.  <span data-ttu-id="49fb2-105">Kliknij formant, który można zmienić jego rozmiaru i przeciągnij jeden z uchwytów osiem zmiany rozmiaru.</span><span class="sxs-lookup"><span data-stu-id="49fb2-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="49fb2-106">Wybierz kontrolkę i klawiszy strzałek, przytrzymując naciśnięty klawisz SHIFT, aby zmienić rozmiar formantu o jeden piksel w danym momencie.</span><span class="sxs-lookup"><span data-stu-id="49fb2-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="49fb2-107">Naciśnij klawisz strzałki w dół lub w prawo, przytrzymując naciśnięty klawisz SHIFT i CTRL kluczy zmiany rozmiaru w większych przyrostach.</span><span class="sxs-lookup"><span data-stu-id="49fb2-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="49fb2-108">Aby zmienić rozmiar wielu formantów w formularzu</span><span class="sxs-lookup"><span data-stu-id="49fb2-108">To resize multiple controls on a form</span></span>

2.  <span data-ttu-id="49fb2-111">Na **Format** menu, wybierz **Wyrównaj rozmiar**i wybierz jedną z czterech opcji.</span><span class="sxs-lookup"><span data-stu-id="49fb2-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="49fb2-112">Pierwsze trzy polecenia Zmień wymiary formantów, aby dopasować wybrany pierwszy formant.</span><span class="sxs-lookup"><span data-stu-id="49fb2-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
