3.  <span data-ttu-id="35c3e-116">W **konstruktora elementu CellStyle** okno dialogowe, zdefiniuj styl przez ustawienie właściwości i użyj **Podgląd** okienko, aby uzgodnić wybrane opcje.</span><span class="sxs-lookup"><span data-stu-id="35c3e-116">In the **CellStyle Builder** dialog box, define the style by setting the properties, and use the **Preview** pane to confirm your choices.</span></span> <span data-ttu-id="35c3e-117">Style, które określisz są używane dla każdego wiersza wyświetlany w formancie, począwszy od drugiej.</span><span class="sxs-lookup"><span data-stu-id="35c3e-117">The styles you specify are used for every other row displayed in the control, starting with the second one.</span></span>  
  
4.  <span data-ttu-id="35c3e-118">Aby zdefiniować style dla pozostałych wierszy, powtórz kroki 2 i 3 przy użyciu <xref:System.Windows.Forms.DataGridView.RowsDefaultCellStyle%2A> właściwości.</span><span class="sxs-lookup"><span data-stu-id="35c3e-118">To define styles for the remaining rows, repeat steps 2 and 3 using the <xref:System.Windows.Forms.DataGridView.RowsDefaultCellStyle%2A> property.</span></span>
