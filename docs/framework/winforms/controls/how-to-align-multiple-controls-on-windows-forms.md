# <a name="how-to-align-multiple-controls-on-windows-forms"></a>Instrukcje: wyrównywanie kontrolek na formularzach systemu Windows

Aby znormalizować układ interfejsu użytkownika aplikacji opartej na systemie Windows, można umieścić grupy kontrolek za pomocą jednego polecenia.

## <a name="to-align-multiple-controls-on-a-form"></a>Aby wyrównać wiele kontrolek w formularzu

1. W programie Visual Studio Otwórz formularz zawierający kontrolki, które chcesz umieścić w **Projektant formularzy systemu Windows**.

2. Wybierz kontrolki, które chcesz wyrównać. Pierwszy wybrany formant to podstawowy formant, do którego pozostałe będą wyrównane.

3. W menu **Format** wybierz pozycję **Wyrównaj**, a następnie wybierz jeden z siedmiu dostępnych opcji.
