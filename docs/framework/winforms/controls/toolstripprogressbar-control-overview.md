# <a name="toolstripprogressbar-control-overview"></a>ToolStripProgressBar — Informacje o formancie
<xref:System.Windows.Forms.ToolStripProgressBar> Łączy rafting i renderowanie funkcje wszystkich <xref:System.Windows.Forms.ToolStrip> kontrolek z jego typowe funkcje śledzenia procesu. A <xref:System.Windows.Forms.ToolStripProgressBar> najbardziej zwykle jest hostowana przez <xref:System.Windows.Forms.StatusStrip>, a w mniejszym stopniu często przez <xref:System.Windows.Forms.ToolStrip>.  
  
 Mimo że <xref:System.Windows.Forms.ToolStripProgressBar> zastępuje i dodaje funkcjonalność do kontroli w poprzednich wersjach <xref:System.Windows.Forms.ToolStripProgressBar> został zachowany na potrzeby zgodności z poprzednimi wersjami i użycia w przyszłości w razie potrzeby.
