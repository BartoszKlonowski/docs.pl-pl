# <a name="how-to-copy-controls-between-windows-forms"></a>Instrukcje: kopiowanie kontrolek pomiędzy formularzami systemu Windows
Kontrolki mogą być kopiowane na tym samym formularzu, na innej formy w projekcie lub do Schowka do użycia w innych rozwiązaniach.  
  
### <a name="to-copy-a-control"></a>Aby skopiować kontrolki  
  
1. Wybierz kontrolkę i z **Edytuj** menu wybierz opcję **kopiowania**.  
  
 Teraz można wkleić ten formant do dowolnej formie, który akceptuje tego typu kontrolki. Ponadto formant został dodany do Schowka.
