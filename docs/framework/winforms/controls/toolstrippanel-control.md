# <a name="toolstrippanel-control"></a>ToolStripPanel — Formant
`ToolStripPanel` Control umożliwia udostępnianie poziomej lub pionowej miejsca w obszarze Narzędzia, gdy jest zadokowany i rozmieszczenie <xref:System.Windows.Forms.ToolStrip> kontrolki, jeśli nie potrzebujesz czterech paneli i środkowym z <xref:System.Windows.Forms.ToolStripContainer>.  
  
 W tematach w tej sekcji opisano pojęć i technik, które służą do tworzenia `ToolStripPanel` funkcji w aplikacjach.
