# <a name="linklabel-control-windows-forms"></a>LinkLabel — Formant (Formularze systemu Windows)
Formant `LinkLabel` Formularze systemu Windows umożliwia dodawanie łączy w stylu sieci Web do aplikacji Windows Forms. Można użyć `LinkLabel` formantu dla wszystkiego, <xref:System.Windows.Forms.Label> dla czego można użyć formantu; Można również ustawić część tekstu jako łącze do obiektu lub strony sieci Web.
