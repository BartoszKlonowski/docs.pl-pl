# <a name="trackbar-control-windows-forms"></a>TrackBar — Formant (Formularze systemu Windows)
Kontrolka `TrackBar` Windows Forms (niekiedy nazywana również kontrolką "Slider") służy do nawigowania w dużej ilości informacji lub wizualnego dostosowywania ustawienia liczbowego. Kontrolka `TrackBar` ma dwie części: pasek przewijania, znany również jako suwak i znaczniki. Kciuk jest częścią, którą można dostosować. Jego pozycja odpowiada właściwości <xref:System.Windows.Forms.TrackBar.Value%2A>. Znaczniki są wskaźnikami wizualizacji, które są odstępie w regularnych odstępach czasu. Pasek śledzenia przenosi się w przyrostach, które można określić i mogą być wyrównane w poziomie lub w pionie. Przykładem użycia paska śledzenia jest ustawienie szybkości migania kursora lub szybkości myszy.
