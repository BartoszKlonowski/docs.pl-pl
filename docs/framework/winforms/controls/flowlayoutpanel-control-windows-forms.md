# <a name="flowlayoutpanel-control-windows-forms"></a>FlowLayoutPanel — Formant (Formularze systemu Windows)
`FlowLayoutPanel` Kontroli rozmieszcza jego zawartość w kierunku przepływu pozioma lub pionowa. Jego zawartość można ich opakować z jednego wiersza do następnego lub jedną kolumnę do następnego. Alternatywnie jego zawartość może zostać obcięty zamiast opakowana.  
  
 Tematy w tej sekcji opisano pojęcia i techniki, które umożliwiają budowanie `FlowLayoutPanel` funkcje do aplikacji.
