# <a name="flowlayoutpanel-control-windows-forms"></a>FlowLayoutPanel — Formant (Formularze systemu Windows)
Formant `FlowLayoutPanel` rozmieszcza swoją zawartość w poziomym lub pionowym kierunku przepływu. Jego zawartość może być opakowana z jednego wiersza do następnego lub z jednej kolumny na następną. Alternatywnie jego zawartość może być przycinana zamiast opakowana.  
  
 W tematach w tej sekcji opisano koncepcje i techniki umożliwiające tworzenie `FlowLayoutPanel` funkcji w aplikacjach.

 Wprowadza ogólne koncepcje `FlowLayoutPanel` formantu, który umożliwia utworzenie układu, który przepływa w poziomie lub w pionie.
