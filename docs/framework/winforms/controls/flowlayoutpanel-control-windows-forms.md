# <a name="flowlayoutpanel-control-windows-forms"></a>FlowLayoutPanel — Formant (Formularze systemu Windows)
`FlowLayoutPanel` Kontroli rozmieszcza jego zawartość w kierunku przepływu poziomej lub pionowej. Jego zawartość może zostać zawinięty, jeden wiersz do następnego lub z jednej kolumny do następnego. Alternatywnie jego zawartość może zostać obcięty zamiast opakowana.  
  
 Tematy w tej sekcji opisano pojęcia i techniki, które pozwalają na tworzenie `FlowLayoutPanel` funkcji w aplikacjach.
