Wiele <xref:System.Windows.Forms.ToolStrip> formanty stosu poziomie czy w pionie, w zależności od <xref:System.Windows.Forms.ToolStripPanelRow.Orientation%2A> z <xref:System.Windows.Forms.ToolStripPanel>.
