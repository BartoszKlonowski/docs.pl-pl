# <a name="tablelayoutpanel-control-windows-forms"></a>TableLayoutPanel — Formant (Formularze systemu Windows)
Formant <xref:System.Windows.Forms.TableLayoutPanel> rozmieszcza swoją zawartość w siatce. Ponieważ układ jest wykonywany zarówno w czasie projektowania, jak i w czasie wykonywania, może zmieniać się dynamicznie w miarę zmiany środowiska aplikacji. Dzięki temu kontrolki w panelu mogą zmieniać rozmiar proporcjonalnie do rozmiaru, dzięki czemu może reagować na zmiany, takie jak zmiana rozmiaru kontrolki nadrzędnej lub długość tekstu zmieniana z powodu lokalizacji.
