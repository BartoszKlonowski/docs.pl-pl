### <a name="to-freeze-a-column-using-the-designer"></a><span data-ttu-id="b6470-115">Aby zablokować kolumnę przy użyciu narzędzia Projektant</span><span class="sxs-lookup"><span data-stu-id="b6470-115">To freeze a column using the designer</span></span>  
  
1. <span data-ttu-id="b6470-116">Kliknij symbol tagu inteligentnego (![symbol tagu inteligentnego](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) w prawym górnym rogu <xref:System.Windows.Forms.DataGridView> sterowania, a następnie wybierz **Edytowanie kolumn**.</span><span class="sxs-lookup"><span data-stu-id="b6470-116">Click the smart tag glyph (![Smart Tag Glyph](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control, and then select **Edit Columns**.</span></span>  
  
2. <span data-ttu-id="b6470-117">Wybierz kolumnę z **wybrane kolumny** listy.</span><span class="sxs-lookup"><span data-stu-id="b6470-117">Select a column from the **Selected Columns** list.</span></span>  
  
3. <span data-ttu-id="b6470-118">W **właściwości kolumny** siatki, ustaw <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> właściwość `true`.</span><span class="sxs-lookup"><span data-stu-id="b6470-118">In the **Column Properties** grid, set the <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> property to `true`.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="b6470-119">Możesz również zablokować kolumnę podczas dodawania go, wybierając **Frozen** pole w **Dodaj kolumnę** okno dialogowe.</span><span class="sxs-lookup"><span data-stu-id="b6470-119">You can also freeze a column when adding it by selecting the **Frozen** box in the **Add Column** dialog box.</span></span>
