# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>Porady: położenie elementu ToolStripItem w formancie ToolStrip
Można przenieść lub dodać <xref:System.Windows.Forms.ToolStripItem> do lewej lub prawej krawędzi <xref:System.Windows.Forms.ToolStrip>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>Aby przenieść lub Dodaj element ToolStripItem po lewej stronie ToolStrip  
  
1.  Ustaw <xref:System.Windows.Forms.ToolStripItemAlignment> właściwość <xref:System.Windows.Forms.ToolStripItem> do <xref:System.Windows.Forms.ToolStripItemAlignment.Left>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>Aby przenieść lub Dodaj element ToolStripItem z prawej strony ToolStrip  
  
1.  Ustaw <xref:System.Windows.Forms.ToolStripItemAlignment> właściwość <xref:System.Windows.Forms.ToolStripItem> do <xref:System.Windows.Forms.ToolStripItemAlignment.Right>.
