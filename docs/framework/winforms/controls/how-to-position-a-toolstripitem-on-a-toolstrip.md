# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>Instrukcje: położenie elementu ToolStripItem w kontrolce ToolStrip
Możesz przenieść lub dodać <xref:System.Windows.Forms.ToolStripItem> do lewej lub prawej strony <xref:System.Windows.Forms.ToolStrip>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>Aby przenieść lub Dodawanie elementu ToolStripItem po lewej stronie ToolStrip  
  
1. Ustaw <xref:System.Windows.Forms.ToolStripItemAlignment> właściwość <xref:System.Windows.Forms.ToolStripItem> do <xref:System.Windows.Forms.ToolStripItemAlignment.Left>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>Aby przenieść lub Dodawanie elementu ToolStripItem w prawą stronę ToolStrip  
  
1. Ustaw <xref:System.Windows.Forms.ToolStripItemAlignment> właściwość <xref:System.Windows.Forms.ToolStripItem> do <xref:System.Windows.Forms.ToolStripItemAlignment.Right>.
