<span data-ttu-id="8881a-103">Często pracując przy użyciu powiązania danych w formularzach Windows Forms, wielu formantów są powiązane z tego samego źródła danych.</span><span class="sxs-lookup"><span data-stu-id="8881a-103">Oftentimes when working with data binding in Windows Forms, multiple controls are bound to the same data source.</span></span> <span data-ttu-id="8881a-104">W niektórych przypadkach może być konieczne wykonać dodatkowe czynności, aby upewnić się, że powiązanych właściwości kontrolek pozostają zsynchronizowane ze sobą i źródła danych.</span><span class="sxs-lookup"><span data-stu-id="8881a-104">In some cases, it may be necessary to take extra steps to ensure that the bound properties of the controls remain synchronized with each other and the data source.</span></span> <span data-ttu-id="8881a-105">Te kroki są niezbędne w dwóch sytuacjach:</span><span class="sxs-lookup"><span data-stu-id="8881a-105">These steps are necessary in two situations:</span></span>
