# <a name="windows-forms-security"></a>Zabezpieczenia formularzy systemu Windows
Windows Forms funkcje modelu zabezpieczeń, który jest oparty na kodzie (poziomy zabezpieczeń są ustawiane dla kodu, niezależnie od tego, czy użytkownik uruchomił kod). Jest to uzupełnienie wszelkich schematów zabezpieczeń, które mogą znajdować się już w systemie komputerowym. Mogą one obejmować te w przeglądarce (takie jak zabezpieczenia oparte na strefach dostępne w programie Internet Explorer) lub systemu operacyjnego (na przykład zabezpieczenia oparte na poświadczeniach systemu Windows NT).
