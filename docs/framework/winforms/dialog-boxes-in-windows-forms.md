# <a name="dialog-boxes-in-windows-forms"></a>Okna dialogowe w formularzach systemu Windows
Okna dialogowe służą do współpracy z użytkownikiem i pobierania informacji. W prostych terminach okno dialogowe jest formularz z właściwością <xref:System.Windows.Forms.FormBorderStyle> wyliczeniem ustawionym na `FixedDialog`. Możesz utworzyć własne niestandardowe okna dialogowe przy użyciu Projektant formularzy systemu Windows w programie Visual Studio. Dodaj kontrolki, takie jak `Label`, `Textbox`i `Button`, aby dostosować okna dialogowe do konkretnych potrzeb. .NET Framework zawiera również wstępnie zdefiniowane okna dialogowe, takie jak **pliki otwierające** i komunikaty, które można dostosować do własnych aplikacji. Aby uzyskać więcej informacji, zobacz [kontrolki i składniki okna dialogowego](./controls/dialog-box-controls-and-components-windows-forms.md).
