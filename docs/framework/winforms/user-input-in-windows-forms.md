# <a name="user-input-in-windows-forms"></a>Dane użytkownika w formularzach systemu Windows
Windows Forms obejmuje model danych wejściowych użytkownika oparty na zdarzeniach, które są wywoływane podczas przetwarzania powiązanych komunikatów systemu Windows. Tematy w tej sekcji zawierają informacje o użytkownikach myszy i klawiatury, w tym przykłady kodu, które demonstrują sposób wykonywania określonych zadań.
