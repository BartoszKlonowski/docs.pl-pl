# <a name="using-a-pen-to-draw-lines-and-shapes"></a>Rysowanie linii i kształtów za pomocą pióra
Użyj interfejsu GDI + `Pen` obiektów, aby narysować segmenty linii, krzywych i przedstawiono kształtów. W tej sekcji *wiersza* odwołuje się do dowolnego z tych opcji, chyba że określono oznacza segment linii. Ustaw właściwości piórem do kontrolowania kolor, szerokość, wyrównania i styl linii za pomocą tego pióra.
