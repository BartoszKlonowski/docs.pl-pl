# <a name="about-gdi-managed-code"></a>Informacje o kodzie zarządzanym GDI+

GDI+ jest częścią systemu operacyjnego Windows, która zapewnia dwuwymiarową grafikę wektorową, tworzenie obrazów i typografię. Interfejs GDI+ ulepsza interfejs GDI (GDI dołączony do wcześniejszych wersji systemu Windows) przez dodanie nowych funkcji i zoptymalizowanie istniejących funkcji.

Interfejs klasy zarządzanej GDI+ (zestaw otok) jest częścią .NET Framework, środowiska do kompilowania, wdrażania i uruchamiania usług sieci Web XML i innych aplikacji.

Ta sekcja zawiera informacje o interfejsie API GDI+ dla programistów korzystających z kodu zarządzanego.
