# <a name="graphics-overview-windows-forms"></a>Przegląd grafiki (formularze systemu Windows)
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] jest graficzny interfejs urządzenia, umożliwiająca programistom do pisania aplikacji niezależnych od urządzenia. Usługi [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] są udostępniane przez zestaw klas zarządzanych.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Omówienie grafiki](../../../../docs/framework/winforms/advanced/overview-of-graphics.md)  
 Zawiera ogólne wprowadzenie do [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 [Trzy kategorie usług grafiki](../../../../docs/framework/winforms/advanced/three-categories-of-graphics-services.md)  
 W tym artykule opisano trzy kategorie, które tworzą programowania w języku [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 [Struktura interfejsu grafiki](../../../../docs/framework/winforms/advanced/structure-of-the-graphics-interface.md)  
 Opisuje interfejs zarządzanej klasy [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
## <a name="reference"></a>Tematy pomocy  
 <xref:System.Drawing>  
 Zapewnia dostęp do [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] funkcje graficzne podstawowe.  
  
 <xref:System.Drawing.Drawing2D>  
 Udostępnia zaawansowane dwuwymiarowa i wektorów funkcje graficzne.  
  
 <xref:System.Drawing.Imaging>  
 Udostępnia zaawansowane [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] funkcje obsługi obrazów.  
  
 <xref:System.Drawing.Text>  
 Udostępnia zaawansowane [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] typografii funkcji.  
  
 <xref:System.Drawing.Printing>  
 Udostępnia związane z drukowaniem usługi.  
  
 <xref:System.Windows.Forms.TextRenderer>  
 Udostępnia [!INCLUDE[ndptecgdi](../../../../includes/ndptecgdi-md.md)] tekst rysowania i pomiaru funkcji.
