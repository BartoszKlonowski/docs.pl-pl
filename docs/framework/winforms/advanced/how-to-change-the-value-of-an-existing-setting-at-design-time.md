### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a><span data-ttu-id="6bea4-104">Aby zmienić wartość istniejącego ustawienia w czasie projektowania w języku C\#</span><span class="sxs-lookup"><span data-stu-id="6bea4-104">To Change the Value of an Existing Setting at Design Time in C\#</span></span>
  
1.  <span data-ttu-id="6bea4-105">W **Eksploratora rozwiązań**, rozwiń węzeł **właściwości** węzła projektu.</span><span class="sxs-lookup"><span data-stu-id="6bea4-105">In **Solution Explorer**, expand the **Properties** node of your project.</span></span>

3.  <span data-ttu-id="6bea4-108">Projektant ustawień zawiera ustawienie, dla którego chcesz zmienić wartość, a następnie wpisz nową wartość w kolumnie wartości.</span><span class="sxs-lookup"><span data-stu-id="6bea4-108">In the Settings designer, find the setting for which you want to change the value and type the new value in the Value column.</span></span>  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-visual-basic"></a><span data-ttu-id="6bea4-109">Aby zmienić wartość istniejącego ustawienia w czasie projektowania w języku Visual Basic</span><span class="sxs-lookup"><span data-stu-id="6bea4-109">To Change the Value of an Existing Setting at Design Time in Visual Basic</span></span>  
  
1.  <span data-ttu-id="6bea4-110">W **Eksploratora rozwiązań**, kliknij prawym przyciskiem myszy węzeł projektu i wybierz polecenie **właściwości**.</span><span class="sxs-lookup"><span data-stu-id="6bea4-110">In **Solution Explorer**, right-click your project node and choose **Properties**.</span></span>  
  
2.  <span data-ttu-id="6bea4-111">W **właściwości** wybierz opcję **ustawienia** kartę.</span><span class="sxs-lookup"><span data-stu-id="6bea4-111">In the **Properties** page, select the **Settings** tab.</span></span>  
  
3.  <span data-ttu-id="6bea4-112">Projektant ustawień zawiera ustawienie, dla którego chcesz zmienić wartość, a następnie wpisz nową wartość w kolumnie wartości.</span><span class="sxs-lookup"><span data-stu-id="6bea4-112">In the Settings designer, find the setting for which you want to change the value and type the new value in the Value column.</span></span>
