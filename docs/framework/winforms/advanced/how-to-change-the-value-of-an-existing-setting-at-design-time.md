### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a><span data-ttu-id="c21f3-104">Aby zmienić wartość istniejącego ustawienia w czasie projektowania w języku C#</span><span class="sxs-lookup"><span data-stu-id="c21f3-104">To Change the Value of an Existing Setting at Design Time in C#</span></span>  
  
1.  <span data-ttu-id="c21f3-105">W **Eksploratora rozwiązań**, rozwiń węzeł **właściwości** węzła projektu.</span><span class="sxs-lookup"><span data-stu-id="c21f3-105">In **Solution Explorer**, expand the **Properties** node of your project.</span></span>

3.  <span data-ttu-id="c21f3-108">W Projektancie ustawienia znaleźć ustawienia, dla którego chcesz zmienić wartość i wpisz nową wartość w kolumnie wartość.</span><span class="sxs-lookup"><span data-stu-id="c21f3-108">In the Settings designer, find the setting for which you want to change the value and type the new value in the Value column.</span></span>

1.  <span data-ttu-id="c21f3-110">W **Eksploratora rozwiązań**, kliknij prawym przyciskiem myszy węzeł projektu i wybierz **właściwości**.</span><span class="sxs-lookup"><span data-stu-id="c21f3-110">In **Solution Explorer**, right-click your project node and choose **Properties**.</span></span>

3.  <span data-ttu-id="c21f3-112">W Projektancie ustawienia znaleźć ustawienia, dla którego chcesz zmienić wartość i wpisz nową wartość w kolumnie wartość.</span><span class="sxs-lookup"><span data-stu-id="c21f3-112">In the Settings designer, find the setting for which you want to change the value and type the new value in the Value column.</span></span>
