# <a name="using-a-brush-to-fill-shapes"></a>Używanie pędzla do wypełniania kształtów
GDI + <xref:System.Drawing.Brush> obiekt jest używany do wypełniania wewnątrz kształtu zamkniętego. GDI + definiuje kilka style wypełnienia: jednolitego koloru, wzorem kreskowanym, obraz tekstury i kolor gradientu.
