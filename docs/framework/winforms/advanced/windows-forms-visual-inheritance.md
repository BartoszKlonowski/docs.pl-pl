# <a name="windows-forms-visual-inheritance"></a>Formularze systemu Windows — dziedziczenie Visual
Czasami możesz zdecydować, że projekt wywołuje formularz podobny do tego, który został utworzony w poprzednim projekcie. Można też utworzyć podstawową formę z ustawieniami, takimi jak znak wodny lub określony układ kontroli, który będzie następnie używany ponownie w projekcie, z każdą iteracją zawierającą modyfikacje oryginalnego szablonu formularza. Dziedziczenie formularzy umożliwia tworzenie formularza podstawowego, a następnie dziedziczenie i dokonywanie modyfikacji oraz zachowanie wszelkich potrzebnych oryginalnych ustawień.  
  
 Można tworzyć formularze klasy pochodnej programowo lub za pomocą selektora dziedziczenia wizualnego.
