# <a name="how-to-create-vertical-text"></a>Instrukcje: Tworzenie pionowego tekstu
Możesz użyć <xref:System.Drawing.StringFormat> obiektu w celu określenia, czy tekst być rysowany pionowo, a nie w poziomie.
