# <a name="install-the-net-framework-on-windows-xp-and-windows-server-2003"></a>Instalowanie programu .NET Framework w systemach Windows XP i Windows Server 2003

> [!NOTE]
> System Windows XP nie jest już obsługiwany przez firmę Microsoft. Zalecamy uaktualnienie do systemu Windows 10, który jest obsługiwany i zawiera najnowszą wersję programu .NET Framework. Ten dokument jest dostarczany wyłącznie jako pomocny przewodnik rozwiązywania problemów.

Program .NET Framework jest wymagany do uruchamiania wielu aplikacji w systemie Windows. Aby go zainstalować, można użyć następujących instrukcji. Być może dotarłeś na tej stronie po próbie uruchomienia aplikacji i wyświetleniu następującego okna dialogowego na komputerze.

![Nie można uruchomić tej aplikacji](./media/this-application-could-not-be-started.png)

Te instrukcje pomogą Ci zainstalować potrzebne wersje programu .NET Framework. [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) jest najnowszą wersją. Nie jest obsługiwana w systemach Windows XP i Windows Server 2003. Jest dołączony do [systemu Windows 10 May 2019 Update](https://support.microsoft.com/help/4028685/windows-10-get-the-update).

## <a name="net-framework-403"></a>.NET Framework 4.0.3

[.NET Framework 4.0.3](https://www.microsoft.com/download/details.aspx?id=29053) to najnowsza obsługiwana wersja programu .NET Framework w systemach Windows XP i Windows Server 2003. .NET Framework 4.0.3 wymaga, aby [.NET Framework 4](https://dotnet.microsoft.com/download/dotnet-framework/net40) jest zainstalowany jako pierwszy. Obie te wersje programu .NET Framework nie są już obsługiwane przez firmę Microsoft.

## <a name="net-framework-4"></a>Program .NET Framework 4

Program [.NET Framework 4](https://dotnet.microsoft.com/download/dotnet-framework/net40) można zainstalować w systemie Windows XP. Nie jest już obsługiwany przez firmę Microsoft.

## <a name="net-framework-35"></a>Program .NET Framework 3,5

Program [.NET Framework 3.5](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1) można zainstalować w systemie Windows XP.

.NET Framework 3.5 może służyć do uruchamiania aplikacji utworzonych dla platformy .NET Framework 1.0 do 3.5.
