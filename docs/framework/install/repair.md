# <a name="repair-the-net-framework"></a>Naprawianie programu .NET Framework

W niektórych sytuacjach instalacja platformy .NET Framework może ulec uszkodzeniu i wymagać naprawy. Może tak być w przypadku, gdy aplikacja ulega awarii zaraz po próbie uruchomienia lub jeśli nie można zainstalować nowsze wersje programu .NET Framework.

Instalację programu .NET Framework można naprawić za pomocą [narzędzia .NET Framework Repair Tool](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe).

Jeśli aplikacja nadal nie uruchamia się po naprawie programu .NET Framework, aplikacja może mieć problem. W takim przypadku należy skontaktować się z wydawcą aplikacji.
