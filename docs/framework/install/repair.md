# <a name="repair-the-net-framework"></a>Naprawianie programu .NET Framework

W niektórych sytuacjach instalacja .NET Framework może ulec uszkodzeniu i wymagać naprawy. Może się tak zdarzyć w przypadku, gdy aplikacja ulegnie awarii bezpośrednio po próbie jej uruchomienia lub jeśli nie można zainstalować nowszych wersji .NET Framework.

Instalację .NET Framework można naprawić za pomocą narzędzia do [naprawy .NET Framework](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe).

Jeśli aplikacja nadal nie uruchamia się po naprawieniu .NET Framework, aplikacja może mieć problem. W takim przypadku należy skontaktować się z wydawcą aplikacji.
