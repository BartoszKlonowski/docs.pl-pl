# <a name="install-the-net-framework-on-windows-vista-and-windows-server-2008"></a>Instalowanie programu .NET Framework w systemach Windows Vista i Windows Server 2008

Program .NET Framework jest wymagany do uruchamiania wielu aplikacji w systemie Windows. Aby go zainstalować, można użyć następujących instrukcji. Być może dotarłeś na tej stronie po próbie uruchomienia aplikacji i wyświetleniu następującego okna dialogowego na komputerze.

![Nie można uruchomić tej aplikacji](./media/this-application-could-not-be-started.png)

Te instrukcje pomogą Ci zainstalować potrzebne wersje programu .NET Framework. [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) jest najnowszą wersją. Nie jest obsługiwana w systemach Windows Vista i Windows Server 2008. Jest dołączony do [systemu Windows 10 May 2019 Update](https://support.microsoft.com/help/4028685/windows-10-get-the-update).

## <a name="net-framework-46"></a>Program .NET Framework 4.6

[.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) to najnowsza obsługiwana wersja programu .NET Framework w systemach Windows Vista i Windows Server 2008.

Program [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) obsługuje aplikacje utworzone dla platformy .NET Framework od 4.0 do 4.6.

## <a name="net-framework-35"></a>Program .NET Framework 3,5

Program [.NET Framework 3.5](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1) można zainstalować w systemie Windows Vista.

Program .NET Framework 3.5 obsługuje aplikacje utworzone dla platformy .NET Framework od 1.0 do 3.5.
