# <a name="install-the-net-framework-on-windows-8-and-windows-server-2012"></a>Instalowanie programu .NET Framework w systemach Windows 8 i Windows Server 2012

Program .NET Framework jest wymagany do uruchamiania wielu aplikacji w systemie Windows. Aby go zainstalować, można użyć następujących instrukcji. Być może dotarłeś na tej stronie po próbie uruchomienia aplikacji i wyświetleniu następującego okna dialogowego na komputerze.

![Nie można uruchomić tej aplikacji](./media/this-application-could-not-be-started.png)

Te instrukcje pomogą Ci zainstalować potrzebne wersje programu .NET Framework. [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) jest najnowszą wersją. Jest obsługiwany w systemie Windows Server 2012, ale nie jest obsługiwany w systemie Windows 8. Jest on dołączony do [windows 10 May 2019 Update](https://support.microsoft.com/help/4028685/windows-10-get-the-update)).

## <a name="net-framework-48"></a> .NET Framework 4.8

> [!div class="button"]
> [Pobierz program .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)

Program .NET Framework 4.8 jest obsługiwany w systemie Windows Server 2012. Nie jest obsługiwany w systemie Windows 8.

[.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) może służyć do uruchamiania aplikacji utworzonych dla platformy .NET Framework 4.0 lub nowszych.

## <a name="net-framework-46"></a>Program .NET Framework 4.6

[.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) to najnowsza obsługiwana wersja programu .NET Framework w systemie Windows 8.

Program .NET Framework 4.6 obsługuje aplikacje utworzone dla platformy .NET Framework od 4.0 do 4.6.

## <a name="net-framework-35"></a>Program .NET Framework 3,5

Postępuj zgodnie z instrukcjami, aby zainstalować [program .NET Framework 3.5 w systemie Windows 8 i 10](dotnet-35-windows-10.md).

Program .NET Framework 3.5 obsługuje aplikacje utworzone dla platformy .NET Framework od 1.0 do 3.5.

## <a name="help"></a>Pomoc

Aby [uzyskać pomoc, można skontaktować się z firmą Microsoft,](mailto:dotnet-install-help@service.microsoft.com?subject=Install-Help) jeśli nie można uzyskać zainstalowanej poprawnej wersji programu .NET Framework.
