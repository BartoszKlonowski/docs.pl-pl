# <a name="deploying-net-framework-applications-with-docker"></a>Wdrażanie aplikacji .NET Framework za pomocą platformy Docker

Możesz wdrożyć aplikację .NET Framework za pomocą platformy Docker, przy użyciu kontenerów Windows. Możesz dowiedzieć się wymagania dotyczące korzystania z [kontenery Windows](/virtualization/windowscontainers/about/) oraz sposób [wprowadzenie do platformy Docker for Windows](https://docs.docker.com/docker-for-windows/). 

Możesz rozpocząć od [uruchamianie aplikacji konsolowej przy użyciu rozwiązania Docker](console.md).
Do uruchamiania aplikacji sieci web na platformie Docker, możesz przeczytać [aplikacji ASP.NET MVC na platformie Docker](/aspnet/mvc/overview/deployment/docker-aspnetmvc).
