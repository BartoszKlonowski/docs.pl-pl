# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback — Metoda
Wykonuje określoną funkcję w apartamencie.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pFunc`  
 [in] Wskaźnik do funkcję, która ma zostać wykonana w obrębie apartamencie.  
  
 `pData`  
 [in] Wskaźnik do argumentu funkcji.
