# <a name="igchostcontrol-interface"></a>IGCHostControl — Interfejs
Zapewnia metodę, która umożliwia modułowi wyrzucania elementów bezużytecznych żądanie zmiany limitów pamięci wirtualnej przez hosta.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[RequestVirtualMemLimit, metoda](igchostcontrol-requestvirtualmemlimit-method.md)|Żąda od hosta zmiany limitów pamięci wirtualnej.|
