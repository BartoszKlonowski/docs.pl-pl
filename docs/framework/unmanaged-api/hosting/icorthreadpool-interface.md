# <a name="icorthreadpool-interface"></a>ICorThreadpool — Interfejs
Udostępnia metody, aby uzyskać dostęp do puli wątków.  
  
> [!NOTE]
>  Ten interfejs jest zastrzeżone wyłącznie do użytku wewnętrznego.

|Metoda|Opis|  
|------------|-----------------|  
|[CorRegisterWaitForSingleObject, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-corregisterwaitforsingleobject-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|  
|[CorUnregisterWait, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-corunregisterwait-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|  
|[CorQueueUserWorkItem, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-corqueueuserworkitem-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|  
|[CorCreateTimer, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-corcreatetimer-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|  
|[CorChangeTimer, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-corchangetimer-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|  
|[CorDeleteTimer, metoda](../../../../docs/framework/unmanaged-api/hosting/icorthreadpool-cordeletetimer-method.md)|Zastrzeżone wyłącznie do użytku wewnętrznego.|
