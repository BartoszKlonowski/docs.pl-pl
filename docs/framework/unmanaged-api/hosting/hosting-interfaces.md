# <a name="hosting-interfaces"></a>Hosting — Interfejsy
W tej sekcji opisano interfejsów, które niezarządzanych hostów można użyć do integracji środowisko uruchomieniowe języka wspólnego (CLR) w swoich aplikacjach.  
  
 Interfejsy hostingu środowiska .NET Framework w wersji 2.0 zastępują interfejsy .NET Framework w wersji 1.0, 1.1. Brak istotnych różnic między dwoma zestawami interfejsów. Mieszanie ich może spowodować nieoczekiwane zachowanie i nie jest zalecane.  
  
 Wersje programu .NET Framework 3.0 i 3.5 używać interfejsów hostingu programu .NET Framework w wersji 2.0, a nie znajdą się wszystkie funkcje hostingu.  
  
 [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)] Interfejsy hostingu zastępują interfejsy .NET Framework 2.0.
