# <a name="ihostsemaphore-interface"></a>IHostSemaphore — Interfejs
Reprezentuje implementację semafora na hoście dla wątków.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[ReleaseSemaphore, metoda](../../../../docs/framework/unmanaged-api/hosting/ihostsemaphore-releasesemaphore-method.md)|Zwiększa liczbę wystąpień bieżącego `IHostSemaphore` o określoną wartość.|  
|[Wait, metoda](../../../../docs/framework/unmanaged-api/hosting/ihostsemaphore-wait-method.md)|Powoduje, że bieżące wystąpienie `IHostSemaphore` zaczeka, aż będzie własnością lub określony czas upłynął.|
