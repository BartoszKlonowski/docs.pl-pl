# <a name="ihostautoevent-interface"></a>IHostAutoEvent — Interfejs

Przedstawia reprezentację implementacji zdarzenia resetowania hosta.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Set, metoda](ihostautoevent-set-method.md)|Ustawia bieżące `IHostAutoEvent` wystąpienie na sygnał.|  
|[Wait — Metoda](ihostautoevent-wait-method.md)|Powoduje, że bieżące `IHostAutoEvent` wystąpienie czeka, aż do momentu, gdy zdarzenie jest własnością lub upłynie określony czas.|
