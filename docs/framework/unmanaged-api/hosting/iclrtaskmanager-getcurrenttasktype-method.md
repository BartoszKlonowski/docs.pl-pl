# <a name="iclrtaskmanagergetcurrenttasktype-method"></a>ICLRTaskManager::GetCurrentTaskType — Metoda
Pobiera typ zadania, które jest aktualnie wykonywane.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetCurrentTaskType(  
    [out] ETaskType *pTaskType  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pTaskType`  
 określoną Wskaźnik do wartości wyliczenia [ETaskType —](../../../../docs/framework/unmanaged-api/hosting/etasktype-enumeration.md) , który wskazuje typ zadania, które jest aktualnie wykonywane.
