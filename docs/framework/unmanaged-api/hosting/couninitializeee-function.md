# <a name="couninitializeee-function"></a>CoUninitializeEE — Funkcja
`CoUninitializeEE` jest przestarzała i nie zapewnia żadnych funkcji.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
void CoUninitializeEE (  
    BOOL fFlags  
);  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie można zwolnić aparatu wykonywania środowiska uruchomieniowego języka wspólnego z procesu. Aby zamknąć wywołanie aparatu wykonywania [CorExitProcess —](../../../../docs/framework/unmanaged-api/hosting/corexitprocess-function.md).
