# <a name="couninitializeee-function"></a>CoUninitializeEE — Funkcja
`CoUninitializeEE` jest przestarzały i nie zawiera żadnych funkcji.  
  
## <a name="syntax"></a>Składnia  
  
```  
void CoUninitializeEE (  
    BOOL fFlags  
);  
```  
  
## <a name="remarks"></a>Uwagi  
 Aparat wykonywania środowiska uruchomieniowego języka wspólnego nie może być zwolnione z procesu. Zamknąć wywołanie aparatu wykonywania [corexitprocess —](../../../../docs/framework/unmanaged-api/hosting/corexitprocess-function.md).
