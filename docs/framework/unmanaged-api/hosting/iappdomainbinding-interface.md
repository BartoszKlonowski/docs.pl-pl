# <a name="iappdomainbinding-interface"></a>IAppDomainBinding — Interfejs

Zapewnia metodę, która jest wywoływana przez środowisko uruchomieniowe języka wspólnego (CLR) do powiadamiania aplikacji hosta o utworzeniu domeny aplikacji.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[OnAppDomain, metoda](iappdomainbinding-onappdomain-method.md)|Wywoływane przez środowisko uruchomieniowe języka wspólnego (CLR) do powiadamiania hosta o utworzeniu domeny aplikacji.|
