# <a name="ihosttaskstart-method"></a>IHostTask::Start — Metoda
Żąda, aby Host przeniesieł zadanie reprezentowane przez bieżące wystąpienie [IHostTask](../../../../docs/framework/unmanaged-api/hosting/ihosttask-interface.md) od zawieszenia do stanu na żywo, w którym można wykonać kod.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT Start ();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
  
|HRESULT|Opis|  
|-------------|-----------------|  
|S_OK|Pomyślnie zwrócono.|  
|E_FAIL|Wystąpił nieznany błąd krytyczny. Gdy metoda zwraca wartość E_FAIL, środowisko uruchomieniowe języka wspólnego (CLR) nie jest już możliwe do użycia w procesie. Kolejne wywołania metod hostingu zwracają HOST_E_CLRNOTAVAILABLE.|  
  
## <a name="remarks"></a>Uwagi  
 `Start` zawsze zwraca wartość HRESULT elementu S_OK, z wyjątkiem przypadków, w których wystąpił błąd krytyczny.
