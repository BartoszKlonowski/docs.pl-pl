# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap — Metoda

## <a name="syntax"></a>Składnia  
  
```  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppv`  
 [out] Wskaźnik do obiektu do odkodowania.
