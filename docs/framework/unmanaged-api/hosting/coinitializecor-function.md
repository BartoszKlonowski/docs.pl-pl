# <a name="coinitializecor-function"></a>CoInitializeCor — Funkcja
`CoInitializeCor` jest przestarzała.  
  
## <a name="syntax"></a>Składnia  
  
```  
STDAPI CoInitializeCor (  
    DWORD fFlags  
);  
```
