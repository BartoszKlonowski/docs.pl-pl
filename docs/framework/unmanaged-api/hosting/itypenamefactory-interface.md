# <a name="itypenamefactory-interface"></a>ITypeNameFactory — Interfejs

Dostarcza metody służące do dekonstrukcji nazwy typu.  
  
 Ten interfejs obsługuje infrastrukturę .NET Framework i nie jest przeznaczony do użycia bezpośrednio w kodzie.
