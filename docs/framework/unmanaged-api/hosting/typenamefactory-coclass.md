# <a name="typenamefactory-coclass"></a>TypeNameFactory — Klasa coclass
Interfejs zarządzania deconstruction nazwy typu.  
  
## <a name="syntax"></a>Składnia  
  
```  
coclass TypeNameFactory {  
    [default] interface ITypeNameFactory;  
};  
```
