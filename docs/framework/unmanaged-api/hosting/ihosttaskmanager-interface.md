 <span data-ttu-id="98b45-144">`IHostTaskManager` Umożliwia środowisku CLR utworzyć i zarządzać zadaniami, aby zapewnić punkty zaczepienia dla hosta podjąć działania, gdy formant przesyła z zarządzanego do kodu niezarządzanego i na odwrót i określić pewne akcje, host może i nie można przyjąć podczas wykonywania kodu.</span><span class="sxs-lookup"><span data-stu-id="98b45-144">`IHostTaskManager` allows the CLR to create and manage tasks, to provide hooks for the host to take action when control transfers from managed to unmanaged code and vice versa, and to specify certain actions the host can and cannot take during code execution.</span></span>
