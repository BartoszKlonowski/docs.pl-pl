# <a name="idebuggerinfo-interface"></a>IDebuggerInfo — Interfejs

Zapewnia metody uzyskiwania informacji o stanie usług debugowania.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IsDebuggerAttached, metoda](idebuggerinfo-isdebuggerattached-method.md)|Pobiera wartość wskazującą, czy zarządzany debuger jest dołączony do tego procesu.|
