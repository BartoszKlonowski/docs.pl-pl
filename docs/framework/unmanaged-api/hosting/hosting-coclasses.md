# <a name="hosting-coclasses"></a>Współklasy hostingu
W tej sekcji opisano klasy coclass niezarządzanego interfejsu API, które są używane przez interfejs API hostingu.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [CLRRuntimeHost, klasa coclass](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Udostępnia interfejsy umożliwiające zarządzanie wykonywaniem kodu przez środowisko uruchomieniowe.  
  
 [ComCallUnmarshal, klasa coclass](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Udostępnia interfejsy służące do zarządzania kierowaniem wskaźników interfejsu.  
  
 [CorRuntimeHost, klasa coclass](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Udostępnia interfejsy do zarządzania aplikacjami, które są wykonywane przez środowisko uruchomieniowe języka wspólnego.  
  
 [TypeNameFactory, klasa coclass](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Udostępnia interfejs do zarządzania dekonstrukcja nazwy typu.
