# <a name="icorprofilerinfogetthreadinfo-method"></a>ICorProfilerInfo::GetThreadInfo — Metoda
Pobiera tożsamość bieżącego wątku Win32 dla określonego wątku.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetThreadInfo(  
    [in]  ThreadID threadId,  
    [out] DWORD    *pdwWin32ThreadId);  
```  
  
#### <a name="parameters"></a>Parametry  
 `threadId`  
 [in] Identyfikator wątku, który można pobrać bieżącego identyfikatora Win32  
  
 `pdwWin32ThreadId`  
 [out] Wskaźnik do bieżącego wątku Win32 wątku określonego identyfikatora.
