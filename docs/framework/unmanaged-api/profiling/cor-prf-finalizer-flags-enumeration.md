# <a name="cor_prf_finalizer_flags-enumeration"></a>COR_PRF_FINALIZER_FLAGS — Wyliczenie
Opisuje finalizator dla obiektu.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
typedef enum {  
    COR_PRF_FINALIZER_CRITICAL = 0x1  
} COR_PRF_FINALIZER_FLAGS;  
```  
  
## <a name="members"></a>Elementy członkowskie  
  
|Element członkowski|Opis|  
|------------|-----------------|  
|`COR_PRF_FINALIZER_CRITICAL`|Finalizator jest krytyczny.|  
  
## <a name="remarks"></a>Uwagi  
 Wyliczenie `COR_PRF_FINALIZER_FLAGS` jest używane przez metodę [ICorProfilerCallback2:: FinalizeableObjectQueued —](icorprofilercallback2-finalizeableobjectqueued-method.md) do opisywania finalizatora dla obiektu.
