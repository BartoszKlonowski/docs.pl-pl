# <a name="icorprofilercallbackruntimeresumestarted-method"></a>ICorProfilerCallback::RuntimeResumeStarted — Metoda

## <a name="syntax"></a>Składnia  
  
```  
HRESULT RuntimeResumeStarted();  
```
