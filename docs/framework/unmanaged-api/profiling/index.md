# <a name="profiling-unmanaged-api-reference"></a>Profilowanie (Niezarządzany wykaz interfejsów API)

Interfejs API profilowania umożliwia profiler do monitorowania wykonywania programu przez środowisko uruchomieniowe języka wspólnego (CLR).

 [Konfigurowanie środowiska profilowania](setting-up-a-profiling-environment.md) W tym artykule opisano kroki, które należy wykonać w celu profilu aplikacji .NET Framework.

 [Profilerzy CLR i aplikacje ze Sklepu Windows](clr-profilers-and-windows-store-apps.md) W tym artykule omówiono sposób przenoszenia narzędzi diagnostycznych, które korzystają z interfejsu API profilowania programu CLR, aby pomyślnie współpracować z aplikacjami ze Sklepu Windows.

 [CORPROF_E_UNSUPPORTED_CALL_SEQUENCE HRESULT](corprof-e-unsupported-call-sequence-hresult.md) Dokumenty warunki, w których wywołanie metody zwraca `CORPROF_E_UNSUPPORTED_CALL_SEQUENCE` HRESULT.

 [Profilowanie globalnych funkcji statycznych](profiling-global-static-functions.md) W tym artykule opisano niezarządzane globalne funkcje statyczne używane przez interfejs API profilowania.
