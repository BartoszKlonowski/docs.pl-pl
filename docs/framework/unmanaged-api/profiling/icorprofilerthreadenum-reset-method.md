# <a name="icorprofilerthreadenumreset-method"></a>ICorProfilerThreadEnum::Reset — Metoda
Moves the enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT Reset();  
```
