# <a name="icorprofilercallback7-interface"></a>Interfejs ICorProfilerCallback7
[Obsługiwane w programie .NET Framework 4.6.1 i nowszych wersjach]  
  
 Podklasa [ICorProfilerCallback6](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback6-interface.md) zapewnia metody wywołania zwrotnego, która środowisko uruchomieniowe języka wspólnego używa do powiadamiania profilera zaktualizowaniu strumienia symbol skojarzone z modułu w pamięci.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[ModuleInMemorySymbolsUpdated — metoda](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback7-moduleinmemorysymbolsupdated-method.md)|Powiadamia profilera zaktualizowaniu strumienia symbol skojarzone z modułu w pamięci.|
