# <a name="icorprofilerinfo10suspendruntime-method"></a>ICorProfilerInfo10:: SuspendRuntime, Metoda
  
 Wstrzymuje środowisko uruchomieniowe bez wykonywania operacji GC.   
  
## <a name="syntax"></a>Składnia  
  
```cpp
HRESULT SuspendRuntime();
```
