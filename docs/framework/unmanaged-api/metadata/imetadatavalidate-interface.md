# <a name="imetadatavalidate-interface"></a>IMetaDataValidate — Interfejs
Zapewnia metody weryfikacji podpisów metadanych.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[ValidateMetaData — Metoda](imetadatavalidate-validatemetadata-method.md)|Sprawdza poprawność podpisów metadanych obiektów w bieżącym zakresie metadanych.|  
|[ValidatorInit — Metoda](imetadatavalidate-validatorinit-method.md)|Ustawia flagę określającą typ modułu w bieżącym zakresie metadanych i rejestruje określoną metodę wywołania zwrotnego dla błędów walidacji.|
