# <a name="imetadataemitsetfieldmarshal-method"></a>IMetaDataEmit::SetFieldMarshal — Metoda
Ustawia PInvoke przekazywanie informacji dla parametru pola, zwracany — metoda lub metody odwołuje się określony token.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT SetFieldMarshal (  
    [in]  mdToken          tk,   
    [in]  PCCOR_SIGNATURE  pvNativeType,   
    [in]  ULONG            cbNativeType   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `tk`  
 [in] Token dla elementu docelowego danych. Jest to `mdFieldDef` lub `mdParamDef` tokenu.  
  
 `pvNativeType`  
 [in] Podpis dla typu niezarządzanego.  
  
 `cbNativeType`  
 [in] Liczba bajtów w `pvNativeType`.
