# <a name="imetadatainfo-interface"></a>IMetaDataInfo — Interfejs
Udostępnia metody, która pobiera informacje o mapowaniu metadanych z pliku na dysku do pamięci.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetFileMapping, metoda](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Pobiera region pamięci zamapowanego pliku, a Typ mapowania.|
