# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile — Metoda
Generuje plik bazy kodu, który zawiera bazę kodu, która jest aktualnie załadowana do tego interfejsu [ICeeGen](iceegen-interface.md) .  
  
 Ta metoda jest przestarzała i nie powinna być używana.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GenerateCeeFile ();  
```  
  
## <a name="parameters"></a>Parametry  
 `GenerateCeeFile`nie przyjmuje żadnych parametrów.
