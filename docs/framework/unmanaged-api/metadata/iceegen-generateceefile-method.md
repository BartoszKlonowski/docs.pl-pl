# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile — Metoda
Generuje plik bazowej kodu, który zawiera kod podstawowy załadowanych obecnie do tego [ICeeGen](../../../../docs/framework/unmanaged-api/metadata/iceegen-interface.md) interfejsu.  
  
 Ta metoda jest przestarzała i nie powinna być używana.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GenerateCeeFile ();  
```  
  
#### <a name="parameters"></a>Parametry  
 `GenerateCeeFile`nie przyjmuje żadnych parametrów.
