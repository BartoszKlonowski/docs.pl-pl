# <a name="iceegengetimaptokeniface-method"></a>ICeeGen::GetIMapTokenIface — Metoda
Pobiera interfejs przywoływany przez określony token.  
  
 Ta metoda jest przestarzała i nie powinna być używana.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetIMapTokenIface (  
    [in, out] IUnknown   **pIMapToken  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pIMapToken`  
 [in. out] Token metadanych dla interfejsu, który ma zostać zwrócony.
