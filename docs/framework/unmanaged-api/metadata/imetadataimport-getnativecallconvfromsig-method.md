# <a name="imetadataimportgetnativecallconvfromsig-method"></a>IMetaDataImport::GetNativeCallConvFromSig — Metoda
Pobiera natywnego Konwencję wywoływania dla metody, która jest reprezentowana przez wskaźnik określoną sygnaturą.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetNativeCallConvFromSig (  
   [in]  void const  *pvSig,  
   [in]  ULONG       cbSig,  
   [out] ULONG       *pCallConv  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pvSig`  
 [in] Wskaźnik do metody do zwrócenia Konwencja wywoływania dla podpisu metadanych.  
  
 `cbSig`  
 [in] Wyrażony w bajtach rozmiar `pvSig`.  
  
 `pCallConv`  
 [out] Wskaźnik do natywną Konwencję wywoływania.
