# <a name="imetadataimportenummodulerefs-method"></a>IMetaDataImport::EnumModuleRefs — Metoda
Wylicza tokeny element ModuleRef, które reprezentują zaimportowanych modułów.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT EnumModuleRefs (  
   [in, out] HCORENUM     *phEnum,  
   [out]     mdModuleRef  rModuleRefs[],  
   [in]      ULONG        cMax,  
   [out]     ULONG        *pcModuleRefs  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `phEnum`  
 [w, out] Wskaźnik do modułu wyliczającego. Musi to być wartość NULL dla pierwsze wywołanie tej metody.  
  
 `rModuleRefs`  
 [out] Tablica używany do przechowywania tokenów element ModuleRef.  
  
 `cMax`  
 [in] Maksymalny rozmiar `rModuleRefs` tablicy.  
  
 `pcModuleRefs`  
 [out] Liczba tokenów element ModuleRef zwracane w `rModuleRefs`.  
  
## <a name="return-value"></a>Wartość zwracana  
  
|HRESULT|Opis|  
|-------------|-----------------|  
|`S_OK`|`EnumModuleRefs`zwrócona pomyślnie.|  
|`S_FALSE`|Nie ma żadnych tokenów do wyliczenia. W takim przypadku `pcModuleRefs` wynosi zero.|
