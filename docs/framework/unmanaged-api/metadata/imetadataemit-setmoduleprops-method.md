# <a name="imetadataemitsetmoduleprops-method"></a>IMetaDataEmit::SetModuleProps — Metoda
Aktualizuje odwołania do modułu zdefiniowanego przez wcześniejsze wywołanie [IMetaDataEmit::DefineModuleRef](../../../../docs/framework/unmanaged-api/metadata/imetadataemit-definemoduleref-method.md).  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT SetModuleProps (
    [in]  LPCWSTR     szName  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `szName`  
 [w] Nazwa modułu w Unicode. Jest to tylko nazwa pliku, a nie pełna nazwa ścieżki.
