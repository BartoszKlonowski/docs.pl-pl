# <a name="imetadataemitapplyeditandcontinue-method"></a>IMetaDataEmit::ApplyEditAndContinue — Metoda
Aktualizuje bieżący zakres zestawu ze zmianami wprowadzonymi w określonych metadanych.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT ApplyEditAndContinue (
    [in]  IUnknown    *pImport  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pImport`  
 \[w\] Wskaźnik do [IUnknown](/cpp/atl/iunknown) obiektu, który reprezentuje metadane delta z pliku pliku wykonywalnego przenośnego (PE).
  
 Metadane różnicowe to blok metadanych, który zawiera zmiany wprowadzone do kopii rzeczywistych metadanych modułu.
