# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken — Metoda
Usuwa określony token z bieżącego zakresu metadanych.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>Parametry  
 `tkObj`  
 [w] Token do usunięcia.
