# <a name="imetadatatablesgetguidheapsize-method"></a>IMetaDataTables::GetGuidHeapSize — Metoda
Pobiera rozmiar sterty identyfikatora GUID w bajtach.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetGuidHeapSize (  
    [out] ULONG   *pcbGuids  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pcbGuids`  
 określoną Wskaźnik do rozmiaru, w bajtach, sterty identyfikatora GUID.
