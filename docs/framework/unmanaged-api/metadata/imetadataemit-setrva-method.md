# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA — Metoda
Ustawia adres względny wirtualnego określonej metody.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,   
    [in]  ULONG        ulRVA   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `md`  
 [in] Token metody docelowej lub w implementacji metody.  
  
 `ulRVA`  
 [in] Adres obszaru kodu lub danych.
