# <a name="imetadataemitdefinemoduleref-method"></a>IMetaDataEmit::DefineModuleRef — Metoda
Tworzy podpis metadanych dla modułu o określonej nazwie.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT DefineModuleRef (
    [in]  LPCWSTR           szName,
    [out] mdModuleRef       *pmur
);  
```  
  
## <a name="parameters"></a>Parametry  
 `szName`  
 [w] Nazwa innego pliku metadanych, zazwyczaj biblioteki DLL. Jest to tylko nazwa pliku. Nie należy używać pełnej nazwy ścieżki.  
  
 `pmur`  
 [na zewnątrz] Przypisany `mdModuleRef` token.
