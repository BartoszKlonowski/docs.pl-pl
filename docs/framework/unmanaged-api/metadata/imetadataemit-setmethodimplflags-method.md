# <a name="imetadataemitsetmethodimplflags-method"></a>IMetaDataEmit::SetMethodImplFlags — Metoda
Ustawia lub aktualizuje podpis implementacji dziedziczonej metody, do której odwołuje się określony token metadanych.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT SetMethodImplFlags (   
    [in]  mdMethodDef   md,   
    [in]  DWORD         dwImplFlags   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `md`  
 [in] Token metody zostanie zmieniony.  
  
 `dwImplFlags`  
 [in] Kombinacja wartości [CorMethodImpl](../../../../docs/framework/unmanaged-api/metadata/cormethodimpl-enumeration.md) wyliczenie określający funkcji w implementacji metody.
