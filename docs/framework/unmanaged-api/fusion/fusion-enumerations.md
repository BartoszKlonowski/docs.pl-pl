# <a name="fusion-enumerations"></a>Wyliczenia łączenia
W tej sekcji opisano niezarządzane wyliczenia używane przez interfejs API Fusion.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [ASM_CACHE_FLAGS, wyliczenie](asm-cache-flags-enumeration.md)  
 Wskazuje źródło zestawu reprezentowanego przez [IAssemblyCacheItem](iassemblycacheitem-interface.md) w globalnej pamięci podręcznej zestawów.  
  
 [ASM_CMP_FLAGS, wyliczenie](asm-cmp-flags-enumeration.md)  
 Wskazuje wersję, kompilację, kulturę, sygnaturę i tak dalej, dwóch zestawów, które mają być porównane przez metodę [IAssemblyName:: IsEqual](iassemblyname-isequal-method.md) .  
  
 [ASM_DISPLAY_FLAGS, wyliczenie](asm-display-flags-enumeration.md)  
 Wskazuje wersję, kompilację, kulturę, sygnaturę i tak dalej, zestawu, którego nazwa wyświetlana zostanie pobrana przez [IAssemblyName:: GetDisplayName](iassemblyname-getdisplayname-method.md) metody.  
  
 [ASM_NAME, wyliczenie](asm-name-enumeration.md)  
 Wskazuje wersję, kompilację, kulturę, sygnaturę i tak dalej, zestawu, którego właściwości będą pobierane lub ustawiane przez metody [IAssemblyName](iassemblyname-interface.md) .  
  
 [AssemblyComparisonResult, wyliczenie](assemblycomparisonresult-enumeration.md)  
 Określa równoważność dwóch tożsamości zestawów określonych przez funkcję [CompareAssemblyIdentity —](compareassemblyidentity-function.md) .  
  
 [CREATE_ASM_NAME_OBJ_FLAGS, wyliczenie](create-asm-name-obj-flags-enumeration.md)  
 Określa atrybuty obiektu [`IAssemblyName`, gdy](createassemblynameobject-function.md) jest konstruowany przez funkcję myfunctionobject.
