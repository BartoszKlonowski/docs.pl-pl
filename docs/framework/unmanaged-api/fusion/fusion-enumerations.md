# <a name="fusion-enumerations"></a>Wyliczenia łączenia
W tej sekcji opisano niezarządzane wyliczenia, których używa fusion interfejsu API.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [ASM_CACHE_FLAGS, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 Wskazuje źródło zestawu reprezentowany przez [iassemblycacheitem —](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md) w globalnej pamięci podręcznej.  
  
 [ASM_CMP_FLAGS, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)  
 Wskazuje, wersji, kompilacji, kultury, podpis i tak dalej, dwóch zestawów, które mają być porównane [iassemblyname::isequal —](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-isequal-method.md) metody.  
  
 [ASM_DISPLAY_FLAGS, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)  
 Wskazuje, wersji, kompilacji, kultury, podpis i tak dalej, zestawu, którego nazwa wyświetlana będzie pobierany przez [iassemblyname::GetDisplayName —](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-getdisplayname-method.md) metody.  
  
 [ASM_NAME, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)  
 Wskazuje, wersji, kompilacji, kultury, podpis i tak dalej, zestawu, którego właściwości będą pobierane lub ustawione przez [iassemblyname —](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) metody.  
  
 [AssemblyComparisonResult, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/assemblycomparisonresult-enumeration.md)  
 Wskazuje na równoważność dwóch tożsamości zestawu, zgodnie z ustaleniami [compareassemblyidentity —](../../../../docs/framework/unmanaged-api/fusion/compareassemblyidentity-function.md) funkcji.  
  
 [CREATE_ASM_NAME_OBJ_FLAGS, wyliczenie](../../../../docs/framework/unmanaged-api/fusion/create-asm-name-obj-flags-enumeration.md)  
 Określa atrybuty elementu `IAssemblyName` obiektu, kiedy jest tworzony przez [createassemblynameobject —](../../../../docs/framework/unmanaged-api/fusion/createassemblynameobject-function.md) funkcji.
