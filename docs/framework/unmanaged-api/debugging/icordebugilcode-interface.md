# <a name="icordebugilcode-interface"></a>Interfejs ICorDebugILCode
[Obsługiwane w .NET Framework 4.5.2 i nowszych wersjach]  
  
 Reprezentuje segment kodu języka pośredniego (IL).  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetEHClauses, metoda](icordebugilcode-getehclauses-method.md)|Zwraca wskaźnik do listy klauzul obsługi wyjątków (EH), które są zdefiniowane dla tego IL.|
