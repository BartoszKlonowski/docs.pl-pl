# <a name="icordebugilcode-interface"></a>Interfejs ICorDebugILCode
[Obsługiwane w programie .NET Framework 4.5.2 i nowszych wersjach]  
  
 Reprezentuje segment kodu w języku pośrednim (IL).  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Metoda GetEHClauses](../../../../docs/framework/unmanaged-api/debugging/icordebugilcode-getehclauses-method.md)|Zwraca wskaźnik do listy klauzule (EH), które są zdefiniowane dla tego IL obsługi wyjątków.|
