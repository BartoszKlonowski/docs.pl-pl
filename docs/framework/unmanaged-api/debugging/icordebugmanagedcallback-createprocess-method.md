# <a name="icordebugmanagedcallbackcreateprocess-method"></a>ICorDebugManagedCallback::CreateProcess — Metoda
Powiadamia debuger został dołączony proces lub uruchomiony po raz pierwszy.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT CreateProcess (  
    [in] ICorDebugProcess *pProcess  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pProcess`  
 [in] Wskaźnik do obiektu ICorDebugProcess, który reprezentuje proces, który jest dołączony lub został rozpoczęty.  
  
## <a name="remarks"></a>Uwagi  
 Ta metoda nie jest wywoływany, dopóki nie została zainicjowana przez środowisko uruchomieniowe języka wspólnego. Większość [ICorDebug](../../../../docs/framework/unmanaged-api/debugging/icordebug-interface.md) metody zwróci CORDBG_E_NOTREADY przed `CreateProcess` wywołania zwrotnego.
