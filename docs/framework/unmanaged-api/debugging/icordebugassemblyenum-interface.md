# <a name="icordebugassemblyenum-interface"></a>ICorDebugAssemblyEnum, interfejs

Implementuje metody ICorDebugEnum i wylicza tablice ICorDebugAssembly.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugassemblyenum-next-method.md)|Pobiera określoną liczbę `ICorDebugAssembly` wystąpień w wyliczeniu, począwszy od bieżącej pozycji.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs może być wywoływany zdalnie, między komputerami ani między procesami.
