# <a name="icordebugassemblyenum-interface1"></a>ICorDebugAssemblyEnum Interface1
Implementuje metody ICorDebugEnum i wylicza ICorDebugAssembly tablic.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugassemblyenum-next-method.md)|Pobiera określoną liczbę `ICorDebugAssembly` wystąpień w wyliczeniu, zaczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
