# <a name="icordebugevalgetthread-method"></a>ICorDebugEval::GetThread — Metoda
Pobiera wątku, w którym ta ocena jest wykonywany lub zostanie wykonana.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetThread (  
    [out] ICorDebugThread   **ppThread  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppThread`  
 [out] Wskaźnik na adres ICorDebugThread obiekt, który reprezentuje wątku.
