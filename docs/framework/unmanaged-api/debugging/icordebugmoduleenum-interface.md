# <a name="icordebugmoduleenum-interface"></a>ICorDebugModuleEnum, interfejs

Implementuje metody ICorDebugEnum i wylicza tablice ICorDebugModule.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Pobiera określoną liczbę `ICorDebugModule` wystąpień z wyliczenia, rozpoczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
> Ten interfejs nie obsługuje wywoływania zdalnego na wielu maszynach ani wielu procesów.
