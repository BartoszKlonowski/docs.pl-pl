# <a name="icordebugmoduleenum-interface1"></a>ICorDebugModuleEnum Interface1
Implementuje metody ICorDebugEnum i wylicza ICorDebugModule tablic.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Pobiera określoną liczbę `ICorDebugModule` wystąpień z wyliczenia, zaczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
