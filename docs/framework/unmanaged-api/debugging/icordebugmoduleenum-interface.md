# <a name="icordebugmoduleenum-interface"></a>ICorDebugModuleEnum — Interfejs

Implementuje metody ICorDebugEnum i wylicza tablice ICorDebugModule.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Pobiera określoną liczbę `ICorDebugModule` wystąpień z wyliczenia, zaczynając od bieżącej pozycji.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs może być wywoływany zdalnie, między komputerami ani między procesami.
