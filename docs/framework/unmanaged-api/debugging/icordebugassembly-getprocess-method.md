# <a name="icordebugassemblygetprocess-method"></a>ICorDebugAssembly::GetProcess — Metoda
Pobiera wskaźnik interfejsu do procesu, w którym działa to wystąpienie ICorDebugAssembly.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppProcess`  
 określoną Wskaźnik do interfejsu ICorDebugProcess, który reprezentuje proces.
