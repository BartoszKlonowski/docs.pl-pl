# <a name="icordebugreferencevaluegetvalue-method"></a>ICorDebugReferenceValue::GetValue — Metoda

Pobiera bieżący adres pamięci obiektu, do którego istnieje odwołanie.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetValue (  
    [out] CORDB_ADDRESS   *pValue  
);  
```  
  
## <a name="parameters"></a>Parametry  

 `pValue`  
 określoną Wskaźnik do `CORDB_ADDRESS` wartości, która określa adres obiektu, do którego odwołuje się ten obiekt ICorDebugReferenceValue.
