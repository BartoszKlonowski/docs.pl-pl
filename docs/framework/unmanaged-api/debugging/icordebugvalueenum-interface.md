# <a name="icordebugvalueenum-interface"></a>ICorDebugValueEnum — Interfejs
Implementuje metody "ICorDebugEnum" i wylicza tablice "ICorDebugValue".  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugvalueenum-next-method.md)|Pobiera określoną liczbę wystąpień `ICorDebugValue` z wyliczenia, rozpoczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
> Ten interfejs nie obsługuje wywoływania zdalnego na wielu maszynach ani wielu procesów.
