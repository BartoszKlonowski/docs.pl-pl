## <a name="method"></a>Metoda  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetObjectForCCW, metoda](icordebugappdomain4-getobjectforccw-method.md)|Pobiera obiekt zarządzany z wskaźnika otoki (CCW) modelu COM.|
