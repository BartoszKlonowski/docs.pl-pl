# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 — interfejs
Logicznie rozszerza interfejs ICorDebugAppDomain w celu uzyskania obiektu zarządzanego z przywywoływanej otoki COM.  
  
## <a name="method"></a>Metoda  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetObjectForCCW, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|Pobiera obiekt zarządzany z wskaźnika otoki (CCW) modelu COM.|
