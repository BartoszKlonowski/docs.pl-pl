# <a name="icordebugcodeenum-interface1"></a>ICorDebugCodeEnum Interface1
Implementuje metody "ICorDebugEnum", a wylicza tablice "ICorDebugCode".  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next — metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugcodeenum-next-method.md)|Pobiera określoną liczbę `ICorDebugCode` wystąpień z wyliczenia, zaczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
