# <a name="icordebugprocess3-interface"></a>ICorDebugProcess3 — Interfejs
Steruje niestandardowymi powiadomieniami debugera.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[SetEnableCustomNotification, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess3-setenablecustomnotification-method.md)|Włącza i wyłącza powiadomienia debugera niestandardowe określonego typu.|  
  
## <a name="remarks"></a>Uwagi  
 Ten interfejs rozszerza logicznie ICorDebugProcess i ICorDebugProcess2 interfejsów.  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
