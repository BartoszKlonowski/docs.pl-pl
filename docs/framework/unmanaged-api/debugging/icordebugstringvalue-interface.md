# <a name="icordebugstringvalue-interface1"></a>ICorDebugStringValue Interface1
Podklasa ICorDebugHeapValue, która ma zastosowanie do wartości ciągu.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetLength, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getlength-method.md)|Pobiera liczbę znaków w ciągu odwołuje się to `ICorDebugStringValue`.|  
|[GetString, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getstring-method.md)|Pobiera ciąg odwołuje się to `ICorDebugStringValue`.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
