# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank — Metoda
Pobiera liczbę wymiarów w typie tablicy.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pnRank`  
 określoną Wskaźnik do liczby wymiarów.
