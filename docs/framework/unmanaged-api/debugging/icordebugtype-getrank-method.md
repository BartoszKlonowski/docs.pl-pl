# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank — Metoda
Pobiera liczbę wymiarów typu tablicowego.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pnRank`  
 [out] Wskaźnik do liczby wymiarów.
