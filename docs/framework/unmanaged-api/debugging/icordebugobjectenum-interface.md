# <a name="icordebugobjectenum-interface1"></a>ICorDebugObjectEnum Interface1
Implementuje metody ICorDebugEnum i wylicza tablice obiektów przez względną wirtualnych adresów (RVAs).  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugobjectenum-next-method.md)|Pobiera RVAs o określoną liczbę obiektów z wyliczenia, zaczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ten interfejs nie obsługuje wywoływany zdalnie, między komputerami lub między procesami.
