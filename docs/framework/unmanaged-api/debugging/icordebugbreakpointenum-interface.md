# <a name="icordebugbreakpointenum-interface"></a>ICorDebugBreakpointEnum, interfejs

Implementuje metody ICorDebugEnum i wylicza tablice ICorDebugBreakpoint.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|Pobiera określoną liczbę `ICorDebugBreakpoint` wystąpień z wyliczenia, rozpoczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
> Ten interfejs nie obsługuje wywoływania zdalnego na wielu maszynach ani wielu procesów.
