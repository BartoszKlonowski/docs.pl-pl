# <a name="icordebugstepperenum-interface"></a>ICorDebugStepperEnum, interfejs
Implementuje metody ICorDebugEnum i wylicza tablice ICorDebugStepper.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Next, metoda](icordebugstepperenum-next-method.md)|Pobiera określoną liczbę wystąpień `ICorDebugStepper` z wyliczenia, rozpoczynając od bieżącego położenia.|  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
> Ten interfejs nie obsługuje wywoływania zdalnego na wielu maszynach ani wielu procesów.
