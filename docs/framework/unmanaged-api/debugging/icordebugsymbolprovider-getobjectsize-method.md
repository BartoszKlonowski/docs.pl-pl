# <a name="icordebugsymbolprovidergetobjectsize-method"></a>ICorDebugSymbolProvider::GetObjectSize — metoda
Zwraca rozmiar obiektu dla obiektu, w oparciu o jego sygnatura elementu typespec.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetObjectSize(  
   [in] ULONG32 cbSignature,  
   [in, size_is(cbSignature)]  BYTE typeSig[],  
   [out] ULONG32 *pObjectSize  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `cbSignature`  
 [in] Liczba bajtów w podpisie elementu typespec.  
  
 typeSig  
 [in] Podpis elementu typespec.  
  
 `pObjectSize`  
 [out] Wskaźnik do rozmiar obiektu.  
  
## <a name="remarks"></a>Uwagi  
  
> [!NOTE]
>  Ta metoda jest tylko dostępne z platformą .NET Native.
