# <a name="getassemblyrefhash-method"></a>GetAssemblyRefHash — Metoda
Pobiera obiekt blob wyznaczania wartości skrótu dla danego zestawu.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetAssemblyRefHash(  
    mdToken FileToken,  
    const void** ppvHash,  
    DWORD* pcbHash  
) PURE;  
```  
  
#### <a name="parameters"></a>Parametry  
 `FileToken`  
 Identyfikator zestawu, do którego będzie można znaleźć skrót.  
  
 `ppvHash`  
 Odbiera wynikowego obiektu blob generowania skrótu.  
  
 `pcbHash`  
 Odbiera rozmiar w bajtach obiektu blob generowania skrótu.  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca wartość S_OK, jeśli metoda zakończy się powodzeniem.  
  
## <a name="requirements"></a>Wymagania  
 Wymaga alink.h
