|Metoda|Opis|  
|------------|-----------------|  
|[AutoAttach, metoda](../../../../docs/framework/unmanaged-api/diagnostics/idebugautoattach-autoattach-method.md)|Wykonuje automatyczne wywoływane serwer debugera dołączania.|
