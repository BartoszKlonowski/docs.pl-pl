# <a name="isymunmanagedscope2-interface"></a>ISymUnmanagedScope2 — Interfejs
Reprezentuje zakres leksykalny w ramach metody. Ten interfejs rozszerza interfejs [ISymUnmanagedScope](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedscope-interface.md) z metodami, które pobierają informacje o stałych zdefiniowanych w zakresie.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetConstantCount, metoda](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedscope2-getconstantcount-method.md)|Pobiera liczbę stałych zdefiniowanych w tym zakresie.|  
|[GetConstants, metoda](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedscope2-getconstants-method.md)|Pobiera stałe lokalne zdefiniowane w tym zakresie.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** CorSym. idl, CorSym. h
