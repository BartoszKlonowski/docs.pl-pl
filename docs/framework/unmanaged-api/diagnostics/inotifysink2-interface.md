# <a name="inotifysink2-interface"></a>INotifySink2 — Interfejs
Deklaruje metody do odbioru powiadomienia.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[OnSyncCallEnter — metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifysink2-onsynccallenter-method.md)|Pobiera wywoływane podczas wchodzenia wywołanie.|  
|[OnSyncCallExit — metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifysink2-onsynccallexit-method.md)|Pobiera wywoływane podczas zamykania wywołania.|  
|[OnSyncCallOut — metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifysink2-onsynccallout-method.md)|Pobiera wywoływane, gdy połączenie jest limit.|  
|[OnSyncCallReturn — metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifysink2-onsynccallreturn-method.md)|Wywołany po wywołaniu zwraca.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** ProtocolNotify2.idl
