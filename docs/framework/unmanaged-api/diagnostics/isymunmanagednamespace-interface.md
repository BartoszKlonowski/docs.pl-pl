# <a name="isymunmanagednamespace-interface"></a>ISymUnmanagedNamespace — Interfejs

Reprezentuje przestrzeń nazw.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[GetName — Metoda](isymunmanagednamespace-getname-method.md)|Pobiera nazwę tej przestrzeni nazw.|  
|[GetNamespaces, metoda](isymunmanagednamespace-getnamespaces-method.md)|Pobiera elementy podrzędne tej przestrzeni nazw.|  
|[GetVariables, metoda](isymunmanagednamespace-getvariables-method.md)|Zwraca wszystkie zmienne zdefiniowane w zakresie globalnym w tej przestrzeni nazw.|  
  
## <a name="requirements"></a>Wymagania  

 **Nagłówek:** CorSym. idl, CorSym. h
