# <a name="inotifyconnection2-interface"></a>INotifyConnection2 — Interfejs
Deklaruje metody zarejestrować lub wyrejestrować źródła powiadomień połączenia.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[RegisterNotifySource, metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifyconnection2-registernotifysource-method.md)|Instaluje obiektu źródłowego określonego powiadomień.|  
|[UnregisterNotifySource, metoda](../../../../docs/framework/unmanaged-api/diagnostics/inotifyconnection2-unregisternotifysource-method.md)|Usuwa obiekt źródłowy określony powiadomień z połączenia.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** ProtocolNotify2.idl
