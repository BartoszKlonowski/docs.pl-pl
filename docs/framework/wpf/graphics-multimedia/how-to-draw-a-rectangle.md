 <span data-ttu-id="c5381-109">W poniższym przykładzie dwa <xref:System.Windows.Shapes.Rectangle> elementy są rysowane <xref:System.Windows.Controls.Canvas>.</span><span class="sxs-lookup"><span data-stu-id="c5381-109">In the following example, two <xref:System.Windows.Shapes.Rectangle> elements are drawn in a <xref:System.Windows.Controls.Canvas>.</span></span> <span data-ttu-id="c5381-110">Pierwszy prostokąt ma <xref:System.Windows.Media.Brushes.Blue%2A> wewnętrznych.</span><span class="sxs-lookup"><span data-stu-id="c5381-110">The first rectangle has a <xref:System.Windows.Media.Brushes.Blue%2A> interior.</span></span> <span data-ttu-id="c5381-111">Drugi prostokąt ma <xref:System.Windows.Media.Brushes.Blue%2A> wewnętrzne, <xref:System.Windows.Media.Brushes.Black%2A> konspektu i zaokrąglone narożniki.</span><span class="sxs-lookup"><span data-stu-id="c5381-111">The second rectangle has a <xref:System.Windows.Media.Brushes.Blue%2A> interior, a <xref:System.Windows.Media.Brushes.Black%2A> outline, and rounded corners.</span></span>
