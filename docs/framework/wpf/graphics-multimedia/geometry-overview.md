 <span data-ttu-id="cebe6-117">Mimo że <xref:System.Windows.Shapes.Shape> obiekty są łatwiej można używać niż <xref:System.Windows.Media.Geometry> obiektów, <xref:System.Windows.Media.Geometry> obiekty są bardziej elastyczne.</span><span class="sxs-lookup"><span data-stu-id="cebe6-117">Although <xref:System.Windows.Shapes.Shape> objects are more readily usable than <xref:System.Windows.Media.Geometry> objects, <xref:System.Windows.Media.Geometry> objects are more versatile.</span></span> <span data-ttu-id="cebe6-118">Gdy <xref:System.Windows.Shapes.Shape> obiekt jest używany do renderowania grafiki 2-D <xref:System.Windows.Media.Geometry> obiekt może służyć do zdefiniować geometrycznych region grafiki 2-D, zdefiniować region dla wycinka lub zdefiniować region testowania trafień, np.</span><span class="sxs-lookup"><span data-stu-id="cebe6-118">While a <xref:System.Windows.Shapes.Shape> object is used to render 2-D graphics, a <xref:System.Windows.Media.Geometry> object can be used to define the geometric region for 2-D graphics, define a region for clipping, or define a region for hit testing, for example.</span></span>
