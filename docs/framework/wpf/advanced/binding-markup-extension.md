## <a name="binding-expression-usage"></a><span data-ttu-id="21bd8-104">Użycie wyrażenia powiązania</span><span class="sxs-lookup"><span data-stu-id="21bd8-104">Binding Expression Usage</span></span>  
  
```  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
