<span data-ttu-id="3d69d-103">Wprowadza wartość właściwości jako wartość powiązaną z danymi, tworząc obiekt wyrażenia pośredniego i interpretuje kontekst danych, który ma zastosowanie do elementu i jego powiązania w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="3d69d-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="3d69d-104">Użycie wyrażenia powiązania</span><span class="sxs-lookup"><span data-stu-id="3d69d-104">Binding Expression Usage</span></span>  
  
```xaml  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
