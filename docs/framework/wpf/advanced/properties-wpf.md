# <a name="properties-wpf"></a>Właściwości (WPF)
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] zawiera zestaw usług, których można użyć do rozszerania funkcjonalności środowiska uruchomieniowego języka wspólnego (CLR).
