# <a name="building-and-deploying-wpf-applications"></a>Kompilowanie i wdrażanie aplikacji WPF
Kompilowanie i wdrażanie modelu zapewnia możliwość tworzenia i wdrażania aplikacji, lokalne i zdalne, takie jak następujące:
