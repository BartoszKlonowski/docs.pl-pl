# <a name="hosting-wpf-applications"></a>Hosting aplikacji WPF
Aplikacje w przeglądarce WPF XAML (XBAP) są rozbudowanymi aplikacjami klienckimi, które można wdrożyć na serwerze sieci Web i uruchomić w przeglądarce. Host WPF (PresentationHost. exe) jest zarejestrowany jako powłoka i obsługa MIME dla plików XBAP i XAML. W związku z tym program Internet Explorer wie, jak uruchomić Host WPF po uruchomieniu aplikacji XBAP. Użytkownicy programu Firefox mogą instalować dodatki Firefox, które umożliwiają również hostowanie aplikacji XBAP. Aplikacja XBAP może być hostowana w innych przeglądarkach lub aplikacjach autonomicznych przy użyciu natywnych interfejsów API hostingowych udostępnianych przez WPF.
