# <a name="windows-in-wpf-applications"></a>Okna w aplikacjach WPF
Użytkownicy pracują z aplikacjami za pomocą systemu Windows. Podstawowym celem okna jest Hostowanie i wyświetlanie zawartości. Typ zawartości, która jest zależna od typu danych, który jest obsługiwany przez aplikację, która może obejmować multimedia, Extensible Application Markup Language (XAML) strony, strony sieci Web, dokumenty, tabele bazy danych i rekordy oraz informacje o systemie.
