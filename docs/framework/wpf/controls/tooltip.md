# <a name="tooltip"></a>ToolTip
Etykietka narzędzia jest mała okno podręczne pojawia się po zatrzymaniu wskaźnika myszy nad elementem, takie jak ponad <xref:System.Windows.Controls.Button>.  
  
 Na poniższej ilustracji przedstawiono wskaźnika myszy, który wskazuje **Zamknij**<xref:System.Windows.Controls.Button>, który następnie wyświetla jego zidentyfikowanie <xref:System.Windows.Controls.ToolTip>.
