# <a name="net-framework-system-requirements"></a>Wymagania systemowe programu .NET Framework

Tabele w tym artykule zawierają wymagania dotyczące sprzętu, systemu operacyjnego i oprogramowania dla następujących wersji programu .NET Framework:

- .NET Framework 4.5 i jego wydania punktowe (4.5.1 i 4.5.2).
- .NET Framework 4.6 i jego wydania punktowe (4.6.1 i 4.6.2).
- .NET Framework 4.7 i jego wydania punktowe (4.7.1 i 4.7.2).
-  .NET Framework 4.8

Aby uzyskać informacje na temat wersji programu .NET Framework wcześniejszych niż .NET Framework 4.5, zobacz [wersje i zależności programu .NET Framework](../migration-guide/versions-and-dependencies.md).

Środowiska programistyczne, które umożliwiają tworzenie aplikacji dla platformy .NET Framework mają oddzielny zestaw wymagań.

[!INCLUDE[net-framework-4-versions](../../../includes/net-framework-4x-versions.md)]

Aby uzyskać informacje o pobieraniu i łączach, zobacz [Instalowanie programu .NET Framework dla deweloperów](../install/guide-for-developers.md).

Aby uzyskać informacje na temat cyklu pomocy technicznej w wersjach programu .NET Framework, zobacz [Cykl pomocy technicznej firmy Microsoft](https://support.microsoft.com/lifecycle/search?sort=PN&alpha=Microsoft%20.NET%20Framework&Filter=FilterNO).

## <a name="hardware-requirements"></a>Wymagania sprzętowe

|                          |        |
| ------------------------ | ------ |
| **Procesor**            | 1 GHz  |
| **Pamięci ram**                  | 512 MB |
| **Miejsce na dysku (minimum)** |        |
| 32-bitowa                   | 4,5 GB |
| 64-bitowy                   | 4,5 GB |

## <a name="installation-requirements"></a>Wymagania dotyczące instalacji

.NET Framework wymaga uprawnień administratora do instalacji. Jeśli nie masz uprawnień administratora do komputera, na którym chcesz zainstalować program .NET Framework, skontaktuj się z administratorem sieci.

 **Notatki:**

- W systemach Windows 7 program .NET Framework wymaga dodatku SP1 dla systemu Windows 7. Jeśli korzystasz z systemu Windows 7 i nie masz jeszcze zainstalowanego dodatku Service Pack 1, musisz to zrobić przed zainstalowaniem programu .NET Framework.

- .NET Framework 4 obsługuje również platformę IA64.

- W przypadku wszystkich platform zaleca się uaktualnienie do najnowszego dodatku Service Pack dla systemu Windows i zainstalowanie aktualizacji krytycznych dostępnych w [witrynie Windows Update w](https://support.microsoft.com/help/12373/windows-update-faq) celu zapewnienia najlepszej zgodności i zabezpieczeń.

- W 64-bitowych systemach operacyjnych program .NET Framework obsługuje zarówno wow64 (przetwarzanie 32-bitowe na komputerze 64-bitowym), jak i natywne przetwarzanie 64-bitowe.
