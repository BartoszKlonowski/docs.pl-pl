Śledzenie zdarzeń dla systemu Windows (ETW) to wysokowydajny, niskonasytowy, skalowalny system śledzenia dostarczany przez systemy operacyjne Windows. Uzupełnia profilowania i debugowania pomocy technicznej świadczonych przez .NET Framework i może służyć do rozwiązywania różnych scenariuszy.
