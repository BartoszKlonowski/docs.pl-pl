## <a name="third-party-serializers"></a><span data-ttu-id="4cfaa-109">Serializatory innych firm</span><span class="sxs-lookup"><span data-stu-id="4cfaa-109">Third-party serializers</span></span>

 <span data-ttu-id="4cfaa-110">Serializatory innych firm, w tym Newtonsoft.JSna, zwykle są oparte na odbiciach.</span><span class="sxs-lookup"><span data-stu-id="4cfaa-110">Third-part serializers, including Newtonsoft.JSON, typically are reflection-based.</span></span> <span data-ttu-id="4cfaa-111">Przy użyciu binarnego dużego obiektu (BLOB) danych serializowanych pola w danych są przypisywane do konkretnego typu przez wyszukanie pól typu docelowego według nazwy.</span><span class="sxs-lookup"><span data-stu-id="4cfaa-111">Given a binary large object (BLOB) of serialized data, the fields in the data are assigned to a concrete type by looking up the fields of the target type by name.</span></span> <span data-ttu-id="4cfaa-112">Co najmniej przy użyciu tych bibliotek powoduje wyjątki [MissingMetadataException](missingmetadataexception-class-net-native.md) dla każdego <xref:System.Type> obiektu, który próbujesz serializować lub deserializować w `List<Type>` kolekcji.</span><span class="sxs-lookup"><span data-stu-id="4cfaa-112">At a minimum, using these libraries causes [MissingMetadataException](missingmetadataexception-class-net-native.md) exceptions for each <xref:System.Type> object that you try to serialize or deserialize in a `List<Type>` collection.</span></span>  
  
 <span data-ttu-id="4cfaa-113">Najprostszym sposobem rozwiązania problemów spowodowanych brakiem metadanych dla tych serializatorów jest zbieranie typów, które będą używane w serializacji w ramach pojedynczej przestrzeni nazw (na przykład `App.Models` ) i zastosowanie `Serialize` dyrektywy Metadata do niej:</span><span class="sxs-lookup"><span data-stu-id="4cfaa-113">The easiest way to address issues caused by missing metadata for these serializers is to collect types that will be used in serialization under a single namespace (such as `App.Models`) and apply a `Serialize` metadata directive to it:</span></span>  
  
```xml

```  
  
 <span data-ttu-id="4cfaa-114">Aby uzyskać informacje o składni używanej w przykładzie, zobacz [ \<Namespace> element](namespace-element-net-native.md).</span><span class="sxs-lookup"><span data-stu-id="4cfaa-114">For information about the syntax used in the example, see [\<Namespace> Element](namespace-element-net-native.md).</span></span>
