## <a name="identifying-resources"></a><span data-ttu-id="9ec6c-113">Identyfikowanie zasobów</span><span class="sxs-lookup"><span data-stu-id="9ec6c-113">Identifying Resources</span></span>

 <span data-ttu-id="9ec6c-117">Gdy serwer odebrał żądanie i przetworzył odpowiedź, zwróci odpowiedź do aplikacji klienckiej.</span><span class="sxs-lookup"><span data-stu-id="9ec6c-117">After the server has received the request and processed the response, it returns the response to the client application.</span></span> <span data-ttu-id="9ec6c-118">Odpowiedź zawiera dodatkowe informacje, takie jak typ zawartości (na przykład nieprzetworzony tekst lub dane XML).</span><span class="sxs-lookup"><span data-stu-id="9ec6c-118">The response includes supplemental information, such as the type of the content (raw text or XML data, for example).</span></span>
