## <a name="identifying-resources"></a><span data-ttu-id="4e8ff-112">Identyfikowanie zasobów</span><span class="sxs-lookup"><span data-stu-id="4e8ff-112">Identifying Resources</span></span>

 <span data-ttu-id="4e8ff-116">Gdy serwer odebrał żądanie i przetworzył odpowiedź, zwróci odpowiedź do aplikacji klienckiej.</span><span class="sxs-lookup"><span data-stu-id="4e8ff-116">After the server has received the request and processed the response, it returns the response to the client application.</span></span> <span data-ttu-id="4e8ff-117">Odpowiedź zawiera dodatkowe informacje, takie jak typ zawartości (na przykład nieprzetworzony tekst lub dane XML).</span><span class="sxs-lookup"><span data-stu-id="4e8ff-117">The response includes supplemental information, such as the type of the content (raw text or XML data, for example).</span></span>
