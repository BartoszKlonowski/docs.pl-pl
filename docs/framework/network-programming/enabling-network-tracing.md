 <span data-ttu-id="3c88b-110">Najczęstsze miejsca docelowe śledzenia, określane również jako detektory śledzenia, są domyślnym odbiornikiem i plikiem dziennika.</span><span class="sxs-lookup"><span data-stu-id="3c88b-110">The most common trace destinations, also referred to as trace listeners, are the default listener and the log file.</span></span>  
  
 <span data-ttu-id="3c88b-111">Śledzenie używa odbiornika domyślnego, jeśli nie określisz odbiornika śledzenia.</span><span class="sxs-lookup"><span data-stu-id="3c88b-111">Tracing uses the default listener if you do not specify a trace listener.</span></span> <span data-ttu-id="3c88b-112">Wiadomości wysyłane do odbiornika domyślnego można wyświetlać, wykonując kod w debugerze z włączoną kodem zarządzanym, takim jak debuger ŚRODOWISKA CLR dostarczany z modułem SDK programu .NET Framework lub program DBwin32.exe dostarczany z modułem Windows SDK.</span><span class="sxs-lookup"><span data-stu-id="3c88b-112">You can view messages sent to the default listener by executing your code in a managed code-enabled debugger such as the CLR Debugger shipped with the .NET Framework SDK, or DBwin32.exe shipped with the Windows SDK.</span></span> <span data-ttu-id="3c88b-113">Za pomocą debugera CLR komunikaty śledzenia są wyświetlane w oknie **Dane wyjściowe.**</span><span class="sxs-lookup"><span data-stu-id="3c88b-113">Using the CLR Debugger, the trace messages appear in the **Output** window.</span></span>  
  
 <span data-ttu-id="3c88b-114">Jeśli wolisz używać pliku do odbierania śladów, możesz określić plik dziennika przy użyciu ustawień konfiguracji, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="3c88b-114">If you prefer to use a file to receive traces, you can specify a log file by using configuration settings, as shown in the following example.</span></span> <span data-ttu-id="3c88b-115">(Aby zapoznać się z ogólną omówień plików konfiguracyjnych, zobacz [Pliki konfiguracyjne](../configure-apps/index.md).)</span><span class="sxs-lookup"><span data-stu-id="3c88b-115">(For a general discussion of configuration files, see [Configuration Files](../configure-apps/index.md).)</span></span>  
  
 <span data-ttu-id="3c88b-116">Aby wysłać ślady do pliku dziennika, `<system.diagnostics>` dodaj następujący węzeł do węzła odpowiedniego pliku konfiguracyjnego (aplikacji lub komputera).</span><span class="sxs-lookup"><span data-stu-id="3c88b-116">To send traces to a log file, add the following node to the `<system.diagnostics>` node of the appropriate configuration file (application or machine).</span></span> <span data-ttu-id="3c88b-117">Nazwę pliku (trace.log) można zmienić zgodnie z potrzebami.</span><span class="sxs-lookup"><span data-stu-id="3c88b-117">You can change the name of the file (trace.log) to suit your needs.</span></span>  
  
```xml

  <trace autoflush="true" indentsize="4">  
    <listeners>  
      <add name="file" type="System.Diagnostics.TextWriterTraceListener" initializeData="trace.log"/>  
    </listeners>
  </trace>  
</system.diagnostics>  
```
