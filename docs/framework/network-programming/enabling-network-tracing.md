<span data-ttu-id="9b60d-104">Śledzenie sieci zapewnia dostęp do informacji o wywołaniach metod i ruchu sieciowym generowanym przez zarządzaną aplikację.</span><span class="sxs-lookup"><span data-stu-id="9b60d-104">Network tracing provides access to information about method invocations and network traffic generated by a managed application.</span></span> <span data-ttu-id="9b60d-105">Aby włączyć śledzenie sieci w aplikacji, należy wykonać następujące zadania:</span><span class="sxs-lookup"><span data-stu-id="9b60d-105">You must complete the following tasks to enable network tracing in your application:</span></span>  
  
- <span data-ttu-id="9b60d-106">Kompiluj swój kod z włączonym śledzeniem.</span><span class="sxs-lookup"><span data-stu-id="9b60d-106">Compile your code with tracing enabled.</span></span> <span data-ttu-id="9b60d-107">Zobacz [jak: Kompiluj warunkowo, śledząc i Debug](../debug-trace-profile/how-to-compile-conditionally-with-trace-and-debug.md) , aby uzyskać więcej informacji na temat przełączników kompilatora wymaganych do włączenia śledzenia.</span><span class="sxs-lookup"><span data-stu-id="9b60d-107">See [How to: Compile Conditionally with Trace and Debug](../debug-trace-profile/how-to-compile-conditionally-with-trace-and-debug.md) for more information about the compiler switches required to enable tracing.</span></span>  
  
- <span data-ttu-id="9b60d-108">Określ miejsce docelowe śledzenia danych wyjściowych.</span><span class="sxs-lookup"><span data-stu-id="9b60d-108">Specify a destination for tracing output.</span></span>  
  
- <span data-ttu-id="9b60d-109">Skonfiguruj zachowanie funkcji Śledzenie sieci.</span><span class="sxs-lookup"><span data-stu-id="9b60d-109">Configure the behavior of network tracing.</span></span> <span data-ttu-id="9b60d-110">Zobacz [jak: Konfigurowanie śledzenia sieci,](how-to-configure-network-tracing.md) Aby uzyskać szczegółowe informacje.</span><span class="sxs-lookup"><span data-stu-id="9b60d-110">See [How to: Configure Network Tracing](how-to-configure-network-tracing.md) for detailed information.</span></span>  
  
 <span data-ttu-id="9b60d-111">Najpopularniejsze miejsca docelowe śledzenia, nazywane również odbiornikami śledzenia, to domyślny odbiornik i plik dziennika.</span><span class="sxs-lookup"><span data-stu-id="9b60d-111">The most common trace destinations, also referred to as trace listeners, are the default listener and the log file.</span></span>  
  
 <span data-ttu-id="9b60d-112">Śledzenie używa domyślnego odbiornika, jeśli nie zostanie określony odbiornik śledzenia.</span><span class="sxs-lookup"><span data-stu-id="9b60d-112">Tracing uses the default listener if you do not specify a trace listener.</span></span> <span data-ttu-id="9b60d-113">Możesz wyświetlić komunikaty wysyłane do domyślnego odbiornika, wykonując kod w debugerze obsługującym kod zarządzany, taki jak debuger CLR dostarczany z zestawem SDK .NET Framework, lub DBwin32.exe dostarczony z Windows SDKem.</span><span class="sxs-lookup"><span data-stu-id="9b60d-113">You can view messages sent to the default listener by executing your code in a managed code-enabled debugger such as the CLR Debugger shipped with the .NET Framework SDK, or DBwin32.exe shipped with the Windows SDK.</span></span> <span data-ttu-id="9b60d-114">Przy użyciu debugera CLR komunikaty śledzenia pojawiają się w oknie **danych wyjściowych** .</span><span class="sxs-lookup"><span data-stu-id="9b60d-114">Using the CLR Debugger, the trace messages appear in the **Output** window.</span></span>  
  
 <span data-ttu-id="9b60d-115">Jeśli wolisz używać pliku do odbierania śladów, możesz określić plik dziennika za pomocą ustawień konfiguracji, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="9b60d-115">If you prefer to use a file to receive traces, you can specify a log file by using configuration settings, as shown in the following example.</span></span> <span data-ttu-id="9b60d-116">(Ogólne omówienie plików konfiguracji można znaleźć w temacie [pliki konfiguracyjne](../configure-apps/index.md)).</span><span class="sxs-lookup"><span data-stu-id="9b60d-116">(For a general discussion of configuration files, see [Configuration Files](../configure-apps/index.md).)</span></span>  
  
 <span data-ttu-id="9b60d-117">Aby wysłać ślady do pliku dziennika, Dodaj następujący węzeł do `<system.diagnostics>` węzła odpowiedniego pliku konfiguracji (aplikacji lub komputera).</span><span class="sxs-lookup"><span data-stu-id="9b60d-117">To send traces to a log file, add the following node to the `<system.diagnostics>` node of the appropriate configuration file (application or machine).</span></span> <span data-ttu-id="9b60d-118">Możesz zmienić nazwę pliku (Trace. log) zgodnie z potrzebami.</span><span class="sxs-lookup"><span data-stu-id="9b60d-118">You can change the name of the file (trace.log) to suit your needs.</span></span>  
  
```xml  
<system.diagnostics>  
  <trace autoflush="true" indentsize="4">  
    <listeners>  
      <add name="file" type="System.Diagnostics.TextWriterTraceListener" initializeData="trace.log"/>  
    </listeners>
  </trace>  
</system.diagnostics>  
```
