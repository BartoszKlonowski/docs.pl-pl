# <a name="network-programming-samples"></a>Przykłady programowania sieciowego
Ta sekcja zawiera opisy i linki do pobrania programowanie dla sieci przykłady używanego przez klasy w <xref:System.Net>, <xref:System.Net.Cache>, <xref:System.Net.Configuration>, <xref:System.Net.Mail>, <xref:System.Net.Mime>, <xref:System.Net.NetworkInformation>, <xref:System.Net.Security>, <xref:System.Net.Sockets> i pokrewnych przestrzeniach nazw. 
  
> [!NOTE]
> Wszystkie przykłady są dostępne w [wersji .NET Framework SDK 2.0 przykłady Pobierz](https://www.microsoft.com/en-us/download/confirmation.aspx?id=22181), która została opublikowana dla programu .NET Framework 2.0 i może być datą.

## <a name="in-this-section"></a>W tej sekcji  
 [Pobierz przykład technologii wskaźnik postępu](https://msdn.microsoft.com/en-us/library/t8w6294a(v=vs.85).aspx)  
 Pokazuje, jak wyświetlić postęp pobierania plików.  
  
 [Przykład technologii klienta FTP](https://msdn.microsoft.com/en-us/library/b7810t5c(v=vs.85).aspx)  
 Pokazuje, jak przekazywać i pobierać pliki do i z serwera FTP.  
  
 [Przykład technologii HttpListener](https://msdn.microsoft.com/en-us/library/y7cbb2y2(v=vs.85).aspx)  
 Pokazuje, jak można przetworzyć żądania HTTP z aplikacji.  
 
 [Przykład aplikacji hosta HttpListener ASPX](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2008/dd767375(v%3dvs.90))   
 Pokazuje, jak używać funkcji <xref:System.Net.HttpListener?displayProperty=nameWithType> klasy w celu utworzenia serwera HTTP, który przekierowuje wywołania do hostowanych aplikacji ASP.NET.
  
 [Przykład technologii masowo wiadomości E-mail](https://msdn.microsoft.com/en-us/library/whw7xbk2(v=vs.85).aspx)  
 Pokazuje, jak wysyłać wiadomości e-mail z aplikacji klienckiej.  
  
 [Przykład technologii NetStat narzędzia](https://msdn.microsoft.com/en-us/library/ks32hs88(v=vs.85).aspx)

 Pokazuje aplikację kliencką, która może wysyłać polecenia ping hosta zdalnego.  
  
 [Przykład technologii WebClient](https://msdn.microsoft.com/en-us/library/fxk992zc(v=vs.85).aspx)  
 Pokazuje, jak wykonywać typowe operacje, takie jak przekazywania i pobierania plików lub danych.
