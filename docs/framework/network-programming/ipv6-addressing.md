- <span data-ttu-id="7be4f-109">**Formularz szesnastkowe dwukropek**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-109">**Colon-hexadecimal form**.</span></span> <span data-ttu-id="7be4f-110">Jest to n:n:n:n:n:n:n:n preferowaną formę.</span><span class="sxs-lookup"><span data-stu-id="7be4f-110">This is the preferred form n:n:n:n:n:n:n:n.</span></span> <span data-ttu-id="7be4f-111">Każdy n oznacza wartość szesnastkową jednego z ośmiu elementów 16-bitowych w adresie.</span><span class="sxs-lookup"><span data-stu-id="7be4f-111">Each n represents the hexadecimal value of one of the eight 16-bit elements of the address.</span></span> <span data-ttu-id="7be4f-112">Na przykład: `3FFE:FFFF:7654:FEDA:1245:BA98:3210:4562`.</span><span class="sxs-lookup"><span data-stu-id="7be4f-112">For example: `3FFE:FFFF:7654:FEDA:1245:BA98:3210:4562`.</span></span>

- <span data-ttu-id="7be4f-113">**Skompresowane formularza**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-113">**Compressed form**.</span></span> <span data-ttu-id="7be4f-114">Ze względu class="sxs-lookup"><span data-stu-id="7be4f-118">The unicast address `3FFE:FFFF:0:0:8:800:20C4:0` in compressed form is `3FFE:FFFF::8:800:20C4:0`.</span></span> <span data-ttu-id="7be4f-119">Adres - <span data-ttu-id="7be4f-121">**Mieszane formularza**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-121">**Mixed form**.</span></span> <span data-ttu-id="7be4f-122">Ten formularz łączy adresów IPv4 i IPv6.</span><span class="sxs-lookup"><span data-stu-id="7be4f-122">This form combines IPv4 and IPv6 addresses.</span></span> <span data-ttu-id="7be4f-123">W tym przypadku format adresu jest n:n:n:n:n:n:d.d.d.d, gdzie każdy n wartości szesnastkowych sześciu elementów wyższego rzędu 16-bitowy adres IPv6, a każdy d reprezentuje wartość dziesiętna adres IPv4.</span><span class="sxs-lookup"><span data-stu-id="7be4f-123">In this case, the address format is n:n:n:n:n:n:d.d.d.d, where each n represents the hexadecimal values of the six IPv6 high-order 16-bit address elements, and each d represents the decimal value of an IPv4 address.</span></span>

## <a name="address-types"></a><span data-ttu-id="7be4f-124">Typy adresów</span><span class="sxs-lookup"><span data-stu-id="7be4f-124">Address Types</span></span>

<span data-ttu-id="7be4f-125">Wiodący bitów w adresie zdefiniować określonego typu adresu IPv6.</span><span class="sxs-lookup"><span data-stu-id="7be4f-125">The leading bits in the address define the specific IPv6 address type.</span></span> <span data-ttu-id="7be4f-126">Pole o zmiennej długości, zawierające te bitów początkowych nosi nazwę formatu prefiksu (FP).</span><span class="sxs-lookup"><span data-stu-id="7be4f-126">The variable-length field containing these leading bits is called a Format Prefix (FP).</span></span>

<span data-ttu-id="7be4f-127">Adres IPv6 emisji pojedynczej jest podzielona na dwie części.</span><span class="sxs-lookup"><span data-stu-id="7be4f-127">An IPv6 unicast address is divided into two parts.</span></span> <span data-ttu-id="7be4f-128">Pierwsza część zawiera prefiks adresu, a druga część identyfikatora interfejsu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-128">The first part contains the address prefix, and the second part contains the interface identifier.</span></span> <span data-ttu-id="7be4f-129">Zwięzły sposób wyrażania kombinację adresu/prefiks IPv6 jest następująca: ipv6 — adres/długością prefiksu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-129">A concise way to express an IPv6 address/prefix combination is as follows: ipv6-address/prefix-length.</span></span>

<span data-ttu-id="7be4f-130">Oto przykład adresu z prefiksem 64-bitowych.</span><span class="sxs-lookup"><span data-stu-id="7be4f-130">The following is an example of an address with a 64-bit prefix.</span></span>

<span data-ttu-id="7be4f-131">`3FFE:FFFF:0:CD30:0:0:0:0/64`.</span><span class="sxs-lookup"><span data-stu-id="7be4f-131">`3FFE:FFFF:0:CD30:0:0:0:0/64`.</span></span>

<span data-ttu-id="7be4f-132">Prefiks, w tym przykładzie jest `3FFE:FFFF:0:CD30`.</span><span class="sxs-lookup"><span data-stu-id="7be4f-132">The prefix in this example is `3FFE:FFFF:0:CD30`.</span></span> <span data-ttu-id="7be4f-133">Adres można również będą zapisywane w postaci skompresowanej jako `3FFE:FFFF:0:CD30::/64`.</span><span class="sxs-lookup"><span data-stu-id="7be4f-133">The address can also be written in a compressed form, as `3FFE:FFFF:0:CD30::/64`.</span></span>

<span data-ttu-id="7be4f-134">Protokół IPv6 definiuje następujące typy adresów:</span><span class="sxs-lookup"><span data-stu-id="7be4f-134">IPv6 defines the following address types:</span></span>

- <span data-ttu-id="7be4f-135">**Adres emisji pojedynczej**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-135">**Unicast address**.</span></span> <span data-ttu-id="7be4f-136">Identyfikator jednego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-136">An identifier for a single interface.</span></span> <span data-ttu-id="7be4f-137">Pakiet wysyłane na ten adres jest przekazywana do interfejsie zidentyfikowany.</span><span class="sxs-lookup"><span data-stu-id="7be4f-137">A packet sent to this address is delivered to the identified interface.</span></span> <span data-ttu-id="7be4f-138">Adresy emisji pojedynczej różnią się od są adresami multiemisji przez wartość octet wyższego rzędu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-138">The unicast addresses are distinguished from the multicast addresses by the value of the high-order octet.</span></span> <span data-ttu-id="7be4f-139">Oktet adresy multiemisji z wartością szesnastkowe FF.</span><span class="sxs-lookup"><span data-stu-id="7be4f-139">The multicast addresses' high-order octet has the hexadecimal value of FF.</span></span> <span data-ttu-id="7be4f-140">Dowolna inna wartość do tego oktetu identyfikuje adresu emisji pojedynczej.</span><span class="sxs-lookup"><span data-stu-id="7be4f-140">Any other value for this octet identifies a unicast address.</span></span> <span data-ttu-id="7be4f-141">Poniżej przedstawiono różne rodzaje adresy emisji pojedynczej:</span><span class="sxs-lookup"><span data-stu-id="7be4f-141">The following are different types of unicast addresses:</span></span>

  - <span data-ttu-id="7be4f-142">**Adresów połączenia lokalnego**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-142">**Link-local addresses**.</span></span> <span data-ttu-id="7be4f-143">Te adresy są używane na jedno łącze i mają następujący format: FE80::*InterfaceID*.</span><span class="sxs-lookup"><span data-stu-id="7be4f-143">These addresses are used on a single link and have the following format: FE80::*InterfaceID*.</span></span> <span data-ttu-id="7be4f-144">Adresów lokalnych łącza są używane między węzłami w link do konfiguracji adres automatycznego odnajdowania sąsiadów lub w przypadku routerów nie istnieje.</span><span class="sxs-lookup"><span data-stu-id="7be4f-144">Link-local addresses are used between nodes on a link for auto-address configuration, neighbor discovery, or when no routers are present.</span></span> <span data-ttu-id="7be4f-145">Adres połączenia lokalnego jest używany głównie podczas uruchamiania i system nie uzyskał jeszcze adresów o większym zakresie.</span><span class="sxs-lookup"><span data-stu-id="7be4f-145">A link-local address is used primarily at startup and when the system has not yet acquired addresses of larger scope.</span></span>

  - <span data-ttu-id="7be4f-146">**Adresy lokacji lokalnej**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-146">**Site-local addresses**.</span></span> <span data-ttu-id="7be4f-147">Te adresy są używane w jednej lokacji i mają następujący format: FEC0::*SubnetID*:*InterfaceID*.</span><span class="sxs-lookup"><span data-stu-id="7be4f-147">These addresses are used on a single site and have the following format: FEC0::*SubnetID*:*InterfaceID*.</span></span> <span data-ttu-id="7be4f-148">Adresy lokacji lokalnej są używane na potrzeby adresowania wewnątrz witryny bez konieczności stosowania prefiksu globalnego.</span><span class="sxs-lookup"><span data-stu-id="7be4f-148">The site-local addresses are used for addressing inside a site without the need for a global prefix.</span></span>

  - <span data-ttu-id="7be4f-149">**Adresy globalne emisji pojedynczej protokołu IPv6**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-149">**Global IPv6 unicast addresses**.</span></span> <span data-ttu-id="7be4f-150">Te adresy mogą być używane w Internecie i ma następujący format: 010 (FP 3 bity) identyfikator TLA (13-bitowy) zarezerwowany (8 bitów), identyfikator umowy SLA ID NLA (24 bity) (16 bitów) *InterfaceID* (64-bitowy).</span><span class="sxs-lookup"><span data-stu-id="7be4f-150">These addresses can be used across the Internet and have the following format: 010(FP, 3 bits) TLA ID (13 bits) Reserved (8 bits) NLA ID (24 bits) SLA ID (16 bits) *InterfaceID* (64 bits).</span></span>

- <span data-ttu-id="7be4f-151">**Adres multiemisji**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-151">**Multicast address**.</span></span> <span data-ttu-id="7be4f-152">Identyfikator zestawu interfejsów (zazwyczaj należących do różnych węzłach).</span><span class="sxs-lookup"><span data-stu-id="7be4f-152">An identifier for a set of interfaces (typically belonging to different nodes).</span></span> <span data-ttu-id="7be4f-153">Pakiet wysyłane na ten adres jest dostarczane do wszystkich interfejsów, które są identyfikowane za pomocą adresu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-153">A packet sent to this address is delivered to all the interfaces identified by the address.</span></span> <span data-ttu-id="7be4f-154">Typy adresów multiemisji zastępują adresów IPv4 emisji.</span><span class="sxs-lookup"><span data-stu-id="7be4f-154">The multicast address types supersede the IPv4 broadcast addresses.</span></span>

- <span data-ttu-id="7be4f-155">**Adres emisji dowolnej**.</span><span class="sxs-lookup"><span data-stu-id="7be4f-155">**Anycast address**.</span></span> <span data-ttu-id="7be4f-156">Identyfikator zestawu interfejsów (zazwyczaj należących do różnych węzłach).</span><span class="sxs-lookup"><span data-stu-id="7be4f-156">An identifier for a set of interfaces (typically belonging to different nodes).</span></span> <span data-ttu-id="7be4f-157">Pakiet wysyłane na ten adres jest przekazywana do tylko jeden interfejs identyfikowanych przez ten adres.</span><span class="sxs-lookup"><span data-stu-id="7be4f-157">A packet sent to this address is delivered to only one interface identified by the address.</span></span> <span data-ttu-id="7be4f-158">Jest to interfejs najbliższej identyfikowane za pomocą metryk routingu.</span><span class="sxs-lookup"><span data-stu-id="7be4f-158">This is the nearest interface as identified by routing metrics.</span></span> <span data-ttu-id="7be4f-159">Adresy emisji dowolnej są pobierane z przestrzeni adresowej emisji pojedynczej i są nie do odróżnienia składniowo.</span><span class="sxs-lookup"><span data-stu-id="7be4f-159">Anycast addresses are taken from the unicast address space and are not syntactically distinguishable.</span></span> <span data-ttu-id="7be4f-160">Interfejs zaadresowane wykonuje rozróżnienie między adresy emisji pojedynczej i emisji dowolnej, w zależności od jego konfiguracji.</span><span class="sxs-lookup"><span data-stu-id="7be4f-160">The addressed interface performs the distinction between unicast and anycast addresses as a function of its configuration.</span></span>
