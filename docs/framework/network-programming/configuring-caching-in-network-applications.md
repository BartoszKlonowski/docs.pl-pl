# <a name="configuring-caching-in-network-applications"></a>Konfigurowanie pamięci podręcznej w aplikacjach sieciowych
Aby skonfigurować buforowanie, należy określić zasady pamięci podręcznej na poziomie aplikacji <xref:System.Net.WebRequest> lub poziomu. W poniższych tematach przedstawiono przykłady kodu, które pokazują, jak skonfigurować aplikacje i żądania do korzystania z pamięci podręcznej.  
  
- [Instrukcje: Ustawianie zasad pamięci podręcznej na podstawie lokalizacji dla aplikacji](how-to-set-a-location-based-cache-policy-for-an-application.md)  
  
- [Instrukcje: Ustawianie domyślnych zasad pamięci podręcznej na podstawie czasu dla aplikacji](how-to-set-the-default-time-based-cache-policy-for-an-application.md)  
  
- [Instrukcje: Dostosowywanie zasad pamięci podręcznej opartej na czasie](how-to-customize-a-time-based-cache-policy.md)  
  
- [Instrukcje: Ustawianie zasad pamięci podręcznej dla żądania](how-to-set-cache-policy-for-a-request.md)  
  
 Zasady pamięci podręcznej można także skonfigurować przy użyciu plików konfiguracji aplikacji lub komputera. Aby uzyskać więcej informacji, &#124; zobacz [ \<requestCaching > element (Ustawienia sieci)](../configure-apps/file-schema/network/requestcaching-element-network-settings.md).
