# <a name="configuring-caching-in-network-applications"></a>Konfigurowanie buforowanie w aplikacjach sieci
Aby skonfigurować buforowanie, należy określić zasady pamięci podręcznej w aplikacji lub <xref:System.Net.WebRequest> poziom. Poniższe tematy zawierają przykłady kodu, które przedstawiają Konfigurowanie aplikacji i żądań pamięci podręcznej.  
  
-   [Porady: Ustawianie zasad na podstawie lokalizacji pamięci podręcznej dla aplikacji](../../../docs/framework/network-programming/how-to-set-a-location-based-cache-policy-for-an-application.md)  
  
-   [Porady: ustawienie domyślnych zasad na podstawie czasu pamięci podręcznej dla aplikacji](../../../docs/framework/network-programming/how-to-set-the-default-time-based-cache-policy-for-an-application.md)  
  
-   [Porady: Dostosowywanie zasad na podstawie czasu pamięci podręcznej](../../../docs/framework/network-programming/how-to-customize-a-time-based-cache-policy.md)  
  
-   [Porady: ustawienie zasad pamięci podręcznej na żądanie](../../../docs/framework/network-programming/how-to-set-cache-policy-for-a-request.md)  
  
 Można również skonfigurować zasady pamięci podręcznej za pomocą aplikacji lub plików konfiguracji maszyny. Aby uzyskać więcej informacji, zobacz &#124; [ \<requestCaching — > elementu (ustawienia sieciowe)](../../../docs/framework/configure-apps/file-schema/network/requestcaching-element-network-settings.md).
