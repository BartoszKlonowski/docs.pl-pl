# <a name="httplistener"></a>HttpListener
<xref:System.Net.HttpListener> Klasa zapewnia programistyczny, kontrolowany przez program odbiornik protokołu HTTP. Odbiornik jest aktywny przez okres istnienia <xref:System.Net.HttpListener> obiektu i działa w aplikacji.  
  
## <a name="httpsys"></a>HTTP.SYS  
 <xref:System.Net.HttpListener> Klasa jest oparta na protokole HTTP. sys, który jest odbiornikiem trybu jądra, który obsługuje cały ruch HTTP dla systemu Windows. HTTP. sys umożliwia zarządzanie połączeniami, ograniczanie przepustowości i rejestrowanie serwera sieci Web. Użyj narzędzia `HttpCfg.exe` , aby dodać certyfikaty SSL. Aby uzyskać więcej informacji, zobacz dokumentację dotyczącą narzędzia [HttpCfg. exe](https://go.microsoft.com/fwlink/?LinkID=178284) w dokumentacji [serwera](https://go.microsoft.com/fwlink/?LinkID=178285) .
