## <a name="pnrp-ids"></a><span data-ttu-id="670c4-118">Identyfikatory PNRP</span><span class="sxs-lookup"><span data-stu-id="670c4-118">PNRP IDs</span></span>  
 <span data-ttu-id="670c4-119">![Identyfikator PNRP](../../../docs/framework/network-programming/media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span><span class="sxs-lookup"><span data-stu-id="670c4-119">![PNRP ID](../../../docs/framework/network-programming/media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span></span>  
  
 <span data-ttu-id="670c4-120">Identyfikatory PNRP składają się z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="670c4-120">PNRP IDs are composed of the following:</span></span>  
  
-   <span data-ttu-id="670c4-121">Znaczących 128 bitów — znane jako identyfikator (P2P) peer-to-peer, są skrót nazwy elementu równorzędnego przypisana do punktu końcowego.</span><span class="sxs-lookup"><span data-stu-id="670c4-121">The high-order 128 bits, known as the peer-to-peer (P2P) ID, are a hash of a peer name assigned to the endpoint.</span></span> <span data-ttu-id="670c4-122">Nazwa elementu równorzędnego ma następujący format: *Authority.Classifier*.</span><span class="sxs-lookup"><span data-stu-id="670c4-122">The peer name has the following format: *Authority.Classifier*.</span></span> <span data-ttu-id="670c4-123">Dla nazwy zabezpieczonych *urzędu* jest skrótu Secure Hash Algorithm 1 (SHA1) klucz publiczny nazwa elementu równorzędnego w znaki szesnastkowe.</span><span class="sxs-lookup"><span data-stu-id="670c4-123">For secured names, *Authority* is the Secure Hash Algorithm 1 (SHA1) hash of the public key of the peer name in hexadecimal characters.</span></span> <span data-ttu-id="670c4-124">Niezabezpieczona nazw *urzędu* jest pojedynczy znak "0".</span><span class="sxs-lookup"><span data-stu-id="670c4-124">For unsecured names, the *Authority* is the single character "0".</span></span> <span data-ttu-id="670c4-125">*Klasyfikator* jest ciągiem, który identyfikuje aplikację.</span><span class="sxs-lookup"><span data-stu-id="670c4-125">*Classifier* is a string that identifies the application.</span></span> <span data-ttu-id="670c4-126">Nie klasyfikatora nazwa elementu równorzędnego może być większa niż 149 znaków, łącznie z `null` terminator.</span><span class="sxs-lookup"><span data-stu-id="670c4-126">No peer name classifier can be greater than 149 characters long, including the `null` terminator.</span></span>  
  
-   <span data-ttu-id="670c4-127">128 bitów znaczącymi bitami są używane do lokalizacji usługi, która jest wygenerowany liczba, która identyfikuje różnych wystąpień tego samego identyfikatora P2P w tej samej chmury.</span><span class="sxs-lookup"><span data-stu-id="670c4-127">The low-order 128 bits are used for the Service Location, which is a generated number that identifies different instances of the same P2P ID in the same cloud.</span></span>  
  
 <span data-ttu-id="670c4-128">Ta kombinacja Identyfikatora P2P i lokalizację usługi umożliwia wielu identyfikatorów PNRP rejestrowana z jednego komputera.</span><span class="sxs-lookup"><span data-stu-id="670c4-128">This combination of P2P ID and Service Location allows multiple PNRP IDs to be registered from a single computer.</span></span>
