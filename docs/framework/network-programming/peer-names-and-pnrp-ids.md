## <a name="pnrp-ids"></a><span data-ttu-id="60022-118">Identyfikatory PNRP</span><span class="sxs-lookup"><span data-stu-id="60022-118">PNRP IDs</span></span>

 <span data-ttu-id="60022-120">Identyfikatory PNRP składają się z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="60022-120">PNRP IDs are composed of the following:</span></span>
