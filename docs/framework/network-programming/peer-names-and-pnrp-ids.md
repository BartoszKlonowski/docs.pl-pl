## <a name="pnrp-ids"></a><span data-ttu-id="e56f1-118">Identyfikatory PNRP</span><span class="sxs-lookup"><span data-stu-id="e56f1-118">PNRP IDs</span></span>

 <span data-ttu-id="e56f1-120">Identyfikatory PNRP składają się z następujących elementów:</span><span class="sxs-lookup"><span data-stu-id="e56f1-120">PNRP IDs are composed of the following:</span></span>  
  
- <span data-ttu-id="e56f1-121">128 bitów wysokiego rzędu, znany jako peer-to-peer (P2P) ID, są skrótem nazwy elementu równorzędnego przypisanego do punktu końcowego.</span><span class="sxs-lookup"><span data-stu-id="e56f1-121">The high-order 128 bits, known as the peer-to-peer (P2P) ID, are a hash of a peer name assigned to the endpoint.</span></span> <span data-ttu-id="e56f1-122">Nazwa elementu równorzędnego ma następujący format: *Authority.Classifier*.</span><span class="sxs-lookup"><span data-stu-id="e56f1-122">The peer name has the following format: *Authority.Classifier*.</span></span> <span data-ttu-id="e56f1-123">W przypadku zabezpieczonych nazw *urząd* jest skrótem bezpiecznego algorytmu mieszania 1 (SHA1) klucza publicznego nazwy elementu równorzędnego w znakach szesnastkowych.</span><span class="sxs-lookup"><span data-stu-id="e56f1-123">For secured names, *Authority* is the Secure Hash Algorithm 1 (SHA1) hash of the public key of the peer name in hexadecimal characters.</span></span> <span data-ttu-id="e56f1-124">W przypadku nazw niezabezpieczonych *Urząd* jest pojedynczym znakiem "0".</span><span class="sxs-lookup"><span data-stu-id="e56f1-124">For unsecured names, the *Authority* is the single character "0".</span></span> <span data-ttu-id="e56f1-125">*Klasyfikator* jest ciągiem identyfikującym aplikację.</span><span class="sxs-lookup"><span data-stu-id="e56f1-125">*Classifier* is a string that identifies the application.</span></span> <span data-ttu-id="e56f1-126">Żaden klasyfikator nazw równorzędnych nie może być `null` większy niż 149 znaków, łącznie z terminatorem.</span><span class="sxs-lookup"><span data-stu-id="e56f1-126">No peer name classifier can be greater than 149 characters long, including the `null` terminator.</span></span>
