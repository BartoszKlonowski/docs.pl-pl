1.  <span data-ttu-id="0a278-115">Użyj poniższego kodu, aby utworzyć nową klasę wyjątków, o nazwie `NoAccessException` i mapowanie ich na HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="0a278-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>
