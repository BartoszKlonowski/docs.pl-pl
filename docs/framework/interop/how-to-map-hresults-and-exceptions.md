### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="bc780-114">Tworzenie nowej klasy wyjątku i Mapuj na HRESULT</span><span class="sxs-lookup"><span data-stu-id="bc780-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1. <span data-ttu-id="bc780-115">Użyj poniższego kodu, aby utworzyć nową klasę wyjątków, o nazwie `NoAccessException` i mapowanie ich na HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="bc780-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
        NoAccessException () {

    throw new NoAccessException();  
    }  
    ```
