 <span data-ttu-id="cd45c-227">Po ustawieniu na 1 flaga COMPLUS_LoaderOptimization środowiska wymusza hosta środowiska wykonawczego, aby załadować wszystkie zestawy w sposób nieo neutralny dla domeny znany jako SingleDomain.</span><span class="sxs-lookup"><span data-stu-id="cd45c-227">When set to 1, the COMPLUS_LoaderOptimization environment flag forces the runtime host to load all assemblies in non-domain-neutral way known as SingleDomain.</span></span> <span data-ttu-id="cd45c-228">SingleDomain nie ładuje żadnych zestawów jako neutralnych dla domeny, z wyjątkiem mscorlib, który jest zawsze ładowany neutralny domeny.</span><span class="sxs-lookup"><span data-stu-id="cd45c-228">SingleDomain loads no assemblies as domain-neutral, except Mscorlib, which is always loaded domain-neutral.</span></span> <span data-ttu-id="cd45c-229">To ustawienie jest nazywane pojedynczą domeną, ponieważ jest często używane, gdy na hoście jest uruchomiona tylko jedna aplikacja w procesie.</span><span class="sxs-lookup"><span data-stu-id="cd45c-229">This setting is called single domain because it is commonly used when the host is running only a single application in the process.</span></span>
