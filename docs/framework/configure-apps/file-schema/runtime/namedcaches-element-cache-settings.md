## <a name="example"></a><span data-ttu-id="99ee7-138">Przykład</span><span class="sxs-lookup"><span data-stu-id="99ee7-138">Example</span></span>  

 <span data-ttu-id="99ee7-139">Poniższy przykład pokazuje, jak ustawić nazwę pamięci podręcznej na domyślną nazwę wpisu pamięci podręcznej przez ustawienie `name` atrybutu na wartość "domyślny".</span><span class="sxs-lookup"><span data-stu-id="99ee7-139">The following example shows how to set the name of the cache to the default cache entry name by setting the `name` attribute to "default".</span></span>  
  
 <span data-ttu-id="99ee7-140">`cacheMemoryLimitMegabytes`Atrybut i `physicalMemoryPercentage` atrybut są ustawione na zero.</span><span class="sxs-lookup"><span data-stu-id="99ee7-140">The `cacheMemoryLimitMegabytes` attribute and the `physicalMemoryPercentage` attribute are set to zero.</span></span> <span data-ttu-id="99ee7-141">Ustawienie tych atrybutów na zero oznacza, że używane są heurystyke autowymiarowania <xref:System.Runtime.Caching.MemoryCache> klasy.</span><span class="sxs-lookup"><span data-stu-id="99ee7-141">Setting these attributes to zero means that the autosizing heuristics of the <xref:System.Runtime.Caching.MemoryCache> class are used.</span></span> <span data-ttu-id="99ee7-142">Implementacja pamięci podręcznej porównuje bieżące obciążenie pamięci dla limitów pamięci bezwzględnej i wartości procentowej co dwie minuty.</span><span class="sxs-lookup"><span data-stu-id="99ee7-142">The cache implementation compares the current memory load against the absolute and percentage-based memory limits every two minutes.</span></span>

```xml  
    <memoryCache>  
      <namedCaches>  
          <add name="default"
               cacheMemoryLimitMegabytes="0"
               physicalMemoryLimitPercentage="0"  
               pollingInterval="00:02:00" />  
      </namedCaches>  
    </memoryCache>  
  </system.runtime.caching>  
  
</configuration>  
```
