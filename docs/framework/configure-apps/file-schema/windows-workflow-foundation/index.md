# <a name="windows-workflow-foundation-configuration-schema"></a>Schemat konfiguracji Foundation przepływu pracy systemu Windows
[!INCLUDE[wf](../../../../../includes/wf-md.md)]elementy konfiguracji służą do konfigurowania aplikacji przepływu pracy. W przypadku aplikacji przepływu pracy można skonfigurować między innymi, śledzenia i śledzenia. Aby uzyskać więcej informacji na temat śledzenia i śledzenie, zobacz [śledzenia przepływu pracy i śledzenie](../../../../../docs/framework/windows-workflow-foundation/workflow-tracking-and-tracing.md). W przypadku usług przepływu pracy można także użyć [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] elementów konfiguracji. Aby uzyskać więcej informacji o [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)], zobacz [schemat konfiguracji programu WCF](../../../../../docs/framework/configure-apps/file-schema/wcf/index.md).  
  
 Ponieważ PLiki konfiguracyjne są w formacie XML, należy zapoznać z danymi XML, aby ręcznie edytować za pomocą edytora tekstu. W przeciwnym razie może wystąpić problemy, takie jak unfound tagu elementu XML lub atrybut. Wynika to z tagów elementu XML i atrybutów jest rozróżniana wielkość liter.
