# <a name="windows-workflow-foundation-configuration-schema"></a>Schemat konfiguracji programu Windows Workflow Foundation
Windows Workflow Foundation (WF), elementy konfiguracji umożliwiają skonfigurowanie aplikacji przepływu pracy. W przypadku aplikacji przepływ pracy można skonfigurować między innymi, kontrola i śledzenie. Aby uzyskać więcej informacji o śledzeniu i śledzenia, zobacz [śledzenia przepływu pracy i śledzenia](../../../../../docs/framework/windows-workflow-foundation/workflow-tracking-and-tracing.md). W przypadku usług przepływu pracy umożliwia również elementy konfiguracji usług Windows Communication Foundation (WCF). Aby uzyskać więcej informacji o technologii WCF, zobacz [schemat konfiguracji programu WCF](../../../../../docs/framework/configure-apps/file-schema/wcf/index.md).  
  
 Ponieważ PLiki konfiguracyjne są w formacie XML, należy zapoznać z danymi XML, aby ręcznie edytować za pomocą edytora tekstu. W przeciwnym razie może wystąpić problemy, takie jak unfound tagu elementu XML lub atrybut. Wynika to z tagów elementu XML i atrybutów jest rozróżniana wielkość liter.
