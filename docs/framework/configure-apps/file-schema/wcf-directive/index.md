# <a name="wcf-directive-syntax"></a>Składnia dyrektyw programu WCF

Określa ustawienia używane przez dyrektywy Windows Communication Foundation (WCF) w plikach SVC do kierowania kompilatorów. Każda dyrektywa może zawierać jeden lub więcej atrybutów (par z wartościami), które są specyficzne dla tej dyrektywy. Funkcja WCF ma tylko dyrektywę [ \@ ServiceHost](servicehost.md) .  
  
## <a name="in-this-section"></a>W tej sekcji  

 [@ServiceHost](servicehost.md)  
 Definiuje atrybuty specyficzne dla strony używane przez kompilator SVC. Mogą być dołączane tylko do plików SVC.
