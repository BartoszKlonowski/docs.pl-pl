## <a name="example"></a><span data-ttu-id="30ca0-154">Przykład</span><span class="sxs-lookup"><span data-stu-id="30ca0-154">Example</span></span>  
 <span data-ttu-id="30ca0-155">Poniższy kod XML przedstawia `<federationConfiguration>` element określający ustawienia dla WSFAM i określa, że domyślny program obsługi plików cookie (wystąpienie <xref:System.IdentityModel.Services.ChunkedCookieHandler> klasy) będzie używany przez sam.</span><span class="sxs-lookup"><span data-stu-id="30ca0-155">The following XML shows a `<federationConfiguration>` element that specifies settings for the WSFAM and specifies that the default cookie handler (an instance of the <xref:System.IdentityModel.Services.ChunkedCookieHandler> class) be used by the SAM.</span></span>  
  
> [!WARNING]
> <span data-ttu-id="30ca0-156">W tym przykładzie żadna procedura obsługi plików cookie i WSFAM nie jest wymagana do korzystania z protokołu HTTPS.</span><span class="sxs-lookup"><span data-stu-id="30ca0-156">In this example, neither the cookie handler nor WSFAM are required to use HTTPS.</span></span> <span data-ttu-id="30ca0-157">Jest to spowodowane tym, że `requireHttps` atrybut `<wsFederation>` elementu i `requireSsl` atrybutu na `<cookieHandlerElement>` `false` .</span><span class="sxs-lookup"><span data-stu-id="30ca0-157">This is because the `requireHttps` attribute on the `<wsFederation>` element and the `requireSsl` attribute on the `<cookieHandlerElement>` are `false`.</span></span> <span data-ttu-id="30ca0-158">Te ustawienia nie są zalecane w przypadku większości środowisk produkcyjnych, ponieważ mogą one stanowić zagrożenie dla bezpieczeństwa.</span><span class="sxs-lookup"><span data-stu-id="30ca0-158">These settings are not recommended for most production environments as they may present a security risk.</span></span>

      signOutReply="http://localhost:50969/SignedOutPage.html"
