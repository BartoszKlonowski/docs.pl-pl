## <a name="example"></a><span data-ttu-id="95bc5-162">Przykład</span><span class="sxs-lookup"><span data-stu-id="95bc5-162">Example</span></span>

> [!WARNING]
>  <span data-ttu-id="95bc5-164">W tym przykładzie obsługa plików cookie ani WSFAM są wymagane do używania protokołu HTTPS.</span><span class="sxs-lookup"><span data-stu-id="95bc5-164">In this example, neither the cookie handler nor WSFAM are required to use HTTPS.</span></span> <span data-ttu-id="95bc5-165">Jest to spowodowane `requireHttps` atrybutu `<wsFederation>` elementu i `requireSsl` atrybutu `<cookieHandlerElement>` są `false`.</span><span class="sxs-lookup"><span data-stu-id="95bc5-165">This is because the `requireHttps` attribute on the `<wsFederation>` element and the `requireSsl` attribute on the `<cookieHandlerElement>` are `false`.</span></span> <span data-ttu-id="95bc5-166">Te ustawienia nie są zalecane dla większości środowisk produkcyjnych, ponieważ mogą one stanowić zagrożenie bezpieczeństwa.</span><span class="sxs-lookup"><span data-stu-id="95bc5-166">These settings are not recommended for most production environments as they may present a security risk.</span></span>
