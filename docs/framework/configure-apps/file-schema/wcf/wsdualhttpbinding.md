 <span data-ttu-id="79199-207">To powiązanie może służyć do niezawodnego komunikowania się przez jeden lub więcej pośredników SOAP.</span><span class="sxs-lookup"><span data-stu-id="79199-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>
