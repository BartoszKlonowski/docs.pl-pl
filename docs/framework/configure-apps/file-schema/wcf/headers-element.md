# <a name="ltheadersgt-element"></a>&lt;headers&gt;, element
Zawiera kolekcję nagłówków adresowych, które są wymagane do prawidłowego adresowania wystawcy lokalnego. Można użyć `add` — słowo kluczowe, aby dodać nagłówek do tej kolekcji.
