# <a name="headers-element"></a>\<nagłówki > element
Zawiera kolekcję nagłówków adresowych, które są wymagane do prawidłowego adresowania wystawcy lokalnego. Możesz użyć `add` — słowo kluczowe, aby dodać nagłówek do tej kolekcji.
