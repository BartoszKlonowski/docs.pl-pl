# <a name="headers-element"></a>\<headers>, element
Zawiera kolekcję nagłówków adresów, które są wymagane w celu prawidłowego adresowania wystawcy lokalnego. Możesz użyć `add` słowa kluczowego, aby dodać nagłówek do tej kolekcji.
