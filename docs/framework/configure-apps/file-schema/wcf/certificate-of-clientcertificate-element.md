## <a name="example"></a><span data-ttu-id="d9089-167">Przykład</span><span class="sxs-lookup"><span data-stu-id="d9089-167">Example</span></span>  
 <span data-ttu-id="d9089-168">Poniższy kod określa, jak można znaleźć odpowiedniego certyfikatu X.509 i niestandardowego sprawdzania poprawności typu w `<authentication>` elementu.</span><span class="sxs-lookup"><span data-stu-id="d9089-168">The following code specifies how to find an appropriate X.509 certificate and a custom validation type in the `<authentication>` element.</span></span>

    <clientCertificate>
      <certificate findValue="www.cohowinery.com"

                   x509FindType="FindByIssuerName" />

                      certificateValidationMode="Custom"
                      revocationMode="Offline"
                      includeWindowsGroups="false"
                      mapClientCertificateToWindowsAccount="true" />
    </clientCertificate>
  </behavior>
