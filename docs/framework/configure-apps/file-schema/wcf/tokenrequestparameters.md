# <a name="tokenrequestparameters"></a>\<tokenRequestParameters>
Ten element konfiguracji zawiera elementy podrzędne, które mogą służyć do określenia dodatkowych elementów XML do wysłania w treści komunikatu do usługi tokenu zabezpieczeń podczas żądania tokenu.
