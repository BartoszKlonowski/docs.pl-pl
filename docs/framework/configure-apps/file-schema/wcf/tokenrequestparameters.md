# <a name="tokenrequestparameters"></a>\<tokenRequestParameters>
Ten element konfiguracji zawiera elementy podrzędne, których można użyć do określenia dodatkowych elementów XML do wysłania w treści wiadomości do usługi tokenu zabezpieczającego podczas żądania tokenu.
