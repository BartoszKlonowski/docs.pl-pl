# <a name="ltprivacynoticeatgt"></a>&lt;privacyNoticeAt&gt;
Reprezentuje element konfiguracji, który określa zachowania poufności, używane w `wsFederationHttp` powiązania.  
  
 \<system.serviceModel>  
\<powiązania >  
\<customBinding>  
\<Powiązanie >  
\<privacyNotice >  
  
## <a name="syntax"></a>Składnia  
  
```xml  
<privacyNotice url="String"
               version="Integer" />
```  
  
## <a name="type"></a>Typ  
 `Type`  
  
## <a name="attributes-and-elements"></a>Atrybuty i elementy  
 W poniższych sekcjach opisano atrybuty, elementy podrzędne i elementy nadrzędne.  
  
### <a name="attributes"></a>Atrybuty  
  
|Atrybut|Opis|  
|---------------|-----------------|  
|`url`|Ciąg określający URI, w której znajduje się powiadomienie dotyczące prywatności.|  
|`version`|Liczba całkowita określająca wersję tego zawiadomienia o prywatności.|  
  
### <a name="child-elements"></a>Elementy podrzędne  
 Brak.  
  
### <a name="parent-elements"></a>Elementy nadrzędne  
  
|Element|Opis|  
|-------------|-----------------|  
|[\<Powiązanie >](../../../../../docs/framework/misc/binding.md)|Definiuje wszystkie funkcje powiązania niestandardowego powiązania.|
