## <a name="partially-qualified-assembly-names-and-side-by-side-execution"></a><span data-ttu-id="cd3fe-190">Częściowo kwalifikowane nazwy zestawów a wykonywanie równoczesne</span><span class="sxs-lookup"><span data-stu-id="cd3fe-190">Partially Qualified Assembly Names and Side-by-Side Execution</span></span>
