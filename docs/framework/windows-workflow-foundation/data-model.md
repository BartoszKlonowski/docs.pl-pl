# <a name="windows-workflow-foundation-data-model"></a>Model danych programu Windows Workflow Foundation

Model danych Windows Workflow Foundation składa się z trzech koncepcji: zmiennych, argumentów i wyrażeń. Zmienne reprezentują magazyn danych i argumenty reprezentują przepływ danych do działania i z niego. Argumenty są powiązane (przypisana wartość) za pomocą wyrażeń, które mogą odwoływać się do zmiennych.
