# <a name="windows-workflow-foundation-data-model"></a>Model danych programu Windows Workflow Foundation
Model danych programu Windows Workflow Foundation składa się z trzech pojęć: zmienne, argumenty i wyrażeń. Zmienne reprezentują magazynowania danych, a argumenty reprezentują przepływ danych do i z działania. Argumenty są powiązane (przypisana wartość) przy użyciu wyrażeń, które mogą odwoływać się do zmiennych.
