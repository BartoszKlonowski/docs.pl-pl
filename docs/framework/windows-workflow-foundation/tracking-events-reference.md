# <a name="tracking-events-reference"></a>Dokumentacja zdarzeń śledzenia

Podczas wykonywania przepływu pracy w programie istnieją [!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] zdarzenia śledzenia, które są przenoszone przez różne etapy okresu istnienia. Host może subskrybować te zdarzenia i aktualizować stan postępu przepływu pracy w trakcie jego okresu istnienia. Zdarzenia śledzenia zgłoszone zostały omówione w tej sekcji.
