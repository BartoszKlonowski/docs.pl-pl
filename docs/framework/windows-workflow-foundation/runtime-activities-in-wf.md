# <a name="runtime-activities-in-wf"></a>Działania środowiska uruchomieniowego w programie WF

[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] zapewnia kilka działań dostarczonych przez system do uzyskiwania dostępu do funkcji środowiska uruchomieniowego przepływu pracy, takich jak trwałość i zakończenie.  
  
|Działanie|Opis|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Persist>|Jawnie żądania są utrwalane przez przepływ pracy.|  
|<xref:System.Activities.Statements.TerminateWorkflow>|Kończy uruchomione wystąpienie przepływu pracy.|  
|<xref:System.Activities.Statements.NoPersistScope>|Działanie kontenera, które uniemożliwia utrwalanie działań podrzędnych.|
