# <a name="1223---switchcasenotfound"></a>1223 — SwitchCaseNotFound
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|1223|  
|słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 Wskazuje, że nie można odnaleźć case działania z wynikiem wyrażenia dopasowania w działaniu przełącznika.  
  
## <a name="message"></a>Komunikat  
 Działanie przełącznika "%1" nie można odnaleźć działania wielkości liter, wynik wyrażenia dopasowania.

|Nazwa elementu danych|Typ elementu danych|Opis|  
|--------------------|--------------------|-----------------|  
|Nazwa wyświetlana|xs:String|Nazwa wyświetlana działania.|  
|AppDomain|xs:String|Ciąg zwracany przez AppDomain.CurrentDomain.FriendlyName.|
