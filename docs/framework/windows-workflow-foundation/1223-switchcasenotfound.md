# <a name="1223---switchcasenotfound"></a>1223 — SwitchCaseNotFound

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|1223|  
|Słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 Wskazuje działanie Case z pasującym wynikiem wyrażenia nie zostało znalezione w działaniu Switch.  
  
## <a name="message"></a>Wiadomość  

 Działanie Switch ' %1 ' nie może znaleźć działania Case pasującego do wyniku wyrażenia.

|Wywołując|XS: ciąg|Ciąg zwracany przez element AppDomain. CurrentDomain —. FriendlyName.|
