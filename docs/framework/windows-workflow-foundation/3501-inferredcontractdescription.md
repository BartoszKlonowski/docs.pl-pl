# <a name="3501---inferredcontractdescription"></a>3501 — InferredContractDescription
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|3501|  
|słowa kluczowe|WFServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Opis  
 Wskazuje, że element ContractDescription została wykryta z WorkflowService.  
  
## <a name="message"></a>Komunikat  
 Element ContractDescription o nazwie = "%1" i Namespace = "%2" została wykryta z WorkflowService.
