|-|-|  
|ID (Identyfikator)|3501|  
|Słowa kluczowe|WFServices|  
|Poziom|Informacje|

## <a name="description"></a>Opis  

 Wskazuje, że element ContractDescription został wywnioskowany z obiektu WorkflowService.

 Element ContractDescription o nazwie "%1" i przestrzeni nazw "%2" został wywnioskowany na podstawie obiektu WorkflowService.
