# <a name="3501---inferredcontractdescription"></a>3501 - InferredContractDescription
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|3501|  
|Słowa kluczowe|WFServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows aplikacji Server aplikacje/analityczne|  
  
## <a name="description"></a>Opis  
 Wskazuje, że element ContractDescription został wywnioskowany na podstawie obiektu WorkflowService.  
  
## <a name="message"></a>Komunikat  
 Element ContractDescription o nazwie "%1" i Namespace = "%2" został wywnioskowany na podstawie obiektu WorkflowService.  
  
## <a name="details"></a>Szczegóły  
  
|Nazwa elementu danych|Typ elementu danych|Opis|  
|--------------------|--------------------|-----------------|  
|Nazwa|xs:String|Nazwa elementu ContractDescription.|  
|Przestrzeń nazw|xs:String|Przestrzeń nazw ContractDescription.|  
|Domeny aplikacji|xs:String|Długość ciągu zwróconego przez AppDomain.CurrentDomain.FriendlyName.|
