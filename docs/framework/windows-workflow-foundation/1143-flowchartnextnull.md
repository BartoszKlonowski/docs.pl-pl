# <a name="1143---flowchartnextnull"></a>1143 — FlowchartNextNull

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|1143|  
|Słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 Wskazuje, że następny węzeł w schemacie blokowym ma wartość null. Wykonanie schematu blokowego zakończy się.  
  
## <a name="message"></a>Wiadomość  

 Flowchart "%1"/FlowStep — następny węzeł ma wartość null. Wykonanie schematu blokowego zakończy się.
