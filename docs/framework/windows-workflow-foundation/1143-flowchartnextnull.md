# <a name="1143---flowchartnextnull"></a>1143 — FlowchartNextNull
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|1143|  
|słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 Wskazuje, że następny węzeł w elemencie Schemat blokowy ma wartość null. Schemat blokowy wykonywania zakończy się.  
  
## <a name="message"></a>Komunikat  
 Schemat blokowy "%1'/FlowStep - kolejnego węzła ma wartość null. Schemat blokowy wykonywania zakończy się.  
  
## <a name="details"></a>Szczegóły  
  
|Nazwa elementu danych|Typ elementu danych|Opis|  
|--------------------|--------------------|-----------------|  
|FlowChart|xs:String|Wyświetlana nazwa schematu blokowego.|  
|AppDomain|xs:String|Ciąg zwracany przez AppDomain.CurrentDomain.FriendlyName.|
