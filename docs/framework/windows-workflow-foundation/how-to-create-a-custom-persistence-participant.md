# <a name="how-to-create-a-custom-persistence-participant"></a><span data-ttu-id="dcf34-102">Instrukcje: Tworzenie niestandardowego uczestnika stanów trwałych</span><span class="sxs-lookup"><span data-stu-id="dcf34-102">How to: Create a Custom Persistence Participant</span></span>
<span data-ttu-id="dcf34-103">Poniższa procedura zawiera kroki, aby utworzyć uczestnika stanów trwałych.</span><span class="sxs-lookup"><span data-stu-id="dcf34-103">The following procedure has steps to create a persistence participant.</span></span> <span data-ttu-id="dcf34-104">Zobacz [uczestniczących w trwałości](https://go.microsoft.com/fwlink/?LinkID=177735) próbki i [rozszerzalności Store](store-extensibility.md) tematu w przykładowej implementacji uczestnicy stanów trwałych.</span><span class="sxs-lookup"><span data-stu-id="dcf34-104">See the [Participating in Persistence](https://go.microsoft.com/fwlink/?LinkID=177735) sample and [Store Extensibility](store-extensibility.md) topic for sample implementations of persistence participants.</span></span>  
  
1.  <span data-ttu-id="dcf34-105">Utwórz klasę pochodząca od <xref:System.Activities.Persistence.PersistenceParticipant> lub <xref:System.Activities.Persistence.PersistenceIOParticipant> klasy.</span><span class="sxs-lookup"><span data-stu-id="dcf34-105">Create a class deriving from the <xref:System.Activities.Persistence.PersistenceParticipant> or the <xref:System.Activities.Persistence.PersistenceIOParticipant> class.</span></span> <span data-ttu-id="dcf34-106">Klasa PersistenceIOParticipant zapewnia ten sam punkty rozszerzeń jako klasa PersistenceParticipant, oprócz możliwości udziału w operacji We/Wy.</span><span class="sxs-lookup"><span data-stu-id="dcf34-106">The PersistenceIOParticipant class offers the same extensibility points as the PersistenceParticipant class in addition to being able to participate in I/O operations.</span></span> <span data-ttu-id="dcf34-107">Postępuj zgodnie z co najmniej jeden z następujących czynności.</span><span class="sxs-lookup"><span data-stu-id="dcf34-107">Follow one or more of the following steps.</span></span>  
  
2.  <span data-ttu-id="dcf34-108">Implementowanie <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> metody.</span><span class="sxs-lookup"><span data-stu-id="dcf34-108">Implement the <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> method.</span></span> <span data-ttu-id="dcf34-109">**CollectValues** metoda ma słownika dwa parametry: jeden do przechowywania wartości odczytu/zapisu, a druga do przechowywania wartości tylko do zapisu (używane w dalszej części zapytania).</span><span class="sxs-lookup"><span data-stu-id="dcf34-109">The **CollectValues** method has two dictionary parameters, one for storing read/write values and the other one for storing write-only values (used later in queries).</span></span> <span data-ttu-id="dcf34-110">W przypadku tej metody należy wypełnić słowników z danymi, które są specyficzne dla uczestnika stanów trwałych.</span><span class="sxs-lookup"><span data-stu-id="dcf34-110">In this method, you should populate these dictionaries with data that is specific to a persistence participant.</span></span> <span data-ttu-id="dcf34-111">Każdy słownik zawiera nazwę wartości klucza oraz sama wartość jak <xref:System.Runtime.DurableInstancing.InstanceValue> obiektu.</span><span class="sxs-lookup"><span data-stu-id="dcf34-111">Each dictionary contains the name of the value as the key and the value itself as an <xref:System.Runtime.DurableInstancing.InstanceValue> object.</span></span>  
  
     <span data-ttu-id="dcf34-112">Wartości w słowniku readWriteValues są dostarczane w **InstanceValue** obiektów.</span><span class="sxs-lookup"><span data-stu-id="dcf34-112">The values in the readWriteValues dictionary are packaged as **InstanceValue** objects.</span></span> <span data-ttu-id="dcf34-113">Wartości w słowniku tylko do zapisu jest spakowany jako **InstanceValue** obiektów z InstanceValueOptions.Optional i InstanceValueOption.WriteOnly zestawu.</span><span class="sxs-lookup"><span data-stu-id="dcf34-113">The values in the write-only dictionary are packaged as **InstanceValue** objects with InstanceValueOptions.Optional and InstanceValueOption.WriteOnly set.</span></span> <span data-ttu-id="dcf34-114">Każdy **InstanceValue** dostarczone przez **CollectValues** implementacje we wszystkich uczestników stanu trwałego musi mieć unikatową nazwę.</span><span class="sxs-lookup"><span data-stu-id="dcf34-114">Each **InstanceValue** provided by the **CollectValues** implementations across all persistence participants must have a unique name.</span></span>  
  
    ```  
    protected virtual void CollectValues (out IDictionary<XName,Object> readWriteValues, out IDictionary<XName,Object> writeOnlyValues)  
    ```  
  
3.  <span data-ttu-id="dcf34-115">Implementowanie <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> metody.</span><span class="sxs-lookup"><span data-stu-id="dcf34-115">Implement the <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> method.</span></span> <span data-ttu-id="dcf34-116">**MapValues** metoda przyjmuje dwa parametry, które są podobne do parametrów, **CollectValues** metoda otrzymuje.</span><span class="sxs-lookup"><span data-stu-id="dcf34-116">The **MapValues** method takes two parameters that are similar to the parameters that the **CollectValues** method receives.</span></span> <span data-ttu-id="dcf34-117">Wszystkie wartości są zbierane w **CollectValues** etapu są przekazywane za pośrednictwem tych parametrów słownika.</span><span class="sxs-lookup"><span data-stu-id="dcf34-117">All the values collected in the **CollectValues** stage are passed through these dictionary parameters.</span></span> <span data-ttu-id="dcf34-118">Nowe wartości dodanej przez **MapValues** etapu są dodawane do wartości tylko do zapisu.</span><span class="sxs-lookup"><span data-stu-id="dcf34-118">The new values added by the **MapValues** stage are added to the write-only values.</span></span>  <span data-ttu-id="dcf34-119">Słownik tylko do zapisu jest używany do przekazywania danych do zewnętrznego źródła, nie są bezpośrednio związane z wartości wystąpienia.</span><span class="sxs-lookup"><span data-stu-id="dcf34-119">The write-only dictionary is used to provide data to an external source not directly associated with the instance values.</span></span> <span data-ttu-id="dcf34-120">Każdej wartości dostarczone przez implementacje **MapValues** metoda we wszystkich uczestników stanu trwałego musi mieć unikatową nazwę.</span><span class="sxs-lookup"><span data-stu-id="dcf34-120">Each value provided by implementations of the **MapValues** method across all persistence participants must have a unique name.</span></span>  
  
    ```  
    protected virtual IDictionary<XName,Object> MapValues (IDictionary<XName,Object> readWriteValues,IDictionary<XName,Object> writeOnlyValues)  
    ```  
  
     <span data-ttu-id="dcf34-121"><xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> Metoda oferuje funkcje, <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> nie, w tym, że umożliwia ona zależność od innej wartości, dostarczone przez inną uczestnika stanów trwałych, które nie zostały przetworzone przez <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> jeszcze.</span><span class="sxs-lookup"><span data-stu-id="dcf34-121">The <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> method provides functionality that <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> does not, in that it allows for a dependency on another value provided by another persistence participant that hasn’t been processed by <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> yet.</span></span>  
  
4.  <span data-ttu-id="dcf34-122">Implementowanie **PublishValues** metody.</span><span class="sxs-lookup"><span data-stu-id="dcf34-122">Implement the **PublishValues** method.</span></span> <span data-ttu-id="dcf34-123">**PublishValues** metoda otrzymuje słownik zawierający wszystkie wartości, które są ładowane w trwałości sklepie.</span><span class="sxs-lookup"><span data-stu-id="dcf34-123">The **PublishValues** method receives a dictionary containing all the values loaded from the persistence store.</span></span>  
  
    ```  
    protected virtual void PublishValues (IDictionary<XName,Object> readWriteValues)  
    ```  
  
5.  <span data-ttu-id="dcf34-124">Implementowanie **BeginOnSave** metodę, jeśli uczestnik jest uczestnika stanów trwałych operacji We/Wy.</span><span class="sxs-lookup"><span data-stu-id="dcf34-124">Implement the **BeginOnSave** method if the participant is a persistence I/O participant.</span></span> <span data-ttu-id="dcf34-125">Ta metoda jest wywoływana podczas operacji zapisu.</span><span class="sxs-lookup"><span data-stu-id="dcf34-125">This method is called during a Save operation.</span></span> <span data-ttu-id="dcf34-126">W przypadku tej metody należy wykonać adjunct operacji We/Wy do utrwalania (wystąpienia przepływu pracy zapisywanie).</span><span class="sxs-lookup"><span data-stu-id="dcf34-126">In this method, you should perform I/O adjunct to the persisting (saving) workflow instances.</span></span>  <span data-ttu-id="dcf34-127">Jeśli host używa transakcji do odpowiedniego polecenia trwałości, Transaction.Current jest podawany tej samej transakcji.</span><span class="sxs-lookup"><span data-stu-id="dcf34-127">If the host is using a transaction for the corresponding persistence command, the same transaction is provided in Transaction.Current.</span></span>  <span data-ttu-id="dcf34-128">Ponadto PersistenceIOParticipants może anonsować wymagane spójności transakcyjnej, w których przypadku hosta tworzy transakcji dla odcinek trwałości, jeśli jeden będzie nie używane.</span><span class="sxs-lookup"><span data-stu-id="dcf34-128">Additionally, PersistenceIOParticipants may advertise a transactional consistency requirement, in which case the host creates a transaction for the persistence episode if one would not otherwise be used.</span></span>  
  
    ```  
    protected virtual IAsyncResult BeginOnSave (IDictionary<XName,Object> readWriteValues, IDictionary<XName,Object> writeOnlyValues, TimeSpan timeout, AsyncCallback callback, Object state)  
    ```  
  
6.  <span data-ttu-id="dcf34-129">Implementowanie **BeginOnLoad** metodę, jeśli uczestnik jest uczestnika stanów trwałych operacji We/Wy.</span><span class="sxs-lookup"><span data-stu-id="dcf34-129">Implement the **BeginOnLoad** method if the participant is a persistence I/O participant.</span></span> <span data-ttu-id="dcf34-130">Ta metoda jest wywoływana podczas operacji ładowania.</span><span class="sxs-lookup"><span data-stu-id="dcf34-130">This method is called during a Load operation.</span></span> <span data-ttu-id="dcf34-131">W przypadku tej metody należy wykonywać operacji We/Wy adjunct ładowanie wystąpienia przepływu pracy.</span><span class="sxs-lookup"><span data-stu-id="dcf34-131">In this method, you should perform I/O adjunct to the loading of workflow instances.</span></span> <span data-ttu-id="dcf34-132">Jeśli host używa transakcji do odpowiedniego polecenia trwałości, Transaction.Current jest podawany tej samej transakcji.</span><span class="sxs-lookup"><span data-stu-id="dcf34-132">If the host is using a transaction for the corresponding persistence command, the same transaction is provided in Transaction.Current.</span></span> <span data-ttu-id="dcf34-133">Ponadto uczestnicy stanów trwałych operacji We/Wy może anonsować wymagane spójności transakcyjnej, w których przypadku hosta tworzy transakcji dla odcinek trwałości, jeśli jeden będzie nie używane.</span><span class="sxs-lookup"><span data-stu-id="dcf34-133">Additionally, Persistence I/O participants may advertise a transactional consistency requirement, in which case the host creates a transaction for the persistence episode if one would not otherwise be used.</span></span>  
  
    ```  
    protected virtual IAsyncResult BeginOnLoad (IDictionary<XName,Object> readWriteValues, TimeSpan timeout, AsyncCallback callback, Object state)  
    ```
