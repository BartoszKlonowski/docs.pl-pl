# <a name="windows-workflow-wf-samples"></a>Przykłady Windows Workflow

Możesz [pobrać przykłady przepływu pracy systemu Windows](https://www.microsoft.com/download/details.aspx?id=21459) , które zawierają instrukcje dotyczące różnych aspektów Windows Workflow Foundation (WF).

Artykuły w tej sekcji zawierają opis niektórych przykładów pakietu do pobrania. Aby zapoznać się z kompletnym zestawem dokumentacji, który obejmuje wszystkie przykłady, należy zapoznać się z sekcją [.NET Framework 4 WF Samples](/previous-versions/dotnet/netframework-4.0/dd483375(v=vs.100)) .

> [!NOTE]
> Próbki do pobrania zostały utworzone za pomocą programu Visual Studio 2010 i .NET Framework 4, ale są zgodne z nowszymi wersjami programu Visual Studio i .NET Framework. Dodatkowe przykłady dla Windows Workflow Foundation w programie [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] można znaleźć w [przykładach kodu w witrynie MSDN](/samples/browse/?redirectedfrom=MSDN-samples).
