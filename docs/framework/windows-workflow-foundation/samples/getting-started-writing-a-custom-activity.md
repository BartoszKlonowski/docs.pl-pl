# <a name="getting-started-writing-a-custom-activity"></a>Wprowadzenie do zapisu działania niestandardowego
Niniejszy przykład pokazuje jak zdefiniować proste działanie niestandardowe w XAML. Działanie jest nadawana nazwa `Rhyme`, i swojej logiki jest sekwencją trzy <xref:System.Activities.Statements.WriteLine> działań.  
  
#### <a name="to-set-up-build-and-run-the-sample"></a>Aby skonfigurować, tworzenie i uruchamianie aplikacji przykładowej  
  
1.  Otwórz **Simple.sln** przykładowe rozwiązanie w [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)].  
  
2.  Skompiluj i uruchom rozwiązanie.
