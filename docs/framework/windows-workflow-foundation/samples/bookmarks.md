# <a name="bookmarks"></a>Zakładki
Niniejszy przykład pokazuje, jak pisać niestandardowe działanie, które tworzy zakładki dla danych wejściowych zewnętrznych. Przykład obejmuje również podstawowa Aplikacja konsoli używa niestandardowego działania w przepływie pracy, który pokazuje, jak odnaleźć i wznowienie zakładki skojarzone z uruchomionego wystąpienia przepływu pracy. Aby uzyskać więcej informacji na temat zakładek, zobacz [zakładki](../../../../docs/framework/windows-workflow-foundation/bookmarks.md).  
  
### <a name="to-set-up-build-and-run-the-sample"></a>Aby skonfigurować, tworzenie i uruchamianie aplikacji przykładowej  
  
1.  Otwórz rozwiązanie przykładowe Bookmarks.sln w [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)].  
  
2.  Aby skompilować rozwiązanie, naciśnij klawisze CTRL + SHIFT + B.  
  
3.  Aby uruchomić przykład, naciśnij klawisze CTRLl + F5.

>  Przykłady może już być zainstalowany na tym komputerze. Przed kontynuowaniem sprawdź, czy są dostępne dla następującego katalogu (ustawienie domyślne).
