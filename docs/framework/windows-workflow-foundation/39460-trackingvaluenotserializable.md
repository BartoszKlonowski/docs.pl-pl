# <a name="39460---trackingvaluenotserializable"></a>39460 - TrackingValueNotSerializable
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|39460|  
|Słowa kluczowe|WFTracking|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 Wskazuje, że wyodrębnionego argumentu/zmiennej w rekordzie śledzenia nie jest możliwy do serializacji.  
  
## <a name="message"></a>Komunikat  
 Wyodrębnionego argumentu/zmiennej "%1" nie jest możliwy do serializacji.

|Domeny aplikacji|xs:String|Długość ciągu zwróconego przez AppDomain.CurrentDomain.FriendlyName.|
