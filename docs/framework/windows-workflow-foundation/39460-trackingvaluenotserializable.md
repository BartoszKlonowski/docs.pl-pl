# <a name="39460---trackingvaluenotserializable"></a>39460 — TrackingValueNotSerializable

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|39460|  
|Słowa kluczowe|WFTracking|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 Wskazuje, że wyodrębniony argument/zmienna w rekordzie śledzenia nie jest możliwy do serializacji.  
  
## <a name="message"></a>Wiadomość  

 Wyodrębnionego argumentu/zmiennej "%1" nie można serializować.
