# <a name="39460---trackingvaluenotserializable"></a>39460 — TrackingValueNotSerializable
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|39460|  
|słowa kluczowe|WFTracking|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 Wskazuje, że wyodrębnione argumentu/zmiennej w rekord śledzenia nie jest możliwy do serializacji.  
  
## <a name="message"></a>Komunikat  
 Wyodrębnione argumentu/zmiennej "%1" nie jest możliwy do serializacji.
