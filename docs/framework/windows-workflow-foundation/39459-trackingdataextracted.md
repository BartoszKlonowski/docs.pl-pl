|||  
|-|-|  
|ID (Identyfikator)|39459|  
|Słowa kluczowe|WFRuntime|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis

 Dane śledzenia %1 wyodrębnione w działaniu %2.
