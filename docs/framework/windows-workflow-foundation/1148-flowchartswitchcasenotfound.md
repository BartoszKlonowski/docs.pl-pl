## <a name="description"></a>Opis  
 Wskazuje, że przypadek domyślny, w przełączniku schemat blokowy ani uwzględniania wielkości liter można odnaleźć. Schemat blokowy wykonywania zakończy się.  
  
## <a name="message"></a>Komunikat  
 Schemat blokowy "%1'/FlowSwitch — można znaleźć, działanie Case ani domyślna wielkość liter dopasowanie wynik wyrażenia. Schemat blokowy wykonywania zakończy się.
