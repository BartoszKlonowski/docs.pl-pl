# <a name="1148---flowchartswitchcasenotfound"></a>1148 — FlowchartSwitchCaseNotFound

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|1148|  
|Słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 Wskazuje, że nie można znaleźć pasującego przypadku ani domyślnego przypadku w przełączniku Flowchart. Wykonanie schematu blokowego zakończy się.  
  
## <a name="message"></a>Wiadomość  

 Flowchart "%1"/FlowSwitch — nie można znaleźć działania Case ani domyślnego przypadku pasującego do wyniku wyrażenia. Wykonanie schematu blokowego zakończy się.

|--------------------|--------------------|-----------------|  
|Schema|XS: ciąg|Nazwa wyświetlana schematu blokowego.|  
|Wywołując|XS: ciąg|Ciąg zwracany przez element AppDomain. CurrentDomain —. FriendlyName.|
