# <a name="net-framework-45-built-in-activity-library"></a>Biblioteka działań wbudowanych programu .NET framework 4.5
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]zawiera nową bibliotekę działania z rozszerzona funkcjonalność. Następujące typy działań:  
  
-   [Przepływ sterowania](../../../docs/framework/windows-workflow-foundation/control-flow-activities-in-wf.md)  
  
-   [Schemat blokowy](../../../docs/framework/windows-workflow-foundation/flowchart-activities-in-wf.md)  
  
-   [Automatu stanów](../../../docs/framework/windows-workflow-foundation/state-machine-activities-in-wf.md)  
  
-   [Działania dotyczące komunikatów](../../../docs/framework/wcf/feature-details/messaging-activities.md)  
  
-   [Środowisko uruchomieniowe](../../../docs/framework/windows-workflow-foundation/runtime-activities-in-wf.md)  
  
-   [Elementy podstawowe](../../../docs/framework/windows-workflow-foundation/primitives-activities-in-wf.md)  
  
-   [Transakcji](../../../docs/framework/windows-workflow-foundation/transaction-activities-in-wf.md)  
  
-   [Kolekcja](../../../docs/framework/windows-workflow-foundation/collection-activities-in-wf.md)  
  
-   [Obsługa błędów](../../../docs/framework/windows-workflow-foundation/error-handling-activities-in-wf.md)  
  
-   [Migracji](../../../docs/framework/windows-workflow-foundation/migration-activity-in-wf.md)  
  
 Działaniami w bibliotece działań wbudowanych mogą być składane w przepływach pracy lub używany w połączeniu z działań niestandardowych. Działania w bibliotece działań wbudowanych są zapieczętowane; nie mają być używane do tworzenia nowych funkcji poprzez dziedziczenie.
