# <a name="primitives-activities-in-wf"></a>Działania pierwotne w WF

[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] Program udostępnia kilka działań dostarczonych przez system, które zapewniają wygodny mechanizm wykonywania typowych zadań.  
  
|Działanie|Opis|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Assign>|Przypisuje wartość do zmiennej w bieżącym zakresie.|  
|<xref:System.Activities.Statements.Delay>|Umieszcza jedną ścieżkę wykonania w stanie bezczynności, co może umożliwić odciążenie przepływu pracy.|  
|<xref:System.Activities.Statements.InvokeDelegate>|Wykonuje delegata, który pochodzi od <xref:System.Activities.ActivityDelegate> i jest uwidoczniony jako właściwość.|  
|<xref:System.Activities.Statements.InvokeMethod>|Wykonuje publiczną metodę obiektu CLR.|  
|<xref:System.Activities.Statements.WriteLine>|Zapisuje określony ciąg do konsoli lub określonego <xref:System.IO.TextWriter> obiektu.|
