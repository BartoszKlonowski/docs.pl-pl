# <a name="primitives-activities-in-wf"></a>Elementy podstawowe działania w WF
[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] zawiera kilka działań dostarczane przez system, które zapewniają wygodny mechanizm do wykonywania typowych zadań.  
  
|Działanie|Opis|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Assign>|Przypisuje wartość do zmiennej w bieżącym zakresie.|  
|<xref:System.Activities.Statements.Delay>|Umieszcza jednej ścieżki wykonywania w stanie bezczynności, potencjalnie umożliwiając przepływ pracy, aby zostać zwolniony.|  
|<xref:System.Activities.Statements.InvokeDelegate>|Wykonuje delegata, która pochodzi od klasy <xref:System.Activities.ActivityDelegate> i jest udostępniany jako właściwość.|  
|<xref:System.Activities.Statements.InvokeMethod>|Wykonuje publiczną metodę obiektu CLR.|  
|<xref:System.Activities.Statements.WriteLine>|Zapisuje określony ciąg na konsoli lub określonym <xref:System.IO.TextWriter> obiektu.|
