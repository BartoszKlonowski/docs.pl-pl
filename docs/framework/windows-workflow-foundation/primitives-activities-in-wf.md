# <a name="primitives-activities-in-wf"></a>Elementy podstawowe działania w WF
[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)]udostępnia kilka dostarczane przez system działań, które zapewniają wygodną mechanizm wykonywania typowych zadań.  
  
|Działanie|Opis|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Assign>|Przypisuje wartość do zmiennej w bieżącym zakresie.|  
|<xref:System.Activities.Statements.Delay>|Umieszcza jedną ścieżkę wykonywania w stanie bezczynności, prawdopodobnie dzięki czemu można zwolnić przepływu pracy.|  
|<xref:System.Activities.Statements.InvokeDelegate>|Wykonuje delegata, która jest pochodną <xref:System.Activities.ActivityDelegate> i jest udostępniany jako właściwość.|  
|<xref:System.Activities.Statements.InvokeMethod>|Wykonuje publiczną metodę obiektu CLR.|  
|<xref:System.Activities.Statements.WriteLine>|Zapisuje określony ciąg w konsoli lub określonej <xref:System.IO.TextWriter> obiektu.|
