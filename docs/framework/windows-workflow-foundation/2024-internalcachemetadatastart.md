# <a name="2024---internalcachemetadatastart"></a>2024 - InternalCacheMetadataStart
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|2024|  
|Słowa kluczowe|WFRuntime|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 Wskazuje początek wykonywanie metody InternalCacheMetadata w działaniu.

 Rozpoczęto wykonywanie metody InternalCacheMetadata w działaniu '%1'.
