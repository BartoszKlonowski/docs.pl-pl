# <a name="connection-string-and-connection-string-name"></a>Parametry połączenia i nazwa parametrów połączenia
**Parametry połączenia** właściwość określa parametry połączenia, która Store wystąpienia przepływu pracy SQL powinna być używana do łączenia z bazą danych trwałości. Ten parametr jest opcjonalny parametr. **Nazwa parametrów połączenia** właściwość określa nazwę parametrów połączenia o nazwie, która Store wystąpienia przepływu pracy SQL powinna być używana do łączenia z bazą danych trwałości. Ten parametr jest opcjonalny parametr. Należy określić wartość dla właściwości Nazwa parametrów połączenia lub właściwość parametrów połączenia, jeśli nie chcesz, aby Store wystąpienia przepływu pracy SQL, aby użyć domyślnego o nazwie parametry połączenia **DefaultSqlWorkflowInstanceStoreConnectionString** .
