# <a name="connection-string-and-connection-string-name"></a>Ciąg połączenia i nazwę ciągu połączenia
**Parametry połączenia** właściwość określa parametry połączenia, które w magazynie wystąpień przepływu pracy SQL ma używać do łączenia z bazą danych trwałości. Ten parametr jest opcjonalny parametr. **Nazwa ciągu połączenia** właściwość określa nazwę parametrów połączenia o nazwie, która powinna być używana do łączenia z bazą danych trwałości w magazynie wystąpień przepływu pracy SQL. Ten parametr jest opcjonalny parametr. Jeśli nie ma w magazynie wystąpień przepływu pracy SQL używanie domyślnych o nazwie parametry połączenia należy określić wartość dla właściwości Nazwa ciągu połączenia lub właściwości parametrów połączenia **DefaultSqlWorkflowInstanceStoreConnectionString** .
