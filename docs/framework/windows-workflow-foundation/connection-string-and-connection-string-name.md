# <a name="connection-string-and-connection-string-name"></a>Parametry połączenia i nazwa parametrów połączenia

Właściwość **parametrów połączenia** określa parametry połączenia, które mają być używane przez magazyn wystąpień przepływu pracy SQL do łączenia się z nietrwałą bazą danych. Ten parametr jest opcjonalnym parametrem. Właściwość **nazwa parametrów połączenia** określa nazwę parametrów połączenia, które mają być używane przez magazyn wystąpienia przepływu pracy SQL do łączenia się z bazą danych trwałości. Ten parametr jest opcjonalnym parametrem. Należy określić wartość właściwości Nazwa parametru połączenia lub właściwość parametrów połączenia, jeśli nie chcesz, aby Magazyn wystąpień programu SQL Workflow używał domyślnie nazwanych parametrów połączenia **DefaultSqlWorkflowInstanceStoreConnectionString**.
