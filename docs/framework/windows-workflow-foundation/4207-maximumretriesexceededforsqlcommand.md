# <a name="4207---maximumretriesexceededforsqlcommand"></a>4207 — MaximumRetriesExceededForSqlCommand

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4207|  
|Słowa kluczowe|Przydział, WFInstanceStore|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 Wskazuje, że dostawca SQL ponowi próbę wykonania polecenia SQL po przeprowadzeniu maksymalnej liczby ponownych prób.  
  
## <a name="message"></a>Wiadomość  

 Ponowienie próby wykonania polecenia SQL jako maksymalnej liczby ponownych prób.
