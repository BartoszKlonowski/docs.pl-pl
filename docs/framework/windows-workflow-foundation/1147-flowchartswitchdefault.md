# <a name="1147---flowchartswitchdefault"></a>1147 — FlowchartSwitchDefault

|||  
|-|-|  
|ID (Identyfikator)|1147|  
|Słowa kluczowe|WFActivities|  
|Poziom|Informacje|

## <a name="description"></a>Opis  

 Wskazuje, że domyślny przypadek został wybrany w przełączniku schematu blokowego.

 Flowchart "%1"/FlowSwitch — wybrano domyślne działanie Case.
