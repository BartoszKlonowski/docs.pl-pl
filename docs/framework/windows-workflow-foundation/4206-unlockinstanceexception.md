|||  
|-|-|  
|ID|4206|  
|Słowa kluczowe|WFInstanceStore|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 Wskazuje, że napotkano wyjątek podczas próby odblokowania wystąpienia.
