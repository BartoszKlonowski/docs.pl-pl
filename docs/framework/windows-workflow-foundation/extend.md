# <a name="extending-windows-workflow-foundation"></a>Rozszerzanie programu Windows Workflow Foundation
W poniższej sekcji opisano sposób rozszerzający Windows Workflow Foundation (WF) z działaniami niestandardowymi i projektantami w środowiskach przemieszczonych poza programem Visual Studio 2010.

## <a name="in-this-section"></a>W tej sekcji
 [Dostosowywanie środowiska projektowania przepływu pracy](customizing-the-workflow-design-experience.md) Wskazuje, w jaki sposób scenariusze projektowania działań niestandardowych i hostowania [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] były znacznie uproszczone w .NET Framework 4. Opracowywanie i wdrażanie jest teraz łatwiejsze i bardziej elastyczne, ponieważ nowy model programowania projektanta działań jest oparty na Windows Presentation Foundation (WPF).
