# <a name="extending-windows-workflow-foundation"></a>Rozszerzanie programu Windows Workflow Foundation
W poniższej sekcji opisano, jak rozszerzyć Windows Workflow Foundation (WF) przy użyciu niestandardowych działań i projektantów w rehostowanym środowisk poza programem Visual Studio 2010.

## <a name="in-this-section"></a>W tej sekcji
 [Dostosowywanie środowiska projektowania przepływu pracy](../../../docs/framework/windows-workflow-foundation/customizing-the-workflow-design-experience.md) wskazuje sposób scenariusze dotyczące projektowania niestandardowych działań i rehosting [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] zostały znacznie uproszczone w [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)]. Programowania i wdrażania są teraz łatwiejsze i bardziej elastyczne, ponieważ nowy model programowania projektanta działań jest zbudowany na Windows Presentation Foundation (WPF).
