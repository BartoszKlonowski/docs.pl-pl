## <a name="examples"></a><span data-ttu-id="892fa-405">Przykłady</span><span class="sxs-lookup"><span data-stu-id="892fa-405">Examples</span></span>  
 <span data-ttu-id="892fa-406">Poniższy przykład powoduje otwarcie interfejsu użytkownika dla obraz (*MageUI.exe*).</span><span class="sxs-lookup"><span data-stu-id="892fa-406">The following example opens the user interface for Mage (*MageUI.exe*).</span></span>

```  
mage
```  

 <span data-ttu-id="892fa-407">W poniższym przykładzie są tworzone domyślne manifesty wdrożenia i aplikacji.</span><span class="sxs-lookup"><span data-stu-id="892fa-407">The following examples create a default deployment manifest and application manifest.</span></span> <span data-ttu-id="892fa-408">Wszystkie te pliki są tworzone w bieżącym katalogu roboczym i są nazwane odpowiednio deploy.application i application.exe.manifest.</span><span class="sxs-lookup"><span data-stu-id="892fa-408">These files are all created in the current working directory and are named deploy.application and application.exe.manifest, respectively.</span></span>

```  
mage -New Deployment  
mage -New Application  
```  
  
 <span data-ttu-id="892fa-409">Poniższy przykład tworzy wypełniane przy użyciu wszystkich zestawów i zasobów plików z katalogu thecurrent manifestu aplikacji.</span><span class="sxs-lookup"><span data-stu-id="892fa-409">The following example creates an application manifest populated with all of the assemblies and resource files from thecurrent directory.</span></span>  
  
```  
mage -New Application -FromDirectory . -Version 1.0.0.0  
```  
  
 <span data-ttu-id="892fa-410">W poniższym przykładzie jest kontynuowany poprzedni przykład i jest określana nazwa wdrożenia oraz docelowy mikroprocesor.</span><span class="sxs-lookup"><span data-stu-id="892fa-410">The following example continues the previous example by specifying the deployment name and target microprocessor.</span></span> <span data-ttu-id="892fa-411">Określany jest także adres URL, pod którym technologia ClickOnce będzie sprawdzać dostępność aktualizacji.</span><span class="sxs-lookup"><span data-stu-id="892fa-411">It also specifies a URL against which ClickOnce should check for updates.</span></span>  
  
```  
mage -New Application -FromDirectory . -Name "Hello, World! Application" -Version 1.0.0.0 -Processor "x86" -ProviderUrl http://internalserver/HelloWorld/  
```  
  
 <span data-ttu-id="892fa-412">W poniższym przykładzie pokazano sposób tworzenia pary manifestów dla wdrożenia aplikacji programu WPF, która będzie obsługiwana w programie Internet Explorer.</span><span class="sxs-lookup"><span data-stu-id="892fa-412">The following example demonstrates how to create a pair of manifests for deploying a WPF application that will be hosted in Internet Explorer.</span></span>  
  
```  
mage -New Application -FromDirectory . -Version 1.0.0.0 -WPFBrowserApp true  
mage -New Deployment -AppManifest 1.0.0.0\application.manifest -WPFBrowserApp true  
```  
  
 <span data-ttu-id="892fa-413">W poniższym przykładzie manifest wdrożenia jest aktualizowany informacjami z manifestu aplikacji, a także jest ustawiana podstawa kodu dla lokalizacji manifestu aplikacji.</span><span class="sxs-lookup"><span data-stu-id="892fa-413">The following example updates a deployment manifest with information from an application manifest, and sets the code base for the location of the application manifest.</span></span>  
  
```  
mage -Update HelloWorld.deploy -AppManifest 1.0.0.0\application.manifest -AppCodeBase http://internalserver/HelloWorld.deploy  
```  
  
 <span data-ttu-id="892fa-414">W poniższym przykładzie manifest wdrożenia jest edytowany w celu wymuszenia aktualizacji zainstalowanej wersji użytkownika.</span><span class="sxs-lookup"><span data-stu-id="892fa-414">The following example edits the deployment manifest to force an update of the user's installed version.</span></span>  
  
```  
mage -Update c:\HelloWorldDeployment\HelloWorld.deploy -MinVersion 1.1.0.0  
```  
  
 <span data-ttu-id="892fa-415">W poniższym przykładzie manifest wdrożenia otrzymuje instrukcję pobrania manifestu aplikacji z innego katalogu.</span><span class="sxs-lookup"><span data-stu-id="892fa-415">The following example tells the deployment manifest to retrieve the application manifest from another directory.</span></span>  
  
```  
mage -Update HelloWorld.deploy -AppCodeBase http://anotherserver/HelloWorld/1.1.0.0/  
```  
  
 <span data-ttu-id="892fa-416">W poniższym przykładzie istniejący manifest wdrożenia jest podpisywany przy użyciu certyfikatu cyfrowego w bieżącym katalogu roboczym.</span><span class="sxs-lookup"><span data-stu-id="892fa-416">The following example signs an existing deployment manifest using a digital certificate in the current working directory.</span></span>
