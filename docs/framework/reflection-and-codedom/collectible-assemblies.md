- <span data-ttu-id="6184a-121">Statyczne odwołanie do `T` przez inny typ dynamicznie definiowane `T1` , jest nadal dostępny przy wykonywaniu kodu.</span><span class="sxs-lookup"><span data-stu-id="6184a-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="6184a-122">Na przykład `T1` może pochodzić od `T`, lub `T` może być typem parametru w metodzie o `T1`.</span><span class="sxs-lookup"><span data-stu-id="6184a-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>

- <span data-ttu-id="6184a-150">**Ładowanie zestawu** </span><span class="sxs-lookup"><span data-stu-id="6184a-150">**Assembly loading** </span></span>
