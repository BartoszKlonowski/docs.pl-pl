<span data-ttu-id="7bd6f-111">Okres istnienia zestawu kolekcjonowanego jest kontrolowany przez istnienie odwołań do typów, które zawiera, oraz obiektów, które są tworzone na podstawie tych typów.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-111">The lifetime of a collectible assembly is controlled by the existence of references to the types it contains and the objects that are created from those types.</span></span> <span data-ttu-id="7bd6f-112">Środowisko uruchomieniowe języka wspólnego nie zwalnia zestawu, tak długo, jak istnieje co najmniej jeden z następujących elementów`T` (jest dowolnym typem zdefiniowanym w zestawie):</span><span class="sxs-lookup"><span data-stu-id="7bd6f-112">The common language runtime does not unload an assembly as long as one or more of the following exist (`T` is any type that is defined in the assembly):</span></span>

- <span data-ttu-id="7bd6f-121">Statyczne odwołanie do `T` przez inny dynamicznie zdefiniowany typ `T1` , który jest nadal osiągalny przez wykonanie kodu.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="7bd6f-122">Na przykład, `T1` może pochodzić od `T`, lub `T` być typem parametru w metodzie `T1`.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>

- <span data-ttu-id="7bd6f-128">Delegat do statycznej metody, która jest zdefiniowana w module zestawu.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-128">A delegate to a static method that is defined in a module of the assembly.</span></span>

<span data-ttu-id="7bd6f-129">Jeśli istnieje tylko jeden element z tej listy tylko dla jednego typu lub jednej metody w zestawie, środowisko uruchomieniowe nie może zwolnić zestawu.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-129">If only one item from this list exists for only one type or one method in the assembly, the runtime cannot unload the assembly.</span></span>

## <a name="restrictions-on-collectible-assemblies"></a><span data-ttu-id="7bd6f-133">Ograniczenia dotyczące zestawów kolekcjonowanych</span><span class="sxs-lookup"><span data-stu-id="7bd6f-133">Restrictions on collectible assemblies</span></span>

<span data-ttu-id="7bd6f-134">Do zestawów kolekcjonowanych dotyczą następujące ograniczenia:</span><span class="sxs-lookup"><span data-stu-id="7bd6f-134">The following restrictions apply to collectible assemblies:</span></span>

- <span data-ttu-id="7bd6f-135">**Odwołania statyczne** Typy w zwykłym zestawie dynamicznym nie mogą zawierać statycznych odwołań do typów, które są zdefiniowane w zestawie kolekcjonowania.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-135">**Static references** Types in an ordinary dynamic assembly cannot have static references to types that are defined in a collectible assembly.</span></span> <span data-ttu-id="7bd6f-136">Na przykład, jeśli zdefiniujesz zwykły typ, który dziedziczy z typu w zestawie kolekcjonowanym, zostanie zgłoszony <xref:System.NotSupportedException> wyjątek.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-136">For example, if you define an ordinary type that inherits from a type in a collectible assembly, a <xref:System.NotSupportedException> exception is thrown.</span></span> <span data-ttu-id="7bd6f-137">Typ w zestawie kolekcjonowania może mieć statyczne odwołania do typu w innym zestawie kolekcjonowanym, ale wydłuży okres istnienia przywoływanego zestawu do okresu istnienia zestawu, do którego się odwołuje.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-137">A type in a collectible assembly can have static references to a type in another collectible assembly, but this extends the lifetime of the referenced assembly to the lifetime of the referencing assembly.</span></span>

- <span data-ttu-id="7bd6f-138">**Międzyoperacyjność modelu COM** W obrębie zestawu kolekcjonowanego nie można definiować interfejsów COM, a żadne wystąpienia typów w ramach zestawu kolekcjonowanego nie mogą być konwertowane na obiekty COM.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-138">**COM interop** No COM interfaces can be defined within a collectible assembly, and no instances of types within a collectible assembly can be converted into COM objects.</span></span> <span data-ttu-id="7bd6f-139">Typ w zestawie kolekcjonowania nie może być obiektem, który jest wywoływany przez COM otokę (CCW) lub otokę wywoływaną przez środowisko uruchomieniowe (RCW).</span><span class="sxs-lookup"><span data-stu-id="7bd6f-139">A type in a collectible assembly cannot serve as a COM callable wrapper (CCW) or runtime callable wrapper (RCW).</span></span> <span data-ttu-id="7bd6f-140">Jednak typy w zestawach kolekcjonowanych mogą korzystać z obiektów, które implementują interfejsy COM.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-140">However, types in collectible assemblies can use objects that implement COM interfaces.</span></span>

- <span data-ttu-id="7bd6f-141">**Wywołanie platformy** Metody, które mają <xref:System.Runtime.InteropServices.DllImportAttribute> atrybut nie zostaną skompilowane, gdy są zadeklarowane w zestawie kolekcjonowanym.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-141">**Platform invoke** Methods that have the <xref:System.Runtime.InteropServices.DllImportAttribute> attribute will not compile when they are declared in a collectible assembly.</span></span> <span data-ttu-id="7bd6f-142"><xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> Instrukcji nie można użyć w implementacji typu w zestawie kolekcjonowanym, a takie typy nie mogą być organizowane w kodzie niezarządzanym.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-142">The <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> instruction cannot be used in the implementation of a type in a collectible assembly, and such types cannot be marshaled to unmanaged code.</span></span> <span data-ttu-id="7bd6f-143">Można jednak wywołać kod natywny przy użyciu punktu wejścia zadeklarowanego w niekolekcjonowanym zestawie.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-143">However, you can call into native code by using an entry point that is declared in a non-collectible assembly.</span></span>

- <span data-ttu-id="7bd6f-144">**Kierowanie** Obiektów (w szczególności delegatów), które są zdefiniowane w zestawach kolekcjonowanych nie można zorganizować.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-144">**Marshaling** Objects (in particular, delegates) that are defined in collectible assemblies cannot be marshaled.</span></span> <span data-ttu-id="7bd6f-145">Jest to ograniczenie dla wszystkich typów emisji przejściowych.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-145">This is a restriction on all transient emitted types.</span></span>

- <span data-ttu-id="7bd6f-146">**Ładowanie zestawu** Emisja odbicia jest jedynym mechanizmem, który jest obsługiwany w przypadku ładowania zestawów kolekcjonowanych.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-146">**Assembly loading** Reflection emit is the only mechanism that is supported for loading collectible assemblies.</span></span> <span data-ttu-id="7bd6f-147">Zestawy, które są ładowane przy użyciu jakiejkolwiek innej formy ładowania zestawu nie mogą zostać zwolnione.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-147">Assemblies that are loaded by using any other form of assembly loading cannot be unloaded.</span></span>

- <span data-ttu-id="7bd6f-148">**Obiekty powiązane z kontekstem** Zmienne kontekstowe nie są obsługiwane.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-148">**Context-bound objects** Context-static variables are not supported.</span></span> <span data-ttu-id="7bd6f-149">Nie można rozciągnąć <xref:System.ContextBoundObject>typów w zestawie kolekcjonowania.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-149">Types in a collectible assembly cannot extend <xref:System.ContextBoundObject>.</span></span> <span data-ttu-id="7bd6f-150">Jednak kod w zestawach kolekcjonowanych może korzystać z obiektów powiązanych z kontekstem, które są zdefiniowane w innym miejscu.</span><span class="sxs-lookup"><span data-stu-id="7bd6f-150">However, code in collectible assemblies can use context-bound objects that are defined elsewhere.</span></span>
