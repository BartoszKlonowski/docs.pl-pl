# <a name="ui-automation-specification-and-community-promise"></a>Specyfikacja automatyzacji interfejsu użytkownika i obietnica dla społeczności
Automatyzacja interfejsu użytkownika zapewnia dostęp programistyczny do większości elementów interfejsu użytkownika na komputerze stacjonarnym, włączenie produktów technologii pomocniczej, takich jak czytniki zawartości ekranu, aby podać informacje o Interfejsie użytkownika dla użytkowników końcowych i manipulować interfejsu użytkownika przy użyciu metod innych niż standardowe dane wejściowe .
