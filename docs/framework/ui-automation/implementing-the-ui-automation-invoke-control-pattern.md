## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="b473e-110">Wytyczne dotyczące implementacji i konwencje</span><span class="sxs-lookup"><span data-stu-id="b473e-110">Implementation Guidelines and Conventions</span></span>  
 <span data-ttu-id="b473e-111">Podczas implementowania Invoke — wzorzec kontrolki, należy zwrócić uwagę następujących wytycznych i konwencje:</span><span class="sxs-lookup"><span data-stu-id="b473e-111">When implementing the Invoke control pattern, note the following guidelines and conventions:</span></span>  
  
-   <span data-ttu-id="b473e-112">Implementowanie kontrolek <xref:System.Windows.Automation.Provider.IInvokeProvider> jeśli takie samo zachowanie nie jest dostępna za pośrednictwem innego dostawcy wzorzec kontroli.</span><span class="sxs-lookup"><span data-stu-id="b473e-112">Controls implement <xref:System.Windows.Automation.Provider.IInvokeProvider> if the same behavior is not exposed through another control pattern provider.</span></span> <span data-ttu-id="b473e-113">Na przykład jeśli <xref:System.Windows.Automation.InvokePattern.Invoke%2A> metoda na formant wykonuje ta sama akcja co <xref:System.Windows.Automation.ExpandCollapsePattern.Expand%2A> lub <xref:System.Windows.Automation.ExpandCollapsePattern.Collapse%2A> metody, formant nie należy implementować <xref:System.Windows.Automation.Provider.IInvokeProvider>.</span><span class="sxs-lookup"><span data-stu-id="b473e-113">For example, if the <xref:System.Windows.Automation.InvokePattern.Invoke%2A> method on a control performs the same action as the <xref:System.Windows.Automation.ExpandCollapsePattern.Expand%2A> or <xref:System.Windows.Automation.ExpandCollapsePattern.Collapse%2A> method, the control should not implement <xref:System.Windows.Automation.Provider.IInvokeProvider>.</span></span>  
  
-   <span data-ttu-id="b473e-114">Wywoływanie kontrolki zwykle odbywa się przez kliknięcie, dwukrotne kliknięcie lub naciśnięcie klawisza ENTER, skrót klawiaturowy wstępnie zdefiniowanych lub alternatywne kombinacja klawiszy.</span><span class="sxs-lookup"><span data-stu-id="b473e-114">Invoking a control is generally performed by clicking or double-clicking or pressing ENTER, a predefined keyboard shortcut, or some alternate combination of keystrokes.</span></span>  
  
-   <span data-ttu-id="b473e-115"><xref:System.Windows.Automation.InvokePatternIdentifiers.InvokedEvent> jest wywoływane na formant, który został aktywowany (jako odpowiedzi do formantu przeprowadzania jego skojarzone z akcją).</span><span class="sxs-lookup"><span data-stu-id="b473e-115"><xref:System.Windows.Automation.InvokePatternIdentifiers.InvokedEvent> is raised on a control that has been activated (as a response to a control carrying out its associated action).</span></span> <span data-ttu-id="b473e-116">Jeśli to możliwe powinno być generowane zdarzenie, po kontrolki ukończył akcję i zwrócony bez blokowania.</span><span class="sxs-lookup"><span data-stu-id="b473e-116">If possible, the event should be raised after the control has completed the action and returned without blocking.</span></span> <span data-ttu-id="b473e-117">Powinno być generowane zdarzenie wywoływany przed obsługi żądania Invoke w następujących scenariuszach:</span><span class="sxs-lookup"><span data-stu-id="b473e-117">The Invoked event should be raised before servicing the Invoke request in the following scenarios:</span></span>  
  
    -   <span data-ttu-id="b473e-118">Nie jest możliwe lub praktyczne czekać, aż do zakończenia akcji.</span><span class="sxs-lookup"><span data-stu-id="b473e-118">It is not possible or practical to wait until the action is complete.</span></span>  
  
    -   <span data-ttu-id="b473e-119">Akcja wymaga interakcji z użytkownikiem.</span><span class="sxs-lookup"><span data-stu-id="b473e-119">The action requires user interaction.</span></span>  
  
    -   <span data-ttu-id="b473e-120">Akcja jest czasochłonne i spowoduje, że klienta wywołującego zablokować na znaczną ilość czasu.</span><span class="sxs-lookup"><span data-stu-id="b473e-120">The action is time-consuming and will cause the calling client to block for a significant amount of time.</span></span>  
  
-   <span data-ttu-id="b473e-121">Jeśli wywołanie formantu znaczący efekty uboczne, te efekty uboczne powinny zostać ujawnione przez <xref:System.Windows.Automation.AutomationElement.AutomationElementInformation.HelpText%2A> właściwości.</span><span class="sxs-lookup"><span data-stu-id="b473e-121">If invoking the control has significant side-effects, those side-effects should be exposed through the <xref:System.Windows.Automation.AutomationElement.AutomationElementInformation.HelpText%2A> property.</span></span> <span data-ttu-id="b473e-122">Na przykład nawet jeśli <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> nie jest skojarzony z zaznaczoną opcję <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> może spowodować, że inna kontrolka stają się wybrać.</span><span class="sxs-lookup"><span data-stu-id="b473e-122">For example, even though <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> is not associated with selection, <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> may cause another control to become selected.</span></span>  
  
-   <span data-ttu-id="b473e-123">Zatrzymaj wskaźnik myszy (lub myszą) efekty ogólnie nie jest uznawany za wywołany.</span><span class="sxs-lookup"><span data-stu-id="b473e-123">Hover (or mouse-over) effects generally do not constitute an Invoked event.</span></span> <span data-ttu-id="b473e-124">Jednak powinna obsługiwać formantów, które wykonują akcję (w przeciwieństwie do Przyczyna efekt wizualny) na podstawie stanu po wskazaniu wskaźnikiem <xref:System.Windows.Automation.InvokePattern> — wzorzec kontrolki.</span><span class="sxs-lookup"><span data-stu-id="b473e-124">However, controls that perform an action (as opposed to cause a visual effect) based on the hover state should support the <xref:System.Windows.Automation.InvokePattern> control pattern.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="b473e-125">Ta implementacja jest uznawana za zgłoszenie ułatwień dostępu, jeśli formant może być wywoływany tylko w wyniku związane z myszy efekt uboczny.</span><span class="sxs-lookup"><span data-stu-id="b473e-125">This implementation is considered an accessibility issue if the control can be invoked only as a result of a mouse-related side effect.</span></span>

-   <span data-ttu-id="b473e-129">Element może zniknąć z [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] drzewa natychmiast po wywoływana.</span><span class="sxs-lookup"><span data-stu-id="b473e-129">An element can disappear from the [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] tree immediately upon being invoked.</span></span> <span data-ttu-id="b473e-130">Trwa żądanie informacji z elementu dostarczone przez wywołanie zwrotne zdarzeń może zakończyć się niepowodzeniem w wyniku.</span><span class="sxs-lookup"><span data-stu-id="b473e-130">Requesting information from the element provided by the event callback may fail as a result.</span></span> <span data-ttu-id="b473e-131">Wstępnie pobieranie informacji o pamięci podręcznej jest zalecaną praktyką.</span><span class="sxs-lookup"><span data-stu-id="b473e-131">Pre-fetching cached information is the recommended workaround.</span></span>  
  
-   <span data-ttu-id="b473e-132">Kontrolki można zaimplementować wiele wzorców kontrolek.</span><span class="sxs-lookup"><span data-stu-id="b473e-132">Controls can implement multiple control patterns.</span></span> <span data-ttu-id="b473e-133">Na przykład kontrolka koloru wypełnienia w [!INCLUDE[TLA#tla_xl](../../../includes/tlasharptla-xl-md.md)] narzędzi implementuje interfejsy <xref:System.Windows.Automation.InvokePattern> i <xref:System.Windows.Automation.ExpandCollapsePattern> kontrolować wzorców.</span><span class="sxs-lookup"><span data-stu-id="b473e-133">For example, the Fill Color control on the [!INCLUDE[TLA#tla_xl](../../../includes/tlasharptla-xl-md.md)] toolbar implements both the <xref:System.Windows.Automation.InvokePattern> and the <xref:System.Windows.Automation.ExpandCollapsePattern> control patterns.</span></span> <span data-ttu-id="b473e-134"><xref:System.Windows.Automation.ExpandCollapsePattern> Udostępnia menu i <xref:System.Windows.Automation.InvokePattern> wypełnia aktywnego zaznaczenia z wybranym kolorze.</span><span class="sxs-lookup"><span data-stu-id="b473e-134"><xref:System.Windows.Automation.ExpandCollapsePattern> exposes the menu and the <xref:System.Windows.Automation.InvokePattern> fills the active selection with the chosen color.</span></span>
