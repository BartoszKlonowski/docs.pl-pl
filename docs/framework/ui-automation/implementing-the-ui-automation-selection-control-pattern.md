 <span data-ttu-id="8bffc-111">Jeśli implementacja wzorca kontrolki wyboru, należy zwrócić uwagę następujących wytycznych i konwencje:</span><span class="sxs-lookup"><span data-stu-id="8bffc-111">When implementing the Selection control pattern, note the following guidelines and conventions:</span></span>  
  
- <span data-ttu-id="8bffc-112">Określa, które implementują <xref:System.Windows.Automation.Provider.ISelectionProvider> zezwalają na jedną lub wiele podrzędnych elementów do wybrania.</span><span class="sxs-lookup"><span data-stu-id="8bffc-112">Controls that implement <xref:System.Windows.Automation.Provider.ISelectionProvider> allow either single or multiple child items to be selected.</span></span> <span data-ttu-id="8bffc-113">Na przykład pola listy, widokiem listy a widokiem drzewa obsługuje wiele zaznaczeń pola kombi, suwaka i Grupa przycisków radiowych wsparcie pojedynczego wyboru.</span><span class="sxs-lookup"><span data-stu-id="8bffc-113">For example, list box, list view, and tree view support multiple selections whereas combo box, slider, and radio button group support single selection.</span></span>
