 <span data-ttu-id="6d1bb-110">Jeśli implementacja wzorca kontrolki przekształcania, należy zwrócić uwagę następujących wytycznych i konwencje:</span><span class="sxs-lookup"><span data-stu-id="6d1bb-110">When implementing the Transform control pattern, note the following guidelines and conventions:</span></span>  
  
-   <span data-ttu-id="6d1bb-111">Obsługa tego wzorca kontrolki nie jest ograniczona do obiektów na pulpicie.</span><span class="sxs-lookup"><span data-stu-id="6d1bb-111">Support for this control pattern is not limited to objects on the desktop.</span></span> <span data-ttu-id="6d1bb-112">Ten wzorzec kontroli muszą być obsługiwane przez element podrzędny obiektu kontenera, jeśli elementy podrzędne mogą być przeniesione, rozmiaru lub obracać za darmo w granicach kontenera.</span><span class="sxs-lookup"><span data-stu-id="6d1bb-112">This control pattern must also be supported by the children of a container object if the children can be moved, resized, or rotated freely within the boundaries of the container.</span></span>

-   <span data-ttu-id="6d1bb-116">Wszystkie parametry i wartości właściwości są bezwzględne i niezależne od ustawień regionalnych.</span><span class="sxs-lookup"><span data-stu-id="6d1bb-116">All parameters and property values are absolute and independent of locale.</span></span>
