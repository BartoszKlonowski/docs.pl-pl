- <span data-ttu-id="043c3-113">Jeśli komórka jest pusta, element automatyzacji interfejsu użytkownika musi nadal być zwracany w celu obsługi właściwości <xref:System.Windows.Automation.Provider.IGridItemProvider.ContainingGrid%2A> dla tej komórki.</span><span class="sxs-lookup"><span data-stu-id="043c3-113">If a cell is empty, a UI Automation element must still be returned in order to support the <xref:System.Windows.Automation.Provider.IGridItemProvider.ContainingGrid%2A> property for that cell.</span></span> <span data-ttu-id="043c3-114">Jest to możliwe, gdy układ elementów podrzędnych w siatce jest podobny do niewyrównanej tablicy (Zobacz przykład poniżej).</span><span class="sxs-lookup"><span data-stu-id="043c3-114">This is possible when the layout of child elements in the grid is similar to a ragged array (see example below).</span></span>

<span data-ttu-id="043c3-116">Przykład kontrolki siatki z pustymi współrzędnymi</span><span class="sxs-lookup"><span data-stu-id="043c3-116">Example of a Grid Control with Empty Coordinates</span></span>  
  
- <span data-ttu-id="043c3-117">Siatka z pojedynczym elementem jest nadal wymagana do zaimplementowania <xref:System.Windows.Automation.Provider.IGridProvider>, jeśli jest logicznie traktowana jako siatka.</span><span class="sxs-lookup"><span data-stu-id="043c3-117">A grid with a single item is still required to implement <xref:System.Windows.Automation.Provider.IGridProvider> if it is logically considered to be a grid.</span></span> <span data-ttu-id="043c3-118">Liczba elementów podrzędnych w siatce nie jest istotna.</span><span class="sxs-lookup"><span data-stu-id="043c3-118">The number of child items in the grid is immaterial.</span></span>  
  
- <span data-ttu-id="043c3-119">Ukryte wiersze i kolumny, w zależności od implementacji dostawcy, mogą zostać załadowane w drzewie [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] i dlatego zostaną odzwierciedlone we właściwościach <xref:System.Windows.Automation.GridPattern.GridPatternInformation.RowCount%2A> i <xref:System.Windows.Automation.GridPattern.GridPatternInformation.ColumnCount%2A>.</span><span class="sxs-lookup"><span data-stu-id="043c3-119">Hidden rows and columns, depending on the provider implementation, may be loaded in the [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] tree and therefore will be reflected in the <xref:System.Windows.Automation.GridPattern.GridPatternInformation.RowCount%2A> and <xref:System.Windows.Automation.GridPattern.GridPatternInformation.ColumnCount%2A> properties.</span></span> <span data-ttu-id="043c3-120">Jeśli ukryte wiersze i kolumny nie zostały jeszcze załadowane, nie powinny być zliczane.</span><span class="sxs-lookup"><span data-stu-id="043c3-120">If the hidden rows and columns have not yet been loaded, they should not be counted.</span></span>  
  
- <span data-ttu-id="043c3-121"><xref:System.Windows.Automation.Provider.IGridProvider> nie umożliwia aktywnego manipulowania siatką; Aby włączyć tę funkcję, należy wdrożyć <xref:System.Windows.Automation.Provider.ITransformProvider>.</span><span class="sxs-lookup"><span data-stu-id="043c3-121"><xref:System.Windows.Automation.Provider.IGridProvider> does not enable active manipulation of a grid; <xref:System.Windows.Automation.Provider.ITransformProvider> must be implemented to enable this functionality.</span></span>  
  
- <span data-ttu-id="043c3-122">Użyj <xref:System.Windows.Automation.StructureChangedEventHandler>, aby nasłuchiwać zmian strukturalnych lub układu w siatce, takich jak komórki, które zostały dodane, usunięte lub scalone.</span><span class="sxs-lookup"><span data-stu-id="043c3-122">Use a <xref:System.Windows.Automation.StructureChangedEventHandler> to listen for structural or layout changes to the grid such as cells that have been added, removed, or merged.</span></span>  
  
- <span data-ttu-id="043c3-123">Użyj <xref:System.Windows.Automation.AutomationFocusChangedEventHandler>, aby śledzić przechodzenie przez elementy lub komórki siatki.</span><span class="sxs-lookup"><span data-stu-id="043c3-123">Use a <xref:System.Windows.Automation.AutomationFocusChangedEventHandler> to track traversal through the items or cells of a grid.</span></span>
