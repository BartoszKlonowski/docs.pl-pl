# <a name="exceptionhelper-class"></a>ExceptionHelper, klasa

Zapewnia wyjątki ze znormalizowanymi komunikatami o błędach. Klasa ta nie może być dziedziczona.

```csharp
internal static class ExceptionHelper
```

> [!WARNING]
> Ta klasa jest wewnętrzna i nie jest przeznaczona do użycia bezpośrednio w kodzie.
>
> Firma Microsoft nie obsługuje korzystania z tej klasy w aplikacji produkcyjnej w żadnej sytuacji.

## <a name="webpermissionunrestricted-field"></a>Pole WebPermissionUnrestricted

Określa, że aplikacja może łączyć się z zasobami internetowymi.

```csharp
internal static readonly WebPermission WebPermissionUnrestricted
```
