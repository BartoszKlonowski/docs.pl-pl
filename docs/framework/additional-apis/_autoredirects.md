# <a name="httpwebrequestautoredirects-field"></a>HttpWebRequest. \_AutoRedirects pola

`HttpWebRequest._AutoRedirects` jest <xref:System.Int32> , które odzwierciedla liczbę przekierowań wprowadzone dla tego <xref:System.Net.HttpWebRequest>.

## <a name="syntax"></a>Składnia  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> `HttpWebRequest._AutoRedirects` Pole jest wewnętrzna i nie są one przeznaczone do użycia bezpośrednio w kodzie.
> 
> Firma Microsoft obsługuje Użyj tego pola w aplikacji produkcyjnej, w żadnym przypadku.

## <a name="requirements"></a>Wymagania

**Namespace:** <xref:System.Net>

**Zestaw:** System (w System.dll)

**Wersje programu .NET framework:** dostępne od wersji 2.0.
