# <a name="datamemberlisteditor-class"></a>Klasa DataMemberListEditor

Udostępnia interfejs użytkownika z listy rozwijanej do edytowania właściwości obiektów powiązanych z danymi (obiektów, które mają inne niż null `DataSource` właściwości) poprzez wyszczególnienie wszystkich właściwości `DataSource` obiektu, aby wybrać źródło wartości.  
  
## <a name="syntax"></a>Składnia
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor` Klasa jest wewnętrzna i nie są one przeznaczone do użycia bezpośrednio w kodzie.
> 
> Microsoft w aplikacji produkcyjnej, w żadnym przypadku nie obsługuje korzystanie z tej klasy.
