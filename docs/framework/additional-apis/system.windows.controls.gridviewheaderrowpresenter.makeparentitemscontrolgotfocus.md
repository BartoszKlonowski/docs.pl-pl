# <a name="makeparentitemscontrolgotfocus-method"></a>MakeParentItemsControlGotFocus, Metoda

Umożliwia skoncentrowanie się na formancie nadrzędnym elementu. Jeśli formant nadrzędny jest <xref:System.Windows.Controls.ListBox> , nadaje fokus do ostatnio używanego elementu w <xref:System.Windows.Controls.ListBox> .

```csharp
internal void MakeParentItemsControlGotFocus()
```

> [!WARNING]
> Ta metoda jest wewnętrzna i nie jest przeznaczona do użycia bezpośrednio w kodzie.
>
> Firma Microsoft nie obsługuje korzystania z tej metody w aplikacji produkcyjnej w żadnej sytuacji.
