# <a name="rtcstate-class"></a>Klasa RtcState

Klasa `RtcState` reprezentuje dane o stanie dla żądania komunikacji w czasie rzeczywistym (RTC).

## <a name="syntax"></a>Składnia
  
```csharp  
internal class RtcState
```

> [!WARNING]
> Klasa `RtcState` jest wewnętrzna i nie jest przeznaczona do użycia bezpośrednio w kodzie.
>
> Firma Microsoft nie obsługuje użycia tej klasy w aplikacji produkcyjnej w żadnych okolicznościach.

## <a name="requirements"></a>Wymagania

**Obszar nazw:**<xref:System.Net>

**Montaż:** System (w pliku System.dll)

**Wersje programu .NET Framework:** Dostępne od 4.5.
