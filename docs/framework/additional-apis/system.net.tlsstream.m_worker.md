# <a name="tlsstreamm_worker-field"></a>TlsStream. m_Worker, pole

Reprezentuje stan strumienia SSL.

## <a name="syntax"></a>Składnia

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Wartość pola

`System.Net.Security.SslState`  
Stan strumienia SSL.

## <a name="remarks"></a>Uwagi

> [!WARNING]
> Pole `TlsStream.m_Worker` jest prywatne i nie jest przeznaczone do użycia bezpośrednio w kodzie.
>
> Firma Microsoft nie obsługuje korzystania z tego pola w aplikacji produkcyjnej w żadnej sytuacji.
