## <a name="syntax"></a>Składnia
  
```csharp  
private Hashtable m_ConnectionGroupList
```

> Pole `ServicePoint.m_ConnectionGroupList` jest prywatne i nie jest przeznaczone do użycia bezpośrednio w kodzie.
