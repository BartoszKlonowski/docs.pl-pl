# <a name="mef-for-net-for-windows-store-apps"></a>MEF dla platformy .NET dla aplikacji do Sklepu Windows
<xref:System.Composition?displayProperty=nameWithType>i jej podrzędne przestrzenie nazw zawierają typy służące do opracowywania rozszerzalnych [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] aplikacji z Managed Extensibility Framework (MEF). Te przestrzenie nazw są częścią [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] podzbioru [!INCLUDE[win8](../../../includes/win8-md.md)] dla systemu operacyjnego.  
  
 Te przestrzenie nazw nie są częścią podstawowej biblioteki klas dystrybuowanej z .NET Framework. Aby zainstalować te przestrzenie nazw, Otwórz projekt w programie Visual Studio, wybierz polecenie **Zarządzaj pakietami NuGet** z menu **projekt** i Wyszukaj w trybie online pakiet Microsoft. kompozycji.  
  
- <xref:System.Composition?displayProperty=nameWithType>zawiera klasy, które stanowią podstawowe MEF for [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] Apps.  
  
- <xref:System.Composition.Convention?displayProperty=nameWithType>zapewnia typy, które obsługują używanie MEF z modelem konfiguracji opartym na Konwencji.  
  
- <xref:System.Composition.Hosting?displayProperty=nameWithType>zapewnia typy MEF, które są przydatne dla deweloperów aplikacji hosta.  
  
- <xref:System.Composition.Hosting.Core?displayProperty=nameWithType>zapewnia typy MEF używane wewnętrznie przez aparat kompozycji.  
  
 Aby uzyskać więcej informacji [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] na temat i listę przestrzeni nazw i typów, które zawiera, zobacz [Omówienie programu .NET dla aplikacji do sklepu Windows](https://go.microsoft.com/fwlink/p/?LinkID=238312) w centrum deweloperów systemu Windows.
