 <span data-ttu-id="1328b-110">Ponieważ `x:TypeArguments` ciąg znaków, a ma zapasowy konwertera typów, zazwyczaj jest zadeklarowany w XAML użycia jako atrybut.</span><span class="sxs-lookup"><span data-stu-id="1328b-110">Because `x:TypeArguments` takes a string, and has type converter backing, it is typically declared in XAML usage as an attribute.</span></span>
