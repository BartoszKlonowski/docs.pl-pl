 <span data-ttu-id="3273d-111">`x:Uid` jest dyskretny od `x:Name` obu z powodu podanego scenariusza lokalizacji XAML i dlatego, że identyfikatory, które są używane dla lokalizacji, nie mają żadnych zależności od wpływu na model programowania `x:Name`.</span><span class="sxs-lookup"><span data-stu-id="3273d-111">`x:Uid` is discrete from `x:Name` both because of the stated XAML localization scenario and so that identifiers that are used for localization have no dependencies on the programming model implications of `x:Name`.</span></span> <span data-ttu-id="3273d-112">Ponadto `x:Name` podlega namescope języka XAML;
