# <a name="windows-communication-foundation-bindings"></a>Wiązania WCF (Windows Communication Foundation)

Powiązania określają sposób, w jaki punkt końcowy usługi WCF (Windows Communication Foundation) komunikuje się z innymi punktami końcowymi. W największej podstawowej powiązaniu należy określić transport (na przykład HTTP lub TCP) do użycia. Możesz również ustawić inne właściwości, takie jak obsługa zabezpieczeń i transakcji, za pomocą powiązań.
