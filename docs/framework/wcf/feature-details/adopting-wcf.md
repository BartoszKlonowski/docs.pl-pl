<span data-ttu-id="2ec38-103">Można użyć Windows Communication Foundation (WCF) dla nowych aplikacji podczas kontynuowanie zachować istniejące aplikacje opracowane za pomocą programu ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-103">You can choose to use Windows Communication Foundation (WCF) for new development, while continuing to maintain existing applications developed using ASP.NET.</span></span> <span data-ttu-id="2ec38-104">Ponieważ WCF ma być najbardziej odpowiednim rozwiązaniem ułatwiające komunikacji dla aplikacji skompilowanej za pomocą programu .NET Framework w jakimkolwiek scenariuszu, może służyć jako standardowe narzędzia do rozwiązywania szerokiej gamy problemów łączności oprogramowania w taki sposób, że ASP.NET Nie można wykonać.</span><span class="sxs-lookup"><span data-stu-id="2ec38-104">Because WCF is intended to be the most suitable choice for facilitating communication with applications built with the .NET Framework in any scenario, it can serve as a standard tool for solving a wide variety of software communications problems in a way that ASP.NET cannot.</span></span>  
  
 <span data-ttu-id="2ec38-105">Nowe aplikacje WCF można wdrożyć na tym samym maszynach jako istniejących usług sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-105">New WCF applications can be deployed on the same machines as existing ASP.NET Web services.</span></span> <span data-ttu-id="2ec38-106">Istniejące usługi sieci Web ASP.NET korzystania z wersji programu .NET Framework, przed w wersji 2.0, można użyć narzędzie rejestracji programu ASP.NET usług IIS do selektywnie wdrożenia programu .NET Framework 2.0 do aplikacji usług IIS, w których są nowe aplikacje WCF, ma być obsługiwana.</span><span class="sxs-lookup"><span data-stu-id="2ec38-106">If the existing ASP.NET Web services use a version of the .NET Framework prior to version 2.0, then you can use the ASP.NET IIS Registration Tool to selectively deploy the .NET Framework 2.0 to IIS applications in which new WCF applications are to be hosted.</span></span> <span data-ttu-id="2ec38-107">Tego narzędzia jest udokumentowany na [narzędzie rejestracji programu ASP.NET usług IIS (Aspnet_regiis.exe)](http://go.microsoft.com/fwlink/?LinkId=94687), i ma wbudowany interfejsu użytkownika, w konsoli zarządzania usług IIS 6.0.</span><span class="sxs-lookup"><span data-stu-id="2ec38-107">That tool is documented at [ASP.NET IIS Registration Tool (Aspnet_regiis.exe)](http://go.microsoft.com/fwlink/?LinkId=94687), and has a user interface built into the IIS 6.0 management console.</span></span>  
  
 <span data-ttu-id="2ec38-108">Usługi WCF może służyć do dodawania nowych funkcji do istniejącej usługi sieci Web ASP.NET przez dodanie skonfigurowanego do uruchamiania w trybie zgodności ASP.NET do istniejącej aplikacji usługi sieci Web ASP.NET w usługach IIS usługi WCF.</span><span class="sxs-lookup"><span data-stu-id="2ec38-108">WCF can be used to add new features to existing ASP.NET Web services by adding WCF services configured to run in ASP.NET compatibility mode to existing ASP.NET Web service applications in IIS.</span></span> <span data-ttu-id="2ec38-109">Ze względu na tryb zgodności ASP.NET, kod dla nowych usług WCF można uzyskać dostęp i zaktualizować przy użyciu tego samego informacji o stanie aplikacji jako istniejącego kodu platformy ASP.NET, <xref:System.Web.HttpContext> klasy.</span><span class="sxs-lookup"><span data-stu-id="2ec38-109">Because of ASP.NET compatibility mode, the code for the new WCF services can access and update the same application state information as the pre-existing ASP.NET code, by using the <xref:System.Web.HttpContext> class.</span></span> <span data-ttu-id="2ec38-110">Aplikacje mogą również współużytkować tego samego biblioteki klas.</span><span class="sxs-lookup"><span data-stu-id="2ec38-110">The applications can also share the same class libraries.</span></span>  
  
 <span data-ttu-id="2ec38-111">Klienci WCF za pomocą usług sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-111">WCF clients can use ASP.NET Web services.</span></span> <span data-ttu-id="2ec38-112">Usługi WCF, które są skonfigurowane przy użyciu <xref:System.ServiceModel.BasicHttpBinding> mogą być używane przez klientów usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-112">WCF services that are configured with the <xref:System.ServiceModel.BasicHttpBinding> can be used by ASP.NET Web service clients.</span></span> <span data-ttu-id="2ec38-113">Usługi sieci Web ASP.NET mogą współistnieć z aplikacjami usługi WCF i WCF nawet służy do dodawania funkcji do istniejącej usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-113">ASP.NET Web services can co-exist with WCF applications, and WCF can even be used to add features to existing ASP.NET Web services.</span></span> <span data-ttu-id="2ec38-114">Możliwość skorzystania z tych sposobów, w których usługi WCF i platformy ASP.NET w sieci Web mogą być używane razem, można przeprowadzić migrację usług sieci Web ASP.NET do programu WCF tylko wtedy, gdy jest to wymagane funkcje, które są udostępniane przez usługi WCF i nie sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="2ec38-114">Given all of these ways in which WCF and ASP.NET Web services can be used together, you may want to migrate ASP.NET Web services to WCF only if you require features that are provided by WCF and not ASP.NET Web services.</span></span>

class="sxs-lookup"><span data-stu-id="2ec38-117">The new design benefits from your experience with the existing system and from wisdom gained since that system was designed.</span></span> <span  <span data-ttu-id="2ec38-120">Kilka przykładów w przypadku, gdy eksportowanie z usług sieci Web ASP.NET do programu WCF jest właściwym rozwiązaniem, poniższa sekcja zawiera niektóre wskazówki dotyczące postępowania.</span><span class="sxs-lookup"><span data-stu-id="2ec38-120">For the few cases where porting from ASP.NET Web services to WCF is the correct solution, the following section provides some guidance on how to proceed.</span></span> <span data-ttu-id="2ec38-121">Brak porady na temat migracji usług i sposobu migracji klientów.</span><span class="sxs-lookup"><span data-stu-id="2ec38-121">There is advice on how to migrate services, and how to migrate clients.</span></span>
