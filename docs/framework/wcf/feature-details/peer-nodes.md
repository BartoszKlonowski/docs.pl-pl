# <a name="peer-nodes"></a>Węzły równorzędne
Węzeł równorzędny jest pojedynczym punktem końcowym w sieci równorzędnej. Aplikacja może mieć wiele węzłów równorzędnych, które uczestniczą w różnych siatkach. Na przykład aplikacja do komunikacji równorzędnej między elementami równorzędnymi może być częścią dwóch oddzielnych siatek: jednej siatki na potrzeby prowadzenia rozmowy z dużymi przyjaciółmi; i inne dla czatów z współpracownikami służbowymi.
