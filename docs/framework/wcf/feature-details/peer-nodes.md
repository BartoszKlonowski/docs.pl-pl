# <a name="peer-nodes"></a>Węzły równorzędne
Węzeł równorzędny jest pojedyncze punkty końcowe w siatce kanał elementu równorzędnego. Aplikacja może mieć wiele węzłów równorzędnych, które uczestniczą w różnych siatki. Na przykład aplikację do obsługi rozmów peer-to-peer mogą należeć do dwóch oddzielnych oczek: jeden siatki na prowadzenie rozmów z przyjaciółmi w szkole; i inny wpis dla rozmowę z współpracownikom.
