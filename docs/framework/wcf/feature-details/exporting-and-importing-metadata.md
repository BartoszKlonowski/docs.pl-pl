### <a name="importing-policies"></a><span data-ttu-id="42f99-124">Importowanie zasad</span><span class="sxs-lookup"><span data-stu-id="42f99-124">Importing Policies</span></span>

 <span data-ttu-id="42f99-126">Logika importu zasad automatycznie obsługuje zasady odwołania do wyrażenia zasad, w tym samym dokumencie WSDL i o `wsu:Id` lub `xml:id` atrybutu.</span><span class="sxs-lookup"><span data-stu-id="42f99-126">The policy import logic automatically handles policy references to policy expressions in the same WSDL document and is identified with a `wsu:Id` or `xml:id` attribute.</span></span> <span data-ttu-id="42f99-127">Logika importu zasad chroni aplikacje przed odwołania cykliczne zasad przez ograniczenie rozmiaru wyrażenie zasad do 4096 węzłów, gdy węzeł ma jeden z następujących elementów: `wsp:Policy`, `wsp:All`, `wsp:ExactlyOne`, `wsp:policyReference`.</span><span class="sxs-lookup"><span data-stu-id="42f99-127">The policy import logic protects applications against circular policy references by limiting the size of a policy expression to 4096 nodes, where a node is a one of the following elements: `wsp:Policy`, `wsp:All`, `wsp:ExactlyOne`, `wsp:policyReference`.</span></span>  
  
 <span data-ttu-id="42f99-128">Logika importu zasad normalizuje automatycznie wyrażenie zasad.</span><span class="sxs-lookup"><span data-stu-id="42f99-128">The policy import logic also automatically normalizes policy expressions.</span></span> <span data-ttu-id="42f99-129">Zagnieżdżone wyrażenia zasad i `wsp:Optional` nie są znormalizować atrybutu.</span><span class="sxs-lookup"><span data-stu-id="42f99-129">Nested policy expressions and the `wsp:Optional` attribute are not normalized.</span></span> <span data-ttu-id="42f99-130">Ilość przetwarzanych danych normalizacji wykonywane jest ograniczona do 4096 kroków, w którym każdy krok daje potwierdzenia zasad lub elementu podrzędnego `wsp:ExactlyOne` elementu.</span><span class="sxs-lookup"><span data-stu-id="42f99-130">The amount of normalization processing done is limited to 4096 steps, where each step yields a policy assertion, or a child element of a `wsp:ExactlyOne` element.</span></span>  
  
 <span data-ttu-id="42f99-131"><xref:System.ServiceModel.Description.WsdlImporter> Typu próbuje maksymalnie 32 kombinacje zasad alternatywnych dołączone do różnych tematów zasad WSDL.</span><span class="sxs-lookup"><span data-stu-id="42f99-131">The <xref:System.ServiceModel.Description.WsdlImporter> type tries up to 32 combinations of policy alternatives attached to the different WSDL policy subjects.</span></span> <span data-ttu-id="42f99-132">Jeśli połączenie nie importuje prawidłowo, pierwsza kombinacja służy do utworzenia częściowego wiązania niestandardowego.</span><span class="sxs-lookup"><span data-stu-id="42f99-132">If no combination imports cleanly, the first combination is used to construct a partial custom binding.</span></span>
