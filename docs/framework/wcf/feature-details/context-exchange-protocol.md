 <span data-ttu-id="0e38f-119">Poniżej znajduje się lista nieodmian w tym trybie:</span><span class="sxs-lookup"><span data-stu-id="0e38f-119">The following is a list of invariants in this mode:</span></span>

 <span data-ttu-id="0e38f-130">Niezależnie od tego, który tryb jest wybierany przez fabrykę kanałów klienta obsługuje <xref:System.ServiceModel.Channels.IRequestChannel> , <xref:System.ServiceModel.Channels.IRequestSessionChannel> oraz <xref:System.ServiceModel.Channels.IDuplexSessionChannel> wzorce wymiany komunikatów.</span><span class="sxs-lookup"><span data-stu-id="0e38f-130">No matter which mode is chosen the client channel factory supports <xref:System.ServiceModel.Channels.IRequestChannel>, <xref:System.ServiceModel.Channels.IRequestSessionChannel>, and <xref:System.ServiceModel.Channels.IDuplexSessionChannel> message exchange patterns.</span></span>  
  
 <span data-ttu-id="0e38f-131">W usłudze wystąpienie kanału jest odpowiedzialne za konwertowanie kontekstu dostarczonego przez klienta na komunikaty przychodzące do  <span data-ttu-id="0e38f-136">Protokół wymiany kontekstu wprowadza nowy `wsc:Context` nagłówek protokołu SOAP do reprezentowania informacji kontekstowych, gdy pliki cookie protokołu HTTP nie są używane do propagowania kontekstu.</span><span class="sxs-lookup"><span data-stu-id="0e38f-136">The context exchange protocol introduces a new `wsc:Context` SOAP header to represent the context information when HTTP cookies are not used to propagate the context.</span></span> <span data-ttu-id="0e38f-137">Schemat nagłówka kontekstu umożliwia dowolną liczbę elementów podrzędnych, z których każdy ma klucz ciągu i zawartość ciągu.</span><span class="sxs-lookup"><span data-stu-id="0e38f-137">The context header schema allows for any number of child elements, each with a string key and string content.</span></span> <span data-ttu-id="0e38f-138">Poniżej znajduje się przykład nagłówka kontekstu.</span><span class="sxs-lookup"><span data-stu-id="0e38f-138">The following is an example of a context header.</span></span>

 `</Context>`
