# <a name="wcf-discovery"></a>Odnajdywanie w programie WCF
Windows Communication Foundation (WCF) zapewnia pomoc techniczną do włączenia usług jako wykrywalne w czasie wykonywania w sposób interoperacyjny przy użyciu protokołu WS Discovery. Usługi WCF można ogłosić ich dostępność sieci przy użyciu komunikatu multiemisji lub serwera proxy odnajdywania. Aplikacje klienckie umożliwia wyszukiwanie w sieci lub serwera proxy odnajdywania można znaleźć usług, które spełniają określone kryteria. Tematy w tej sekcji omówiono i model programowania dla tej funkcji, które szczegółowo opisano.
