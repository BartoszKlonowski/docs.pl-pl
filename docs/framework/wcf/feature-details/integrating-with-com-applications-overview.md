 <span data-ttu-id="b845d-112">Wspólne innych klienci WCF, korzystając z monikerem aplikacje Określ address, binding i contract do komunikowania się z usługą.</span><span class="sxs-lookup"><span data-stu-id="b845d-112">In common with other WCF clients, when using the moniker, applications specify the address, binding and contract to communicate with a service.</span></span> <span data-ttu-id="b845d-113">Kontrakt można określić w jednym z następujących sposobów:</span><span class="sxs-lookup"><span data-stu-id="b845d-113">The contract can be specified in one of the following ways:</span></span>
