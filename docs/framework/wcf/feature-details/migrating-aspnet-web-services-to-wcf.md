<span data-ttu-id="ecae5-103">Program ASP.NET udostępnia biblioteki klas .NET Framework i narzędzia do tworzenia usług sieci Web, a także urządzenia do hostingu usług w ramach usługi Internet Information Services (IIS).</span><span class="sxs-lookup"><span data-stu-id="ecae5-103">ASP.NET provides .NET Framework class libraries and tools for building Web services, as well as facilities for hosting services within Internet Information Services (IIS).</span></span> <span data-ttu-id="ecae5-104">Windows Communication Foundation (WCF) oferuje biblioteki klas .NET Framework, narzędzia i hostingu urządzenia do włączania jednostek oprogramowania do komunikowania się przy użyciu protokołów, w tym używanych przez usługi sieci Web.</span><span class="sxs-lookup"><span data-stu-id="ecae5-104">Windows Communication Foundation (WCF) provides .NET Framework class libraries, tools and hosting facilities for enabling software entities to communicate using any protocols, including those used by Web services.</span></span>  <span data-ttu-id="ecae5-105">Migrowanie usług sieci Web ASP.NET do programu WCF umożliwia aplikacjom korzystać z nowych funkcji i ulepszeń, które są unikatowe dla usługi WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-105">Migrating ASP.NET Web Services to WCF allows your applications to take advantage of new features and improvements that are unique to WCF.</span></span>  
  
 <span data-ttu-id="ecae5-106">Usługi WCF ma kilka zalet ważne względem usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-106">WCF has several important advantages relative to ASP.NET Web services.</span></span> <span data-ttu-id="ecae5-107">Narzędzia usług sieci Web ASP.NET są przeznaczone wyłącznie na potrzeby tworzenia usług sieci Web, usługi WCF udostępnia narzędzia, które mogą być używane, gdy oprogramowanie jednostek muszą być wprowadzane do komunikowania się ze sobą.</span><span class="sxs-lookup"><span data-stu-id="ecae5-107">While ASP.NET Web services tools are solely for building Web services, WCF provides tools that can be used when software entities must be made to communicate with one another.</span></span> <span data-ttu-id="ecae5-108">Zmniejszy to liczbę technologie wymagane do uwzględnienia scenariusze komunikacji innego oprogramowania, które z kolei zostanie obniżony koszt zasobów rozwoju oprogramowania, a także czas wymagany do ukończenia oprogramowania deweloperów rozwoju projektów.</span><span class="sxs-lookup"><span data-stu-id="ecae5-108">This will reduce the number of technologies that developers are required to know in order to accommodate different software communication scenarios, which in turn will reduce the cost of software development resources, as well as the time to complete software development projects.</span></span>  
  
 <span data-ttu-id="ecae5-109">Nawet w przypadku projektów rozwoju usług sieci Web usługi WCF obsługuje więcej protokoły usług sieci Web niż Obsługa usług sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-109">Even for Web service development projects, WCF supports more Web service protocols than ASP.NET Web services support.</span></span> <span data-ttu-id="ecae5-110">Te dodatkowe protokoły zapewniają bardziej zaawansowanych rozwiązań obejmujące, między inne rzeczy, niezawodne sesje i transakcji.</span><span class="sxs-lookup"><span data-stu-id="ecae5-110">These additional protocols provide for more sophisticated solutions involving, amongst other things, reliable sessions and transactions.</span></span>  
  
 <span data-ttu-id="ecae5-111">Usługi WCF obsługuje protokoły więcej do transportu wiadomości niż usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-111">WCF supports more protocols for transporting messages than ASP.NET Web services.</span></span> <span data-ttu-id="ecae5-112">Usługi sieci Web programu ASP.NET obsługują tylko wysyłanie wiadomości przy użyciu protokołu HTTP (Hypertext Transfer).</span><span class="sxs-lookup"><span data-stu-id="ecae5-112">ASP.NET Web services only support sending messages by using the Hypertext Transfer Protocol (HTTP).</span></span> <span data-ttu-id="ecae5-113">Usługi WCF obsługuje wysyłanie wiadomości przy użyciu protokołu HTTP, a także Transmission Control Protocol (TCP), nazwanych potoków i usługi kolejkowania wiadomości firmy Microsoft (MSMQ).</span><span class="sxs-lookup"><span data-stu-id="ecae5-113">WCF supports sending messages by using HTTP, as well as the Transmission Control Protocol (TCP), named pipes, and Microsoft Message Queuing (MSMQ).</span></span> <span data-ttu-id="ecae5-114">Większe znaczenie WCF może zostać rozszerzony do obsługi dodatkowych protokołów.</span><span class="sxs-lookup"><span data-stu-id="ecae5-114">More important, WCF can be extended to support additional transport protocols.</span></span> <span data-ttu-id="ecae5-115">W związku z tym oprogramowania utworzony przy użyciu programu WCF można dostosować do współdziała z szeroką gamę innego oprogramowania, zwiększając potencjalnych zwrot z inwestycji.</span><span class="sxs-lookup"><span data-stu-id="ecae5-115">Therefore, software developed using WCF can be adapted to work together with a wider variety of other software, thereby increasing the potential return on the investment.</span></span>  
  
 <span data-ttu-id="ecae5-116">WCF oferuje znacznie bardziej zaawansowane funkcje urządzeń do wdrażania i zarządzania aplikacjami niż usługi sieci Web programu ASP.NET zapewnia.</span><span class="sxs-lookup"><span data-stu-id="ecae5-116">WCF provides much richer facilities for deploying and managing applications than ASP.NET Web services provides.</span></span> <span data-ttu-id="ecae5-117">Oprócz system konfiguracji, który ma także ASP.NET, usługi WCF oferuje edytora konfiguracji, śledzenie działania od nadawcy do odbiorcy i z powrotem przy użyciu dowolnej liczby pośredników, przeglądarka śledzenia, rejestrowanie komunikatów, ogromną liczbę liczników wydajności i Obsługa Instrumentacji zarządzania Windows.</span><span class="sxs-lookup"><span data-stu-id="ecae5-117">In addition to a configuration system, which ASP.NET also has, WCF offers a configuration editor, activity tracing from senders to receivers and back through any number of intermediaries, a trace viewer, message logging, a vast number of performance counters, and support for Windows Management Instrumentation.</span></span>  
  
 <span data-ttu-id="ecae5-118">Podane tych potencjalnych korzyści WCF względem sieci Web ASP.NET usługi, jeśli są używane lub są biorąc pod uwagę przy użyciu usługi sieci Web ASP.NET, dostępnych jest kilka opcji:</span><span class="sxs-lookup"><span data-stu-id="ecae5-118">Given these potential benefits of WCF relative to ASP.NET Web services, if you are using, or are considering using ASP.NET Web services you have several options:</span></span>  
  
-   <span data-ttu-id="ecae5-119">Nadal używać usług sieci Web ASP.NET i przyznano korzyści proffered przez usługę WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-119">Continue to use ASP.NET Web services, and forego the benefits proffered by WCF.</span></span>  
  
-   <span data-ttu-id="ecae5-120">Zachowaj przy użyciu usług sieci Web ASP.NET w celu przyjęcia usługi WCF w czasie w przyszłości.</span><span class="sxs-lookup"><span data-stu-id="ecae5-120">Keep using ASP.NET Web services with the intention of adopting WCF at some time in the future.</span></span> <span data-ttu-id="ecae5-121">Tematy w tej sekcji opisano sposób zmaksymalizować perspektyw było go używać nowej aplikacji usługi sieci Web ASP.NET oraz przyszłych aplikacji WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-121">The topics in this section explain how to maximize the prospects for being able to use new ASP.NET Web service applications together with future WCF applications.</span></span> <span data-ttu-id="ecae5-122">Tematy w tej sekcji opisano również sposób tworzenia nowej sieci Web ASP.NET usług, aby ułatwić ich migracja do programu WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-122">The topics in this section also explain how to build new ASP.NET Web services so as to make it easier to migrate them to WCF.</span></span> <span data-ttu-id="ecae5-123">Jednak jeśli ważne jest zabezpieczenie usługi lub gwarancje niezawodności lub transakcji są wymagane, czy niestandardowych obiektów zarządzania musi być skonstruowany, a następnie jest lepszym rozwiązaniem do przyjęcia usługi WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-123">However, if securing the services is important, or reliability or transaction assurances are required, or if custom management facilities will have to be constructed, then it is a better option to adopt WCF.</span></span> <span data-ttu-id="ecae5-124">Usługi WCF jest przeznaczona dla dokładnie takich scenariuszy.</span><span class="sxs-lookup"><span data-stu-id="ecae5-124">WCF is designed for precisely such scenarios.</span></span>  
  
-   <span data-ttu-id="ecae5-125">Przyjmuje WCF dla nowych aplikacji, pozostawiając Obsługa istniejących aplikacji usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-125">Adopt WCF for new development, while continuing to maintain your existing ASP.NET Web service applications.</span></span> <span data-ttu-id="ecae5-126">Ten wybór jest bardzo prawdopodobne optymalny.</span><span class="sxs-lookup"><span data-stu-id="ecae5-126">This choice is very likely the optimal one.</span></span> <span data-ttu-id="ecae5-127">Zalety usługi WCF, go daje podczas Reserve Sparing Table koszt modyfikacji istniejących aplikacji z niego korzystać.</span><span class="sxs-lookup"><span data-stu-id="ecae5-127">It yields the benefits of WCF, while sparing the cost of modifying the existing applications to use it.</span></span> <span data-ttu-id="ecae5-128">W tym scenariuszu nowych aplikacji WCF mogą współistnieć z istniejących aplikacji ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-128">In this scenario, new WCF applications can co-exist with existing ASP.NET applications.</span></span> <span data-ttu-id="ecae5-129">Nowe aplikacje WCF będą mogli używać istniejących usług sieci Web ASP.NET i WCF może służyć do programu nowe funkcje operacyjne w aplikacji ASP.NET z tryb zgodności WCF ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-129">New WCF applications will be able to use existing ASP.NET Web services, and WCF can be used to program new operational capabilities into existing ASP.NET applications by virtue of WCF ASP.NET compatibility mode.</span></span>  
  
-   <span data-ttu-id="ecae5-130">Przyjmuje WCF i migracji istniejącej aplikacji usługi sieci Web ASP.NET do programu WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-130">Adopt WCF and migrate existing ASP.NET Web service applications to WCF.</span></span> <span data-ttu-id="ecae5-131">Możesz wybrać tę opcję w celu zwiększenia istniejących aplikacji z funkcji udostępnianych przez usługi WCF lub do odtworzenia funkcjonalności istniejących usług sieci Web ASP.NET w ramach nowych, bardziej zaawansowanych aplikacji WCF.</span><span class="sxs-lookup"><span data-stu-id="ecae5-131">You may choose this option to enhance the existing applications with features provided by WCF, or to reproduce the functionality of existing ASP.NET Web services within new, more powerful WCF applications.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="ecae5-132">Należy zwrócić uwagę, jeśli usługa WCF jest hostowana przez usługi IIS 5.x i platformy ASP.NET został odinstalowany.</span><span class="sxs-lookup"><span data-stu-id="ecae5-132">Care must be taken if a WCF service is hosted by IIS 5.x and ASP.NET is uninstalled.</span></span> <span data-ttu-id="ecae5-133">Gdy usługa WCF jest hostowana przez usługi IIS 5.x, kod usługi może wystąpić w przypadku odinstalowania programu ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="ecae5-133">When a WCF service is hosted by IIS 5.x, the code for the service can be requested if ASP.NET is uninstalled.</span></span> <span data-ttu-id="ecae5-134">Po odinstalowaniu programu ASP.NET w systemie operacyjnym, który jest uruchomiony program IIS 5.x i WCF zostanie odinstalowana, plik z rozszerzeniem .svc jest traktowane jako plik tekstowy i zawartość, łącznie z dowolnego kodu źródłowego, jest zwracany do obiektu żądającego.</span><span class="sxs-lookup"><span data-stu-id="ecae5-134">When ASP.NET is uninstalled on an operating system that is running IIS 5.x and WCF is uninstalled, a file with the .svc extension is considered a text file and the contents, including any source code, is returned to the requester.</span></span>
