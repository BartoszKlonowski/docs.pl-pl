  - <span data-ttu-id="e8561-131"><xref:System.ServiceModel.Discovery.FindCriteria.ScopeMatchByLdap?displayProperty=nameWithType> dopasowuje zakresy przez segmenty przy użyciu adresu URL LDAP.</span><span class="sxs-lookup"><span data-stu-id="e8561-131"><xref:System.ServiceModel.Discovery.FindCriteria.ScopeMatchByLdap?displayProperty=nameWithType> matches scopes by segments using an LDAP URL.</span></span>

  <span data-ttu-id="e8561-134">Jeśli nie określono reguły dopasowywania zakresu, <xref:System.ServiceModel.Discovery.FindCriteria.ScopeMatchByPrefix> jest używana.</span><span class="sxs-lookup"><span data-stu-id="e8561-134">If a scope-matching rule is not specified, <xref:System.ServiceModel.Discovery.FindCriteria.ScopeMatchByPrefix> is used.</span></span>

## <a name="findresponse"></a><span data-ttu-id="e8561-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="e8561-140">FindResponse</span></span>

<span data-ttu-id="e8561-141"><xref:System.ServiceModel.Discovery.FindResponse> ma właściwość kolekcji <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A>, która zawiera odpowiedzi wysyłane przez pasujące usługi w sieci.</span><span class="sxs-lookup"><span data-stu-id="e8561-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="e8561-142">Jeśli żadna usługa nie odpowiedziała, kolekcja jest pusta.</span><span class="sxs-lookup"><span data-stu-id="e8561-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="e8561-143">Jeśli co najmniej jedna usługa odpowiedziała, Każda odpowiedź jest przechowywana w obiekcie <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata>, który zawiera adres, kontrakt i dodatkowe informacje dotyczące usługi.</span><span class="sxs-lookup"><span data-stu-id="e8561-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>

<span data-ttu-id="e8561-144">Poniższy przykład pokazuje, jak wykonać operację znajdowania w kodzie.</span><span class="sxs-lookup"><span data-stu-id="e8561-144">The following example shows how to perform a find operation in code.</span></span>

```csharp
// Create DiscoveryClient
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());

// Create FindCriteria
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));
findCriteria.Duration = TimeSpan.FromSeconds(10);

// Find ICalculatorService endpoints
FindResponse findResponse = discoveryClient.Find(findCriteria);

Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)
```
