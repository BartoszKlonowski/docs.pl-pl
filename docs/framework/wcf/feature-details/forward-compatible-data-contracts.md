 <span data-ttu-id="7a6b7-109">Gdy dane przechodzą z nowej wersji do starej wersji i z powrotem do nowej wersji kontraktu danych, nastąpi przekroczenie.</span><span class="sxs-lookup"><span data-stu-id="7a6b7-109">Round-tripping occurs when data passes from a new version to an old version and back to the new version of a data contract.</span></span> <span data-ttu-id="7a6b7-110">Dwukierunkowa gwarancja gwarantuje, że żadne dane nie zostaną utracone.</span><span class="sxs-lookup"><span data-stu-id="7a6b7-110">Round-tripping guarantees that no data is lost.</span></span> <span data-ttu-id="7a6b7-111">Włączenie operacji okrężnej powoduje, że typ do przodu jest zgodny z wszelkimi przyszłymi zmianami obsługiwanymi przez model przechowywania wersji kontraktu danych.</span><span class="sxs-lookup"><span data-stu-id="7a6b7-111">Enabling round-tripping makes the type forward-compatible with any future changes supported by the data contract versioning model.</span></span>
