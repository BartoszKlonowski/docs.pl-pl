### <a name="transfer-security"></a><span data-ttu-id="2a524-172">Przeniesienie zabezpieczeń</span><span class="sxs-lookup"><span data-stu-id="2a524-172">Transfer Security</span></span>

 <span data-ttu-id="2a524-185">Aby uzyskać więcej informacji dotyczących tych różnic, zobacz [zabezpieczania usług i klientów](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span><span class="sxs-lookup"><span data-stu-id="2a524-185">For more information about these differences, see [Securing Services and Clients](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span></span>
