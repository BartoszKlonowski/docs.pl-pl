# <a name="message-security-with-issued-tokens"></a>Zabezpieczenia komunikatów z wystawionymi tokenami
Klient wystawiony token umożliwia scenariusz zabezpieczeń federacyjnych w programie Windows Communication Foundation (WCF).  
  
 Zabezpieczenia federacyjne umożliwiają ustanowienie zaufania między domenami niezależnymi. Każda domena scentralizowana uwierzytelnianie i autoryzację w usłudze tokenu zabezpieczającego. W przypadku ustanowienia zaufania między dwoma takimi usługami członkowie każdej z nich mogą korzystać z usług innych.  
  
 Szczegółowe wyjaśnienie scenariusza federacyjnego można znaleźć w temacie [federacyjne i wystawione tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
