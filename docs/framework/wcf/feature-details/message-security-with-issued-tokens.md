# <a name="message-security-with-issued-tokens"></a>Zabezpieczenia komunikatów z wystawionymi tokenami
Wystawiony token klienta umożliwia scenariusza federacyjnego zabezpieczenia w Windows Communication Foundation (WCF).  
  
 Zabezpieczeń umożliwia ustanowienie relacji zaufania między domenami niezależne. Każda domena centralizuje, uwierzytelnianie i autoryzacja w usługę tokenu zabezpieczającego. Po ustanowieniu zaufania między dwoma takich usług, członkowie dla każdej domeny, mogą korzystać z usług innych.  
  
 Aby uzyskać szczegółowe wyjaśnienie scenariusza federacyjnego, zobacz [Federacja i wystawione tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
