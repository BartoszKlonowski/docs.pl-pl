# <a name="message-security-with-issued-tokens"></a>Zabezpieczenia komunikatów z wystawionymi tokenami
Wystawiony token klienta umożliwia scenariusz zabezpieczeń w systemie Windows Communication Foundation (WCF).  
  
 Zabezpieczeń umożliwia ustanowienie zaufania między domenami niezależne. Każda domena centralizuje uwierzytelnianie i autoryzację w usługi tokenu zabezpieczającego. Podczas ustanawiania relacji zaufania między dwoma takich usług, członkowie każdej domeny, można użyć usług innych.  
  
 Aby uzyskać szczegółowe wyjaśnienie scenariusza federacyjnych, zobacz [Federacja i wystawione tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
