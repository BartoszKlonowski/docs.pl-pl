 <span data-ttu-id="65542-117">Model programowania sieci Web, które są opisane w [programowania omówienie modelu WCF Web HTTP](../../../../docs/framework/wcf/feature-details/wcf-web-http-programming-model-overview.md) może być używany z usługami ASP.NET AJAX.</span><span class="sxs-lookup"><span data-stu-id="65542-117">The Web Programming Model described in the [WCF Web HTTP Programming Model Overview](../../../../docs/framework/wcf/feature-details/wcf-web-http-programming-model-overview.md) may be used with ASP.NET AJAX services.</span></span> <span data-ttu-id="65542-118">W szczególności:</span><span class="sxs-lookup"><span data-stu-id="65542-118">Specifically:</span></span>
