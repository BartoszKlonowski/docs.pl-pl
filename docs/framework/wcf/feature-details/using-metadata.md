# <a name="using-metadata"></a>Używanie metadanych
Metadane usługi zawierają opis usługi do odczytu maszynowego. Metadane usługi obejmują opisy punktów końcowych, powiązań, kontraktów, operacji i komunikatów usługi. Możesz użyć metadanych usługi do różnych celów, w tym automatycznego generowania klienta do korzystania z usługi, implementowania opisu usługi i dynamicznego aktualizowania powiązania dla klienta.
