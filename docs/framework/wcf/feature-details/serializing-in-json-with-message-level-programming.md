## <a name="untyped-message-programming"></a><span data-ttu-id="efc6a-115">Komunikat bez programowania</span><span class="sxs-lookup"><span data-stu-id="efc6a-115">Untyped Message Programming</span></span>  
 <span data-ttu-id="efc6a-116">Podczas pracy bezpośrednio z obiektów bez typu komunikatu, musisz jawnie ustawić właściwości na komunikat bez typu, który ma serializować go jako plik JSON.</span><span class="sxs-lookup"><span data-stu-id="efc6a-116">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="efc6a-117">Poniższy fragment kodu pokazuje, jak to zrobić.</span><span class="sxs-lookup"><span data-stu-id="efc6a-117">The following code snippet shows how to do this.</span></span>  
  
```  
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```
