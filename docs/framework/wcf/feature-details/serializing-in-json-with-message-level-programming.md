## <a name="untyped-message-programming"></a><span data-ttu-id="1004d-116">Programowanie komunikatów niewpisanych</span><span class="sxs-lookup"><span data-stu-id="1004d-116">Untyped Message Programming</span></span>  
 <span data-ttu-id="1004d-117">Podczas pracy bezpośrednio z niewpisanymi obiektami komunikatów należy jawnie ustawić właściwości niewpisanej wiadomości, aby serializować ją jako kod JSON.</span><span class="sxs-lookup"><span data-stu-id="1004d-117">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="1004d-118">Poniższy fragment kodu przedstawia, jak to zrobić.</span><span class="sxs-lookup"><span data-stu-id="1004d-118">The following code snippet shows how to do this.</span></span>  
  
```  
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```
