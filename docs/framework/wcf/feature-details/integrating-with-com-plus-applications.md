# <a name="integrating-with-com-applications"></a>Współdziałanie z aplikacjami COM+

Windows Communication Foundation (WCF) oferuje bogate środowisko do tworzenia aplikacji rozproszonych. Jeśli masz znaczną inwestycyjną w logice aplikacji opartej na składnikach hostowanej w modelu COM+, możesz użyć programu WCF do rozbudowania istniejącej logiki zamiast konieczności jej ponownego zapisywania. W tematach w tej sekcji opisano sposób korzystania z modelu COM+ z programem WCF.
