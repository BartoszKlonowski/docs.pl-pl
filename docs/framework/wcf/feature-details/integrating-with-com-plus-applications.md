# <a name="integrating-with-com-applications"></a>Współdziałanie z aplikacjami COM+
Windows Communication Foundation (WCF) zapewnia bogate środowisko tworzenia aplikacji rozproszonych. Jeśli masz znaczne inwestycje w logice oparty na komponentach aplikacji hostowanej w modelu COM +, można użyć WCF do rozszerzenia Twojej istniejącej logiki, a nie po prostu go do. Tematy w tej sekcji opisano, jak za pomocą modelu COM + WCF.
