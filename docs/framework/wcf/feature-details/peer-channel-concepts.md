# <a name="peer-channel-concepts"></a>Pojęcia kanałów równorzędnych
W tej sekcji opisano skrótowo pojęcia, które tworzą kanał elementu równorzędnego Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Komunikacja równorzędna siatek](../../../../docs/framework/wcf/feature-details/peer-meshes.md):  Opisuje charakterystykę siatek utworzone za pomocą kanału równorzędnego.  
  
 [Komunikacja równorzędna węzłów](../../../../docs/framework/wcf/feature-details/peer-nodes.md):  W tym artykule opisano węzły równorzędne.  
  
 [Zabezpieczenia kanału równorzędnego](../../../../docs/framework/wcf/feature-details/peer-channel-security.md):  Opisuje scenariusze zabezpieczeń i rozwiązaniach, dostarczane przez kanał elementu równorzędnego.  
  
 [Komunikacja równorzędna rozwiązujący](../../../../docs/framework/wcf/feature-details/peer-resolvers.md):  W tym artykule omówiono używanie równorzędnej nazwę rozwiązania protokołu (PRNP) i innych niestandardowych mechanizmów rozpoznawania.
