# <a name="support-for-json-and-other-data-transfer-formats"></a>Obsługa formatu JSON i innych formatów transferowania danych

JSON (JavaScript Object Notation) to wydajny format kodowania danych, który umożliwia szybką wymianę małych ilości danych między przeglądarkami klientów a usługami sieci Web obsługującymi technologię AJAX (asynchroniczne skrypty JavaScript i XML).  
  
## <a name="in-this-section"></a>W tej sekcji  

 [Autonomiczna serializacja kodu JSON](stand-alone-json-serialization.md)  
 Opisuje autonomiczną Serializacja kodu JSON.  
  
 [Instrukcje: serializowanie i deserializowanie danych JSON](how-to-serialize-and-deserialize-json-data.md)  
 Demonstruje sposób serializacji obiektów typu .NET do danych zakodowanych w formacie JSON, a następnie deserializacji danych z powrotem do wystąpień typów .NET przy użyciu <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> .  
  
 [Mapowanie między formatami JSON i XML](mapping-between-json-and-xml.md)  
 Opisuje zgodność z kodowaniem JavaScript Object Notation (JSON) i sprawdzonych XML, który jest używany wewnętrznie przez Windows Communication Foundation (WCF) do reprezentowania danych i dokumentów zakodowanych w formacie JSON.
