## <a name="port-sharing-architecture"></a><span data-ttu-id="71418-121">Architektura współużytkowania portów</span><span class="sxs-lookup"><span data-stu-id="71418-121">Port Sharing Architecture</span></span>

- <span data-ttu-id="71418-124">Transportu TCP usługi WCF: Implementuje protokół net.tcp://.</span><span class="sxs-lookup"><span data-stu-id="71418-124">The WCF TCP transport: Implements the net.tcp:// protocol.</span></span>

 <span data-ttu-id="71418-126">Usługi udostępniania portów Net.TCP jest usługa Windows trybu użytkownika, która akceptuje połączenia net.tcp:// imieniu procesów roboczych, łączących się za jego pośrednictwem.</span><span class="sxs-lookup"><span data-stu-id="71418-126">The Net.TCP Port Sharing Service is a user-mode Windows service that accepts net.tcp:// connections on behalf of the worker processes that connect through it.</span></span> <span data-ttu-id="71418-127">Po odebraniu połączenia z gniazdem współużytkowania portów bada strumienia komunikatów przychodzących w celu uzyskania adresu docelowego.</span><span class="sxs-lookup"><span data-stu-id="71418-127">When a socket connection arrives, the port sharing service inspects the incoming message stream to obtain its destination address.</span></span> <span data-ttu-id="71418-128">Oparte na ten adres, port udostępnianej usługi może kierować strumień danych do aplikacji, które ostatecznie przetwarza je.</span><span class="sxs-lookup"><span data-stu-id="71418-128">Based on this address, the port sharing service can route the data stream to the application that ultimately processes it.</span></span>
