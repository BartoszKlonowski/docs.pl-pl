# <a name="partial-trust"></a>Zaufanie częściowe
Począwszy od [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)], częściowo zaufanych wywołań mogą uzyskiwać dostęp do typów publicznych i metody implementowane w <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>, i <xref:System.ServiceModel.Web>. W tej sekcji opisano obsługiwane scenariusze użycia usługi Windows Communication Foundation (WCF) w aplikacji częściowo zaufanej, jak i ograniczony podzestaw funkcji WCF dostępnych dla aplikacji działających z zabezpieczeniami dostępu zmniejszenie kodu (CAS) uprawnienia.
