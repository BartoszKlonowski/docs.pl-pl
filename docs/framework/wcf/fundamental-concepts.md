 <span data-ttu-id="317b5-246">zabezpieczenia</span><span class="sxs-lookup"><span data-stu-id="317b5-246">security</span></span>

 <span data-ttu-id="317b5-249">Tryb zabezpieczeń Transport</span><span class="sxs-lookup"><span data-stu-id="317b5-249">transport security mode</span></span>

approach is the complexity of the cryptographic mechanisms employed, resulting in performance implications.</span></span>
