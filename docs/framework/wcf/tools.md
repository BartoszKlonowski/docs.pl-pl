# <a name="windows-communication-foundation-tools"></a>Narzędzia programu Windows Communication Foundation
Narzędzia Microsoft Windows Communication Foundation (WCF) są przeznaczone do ułatwienia umożliwiające tworzenie, wdrażanie i zarządzanie aplikacjami usługi WCF. Ta sekcja zawiera szczegółowe informacje na temat narzędzia. Należy pamiętać, że nie są obsługiwane narzędzia.  
  
 Wszystkie narzędzia, które można uruchomić z wiersza polecenia.  
  
 W poniższej tabeli wymieniono te narzędzia i krótki opis.  
  
|Narzędzie|Opis|  
|----------|-----------------|  
|[Narzędzie do obsługi metadanych elementu ServiceModel (Svcutil.exe)](../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md)|Generuje kod modelu usługi na podstawie dokumentów metadanych i dokumentów metadanych z kodu modelu usługi.|  
|[Narzędzie do znajdowania klucza prywatnego (FindPrivateKey.exe)](../../../docs/framework/wcf/find-private-key-tool-findprivatekey-exe.md)|Pobiera klucz prywatny z określonego magazynu.|  
|[Narzędzie do rejestracji modelu ServiceModel (ServiceModelReg.exe)](../../../docs/framework/wcf/servicemodelreg-exe.md)|Zarządza rejestracji i wyrejestrować rejestracji modelu ServiceModel na jednym komputerze.|  
|[Narzędzie konfiguracji modelu usług COM+ (ComSvcConfig.exe)](../../../docs/framework/wcf/com-service-model-configuration-tool-comsvcconfig-exe.md)|Konfiguruje interfejsy modelu COM + być udostępniane jako usługi sieci Web.|  
|[Narzędzie edytora konfiguracji (SvcConfigEditor.exe)](../../../docs/framework/wcf/configuration-editor-tool-svcconfigeditor-exe.md)|Tworzy i modyfikuje ustawienia konfiguracji dla usługi WCF.|  
|[Narzędzie do przeglądania danych śledzenia usług (SvcTraceViewer.exe)](../../../docs/framework/wcf/service-trace-viewer-tool-svctraceviewer-exe.md)|Ułatwia przeglądanie, grupy i filtr śledzenia komunikatów, dzięki czemu możesz zdiagnozować, naprawy i sprawdź problemy z usługami WCF.|  
|[Narzędzie do konfiguracji elementu WS-AtomicTransaction (wsatConfig.exe)](../../../docs/framework/wcf/ws-atomictransaction-configuration-utility-wsatconfig-exe.md)|Konfiguruje ustawienia pomocy technicznej WS-AtomicTransaction podstawowe za pomocą narzędzia wiersza polecenia.|  
|[Przystawka programu MMC do konfigurowania elementu WS-AtomicTransaction](../../../docs/framework/wcf/ws-atomictransaction-configuration-mmc-snap-in.md)|Konfiguruje ustawienia pomocy technicznej WS-AtomicTransaction podstawowe za pomocą przystawki programu MMC.|  
|[Narzędzie rejestracji usług przepływu pracy (WFServicesReg.exe)](../../../docs/framework/wcf/workflow-service-registration-tool-wfservicesreg-exe.md)|Rejestruje usługę Windows Workflow.|  
|[Host usługi WCF (WcfSvcHost.exe)](../../../docs/framework/wcf/wcf-service-host-wcfsvchost-exe.md)|Usługi WCF hostów zawartych w bibliotekach plików (*.dll)|  
|[Testowy klient WCF (WcfTestClient.exe)](../../../docs/framework/wcf/wcf-test-client-wcftestclient-exe.md)|Narzędzie graficznego interfejsu użytkownika, które pozwala na parametry wejściowe arbitralnie wybrane typy przedstawia te dane wejściowe do usługi i widok, który odpowiedzi usługa wysyła z powrotem.|  
|[Narzędzie Contract-First](../../../docs/framework/wcf/contract-first-tool.md)|Visual Studio, twórz zadanie, które tworzy kod klasy na podstawie kontraktów danych XSD.|  
  
 Wszystkich poprzednich narzędzi z wyjątkiem ServiceModelReg.exe, WsatConfig.exe i ComSvcConfig.exe dostarczanie za pomocą zestawu Windows SDK i znajduje się w folderze C:\Program Files\Microsoft SDKs\Windows\v6.0\Bin.  Określone narzędzia 3 znajdują się w obszarze C:\Windows\Microsoft.NET\Framework\v3.0\Windows Communication Foundation.
