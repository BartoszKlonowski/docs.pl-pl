# <a name="custom-binding"></a>Wiązanie niestandardowe
Ta sekcja zawiera przykłady demonstrujące użycie niestandardowego atrybutu powiązania elementu punktu końcowego.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Powiązanie niestandardowe imperatywne](custom-binding-imperative.md)  
 Pokazuje, jak napisać sam kod w celu zdefiniowania i użycia niestandardowych powiązań bez użycia pliku konfiguracji ani klienta wygenerowanego przez program WCF.  
  
 [Transport i kodowanie powiązań niestandardowych](custom-binding-transport-and-encoding.md)  
 Pokazuje, jak skonfigurować powiązanie niestandardowe z różnymi elementami transportu i przesyłania komunikatów.  
  
 [Sesja niezawodna powiązania niestandardowego](custom-binding-reliable-session.md)  
 Pokazuje, jak skonfigurować powiązanie niestandardowe z różnymi elementami transportu i transportem komunikatów, szczególnie w przypadku włączania niezawodnych sesji.  
  
 [Niestandardowe powiązanie niezawodnej sesji przez protokół HTTPS](custom-binding-reliable-session-over-https.md)  
 Demonstruje użycie zabezpieczeń transportu SSL z niezawodnymi sesjami.
