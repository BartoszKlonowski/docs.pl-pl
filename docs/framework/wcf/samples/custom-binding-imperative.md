# <a name="custom-binding-imperative"></a>Powiązanie niestandardowe imperatywne
Przykład pokazuje, jak napisać kod imperatywny do definiowania i używania niestandardowych powiązań bez użycia pliku konfiguracji lub klienta generowanego przez Program Windows Communication Foundation (WCF). Ten przykład łączy funkcje dostarczane przez transport HTTP i niezawodny kanał sesji, aby utworzyć niezawodne powiązanie oparte na protoko. Ten przykład jest oparty na [wprowadzenie,](../../../../docs/framework/wcf/samples/getting-started-sample.md) który implementuje usługę kalkulatora.  
  
> [!NOTE]
> Procedura konfiguracji i instrukcje kompilacji dla tego przykładu znajdują się na końcu tego tematu.  
  
 Zarówno na kliencie, jak i w usłudze tworzone jest niestandardowe powiązanie, które zawiera dwa elementy wiązania (niezawodna sesja i HTTP):  

```csharp
ReliableSessionBindingElement reliableSession = new ReliableSessionBindingElement();  
reliableSession.Ordered = true;  
  
HttpTransportBindingElement httpTransport = new HttpTransportBindingElement();  
httpTransport.AuthenticationScheme = System.Net.AuthenticationSchemes.Anonymous;  
httpTransport.HostNameComparisonMode = HostNameComparisonMode.StrongWildcard;  
  
CustomBinding binding = new CustomBinding(reliableSession, httpTransport);  
```
  
 W usłudze powiązanie jest używane przez dodanie punktu końcowego do ServiceHost:  

```csharp
serviceHost.AddServiceEndpoint(typeof(ICalculator), binding, "");  
```

 Na kliencie powiązanie jest <xref:System.ServiceModel.ChannelFactory> używane przez a do utworzenia kanału do usługi:  

```csharp
EndpointAddress address = new EndpointAddress("http://localhost:8000/servicemodelsamples/service");  
ChannelFactory<ICalculator> channelFactory = new ChannelFactory<ICalculator>(binding, address);  
ICalculator channel = channelFactory.CreateChannel();  
```

 Ten kanał jest następnie używany do interakcji z usługą:  

```csharp
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = channel.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
```

 Po uruchomieniu próbki żądania operacji i odpowiedzi są wyświetlane w oknie konsoli klienta. Naciśnij klawisz ENTER w oknie klienta, aby zamknąć klienta.  
  
```console  
Add(100,15.99) = 115.99  
Subtract(145,76.54) = 68.46  
Multiply(9,81.25) = 731.25  
Divide(22,7) = 3.14285714285714  
  
Press <ENTER> to terminate client.  
```
