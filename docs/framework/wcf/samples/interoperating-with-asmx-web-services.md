<span data-ttu-id="22217-103">Ten przykład pokazuje, jak zintegrować aplikację kliencką Windows Communication Foundation (WCF) z istniejącą usługą sieci Web ASMX.</span><span class="sxs-lookup"><span data-stu-id="22217-103">This sample demonstrates how to integrate a Windows Communication Foundation (WCF) client application with an existing ASMX Web service.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="22217-104">Procedura instalacji i instrukcje dotyczące kompilacji dla tego przykładu znajdują się na końcu tego tematu.</span><span class="sxs-lookup"><span data-stu-id="22217-104">The setup procedure and build instructions for this sample are located at the end of this topic.</span></span>  
  
 <span data-ttu-id="22217-105">Ten przykład składa się z programu konsolowego klienta (exe) i biblioteki usług (. dll) hostowanej przez Internet Information Services (IIS).</span><span class="sxs-lookup"><span data-stu-id="22217-105">This sample consists of a client console program (.exe) and a service library (.dll) hosted by Internet Information Services (IIS).</span></span> <span data-ttu-id="22217-106">Usługa jest usługą sieci Web ASMX, która implementuje kontrakt definiujący wzorzec komunikacji z odpowiedzią na żądanie.</span><span class="sxs-lookup"><span data-stu-id="22217-106">The service is an ASMX Web Service that implements a contract that defines a request-reply communication pattern.</span></span> <span data-ttu-id="22217-107">Usługa ujawnia operacje matematyczne (`Add`, `Subtract`, `Multiply`i `Divide`).</span><span class="sxs-lookup"><span data-stu-id="22217-107">The service exposes math operations (`Add`, `Subtract`, `Multiply`, and `Divide`).</span></span> <span data-ttu-id="22217-108">Klient wysyła żądania synchroniczne do operacji matematycznej, a usługa w odpowiedzi na wynik.</span><span class="sxs-lookup"><span data-stu-id="22217-108">The client makes synchronous requests to a math operation and the service replies with the result.</span></span> <span data-ttu-id="22217-109">Aktywność klienta jest widoczna w oknie konsoli.</span><span class="sxs-lookup"><span data-stu-id="22217-109">Client activity is visible in the console window.</span></span>  
  
 <span data-ttu-id="22217-110">Implementacja usługi sieci Web ASMX pokazana w poniższym przykładowym kodzie oblicza i zwraca odpowiedni wynik.</span><span class="sxs-lookup"><span data-stu-id="22217-110">The ASMX Web service implementation shown in the following sample code calculates and returns the appropriate result.</span></span>  
  
```csharp  
[WebService(Namespace="http://Microsoft.ServiceModel.Samples")]  
public class CalculatorService : System.Web.Services.WebService  
    {  
        [WebMethod]  
        public double Add(double n1, double n2)  
        {  
            return n1 + n2;  
        }  
        [WebMethod]  
        public double Subtract(double n1, double n2)  
        {  
            return n1 - n2;  
        }  
        [WebMethod]  
        public double Multiply(double n1, double n2)  
        {  
            return n1 * n2;  
        }  
        [WebMethod]  
        public double Divide(double n1, double n2)  
        {  
            return n1 / n2;  
        }  
    }  
```  
  
 <span data-ttu-id="22217-111">Zgodnie z konfiguracją usługa może być dostępna na `http://localhost/servicemodelsamples/service.asmx` przez klienta programu na tym samym komputerze.</span><span class="sxs-lookup"><span data-stu-id="22217-111">As configured, the service can be accessed at `http://localhost/servicemodelsamples/service.asmx` by a client on the same machine.</span></span> <span data-ttu-id="22217-112">Aby klienci na komputerach zdalnych mogli uzyskać dostęp do usługi, należy określić kwalifikowaną nazwę domeny zamiast hosta lokalnego.</span><span class="sxs-lookup"><span data-stu-id="22217-112">For clients on remote machines to access the service, a qualified domain name must be specified instead of localhost.</span></span>  
  
 <span data-ttu-id="22217-113">Komunikacja odbywa się za pomocą klienta wygenerowanego przez narzędzie do obsługi [metadanych ServiceModel (Svcutil. exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span><span class="sxs-lookup"><span data-stu-id="22217-113">Communication is done through a client generated by the [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span></span> <span data-ttu-id="22217-114">Klient jest zawarty w pliku generatedClient.cs.</span><span class="sxs-lookup"><span data-stu-id="22217-114">The client is contained in the file generatedClient.cs.</span></span> <span data-ttu-id="22217-115">Usługa ASMX musi być dostępna do wygenerowania kodu serwera proxy, ponieważ służy do pobierania zaktualizowanych metadanych.</span><span class="sxs-lookup"><span data-stu-id="22217-115">The ASMX service must be available to generate the proxy code, because it is used to retrieve the updated metadata.</span></span> <span data-ttu-id="22217-116">Uruchom następujące polecenie w wierszu polecenia w katalogu klienta, aby wygenerować serwer proxy z określonym typem.</span><span class="sxs-lookup"><span data-stu-id="22217-116">Run the following command from a command prompt in the client directory to generate the typed proxy.</span></span>  
  
```console  
svcutil.exe /n:http://Microsoft.ServiceModel.Samples,Microsoft.ServiceModel.Samples http://localhost/servicemodelsamples/service.svc?wsdl /out:generatedClient.cs  
```  
  
 <span data-ttu-id="22217-117">Korzystając z wygenerowanego klienta, można uzyskać dostęp do punktu końcowego usługi przez skonfigurowanie odpowiedniego adresu i powiązania.</span><span class="sxs-lookup"><span data-stu-id="22217-117">By using the generated client, you can access a service endpoint by configuring the appropriate address and binding.</span></span> <span data-ttu-id="22217-118">Podobnie jak w przypadku usługi, klient używa pliku konfiguracji (App. config), aby określić punkt końcowy do komunikacji.</span><span class="sxs-lookup"><span data-stu-id="22217-118">Like the service, the client uses a configuration file (App.config) to specify the endpoint to communicate with.</span></span> <span data-ttu-id="22217-119">Konfiguracja punktu końcowego klienta składa się z adresu bezwzględnego dla punktu końcowego usługi, powiązania i kontraktu, jak pokazano w poniższej konfiguracji przykładowej.</span><span class="sxs-lookup"><span data-stu-id="22217-119">The client endpoint configuration consists of an absolute address for the service endpoint, the binding, and the contract, as shown in the following sample configuration.</span></span>  
  
```xml  
<client>  
   <endpoint   
      address="http://localhost/ServiceModelSamples/service.asmx"   
      binding="basicHttpBinding"   
      contract="Microsoft.ServiceModel.Samples.CalculatorServiceSoap" />  
</client>  
```  
  
 <span data-ttu-id="22217-120">Implementacja klienta tworzy wystąpienie wygenerowanego klienta.</span><span class="sxs-lookup"><span data-stu-id="22217-120">The client implementation constructs an instance of the generated client.</span></span> <span data-ttu-id="22217-121">Wygenerowany klient może być następnie używany do komunikowania się z usługą.</span><span class="sxs-lookup"><span data-stu-id="22217-121">The generated client can then be used to communicate with the service.</span></span>  
  
```csharp  
// Create a client.  
CalculatorServiceSoapClient client = new CalculatorServiceSoapClient();  
  
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = client.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
// Call the Subtract service operation.  
value1 = 145.00D;  
value2 = 76.54D;  
result = client.Subtract(value1, value2);  
Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
// Call the Multiply service operation.  
value1 = 9.00D;  
value2 = 81.25D;  
result = client.Multiply(value1, value2);  
Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
// Call the Divide service operation.  
value1 = 22.00D;  
value2 = 7.00D;  
result = client.Divide(value1, value2);  
Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
  
Console.WriteLine();  
Console.WriteLine("Press <ENTER> to terminate client.");  
Console.ReadLine();  
```
