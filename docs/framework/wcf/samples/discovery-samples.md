# <a name="discovery-samples"></a>Odnajdywanie (przykłady)
Ta sekcja zawiera przykłady powiązanych z tą funkcją odnajdywania.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Anonse](../../../../docs/framework/wcf/samples/announcements-sample.md)  
 Demonstracja anonsów za pomocą funkcji odnajdywania.  
  
 [Znajdowanie asynchroniczne](../../../../docs/framework/wcf/samples/asynchronous-find-sample.md)  
 Pokazuje, jak wykonać znajdowanie asynchroniczne, aby wyszukać wszystkie usługi Kalkulator.  
  
 [Podstawy](../../../../docs/framework/wcf/samples/basic-sample.md)  
 Pokazuje, jak utworzyć wykrywalny programowo, a także wyszukiwania dla tej usługi przy użyciu usługi <xref:System.ServiceModel.Discovery.DynamicEndpoint>.  
  
 [Konfiguracja](../../../../docs/framework/wcf/samples/configuration-sample.md)  
 Pokazuje, jak sprawiają, że usługa wykrywania urządzeń przy użyciu pliku konfiguracji, a także sposób użycia <xref:System.ServiceModel.Discovery.DynamicEndpoint> za pomocą konfiguracji do wyszukiwania dla tej usługi.  
  
 [Przykład elementu powiązania odnajdywania](../../../../docs/framework/wcf/samples/discovery-binding-element-sample.md)  
 Pokazuje, jak dodać element powiązania odnajdywania do niestandardowego powiązania. To umożliwia klientom odnajdowanie usługi, po otwarciu kanału.  
  
 [Przykład serwera proxy odnajdywania](../../../../docs/framework/wcf/samples/discovery-proxy-sample.md)  
 Pokazuje, jak utworzyć wdrożenia serwera Proxy odnajdywania do przechowywania informacji o istniejących usług i jak klienci mogą wykonywać kwerendę czy serwera proxy, aby uzyskać informacje.  
  
 [Odnajdywanie usługi działającej w trybie unikatowego identyfikatora URI nasłuchiwania — przykład](../../../../docs/framework/wcf/samples/discover-a-service-with-unique-listen-uri-mode-sample.md)  
 Pokazuje, jak odnajdywanie usługi, która ma <xref:System.ServiceModel.Channels.BindingContext.ListenUriMode%2A> ustawioną właściwość <xref:System.ServiceModel.Description.ListenUriMode.Unique>.  
  
 [Odnajdywanie z zakresami](../../../../docs/framework/wcf/samples/discovery-with-scopes-sample.md)  
 Pokazuje, jak korzystanie z zakresów kategoryzację wykrywalny punktów końcowych, jak również sposób użycia <xref:System.ServiceModel.Discovery.DiscoveryClient> do wykonywania asynchronicznych wyszukiwania dla punktów końcowych.  
  
 [Niestandardowe kryteria znajdowania](../../../../docs/framework/wcf/samples/custom-find-criteria.md)  
 Przedstawiono sposób tworzenia dopasowanie niestandardowy zakres przy użyciu logiki oraz implementacji usługi odnajdywania niestandardowych.  
  
 [Odnajdywanie przepływu pracy — przykład](../../../../docs/framework/wcf/samples/workflow-discovery-sample.md)  
 Przedstawiono sposób wprowadzania wykrywalny usługi przepływu pracy oraz tworzyć działania niestandardowego kodu, która wyszukuje określonej usługi.  
  
 [Usługa routera odnajdywania](../../../../docs/framework/wcf/samples/discovery-router-service.md)
