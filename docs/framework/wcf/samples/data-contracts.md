# <a name="data-contracts"></a>Kontrakty danych

Ta sekcja zawiera przykłady demonstrujące kontrakty dotyczące danych w programie Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  

 [Podstawowy kontrakt danych](basic-data-contract.md)  
 Demonstruje sposób implementacji kontraktu danych.  
  
 [Przykład elementu DataContractSerializer](datacontractserializer-sample.md)  
 Pokazuje <xref:System.Runtime.Serialization.DataContractSerializer> , który wykonuje ogólne usługi serializacji i deserializacji dla klas kontraktu danych.  
  
 [Znane typy](known-types.md)  
 Pokazuje, jak określić informacje o typach pochodnych w kontrakcie danych.  
  
 [Odwołania do obiektów](object-references.md)  
 Demonstruje sposób przekazywania obiektów przez odwołania między serwerem a klientem.  
  
 [Obsługa obiektów POCO](poco-support.md)  
 Demonstruje obsługę serializacji dla nieoznaczonych typów; oznacza to, że typy, do których nie zastosowano atrybutów serializacji, czasami określane jako zwykły typ obiektów CLR (POCO).  
  
 [Używanie obiektu wiążącego serializacji](usage-of-serialization-binder.md)  
 Ten przykład pokazuje, jak użyć <xref:System.Runtime.Serialization.SerializationBinder> do zmiany wersji typu ogólnego podczas serializacji.
