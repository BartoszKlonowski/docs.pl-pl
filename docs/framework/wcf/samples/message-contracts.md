# <a name="message-contracts"></a>Kontrakty komunikatów
Ta sekcja zawiera przykłady ilustrujące kontraktów komunikatu w systemie Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Domyślny kontrakt komunikatów](../../../../docs/framework/wcf/samples/default-message-contract.md)  
 Pokazuje usługi, gdy wiadomość niestandardowe zdefiniowane przez użytkownika jest przekazywany do i z operacji usługi.  
  
 Bez typu żądanie odpowiedź.  
 Pokazuje, jak zdefiniować kontrakty operacji, które używają klasy wiadomości.  
  
 [Nieopakowane komunikaty](../../../../docs/framework/wcf/samples/unwrapped-messages.md)  
 Pokazuje nieopakowane komunikaty. Domyślnie treść komunikatu jest sformatowany w taki sposób, że są opakowywane parametrów do operacji usługi.  
  
 [Ustawianie właściwości Use i Style](../../../../docs/framework/wcf/samples/setting-the-use-and-style-properties.md)  
 Przedstawiono sposób użycia właściwości Use i Style na <xref:System.ServiceModel.XmlSerializerFormatAttribute> i <xref:System.ServiceModel.DataContractFormatAttribute>.  
  
 [XmlReader — przykład](../../../../docs/framework/wcf/samples/xmlreader-sample.md)  
 Pokazuje przetwarzania treści wiadomości przy użyciu <xref:System.Xml.XmlReader>.
