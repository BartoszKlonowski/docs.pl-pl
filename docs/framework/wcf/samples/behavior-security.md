Ta sekcja zawiera przykłady, które pokazują, jak skonfigurować zabezpieczenia dla zachowań usługi.

 W tym przykładzie pokazano, jak usługa może używać członkostwa ASP.NET i dostawców ról do uwierzytelniania i autoryzowania klientów.

 [Personifikowanie klienta](impersonating-the-client.md)  
 Ten przykład pokazuje, jak personifikować aplikację wywołującą w usłudze, aby usługa mogła uzyskać dostęp do zasobów systemowych w imieniu obiektu wywołującego.
