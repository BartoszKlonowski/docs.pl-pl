# <a name="syndication-extensibility-samples"></a>Przykłady rozszerzania syndykacji
Ta sekcja zawiera przykłady pokazujące, niestandardowe syndykacji w Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Rozszerzenia z silną kontrolą typów](../../../../docs/framework/wcf/samples/strongly-typed-extensions-sample.md)  
 Demonstruje sposób implementacji elementu rozszerzenia zdefiniowane w RFC proponowanych rozszerzeń wątkowości Atom.  
  
 [Program formatujący źródła danych (format JSON)](../../../../docs/framework/wcf/samples/feed-formatter-json.md)  
 Pokazuje, jak do serializacji wystąpienia <xref:System.ServiceModel.Syndication.SyndicationFeed> klasy w formacie JavaScript Object Notation (JSON) za pomocą niestandardowego <xref:System.ServiceModel.Syndication.SyndicationFeedFormatter> i <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer>.  
  
 [Strumieniowe źródła danych](../../../../docs/framework/wcf/samples/streaming-feeds-sample.md)
