# <a name="behaviors"></a>Zachowania

Ta sekcja zawiera przykłady przedstawiające zachowania usługi Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  

 [Współbieżność](concurrency.md)  
 Demonstruje użycie <xref:System.ServiceModel.ServiceBehaviorAttribute> z <xref:System.ServiceModel.ConcurrencyMode> wyliczeniem, które określa, czy wystąpienie usługi przetwarza wiadomości sekwencyjnie czy współbieżnie.  
  
 [Domyślne zachowanie usługi](default-service-behavior.md)  
 Pokazuje, jak można skonfigurować ustawienia zachowania usługi.  
  
 [Tworzenie wystąpienia](instancing.md)  
 Pokazuje ustawienie zachowania tworzenia wystąpień, które kontroluje sposób, w jaki wystąpienia klasy usługi są tworzone w odpowiedzi na żądania klientów.  
  
 [Zachowanie publikowania metadanych](metadata-publishing-behavior.md)  
 Pokazuje, jak sterować funkcjami publikowania metadanych usługi.  
  
 [Zachowanie transakcji usługi](service-transaction-behavior.md)  
 Demonstruje użycie transakcji skoordynowanej przez klienta oraz ustawień <xref:System.ServiceModel.ServiceBehaviorAttribute> i <xref:System.ServiceModel.OperationBehaviorAttribute> sterowania zachowaniem transakcji usługi.  
  
 [Zachowanie debugowania usług](service-debug-behavior.md)  
 Pokazuje, jak można skonfigurować ustawienia zachowania debugowania usługi.  
  
 [Ograniczanie przepływności](throttling.md)  
 Demonstruje użycie formantów ograniczających.  
  
 [Zabezpieczenia zachowania](behavior-security.md)  
 Demonstruje konfigurację zabezpieczeń zachowania usługi.
