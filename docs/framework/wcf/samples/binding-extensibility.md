# <a name="binding-extensibility"></a>Rozszerzalność wiązań

Ta sekcja zawiera przykłady demonstrujące niestandardowe powiązania w Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  

 <xref:System.ServiceModel.NetHttpBinding>  
 Demonstruje sposób implementacji powiązania, które stosy <xref:System.ServiceModel.Channels.HttpTransportBindingElement> lub <xref:System.ServiceModel.Channels.HttpsTransportBindingElement> na górze <xref:System.ServiceModel.Channels.BinaryMessageEncodingBindingElement> .  
  
 [WSStreamedHttpBinding](wsstreamedhttpbinding.md)  
 Pokazuje, jak utworzyć powiązanie, które jest przeznaczone do obsługi scenariuszy przesyłania strumieniowego, gdy używany jest transport HTTP.
