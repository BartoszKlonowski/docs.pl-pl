# <a name="tool-samples"></a>Przykłady narzędzi
Ta sekcja zawiera narzędzia Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>W tej sekcji  
 [ConfigurationCodeGenerator](../../../../docs/framework/wcf/samples/configurationcodegenerator.md)  
 W tym artykule opisano sposób użycia narzędzia ConfigurationCodeGenerator.exe do udostępnienia implementacji usługi niestandardowym kanale w celu system konfiguracji.  
  
 [CustomChannelsTester](../../../../docs/framework/wcf/samples/customchannelstester.md)  
 W tym artykule opisano sposób użycia narzędzia CustomChannelsTester.exe do testowania implementacji usługi niestandardowym kanale z zestawem kontraktów predefiniowaną usługę.  
  
 [FindPrivateKey](../../../../docs/framework/wcf/samples/findprivatekey.md)  
 W tym artykule opisano sposób użycia narzędzia FindPrivateKey.exe ułatwienie procesu odnajdywania, lokalizację i nazwę pliku klucza prywatnego skojarzonego z certyfikatem X.509 określonych w magazynie certyfikatów.
