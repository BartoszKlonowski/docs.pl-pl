# <a name="wcf-and-net-framework-client-profile"></a>WCF i .NET Framework Client Profile

.NET Framework profile klienta to uproszczona wersja pełnego .NET Framework przeznaczona dla klientów, którzy nie potrzebują całej struktury. Struktura klienta nie obsługuje wszystkich Windows Communication Foundation.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Funkcje WCF obsługiwane przez .NET Framework profilu klienta  

 .NET Framework profilu klienta obsługiwane są następujące funkcje Windows Communication Foundation:  
  
- Wszystkie usługi WCF są obsługiwane z wyjątkiem programu CardSpace i hostingu w sieci Web.  
  
- Obsługiwane są zdalne kanały TCP/IP.  
  
- Usługa asmx (usługi sieci Web) nie jest obsługiwana.
