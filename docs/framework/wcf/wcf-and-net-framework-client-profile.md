# <a name="wcf-and-net-framework-client-profile"></a>WCF i .NET Framework Client Profile
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] to Uproszczona wersja pełnego środowiska .NET Framework, przeznaczona dla klientów, którzy nie potrzebują całej struktury framework. Nie wszystkie Windows Communication Foundation jest obsługiwana przez platformę klienta.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Funkcji WCF, które są obsługiwane przez .NET Framework Client Profile  
 Następujące funkcje programu Windows Communication Foundation są obsługiwane przez [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]:  
  
-   Obsługiwane są wszystkie WCF z wyjątkiem Cardspace i hosting sieci web.  
  
-   Kanałów komunikacji zdalnej TCP/IP są obsługiwane.  
  
-   Asmx (usługi sieci Web) nie są obsługiwane.
