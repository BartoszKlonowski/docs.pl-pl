# <a name="wcf-and-net-framework-client-profile"></a>WCF i .NET Framework Client Profile
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]jest to Uproszczona wersja programu .net pełne Framework przeznaczona dla klientów, którzy nie potrzebują całego framework. Nie wszystkie usługi Windows Communication Foundation jest obsługiwana przez platformę klienta.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Usługi WCF funkcje obsługiwane przez programu .net Framework Client Profile  
 Następujące funkcje systemu Windows Communication Foundation są obsługiwane przez [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]:  
  
-   Wszystkie usługi WCF jest obsługiwane z wyjątkiem Cardspace i hostingu sieci web.  
  
-   Kanałów komunikacji zdalnej TCP/IP są obsługiwane.  
  
-   Asmx (usługi sieci Web) nie są obsługiwane.
