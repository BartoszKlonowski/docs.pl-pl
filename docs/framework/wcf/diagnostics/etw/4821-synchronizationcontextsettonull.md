# <a name="4821---synchronizationcontextsettonull"></a>4821 — SynchronizationContextSetToNull
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4821|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, jeśli SynchronizationContext została ustawiona na wartość null przez DiscoveryClientChannel przed rozpoczęciem operacji wyszukiwania.  
  
## <a name="message"></a>Komunikat  
 SynchronizationContext została ustawiona na wartość null przez DiscoveryClientChannel przed rozpoczęciem operacji wyszukiwania.
