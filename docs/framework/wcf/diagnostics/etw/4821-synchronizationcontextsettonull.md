# <a name="4821---synchronizationcontextsettonull"></a>4821 — SynchronizationContextSetToNull

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4821|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy SynchronizationContext ma wartość null przez DiscoveryClientChannel przed zainicjowaniem operacji znajdowania.  
  
## <a name="message"></a>Wiadomość  

 SynchronizationContext została ustawiona na wartość null przez DiscoveryClientChannel przed zainicjowaniem operacji znajdowania.
