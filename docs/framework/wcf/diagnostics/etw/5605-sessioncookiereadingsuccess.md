# <a name="5605---sessioncookiereadingsuccess"></a>5605 — SessionCookieReadingSuccess
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|5605|  
|słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane po tokenu sesji odczytu z pliku cookie sesji, które zakończyło się pomyślnie.  
  
## <a name="message"></a>Komunikat  
 Odczytywanie tokenu sesji z pliku cookie sesji zakończyło się pomyślnie.
