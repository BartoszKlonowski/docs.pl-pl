# <a name="5605---sessioncookiereadingsuccess"></a>5605 — SessionCookieReadingSuccess

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|5605|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane w przypadku pomyślnego przeczytania tokenu sesji z pliku cookie sesji.  
  
## <a name="message"></a>Wiadomość  

 Odczytywanie tokenu sesji z pliku cookie sesji zakończyło się pomyślnie.
