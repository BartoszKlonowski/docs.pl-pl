# <a name="603---aspnetroutingservice"></a>603 — AspNetRoutingService

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|603|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy żądanie przychodzące pasuje do usługi WCF zdefiniowanej w marszrucie ASP.NET o adresie %1.  
  
## <a name="message"></a>Wiadomość  

 Żądanie przychodzące jest zgodne z usługą WCF zdefiniowaną w marszrucie ASP.NET o adresie %1.
