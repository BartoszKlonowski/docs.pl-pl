# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 — DiscoveryMessageWithInvalidReplyTo
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4807|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.  
  
## <a name="message"></a>Komunikat  
 Komunikat z żądaniem odnajdowania o atrybucie messageId = "%1" został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.
