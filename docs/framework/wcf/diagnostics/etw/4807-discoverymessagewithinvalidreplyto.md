# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 — DiscoveryMessageWithInvalidReplyTo
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4807|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, podczas odnajdywania komunikat został porzucony, ponieważ miał nieprawidłowy adres.  
  
## <a name="message"></a>Komunikat  
 Komunikat żądania odnajdywania z messageId = "%1" został porzucony, ponieważ miał nieprawidłowy adres.
