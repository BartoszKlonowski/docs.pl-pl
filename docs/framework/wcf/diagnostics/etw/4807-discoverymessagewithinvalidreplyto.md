# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 — DiscoveryMessageWithInvalidReplyTo

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4807|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy komunikat odnajdowania został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.  
  
## <a name="message"></a>Wiadomość  

 Komunikat z żądaniem odnajdowania o atrybucie messageId równym "%1" został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.
