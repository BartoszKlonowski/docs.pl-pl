# <a name="207---faultproviderinvoked"></a>207 — FaultProviderInvoked
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|207|  
|słowa kluczowe|Rozwiązywanie problemów, modelu ServiceModel|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane po `FaultProvider` zostało wywołane.  
  
## <a name="message"></a>Komunikat  
 Dyspozytor wywoływane FaultProvider typu "%1", z wyjątkiem typu "%2".

|ExceptionTypeName|`xs:string`|Imię i nazwisko CLR wyjątku, który `FaultProvider` działał na.|  
|HostReference|`xs:string`|W przypadku usług hostowanych w sieci Web to pole jednoznacznie identyfikuje usługę w hierarchii w sieci Web.
