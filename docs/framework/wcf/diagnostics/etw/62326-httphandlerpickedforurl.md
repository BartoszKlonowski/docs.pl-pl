# <a name="62326---httphandlerpickedforurl"></a>62326 — HttpHandlerPickedForUrl

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|62326|  
|Słowa kluczowe|WebHost|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy program obsługi protokołu HTTP zostanie pobrany do obsługi wszystkich żądań wysyłanych do tego adresu URL.  
  
## <a name="message"></a>Wiadomość  

 Adres URL ' %1 ' zawiera dokument XAML z typem elementu głównego ' %2 '. Typ procedury obsługi HTTP ' %3 ' jest wybrany do obsługi wszystkich żądań wysyłanych do tego adresu URL.
