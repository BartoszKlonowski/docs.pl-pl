# <a name="3351---servermaxpooledconnectionsquotareached"></a>3351 — ServerMaxPooledConnectionsQuotaReached

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3351|  
|Słowa kluczowe|limit przydziału|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytics|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy zostanie osiągnięty limit przydziału połączeń w puli serwera.  
  
## <a name="message"></a>Wiadomość  

 Osiągnięto przydział maksymalnej liczby połączeń obsługiwanych przez pule serwera.
