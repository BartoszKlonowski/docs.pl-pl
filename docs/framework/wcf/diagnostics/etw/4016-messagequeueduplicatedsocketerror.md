# <a name="4016---messagequeueduplicatedsocketerror"></a>4016 — MessageQueueDuplicatedSocketError

|||  
|-|-|  
|ID (Identyfikator)|4016|  
|Słowa kluczowe|ActivationServices|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytics|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy kolejka komunikatów nie może zduplikować gniazda.  
  
## <a name="message"></a>Wiadomość  

 Kolejka komunikatów nie może zduplikować gniazda.
