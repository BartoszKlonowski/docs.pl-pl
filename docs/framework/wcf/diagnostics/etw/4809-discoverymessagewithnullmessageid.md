# <a name="4809---discoverymessagewithnullmessageid"></a>4809 — DiscoveryMessageWithNullMessageId
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4809|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony, ponieważ nagłówek komunikatu nie zawierał wymaganej właściwości MessageId.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 został porzucony, ponieważ nagłówek komunikatu nie zawierał wymaganej właściwości MessageId.
