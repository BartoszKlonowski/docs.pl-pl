# <a name="4809---discoverymessagewithnullmessageid"></a>4809 — DiscoveryMessageWithNullMessageId
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4809|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, podczas odnajdywania komunikat został porzucony, ponieważ nagłówek wiadomości nie zawiera wymaganej właściwości MessageId.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 zostało porzucone, ponieważ nagłówek wiadomości nie zawiera wymaganej właściwości MessageId.
