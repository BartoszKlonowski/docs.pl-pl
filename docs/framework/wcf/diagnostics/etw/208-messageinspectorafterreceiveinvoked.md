# <a name="208---messageinspectorafterreceiveinvoked"></a>208 — MessageInspectorAfterReceiveInvoked
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|208|  
|słowa kluczowe|Rozwiązywanie problemów, modelu ServiceModel|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane po modelu usługi zostało wywołane `AfterReceive` metody w Inspektorze wiadomości.  
  
## <a name="message"></a>Komunikat  
 Dyspozytor wywoływane "AfterReceiveReply" na MessageInspector typu "%1".
