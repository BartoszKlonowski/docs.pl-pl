# <a name="4805---discoverymessagewithinvalidcontent"></a>4805 — DiscoveryMessageWithInvalidContent

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4805|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy komunikat odnajdywania zawiera nieprawidłową zawartość.  
  
## <a name="message"></a>Wiadomość  

 Komunikat %1 o atrybucie messageId równym "%2" został porzucony, ponieważ miał nieprawidłową zawartość.
