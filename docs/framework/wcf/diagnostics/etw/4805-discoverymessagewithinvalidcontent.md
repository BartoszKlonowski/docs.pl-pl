# <a name="4805---discoverymessagewithinvalidcontent"></a>4805 — DiscoveryMessageWithInvalidContent
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4805|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy komunikatu odnajdowania zawiera nieprawidłową zawartość.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 o atrybucie messageId = "%2" został porzucony, ponieważ miał nieprawidłową zawartość.
