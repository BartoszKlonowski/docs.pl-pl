# <a name="4810---discoverymessagewithnullmessagesequence"></a>4810 — DiscoveryMessageWithNullMessageSequence
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4810|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony przez obiekt DiscoveryClient, ponieważ nie miał właściwości DiscoveryMessageSequence.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 o atrybucie messageId = "%2" został porzucony przez obiekt DiscoveryClient, ponieważ nie miał właściwości DiscoveryMessageSequence.
