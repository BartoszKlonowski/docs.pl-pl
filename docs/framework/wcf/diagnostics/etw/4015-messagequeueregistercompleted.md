|Słowa kluczowe|ActivationServices|

## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po zakończeniu rejestracji kolejki komunikatów.

 Rejestracja kolejki komunikatów dla identyfikatora URI "%1" została zakończona.
