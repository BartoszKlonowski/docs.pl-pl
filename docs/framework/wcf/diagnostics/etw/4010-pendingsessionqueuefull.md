# <a name="4010---pendingsessionqueuefull"></a>4010 — PendingSessionQueueFull
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4010|  
|Słowa kluczowe|ActivationServices|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji Server aplikacje/analityczne|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy dystrybucja sesji nie powiodło się, ponieważ kolejka sesji oczekujących jest pełna.  
  
## <a name="message"></a>Komunikat  
 Dystrybucja sesji dla "%1" nie powiodło się, ponieważ kolejka sesji oczekujących jest pełna z "%2" oczekujących elementów.
