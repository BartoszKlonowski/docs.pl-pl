|||  
|-|-|  
|Identyfikator|4010|  
|słowa kluczowe|ActivationServices|  
|Poziom|Ostrzeżenie|

## <a name="description"></a>Opis  
 To zdarzenie jest emitowane podczas wysyłania sesji nie powiodło się, ponieważ kolejka oczekujących sesji jest pełna.  
  
## <a name="message"></a>Komunikat  
 Wysyłania sesji dla "%1" nie powiodło się, ponieważ oczekujących sesji kolejka jest pełna z "%2", oczekujące elementy.
