# <a name="4819---innerchannelopensucceeded"></a>4819 — InnerChannelOpenSucceeded
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4819|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy DiscoveryClientChannel pomyślnie odnaleźć punktu końcowego i otwarty kanał korzystania z niego.  
  
## <a name="message"></a>Komunikat  
 DiscoveryClientChannel pomyślnie odnaleźć punktu końcowego i otwarty kanał korzystania z niego. Klient jest połączony z usługą za pomocą EndpointAddress = "%1" i za pośrednictwem = "%2".
