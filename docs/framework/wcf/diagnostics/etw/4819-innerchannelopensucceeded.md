# <a name="4819---innerchannelopensucceeded"></a>4819 — InnerChannelOpenSucceeded

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4819|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po pomyślnym odnalezieniu punktu końcowego przez DiscoveryClientChannel i otwarciu kanału przy użyciu go.  
  
## <a name="message"></a>Wiadomość  

 DiscoveryClientChannel pomyślnie wykrył punkt końcowy i otworzył kanał przy jego użyciu. Klient jest połączony z usługą przy użyciu elementu EndpointAddress = ' %1 ' i za pośrednictwem = ' %2 '.
