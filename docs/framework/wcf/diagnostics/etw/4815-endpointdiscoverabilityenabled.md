# <a name="4815---endpointdiscoverabilityenabled"></a>4815 — EndpointDiscoverabilityEnabled
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4815|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, jeśli włączono odnajdywania punktu końcowego.  
  
## <a name="message"></a>Komunikat  
 Odnajdowanie punktu końcowego za pomocą EndpointAddress = "%1" i identyfikatorze ListenUri = "%2" został włączony.
