# <a name="4815---endpointdiscoverabilityenabled"></a>4815 — EndpointDiscoverabilityEnabled

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4815|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po włączeniu odnajdowania punktu końcowego.  
  
## <a name="message"></a>Wiadomość  

 Możliwość odnajdowania punktu końcowego z atrybutem EndpointAddress o wartości "%1" i ListenUri o wartości "%2" została włączona.
