# <a name="4815---endpointdiscoverabilityenabled"></a>4815 — EndpointDiscoverabilityEnabled
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4815|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany po włączeniu możliwość odnajdowania punktu końcowego.  
  
## <a name="message"></a>Komunikat  
 Możliwość odnajdowania punktu końcowego EndpointAddress o wartości "%1" i ListenUri o wartości = "%2" został włączony.
