# <a name="3408---httppipelinebeginprocessinboundrequeststart"></a>3408 — HttpPipelineBeginProcessInboundRequestStart

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3408|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy procedura obsługi komunikatów http rozpoczęła asynchroniczne przetwarzanie żądania przychodzącego.  
  
## <a name="message"></a>Wiadomość  

 Obsługa komunikatów http rozpoczęła asynchroniczne przetwarzanie żądania przychodzącego.
