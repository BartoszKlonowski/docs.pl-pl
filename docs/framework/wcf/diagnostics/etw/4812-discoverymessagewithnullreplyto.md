# <a name="4812---discoverymessagewithnullreplyto"></a>4812 — DiscoveryMessageWithNullReplyTo

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4812|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy komunikat odnajdowania został porzucony przez obiekt DiscoveryClient, ponieważ nie miał adresu ReplyTo.  
  
## <a name="message"></a>Wiadomość  

 Komunikat z żądaniem odnajdowania o atrybucie messageId równym "%1" został porzucony, ponieważ nie miał adresu ReplyTo.
