# <a name="4812---discoverymessagewithnullreplyto"></a>4812 — DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4812|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony przez obiekt DiscoveryClient, ponieważ nie miał adresu ReplyTo.  
  
## <a name="message"></a>Komunikat  
 Komunikat z żądaniem odnajdowania o atrybucie messageId = "%1" został porzucony, ponieważ nie miał adresu ReplyTo.
