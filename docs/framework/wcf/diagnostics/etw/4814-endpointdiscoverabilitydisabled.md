# <a name="4814---endpointdiscoverabilitydisabled"></a>4814 — EndpointDiscoverabilityDisabled

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4814|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy wykrywalność punktu końcowego została wyłączona.  
  
## <a name="message"></a>Wiadomość  

 Możliwość odnajdowania punktu końcowego z atrybutem EndpointAddress = ' %1 ' i ListenUri = ' %2 ' została wyłączona.
