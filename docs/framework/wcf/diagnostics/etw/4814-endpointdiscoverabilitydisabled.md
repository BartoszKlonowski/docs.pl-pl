# <a name="4814---endpointdiscoverabilitydisabled"></a>4814 — EndpointDiscoverabilityDisabled
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4814|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, jeśli wyłączono możliwość odnajdowania punktu końcowego.  
  
## <a name="message"></a>Komunikat  
 Możliwość odnajdowania punktu końcowego EndpointAddress o wartości "%1" i ListenUri o wartości = "%2" zostało wyłączone.
