# <a name="4814---endpointdiscoverabilitydisabled"></a>4814 — EndpointDiscoverabilityDisabled
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4814|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, jeśli wyłączono odnajdywania punktu końcowego.  
  
## <a name="message"></a>Komunikat  
 Odnajdowanie punktu końcowego za pomocą EndpointAddress = "%1" i identyfikatorze ListenUri = "%2" został wyłączony.
