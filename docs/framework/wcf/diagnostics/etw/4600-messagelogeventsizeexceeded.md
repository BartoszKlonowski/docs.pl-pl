# <a name="4600---messagelogeventsizeexceeded"></a>4600 — MessageLogEventSizeExceeded

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4600|  
|Słowa kluczowe|WCFMessageLogging|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy nie można zarejestrować komunikatu, ponieważ przekracza on rozmiar zdarzenia ETW.  
  
## <a name="message"></a>Wiadomość  

 Nie można zarejestrować komunikatu, ponieważ przekracza on rozmiar zdarzenia ETW
