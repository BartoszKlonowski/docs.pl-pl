|||  
|-|-|

|Słowa kluczowe|RoutingServices|  
|Poziom|Informacje|

## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy usługa routingu przesyła komunikat.

 Usługa routingu przesyła komunikat o identyfikatorze: "%1" [operacji %2], do "%3".
