# <a name="3385---windowsstreamsecurityonacceptupgrade"></a>3385 — WindowsStreamSecurityOnAcceptUpgrade
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|3385|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows aplikacji Server aplikacje/analityczne|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany na zabezpieczenia strumienia systemu Windows, akceptując uaktualnienie zabezpieczeń.  
  
## <a name="message"></a>Komunikat  
 Przesyłania zabezpieczeń przy akceptowaniu uaktualnienia systemu Windows.
