# <a name="3416---websocketconnectionrequestsendstop"></a>3416 — WebSocketConnectionRequestSendStop

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3416|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy żądanie połączenia WebSocket do wysłania zostało zatrzymane.  
  
## <a name="message"></a>Wiadomość  

 Wysłano żądanie połączenia gniazda WebSocketId: %1.
