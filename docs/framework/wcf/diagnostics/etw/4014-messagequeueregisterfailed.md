## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, jeżeli nie powiodła się rejestracja kolejki komunikatów.  
  
## <a name="message"></a>Komunikat  
 Rejestracja kolejki komunikatów dla identyfikatora uri: "%1" nie powiodła się ze stanem: "%2".
