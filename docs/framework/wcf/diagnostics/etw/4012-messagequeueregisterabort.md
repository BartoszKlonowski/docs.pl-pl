# <a name="4012---messagequeueregisterabort"></a>4012 — MessageQueueRegisterAbort

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4012|  
|Słowa kluczowe|ActivationServices|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytics|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy rejestracja kolejki komunikatów zostanie przerwana.  
  
## <a name="message"></a>Wiadomość  

 Rejestracja kolejki komunikatów została przerwana ze stanem "%1" dla identyfikatora URI: "%2".
