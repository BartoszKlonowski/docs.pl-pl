# <a name="4801---discoveryclientinclientchannelfailedtoclose"></a>4801 — DiscoveryClientInClientChannelFailedToClose
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4801|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy klasa DiscoveryClient utworzony wewnątrz DiscoveryClientChannel nie można zamknąć i dlatego została przerwana.  
  
## <a name="message"></a>Komunikat  
 Klasa DiscoveryClient utworzony wewnątrz DiscoveryClientChannel nie powiodło się zamknięcie i dlatego została przerwana.
