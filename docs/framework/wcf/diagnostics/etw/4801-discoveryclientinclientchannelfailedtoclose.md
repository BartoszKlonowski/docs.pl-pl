# <a name="4801---discoveryclientinclientchannelfailedtoclose"></a>4801 — DiscoveryClientInClientChannelFailedToClose

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4801|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy obiekt DiscoveryClient utworzone wewnątrz DiscoveryClientChannel nie zostało zamknięte i dlatego zostało przerwane.  
  
## <a name="message"></a>Wiadomość  

 Nie można zamknąć obiekt DiscoveryClient utworzonego wewnątrz DiscoveryClientChannel i w związku z tym został przerwany.
