|||  
|-|-|  
|ID (Identyfikator)|3395|  
|Słowa kluczowe|Kanał|  
|Poziom|Informacje|

## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy komunikat przesyłany strumieniowo został zapisany asynchronicznie przez koder.  
  
## <a name="message"></a>Wiadomość
