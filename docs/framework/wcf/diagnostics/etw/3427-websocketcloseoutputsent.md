# <a name="3427---websocketcloseoutputsent"></a>3427 — WebSocketCloseOutputSent

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3427|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy jest wysyłane wyjście z zamknięcia obiektu WebSocket.  
  
## <a name="message"></a>Wiadomość  

 Gniazdo WebSocketId: %1 wysyła komunikat zamknięcia wyjścia do "%2" ze stanem zamknięcia "%3".
