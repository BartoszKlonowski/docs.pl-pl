## <a name="enabling-tracing"></a><span data-ttu-id="66f96-108">Włączenie debugowania</span><span class="sxs-lookup"><span data-stu-id="66f96-108">Enabling Tracing</span></span>  
 <span data-ttu-id="66f96-109">Można włączyć śledzenia przepływu komunikatów przez ustawienie dla platformy .NET Framework 4 `messageFlowTracing` element konfiguracji do `true`, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="66f96-109">You can enable message flow tracing by setting the .NET Framework 4 `messageFlowTracing` configuration element to `true`, as shown in the following example.</span></span>  
  
```xml  
<system.servicemodel>  
  <diagnostics>  
    <endToEndTracing propagateActivity="true" messageFlowTracing="true" />  
  </diagnostics>  
</system.servicemodel>  
```

 <span data-ttu-id="66f96-112">Działania są skorelowane przez wymianę identyfikatora identyfikator działania.</span><span class="sxs-lookup"><span data-stu-id="66f96-112">Activities are correlated by the interchange of an identifier called the activity ID.</span></span> <span data-ttu-id="66f96-113">Ten identyfikator jest identyfikatorem GUID i jest generowany przez klasę System.Diagnostics.CorrelationManager.</span><span class="sxs-lookup"><span data-stu-id="66f96-113">This identifier is a GUID, and is generated by the System.Diagnostics.CorrelationManager class.</span></span> <span data-ttu-id="66f96-114">Manipulowanie System.Diagnostics.Trace.CorrelationManager.ActivityID, upewnij się, że ma wartość z oryginalną przesyłania Kontrola wykonywania przywracając WCF kodu.</span><span class="sxs-lookup"><span data-stu-id="66f96-114">If you manipulate System.Diagnostics.Trace.CorrelationManager.ActivityID, ensure that the value is set to original when execution control transfers back to WCF code.</span></span>  <span data-ttu-id="66f96-115">Ponadto jeśli używasz modelu programowania asynchronicznego WCF upewnij się, że System.Diagnostics.Trace.CorrelationManager.ActivityID są przesyłane między wątkami.</span><span class="sxs-lookup"><span data-stu-id="66f96-115">Also, if you use an asynchronous WCF programming model ensure that System.Diagnostics.Trace.CorrelationManager.ActivityID is transferred between the threads.</span></span>
