# <a name="203---clientparameterinspectoraftercallinvoked"></a>203 — ClientParameterInspectorAfterCallInvoked

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|203|  
|Słowa kluczowe|Rozwiązywanie problemów, ServiceModel|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Analytics|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po wywołaniu metody przez model usługi `AfterCall` w Inspektorze parametrów klienta.  
  
## <a name="message"></a>Wiadomość  

 Dyspozytor wywołał element "AfterCall" na ClientParameterInspector typu "%1".
