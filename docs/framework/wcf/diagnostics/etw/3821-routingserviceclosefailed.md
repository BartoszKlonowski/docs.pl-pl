# <a name="3821---routingserviceclosefailed"></a>3821 — RoutingServiceCloseFailed

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3821|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu zakończyła się niepowodzeniem podczas zamykania klienta wychodzącego.  
  
## <a name="message"></a>Wiadomość  

 Wystąpił błąd usługi routingu podczas zamykania klienta wychodzącego ' %1 '.
