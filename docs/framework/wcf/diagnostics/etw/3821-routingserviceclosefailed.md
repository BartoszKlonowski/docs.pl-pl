# <a name="3821---routingserviceclosefailed"></a>3821 — RoutingServiceCloseFailed
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|3821|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy usługa routingu nie powiodło się podczas zamykania klienta w ruchu wychodzącym.  
  
## <a name="message"></a>Komunikat  
 Usługa routingu wystąpił błąd podczas zamykania klienta w ruchu wychodzącym '%1'.
