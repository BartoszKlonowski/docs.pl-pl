# <a name="3805---routingservicedisplayconfig"></a>3805 — RoutingServiceDisplayConfig

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3805|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu jest skonfigurowana z RouteOnHeadersOnly.  
  
## <a name="message"></a>Wiadomość  

 Usługa routingu jest skonfigurowana z RouteOnHeadersOnly: %1, SoapProcessingEnabled: %2, EnsureOrderedDispatch: %3.
