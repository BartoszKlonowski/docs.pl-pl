# <a name="4811---discoverymessagewithnullrelatesto"></a>4811 — DiscoveryMessageWithNullRelatesTo
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4811|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony przez obiekt DiscoveryClient, ponieważ nagłówek komunikatu nie zawierał wymaganej właściwości RelatesTo.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 o atrybucie messageId = "%2" został porzucony przez obiekt DiscoveryClient, ponieważ nagłówek komunikatu nie zawierał wymaganej właściwości RelatesTo.
