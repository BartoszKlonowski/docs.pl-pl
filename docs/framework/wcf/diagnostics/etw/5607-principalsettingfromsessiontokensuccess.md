# <a name="5607---principalsettingfromsessiontokensuccess"></a>5607 — PrincipalSettingFromSessionTokenSuccess

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|5607|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy ustawienie podmiotu zabezpieczeń na podstawie tokenu sesji zakończyło się pomyślnie.  
  
## <a name="message"></a>Wiadomość  

 Ustawienie podmiotu zabezpieczeń na podstawie tokenu sesji zakończyło się pomyślnie.
