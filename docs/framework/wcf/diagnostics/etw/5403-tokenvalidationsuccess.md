# <a name="5403---tokenvalidationsuccess"></a>5403 — TokenValidationSuccess

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|5403|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy Walidacja elementu SecurityToken zakończyła się pomyślnie.  
  
## <a name="message"></a>Wiadomość  

 Walidacja elementu SecurityToken (typ ' %1 ' i identyfikator ' %2 ') zakończyła się pomyślnie.
