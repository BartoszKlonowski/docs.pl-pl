# <a name="3803---routingserviceprocessingfailure"></a>3803 — RoutingServiceProcessingFailure
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|3803|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy usługa routingu nie powiodło się podczas przetwarzania komunikatu w punkcie końcowym.  
  
## <a name="message"></a>Komunikat  
 Usługa routingu nie powiodło się podczas przetwarzania komunikatu w punkcie końcowym o adresie "%1".
