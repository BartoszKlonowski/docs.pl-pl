## <a name="description"></a>Opis  
 To zdarzenie jest emitowany do zgłaszania liczba fabryki kanałów podstawowej klienta pamięci podręcznej.  
  
## <a name="message"></a>Komunikat  
 Liczba fabryki kanałów pamięci podręcznej wynosi: "%1".  Co najwyżej fabryki kanałów "%2" mogą być buforowane.
