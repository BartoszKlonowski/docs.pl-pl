# <a name="3429---websocketclosestatusreceived"></a>3429 — WebSocketCloseStatusReceived

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3429|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po odebraniu stanu zamknięcia obiektu WebSocket.  
  
## <a name="message"></a>Wiadomość  

 Gniazdo WebSocketId: %1 Odebrano komunikat zamknięcia połączenia ze stanem "%2".
