# <a name="4804---discoverymessagereceivedafteroperationcompleted"></a>4804 — DiscoveryMessageReceivedAfterOperationCompleted

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4804|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po odebraniu komunikatu odnajdowania po zakończeniu operacji.  
  
## <a name="message"></a>Wiadomość  

 Komunikat %1 o atrybucie messageId równym "%2" został porzucony przez obiekt DiscoveryClient, ponieważ operacja %3 została ukończona.
