# <a name="3825---routingserviceabandoningreceivecontext"></a>3825 — RoutingServiceAbandoningReceiveContext

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3825|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu wywołuje metodę ReceiveContext. Abandon dla komunikatu o określonym IDENTYFIKATORze.  
  
## <a name="message"></a>Wiadomość  

 Usługa routingu wywołuje metodę ReceiveContext. Abandon dla komunikatu o IDENTYFIKATORze: %1.
