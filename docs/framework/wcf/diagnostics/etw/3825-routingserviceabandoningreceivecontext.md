# <a name="3825---routingserviceabandoningreceivecontext"></a>3825 — RoutingServiceAbandoningReceiveContext
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|3825|  
|słowa kluczowe|RoutingServices|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy usługa routingu wywołuje ReceiveContext.Abandon wiadomości o określonym identyfikatorze.  
  
## <a name="message"></a>Komunikat  
 Usługa routingu wywołuje ReceiveContext.Abandon komunikatu o identyfikatorze: "%1".
