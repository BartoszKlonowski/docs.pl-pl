# <a name="3413---httppipelinebeginprocessresponsestart"></a>3413 — HttpPipelineBeginProcessResponseStart

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3413|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy program obsługi komunikatów http rozpoczął przetwarzanie odpowiedzi asynchronicznie.  
  
## <a name="message"></a>Wiadomość  

 Obsługa komunikatów http rozpoczęła asynchroniczne przetwarzanie odpowiedzi.
