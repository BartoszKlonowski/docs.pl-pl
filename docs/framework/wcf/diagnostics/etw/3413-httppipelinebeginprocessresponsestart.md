# <a name="3413---httppipelinebeginprocessresponsestart"></a>3413 — HttpPipelineBeginProcessResponseStart
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|3413|  
|słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, jeśli program obsługi komunikatów Http rozpoczęła się asynchronicznie przetwarzania odpowiedzi.  
  
## <a name="message"></a>Komunikat  
 Program obsługi komunikatów HTTP do przetwarzania odpowiedzi asynchronicznie.
