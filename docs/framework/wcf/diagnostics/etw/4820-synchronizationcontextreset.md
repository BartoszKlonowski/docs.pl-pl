# <a name="4820---synchronizationcontextreset"></a>4820 — SynchronizationContextReset
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4820|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane po SynchronizationContext zostało zresetowane do wartości oryginalnej przez DiscoveryClientChannel.  
  
## <a name="message"></a>Komunikat  
 SynchronizationContext została zresetowana do wartości oryginalnej %1 DiscoveryClientChannel.
