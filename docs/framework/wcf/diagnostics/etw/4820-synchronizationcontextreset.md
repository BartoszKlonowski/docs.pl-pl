# <a name="4820---synchronizationcontextreset"></a>4820 — SynchronizationContextReset

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4820|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane po zresetowaniu SynchronizationContext do jego oryginalnej wartości przez DiscoveryClientChannel.  
  
## <a name="message"></a>Wiadomość  

 SynchronizationContext został zresetowany do oryginalnej wartości %1 przez DiscoveryClientChannel.
