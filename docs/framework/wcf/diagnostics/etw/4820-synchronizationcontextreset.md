# <a name="4820---synchronizationcontextreset"></a>4820 — SynchronizationContextReset
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4820|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy zostało zresetowane obiektu SynchronizationContext to oryginalnej wartości przez obiekt DiscoveryClientChannel.  
  
## <a name="message"></a>Komunikat  
 Obiekt DiscoveryClientChannel zresetowano obiektu SynchronizationContext to oryginalnej wartości %1.
