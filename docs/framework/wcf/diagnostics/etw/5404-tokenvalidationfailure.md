# <a name="5404---tokenvalidationfailure"></a>5404 — TokenValidationFailure
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|5404|  
|słowa kluczowe|Zabezpieczenia|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy weryfikacja SecurityToken nie powiodła się.  
  
## <a name="message"></a>Komunikat  
 Nie można zweryfikować SecurityToken (typ "%1" i identyfikatorze "%2"). %3
