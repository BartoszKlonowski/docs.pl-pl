# <a name="5404---tokenvalidationfailure"></a>5404 — TokenValidationFailure

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|5404|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy Walidacja elementu SecurityToken nie powiodła się.  
  
## <a name="message"></a>Wiadomość  

 Walidacja elementu SecurityToken (typ ' %1 ' i identyfikator ' %2 ') nie powiodła się. %3
