# <a name="4808---discoverymessagewithnocontent"></a>4808 — DiscoveryMessageWithNoContent
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4808|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany podczas odnajdywania komunikat został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.  
  
## <a name="message"></a>Komunikat  
 Komunikat %1 został porzucony, ponieważ nie ma żadnej zawartości.
