# <a name="4808---discoverymessagewithnocontent"></a>4808 — DiscoveryMessageWithNoContent

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4808|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy komunikat odnajdowania został porzucony, ponieważ miał nieprawidłowy adres ReplyTo.  
  
## <a name="message"></a>Wiadomość  

 Komunikat %1 został porzucony, ponieważ nie miał zawartości.
