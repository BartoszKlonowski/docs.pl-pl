# <a name="3430---websocketuseversionfromclientwebsocketfactory"></a>3430 — WebSocketUseVersionFromClientWebSocketFactory

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3430|  
|Słowa kluczowe|HTTP|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane podczas korzystania z wartości WebSocketVersion z fabryki protokołu WebSocket klienta.  
  
## <a name="message"></a>Wiadomość  

 Używanie wartości WebSocketVersion z fabryki protokołu WebSocket klienta typu ' %1 '.
