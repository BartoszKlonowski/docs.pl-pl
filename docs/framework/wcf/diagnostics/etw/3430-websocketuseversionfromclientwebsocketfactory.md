|-|-|

|Słowa kluczowe|HTTP|

## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy używanie wartości WebSocketVersion z fabryki obiektów WebSocket klienta.

 Używanie wartości WebSocketVersion z fabryki obiektów WebSocket klienta typu "%1".
