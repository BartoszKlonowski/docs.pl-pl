## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3818|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu napotkała wyjątek dwustronnego wywołania zwrotnego.  
  
## <a name="message"></a>Wiadomość  

 Składnik usługi routingu %1 napotkał wyjątek dwustronnego wywołania zwrotnego.
