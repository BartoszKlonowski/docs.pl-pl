# <a name="3820---routingservicecreatingtransaction"></a>3820 — RoutingServiceCreatingTransaction

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3820|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu utworzyła nową transakcję w celu przetworzenia komunikatu.  
  
## <a name="message"></a>Wiadomość  

 Usługa routingu utworzyła nową transakcję o identyfikatorze ' %1 ' na potrzeby przetwarzania komunikatów.
