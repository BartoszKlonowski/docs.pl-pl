# <a name="3820---routingservicecreatingtransaction"></a>3820 — RoutingServiceCreatingTransaction
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|3820|  
|słowa kluczowe|RoutingServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy usługa routingu utworzył nową transakcję do przetwarzania wiadomości.  
  
## <a name="message"></a>Komunikat  
 Usługa routingu utworzyć nową transakcję o identyfikatorze "%1" do przetwarzania komunikatów.
