# <a name="5606---principalsettingfromsessiontokenstarted"></a>5606 — PrincipalSettingFromSessionTokenStarted
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|5606|  
|Słowa kluczowe|Zabezpieczenia|  
|Poziom|Pełny|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowany przy uruchamianiu ustawienie podmiotu zabezpieczeń z tokenu sesji.  
  
## <a name="message"></a>Komunikat  
 Ustawianie podmiotu zabezpieczeń na podstawie tokenu sesji uruchomiona.
