# <a name="3819---routingservicemovedtobackup"></a>3819 — RoutingServiceMovedToBackup
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|3819|  
|słowa kluczowe|RoutingServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy usługa routingu została przeniesiona do tworzenia kopii zapasowych punktów końcowych.  
  
## <a name="message"></a>Komunikat  
 Routing wiadomości usługi o identyfikatorze: "%1" [operacji %2] przeniesione do endpoint kopii zapasowych "%3".
