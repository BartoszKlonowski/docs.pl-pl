|||  
|-|-|

|Słowa kluczowe|RoutingServices|  
|Poziom|Informacje|

## <a name="description"></a>Opis  
 To zdarzenie jest emitowany, gdy usługa routingu została przeniesiona do tworzenia kopii zapasowej punktów końcowych.  
  
## <a name="message"></a>Komunikat  
 Komunikat usługi routingu o identyfikatorze: "%1" [operacji %2] przeniesiony do zapasowego punktu końcowego "%3".
