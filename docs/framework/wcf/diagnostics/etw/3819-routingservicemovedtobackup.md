# <a name="3819---routingservicemovedtobackup"></a>3819 — RoutingServiceMovedToBackup

## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|3819|  
|Słowa kluczowe|RoutingServices|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows-Application Server — aplikacje/debugowanie|  
  
## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy usługa routingu została przeniesiona do punktów końcowych kopii zapasowej.  
  
## <a name="message"></a>Wiadomość  

 Komunikat usługi routingu o IDENTYFIKATORze: "%1" [operacja %2] został przeniesiony do kopii zapasowej punktu końcowego "%3".
