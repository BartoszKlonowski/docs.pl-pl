## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID (Identyfikator)|4813|  
|Słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|

## <a name="description"></a>Opis  

 To zdarzenie jest emitowane, gdy wiadomość została porzucona, ponieważ była duplikatem.  
  
## <a name="message"></a>Wiadomość  

 Komunikat %1 o atrybucie messageId równym "%2" został porzucony, ponieważ był duplikatem.
