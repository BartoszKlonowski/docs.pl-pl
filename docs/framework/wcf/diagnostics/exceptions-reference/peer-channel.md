# <a name="peer-channel"></a>Kanał elementu równorzędnego

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez kanał równorzędny programu Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|InsufficientCredentials|Podane poświadczenia nie są wystarczające, aby można było wykonać żądaną operację. Określ prawidłową wartość dla określonej operacji|  
|InvalidResolverMode|Określona wartość PeerResolverMode jest nieprawidłowa. Określ wartość PeerResolveMode. Auto, Default lub PNRP.|  
|PeerNodeAborted|Nie można otworzyć węzła równorzędnego, ponieważ został przerwany.|  
|PeerNullRegistrationInfo|Informacje o rejestracji nie mogą mieć wartości null. Upewnij się, że operacja Register została wywołana z prawidłowym obiektem RegistrationInfo.|  
|PeerNullResolveInfo|Nie można rozpoznać informacji `null` . Upewnij się, że operacja rozpoznawania jest wywołana z prawidłowym obiektem ResolveInfo.|
