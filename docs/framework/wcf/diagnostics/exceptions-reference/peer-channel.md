# <a name="peer-channel"></a>Kanał elementu równorzędnego
W tym temacie wymieniono wszystkie wyjątki generowane przez [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] kanał elementu równorzędnego.  
  
## <a name="exception-list"></a>Listy wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|InsufficientCredentials|Podane poświadczenia są niewystarczające do wykonania żądanej operacji. Określ prawidłową wartość dla określonej operacji|  
|InvalidResolverMode|Określona wartość PeerResolverMode jest nieprawidłowa. Określ wartość PeerResolveMode.Auto, Default lub PNRP.|  
|PeerNodeAborted|Nie można otworzyć węzła PeerNode, ponieważ operacja została przerwana.|  
|PeerNullRegistrationInfo|Informacje o rejestracji nie może mieć wartości null. Upewnij się, że operacja rejestracji została wywołana przy użyciu prawidłowego obiektu RegistrationInfo.|  
|PeerNullResolveInfo|Rozwiąż informacji nie może być `null`. Upewnij się, że operacja rozpoznawania została wywołana przy użyciu prawidłowego ResolveInfo.|
