# <a name="peer-channel"></a>Kanał elementu równorzędnego
Ten temat zawiera listę wszystkich wyjątków generowanych przez kanał elementu równorzędnego Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|InsufficientCredentials|Podane poświadczenia są niewystarczające do wykonania żądanej operacji. Określ prawidłową wartość dla określonej operacji|  
|InvalidResolverMode|Określona wartość PeerResolverMode jest nieprawidłowa. Określ wartość PeerResolveMode.Auto, domyślnymi lub PNRP.|  
|PeerNodeAborted|Nie można otworzyć węzeł równorzędny, ponieważ operacja została przerwana.|  
|PeerNullRegistrationInfo|Informacje o rejestracji nie może mieć wartości null. Upewnij się, że operacja rejestracji została wywołana z prawidłowym obiektem RegistrationInfo.|  
|PeerNullResolveInfo|Rozwiązania informacji nie może być `null`. Upewnij się, że operacja rozpoznawania została wywołana przy użyciu prawidłowego ResolveInfo.|
