# <a name="core-communications-internal-duplex-transport-channels"></a>Komunikacja podstawowa: Wewnętrzne dwukierunkowe kanały transportu

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez wewnętrzne kanały transportowe Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Aby wysłać wiadomość w kanale kompozytowym serwera, komunikat musi mieć ustawioną właściwość "Via" lub "do".|
