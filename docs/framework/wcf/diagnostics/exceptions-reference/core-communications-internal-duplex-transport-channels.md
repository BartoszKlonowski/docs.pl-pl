## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Do wysyłania wiadomości w kanałach dwukierunkowego złożonej z serwera, wiadomości musi mieć właściwość "Przez" lub nagłówku "Do" Ustaw.|
