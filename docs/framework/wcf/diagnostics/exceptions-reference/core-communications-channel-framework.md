# <a name="core-communications-channel-framework"></a>Podstawowe wiadomości: Struktura kanału
W tym temacie wymieniono wszystkie wyjątki generowane przez platformę kanał Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Listy wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|ReceiveRequestTimedOut|Odebrano żądanie dla określonego adresu lokalnego został przekroczony po określonym czasie. Czas przydzielony na tę operację mógł stanowi część większego limitu czasu.|  
|ReceiveTimedOutNoLocalAddress|Operacja odbioru został przekroczony po określonym czasie. Czas przydzielony na tę operację mógł stanowi część większego limitu czasu.|  
|UnableToDemuxChannel|Kanał nie jest dostępny zaakceptować komunikat z określonej akcji.|
