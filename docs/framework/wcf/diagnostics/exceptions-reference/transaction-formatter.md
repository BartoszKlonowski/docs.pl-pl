# <a name="transaction-formatter"></a>Element formatujący transakcji
Ten temat zawiera listę wszystkich wyjątków generowanych przez element formatujący transakcji.  
  
## <a name="exception"></a>Wyjątek  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|TMCommunicationError|Wystąpił błąd podczas komunikowania się z menedżerem transakcji rozproszonych.|  
|WhereaboutsNoHostName|Informacje o lokalizacji usługi MSDTC nie zawiera nazwę hosta.|  
|WhereaboutsReadFailed|Nie można zdeserializować informacji o lokalizacji usługi MSDTC.|  
|WhereaboutsSignatureMissing|Informacje o lokalizacji usługi MSDTC brakuje podpisu standardowe miejsce.|  
|WsatUriCreationFailed|Nie można utworzyć adresu usługi rejestracji z informacji o lokalizacji usługi MSDTC.|
