# <a name="transaction-formatter"></a>Element formatujący transakcji

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez program formatujący transakcji.  
  
## <a name="exception"></a>Wyjątek  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|TMCommunicationError|Wystąpił błąd podczas komunikacji z menedżerem transakcji rozproszonych.|  
|WhereaboutsNoHostName|Informacje o lokalizacji usługi MSDTC nie zawierały nazwy hosta.|  
|WhereaboutsReadFailed|Nie można deserializować informacji o lokalizacji usługi MSDTC.|  
|WhereaboutsSignatureMissing|Brak sygnatury lokalizacji standardowej w informacjach o lokalizacji usługi MSDTC.|  
|WsatUriCreationFailed|Nie można utworzyć adresu usługi rejestracji z informacji o lokalizacji MSDTC.|
