# <a name="transaction-formatter"></a>Element formatujący transakcji
Ten temat zawiera listę wszystkich wyjątków generowanych przez element formatujący transakcji.  
  
## <a name="exception"></a>Wyjątek  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|TMCommunicationError|Wystąpił błąd podczas komunikowania się z menedżerem transakcji rozproszonej.|  
|WhereaboutsNoHostName|Informacje o lokalizacji usługi MSDTC nie zawiera nazwy hosta.|  
|WhereaboutsReadFailed|Nie można deserializować informacji o lokalizacji usługi MSDTC.|  
|WhereaboutsSignatureMissing|Podpis lokalizacji brakowało informacji o lokalizacji usługi MSDTC.|  
|WsatUriCreationFailed|Nie można utworzyć adresu usługi rejestracji z informacji o lokalizacji usługi MSDTC.|
