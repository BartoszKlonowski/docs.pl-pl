# <a name="com-integration"></a>Integracja modelu COM+

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez integrację modelu COM+.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|Nie można rozpoznać typu określonego parametru w określonym obszarze nazw.|  
|NoEqualSignFound|Określone słowo kluczowe nie ma znaku równości po nim. Upewnij się, że każde słowo kluczowe jest poprzedzone znakiem równości i wartością.|  
|UdtNotFoundInAssembly|Nie można znaleźć określonego typu zdefiniowanego przez użytkownika. Upewnij się, że poprawny typ i biblioteka typów są zarejestrowane i określone.|
