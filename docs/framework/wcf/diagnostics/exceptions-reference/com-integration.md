# <a name="com-integration"></a>Integracja modelu COM+
W tym temacie wymieniono wszystkie wyjątki generowane przez integracja modelu COM +.  
  
## <a name="exception-list"></a>Listy wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|Nie można rozpoznać typu parametru określonego w ramach określonego obszaru nazw.|  
|NoEqualSignFound|Określone słowa kluczowe ma nie umieszczono znaku równości. Upewnij się, że każde słowo kluczowe następuje znak równości i wartość.|  
|UdtNotFoundInAssembly|Nie można odnaleźć określonego typu zdefiniowane przez użytkownika. Upewnij się, że prawidłowy typ oraz bibliotekę typów zarejestrowano i określono.|
