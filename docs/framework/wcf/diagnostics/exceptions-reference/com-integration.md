# <a name="com-integration"></a>Integracja modelu COM+
Ten temat zawiera listę wszystkich wyjątków generowanych przez integracja modelu COM +.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|Nie można rozpoznać typu dla określonego parametru w ramach określonego obszaru nazw.|  
|NoEqualSignFound|Określonego słowa kluczowego ma nie znaku równości. Upewnij się, że każde słowo kluczowe następuje znak równości i wartość.|  
|UdtNotFoundInAssembly|Nie można odnaleźć określonego typu zdefiniowanego przez użytkownika. Upewnij się, że poprawnego typu i biblioteki typów są zarejestrowane i określony.|
