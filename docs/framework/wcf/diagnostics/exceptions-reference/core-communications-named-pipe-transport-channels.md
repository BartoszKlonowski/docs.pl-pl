# <a name="core-communications-named-pipe-transport-channels"></a>Komunikacja podstawowa: Kanały transportu nazwanego potoku

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez kanały transportu nazwanego potoku.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Nie można zamknąć potoku podczas oczekiwania na operację zapisu w potoku.|  
|PipeReadPending|Operacja odczytu jest w toku dla potoku.|  
|PipeShutdownReadError|Operacja odczytu wskaźnika "Shutdown" potoku nie powiodła się.|  
|PipeShutdownWriteError|Nie można wykonać operacji zapisu wskaźnika "Shutdown" potoku.|  
|PipeWritePending|Operacja zapisu jest w toku dla potoku.|
