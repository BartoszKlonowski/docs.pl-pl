# <a name="core-communications-named-pipe-transport-channels"></a>Podstawowe wiadomości: Kanały transportu nazwanego potoku
W tym temacie wymieniono wszystkie wyjątki generowane przez kanały transportu nazwanego potoku.  
  
## <a name="exception-list"></a>Listy wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Nie można zamknąć potoku, gdy oczekuje operacji zapisu do potoku.|  
|PipeReadPending|Trwa operacja odczytu dla potoku.|  
|PipeShutdownReadError|Operacja odczytu z potoku sygnału "zamknięcia" nie powiodło się.|  
|PipeShutdownWriteError|Operacja zapisu wskaźnika "Zamknij" potoku nie powiodło się.|  
|PipeWritePending|Trwa operacja zapisu do potoku.|
