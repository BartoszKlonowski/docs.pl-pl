# <a name="core-communications-named-pipe-transport-channels"></a>Komunikacja podstawowa: Kanały transportu nazwanego potoku
Ten temat zawiera listę wszystkich wyjątków generowanych przez kanały transportu nazwanego potoku.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Nie można zamknąć potoku, gdy operacja zapisu do potoku jest w stanie oczekiwania.|  
|PipeReadPending|Trwa operacja odczytu dla potoku.|  
|PipeShutdownReadError|Operacja odczytu wskaźnika "Zamknij" w potoku nie powiodło się.|  
|PipeShutdownWriteError|Operacja zapisu wskaźnika "Zamknij" w potoku nie powiodło się.|  
|PipeWritePending|Operacja zapisu jest w toku dla potoku.|
