# <a name="core-communications-tcp-transport-channels"></a>Komunikacja podstawowa: Kanały transportu TCP
Ten temat zawiera listę wszystkich wyjątków generowanych przez kanały transportu TCP.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|Zdalny punkt końcowy gniazda określonego nie odpowiedział na żądanie zamknięcia w ramach określonego przydzielonego limitu czasu. Istnieje możliwość, że zdalny punkt końcowy jest nie wywołuje metody Close po odebraniu sygnału EOF (null) z operacji odbierania. Czas przydzielony na tę operację mógł stanowić część większego limitu czasu.|
