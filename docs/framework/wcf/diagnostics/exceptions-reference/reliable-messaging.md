# <a name="reliable-messaging"></a>Niezawodna obsługa komunikatów

W tym temacie wymieniono wszystkie wyjątki wygenerowane przez niezawodne komunikaty.  
  
## <a name="exception-list"></a>Lista wyjątków  
  
|Kod zasobu|Ciąg zasobu|  
|-------------------|---------------------|  
|InvalidBufferRemaining|Bufor odebrał komunikat o określonym rozmiarze. Bufor, który pozostanie niezmieniony, nie może być mniejszy od zera.|
