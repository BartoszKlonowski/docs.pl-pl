# <a name="deploying-services"></a>Wdrażanie usług
W tym temacie opisano, jak można wdrożyć aplikację systemu Windows Communication Foundation (WCF) w środowisku czasu wykonywania.  
  
## <a name="choosing-the-hosting-environment-for-your-application"></a>Wybieranie środowiska macierzystego dla aplikacji  
 Usługi WCF zostały zaprojektowane do uruchamiania w dowolnym procesie systemu Windows obsługuje kodu zarządzanego. Staje się aktywny, usługa musi być hostowany w środowisku czasu wykonywania, która tworzy go i kontroluje jego kontekstu i okresem istnienia. Hosting zakres opcje uruchamiania wewnątrz najprostszym aplikacji konsoli do środowiska serwera, takich jak usługi systemu Windows, Internet Information Services (IIS), lub w ramach procesu roboczego zarządzane przez usługę aktywacji systemu Windows (WAS). Aby zapoznać się z różnych opcji hostingu aplikacji WCF, zobacz [Hosting usług](../../../../docs/framework/wcf/hosting-services.md).
