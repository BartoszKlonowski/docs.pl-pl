# <a name="message-logging"></a>Rejestrowanie komunikatów

Windows Communication Foundation (WCF) zapewnia możliwość rejestrowania przychodzących i wychodzących komunikatów do użycia w trybie offline. Rejestrowanie komunikatów pozwala zobaczyć, jak wygląda komunikat i treść wiadomości. Ten typ rejestrowania jest szczególnie przydatny do informowania, jakie argumenty zostały przekazane i w jaki sposób punkt końcowy punktu końcowego wykrywa argumenty wyrażone jako XML. Ponadto rejestrowanie wiadomości w miarę odebrania umożliwia zdiagnozowanie źle sformułowanych komunikatów, a także sprawdzenie, jak dotarł komunikat. Można również przeanalizować tokeny zabezpieczające używane, części zaszyfrowane i podpisane oraz częściowo pozostawione.
