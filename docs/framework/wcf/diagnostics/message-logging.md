# <a name="message-logging"></a>Rejestrowanie komunikatów
Windows Communication Foundation (WCF) zapewnia możliwość rejestrowania komunikatów przychodzących i wychodzących do użycia w trybie offline. Rejestrowanie komunikatów temu można zobaczyć, jakie komunikatów i komunikat o treści wygląda. Tego rodzaju rejestrowanie jest szczególnie przydatne w informacją o tym, co przekazano argumentów i jak odbieranie punktu końcowego usłyszeć argumenty wyrażonej w postaci XML. Oprócz rejestrowania komunikatu, ponieważ zostało przesłane umożliwia diagnozowanie źle sformułowane komunikaty, a także aby zobaczyć, jak odebrania wiadomości. Można również sprawdzić zabezpieczeń tokeny używane, części zaszyfrowana i podpisana i części niezmienione.
