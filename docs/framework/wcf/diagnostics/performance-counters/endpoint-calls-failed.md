# <a name="endpoint-calls-failed"></a>Punkt końcowy: Wywołania zakończone niepowodzeniem

Nazwa licznika: wywołania zakończone niepowodzeniem.  
  
## <a name="description"></a>Opis  

 Liczba wywołań z nieobsługiwanymi wyjątkami w tym punkcie końcowym. Licznik jest zwiększany za każdym razem, gdy w tym punkcie końcowym występuje nieobsługiwany wyjątek.
