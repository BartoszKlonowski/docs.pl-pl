# <a name="call-duration"></a>Czas trwania wywołania
Nazwa licznika: Czas trwania wywołania  
  
## <a name="description"></a>Opis  
 Średni czas trwania wywołań tej operacji. Średni czas trwania jest obliczany dla równanie: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Dla usługi WCF asynchronicznego liczników Wywołaj czas trwania zawsze zwraca wartość -1.
