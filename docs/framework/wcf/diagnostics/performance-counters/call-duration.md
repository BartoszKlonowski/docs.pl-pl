# <a name="call-duration"></a>Czas trwania wywołania
Nazwa licznika: Czas trwania wywołania  
  
## <a name="description"></a>Opis  
 Średni czas trwania wywołań tej operacji. Średni czas trwania jest obliczana oparte na równanie: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Stosowania na asynchroniczne usługi WCF licznika Czas wywołania zawsze zwraca wartość -1.
