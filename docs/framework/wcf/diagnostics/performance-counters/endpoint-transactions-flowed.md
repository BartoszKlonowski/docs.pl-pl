# <a name="endpoint-transactions-flowed"></a>Punkt końcowy: Przekazane transakcje
Nazwa licznika: Przekazane transakcje.  
  
## <a name="description"></a>Opis  
 Liczba transakcji skierowana do operacji w tym punkcie końcowym. Ten licznik jest zwiększany za każdym razem jest obecny w komunikatu wysłanego do punktu końcowego zawiera identyfikator transakcji.
