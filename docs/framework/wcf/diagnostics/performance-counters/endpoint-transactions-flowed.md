# <a name="endpoint-transactions-flowed"></a>Punkt końcowy: Przepływ transakcji
Nazwa komputera: Przepływ transakcji.  
  
## <a name="description"></a>Opis  
 Liczba transakcji przekazane do operacji w tym punkcie końcowym. Ten licznik jest zwiększany ilekroć dowolny identyfikator transakcji znajduje się w wiadomości wysyłane do punktu końcowego.
