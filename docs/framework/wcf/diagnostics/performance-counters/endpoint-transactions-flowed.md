# <a name="endpoint-transactions-flowed"></a>Punkt końcowy: Przekazane transakcje

Nazwa licznika: przepływają transakcje.  
  
## <a name="description"></a>Opis  

 Liczba transakcji przepływających do operacji w tym punkcie końcowym. Ten licznik jest zwiększany za każdym razem, gdy w komunikacie wysyłanym do punktu końcowego jest obecny identyfikator transakcji.
