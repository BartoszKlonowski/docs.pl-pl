# <a name="queue-dropped-messages"></a>Porzucone komunikaty w kolejce
Nazwa licznika: Porzucone wiadomości w kolejce.  
  
## <a name="description"></a>Opis  
 Liczba komunikatów, które są usuwane przez transport w kolejce w tej usłudze.  
  
 Aby uzyskać więcej informacji o tym, kiedy wiadomości są odrzucane, zobacz [Obsługa poison-message](../../feature-details/poison-message-handling.md).
