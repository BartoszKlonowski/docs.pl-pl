# <a name="queue-dropped-messages"></a>Porzucone komunikaty w kolejce
Nazwa licznika: Porzucone komunikaty w kolejce.  
  
## <a name="description"></a>Opis  
 Liczba wiadomości, które są odrzucane przez umieszczonych w kolejce transportu na tę usługę.  
  
 Zobacz [obsługi komunikatów Poison](https://go.microsoft.com/fwlink/?LinkId=96233) więcej informacji na temat gdy komunikaty są odrzucane.
