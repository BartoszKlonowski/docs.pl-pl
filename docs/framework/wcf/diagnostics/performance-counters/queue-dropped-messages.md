# <a name="queue-dropped-messages"></a>Porzucone komunikaty w kolejce

Nazwa licznika: porzucone wiadomości w kolejce.  
  
## <a name="description"></a>Opis  

 Liczba komunikatów porzuconych przez transport w kolejce w tej usłudze.  
  
 Aby uzyskać więcej informacji na temat usuwania komunikatów, zobacz [Obsługa komunikatów trujących](../../feature-details/poison-message-handling.md).
