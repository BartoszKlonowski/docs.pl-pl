# <a name="queued-rejected-messages"></a>Zakolejkowane komunikaty odrzucone
Nazwa licznika: Zakolejkowane komunikaty odrzucone.  
  
## <a name="description"></a>Opis  
 Liczba komunikatów, które zostały odrzucone przez transport z kolejką w tej usłudze.  
  
 Zobacz [obsługi wiadomości Poison](http://go.microsoft.com/fwlink/?LinkID=96233) uzyskać więcej informacji o podczas wiadomości zostały odrzucone.
