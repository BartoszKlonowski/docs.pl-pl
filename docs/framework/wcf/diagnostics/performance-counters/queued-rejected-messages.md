# <a name="queued-rejected-messages"></a>Zakolejkowane komunikaty odrzucone
Nazwa licznika: W kolejce komunikaty odrzucone.  
  
## <a name="description"></a>Opis  
 Liczba wiadomości, które zostały odrzucone przez umieszczonych w kolejce transportu na tę usługę.  
  
 Zobacz [obsługi komunikatów Poison](https://go.microsoft.com/fwlink/?LinkID=96233) więcej informacji na temat po wiadomości są odrzucane.
