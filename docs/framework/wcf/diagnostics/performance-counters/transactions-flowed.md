# <a name="transactions-flowed"></a>Przepływ transakcji

Nazwa licznika: przepływające transakcje  
  
## <a name="description"></a>Opis  

 Liczba transakcji przepływających do tej operacji. Licznik jest zwiększany dla każdej wiadomości wysyłanej do operacji i zawiera identyfikator transakcji.
