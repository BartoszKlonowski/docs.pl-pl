# <a name="calls-duration"></a>Czas trwania wywołań
Nazwa licznika: Czas trwania połączeń  
  
## <a name="description"></a>Opis  
 Średni czas trwania wywołań tej operacji. Średni czas trwania jest obliczany na podstawie tego równania: (N1-N0)/(D1-D0).  
  
> [!WARNING]
> Gdy jest używany w asynchronicznej usługi WCF wywołanie licznik duration zawsze zwróci -1.
