# <a name="calls-duration"></a>Czas trwania wywołań
Nazwa komputera: Czas trwania wywołań  
  
## <a name="description"></a>Opis  
 Średni czas trwania wywołań tej operacji. Średni czas trwania jest obliczany na podstawie równanie: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Stosowania na asynchroniczne usługi WCF licznik Czas trwania wywołania zawsze zwraca wartość -1.
