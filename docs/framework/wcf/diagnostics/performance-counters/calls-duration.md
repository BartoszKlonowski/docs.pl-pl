# <a name="calls-duration"></a>Czas trwania wywołań

Nazwa licznika: czas trwania wywołań  
  
## <a name="description"></a>Opis  

 Średni czas trwania wywołań tej operacji. Średni czas trwania jest obliczany na podstawie tego równania: (N1-N0)/(D1-d0).  
  
> [!WARNING]
> W przypadku korzystania z asynchronicznej usługi WCF licznik czas trwania wywołań zawsze zwróci wartość-1.
