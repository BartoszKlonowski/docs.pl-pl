# <a name="queued-poison-messages"></a>Skażone wiadomości w kolejce
Nazwa licznika: Skażone wiadomości w kolejce.  
  
## <a name="description"></a>Opis  
 Liczba wiadomości oznaczonych jako uszkodzone przez umieszczonych w kolejce transportu na tę usługę z kolejką.  
  
 Zobacz [obsługi komunikatów Poison](https://go.microsoft.com/fwlink/?LinkID=96233) więcej informacji na temat po wiadomości są oznaczone uszkodzone kolejką.
