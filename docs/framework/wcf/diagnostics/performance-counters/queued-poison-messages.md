# <a name="queued-poison-messages"></a>Skażone wiadomości w kolejce

Nazwa licznika: skażone komunikaty w kolejce.  
  
## <a name="description"></a>Opis  

 Liczba komunikatów oznaczonych jako skażone przez transport w kolejce w tej usłudze.  
  
 Aby uzyskać więcej informacji o tym, kiedy wiadomości są oznaczone jako trujące, zobacz [Obsługa komunikatów trujących](../../feature-details/poison-message-handling.md).
