# <a name="queued-poison-messages"></a>Skażone wiadomości w kolejce
Nazwa licznika: Wiadomości poison w kolejce.  
  
## <a name="description"></a>Opis  
 Liczba wiadomości, które są oznaczone zatrute przez transport w kolejce w tej usłudze.  
  
 Aby uzyskać więcej informacji o tym, kiedy wiadomości są oznaczone jako zatrute, zobacz [Obsługa trujących wiadomości](../../feature-details/poison-message-handling.md).
