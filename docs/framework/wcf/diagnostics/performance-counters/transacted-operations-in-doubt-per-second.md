# <a name="transacted-operations-in-doubt-per-second"></a>Wątpliwe operacje transakcyjne na sekundę
Nazwa licznika: Operacje transakcyjne w stanie wątpliwy na sekundę.  
  
## <a name="description"></a>Opis  
 Liczba operacji transakcyjnych z wynikami wątpliwymi w tej usłudze w drugim.  
  
 Ten licznik jest typem licznika wydajności [PERF_COUNTER_COUNTER](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10)), którego wartość jest obliczana przy użyciu następującej formuły.  
  
 (N 1-N 0)/((D 1-D 0)/F)
