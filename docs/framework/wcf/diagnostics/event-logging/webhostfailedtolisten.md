# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Identyfikator: 112  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas próby nasłuchiwania na określony adres URL. Ten proces roboczy zostanie zakończony. Zdarzenie zawiera wyjątek, nazwa procesu i identyfikatora procesu.
