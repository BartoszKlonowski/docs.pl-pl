# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
#C1 112  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że wystąpił błąd podczas próby nasłuchiwania określonego adresu URL. Ten proces roboczy jest zakończony. Zdarzenie zawiera listę wyjątku, nazwę procesu i identyfikator procesu.
