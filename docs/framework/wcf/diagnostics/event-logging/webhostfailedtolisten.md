# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id: 112  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas próby nasłuchiwania na określony adres URL. Ten proces zostanie zakończony. Zdarzenie Wyświetla listę wyjątków, nazwa procesu i identyfikatora procesu.
