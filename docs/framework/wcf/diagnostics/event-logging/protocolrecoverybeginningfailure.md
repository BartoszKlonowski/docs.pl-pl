# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Identyfikator: 144  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że protokołu MSDTC WS-Atomic Transaction nie powiodło się na początku odzyskiwania. W związku z tym funkcja protokołu WS-AT jest wyłączona. Zdarzenia zawiera identyfikator protokołu, nazwy protokołu, wyjątek, nazwa procesu i identyfikatora procesu.
