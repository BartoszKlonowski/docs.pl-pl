# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Id: 144  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że protokół WS-Atomic Transaction MSDTC nie powiodło się na początku odzyskiwania. W rezultacie WS-Atomic Transaction funkcja jest wyłączona. Zdarzenie Wyświetla identyfikator protokołu, nazwy protokołu, wyjątków, nazwa procesu i identyfikatora procesu.
