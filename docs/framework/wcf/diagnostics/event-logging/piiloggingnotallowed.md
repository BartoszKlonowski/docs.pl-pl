# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed

Identyfikator: 108  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że nie jest rejestrowane żadne znane dane OSOBowe. Rejestrowanie znanego elementu dane OSOBowe jest niedozwolone. Aby można było rejestrować znane dane OSOBowe, ustaw wartość "enableLoggingKnownPii" na `true` Machine.config. Zdarzenie Wyświetla nazwę procesu i identyfikator procesu.
