# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed
#C1 108  
  
 Obrażeń Błąd  
  
 Kategorii Śledzenie  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie jest rejestrowane żadne znane dane OSOBowe. Rejestrowanie znanego elementu dane OSOBowe jest niedozwolone. Aby można było rejestrować znane dane osobowe, ustaw wartość " `true` enableLoggingKnownPii" na w pliku Machine. config. Zdarzenie Wyświetla nazwę procesu i identyfikator procesu.
