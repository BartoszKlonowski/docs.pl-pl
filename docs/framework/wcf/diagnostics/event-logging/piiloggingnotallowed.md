# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed
Id: 108  
  
 Ważność: Błąd  
  
 Kategoria: Śledzenie  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, jest rejestrowane nie znane dane osobowe. Rejestrowanie znane dane osobowe nie jest dozwolone. Aby zezwolić na rejestrowanie znane też danych osobowych, ustaw "enableLoggingKnownPii" `true` w pliku Machine.config. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
