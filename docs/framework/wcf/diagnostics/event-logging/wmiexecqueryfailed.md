# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
Identyfikator: 124  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie WMI ExecQuery nie została przetworzona. Wyświetlane są zdarzenia wyjątków, nazwa procesu i identyfikatora procesu.
