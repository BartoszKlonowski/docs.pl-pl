# <a name="messageauthenticationfailure"></a>MessageAuthenticationFailure

Identyfikator: 170  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że uwierzytelnianie wiadomości nie powiodło się. Zdarzenie Wyświetla listę usługi, akcję, tożsamość klienta i identyfikator działania.
