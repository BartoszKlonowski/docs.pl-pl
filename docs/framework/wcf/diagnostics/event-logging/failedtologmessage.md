# <a name="failedtologmessage"></a>FailedToLogMessage
Identyfikator: 113  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że komunikat nie został zarejestrowany. Wyświetlane są zdarzenia wyjątków, nazwa procesu i identyfikatora procesu.
