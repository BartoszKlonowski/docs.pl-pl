# <a name="impersonationfailure"></a>ImpersonationFailure

Identyfikator: 176  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że personifikacja zakończyła się niepowodzeniem, wymieniając nazwę metody, tożsamość klienta i identyfikator działania.
