# <a name="impersonationfailure"></a>ImpersonationFailure
Id: 176  
  
 Ważność: Błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że personifikacja zakończyła się niepowodzeniem, zawierające nazwę metody, tożsamość klienta i identyfikator działania.
