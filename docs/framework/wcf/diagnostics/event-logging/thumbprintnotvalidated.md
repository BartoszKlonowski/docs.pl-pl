Identyfikator: 153  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge

 To zdarzenie oznacza, że nie można zweryfikować certyfikatu tożsamości o określonym odcisku palca. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
