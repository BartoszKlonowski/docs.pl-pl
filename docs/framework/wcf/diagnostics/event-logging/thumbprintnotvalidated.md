# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
Id: 153  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można zweryfikować certyfikatu tożsamości przy użyciu określonego odcisku palca. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
