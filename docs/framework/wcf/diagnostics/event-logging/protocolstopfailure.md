# <a name="protocolstopfailure"></a>ProtocolStopFailure
Identyfikator: 147  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można zatrzymać usługi protokołu WS-Atomic Transaction. Zdarzenia zawiera identyfikator protokołu, nazwy protokołu, wyjątek, nazwa procesu i identyfikatora procesu.
