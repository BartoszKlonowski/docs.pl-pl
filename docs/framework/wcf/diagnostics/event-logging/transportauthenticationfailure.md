# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure

Identyfikator: 174  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że uwierzytelnianie transportu zakończyło się niepowodzeniem, wymieniając tożsamość klienta i identyfikator działania.
