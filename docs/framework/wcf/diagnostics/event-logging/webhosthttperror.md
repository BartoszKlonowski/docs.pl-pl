# <a name="webhosthttperror"></a>WebHostHttpError
Identyfikator: 110  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił wyjątek HTTP hosta sieci Web. Zdarzenia wyświetlane są informacje o nadawcy, HtmlErrorMessage, wyjątek, nazwa procesu i identyfikatora procesu.
