# <a name="webhostunhandledexception"></a>WebHostUnhandledException
Id: 109  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że WebHost nieobsłużony wyjątek wystąpił. Zdarzenie Wyświetla informacje o nadawcy, wyjątków, nazwa procesu i proces identyfikatora.
