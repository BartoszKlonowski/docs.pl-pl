# <a name="webhostunhandledexception"></a>WebHostUnhandledException

Identyfikator: 109  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że wystąpił nieobsługiwany wyjątek WebHost. Zdarzenie Wyświetla informacje o nadawcy, wyjątek, nazwę procesu i identyfikator procesu.
