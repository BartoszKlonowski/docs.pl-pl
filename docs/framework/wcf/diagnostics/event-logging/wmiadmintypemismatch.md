 Ważność: błąd  
  
 Kategoria: ServiceModel

 To zdarzenie wskazuje, że wystąpił niezgodność typów. Określona właściwość klasy CIM nie może akceptować wartości określonego typu. Zdarzenie Wyświetla nazwę procesu i identyfikator procesu.
