# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch
Id: 128  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpiła niezgodność typów. Określona właściwość klasy modelu wspólnych informacji nie może akceptować wartości określonego typu. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
