# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch
Identyfikator: 128  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpiła niezgodność typów. Określona właściwość klasy modelu wspólnych informacji nie można zaakceptować wartości określonego typu. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
