# <a name="messageloggingon"></a>MessageLoggingOn
Id: 116  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 Włączono rejestrowanie komunikatów. Poufne informacje mogą być rejestrowane, nawet jeśli została zaszyfrowana na łączu. Na przykład komunikat o treści. Zdarzenia zawiera identyfikator aplikacji, wyjątków, nazwa procesu i identyfikatora procesu.
