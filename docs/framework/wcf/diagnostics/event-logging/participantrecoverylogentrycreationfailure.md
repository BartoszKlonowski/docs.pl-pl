# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
Id: 141  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można wygenerować wpisu dziennika arecovery dla uczestnika rejestracji. Transakcja zostanie przerwana. Listy zdarzeń identyfikator transakcji identyfikator rejestracji, wyjątków, nazwa procesu i identyfikatora procesu.
