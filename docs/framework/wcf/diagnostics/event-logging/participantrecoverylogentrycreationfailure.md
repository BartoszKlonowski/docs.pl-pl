# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
Identyfikator: 141  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można wygenerować wpisu dziennika arecovery dla rejestracji uczestnika. Transakcja zostanie przerwana. Listy zdarzeń identyfikator transakcji identyfikator rejestracji, wyjątek, nazwa procesu i identyfikatora procesu.
