# <a name="performancecounterinitializationfailure"></a>PerformanceCounterInitializationFailure

Identyfikator: 149  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że nie można zainicjować licznika wydajności. Zdarzenie zawiera listę Nazwa licznika, wyjątek, nazwę procesu i identyfikator procesu.
