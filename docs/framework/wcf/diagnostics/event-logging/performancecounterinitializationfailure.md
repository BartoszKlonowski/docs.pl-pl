Id: 149  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge

 To zdarzenie oznacza, że można zainicjować licznika wydajności. Zdarzenie Wyświetla nazwę licznika, wyjątków, nazwa procesu i identyfikatora procesu.
