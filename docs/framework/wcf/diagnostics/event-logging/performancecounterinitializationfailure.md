# <a name="performancecounterinitializationfailure"></a>PerformanceCounterInitializationFailure
Identyfikator: 149  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że można zainicjować licznika wydajności. Zdarzenie zawiera nazwę licznika, wyjątek, nazwa procesu i identyfikatora procesu.
