# <a name="unhandledstatemachineexceptionrecorddescription"></a>UnhandledStateMachineExceptionRecordDescription
Identyfikator: 136  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że zgłoszono nieobsługiwany wyjątek podczas przetwarzania zdarzenia automatu stanów. Listy zdarzeń identyfikator transakcji, nazwa komputera stanu, bieżący stan, Historia, Identyfikatora rejestracji, wyjątek, nazwa procesu i procesu identyfikatora.
