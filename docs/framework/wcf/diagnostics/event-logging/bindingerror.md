# <a name="bindingerror"></a>BindingError
Id: 159  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że wystąpił błąd podczas analizowania określone powiązanie określonej lokacji, dlatego protokół jest wyłączona dla witryny tymczasowo. Zdarzenie Wyświetla powiązania, źródło, wyjątków, nazwa procesu i identyfikatora procesu.
