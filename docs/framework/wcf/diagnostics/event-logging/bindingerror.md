# <a name="bindingerror"></a>BindingError
#C1 159  
  
 Obrażeń Błąd  
  
 Kategorii SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że wystąpił błąd podczas analizowania określonego powiązania określonej lokacji, w tym przypadku protokół jest tymczasowo wyłączony dla lokacji. Zdarzenie zawiera listę powiązań, źródła, wyjątku, nazwy procesu i identyfikatora procesu.
