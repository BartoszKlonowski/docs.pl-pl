# <a name="bindingerror"></a>BindingError
Identyfikator: 159  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że wystąpił błąd podczas analizowania określone powiązanie określonej lokacji, w związku z tym protokołem jest witryny tymczasowo wyłączone. Zdarzenia wyświetlane powiązanie, źródło, wyjątek, nazwa procesu i identyfikatora procesu.
