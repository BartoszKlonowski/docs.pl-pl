# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Id: 137  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza automatu stanów próba przetworzenia nieoczekiwane zdarzenie. Zdarzenie zostało uważane za krytyczne. Zdarzenie Wyświetla identyfikator transakcji, stan maszyny, bieżący stan, historii, zdarzeń nazwy, zdarzeń szczegółowe informacje, nazwa procesu i identyfikatora procesu.
