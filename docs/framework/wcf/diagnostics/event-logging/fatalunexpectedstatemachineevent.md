# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Identyfikator: 137  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że komputer stanu próbował przetworzyć nieoczekiwane zdarzenie. Zdarzenie zostało on traktowany jako krytyczny. Zdarzenie Wyświetla identyfikator transakcji, stan maszyny, bieżący stan, historii, zdarzenia nazwę, zdarzenie szczegóły, nazwa procesu i identyfikatora procesu.
