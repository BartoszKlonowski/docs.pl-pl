# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent

Identyfikator: 137  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że komputer stanu podjął próbę przetworzenia nieoczekiwanego zdarzenia. Zdarzenie zostało uznane za krytyczne. W tym zdarzeniu znajduje się lista identyfikator transakcji, automat stanu, bieżący stan, historia, nazwa zdarzenia, szczegóły zdarzenia, nazwa procesu i identyfikator procesu.
