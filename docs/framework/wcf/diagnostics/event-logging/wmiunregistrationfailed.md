# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
Id: 127  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że dostawca usługi WMI nie został wyrejestrowany. Zdarzenie wyświetla obiekt usługi WMI, błąd, nazwa procesu i identyfikatora procesu.
