# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
Identyfikator: 127  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że dostawca WMI nie został wyrejestrowany. Zdarzenia wyświetlane obiektu WMI, błąd, nazwa procesu i identyfikatora procesu.
