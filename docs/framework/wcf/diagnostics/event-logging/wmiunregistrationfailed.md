# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed

Identyfikator: 127  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że dostawca WMI nie został wyrejestrowany. Zdarzenie zawiera listę obiektów WMI, błąd, nazwa procesu i identyfikator procesu.
