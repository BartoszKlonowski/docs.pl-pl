# <a name="coordinatorrecoverylogentrycreationfailure"></a>CoordinatorRecoveryLogEntryCreationFailure
Identyfikator: 140  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 Nie można wygenerować wpisu dziennika odzyskiwania dla rejestracji koordynatora. Transakcja zostanie przerwana. Listy zdarzeń identyfikator transakcji, przyczyny, wyjątek, nazwa procesu i procesu identyfikatora.
