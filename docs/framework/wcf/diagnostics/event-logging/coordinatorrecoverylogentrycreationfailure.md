# <a name="coordinatorrecoverylogentrycreationfailure"></a>CoordinatorRecoveryLogEntryCreationFailure
Id: 140  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 Nie można wygenerować wpisów dziennika odzyskiwania dla rejestracji koordynatora. Transakcja zostanie przerwana. Listy zdarzeń identyfikator transakcji, przyczyna, wyjątków, nazwa procesu i proces identyfikatora.
