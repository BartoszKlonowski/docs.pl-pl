# <a name="wasdisconnected"></a>WasDisconnected
Identyfikator: 162  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że aktywacji procesów systemu Windows (WAS) została rozłączona. Zdarzenia wyświetlane HRESULT, nazwa procesu i identyfikatora procesu.
