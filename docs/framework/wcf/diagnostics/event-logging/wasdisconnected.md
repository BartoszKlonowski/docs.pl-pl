# <a name="wasdisconnected"></a>WasDisconnected
Id: 162  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że Windows Process Activation (WAS) została odłączona. Zdarzenie Wyświetla wartość HRESULT, nazwa procesu i identyfikatora procesu.
