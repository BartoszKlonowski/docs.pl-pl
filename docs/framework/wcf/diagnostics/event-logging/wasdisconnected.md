# <a name="wasdisconnected"></a>WasDisconnected
#C1 162  
  
 Obrażeń Błąd  
  
 Kategorii SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że aktywacja procesów systemu Windows (WAS) została odłączona. Zdarzenie zawiera listę wartości HRESULT, nazwa procesu i identyfikator procesu.
