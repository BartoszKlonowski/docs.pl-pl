# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
#C1 146  
  
 Obrażeń Błąd  
  
 Kategorii TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że podczas odzyskiwania TransactionBridge MSDTC zakończyło się niepowodzeniem. Jest to warunek krytyczny, więc usługa MSDTC została przerwana. Zdarzenie zawiera listę wyjątku, nazwę procesu i identyfikator procesu. .
