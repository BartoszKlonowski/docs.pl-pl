# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Identyfikator: 146  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usługa MSDTC TransactionBridge nie powiodło się podczas odzyskiwania. Jest to stan krytyczny, więc usługa MSDTC została zakończona. Zdarzenie zawiera wyjątek, nazwa procesu i identyfikatora procesu. .
