# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Id: 146  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że MSDTC TransactionBridge nie powiodło się podczas odzyskiwania. Jest to warunek krytyczny, aby usługa MSDTC została zakończona. Zdarzenie Wyświetla listę wyjątków, nazwa procesu i identyfikatora procesu. .
