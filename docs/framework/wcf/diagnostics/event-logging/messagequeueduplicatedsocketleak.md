 Ważność: błąd  
  
 Kategoria: SMSvcHost

 To zdarzenie oznacza, że wystąpił błąd podczas wysyłania zduplikowanego gniazda. Ta dojścia jest teraz ujawnione w procesie. Zdarzenie zawiera źródła, wyjątek, nazwa procesu i identyfikatora procesu.
