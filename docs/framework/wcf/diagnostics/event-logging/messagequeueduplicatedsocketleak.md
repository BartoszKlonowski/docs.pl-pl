# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
Id: 165  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas wysyłania zduplikowane gniazda. Tego dojścia jest teraz ujawnione w procesie. Zdarzenie zawiera źródła, wyjątków, nazwa procesu i identyfikatora procesu.
