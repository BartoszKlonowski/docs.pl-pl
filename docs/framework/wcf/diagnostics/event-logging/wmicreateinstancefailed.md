# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Identyfikator: 123  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie WMI CreateInstance nie zostało przetworzone. Zdarzenie zawiera nazwę klasy, wyjątek, nazwa procesu i identyfikatora procesu.
