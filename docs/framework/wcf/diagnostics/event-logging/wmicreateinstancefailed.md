# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Id: 123  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, żądanie CreateInstance — WMI nie została przetworzona. Zdarzenie zawiera nazwę klasy, wyjątków, nazwa procesu i identyfikatora procesu.
