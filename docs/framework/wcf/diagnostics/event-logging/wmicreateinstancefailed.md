# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed

Identyfikator: 123  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że żądanie wystąpienia usługi WMI nie zostało przetworzone. Zdarzenie Wyświetla nazwę klasy, wyjątek, nazwę procesu i identyfikator procesu.
