# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess

Identyfikator: 171  
  
 Ważność: informacyjne  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że negocjowanie zabezpieczeń zakończyło się pomyślnie, wyświetlając nazwę usługi, akcję, tożsamość klienta, ActivityId i negocjacje.
