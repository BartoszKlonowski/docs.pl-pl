# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess
Id: 171  
  
 Ważność: Informacyjny  
  
 Kategoria: SecurityAudit

 To zdarzenie oznacza, że negocjowanie zabezpieczeń zakończyła się pomyślnie, nazwę usługi, działania, tożsamość klienta, identyfikator działania i negocjacji.
