# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
Identyfikator: 119  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że licznik wydajności nie został usunięty. Zdarzenie Wyświetla nazwę kategorii, nazwa licznika, nazwę wystąpienia, wyjątek, nazwa procesu i identyfikatora procesu.
