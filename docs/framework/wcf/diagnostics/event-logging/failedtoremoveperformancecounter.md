# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
#C1 119  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że licznik wydajności nie został usunięty. Zdarzenie zawiera listę kategorii Nazwa, Nazwa licznika, nazwa wystąpienia, wyjątek, nazwa procesu i identyfikator procesu.
