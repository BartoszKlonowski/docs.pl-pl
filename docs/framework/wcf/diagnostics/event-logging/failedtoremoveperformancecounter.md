# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
Id: 119  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że licznika wydajności, który nie został usunięty. Zdarzenie Wyświetla nazwę kategorii, nazwa licznika, nazwę wystąpienia, wyjątek, nazwa procesu i identyfikatora procesu.
