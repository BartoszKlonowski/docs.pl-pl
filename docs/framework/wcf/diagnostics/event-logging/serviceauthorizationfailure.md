# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
Id: 168  
  
 Ważność: Błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usługa Autoryzacja nie powiodła się. Zdarzenie Wyświetla listę usług, działania, tożsamość klienta autoryzacji, identyfikator działania i ServiceAuthorizationManager.
