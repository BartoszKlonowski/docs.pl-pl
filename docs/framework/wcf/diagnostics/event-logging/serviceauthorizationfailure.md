# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
Identyfikator: 168  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usługa Autoryzacja nie powiodła się. Zdarzenie zawiera usługi, działania, tożsamość klienta elementu AuthorizationContext, identyfikator działania i ServiceAuthorizationManager.
