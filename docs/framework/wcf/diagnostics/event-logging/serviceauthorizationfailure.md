# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure

Identyfikator: 168  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że autoryzacja usługi nie powiodła się. Zdarzenie Wyświetla listę usługi, akcję, tożsamość klienta, AuthorizationContext, identyfikator działania i ServiceAuthorizationManager.
