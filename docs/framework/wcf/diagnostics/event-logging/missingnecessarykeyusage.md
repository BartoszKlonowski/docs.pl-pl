# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage

Identyfikator: 156  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że certyfikat tożsamości z określoną nazwą podmiotu i odciskiem palca nie zawiera określonych danych między jego użyciami. Wyświetla również nazwę procesu i identyfikator procesu.
