# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
Identyfikator: 156  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że z określoną nazwą podmiotu i odcisk palca certyfikatu tożsamości nie dostarcza szczegółowe dane między jego elementów Keyusage. Przedstawia on także nazwę procesu i identyfikatora procesu.
