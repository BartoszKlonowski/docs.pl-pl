# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
Identyfikator: 152  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można odnaleźć certyfikatu tożsamości o określonych odcisku palca. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
