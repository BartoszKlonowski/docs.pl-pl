# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
Id: 152  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można odnaleźć certyfikatu tożsamości przy użyciu określonego odcisku palca. Zdarzenie Wyświetla listę identyfikator procesu i nazwę procesu.
