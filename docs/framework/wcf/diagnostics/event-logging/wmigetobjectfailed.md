# <a name="wmigetobjectfailed"></a>WmiGetObjectFailed
#C1 120  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że żądanie GetObject usługi WMI nie zostało przetworzone. Zdarzenie zawiera listę obiektu WMI, wyjątek, nazwę procesu i identyfikator procesu.
