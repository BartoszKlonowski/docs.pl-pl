# <a name="wmipropertymissing"></a>WmiPropertyMissing
Identyfikator: 129  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpiła niezgodność typów. Określona klasa CIM nie ma określonej właściwości określonego typu. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
