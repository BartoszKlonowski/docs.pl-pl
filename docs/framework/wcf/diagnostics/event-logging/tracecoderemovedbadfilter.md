# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
Identyfikator: 114  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usunięto nieprawidłowy filtr rejestrowania komunikatów dostęp do treści wiadomości. Zdarzenie zawiera wyrażenie XPath, nazwa procesu i identyfikatora procesu.
