# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
#C1 114  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje zły Filtr rejestrowania komunikatów, który miał dostęp do treści komunikatu. Zdarzenie zawiera listę XPath, nazwę procesu i identyfikator procesu.
