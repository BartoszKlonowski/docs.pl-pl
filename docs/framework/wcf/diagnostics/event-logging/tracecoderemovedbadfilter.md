# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
Id: 114  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usunięto filtr rejestrowania zły komunikat, który został dostęp do treści wiadomości. Zdarzenie zawiera wyrażenie XPath, nazwa procesu i identyfikatora procesu.
