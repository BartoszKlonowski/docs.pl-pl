# <a name="servicestartfailed"></a>ServiceStartFailed
Id: 164  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie, aby uruchomić usługę nie powiodło się. Zdarzenie Wyświetla kod błędu, nazwa procesu i identyfikatora procesu.
