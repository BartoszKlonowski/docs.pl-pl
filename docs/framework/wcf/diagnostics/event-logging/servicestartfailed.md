# <a name="servicestartfailed"></a>ServiceStartFailed
Identyfikator: 164  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie uruchomienia usługi nie powiodło się. Zdarzenie zawiera kod błędu, nazwa procesu i identyfikatora procesu.
