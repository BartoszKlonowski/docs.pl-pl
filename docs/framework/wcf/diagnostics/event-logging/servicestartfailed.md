# <a name="servicestartfailed"></a>ServiceStartFailed

Identyfikator: 164  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że żądanie uruchomienia usługi nie powiodło się. Zdarzenie zawiera kod błędu, nazwę procesu i identyfikator procesu.
