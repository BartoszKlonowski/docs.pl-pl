# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
Identyfikator: 118  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że licznik wydajności nie został załadowany. Zdarzenie Wyświetla nazwę kategorii, nazwa licznika, nazwę wystąpienia, wyjątek, nazwa procesu i identyfikatora procesu.
