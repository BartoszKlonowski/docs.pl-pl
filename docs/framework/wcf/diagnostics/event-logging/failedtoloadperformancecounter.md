# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
Id: 118  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że licznika wydajności, który nie został załadowany. Zdarzenie Wyświetla nazwę kategorii, nazwa licznika, nazwę wystąpienia, wyjątek, nazwa procesu i identyfikatora procesu.
