# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
#C1 118  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że licznik wydajności nie został załadowany. Zdarzenie zawiera listę kategorii Nazwa, Nazwa licznika, nazwa wystąpienia, wyjątek, nazwa procesu i identyfikator procesu.
