# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
Identyfikator: 131  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas uruchamiania inicjatora DllHost. Zdarzenia wyświetlane AppID, wyjątek, nazwa procesu i identyfikatora procesu.
