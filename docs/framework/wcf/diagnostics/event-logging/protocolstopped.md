# <a name="protocolstopped"></a>ProtocolStopped

Identyfikator: 151  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że usługa protokołu Transaction WS-Atomic została zatrzymana. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, nazwa procesu i identyfikator procesu. .
