# <a name="protocolstopped"></a>ProtocolStopped
#C1 151  
  
 Obrażeń Błąd  
  
 Kategorii TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że usługa protokołu transakcji WS-AT została zatrzymana. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, nazwa procesu i identyfikator procesu. .
