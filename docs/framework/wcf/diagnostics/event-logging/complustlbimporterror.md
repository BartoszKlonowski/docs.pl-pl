# <a name="complustlbimporterror"></a>ComPlusTLBImportError

Identyfikator: 132  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że wystąpił błąd podczas importowania biblioteki typów. Zdarzenie zawiera listę identyfikatorów IID, identyfikator biblioteki typów, wyjątek, nazwę procesu i identyfikator procesu.
