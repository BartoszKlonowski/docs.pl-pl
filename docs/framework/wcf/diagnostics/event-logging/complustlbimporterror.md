# <a name="complustlbimporterror"></a>ComPlusTLBImportError
Identyfikator: 132  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas importowania biblioteki typów. Zdarzenia zawiera identyfikator IID, identyfikator biblioteki typów, wyjątek, nazwa procesu i identyfikatora procesu.
