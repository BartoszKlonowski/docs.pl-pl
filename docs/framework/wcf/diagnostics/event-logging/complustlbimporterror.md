# <a name="complustlbimporterror"></a>ComPlusTLBImportError
Id: 132  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas importowania biblioteki typów. Zdarzenie Wyświetla identyfikatora IID, identyfikator typu biblioteki, wyjątków, nazwa procesu i identyfikatora procesu.
