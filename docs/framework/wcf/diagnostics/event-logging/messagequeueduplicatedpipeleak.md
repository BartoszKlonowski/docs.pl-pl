# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak
Id: 166  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas wysyłania zduplikowany nazwany potok. Tego dojścia jest teraz ujawnione w procesie. Zdarzenie zawiera źródła, wyjątków, nazwa procesu i identyfikatora procesu.
