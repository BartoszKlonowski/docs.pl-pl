# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
Identyfikator: 157  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza certyfikatu tożsamości z nazw określonych podmiotu i odcisk palca nie ma określonych danych między jego elementów EnhancedKeyUsage. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
