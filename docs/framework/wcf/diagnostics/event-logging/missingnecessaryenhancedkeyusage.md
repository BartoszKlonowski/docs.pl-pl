# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage

Identyfikator: 157  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że certyfikat tożsamości z określoną nazwą podmiotu i odciskiem palca nie udostępnia określonych danych między jego EnhancedKeyUsages. Zdarzenie Wyświetla nazwę procesu i identyfikator procesu.
