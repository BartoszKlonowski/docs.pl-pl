# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
Id: 157  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, certyfikat tożsamości, z nazwą określonego podmiotu i odcisk palca nie zapewnia określonych danych między jego EnhancedKeyUsages. Zdarzenie Wyświetla nazwę procesu i identyfikatora procesu.
