# <a name="failfastexception"></a>FailFastException
Identyfikator: 103  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  
 Wystąpił wyjątek podczas natychmiastowy błąd. Zdarzenie zawiera wyjątek, nazwa procesu i identyfikatora procesu.
