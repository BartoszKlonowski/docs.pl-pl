# <a name="failfastexception"></a>FailFastException

Identyfikator: 103  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  

 Zgłoszono wyjątek podczas FailFast. Zdarzenie zawiera listę wyjątku, nazwę procesu i identyfikator procesu.
