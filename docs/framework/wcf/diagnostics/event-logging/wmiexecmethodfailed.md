# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed
Identyfikator: 125  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie WMI ExecMethod nie zostało przetworzone. Wyświetlane są zdarzenia wyjątków, nazwa procesu i identyfikatora procesu.
