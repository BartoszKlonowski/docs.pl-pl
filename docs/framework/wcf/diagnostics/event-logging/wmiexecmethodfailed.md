# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed

Identyfikator: 125  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że żądanie usługi WMI ExecMethod nie zostało przetworzone. Zdarzeń zawiera wyjątek, nazwę procesu i identyfikator procesu.
