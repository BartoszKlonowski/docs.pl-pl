# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed
Id: 125  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, czy żądanie usługi WMI, którym nie została przetworzona. Zdarzenie Wyświetla listę wyjątków, nazwa procesu i identyfikatora procesu.
