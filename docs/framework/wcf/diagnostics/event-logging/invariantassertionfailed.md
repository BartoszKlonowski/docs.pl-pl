# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Identyfikator: 106  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że potwierdzenie niezmiennej ma wartość false. Zdarzenia wyświetla komunikat, nazwa procesu i identyfikatora procesu.
