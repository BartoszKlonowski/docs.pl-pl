# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Id: 106  
  
 Ważność: Błąd  
  
 Kategoria: Śledzenie  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że potwierdzenie niezmiennej ma wartość false. Zdarzenie wyświetla komunikat, nazwa procesu i identyfikatora procesu.
