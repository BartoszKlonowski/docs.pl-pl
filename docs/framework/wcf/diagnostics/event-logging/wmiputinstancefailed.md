# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
Id: 121  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie PutInstance usługi WMI nie została przetworzona. Wyświetla zdarzenia obiektu WMI, wyjątków, nazwa procesu i procesu identyfikatora.
