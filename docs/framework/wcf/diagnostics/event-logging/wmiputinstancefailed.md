# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
#C1 121  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że żądanie WMI PutInstance nie zostało przetworzone. Zdarzenie zawiera listę obiektu WMI, wyjątek, nazwę procesu i identyfikator procesu.
