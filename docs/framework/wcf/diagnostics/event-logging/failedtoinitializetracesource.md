# <a name="failedtoinitializetracesource"></a>FailedToInitializeTraceSource
Identyfikator: 101  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  
 Nie można zainicjować źródła śledzenia. Śledzenie jest wyłączone. Zdarzenie zawiera wyjątek, nazwa procesu i identyfikatora procesu.
