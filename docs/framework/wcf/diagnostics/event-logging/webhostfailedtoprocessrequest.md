# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
Id: 111  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że hostem sieci Web nie można przetworzyć żądania. Zdarzenie Wyświetla informacje o nadawcy, wyjątków, nazwa procesu i proces identyfikatora.
