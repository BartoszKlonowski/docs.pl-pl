# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest

Identyfikator: 111  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że przetworzenie żądania przez element WebHost nie powiodło się. Zdarzenie Wyświetla informacje o nadawcy, wyjątek, nazwę procesu i identyfikator procesu.
