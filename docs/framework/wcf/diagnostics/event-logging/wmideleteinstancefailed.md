# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
Id: 122  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, czy żądanie WMI DeleteInstance nie została przetworzona. Zdarzenie Wyświetla listę wyjątków, nazwa procesu i identyfikatora procesu.
