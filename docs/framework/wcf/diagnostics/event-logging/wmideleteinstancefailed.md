# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
#C1 122  
  
 Obrażeń Błąd  
  
 Kategorii Modelu  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że żądanie WMI DeleteInstance nie zostało przetworzone. Zdarzenie zawiera listę wyjątku, nazwę procesu i identyfikator procesu.
