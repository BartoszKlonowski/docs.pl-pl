# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
Identyfikator: 122  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że żądanie WMI DeleteInstance nie zostało przetworzone. Zdarzenie zawiera wyjątek, nazwa procesu i identyfikatora procesu.
