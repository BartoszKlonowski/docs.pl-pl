# <a name="piiloggingon"></a>PiiLoggingOn

Identyfikator: 107  
  
 Ważność: błąd  
  
 Kategoria: śledzenie  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że logowanie z danymi OSOBowymi zostało włączone. Informacje poufne zostaną zarejestrowane. Zdarzenie Wyświetla nazwę procesu i identyfikator procesu.
