# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
#C1 148  
  
 Obrażeń Błąd  
  
 Kategorii TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie wskazuje, że komputer stanu podjął próbę przetworzenia nieoczekiwanego zdarzenia. Zdarzenie nie jest uznawane za krytyczne. W tym zdarzeniu znajduje się lista identyfikator transakcji, automat stanu, bieżący stan, historia, nazwa zdarzenia, szczegóły zdarzenia, nazwa procesu i identyfikator procesu.
