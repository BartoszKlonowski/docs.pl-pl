# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
Identyfikator: 148  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że komputer stanu próbował przetworzyć nieoczekiwane zdarzenie. Zdarzenie nie jest uznawane za krytyczne. Zdarzenie Wyświetla identyfikator transakcji, stan maszyny, bieżący stan, historii, zdarzenia nazwę, zdarzenie szczegóły, nazwa procesu i identyfikatora procesu.
