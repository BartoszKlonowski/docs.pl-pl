# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
Id: 148  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza automatu stanów próba przetworzenia nieoczekiwane zdarzenie. Zdarzenia nie są uważane za krytyczne. Zdarzenie Wyświetla identyfikator transakcji, stan maszyny, bieżący stan, historii, zdarzeń nazwy, zdarzeń szczegółowe informacje, nazwa procesu i identyfikatora procesu.
