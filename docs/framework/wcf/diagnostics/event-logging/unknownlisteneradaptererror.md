# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Identyfikator: 161  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił nieoczekiwany błąd w adapterze odbiornika podczas obsługi powiadomienia usługi aktywacji procesów systemu Windows (WAS). Proces jest zakończony. Zdarzenia wyświetlane protokołu, wyjątek, nazwa procesu i identyfikatora procesu.
