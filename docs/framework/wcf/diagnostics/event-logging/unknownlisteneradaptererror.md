# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError

Identyfikator: 161  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że wystąpił nieoczekiwany błąd w adapterze odbiornika podczas obsługi powiadomienia usługi aktywacji procesów systemu Windows (WAS). Proces został zakończony. Zdarzenie Wyświetla listę protokołów, wyjątków, nazwy procesu i identyfikatora procesu.
