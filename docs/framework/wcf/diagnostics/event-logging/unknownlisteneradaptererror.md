# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Id: 161  
  
 Ważność: Błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił nieoczekiwany błąd karty odbiornika podczas obsługi powiadomienia Windows Process Activation Service (WAS). Ten proces zostanie zakończony. Zdarzenie Wyświetla protokołu, wyjątków, nazwa procesu i identyfikatora procesu.
