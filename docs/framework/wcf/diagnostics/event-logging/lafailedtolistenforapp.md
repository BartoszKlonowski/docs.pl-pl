# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Identyfikator: 160  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd w określonej usługi aktywacji określonego protokołu podczas próby nasłuchiwania dla określonej lokacji, w związku z tym protokołem jest witryny tymczasowo wyłączone. Zdarzenie zawiera adres URL, stan, wyjątek, nazwa procesu i identyfikatora procesu.
