# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
Id: 130  
  
 Ważność: Błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas uruchamiania usług COM +. Zdarzenie Wyświetla identyfikator ClsID, zastępczy, wyjątków, nazwa procesu i identyfikatora procesu.
