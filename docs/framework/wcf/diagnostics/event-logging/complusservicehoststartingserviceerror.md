# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError

Identyfikator: 130  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że wystąpił błąd podczas uruchamiania usługi COM+. Zdarzenie Wyświetla identyfikator AppID, identyfikator ClsID, Surogat, wyjątek, nazwę procesu i identyfikator procesu.
