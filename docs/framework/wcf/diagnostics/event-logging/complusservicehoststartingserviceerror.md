# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
Identyfikator: 130  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wystąpił błąd podczas uruchamiania usług COM +. Zdarzenia zawiera identyfikator ClsID, Surogat wyjątek, nazwa procesu i identyfikatora procesu.
