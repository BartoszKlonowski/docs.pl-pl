# <a name="protocolstartfailure"></a>ProtocolStartFailure
Identyfikator: 143  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można uruchomić usługi protokołu WS-Atomic Transaction. W związku z tym funkcja protokołu WS-AT jest wyłączona. Zdarzenia zawiera identyfikator protokołu, nazwy protokołu, wyjątek, nazwa procesu i identyfikatora procesu. .
