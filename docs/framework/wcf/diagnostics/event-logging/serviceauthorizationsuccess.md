# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
Identyfikator: 176  
  
 Ważność: informacyjny  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że usługa autoryzacji zakończyła się pomyślnie. Zdarzenie zawiera usługi, działania, tożsamość klienta, kontekst autoryzacji, identyfikator działania i ServiceAuthorizationManager.
