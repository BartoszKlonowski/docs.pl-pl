# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess

Identyfikator: 176  
  
 Ważność: informacyjne  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że autoryzacja usługi powiodła się. Zdarzenie Wyświetla listę usługi, akcję, tożsamość klienta, kontekst autoryzacji, identyfikator działania i ServiceAuthorizationManager.
