# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Identyfikator: 126  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że dostawca WMI nie został zarejestrowany. Zdarzenia wyświetlane obiektu WMI, błąd, nazwa procesu i identyfikatora procesu.
