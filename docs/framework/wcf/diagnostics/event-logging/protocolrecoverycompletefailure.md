# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
Identyfikator: 145  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że Usługa protokołu WS-Atomic Transaction nie została wykonana, uruchamianie i odzyskiwanie. W związku z tym funkcja protokołu WS-AT jest wyłączona. Zdarzenia zawiera identyfikator protokołu, nazwy protokołu, wyjątek, nazwa procesu i identyfikatora procesu.
