# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure

Identyfikator: 145  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że usługa protokołu transakcji WS-Atomic nie może zakończyć uruchamiania i odzyskiwania. W efekcie funkcja WS-Atomic Transaction jest wyłączona. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, wyjątek, nazwa procesu i identyfikator procesu.
