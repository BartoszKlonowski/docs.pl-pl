# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
Identyfikator: 172  
  
 Ważność: błąd  
  
 Kategoria: SecurityAudit  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można wynegocjować zabezpieczeń. Zdarzenie zawiera usługi, działania, tożsamość klienta, identyfikator działania i negocjacji.
