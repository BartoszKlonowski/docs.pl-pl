# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete

Identyfikator: 150  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie wskazuje, że usługa protokołu Transaction WS-Atomic pomyślnie zakończyła działanie i odzyskiwanie. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, nazwa procesu i identyfikator procesu.
