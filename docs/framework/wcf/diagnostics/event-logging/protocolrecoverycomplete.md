# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
Identyfikator: 150  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że Usługa protokołu WS-AT zostało pomyślnie ukończone uruchamiania i odzyskiwania. Zdarzenia zawiera identyfikator protokołu, nazwy protokołu i nazwy procesu identyfikatora procesu.
