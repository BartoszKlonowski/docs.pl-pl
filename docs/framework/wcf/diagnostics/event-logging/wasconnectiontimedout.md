# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
Identyfikator: 163  
  
 Ważność: błąd  
  
 Kategoria: SMSvcHost  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że wymagającego Usługa aktywacji procesów systemu Windows (WAS) upłynął limit czasu połączenia, nazwa procesu i identyfikator.
