# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
Identyfikator: 115  
  
 Ważność: błąd  
  
 Kategoria: ServiceModel  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że rejestrowanie komunikatów nie można utworzyć źródła śledzenia. Wyświetlane są zdarzenia wyjątków, nazwa procesu i identyfikatora procesu.
