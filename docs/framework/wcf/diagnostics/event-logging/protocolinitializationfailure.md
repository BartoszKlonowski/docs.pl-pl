# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
#C1 142  
  
 Obrażeń Błąd  
  
 Kategorii TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można zainicjować usługi protokołu transakcji WS-AT. W efekcie funkcja transakcji WS-AT jest wyłączona. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, wyjątek, nazwa procesu i identyfikator procesu. .
