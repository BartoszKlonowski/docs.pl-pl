# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
Id: 142  
  
 Ważność: Błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  
 To zdarzenie oznacza, że nie można zainicjować usługi protokołu WS-Atomic Transaction. W rezultacie WS-Atomic Transaction funkcja jest wyłączona. Zdarzenie Wyświetla identyfikator protokołu, nazwy protokołu, wyjątków, nazwa procesu i identyfikatora procesu. .
