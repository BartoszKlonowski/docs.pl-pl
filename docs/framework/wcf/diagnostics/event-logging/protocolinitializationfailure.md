# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure

Identyfikator: 142  
  
 Ważność: błąd  
  
 Kategoria: TransactionBridge  
  
## <a name="description"></a>Opis  

 To zdarzenie oznacza, że nie można zainicjować usługi protokołu Transaction WS-Atomic. W efekcie funkcja WS-Atomic Transaction jest wyłączona. W tym zdarzeniu znajduje się lista Identyfikator protokołu, nazwa protokołu, wyjątek, nazwa procesu i identyfikator procesu. .
