# <a name="systemservicemodelchannelspnrpresolvedaddresses"></a>System.ServiceModel.Channels.PnrpResolvedAddresses

Program rozpoznawania protokołu PNRP ukończył operację rozwiązania.  
  
## <a name="description"></a>Opis  

 To śledzenie ostrzeżeń wskazuje, że program rozpoznawania nazw PNRP nie może znaleźć żadnych adresów. Ten ślad występuje podczas operacji rozpoznawania przez program rozpoznawania nazw PNRP.
