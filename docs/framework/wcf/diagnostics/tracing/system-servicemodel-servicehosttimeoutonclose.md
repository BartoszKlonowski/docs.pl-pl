# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Opis  
 Operacja zamykania elementu ServiceHost jest został przekroczony. Oznacza to, że niektórych zamykania kanałów nie zostały prawidłowo zamknięcia przez klienta i może zostać przerwane. Zmodyfikuj domyślny czas, aby dostosować limit czasu zamknięcia.
