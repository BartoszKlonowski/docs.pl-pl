# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose

System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Opis  

 Przekroczono limit czasu operacji zamykania hosta ServiceHost. Oznacza to, że niektóre kanały sesji nie zostały prawidłowo zamknięte przez klienta i mogły zostać przerwane. Zmodyfikuj domyślny czas, aby dostosować limit czasu zamknięcia.
