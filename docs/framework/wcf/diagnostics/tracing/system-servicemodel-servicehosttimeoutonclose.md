# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Opis  
 Operacja zamykania elementu ServiceHost przekroczyła limit czasu. Oznacza to, że niektóre sesji kanały nie zostały prawidłowo zamknięte przez klienta i może zostać przerwane. Zmodyfikuj domyślny czas, aby dostosować limit czasu zamknięcia.
