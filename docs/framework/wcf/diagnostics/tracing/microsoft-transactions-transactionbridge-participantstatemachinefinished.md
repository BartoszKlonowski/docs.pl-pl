# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
Komputer stanu dla rejestracji uczestnika przeszedł w stan zakończenia.  
  
## <a name="description"></a>Opis  
 Śledzone po zakończeniu przetwarzania 2pc podrzędnego rejestracji uczestnika. Wynik dla rejestracji mogą być zatwierdzone lub anulowane. Również śledzonego, jeśli każdy uczestnik głosów tylko do odczytu podczas przygotowania.
