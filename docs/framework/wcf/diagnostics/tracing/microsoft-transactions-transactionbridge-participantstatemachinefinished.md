# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
Automatu stanów dla uczestnika rejestracji została zakończona stanu.  
  
## <a name="description"></a>Opis  
 Śledzone, gdy podrzędny rejestracji uczestnika zakończy przetwarzanie 2pc. Wynik dla rejestracji może być przydzielony lub Aborted. Również śledzona jest, gdy każdy uczestnik głosów tylko do odczytu podczas przygotowywania.
