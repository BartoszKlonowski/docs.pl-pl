# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished

Komputer stanu dla rejestracji uczestnika przeszedł w stan zakończenia.  
  
## <a name="description"></a>Opis  

 Śledzone, gdy Rejestracja uczestnika podrzędnego zakończyła przetwarzanie 2PC. Wynik rejestracji można zatwierdzić lub przerwać. Jest on także śledzony, jeśli którykolwiek uczestnik odgłosów w trakcie przygotowania.
