# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Nie można otworzyć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Obiektu komunikacyjnego nie można otworzyć, ponieważ wystąpił wyjątek podczas wykonywania metody Open. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
