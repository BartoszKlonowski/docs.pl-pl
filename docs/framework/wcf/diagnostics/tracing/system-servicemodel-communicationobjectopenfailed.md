# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Nie można otworzyć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można otworzyć, ponieważ wystąpił wyjątek podczas wykonywania metody Open obiektu komunikacji. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
