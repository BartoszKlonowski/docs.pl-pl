# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Nie można otworzyć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można otworzyć obiektu komunikacyjnego, ponieważ wystąpił wyjątek podczas wykonywania otwartej metody. Inny ślad w pobliżu może zawierać szczegółowe informacje o wyjątku.
