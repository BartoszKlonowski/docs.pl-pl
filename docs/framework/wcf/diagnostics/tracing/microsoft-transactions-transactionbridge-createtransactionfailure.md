# <a name="microsofttransactionstransactionbridgecreatetransactionfailure"></a>Microsoft.Transactions.TransactionBridge.CreateTransactionFailure

Nie można utworzyć transakcji.  
  
## <a name="description"></a>Opis  

 Ten ślad jest generowany, gdy usługa MSDTC nie może utworzyć transakcji. Może to być spowodowane małą ilością zasobów, niewystarczającą ilością miejsca w dzienniku lub innymi błędami.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  

 Sprawdź ciąg stanu w komunikacie śledzenia, aby określić, czy istnieje jakikolwiek element możliwy do wykonania.
