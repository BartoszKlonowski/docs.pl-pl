# <a name="microsofttransactionstransactionbridgecreatetransactionfailure"></a>Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
Nie można utworzyć transakcji.  
  
## <a name="description"></a>Opis  
 Ślad jest generowany po nie można utworzyć transakcji usługi MSDTC. Może to być spowodowane małą ilością zasobów, miejsca w dzienniku niewystarczające lub inne błędy.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy ciąg stanu w komunikat śledzenia do ustalenia, czy istnieje dowolny element przydatnych wyników.
