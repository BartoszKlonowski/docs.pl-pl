# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Uzgadnianie zabezpieczeń nie powiodła się przed upływem określonego limitu czasu.  
  
## <a name="description"></a>Opis  
 Ślad występuje podczas próby ustanowienia połączenia z elementem sąsiednim bezpieczne.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że sąsiadów wszystkich poświadczeń zabezpieczeń odpowiednich i odpowiadanie na wszystkie wiadomości w mniej niż minutę.
