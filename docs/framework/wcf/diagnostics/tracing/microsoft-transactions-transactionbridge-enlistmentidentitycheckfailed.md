# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed

Tożsamość nadawcy wiadomości przychodzącej nie jest zgodna z tożsamością pochodzącą z odwołania do punktu końcowego rejestracji.  
  
## <a name="description"></a>Opis  

 Śledzone, jeśli sprawdzanie poprawności zabezpieczeń wiadomości nie powiodło się.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  

 Upewnij się, że zabezpieczenia transakcji są poprawnie skonfigurowane dla wszystkich komputerów uczestniczących w transakcji.
