# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Tożsamość nadawcy wiadomości przychodzące nie były zgodne z tożsamością pochodzącą z odwołania do punktu końcowego rejestracji.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli Weryfikacja zabezpieczeń komunikatu nie powiodła się.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że zabezpieczenia transakcji został poprawnie skonfigurowany dla wszystkich komputerów w transakcji.
