# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Tożsamość nadawcy komunikatu przychodzącego nie był zgodny z tożsamością pochodzącą z odwołania do punktu końcowego rejestracji.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli sprawdzania zabezpieczeń wiadomości nie powiodło się.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że zabezpieczenia transakcji został poprawnie skonfigurowany dla wszystkich maszyn uczestniczących w transakcji.
