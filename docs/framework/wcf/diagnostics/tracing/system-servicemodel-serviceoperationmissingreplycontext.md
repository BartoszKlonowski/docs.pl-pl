# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Opis  
 ReplyContext ma wartość null dla operacji dwukierunkowe. Dane śledzenia wyświetla następujący komunikat: "Operacji odpowiedzi jest nie IRequestContext na potrzeby odpowiedzi."  
  
 Ślad jest przydatne podczas debugowania w czasie projektowania.
