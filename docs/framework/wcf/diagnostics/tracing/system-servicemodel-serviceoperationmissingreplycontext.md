# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext

System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Opis  

 ReplyContext ma wartość null dla operacji dwukierunkowych. Dane śledzenia wyświetlają następujący komunikat: "odpowiedź nie ma IRequestContext do użycia w odpowiedzi".  
  
 Ten ślad jest przydatny do debugowania podczas opracowywania.
