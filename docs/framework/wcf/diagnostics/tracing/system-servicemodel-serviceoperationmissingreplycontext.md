# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Opis  
 ReplyContext ma wartość null dla operacji dwukierunkowego. Dane śledzenia wyświetli następujący komunikat: "Operacja odpowiedź ma nie IRequestContext do użycia w odpowiedzi."  
  
 Ślad jest przydatne w przypadku debugowania w czasie projektowania.
