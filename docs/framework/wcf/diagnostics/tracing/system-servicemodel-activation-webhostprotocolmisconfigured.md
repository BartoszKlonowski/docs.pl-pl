# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Protokół jest niepoprawnie skonfigurowany w usłudze aktywacji procesów systemu Windows (WAS).  
  
## <a name="description"></a>Opis  
 Ten błąd występuje, gdy protokół transportowy jest niepoprawnie skonfigurowany.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź i napraw parametry konfiguracji transportu w pliku Web. config.
