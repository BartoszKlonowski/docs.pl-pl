# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Nie można zamknąć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można zamknąć, ponieważ wystąpił wyjątek podczas wykonywania jego metody Close obiektu komunikacyjnego. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
