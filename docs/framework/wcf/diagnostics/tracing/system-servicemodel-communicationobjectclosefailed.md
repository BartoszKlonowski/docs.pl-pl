# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Nie można zamknąć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można zamknąć, ponieważ wystąpił wyjątek podczas wykonywania metody Close obiektu komunikacji. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
