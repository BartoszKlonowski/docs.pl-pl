# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Nie można zamknąć obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można zamknąć obiektu komunikacyjnego, ponieważ wystąpił wyjątek podczas wykonywania jego metody Close. Inny ślad w pobliżu może zawierać szczegółowe informacje o wyjątku.
