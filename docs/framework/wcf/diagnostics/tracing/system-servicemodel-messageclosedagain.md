# <a name="systemservicemodelmessageclosedagain"></a>System.ServiceModel.MessageClosedAgain
System.ServiceModel.MessageClosedAgain  
  
## <a name="description"></a>Opis  
 Zamknięto wiadomość ponownie.  
  
 Komunikat powinien zostać zamknięty, tylko raz. Trwa wysyłanego tego śledzenia w kodzie użytkownika rozszerzenia wskazuje, że kod rozszerzenia użytkownika zostanie zamknięty komunikat, który został już zamknięty. Ślad jest są emitowane przez kod produktu, wskazuje, że kod rozszerzenia użytkownika może potencjalnie można zamknięcia komunikat zbyt wcześnie.
