# <a name="debugging-on-the-client"></a>Debugowanie na kliencie

Aby ułatwić użytkownikom pisanie aplikacji klienckich dla usługi WCF, możesz dodać [\<serviceDebug>](../../../configure-apps/file-schema/wcf/servicedebug.md) zachowanie usługi do pliku konfiguracji usługi. To zachowanie może służyć do publikowania stron pomocy i zwracania informacji o wyjątku zarządzanym w szczegółowych informacjach o błędach SOAP zwracanych do klienta.
