# <a name="debugging-on-the-client"></a>Debugowanie na kliencie
Aby ułatwić użytkownikom na zapis aplikacje klienckie usługi WCF, można dodać [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) usługi zachowania do pliku konfiguracji usługi. To zachowanie może służyć do publikowania strony pomocy i zwraca informacje o zarządzanym wyjątku w szczegółowe informacje o błędach SOAP zwracanych do klienta.
