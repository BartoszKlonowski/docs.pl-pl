# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Opis  
 Deserializator nie wie, jakiego typu użyć podczas deserializacji. Rozważ dodanie typu określonego jako typ fabryczny w strumieniu XML do listy znanych typów — na przykład za pomocą atrybutu KnownTypeAttribute.
