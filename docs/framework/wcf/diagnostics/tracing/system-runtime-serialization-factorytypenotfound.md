# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound

System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Opis  

 Deserializacja nie ma informacji o typie, który ma być używany podczas deserializacji. Rozważ dodanie typu określonego jako typ fabryki w kodzie XML do listy znanych typów — na przykład przy użyciu atrybutu KnownTypeattribute.
