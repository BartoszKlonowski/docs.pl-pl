# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Opis  
 Deserializator nie zna tego typu do użycia podczas deserializacji. Rozważ dodanie typ określony jako typ fabryki w formacie XML do listy znanych typów — na przykład za pomocą atrybutu KnownTypeAttribute.
