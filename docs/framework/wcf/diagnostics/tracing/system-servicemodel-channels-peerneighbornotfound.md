# <a name="systemservicemodelchannelspeerneighbornotfound"></a>System.ServiceModel.Channels.PeerNeighborNotFound

Nie można wykonać operacji na sąsiadu, ponieważ nie można odnaleźć tego sąsiada na liście sąsiadów.  
  
## <a name="description"></a>Opis  

 Ten błąd występuje podczas nawiązywania połączenia z sąsiadem lub zamykania go.
