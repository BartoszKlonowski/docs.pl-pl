# <a name="systemservicemodeltxsourcetxscoperequiredistransactedtransport"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
Podano określoną transakcję dla określonej operacji z transportem transakcyjnym, takim jak usługa MSMQ.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli transakcja udostępniona dla kodu użytkownika została utworzona przez sam transport.
