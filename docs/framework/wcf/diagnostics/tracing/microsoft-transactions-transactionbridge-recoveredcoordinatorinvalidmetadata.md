# <a name="microsofttransactionstransactionbridgerecoveredcoordinatorinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata
Odwołanie do punktu końcowego w pozycji dziennika koordynator odzyskiwania zawiera nieprawidłowe lub niezgodne metadane.  
  
## <a name="description"></a>Opis  
 System nie może utworzyć serwer proxy dla rejestracji koordynatora odzyskane.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Jeśli problem będzie się powtarzać, skontaktuj się z firmy Microsoft.
