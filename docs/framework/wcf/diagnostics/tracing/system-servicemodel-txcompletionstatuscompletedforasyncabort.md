# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Transakcja określona dla określonej operacji została wykonana wskutek asynchronicznego przerwania.  
  
## <a name="description"></a>Opis  
 Bieżąca transakcja została przerwana z powodu innego uczestnika głosowanie w celu przerwania, limity czasu wystąpienia, lub inny błąd wewnątrz uczestnika transakcji.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Jeśli ta przerwania jest nieoczekiwany, sprawdź wszystkie dzienniki systemu, aby określić przyczynę rzeczywistych w celu przerwania.
