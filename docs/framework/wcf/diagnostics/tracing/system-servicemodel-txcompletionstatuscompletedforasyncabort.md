# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Określona transakcja dla określonej operacji została ukończona z powodu asynchronicznego przerwania.  
  
## <a name="description"></a>Opis  
 Bieżąca transakcja została przerwana z powodu poprawnego głosu uczestnika na przerwanie, przekroczenie limitu czasu lub inny błąd wewnątrz uczestnika transakcji.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Jeśli to przerwanie jest nieoczekiwane, sprawdź wszystkie dzienniki systemu, aby ustalić rzeczywistą przyczynę przerwania.
