# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Znaleziono istniejący niezgodny menedżer transportu dla określonego identyfikatora URI.  
  
## <a name="description"></a>Opis  
 Ślad informacyjną wskazuje, że pasujące rejestracji został znaleziony dla odbiornika otwierany jednak istniejącego menedżera transportu jest niezgodny. Odbiornik będzie podejmować próby przy użyciu innego menedżera transportu i zgłosić wyjątek, jeśli nie powiedzie się.
