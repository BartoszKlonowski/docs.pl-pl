# <a name="microsofttransactionstransactionbridgeparticipantrecovered"></a>Microsoft.Transactions.TransactionBridge.ParticipantRecovered
Rejestracja uczestnika została odzyskana z dziennika Menedżera transakcji.  
  
## <a name="description"></a>Opis  
 Śledzenie, gdy lokalny Menedżer transakcji pomyślnie wykonał odzyskiwanie dla danej rejestracji podrzędnego uczestnika.
