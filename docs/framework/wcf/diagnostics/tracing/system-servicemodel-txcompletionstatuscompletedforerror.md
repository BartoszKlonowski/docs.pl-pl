# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Transakcja określona dla określonej operacji została wykonana wskutek wystąpienia nieobsłużonego wyjątku.  
  
## <a name="description"></a>Opis  
 Śledzone, gdy wystąpi błąd podczas próby wykonania bieżącej transakcji. Dzieje się to przed odpowiedzi lub błędów są wysyłane do obiektu wywołującego.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy w komunikacie śledzonych komunikat o wyjątku oraz dowolne elementy przydatnych wyników.
