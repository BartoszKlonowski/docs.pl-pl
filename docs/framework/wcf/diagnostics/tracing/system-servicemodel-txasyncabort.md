# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
Określona transakcja została przerwana asynchronicznie.  
  
## <a name="description"></a>Opis  
 Bieżąca transakcja została przerwana z powodu poprawnego głosu uczestnika na przerwanie, przekroczenie limitu czasu lub inny błąd wewnątrz uczestnika transakcji.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź wszystkie dzienniki systemu w przypadku nieoczekiwanego przerwania, aby określić rzeczywistą przyczynę przerwania.
