# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
Określonej transakcji asynchronicznie zostało przerwane.  
  
## <a name="description"></a>Opis  
 Bieżąca transakcja została przerwana z powodu innej uczestnika oddanie głosu na przerwanie, limity czasu wystąpienia lub inny błąd wewnątrz uczestnika transakcji.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź wszystkie dzienniki systemu, jeśli to przerwania jest nieoczekiwany w celu określenia rzeczywistego powód przerwania.
