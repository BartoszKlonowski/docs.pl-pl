# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
Określona transakcja została przerwana asynchronicznie.  
  
## <a name="description"></a>Opis  
 Bieżąca transakcja została przerwana z powodu innego uczestnika głosowanie w celu przerwania, limity czasu wystąpienia, lub inny błąd wewnątrz uczestnika transakcji.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź wszystkie dzienniki systemu, jeśli to przerwania jest nieoczekiwane, aby określić przyczynę rzeczywistych w celu przerwania.
