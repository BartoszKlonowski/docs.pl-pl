# <a name="systemservicemodelchannelsprematuredatagrameof"></a>System.ServiceModel.Channels.PrematureDatagramEof
System.ServiceModel.Channels.PrematureDatagramEof  
  
## <a name="description"></a>Opis  
 Odebrano komunikat o wartości null (sygnalizowanie koniec kanału) z kanału datagramu, ale kanał nadal jest w stanie otwartym. Wskazuje to na błąd w kanale datagramu, a pętla odbioru demultipleksera została przedwcześnie zakończona.
