# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Opis  
 Osiągnięto jeden ograniczania MaxConcurrentCalls/wystąpień/sesje i wiadomości są umieszczane w kolejce.  
  
 Ślad umożliwia dostosowanie wartości ograniczania na podstawie obciążenia produkcyjne oczekiwanymi i rzeczywistymi.
