# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached

System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Opis  

 Jedna z przepustnicy MaxConcurrentCalls/Instances/Sessions została osiągnięta, a komunikat jest umieszczany w kolejce.  
  
 Za pomocą tego śladu można dostosować wartości ograniczania na podstawie przewidywanych i rzeczywistych obciążeń produkcyjnych.
