# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC

Określona transakcja została ukończona, gdy sesja została zamknięta.  
  
## <a name="description"></a>Opis  

 Śledzony, gdy bieżąca aktywna sesja została zamknięta i `TransactionAutoCompleteOnSessionClose` ma ustawioną wartość `true` .
