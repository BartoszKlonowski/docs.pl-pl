# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
Określonej transakcji zostało ukończone, gdy sesja została zamknięta.  
  
## <a name="description"></a>Opis  
 Śledzone podczas bieżącej aktywnej sesji zostało zamknięte i `TransactionAutoCompleteOnSessionClose` ustawiono `true`.
