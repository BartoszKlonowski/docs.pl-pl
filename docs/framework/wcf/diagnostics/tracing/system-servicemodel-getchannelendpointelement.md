# <a name="systemservicemodelgetchannelendpointelement"></a>System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## <a name="description"></a>Opis  
 Ta śledzenia jest emitowane przy konstruowaniu fabryki kanałów, na poziomie informacji. Zawiera on opis punktu końcowego, który klient rozmawia (na przykład adresów zdalnych, powiązanie, Nazwa kontraktu).
