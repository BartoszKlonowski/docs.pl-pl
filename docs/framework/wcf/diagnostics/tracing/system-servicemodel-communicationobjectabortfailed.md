# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Nie można przerwać obiektu komunikacji.  
  
## <a name="description"></a>Opis  
 Nie można przerwać, ponieważ wystąpił wyjątek podczas wykonywania metody Abort obiektu komunikacji. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
