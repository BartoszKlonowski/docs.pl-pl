# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Przerwanie obiektu komunikacyjnego nie powiodło się.  
  
## <a name="description"></a>Opis  
 Przerwanie obiektu komunikacyjnego nie powiodło się, ponieważ wystąpił wyjątek podczas wykonywania metody Abort. Inny ślad w pobliżu może zawierać szczegółowe informacje o wyjątku.
