# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Obiektu komunikacyjnego nie można przerwać.  
  
## <a name="description"></a>Opis  
 Obiektu komunikacyjnego nie można przerwać, ponieważ wystąpił wyjątek podczas wykonywania metody jego przerwanie. Z śledzenia mogą zawierać szczegółowe informacje o wyjątku.
