# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Nie zaakceptowano sąsiedniego elementu równorzędnego.  
  
## <a name="description"></a>Opis  
 Ślad występuje, gdy połączenie sąsiada przetwarzania. Możliwe przyczyny niezgodnych ustawień, przekroczono limit czasu, sąsiada nie może akceptować połączenia lub przejściowe problemy z siecią.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy ustawienia zabezpieczeń niedopasowane, przekroczono limit czasu niemożność sąsiada akceptuje połączenia lub przejściowe problemy z siecią.
