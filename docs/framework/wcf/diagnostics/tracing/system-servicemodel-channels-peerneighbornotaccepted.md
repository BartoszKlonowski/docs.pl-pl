# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Nie zaakceptowano sąsiadujących elementów równorzędnych.  
  
## <a name="description"></a>Opis  
 Ślad występuje, gdy połączenie sąsiada przetwarzania. Możliwe przyczyny niezgodnych ustawień, przekroczono limit czasu, sąsiada, które nie akceptują połączenia lub problemy przejściowe problemy z siecią.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy ustawienia zabezpieczeń niezgodne, przekroczono limit czasu sąsiada brakiem, aby akceptować połączenia lub przejściowe problemy z siecią.
