# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Sąsiedni element równorzędny nie został zaakceptowany.  
  
## <a name="description"></a>Opis  
 Ten ślad występuje podczas przetwarzania połączenia z sąsiadem. Możliwe przyczyny to: niezgodne ustawienia zabezpieczeń, Przekroczono limit czasu, sąsiada, który nie może akceptować połączenia lub przejściowych problemów z siecią.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy istnieją niezgodne ustawienia zabezpieczeń, Przekroczono limit czasu, brak możliwości zaakceptowania połączenia lub przejściowych problemów z siecią.
