# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
Nie można wykonać operacji nawiązywania połączeń TCP.  
  
## <a name="description"></a>Opis  
 To ostrzeżenie śledzenia poziomu oznacza błąd do nawiązania połączenia z punktem końcowym protokołu TCP. Może to się zdarzyć, jeśli zdalny punkt końcowy nie odpowiada na danego adresu IP i portu. Ślad można zignorować, jeśli kolejne próbuje połączyć się powieść adresów (np. adresy IPv4 lub IPv6, lub inne adresy IP reprezentujący nazwę danego hosta) innych prawidłowymi adresami IP. Wyjątek w obrębie tego śledzenia mogą zawierać dodatkowe informacje o tym błędzie.
