# <a name="systemservicemodeltxsourcetxscoperequirediscreatenewtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsCreateNewTransaction

Określona transakcja dla określonej operacji została nowo utworzona.  
  
## <a name="description"></a>Opis  

 Śledzone, jeśli transakcja udostępniona dla kodu użytkownika została nowo utworzona przez infrastrukturę.
