# <a name="systemservicemodeltxsourcetxscoperequirediscreatenewtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsCreateNewTransaction
Transakcja określona dla określonej operacji była nowo utworzona.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli transakcja udostępnione do kodu użytkownika była nowo utworzona przez infrastrukturę.
