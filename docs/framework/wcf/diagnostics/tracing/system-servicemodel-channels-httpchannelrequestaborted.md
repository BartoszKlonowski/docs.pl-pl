# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Nie można wysłać komunikatu żądania za pośrednictwem protokołu HTTP.  
  
## <a name="description"></a>Opis  
 Tego poziomu śledzenia ostrzeżenie wskazuje, że żądanie kanału HTTP zostało przerwane. Dodatkowe informacje w śledzeniu ujawni przyczynę lub komunikat przekazany do metody przerwania.
