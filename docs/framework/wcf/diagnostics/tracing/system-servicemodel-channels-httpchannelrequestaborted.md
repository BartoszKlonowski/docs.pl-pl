# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Nie można wysłać komunikatu żądania za pośrednictwem protokołu HTTP.  
  
## <a name="description"></a>Opis  
 Tego poziomu śledzenia ostrzeżenie wskazuje, czy żądanie kanał HTTP zostało przerwane. Dodatkowe informacje w tym śladzie wyświetli przyczyny lub komunikat przekazywany do metody przerwania.
