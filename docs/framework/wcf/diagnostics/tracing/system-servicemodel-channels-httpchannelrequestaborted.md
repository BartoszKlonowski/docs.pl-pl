# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Nie można wysłać komunikatu żądania za pośrednictwem protokołu HTTP.  
  
## <a name="description"></a>Opis  
 Ten ślad poziomu ostrzeżeń wskazuje, że żądanie kanału HTTP zostało przerwane. Dodatkowe informacje w tym śledzeniu ujawniają przyczynę lub komunikat przesłany do metody Abort.
