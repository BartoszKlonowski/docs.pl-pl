# <a name="systemservicemodelchannelspeerfloodedmessagenotmatched"></a>System.ServiceModel.Channels.PeerFloodedMessageNotMatched
Węzeł równorzędny odebrał komunikat, które nie pasuje do żadnego lokalnego kanału.  
  
## <a name="description"></a>Opis  
 Ślad występuje podczas przetwarzania komunikatu przychodzącego powódź.
