# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
Transakcja określona dla określonej operacji została wykonana z powodu `OperationBehaviorAttribute.TransactionAutoComplete` element członkowski jest ustawiany `true`.  
  
## <a name="description"></a>Opis  
 Śledzone, gdy infrastruktura automatycznie wykonuje transakcji w imieniu użytkownika.
