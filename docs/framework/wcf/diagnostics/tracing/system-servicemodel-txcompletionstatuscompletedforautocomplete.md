# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete

Określona transakcja dla określonej operacji została ukończona z powodu `OperationBehaviorAttribute.TransactionAutoComplete` Ustawienia elementu członkowskiego `true` .  
  
## <a name="description"></a>Opis  

 Śledzenie, gdy infrastruktura automatycznie kończy transakcję w imieniu użytkownika.
