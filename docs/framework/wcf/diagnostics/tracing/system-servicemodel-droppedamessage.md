# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Opis  
 Odebrano komunikat odpowiedzi, ale Brak oczekujących żądań oczekujących na ten komunikat. Wiadomość została utracona przez środowisko wykonawcze Windows Communication Foundation (WCF).  
  
 Ślad może służyć do debugowania utracone wiadomości.
