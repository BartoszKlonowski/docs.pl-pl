# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Opis  
 Otrzymano komunikat odpowiedzi, ale brak zaległych żądań oczekujących dla tej wiadomości. Komunikat jest usuwane przez środowisko wykonawcze systemu Windows Communication Foundation (WCF).  
  
 Ślad może służyć do debugowania utratą komunikatów.
