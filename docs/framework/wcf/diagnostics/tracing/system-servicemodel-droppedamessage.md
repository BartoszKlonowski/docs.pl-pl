# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Opis  
 Odebrano komunikat odpowiedzi, ale nie ma oczekujących żądań oczekujących na tę wiadomość. Komunikat jest usuwany przez środowisko uruchomieniowe Windows Communication Foundation (WCF).  
  
 Ten ślad może służyć do debugowania utraconych wiadomości.
