# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfailed"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
Usługa protokołu WS-AT nie może wysłać komunikatu rejestracji do jego koordynatora  
  
## <a name="description"></a>Opis  
 Śledzony, jeśli lokalny element TransactionManager nie może zarejestrować się w jego nadrzędnym, ponieważ nie ma możliwości wysłania wiadomości.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź komunikat śledzenia wyjątku powodującego błąd wysyłania komunikatu
