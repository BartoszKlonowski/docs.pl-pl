# <a name="microsofttransactionstransactionbridgecoordinatorrecovered"></a>Microsoft.Transactions.TransactionBridge.CoordinatorRecovered

Funkcja rejestracji koordynatora została odzyskana z dziennika Menedżera transakcji.  
  
## <a name="description"></a>Opis  

 Śledzony, gdy lokalny Menedżer transakcji pomyślnie wykonał odzyskiwanie dla danej rejestracji z koordynatorem wyższego poziomu.
