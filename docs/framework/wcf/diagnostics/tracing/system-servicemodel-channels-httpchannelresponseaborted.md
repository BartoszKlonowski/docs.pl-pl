# <a name="systemservicemodelchannelshttpchannelresponseaborted"></a>System.ServiceModel.Channels.HttpChannelResponseAborted
Nie można wysłać komunikatu odpowiedzi przez HTTP.  
  
## <a name="description"></a>Opis  
 Tego poziomu śledzenia ostrzeżenie wskazuje, że odpowiedź kanału HTTP zostało przerwane. Dodatkowe informacje w śledzeniu ujawni przyczynę lub komunikat przekazany do metody przerwania.
