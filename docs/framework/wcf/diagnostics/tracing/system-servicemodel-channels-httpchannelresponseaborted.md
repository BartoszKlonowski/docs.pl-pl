# <a name="systemservicemodelchannelshttpchannelresponseaborted"></a>System.ServiceModel.Channels.HttpChannelResponseAborted
Nie można wysłać komunikatu odpowiedzi za pośrednictwem protokołu HTTP.  
  
## <a name="description"></a>Opis  
 Tego poziomu śledzenia ostrzeżenie wskazuje, czy odpowiedź HTTP kanału zostało przerwane. Dodatkowe informacje w tym śladzie wyświetli przyczyny lub komunikat przekazywany do metody przerwania.
