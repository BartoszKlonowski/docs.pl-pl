# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Próba połączenia się z punktem końcowym nazwanego potoku nie powiodło się. Podejmowana jest kolejna przed upływem określonego limitu czasu.  
  
## <a name="description"></a>Opis  
 Ślad informacyjny oznacza błąd nawiązywania połączenia z punktem końcowym nazwanego potoku. Może to się zdarzyć, jeśli punktem końcowym nazwanego potoku nie zostanie znaleziony lub jest zajęty. Dodatkowe prób, oddzielonych krótkich ilość czasu, aż do znalezienia właściwego konta lub OpenTimeout wygaśnie.
