# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Wymagany jest certyfikat klienta. Nie znaleziono certyfikatu w żądaniu.  
  
## <a name="description"></a>Opis  
 Ślad wskazuje, czy odbiornik HTTPS odebrane żądanie HTTPS, który nie został skojarzony z certyfikatem klienta. Ponieważ odbiornik został skonfigurowany, aby wymagać certyfikaty klienta dla wszystkich żądań HTTPS, odbiornik nie można zweryfikować autentyczności klienta.
