# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Wymagany jest certyfikat klienta. W żądaniu nie znaleziono żadnego certyfikatu.  
  
## <a name="description"></a>Opis  
 Ten ślad wskazuje, że odbiornik protokołu HTTPS otrzymał żądanie HTTPS, które nie zostało skojarzone z certyfikatem klienta. Ponieważ odbiornik został skonfigurowany tak, aby wymagał certyfikatów klienta na wszystkich żądaniach HTTPS, odbiornik nie mógł zweryfikować autentyczności klienta.
