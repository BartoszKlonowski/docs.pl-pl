# <a name="systemservicemodelchannelspeerfloodedmessagenotpropagated"></a>System.ServiceModel.Channels.PeerFloodedMessageNotPropagated

Węzeł równorzędny odebrał komunikat, ale nie jest propagowany do sąsiadów.  
  
## <a name="description"></a>Opis  

 Ten ślad występuje podczas przetwarzania przychodzącego komunikatu o zalewania.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  

 Sprawdź, czy istnieje filtr propagacji komunikatów.
