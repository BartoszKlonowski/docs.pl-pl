# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfaulted"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFaulted
Usługa protokołu WS-AT odebrał błąd od koordynatora w odpowiedzi na komunikat rejestrowania.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli lokalny Menedżer transakcji nie można zarejestrować w usłudze jego wyższego poziomu Menedżera transakcji z powodu błędu, do którego są zwracane.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Sprawdź, czy komunikat śledzenia błędu zwrócony.
