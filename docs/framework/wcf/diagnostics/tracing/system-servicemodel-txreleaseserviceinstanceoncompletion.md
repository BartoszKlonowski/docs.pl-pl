# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
Wystąpienie usługi zostało zwolnione po zakończeniu transakcji "{0}" ponieważ ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute została ustawiona na wartość true.  
  
## <a name="description"></a>Opis  
 Śledzone, gdy bieżące wystąpienie usługi jest wydane lub usunięte z powodu zakończenia to bieżąca aktywna transakcja ReleaseServiceInstanceOnTransactionComplete jest ustawiona na `true`.
