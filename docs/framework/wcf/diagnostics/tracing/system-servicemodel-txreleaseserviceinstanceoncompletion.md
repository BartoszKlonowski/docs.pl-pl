# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion

Wystąpienie usługi zostało wydane po zakończeniu transakcji " {0} ", ponieważ element ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute został ustawiony na wartość true.  
  
## <a name="description"></a>Opis  

 Śledzone po wydaniu lub usunięciu bieżącego wystąpienia usługi ze względu na zakończenie bieżącej aktywnej transakcji, a ReleaseServiceInstanceOnTransactionComplete ma ustawioną wartość `true` .
