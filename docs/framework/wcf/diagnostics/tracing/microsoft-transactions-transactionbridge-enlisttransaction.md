# <a name="microsofttransactionstransactionbridgeenlisttransaction"></a>Microsoft.Transactions.TransactionBridge.EnlistTransaction
Usługa protokołu WS-Atomic Transaction została pomyślnie zarejestrowana w transakcji, która używa dostępnego kontekstu koordynacji.  
  
## <a name="description"></a>Opis  
 Transakcja dla ruchu przychodzącego zostało pomyślnie przetworzone.
