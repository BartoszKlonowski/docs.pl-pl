# <a name="end-to-end-tracing-scenarios"></a>Scenariusze kompleksowego śledzenia
Ta sekcja zawiera tematy opisujące różne scenariusze dotyczące korzystania ze śledzenia.  
  
 Śledzenie działań jest implementowane dla scenariuszy odpowiedzi/odpowiedzi, wielokierunkowych i dupleksowych przy użyciu następujących elementów:  
  
- Synchroniczne lub asynchroniczne  
  
- BasicHttpBinding, netNamedPipeBinding, netTcpBinding, wsHttpBinding lub Msmqbinding  
  
- Usługa COM  
  
- Propagacja identyfikatora działania użytkownika
