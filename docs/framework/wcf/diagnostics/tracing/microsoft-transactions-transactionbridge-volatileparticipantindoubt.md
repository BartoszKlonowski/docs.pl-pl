# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
Usługa protokołu WS-AT odebrała przygotowany lub powtarzający się komunikat od nierozpoznanego uczestnika trwałego. Do uczestnika został zwrócony błąd, deklaruje, że wynik transakcji jest wątpliwy.  
  
## <a name="description"></a>Opis  
 Śledzony, gdy lokalny Menedżer transakcji odbiera przygotowany lub powtarzający się komunikat z nietrwałej rejestracji, że już został zapomniany.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Zbadaj potencjalne przyczyny późnych komunikatów pochodzących od uczestnika nietrwałego.
