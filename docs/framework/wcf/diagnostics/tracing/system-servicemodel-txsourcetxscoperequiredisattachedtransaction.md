Pobrano określoną transakcję dla określonej operacji z transakcji InstanceContext.

 Śledzone, jeśli transakcja udostępniona dla kodu użytkownika została pozostawiona za pomocą metody Autouzupełnianie = false wywołanej przed tym wywołaniem.
