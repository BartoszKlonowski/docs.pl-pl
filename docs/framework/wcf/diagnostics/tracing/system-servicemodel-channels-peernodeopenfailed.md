# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Próba otwarcia połączenia sąsiada nie powiodła się.  
  
## <a name="description"></a>Opis  
 Ślad ten występuje po podjęciu próby ustanowienia połączenia sąsiada. Możliwe przyczyny mogą być następujące: nieodpowiadający program rozpoznawania nazw, nieprawidłowe ustawienie powiązania lub przejściowe awarie sieci.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że oba węzły mają prawidłowe adresy i zostały odpowiednio skonfigurowane.
