# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Podjęto próbę otwarcia połączenia sąsiada nie powiodło się.  
  
## <a name="description"></a>Opis  
 Ślad występuje po próby nawiązania połączenia sąsiadów. Możliwe przyczyny mogą obejmować odpowiadać rozpoznawania nazw, ustawienie nieprawidłowe powiązanie lub awarie przejściowe problemy z siecią.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że oba węzły mają prawidłowe adresy i prawidłowo skonfigurowany.
