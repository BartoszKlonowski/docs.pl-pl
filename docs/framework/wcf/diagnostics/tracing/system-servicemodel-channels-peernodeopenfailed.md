# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Próba otwarcia połączenia z elementem sąsiednim nie powiodło się.  
  
## <a name="description"></a>Opis  
 Ślad występuje po próby nawiązania połączenia z elementem sąsiednim. Możliwe przyczyny mogą obejmować odpowiadać program rozpoznawania nazw, ustawienie nieprawidłowe powiązanie lub awarie sieci.  
  
## <a name="troubleshooting"></a>Rozwiązywanie problemów  
 Upewnij się, że oba węzły mają prawidłowe adresy i prawidłowo skonfigurowany.
