# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Określonej transakcji dla określonej operacji zostało zakończone z powodu wywołania metody SetTransactionComplete.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli użytkownik jawnie zostanie podjęta decyzja ukończyć bieżącej transakcji za pośrednictwem elementu OperationContext.
