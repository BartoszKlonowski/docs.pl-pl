# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Określona transakcja dla określonej operacji została ukończona z powodu wywołania metody SetTransactionComplete.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli użytkownik jawnie zdecydował się ukończyć bieżącą transakcję za pomocą OperationContext.
