# <a name="systemservicemodelactivationwebhostserviceactivated"></a>System.ServiceModel.Activation.WebHostServiceActivated

System.ServiceModel.Activation.WebHostServiceActivated  
  
## <a name="description"></a>Opis  

 Występuje, gdy usługa hostowana w Internet Information Services (IIS) lub usługa aktywacji procesów systemu Windows (WAS) jest w pełni aktywowana i rozpoczyna nasłuchiwanie komunikatów.
