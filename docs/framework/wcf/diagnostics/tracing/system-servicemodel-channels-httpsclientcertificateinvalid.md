# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certyfikat klienta jest nieprawidłowy.  
  
## <a name="description"></a>Opis  
 Ślad wskazuje, że certyfikat podany przez klienta odnaleziono nieprawidłowy przez odbiornik HTTPS. Odbiornik HTTPS próby weryfikacji autentyczności klienta przy użyciu tego certyfikatu. Certyfikat może być nieprawidłowy, jeśli jego urząd certyfikacji nie jest rozpoznawany przez serwer obsługujący usługę.
