# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certyfikat klienta jest nieprawidłowy.  
  
## <a name="description"></a>Opis  
 Ślad wskazuje, że certyfikat dostarczony przez klienta została uznana za nieprawidłową przez odbiornik HTTPS. Odbiornik HTTPS próby sprawdzenie oryginalności klienta przy użyciu tego certyfikatu. Certyfikat może być nieprawidłowy, jeśli jego urzędu certyfikacji nie jest rozpoznawane przez serwer obsługujący usługę.
