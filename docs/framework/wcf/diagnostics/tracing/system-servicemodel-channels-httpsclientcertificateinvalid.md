# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certyfikat klienta jest nieprawidłowy.  
  
## <a name="description"></a>Opis  
 Ten ślad wskazuje, że certyfikat dostarczony przez klienta okazał się nieprawidłowy przez odbiornik HTTPS. Odbiornik HTTPS podjął próbę zweryfikowania autentyczności klienta przy użyciu tego certyfikatu. Certyfikat może być nieprawidłowy, jeśli jego urząd certyfikacji nie jest rozpoznawany przez serwer hostujący usługę.
