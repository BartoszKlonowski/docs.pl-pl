# <a name="msmqintegrationbindingelement"></a>MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## <a name="syntax"></a>Składnia  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa MsmqIntegrationBindingElement nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa MsmqIntegrationBindingElement ma następującą właściwość:  
  
### <a name="serializationformat"></a>Wartości właściwości SerializationFormat  
 Typ danych: ciąg  
  
 Dostęp typu: tylko do odczytu  
  
 Format używany przez powiązanie do serializacji komunikatów.
