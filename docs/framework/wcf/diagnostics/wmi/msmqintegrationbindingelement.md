# <a name="msmqintegrationbindingelement"></a>MsmqIntegrationBindingElement
MsmqIntegrationBindingElement

```csharp  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa MsmqIntegrationBindingElement nie definiuje żadnych metod.

### <a name="serializationformat"></a>serializationFormat  
 Typ danych: ciąg

 Format używany przez powiązanie do serializacji wiadomości.
