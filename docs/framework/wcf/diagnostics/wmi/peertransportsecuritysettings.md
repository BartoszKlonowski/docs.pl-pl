# <a name="peertransportsecuritysettings"></a>PeerTransportSecuritySettings

PeerTransportSecuritySettings  
  
## <a name="syntax"></a>Składnia  
  
```csharp
class PeerTransportSecuritySettings  
{  
  string CredentialType;  
};  
```  
  
## <a name="methods"></a>Metody  

 Klasa PeerTransportSecuritySettings nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  

 Klasa PeerTransportSecuritySettings ma następującą właściwość:  
  
### <a name="credentialtype"></a>CredentialType  

 Typ danych: ciąg  
  
 Typ dostępu: tylko do odczytu  
  
 Typ poświadczenia transportu dla elementu zabezpieczeń elementów równorzędnych.
