# <a name="transactionflowattribute"></a>TransactionFlowAttribute
TransactionFlowAttribute  
  
## <a name="syntax"></a>Składnia  
  
```csharp
class TransactionFlowAttribute : Behavior  
{  
  string TransactionFlowOption;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa TransactionFlowAttribute nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa TransactionFlowAttribute ma następującą właściwość:  
  
### <a name="transactionflowoption"></a>Parametru TransactionFlowOption  
 Typ danych: ciąg  
  
 Dostęp do typu: tylko do odczytu  
  
 Wskazuje, czy przepływ transakcji.

|---------|-----------------------------------|
