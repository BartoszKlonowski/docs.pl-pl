# <a name="messageencodingbindingelement"></a>MessageEncodingBindingElement
MessageEncodingBindingElement  
  
## <a name="syntax"></a>Składnia  
 `class MessageEncodingBindingElement : BindingElement`  
  
 `{`  
  
 `string MessageVersion;`  
  
 `};`  
  
## <a name="methods"></a>Metody  
 Klasa element MessageEncodingBindingElement nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa element MessageEncodingBindingElement ma następującą właściwość:  
  
### <a name="messageversion"></a>Element MessageVersion  
 Typ danych: ciąg  
  
 Dostęp typu: tylko do odczytu  
  
 Wersja SOAP komunikatów wysyłanych za pomocą tego powiązania.
