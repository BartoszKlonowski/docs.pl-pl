# <a name="compositeduplexbindingelement"></a>CompositeDuplexBindingElement
CompositeDuplexBindingElement  
  
## <a name="syntax"></a>Składnia  
  
```  
class CompositeDuplexBindingElement : BindingElement  
{  
  string ClientBaseAddress;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa CompositeDuplexBindingElement nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa CompositeDuplexBindingElement ma następującą właściwość:  
  
### <a name="clientbaseaddress"></a>ClientBaseAddress  
 Typ danych: ciąg  
  
 Dostęp typu: tylko do odczytu  
  
 Adres podstawowy klienta.
