# <a name="peerresolverbindingelement"></a>PeerResolverBindingElement
PeerResolverBindingElement  
  
## <a name="syntax"></a>Składnia  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa element PeerResolverBindingElement nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa element PeerResolverBindingElement ma następującą właściwość:  
  
### <a name="referralpolicy"></a>ReferralPolicy  
 Typ danych: ciąg  
  
 Dostęp typu: tylko do odczytu  
  
 Określa, jak referencje są współużytkowane przez elementy równorzędne.
