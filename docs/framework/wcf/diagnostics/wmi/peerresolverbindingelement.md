# <a name="peerresolverbindingelement"></a>PeerResolverBindingElement

PeerResolverBindingElement  
  
## <a name="syntax"></a>Składnia  
  
```csharp
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## <a name="methods"></a>Metody  

 Klasa PeerResolverBindingElement nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  

 Klasa PeerResolverBindingElement ma następującą właściwość:  
  
### <a name="referralpolicy"></a>ReferralPolicy  

 Typ danych: ciąg  
  
 Typ dostępu: tylko do odczytu  
  
 Określa, jak odwołania są współużytkowane przez elementy równorzędne.
