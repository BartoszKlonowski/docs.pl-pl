## <a name="syntax"></a>Składnia  
  
```csharp
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>Parametry  
  
### <a name="operation"></a>Operacja  

 [in] ciąg
