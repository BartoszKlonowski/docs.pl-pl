# <a name="xmlserializeroperationbehavior"></a>XmlSerializerOperationBehavior
XmlSerializerOperationBehavior  
  
## <a name="syntax"></a>Składnia  
  
```csharp
class XmlSerializerOperationBehavior : Behavior  
{  
  string Style;  
  string Use;  
};  
```  
  
## <a name="methods"></a>Metody  
 Klasa XmlSerializerOperationBehavior nie definiuje żadnych metod.  
  
## <a name="properties"></a>Właściwości  
 Klasa XmlSerializerOperationBehavior ma następujące właściwości:  
  
### <a name="style"></a>Styl  
 Typ danych: ciąg  
  
 Dostęp do typu: tylko do odczytu  
  
 Określa styl komunikatu protokołu SOAP.  
  
### <a name="use"></a>Zastosowanie  
 Typ danych: ciąg  
  
 Dostęp do typu: tylko do odczytu  
  
 Określa styl kodowania SOAP.

|---------|-----------------------------------|  
|Przestrzeń nazw|Zdefiniowane w root\ServiceModel|
