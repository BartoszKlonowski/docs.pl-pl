# <a name="how-to-implement-a-windows-communication-foundation-service-contract"></a>Instrukcje: Implementowanie kontraktu usługi WCF (Windows Communication Foundation)

Jest to drugi z sześciu zadań podrzędnych, wymagane do utworzenia podstawowej usługi Windows Communication Foundation (WCF) i klienta, który można wywołać usługi. Aby uzyskać omówienie wszystkich sześciu zadań podrzędnych, zobacz [Samouczek wprowadzający](../../../docs/framework/wcf/getting-started-tutorial.md) tematu.

Następnym krokiem w tworzeniu aplikacji WCF jest zaimplementowanie interfejsu usługi. To związane z utworzeniem klasy o nazwie `CalculatorService` implementującej zdefiniowany przez użytkownika `ICalculator` interfejs...

## <a name="to-implement-a-wcf-service-contract"></a>Aby zaimplementować kontraktu usługi WCF

Otwórz plik Service1.cs lub Service1.vb i Dodaj następujący kod:

```csharp
using System;
using System.ServiceModel;

namespace GettingStartedLib
{
    public class CalculatorService : ICalculator
    {
        public double Add(double n1, double n2)
        {
            double result = n1 + n2;
            Console.WriteLine("Received Add({0},{1})", n1, n2);
            // Code added to write output to the console window.
            Console.WriteLine("Return: {0}", result);
            return result;
        }

        public double Subtract(double n1, double n2)
        {
            double result = n1 - n2;
            Console.WriteLine("Received Subtract({0},{1})", n1, n2);
            Console.WriteLine("Return: {0}", result);
            return result;
        }

        public double Multiply(double n1, double n2)
        {
            double result = n1 * n2;
            Console.WriteLine("Received Multiply({0},{1})", n1, n2);
            Console.WriteLine("Return: {0}", result);
            return result;
        }

        public double Divide(double n1, double n2)
        {
            double result = n1 / n2;
            Console.WriteLine("Received Divide({0},{1})", n1, n2);
            Console.WriteLine("Return: {0}", result);
            return result;
        }
    }
}
```

## <a name="example"></a>Przykład

Poniższy kod przedstawia obie interfejs który definiuje kontrakt i implementacja interfejsu.

```csharp
using System;
using System.ServiceModel;

namespace GettingStartedLib
{
    [ServiceContract(Namespace = "http://Microsoft.ServiceModel.Samples")]
    public interface ICalculator
    {
        [OperationContract]
        double Add(double n1, double n2);
        [OperationContract]
        double Subtract(double n1, double n2);
        [OperationContract]
        double Multiply(double n1, double n2);
        [OperationContract]
        double Divide(double n1, double n2);
    }
}
```
