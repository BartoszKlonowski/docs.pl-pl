# <a name="wcf-system-requirements"></a>Wymagania systemowe architektury WCF
Windows Communication Foundation (WCF) jest infrastruktury komunikacji, który służy do tworzenia aplikacji rozproszonych. Poniższa lista zawiera wymagania, które umożliwiają [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] do uruchomienia.  
  
## <a name="system-requirements"></a>Wymagania systemowe  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] jest instalowana domyślnie na [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] można również zainstalować na [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], lub [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Uwaga** funkcjonalność usługi kolejkowania komunikatów (MSMQ) [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] jest obsługiwana tylko w systemach [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], i [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
