 <span data-ttu-id="09363-110">Po dodaniu Instalatora do projektu w projekcie zostanie utworzona nowa klasa (która domyślnie nosi nazwę `ProjectInstaller` ), a w nim zostaną utworzone wystąpienia odpowiednich składników instalacyjnych.</span><span class="sxs-lookup"><span data-stu-id="09363-110">When you add an installer to your project, a new class (which, by default, is named `ProjectInstaller`) is created in the project, and instances of the appropriate installation components are created within it.</span></span> <span data-ttu-id="09363-111">Ta klasa działa jako punkt centralny dla wszystkich składników instalacji potrzebnych przez projekt.</span><span class="sxs-lookup"><span data-stu-id="09363-111">This class acts as a central point for all of the installation components your project needs.</span></span> <span data-ttu-id="09363-112">Jeśli na przykład dodasz drugą usługę do aplikacji i klikniesz link Dodaj Instalatora, nie zostanie utworzona druga Klasa Instalatora; Zamiast tego niezbędny dodatkowy składnik instalacyjny dla drugiej usługi jest dodawany do istniejącej klasy.</span><span class="sxs-lookup"><span data-stu-id="09363-112">For example, if you add a second service to your application and click the Add Installer link, a second installer class is not created; instead, the necessary additional installation component for the second service is added to the existing class.</span></span>  
  
 <span data-ttu-id="09363-113">Nie trzeba wykonywać żadnych specjalnych kodowania w ramach instalatorów, aby zapewnić prawidłowe działanie usług.</span><span class="sxs-lookup"><span data-stu-id="09363-113">You do not need to do any special coding within the installers to make your services install correctly.</span></span> <span data-ttu-id="09363-114">Jednak czasami trzeba zmodyfikować zawartość instalatorów, jeśli trzeba dodać specjalną funkcjonalność do procesu instalacji.</span><span class="sxs-lookup"><span data-stu-id="09363-114">However, you may occasionally need to modify the contents of the installers if you need to add special functionality to the installation process.</span></span>

1. <span data-ttu-id="09363-119">W **Eksplorator rozwiązań**dostęp do widoku **projektu** dla usługi, dla której chcesz dodać składnik instalacji.</span><span class="sxs-lookup"><span data-stu-id="09363-119">In **Solution Explorer**, access **Design** view for the service for which you want to add an installation component.</span></span>  
  
2. <span data-ttu-id="09363-120">Kliknij tło projektanta, aby wybrać samą usługę, a nie jej zawartość.</span><span class="sxs-lookup"><span data-stu-id="09363-120">Click the background of the designer to select the service itself, rather than any of its contents.</span></span>  
  
3. <span data-ttu-id="09363-121">Korzystając z projektanta w fokusie, kliknij prawym przyciskiem myszy, a następnie kliknij polecenie **Dodaj Instalatora**.</span><span class="sxs-lookup"><span data-stu-id="09363-121">With the designer in focus, right-click, and then click **Add Installer**.</span></span>  
  
     <span data-ttu-id="09363-122">Nowa klasa, `ProjectInstaller` i dwa składniki instalacyjne, <xref:System.ServiceProcess.ServiceProcessInstaller> i <xref:System.ServiceProcess.ServiceInstaller> , są dodawane do projektu, a wartości właściwości usługi są kopiowane do składników programu.</span><span class="sxs-lookup"><span data-stu-id="09363-122">A new class, `ProjectInstaller`, and two installation components, <xref:System.ServiceProcess.ServiceProcessInstaller> and <xref:System.ServiceProcess.ServiceInstaller>, are added to your project, and property values for the service are copied to the components.</span></span>  
  
4. <span data-ttu-id="09363-123">Kliknij <xref:System.ServiceProcess.ServiceInstaller> składnik i sprawdź, czy wartość <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> właściwości jest ustawiona na taką samą wartość jak <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> Właściwość w samej usłudze.</span><span class="sxs-lookup"><span data-stu-id="09363-123">Click the <xref:System.ServiceProcess.ServiceInstaller> component and verify that the value of the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property is set to the same value as the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property on the service itself.</span></span>  
  
5. <span data-ttu-id="09363-124">Aby określić, jak zostanie uruchomiona usługa, kliknij <xref:System.ServiceProcess.ServiceInstaller> składnik i ustaw <xref:System.ServiceProcess.ServiceInstaller.StartType%2A> Właściwość na odpowiednią wartość.</span><span class="sxs-lookup"><span data-stu-id="09363-124">To determine how your service will be started, click the <xref:System.ServiceProcess.ServiceInstaller> component and set the <xref:System.ServiceProcess.ServiceInstaller.StartType%2A> property to the appropriate value.</span></span>

6. <span data-ttu-id="09363-131">Aby określić kontekst zabezpieczeń, w którym zostanie uruchomiona usługa, kliknij <xref:System.ServiceProcess.ServiceProcessInstaller> składnik i ustaw odpowiednie wartości właściwości.</span><span class="sxs-lookup"><span data-stu-id="09363-131">To determine the security context in which your service will run, click the <xref:System.ServiceProcess.ServiceProcessInstaller> component and set the appropriate property values.</span></span> <span data-ttu-id="09363-132">Aby uzyskać więcej informacji, zobacz [How to: określanie kontekstu zabezpieczeń dla usług](how-to-specify-the-security-context-for-services.md).</span><span class="sxs-lookup"><span data-stu-id="09363-132">For more information, see [How to: Specify the Security Context for Services](how-to-specify-the-security-context-for-services.md).</span></span>  
  
7. <span data-ttu-id="09363-133">Zastąp wszystkie metody, dla których należy wykonać przetwarzanie niestandardowe.</span><span class="sxs-lookup"><span data-stu-id="09363-133">Override any methods for which you need to perform custom processing.</span></span>  
  
8. <span data-ttu-id="09363-134">Wykonaj kroki od 1 do 7 dla każdej dodatkowej usługi w projekcie.</span><span class="sxs-lookup"><span data-stu-id="09363-134">Perform steps 1 through 7 for each additional service in your project.</span></span>
