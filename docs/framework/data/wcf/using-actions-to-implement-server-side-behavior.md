## <a name="invoking-a-wcf-data-service-action"></a><span data-ttu-id="36ec3-163">Wywołanie akcji usług danych WCF</span><span class="sxs-lookup"><span data-stu-id="36ec3-163">Invoking a WCF Data Service Action</span></span>

 `http://MovieServer/MovieService.svc/Movies(1)/Rate`
  
 <span data-ttu-id="36ec3-169">Movies(1) określa filmu, którą chcesz nawiązać za oraz stawkę za określa szybkość akcji.</span><span class="sxs-lookup"><span data-stu-id="36ec3-169">Movies(1) specifies the movie you wish to rate and Rate specifies the Rate action.</span></span> <span data-ttu-id="36ec3-170">Rzeczywistej wartości ocena będzie w treści żądania HTTP, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="36ec3-170">The actual value of the rating will be in the body of the HTTP request as shown in the following example:</span></span>

```  
Content-Type: application/json   
Content-Length: 20   
Host: localhost:15238  
{   
   "rating": 4   
}  
```  
  
> [!WARNING]
> <span data-ttu-id="36ec3-171">Powyższy kod przykładowy będzie działać tylko w przypadku 5.2 usług danych WCF i później, który ma pomocy technicznej dla lekkiej serializacji JSON.</span><span class="sxs-lookup"><span data-stu-id="36ec3-171">The above sample code will work only with WCF Data Services 5.2 and later which has support for JSON light.</span></span> <span data-ttu-id="36ec3-172">Jeśli przy użyciu starszej wersji usługi danych WCF, należy określić właściwość json pełne content-type w następujący sposób: `application/json;odata=verbose`.</span><span class="sxs-lookup"><span data-stu-id="36ec3-172">If using an earlier version of WCF Data Services, you must specify the json verbose content-type as follows: `application/json;odata=verbose`.</span></span>

 <span data-ttu-id="36ec3-174">W powyższym fragmencie kodu `MoviesModel` klasy został wygenerowany przy użyciu programu Visual Studio Dodaj odwołanie do usługi z usługą danych programu WCF.</span><span class="sxs-lookup"><span data-stu-id="36ec3-174">In the code snippet above, the `MoviesModel` class was generated by using Visual Studio to Add Service Reference to a WCF Data Service.</span></span>
