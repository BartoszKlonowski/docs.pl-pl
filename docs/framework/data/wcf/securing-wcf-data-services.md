-   <span data-ttu-id="975ba-215">Biblioteka klienta zakłada, że protokoły używane w celu uzyskania dostępu do usługi danych zapewniają odpowiedni poziom zabezpieczeń.</span><span class="sxs-lookup"><span data-stu-id="975ba-215">The client library assumes that the protocols used to access the data service provide an appropriate level of security.</span></span>  
  
-   <span data-ttu-id="975ba-216">Biblioteka klienta używa wszystkich wartości domyślnych dla limitów czasu i opcji analizy podstawowych stosów transportu zapewnianych przez platformę.</span><span class="sxs-lookup"><span data-stu-id="975ba-216">The client library uses all the default values for timeouts and parsing options of the underlying platform-provided transport stacks.</span></span>

-   <span data-ttu-id="975ba-218">Biblioteka klienta nie implementuje żadnych mechanizmów dostępu między domenami.</span><span class="sxs-lookup"><span data-stu-id="975ba-218">The client library does not implement any cross domain access mechanisms.</span></span> <span data-ttu-id="975ba-219">W zamian opiera się na mechanizmach zapewnianych przez podstawowy stos HTTP.</span><span class="sxs-lookup"><span data-stu-id="975ba-219">Instead, it relies on the mechanisms provided by the underlying HTTP stack.</span></span>
