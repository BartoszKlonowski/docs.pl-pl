# <a name="query-examples"></a>Przykłady zapytań
Ta sekcja zawiera przykłady kodu języka Visual Basic i C# dla typowych [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] zapytania. Deweloperzy korzystający z programu Visual Studio można znaleźć więcej przykładów w przykładowe rozwiązanie dostępne w sekcji przykładów. Aby uzyskać więcej informacji, zobacz [przykłady](../../../../../../docs/framework/data/adonet/sql/linq/samples.md).  
  
> [!IMPORTANT]
>  *DB* jest często używana w przykładach kodu w [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] dokumentacji. *DB* zakłada, że wystąpienie *Northwind* klasy, która dziedziczy po elemencie <xref:System.Data.Linq.DataContext>.
