# <a name="data-types-and-functions"></a>Typy danych i funkcje
Tematy wymienione w poniższej tabeli opisano LINQ do SQL obsługę elementów członkowskich, konstrukcji i rzutowania środowisko uruchomieniowe języka wspólnego (CLR). Obsługiwane elementy Członkowskie oraz elementów składowych są dostępne do użycia w Twojej LINQ do zapytania SQL.  
  
 Nieobsługiwany element w tabeli oznacza, że CLR elementu członkowskiego, konstrukcja lub rzutowania nie może dokonywać translacji LINQ do SQL do wykonania na serwerze SQL. Może być nadal z nich korzystać w kodzie, ale muszą zostać obliczone przed zapytania jest translacja języka Transact-SQL lub po wyniki zostały pobrane z bazy danych.
