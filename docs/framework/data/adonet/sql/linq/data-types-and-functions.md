# <a name="data-types-and-functions"></a>Typy danych i funkcje
Tematy wymienione w poniższej tabeli opisano LINQ do SQL działu pomocy technicznej dla członków, konstrukcje i rzutowania środowisko uruchomieniowe języka wspólnego (CLR). Obsługiwane elementy członkowskie i konstrukcje są dostępne do użycia w Twojej zapytania LINQ to SQL.  
  
 Element nieobsługiwany w tabeli oznacza, że LINQ to SQL nie może dokonywać translacji CLR elementu członkowskiego, konstrukcja lub Rzutowanie do wykonania w programie SQL Server. Nadal mieć możliwość ich używać w kodzie, ale muszą być ocenione, zanim zapytania jest tłumaczony języka Transact-SQL lub po wyniki zostały pobrane z bazy danych.
