# <a name="clr-triggers"></a>Wyzwalacze CLR

Wyzwalacz jest specjalnym typem procedury składowanej, która jest uruchamiana automatycznie podczas wykonywania zdarzenia języka. Ze względu na Microsoft SQL Server integrację z .NET Framework środowiska uruchomieniowego języka wspólnego (CLR) można użyć dowolnego języka .NET Framework, aby utworzyć wyzwalacze środowiska CLR.  
  
Aby uzyskać więcej informacji, zobacz [Tworzenie wyzwalaczy CLR](/sql/relational-databases/triggers/create-clr-triggers).
