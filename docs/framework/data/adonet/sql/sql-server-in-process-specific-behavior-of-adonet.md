# <a name="sql-server-in-process-specific-behavior-of-adonet"></a>Zachowanie w procesie specyficzne dla serwera SQL ADO.NET
Istnieją cztery główne rozszerzenia funkcjonalne do ADO.NET, które są dostępne w przestrzeni nazw <xref:Microsoft.SqlServer.Server>, które są przeznaczone do użycia w procesie: <xref:Microsoft.SqlServer.Server.SqlContext>, <xref:Microsoft.SqlServer.Server.SqlPipe>, <xref:Microsoft.SqlServer.Server.SqlTriggerContext>i <xref:Microsoft.SqlServer.Server.SqlDataRecord>.  
  
 Aby uzyskać bardziej szczegółowe informacje, zapoznaj się z wersją usługi SQL Server Books Online dla używanej wersji SQL Server.
