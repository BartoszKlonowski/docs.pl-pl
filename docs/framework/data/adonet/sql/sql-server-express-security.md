## <a name="user-instances"></a><span data-ttu-id="9425e-114">Wystąpienia użytkownika</span><span class="sxs-lookup"><span data-stu-id="9425e-114">User Instances</span></span>  
 <span data-ttu-id="9425e-115">Wystąpienia użytkownika jest osobnego wystąpienia programu SQL Server Express aparatu bazy danych jest generowany przez nadrzędny wystąpienie programu SQL Server Express.</span><span class="sxs-lookup"><span data-stu-id="9425e-115">A user instance is a separate instance of the SQL Server Express database engine that is generated by a parent instance of SQL Server Express.</span></span> <span data-ttu-id="9425e-116">Podstawowym celem wystąpienia użytkownika jest umożliwienie użytkownicy korzystający z systemu Windows przy użyciu konta użytkownika uprawnień, poproś administratora systemu (`sysadmin`) uprawnienia w wystąpieniu programu SQL Server Express na komputerze lokalnym.</span><span class="sxs-lookup"><span data-stu-id="9425e-116">The primary goal of a user instance is to allow users who are running Windows under a least-privilege user account to have system administrator (`sysadmin`) privileges on the SQL Server Express instance on their local computer.</span></span> <span data-ttu-id="9425e-117">Wystąpienia użytkownika nie są przeznaczone dla użytkowników, którzy są administratorami systemu na swoich komputerach.</span><span class="sxs-lookup"><span data-stu-id="9425e-117">User instances are not intended for users who are system administrators on their own computers.</span></span>  
  
 <span data-ttu-id="9425e-118">Wystąpienia użytkownika jest generowany na podstawie podstawowego wystąpienia programu SQL Server lub SQL Server Express w imieniu użytkownika.</span><span class="sxs-lookup"><span data-stu-id="9425e-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="9425e-119">Jest uruchamiana jako proces użytkownika w kontekście zabezpieczeń systemu Windows użytkownika, nie jako usługa.</span><span class="sxs-lookup"><span data-stu-id="9425e-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="9425e-120">Logowania do programu SQL Server są niedozwolone; obsługiwane są tylko nazwy logowania systemu Windows.</span><span class="sxs-lookup"><span data-stu-id="9425e-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="9425e-121">Zapobiega to oprogramowanie wykonywania w wystąpieniu użytkownika z wprowadzania zmian całego systemu, które użytkownik nie będą miały uprawnienia do wprowadzania.</span><span class="sxs-lookup"><span data-stu-id="9425e-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="9425e-122">Wystąpienia użytkownika jest nazywane również wystąpienia podrzędnej lub klienta, a jest czasami określana za pomocą akronim RANU ("Uruchom jako zwykłego użytkownika").</span><span class="sxs-lookup"><span data-stu-id="9425e-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>  
  
 <span data-ttu-id="9425e-123">Każde wystąpienie użytkownika jest izolowane od jego wystąpienie nadrzędne i z innych wystąpień użytkownika uruchomionych na tym samym komputerze.</span><span class="sxs-lookup"><span data-stu-id="9425e-123">Each user instance is isolated from its parent instance and from other user instances running on the same computer.</span></span> <span data-ttu-id="9425e-124">Baza danych zainstalowana na wystąpieniach użytkowników są otwarte w trybie jednego użytkownika. wielu użytkowników nie można się z nimi łączyć.</span><span class="sxs-lookup"><span data-stu-id="9425e-124">Databases installed on user instances are opened in single-user mode only; multiple users cannot connect to them.</span></span> <span data-ttu-id="9425e-125">Funkcje replikacji, zapytania rozproszone i połączenia zdalne są wyłączone dla wystąpienia użytkownika.</span><span class="sxs-lookup"><span data-stu-id="9425e-125">Replication, distributed queries and remote connections are disabled for user instances.</span></span> <span data-ttu-id="9425e-126">Po podłączeniu do wystąpienia użytkownika, użytkownicy nie mają żadnych specjalnych uprawnień na wystąpienie programu SQL Server Express nadrzędne.</span><span class="sxs-lookup"><span data-stu-id="9425e-126">When connected to a user instance, users do not have any special privileges on the parent SQL Server Express instance.</span></span>
