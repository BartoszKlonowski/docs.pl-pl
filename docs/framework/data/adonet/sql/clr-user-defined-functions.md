# <a name="clr-user-defined-functions"></a>Zdefiniowane przez użytkownika funkcje CLR

Funkcje zdefiniowane przez użytkownika to procedury, które mogą przyjmować parametry, wykonywać obliczenia lub inne akcje i zwracać wynik. Funkcje zdefiniowane przez użytkownika można napisać w dowolnym języku programowania Microsoft .NET Framework, takim jak Microsoft Visual Basic .NET lub Microsoft Visual C#.  
  
 Aby uzyskać szczegółowe informacje, zobacz [CLR — funkcje zdefiniowane przez użytkownika](/sql/relational-databases/clr-integration-database-objects-user-defined-functions/clr-user-defined-functions).
