## <a name="application-role-features"></a><span data-ttu-id="83a01-109">Funkcje roli aplikacji</span><span class="sxs-lookup"><span data-stu-id="83a01-109">Application Role Features</span></span>  
 <span data-ttu-id="83a01-110">Role aplikacji mają następujące funkcje:</span><span class="sxs-lookup"><span data-stu-id="83a01-110">Application roles have the following features:</span></span>

-   <span data-ttu-id="83a01-113">Hasło musi być przechowywane na komputerze klienckim i dostarczane w czasie wykonywania. do roli aplikacji nie można aktywować z wewnątrz programu SQL Server.</span><span class="sxs-lookup"><span data-stu-id="83a01-113">The password must be stored on the client computer and supplied at run time; an application role cannot be activated from inside of SQL Server.</span></span>

-   <span data-ttu-id="83a01-118">Jeśli członek `sysadmin` stałej roli serwera aktywuje roli aplikacji, kontekst zabezpieczeń przełącza się do tej roli aplikacji na czas trwania połączenia.</span><span class="sxs-lookup"><span data-stu-id="83a01-118">If a member of the `sysadmin` fixed server role activates an application role, the security context switches to that of the application role for the duration of the connection.</span></span>

-   <span data-ttu-id="83a01-121">Wbudowane funkcje, które zwracają nazwy logowania, takie jak SYSTEM_USER, zwraca nazwę logowania, która wywołała roli aplikacji.</span><span class="sxs-lookup"><span data-stu-id="83a01-121">Built-in functions that return login names, such as SYSTEM_USER, return the name of the login that invoked the application role.</span></span> <span data-ttu-id="83a01-122">Wbudowane funkcje, które zwracają nazwy użytkownika bazy danych zwraca nazwę roli aplikacji.</span><span class="sxs-lookup"><span data-stu-id="83a01-122">Built-in functions that return database user names return the name of the application role.</span></span>
