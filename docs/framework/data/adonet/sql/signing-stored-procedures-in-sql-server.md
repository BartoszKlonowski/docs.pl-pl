 <span data-ttu-id="dc97f-109">Możesz można również utworzyć identyfikator logowania mapowany na ten sam certyfikat, a następnie przyznać wszelkich niezbędnych uprawnień na poziomie serwera do tej nazwy logowania lub Dodaj logowanie do co najmniej jednej z ról stałej.</span><span class="sxs-lookup"><span data-stu-id="dc97f-109">You can also create a login mapped to the same certificate, and then grant any necessary server-level permissions to that login, or add the login to one or more of the fixed server roles.</span></span> <span data-ttu-id="dc97f-110">Służy to umożliwienie `TRUSTWORTHY` bazy danych ustawienie dla scenariuszy, w których są wymagane uprawnienia na wyższym poziomie.</span><span class="sxs-lookup"><span data-stu-id="dc97f-110">This is designed to avoid enabling the `TRUSTWORTHY` database setting for scenarios in which higher level permissions are needed.</span></span>  
  
 <span data-ttu-id="dc97f-111">Podczas wykonywania procedury składowanej uprawnienia użytkownika certyfikatu i/lub logowania programu SQL Server łączy się z udostępnianych przez obiekt wywołujący.</span><span class="sxs-lookup"><span data-stu-id="dc97f-111">When the stored procedure is executed, SQL Server combines the permissions of the certificate user and/or login with those of the caller.</span></span> <span data-ttu-id="dc97f-112">W odróżnieniu od `EXECUTE AS` klauzuli nie zmienia kontekst wykonywania procedury.</span><span class="sxs-lookup"><span data-stu-id="dc97f-112">Unlike the `EXECUTE AS` clause, it does not change the execution context of the procedure.</span></span> <span data-ttu-id="dc97f-113">Wbudowane funkcje tej zwrotu nazwy logowania i nazwy użytkowników zwrócić nazwę obiektu wywołującego, a nie nazwa użytkownika certyfikatu.</span><span class="sxs-lookup"><span data-stu-id="dc97f-113">Built-in functions that return login and user names return the name of the caller, not the certificate user name.</span></span>

data-ttu-id="dc97f-116">W czasie wykonywania szyfrowanie danych jest odszyfrowywany przy użyciu klucza publicznego i porównywana z wartością skrótu procedury składowanej.</span><span class="sxs-lookup"><span data-stu-id="dc97f-116">At run time, the data digest is decrypted with the public key and compared with the hash value of the stored procedure.</span></span> <span  <span data-ttu-id="dc97f-120">Istnieją dwie czynności związane z rejestracją modułu:</span><span class="sxs-lookup"><span data-stu-id="dc97f-120">There are two required steps involved in signing a module:</span></span>  
  
1. <span data-ttu-id="dc97f-121">Utwórz certyfikat za pomocą instrukcji języka Transact-SQL `CREATE CERTIFICATE [certificateName]` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="dc97f-121">Create a certificate using the Transact-SQL `CREATE CERTIFICATE [certificateName]` statement.</span></span> <span data-ttu-id="dc97f-122">Ta instrukcja oferuje kilka opcji do ustawiania datę początkową i końcową i hasła.</span><span class="sxs-lookup"><span data-stu-id="dc97f-122">This statement has several options for setting a start and end date and a password.</span></span> <span data-ttu-id="dc97f-123">Data wygaśnięcia domyślną jest jeden rok.</span><span class="sxs-lookup"><span data-stu-id="dc97f-123">The default expiration date is one year.</span></span>  
  
1. <span data-ttu-id="dc97f-124">Utwórz procedurę za pomocą certyfikatu za pomocą języka Transact-SQL `ADD SIGNATURE TO [procedureName] BY CERTIFICATE [certificateName]` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="dc97f-124">Sign the procedure with the certificate using the Transact-SQL `ADD SIGNATURE TO [procedureName] BY CERTIFICATE [certificateName]` statement.</span></span>

<span data-ttu-id="dc97f-130">Jeśli moduł wymaga dodatkowych uprawnień na poziomie serwera:</span><span class="sxs-lookup"><span data-stu-id="dc97f-130">If the module needs additional server-level permissions:</span></span>  
  
1. <span data-ttu-id="dc97f-131">Skopiuj certyfikat `master` bazy danych.</span><span class="sxs-lookup"><span data-stu-id="dc97f-131">Copy the certificate to the `master` database.</span></span>
