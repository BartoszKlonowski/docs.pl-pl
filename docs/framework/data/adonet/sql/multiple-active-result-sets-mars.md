# <a name="multiple-active-result-sets-mars"></a>Wiele aktywnych zestawów wyników (MARS)
Wiele aktywnych zestawów wyników (MARS) to funkcja umożliwiająca wykonywanie wielu partii przy użyciu jednego połączenia. W poprzednich wersjach można było wykonać tylko jedną partię jednocześnie dla jednego połączenia. Wykonywanie wielu partii przy użyciu MARS nie implikuje jednoczesnego wykonywania operacji.
