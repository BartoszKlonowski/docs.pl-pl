# <a name="clr-user-defined-types"></a>Zdefiniowane przez użytkownika typy CLR

Microsoft SQL Server zapewnia obsługę typów zdefiniowanych przez użytkownika (UDTs) implementowanych przy użyciu środowiska uruchomieniowego języka wspólnego (CLR) Microsoft .NET Framework. Środowisko CLR jest zintegrowane z SQL Server, a mechanizm ten umożliwia rozbudowanie systemu typu bazy danych. UDTs zapewniają rozszerzalność użytkowników systemu SQL Server typu danych, a także możliwość definiowania złożonych typów strukturalnych.  
  
 UDTs mogą zapewnić dwie kluczowe korzyści z perspektywy architektury aplikacji:  
  
- Silne hermetyzacja (zarówno w kliencie, jak i na serwerze) między stanem wewnętrznym i zachowaniami zewnętrznymi.  
  
- Ścisła integracja z innymi powiązanymi funkcjami serwera. Po zdefiniowaniu własnego UDT można użyć go we wszystkich kontekstach, w których można używać typu systemu w SQL Server, w tym definicji kolumn, oraz jako zmienne, parametry, wyniki funkcji, kursory, wyzwalacze i replikacja.  
  
 Aby uzyskać szczegółowe informacje, zapoznaj się z [dokumentacją SQL Server](/sql) dla używanej wersji SQL Server.
