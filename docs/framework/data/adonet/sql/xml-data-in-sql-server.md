# <a name="xml-data-in-sql-server"></a>Dane XML w programie SQL Server
SQL Server udostępnia funkcje SQLXML wewnątrz programu .NET Framework. Deweloperzy mogą pisać aplikacje, które dostęp do danych XML z wystąpienia programu SQL Server, przenoszenie danych do środowiska .NET Framework, przetwarzania danych i wysyłać aktualizacje z powrotem do programu SQL Server. Dane XML można na wiele sposobów, w programie SQL Server, łącznie z magazynem danych i jako wartości parametrów do pobierania danych. **SqlXml** klasa w programie .NET Framework zapewnia obsługę po stronie klienta do pracy z danymi przechowywanymi w kolumnie XML w programie SQL Server. Aby uzyskać więcej informacji zobacz "SQLXML klas zarządzanych" w dokumentacji SQL Server — książki Online.
