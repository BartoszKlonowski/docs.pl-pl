# <a name="xml-data-in-sql-server"></a>Dane XML w programie SQL Server
Program SQL Server udostępnia funkcje języka SQLXML wewnątrz programu .NET Framework. Programiści mogą pisać aplikacje, które dostęp do danych XML z wystąpienia programu SQL Server, przenieść dane do środowiska .NET Framework, a przetwarzanie tych danych i Wyślij aktualizacje z powrotem do programu SQL Server. Dane XML może służyć na kilka sposobów, w programie SQL Server, łącznie z magazynem danych oraz jako wartości parametrów dla pobierania danych. **SqlXml** klasy w .NET Framework zapewnia obsługę po stronie klienta do pracy z danymi przechowywanymi w kolumnie XML w programie SQL Server. Aby uzyskać więcej informacji zobacz "SQLXML klasy zarządzane" w podręcznikach Online programu SQL Server.
