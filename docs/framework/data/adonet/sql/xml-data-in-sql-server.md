# <a name="xml-data-in-sql-server"></a>Dane XML w programie SQL Server
SQL Server udostępnia funkcje SQLXML w .NET Framework. Deweloperzy mogą pisać aplikacje, które uzyskują dostęp do danych XML z wystąpienia SQL Server, przenosić dane do środowiska .NET Framework, przetwarzać dane i wysyłać aktualizacje z powrotem do SQL Server. Dane XML mogą być używane na kilka sposobów w SQL Server, w tym magazyn danych, oraz jako wartości parametrów na potrzeby pobierania danych. Klasa **SQLXML** w .NET Framework zapewnia obsługę po stronie klienta do pracy z danymi przechowywanymi w kolumnie XML w SQL Server. Aby uzyskać więcej informacji, zobacz "klasy zarządzane w programie SQLXML" w SQL Server Books Online.
