# <a name="download-sample-databases-linq-to-dataset"></a>Pobieranie przykładowych baz danych (LINQ to DataSet)

Przykłady i wskazówki w linq do dataset dokumentacji użyć adventureworks przykładowej bazy danych. Możesz pobrać tę przykładową bazę danych bezpłatnie. Przejdź do [adventureworks przykładowych baz danych,](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks) aby pobrać bazę danych. Następnie postępuj zgodnie z instrukcjami na tej stronie, aby dołączyć lub przywrócić bazę danych.
  
## <a name="sql-server-express"></a>SQL Server Express

Przykłady i wskazówki w linq do dataset sekcji używać SQL Server 2005 jako magazyn danych, ale można zmodyfikować je do korzystania z [programu SQL Server Express](https://go.microsoft.com/fwlink/?linkid=866658) zamiast. Program SQL Server Express jest bezpłatny i można go rozpowszechniać za pomocą aplikacji.
