# <a name="entity-key"></a><span data-ttu-id="e9a90-102">klucz jednostki</span><span class="sxs-lookup"><span data-stu-id="e9a90-102">entity key</span></span>
<span data-ttu-id="e9a90-103">*Klucz jednostki* jest [właściwość](../../../../docs/framework/data/adonet/property.md) lub zbiór właściwości [typu jednostki](../../../../docs/framework/data/adonet/entity-type.md) które są używane w celu ustalenia tożsamości.</span><span class="sxs-lookup"><span data-stu-id="e9a90-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="e9a90-104">Właściwości, które tworzą klucz jednostki są wybierane w czasie projektowania.</span><span class="sxs-lookup"><span data-stu-id="e9a90-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="e9a90-105">Wartości właściwości klucza jednostki musi jednoznacznie identyfikować wystąpienia typu jednostki, w ramach [zestaw jednostek](../../../../docs/framework/data/adonet/entity-set.md) w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="e9a90-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="e9a90-106">Należy wybrać właściwości, które tworzą klucz jednostki w celu zagwarantowania unikatowości wystąpień w zestawie jednostek.</span><span class="sxs-lookup"><span data-stu-id="e9a90-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="e9a90-107">Poniżej przedstawiono wymagania dotyczące zbiór właściwości jako klucz jednostki:</span><span class="sxs-lookup"><span data-stu-id="e9a90-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="e9a90-108">Brak kluczy dwie jednostki w ramach zestawu jednostek, mogą być identyczne.</span><span class="sxs-lookup"><span data-stu-id="e9a90-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="e9a90-109">Oznacza to, że dwie jednostki w ramach zestawu jednostek, aby uzyskać wartości dla wszystkich właściwości, które tworzą klucz nie może być taka sama.</span><span class="sxs-lookup"><span data-stu-id="e9a90-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="e9a90-110">Jednak niektóre (ale nie wszystkie) wartości, które tworzą jednostki klucz może być taki sam.</span><span class="sxs-lookup"><span data-stu-id="e9a90-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="e9a90-111">Klucz jednostki musi składać się z szeregu innych niż null, niezmienne, [pierwotny typ właściwości](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="e9a90-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="e9a90-112">Nie można zmienić właściwości, które tworzą klucz jednostki dla typu danej jednostki.</span><span class="sxs-lookup"><span data-stu-id="e9a90-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="e9a90-113">Nie zezwalaj na więcej niż jeden klucz jednostki możliwe dla danej jednostki typu; zastępczy klucze nie są obsługiwane.</span><span class="sxs-lookup"><span data-stu-id="e9a90-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="e9a90-114">Gdy jednostka jest zaangażowana w hierarchii dziedziczenia, jednostkę główną musi zawierać wszystkie właściwości, które tworzą klucz jednostki, a klucz musi być zdefiniowany dla typu jednostki głównej.</span><span class="sxs-lookup"><span data-stu-id="e9a90-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="e9a90-115">Aby uzyskać więcej informacji, zobacz [modelu danych jednostki: Dziedziczenie](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="e9a90-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>
