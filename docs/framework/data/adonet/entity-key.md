# <a name="entity-key"></a><span data-ttu-id="0c0af-102">klucz jednostki</span><span class="sxs-lookup"><span data-stu-id="0c0af-102">entity key</span></span>
<span data-ttu-id="0c0af-103">*Klucz jednostki* jest [właściwości](../../../../docs/framework/data/adonet/property.md) lub zbiór właściwości [typu jednostki](../../../../docs/framework/data/adonet/entity-type.md) które są używane do określania tożsamości.</span><span class="sxs-lookup"><span data-stu-id="0c0af-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="0c0af-104">Właściwości, które tworzą klucz jednostki są wybierane w czasie projektowania.</span><span class="sxs-lookup"><span data-stu-id="0c0af-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="0c0af-105">Wartości właściwości klucza obiektu musi jednoznacznie identyfikować wystąpienia typu jednostki, w ramach [zestaw jednostek](../../../../docs/framework/data/adonet/entity-set.md) w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="0c0af-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="0c0af-106">Właściwości, które tworzą klucz jednostki powinna być wybrana w celu zagwarantowania unikatowości wystąpień w zestawie jednostek.</span><span class="sxs-lookup"><span data-stu-id="0c0af-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="0c0af-107">Poniżej przedstawiono wymagania dotyczące zbiór właściwości jako klucz jednostki:</span><span class="sxs-lookup"><span data-stu-id="0c0af-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="0c0af-108">Brak kluczy dwie jednostki w zestawie jednostek mogą być identyczne.</span><span class="sxs-lookup"><span data-stu-id="0c0af-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="0c0af-109">Oznacza to, na dwie jednostki w zestawie jednostek, wartości dla wszystkich właściwości, które stanowią klucza nie może być taka sama.</span><span class="sxs-lookup"><span data-stu-id="0c0af-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="0c0af-110">Jednak niektóre (ale nie wszystkie) wartości, które tworzą jednostki klucza może być taka sama.</span><span class="sxs-lookup"><span data-stu-id="0c0af-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="0c0af-111">Klucz jednostki musi składać się z zestawu wartości null, modyfikować, [właściwości typu pierwotnego](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="0c0af-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="0c0af-112">Nie można zmienić właściwości tworzące klucz jednostki dla typu danej jednostki.</span><span class="sxs-lookup"><span data-stu-id="0c0af-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="0c0af-113">Nie można zezwolić więcej niż jeden klucz jednostki możliwe dla danej jednostki typu; klucze surogat nie są obsługiwane.</span><span class="sxs-lookup"><span data-stu-id="0c0af-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="0c0af-114">Gdy jednostka jest elementem hierarchii dziedziczenia, obiekt główny musi zawierać wszystkie właściwości, które tworzą klucz jednostki, a klucz jednostki musi być zdefiniowana w ramach typu jednostki głównego.</span><span class="sxs-lookup"><span data-stu-id="0c0af-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="0c0af-115">Aby uzyskać więcej informacji, zobacz [modelu Entity Data Model: dziedziczenie](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="0c0af-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>
