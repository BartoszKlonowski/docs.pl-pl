# <a name="navigation-property"></a><span data-ttu-id="39b91-102">Właściwość nawigacji</span><span class="sxs-lookup"><span data-stu-id="39b91-102">navigation property</span></span>
<span data-ttu-id="39b91-103">A *właściwość nawigacji* jest opcjonalną właściwością na [typu jednostki](../../../../docs/framework/data/adonet/entity-type.md) umożliwiającą nawigacji z jednego [zakończenia](../../../../docs/framework/data/adonet/association-end.md) z [skojarzenia](../../../../docs/framework/data/adonet/association-type.md) do Data zakończenia.</span><span class="sxs-lookup"><span data-stu-id="39b91-103">A *navigation property* is an optional property on an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that allows for navigation from one [end](../../../../docs/framework/data/adonet/association-end.md) of an [association](../../../../docs/framework/data/adonet/association-type.md) to the other end.</span></span> <span data-ttu-id="39b91-104">W odróżnieniu od innych [właściwości](../../../../docs/framework/data/adonet/property.md), właściwości nawigacji nie zawierają danych.</span><span class="sxs-lookup"><span data-stu-id="39b91-104">Unlike other [properties](../../../../docs/framework/data/adonet/property.md), navigation properties do not carry data.</span></span>  
  
 <span data-ttu-id="39b91-105">Definicja właściwości nawigacji obejmuje następujące funkcje:</span><span class="sxs-lookup"><span data-stu-id="39b91-105">A navigation property definition includes the following:</span></span>  
  
-   <span data-ttu-id="39b91-106">Nazwa.</span><span class="sxs-lookup"><span data-stu-id="39b91-106">A name.</span></span> <span data-ttu-id="39b91-107">(Wymagane)</span><span class="sxs-lookup"><span data-stu-id="39b91-107">(Required)</span></span>  
  
-   <span data-ttu-id="39b91-108">Skojarzenie go przechodzi.</span><span class="sxs-lookup"><span data-stu-id="39b91-108">The association that it navigates.</span></span> <span data-ttu-id="39b91-109">(Wymagane)</span><span class="sxs-lookup"><span data-stu-id="39b91-109">(Required)</span></span>  
  
-   <span data-ttu-id="39b91-110">Końcach asocjacji, prowadzącego go.</span><span class="sxs-lookup"><span data-stu-id="39b91-110">The ends of the association that it navigates.</span></span> <span data-ttu-id="39b91-111">(Wymagane)</span><span class="sxs-lookup"><span data-stu-id="39b91-111">(Required)</span></span>  
  
 <span data-ttu-id="39b91-112">Należy pamiętać, że właściwości nawigacji są opcjonalne na obu typów jednostek na końcach asocjacji.</span><span class="sxs-lookup"><span data-stu-id="39b91-112">Note that navigation properties are optional on both entity types at the ends of an association.</span></span> <span data-ttu-id="39b91-113">Jeśli zdefiniujesz właściwości nawigacji na jednym typie jednostek na końcu asocjacji, ma Zdefiniuj właściwość nawigacji typu jednostki na drugim końcu skojarzenia.</span><span class="sxs-lookup"><span data-stu-id="39b91-113">If you define a navigation property on one entity type at the end of an association, you do not have to define a navigation property on the entity type at the other end of the association.</span></span>  
  
 <span data-ttu-id="39b91-114">Typ danych właściwości nawigacji jest określana przez [liczebność](../../../../docs/framework/data/adonet/association-end-multiplicity.md) jego zdalnego [end skojarzenia](../../../../docs/framework/data/adonet/association-end.md).</span><span class="sxs-lookup"><span data-stu-id="39b91-114">The data type of a navigation property is determined by the [multiplicity](../../../../docs/framework/data/adonet/association-end-multiplicity.md) of its remote [association end](../../../../docs/framework/data/adonet/association-end.md).</span></span> <span data-ttu-id="39b91-115">Na przykład, załóżmy, że właściwość nawigacji, `OrdersNavProp`, istnieje na `Customer` typu jednostki i przechodzi skojarzenia typu jeden do wielu między `Customer` i `Order`.</span><span class="sxs-lookup"><span data-stu-id="39b91-115">For example, suppose a navigation property, `OrdersNavProp`, exists on a `Customer` entity type and navigates a one-to-many association between `Customer` and `Order`.</span></span> <span data-ttu-id="39b91-116">Ponieważ punkt końcowy skojarzenia zdalnego dla właściwości nawigacji ma liczebność wielu (\*), jego typ danych to kolekcja (z `Order`).</span><span class="sxs-lookup"><span data-stu-id="39b91-116">Because the remote association end for the navigation property has multiplicity of many (\*), its data type is a collection (of `Order`).</span></span> <span data-ttu-id="39b91-117">Podobnie, jeśli właściwość nawigacji, `CustomerNavProp`, istnieje na `Order` typu jednostki, będzie jego typu danych `Customer`, ponieważ liczebność drugim końcu jest jeden (1).</span><span class="sxs-lookup"><span data-stu-id="39b91-117">Similarly, if a navigation property, `CustomerNavProp`, exists on the `Order` entity type, its data type would be `Customer`, because the multiplicity of the remote end is one (1).</span></span>
