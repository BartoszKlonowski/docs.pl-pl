-   <span data-ttu-id="7a102-112">Zależnego końca ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="7a102-112">The dependent end of the constraint.</span></span> <span data-ttu-id="7a102-113">(Typ jednostki, który ma właściwość lub właściwości, które odwołują się klucz główny koniec.)</span><span class="sxs-lookup"><span data-stu-id="7a102-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>
