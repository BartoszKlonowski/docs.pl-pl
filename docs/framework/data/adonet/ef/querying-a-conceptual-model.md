# <a name="querying-a-conceptual-model"></a>Wykonywanie zapytania do modelu koncepcyjnego
ADO.NET Entity Framework umożliwia wykonywanie zapytań względem modelu koncepcyjnego. Aby zbadać model koncepcyjny przy użyciu najnowszej wersji Entity Framework, zobacz [wykonywanie zapytań dotyczących danych](https://go.microsoft.com/fwlink/?LinkId=235282).
