<span data-ttu-id="6a2d8-103">Jeśli masz aplikację, która wykonuje strukturalnie podobne zapytania wiele razy w entity framework, często można zwiększyć wydajność, kompilując kwerendę jeden raz i wykonując go kilka razy z różnymi parametrami.</span><span class="sxs-lookup"><span data-stu-id="6a2d8-103">When you have an application that executes structurally similar queries many times in the Entity Framework, you can frequently increase performance by compiling the query one time and executing it several times with different parameters.</span></span> <span data-ttu-id="6a2d8-104">Na przykład aplikacja może być konieczności pobierania wszystkich klientów w określonym mieście; miasto jest określone w czasie wykonywania przez użytkownika w formularzu.</span><span class="sxs-lookup"><span data-stu-id="6a2d8-104">For example, an application might have to retrieve all the customers in a particular city; the city is specified at runtime by the user in a form.</span></span> <span data-ttu-id="6a2d8-105">LINQ do jednostek obsługuje przy użyciu skompilowanych zapytań w tym celu.</span><span class="sxs-lookup"><span data-stu-id="6a2d8-105">LINQ to Entities supports using compiled queries for this purpose.</span></span>
