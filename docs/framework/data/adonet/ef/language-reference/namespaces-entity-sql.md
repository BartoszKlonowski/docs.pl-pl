[!INCLUDE[esql](../../../../../../includes/esql-md.md)] <span data-ttu-id="f8872-103">wprowadza przestrzenie nazw, aby uniknąć konfliktów nazw dla identyfikatorów globalnych, takich jak nazwy typów, zestawy jednostek, funkcje i tak dalej.</span><span class="sxs-lookup"><span data-stu-id="f8872-103">introduces namespaces to avoid name conflicts for global identifiers such as type names, entity sets, functions, and so on.</span></span> <span data-ttu-id="f8872-104">Przestrzeń nazw obsługuje [!INCLUDE[esql](../../../../../../includes/esql-md.md)] jest podobna do obsługi przestrzeni nazw w .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="f8872-104">The namespace support in [!INCLUDE[esql](../../../../../../includes/esql-md.md)] is similar to the namespace support in the .NET Framework.</span></span>

