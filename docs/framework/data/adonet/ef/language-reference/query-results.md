<span data-ttu-id="2b2a7-103">Po [!INCLUDE[linq_entities](../../../../../../includes/linq-entities-md.md)] zapytania jest konwertowana na drzew poleceń i wykonywane, wyniki zapytania zwykle są zwracane jako jeden z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="2b2a7-103">After a [!INCLUDE[linq_entities](../../../../../../includes/linq-entities-md.md)] query is converted to command trees and executed, the query results are usually returned as one of the following:</span></span>

 <span data-ttu-id="2b2a7-108">Gdy zapytanie zostało wykonane względem źródła danych, wyniki są zmaterializowanego na typy CLR i zwracany do klienta.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-108">When the query has executed against the data source, the results are materialized into CLR types and returned to the client.</span></span> <span data-ttu-id="2b2a7-109">Wszystkie materializacja obiektu odbywa się przez [!INCLUDE[adonet_ef](../../../../../../includes/adonet-ef-md.md)].</span><span class="sxs-lookup"><span data-stu-id="2b2a7-109">All object materialization is performed by the [!INCLUDE[adonet_ef](../../../../../../includes/adonet-ef-md.md)].</span></span> <span data-ttu-id="2b2a7-110">Wszelkie błędy, które wynikają z brakiem połączenia do mapowania miedzy [!INCLUDE[adonet_ef](../../../../../../includes/adonet-ef-md.md)] i środowiska CLR spowoduje, że wyjątki zostanie wygenerowany podczas materializacja obiektu.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-110">Any errors that result from an inability to map between the [!INCLUDE[adonet_ef](../../../../../../includes/adonet-ef-md.md)] and the CLR will cause exceptions to be thrown during object materialization.</span></span>  
  
 <span data-ttu-id="2b2a7-111">Jeśli wykonanie zapytania zwraca typów pierwotnych modelu koncepcyjnego, wyniki składają się z typów CLR, które są autonomiczne i jest odłączony od class="sxs-lookup"><span data-stu-id="2b2a7-112">However, if the query returns a collection of typed entity objects, represented by <xref:System.Data.Objects.ObjectQuery%601>, those types are tracked jego możliwości, zgodnie z definicją w [!INCLUDE[adonet_ef](../../../../../../includes/adonet-ef-md.md)].</span><span class="sxs-lookup"><span data-stu-id="2b2a7-114">This functionality can be used in  <span data-ttu-id="2b2a7-116">Struktura typy zwracane z zapytań (na przykład typy anonimowe i typy złożone dopuszczające wartości zerowe) mogą być `null` wartość.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-116">Struct types returned from queries (such as anonymous types and nullable complex types) can be of `null` value.</span></span> <span data-ttu-id="2b2a7-117"><xref:System.Data.Objects.DataClasses.EntityCollection%601> Również mogą być właściwość zwróconą jednostkę `null` wartość.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-117">An <xref:System.Data.Objects.DataClasses.EntityCollection%601> property of a returned entity can also be of `null` value.</span></span> <span data-ttu-id="2b2a7-118">Może to wynikać z projekcji kolekcji właściwości jednostki, która jest `null` wartości, takich jak wywoływanie <xref:System.Linq.Queryable.FirstOrDefault%2A> na <xref:System.Data.Objects.ObjectQuery%601> , nie ma żadnych elementów.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-118">This can result from projecting the collection property of an entity that is of `null` value, such as calling <xref:System.Linq.Queryable.FirstOrDefault%2A> on an <xref:System.Data.Objects.ObjectQuery%601> that has no elements.</span></span>  
  
 <span data-ttu-id="2b2a7-119">W niektórych sytuacjach zapytanie, może pojawić się do generowania zmaterializowanych wynik podczas jego realizacji, ale będzie można wykonać zapytania na serwerze i obiekt jednostki nigdy nie będzie można zmaterializowanego w CLR.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-119">In certain situations, a query might appear to generate a materialized result during its execution, but the query will be executed on the server and the entity object will never be materialized in the CLR.</span></span> <span data-ttu-id="2b2a7-120">Może to powodować problemy, jeśli zależy efekty uboczne materializacja obiektu.</span><span class="sxs-lookup"><span data-stu-id="2b2a7-120">This can cause problems if you are depending on side effects of object materialization.</span></span>
