# <a name="sqlclient-for-entity-framework-functions"></a>Klient SQL dla funkcji programu Entity Framework

.NET Framework Dostawca danych for SQL Server (SqlClient) dla Entity Framework zawiera zestaw funkcji do wykonywania obliczeń matematycznych i agregacji, a także funkcje do wykonania `System.DateTime` i `string` działania. Te funkcje znajdują się w `SQLServer` przestrzeni nazw.  
  
 Aby uzyskać listę funkcji, które powinny działać z dowolnym dostawcą, zobacz [funkcje kanoniczne](./language-reference/canonical-functions.md).  
  
 Aby uzyskać informacje o sposobach mapowania funkcji kanonicznych na funkcje SQL Server, zobacz [model koncepcyjny kanoniczny dla SQL Server mapowania funkcji](conceptual-model-canonical-to-sql-server-functions-mapping.md).
