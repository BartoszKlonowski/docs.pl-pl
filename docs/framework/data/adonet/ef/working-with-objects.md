# <a name="working-with-objects"></a>Praca z obiektami

Entity Framework umożliwia wykonywanie zapytań, wstawianie, aktualizowanie i usuwanie danych, które są wyrażane jako obiekty środowiska uruchomieniowego języka wspólnego (CLR), które są wystąpieniami typów jednostek. Typy jednostek reprezentują jednostki zdefiniowane w modelu koncepcyjnym. Entity Framework mapuje jednostki i relacje, które są zdefiniowane w modelu koncepcyjnym ze źródłem danych. Entity Framework zapewnia następujące funkcje: dane zmaterializowania zwracane ze źródła danych jako obiekty; Śledź zmiany wprowadzone w obiektach; Obsługa współbieżności; Propaguj zmiany obiektu z powrotem do źródła danych; i powiąż obiekty z kontrolkami.  
  
 Aby uzyskać więcej informacji na temat pracy z obiektami w najnowszej wersji Entity Framework, zobacz [Praca z obiektami](/previous-versions/gg696163(v=vs.103)).
