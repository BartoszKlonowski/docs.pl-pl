# <a name="entity-framework-data-providers"></a>Dostawcy danych programu Entity Framework
Ta sekcja zawiera informacje o dostawcach danych, które [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]obsługują program.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Dostawca EntityClient dla programu Entity Framework](entityclient-provider-for-the-entity-framework.md)  
 Opisuje dostawcę danych EntityClient. Ten dostawca przekształca zapytania z modelem danych do drzewa poleceń w postaci kanonicznej. Drzewo poleceń może być następnie używane przez .NET Framework dostawcę danych dla [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
 [Element SqlClient programu Entity Framework](sqlclient-for-the-entity-framework.md)  
 Opisuje dostawcę danych .NET Framework, który obsługuje [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] do użycia z SQL Server bazą danych.
