# <a name="getting-started-entity-framework"></a>Wprowadzenie (Entity Framework)
ADO.NET Entity Framework obsługuje aplikacje i usługi zorientowane na dane oraz udostępnia platformę do programowania w oparciu o dane, które wzbudzają poziom abstrakcji z logicznego poziomu relacyjnego do poziomu koncepcyjnego. Dzięki umożliwieniu deweloperom pracy z danymi na wyższym poziomie abstrakcji Entity Framework obsługuje kod, który jest niezależny od określonego aparatu magazynu danych lub schematu relacyjnego. Aby uzyskać więcej informacji, zobacz [Entity Framework przegląd](overview.md).  
  
 Aby szybko rozpocząć korzystanie z najnowszej wersji Entity Framework, zobacz [wprowadzenie](https://go.microsoft.com/fwlink/?LinkId=235280)
