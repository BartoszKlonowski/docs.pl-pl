# <a name="getting-started-entity-framework"></a>Wprowadzenie (Entity Framework)

Umożliwia deweloperom pracę z danymi na wyższym poziomie abstrakcji, [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] obsługuje kod, który jest niezależny od dowolnego aparatu magazynu danych lub schemat relacyjny.
