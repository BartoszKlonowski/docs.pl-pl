### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="ca618-116">Generowanie modelu szkoły w projekcie języka Visual Basic, za pomocą EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="ca618-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="ca618-119">W wierszu polecenia Uruchom następujące polecenie bez podziałów wiersza:</span><span class="sxs-lookup"><span data-stu-id="ca618-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
