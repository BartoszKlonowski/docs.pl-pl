### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="65b5d-116">Aby wygenerować model szkoły dla projektu Visual Basic przy użyciu programu EdmGen. exe</span><span class="sxs-lookup"><span data-stu-id="65b5d-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2. <span data-ttu-id="65b5d-119">W wierszu polecenia wykonaj następujące polecenie bez podziałów wierszy:</span><span class="sxs-lookup"><span data-stu-id="65b5d-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```console  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
