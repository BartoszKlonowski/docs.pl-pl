### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="6e8d5-116">Aby wygenerować model służbowych przy użyciu EdmGen.exe projektu Visual Basic</span><span class="sxs-lookup"><span data-stu-id="6e8d5-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1.  <span data-ttu-id="6e8d5-117">Utwórz bazę danych służbowych.</span><span class="sxs-lookup"><span data-stu-id="6e8d5-117">Create the School database.</span></span> <span data-ttu-id="6e8d5-118">Aby uzyskać więcej informacji, zobacz [tworzenie przykładowej bazy danych służbowych](http://msdn.microsoft.com/en-us/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span><span class="sxs-lookup"><span data-stu-id="6e8d5-118">For more information, see [Creating the School Sample Database](http://msdn.microsoft.com/en-us/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span></span>  
  
2.  <span data-ttu-id="6e8d5-119">W wierszu polecenia Uruchom następujące polecenie bez podziałów wierszy:</span><span class="sxs-lookup"><span data-stu-id="6e8d5-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
