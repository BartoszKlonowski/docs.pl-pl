### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="4722a-116">Aby wygenerować model szkoły dla projektu Visual Basic przy użyciu programu EdmGen. exe</span><span class="sxs-lookup"><span data-stu-id="4722a-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1. <span data-ttu-id="4722a-117">Utwórz bazę danych szkoły.</span><span class="sxs-lookup"><span data-stu-id="4722a-117">Create the School database.</span></span> <span data-ttu-id="4722a-118">Aby uzyskać więcej informacji, zobacz [Tworzenie przykładowej bazy danych szkoły](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100)).</span><span class="sxs-lookup"><span data-stu-id="4722a-118">For more information, see [Creating the School Sample Database](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100)).</span></span>  
  
2. <span data-ttu-id="4722a-119">W wierszu polecenia wykonaj następujące polecenie bez podziałów wierszy:</span><span class="sxs-lookup"><span data-stu-id="4722a-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
