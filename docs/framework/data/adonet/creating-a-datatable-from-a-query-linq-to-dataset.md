 <span data-ttu-id="ca7ed-106">Metoda <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> używa następującego procesu do <xref:System.Data.DataTable> utworzenia kwerendy:</span><span class="sxs-lookup"><span data-stu-id="ca7ed-106">The <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method uses the following process to create a <xref:System.Data.DataTable> from a query:</span></span>

<span data-ttu-id="ca7ed-115">Wywołanie <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> metody powoduje, że kwerenda powiązana z tabelą źródłową do wykonania.</span><span class="sxs-lookup"><span data-stu-id="ca7ed-115">Calling the <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method causes the query bound to the source table to execute.</span></span>  
  
 <span data-ttu-id="ca7ed-116">Gdy <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> metoda napotka odwołanie zerowe lub typ wartości nullable w wierszu w <xref:System.DBNull.Value>tabeli źródłowej, zastępuje wartość .</span><span class="sxs-lookup"><span data-stu-id="ca7ed-116">When the <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method encounters either a null reference or nullable value type in a row in the source table, it replaces the value with <xref:System.DBNull.Value>.</span></span> <span data-ttu-id="ca7ed-117">W ten sposób wartości null są <xref:System.Data.DataTable>obsługiwane poprawnie w zwróconym .</span><span class="sxs-lookup"><span data-stu-id="ca7ed-117">This way, null values are handled correctly in the returned <xref:System.Data.DataTable>.</span></span>  
  
 <span data-ttu-id="ca7ed-118">Uwaga: <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> Metoda akceptuje jako dane wejściowe kwerendy, <xref:System.Data.DataTable> <xref:System.Data.DataSet> która może zwracać wiersze z wielu lub obiektów.</span><span class="sxs-lookup"><span data-stu-id="ca7ed-118">Note: The <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method accepts as input a query that can return rows from multiple <xref:System.Data.DataTable> or <xref:System.Data.DataSet> objects.</span></span> <span data-ttu-id="ca7ed-119">Metoda <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> skopiuje dane, ale nie <xref:System.Data.DataTable> właściwości <xref:System.Data.DataSet> ze źródła <xref:System.Data.DataTable>lub obiektów do zwracanych .</span><span class="sxs-lookup"><span data-stu-id="ca7ed-119">The <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method will copy the data but not the properties from the source <xref:System.Data.DataTable> or <xref:System.Data.DataSet> objects to the returned <xref:System.Data.DataTable>.</span></span> <span data-ttu-id="ca7ed-120">Należy jawnie ustawić właściwości na zwróconych <xref:System.Data.DataTable>, <xref:System.Data.DataTable.Locale%2A> takich <xref:System.Data.DataTable.TableName%2A>jak i .</span><span class="sxs-lookup"><span data-stu-id="ca7ed-120">You will need to explicitly set the properties on the returned <xref:System.Data.DataTable>, such as <xref:System.Data.DataTable.Locale%2A> and <xref:System.Data.DataTable.TableName%2A>.</span></span>
