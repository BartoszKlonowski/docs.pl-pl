 <span data-ttu-id="0dec4-116">Operatory <xref:System.Linq.Enumerable.GroupJoin%2A> nie mają bezpośredniego odpowiednika w warunkach relacyjnych baz danych; implementują nadzbiór sprzężeń wewnętrznych i Lewe sprzężenia zewnętrzne.</span><span class="sxs-lookup"><span data-stu-id="0dec4-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="0dec4-117">Lewe sprzężenie zewnętrzne to sprzężenie zwracające każdy element pierwszej (lewej) kolekcji, nawet jeśli nie ma skorelowanych elementów w drugiej kolekcji.</span><span class="sxs-lookup"><span data-stu-id="0dec4-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>
