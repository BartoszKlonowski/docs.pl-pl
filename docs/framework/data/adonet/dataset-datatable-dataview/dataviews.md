# <a name="dataviews"></a><span data-ttu-id="d22e7-102">Elementy DataView</span><span class="sxs-lookup"><span data-stu-id="d22e7-102">DataViews</span></span>

<span data-ttu-id="d22e7-103">A <xref:System.Data.DataView> umożliwia tworzenie różnych widoków danych przechowywanych w programie <xref:System.Data.DataTable> , które są często używane w aplikacjach do wiązania danych.</span><span class="sxs-lookup"><span data-stu-id="d22e7-103">A <xref:System.Data.DataView> enables you to create different views of the data stored in a <xref:System.Data.DataTable>, a capability that is often used in data-binding applications.</span></span> <span data-ttu-id="d22e7-104">Za pomocą elementu **DataView**można uwidocznić dane w tabeli z różnymi kolejności sortowania i można filtrować dane według stanu wiersza lub w oparciu o wyrażenie filtru.</span><span class="sxs-lookup"><span data-stu-id="d22e7-104">Using a **DataView**, you can expose the data in a table with different sort orders, and you can filter the data by row state or based on a filter expression.</span></span>

 <span data-ttu-id="d22e7-105">**Element DataView** udostępnia dynamiczny widok danych w źródłowej **tabeli DataTable**: zawartość, kolejność i członkostwo odzwierciedlają zmiany w miarę ich występowania.</span><span class="sxs-lookup"><span data-stu-id="d22e7-105">A **DataView** provides a dynamic view of data in the underlying **DataTable**: the content, ordering, and membership reflect changes as they occur.</span></span> <span data-ttu-id="d22e7-106">To zachowanie różni się od metody **SELECT** **elementu DataTable**, która zwraca <xref:System.Data.DataRow> tablicę z tabeli na podstawie określonego filtru i/lub porządku sortowania: Ta zawartość odzwierciedla zmiany w tabeli źródłowej, ale jej członkostwo i porządkowanie pozostają statyczne.</span><span class="sxs-lookup"><span data-stu-id="d22e7-106">This behavior differs from the **Select** method of the **DataTable**, which returns a <xref:System.Data.DataRow> array from a table based on a particular filter and/or sort order: this content reflects changes to the underlying table, but its membership and ordering remain static.</span></span> <span data-ttu-id="d22e7-107">Dynamiczne możliwości obiektu **DataView** sprawiają, że są idealnym rozwiązaniem dla aplikacji do wiązania danych.</span><span class="sxs-lookup"><span data-stu-id="d22e7-107">The dynamic capabilities of the **DataView** make it ideal for data-binding applications.</span></span>

 <span data-ttu-id="d22e7-108">**Element DataView** udostępnia dynamiczny widok pojedynczego zestawu danych, podobnie jak widok bazy danych, do którego można zastosować różne kryteria sortowania i filtrowania.</span><span class="sxs-lookup"><span data-stu-id="d22e7-108">A **DataView** provides you with a dynamic view of a single set of data, much like a database view, to which you can apply different sorting and filtering criteria.</span></span> <span data-ttu-id="d22e7-109">W przeciwieństwie do widoku bazy danych, jednak **element DataView** nie może być traktowany jako tabela i nie może zawierać widoku sprzężonych tabel.</span><span class="sxs-lookup"><span data-stu-id="d22e7-109">Unlike a database view, however, a **DataView** cannot be treated as a table and cannot provide a view of joined tables.</span></span> <span data-ttu-id="d22e7-110">Nie można również wykluczyć kolumn, które istnieją w tabeli źródłowej, lub dołączać kolumny, które nie istnieją w tabeli źródłowej, takie jak kolumny obliczeniowe.</span><span class="sxs-lookup"><span data-stu-id="d22e7-110">You also cannot exclude columns that exist in the source table or append columns that do not exist in the source table, such as computational columns.</span></span>

 <span data-ttu-id="d22e7-111"><xref:System.Data.DataView.DataViewManager%2A>Aby zarządzać ustawieniami widoku dla wszystkich tabel w **zestawie danych**, można użyć elementu.</span><span class="sxs-lookup"><span data-stu-id="d22e7-111">You can use a <xref:System.Data.DataView.DataViewManager%2A> to manage view settings for all the tables in a **DataSet**.</span></span> <span data-ttu-id="d22e7-112">Element **DataViewManager** zapewnia wygodny sposób zarządzania domyślnymi ustawieniami widoku dla każdej tabeli.</span><span class="sxs-lookup"><span data-stu-id="d22e7-112">The **DataViewManager** provides you with a convenient way to manage default view settings for each table.</span></span> <span data-ttu-id="d22e7-113">W przypadku wiązania kontrolki z więcej niż jedną tabelą **zestawu danych**powiązanie z elementem **DataViewManager** jest idealnym wyborem.</span><span class="sxs-lookup"><span data-stu-id="d22e7-113">When binding a control to more than one table of a **DataSet**, binding to a **DataViewManager** is the ideal choice.</span></span>
