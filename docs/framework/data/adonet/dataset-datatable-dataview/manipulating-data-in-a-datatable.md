# <a name="manipulating-data-in-a-datatable"></a>Operowanie na danych w elemencie DataTable

Po utworzeniu elementu <xref:System.Data.DataTable> w programie <xref:System.Data.DataSet> można wykonać te same działania, które będą używane w przypadku korzystania z tabeli w bazie danych. Można dodawać, wyświetlać, edytować i usuwać dane w tabeli; można monitorować błędy i zdarzenia; i można wykonywać zapytania dotyczące danych w tabeli. Podczas modyfikowania danych w **elemencie DataTable**można także sprawdzić, czy zmiany są dokładne, oraz określić, czy zmiany mają być programowo akceptowane czy odrzucane.
