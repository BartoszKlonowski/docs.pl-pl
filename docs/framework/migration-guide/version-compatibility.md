## <a name="version-compatibility-for-apps"></a><a name="Apps"></a><span data-ttu-id="ac641-105">Zgodność wersji dla aplikacji</span><span class="sxs-lookup"><span data-stu-id="ac641-105">Version compatibility for apps</span></span>

<span data-ttu-id="ac641-109">Aby zdefiniować określone wersje, na których działa aplikacja, dodaj jeden lub więcej [ \<obsługiwanychruntime>](../configure-apps/file-schema/startup/supportedruntime-element.md) elementów do pliku konfiguracji aplikacji.</span><span class="sxs-lookup"><span data-stu-id="ac641-109">To define the specific versions on which your app runs, add one or more [\<supportedRuntime>](../configure-apps/file-schema/startup/supportedruntime-element.md) elements to your app's configuration file.</span></span> <span data-ttu-id="ac641-110">Każdy `<supportedRuntime>` element zawiera listę obsługiwanej wersji środowiska wykonawczego, z pierwszym określania najbardziej preferowanej wersji i ostatni określania najmniej preferowanej wersji.</span><span class="sxs-lookup"><span data-stu-id="ac641-110">Each `<supportedRuntime>` element lists a supported version of the runtime, with the first specifying the most preferred version and the last specifying the least preferred version.</span></span>

```xml
   <startup>
      <supportedRuntime version="v2.0.50727" />
      <supportedRuntime version="v4.0" />
   </startup>
</configuration>
```

<span data-ttu-id="ac641-119">.NET Framework 4.5 i nowsze wersje są wstecznie zgodne z aplikacjami, które zostały utworzone we wcześniejszych wersjach programu .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="ac641-119">The .NET Framework 4.5 and later versions are backward-compatible with apps that were built with earlier versions of the .NET Framework.</span></span> <span data-ttu-id="ac641-120">Innymi słowy aplikacje i składniki utworzone z poprzednich wersji będzie działać bez modyfikacji w .NET Framework 4.5 i nowszych wersjach.</span><span class="sxs-lookup"><span data-stu-id="ac641-120">In other words, apps and components built with previous versions will work without modification on the .NET Framework 4.5 and later versions.</span></span> <span data-ttu-id="ac641-121">Jednak domyślnie aplikacje są uruchamiane w wersji środowiska wykonawczego języka wspólnego, dla którego zostały opracowane, więc może być wymagane podanie pliku konfiguracji, aby umożliwić uruchamianie aplikacji w wersjach .NET Framework 4.5 lub nowszych.</span><span class="sxs-lookup"><span data-stu-id="ac641-121">However, by default, apps run on the version of the common language runtime for which they were developed, so you may have to provide a configuration file to enable your app to run on the .NET Framework 4.5 or later versions.</span></span> <span data-ttu-id="ac641-122">Aby uzyskać więcej informacji, zobacz [zgodność wersji dla aplikacji](#Apps) sekcji wcześniej w tym artykule.</span><span class="sxs-lookup"><span data-stu-id="ac641-122">For more information, see the [Version compatibility for apps](#Apps) section earlier in this article.</span></span>
