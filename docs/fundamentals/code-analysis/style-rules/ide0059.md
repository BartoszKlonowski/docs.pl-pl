## <a name="overview"></a><span data-ttu-id="3ce7d-116">Omówienie</span><span class="sxs-lookup"><span data-stu-id="3ce7d-116">Overview</span></span>

<span data-ttu-id="3ce7d-117">Ta reguła flaguje niepotrzebne przypisania wartości.</span><span class="sxs-lookup"><span data-stu-id="3ce7d-117">This rule flags unnecessary value assignments.</span></span> <span data-ttu-id="3ce7d-118">Na przykład:</span><span class="sxs-lookup"><span data-stu-id="3ce7d-118">For example:</span></span>

<span data-ttu-id="3ce7d-119">Aby rozwiązać ten problem, użytkownicy mogą wykonać jedną z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="3ce7d-119">Users can take one of the following actions to fix this violation:</span></span>

- <span data-ttu-id="3ce7d-122">Jeśli wyrażenie po prawej stronie przypisywania ma efekty uboczne, Zastąp lewą część przypisania z [odrzuceniem](../../../csharp/discards.md) lub zmienną lokalną, która nigdy nie jest używana.</span><span class="sxs-lookup"><span data-stu-id="3ce7d-122">If the expression on the right side of the assignment has side effects, replace the left side of the assignment with a [discard](../../../csharp/discards.md) or a local variable that's never used.</span></span> <span data-ttu-id="3ce7d-123">Poprawia to przejrzystość kodu i jawne zamiar odrzucania nieużywanej wartości.</span><span class="sxs-lookup"><span data-stu-id="3ce7d-123">This improves code clarity and explicit intent of discarding an unused value.</span></span> <span data-ttu-id="3ce7d-124">Opcja dla tej reguły dotyczy użycia zmiennej lokalnej Odrzuć i nieużywaną.</span><span class="sxs-lookup"><span data-stu-id="3ce7d-124">The option for this rule concerns the use of a discard versus unused local variable.</span></span>

### <a name="example"></a><span data-ttu-id="3ce7d-136">Przykład</span><span class="sxs-lookup"><span data-stu-id="3ce7d-136">Example</span></span>

```csharp
// csharp_style_unused_value_assignment_preference = discard_variable

{
    _ = wordCount.TryGetValue(searchWord, out var count);
    return count;
}

// csharp_style_unused_value_assignment_preference = unused_local_variable

{
    var unused = wordCount.TryGetValue(searchWord, out var count);
    return count;
}
```
