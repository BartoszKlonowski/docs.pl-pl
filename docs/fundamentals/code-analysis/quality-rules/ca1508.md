<span data-ttu-id="7ac1e-115">Metody mogą mieć kod warunkowy, taki jak instrukcje if, wyrażenia binarne ( `==` , `!=` ,, `<` `>` ), sprawdzenia wartości null itd. Rozważmy na przykład następujący kod:</span><span class="sxs-lookup"><span data-stu-id="7ac1e-115">Methods can have conditional code, such as if statements, binary expressions (`==`, `!=`, `<`, `>`), null checks, etc. For example, consider the following code:</span></span>

```csharp
public void M(int i, int j)
{
    if (i != 0)
    {
        return;
    }

    if (j != 0)
    {
        return;
    }

    // Below condition will always evaluate to 'false' as 'i' and 'j' are both '0' here.

    {
        // Code in this 'if' branch is dead code.
        // It can either be removed or refactored.
        ...
    }
}
```
