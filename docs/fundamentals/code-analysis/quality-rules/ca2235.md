## <a name="rule-description"></a><span data-ttu-id="3c4a4-113">Opis reguły</span><span class="sxs-lookup"><span data-stu-id="3c4a4-113">Rule description</span></span>

> [!TIP]
> <span data-ttu-id="3c4a4-116">CA2235 nie uruchamia się w przypadku pól wystąpienia typów, które implementują, <xref:System.Runtime.Serialization.ISerializable> ponieważ udostępniają własne logiki serializacji.</span><span class="sxs-lookup"><span data-stu-id="3c4a4-116">CA2235 does not fire for instance fields of types that implement <xref:System.Runtime.Serialization.ISerializable> because they provide their own serialization logic.</span></span>
