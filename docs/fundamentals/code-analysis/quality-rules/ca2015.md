## <a name="how-to-fix-violations"></a><span data-ttu-id="7e43d-117">Jak naprawić naruszenia</span><span class="sxs-lookup"><span data-stu-id="7e43d-117">How to fix violations</span></span>

```csharp
class DerivedClass <T> : MemoryManager<T>
{
    public override bool Dispose(bool disposing)
    {
        if (disposing)
        {
            _handle.Dispose();
        }
    }

    ...

    // Violation occurs, remove the finalizer to fix the warning.
    ~DerivedClass() => Dispose(false);
}
```
