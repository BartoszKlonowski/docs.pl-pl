## <a name="cause"></a><span data-ttu-id="b8a57-111">Przyczyna</span><span class="sxs-lookup"><span data-stu-id="b8a57-111">Cause</span></span>

<span data-ttu-id="b8a57-112">Korzystanie z <xref:System.Net.Http.HttpClient?displayProperty=fullName> programu przy jednoczesnym udostępnieniu obsługi specyficznej dla platformy ( <xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> lub), <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName> której `CheckCertificateRevocationList` Właściwość jest prawdopodobnie ustawiona, `false` zezwala na akceptowanie odwołanych certyfikatów przez <xref:System.Net.Http.HttpClient> prawidłowy.</span><span class="sxs-lookup"><span data-stu-id="b8a57-112">Using <xref:System.Net.Http.HttpClient?displayProperty=fullName> while providing a platform specific handler (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> or <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) whose `CheckCertificateRevocationList` property is possibly set to `false` will allow revoked certificates to be accepted by the <xref:System.Net.Http.HttpClient> as valid.</span></span>

## <a name="pseudo-code-examples"></a><span data-ttu-id="b8a57-128">Przykłady pseudo kodu</span><span class="sxs-lookup"><span data-stu-id="b8a57-128">Pseudo-code examples</span></span>

```csharp
using System;
using System.Net.Http;

class ExampleClass
{
    void ExampleMethod(bool checkCertificateRevocationList)
    {
        WinHttpHandler winHttpHandler = new WinHttpHandler();
        winHttpHandler.CheckCertificateRevocationList = checkCertificateRevocationList;
        Random r = new Random();

        if (r.Next(6) == 4)
        {
            winHttpHandler.CheckCertificateRevocationList = true;
        }

        HttpClient httpClient = new HttpClient(winHttpHandler);
    }
}
```
