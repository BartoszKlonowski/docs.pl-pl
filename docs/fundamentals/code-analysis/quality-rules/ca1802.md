## <a name="cause"></a><span data-ttu-id="965a2-111">Przyczyna</span><span class="sxs-lookup"><span data-stu-id="965a2-111">Cause</span></span>

<span data-ttu-id="965a2-112">Pole jest zadeklarowane `static` i `readonly` ( `Shared` i `ReadOnly` w Visual Basic) i jest inicjowane z wartością, która jest obliczanej w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="965a2-112">A field is declared `static` and `readonly` (`Shared` and `ReadOnly` in Visual Basic), and is initialized with a value that is computable at compile time.</span></span>

## <a name="rule-description"></a><span data-ttu-id="965a2-114">Opis reguły</span><span class="sxs-lookup"><span data-stu-id="965a2-114">Rule description</span></span>

<span data-ttu-id="965a2-115">Wartość `static readonly` pola jest obliczana w czasie wykonywania, gdy wywoływany jest Konstruktor statyczny dla typu deklarującego.</span><span class="sxs-lookup"><span data-stu-id="965a2-115">The value of a `static readonly` field is computed at run time when the static constructor for the declaring type is called.</span></span> <span data-ttu-id="965a2-116">Jeśli `static readonly` pole jest inicjowane, gdy jest zadeklarowany, a Konstruktor statyczny nie jest zadeklarowany jawnie, kompilator emituje konstruktora statycznego, aby zainicjować pole.</span><span class="sxs-lookup"><span data-stu-id="965a2-116">If the `static readonly` field is initialized when it is declared and a static constructor is not declared explicitly, the compiler emits a static constructor to initialize the field.</span></span>

<span data-ttu-id="965a2-117">Wartość `const` pola jest obliczana w czasie kompilacji i przechowywana w metadanych, co zwiększa wydajność w czasie wykonywania w porównaniu do `static readonly` pola.</span><span class="sxs-lookup"><span data-stu-id="965a2-117">The value of a `const` field is computed at compile time and stored in the metadata, which increases run-time performance when it is compared to a `static readonly` field.</span></span>

<span data-ttu-id="965a2-118">Ponieważ wartość przypisana do pola Target jest obliczanej w czasie kompilacji, należy zmienić deklarację na `const` pole, aby wartość była obliczana w czasie kompilacji, a nie w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="965a2-118">Because the value assigned to the targeted field is computable at compile time, change the declaration to a `const` field so that the value is computed at compile time instead of at run time.</span></span>
