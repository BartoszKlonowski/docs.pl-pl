<span data-ttu-id="25bd8-113">To ostrzeżenie jest zgłaszane, gdy ciąg literału zostanie przesunięty jako wartość do parametru lub właściwości, a co najmniej jedna z następujących sytuacji jest prawdziwa:</span><span class="sxs-lookup"><span data-stu-id="25bd8-113">This warning is raised when a literal string is passed as a value to a parameter or property and one or more of the following situations is true:</span></span>
