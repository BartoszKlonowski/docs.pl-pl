## <a name="how-to-fix-violations"></a><span data-ttu-id="346d4-119">Jak naprawić naruszenia</span><span class="sxs-lookup"><span data-stu-id="346d4-119">How to fix violations</span></span>

```csharp
using System.Collections.Generic;
using System.Linq;

class C
{
    public int GetCount(int[] array)
        => array.Count();

    public int GetCount(ICollection<int> collection)
        => collection.Count();
}
```

```csharp
using System.Collections.Generic;

class C
{
    public int GetCount(int[] array)
        => array.Length;

    public int GetCount(ICollection<int> collection)
        => collection.Count;
}
```
