### <a name="enumeration-name-violation"></a><span data-ttu-id="6f317-133">Naruszenie nazwy wyliczenia</span><span class="sxs-lookup"><span data-stu-id="6f317-133">Enumeration name violation</span></span>

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5397 violation for using Tls11
        SslProtocols protocols = SslProtocols.Tls11 | SslProtocols.Tls12;
    }
}
```

### <a name="solution"></a><span data-ttu-id="6f317-135">Rozwiązanie</span><span class="sxs-lookup"><span data-stu-id="6f317-135">Solution</span></span>

```csharp
using System;
using System.Security.Authentication;

public class TestClass
{
    public void Method()
    {
        // Let the operating system decide what TLS protocol version to use.

        SslProtocols sslProtocols = SslProtocols.None;
    }
}
```
