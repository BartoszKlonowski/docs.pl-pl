## <a name="how-to-fix-violations"></a><span data-ttu-id="0dc19-116">Jak naprawić naruszenia</span><span class="sxs-lookup"><span data-stu-id="0dc19-116">How to fix violations</span></span>

<span data-ttu-id="0dc19-117">Jeśli jest wymagane kierowanie zmodyfikowanych danych ciągu do obiektu wywołującego, należy przekazać ciąg przez odwołanie.</span><span class="sxs-lookup"><span data-stu-id="0dc19-117">If marshaling of modified string data back to the caller is required, pass the string by reference instead.</span></span> <span data-ttu-id="0dc19-118">W przeciwnym razie <xref:System.Runtime.InteropServices.OutAttribute> można je usunąć bez żadnych innych zmian.</span><span class="sxs-lookup"><span data-stu-id="0dc19-118">Otherwise, the <xref:System.Runtime.InteropServices.OutAttribute> can be removed without any other changes.</span></span>

```csharp
 // Violation
[DllImport("MyLibrary")]
private static extern void Foo([Out] string s);

// Fixed: passed by reference
[DllImport("MyLibrary")]
private static extern void Foo(out string s);

// Fixed: marshaling data back to caller is not required
[DllImport("MyLibrary")]
private static extern void Foo(string s);
```
