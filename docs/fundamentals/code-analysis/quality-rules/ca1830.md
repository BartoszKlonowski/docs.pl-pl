## <a name="how-to-fix-violations"></a><span data-ttu-id="05593-116">Jak naprawić naruszenia</span><span class="sxs-lookup"><span data-stu-id="05593-116">How to fix violations</span></span>

<span data-ttu-id="05593-117">Usuń niepotrzebne `ToString()` z wywołania.</span><span class="sxs-lookup"><span data-stu-id="05593-117">Delete the unnecessary `ToString()` from the invocation.</span></span>

```csharp
using System.Text;

class C
{
    int _value;

    // Violation
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value.ToString()).AppendLine();
    }

    // Fixed
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value).AppendLine();
    }
}
```
