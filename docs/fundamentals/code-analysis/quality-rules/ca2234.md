## <a name="cause"></a><span data-ttu-id="8ef2c-111">Przyczyna</span><span class="sxs-lookup"><span data-stu-id="8ef2c-111">Cause</span></span>

<span data-ttu-id="8ef2c-112">Wykonano wywołanie metody, która ma parametr ciągu, którego nazwa zawiera "URI", "URI", "urn", "urn", "URL" lub "URL", a typ deklarujący metody zawiera odpowiednie Przeciążenie metody, która ma <xref:System.Uri?displayProperty=fullName> parametr.</span><span class="sxs-lookup"><span data-stu-id="8ef2c-112">A call is made to a method that has a string parameter whose name contains "uri", "Uri", "urn", "Urn", "url", or "Url" and the declaring type of the method contains a corresponding method overload that has a <xref:System.Uri?displayProperty=fullName> parameter.</span></span>

## <a name="rule-description"></a><span data-ttu-id="8ef2c-114">Opis reguły</span><span class="sxs-lookup"><span data-stu-id="8ef2c-114">Rule description</span></span>

<span data-ttu-id="8ef2c-115">Nazwa parametru jest dzielona na tokeny na podstawie Konwencji wielkości liter notacji CamelCase, a następnie sprawdzana jest wartość "URI", "URI", "urn", "urn", "URL" lub "URL".</span><span class="sxs-lookup"><span data-stu-id="8ef2c-115">A parameter name is split into tokens based on the camel casing convention, and then each token is checked to see whether it equals "uri", "Uri", "urn", "Urn", "url", or "Url".</span></span> <span data-ttu-id="8ef2c-116">Jeśli istnieje dopasowanie, przyjęto, że parametr reprezentuje identyfikator URI (Uniform Resource Identifier).</span><span class="sxs-lookup"><span data-stu-id="8ef2c-116">If there is a match, the parameter is assumed to represent a uniform resource identifier (URI).</span></span> <span data-ttu-id="8ef2c-117">Reprezentacja ciągu identyfikatora URI jest podatna na analizowanie i kodowanie błędów i może prowadzić do powstawania luk w zabezpieczeniach.</span><span class="sxs-lookup"><span data-stu-id="8ef2c-117">A string representation of a URI is prone to parsing and encoding errors, and can lead to security vulnerabilities.</span></span> <span data-ttu-id="8ef2c-118"><xref:System.Uri>Klasa udostępnia te usługi w bezpieczny i bezpieczny sposób.</span><span class="sxs-lookup"><span data-stu-id="8ef2c-118">The <xref:System.Uri> class provides these services in a safe and secure manner.</span></span> <span data-ttu-id="8ef2c-119">Gdy istnieje wybór między dwoma przeciążeniami, które różnią się tylko reprezentacją identyfikatora URI, użytkownik powinien wybrać Przeciążenie przyjmujące <xref:System.Uri> argument.</span><span class="sxs-lookup"><span data-stu-id="8ef2c-119">When there is a choice between two overloads that differ only regarding the representation of a URI, the user should choose the overload that takes a <xref:System.Uri> argument.</span></span>
