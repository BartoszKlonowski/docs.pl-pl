## <a name="cause"></a><span data-ttu-id="134fb-111">Przyczyna</span><span class="sxs-lookup"><span data-stu-id="134fb-111">Cause</span></span>

<span data-ttu-id="134fb-112">`Count`Właściwość or `Length` lub <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> Metoda rozszerzenia została użyta do określenia, czy obiekt zawiera jakiekolwiek elementy, porównując wartość z `0` lub `1` , a obiekt ma bardziej wydajną `IsEmpty` Właściwość, której można użyć zamiast tego.</span><span class="sxs-lookup"><span data-stu-id="134fb-112">The `Count` or `Length` property or the <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> extension method was used to determine whether or not the object contains any items by comparing the value to `0` or `1`, and the object has a more efficient `IsEmpty` property that could be used instead.</span></span>

## <a name="rule-description"></a><span data-ttu-id="134fb-113">Opis reguły</span><span class="sxs-lookup"><span data-stu-id="134fb-113">Rule description</span></span>

<span data-ttu-id="134fb-114">Ta zasada flaguje wywołania `Count` `Length` właściwości i <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> i i <xref:System.Linq.Enumerable.LongCount%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> metod LINQ, gdy są one używane do określenia, czy obiekt zawiera jakiekolwiek elementy, a obiekt ma bardziej wydajną `IsEmpty` Właściwość.</span><span class="sxs-lookup"><span data-stu-id="134fb-114">This rule flags the calls to the `Count` and `Length` properties or <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> and <xref:System.Linq.Enumerable.LongCount%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> LINQ methods when they are used to determine if the object contains any items and the object has a more efficient `IsEmpty` property.</span></span>

<span data-ttu-id="134fb-115">Analiza tej reguły początkowo pokrywa się z podobnymi regułami CA1827, CA1828 i CA1829; Analizatory takich reguł zostały scalone wraz z tą dla CA1836, aby zgłosić najlepszą diagnostykę w przypadku nakładania się.</span><span class="sxs-lookup"><span data-stu-id="134fb-115">The analysis of this rule originally overlapped with similar rules CA1827, CA1828, and CA1829; the analyzers of such rules were merged along with the one for CA1836 to report the best diagnosis in case of overlap.</span></span>

## <a name="how-to-fix-violations"></a><span data-ttu-id="134fb-116">Jak naprawić naruszenia</span><span class="sxs-lookup"><span data-stu-id="134fb-116">How to fix violations</span></span>

<span data-ttu-id="134fb-117">Aby naprawić naruszenie, Zastąp <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> wywołanie metody lub lub dostęp do właściwości, <xref:System.Linq.Enumerable.LongCount%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> `Length` `Count` gdy jest używany w operacji, która określa, czy obiekt jest pusty przy użyciu `IsEmpty` dostępu do właściwości.</span><span class="sxs-lookup"><span data-stu-id="134fb-117">To fix a violation, replace the <xref:System.Linq.Enumerable.Count%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> or <xref:System.Linq.Enumerable.LongCount%60%601%28System.Collections.Generic.IEnumerable%7B%60%600%7D%29> method call or the `Length` or `Count` property access when it's used in an operation that determines if the object is empty with use of the `IsEmpty` property access.</span></span> <span data-ttu-id="134fb-118">Na przykład następujące dwa fragmenty kodu pokazują naruszenie reguły i sposób jej naprawy:</span><span class="sxs-lookup"><span data-stu-id="134fb-118">For example, the following two code snippets show a violation of the rule and how to fix it:</span></span>

```csharp
using System.Collections.Concurrent;

class C
{
    ConcurrentQueue<int> _queue;
    public bool IsEmpty => _queue.Count == 0;
}
```

```csharp
using System.Collections.Concurrent;

class C
{
    ConcurrentQueue<int> _queue;
    public bool IsEmpty => _queue.IsEmpty;
}
```
