## <a name="cause"></a><span data-ttu-id="1e807-111">Przyczyna</span><span class="sxs-lookup"><span data-stu-id="1e807-111">Cause</span></span>

<span data-ttu-id="1e807-112">Użycie w <xref:System.Net.Http.HttpClient?displayProperty=fullName> czasie, gdy zapewnia obsługę specyficzną dla platformy ( <xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> lub <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName> ), której `CheckCertificateRevocationList` Właściwość nie jest ustawiona na, `true` zezwala na akceptowanie odwołanych certyfikatów przez <xref:System.Net.Http.HttpClient> prawidłowy.</span><span class="sxs-lookup"><span data-stu-id="1e807-112">Using <xref:System.Net.Http.HttpClient?displayProperty=fullName> while providing a platform specific handler (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> or <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) whose `CheckCertificateRevocationList` property is not set to `true` will allow revoked certificates to be accepted by the <xref:System.Net.Http.HttpClient> as valid.</span></span>

<span data-ttu-id="1e807-113">Ta reguła jest podobna do [CA5400](ca5400.md), ale analiza może stwierdzić, że `CheckCertificateRevocationList` Właściwość jest w nieskończoność `false` lub nie została ustawiona.</span><span class="sxs-lookup"><span data-stu-id="1e807-113">This rule is similar to [CA5400](ca5400.md), but analysis can determine that the `CheckCertificateRevocationList` property is definitely `false` or not set.</span></span>
