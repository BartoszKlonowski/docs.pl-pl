# <a name="azure-serverless-platform"></a>Platforma bezserwerowa platformy Azure

Platforma bezserwerowa platformy Azure zawiera funkcje azure, aplikacje logiki i siatkę zdarzeń. Usługi te współpracują ze sobą i łączą się z niezliczonymi innymi zasobami. Platforma bezserwerowa współpracuje ze wszystkim, od baz danych i pamięci masowej po analitykę i uczenie maszynowe/sztuczną inteligencję.

Można również użyć usługi Application Insights, platformy bezserwerowej do przechwytywania śladów diagnostycznych i telemetrii. Usługa Application Insights jest dostępna dla aplikacji wszystkich typów (komputerów stacjonarnych, mobilnych lub internetowych) oraz implementacji bezserwerowych.
