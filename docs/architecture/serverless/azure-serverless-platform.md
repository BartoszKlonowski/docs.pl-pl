# <a name="azure-serverless-platform"></a>Platforma bezserwerowa platformy Azure

Platforma bezserwerowa platformy Azure zawiera Azure Functions, Logic Apps i Event Grid. Te usługi współpracują ze sobą i łączą się z wyposażono innymi zasobami. Platforma bezserwerowa współpracuje ze wszystkimi bazami danych i magazynem na potrzeby analiz i uczenia maszynowego/sztucznej analizy.

Można również użyć Application Insights, platformy bezserwerowej do przechwytywania śladów diagnostycznych i danych telemetrycznych. Application Insights są dostępne dla aplikacji wszystkich typów (klasycznych, mobilnych lub sieci Web), jak również implementacji bezserwerowych. Platforma jest wizualizowana na poniższym diagramie:
