# <a name="build-resilient-services-ready-for-the-cloud-embrace-transient-failures-in-the-cloud"></a><span data-ttu-id="57f8a-105">Kompilowanie odpornych usług gotowych do użycia w chmurze: obsługa przejściowych błędów w chmurze</span><span class="sxs-lookup"><span data-stu-id="57f8a-105">Build resilient services ready for the cloud: Embrace transient failures in the cloud</span></span>

<span data-ttu-id="57f8a-109">Aplikacja jest gotowa do chmury, gdy co najmniej implementuje oparty na oprogramowaniu model odporności, a nie model oparty na sprzęcie.</span><span class="sxs-lookup"><span data-stu-id="57f8a-109">Your application is ready for the cloud when, at a minimum, it implements a software-based model of resiliency, rather than a hardware-based model.</span></span> <span data-ttu-id="57f8a-110">Aplikacja w chmurze musi obejmować częściowe błędy, które z pewnością wystąpią.</span><span class="sxs-lookup"><span data-stu-id="57f8a-110">Your cloud application must embrace the partial failures that will certainly occur.</span></span> <span data-ttu-id="57f8a-111">Zaprojektuj lub częściowo refaktoryzuj aplikację, aby osiągnąć odporność z oczekiwanymi częściowymi awariami.</span><span class="sxs-lookup"><span data-stu-id="57f8a-111">Design or partially refactor your application to achieve resiliency with expected partial failures.</span></span> <span data-ttu-id="57f8a-112">Powinien być zaprojektowany do radzenia sobie z częściowymi awariami, takimi jak przejściowe awarie sieci i węzły lub maszyny wirtualne ulegające awarii w chmurze.</span><span class="sxs-lookup"><span data-stu-id="57f8a-112">It should be designed to cope with partial failures, like transient network outages and nodes, or VMs crashing in the cloud.</span></span> <span data-ttu-id="57f8a-113">Nawet kontenery są przenoszone do innego węzła w klastrze koordynatora może spowodować sporadyczne krótkie błędy w aplikacji.</span><span class="sxs-lookup"><span data-stu-id="57f8a-113">Even containers being moved to a different node within an orchestrator cluster can cause intermittent short failures within the application.</span></span>

## <a name="handling-partial-failure"></a><span data-ttu-id="57f8a-114">Obsługa częściowych niepowodzeń</span><span class="sxs-lookup"><span data-stu-id="57f8a-114">Handling partial failure</span></span>

<span data-ttu-id="57f8a-118">Ponieważ klienci i usługi są oddzielnymi procesami, usługa może nie być w stanie odpowiedzieć w odpowiednim czasie na żądanie klienta.</span><span class="sxs-lookup"><span data-stu-id="57f8a-118">Because clients and services are separate processes, a service might not be able to respond in a timely manner to a client's request.</span></span> <span data-ttu-id="57f8a-119">Usługa może być przeciążona i reagować powoli na żądania lub może nie być dostępna przez krótki czas z powodu problemów z siecią.</span><span class="sxs-lookup"><span data-stu-id="57f8a-119">The service might be overloaded and respond slowly to requests, or it might not be accessible for a short time because of network issues.</span></span>

<span data-ttu-id="57f8a-120">Rozważmy na przykład monolityczną aplikację .NET, która uzyskuje dostęp do bazy danych w usłudze Azure SQL Database.</span><span class="sxs-lookup"><span data-stu-id="57f8a-120">For example, consider a monolithic .NET application that accesses a database in Azure SQL Database.</span></span> <span data-ttu-id="57f8a-121">Jeśli baza danych SQL platformy Azure lub jakakolwiek inna usługa innej firmy nie odpowiada przez krótki czas (baza danych SQL platformy Azure może zostać przeniesiona do innego węzła lub serwera i nie odpowiada przez kilka sekund), gdy użytkownik próbuje wykonać dowolną akcję, aplikacja może ulec awarii i w tym samym momencie.</span><span class="sxs-lookup"><span data-stu-id="57f8a-121">If the Azure SQL database or any other third-party service is unresponsive for a brief time (an Azure SQL database might be moved to a different node or server, and be unresponsive for a few seconds), when the user tries to do any action, the application might crash and show an exception at the same moment.</span></span>

<span data-ttu-id="57f8a-122">Podobny scenariusz może wystąpić w aplikacji, która korzysta z usług HTTP.</span><span class="sxs-lookup"><span data-stu-id="57f8a-122">A similar scenario might occur in an app that consumes HTTP services.</span></span> <span data-ttu-id="57f8a-123">Sieć lub sama usługa może nie być dostępna w chmurze podczas krótkiego, przejściowego błędu.</span><span class="sxs-lookup"><span data-stu-id="57f8a-123">The network or the service itself might not be available in the cloud during a short, transient failure.</span></span>
