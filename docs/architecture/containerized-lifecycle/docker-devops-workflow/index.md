- <span data-ttu-id="6c665-114">Azure DevOps Services i Team Foundation Server obejmują zaawansowane i rosnące ekosystemy rozszerzeń pierwszej i innej firmy, za pomocą których można łatwo skonstruować potok CI, kompilację, testowanie, dostarczanie i Zarządzanie zleceniami dla mikrousług.</span><span class="sxs-lookup"><span data-stu-id="6c665-114">Azure DevOps Services and Team Foundation Server include a powerful and growing ecosystem of first and third-party extensions with which you easily can construct a CI, build, test, delivery, and release management pipeline for microservices.</span></span>
