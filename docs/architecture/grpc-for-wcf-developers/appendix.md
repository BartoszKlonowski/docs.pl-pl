<span data-ttu-id="6846c-104">Windows Communication Foundation (WCF) obsługuje transakcje rozproszone, umożliwiając wykonywanie operacji niepodzielnych w wielu usługach.</span><span class="sxs-lookup"><span data-stu-id="6846c-104">Windows Communication Foundation (WCF) supports distributed transactions, allowing you to perform atomic operations across multiple services.</span></span> <span data-ttu-id="6846c-105">Ta funkcja jest oparta na [Distributed Transaction Coordinator firmy Microsoft](https://docs.microsoft.com/previous-versions/windows/desktop/ms684146(v=vs.85)).</span><span class="sxs-lookup"><span data-stu-id="6846c-105">This functionality is based on the [Microsoft Distributed Transaction Coordinator](https://docs.microsoft.com/previous-versions/windows/desktop/ms684146(v=vs.85)).</span></span>

<span data-ttu-id="6846c-106">W przypadku nowszych mikrousług na poziomie, ten typ zautomatyzowanego przetwarzania transakcji rozproszonych nie jest możliwy.</span><span class="sxs-lookup"><span data-stu-id="6846c-106">In the newer microservices landscape, this type of automated distributed transaction processing isn't possible.</span></span> <span data-ttu-id="6846c-107">Istnieje zbyt wiele różnych technologii, w tym relacyjne bazy danych, NoSQLe magazyny danych i systemy obsługi komunikatów.</span><span class="sxs-lookup"><span data-stu-id="6846c-107">There are too many different technologies involved, including relational databases, NoSQL data stores, and messaging systems.</span></span> <span data-ttu-id="6846c-108">Może to być również kombinacja systemów operacyjnych, języków programowania i platform używanych w jednym środowisku.</span><span class="sxs-lookup"><span data-stu-id="6846c-108">There might also be a mix of operating systems, programming languages, and frameworks in use in a single environment.</span></span>

<span data-ttu-id="6846c-109">Transakcja rozproszona WCF to implementacja tego, co jest znane jako [dwufazowe zatwierdzanie (2PC)](https://en.wikipedia.org/wiki/Two-phase_commit_protocol).</span><span class="sxs-lookup"><span data-stu-id="6846c-109">WCF distributed transaction is an implementation of what is known as a [two-phase commit (2PC)](https://en.wikipedia.org/wiki/Two-phase_commit_protocol).</span></span> <span data-ttu-id="6846c-110">Transakcje 2PC można zaimplementować ręcznie przez koordynowanie komunikatów między usługami, tworzenie otwartych transakcji w ramach każdej usługi i wysyłanie komunikatów zatwierdzania lub wycofywania, w zależności od sukcesu lub niepowodzenia.</span><span class="sxs-lookup"><span data-stu-id="6846c-110">You can implement 2PC transactions manually by coordinating messages across services, creating open transactions within each service, and sending commit or rollback messages, depending upon success or failure.</span></span> <span data-ttu-id="6846c-111">Jednak stopień złożoności związany z zarządzaniem 2PC może wzrosnąć w miarę rozwoju systemów.</span><span class="sxs-lookup"><span data-stu-id="6846c-111">However, the complexity involved in managing 2PC can increase exponentially as systems evolve.</span></span> <span data-ttu-id="6846c-112">Otwarte transakcje przechowują blokady bazy danych, które mogą negatywnie wpłynąć na wydajność, lub gorszą, powodują zakleszczenie między usługami.</span><span class="sxs-lookup"><span data-stu-id="6846c-112">Open transactions hold database locks that can negatively affect performance, or, worse, cause cross-service deadlocks.</span></span>

<span data-ttu-id="6846c-113">Jeśli to możliwe, najlepiej jest unikać transakcji rozproszonych.</span><span class="sxs-lookup"><span data-stu-id="6846c-113">If possible, it's best to avoid distributed transactions altogether.</span></span> <span data-ttu-id="6846c-114">Jeśli dwa elementy danych są tak połączone jak w celu żądania niepodzielnych aktualizacji, należy rozważyć ich obsługę zarówno w ramach tej samej usługi.</span><span class="sxs-lookup"><span data-stu-id="6846c-114">If two items of data are so linked as to require atomic updates, consider handling them both with the same service.</span></span> <span data-ttu-id="6846c-115">Zastosuj te niepodzielne zmiany za pomocą pojedynczego żądania lub wiadomości do tej usługi.</span><span class="sxs-lookup"><span data-stu-id="6846c-115">Apply those atomic changes by using a single request or message to that service.</span></span>
