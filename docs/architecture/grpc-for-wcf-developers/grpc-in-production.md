# <a name="grpc-in-production"></a>Usługa gRPC w środowisku produkcyjnym

Aplikacje ASP.NET Core 3,0, w tym usługi gRPC Services, można uruchamiać w systemach Windows, Linux i w kontenerach przy użyciu nowoczesnych platform, takich jak Docker i Kubernetes. W tym rozdziale zapoznają się z różnymi opcjami dotyczącymi uruchamiania usług gRPC w środowisku produkcyjnym i przyjrzyj się opcjom monitorowania i rejestrowania, aby zapewnić optymalne działanie systemów.
