# <a name="grpc-in-production"></a>Usługa gRPC w środowisku produkcyjnym

Można uruchamiać aplikacje ASP.NET Core 3,0, w tym usługi gRPC Services, w systemie Windows, na Linux i w kontenerach korzystających z nowoczesnych platform, takich jak Docker i Kubernetes. Ten rozdział eksploruje różne opcje uruchamiania usług gRPC Services w środowisku produkcyjnym oraz sprawdza opcje monitorowania i rejestrowania, aby zapewnić optymalne działanie systemów.
