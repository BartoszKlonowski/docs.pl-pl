<span data-ttu-id="2d05b-109">Powiązanie NetTCP programu WCF umożliwia tworzenie trwałych połączeń, małych komunikatów i komunikatów dwukierunkowych.</span><span class="sxs-lookup"><span data-stu-id="2d05b-109">WCF's NetTCP binding allows for persistent connections, small messages, and two-way messaging.</span></span> <span data-ttu-id="2d05b-110">Działa jednak tylko między klientami i serwerami platformy .NET.</span><span class="sxs-lookup"><span data-stu-id="2d05b-110">But it works only between .NET clients and servers.</span></span> <span data-ttu-id="2d05b-111">gRPC umożliwia korzystanie z tych samych funkcji, ale jest obsługiwana w wielu językach programowania i platformach.</span><span class="sxs-lookup"><span data-stu-id="2d05b-111">gRPC allows the same functionality but is supported across multiple programming languages and platforms.</span></span>
