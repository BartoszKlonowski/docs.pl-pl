### <a name="the-application-layer"></a><span data-ttu-id="74591-179">Warstwa aplikacji</span><span class="sxs-lookup"><span data-stu-id="74591-179">The application layer</span></span>

<span data-ttu-id="74591-180">Przechodząc do warstwy aplikacji, możemy ponownie przytoczyć książkę Erica Evansa [Domain Driven Design:](https://domainlanguage.com/ddd/)</span><span class="sxs-lookup"><span data-stu-id="74591-180">Moving on to the application layer, we can again cite Eric Evans's book [Domain Driven Design](https://domainlanguage.com/ddd/):</span></span>

<span data-ttu-id="74591-181">**Warstwa aplikacji:** Definiuje zadania, które oprogramowanie ma wykonać i kieruje wyraziste obiekty domeny do rozwiązywania problemów.</span><span class="sxs-lookup"><span data-stu-id="74591-181">**Application Layer:** Defines the jobs the software is supposed to do and directs the expressive domain objects to work out problems.</span></span> <span data-ttu-id="74591-182">Zadania, za które odpowiedzialna jest ta warstwa, mają znaczenie dla firmy lub są niezbędne do interakcji z warstwami aplikacji innych systemów.</span><span class="sxs-lookup"><span data-stu-id="74591-182">The tasks this layer is responsible for are meaningful to the business or necessary for interaction with the application layers of other systems.</span></span> <span data-ttu-id="74591-183">Warstwa ta jest cienka.</span><span class="sxs-lookup"><span data-stu-id="74591-183">This layer is kept thin.</span></span> <span data-ttu-id="74591-184">Nie zawiera reguł biznesowych ani wiedzy, ale tylko koordynuje zadania i delegatów pracy do współpracy obiektów domeny w następnej warstwie w dół.</span><span class="sxs-lookup"><span data-stu-id="74591-184">It does not contain business rules or knowledge, but only coordinates tasks and delegates work to collaborations of domain objects in the next layer down.</span></span> <span data-ttu-id="74591-185">Nie ma stanu odzwierciedlającego sytuację biznesową, ale może mieć stan, który odzwierciedla postęp zadania dla użytkownika lub programu.</span><span class="sxs-lookup"><span data-stu-id="74591-185">It does not have state reflecting the business situation, but it can have state that reflects the progress of a task for the user or the program.</span></span>
