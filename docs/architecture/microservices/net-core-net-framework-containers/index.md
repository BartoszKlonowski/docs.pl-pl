# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Wybieranie między .NET Core i .NET Framework dla kontenerów platformy Docker

Istnieją dwie obsługiwane struktury do tworzenia konteneryzowanych aplikacji dodocker po stronie serwera za pomocą .NET: [.NET Framework i .NET Core](https://dotnet.microsoft.com/download). Współużytkują one wiele składników platformy .NET i można udostępniać kod między nimi. Istnieją jednak podstawowe różnice między nimi, a które struktury, których używasz, będą zależeć od tego, co chcesz osiągnąć. Ta sekcja zawiera wskazówki dotyczące wyboru każdej struktury.
