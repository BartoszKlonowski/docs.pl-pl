<span data-ttu-id="992ce-105">*Mikrousługi i aplikacji opartych na chmurze musi obejmować częściowe błędy, które z pewnością wystąpi po pewnym czasie. You must design your application to be resilient to those partial failures.*</span></span>

<span data-ttu-id="992ce-114">Wiele poszczególnych składników aplikacji powinny również zawierać funkcje monitorowania kondycji.</span><span class="sxs-lookup"><span data-stu-id="992ce-114">The many individual components of your application should also incorporate health monitoring features.</span></span> <span data-ttu-id="992ce-115">Postępując zgodnie z wytycznymi w tym rozdziale, można utworzyć aplikację, która może działać płynnie pomimo przejściowych przestojów lub normalnej czkawki, które występują w złożonych i opartych na chmurze wdrożeń.</span><span class="sxs-lookup"><span data-stu-id="992ce-115">By following the guidelines in this chapter, you can create an application that can work smoothly in spite of transient downtime or the normal hiccups that occur in complex and cloud-based deployments.</span></span>
