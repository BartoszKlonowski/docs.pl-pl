<span data-ttu-id="9b109-117">W niektórych środowiskach wdrażania mikrousług (nazywanych klastrami, które mają być omówione w późniejszej sekcji), odnajdowanie usługi jest wbudowane.</span><span class="sxs-lookup"><span data-stu-id="9b109-117">In some microservice deployment environments (called clusters, to be covered in a later section), service discovery is built-in.</span></span> <span data-ttu-id="9b109-118">Na przykład usługa Azure Container Service ze środowiskiem Kubernetes (AKS) może obsługiwać rejestrację i wyrejestrowanie wystąpienia usługi.</span><span class="sxs-lookup"><span data-stu-id="9b109-118">For example, an Azure Container Service with Kubernetes (AKS) environment can handle service instance registration and deregistration.</span></span> <span data-ttu-id="9b109-119">Uruchamia również serwer proxy na każdym hoście klastra, który odgrywa rolę routera odnajdowania po stronie serwera.</span><span class="sxs-lookup"><span data-stu-id="9b109-119">It also runs a proxy on each cluster host that plays the role of server-side discovery router.</span></span>
