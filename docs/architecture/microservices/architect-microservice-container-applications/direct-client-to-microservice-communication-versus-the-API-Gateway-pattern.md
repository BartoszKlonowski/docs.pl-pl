## <a name="drawbacks-of-the-api-gateway-pattern"></a><span data-ttu-id="84708-250">Wady wzorca interfejsu API</span><span class="sxs-lookup"><span data-stu-id="84708-250">Drawbacks of the API Gateway pattern</span></span>
