# <a name="local-environment"></a>Środowisko lokalne

Pierwszy krok w celu szybkiego startu lokalnie w trakcie jest można skonfigurować środowisko deweloperskie na komputerze lokalnym.
Temat .NET [wprowadzenie w ciągu 10 minut](https://www.microsoft.com/net/core) zawiera instrukcje dotyczące konfigurowania środowiska deweloperskiego lokalnego Mac, komputera lub Linux.

Alternatywnie możesz zainstalować [.NET Core SDK](http://dot.net/core) i [Visual Studio Code](https://code.visualstudio.com/).

## <a name="basic-application-development-flow"></a>Przepływ programowanie aplikacji w warstwie podstawowa

Utworzysz aplikacji przy użyciu [ `dotnet new` ](../../core/tools/dotnet-new.md) polecenia. To polecenie generuje pliki i zasoby niezbędne do aplikacji. Przewodniki Szybki Start, wszystkie użyj `console` typu aplikacji.

Inne polecenia użyjesz są [ `dotnet build` ](../../core/tools/dotnet-build.md) do tworzenia pliku wykonywalnego, a [ `dotnet run` ](../../core/tools/dotnet-run.md) uruchomienia pliku wykonywalnego.
