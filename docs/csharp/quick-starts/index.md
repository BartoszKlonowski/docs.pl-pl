# <a name="c-quickstarts"></a>Przewodniki Szybki Start C# #

Zapraszamy do skrócone podręczniki C#. Te rozpoczynać interakcyjne — lekcje, które można uruchomić w przeglądarce.

Wszystkie elementy szybkiego startu po lekcji Hello World jest dostępnych za pośrednictwem online przeglądania lub [w środowisku lokalnym projektowania](local-environment.md). Istnieją linki ułatwiające konfiguracji środowiska i przejdź do następnej procedury szybkiego startu na komputerze.
