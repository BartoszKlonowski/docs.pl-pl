# <a name="out-c-reference"></a>out (odwołanie w C#)

Możesz użyć `out` — słowo kluczowe w dwóch kontekstów:

- Jako [modyfikator parametru](out-parameter-modifier.md), pozwalającej można przekazać argument do metody przez odwołanie, a nie przez wartość.

- W [deklaracji parametru typu ogólnego](out-generic-modifier.md) dla interfejsów i delegatów, która określa, że parametr typu jest kowariantny.
