# <a name="out-c-reference"></a>out (odwołanie w C#)

Słowa kluczowego `out` można używać w dwóch kontekstach:

- Jako [modyfikator parametrów](out-parameter-modifier.md), który pozwala przekazać argument do metody przez odwołanie, a nie przez wartość.

- W [deklaracji parametrów typu ogólnego](out-generic-modifier.md) dla interfejsów i delegatów, który określa, że parametr typu jest kowariant.
