# <a name="out-c-reference"></a>out (odwołanie w C#)

Możesz użyć `out` słowa kluczowego w dwóch kontekstach:

- Jako [modyfikator parametru](out-parameter-modifier.md), który umożliwia przekazywanie argumentu do metody przez odwołanie, a nie przez wartość.

- W [deklaracjach parametrów typu ogólnego](out-generic-modifier.md) dla interfejsów i delegatów, które określają, że parametr typu jest współvariant.
