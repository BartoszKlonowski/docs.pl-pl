# <a name="out-c-reference"></a>out (odwołanie w C#)
Można użyć `out` — słowo kluczowe w dwóch kontekstów:

- Jako [modyfikator parametru](../../../csharp/language-reference/keywords/out-parameter-modifier.md), pozwalającej przekazujesz argument do metody przez odwołanie, a nie przez wartość.

- W [deklaracji parametru typu ogólnego](../../../csharp/language-reference/keywords/out-generic-modifier.md) dla interfejsów i delegatów, który określa, że parametr typu jest kowariantny.
