# <a name="ascending-c-reference"></a>ascending (odwołanie w C#)

`ascending`Kontekstowe słowo kluczowe jest używane w [klauzuli OrderBy](./orderby-clause.md) w wyrażeniach zapytania, aby określić, że kolejność sortowania jest od najmniejszych do największych. Ponieważ `ascending` jest to domyślny porządek sortowania, nie trzeba go określać.
