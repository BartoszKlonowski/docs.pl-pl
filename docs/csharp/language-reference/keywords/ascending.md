# <a name="ascending-c-reference"></a>ascending (odwołanie w C#)
`ascending` Kontekstowe słowo kluczowe jest używane w [klauzula orderby](../../../csharp/language-reference/keywords/orderby-clause.md) w wyrażeniach zapytań, aby określić, że kolejność sortowania jest od wartości najmniejszej do największej. Ponieważ `ascending` jest sortowanie domyślne kolejność, nie trzeba podać go.
