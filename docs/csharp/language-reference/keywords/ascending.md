# <a name="ascending-c-reference"></a>ascending (odwołanie w C#)
Kontekstowe `ascending` słowo kluczowe jest używane w [orderby klauzuli](./orderby-clause.md) w wyrażeniach kwerendy, aby określić, że kolejność sortowania jest od najmniejszego do największego. Ponieważ `ascending` jest to domyślna kolejność sortowania, nie trzeba jej określać.
