# <a name="operator-keywords-c-reference"></a>Słowa kluczowe operatora (odwołanie w C#)

Używane do wykonywania różnych akcji, takich jak tworzenie obiektów, sprawdzanie typu run-time obiektu uzyskiwania rozmiaru typu, a także innych działaniach. W tej sekcji przedstawiono następujące słowa kluczowe:

- [jako](as.md) Konwertuje obiekt typu zgodny.

- [await](await.md) wstrzymuje [async](async.md) metoda aż oczekiwane zadanie jest ukończone.

- [jest](is.md) sprawdza typu run-time obiektu, lub (począwszy od C# 7.0) sprawdza wyrażenie do wzorca.

- [new](new.md)

  - [New Operator](new-operator.md) tworzy obiekty.

  - [New — modyfikator](new-modifier.md) ukrywa dziedziczoną składową.

  - [New — ograniczenie](new-constraint.md) kwalifikuje się do parametru typu.

- [nameof](nameof.md) uzyskuje prostego ciągu (niekwalifikowanej) Nazwa zmiennej, typu lub elementu członkowskiego.

- [Operator sizeof](sizeof.md) uzyskuje rozmiar typu niezarządzanego.  

- [TypeOf](typeof.md) uzyskiwany <xref:System.Type?displayProperty=nameWithType> obiektu dla typu.  

- [stackalloc](stackalloc.md) przydziela blok pamięci na stosie.  

Następujące słowa kluczowe, które mogą być używane jako operatory i instrukcje, są objęte [instrukcji](statement-keywords.md) sekcji:

- [zaznaczone](checked.md) kontekstu określa zaznaczone.  

- [unchecked](unchecked.md) kontekście niesprawdzanym określa.
