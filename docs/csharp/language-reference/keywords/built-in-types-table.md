<span data-ttu-id="2bdc4-126">Użyj [typeof](typeof.md) operatorem w celu uzyskania <xref:System.Type?displayProperty=nameWithType> wystąpienia, który reprezentuje określony typ:</span><span class="sxs-lookup"><span data-stu-id="2bdc4-126">Use the [typeof](typeof.md) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>
