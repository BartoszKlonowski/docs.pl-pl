
<span data-ttu-id="dffed-126">Użyj operatora [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) , aby uzyskać <xref:System.Type?displayProperty=nameWithType> wystąpienie, które reprezentuje określony typ:</span><span class="sxs-lookup"><span data-stu-id="dffed-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Console.WriteLine(stringType.FullName);

// System.String
// System.Double
```
