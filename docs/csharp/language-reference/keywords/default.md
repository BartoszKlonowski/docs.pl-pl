# <a name="default-c-reference"></a>default (odwołanie w C#)

Słowa kluczowego `default` można użyć na dwa sposoby:

- Aby określić etykietę domyślną w [instrukcji`switch`](switch.md).
- Jako [operatora domyślnego lub literału](../operators/default.md) w celu utworzenia wartości domyślnej typu.
