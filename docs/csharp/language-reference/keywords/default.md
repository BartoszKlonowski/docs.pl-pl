# <a name="default-c-reference"></a>default (odwołanie w C#)

Słowa `default` kluczowego można używać na dwa sposoby:

- Aby określić etykietę domyślną w [ `switch` instrukcji](switch.md).
- Jako [domyślny operator lub literał](../operators/default.md) do produkcji wartości domyślnej typu.
