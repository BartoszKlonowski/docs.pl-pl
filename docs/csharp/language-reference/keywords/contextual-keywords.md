# <a name="contextual-keywords-c-reference"></a>Kontekstowe słowa kluczowe (odwołanie w C#)

Kontekstowe słowo kluczowe służy do zapewnienia określonego znaczenia w kodzie, ale nie jest słowem zastrzeżonym w języku C#. W tej sekcji wprowadzono następujące kontekstowe słowa kluczowe:  
  
|Słowo kluczowe|Opis|  
|-------------|-----------------|  
|[Dodaj](./add.md)|Definiuje akcesor zdarzenia niestandardowego, który jest wywoływany, gdy kod klienta subskrybuje zdarzenie.|  
|[async](./async.md)|Wskazuje, że modyfikowana metoda, wyrażenie lambda lub metoda anonimowa jest asynchroniczna.|  
|[await](../operators/await.md)|Zawiesza metodę asynchronia, dopóki nie zostanie ukończone oczekiwane zadanie.|  
|[Dynamiczne](../builtin-types/reference-types.md)|Definiuje typ odwołania, który włącza operacje, w których występuje, aby ominąć sprawdzanie typu kompilacji w czasie.|  
|[get](./get.md)|Definiuje metodę akcesora dla właściwości lub indeksatora.|  
|[Globalne](../operators/namespace-alias-qualifier.md)|Alias globalnego obszaru nazw, który w przeciwnym razie nie jest nazwany.|  
|[Częściowe](./partial-type.md)|Definiuje klasy częściowe, struktury i interfejsy w tej samej jednostce kompilacji.|  
|[Usunąć](./remove.md)|Definiuje akcesor zdarzenia niestandardowego, który jest wywoływany, gdy kod klienta anuluje subskrypcję ze zdarzenia.|  
|[Ustawić](./set.md)|Definiuje metodę akcesora dla właściwości lub indeksatora.|  
|[Wartość](./value.md)|Służy do ustawiania akcesorów i dodawania lub usuwania programów obsługi zdarzeń.|  
|[Var](./var.md)|Włącza typ zmiennej zadeklarowanej w zakresie metody, które mają być określone przez kompilator.|  
|[Kiedy](when.md)|Określa warunek filtru `catch` dla `case` bloku lub `switch` etykiety instrukcji.|
|[Gdzie](./where-generic-type-constraint.md)|Dodaje ograniczenia do deklaracji ogólnej. (Zobacz [też, gdzie](./where-clause.md)).|  
|[yield](./yield.md)|Używany w bloku iteratorem, aby zwrócić wartość do obiektu wyliczacza lub zasygnalizować koniec iteracji.|  
  
 Wszystkie słowa kluczowe kwerendy wprowadzone w języku C# 3.0 są również kontekstowe. Aby uzyskać więcej informacji, zobacz [Słowa kluczowe kwerendy (LINQ)](./query-keywords.md).
