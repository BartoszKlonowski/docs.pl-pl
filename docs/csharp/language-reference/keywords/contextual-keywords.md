# <a name="contextual-keywords-c-reference"></a>Kontekstowe słowa kluczowe (odwołanie w C#)
Kontekstowe słowo kluczowe służy do zapewnienia określone znaczenie w kodzie, ale nie jest ona słowem zastrzeżonym w języku C#. Kontekstowe słowa kluczowe następujące zostały wprowadzone w tej sekcji:  
  
|Słowo kluczowe|Opis|  
|-------------|-----------------|  
|[Dodaj](../../../csharp/language-reference/keywords/add.md)|Definiuje metody dostępu zdarzeń niestandardowych, które jest wywoływane, gdy kod klienta subskrybuje zdarzenia.|  
|[asynchroniczne](../../../csharp/language-reference/keywords/async.md)|Wskazuje, że modyfikowana metoda, wyrażenie lambda lub metoda anonimowa jest asynchroniczna.|  
|[await](../../../csharp/language-reference/keywords/await.md)|Do czasu ukończenia zadania oczekiwano wstrzymuje wykonywanie metody asynchronicznej.|  
|[dynamiczne](../../../csharp/language-reference/keywords/dynamic.md)|Definiuje typ odwołania, który umożliwia operacji, w których występuje Pomiń sprawdzanie typów w czasie kompilacji.|  
|[Pobierz](../../../csharp/language-reference/keywords/get.md)|Definiuje metodę dostępu do właściwości lub indeksatora.|  
|[globalne](../../../csharp/language-reference/keywords/global.md)|Określa domyślną globalnej przestrzeni nazw, które w przeciwnym razie jest bez nazwy.|  
|[częściowe](../../../csharp/language-reference/keywords/partial-type.md)|Definiuje częściowej klasy, struktury i interfejsy w tej samej jednostce kompilacyjnej.|  
|[Usuń](../../../csharp/language-reference/keywords/remove.md)|Definiuje akcesor zdarzenie niestandardowe, które jest wywoływane, gdy kod klienta cofnięć subskrypcji z zdarzenia.|  
|[zestaw](../../../csharp/language-reference/keywords/set.md)|Definiuje metodę dostępu do właściwości lub indeksatora.|  
|[wartość](../../../csharp/language-reference/keywords/value.md)|Pozwala ustawić metody dostępu i Dodaj lub usuń programy obsługi zdarzeń.|  
|[var](../../../csharp/language-reference/keywords/var.md)|Umożliwia typu zmiennej zadeklarowanej w zakresie metoda będzie określany przez kompilator.|  
|[Kiedy](when.md)|Określa warunek filtrowania `catch` bloku lub `case` etykietę `switch` instrukcji.|
|[gdzie](../../../csharp/language-reference/keywords/where-generic-type-constraint.md)|Dodaje ograniczenia do deklaracji ogólnej. (Zobacz też [gdzie](../../../csharp/language-reference/keywords/where-clause.md)).|  
|[YIELD](../../../csharp/language-reference/keywords/yield.md)|Używany w blokiem iteratora, aby zwrócić wartość do obiektu modułu wyliczającego lub sygnalizuje koniec iteracji.|  
  
 Wszystkie wprowadzone w języku C# 3.0 słowa kluczowe zapytania są również kontekstowych. Aby uzyskać więcej informacji, zobacz [słowa kluczowe zapytania (LINQ)](../../../csharp/language-reference/keywords/query-keywords.md).
