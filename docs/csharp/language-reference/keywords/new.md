# <a name="new-c-reference"></a>new (odwołanie w C#)
W języku C# `new` — słowo kluczowe może służyć jako operatora, modyfikujący lub ograniczenie.  
  
 [new, operator](../../../csharp/language-reference/keywords/new-operator.md)  
 Używany do tworzenia obiektów i wywołania konstruktorów.  
  
 [new, modyfikator](../../../csharp/language-reference/keywords/new-modifier.md)  
 Używane do ukrywania dziedziczonego elementu członkowskiego z elementem członkowskim klasy podstawowej.  
  
 [new, ograniczenie](../../../csharp/language-reference/keywords/new-constraint.md)  
 Używany do ograniczania typów, które mogą być używane jako argumenty dla parametru typu w deklaracji ogólnej.
