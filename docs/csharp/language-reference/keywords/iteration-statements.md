# <a name="iteration-statements-c-reference"></a>Instrukcje iteracji (odwołanie w C#)

Pętle można utworzyć za pomocą instrukcji iteracji. Instrukcje Przyczyna osadzone instrukcje iteracji za wykonywane wiele razy z zastrzeżeniem kryteriów zakończenia pętli. Te instrukcje są wykonywane w kolejności, chyba że [instrukcji jump](jump-statements.md) zostanie osiągnięty.

Następujące słowa kluczowe są używane w instrukcjach iteracji:

- [do](do.md)

- [for](for.md)

- [foreach, in](foreach-in.md)

- [while](while.md)
