# <a name="types-c-reference"></a>Types (odwołanie w C#)

Wpisywanie system C# zawiera następujące kategorie:

- [Typy wartości](value-types.md)

- [Typy odwołań](reference-types.md)

- [Typy wskaźników](../../programming-guide/unsafe-code-pointers/pointer-types.md)

 Zmienne, które są typami wartości przechowywania danych, a te, które są typami odwołań są przechowywane odwołania do danych rzeczywistych. Wystąpienia typu referencyjnego są również określane jako obiekty. Typy wskaźnika, które mogą być używane tylko w [niebezpieczne](unsafe.md) trybu.

 Można przekonwertować typu wartości na typ odwołania i z powrotem do typu wartości przy użyciu [opakowywanie i rozpakowywanie](../../../csharp/programming-guide/types/boxing-and-unboxing.md). Z wyjątkiem opakowanym typem wartościowym nie można przekonwertować typu odwołania do typu wartości.

 Ta sekcja wprowadza również [void](void.md).
