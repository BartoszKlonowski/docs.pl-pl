## <a name="remarks"></a><span data-ttu-id="dd646-132">Uwagi</span><span class="sxs-lookup"><span data-stu-id="dd646-132">Remarks</span></span>  
  
-   <span data-ttu-id="dd646-133">Jawna konwersja liczbowa może spowodować utratę dokładności lub wynik przerzucane wyjątków.</span><span class="sxs-lookup"><span data-stu-id="dd646-133">The explicit numeric conversion may cause loss of precision or result in throwing exceptions.</span></span>  
  
-   <span data-ttu-id="dd646-134">Podczas konwertowania `decimal` wartości na typ całkowity, ta wartość jest zaokrąglana do zero do najbliższej wartości całkowitej.</span><span class="sxs-lookup"><span data-stu-id="dd646-134">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="dd646-135">Jeśli wynikowej wartości całkowite jest spoza zakresu typu docelowego <xref:System.OverflowException> jest generowany.</span><span class="sxs-lookup"><span data-stu-id="dd646-135">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
-   <span data-ttu-id="dd646-136">Podczas konwersji z `double` lub `float` wartości na typ całkowity, wartość zostanie obcięta.</span><span class="sxs-lookup"><span data-stu-id="dd646-136">When you convert from a `double` or `float` value to an integral type, the value is truncated.</span></span> <span data-ttu-id="dd646-137">Jeśli całkowitą wartość wynikowa znajduje się poza zakresem wartości docelowej, wynik zależy od przepełnienie sprawdzanie kontekstu.</span><span class="sxs-lookup"><span data-stu-id="dd646-137">If the resulting integral value is outside the range of the destination value, the result depends on the overflow checking context.</span></span> <span data-ttu-id="dd646-138">W kontekście zaznaczone `OverflowException` jest zgłoszony w kontekście niezaznaczone, wynik jest wartością nieokreślonego typu docelowego.</span><span class="sxs-lookup"><span data-stu-id="dd646-138">In a checked context, an `OverflowException` is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
-   <span data-ttu-id="dd646-139">Podczas konwertowania `double` do `float`, `double` wartość jest zaokrąglana do najbliższego `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="dd646-139">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="dd646-140">Jeśli `double` wartość jest zbyt mała lub zbyt duży, aby zmieścić na typ docelowy zostanie zero lub nieskończoności.</span><span class="sxs-lookup"><span data-stu-id="dd646-140">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
-   <span data-ttu-id="dd646-141">Podczas konwertowania `float` lub `double` do `decimal`, wartości źródłowej jest konwertowana na `decimal` reprezentacja i zaokrąglona do najbliższej liczby po przecinku 28, jeśli jest to wymagane.</span><span class="sxs-lookup"><span data-stu-id="dd646-141">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="dd646-142">W zależności od wartości wartość źródła może wystąpić jeden z następujących wyników:</span><span class="sxs-lookup"><span data-stu-id="dd646-142">Depending on the value of the source value, one of the following results may occur:</span></span>  
  
    -   <span data-ttu-id="dd646-143">Jeśli wartość źródła jest za mały, aby mogły być reprezentowane jako `decimal`, wynik wynosi zero.</span><span class="sxs-lookup"><span data-stu-id="dd646-143">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  
  
    -   <span data-ttu-id="dd646-144">Jeśli wartość źródła jest wartością typu NaN (nieliczbową), nieskończoności, lub zbyt duży, może być reprezentowana jako `decimal`, `OverflowException` jest generowany.</span><span class="sxs-lookup"><span data-stu-id="dd646-144">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an `OverflowException` is thrown.</span></span>  
  
-   <span data-ttu-id="dd646-145">Podczas konwertowania `decimal` do `float` lub `double`, `decimal` wartość jest zaokrąglana do najbliższego `double` lub `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="dd646-145">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
