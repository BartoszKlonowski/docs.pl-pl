## <a name="remarks"></a><span data-ttu-id="f94d6-132">Uwagi</span><span class="sxs-lookup"><span data-stu-id="f94d6-132">Remarks</span></span>  
  
-   <span data-ttu-id="f94d6-133">Jawna konwersja liczbowa może spowodować utratę precyzji lub wynik zgłaszanie wyjątków.</span><span class="sxs-lookup"><span data-stu-id="f94d6-133">The explicit numeric conversion may cause loss of precision or result in throwing exceptions.</span></span>  
  
-   <span data-ttu-id="f94d6-134">Podczas konwertowania `decimal` wartości na typ całkowity, ta wartość jest zaokrąglana w kierunku zera do najbliższej wartości całkowitej.</span><span class="sxs-lookup"><span data-stu-id="f94d6-134">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="f94d6-135">Jeśli wynikową wartość całkowitą znajduje się poza zakresem typu docelowego <xref:System.OverflowException> zgłaszany.</span><span class="sxs-lookup"><span data-stu-id="f94d6-135">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
-   <span data-ttu-id="f94d6-136">Podczas konwersji z `double` lub `float` wartości na typ całkowity, wartość zostanie obcięta.</span><span class="sxs-lookup"><span data-stu-id="f94d6-136">When you convert from a `double` or `float` value to an integral type, the value is truncated.</span></span> <span data-ttu-id="f94d6-137">Jeśli całkowitą wartość wynikowa znajduje się poza zakresem wartości docelowej, wynik zależy od przepełnienia, sprawdzając kontekst.</span><span class="sxs-lookup"><span data-stu-id="f94d6-137">If the resulting integral value is outside the range of the destination value, the result depends on the overflow checking context.</span></span> <span data-ttu-id="f94d6-138">W kontekście sprawdzanym `OverflowException` jest zgłaszany, gdy w kontekście niesprawdzanym, wynik jest wartością nieokreślonego typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="f94d6-138">In a checked context, an `OverflowException` is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
-   <span data-ttu-id="f94d6-139">Podczas konwertowania `double` do `float`, `double` wartość jest zaokrąglana do najbliższej `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="f94d6-139">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="f94d6-140">Jeśli `double` wartość jest zbyt mała lub zbyt duży, aby zmieścić w typ docelowy wynik będzie zero lub nieskończoności.</span><span class="sxs-lookup"><span data-stu-id="f94d6-140">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
-   <span data-ttu-id="f94d6-141">Podczas konwertowania `float` lub `double` do `decimal`, wartość źródłowa jest konwertowany na `decimal` reprezentacji i zaokrąglony do najbliższej liczby po 28 miejsce dziesiętne, jeśli jest to wymagane.</span><span class="sxs-lookup"><span data-stu-id="f94d6-141">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="f94d6-142">W zależności od wartości wartość źródła może wystąpić jeden z poniższych wyników:</span><span class="sxs-lookup"><span data-stu-id="f94d6-142">Depending on the value of the source value, one of the following results may occur:</span></span>  
  
    -   <span data-ttu-id="f94d6-143">Jeśli wartość źródłowa jest za mały, może być reprezentowana jako `decimal`, wynik staje się zerem.</span><span class="sxs-lookup"><span data-stu-id="f94d6-143">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  
  
    -   <span data-ttu-id="f94d6-144">Jeśli wartość źródłowa jest wartością typu NaN (nieliczbową), nieskończoność, lub zbyt duży, może być reprezentowana jako `decimal`, `OverflowException` zgłaszany.</span><span class="sxs-lookup"><span data-stu-id="f94d6-144">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an `OverflowException` is thrown.</span></span>  
  
-   <span data-ttu-id="f94d6-145">Podczas konwertowania `decimal` do `float` lub `double`, `decimal` wartość jest zaokrąglana do najbliższej `double` lub `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="f94d6-145">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
