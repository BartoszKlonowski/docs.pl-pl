## <a name="remarks"></a><span data-ttu-id="96772-130">Uwagi</span><span class="sxs-lookup"><span data-stu-id="96772-130">Remarks</span></span>  
  
- <span data-ttu-id="96772-131">Jawna konwersja liczbowa może spowodować utratę precyzji lub wynik w zostanie zgłoszony wyjątek, zwykle <xref:System.OverflowException>.</span><span class="sxs-lookup"><span data-stu-id="96772-131">The explicit numeric conversion may cause loss of precision or result in throwing an exception, typically an <xref:System.OverflowException>.</span></span>  

- <span data-ttu-id="96772-132">Podczas konwertowania wartości typu całkowitego innego typu całkowitego, wynik zależy od przeciążenia [sprawdzanie kontekstu](checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="96772-132">When you convert a value of an integral type to another integral type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="96772-133">W kontekście sprawdzanym konwersja powiedzie się, gdy wartość źródła znajduje się w zakresie typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="96772-133">In a checked context, the conversion succeeds if the source value is within the range of the destination type.</span></span> <span data-ttu-id="96772-134">W przeciwnym razie <xref:System.OverflowException> zgłaszany.</span><span class="sxs-lookup"><span data-stu-id="96772-134">Otherwise, an <xref:System.OverflowException> is thrown.</span></span> <span data-ttu-id="96772-135">W kontekście niesprawdzanym konwersji zawsze powiedzie się i będzie kontynuowane w następujący sposób:</span><span class="sxs-lookup"><span data-stu-id="96772-135">In an unchecked context, the conversion always succeeds, and proceeds as follows:</span></span>

  - <span data-ttu-id="96772-136">Jeśli typ źródła jest większy niż typ docelowy, a następnie wartość źródłowa zostanie obcięta przez odrzucenie jego "dodatkowe" najbardziej znaczące bity.</span><span class="sxs-lookup"><span data-stu-id="96772-136">If the source type is larger than the destination type, then the source value is truncated by discarding its "extra" most significant bits.</span></span> <span data-ttu-id="96772-137">Wynik jest traktowane jako wartość typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="96772-137">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="96772-138">Jeśli typ źródła jest mniejszy niż typ docelowy, następnie wartość źródłowa jest rozszerzona o znak lub rozszerzone zero tak, aby jest taki sam rozmiar jak typ docelowy.</span><span class="sxs-lookup"><span data-stu-id="96772-138">If the source type is smaller than the destination type, then the source value is either sign-extended or zero-extended so that it is the same size as the destination type.</span></span> <span data-ttu-id="96772-139">Rozszerzenia znaku jest używana, jeśli typ źródła jest podpisany; rozszerzenie zero jest używana, jeśli typ źródła jest niepodpisany.</span><span class="sxs-lookup"><span data-stu-id="96772-139">Sign-extension is used if the source type is signed; zero-extension is used if the source type is unsigned.</span></span> <span data-ttu-id="96772-140">Wynik jest traktowane jako wartość typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="96772-140">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="96772-141">Jeśli typ źródła jest taki sam rozmiar jak typ docelowy, wartość źródłowa jest traktowany jako wartość typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="96772-141">If the source type is the same size as the destination type, then the source value is treated as a value of the destination type.</span></span>
  
- <span data-ttu-id="96772-142">Podczas konwertowania `decimal` wartości na typ całkowity, ta wartość jest zaokrąglana w kierunku zera do najbliższej wartości całkowitej.</span><span class="sxs-lookup"><span data-stu-id="96772-142">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="96772-143">Jeśli wynikową wartość całkowitą znajduje się poza zakresem typu docelowego <xref:System.OverflowException> zgłaszany.</span><span class="sxs-lookup"><span data-stu-id="96772-143">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="96772-144">Podczas konwertowania `double` lub `float` wartości na typ całkowity, ta wartość jest zaokrąglana w kierunku zera do najbliższej wartości całkowitej.</span><span class="sxs-lookup"><span data-stu-id="96772-144">When you convert a `double` or `float` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="96772-145">Jeśli całkowitą wartość wynikowa znajduje się poza zakresem typu miejsca docelowego, wynik zależy od przeciążenia [sprawdzanie kontekstu](checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="96772-145">If the resulting integral value is outside the range of the destination type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="96772-146">W kontekście sprawdzanym <xref:System.OverflowException> jest zgłaszany, gdy w kontekście niesprawdzanym, wynik jest wartością nieokreślonego typu miejsca docelowego.</span><span class="sxs-lookup"><span data-stu-id="96772-146">In a checked context, an <xref:System.OverflowException> is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
- <span data-ttu-id="96772-147">Podczas konwertowania `double` do `float`, `double` wartość jest zaokrąglana do najbliższej `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="96772-147">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="96772-148">Jeśli `double` wartość jest zbyt mała lub zbyt duży, aby zmieścić w typ docelowy wynik będzie zero lub nieskończoności.</span><span class="sxs-lookup"><span data-stu-id="96772-148">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
- <span data-ttu-id="96772-149">Podczas konwertowania `float` lub `double` do `decimal`, wartość źródłowa jest konwertowany na `decimal` reprezentacji i zaokrąglony do najbliższej liczby po 28 miejsce dziesiętne, jeśli jest to wymagane.</span><span class="sxs-lookup"><span data-stu-id="96772-149">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="96772-150">W zależności od wartości wartość źródła może wystąpić jeden z poniższych wyników:</span><span class="sxs-lookup"><span data-stu-id="96772-150">Depending on the value of the source value, one of the following results may occur:</span></span>  

  - <span data-ttu-id="96772-151">Jeśli wartość źródłowa jest za mały, może być reprezentowana jako `decimal`, wynik staje się zerem.</span><span class="sxs-lookup"><span data-stu-id="96772-151">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  

  - <span data-ttu-id="96772-152">Jeśli wartość źródłowa jest wartością typu NaN (nieliczbową), nieskończoność, lub zbyt duży, może być reprezentowana jako `decimal`, <xref:System.OverflowException> zgłaszany.</span><span class="sxs-lookup"><span data-stu-id="96772-152">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="96772-153">Podczas konwertowania `decimal` do `float` lub `double`, `decimal` wartość jest zaokrąglana do najbliższej `double` lub `float` wartość.</span><span class="sxs-lookup"><span data-stu-id="96772-153">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
