# <a name="using-c-reference"></a>using (odwołanie w C#)
`using` — Słowo kluczowe ma dwa główne zastosowania:  
  
-   Jako dyrektywy, gdy jest używany do tworzenia aliasów przestrzeni nazw lub importowania typów zdefiniowanych w innych przestrzeniach nazw. Zobacz [dyrektywa using](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Za pomocą instrukcji, gdy definiuje zakres przed upływem którego obiekt zostanie zlikwidowany. Zobacz [za pomocą instrukcji](../../../csharp/language-reference/keywords/using-statement.md).  
  
Ponadto [przy użyciu statycznego](using-static.md) dyrektywy pozwala zdefiniować typu którego statyczne elementy członkowskie użytkownik ma dostęp bez określania nazwy typu.
