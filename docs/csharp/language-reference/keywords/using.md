# <a name="using-c-reference"></a>using (odwołanie w C#)
`using` — Słowo kluczowe ma dwa główne zastosowań:  
  
-   Jak dyrektywa, gdy jest używany, aby utworzyć alias dla przestrzeni nazw lub importowania typów zdefiniowanych w innych przestrzeniach nazw. Zobacz [użycie dyrektywy](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Jako instrukcji, gdy definiuje zakres na końcu którego obiekt zostanie usunięty. Zobacz [za pomocą instrukcji](../../../csharp/language-reference/keywords/using-statement.md).  
  
Ponadto [przy użyciu statycznej](using-static.md) dyrektywy pozwala zdefiniować typ którego statyczne elementy członkowskie, możesz uzyskać dostęp bez określenia nazwy typu.
