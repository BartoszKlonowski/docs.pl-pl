# <a name="using-c-reference"></a>using (odwołanie w C#)

Słowo `using` kluczowe ma trzy główne zastosowania:

- [Using Instrukcji](using-statement.md) definiuje zakres, na końcu którego obiekt zostanie usunięty.
- Using [dyrektywy](using-directive.md) tworzy alias dla obszaru nazw lub importuje typy zdefiniowane w innych obszarach nazw.
- [Using static dyrektywy](using-static.md) importuje elementy członkowskie jednej klasy.
