# <a name="query-keywords-c-reference"></a>Słowa kluczowe zapytania (odwołanie w C#)
Ta sekcja zawiera kontekstowe słowa kluczowe, używać w wyrażeniach zapytań.  
  
## <a name="in-this-section"></a>W tej sekcji  
  
|Klauzula|Opis|  
|------------|-----------------|  
|[z](../../../csharp/language-reference/keywords/from-clause.md)|Określa źródło danych i zmienną zakresu (podobnie jak zmienna iteracji).|  
|[gdzie](../../../csharp/language-reference/keywords/where-clause.md)|Filtry źródła oparte na co najmniej jednego wyrażenia logiczne rozdzielone i operatory logiczne i elementy ( `&&` lub <code>&#124;&#124;</code> ).|  
|[Wybierz](../../../csharp/language-reference/keywords/select-clause.md)|Określa typ i kształtu, który ma elementy w sekwencji zwrócony podczas wykonywania zapytania.|  
|[grupy](../../../csharp/language-reference/keywords/group-clause.md)|Wyniki zapytania grup zgodnie z określoną wartość klucza.|  
|[do](../../../csharp/language-reference/keywords/into.md)|Zawiera identyfikator, który może służyć jako odwołanie do wyników sprzężenia, grupy lub w klauzuli select.|  
|[OrderBy](../../../csharp/language-reference/keywords/orderby-clause.md)|Sortuj zapytania wyniki w kolejności rosnącej lub malejącej według domyślna funkcja porównująca dla typu elementu.|  
|[sprzężenia](../../../csharp/language-reference/keywords/join-clause.md)|Łączy dwa źródła danych na podstawie porównania równości między dwoma określone kryteria dopasowywania.|  
|[Let](../../../csharp/language-reference/keywords/let-clause.md)|Wprowadza zmienną zakresu do przechowywania wyników wyrażenia podrzędnego w wyrażeniu zapytania.|  
|[w](../../../csharp/language-reference/keywords/in.md)|Kontekstowe słowo kluczowe w [sprzężenia](../../../csharp/language-reference/keywords/join-clause.md) klauzuli.|  
|[na](../../../csharp/language-reference/keywords/on.md)|Kontekstowe słowo kluczowe w [sprzężenia](../../../csharp/language-reference/keywords/join-clause.md) klauzuli.|  
|[równa się](../../../csharp/language-reference/keywords/equals.md)|Kontekstowe słowo kluczowe w [sprzężenia](../../../csharp/language-reference/keywords/join-clause.md) klauzuli.|  
|[przez](../../../csharp/language-reference/keywords/by.md)|Kontekstowe słowo kluczowe w [grupy](../../../csharp/language-reference/keywords/group-clause.md) klauzuli.|  
|[Rosnąca](../../../csharp/language-reference/keywords/ascending.md)|Kontekstowe słowo kluczowe w [orderby](../../../csharp/language-reference/keywords/orderby-clause.md) klauzuli.|  
|[Malejąco](../../../csharp/language-reference/keywords/descending.md)|Kontekstowe słowo kluczowe w [orderby](../../../csharp/language-reference/keywords/orderby-clause.md) klauzuli.|
