# <a name="endregion-c-reference"></a>#endregion (odwołanie w C#)

`#endregion` oznacza koniec bloku [#region](./preprocessor-region.md) . Na przykład:  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
