# <a name="lt-operator-c-reference"></a>&lt;= — Operator (odwołanie w C#)
Wszystkie typy numeryczne i wyliczenie zdefiniować "mniejsze niż lub równe" operator relacyjny (`<=`) zwracającego `true` Jeśli pierwszy operand jest mniejsza niż druga Strona, `false` inaczej.  
  
## <a name="remarks"></a>Uwagi  
 Typy definiowane przez użytkownika mogą przeciążać operator `<=`. Aby uzyskać więcej informacji, zobacz [operator](../../../csharp/language-reference/keywords/operator.md). Jeśli `<=` jest przeciążona, [ >= ](../../../csharp/language-reference/operators/greater-than-equal-operator.md) musi również być przeciążony. Operacje na typach całkowitych w wyliczeniach są zazwyczaj dozwolone.
