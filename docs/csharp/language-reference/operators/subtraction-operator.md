# <a name="--operator-c-reference"></a>Operator - (odwołanie w C#)
`-` Operator może działać jako jednoargumentowy lub operator binarny.  
  
## <a name="remarks"></a>Uwagi  
 Jednoargumentowy `-` operatory są wstępnie zdefiniowane dla wszystkich typów numerycznych. Wynik jednoargumentowy `-` operacji na typ liczbowy jest liczbowych Negacja argument.  
  
 Binarny `-` operatory są wstępnie zdefiniowane dla wszystkich typów liczbowych i wyliczenia do odjęcia drugiego operandu od pierwszego.  
  
 Typy delegatów również udostępniać dane binarne `-` operatora, który przeprowadza usunięcie obiektu delegowanego.  
  
 Typy definiowane przez użytkownika można przeciążać jednoargumentowego `-` i binarne `-` operatorów. Aby uzyskać więcej informacji, zobacz [— operator (odwołanie w C#)](../../../csharp/language-reference/keywords/operator.md).
