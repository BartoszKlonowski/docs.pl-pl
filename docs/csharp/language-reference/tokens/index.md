# <a name="c-special-characters"></a>Znaki specjalne C#

Znaki specjalne są wstępnie zdefiniowane, kontekstowe znaki, które modyfikuje elementu programu (literałem, identyfikator lub nazwa atrybutu), do którego one dołączany początku. C# obsługuje następujące znaki specjalne: 

- [@](../../../csharp/language-reference/tokens/verbatim.md), znak identyfikator dosłownego wyrażenia. 

- [$](../../../csharp/language-reference/tokens/interpolated.md), znaków ciągu interpolowanym.
