# <a name="compiler-error-cs0134"></a>Błąd kompilatora CS0134
'Zmienna' jest typu "type". Pole stałej typu referencyjnego innego niż string można zainicjować tylko przy użyciu wartości null.  
  
 Wyrażenie stałe jest wyrażeniem, które mogą zostać w pełni oszacowane w czasie kompilacji. Ponieważ jedynym sposobem, aby utworzyć inną niż null wartości typu odwołania się do zastosowania operatora new i operatora new nie jest dozwolona w wyrażeniu stałej, jedyną możliwą wartością dla stałych typu referencyjnego innego niż ciąg ma wartość null.  
  
 Jeśli napotkasz ten błąd przy próbie utworzenia [const](../../../csharp/language-reference/keywords/const.md) tablicy ciągów rozwiązaniem jest utworzenie tablicy [tylko do odczytu](../../../csharp/language-reference/keywords/readonly.md)i zainicjować go w konstruktorze.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0134.  
  
```csharp  
// CS0134.cs  
// compile with: /target:library  
class MyTest {}   
  
class MyClass  
{  
   const MyTest test = new MyTest();   // CS0134  
  
   //OK  
   const MyTest test2 = null;  
   const System.String test3 = "test";  
}  
```
