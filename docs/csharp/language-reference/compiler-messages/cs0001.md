# <a name="compiler-error-cs0001"></a>Błąd kompilatora CS0001
Wewnętrzny błąd kompilatora  
  
 Spróbuj określić, czy kompilacja nie kończy się niepowodzeniem, ponieważ nie można przeanalizować nieoczekiwanej składni. Jeśli ten błąd występuje często, skontaktuj się z firmą Microsoft.
