# <a name="compiler-error-cs0001"></a>Błąd kompilatora CS0001
Wewnętrzny błąd kompilatora  
  
 Spróbuj określić, czy kompilacja nie kończy się niepowodzeniem z powodu niemożności przeanalizowania nieoczekiwanej składni. Jeśli otrzymujesz ten błąd często, skontaktuj się z firmą Microsoft.
