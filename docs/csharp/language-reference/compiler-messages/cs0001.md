# <a name="compiler-error-cs0001"></a>Błąd kompilatora CS0001

Wewnętrzny błąd kompilatora

 Spróbuj określić, czy kompilator kończy się niepowodzeniem z powodu niemożności przeanalizowania nieoczekiwanej składni. Jeśli ten błąd wystąpi wielokrotnie, skontaktuj się z firmą Microsoft.
