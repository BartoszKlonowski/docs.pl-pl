# <a name="compiler-error-cs0001"></a>Błąd kompilatora CS0001
Błąd kompilatora wewnętrznego

 Spróbuj ustalić, czy kompilator nie działa z powodu jego niemożności przeanalizowania nieoczekiwanej składni. Jeśli ten błąd jest wielokrotnie wyświetlany, skontaktuj się z firmą Microsoft.
