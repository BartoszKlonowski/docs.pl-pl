<span data-ttu-id="870f9-103">Oczekiwano wartości typu całkowitego</span><span class="sxs-lookup"><span data-stu-id="870f9-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="870f9-104">Zmienna została użyta w sytuacji, w której był wymagany typ całkowity.</span><span class="sxs-lookup"><span data-stu-id="870f9-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="870f9-105">Aby uzyskać więcej informacji, zobacz [Typy](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="870f9-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="870f9-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="870f9-106">Example</span></span>  
 <span data-ttu-id="870f9-107">Ten błąd może wystąpić, gdy konwersja jest niedostępna lub jeśli dostępna konwersja niejawna spowodowała zwrócenie niejednoznacznego wyniku.</span><span class="sxs-lookup"><span data-stu-id="870f9-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="870f9-108">Poniższy przykład generuje błąd CS0151.</span><span class="sxs-lookup"><span data-stu-id="870f9-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```
