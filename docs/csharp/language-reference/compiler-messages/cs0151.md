## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="33af4-107">Przykład niejednoznacznej konwersji</span><span class="sxs-lookup"><span data-stu-id="33af4-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="33af4-108">Ten błąd może wystąpić, gdy nie ma konwersji lub jeśli dostępne niejawne konwersje powodują powstanie niejednoznacznej sytuacji.</span><span class="sxs-lookup"><span data-stu-id="33af4-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="33af4-109">Poniższy przykład generuje CS0151:</span><span class="sxs-lookup"><span data-stu-id="33af4-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="33af4-110">Przykład metody void</span><span class="sxs-lookup"><span data-stu-id="33af4-110">Example of void method</span></span>
