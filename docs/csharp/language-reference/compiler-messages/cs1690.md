# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="165ec-102">Ostrzeżenie kompilatora (poziom 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="165ec-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="165ec-103">Dostępu do członka "członek" może spowodować wyjątek czasu wykonywania, ponieważ jest to pole klasy marshal-by-reference</span><span class="sxs-lookup"><span data-stu-id="165ec-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="165ec-104">To ostrzeżenie występuje podczas próby wywołania metody, właściwości lub indeksatora w elemencie członkowskim klasy, która jest pochodną <xref:System.MarshalByRefObject>, a element członkowski jest typem wartości.</span><span class="sxs-lookup"><span data-stu-id="165ec-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="165ec-105">Obiekty, które dziedziczą z `MarshalByRefObject` zwykle mają być przekazywane przez odwołanie w domenie aplikacji.</span><span class="sxs-lookup"><span data-stu-id="165ec-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="165ec-106">Jeśli kiedykolwiek dowolny kod próbuje uzyskać bezpośredniego dostępu do elementu członkowskiego typu wartości takiego obiektu w domenie aplikacji, wystąpi wyjątek czasu wykonywania.</span><span class="sxs-lookup"><span data-stu-id="165ec-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="165ec-107">Aby usunąć to ostrzeżenie, skopiuj element członkowski do zmiennej lokalnej i wywołać metodę dla tej zmiennej.</span><span class="sxs-lookup"><span data-stu-id="165ec-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="165ec-108">Poniższy przykład generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="165ec-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
