# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="dee44-102">Ostrzeżenie kompilatora (poziom 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="dee44-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="dee44-103">Dostęp do składowej w elemencie "member" może spowodować wyjątek czasu wykonywania, ponieważ jest to pole klasy marshal-by-Reference</span><span class="sxs-lookup"><span data-stu-id="dee44-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="dee44-104">To ostrzeżenie występuje podczas próby wywołania metody, właściwości lub indeksatora w składowej klasy, która dziedziczy z <xref:System.MarshalByRefObject>, a element członkowski jest typem wartości.</span><span class="sxs-lookup"><span data-stu-id="dee44-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="dee44-105">Obiekty dziedziczące z `MarshalByRefObject` są zwykle przeznaczone do organizowania przez odwołanie w domenie aplikacji.</span><span class="sxs-lookup"><span data-stu-id="dee44-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="dee44-106">Jeśli jakikolwiek kod kiedykolwiek próbuje bezpośrednio uzyskać dostęp do elementu członkowskiego typu wartości takiego obiektu w domenie aplikacji, wystąpi <xref:System.InvalidOperationException> środowiska uruchomieniowego.</span><span class="sxs-lookup"><span data-stu-id="dee44-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="dee44-107">Aby usunąć ostrzeżenie, najpierw skopiuj element członkowski do zmiennej lokalnej i Wywołaj metodę dla tej zmiennej.</span><span class="sxs-lookup"><span data-stu-id="dee44-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="dee44-108">Poniższy przykład generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="dee44-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {            
        AppDomain domain = AppDomain.CreateDomain("MyDomain");                
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690    

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;        
    }
}
```
