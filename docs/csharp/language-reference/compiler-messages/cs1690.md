# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="e0a87-102">Ostrzeżenie kompilatora (poziom 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="e0a87-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="e0a87-103">Uzyskiwanie dostępu do elementu członkowskiego na "element członkowski" może spowodować wyjątek w czasie wykonywania, ponieważ jest to pole klasy marshal-by-reference</span><span class="sxs-lookup"><span data-stu-id="e0a87-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="e0a87-104">To ostrzeżenie występuje podczas próby wywołania metody, właściwości lub indeksatora <xref:System.MarshalByRefObject>na element członkowski klasy, która pochodzi od , a element członkowski jest typem wartości.</span><span class="sxs-lookup"><span data-stu-id="e0a87-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="e0a87-105">Obiekty, które `MarshalByRefObject` dziedziczą z są zazwyczaj przeznaczone do organizowania przez odwołanie w domenie aplikacji.</span><span class="sxs-lookup"><span data-stu-id="e0a87-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="e0a87-106">Jeśli kod kiedykolwiek próbuje uzyskać bezpośredni dostęp do elementu członkowskiego typu wartości <xref:System.InvalidOperationException> takiego obiektu w domenie aplikacji, wystąpi czas wykonywania.</span><span class="sxs-lookup"><span data-stu-id="e0a87-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="e0a87-107">Aby rozwiązać ostrzeżenie, najpierw skopiuj element członkowski do zmiennej lokalnej i wywołaj metodę na tej zmiennej.</span><span class="sxs-lookup"><span data-stu-id="e0a87-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="e0a87-108">Poniższy przykład generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="e0a87-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
