# <a name="compiler-error-cs8400"></a>Błąd kompilatora CS8400

Funkcja "feature" nie jest dostępna w języku C# 8,0. Użyj wersji językowej \<version> lub nowszej.

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

Uaktualnij projekt do co najmniej wersji wymienionej w błędzie.
