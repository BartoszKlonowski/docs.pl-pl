# <a name="compiler-error-cs0050"></a>Błąd kompilatora CS0050
Niespójność dostępności: typ zwracany "Type" jest mniej dostępny niż metoda "Method"  
  
 Zwracany typ oraz każdy z typów przywoływanych na liście parametrów formalnych metody muszą być co najmniej na tym samym poziomie dostępności co metoda. Aby uzyskać więcej informacji, zobacz [modyfikatory dostępu](../../programming-guide/classes-and-structs/access-modifiers.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0050, ponieważ nie podano modyfikatora dostępności `MyClass` dla i jego dostępność jest domyślnie `private`ustawiona na.  
  
```csharp  
// CS0050.cs  
class MyClass //accessibility defaults to private  
// try the following line instead  
// public class MyClass   
{  
}  
  
public class MyClass2  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
