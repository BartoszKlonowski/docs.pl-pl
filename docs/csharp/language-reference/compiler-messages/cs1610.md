# <a name="compiler-warning-level-4-cs1610"></a>Ostrzeżenie kompilatora (poziom 4) CS1610
Nie można usunąć pliku tymczasowego 'Plik' używany jako domyślny zasób Win32 — zasób  
  
 Korzystając z [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md) — opcja kompilatora i kiedy Twoja **% TEMP %** katalogu nie ma uprawnień usuwania, to ostrzeżenie wskazuje, że kompilator nie można usunąć pliku tymczasowego, którego utworzona.  
  
 Upewnij się, że masz uprawnienia odczytu/zapisu/usuwania **% TEMP %** katalogu.  
  
 Jeśli to konieczne, można ręcznie usunąć te pliki, i nie będzie żadnych szkód do C# lub dowolnego programu.
