## <a name="example"></a><span data-ttu-id="0129d-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="0129d-110">Example</span></span>

```csharp  
// CS0178.cs  
class MyClass  
{  
   public static void Main()  
   {  
      int a = new int[5][,][][5;   // CS0178  
      int[,] b = new int[3,2];   // OK  
  
      int[][] c = new int[10][];  
      c[0] = new int[5][5];   // CS0178  
      c[0] = new int[2];   // OK  
      c[1] = new int[2]{1,2};   // OK  
   }  
}  
```
