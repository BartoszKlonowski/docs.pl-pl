# <a name="compiler-error-cs0178"></a>Błąd kompilatora CS0178
Nieprawidłowy specyfikator rangi: oczekiwano "," lub "]"  
  
 Inicjowanie tablicy zostało źle sformułowane. Na przykład podczas określania wymiarów tablicy, można użyć następujących wyrażeń:  
  
-   Numer w nawiasie  
  
-   Pusty nawias  
  
-   Przecinek w nawiasie  
  
 Aby uzyskać więcej informacji, zobacz [tablice](../../../csharp/programming-guide/arrays/index.md) i specyfikację języka C# ([specyfikacja języka C#](../../../csharp/language-reference/language-specification/index.md)) (sekcja o inicjatorach tablic).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0178.  
  
```csharp  
// CS0178.cs  
class MyClass  
{  
   public static void Main()  
   {  
      int a = new int[5][,][][5;   // CS0178  
      int[,] b = new int[3,2];   // OK  
  
      int[][] c = new int[10][];  
      c[0] = new int[5][5];   // CS0178  
      c[0] = new int[2];   // OK  
      c[1] = new int[2]{1,2};   // OK  
   }  
}  
```
