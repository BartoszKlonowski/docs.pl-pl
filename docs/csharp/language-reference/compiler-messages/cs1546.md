## <a name="example"></a><span data-ttu-id="aa770-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="aa770-108">Example</span></span>  
 <span data-ttu-id="aa770-109">Ten przykładowy kod składa się z pliku .cpp, który kompiluje do pliku dll, oraz pliku .cs, który używa tego pliku dll.</span><span class="sxs-lookup"><span data-stu-id="aa770-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="aa770-110">Poniższy kod jest dla pliku .dll i definiuje właściwość, do accessed przez kod w pliku .cs.</span><span class="sxs-lookup"><span data-stu-id="aa770-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp

// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
