 <span data-ttu-id="17b80-110">Ten przykładowy kod składa się z pliku. cpp, który kompiluje plik. dll, i pliku. cs, który używa biblioteki DLL.</span><span class="sxs-lookup"><span data-stu-id="17b80-110">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="17b80-111">Poniższy kod dotyczy pliku dll i definiuje właściwość, do której ma dostęp kod w pliku CS.</span><span class="sxs-lookup"><span data-stu-id="17b80-111">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>

```cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
