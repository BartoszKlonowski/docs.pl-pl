# <a name="compiler-error-cs1546"></a>Błąd kompilatora CS1546
Właściwość, indeksator lub zdarzenie "właściwość" nie jest obsługiwany przez język; Spróbuj bezpośrednio wywołać metody dostępu "metody dostępu"  
  
 Twój kod zużywa obiekt, który ma domyślnie indeksowanej właściwości i próbowano użyć składni indeksowanego. Aby rozwiązać ten problem, należy wywołać metody dostępu właściwości. Aby uzyskać więcej informacji na temat właściwości i indeksatorów, zobacz [indeksatory](../../../csharp/programming-guide/indexers/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS1546.  
  
## <a name="example"></a>Przykład  
 Ten przykładowy kod składa się z pliku .cpp, który kompiluje do biblioteki .dll, i w pliku .cs, który używa tego .dll. Poniższy kod służy do pliku dll i określa właściwość, która ma zostać oceniony przez kod w pliku CS.  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```  
  
## <a name="example"></a>Przykład  
 Jest to C# pliku.  
  
```csharp  
// CS1546.cs  
// compile with: /r:CPP1546.dll   
using System;  
public class Test  
{  
    public static void Main()  
    {  
        int i = 0;  
        MCPP mcpp = new MCPP();  
        i = mcpp.Prop(1,1); // CS1546  
        // Try the following line instead:  
        // i = mcpp.get_Prop(1,1);  
    }  
}  
```
