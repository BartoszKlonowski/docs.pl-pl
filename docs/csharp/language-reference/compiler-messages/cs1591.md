# <a name="compiler-warning-level-4-cs1591"></a>Ostrzeżenie kompilatora (poziom 4) CS1591
Brak komentarza XML dla widocznego publicznie typu lub elementu członkowskiego "Type_or_Member"  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) określono opcję kompilatora, ale co najmniej jedna konstrukcja nie ma komentarzy.  
  
 Poniższy przykład spowoduje wygenerowanie CS1591:  
  
```csharp  
// CS1591.cs  
// compile with: /W:4 /doc:x.xml  
  
/// text  
public class Test  
{  
   // /// text  
   public static void Main()   // CS1591, remove "//" from previous line  
   {  
   }  
}  
```
