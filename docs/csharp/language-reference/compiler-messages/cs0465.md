<span data-ttu-id="e15e7-104">Wprowadzenie metody "Finalize" może zakłócać wywołanie destruktora.</span><span class="sxs-lookup"><span data-stu-id="e15e7-104">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="e15e7-105">Czy zamierzasz zadeklarować destruktor?</span><span class="sxs-lookup"><span data-stu-id="e15e7-105">Did you intend to declare a destructor?</span></span>  
  
 <span data-ttu-id="e15e7-106">To ostrzeżenie występuje, gdy tworzysz klasę przy użyciu metody, której sygnatura to `public virtual void Finalize` .</span><span class="sxs-lookup"><span data-stu-id="e15e7-106">This warning occurs when you create a class with a method whose signature is `public virtual void Finalize`.</span></span>  
  
 <span data-ttu-id="e15e7-107">Jeśli taka Klasa jest używana jako klasa bazowa i jeśli Klasa pochodna definiuje destruktor, destruktor przesłoni metodę klasy bazowej `Finalize` , a nie <xref:System.Object.Finalize%2A> .</span><span class="sxs-lookup"><span data-stu-id="e15e7-107">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e15e7-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="e15e7-108">Example</span></span>  

 <span data-ttu-id="e15e7-109">Poniższy przykład generuje CS0465.</span><span class="sxs-lookup"><span data-stu-id="e15e7-109">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
