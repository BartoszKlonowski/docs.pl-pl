<span data-ttu-id="4b870-104">Wprowadzenie metody "Finalize" może zakłócać wywołanie destruktora.</span><span class="sxs-lookup"><span data-stu-id="4b870-104">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="4b870-105">Czy zamierzasz zadeklarować destruktor?</span><span class="sxs-lookup"><span data-stu-id="4b870-105">Did you intend to declare a destructor?</span></span>

## <a name="example"></a><span data-ttu-id="4b870-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="4b870-108">Example</span></span>  
 <span data-ttu-id="4b870-109">Poniższy przykład generuje CS0465.</span><span class="sxs-lookup"><span data-stu-id="4b870-109">The following sample generates CS0465.</span></span>

// CS0465.cs

{  
   public virtual void Finalize() {}   // CS0465

{  
   ~B() {}
