## <a name="example"></a><span data-ttu-id="9a0a7-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="9a0a7-107">Example</span></span>

```csharp

class A  
{  
   public virtual void Finalize() {}   // CS0465

{  
   ~B() {}  
}
```
