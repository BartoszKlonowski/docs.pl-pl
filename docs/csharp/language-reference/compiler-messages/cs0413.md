# <a name="compiler-error-cs0413"></a><span data-ttu-id="94dcd-103">Błąd kompilatora CS0413</span><span class="sxs-lookup"><span data-stu-id="94dcd-103">Compiler Error CS0413</span></span>
<span data-ttu-id="94dcd-104">Parametr typu "Type" nie może być używany z operatorem "As", ponieważ nie ma ograniczenia typu klasy ani ograniczenia "Class"</span><span class="sxs-lookup"><span data-stu-id="94dcd-104">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="94dcd-105">Ten błąd występuje, gdy typ generyczny używa operatora [as](../operators/type-testing-and-cast.md#as-operator) , ale ten typ generyczny nie ma ograniczenia typu klasy.</span><span class="sxs-lookup"><span data-stu-id="94dcd-105">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="94dcd-106">`as`Operator jest dozwolony tylko w przypadku typów wartości referencyjnych i dopuszczających wartość null, więc parametr type musi być ograniczony, aby zagwarantować, że nie jest typem wartości.</span><span class="sxs-lookup"><span data-stu-id="94dcd-106">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="94dcd-107">Aby uniknąć tego błędu, użyj ograniczenia typu klasy lub ograniczenia typu odwołania.</span><span class="sxs-lookup"><span data-stu-id="94dcd-107">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="94dcd-108">Wynika to z faktu `as` , że operator może zwrócić `null` , który nie jest wartością możliwej dla typu wartości, a parametr typu musi być traktowany jako typ wartości, chyba że jest to ograniczenie typu klasy lub ograniczenie typu odwołania.</span><span class="sxs-lookup"><span data-stu-id="94dcd-108">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="94dcd-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="94dcd-109">Example</span></span>

<span data-ttu-id="94dcd-110">Poniższy przykład generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="94dcd-110">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
