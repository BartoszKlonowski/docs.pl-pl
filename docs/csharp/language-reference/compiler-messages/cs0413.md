# <a name="compiler-error-cs0413"></a>Błąd kompilatora CS0413
Parametr typu "parametr typu" nie można używać z operatorem "as", ponieważ nie ma ograniczenia typu klasy ani ograniczenia "class"  
  
 Ten błąd występuje, jeśli korzysta z typem ogólnym [jako](../../../csharp/language-reference/keywords/as.md) operatora, ale typ rodzajowy nie ma klasy typu ograniczenia. `as` Operator jest dozwolona tylko w przypadku typów referencyjnych, więc parametru typu muszą być ograniczone do zagwarantowania, że nie jest typem wartości. Aby uniknąć tego błędu, należy użyć ograniczenie typu klasy lub ograniczenie typu odwołania.  
  
 Jest to spowodowane `as` operator może zwrócić `null`, który nie jest możliwa wartość dla typu wartości i parametru typu muszą być traktowane jako typ wartości, chyba że jest to ograniczenie typu klasy lub ograniczenie typu odwołania.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0413.  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
