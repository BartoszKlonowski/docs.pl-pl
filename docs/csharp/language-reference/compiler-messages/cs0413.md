## <a name="example"></a><span data-ttu-id="8e96b-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="8e96b-108">Example</span></span>

<span data-ttu-id="8e96b-109">Poniższy przykład generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="8e96b-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
