# <a name="compiler-error-cs0016"></a>Błąd kompilatora CS0016

Nie można zapisać do pliku wyjściowego 'plik' — 'reason'

 Kompilator nie może zapisać do pliku wyjściowego. Sprawdź ścieżkę do pliku, aby upewnić się, że istnieje. Jeśli wcześniej utworzony plik jest już w lokalizacji, upewnij się, że jest zapisywalny i że żaden proces nie ma obecnie blokady pliku. Na przykład upewnij się, że plik wykonywalny nie jest uruchomiony podczas próby kompilacji.
