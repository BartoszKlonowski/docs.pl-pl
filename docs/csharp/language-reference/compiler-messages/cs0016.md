# <a name="compiler-error-cs0016"></a>Błąd kompilatora CS0016

Nie można zapisać do pliku wyjściowego "File" — "Przyczyna"

 Kompilator nie może zapisać do pliku wyjściowego. Sprawdź ścieżkę do pliku, aby upewnić się, że istnieje. Jeśli wcześniej skompilowany plik znajduje się już w lokalizacji, upewnij się, że jest on zapisywalny, a żaden proces nie ma obecnie blokady dla tego pliku. Na przykład upewnij się, że plik wykonywalny nie jest uruchomiony podczas próby skompilowania.
