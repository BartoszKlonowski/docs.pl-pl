# <a name="compiler-error-cs0016"></a>Błąd kompilatora CS0016
Nie można zapisać do pliku 'Plik' wyjściowego — "Przyczyna"  
  
 Kompilator nie można zapisać do pliku wyjściowego. Sprawdź ścieżkę do pliku, aby upewnić się, że istnieje. Jeśli wcześniej skompilowany plik znajduje się już w tej lokalizacji, upewnij się, jest zapisywalny, i że żaden proces jest obecnie zablokowany plik. Na przykład upewnij się, że podczas próby kompilacji pliku wykonywalnego nie jest uruchomiona.
