# <a name="compiler-error-cs0016"></a>Błąd kompilatora CS0016

Nie można zapisać do pliku wyjściowego „plik” — „przyczyna”

 Kompilatorowi nie udało się zapisać danych w pliku wyjściowym. Sprawdź ścieżkę do pliku, aby upewnić się, że plik istnieje. Jeśli skompilowany wcześniej plik znajduje się już w tej lokalizacji, upewnij się, że można go zapisywać i że żaden proces go nie blokuje. Na przykład upewnij się, że plik wykonywalny nie jest uruchomiony podczas próby skompilowania.
