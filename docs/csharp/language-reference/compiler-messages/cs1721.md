# <a name="compiler-error-cs1721"></a>Błąd kompilatora CS1721

Klasa "Class" nie może mieć wielu klas bazowych: "class_1" i "class_2"

Najbardziej typową przyczyną tego komunikatu o błędzie jest próba użycia wielokrotnego dziedziczenia. Klasa w C# może dziedziczyć tylko bezpośrednio z jednej klasy. Jednak Klasa może implementować dowolną liczbę interfejsów.

## <a name="example"></a>Przykład

Poniższy przykład pokazuje, w jaki sposób CS1721 jest generowany:

```csharp
// CS1721.cs
public class A {}
public class B {}
public class MyClass : A, B {}   // CS1721
```

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

Poniżej przedstawiono różne sposoby korygowania tego błędu:

- Ustaw `B` dziedziczyć po klasie `A`i `MyClass` dziedziczyć po `B`:

    ```csharp
    public class A {}
    public class B : A {}
    public class MyClass : B {}
    ```

- Zadeklaruj `B` jako interfejs. Ustaw `MyClass` dziedziczyć po `B`interfejsu i klasie `A`:

    ```csharp
    public class A {}
    public interface B {}
    public class MyClass : A, B {}
    ```
