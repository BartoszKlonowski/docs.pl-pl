# <a name="compiler-error-cs0052"></a><span data-ttu-id="08b60-102">Błąd kompilatora CS0052</span><span class="sxs-lookup"><span data-stu-id="08b60-102">Compiler Error CS0052</span></span>

<span data-ttu-id="08b60-103">Niespójność dostępności: typ pola "type" jest mniej dostępny niż pole "field"</span><span class="sxs-lookup"><span data-stu-id="08b60-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>  
  
 <span data-ttu-id="08b60-104">Typ pola nie może być mniej dostępny niż samo pole, ponieważ wszystkie konstrukcje publiczne muszą zwracać obiekt dostępny publicznie.</span><span class="sxs-lookup"><span data-stu-id="08b60-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="08b60-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="08b60-105">Example</span></span>

 <span data-ttu-id="08b60-106">Poniższy przykład spowoduje wygenerowanie CS0052:</span><span class="sxs-lookup"><span data-stu-id="08b60-106">The following sample generates CS0052:</span></span>  

```csharp
// CS0052.cs  
public class MyClass2  
{  
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.  
    public MyClass M;   // CS0052  
  
    private class MyClass  
    {  
    }  
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}  
  
    public class MainClass  
    {  
        public static void Main()  
        {  
        }  
    }  
```
