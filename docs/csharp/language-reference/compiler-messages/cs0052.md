# <a name="compiler-error-cs0052"></a><span data-ttu-id="04f37-103">Błąd kompilatora CS0052</span><span class="sxs-lookup"><span data-stu-id="04f37-103">Compiler Error CS0052</span></span>

<span data-ttu-id="04f37-104">Niespójność dostępności: typ pola "Type" jest mniej dostępny niż pole "Field"</span><span class="sxs-lookup"><span data-stu-id="04f37-104">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="04f37-105">Typ pola nie może być mniej dostępny niż pole, ponieważ wszystkie konstrukcje publiczne muszą zwracać publicznie dostępny obiekt.</span><span class="sxs-lookup"><span data-stu-id="04f37-105">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="04f37-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="04f37-106">Example</span></span>

 <span data-ttu-id="04f37-107">Poniższy przykład generuje CS0052:</span><span class="sxs-lookup"><span data-stu-id="04f37-107">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
