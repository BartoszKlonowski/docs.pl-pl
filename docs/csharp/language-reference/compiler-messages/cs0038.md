<span data-ttu-id="a8a26-104">Nie można uzyskać dostępu do niestatycznej składowej typu zewnętrznego "type1" za pośrednictwem typu zagnieżdżonego "type2"</span><span class="sxs-lookup"><span data-stu-id="a8a26-104">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

```csharp

    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
