<span data-ttu-id="10ec6-103">Nie można uzyskać dostępu do niestatycznego członka typu zewnętrznego "type1" za pośrednictwem typu zagnieżdżonego "type2".</span><span class="sxs-lookup"><span data-stu-id="10ec6-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

```csharp

    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();

        }
    }

    public static void Main()
    {
    }
}
```
