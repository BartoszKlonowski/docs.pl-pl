<span data-ttu-id="8d161-103">Nie można uzyskać dostępu do niestatycznego członka typu zewnętrznego "type1" za pośrednictwem typu zagnieżdżonego "type2".</span><span class="sxs-lookup"><span data-stu-id="8d161-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="8d161-104">Pole w klasie nie jest automatycznie dostępne dla klasy zagnieżdżonej.</span><span class="sxs-lookup"><span data-stu-id="8d161-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="8d161-105">Aby mogło być dla niej dostępne, musi być [statyczne](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="8d161-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="8d161-106">W przeciwnym razie należy utworzyć wystąpienie klasy zewnętrznej.</span><span class="sxs-lookup"><span data-stu-id="8d161-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="8d161-107">Aby uzyskać więcej informacji, zobacz [Typy zagnieżdżone](../../programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="8d161-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="8d161-108">Poniższy przykład generuje błąd CS0038:</span><span class="sxs-lookup"><span data-stu-id="8d161-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
