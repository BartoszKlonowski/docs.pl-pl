# <a name="compiler-error-cs0507"></a><span data-ttu-id="48ccb-102">Błąd kompilatora CS0507</span><span class="sxs-lookup"><span data-stu-id="48ccb-102">Compiler Error CS0507</span></span>
<span data-ttu-id="48ccb-103">'function1': nie można zmienić modyfikatorów dostępu podczas zastępowania "access" dziedziczonego elementu członkowskiego "function2"</span><span class="sxs-lookup"><span data-stu-id="48ccb-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="48ccb-104">Podjęto próbę zmiany specyfikacji dostępu w zastąpionym metodzie.</span><span class="sxs-lookup"><span data-stu-id="48ccb-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="48ccb-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="48ccb-105">Example</span></span>  
 <span data-ttu-id="48ccb-106">Poniższy przykład generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="48ccb-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="48ccb-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="48ccb-107">Example</span></span>  
 <span data-ttu-id="48ccb-108">CS0507 może również wystąpić, jeśli klasa próbuje `protected internal` zastąpić metodę oznaczoną jako zdefiniowaną w metadanych, do których istnieje odwołanie.</span><span class="sxs-lookup"><span data-stu-id="48ccb-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="48ccb-109">W tej sytuacji metoda nadrzędnego powinna `protected`być oznaczona jako .</span><span class="sxs-lookup"><span data-stu-id="48ccb-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="48ccb-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="48ccb-110">Example</span></span>  
 <span data-ttu-id="48ccb-111">Poniższy przykład generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="48ccb-111">The following sample generates CS0507.</span></span>
