# <a name="compiler-error-cs0507"></a><span data-ttu-id="bc1cf-102">Błąd kompilatora CS0507</span><span class="sxs-lookup"><span data-stu-id="bc1cf-102">Compiler Error CS0507</span></span>
<span data-ttu-id="bc1cf-103">"function1": nie można zmienić modyfikatorów dostępu podczas przesłaniania, "dostęp" dziedziczonej składowej "function2"</span><span class="sxs-lookup"><span data-stu-id="bc1cf-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="bc1cf-104">Próbowano zmienić specyfikacji dostępu w zastąpienie metody.</span><span class="sxs-lookup"><span data-stu-id="bc1cf-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bc1cf-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="bc1cf-105">Example</span></span>  
 <span data-ttu-id="bc1cf-106">Poniższy przykład spowoduje wygenerowanie CS0507.</span><span class="sxs-lookup"><span data-stu-id="bc1cf-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="bc1cf-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="bc1cf-107">Example</span></span>  
 <span data-ttu-id="bc1cf-108">CS0507 może również wystąpić, jeśli klasa próbuje zastąpić metoda oznaczona jako `protected internal` zdefiniowane w metadanych odwołania.</span><span class="sxs-lookup"><span data-stu-id="bc1cf-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="bc1cf-109">W takiej sytuacji metoda przesłaniania powinien być oznaczony jako `protected`.</span><span class="sxs-lookup"><span data-stu-id="bc1cf-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="bc1cf-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="bc1cf-110">Example</span></span>  
 <span data-ttu-id="bc1cf-111">Poniższy przykład spowoduje wygenerowanie CS0507.</span><span class="sxs-lookup"><span data-stu-id="bc1cf-111">The following sample generates CS0507.</span></span>
