# <a name="compiler-error-cs0507"></a><span data-ttu-id="3f800-103">Błąd kompilatora CS0507</span><span class="sxs-lookup"><span data-stu-id="3f800-103">Compiler Error CS0507</span></span>
<span data-ttu-id="3f800-104">"function1": nie można zmienić modyfikatorów dostępu podczas przesłaniania dziedziczonej składowej "function2" "Access"</span><span class="sxs-lookup"><span data-stu-id="3f800-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="3f800-105">Podjęto próbę zmiany specyfikacji dostępu w przesłonięciu metody.</span><span class="sxs-lookup"><span data-stu-id="3f800-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3f800-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="3f800-106">Example</span></span>  
 <span data-ttu-id="3f800-107">Poniższy przykład generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="3f800-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="3f800-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="3f800-108">Example</span></span>  
 <span data-ttu-id="3f800-109">CS0507 może również wystąpić, jeśli klasa próbuje zastąpić metodę oznaczoną jako `protected internal` zdefiniowana w metadanych przywoływanych.</span><span class="sxs-lookup"><span data-stu-id="3f800-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="3f800-110">W tej sytuacji metoda przesłaniania powinna być oznaczona jako `protected` .</span><span class="sxs-lookup"><span data-stu-id="3f800-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="3f800-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="3f800-111">Example</span></span>  
 <span data-ttu-id="3f800-112">Poniższy przykład generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="3f800-112">The following sample generates CS0507.</span></span>
