# <a name="compiler-error-cs0845"></a><span data-ttu-id="ea154-103">Błąd kompilatora CS0845</span><span class="sxs-lookup"><span data-stu-id="ea154-103">Compiler Error CS0845</span></span>

<span data-ttu-id="ea154-104">Drzewo wyrażenia lambda nie może zawierać operatora łączenia z literałem o wartości null po lewej stronie.</span><span class="sxs-lookup"><span data-stu-id="ea154-104">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>

1. <span data-ttu-id="ea154-107">Rzutowanie literału o wartości null na obiekt.</span><span class="sxs-lookup"><span data-stu-id="ea154-107">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ea154-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="ea154-108">Example</span></span>  

 <span data-ttu-id="ea154-109">Poniższy kod generuje CS0845:</span><span class="sxs-lookup"><span data-stu-id="ea154-109">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
