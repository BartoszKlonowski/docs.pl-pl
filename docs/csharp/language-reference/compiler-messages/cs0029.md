 <span data-ttu-id="dd369-118">Zmienna `lng` zawiera teraz wartość, która nie może być przechowywana w zmiennej, `i` ponieważ jest zbyt duża.</span><span class="sxs-lookup"><span data-stu-id="dd369-118">The variable `lng` now contains a value that cannot be stored in the variable `i` because it is too large.</span></span> <span data-ttu-id="dd369-119">Jeśli przekonwertujemy tę wartość na typ int, będziemy tracić część naszych danych, a przekonwertowana wartość nie będzie taka sama jak wartość przed konwersją.</span><span class="sxs-lookup"><span data-stu-id="dd369-119">If we were to convert this value to an int type we would be losing some of our data and the converted value would not be the same as the value before the conversion.</span></span>

 <span data-ttu-id="dd369-123">Zwróć uwagę na różnice między tym przykładem kodu a pierwszym.</span><span class="sxs-lookup"><span data-stu-id="dd369-123">Notice the difference between this code sample and the first.</span></span> <span data-ttu-id="dd369-124">Tym razem zmienna `lng` znajduje się po lewej stronie operatora przypisania, więc jest to element docelowy tego przydziału.</span><span class="sxs-lookup"><span data-stu-id="dd369-124">This time the variable `lng` is on the left-hand side of the assignment operator, so it is the target of our assignment.</span></span> <span data-ttu-id="dd369-125">Aby można było wykonać przypisanie, kompilator musi niejawnie skonwertować zmienną `i` , która jest typu int, na typ Long.</span><span class="sxs-lookup"><span data-stu-id="dd369-125">Before the assignment can be made, the compiler must implicitly convert the variable `i`, which is of type int, to type long.</span></span> <span data-ttu-id="dd369-126">Jest to konwersja rozszerzająca, ponieważ Konwertujemy z typu, który zajmuje 4 bajty pamięci (int) do typu, który zajmuje 8 bajtów pamięci (Long).</span><span class="sxs-lookup"><span data-stu-id="dd369-126">This is a widening conversion since we are converting from a type that occupies 4 bytes of memory (an int) to a type that occupies 8 bytes of memory (a long).</span></span> <span data-ttu-id="dd369-127">Niejawne konwersje rozszerzające są dozwolone, ponieważ nie ma potencjalnej utraty danych.</span><span class="sxs-lookup"><span data-stu-id="dd369-127">Implicit widening conversions are allowed because there is no potential loss of data.</span></span> <span data-ttu-id="dd369-128">Każda wartość, która może być przechowywana w int, może być również przechowywana w długim.</span><span class="sxs-lookup"><span data-stu-id="dd369-128">Any value that can be stored in an int can also be stored in a long.</span></span>
