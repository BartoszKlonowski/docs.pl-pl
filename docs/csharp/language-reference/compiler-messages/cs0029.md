 <span data-ttu-id="f90da-117">Zmienna `lng` teraz zawiera wartość, która nie może być przechowywana w zmiennej `i` ponieważ jest zbyt duży.</span><span class="sxs-lookup"><span data-stu-id="f90da-117">The variable `lng` now contains a value that cannot be stored in the variable `i` because it is too large.</span></span> <span data-ttu-id="f90da-118">Gdybyśmy wybrali przekonwertować tę wartość na typ int, firma Microsoft może być utraty niektóre z naszych danych i przekonwertowana wartości nie może być taka sama jak wartość przed dokonaniem konwersji.</span><span class="sxs-lookup"><span data-stu-id="f90da-118">If we were to convert this value to an int type we would be losing some of our data and the converted value would not be the same as the value before the conversion.</span></span>
