# <a name="compiler-warning-level-1-cs1598"></a>Ostrzeżenie kompilatora (poziom 1) CS1598
Nie można załadować parsera XML z następującej przyczyny: "Przyczyna". Plik dokumentacji XML "File" nie zostanie wygenerowany.  
  
 Określono opcję [/doc](../compiler-options/doc-compiler-option.md) , ale kompilator nie mógł znaleźć i załadować biblioteki Msxml3. dll. Upewnij się, że plik msxml3. dll jest zainstalowany i zarejestrowany.
