# <a name="compiler-warning-level-1-cs1598"></a>Ostrzeżenie kompilatora (poziom 1) CS1598
Analizator składni XML nie można załadować z następującej przyczyny: "Przyczyna". Plik dokumentacji XML 'Plik' nie zostanie wygenerowany.  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) określono opcję, ale kompilator nie można odnaleźć i załadować msxml3.dll. Upewnij się, że plik msxml3.dll jest zainstalowany i zarejestrowany.
