# <a name="compiler-error-cs1519"></a><span data-ttu-id="a9a67-102">Błąd kompilatora CS1519</span><span class="sxs-lookup"><span data-stu-id="a9a67-102">Compiler Error CS1519</span></span>
<span data-ttu-id="a9a67-103">Nieprawidłowy token "token" w deklaracji elementu członkowskiego klasy, struktury lub interfejsu</span><span class="sxs-lookup"><span data-stu-id="a9a67-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="a9a67-104">Ten błąd jest generowany zawsze, gdy token zostanie napotkany w lokalizacji, w której nie należy.</span><span class="sxs-lookup"><span data-stu-id="a9a67-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="a9a67-105">*Token* jest słowem kluczowym; identyfikator (nazwa klasy, struktury, metody itd.); ciąg, znak lub wartość literału liczbowego, taka jak 108, "Hello" lub "A"; lub operatora lub punktuacza, `==` takiego `;`jak lub .</span><span class="sxs-lookup"><span data-stu-id="a9a67-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="a9a67-106">Każda [klasa,](../keywords/class.md)struktura lub deklaracja elementu członkowskiego interfejsu, która zawiera nieprawidłowe modyfikatory, zanim typ wygeneruje ten błąd.</span><span class="sxs-lookup"><span data-stu-id="a9a67-106">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="a9a67-107">Aby naprawić błąd, usuń nieprawidłowe modyfikatory.</span><span class="sxs-lookup"><span data-stu-id="a9a67-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="a9a67-108">Poniższy przykład generuje CS1519 w pięciu miejscach, ponieważ tokeny są umieszczane w miejscach, w których są nieprawidłowe:</span><span class="sxs-lookup"><span data-stu-id="a9a67-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
