# <a name="compiler-error-cs1519"></a><span data-ttu-id="33df9-102">Błąd kompilatora CS1519</span><span class="sxs-lookup"><span data-stu-id="33df9-102">Compiler Error CS1519</span></span>
<span data-ttu-id="33df9-103">Nieprawidłowy token 'token' w klasy, struktury lub deklaracji elementu członkowskiego interfejsu</span><span class="sxs-lookup"><span data-stu-id="33df9-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="33df9-104">Ten błąd jest generowany napotkaniu tokenu w lokalizacji, w których nie należy.</span><span class="sxs-lookup"><span data-stu-id="33df9-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="33df9-105">A *tokenu* jest słowo kluczowe; identyfikator (Nazwa klasy, struktury, metody i tak dalej); ciąg, znak lub wartość liczbowa literału, takich jak 108, "Hello", lub ";" lub operator lub znak interpunkcyjny, takich jak `==` lub `;`.</span><span class="sxs-lookup"><span data-stu-id="33df9-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="33df9-106">Wszelkie [klasy](../../../csharp/language-reference/keywords/class.md), struktury lub deklaracji elementu członkowskiego interfejsu, który zawiera nieprawidłowe modyfikatory przed typem spowoduje wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="33df9-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="33df9-107">Aby naprawić błąd, należy usunąć nieprawidłowy modyfikatorów.</span><span class="sxs-lookup"><span data-stu-id="33df9-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="33df9-108">Poniższy przykład generuje CS1519 w pięciu miejscach, ponieważ tokeny są umieszczane w lokalizacjach, w którym nie są prawidłowe:</span><span class="sxs-lookup"><span data-stu-id="33df9-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
