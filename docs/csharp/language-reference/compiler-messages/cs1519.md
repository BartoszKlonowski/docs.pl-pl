# <a name="compiler-error-cs1519"></a><span data-ttu-id="906d8-102">Błąd kompilatora CS1519</span><span class="sxs-lookup"><span data-stu-id="906d8-102">Compiler Error CS1519</span></span>
<span data-ttu-id="906d8-103">Nieprawidłowy token 'token' w klasy, struktury lub deklaracją składowej interfejsu</span><span class="sxs-lookup"><span data-stu-id="906d8-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="906d8-104">Ten błąd jest generowany zawsze wtedy, gdy token zostanie napotkany w lokalizacji, w którym nie należy.</span><span class="sxs-lookup"><span data-stu-id="906d8-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="906d8-105">A *tokenu* jest słowem kluczowym; identyfikatora (nazwy klasy, struktury, metody i tak dalej); ciągu, znaku lub wartość liczbowa literału, takich jak 108, "Hello", lub ";" lub operatora lub znak interpunkcyjny, takie jak `==` lub `;`.</span><span class="sxs-lookup"><span data-stu-id="906d8-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="906d8-106">Wszelkie [klasy](../../../csharp/language-reference/keywords/class.md), struktury lub deklaracją składowej interfejsu, który zawiera nieprawidłowe modyfikatory, które przed typem spowoduje wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="906d8-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="906d8-107">Aby naprawić błąd, Usuń nieprawidłowe modyfikatory.</span><span class="sxs-lookup"><span data-stu-id="906d8-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="906d8-108">Poniższy przykład generuje CS1519 w pięciu miejsca, ponieważ tokeny są umieszczane w lokalizacji, w którym są nieprawidłowe:</span><span class="sxs-lookup"><span data-stu-id="906d8-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
