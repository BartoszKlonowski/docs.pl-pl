# <a name="compiler-error-cs1519"></a><span data-ttu-id="30228-103">Błąd kompilatora CS1519</span><span class="sxs-lookup"><span data-stu-id="30228-103">Compiler Error CS1519</span></span>

<span data-ttu-id="30228-104">Nieprawidłowy token "token" w deklaracji składowej klasy, struktury lub interfejsu</span><span class="sxs-lookup"><span data-stu-id="30228-104">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="30228-105">Ten błąd jest generowany za każdym razem, gdy token zostanie napotkany w lokalizacji, w której nie należy.</span><span class="sxs-lookup"><span data-stu-id="30228-105">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="30228-106">*Token* jest słowem kluczowym; Identyfikator (nazwa klasy, struktury, metody itd.); ciąg, znak lub wartość literału liczbowego, taka jak 108, "Hello" lub "A"; operator lub punctuator, taki jak `==` lub `;` .</span><span class="sxs-lookup"><span data-stu-id="30228-106">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="30228-107">Wszystkie [klasy](../keywords/class.md), struktury lub deklaracje elementów członkowskich interfejsów, które zawierają nieprawidłowe modyfikatory przed typem, generują ten błąd.</span><span class="sxs-lookup"><span data-stu-id="30228-107">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="30228-108">Aby naprawić ten błąd, Usuń nieprawidłowe modyfikatory.</span><span class="sxs-lookup"><span data-stu-id="30228-108">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="30228-109">Poniższy przykład generuje CS1519 w pięciu miejscach, ponieważ tokeny są umieszczane w lokalizacjach, gdzie są nieprawidłowe:</span><span class="sxs-lookup"><span data-stu-id="30228-109">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
