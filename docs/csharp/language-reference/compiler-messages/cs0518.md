# <a name="compiler-error-cs0518"></a>Błąd kompilatora CS0518
Wstępnie zdefiniowany typ "type" nie został zdefiniowany ani zaimportowany  
  
 Główną przyczyną tego problemu jest projekt nie importuje biblioteki mscorlib.dll, która określa całą przestrzeń nazw System. Może to być spowodowane jedną z następujących:  
  
- [/Nostdlib](../../../csharp/language-reference/compiler-options/nostdlib-compiler-option.md) określono opcję kompilatora wiersza polecenia. / Nostdlib — opcja zapobiega Importuj biblioteki mscorlib.dll. Użyj tej opcji, jeśli chcesz zdefiniować lub utworzyć przestrzeń nazw System specyficzne dla użytkownika.  
  
- Niepoprawna Biblioteka mscorlib.dll ma odwołania.  
  
- Istnieje uszkodzenie programu Visual Studio .NET lub .NET Framework wspólnego języka środowiska uruchomieniowego instalacji.  
  
- Pozostają pozostałe składniki z poprzedniej instalacji, które nie są zgodne z najnowszą instalacji.  
  
 Aby rozwiązać ten problem, wykonaj jedną z następujących czynności:  
  
- Nie określaj / nostdlib — opcja kompilatora wiersza polecenia.  
  
- Upewnij się, że projekt odwołuje się do poprawne mscorlib.dll.  
  
- (Jeśli poprzednie rozwiązania nie wyeliminują problemu), należy ponownie zainstalować .NET Framework środowisko uruchomieniowe języka wspólnego.
