# <a name="compiler-error-cs0188"></a><span data-ttu-id="b48eb-103">Błąd kompilatora CS0188</span><span class="sxs-lookup"><span data-stu-id="b48eb-103">Compiler Error CS0188</span></span>

<span data-ttu-id="b48eb-104">Nie można użyć obiektu "This" przed przypisaniem wszystkich jego pól</span><span class="sxs-lookup"><span data-stu-id="b48eb-104">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="b48eb-105">Wszystkie pola w elemencie `struct` muszą być przypisane przez konstruktora, aby Konstruktor mógł wywołać metodę w `struct` .</span><span class="sxs-lookup"><span data-stu-id="b48eb-105">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="b48eb-106">Jeśli ten błąd występuje podczas próby zainicjowania właściwości w konstruktorze struktury, rozwiązanie to zainicjowanie pola zapasowego tej właściwości bezpośrednio zamiast samej właściwości.</span><span class="sxs-lookup"><span data-stu-id="b48eb-106">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="b48eb-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="b48eb-107">Example</span></span>

 <span data-ttu-id="b48eb-108">Poniższy przykład generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="b48eb-108">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
