# <a name="compiler-error-cs0188"></a><span data-ttu-id="68793-102">Błąd kompilatora CS0188</span><span class="sxs-lookup"><span data-stu-id="68793-102">Compiler Error CS0188</span></span>

<span data-ttu-id="68793-103">Nie można użyć obiektu "this" przed przypisaniem wszystkich jego pól do</span><span class="sxs-lookup"><span data-stu-id="68793-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="68793-104">Wszystkie pola `struct` w konstruktorze muszą być przypisane przez konstruktora, zanim konstruktor może wywołać metodę w pliku `struct`.</span><span class="sxs-lookup"><span data-stu-id="68793-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="68793-105">Jeśli ten błąd zostanie wyświetlony podczas próby zainicjowania właściwości w konstrutorze struktury, rozwiązaniem jest zainicjowanie pola zapasowego tej właściwości bezpośrednio zamiast samej właściwości.</span><span class="sxs-lookup"><span data-stu-id="68793-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="68793-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="68793-106">Example</span></span>

 <span data-ttu-id="68793-107">Poniższy przykład generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="68793-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
