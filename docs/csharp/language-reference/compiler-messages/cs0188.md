<span data-ttu-id="e6b32-103">Nie można użyć obiektu "This" przed przypisaniem wszystkich jego pól</span><span class="sxs-lookup"><span data-stu-id="e6b32-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="e6b32-104">Wszystkie pola w `struct` elemencie muszą być przypisane przez konstruktora, aby Konstruktor mógł wywołać metodę `struct`w.</span><span class="sxs-lookup"><span data-stu-id="e6b32-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="e6b32-105">Jeśli ten błąd występuje podczas próby zainicjowania właściwości w konstruktorze struktury, rozwiązanie to zmienia parametr konstruktora, aby określić pole zapasowe zamiast samej właściwości.</span><span class="sxs-lookup"><span data-stu-id="e6b32-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="e6b32-106">W strukturach należy unikać implementowania właściwości, ponieważ nie mają one pola zapasowego i w związku z tym nie można inicjować ich w żaden sposób z konstruktora.</span><span class="sxs-lookup"><span data-stu-id="e6b32-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="e6b32-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="e6b32-108">Example</span></span>

 <span data-ttu-id="e6b32-109">Poniższy przykład generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="e6b32-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
