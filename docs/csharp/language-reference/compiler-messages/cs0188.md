<span data-ttu-id="04e6f-103">Nie można użyć obiektu "this", zanim wszystkie jej pola są przypisane do</span><span class="sxs-lookup"><span data-stu-id="04e6f-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="04e6f-104">Wszystkie pola w `struct` muszą być przypisane przez konstruktora przed konstruktora można wywołać metody `struct`.</span><span class="sxs-lookup"><span data-stu-id="04e6f-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="04e6f-105">Jeśli zostanie wyświetlony ten błąd podczas próby zainicjowania właściwości w konstruktorze struktury rozwiązania jest zmiana parametru konstruktora do określenia pola zapasowego zamiast samej właściwości.</span><span class="sxs-lookup"><span data-stu-id="04e6f-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="04e6f-106">Właściwości zaimplementowane automatycznie należy unikać w strukturach, ponieważ nie pole zapasowy i nie można zainicjować w żadnym z konstruktora.</span><span class="sxs-lookup"><span data-stu-id="04e6f-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="04e6f-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="04e6f-108">Example</span></span>  
 <span data-ttu-id="04e6f-109">Poniższy przykład generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="04e6f-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
