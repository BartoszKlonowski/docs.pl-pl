# <a name="compiler-error-cs1946"></a><span data-ttu-id="ec2a6-102">Błąd kompilatora CS1946</span><span class="sxs-lookup"><span data-stu-id="ec2a6-102">Compiler Error CS1946</span></span>

<span data-ttu-id="ec2a6-103">Nie można przekonwertować wyrażenia metody anonimowej na drzewo wyrażeń.</span><span class="sxs-lookup"><span data-stu-id="ec2a6-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="ec2a6-104">[Metoda anonimowa](../operators/delegate-operator.md) reprezentuje zestaw instrukcji, ale drzewo wyrażeń nie może zawierać instrukcji.</span><span class="sxs-lookup"><span data-stu-id="ec2a6-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="ec2a6-105">W związku z tym metoda anonimowa nie może być reprezentowana przez drzewo wyrażeń.</span><span class="sxs-lookup"><span data-stu-id="ec2a6-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="ec2a6-106">Aby poprawić ten błąd, zmień metodę anonimową na [wyrażenie lambda](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="ec2a6-106">To correct this error, change the anonymous method to a [lambda expression](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="ec2a6-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="ec2a6-107">Example</span></span>

<span data-ttu-id="ec2a6-108">Poniższy przykład generuje CS1946:</span><span class="sxs-lookup"><span data-stu-id="ec2a6-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
