# <a name="compiler-error-cs1946"></a><span data-ttu-id="886b5-103">Błąd kompilatora CS1946</span><span class="sxs-lookup"><span data-stu-id="886b5-103">Compiler Error CS1946</span></span>

<span data-ttu-id="886b5-104">Nie można przekonwertować wyrażenia metody anonimowej na drzewo wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="886b5-104">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="886b5-105">[Metoda anonimowa](../operators/delegate-operator.md) reprezentuje zestaw instrukcji, ale drzewo wyrażenia nie może zawierać instrukcji.</span><span class="sxs-lookup"><span data-stu-id="886b5-105">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="886b5-106">W związku z tym anonimowa Metoda nie może być reprezentowana przez drzewo wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="886b5-106">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="886b5-107">Aby naprawić ten błąd, Zmień metodę anonimową na [wyrażenie lambda](../operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="886b5-107">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="886b5-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="886b5-108">Example</span></span>

<span data-ttu-id="886b5-109">Poniższy przykład generuje CS1946:</span><span class="sxs-lookup"><span data-stu-id="886b5-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
