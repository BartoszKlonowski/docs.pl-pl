# <a name="compiler-error-cs1946"></a>Błąd kompilatora CS1946
Nie można przekonwertować metody anonimowej na drzewo wyrażeń.  
  
 Metody anonimowe reprezentują zestawy instrukcji, a drzewa wyrażeń nie mogą zawierać instrukcji. W związku z tym metody anonimowej nie można reprezentować poprzez drzewo wyrażeń.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmień metodę anonimową na wyrażenie lambda.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1946:  
  
```csharp  
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
