## <a name="example"></a><span data-ttu-id="c29af-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="c29af-108">Example</span></span>  
 <span data-ttu-id="c29af-109">Poniższy przykład generuje CS1946:</span><span class="sxs-lookup"><span data-stu-id="c29af-109">The following example generates CS1946:</span></span>

    public delegate void D();

    }
