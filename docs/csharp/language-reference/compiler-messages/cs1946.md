# <a name="compiler-error-cs1946"></a><span data-ttu-id="b0705-102">Błąd kompilatora CS1946</span><span class="sxs-lookup"><span data-stu-id="b0705-102">Compiler Error CS1946</span></span>

<span data-ttu-id="b0705-103">Nie można przekonwertować wyrażenia metody anonimowej na drzewo wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="b0705-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="b0705-104">Metoda anonimowa reprezentuje zestaw instrukcji, a drzewo wyrażeń nie może zawierać instrukcji.</span><span class="sxs-lookup"><span data-stu-id="b0705-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="b0705-105">Dlatego też drzewo wyrażeń nie może reprezentować metody anonimowej.</span><span class="sxs-lookup"><span data-stu-id="b0705-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="b0705-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="b0705-106">To correct this error</span></span>

1. <span data-ttu-id="b0705-107">Zmień metodę anonimową na wyrażenie lambda.</span><span class="sxs-lookup"><span data-stu-id="b0705-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b0705-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="b0705-108">Example</span></span>

<span data-ttu-id="b0705-109">Poniższy przykład generuje CS1946:</span><span class="sxs-lookup"><span data-stu-id="b0705-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
