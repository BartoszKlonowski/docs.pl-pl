 <span data-ttu-id="c8427-104">Jeśli zmienna jest stała ([const](../../../csharp/language-reference/keywords/const.md)), to jest również statyczna ([static](../../../csharp/language-reference/keywords/static.md)).</span><span class="sxs-lookup"><span data-stu-id="c8427-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="c8427-105">Jeśli chcesz, aby zmienna miała oba atrybuty ( **const** i **static**), zadeklaruj ją jako **const**.
