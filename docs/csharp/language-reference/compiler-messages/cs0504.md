<span data-ttu-id="50600-104">Stała "zmienna" nie może być oznaczona jako statyczna</span><span class="sxs-lookup"><span data-stu-id="50600-104">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="50600-105">Jeśli zmienna jest [stałą](../keywords/const.md), również jest [statyczna](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="50600-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="50600-106">Jeśli chcesz, aby zmienna **stała** i **statyczna** , po prostu Zadeklaruj tę zmienną jako **stałą**; Jeśli wszystko, co chcesz, jest zmienną **statyczną** , po prostu oznacz ją jako **statyczną**.</span><span class="sxs-lookup"><span data-stu-id="50600-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="50600-107">Poniższy przykład generuje CS0504:</span><span class="sxs-lookup"><span data-stu-id="50600-107">The following sample generates CS0504:</span></span>

```csharp  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
