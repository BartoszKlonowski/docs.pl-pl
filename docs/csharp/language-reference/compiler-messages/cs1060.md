# <a name="compiler-warning-level-1-cs1060"></a><span data-ttu-id="ea56d-102">Ostrzeżenie kompilatora (poziom 1) CS1060</span><span class="sxs-lookup"><span data-stu-id="ea56d-102">Compiler Warning (level 1) CS1060</span></span>
<span data-ttu-id="ea56d-103">Użycie prawdopodobnie nieprzypisanego pola "name".</span><span class="sxs-lookup"><span data-stu-id="ea56d-103">Use of possibly unassigned field 'name'.</span></span> <span data-ttu-id="ea56d-104">Zmienne wystąpienia struktury są początkowo nieprzypisane, jeśli struktura nie została przypisana.</span><span class="sxs-lookup"><span data-stu-id="ea56d-104">Struct instance variables are initially unassigned if struct is unassigned.</span></span>  
  
 <span data-ttu-id="ea56d-105">Elementy członkowskie struktury są inicjowane do ich wartości domyślnych, jeśli nie są one jawnie inicjowane.</span><span class="sxs-lookup"><span data-stu-id="ea56d-105">Struct members are initialized to their default value if you do not explicitly initialize them.</span></span> <span data-ttu-id="ea56d-106">Wartość domyślna dla typów klas (i innych typów referencyjnych) ma wartość null.</span><span class="sxs-lookup"><span data-stu-id="ea56d-106">The default value for class types (and other reference types) is null.</span></span> <span data-ttu-id="ea56d-107">Jeśli Klasa nie została zainicjowana przed próbą uzyskania dostępu do niej `NullReferenceException` , zostanie wygenerowane w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="ea56d-107">If the class is not initialized before any attempt to access it, a `NullReferenceException` will be thrown at runtime.</span></span> <span data-ttu-id="ea56d-108">Kompilator nie może ostatecznie określić, czy element członkowski klasy zostanie zainicjowany, a więc CS1060 jest ostrzeżeniem, a nie błędem.</span><span class="sxs-lookup"><span data-stu-id="ea56d-108">The compiler cannot determine definitively whether the class member will be initialized or not, and so CS1060 is a warning and not an error.</span></span>

## <a name="example"></a><span data-ttu-id="ea56d-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="ea56d-111">Example</span></span>  
 <span data-ttu-id="ea56d-112">Poniższy kod generuje CS1060, ponieważ typ `U` klasy jest członkiem, `struct S` ale nigdy nie został zainicjowany.</span><span class="sxs-lookup"><span data-stu-id="ea56d-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
