# <a name="compiler-warning-level-1-cs1060"></a>Ostrzeżenie kompilatora (poziom 1) CS1060
Użycie prawdopodobnie nieprzypisanego pola "name". Zmienne wystąpienia struktury są początkowo nieprzypisane, jeśli struktura nie została przypisana.  
  
 Elementy członkowskie struktury są inicjowane do ich wartości domyślnych, jeśli nie są one jawnie inicjowane. Wartość domyślna dla typów klas (i innych typów referencyjnych) ma wartość null. Jeśli Klasa nie została zainicjowana przed żadną próbą uzyskania dostępu do niej, zostanie wygenerowany `NullReferenceException` w czasie wykonywania. Kompilator nie może ostatecznie określić, czy element członkowski klasy zostanie zainicjowany, a więc CS1060 jest ostrzeżeniem, a nie błędem.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Podaj konstruktora dla `struct`, który inicjuje wszystkie jego elementy członkowskie.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1060, ponieważ typ klasy `U` jest członkiem `struct S`, ale nigdy nie został zainicjowany.  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
