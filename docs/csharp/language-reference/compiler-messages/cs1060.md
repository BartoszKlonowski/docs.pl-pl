 <span data-ttu-id="857e1-105">Elementy członkowskie struktury są inicjowane przywrócić wartości domyślne, jeśli użytkownik nie jawnie zainicjować je.</span><span class="sxs-lookup"><span data-stu-id="857e1-105">Struct members are initialized to their default value if you do not explicitly initialize them.</span></span> <span data-ttu-id="857e1-106">Wartość domyślna dla typu klasy (i inne typy referencyjne) ma wartość null.</span><span class="sxs-lookup"><span data-stu-id="857e1-106">The default value for class types (and other reference types) is null.</span></span> <span data-ttu-id="857e1-107">Jeśli klasa nie został zainicjowany przed wszystkie próby dostępu do niego, `NullReferenceException` zostanie zgłoszony w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="857e1-107">If the class is not initialized before any attempt to access it, a `NullReferenceException` will be thrown at runtime.</span></span> <span data-ttu-id="857e1-108">Kompilator nie może określić ostatecznie czy elementu członkowskiego klasy zostaną zainicjowane, lub nie, a więc CS1060 ostrzeżenie i nie jest błąd.</span><span class="sxs-lookup"><span data-stu-id="857e1-108">The compiler cannot determine definitively whether the class member will be initialized or not, and so CS1060 is a warning and not an error.</span></span>

## <a name="example"></a><span data-ttu-id="857e1-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="857e1-111">Example</span></span>  
 <span data-ttu-id="857e1-112">Poniższy kod generuje CS1060, ponieważ typ klasy `U` jest elementem członkowskim `struct S` , ale nigdy nie został zainicjowany.</span><span class="sxs-lookup"><span data-stu-id="857e1-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
