 <span data-ttu-id="d05fd-105">Podczas definiowania atrybutu należy określić, do jakich konstrukcji można zastosować, określając `AttributeTargets` wartość.</span><span class="sxs-lookup"><span data-stu-id="d05fd-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="d05fd-106">W poniższym przykładzie `MyAttribute` atrybut może być stosowany tylko do interfejsów, `AttributeUsage` ponieważ określa `AttributeTargets.Interface`atrybut.</span><span class="sxs-lookup"><span data-stu-id="d05fd-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Interface`.</span></span> <span data-ttu-id="d05fd-107">Ten błąd jest generowany, ponieważ atrybut jest stosowany do klasy (klasy `A`).</span><span class="sxs-lookup"><span data-stu-id="d05fd-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="d05fd-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="d05fd-108">Example</span></span>

 <span data-ttu-id="d05fd-109">Poniższy przykład generuje CS0592:</span><span class="sxs-lookup"><span data-stu-id="d05fd-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
