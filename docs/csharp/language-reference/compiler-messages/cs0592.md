 <span data-ttu-id="5c900-105">Podczas definiowania atrybutu można zdefiniować konstrukcje, do których `AttributeTargets` można go zastosować, określając wartość.</span><span class="sxs-lookup"><span data-stu-id="5c900-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="5c900-106">W poniższym `MyAttribute` przykładzie atrybut można zastosować tylko do interfejsów, ponieważ `AttributeUsage` atrybut określa `AttributeTargets.Interface`.</span><span class="sxs-lookup"><span data-stu-id="5c900-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Interface`.</span></span> <span data-ttu-id="5c900-107">Błąd jest generowany, ponieważ atrybut jest stosowany do `A`klasy (klasy).</span><span class="sxs-lookup"><span data-stu-id="5c900-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="5c900-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5c900-108">Example</span></span>

 <span data-ttu-id="5c900-109">Poniższy przykład generuje CS0592:</span><span class="sxs-lookup"><span data-stu-id="5c900-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
