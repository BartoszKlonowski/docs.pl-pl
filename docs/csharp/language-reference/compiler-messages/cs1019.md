# <a name="compiler-error-cs1019"></a><span data-ttu-id="a22b2-102">Błąd kompilatora CS1019</span><span class="sxs-lookup"><span data-stu-id="a22b2-102">Compiler Error CS1019</span></span>
<span data-ttu-id="a22b2-103">Oczekiwano operatora jednoargumentowego z możliwością przeciążenia</span><span class="sxs-lookup"><span data-stu-id="a22b2-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="a22b2-104">Rzecz, która wygląda Przeciążony operator jednoargumentowy został zadeklarowany, ale operator nie istnieje lub jest w niewłaściwej lokalizacji w podpisie.</span><span class="sxs-lookup"><span data-stu-id="a22b2-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="a22b2-105">A *operatora jednoargumentowego* jest operatora, który działa na jeden argument.</span><span class="sxs-lookup"><span data-stu-id="a22b2-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="a22b2-106">Na przykład `++` jest operatora jednoargumentowego.</span><span class="sxs-lookup"><span data-stu-id="a22b2-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="a22b2-107">Niektóre operatory jednoargumentowe można przeciążać przy użyciu `operator` — słowo kluczowe i określenie jednego parametru typu, który operator działa na.</span><span class="sxs-lookup"><span data-stu-id="a22b2-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="a22b2-108">Na przykład, jeśli chcesz przeciążyć operator `++` dla klasy zdefiniowanej przez użytkownika `Temperature` , aby mogły zapisywać `Temperature++`, można zdefiniować ją w ten sposób:</span><span class="sxs-lookup"><span data-stu-id="a22b2-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="a22b2-109">Gdy zostanie wyświetlony ten błąd, zadeklarowaniu coś, która wygląda jak Przeciążony operator jednoargumentowy, z wyjątkiem tego samego operator brakuje lub jest w niewłaściwej lokalizacji w podpisie.</span><span class="sxs-lookup"><span data-stu-id="a22b2-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="a22b2-110">Jeśli usuniesz `++` podpisu w poprzednim przykładzie, spowoduje wygenerowanie CS1019.</span><span class="sxs-lookup"><span data-stu-id="a22b2-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="a22b2-111">Poniższy kod generuje CS1019:</span><span class="sxs-lookup"><span data-stu-id="a22b2-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
