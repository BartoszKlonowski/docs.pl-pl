# <a name="compiler-error-cs1019"></a><span data-ttu-id="91f4b-103">Błąd kompilatora CS1019</span><span class="sxs-lookup"><span data-stu-id="91f4b-103">Compiler Error CS1019</span></span>
<span data-ttu-id="91f4b-104">Oczekiwano operatora jednoargumentowego z możliwością przeciążenia</span><span class="sxs-lookup"><span data-stu-id="91f4b-104">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="91f4b-105">Element, który wygląda jak przeciążony operator jednoargumentowy, został zadeklarowany, ale brakuje operatora lub znajduje się w niewłaściwej lokalizacji w podpisie.</span><span class="sxs-lookup"><span data-stu-id="91f4b-105">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="91f4b-106">*Operator jednoargumentowy* jest operatorem, który działa na pojedynczym operandzie.</span><span class="sxs-lookup"><span data-stu-id="91f4b-106">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="91f4b-107">Na przykład `++` jest operatorem jednoargumentowym.</span><span class="sxs-lookup"><span data-stu-id="91f4b-107">For example, `++` is a unary operator.</span></span> <span data-ttu-id="91f4b-108">Można przeciążać niektóre operatory jednoargumentowe za pomocą `operator` słowa kluczowego i określając pojedynczy parametr typu, na którym działa operator.</span><span class="sxs-lookup"><span data-stu-id="91f4b-108">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="91f4b-109">Na przykład jeśli chcesz przeciążyć operatora `++` dla klasy zdefiniowanej przez użytkownika `Temperature` , tak aby można było pisać `Temperature++` , możesz zdefiniować ją w następujący sposób:</span><span class="sxs-lookup"><span data-stu-id="91f4b-109">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="91f4b-110">Gdy pojawia się ten błąd, zadeklarowano coś, który wygląda jak przeciążony operator jednoargumentowy, z tą różnicą, że brakuje samego operatora lub znajduje się w niewłaściwej lokalizacji w podpisie.</span><span class="sxs-lookup"><span data-stu-id="91f4b-110">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="91f4b-111">Po usunięciu `++` z podpisu w poprzednim przykładzie zostanie wygenerowane CS1019.</span><span class="sxs-lookup"><span data-stu-id="91f4b-111">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="91f4b-112">Poniższy kod generuje CS1019:</span><span class="sxs-lookup"><span data-stu-id="91f4b-112">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }  
}  
```
