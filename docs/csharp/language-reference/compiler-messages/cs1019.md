# <a name="compiler-error-cs1019"></a><span data-ttu-id="1987e-102">Błąd kompilatora CS1019</span><span class="sxs-lookup"><span data-stu-id="1987e-102">Compiler Error CS1019</span></span>
<span data-ttu-id="1987e-103">Oczekiwano operatora jednoargumentowego z możliwością przeciążenia</span><span class="sxs-lookup"><span data-stu-id="1987e-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="1987e-104">Został zadeklarowany element wyglądający na przeciążony operator jednoargumentowy, ale operator nie istnieje lub jego lokalizacja w podpisie jest nieprawidłowa.</span><span class="sxs-lookup"><span data-stu-id="1987e-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="1987e-105">*Operator jednoargumentowy* to operator, który operuje tylko na jednym argumencie operacji.</span><span class="sxs-lookup"><span data-stu-id="1987e-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="1987e-106">Przykładem takiego operatora jest `++`.</span><span class="sxs-lookup"><span data-stu-id="1987e-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="1987e-107">Niektóre operatory jednoargumentowe można przeciążać, używając słowa kluczowego `operator` i określając jeden parametr takiego typu, na jakim działa operator.</span><span class="sxs-lookup"><span data-stu-id="1987e-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="1987e-108">Na przykład — aby przeciążyć operator `++` dla klasy `Temperature` zdefiniowanej przez użytkownika i użyć zapisu `Temperature++`, można użyć takiego rozwiązania:</span><span class="sxs-lookup"><span data-stu-id="1987e-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="1987e-109">Jeśli wystąpi ten błąd, oznacza to, że został zadeklarowany element, który wygląda jak przeciążony operator jednoargumentowy, ale brakuje samego operatora lub znajduje się on w niewłaściwej lokalizacji w podpisie.</span><span class="sxs-lookup"><span data-stu-id="1987e-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="1987e-110">Jeśli usuniesz `++` z podpisu w poprzednim przykładzie, spowoduje to wygenerowanie błędu CS1019.</span><span class="sxs-lookup"><span data-stu-id="1987e-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="1987e-111">Poniższy kod generuje CS1019:</span><span class="sxs-lookup"><span data-stu-id="1987e-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
