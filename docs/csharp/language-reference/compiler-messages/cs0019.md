# <a name="compiler-error-cs0019"></a>Błąd kompilatora CS0019

Nie można zastosować operatora "operator" do argumentów operacji typu "Type" i "Type"

 Operator binarny jest stosowany do typów danych, które nie obsługują tej funkcji. Nie można na [ &#124; ](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) przykład użyć operatora na ciągach, nie można użyć operatorów [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-)ani [>](../operators/comparison-operators.md#greater-than-operator-) w zmiennych [bool](../builtin-types/bool.md) i nie można użyć operatora [==](../operators/equality-operators.md#equality-operator-) z typem `struct`, chyba że typ jawnie przeciążuje tego operatora.

 Można przeciążyć operatora, aby obsługiwał operandy niektórych typów. Aby uzyskać więcej informacji, zobacz [przeciążanie operatora](../operators/operator-overloading.md).

## <a name="example"></a>Przykład

 W poniższym przykładzie CS0019 jest generowana w dwóch miejscach, ponieważ [bool](../builtin-types/bool.md) w C# nie jest konwertowany na [int](../builtin-types/integral-numeric-types.md). CS0019 jest również generowany, gdy operator odejmowania `-` jest stosowany do ciągu. Operatora dodawania `+` można używać z argumentami operacji typu String, ponieważ ten operator jest przeciążony przez klasę `String` do wykonywania łączenia ciągów.

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }
    
    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```

## <a name="example"></a>Przykład

 W poniższym przykładzie logika warunkowa musi być określona poza <xref:System.Diagnostics.ConditionalAttribute>. Do <xref:System.Diagnostics.ConditionalAttribute>można przekazać tylko jeden wstępnie zdefiniowany symbol.

 Poniższy przykład generuje CS0019:

```csharp
// CS0019_a.cs
// compile with: /target:library
using System.Diagnostics;

public class MyClass
{
   [ConditionalAttribute("DEBUG" || "TRACE")]   // CS0019
   public void TestMethod() {}

   // OK
   [ConditionalAttribute("DEBUG"), ConditionalAttribute("TRACE")]
   public void TestMethod2() {}
}
```
