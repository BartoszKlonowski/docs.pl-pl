## <a name="example"></a><span data-ttu-id="f2a65-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="f2a65-108">Example</span></span>

 <span data-ttu-id="f2a65-109">W poniższym przykładzie błąd CS0019 jest generowany w dwóch miejscach, ponieważ w języku C# nie można przekonwertować wartości [bool](../keywords/bool.md) na [int](../builtin-types/integral-numeric-types.md). Błąd CS0019 występuje również, gdy operator odejmowania jest stosowany dla ciągu znakowego.</span><span class="sxs-lookup"><span data-stu-id="f2a65-109">In the following example, CS0019 is generated in two places because [bool](../keywords/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="f2a65-110">Operatora dodawania (+) można używać z tekstowym argumentami operacji, ponieważ ten operator jest przeciążony przez klasę `String` w celu połączenia ciągu znaków.</span><span class="sxs-lookup"><span data-stu-id="f2a65-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```

 <span data-ttu-id="f2a65-114">Poniższy przykład generuje CS0019.</span><span class="sxs-lookup"><span data-stu-id="f2a65-114">The following sample generates CS0019.</span></span>
