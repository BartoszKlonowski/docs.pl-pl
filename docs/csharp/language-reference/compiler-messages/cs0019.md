# <a name="compiler-error-cs0019"></a>Błąd kompilatora CS0019

Nie można zastosować operatora "operator" do argumentów operacji typu "Type" i "Type"

 Operator binarny jest stosowany do typów danych, które go nie obsługują. Na przykład [ &#124; ](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) nie można użyć operatora dla ciągów, nie można użyć [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-)lub [0](../operators/comparison-operators.md#greater-than-operator-) operatorów w zmiennych [bool](../keywords/bool.md) i nie można użyć [3](../operators/equality-operators.md#equality-operator-) operator z typem 4, chyba że typ jawnie przeciążuje tego operatora.

 Można przeciążyć operatora, aby obsługiwał operandy niektórych typów. Aby uzyskać więcej informacji, zobacz przeciążanie [operatora](../operators/operator-overloading.md).

## <a name="example"></a>Przykład

 W poniższym przykładzie błąd CS0019 jest generowany w dwóch miejscach, ponieważ w języku C# nie można przekonwertować wartości [bool](../keywords/bool.md) na [int](../builtin-types/integral-numeric-types.md). Błąd CS0019 występuje również, gdy operator odejmowania jest stosowany dla ciągu znakowego. Operatora dodawania (+) można używać z tekstowym argumentami operacji, ponieważ ten operator jest przeciążony przez klasę `String` w celu połączenia ciągu znaków.

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }
    
    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```

## <a name="example"></a>Przykład

 W poniższym przykładzie należy określić logikę warunkową poza klasą <xref:System.Diagnostics.ConditionalAttribute>. Do klasy <xref:System.Diagnostics.ConditionalAttribute> można przekazać tylko jeden wstępnie zdefiniowany symbol.

 Poniższy przykład generuje CS0019:

```csharp
// CS0019_a.cs
// compile with: /target:library
using System.Diagnostics;

public class MyClass
{
   [ConditionalAttribute("DEBUG" || "TRACE")]   // CS0019
   public void TestMethod() {}

   // OK
   [ConditionalAttribute("DEBUG"), ConditionalAttribute("TRACE")]
   public void TestMethod2() {}
}
```
