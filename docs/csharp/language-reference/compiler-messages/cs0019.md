## <a name="example"></a><span data-ttu-id="762c7-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="762c7-109">Example</span></span>

<span data-ttu-id="762c7-110">W poniższym przykładzie CS0019 jest generowany w trzech miejscach, ponieważ [bool](../builtin-types/bool.md) w języku C# nie jest konwertowany na [int](../builtin-types/integral-numeric-types.md). CS0019 is also generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="762c7-111">Operatora dodawania `+` można używać z argumentami operandów ciągu, ponieważ ten operator jest przeciążony przez `String` klasę w celu wykonywania łączenia ciągów.</span><span class="sxs-lookup"><span data-stu-id="762c7-111">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
