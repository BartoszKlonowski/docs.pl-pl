## <a name="example"></a><span data-ttu-id="48d05-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="48d05-108">Example</span></span>

 <span data-ttu-id="48d05-109">W poniższym przykładzie błąd CS0019 jest generowany w dwóch miejscach, ponieważ w języku C# nie można przekonwertować wartości [bool](../../../csharp/language-reference/keywords/bool.md) na [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md). Błąd CS0019 występuje również, gdy operator odejmowania jest stosowany dla ciągu znakowego.</span><span class="sxs-lookup"><span data-stu-id="48d05-109">In the following example, CS0019 is generated in two places because [bool](../../../csharp/language-reference/keywords/bool.md) in C# is not convertible to [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="48d05-110">Operatora dodawania (+) można używać z tekstowym argumentami operacji, ponieważ ten operator jest przeciążony przez klasę `String` w celu połączenia ciągu znaków.</span><span class="sxs-lookup"><span data-stu-id="48d05-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```

## <a name="example"></a><span data-ttu-id="48d05-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="48d05-111">Example</span></span>

 <span data-ttu-id="48d05-114">Poniższy przykład spowoduje wygenerowanie CS0019.</span><span class="sxs-lookup"><span data-stu-id="48d05-114">The following sample generates CS0019.</span></span>  

```csharp
// CS0019_a.cs  
// compile with: /target:library  
using System.Diagnostics;  
public class MyClass  
{  
   [ConditionalAttribute("DEBUG" || "TRACE")]   // CS0019

   // OK  
   [ConditionalAttribute("DEBUG"), ConditionalAttribute("TRACE")]  
   public void TestMethod2() {}  
}  
```
