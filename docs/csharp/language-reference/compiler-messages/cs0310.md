 <span data-ttu-id="8fbe3-104">Ogólny typ lub metoda definiuje nowego ograniczenia w jego miejsce klauzuli, dlatego dowolny typ musi mieć publicznego konstruktora bez parametrów, aby mogły być używane jako argument typu dla tego typu ogólnego lub metody.</span><span class="sxs-lookup"><span data-stu-id="8fbe3-104">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="8fbe3-105">Aby uniknąć tego błędu, upewnij się, że typ ma konstruktora prawidłowe, lub zmodyfikuj Klauzula ograniczenia typu ogólnego lub metody.</span><span class="sxs-lookup"><span data-stu-id="8fbe3-105">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8fbe3-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="8fbe3-106">Example</span></span>  
 <span data-ttu-id="8fbe3-107">Poniższy przykład generuje CS0310:</span><span class="sxs-lookup"><span data-stu-id="8fbe3-107">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
