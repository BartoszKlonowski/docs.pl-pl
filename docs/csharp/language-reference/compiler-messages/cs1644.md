# <a name="compiler-error-cs1644"></a>Błąd kompilatora CS1644
Funkcja „funkcja” nie jest częścią standardowej specyfikacji ISO języka C# i inne kompilatory mogą jej nie akceptować  
  
 Ten błąd występuje, jeśli określono opcję [/langversion](../compiler-options/langversion-compiler-option.md) ISO-1, a w kompilowanym kodzie są używane funkcje spoza standardu ISO 1.0. Aby rozwiązać ten problem, nie używaj żadnych funkcji kompilatora C# w wersji 2.0, takich jak typy ogólne lub metody anonimowe z opcją zgodności z ISO-1.  
  
 Poniższy przykład generuje CS1644:  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```
