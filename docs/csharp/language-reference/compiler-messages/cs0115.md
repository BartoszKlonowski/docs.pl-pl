## <a name="example"></a><span data-ttu-id="25d65-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="25d65-106">Example</span></span>  
 <span data-ttu-id="25d65-107">Poniższy przykład generuje błąd CS0115.</span><span class="sxs-lookup"><span data-stu-id="25d65-107">The following sample generates CS0115.</span></span> <span data-ttu-id="25d65-108">Można się go pozbyć w jeden z dwóch sposobów:</span><span class="sxs-lookup"><span data-stu-id="25d65-108">You can resolve CS0115 in one of two ways:</span></span>  
  
- <span data-ttu-id="25d65-109">Usuń słowo kluczowe `override` z metody w bloku `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="25d65-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
- <span data-ttu-id="25d65-110">Użyj `MyClass1` jako klasy bazowej dla `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="25d65-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
