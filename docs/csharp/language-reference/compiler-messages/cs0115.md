# <a name="compiler-error-cs0115"></a>Błąd kompilatora CS0115
"Function": nie znaleziono odpowiedniej metody do przesłonięcia  
  
 Metoda została oznaczona jako przesłonięcie, ale kompilator nie odnalazł żadnej metody do przesłonięcia. Aby uzyskać więcej informacji, zobacz [przesłonięcie](../keywords/override.md) oraz [Korzystanie ze słów kluczowych „override” i „new”](../../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje błąd CS0115. Można się go pozbyć w jeden z dwóch sposobów:  
  
- Usuń słowo kluczowe `override` z metody w bloku `MyClass2`.  
  
- Użyj `MyClass1` jako klasy bazowej dla `MyClass2`.  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
