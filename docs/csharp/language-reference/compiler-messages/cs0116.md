# <a name="compiler-error-cs0116"></a><span data-ttu-id="dd763-102">Błąd kompilatora CS0116</span><span class="sxs-lookup"><span data-stu-id="dd763-102">Compiler Error CS0116</span></span>

<span data-ttu-id="dd763-103">Przestrzeń nazw nie może bezpośrednio zawierać składowych, takich jak pola lub metody.</span><span class="sxs-lookup"><span data-stu-id="dd763-103">A namespace cannot directly contain members such as fields or methods.</span></span>

<span data-ttu-id="dd763-104">Przestrzeń nazw może zawierać inne przestrzenie nazw, struktury i klasy.</span><span class="sxs-lookup"><span data-stu-id="dd763-104">A namespace can contain other namespaces, structs, and classes.</span></span> <span data-ttu-id="dd763-105">Aby uzyskać więcej informacji, zobacz artykuł [słowo kluczowe przestrzeni nazw](../keywords/namespace.md) .</span><span class="sxs-lookup"><span data-stu-id="dd763-105">For more information, see the [namespace keyword](../keywords/namespace.md) article.</span></span>

## <a name="example"></a><span data-ttu-id="dd763-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="dd763-106">Example</span></span>

<span data-ttu-id="dd763-107">Poniższy przykład powoduje, że program Visual Studio będzie flagować części kodu jako naruszenia CS0116.</span><span class="sxs-lookup"><span data-stu-id="dd763-107">The following sample will cause Visual Studio to flag parts of the code as being in violation of CS0116.</span></span> <span data-ttu-id="dd763-108">Próba skompilowania tego kodu spowoduje niepowodzenie kompilacji:</span><span class="sxs-lookup"><span data-stu-id="dd763-108">Attempting to build this code will result in build failure:</span></span>

```csharp
// CS0116.cs
namespace x
{
    // A namespace can be placed within another namespace.
    using System;

    // These variables trigger the CS0116 error as they are declared outside of a struct or class.
    public int latitude;
    public int longitude;
    Coordinate coord;

    // Auto-properties also fall under the definition of this rule.
    public string LocationName { get; set; }

    // This method as well: if it isn't in a class or a struct, it's violating CS0116.
    public void DisplayLatitude()
    {
        Console.WriteLine($"Lat: {latitude}");
    }

    public struct Coordinate
    {
    }

    public class CoordinatePrinter
    {
        public void DisplayLongitude()
        {
            Console.WriteLine($"Longitude: {longitude}");
        }

        public void DisplayLocation()
        {
            Console.WriteLine($"Location: {LocationName}");
        }
    }
}
```

<span data-ttu-id="dd763-109">Należy pamiętać, C#że w metodzie, metody i zmienne muszą być zadeklarowane i zdefiniowane w obrębie struktury lub klasy.</span><span class="sxs-lookup"><span data-stu-id="dd763-109">Note that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="dd763-110">Aby uzyskać więcej informacji na temat struktury C#programu w programie, zobacz [ogólna struktura C# artykułu programu](../../programming-guide/inside-a-program/general-structure-of-a-csharp-program.md) .</span><span class="sxs-lookup"><span data-stu-id="dd763-110">For more information on program structure in C#, see the [General Structure of a C# Program](../../programming-guide/inside-a-program/general-structure-of-a-csharp-program.md) article.</span></span> <span data-ttu-id="dd763-111">Aby naprawić ten błąd, należy ponownie napisać kod, aby wszystkie metody i pola były zawarte w strukturze lub klasie:</span><span class="sxs-lookup"><span data-stu-id="dd763-111">To fix this error, rewrite your code such that all methods and fields are contained within either a struct or a class:</span></span>

```csharp
namespace x
{
    // A namespace can be placed within another namespace.
    using System;

    // These variables are now placed within a struct, so CS0116 is no longer violated.
    public struct Coordinate
    {
        public int Latitude;
        public int Longitude;
    }

    // The methods and fields are now placed within a class, and the compiler is satisfied.
    public class CoordinatePrinter
    {
        Coordinate coord;
        public string LocationName { get; set; }

        public void DisplayLatitude()
        {
            Console.WriteLine($"Lat: {coord.Latitude}");
        }

        public void DisplayLongitude()
        {
            Console.WriteLine($"Longitude: {coord.Longitude}");
        }

        public void DisplayLocation()
        {
            Console.WriteLine($"Location: {LocationName}");
        }
    }
}
```
