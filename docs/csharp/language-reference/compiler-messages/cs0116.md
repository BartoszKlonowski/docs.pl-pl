# <a name="compiler-error-cs0116"></a><span data-ttu-id="3ba2a-102">Błąd kompilatora CS0116</span><span class="sxs-lookup"><span data-stu-id="3ba2a-102">Compiler Error CS0116</span></span>

<span data-ttu-id="3ba2a-103">Przestrzeń nazw nie zawiera bezpośrednio składowych, takich jak pola lub metody</span><span class="sxs-lookup"><span data-stu-id="3ba2a-103">A namespace does not directly contain members such as fields or methods</span></span>  
  
 <span data-ttu-id="3ba2a-104">Wewnątrz [przestrzeni nazw](../../../csharp/language-reference/keywords/namespace.md), kompilator akceptuje tylko klasy, struktury, złożenia, wyliczenia, interfejsów i delegatów.</span><span class="sxs-lookup"><span data-stu-id="3ba2a-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="3ba2a-105">Ten błąd często jest generowany przez deweloperów w tle języka C/C++, którzy należy zapominać, że w języku C#, metody i zmienne, musi być zadeklarowana i zdefiniowane w obrębie struktury lub klasy.</span><span class="sxs-lookup"><span data-stu-id="3ba2a-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="3ba2a-106">Aby uzyskać więcej informacji, zobacz [ogólna struktura programu w języku C#](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="3ba2a-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="3ba2a-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="3ba2a-107">Example</span></span>

 <span data-ttu-id="3ba2a-108">Poniższy przykład spowoduje wygenerowanie CS0116:</span><span class="sxs-lookup"><span data-stu-id="3ba2a-108">The following sample generates CS0116:</span></span>  

```csharp
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
