# <a name="compiler-error-cs0116"></a><span data-ttu-id="d72bf-103">Błąd kompilatora CS0116</span><span class="sxs-lookup"><span data-stu-id="d72bf-103">Compiler Error CS0116</span></span>

<span data-ttu-id="d72bf-104">Przestrzeń nazw nie może bezpośrednio zawierać składowych, takich jak pola lub metody.</span><span class="sxs-lookup"><span data-stu-id="d72bf-104">A namespace cannot directly contain members such as fields or methods.</span></span>

<span data-ttu-id="d72bf-105">Przestrzeń nazw może zawierać inne przestrzenie nazw, struktury i klasy.</span><span class="sxs-lookup"><span data-stu-id="d72bf-105">A namespace can contain other namespaces, structs, and classes.</span></span> <span data-ttu-id="d72bf-106">Aby uzyskać więcej informacji, zobacz artykuł [słowo kluczowe przestrzeni nazw](../keywords/namespace.md) .</span><span class="sxs-lookup"><span data-stu-id="d72bf-106">For more information, see the [namespace keyword](../keywords/namespace.md) article.</span></span>

## <a name="example"></a><span data-ttu-id="d72bf-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d72bf-107">Example</span></span>

<span data-ttu-id="d72bf-108">Poniższy przykład powoduje, że program Visual Studio będzie flagować części kodu jako naruszenia CS0116.</span><span class="sxs-lookup"><span data-stu-id="d72bf-108">The following sample will cause Visual Studio to flag parts of the code as being in violation of CS0116.</span></span> <span data-ttu-id="d72bf-109">Próba skompilowania tego kodu spowoduje niepowodzenie kompilacji:</span><span class="sxs-lookup"><span data-stu-id="d72bf-109">Attempting to build this code will result in build failure:</span></span>

```csharp
// CS0116.cs
namespace x
{
    // A namespace can be placed within another namespace.
    using System;

    // These variables trigger the CS0116 error as they are declared outside of a struct or class.
    public int latitude;
    public int longitude;
    Coordinate coord;

    // Auto-properties also fall under the definition of this rule.
    public string LocationName { get; set; }

    // This method as well: if it isn't in a class or a struct, it's violating CS0116.
    public void DisplayLatitude()
    {
        Console.WriteLine($"Lat: {latitude}");
    }

    public struct Coordinate
    {
    }

    public class CoordinatePrinter
    {
        public void DisplayLongitude()
        {
            Console.WriteLine($"Longitude: {longitude}");
        }

        public void DisplayLocation()
        {
            Console.WriteLine($"Location: {LocationName}");
        }
    }
}
```

<span data-ttu-id="d72bf-110">Należy pamiętać, że w języku C# metody i zmienne muszą być zadeklarowane i zdefiniowane w obrębie struktury lub klasy.</span><span class="sxs-lookup"><span data-stu-id="d72bf-110">Note that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="d72bf-111">Aby uzyskać więcej informacji na temat struktury programu w języku C#, zobacz [ogólną strukturę artykułu w języku c#](../../programming-guide/inside-a-program/general-structure-of-a-csharp-program.md) .</span><span class="sxs-lookup"><span data-stu-id="d72bf-111">For more information on program structure in C#, see the [General Structure of a C# Program](../../programming-guide/inside-a-program/general-structure-of-a-csharp-program.md) article.</span></span> <span data-ttu-id="d72bf-112">Aby naprawić ten błąd, należy ponownie napisać kod, aby wszystkie metody i pola były zawarte w strukturze lub klasie:</span><span class="sxs-lookup"><span data-stu-id="d72bf-112">To fix this error, rewrite your code such that all methods and fields are contained within either a struct or a class:</span></span>

```csharp
namespace x
{
    // A namespace can be placed within another namespace.
    using System;

    // These variables are now placed within a struct, so CS0116 is no longer violated.
    public struct Coordinate
    {
        public int Latitude;
        public int Longitude;
    }

    // The methods and fields are now placed within a class, and the compiler is satisfied.
    public class CoordinatePrinter
    {
        Coordinate coord;
        public string LocationName { get; set; }

        public void DisplayLatitude()
        {
            Console.WriteLine($"Lat: {coord.Latitude}");
        }

        public void DisplayLongitude()
        {
            Console.WriteLine($"Longitude: {coord.Longitude}");
        }

        public void DisplayLocation()
        {
            Console.WriteLine($"Location: {LocationName}");
        }
    }
}
```
