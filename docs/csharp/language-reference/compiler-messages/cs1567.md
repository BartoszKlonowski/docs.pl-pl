# <a name="compiler-error-cs1567"></a>Błąd kompilatora CS1567
Błąd podczas generowania pliku zasobu Win32: 'Plik'.  
  
 Twoja kompilacja, albo użyła opcji kompilatora - [/win32icon](../../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) lub nie użyła [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), co spowodowało, że kompilator, wygenerował plik zawierający informacje o zasobie, ale nie był w stanie utowrzyć pliku z powodu niewystarczającego miejsca na dysku lub innego błędu.  
  
 Jeśli nie możesz rozwiązać problemu generowania plików, możesz użyć opcji [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), która nie generuje pliku zawierającego informacje o zasobie.
