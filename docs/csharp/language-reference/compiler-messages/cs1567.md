# <a name="compiler-error-cs1567"></a>Błąd kompilatora CS1567
Błąd podczas generowania zasobu Win32: „plik”.  
  
 Twoja kompilacja albo użyła opcji kompilatora [/win32icon](../compiler-options/win32icon-compiler-option.md), albo nie użyła opcji [/win32res](../compiler-options/win32res-compiler-option.md), co spowodowało, że kompilator wygenerował plik zawierający informacje o zasobie, ale nie był w stanie utworzyć pliku z powodu niewystarczającego miejsca na dysku lub innego błędu.  
  
 Jeśli nie możesz rozwiązać problemu generowania plików, możesz użyć opcji [/win32res](../compiler-options/win32res-compiler-option.md), która nie generuje pliku zawierającego informacje o zasobie.
