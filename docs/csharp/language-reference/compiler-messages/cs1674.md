 <span data-ttu-id="91811-104">Instrukcja [using](../../../csharp/language-reference/keywords/using-statement.md) służy do zapewnienia, że obiekt na końcu bloku `using` zostanie usunięty, w związku można jej używać tylko z tymi typami, które są usuwalne. </span><span class="sxs-lookup"><span data-stu-id="91811-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="91811-105"> Usunąć nie można na przykład typów wartości ani parametrów typu, które nie są ograniczone jako klasy.</span><span class="sxs-lookup"><span data-stu-id="91811-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="91811-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="91811-106">Example</span></span>

## <a name="example"></a><span data-ttu-id="91811-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="91811-108">Example</span></span>  
 <span data-ttu-id="91811-109">Kod ten generuje błąd CS1674.</span><span class="sxs-lookup"><span data-stu-id="91811-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
