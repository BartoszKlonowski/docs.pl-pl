## <a name="example"></a><span data-ttu-id="9d9a4-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="9d9a4-108">Example</span></span>

```csharp

   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }

class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}

      using (D d = new D()) {}  
   }

```
