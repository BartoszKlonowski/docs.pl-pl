 <span data-ttu-id="ce50d-105">[Instrukcja using](../keywords/using-statement.md) jest przeznaczona do użycia w celu zapewnienia usuwania obiektu na końcu `using` bloku, w związku z tym, w takiej instrukcji można używać tylko typów, które są jednorazowe.</span><span class="sxs-lookup"><span data-stu-id="ce50d-105">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="ce50d-106">Na przykład typy wartości nie są jednorazowe i parametry typu, które nie są ograniczone do klas, nie mogą być traktowane jako jednorazowe.</span><span class="sxs-lookup"><span data-stu-id="ce50d-106">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>

## <a name="example"></a><span data-ttu-id="ce50d-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="ce50d-109">Example</span></span>  
 <span data-ttu-id="ce50d-110">Poniższy przykład generuje CS1674.</span><span class="sxs-lookup"><span data-stu-id="ce50d-110">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
