# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="73cec-102">Ostrzeżenie kompilatora (poziom 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="73cec-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="73cec-103">Użycie metody przeciążonej „metoda”, która różni się tylko nienazwanymi typami tablic, nie jest zgodne ze specyfikacją CLS.</span><span class="sxs-lookup"><span data-stu-id="73cec-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="73cec-104">Ten błąd występuje, gdy jest używana przeciążona metoda pobierająca tablicę nieregularną, a jedyną różnicą między podpisami metod jest typ elementu tablicy.</span><span class="sxs-lookup"><span data-stu-id="73cec-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="73cec-105">Aby uniknąć tego błędu, należy rozważyć użycie tablicy prostokątnej, a nie postrzępionej tablicy; użyć dodatkowego parametru, aby rozróżnić wywołanie funkcji; zmienić nazwę jednej lub więcej przeciążonych metod; lub, jeśli zgodność ze schyłą nie jest potrzebna, usuń <xref:System.CLSCompliantAttribute> atrybut.</span><span class="sxs-lookup"><span data-stu-id="73cec-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="73cec-106">Aby uzyskać więcej informacji na temat zgodności z cls, zobacz [Niezależność od języka i składniki niezależne od języka](../../../standard/language-independence-and-language-independent-components.md).</span><span class="sxs-lookup"><span data-stu-id="73cec-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="73cec-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="73cec-107">Example</span></span>  
 <span data-ttu-id="73cec-108">W poniższym przykładzie jest generowany błąd kompilatora CS3007:</span><span class="sxs-lookup"><span data-stu-id="73cec-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
