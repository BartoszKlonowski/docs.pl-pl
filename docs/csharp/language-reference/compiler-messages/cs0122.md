# <a name="compiler-error-cs0122"></a>Błąd kompilatora CS0122
"członek" jest niedostępny z powodu swojego poziomu ochrony  
  
 [Modyfikator dostępu](../../../csharp/language-reference/keywords/modifiers.md) klasy elementu członkowskiego uniemożliwia dostęp do elementu członkowskiego. Aby uzyskać więcej informacji, zobacz [Modyfikatory dostępu](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).   
  
 Jedną z przyczyn to (nie pokazano w poniższym przykładzie) można pominięcie **/out** flagi kompilatora w elemencie docelowym przyjaznego zestawu. Aby uzyskać więcej informacji, zobacz [przyjaznych zestawów](../../../standard/assembly/friend-assemblies.md) i [/out (C# opcje kompilatora)](../../../csharp/language-reference/compiler-options/out-compiler-option.md)  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0122:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
