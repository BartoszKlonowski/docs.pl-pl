# <a name="compiler-error-cs0122"></a>Błąd kompilatora CS0122
element "member" jest niedostępny z powodu swojego poziomu ochrony  
  
 [Modyfikator dostępu](../keywords/modifiers.md) dla elementu członkowskiego klasy uniemożliwia dostęp do elementu członkowskiego. Aby uzyskać więcej informacji, zobacz [Modyfikatory dostępu](../../programming-guide/classes-and-structs/access-modifiers.md).  
  
 Jedną z przyczyn tego błędu (nie pokazano w przykładzie poniżej) można pominąć flagi kompilatora **/out** w elemencie docelowym zestawu zaprzyjaźnionego. Aby uzyskać więcej informacji, zobacz [zaprzyjaźnione zestawy](../../../standard/assembly/friend-assemblies.md) i [/out (C# opcje kompilatora)](../compiler-options/out-compiler-option.md)  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0122:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
