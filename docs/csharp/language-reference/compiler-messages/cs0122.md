## <a name="example"></a><span data-ttu-id="0a4a6-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="0a4a6-108">Example</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122

   }
}
```
