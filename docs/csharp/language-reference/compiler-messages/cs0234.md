# <a name="compiler-error-cs0234"></a>Błąd kompilatora CS0234
Typu lub przestrzeni nazw nazwa "name" nie istnieje w przestrzeni nazw "namespace" (czy nie brakuje odwołania do zestawu?)  
  
 Oczekiwano typu. Możliwe przyczyny tego błędu są następujące:  
  
- Zestaw, który zawiera definicję typu nie jest przywoływany w kompilacji; Użyj [/Reference (Importuj metadane)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) do określenia zestawu  
  
- Przekazana nazwa zmiennej, do [typeof](../../../csharp/language-reference/keywords/typeof.md) operatora.  
  
- Próbowano odwołanie do zestawu, który nie jest częścią docelowej profilu .NET Framework. Aby uzyskać więcej informacji, zobacz [Rozwiązywanie problemów z celem błędy programu .NET Framework](/visualstudio/msbuild/troubleshooting-dotnet-framework-targeting-errors).  
  
 Jeśli zostanie wyświetlony ten błąd, po przeniesieniu kodu od komputera deweloperskiego z jednego do drugiego, upewnij się, że projekt na nowej maszynie ma poprawne odwołania i wersje zestawów są takie same jak w starym maszyny. Przeglądarka obiektów umożliwia również sprawdzić zestawu i sprawdź, czy zawiera typy, które mogą zawierać.  
  
 Poniższy przykład spowoduje wygenerowanie CS0234:  
  
```csharp  
// CS0234.cs  
public class C  
{  
   public static void Main()  
   {  
      System.DateTime x = new System.DateTim();   // CS0234  
      // try the following line instead  
      // System.DateTime x = new System.DateTime();  
   }  
}  
```
