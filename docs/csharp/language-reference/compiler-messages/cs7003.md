# <a name="compiler-error-cs7003"></a>Błąd kompilatora CS7003

Nieoczekiwane użycie niepowiązanej nazwy ogólnej

Ten błąd występuje, jeśli używasz wymagające jeden parametr typu ogólnego typu bez przekazywania dowolną nazwę typu parametru ogólnego między nawiasami. Użyj tego może być deklaracji zmiennej lub tworzenia wystąpienia obiektu.

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  

Podczas korzystania z typu ogólnego, należy podać jedną nazwę typu parametru w nawiasy ostre.

## <a name="example"></a>Przykład

Poniższy przykład generuje CS7003:

```csharp
// CS7003.cs  
    class Program
    {
        static void Main(string[] args)
        {
            var myVar1 = new MyGenericClass<>();  //CS7003

            MyGenericClass<> var2;                //CS7003
        }
    }

    public class MyGenericClass<T> { }
```
