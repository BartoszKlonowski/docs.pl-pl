# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="9ed67-102">Ostrzeżenie kompilatora (poziom 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="9ed67-102">Compiler Warning (level 2) CS0618</span></span>

 <span data-ttu-id="9ed67-104">Element członkowski klasy została oznaczona atrybutem `Obsolete` atrybutu w taki sposób, że gdy odwołuje się do elementu członkowskiego klasy zostanie wygenerowane ostrzeżenie.</span><span class="sxs-lookup"><span data-stu-id="9ed67-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="9ed67-105">Aby uzyskać więcej informacji, zobacz [takie same atrybuty wspólne](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="9ed67-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="9ed67-106">Poniższy przykład generuje CS0618:</span><span class="sxs-lookup"><span data-stu-id="9ed67-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
