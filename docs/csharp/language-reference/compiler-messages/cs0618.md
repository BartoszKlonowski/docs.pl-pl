# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="418d5-103">Ostrzeżenie kompilatora (poziom 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="418d5-103">Compiler Warning (level 2) CS0618</span></span>

<span data-ttu-id="418d5-104">element "member" jest przestarzały: "text"</span><span class="sxs-lookup"><span data-stu-id="418d5-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="418d5-105">Element członkowski klasy został oznaczony `Obsolete` atrybutem, w taki sposób, że w przypadku odwołania do elementu członkowskiego klasy zostanie wygenerowane ostrzeżenie.</span><span class="sxs-lookup"><span data-stu-id="418d5-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="418d5-106">Aby uzyskać więcej informacji, zobacz [Common Attributes](../attributes/global.md).</span><span class="sxs-lookup"><span data-stu-id="418d5-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="418d5-107">Poniższy przykład generuje CS0618:</span><span class="sxs-lookup"><span data-stu-id="418d5-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
