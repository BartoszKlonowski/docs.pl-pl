# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="7ffd6-102">Ostrzeżenie kompilatora (poziom 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="7ffd6-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="7ffd6-103">"identyfikator": odniesienie do pola lotnego nie będzie traktowane jako lotne</span><span class="sxs-lookup"><span data-stu-id="7ffd6-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="7ffd6-104">Pole lotne zwykle nie powinno `ref` być przekazywane przy użyciu parametru lub **wyjścia,** ponieważ nie będzie traktowane jako lotne w zakresie funkcji.</span><span class="sxs-lookup"><span data-stu-id="7ffd6-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="7ffd6-105">Istnieją wyjątki od tej reguły, takie jak podczas wywoływania zablokowanego interfejsu API.</span><span class="sxs-lookup"><span data-stu-id="7ffd6-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="7ffd6-106">Podobnie jak w przypadku każdego ostrzeżenia, można użyć [ostrzeżenia #pragma,](../preprocessor-directives/preprocessor-pragma-warning.md) aby wyłączyć to ostrzeżenie w tych rzadkich przypadkach, gdy celowo używasz pola lotnego jako parametr ulotny.</span><span class="sxs-lookup"><span data-stu-id="7ffd6-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="7ffd6-107">Poniższy przykład generuje CS0420:</span><span class="sxs-lookup"><span data-stu-id="7ffd6-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
