# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="4d293-102">Ostrzeżenie kompilatora (poziom 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="4d293-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="4d293-103">'Identyfikator': odwołanie do pola nietrwałego nie będzie traktowane jako nietrwałe</span><span class="sxs-lookup"><span data-stu-id="4d293-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="4d293-104">Pola nietrwałego nie należy zwykle przekazywać przy użyciu `ref` lub **się** parametru, ponieważ nie będzie traktowane jako nietrwałe w zakresie funkcji.</span><span class="sxs-lookup"><span data-stu-id="4d293-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="4d293-105">Istnieją wyjątki od tej, takie jak podczas wywołanie blokowanego interfejsu API.</span><span class="sxs-lookup"><span data-stu-id="4d293-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="4d293-106">Wszelkie ostrzeżenia mogą używać [ostrzeżenie #pragma](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) Aby wyłączyć to ostrzeżenie w tych rzadkich przypadkach, w których umyślnie używasz pola nietrwałego jako parametr przekazany przez odwołanie.</span><span class="sxs-lookup"><span data-stu-id="4d293-106">As with any warning, you may use the [#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="4d293-107">Poniższy przykład spowoduje wygenerowanie CS0420:</span><span class="sxs-lookup"><span data-stu-id="4d293-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
