# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="58d1f-103">Ostrzeżenie kompilatora (poziom 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="58d1f-103">Compiler Warning (level 1) CS0420</span></span>

<span data-ttu-id="58d1f-104">"Identyfikator": odwołanie do pola nietrwałego nie będzie traktowane jako nietrwałe</span><span class="sxs-lookup"><span data-stu-id="58d1f-104">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="58d1f-105">Pole nietrwałe nie powinno być zwykle przenoszone przy użyciu `ref` parametru lub **out** , ponieważ nie będzie traktowane jako nietrwały w zakresie funkcji.</span><span class="sxs-lookup"><span data-stu-id="58d1f-105">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="58d1f-106">Istnieją między innymi wyjątki, takie jak podczas wywoływania interfejsu API z blokadą.</span><span class="sxs-lookup"><span data-stu-id="58d1f-106">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="58d1f-107">Podobnie jak w przypadku dowolnego ostrzeżenia, można użyć [ostrzeżenia #pragma](../preprocessor-directives/preprocessor-pragma-warning.md) , aby wyłączyć to ostrzeżenie w rzadkich przypadkach, gdy celowo używasz pola nietrwałego jako parametru odwołania.</span><span class="sxs-lookup"><span data-stu-id="58d1f-107">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="58d1f-108">Poniższy przykład generuje CS0420:</span><span class="sxs-lookup"><span data-stu-id="58d1f-108">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
