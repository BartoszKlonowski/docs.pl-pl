## <a name="example"></a><span data-ttu-id="dedd6-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="dedd6-109">Example</span></span>  

 <span data-ttu-id="dedd6-110">Poniższy przykładowy kod generuje CS0650.</span><span class="sxs-lookup"><span data-stu-id="dedd6-110">The following example code generates CS0650.</span></span>  
  
```csharp  
// CS0650.cs  
public class MyClass  
{  
   public static void Main()  
   {  
// Generates CS0650. Incorrect array declaration syntax:  
      int myarray[2];
  
      // Correct declaration.  
      int[] myarray2;  
  
      // Declaration and initialization in one statement  
      int[] myArray3= new int[2] {1,2}  
  
      // Access an array element.  
      myarray3[0] = 0;  
    }  
}  
```
