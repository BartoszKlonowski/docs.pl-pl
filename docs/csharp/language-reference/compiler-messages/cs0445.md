# <a name="compiler-error-cs0445"></a><span data-ttu-id="a0cf0-103">Błąd kompilatora CS0445</span><span class="sxs-lookup"><span data-stu-id="a0cf0-103">Compiler Error CS0445</span></span>

<span data-ttu-id="a0cf0-104">Nie można zmodyfikować wyniku konwersji rozpakowywania</span><span class="sxs-lookup"><span data-stu-id="a0cf0-104">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="a0cf0-105">Wynikiem konwersji rozpakowywania jest zmienna tymczasowa.</span><span class="sxs-lookup"><span data-stu-id="a0cf0-105">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="a0cf0-106">Kompilator zapobiega modyfikowaniu takich zmiennych, ponieważ wszelkie modyfikacje byłyby, gdy zmienna tymczasowa zostanie wysunięta.</span><span class="sxs-lookup"><span data-stu-id="a0cf0-106">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="a0cf0-107">Aby rozwiązać ten problem, Zadeklaruj nową zmienną typu wartości do przechowywania wyrażenia pośredniego i przypisz wynik konwersji rozpakowywania do tej zmiennej.</span><span class="sxs-lookup"><span data-stu-id="a0cf0-107">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="a0cf0-108">Poniższy kod generuje CS0455.</span><span class="sxs-lookup"><span data-stu-id="a0cf0-108">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
