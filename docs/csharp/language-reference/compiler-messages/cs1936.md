# <a name="compiler-error-cs1936"></a><span data-ttu-id="45d53-102">Błąd kompilatora CS1936</span><span class="sxs-lookup"><span data-stu-id="45d53-102">Compiler Error CS1936</span></span>

<span data-ttu-id="45d53-103">Nie można odnaleźć implementacji wzorca kwerendy dla typu źródłowego "type".</span><span class="sxs-lookup"><span data-stu-id="45d53-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="45d53-104">"metody" nie znaleziono.</span><span class="sxs-lookup"><span data-stu-id="45d53-104">'method' not found.</span></span>  
  
<span data-ttu-id="45d53-105">Aby zbadać typ źródła, ten typ musi implementować standardowe metody operatora kwerendy, które są wywoływane w kwerendzie.</span><span class="sxs-lookup"><span data-stu-id="45d53-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="45d53-106">Implementacja może być w postaci elementów członkowskich klasy lub metod rozszerzenia, `using` które są wprowadzane w zakres z odpowiedniej dyrektywy.</span><span class="sxs-lookup"><span data-stu-id="45d53-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="45d53-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="45d53-110">Example</span></span>

<span data-ttu-id="45d53-111">Poniższy przykład daje CS1936:</span><span class="sxs-lookup"><span data-stu-id="45d53-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="45d53-112">Ten błąd zazwyczaj występuje, gdy przypadkowo spróbujesz zbadać obiekt pewnego typu zamiast kolekcji tych obiektów.</span><span class="sxs-lookup"><span data-stu-id="45d53-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
