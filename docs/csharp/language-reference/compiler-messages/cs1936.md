# <a name="compiler-error-cs1936"></a><span data-ttu-id="2b8f5-103">Błąd kompilatora CS1936</span><span class="sxs-lookup"><span data-stu-id="2b8f5-103">Compiler Error CS1936</span></span>

<span data-ttu-id="2b8f5-106">Aby można było wykonać zapytanie o typ źródłowy, ten typ musi implementować standardowe metody operatora zapytania, które są wywoływane w zapytaniu.</span><span class="sxs-lookup"><span data-stu-id="2b8f5-106">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="2b8f5-107">Implementacja może być w postaci elementów członkowskich klasy lub metod rozszerzających, które są wprowadzane do zakresu z odpowiednią `using` dyrektywą.</span><span class="sxs-lookup"><span data-stu-id="2b8f5-107">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="2b8f5-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="2b8f5-111">Example</span></span>

<span data-ttu-id="2b8f5-112">Poniższy przykład generuje CS1936:</span><span class="sxs-lookup"><span data-stu-id="2b8f5-112">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="2b8f5-113">Ten błąd występuje zazwyczaj, gdy przypadkowo próbujesz zbadać obiekt pewnego typu zamiast kolekcji tych obiektów.</span><span class="sxs-lookup"><span data-stu-id="2b8f5-113">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
