<span data-ttu-id="1cf7a-105">Aby można było wykonać zapytanie o typ źródła, typ musi implementować standardowe metody operatorów zapytania, które są wywoływane w zapytaniu.</span><span class="sxs-lookup"><span data-stu-id="1cf7a-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="1cf7a-106">Implementacja może być w formie elementów członkowskich klasy lub metod rozszerzenia, które są wprowadzane za pomocą odpowiedniej dyrektywy `using`.</span><span class="sxs-lookup"><span data-stu-id="1cf7a-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="1cf7a-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="1cf7a-110">Example</span></span>

<span data-ttu-id="1cf7a-111">Poniższy przykład tworzy CS1936:</span><span class="sxs-lookup"><span data-stu-id="1cf7a-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="1cf7a-112">Ten błąd występuje zazwyczaj podczas próby przypadkowego wykonania zapytania do obiektu określonego typu zamiast do kolekcji tych obiektów.
</span><span class="sxs-lookup"><span data-stu-id="1cf7a-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
