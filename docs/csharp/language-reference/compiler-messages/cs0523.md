# <a name="compiler-error-cs0523"></a>Błąd kompilatora CS0523
Element członkowski struktury "pole struktury struct2" typu "struct1" powoduje wystąpienie cyklu w układzie struktury  
  
 Definicje dwóch struktur zawiejają odwołanie cykliczne. Zmień definicje [struktury](../../../csharp/language-reference/keywords/struct.md) tak, aby żadna nie definiowała się w drugiej. Powyższe ograniczenie ma zastosowanie tylko do struktur, ponieważ struktury są typem wartościowym. Jeśli chcesz użyć odwołań cyklicznych, musisz zadeklarować swój typ jako klasę.  
  
 Poniższy przykład generuje CS0523:  
  
```csharp  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
