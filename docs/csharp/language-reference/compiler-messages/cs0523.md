# <a name="compiler-error-cs0523"></a>Błąd kompilatora CS0523
Element członkowski struktury „pole struct2” typu „struct1” powoduje wystąpienie cyklu w układzie struktury  
  
 Definicje dwóch struktur zawierają odwołania cykliczne. Zmień definicje [struktury](../builtin-types/struct.md) , tak aby każda z nich nie została zdefiniowana na drugim. To ograniczenie dotyczy tylko struktur, ponieważ są one typem wartościowym. Aby użyć odwołań cyklicznych, zadeklaruj typy jako klasy.  
  
 Poniższy przykład generuje CS0523:  
  
```csharp  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
