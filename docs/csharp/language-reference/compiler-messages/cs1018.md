# <a name="compiler-error-cs1018"></a>Błąd kompilatora CS1018
Słowo kluczowe "to" lub "podstawa" oczekiwane  
  
 Kompilator napotkał niekompletną deklarację konstruktora.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1018 i sugeruje kilka sposobów rozwiązania błędu:  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
