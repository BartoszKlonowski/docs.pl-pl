# <a name="compiler-error-cs1018"></a>Błąd kompilatora CS1018
Oczekiwano słowa kluczowego "this" lub "base"  
  
 Kompilator napotkał niekompletną deklarację konstruktora.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1018, a także sugeruje usunięcie przyczyny błędu na kilka sposobów:  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // możliwe rozwiązania:  
    // public a () usunięcie dwukropka
    // public a () : base() wywołuje domyślny konstruktor C 
    // public a () : this(1) wywołuje przypisany konstruktor klasy a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
