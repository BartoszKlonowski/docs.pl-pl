# <a name="compiler-error-cs1612"></a><span data-ttu-id="2e316-102">Błąd kompilatora CS1612</span><span class="sxs-lookup"><span data-stu-id="2e316-102">Compiler Error CS1612</span></span>
<span data-ttu-id="2e316-103">Nie można zmodyfikować zwracanej wartości "expression", ponieważ nie jest to zmienna</span><span class="sxs-lookup"><span data-stu-id="2e316-103">Cannot modify the return value of 'expression' because it is not a variable</span></span>  
  
 <span data-ttu-id="2e316-104">Nastąpiła próba do modyfikowania typu wartości, który jest generowany w wyniku wyrażenia pośredniego, ale nie są przechowywane w zmiennej.</span><span class="sxs-lookup"><span data-stu-id="2e316-104">An attempt was made to modify a value type that is produced as the result of an intermediate expression but is not stored in a variable.</span></span> <span data-ttu-id="2e316-105">Ten błąd może wystąpić przy próbie bezpośrednio modyfikować struktury w kolekcji ogólnej, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="2e316-105">This error can occur when you attempt to directly modify a struct in a generic collection, as shown in the following example:</span></span>  
  
```csharp  
List<myStruct> list = {…};  
list[0].Name = "MyStruct42"; //CS1612  
```  
  
 <span data-ttu-id="2e316-106">Aby zmodyfikować struktury, najpierw przypisać ją do zmiennej lokalnej, zmodyfikować zmienną, a następnie przypisać zmiennej wstecz do elementu w kolekcji.</span><span class="sxs-lookup"><span data-stu-id="2e316-106">To modify the struct, first assign it to a local variable, modify the variable, then assign the variable back to the item in the collection.</span></span>  
  
```csharp  
List<myStruct> list = {…};  
MyStruct ms = list[0];  
ms.Name = "MyStruct42";  
list[0] = ms;  
```  
  
 <span data-ttu-id="2e316-107">Ten błąd występuje, ponieważ typy wartości są kopiowane w przydziale.</span><span class="sxs-lookup"><span data-stu-id="2e316-107">This error occurs because value types are copied on assignment.</span></span> <span data-ttu-id="2e316-108">Podczas pobierania typu wartości z właściwości lub indeksatora, otrzymujesz kopię obiektu, a nie odwołanie do samego obiektu.</span><span class="sxs-lookup"><span data-stu-id="2e316-108">When you retrieve a value type from a property or indexer, you are getting a copy of the object, not a reference to the object itself.</span></span> <span data-ttu-id="2e316-109">Kopia, która jest zwracana nie jest przechowywany przez właściwość lub indeksator, ponieważ są one faktycznie metod, nie lokalizacjom magazynu (zmienne).</span><span class="sxs-lookup"><span data-stu-id="2e316-109">The copy that is returned is not stored by the property or indexer because they are actually methods, not storage locations (variables).</span></span> <span data-ttu-id="2e316-110">Należy zapisać kopię do zmiennej możesz deklarować zanim można go zmodyfikować.</span><span class="sxs-lookup"><span data-stu-id="2e316-110">You must store the copy into a variable that you declare before you can modify it.</span></span>  
  
 <span data-ttu-id="2e316-111">Ten błąd nie występuje w przypadku typów referencyjnych, ponieważ właściwość lub indeksator w takiej sytuacji zwraca odwołanie do istniejącego obiektu, który prowadzi do lokalizacji magazynu.</span><span class="sxs-lookup"><span data-stu-id="2e316-111">The error does not occur with reference types because a property or indexer in that case returns a reference to an existing object, which is a storage location.</span></span>  
  
 <span data-ttu-id="2e316-112">Jeśli definiujesz, klasy lub struktury, możesz rozwiązać ten problem przez zmodyfikowanie Twojego deklaracja właściwości w celu zapewnienia dostępu do elementów członkowskich struktury.</span><span class="sxs-lookup"><span data-stu-id="2e316-112">If you are defining the class or struct, you can resolve this error by modifying your property declaration to provide access to the members of a struct.</span></span> <span data-ttu-id="2e316-113">Jeśli piszesz kod klienta można rozwiązać błąd, tworzenia wystąpienia struktury, modyfikowania jej pola, a następnie przypisując całej strukturze, wróć do właściwości.</span><span class="sxs-lookup"><span data-stu-id="2e316-113">If you are writing client code, you can resolve the error by creating your own instance of the struct, modifying its fields, and then assigning the entire struct back to the property.</span></span> <span data-ttu-id="2e316-114">Trzeci alternatywę można zmienić swoje struktury do klasy.</span><span class="sxs-lookup"><span data-stu-id="2e316-114">As a third alternative, you can change your struct to a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2e316-115">Przykład</span><span class="sxs-lookup"><span data-stu-id="2e316-115">Example</span></span>  
 <span data-ttu-id="2e316-116">CS1612 występuje także w przypadku, gdy próba dostępu do elementu członkowskiego struktury za pomocą właściwości w otaczającej klasy, która zwraca całej struktury, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="2e316-116">CS1612 also occurs when you attempt to access the member of a struct through a property on an enclosing class that is returning the entire struct, as shown in the following example:</span></span>  
  
```csharp  
// CS1612.cs  
using System;  
  
public struct MyStruct  
{  
    public int Width;  
}  
  
public class ListView  
{  
    MyStruct ms;  
    public MyStruct Size  
    {  
        get { return ms; }  
        set { ms = value; }  
    }  
}  
  
public class MyClass  
{  
    public MyClass()  
    {  
        ListView lvi;  
        lvi = new ListView();  
        lvi.Size.Width = 5; // CS1612  
  
        // You can use the following lines instead.  
        // MyStruct ms;  
        // ms.Width = 5;  
        // lvi.Size = ms;  // CS1612  
    }  
  
    public static void Main()   
    {  
        MyClass mc = new MyClass();  
        // Keep the console open in debug mode.  
        Console.WriteLine("Press any key to exit.");  
        Console.ReadKey();     
    }  
}  
```
