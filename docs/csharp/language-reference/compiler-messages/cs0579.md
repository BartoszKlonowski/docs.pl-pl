# <a name="compiler-error-cs0579"></a>Błąd kompilatora CS0579
Zduplikowany atrybut „atrybut”.  
  
 Tego samego atrybutu nie można określić więcej niż jeden raz, chyba że dla [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md) określono **AllowMultiple = true**.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0579.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
