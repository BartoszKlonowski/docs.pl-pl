# <a name="compiler-error-cs1001"></a><span data-ttu-id="8d0e4-102">Błąd kompilatora CS1001</span><span class="sxs-lookup"><span data-stu-id="8d0e4-102">Compiler Error CS1001</span></span>
<span data-ttu-id="8d0e4-103">Oczekiwano identyfikatora</span><span class="sxs-lookup"><span data-stu-id="8d0e4-103">Identifier expected</span></span>  
  
 <span data-ttu-id="8d0e4-104">Nie podano identyfikatora.</span><span class="sxs-lookup"><span data-stu-id="8d0e4-104">You did not supply an identifier.</span></span> <span data-ttu-id="8d0e4-105">Identyfikator jest nazwa klasy, struktury, przestrzeni nazw, metody, zmiennej i tak dalej, podane.</span><span class="sxs-lookup"><span data-stu-id="8d0e4-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="8d0e4-106">W poniższym przykładzie deklaruje prostą klasę, ale nie nadaj klasie nazwę:</span><span class="sxs-lookup"><span data-stu-id="8d0e4-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="8d0e4-107">Poniższy przykład generuje CS1001, ponieważ przy deklarowaniu wyliczenia, należy określić członków:</span><span class="sxs-lookup"><span data-stu-id="8d0e4-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="8d0e4-108">Nazwy parametrów są wymagane, nawet jeśli kompilator nie używa ich, na przykład w definicji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8d0e4-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="8d0e4-109">Te parametry są wymagane, aby poinformować programistów, którzy zużywają interfejsu coś o co oznaczają parametrów.</span><span class="sxs-lookup"><span data-stu-id="8d0e4-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
