# <a name="compiler-error-cs1001"></a><span data-ttu-id="c788e-102">Błąd kompilatora CS1001</span><span class="sxs-lookup"><span data-stu-id="c788e-102">Compiler Error CS1001</span></span>

<span data-ttu-id="c788e-103">Oczekiwano identyfikatora</span><span class="sxs-lookup"><span data-stu-id="c788e-103">Identifier expected</span></span>

<span data-ttu-id="c788e-104">Nie dostarczono identyfikatora.</span><span class="sxs-lookup"><span data-stu-id="c788e-104">You did not supply an identifier.</span></span> <span data-ttu-id="c788e-105">Identyfikator to nazwa klasy, struktury, przestrzeni nazw, metody, zmiennej i tak dalej.</span><span class="sxs-lookup"><span data-stu-id="c788e-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="c788e-106">Poniższy przykład deklaruje prostą klasę, ale nie przydaje klasy a nazwy:</span><span class="sxs-lookup"><span data-stu-id="c788e-106">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="c788e-107">Poniższy przykład generuje CS1001, ponieważ w przypadku deklarowania wyliczenia należy określić członków:</span><span class="sxs-lookup"><span data-stu-id="c788e-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="c788e-108">Nazwy parametrów są wymagane nawet wtedy, gdy kompilator nie korzysta z nich, na przykład w definicji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="c788e-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="c788e-109">Parametry te są wymagane, aby programiści korzystający z interfejsu wiedzieli coś o znaczeniu parametrów.</span><span class="sxs-lookup"><span data-stu-id="c788e-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```
