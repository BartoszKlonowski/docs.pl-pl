# <a name="compiler-error-cs1001"></a><span data-ttu-id="5cf06-102">Błąd kompilatora CS1001</span><span class="sxs-lookup"><span data-stu-id="5cf06-102">Compiler Error CS1001</span></span>
<span data-ttu-id="5cf06-103">Oczekiwano identyfikatora</span><span class="sxs-lookup"><span data-stu-id="5cf06-103">Identifier expected</span></span>  
  
 <span data-ttu-id="5cf06-104">Nie dostarczono identyfikatora.</span><span class="sxs-lookup"><span data-stu-id="5cf06-104">You did not supply an identifier.</span></span> <span data-ttu-id="5cf06-105">Identyfikator to nazwa klasy, struktury, przestrzeni nazw, metody, zmiennej i tak dalej.</span><span class="sxs-lookup"><span data-stu-id="5cf06-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="5cf06-106">Poniższy przykład deklaruje prostą klasę, ale nie przydaje klasy a nazwy:</span><span class="sxs-lookup"><span data-stu-id="5cf06-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="5cf06-107">Poniższy przykład generuje CS1001, ponieważ w przypadku deklarowania wyliczenia należy określić członków:</span><span class="sxs-lookup"><span data-stu-id="5cf06-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="5cf06-108">Nazwy parametrów są wymagane nawet wtedy, gdy kompilator nie korzysta z nich, na przykład w definicji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="5cf06-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="5cf06-109">Parametry te są wymagane, aby programiści korzystający z interfejsu wiedzieli coś o znaczeniu parametrów.</span><span class="sxs-lookup"><span data-stu-id="5cf06-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
