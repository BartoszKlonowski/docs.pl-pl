 <span data-ttu-id="e3fa4-104">Bez [](../keywords/unsafe.md) niebezpiecznego kontekstu operatora [sizeof](../../../csharp/language-reference/operators/sizeof.md) można używać tylko dla typów, których rozmiar jest stałą czasu kompilacji.</span><span class="sxs-lookup"><span data-stu-id="e3fa4-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../../../csharp/language-reference/operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="e3fa4-105">Jeśli ten błąd jest wyświetlany, użyj niebezpiecznego kontekstu.</span><span class="sxs-lookup"><span data-stu-id="e3fa4-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="e3fa4-106">Poniższy przykład generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="e3fa4-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
