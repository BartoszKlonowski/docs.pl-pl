# <a name="compiler-error-cs0233"></a><span data-ttu-id="bae01-102">Błąd kompilatora CS0233</span><span class="sxs-lookup"><span data-stu-id="bae01-102">Compiler Error CS0233</span></span>

## <a name="example"></a><span data-ttu-id="bae01-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="bae01-107">Example</span></span>  
 <span data-ttu-id="bae01-108">Poniższy przykład generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="bae01-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
