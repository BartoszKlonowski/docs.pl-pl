# <a name="compiler-error-cs0233"></a>Błąd kompilatora CS0233
'Identyfikator' nie ma wstępnie zdefiniowanego rozmiaru, w związku z tym sizeof można używać tylko w niebezpiecznym kontekście (Rozważ użycie metody System.Runtime.InteropServices.Marshal.SizeOf)  
  
Operatora [Sizeof](../../../csharp/language-reference/keywords/sizeof.md) można używać tylko typów, które są stałymi podczas kompilacji. W przypadku uzyskania tego błędu, upewnij się, że rozmiar identyfikatora można określić w czasie kompilacji. Jeżeli nie jest to możliwe to rozważ użycie <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> zamiast `sizeof`.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0233:  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
