# <a name="compiler-error-cs0233"></a>Błąd kompilatora CS0233
element "identifier" nie ma wstępnie zdefiniowanego rozmiaru, dlatego operatora sizeof można używać tylko w niebezpiecznym kontekście
  
 Bez [niebezpiecznego](../keywords/unsafe.md) kontekstu operatora [sizeof](../operators/sizeof.md) można używać tylko dla typów, których rozmiar jest stałą czasu kompilacji. Jeśli ten błąd jest wyświetlany, użyj niebezpiecznego kontekstu.
  
Poniższy przykład generuje CS0233:
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
