# <a name="compiler-error-cs0071"></a>Błąd kompilatora CS0071

Jawna implementacja interfejsu zdarzenia musi używać składni metody dostępu zdarzenia

W przypadku jawnej implementacji [zdarzenia](../keywords/event.md) , które zostało zadeklarowane w interfejsie, należy ręcznie dostarczyć `add` i `remove` metody dostępu do zdarzeń, które są zazwyczaj udostępniane przez kompilator. Kod dostępu może połączyć zdarzenie interfejsu z innym zdarzeniem w klasie (pokazanym w dalszej części tego tematu) lub z własnym typem delegata. Aby uzyskać więcej informacji, zobacz [jak zaimplementować zdarzenia interfejsu](../../programming-guide/events/how-to-implement-interface-events.md).
  
## <a name="example"></a>Przykład

 Poniższy przykład generuje CS0071.

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
