# <a name="compiler-error-cs0071"></a>Błąd kompilatora CS0071
Jawna implementacja interfejsu zdarzenia, należy użyć składni metody dostępu zdarzeń  
  
 Podczas implementowania jawnie [zdarzeń](../../../csharp/language-reference/keywords/event.md) , została zadeklarowana w interfejsie, należy użyć ręcznie podać `add` i `remove` metod dostępu zdarzeń, które zwykle są dostarczane przez kompilator. Kod dostępu można połączyć zdarzenia interfejsu do innego zdarzenia w klasie (przedstawiony w dalszej części tego tematu) lub na swój własny typ delegata. Aby uzyskać więcej informacji, zobacz [jak:  Zdarzenia implementowania interfejsu](../../../csharp/programming-guide/events/how-to-implement-interface-events.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0071.  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
