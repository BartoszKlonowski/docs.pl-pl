## <a name="example"></a><span data-ttu-id="0e221-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="0e221-106">Example</span></span>  
 <span data-ttu-id="0e221-107">Poniższy przykład generuje błąd CS0039.</span><span class="sxs-lookup"><span data-stu-id="0e221-107">The following example generates CS0039.</span></span>  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
