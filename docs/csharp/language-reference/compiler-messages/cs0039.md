# <a name="compiler-error-cs0039"></a>Błąd kompilatora CS0039
Typu "type1" nie można przekonwertować na typ "type2" za pomocą konwersji referencyjnej, pakującej, rozpakowującej, opakowującej ani konwersji typu null.
  
  
 Konwersja z operatorem [as](../../../csharp/language-reference/keywords/as.md) jest dozwolona przez dziedziczenie oraz konwersje referencyjne i pakujące.
 Aby uzyskać więcej informacji, zobacz [operatory konwersji](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje błąd CS0039.  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
