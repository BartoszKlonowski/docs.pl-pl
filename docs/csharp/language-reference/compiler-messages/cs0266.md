# <a name="compiler-error-cs0266"></a><span data-ttu-id="c9b6e-102">Błąd kompilatora CS0266</span><span class="sxs-lookup"><span data-stu-id="c9b6e-102">Compiler Error CS0266</span></span>
<span data-ttu-id="c9b6e-103">Nie można niejawnie przekonwertować typu 'type1' na 'Typ2'.</span><span class="sxs-lookup"><span data-stu-id="c9b6e-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="c9b6e-104">Istnieje jawna konwersja (czy brakuje rzutowania?)</span><span class="sxs-lookup"><span data-stu-id="c9b6e-104">An explicit conversion exists (are you missing a cast?)</span></span>  
  
 <span data-ttu-id="c9b6e-105">Ten błąd występuje, gdy kod podejmuje próbę przekonwertowania między dwoma typami, których nie można niejawnie przekonwertować, ale dla których jest dostępna jawna konwersja.</span><span class="sxs-lookup"><span data-stu-id="c9b6e-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="c9b6e-106">Aby uzyskać więcej informacji, zobacz [Rzutowanie i konwersje typów](../../../csharp/programming-guide/types/casting-and-type-conversions.md). </span><span class="sxs-lookup"><span data-stu-id="c9b6e-106">For more information, see [Casting and Type Conversions](../../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="c9b6e-107">Poniższy kod przedstawia przykłady, które generują CS0266.</span><span class="sxs-lookup"><span data-stu-id="c9b6e-107">The following code shows examples that generate CS0266.</span></span>  
  
```csharp  
// CS0266.cs  
class MyClass  
{  
    public static void Main()  
    {  
        // You cannot implicitly convert a double to an integer.  
        double d = 3.2;  
  
        // The following line causes compiler error CS0266.  
        int i1 = d;  
  
        // However, you can resolve the error by using an explicit conversion.  
        int i2 = (int)d;  
  
        // You cannot implicitly convert an object to a class type.  
        object obj = "MyString";  
  
        // The following assignment statement causes error CS0266.  
        MyClass myClass = obj;    
  
        // You can resolve the error by using an explicit conversion.  
        MyClass c = ( MyClass )obj;  
  
        // You cannot implicitly convert a base class object to a derived class type.  
        MyClass mc = new MyClass();  
        DerivedClass dc = new DerivedClass();  
  
        // The following line causes compiler error CS0266.  
        dc = mc;  
  
        // You can resolve the error by using an explicit conversion.  
        dc = (DerivedClass)mc;  
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
