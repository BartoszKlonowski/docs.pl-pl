# <a name="compiler-warning-level-1-cs1616"></a>Ostrzeżenie kompilatora (poziom 1) CS1616
Opcja "Option" przesłania atrybut "Attribute" określony w pliku źródłowym lub dodanym module  
  
 To ostrzeżenie występuje <xref:System.Reflection.AssemblyKeyFileAttribute> , jeśli atrybuty zestawu lub <xref:System.Reflection.AssemblyKeyNameAttribute> Znalezione w źródle powodują konflikt z opcją wiersza polecenia [/KeyFile](../compiler-options/keyfile-compiler-option.md) lub [/keycontainer](../compiler-options/keycontainer-compiler-option.md) lub nazwą pliku klucza lub kontenerem kluczy określonym we właściwościach projektu.  
  
 W poniższym przykładzie Załóżmy, że masz plik klucza o nazwie `cs1616.snk`. Generuj ten plik przy użyciu wiersza polecenia:  
  
```console  
sn -k CS1616.snk  
```  
  
 Poniższy przykład generuje CS1616:  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
