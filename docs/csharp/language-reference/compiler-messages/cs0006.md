# <a name="compiler-error-cs0006"></a>Błąd kompilatora CS0006

Nie można znaleźć pliku metadanych "dll_name"
  
 Program został skompilowany i jawnie przekazano nazwę pliku, który zawiera metadane; nie znaleziono jednak biblioteki. dll. Aby uzyskać więcej informacji, zobacz [-Reference (opcje kompilatora C#)](../compiler-options/reference-compiler-option.md).
