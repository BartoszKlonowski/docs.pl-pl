 <span data-ttu-id="c1efd-105">Ten błąd jest generowany podczas próby użycia inicjatora kolekcji z klasy, która nie ma publicznej niestatycznej metody `Add`.</span><span class="sxs-lookup"><span data-stu-id="c1efd-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="c1efd-106">Jeśli metoda `Add` jest niedostępna z powodu jej poziomu ochrony (`private`, `protected`, `internal`) i pojawia się błąd CS0122, to prawdopodobnie oznacza to, że jest zdefiniowana jako `static`.</span><span class="sxs-lookup"><span data-stu-id="c1efd-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
