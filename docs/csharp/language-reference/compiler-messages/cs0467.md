# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="6d73d-102">Ostrzeżenie kompilatora (poziom 2) CS0467</span><span class="sxs-lookup"><span data-stu-id="6d73d-102">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="6d73d-103">Niejednoznaczność między metodą "metoda" a metodą nieprzyokiego "metodą".</span><span class="sxs-lookup"><span data-stu-id="6d73d-103">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="6d73d-104">Korzystanie z grupy metod.</span><span class="sxs-lookup"><span data-stu-id="6d73d-104">Using method group.</span></span>  
  
 <span data-ttu-id="6d73d-105">Odziedziczone elementy członkowskie z różnych interfejsów, które mają ten sam podpis spowodować błąd niejednoznaczności.</span><span class="sxs-lookup"><span data-stu-id="6d73d-105">Inherited members from different interfaces that have the same signature  cause an ambiguity error.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6d73d-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="6d73d-106">Example</span></span>  
 <span data-ttu-id="6d73d-107">Poniższy przykład generuje CS0467.</span><span class="sxs-lookup"><span data-stu-id="6d73d-107">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You can also disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
