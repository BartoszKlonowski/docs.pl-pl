# <a name="compiler-error-cs8411"></a><span data-ttu-id="0dceb-102">Błąd kompilatora CS8411</span><span class="sxs-lookup"><span data-stu-id="0dceb-102">Compiler Error CS8411</span></span>

<span data-ttu-id="0dceb-103">Asynchroniczna instrukcja foreach nie może działać na zmiennych typu "Type", ponieważ typ "Type" nie zawiera odpowiedniej definicji wystąpienia publicznego dla elementu "GetAsyncEnumerator"</span><span class="sxs-lookup"><span data-stu-id="0dceb-103">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="0dceb-104">`await foreach` Instrukcja działa tylko na typach z definicją `GetAsyncEnumerator` , taką jak `IAsyncEnumerable<T>` .</span><span class="sxs-lookup"><span data-stu-id="0dceb-104">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="0dceb-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="0dceb-105">To correct this error</span></span>

<span data-ttu-id="0dceb-106">Zastąp element `await foreach` pytaniem `foreach`.</span><span class="sxs-lookup"><span data-stu-id="0dceb-106">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="0dceb-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="0dceb-107">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
```
