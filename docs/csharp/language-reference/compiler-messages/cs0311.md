## <a name="example"></a><span data-ttu-id="75ce7-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="75ce7-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }

    static void Main()
```  
