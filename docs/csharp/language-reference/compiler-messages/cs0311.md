# <a name="compiler-error-cs0311"></a><span data-ttu-id="33713-102">Błąd kompilatora CS0311</span><span class="sxs-lookup"><span data-stu-id="33713-102">Compiler Error CS0311</span></span>

<span data-ttu-id="33713-103">Typ "type1" nie może być używany jako parametr typu "t" w typie ogólnym lub metodzie "\<Name >".</span><span class="sxs-lookup"><span data-stu-id="33713-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="33713-104">Nie istnieje niejawna konwersja odwołań z "type1" na "type2".</span><span class="sxs-lookup"><span data-stu-id="33713-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="33713-105">Gdy ograniczenie jest stosowane do parametru typu ogólnego, niejawna tożsamość lub konwersja odwołania musi istnieć z argumentu konkretny dla typu ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="33713-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="33713-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="33713-106">To correct this error</span></span>  
  
1. <span data-ttu-id="33713-107">Zmień argument używany do tworzenia klasy.</span><span class="sxs-lookup"><span data-stu-id="33713-107">Change the argument you are using to create the class.</span></span>  
  
2. <span data-ttu-id="33713-108">Jeśli jesteś własnością klasy, możesz usunąć ograniczenie lub wykonać coś w celu włączenia niejawnego odwołania lub konwersji tożsamości.</span><span class="sxs-lookup"><span data-stu-id="33713-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="33713-109">Na przykład, można sprawić, aby drugi typ dziedziczył od pierwszego.</span><span class="sxs-lookup"><span data-stu-id="33713-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33713-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="33713-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="33713-111">Jeśli ten błąd występuje podczas próby użycia argumentu typu wartości, Zauważ, że niejawna konwersja numeryczna, na przykład z `short` do `int`, nie spełnia parametru typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="33713-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
