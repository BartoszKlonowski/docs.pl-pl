# <a name="compiler-error-cs0311"></a><span data-ttu-id="8c4bc-103">Błąd kompilatora CS0311</span><span class="sxs-lookup"><span data-stu-id="8c4bc-103">Compiler Error CS0311</span></span>

<span data-ttu-id="8c4bc-104">Typ "type1" nie może być używany jako parametr typu "t" w typie ogólnym lub metodzie " \<name> ".</span><span class="sxs-lookup"><span data-stu-id="8c4bc-104">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="8c4bc-105">Nie istnieje niejawna konwersja odwołań z "type1" na "type2".</span><span class="sxs-lookup"><span data-stu-id="8c4bc-105">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="8c4bc-106">Gdy ograniczenie jest stosowane do parametru typu ogólnego, niejawna tożsamość lub konwersja odwołania musi istnieć z argumentu konkretny dla typu ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="8c4bc-106">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8c4bc-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="8c4bc-107">To correct this error</span></span>  
  
1. <span data-ttu-id="8c4bc-108">Zmień argument używany do tworzenia klasy.</span><span class="sxs-lookup"><span data-stu-id="8c4bc-108">Change the argument you are using to create the class.</span></span>  
  
2. <span data-ttu-id="8c4bc-109">Jeśli jesteś własnością klasy, możesz usunąć ograniczenie lub wykonać coś w celu włączenia niejawnego odwołania lub konwersji tożsamości.</span><span class="sxs-lookup"><span data-stu-id="8c4bc-109">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="8c4bc-110">Na przykład, można sprawić, aby drugi typ dziedziczył od pierwszego.</span><span class="sxs-lookup"><span data-stu-id="8c4bc-110">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8c4bc-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="8c4bc-111">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="8c4bc-112">Jeśli ten błąd występuje podczas próby użycia argumentu typu wartości, Zauważ, że niejawna konwersja numeryczna, na przykład z `short` do `int` , nie spełnia parametru typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="8c4bc-112">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
