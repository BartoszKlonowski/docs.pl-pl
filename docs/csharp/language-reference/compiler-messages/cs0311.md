## <a name="example"></a><span data-ttu-id="966ea-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="966ea-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="966ea-111">Jeśli ten błąd występuje podczas próby użycia argumentów typu wartości, należy zauważyć, że niejawnej konwersji liczbowych, na przykład z `short` do `int`, nie spełnia parametr typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="966ea-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
