# <a name="compiler-error-cs0311"></a><span data-ttu-id="c831f-102">Błąd kompilatora CS0311</span><span class="sxs-lookup"><span data-stu-id="c831f-102">Compiler Error CS0311</span></span>

<span data-ttu-id="c831f-103">Nie można użyć typu "type1" jako parametru typu "T" w typie ogólnym lub metodzie "nazwa\<>".</span><span class="sxs-lookup"><span data-stu-id="c831f-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="c831f-104">Nie ma niejawnej konwersji odniesienia z "type1" na "type2".</span><span class="sxs-lookup"><span data-stu-id="c831f-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="c831f-105">Gdy ograniczenie jest stosowane do parametru typu ogólnego, niejawna tożsamość lub konwersja odwołania musi istnieć od konkretnego argumentu do typu ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="c831f-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c831f-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="c831f-106">To correct this error</span></span>  
  
1. <span data-ttu-id="c831f-107">Zmień argument, którego używasz do utworzenia klasy.</span><span class="sxs-lookup"><span data-stu-id="c831f-107">Change the argument you are using to create the class.</span></span>  
  
2. <span data-ttu-id="c831f-108">Jeśli jesteś właścicielem klasy, można usunąć ograniczenie lub zrobić coś, aby włączyć niejawne odwołanie lub konwersji tożsamości.</span><span class="sxs-lookup"><span data-stu-id="c831f-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="c831f-109">Na przykład można sprawić, że drugi typ dziedziczy ć od pierwszego.</span><span class="sxs-lookup"><span data-stu-id="c831f-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c831f-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="c831f-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="c831f-111">Jeśli ten błąd występuje podczas próby użycia argumentu typu wartości, należy `short` zauważyć, że niejawna konwersja liczbowa, na przykład z do `int`, nie spełnia parametru typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="c831f-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
