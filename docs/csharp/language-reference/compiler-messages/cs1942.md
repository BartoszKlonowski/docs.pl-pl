## <a name="to-correct-this-error"></a><span data-ttu-id="399c6-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="399c6-106">To correct this error</span></span>

1. <span data-ttu-id="399c6-107">Jeśli zmienna zakresu jest jawnie wpisana, upewnij się, że typ jest taki sam jak lub niejawnie konwertowany z, typ elementów w kolekcji, do której wykonuje iteracje.</span><span class="sxs-lookup"><span data-stu-id="399c6-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="399c6-108">Jeśli zmienna zakresu jest poprzedzona słowem kluczowym `var`, Usuń `var`.</span><span class="sxs-lookup"><span data-stu-id="399c6-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="399c6-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="399c6-109">Example</span></span>

<span data-ttu-id="399c6-110">Poniższy kod generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="399c6-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="399c6-111">CS1942 jest powiązany z CS1949, ponieważ użycie `var` z zmienną zakresu powoduje niepowodzenie podstawowej operacji `Cast<T>`, ponieważ `var` nie jest typem.</span><span class="sxs-lookup"><span data-stu-id="399c6-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
