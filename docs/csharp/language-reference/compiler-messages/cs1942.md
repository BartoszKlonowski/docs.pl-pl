<span data-ttu-id="1a48f-103">Typ wyrażenia w klauzuli „clause” jest nieprawidłowy.</span><span class="sxs-lookup"><span data-stu-id="1a48f-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="1a48f-104">Wnioskowanie o typie nie powiodło się w wywołaniu metody „method”.</span><span class="sxs-lookup"><span data-stu-id="1a48f-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="1a48f-105">Ten błąd zazwyczaj jest generowany wtedy, gdy zmienna zakresu ma nieprawidłowy typ jawny.</span><span class="sxs-lookup"><span data-stu-id="1a48f-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="1a48f-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="1a48f-106">To correct this error</span></span>

1. <span data-ttu-id="1a48f-107">Jeśli zmienna zakresu jest jawnie określona pod kątem typu, upewnij się, że jej typ jest taki sam lub niejawnie konwertowalny na typ kolekcji, w której dochodzi do iteracji.</span><span class="sxs-lookup"><span data-stu-id="1a48f-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="1a48f-108">Jeśli zmienna zakresu jest poprzedzona słowem kluczowym `var`, usuń je.`var`</span><span class="sxs-lookup"><span data-stu-id="1a48f-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1a48f-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="1a48f-109">Example</span></span>

<span data-ttu-id="1a48f-110">Poniższy kod generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="1a48f-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="1a48f-111">Błąd CS1942 jest powiązany z błędem CS1949, ponieważ użycie `var` dla określenia zakresu zmiennej powoduje, że wykonanie operacji `Cast<T>` kończy się niepowodzeniem ze względu na to, że `var` nie jest typem.</span><span class="sxs-lookup"><span data-stu-id="1a48f-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
