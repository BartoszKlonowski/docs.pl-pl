## <a name="to-correct-this-error"></a><span data-ttu-id="06e67-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="06e67-107">To correct this error</span></span>

1. <span data-ttu-id="06e67-108">Jeśli zmienna zakresu jest jawnie wpisana, upewnij się, że typ jest taki sam jak lub niejawnie konwertowany z, typ elementów w kolekcji, do której wykonuje iteracje.</span><span class="sxs-lookup"><span data-stu-id="06e67-108">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="06e67-109">Jeśli zmienna zakresu jest poprzedzona `var` słowem kluczowym, Usuń `var` .</span><span class="sxs-lookup"><span data-stu-id="06e67-109">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="06e67-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="06e67-110">Example</span></span>

<span data-ttu-id="06e67-111">Poniższy kod generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="06e67-111">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="06e67-112">CS1942 jest powiązany z CS1949, ponieważ użycie `var` z zmienną zakresu powoduje `Cast<T>` Niepowodzenie operacji, ponieważ `var` nie jest typem.</span><span class="sxs-lookup"><span data-stu-id="06e67-112">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
