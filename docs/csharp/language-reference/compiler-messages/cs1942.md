## <a name="example"></a><span data-ttu-id="fba6b-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="fba6b-109">Example</span></span>

```csharp

class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```  
  
 <span data-ttu-id="fba6b-111">CS1942 jest powiązany z CS1949, ponieważ użycie `var` zakresu zmiennej powoduje, że podstawowa `Cast<T>` operacji się niepowodzenie, ponieważ `var` nie jest typem.</span><span class="sxs-lookup"><span data-stu-id="fba6b-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
