# <a name="compiler-error-cs8403"></a>Błąd kompilatora CS8403

Metoda "Method" z blokiem iteratora musi mieć atrybut "Async", aby można było zwrócić element "{IAsyncEnumerable \<T> | IAsyncEnumerator \<T> } "

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

Oznacz metodę za pomocą `async` modyfikatora.
