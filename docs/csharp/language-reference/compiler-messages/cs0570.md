## <a name="example"></a><span data-ttu-id="03d6a-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="03d6a-106">Example</span></span>  
 <span data-ttu-id="03d6a-107">Poniższy program w C++ używa atrybutu RequiredAttributeAttribute, którego nie można używać w innych językach.</span><span class="sxs-lookup"><span data-stu-id="03d6a-107">The following C++ program uses an attribute, RequiredAttributeAttribute, which may not be consumed by other languages.</span></span>  
  
```cpp
// CPP0570.cpp  
// compile with: /clr /LD  
  
using namespace System;  
using namespace System::Runtime::CompilerServices;  
  
namespace CS0570_Server {  
   [RequiredAttributeAttribute(Int32::typeid)]    
   public ref struct Scenario1 {  
      int intVar;  
   };  
  
   public ref struct CS0570Class {  
      Scenario1 ^ sc1_field;  
  
      property virtual Scenario1 ^ sc1_prop {  
         Scenario1 ^ get() { return sc1_field; }  
      }  
  
      Scenario1 ^ sc1_method() { return sc1_field; }  
   };  
};  
```  
  
## <a name="example"></a><span data-ttu-id="03d6a-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="03d6a-108">Example</span></span>  
 <span data-ttu-id="03d6a-109">Poniższy przykład spowoduje wygenerowanie CS0570.</span><span class="sxs-lookup"><span data-stu-id="03d6a-109">The following sample generates CS0570.</span></span>

```csharp  
// CS0570.cs

using System;  
using CS0570_Server;

      r.sc1_method();   // CS0570  
   }  
}  
```
