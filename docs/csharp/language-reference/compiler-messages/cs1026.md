# <a name="compiler-error-cs1026"></a>Błąd kompilatora CS1026
) oczekiwane  
  
 Znaleziono niekompletne oświadczenie.  
  
 Częstą przyczyną tego błędu jest umieszczenie instrukcji, a nie wyrażenia, w wyrażeniu wbudowanym na stronie ASP.NET. Na przykład następujące jest niepoprawne:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Poniższe czynności są poprawne:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Jest on interpretowany w następujący sposób:  
  
```aspx-csharp  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Poniższy przykład generuje CS1026:  
  
```csharp  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
