# <a name="compiler-error-cs1026"></a>Błąd kompilatora CS1026
) oczekiwano  
  
 Znaleziono instrukcji niekompletne.  
  
 Typową przyczyną tego błędu jest umieszczenie instrukcję, zamiast wyrażenia w wyrażeniu wbudowanego strony ASP.NET. Na przykład następujące jest niepoprawna:  
  
```  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Następujące ustawienia są poprawne:  
  
```  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Jest interpretowana w następujący sposób:  
  
```  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Poniższy przykład generuje CS1026:  
  
```csharp  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
