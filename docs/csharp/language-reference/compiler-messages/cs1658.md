# <a name="compiler-warning-level-1-cs1658"></a>Ostrzeżenie kompilatora (poziom 1) CS1658
"tekst ostrzegawczy". Zobacz również błąd "kod błędu"  
  
 Kompilator emituje to ostrzeżenie, gdy zastąpi błąd ostrzeżeniem. Aby uzyskać informacje o tym problemie, zapoznaj się z wymienionym błędem. Aby znaleźć odpowiedni błąd z poziomu środowiska IDE programu Visual Studio, Użyj indeksu. Na przykład, jeśli tekst powyżej odczytuje "Zobacz także błąd" CS1037 "," Szukaj CS1037 w indeksie.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1658.  
  
```csharp  
// CS1658.cs  
// compile with: /doc:x.xml  
// CS1584 expected  
/// <summary>  
/// </summary>  
public class C  
{  
    /// <see cref="C.F(params object[])"/>  // CS1658  
    public static void M()  
    {  
    }  
  
    /// <summary>  
    /// </summary>  
    public void F(params object[] o)  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
