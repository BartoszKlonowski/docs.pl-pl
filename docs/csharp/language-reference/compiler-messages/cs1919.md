 <span data-ttu-id="c24aa-104">`new` Operator tworzy obiekty tylko na stercie zarządzanej.</span><span class="sxs-lookup"><span data-stu-id="c24aa-104">The `new` operator creates objects only on the managed heap.</span></span> <span data-ttu-id="c24aa-105">Jednak można utworzyć obiektów w pamięci niezarządzanej pośrednio przy użyciu możliwości współdziałanie języka aby wywołać metod natywnych, które zwracają wskaźników.</span><span class="sxs-lookup"><span data-stu-id="c24aa-105">However, you can create objects in unmanaged memory indirectly by using the interoperability capabilities of the language to call native methods that return pointers.</span></span>

## <a name="example"></a><span data-ttu-id="c24aa-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="c24aa-110">Example</span></span>
