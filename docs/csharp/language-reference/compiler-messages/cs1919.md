## <a name="example"></a><span data-ttu-id="aa424-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="aa424-110">Example</span></span>

 <span data-ttu-id="aa424-111">Poniższy przykład generuje błąd CS1919, ponieważ wskaźnik jest niezabezpieczony:</span><span class="sxs-lookup"><span data-stu-id="aa424-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp
{  
    public static int Main()

    }

```
