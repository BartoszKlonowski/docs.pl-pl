 <span data-ttu-id="0194f-105">`new`Operator tworzy obiekty tylko na zarządzanym stosie.</span><span class="sxs-lookup"><span data-stu-id="0194f-105">The `new` operator creates objects only on the managed heap.</span></span> <span data-ttu-id="0194f-106">Można jednak tworzyć obiekty w pamięci niezarządzanej pośrednio przy użyciu funkcji współdziałania języka do wywoływania metod natywnych, które zwracają wskaźniki.</span><span class="sxs-lookup"><span data-stu-id="0194f-106">However, you can create objects in unmanaged memory indirectly by using the interoperability capabilities of the language to call native methods that return pointers.</span></span>

## <a name="example"></a><span data-ttu-id="0194f-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="0194f-111">Example</span></span>

```csharp
unsafe public class C  
{  
    public static int Main()

    }

```
