# <a name="compiler-error-cs0702"></a>Błąd kompilatora CS0702
Ograniczenie nie może być klasą specjalną "identyfikator"  
  
 Następujące typy nie mogą służyć jako ograniczeń: `System.Object,``System.Array`, `System.Delegate`, `System.Enum`, lub `System.ValueType`.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0702:  
  
```csharp  
// CS0702.cs  
class C<T> where T : System.Array  // CS0702  
{  
}  
```
