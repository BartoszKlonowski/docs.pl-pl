# <a name="compiler-error-cs2032"></a>Błąd kompilatora CS2032
Znak "character" nie jest dozwolony w wierszu poleceń lub w plikach odpowiedzi  
  
 Pliki odpowiedzi i opcje wiersza poleceń dla csc.exe nie mogą zawierać znaków kontrolnych ASCII w zakresie 0-31 oraz znaku (`|`).  
  
 Błąd kompilatora CS2032 jest trudny do zaprezentowania w wierszu poleceń, ponieważ procesor wiersza poleceń i zintegrowane środowisko programistyczne (IDE) odfiltrowywuje znaki, które nie są prawidłowe. Poniższa procedura generuje błąd przy użyciu [pliku odpowiedzi](../../../csharp/language-reference/compiler-options/response-file-compiler-option.md).  
  
### <a name="to-generate-this-error"></a>Aby wygenerować ten błąd  
  
1.  W  folderze **Moje dokumenty**, utwórz plik tekstowy o nazwie CS2032.rsp, a następnie wprowadź w nim następujące opcje kompilatora:  
  
    ```csharp  
    /target:exe /out:cs|2032.exe cs2032.cs  
    ```  
  
2.  W folderze **Moje dokumenty**, utwórz plik tekstowy o nazwie cs2032.cs o dowolnej zawartości.  
  
3.  Otwórz menu **Start**. Znajdź i zaznacz [wiersz poleceń dla deweloperów programu Visual Studio](../../../framework/tools/developer-command-prompt-for-vs.md).
  
4.  Zmień bieżący katalog na `C:\Users\\<TwojaNazwaUzytkownika>\Documents`.  
  
5.  Uruchom następujące polecenie w wierszu poleceń: `csc @cs2032.rsp`  
  
6.  Komunikat o błędzie CS2032 pojawia się, ponieważ plik odpowiedzi, CS2032.rsp, zawiera znak kreski pionowej.
