# <a name="compiler-error-cs0433"></a>Błąd kompilatora CS0433
Typ TypeName1 istnieje zarówno TypeName2, jak i TypeName3  
  
 Dwa różne zestawy, do których odwołuje się do aplikacji zawierać tej samej przestrzeni nazw i typ, który powoduje niejednoznaczność.  
  
 Aby rozwiązać ten problem, użyj funkcji alias [/reference (C# opcje kompilatora)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) kompilatora opcję lub nie odwołują się do jednego z zestawów.  
 
Ten błąd może również wystąpić w formularzach sieci Web platformy ASP.NET, jeśli:

* `@ Page` Ma dyrektywy `CodeFile` atrybutu powinna być `CodeBehind` atrybutu.
* Kod jest umieszczany w *App_Code* folder, który nie powinien znajdować się w.
 
 Ten błąd może również wystąpić, jeśli kod znajduje się w folderze APP_CODE, które nie powinny być w folderze APP_CODE.
  
## <a name="example"></a>Przykład  
 Ten kod tworzy bibliotekę DLL za pierwszym razem typu niejednoznaczne.  
  
```csharp  
// CS0433_1.cs  
// compile with: /target:library  
namespace TypeBindConflicts   
{  
   public class AggPubImpAggPubImp {}  
}  
```  
  
## <a name="example"></a>Przykład  
 Ten kod tworzy bibliotekę DLL przy użyciu drugiej kopii typu niejednoznaczne.  
  
```csharp  
// CS0433_2.cs  
// compile with: /target:library  
namespace TypeBindConflicts   
{  
   public class AggPubImpAggPubImp {}  
}  
```  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0433.  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a>Przykład  
 Poniższy przykład pokazuje, jak można użyć funkcji alias **/reference** opcję kompilatora, aby rozwiązać ten problem CS0433.  
  
```csharp  
// CS0433_4.cs  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
