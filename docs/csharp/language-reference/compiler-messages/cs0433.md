## <a name="example"></a><span data-ttu-id="a9f14-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="a9f14-110">Example</span></span>  
 <span data-ttu-id="a9f14-111">Poniższy przykład generuje CS0433.</span><span class="sxs-lookup"><span data-stu-id="a9f14-111">The following example generates CS0433.</span></span>

{  
   public static void Main()

      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433
