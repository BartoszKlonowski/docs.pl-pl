# <a name="compiler-error-cs0843"></a>Błąd kompilatora CS0843
Pole zapasowe dla automatycznie implementowanej właściwości „name” musi być w pełni przypisane przed zwróceniem sterowania obiektowi wywołującemu. Należy wziąć pod uwagę podczas wywoływania konstruktora bez parametrów z inicjatora konstruktora.  
  
 Aby przypisać wartość do automatycznie implementowana właściwość z konstruktorem, należy najpierw wywołać konstruktora bez parametrów do utworzenia obiektu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj wywołanie konstruktora bez parametrów w inicjatora konstruktora, jak pokazano w poniższym przykładzie. Zwróć uwagę na użycie słowa kluczowego`: this()`. Aby dowiedzieć się więcej, zobacz informacje o słowie kluczowym [this](../../../csharp/language-reference/keywords/this.md).  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0843:  
  
```csharp  
// cs0843.cs  
struct S  
{  
    public int AIProp { get; set; }  
    public S(int i){} //CS0843  
    // Try the following lines instead.  
   // public S(int i) : this()  
   // {  
   //     AIProp = i;  
   // }  
}  
  
class Test  
{  
    static int Main()  
    {  
        return 1;  
    }  
}  
```
