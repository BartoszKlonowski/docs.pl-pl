# <a name="compiler-error-cs1708"></a><span data-ttu-id="0c04e-103">Błąd kompilatora CS1708</span><span class="sxs-lookup"><span data-stu-id="0c04e-103">Compiler Error CS1708</span></span>

<span data-ttu-id="0c04e-104">Bufory o ustalonym rozmiarze mogą być dostępne tylko za poorednictwem lokalnych lub pól</span><span class="sxs-lookup"><span data-stu-id="0c04e-104">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="0c04e-105">Nową funkcją w języku C# 2,0 jest możliwość zdefiniowania tablicy w wierszu wewnątrz `struct` .</span><span class="sxs-lookup"><span data-stu-id="0c04e-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="0c04e-106">Dostęp do takich tablic można uzyskać tylko za pośrednictwem lokalnych zmiennych lub pól i nie można się do nich odwoływać jako wartości pośrednie po lewej stronie wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="0c04e-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="0c04e-107">Ponadto nie można uzyskać dostępu do tablic przez pola, które są `static` lub `readonly` .</span><span class="sxs-lookup"><span data-stu-id="0c04e-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="0c04e-108">Aby rozwiązać ten problem, zdefiniuj zmienną tablicową i przypisz tablicę w wierszu do zmiennej.</span><span class="sxs-lookup"><span data-stu-id="0c04e-108">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="0c04e-109">Lub Usuń `static` `readonly` modyfikator or z pola reprezentującego tablicę w wierszu.</span><span class="sxs-lookup"><span data-stu-id="0c04e-109">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0c04e-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="0c04e-110">Example</span></span>  

 <span data-ttu-id="0c04e-111">Poniższy przykład generuje CS1708.</span><span class="sxs-lookup"><span data-stu-id="0c04e-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
