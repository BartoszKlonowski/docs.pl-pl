# <a name="compiler-error-cs1708"></a>Błąd kompilatora CS1708
Bufory o ustalonym rozmiarze można uzyskać tylko przez elementy lokalne lub pola  
  
 Nową funkcją w C# 2.0 jest możliwość zdefiniowania tablicy wewnątrz `struct`. Takie tablice są możliwe do odczytu tylko za pośrednictwem zmiennych lokalnych lub pól, i nie mogą być używane jako wartości pośrednie po lewej stronie wyrażeń. Ponadto nie można uzyskać dostępu do pól tablic, które są `static` lub `readonly`.  
  
 Aby rozwiązać ten problem, zdefiniuj zmienną tablicową i przypisz zmienną tablicy w wierszu. Lub usuń modyfikator `static` lub `readonly` reprezentujący tablice.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1708.  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
