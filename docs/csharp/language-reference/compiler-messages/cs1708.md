# <a name="compiler-error-cs1708"></a><span data-ttu-id="d5ea7-102">Błąd kompilatora CS1708</span><span class="sxs-lookup"><span data-stu-id="d5ea7-102">Compiler Error CS1708</span></span>
<span data-ttu-id="d5ea7-103">Bufory o ustalonym rozmiarze może zostać oceniony jedynie przez elementy lokalne lub pola</span><span class="sxs-lookup"><span data-stu-id="d5ea7-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="d5ea7-104">W C# 2.0 dostępna jest nowa funkcja umożliwiająca zdefiniowanie tablicy w deklaracji `struct`.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="d5ea7-105">Takie tablice są dostępne tylko za pośrednictwem zmiennych lub pól lokalnych. Nie mogą być one używane jako wartości pośrednie po lewej stronie wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="d5ea7-106">Dostępu do tablic nie można też uzyskać z pól o wartości `static` lub `readonly`.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="d5ea7-107">Aby rozwiązać ten problem, zdefiniuj zmienną tablicową i przypisz do niej wbudowaną tablicę.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="d5ea7-108">Możesz też usunąć modyfikator `static` lub `readonly` z pola reprezentującego tablicę wbudowaną.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d5ea7-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="d5ea7-109">Example</span></span>  
 <span data-ttu-id="d5ea7-110">Poniższy przykład spowoduje wygenerowanie CS1708.</span><span class="sxs-lookup"><span data-stu-id="d5ea7-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
