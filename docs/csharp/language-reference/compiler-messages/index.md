<span data-ttu-id="866d4-103">Błędy kompilatora C# mają odpowiednie tematy, które wyjaśniają, dlaczego zostanie wygenerowany błąd i, w niektórych przypadkach, jak naprawić błąd.</span><span class="sxs-lookup"><span data-stu-id="866d4-103">Some C# compiler errors have corresponding topics that explain why the error is generated, and, in some cases, how to fix the error.</span></span> <span data-ttu-id="866d4-104">Należy użyć jednej z następujących czynności, aby zobaczyć, czy pomoc jest dostępna dla danego komunikatu o błędzie.</span><span class="sxs-lookup"><span data-stu-id="866d4-104">Use one of the following steps to see whether help is available for a particular error message.</span></span>  
  
- <span data-ttu-id="866d4-105">Znajdź numer błędu (na przykład CS0029) w [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie wyszukaj go w Microsoft Docs.</span><span class="sxs-lookup"><span data-stu-id="866d4-105">Find the error number (for example, CS0029) in the [Output Window](/visualstudio/ide/reference/output-window), and then search for it on Microsoft Docs.</span></span>  
  
- <span data-ttu-id="866d4-106">Wybierz numer błędu (na przykład CS0029) [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie naciśnij klawisz F1.</span><span class="sxs-lookup"><span data-stu-id="866d4-106">Choose the error number (for example, CS0029) in the [Output Window](/visualstudio/ide/reference/output-window), and then choose the F1 key.</span></span>  
  
- <span data-ttu-id="866d4-107">W indeksie, wprowadź numer błędu w **poszukaj** pole.</span><span class="sxs-lookup"><span data-stu-id="866d4-107">In the Index, enter the error number in the **Look for** box.</span></span>  
  
 <span data-ttu-id="866d4-108">Jeśli żaden z tych kroków prowadzi do informacji na temat błędu, przejdź na koniec na tej stronie, a następnie wysłać opinię, który zawiera numer lub tekst błędu.</span><span class="sxs-lookup"><span data-stu-id="866d4-108">If none of these steps leads to information about your error, go to the end of this page, and send feedback that includes the number or text of the error.</span></span>
