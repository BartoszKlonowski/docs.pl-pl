# <a name="c-compiler-errors"></a>Błędy kompilatora C#

Błędy kompilatora C# mają odpowiednie tematy, które wyjaśniają, dlaczego zostanie wygenerowany błąd i, w niektórych przypadkach, jak naprawić błąd. Należy użyć jednej z następujących czynności, aby zobaczyć, czy pomoc jest dostępna dla danego komunikatu o błędzie.  
  
- Znajdź numer błędu (na przykład CS0029) w [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie wyszukaj go w Microsoft Docs.  
  
- Wybierz numer błędu (na przykład CS0029) [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie naciśnij klawisz F1.  
  
- W indeksie, wprowadź numer błędu w **poszukaj** pole.  
  
 Jeśli żaden z tych kroków prowadzi do informacji na temat błędu, przejdź na koniec na tej stronie, a następnie wysłać opinię, który zawiera numer lub tekst błędu.  
  
 Aby uzyskać informacje o sposobie konfigurowania opcji błędów i ostrzeżeń w języku C#, zobacz [Stroka kompilacji, Projektant projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
