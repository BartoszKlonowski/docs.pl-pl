# <a name="c-compiler-errors"></a>Błędy kompilatora C#
Błędy kompilatora C# ma odpowiednich tematów, dlatego zostanie wygenerowany błąd, i, w niektórych przypadkach sposób naprawić błąd. Użyj jednej z poniższych czynności, aby sprawdzić, czy pomoc jest dostępna dla danego komunikatu o błędzie.  
  
-   Znajdź kod błędu (na przykład CS0029) w [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie wyszukaj go na Docs firmy Microsoft.  
  
-   Wybierz liczbę błędów (na przykład CS0029) w [okno danych wyjściowych](/visualstudio/ide/reference/output-window), a następnie wybierz klawisz F1.  
  
-   W indeksie, wprowadź numer błędu w **Wyszukaj** pole.  
  
 Jeśli żaden z tych kroków prowadzą do informacji na temat błędu, przejdź na końcu tej strony, a następnie Wyślij opinię, zawierającą numer lub tekst błędu.  
  
 Aby uzyskać informacje o sposobie konfigurowania opcje ostrzeżenia i błędu w języku C#, zobacz [strona kompilacji, Projektant projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
