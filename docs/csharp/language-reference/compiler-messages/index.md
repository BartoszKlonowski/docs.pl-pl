# <a name="c-compiler-errors"></a>Błędy kompilatora C#

Niektóre błędy kompilatora C# mają odpowiednie tematy, które wyjaśniają, dlaczego błąd jest generowany, a w niektórych przypadkach, jak naprawić błąd. Użyj jednej z następujących czynności, aby sprawdzić, czy pomoc jest dostępna dla określonego komunikatu o błędzie.  
  
- Znajdź numer błędu (na przykład CS0029) w [oknie wyjściowym](/visualstudio/ide/reference/output-window), a następnie wyszukaj go w programach Microsoft Docs.  
  
- Wybierz numer błędu (na przykład CS0029) w [oknie wyjście](/visualstudio/ide/reference/output-window), a następnie wybierz klawisz F1.  
  
- W indeksie wprowadź numer błędu w polu **Szukaj.**  
  
 Jeśli żaden z tych kroków nie prowadzi do uzyskania informacji o błędzie, przejdź do końca tej strony i wyślij opinię zawierającą liczbę lub tekst błędu.  
  
 Aby uzyskać informacje dotyczące konfigurowania opcji błędów i ostrzeżeń w języku C#, zobacz [Tworzenie strony, projektanta projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
