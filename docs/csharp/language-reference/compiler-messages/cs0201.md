## <a name="example"></a><span data-ttu-id="0f6f4-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="0f6f4-110">Example</span></span>

 <span data-ttu-id="0f6f4-111">Poniższy przykład generuje CS0201, ponieważ sprawdzono same nie jest instrukcją, mimo że jest ona sparametryzowane przez operację przyrostu.</span><span class="sxs-lookup"><span data-stu-id="0f6f4-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>  

```csharp

// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
