# <a name="compiler-error-cs1579"></a><span data-ttu-id="993cc-102">Błąd kompilatora CS1579</span><span class="sxs-lookup"><span data-stu-id="993cc-102">Compiler Error CS1579</span></span>
<span data-ttu-id="993cc-103">Instrukcja foreach nie może operować na zmiennych typu "type1", ponieważ 'Typ2' nie zawiera definicji publicznej 'Identyfikator'</span><span class="sxs-lookup"><span data-stu-id="993cc-103">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="993cc-104">Do iterowania po kolekcji za pomocą [foreach](../keywords/foreach-in.md) instrukcji, Kolekcja musi spełniać następujące wymagania:</span><span class="sxs-lookup"><span data-stu-id="993cc-104">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="993cc-105">Jego typ musi zawierać publiczny bez parametrów `GetEnumerator` metody, którego typem zwracanym jest klasy, struktury lub typ interfejsu.</span><span class="sxs-lookup"><span data-stu-id="993cc-105">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="993cc-106">Zwracany typ `GetEnumerator` metoda musi zawierać publiczny właściwość o nazwie `Current` i publiczny bez parametrów metody o nazwie `MoveNext` którego typem zwracanym jest <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="993cc-106">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="993cc-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="993cc-107">Example</span></span>

<span data-ttu-id="993cc-108">Poniższy przykład generuje CS1579, ponieważ `MyCollection` klasy nie zawiera publicznego `GetEnumerator` metody:</span><span class="sxs-lookup"><span data-stu-id="993cc-108">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection   
{  
   int[] items;  
   public MyCollection()   
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()   
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator   
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)   
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()   
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()   
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
