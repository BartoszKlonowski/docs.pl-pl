# <a name="compiler-error-cs1579"></a><span data-ttu-id="3d822-102">Błąd kompilatora CS1579</span><span class="sxs-lookup"><span data-stu-id="3d822-102">Compiler Error CS1579</span></span>
<span data-ttu-id="3d822-103">instrukcja foreach nie może działać na zmiennych typu "type1", ponieważ "type2" nie zawiera publicznej definicji "identyfikatora"</span><span class="sxs-lookup"><span data-stu-id="3d822-103">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="3d822-104">Aby iterate za pomocą kolekcji przy użyciu [foreach](../keywords/foreach-in.md) instrukcji, kolekcja musi spełniać następujące wymagania:</span><span class="sxs-lookup"><span data-stu-id="3d822-104">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="3d822-105">Jego typ musi zawierać `GetEnumerator` publiczną metodę bezparametrową, której typem zwracanym jest typ klasy, struktury lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="3d822-105">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="3d822-106">Zwracany typ `GetEnumerator` metody musi zawierać właściwość `Current` publiczną o nazwie `MoveNext` i publiczną metodę bezparametrową o nazwie, której typem zwracanym jest <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="3d822-106">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="3d822-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="3d822-107">Example</span></span>

<span data-ttu-id="3d822-108">Poniższy przykład generuje CS1579, `MyCollection` ponieważ klasa nie `GetEnumerator` zawiera metody publicznej:</span><span class="sxs-lookup"><span data-stu-id="3d822-108">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection
{  
   int[] items;  
   public MyCollection()
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
