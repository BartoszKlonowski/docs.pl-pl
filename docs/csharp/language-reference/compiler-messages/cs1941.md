# <a name="compiler-error-cs1941"></a>Błąd kompilatora CS1941
Typ jednego z wyrażeń w klauzuli "clause" jest nieprawidłowy. Wnioskowanie typu w wywołaniu metody "method" nie powiodło się.  
  
 Wnioskowanie typów opiera sie na typach elementów w źródle danych zapytania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Jeśli przyczyna błedu nie jest widoczna od razu, należy dokładnie zbadać zapytanie i typ wyniku każdej klauzuli od źródła danych do punktu, w któym występuje błąd.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1941, ponieważ operator `equals` próbuje porównać `int` ze `string`.  
  
```csharp  
// cs1941.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        var nums = new[] { 1, 2, 3, 4, 5, 6 };  
        var words = new string[] { "lake", "mountain", "sky" };  
        IEnumerable e = from n in nums  
                        join w in words on n equals w // CS1941  
                        select w;  
        return 0;  
    }  
}  
```  
  
 Metoda, w której zawodzi wnioskowanie typów jest tłumaczona w czasie kompilacji.
