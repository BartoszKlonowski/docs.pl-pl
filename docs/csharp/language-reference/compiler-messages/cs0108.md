# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="a8886-102">Ostrzeżenie kompilatora (poziom 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="a8886-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="a8886-103">"Członek1" ukrywa dziedziczoną składową 'członek2'.</span><span class="sxs-lookup"><span data-stu-id="a8886-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="a8886-104">Użyj słowa kluczowego new, jeżeli ukrywanie było zamierzone.</span><span class="sxs-lookup"><span data-stu-id="a8886-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="a8886-105">Zmienna została zadeklarowana o takiej samej nazwie co zmienna w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="a8886-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="a8886-106">Jednak [nowe](../../../csharp/language-reference/keywords/new.md) nie użyto słowa kluczowego.</span><span class="sxs-lookup"><span data-stu-id="a8886-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="a8886-107">To ostrzeżenie informuje, że należy używać **nowe**; ta zmienna została zgłoszona tak, jakby **nowe** została wykorzystana w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="a8886-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="a8886-108">Poniższy przykład spowoduje wygenerowanie CS0108:</span><span class="sxs-lookup"><span data-stu-id="a8886-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
