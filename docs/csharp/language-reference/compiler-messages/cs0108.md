# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="2dad6-102">Ostrzeżenie kompilatora (poziom 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="2dad6-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="2dad6-103">"member1" ukrywa odziedziczonego członka "member2".</span><span class="sxs-lookup"><span data-stu-id="2dad6-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="2dad6-104">Użyj nowego słowa kluczowego, jeśli ukrywanie było przeznaczone.</span><span class="sxs-lookup"><span data-stu-id="2dad6-104">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="2dad6-105">Zmienna została zadeklarowana o tej samej nazwie co zmienna w klasie podstawowej.</span><span class="sxs-lookup"><span data-stu-id="2dad6-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="2dad6-106">Jednak [nowe](../keywords/new-modifier.md) słowo kluczowe nie zostało użyte.</span><span class="sxs-lookup"><span data-stu-id="2dad6-106">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="2dad6-107">To ostrzeżenie informuje, że należy używać **nowych**; zmienna jest zadeklarowana tak, jakby w deklaracji użyto **nowej.**</span><span class="sxs-lookup"><span data-stu-id="2dad6-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="2dad6-108">Poniższy przykład generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="2dad6-108">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
