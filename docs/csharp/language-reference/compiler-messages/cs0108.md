# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="e63d6-102">Ostrzeżenie kompilatora (poziom 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="e63d6-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="e63d6-103">element "member1" ukrywa dziedziczoną składową "member2".</span><span class="sxs-lookup"><span data-stu-id="e63d6-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="e63d6-104">Użyj słowa kluczowego new, jeśli ukrywanie było zamierzone.</span><span class="sxs-lookup"><span data-stu-id="e63d6-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="e63d6-105">Zmienna została zadeklarowana z taką samą nazwą jak zmienna w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="e63d6-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="e63d6-106">Jednak słowo kluczowe [New](../keywords/new-modifier.md) nie zostało użyte.</span><span class="sxs-lookup"><span data-stu-id="e63d6-106">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="e63d6-107">To ostrzeżenie informuje o konieczności użycia **nowego**; Zmienna jest zadeklarowana tak, jakby była używana w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="e63d6-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="e63d6-108">Poniższy przykład generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="e63d6-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
