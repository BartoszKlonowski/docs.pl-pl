# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="ffe2c-102">Ostrzeżenie kompilatora (poziom 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="ffe2c-102">Compiler Warning (level 2) CS0108</span></span>
<span data-ttu-id="ffe2c-103">"Członek1" ukrywa dziedziczony element członkowski "member2".</span><span class="sxs-lookup"><span data-stu-id="ffe2c-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="ffe2c-104">Użyj kluczowego new, jeżeli ukrywanie zamierzony sposób.</span><span class="sxs-lookup"><span data-stu-id="ffe2c-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="ffe2c-105">Zmienna została zadeklarowana z tej samej nazwy co zmienna w klasie podstawowej.</span><span class="sxs-lookup"><span data-stu-id="ffe2c-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="ffe2c-106">Jednak [nowe](../../../csharp/language-reference/keywords/new.md) nie użyto słowa kluczowego.</span><span class="sxs-lookup"><span data-stu-id="ffe2c-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="ffe2c-107">To ostrzeżenie informuje, że należy używać **nowe**; zmienna została zadeklarowana tak, jakby **nowe** zostało użyte w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="ffe2c-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="ffe2c-108">Poniższy przykład generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="ffe2c-108">The following sample generates CS0108:</span></span>  
  
```csharp  
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
