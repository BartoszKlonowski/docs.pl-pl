# <a name="compiler-error-cs1943"></a><span data-ttu-id="2b250-102">Błąd kompilatora CS1943</span><span class="sxs-lookup"><span data-stu-id="2b250-102">Compiler Error CS1943</span></span>
<span data-ttu-id="2b250-103">Wyrażenie typu „typ” jest niedozwolone w kolejnej klauzuli from w wyrażeniu zapytania z typem źródła „typ”.</span><span class="sxs-lookup"><span data-stu-id="2b250-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="2b250-104">Wnioskowanie typu w wywołaniu elementu „metoda” nie powiodło się.</span><span class="sxs-lookup"><span data-stu-id="2b250-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="2b250-105">Wszystkie zmienne zakresu musi reprezentować typami odpytywalnymi.</span><span class="sxs-lookup"><span data-stu-id="2b250-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2b250-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="2b250-106">To correct this error</span></span>  
  
1. <span data-ttu-id="2b250-107">Upewnij się, że typ jest typem obsługującym zapytania i implementującym interfejs `IEnumerable`, `IEnumerable<T>`lub interfejs pochodny albo jest innym dowolnym typem ze zdefiniowanym wzorcem zapytania dla niego.</span><span class="sxs-lookup"><span data-stu-id="2b250-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="2b250-108">Jeśli typ jest nieogólnym interfejsem `IEnumerable`, podaj jawny typ zmiennej zakresu.</span><span class="sxs-lookup"><span data-stu-id="2b250-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2b250-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="2b250-109">Example</span></span>  
 <span data-ttu-id="2b250-110">Poniższy kod generuje CS1943:</span><span class="sxs-lookup"><span data-stu-id="2b250-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
