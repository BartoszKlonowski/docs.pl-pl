# <a name="compiler-error-cs1943"></a><span data-ttu-id="97afa-103">Błąd kompilatora CS1943</span><span class="sxs-lookup"><span data-stu-id="97afa-103">Compiler Error CS1943</span></span>

<span data-ttu-id="97afa-104">Wyrażenie typu "Type" jest niedozwolone w kolejnej klauzuli FROM w wyrażeniu zapytania z typem źródła "Type".</span><span class="sxs-lookup"><span data-stu-id="97afa-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="97afa-105">Wnioskowanie typu nie powiodło się w wywołaniu metody.</span><span class="sxs-lookup"><span data-stu-id="97afa-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="97afa-106">Wszystkie zmienne zakresu muszą reprezentować typy queryable.</span><span class="sxs-lookup"><span data-stu-id="97afa-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="97afa-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="97afa-107">To correct this error</span></span>  
  
1. <span data-ttu-id="97afa-108">Upewnij się, że typ jest typem queryable, który implementuje `IEnumerable` , `IEnumerable<T>` lub pochodny interfejs lub inny typ, dla którego zdefiniowano dla niego wzorzec zapytania.</span><span class="sxs-lookup"><span data-stu-id="97afa-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="97afa-109">Jeśli typ jest nieogólny `IEnumerable` , podaj jawny typ zmiennej zakresu.</span><span class="sxs-lookup"><span data-stu-id="97afa-109">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="97afa-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="97afa-110">Example</span></span>  

 <span data-ttu-id="97afa-111">Poniższy kod generuje CS1943:</span><span class="sxs-lookup"><span data-stu-id="97afa-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
