# <a name="compiler-error-cs1943"></a>Błąd kompilatora CS1943
Wyrażenie typu "type" nie może zostać poddane działaniu klauzuli from w wyrażeniu zapytania o typie źródła "type". Wnioskowanie typu w wywołaniu metody "method" nie powiodło się.  
  
 Wszystkie zmienne zakresu musi reprezentować typy zapytań.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że typ może być wykorzystany w zapytaniach, czyli implementuje interfejs `IEnumerable`, `IEnumerable<T>` lub jego interfejs pochodny, który umożliwia poddanie zapytaniu.  
  
2.  Jeśli typ jest inny niż generyczny `IEnumerable`, podaj jawnie zmienną zakresu.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1943:  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
