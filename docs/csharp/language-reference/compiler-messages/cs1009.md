# <a name="compiler-error-cs1009"></a><span data-ttu-id="c4d65-102">Błąd kompilatora CS1009</span><span class="sxs-lookup"><span data-stu-id="c4d65-102">Compiler Error CS1009</span></span>
<span data-ttu-id="c4d65-103">Nierozpoznana sekwencja ucieczki</span><span class="sxs-lookup"><span data-stu-id="c4d65-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="c4d65-104">Nieoczekiwany znak następujący po ukośniku odwrotnym (\\) w [ciągu znaków](../keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="c4d65-104">An unexpected character follows a backslash (\\) in a [string](../keywords/string.md).</span></span> <span data-ttu-id="c4d65-105">Kompilator oczekuje jednego z prawidłowych znaków ucieczki.</span><span class="sxs-lookup"><span data-stu-id="c4d65-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="c4d65-106">Aby uzyskać więcej informacji, zobacz [Znaki specjalne](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="c4d65-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="c4d65-107">Poniższy przykład generuje CS1009.</span><span class="sxs-lookup"><span data-stu-id="c4d65-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="c4d65-108">Typową przyczyną tego błędu jest użycie ukośnika w ścieżce pliku, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="c4d65-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="c4d65-109">Aby rozwiązać problem, należy wpisać "\\\\" lub poprzedzić literał ciągu znakiem @-quoted — jak w przykładzie poniżej.</span><span class="sxs-lookup"><span data-stu-id="c4d65-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
