# <a name="compiler-error-cs0304"></a><span data-ttu-id="c700f-102">Błąd kompilatora CS0304</span><span class="sxs-lookup"><span data-stu-id="c700f-102">Compiler Error CS0304</span></span>

<span data-ttu-id="c700f-103">Nie można utworzyć wystąpienia typu zmiennej "Type", ponieważ nie ma ono ograniczenia New ()</span><span class="sxs-lookup"><span data-stu-id="c700f-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="c700f-104">Podczas implementowania `new` klasy generycznej i chcesz użyć słowa kluczowego w celu utworzenia nowego wystąpienia dowolnego typu, który jest dostarczany dla parametru `T`typu, należy zastosować [ograniczenie New ()](../keywords/new-constraint.md) do `T` w deklaracji klasy, jako pokazano w następującym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="c700f-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="c700f-105">Ograniczenie wymusza bezpieczeństwo typu przez zagwarantowanie, że każdy konkretny typ, który jest dostarczany `T` dla ma domyślny konstruktor bez parametrów. `new()`</span><span class="sxs-lookup"><span data-stu-id="c700f-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="c700f-106">CS0304 występuje, `new` Jeśli próbujesz użyć operatora w treści klasy w celu utworzenia wystąpienia parametru `T` typu, `new()` gdy `T` nie określono ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="c700f-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="c700f-107">Po stronie klienta, jeśli kod próbuje utworzyć wystąpienie klasy generycznej z typem, który nie ma konstruktora bez parametrów, ten kod spowoduje wygenerowanie [błędu kompilatora CS0310](./cs0310.md).</span><span class="sxs-lookup"><span data-stu-id="c700f-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="c700f-108">Poniższy przykład generuje CS0304.</span><span class="sxs-lookup"><span data-stu-id="c700f-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="c700f-109">`new` Operator nie jest również dozwolony w metodach klasy.</span><span class="sxs-lookup"><span data-stu-id="c700f-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="c700f-110">Aby uniknąć tego błędu, zadeklaruj klasę przy użyciu `new()` ograniczenia, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="c700f-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
