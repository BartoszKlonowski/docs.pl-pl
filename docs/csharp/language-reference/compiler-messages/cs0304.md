# <a name="compiler-error-cs0304"></a><span data-ttu-id="31a62-102">Błąd kompilatora CS0304</span><span class="sxs-lookup"><span data-stu-id="31a62-102">Compiler Error CS0304</span></span>

<span data-ttu-id="31a62-103">Nie można utworzyć wystąpienia typu zmiennej "type", ponieważ nie ma ono ograniczenia new()</span><span class="sxs-lookup"><span data-stu-id="31a62-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="31a62-104">Podczas implementowania klasy ogólnej i chcesz `new` użyć słowa kluczowego, aby utworzyć nowe wystąpienie `T`dowolnego typu, który jest dostarczany dla `T` parametru typu, należy zastosować [ograniczenie new()](../keywords/new-constraint.md) do deklaracji klasy, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="31a62-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="31a62-105">Ograniczenie `new()` wymusza bezpieczeństwo typu, gwarantując, że każdy typ betonu, który jest dostarczany dla `T` ma domyślny, konstruktor bez parametrów.</span><span class="sxs-lookup"><span data-stu-id="31a62-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="31a62-106">CS0304 występuje, jeśli spróbujesz `new` użyć operatora w treści klasy, `T` `T` aby utworzyć `new()` wystąpienie parametru typu, gdy nie określa ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="31a62-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="31a62-107">Po stronie klienta, jeśli kod próbuje utworzyć wystąpienie klasy ogólnej z typem, który nie ma konstruktora bez parametrów, kod ten wygeneruje [błąd kompilatora CS0310](./cs0310.md).</span><span class="sxs-lookup"><span data-stu-id="31a62-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="31a62-108">Poniższy przykład generuje CS0304.</span><span class="sxs-lookup"><span data-stu-id="31a62-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="31a62-109">Operator `new` również nie jest dozwolone w metodach klasy.</span><span class="sxs-lookup"><span data-stu-id="31a62-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="31a62-110">Aby uniknąć błędu, zadeklarować `new()` klasę przy użyciu ograniczenia, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="31a62-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
