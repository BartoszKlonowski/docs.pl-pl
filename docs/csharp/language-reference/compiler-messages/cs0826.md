 <span data-ttu-id="f292d-104">Elementy tablicy wszystkie muszą być tego samego typu, albo niejawnie możliwe do przekonwertowania na ten sam typ, zgodnie z regułami wnioskowania typu używanego przez kompilator.</span><span class="sxs-lookup"><span data-stu-id="f292d-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="f292d-105">Najlepszy typ musi być jednego z typów w wyrażeniu tablicy.</span><span class="sxs-lookup"><span data-stu-id="f292d-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="f292d-106">Elementy będą nie można przekonwertować na nowy typ takich jak `object`.</span><span class="sxs-lookup"><span data-stu-id="f292d-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="f292d-107">Niejawnie wpisane tablicy kompilator musi rozpoznać tablicy typu na podstawie typu elementów przypisane do niej.</span><span class="sxs-lookup"><span data-stu-id="f292d-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="f292d-112">Przykład</span><span class="sxs-lookup"><span data-stu-id="f292d-112">Example</span></span>  
 <span data-ttu-id="f292d-113">Poniższy kod generuje CS0826, ponieważ są elementy tablicy nie te same typ i przez kompilator typu wnioskowania logiki brakuje jednego typu najlepsze:</span><span class="sxs-lookup"><span data-stu-id="f292d-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
