# <a name="compiler-error-cs0826"></a>Błąd kompilatora CS0826
Nie odnaleziono najlepszego typu dla niejawnie określonej.  
  
 Wszystkie elementy tablicy muszą być tego samego typu, albo muszą być niejawnie przekonwertowane na ten sam typ, zgodnie z regułami wnioskowania typu; używanego przez kompilator. Najlepszy typ musi być jednym z typów występujących w wyrażeniu tablicy. Elementy nie będą mogły zostać przekonwertować na nowy typ taki jak `object`. Dla tablicy określonej niejawnie, kompilator musi rozpoznać typ tablicy na podstawie typów elementów do niej przypisanych.
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nadaj tablicy jawny typ.  
  
-   Wszystkie elementy tablicy muszą być tego samego typu.  
  
-   Podaj jawne rzutowanie dla tych elementów, które mogą być przyczyną problemu.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0826, ponieważ elementy tablicy nie są tego samego typu i logika kompilatora odnotowuje brak jednego najlepszego typu: 
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
