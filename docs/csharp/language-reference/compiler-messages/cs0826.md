 <span data-ttu-id="04273-104">Elementy tablicy muszą być tego samego typu lub niejawnie konwertowalne na ten sam typ zgodnie z regułami wnioskowania o typie używanymi przez kompilator.</span><span class="sxs-lookup"><span data-stu-id="04273-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="04273-105">Najlepszy typ musi być jednym z typów obecnych w wyrażeniu tablicy.</span><span class="sxs-lookup"><span data-stu-id="04273-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="04273-106">Elementy nie zostaną przekonwertowane na `object`nowy typ, taki jak .</span><span class="sxs-lookup"><span data-stu-id="04273-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="04273-107">W przypadku tablicy wpisanej niejawnie kompilator musi wywnioskować typ tablicy na podstawie typu przypisanych do niej elementów.</span><span class="sxs-lookup"><span data-stu-id="04273-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="04273-112">Przykład</span><span class="sxs-lookup"><span data-stu-id="04273-112">Example</span></span>  
 <span data-ttu-id="04273-113">Poniższy kod generuje CS0826, ponieważ elementy tablicy nie są wszystkie tego samego typu, a logika wnioskowania typu kompilatora nie znajduje jeden najlepszy typ:</span><span class="sxs-lookup"><span data-stu-id="04273-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
