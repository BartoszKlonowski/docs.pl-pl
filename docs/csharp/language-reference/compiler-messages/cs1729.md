 <span data-ttu-id="e1c07-104">Ten błąd występuje, gdy bezpośrednio lub pośrednio wywołać konstruktora klasy, ale kompilator nie może znaleźć żadnych konstruktorów z taką samą liczbę parametrów.</span><span class="sxs-lookup"><span data-stu-id="e1c07-104">This error occurs when you either directly or indirectly invoke the constructor of a class but the compiler cannot find any constructors with the same number of parameters.</span></span> <span data-ttu-id="e1c07-105">W poniższym `test` przykładzie klasa nie ma konstruktorów, które przyjmują wszelkie argumenty.</span><span class="sxs-lookup"><span data-stu-id="e1c07-105">In the following example, the `test` class has no constructors that take any arguments.</span></span> <span data-ttu-id="e1c07-106">W związku z tym ma tylko konstruktora bez parametrów, który przyjmuje zero argumentów.</span><span class="sxs-lookup"><span data-stu-id="e1c07-106">It therefore has only a parameterless constructor that takes zero arguments.</span></span> <span data-ttu-id="e1c07-107">Ponieważ w drugim wierszu, w którym jest generowany błąd, klasa pochodna deklaruje żadnych konstruktorów własnych, kompilator zapewnia konstruktora bezparametrów.</span><span class="sxs-lookup"><span data-stu-id="e1c07-107">Because in the second line in which the error is generated, the derived class declares no constructors of its own, the compiler provides a parameterless constructor.</span></span> <span data-ttu-id="e1c07-108">Ten konstruktor wywołuje konstruktora bezparametrów w klasie podstawowej.</span><span class="sxs-lookup"><span data-stu-id="e1c07-108">That constructor invokes a parameterless constructor in the base class.</span></span> <span data-ttu-id="e1c07-109">Ponieważ klasa podstawowa nie ma takiego konstruktora, cs1729 jest generowany.</span><span class="sxs-lookup"><span data-stu-id="e1c07-109">Because the base class has no such constructor, CS1729 is generated.</span></span>

## <a name="example"></a><span data-ttu-id="e1c07-114">Przykład</span><span class="sxs-lookup"><span data-stu-id="e1c07-114">Example</span></span>  
 <span data-ttu-id="e1c07-115">Poniższy przykład generuje CS1729:</span><span class="sxs-lookup"><span data-stu-id="e1c07-115">The following example generates CS1729:</span></span>  
  
```csharp  
// cs1729.cs  
class Test  
{  
    static int Main()  
    {  
        // Class Test has only a parameterless constructor, which takes no arguments.  
        Test test1 = new Test(2); // CS1729  
        // The following line resolves the error.  
        Test test2 = new Test();  
  
        // Class Parent has only one constructor, which takes two int parameters.  
        Parent exampleParent1 = new Parent(10); // CS1729  
        // The following line resolves the error.  
        Parent exampleParent2 = new Parent(10, 1);  
  
        return 1;  
    }  
}  
  
public class Parent  
{  
    // The only constructor for this class has two parameters.  
    public Parent(int i, int j) { }  
}  
  
// The following declaration causes a compiler error because class Parent  
// does not have a constructor that takes no arguments. The declaration of  
// class Child2 shows how to resolve this error.  
public class Child : Parent { } // CS1729  
  
public class Child2 : Parent  
{  
    // The constructor for Child2 has only one parameter. To access the
    // constructor in Parent, and prevent this compiler error, you must provide
    // a value for the second parameter of Parent. The following example provides 0.  
    public Child2(int k)  
        : base(k, 0)  
    {  
        // Add the body of the constructor here.  
    }  
}  
```
