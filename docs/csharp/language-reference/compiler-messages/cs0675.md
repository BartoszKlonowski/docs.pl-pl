# <a name="compiler-warning-level-3-cs0675"></a><span data-ttu-id="91de7-103">Ostrzeżenie kompilatora (poziom 3) CS0675</span><span class="sxs-lookup"><span data-stu-id="91de7-103">Compiler Warning (level 3) CS0675</span></span>

<span data-ttu-id="91de7-104">Użyto operatora bitowego or w argumencie operacji z rozszerzonym znakiem. najpierw Rozważ rzutowanie na mniejszy typ bez znaku</span><span class="sxs-lookup"><span data-stu-id="91de7-104">Bitwise-or operator used on a sign-extended operand; consider casting to a smaller unsigned type first</span></span>  
  
 <span data-ttu-id="91de7-105">Kompilator niejawnie poszerzyć i podpisywać zmiennej, a następnie użył wartości otrzymanej w operacji bitowej.</span><span class="sxs-lookup"><span data-stu-id="91de7-105">The compiler implicitly widened and sign-extended a variable, and then used the resulting value in a bitwise OR operation.</span></span> <span data-ttu-id="91de7-106">Może to spowodować nieoczekiwane zachowanie.</span><span class="sxs-lookup"><span data-stu-id="91de7-106">This can result in unexpected behavior.</span></span>  
  
 <span data-ttu-id="91de7-107">Poniższy przykład generuje CS0675:</span><span class="sxs-lookup"><span data-stu-id="91de7-107">The following sample generates CS0675:</span></span>  
  
```csharp  
// CS0675.cs  
// compile with: /W:3  
using System;  
  
public class sign  
{  
   public static void Main()  
   {  
      int hi = 1;  
      int lo = 1;  
      long value = (((long)hi) << 32) | lo;              // CS0675  
      // try the following line instead  
      // long value = (((long)hi) << 32) | ((uint)lo);   // correct  
   }  
}  
```
