# <a name="compiler-error-cs0703"></a><span data-ttu-id="fc67a-102">Błąd kompilatora CS0703</span><span class="sxs-lookup"><span data-stu-id="fc67a-102">Compiler Error CS0703</span></span>
<span data-ttu-id="fc67a-103">Niespójna dostępność: typ ograniczenia "identyfikator" jest mniej dostępny niż "identyfikator"</span><span class="sxs-lookup"><span data-stu-id="fc67a-103">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="fc67a-104">Ograniczenie nie może wymusić parametr ogólny być mniej dostępne niż klasy ogólnej, sama.</span><span class="sxs-lookup"><span data-stu-id="fc67a-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="fc67a-105">W poniższym przykładzie, podczas\<gdy klasa ogólna C T> jest zadeklarowana jako publiczna, ograniczenie próbuje wymusić T do zaimplementowania interfejsu wewnętrznego.</span><span class="sxs-lookup"><span data-stu-id="fc67a-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="fc67a-106">Nawet jeśli było to dozwolone, tylko klienci z dostępem wewnętrznym będą mogli utworzyć parametr dla klasy, więc w efekcie klasa może być używana tylko przez klientów z dostępem wewnętrznym.</span><span class="sxs-lookup"><span data-stu-id="fc67a-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="fc67a-107">Aby pozbyć się tego błędu, upewnij się, że poziom dostępu klasy ogólnej nie jest mniej restrykcyjne niż wszelkie klasy lub interfejsy, które pojawiają się w granicach.</span><span class="sxs-lookup"><span data-stu-id="fc67a-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="fc67a-108">Poniższy przykład generuje CS0703:</span><span class="sxs-lookup"><span data-stu-id="fc67a-108">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
