# <a name="compiler-error-cs0703"></a><span data-ttu-id="0a1a6-103">Błąd kompilatora CS0703</span><span class="sxs-lookup"><span data-stu-id="0a1a6-103">Compiler Error CS0703</span></span>
<span data-ttu-id="0a1a6-104">Niespójność dostępności: typ ograniczenia "identifier" jest mniej dostępny niż "Identyfikator"</span><span class="sxs-lookup"><span data-stu-id="0a1a6-104">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="0a1a6-105">Ograniczenie nie może wymusić, aby parametr generyczn był mniej dostępny niż sama Klasa ogólna.</span><span class="sxs-lookup"><span data-stu-id="0a1a6-105">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="0a1a6-106">W poniższym przykładzie, podczas gdy Klasa ogólna C \<T> jest zadeklarowana jako publiczna, ograniczenie próbuje wymusić T, aby zaimplementować wewnętrzny interfejs.</span><span class="sxs-lookup"><span data-stu-id="0a1a6-106">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="0a1a6-107">Nawet jeśli jest to dozwolone, tylko klienci z dostępem wewnętrznym mogą utworzyć parametr dla klasy, dlatego w efekcie klasy mogą być używane tylko przez klientów z dostępem wewnętrznym.</span><span class="sxs-lookup"><span data-stu-id="0a1a6-107">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="0a1a6-108">Aby usunąć ten błąd, upewnij się, że poziom dostępu klasy generycznej nie jest mniej restrykcyjny niż wszystkie klasy lub interfejsy, które znajdują się w granicach.</span><span class="sxs-lookup"><span data-stu-id="0a1a6-108">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="0a1a6-109">Poniższy przykład generuje CS0703:</span><span class="sxs-lookup"><span data-stu-id="0a1a6-109">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
