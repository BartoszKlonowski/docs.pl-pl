# <a name="compiler-error-cs0260"></a><span data-ttu-id="526c2-102">Błąd kompilatora CS0260</span><span class="sxs-lookup"><span data-stu-id="526c2-102">Compiler Error CS0260</span></span>

<span data-ttu-id="526c2-103">Brak częściowego modyfikatora w deklaracji typu 'type'; istnieje inna częściowa deklaracja tego typu</span><span class="sxs-lookup"><span data-stu-id="526c2-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="526c2-104">Ten błąd wskazuje, że zadeklarowano wiele klas o tej samej nazwie.</span><span class="sxs-lookup"><span data-stu-id="526c2-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="526c2-105">Dodatkowo co najmniej jedna deklaracja (ale nie wszystkie) zawiera modyfikator `partial`.</span><span class="sxs-lookup"><span data-stu-id="526c2-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="526c2-106">Aby zdefiniować klasę w kilku częściach, musisz deklarować każdą część za pomocą słowa kluczowego `partial`.</span><span class="sxs-lookup"><span data-stu-id="526c2-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="526c2-107">Ten błąd występuje także w przypadku zadeklarowania klasy i przypadkowego nadania jej takiej samej nazwy jak innej klasie częściowej, która jest zadeklarowana w innym miejscu w tej samej przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="526c2-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="526c2-108">Poniższy przykład spowoduje wygenerowanie CS0260:</span><span class="sxs-lookup"><span data-stu-id="526c2-108">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
