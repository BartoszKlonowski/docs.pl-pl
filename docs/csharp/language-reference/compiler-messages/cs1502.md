# <a name="compiler-error-cs1502"></a><span data-ttu-id="cf572-103">Błąd kompilatora CS1502</span><span class="sxs-lookup"><span data-stu-id="cf572-103">Compiler Error CS1502</span></span>

<span data-ttu-id="cf572-104">Najlepsza przeciążona metoda dopasowania dla elementu "name" ma nieprawidłowe argumenty.</span><span class="sxs-lookup"><span data-stu-id="cf572-104">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="cf572-105">Ten błąd występuje, gdy typy argumentów, które są przesyłane do metody, nie pasują do typów parametrów tej metody.</span><span class="sxs-lookup"><span data-stu-id="cf572-105">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="cf572-106">Jeśli wywołana metoda jest przeciążona, żadna ze przeciążonych wersji nie ma sygnatury zgodnej z typami argumentów, które są przesyłane.</span><span class="sxs-lookup"><span data-stu-id="cf572-106">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="cf572-107">Aby rozwiązać ten problem, wykonaj jedną z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="cf572-107">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="cf572-108">Sprawdź podwójne typy argumentów, które są przesyłane.</span><span class="sxs-lookup"><span data-stu-id="cf572-108">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="cf572-109">Upewnij się, że są one zgodne z argumentami wywoływanej metody.</span><span class="sxs-lookup"><span data-stu-id="cf572-109">Make sure that they match the arguments of the method being called.</span></span>  
  
- <span data-ttu-id="cf572-110">W razie potrzeby przekonwertuj wszystkie niezgodne parametry przy użyciu <xref:System.Convert> klasy.</span><span class="sxs-lookup"><span data-stu-id="cf572-110">If appropriate, convert any mismatched parameters using the <xref:System.Convert> class.</span></span>  
  
- <span data-ttu-id="cf572-111">W razie potrzeby należy rzutować wszystkie niezgodne parametry, aby odpowiadały typowi oczekiwanej metody.</span><span class="sxs-lookup"><span data-stu-id="cf572-111">If appropriate, cast any mismatched parameters to match the type that the method is expecting.</span></span>  
  
- <span data-ttu-id="cf572-112">Jeśli to konieczne, zdefiniuj kolejną przeciążoną wersję metody w celu dopasowania do typów parametrów, które są wysyłane.</span><span class="sxs-lookup"><span data-stu-id="cf572-112">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>

      }  
   }
