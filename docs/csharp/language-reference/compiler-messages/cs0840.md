## <a name="to-correct-this-error"></a><span data-ttu-id="cc796-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="cc796-109">To correct this error</span></span>  
  
1. <span data-ttu-id="cc796-110">Podaj brakującą treść lub metodę dostępu albo użyj modyfikatorów [abstract](../keywords/abstract.md), [extern](../keywords/extern.md)lub [częściowe (Type)](../keywords/partial-type.md) dla tego i/lub jego typu otaczającego.</span><span class="sxs-lookup"><span data-stu-id="cc796-110">Supply the missing body or accessor or else use the [abstract](../keywords/abstract.md), [extern](../keywords/extern.md), or [partial (Type)](../keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cc796-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="cc796-111">Example</span></span>  

 <span data-ttu-id="cc796-112">Poniższy przykład generuje CS0840:</span><span class="sxs-lookup"><span data-stu-id="cc796-112">The following example generates CS0840:</span></span>  
  
```csharp

class Test  
{  
    public int myProp { get; } // CS0840

    // try the following line instead

}  
```
