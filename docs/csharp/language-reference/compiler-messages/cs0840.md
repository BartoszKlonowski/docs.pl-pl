## <a name="to-correct-this-error"></a><span data-ttu-id="b67fe-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="b67fe-109">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="b67fe-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="b67fe-111">Example</span></span>  
 <span data-ttu-id="b67fe-112">Poniższy przykład generuje CS0840:</span><span class="sxs-lookup"><span data-stu-id="b67fe-112">The following example generates CS0840:</span></span>

class Test  
{  
    public int myProp { get; } // CS0840
