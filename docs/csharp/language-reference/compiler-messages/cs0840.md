# <a name="compiler-error-cs0840"></a><span data-ttu-id="2738f-102">Błąd kompilatora CS0840</span><span class="sxs-lookup"><span data-stu-id="2738f-102">Compiler Error CS0840</span></span>
<span data-ttu-id="2738f-103">„Nazwa właściwości” musi deklarować treść, ponieważ nie jest ona oznaczona jako abstrakcyjna ani zewnętrzna.</span><span class="sxs-lookup"><span data-stu-id="2738f-103">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="2738f-104">Automatycznie implementowane właściwości muszą definiować obie metody dostępu — „get” i „set”.</span><span class="sxs-lookup"><span data-stu-id="2738f-104">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="2738f-105">Jeśli właściwość regularna nie jest oznaczona jako `abstract` lub `extern` albo jest elementem członkowskim typu `partial`, musi zawierać treść.</span><span class="sxs-lookup"><span data-stu-id="2738f-105">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="2738f-106">Właściwości implementowane automatycznie nie zawierają treści metod dostępu, ale muszą określać obie metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="2738f-106">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="2738f-107">Aby utworzyć implementowaną automatycznie właściwość tylko do odczytu, metodę dostępu „set” należy określić jako `private`.</span><span class="sxs-lookup"><span data-stu-id="2738f-107">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2738f-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="2738f-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="2738f-109">Dodaj brakującą treść lub metodę dostępu albo użyj na niej i jej typie otaczającym jednego z tych modyfikatorów: [abstract](../../../csharp/language-reference/keywords/abstract.md), [extern](../../../csharp/language-reference/keywords/extern.md) lub [partial (typ)](../../../csharp/language-reference/keywords/partial-type.md).</span><span class="sxs-lookup"><span data-stu-id="2738f-109">Supply the missing body or accessor or else use the [abstract](../../../csharp/language-reference/keywords/abstract.md), [extern](../../../csharp/language-reference/keywords/extern.md), or [partial (Type)](../../../csharp/language-reference/keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2738f-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="2738f-110">Example</span></span>  
 <span data-ttu-id="2738f-111">Poniższy przykład generuje CS0840:</span><span class="sxs-lookup"><span data-stu-id="2738f-111">The following example generates CS0840:</span></span>  
  
```csharp  
// cs0840.cs  
// Compile with /target:library  
using System;  
class Test  
{  
    public int myProp { get; } // CS0840  
  
    // to create a read-only property  
    // try the following line instead  
    public int myProp2 { get; private set; }  
  
}  
```
