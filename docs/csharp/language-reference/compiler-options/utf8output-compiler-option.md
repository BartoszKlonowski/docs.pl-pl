# <a name="-utf8output-c-compiler-options"></a>-utf8output — (opcje kompilatora C#)
Opcja **-utf8output —** wyświetla dane wyjściowe kompilatora przy użyciu kodowania UTF-8.  
  
## <a name="syntax"></a>Składnia  
  
```console  
-utf8output  
```  
  
## <a name="remarks"></a>Uwagi  
 W niektórych konfiguracjach międzynarodowych dane wyjściowe kompilatora nie mogą być prawidłowo wyświetlane w konsoli programu. W tych konfiguracjach należy użyć parametrów **-utf8output —** i przekierować dane wyjściowe kompilatora do pliku.  
  
 Ta opcja kompilatora jest niedostępna w programie Visual Studio i nie można jej zmienić programowo.
