# <a name="-fullpaths-c-compiler-options"></a>-fullpaths — (C# opcje kompilatora)
Opcja **-fullpaths —** powoduje, że kompilator Określa pełną ścieżkę do pliku podczas wyświetlania listy błędów kompilacji i ostrzeżeń.  
  
## <a name="syntax"></a>Składnia  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Uwagi  
 Domyślnie błędy i ostrzeżenia wynikające z kompilacji określają nazwę pliku, w którym został znaleziony błąd. Opcja **-fullpaths —** powoduje, że kompilator Określa pełną ścieżkę do pliku.  
  
 Ta opcja kompilatora jest niedostępna w programie Visual Studio i nie można jej zmienić programowo.
