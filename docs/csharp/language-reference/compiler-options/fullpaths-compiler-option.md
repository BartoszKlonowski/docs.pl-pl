# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (Opcje kompilatora C#)
Opcja **-fullpaths** powoduje, że kompilator określa pełną ścieżkę do pliku podczas wyświetlania błędów kompilacji i ostrzeżeń.  
  
## <a name="syntax"></a>Składnia  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Uwagi  
 Domyślnie błędy i ostrzeżenia wynikające z kompilacji określają nazwę pliku, w którym znaleziono błąd. Opcja **-fullpaths** powoduje, że kompilator określa pełną ścieżkę do pliku.  
  
 Ta opcja kompilatora jest niedostępna w programie Visual Studio i nie można jej zmienić programowo.
