# <a name="-nologo-c-compiler-options"></a>-nologo (Opcje kompilatora C#)
Opcja **-nologo** pomija wyświetlanie banera logowania podczas uruchamiania kompilatora i wyświetlanie komunikatów informacyjnych podczas kompilowania.  
  
## <a name="syntax"></a>Składnia  
  
```console  
-nologo  
```  
  
## <a name="remarks"></a>Uwagi  
 Ta opcja nie jest dostępna w środowisku programistycznym; jest dostępna tylko podczas kompilowania z wiersza polecenia.  
  
 Ta opcja kompilatora jest niedostępna w programie Visual Studio i nie można jej zmienić programowo.
