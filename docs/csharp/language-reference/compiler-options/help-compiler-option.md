# <a name="-help---c-compiler-options"></a>-help, -? (Opcje kompilatora Języka C#)
Ta opcja wysyła listę opcji kompilatora i krótki opis każdej opcji, aby stdout.  
  
## <a name="syntax"></a>Składnia  
  
```console  
-help  
-?  
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli ta opcja jest uwzględniona w kompilacji, nie zostanie utworzony żaden plik wyjściowy i nie będzie kompilacji.  
  
 Ta opcja kompilatora jest niedostępna w programie Visual Studio i nie można jej zmienić programowo.
