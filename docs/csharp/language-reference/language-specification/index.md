# <a name="c-60-draft-language-specification"></a>Projekt C# w wersji 6.0 specyfikacja języka
Specyfikacja języka C# jest ostatecznym źródłem informacji o składni i użyciu C#. Określenie tej wartości zawiera szczegółowe informacje dotyczące wszystkich aspektów języka, w tym wiele punktów, które nie obejmuje dokumentacji Visual C#.

Możesz pobrać wersję 5.0 tej specyfikacji z [Microsoft Download Center](http://www.microsoft.com/download/details.aspx?id=7029). Jeśli po zainstalowaniu programu Visual Studio 2015, można również znaleźć specyfikację na komputerze w folderze Program Files (x86)/Microsoft Visual Studio 14.0/VC#/Specifications/1033. Jeśli zainstalowana jest inna wersja programu Visual Studio lub Visual Studio jest zainstalowany w innym języku niż angielski, należy zmienić ścieżkę zależnie od potrzeb.

W wersji 6.0 specyfikacji nie została zatwierdzona jako standard. Ta witryna zawiera [ *projekt* specyfikacji języka C# w wersji 6.0](../../../../_csharplang/spec/lexical-structure.md). Jest zbudowany zawartych w plikach markdown [repozytorium GitHub dotnet/csharplang](https://github.com/dotnet/csharplang/blob/master/spec/README.md).

Kwestie dotyczące specyfikacji wersji roboczej należy utworzyć w [dotnet/csharplang](https://github.com/dotnet/csharplang/issues) repozytorium. Lub, jeśli interesuje Cię poprawiania błędów można znaleźć, możesz przesłać [żądanie ściągnięcia](https://github.com/dotnet/csharplang/pulls) do tego samego repozytorium.
