 <span data-ttu-id="8fe9b-113">Wyjątki mają następujące właściwości:</span><span class="sxs-lookup"><span data-stu-id="8fe9b-113">Exceptions have the following properties:</span></span>

- <span data-ttu-id="8fe9b-116">Gdy wystąpi wyjątek w `try` zablokować, przepływu sterowania przechodzi do pierwszego programu obsługi wyjątków skojarzone, który znajduje się w dowolnym miejscu w stosie wywołań.</span><span class="sxs-lookup"><span data-stu-id="8fe9b-116">Once an exception occurs in the `try` block, the flow of control jumps to the first associated exception handler that is present anywhere in the call stack.</span></span> <span data-ttu-id="8fe9b-117">W języku C# `catch` — słowo kluczowe jest używane do definiowania obsługi wyjątków.</span><span class="sxs-lookup"><span data-stu-id="8fe9b-117">In C#, the `catch` keyword is used to define an exception handler.</span></span>

- <span data-ttu-id="8fe9b-121">Jeśli `catch` bloku definiuje zmienną wyjątek, służy do uzyskania dalszych informacji o typ wyjątku, który wystąpił.</span><span class="sxs-lookup"><span data-stu-id="8fe9b-121">If a `catch` block defines an exception variable, you can use it to obtain more information about the type of exception that occurred.</span></span>  
  
- <span data-ttu-id="8fe9b-122">Wyjątki mogą jawnie generowane przez program przy użyciu `throw` — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="8fe9b-122">Exceptions can be explicitly generated by a program by using the `throw` keyword.</span></span>  
  
- <span data-ttu-id="8fe9b-123">Obiekty wyjątków zawierają szczegółowe informacje o błędzie, takie jak stan stos wywołań i tekst opisu błędu.</span><span class="sxs-lookup"><span data-stu-id="8fe9b-123">Exception objects contain detailed information about the error, such as the state of the call stack and a text description of the error.</span></span>
