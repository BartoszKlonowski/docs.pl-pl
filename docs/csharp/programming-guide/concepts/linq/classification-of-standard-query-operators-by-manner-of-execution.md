## <a name="manners-of-execution"></a><span data-ttu-id="840a4-108">Maniery wykonania</span><span class="sxs-lookup"><span data-stu-id="840a4-108">Manners of Execution</span></span>  
  
### <a name="immediate"></a><span data-ttu-id="840a4-109">Natychmiastowe</span><span class="sxs-lookup"><span data-stu-id="840a4-109">Immediate</span></span>  
 <span data-ttu-id="840a4-110">Natychmiastowe wykonanie oznacza, że źródło danych jest odczytywane, a operacja jest wykonywana w punkcie w kodzie, w którym jest zadeklarowana kwerenda.</span><span class="sxs-lookup"><span data-stu-id="840a4-110">Immediate execution means that the data source is read and the operation is performed at the point in the code where the query is declared.</span></span> <span data-ttu-id="840a4-111">Wszystkie standardowe operatory zapytań, które zwracają pojedynczy, niewyliczony wynik, są wykonywane natychmiast.</span><span class="sxs-lookup"><span data-stu-id="840a4-111">All the standard query operators that return a single, non-enumerable result execute immediately.</span></span>  
  
### <a name="deferred"></a><span data-ttu-id="840a4-112">Odroczonego</span><span class="sxs-lookup"><span data-stu-id="840a4-112">Deferred</span></span>  
 <span data-ttu-id="840a4-113">Odroczone wykonanie oznacza, że operacja nie jest wykonywana w punkcie w kodzie, w którym jest zadeklarowana kwerenda.</span><span class="sxs-lookup"><span data-stu-id="840a4-113">Deferred execution means that the operation is not performed at the point in the code where the query is declared.</span></span> <span data-ttu-id="840a4-114">Operacja jest wykonywana tylko wtedy, gdy zmienna kwerendy jest `foreach` wyliczana, na przykład przy użyciu instrukcji.</span><span class="sxs-lookup"><span data-stu-id="840a4-114">The operation is performed only when the query variable is enumerated, for example by using a `foreach` statement.</span></span> <span data-ttu-id="840a4-115">Oznacza to, że wyniki wykonywania kwerendy zależą od zawartości źródła danych, gdy kwerenda jest wykonywana, a nie po zdefiniowaniu kwerendy.</span><span class="sxs-lookup"><span data-stu-id="840a4-115">This means that the results of executing the query depend on the contents of the data source when the query is executed rather than when the query is defined.</span></span> <span data-ttu-id="840a4-116">Jeśli zmienna kwerendy jest wyliczana wiele razy, wyniki mogą się różnić za każdym razem.</span><span class="sxs-lookup"><span data-stu-id="840a4-116">If the query variable is enumerated multiple times, the results might differ every time.</span></span> <span data-ttu-id="840a4-117">Prawie wszystkie standardowe operatory zapytań, których typ zwracany jest <xref:System.Collections.Generic.IEnumerable%601> lub <xref:System.Linq.IOrderedEnumerable%601> wykonać w sposób odroczony.</span><span class="sxs-lookup"><span data-stu-id="840a4-117">Almost all the standard query operators whose return type is <xref:System.Collections.Generic.IEnumerable%601> or <xref:System.Linq.IOrderedEnumerable%601> execute in a deferred manner.</span></span>  
  
 <span data-ttu-id="840a4-118">Operatory zapytań, które używają odroczonego wykonania można dodatkowo sklasyfikować jako przesyłanie strumieniowe lub nieprzesyłania strumieniowego.</span><span class="sxs-lookup"><span data-stu-id="840a4-118">Query operators that use deferred execution can be additionally classified as streaming or non-streaming.</span></span>  
  
#### <a name="streaming"></a><span data-ttu-id="840a4-119">Przesyłanie strumieniowe</span><span class="sxs-lookup"><span data-stu-id="840a4-119">Streaming</span></span>  
 <span data-ttu-id="840a4-120">Operatory przesyłania strumieniowego nie muszą odczytywać wszystkich danych źródłowych, zanim wydadzą elementy.</span><span class="sxs-lookup"><span data-stu-id="840a4-120">Streaming operators do not have to read all the source data before they yield elements.</span></span> <span data-ttu-id="840a4-121">W czasie wykonywania operator przesyłania strumieniowego wykonuje swoją operację na każdym elemencie źródłowym, ponieważ jest odczytywany i daje element, jeśli jest to właściwe.</span><span class="sxs-lookup"><span data-stu-id="840a4-121">At the time of execution, a streaming operator performs its operation on each source element as it is read and yields the element if appropriate.</span></span> <span data-ttu-id="840a4-122">Operator przesyłania strumieniowego kontynuuje odczytywanie elementów źródłowych, dopóki nie zostanie wyprodukowany element wynikowy.</span><span class="sxs-lookup"><span data-stu-id="840a4-122">A streaming operator continues to read source elements until a result element can be produced.</span></span> <span data-ttu-id="840a4-123">Oznacza to, że więcej niż jeden element źródłowy może być odczytywany w celu uzyskania jednego elementu wynikowego.</span><span class="sxs-lookup"><span data-stu-id="840a4-123">This means that more than one source element might be read to produce one result element.</span></span>  
  
#### <a name="non-streaming"></a><span data-ttu-id="840a4-124">Brak przesyłania strumieniowego</span><span class="sxs-lookup"><span data-stu-id="840a4-124">Non-Streaming</span></span>  
 <span data-ttu-id="840a4-125">Operatory nieprzesyłate strumieniowo muszą odczytać wszystkie dane źródłowe, zanim mogą udawać element wynikowy.</span><span class="sxs-lookup"><span data-stu-id="840a4-125">Non-streaming operators must read all the source data before they can yield a result element.</span></span> <span data-ttu-id="840a4-126">Operacje, takie jak sortowanie lub grupowanie należą do tej kategorii.</span><span class="sxs-lookup"><span data-stu-id="840a4-126">Operations such as sorting or grouping fall into this category.</span></span> <span data-ttu-id="840a4-127">W czasie wykonywania operatory zapytań nieprzesyłających strumieniowo odczytać wszystkie dane źródłowe, umieścić go w strukturze danych, wykonać operację i przynieść wynikowe elementy.</span><span class="sxs-lookup"><span data-stu-id="840a4-127">At the time of execution, non-streaming query operators read all the source data, put it into a data structure, perform the operation, and yield the resulting elements.</span></span>
