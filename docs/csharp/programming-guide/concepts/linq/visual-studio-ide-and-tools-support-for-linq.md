# <a name="visual-studio-ide-and-tools-support-for-linq-c"></a>Obsługa IDE programu Visual Studio dla LINQ (C#)
Zintegrowane środowisko programistyczne (IDE) programu Visual Studio udostępnia następujące funkcje obsługujące tworzenie aplikacji LINQ:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 Object Relational Designer to narzędzie do projektowania wizualnego, którego można używać w [linq do](../../../../framework/data/adonet/sql/linq/index.md) aplikacji SQL do generowania klas w języku C#, które reprezentują dane relacyjne w podstawowej bazie danych. Aby uzyskać więcej informacji, zobacz [LINQ to SQL Tools w programie Visual Studio](/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2).  
  
## <a name="sqlmetal-command-line-tool"></a>Narzędzie wiersza polecenia SQLMetal  
 SQLMetal jest narzędziem wiersza polecenia, które mogą być używane w procesach kompilacji do generowania klas z istniejących baz danych do użycia w LINQ do aplikacji SQL. Aby uzyskać więcej informacji, zobacz [SqlMetal.exe (Narzędzie do generowania kodu)](../../../../framework/tools/sqlmetal-exe-code-generation-tool.md).  
  
## <a name="linq-aware-code-editor"></a>Edytor kodu z obsługą LINQ  
 Edytor kodu C# obsługuje linq intensywnie z IntelliSense i możliwości formatowania.  
  
## <a name="visual-studio-debugger-support"></a>Obsługa debugera programu Visual Studio  
 Debuger programu Visual Studio obsługuje debugowanie wyrażeń zapytań. Aby uzyskać więcej informacji, zobacz [Debugowanie LINQ](/visualstudio/debugger/debugging-linq).
