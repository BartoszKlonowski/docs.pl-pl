## <a name="what-is-pure-functional-transformation"></a><span data-ttu-id="55ddf-107">Co to jest czysta transformacja funkcjonalna?</span><span class="sxs-lookup"><span data-stu-id="55ddf-107">What Is Pure Functional Transformation?</span></span>

<span data-ttu-id="55ddf-108">W *czystej transformacji funkcjonalnej*, zestaw funkcji, *zwanyczystych funkcji*, zdefiniować, jak przekształcić zestaw danych strukturalnych z jego pierwotnej postaci w innej formie.</span><span class="sxs-lookup"><span data-stu-id="55ddf-108">In *pure functional transformation*, a set of functions, called *pure functions*, define how to transform a set of structured data from its original form into another form.</span></span> <span data-ttu-id="55ddf-109">Słowo "czysty" wskazuje, że funkcje są *komponowane,* co wymaga, że są:</span><span class="sxs-lookup"><span data-stu-id="55ddf-109">The word "pure" indicates that the functions are *composable*, which requires that they are:</span></span>

- <span data-ttu-id="55ddf-110">*Niezależne*, tak, że mogą być swobodnie uporządkowane i przegrupowane bez splątania lub współzależności z resztą programu.</span><span class="sxs-lookup"><span data-stu-id="55ddf-110">*Self-contained*, so that they can be freely ordered and rearranged without entanglement or interdependencies with the rest of the program.</span></span> <span data-ttu-id="55ddf-111">Czyste przemiany nie mają wiedzy ani wpływu na ich środowisko.</span><span class="sxs-lookup"><span data-stu-id="55ddf-111">Pure transformations have no knowledge of or effect upon their environment.</span></span> <span data-ttu-id="55ddf-112">Oznacza to, że funkcje używane w transformacji nie mają *skutków ubocznych*.</span><span class="sxs-lookup"><span data-stu-id="55ddf-112">That is, the functions used in the transformation have no *side effects*.</span></span>

- <span data-ttu-id="55ddf-113">*Bezstanowe*, tak aby wykonywanie tej samej funkcji lub określonego zestawu funkcji na tym samym wejściu zawsze spowoduje to samo dane wyjściowe.</span><span class="sxs-lookup"><span data-stu-id="55ddf-113">*Stateless*, so that executing the same function or specific set of functions on the same input will always result in the same output.</span></span> <span data-ttu-id="55ddf-114">Czyste transformacje nie mają pamięci o ich uprzednim użyciu.</span><span class="sxs-lookup"><span data-stu-id="55ddf-114">Pure transformations have no memory of their prior use.</span></span>
