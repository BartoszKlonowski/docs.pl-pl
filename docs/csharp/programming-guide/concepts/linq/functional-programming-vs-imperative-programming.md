## <a name="advantages-of-pure-functions"></a><span data-ttu-id="01060-149">Zalety czystych funkcji</span><span class="sxs-lookup"><span data-stu-id="01060-149">Advantages of Pure Functions</span></span>  
 <span data-ttu-id="01060-150">Podstawową przyczyną wdrożenia przekształceń funkcjonalnych jako czystych funkcji jest możliwość tworzenia czystych funkcji: to jest, samodzielne i bezstanowe.</span><span class="sxs-lookup"><span data-stu-id="01060-150">The primary reason to implement functional transformations as pure functions is that pure functions are composable: that is, self-contained and stateless.</span></span> <span data-ttu-id="01060-151">Te cechy obejmują szereg korzyści, w tym następujące:</span><span class="sxs-lookup"><span data-stu-id="01060-151">These characteristics bring a number of benefits, including the following:</span></span>  
  
- <span data-ttu-id="01060-152">Zwiększ czytelność i łatwość utrzymania.</span><span class="sxs-lookup"><span data-stu-id="01060-152">Increased readability and maintainability.</span></span> <span data-ttu-id="01060-153">Jest to spowodowane tym, że każda funkcja została zaprojektowana w celu wykonania określonego zadania z uwzględnieniem argumentów.</span><span class="sxs-lookup"><span data-stu-id="01060-153">This is because each function is designed to accomplish a specific task given its arguments.</span></span> <span data-ttu-id="01060-154">Funkcja nie polega na żadnym stanie zewnętrznym.</span><span class="sxs-lookup"><span data-stu-id="01060-154">The function does not rely on any external state.</span></span>
