 <span data-ttu-id="25929-161">W poniższym przykładzie jest tworzony i uruchamiane przez wątek `Main` funkcji.</span><span class="sxs-lookup"><span data-stu-id="25929-161">In the following example, a thread is created and started by the `Main` function.</span></span> <span data-ttu-id="25929-162">Nowy wątek czeka na zdarzenia przy użyciu <xref:System.Threading.WaitHandle.WaitOne%2A> metody.</span><span class="sxs-lookup"><span data-stu-id="25929-162">The new thread waits on an event using the <xref:System.Threading.WaitHandle.WaitOne%2A> method.</span></span> <span data-ttu-id="25929-163">Wątek jest zawieszony, aż zdarzenie stanie się zasygnalizowane przez wątek główny, który jest wykonywany `Main` funkcji.</span><span class="sxs-lookup"><span data-stu-id="25929-163">The thread is suspended until the event becomes signaled by the primary thread that is executing the `Main` function.</span></span> <span data-ttu-id="25929-164">Gdy zdarzenie jest sygnalizowane, zwraca pomocniczego wątku.</span><span class="sxs-lookup"><span data-stu-id="25929-164">Once the event becomes signaled, the auxiliary thread returns.</span></span> <span data-ttu-id="25929-165">W tym przypadku ponieważ zdarzenie jest używana tylko jeden wątek aktywacji, albo <xref:System.Threading.AutoResetEvent> lub <xref:System.Threading.ManualResetEvent> klasy mogą być używane.</span><span class="sxs-lookup"><span data-stu-id="25929-165">In this case, because the event is only used for one thread activation, either the <xref:System.Threading.AutoResetEvent> or <xref:System.Threading.ManualResetEvent> classes could be used.</span></span>  
  
```csharp  
using System;  
using System.Threading;  
  
class ThreadingExample  
{  
    static AutoResetEvent autoEvent;  
  
    static void DoWork()  
    {  
        Console.WriteLine("   worker thread started, now waiting on event...");  
        autoEvent.WaitOne();  
        Console.WriteLine("   worker thread reactivated, now exiting...");  
    }  
  
    static void Main()  
    {  
        autoEvent = new AutoResetEvent(false);  
  
        Console.WriteLine("main thread starting worker thread...");  
        Thread t = new Thread(DoWork);  
        t.Start();  
  
        Console.WriteLine("main thread sleeping for 1 second...");  
        Thread.Sleep(1000);  
  
        Console.WriteLine("main thread signaling worker thread...");  
        autoEvent.Set();  
    }  
}  
```
