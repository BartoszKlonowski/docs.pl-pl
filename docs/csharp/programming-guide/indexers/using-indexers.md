array.</span></span> <span data-ttu-id="adff3-104">Indeksatory najczęściej są implementowane w typach, której głównym celem jest hermetyzacji wewnętrznej kolekcji lub tablicy.</span><span class="sxs-lookup"><span data-stu-id="adff3-104">Indexers are most frequently implemented in types whose primary purpose is to encapsulate an internal collection or array.</span></span> <span data-ttu-id="adff3-105">Załóżmy na przykład, posiadasz klasę `TempRecord` reprezentujący temperaturę w Farenheit zarejestrowanej w 10 różnych razy w okresie 24-godzinnym.</span><span class="sxs-lookup"><span data-stu-id="adff3-105">For example, suppose you have a class `TempRecord` that represents the temperature in Farenheit as recorded at 10 different times during a 24 hour ## <a name="remarks"></a><span data-ttu-id="adff3-110">Uwagi</span><span class="sxs-lookup"><span data-stu-id="adff3-110">Remarks</span></span>

 <span data-ttu-id="adff3-118">Aby zapewnić indeksatora o nazwie używanego w innych językach, należy użyć <xref:System.Runtime.CompilerServices.IndexerNameAttribute?displayProperty=nameWithType>, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="adff3-118">To provide the indexer with a name that other languages can use, use <xref:System.Runtime.CompilerServices.IndexerNameAttribute?displayProperty=nameWithType>, as the following example shows:</span></span>

```csharp
    // get and set accessors

```

<span data-ttu-id="adff3-119">Ten indeksator będzie mieć taką nazwę `TheItem`.</span><span class="sxs-lookup"><span data-stu-id="adff3-119">This indexer will have the name `TheItem`.</span></span> <span data-ttu-id="adff3-120">Nie udostępniają atrybut name czyniłyby `Item` nazwę domyślną.</span><span class="sxs-lookup"><span data-stu-id="adff3-120">Not providing the name attribute would make `Item` the default name.</span></span>
