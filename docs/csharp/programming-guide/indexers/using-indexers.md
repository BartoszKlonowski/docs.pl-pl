array.</span></span> <span data-ttu-id="3030b-104">Indeksatory najczęściej są implementowane w typach, których podstawowym celem jest hermetyzować wewnętrznej kolekcji lub tablicy.</span><span class="sxs-lookup"><span data-stu-id="3030b-104">Indexers are most frequently implemented in types whose primary purpose is to encapsulate an internal collection or array.</span></span> <span data-ttu-id="3030b-105">Na przykład załóżmy, że istnieje klasa o nazwie TempRecord reprezentujący temperatury w Farenheit zarejestrowane w różnym czasie 10 w okresie 24 godzin.</span><span period.</span></span> <span data-ttu-id="3030b-106">Klasa zawiera tablicę o nazwie "temps" typ float do reprezentowania temperatury, a <xref:System.DateTime> reprezentujący datę temperatury zostały
