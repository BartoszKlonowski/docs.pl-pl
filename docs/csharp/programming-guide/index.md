# <a name="c-programming-guide"></a>Podręcznik programowania C#
Ta sekcja zawiera szczegółowe informacje o kluczowych właściwościach języka C# oraz o funkcjonalnościach dostępnych w tym języku programowania dzięki platformie .NET Framework.  
  
 W większej części tej sekcji przyjęto założenie, że wiesz już nieco o C# i ogólnie o programowaniu. Jeśli jesteś zupełnie początkującym programistą lub początkującym w języku C#, możesz odwiedzić [Wprowadzenie do języka C#](https://www.microsoft.com/net/tutorials/csharp/getting-started). Jest to interaktywny samouczek, w którym nie jest wymagana żadna znajomość programowania.  
  
 Aby uzyskać informacje dotyczące określonych słów kluczowych, operatorów i dyrektyw preprocesora, zobacz [Dokumentację Języka C#](../../csharp/language-reference/index.md). Aby uzyskać informacje o specyfikacji języka C#, zobacz [Specyfikację Języka C#](../../csharp/language-reference/language-specification/index.md).
