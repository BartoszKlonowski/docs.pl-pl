<span data-ttu-id="d48ad-118">Typy wartości null mają następującą charakterystykę:</span><span class="sxs-lookup"><span data-stu-id="d48ad-118">Nullable value types have the following characteristics:</span></span>

- <span data-ttu-id="d48ad-122">Przypisz wartość do typu wartości null tak samo jak dla bazowego typu wartości: `int? x = 10;` lub `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-122">Assign a value to a nullable value type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="d48ad-123">Można również przypisać wartość `null`: `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-123">You also can assign the `null` value: `int? x = null;`.</span></span>

  - <span data-ttu-id="d48ad-125">Właściwość <xref:System.Nullable%601.HasValue%2A> zwraca `true`, jeśli zmienna zawiera wartość lub `false`, jeśli jest `null`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-125">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>

  - <span data-ttu-id="d48ad-126">Właściwość <xref:System.Nullable%601.Value%2A> zwraca wartość, jeśli <xref:System.Nullable%601.HasValue%2A> zwróci `true`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-126">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="d48ad-127">W przeciwnym razie zostanie zgłoszony <xref:System.InvalidOperationException>.</span><span class="sxs-lookup"><span data-stu-id="d48ad-127">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>

- <span data-ttu-id="d48ad-128">Można również użyć operatorów `==` i `!=` z typem wartości null, jak pokazano w następującym przykładzie: `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-128">You can also use the `==` and `!=` operators with a nullable value type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="d48ad-129">Jeśli `a` i `b` są równe null, `a == b` szacuje się do `true`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-129">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>

- <span data-ttu-id="d48ad-131">Wartość domyślna `T?` to wystąpienie, którego właściwość <xref:System.Nullable%601.HasValue%2A> zwraca `false`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-131">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>

- <span data-ttu-id="d48ad-132">Użyj metody <xref:System.Nullable%601.GetValueOrDefault> do zwrócenia przypisanej wartości lub wartości [domyślnej](../../language-reference/keywords/default-values-table.md) bazowego typu wartości, jeśli wartość tego typu jest `null`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-132">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>

- <span data-ttu-id="d48ad-133">Użyj metody <xref:System.Nullable%601.GetValueOrDefault(%600)> do zwrócenia przypisanej wartości lub podanej wartości domyślnej, jeśli wartość tego typu jest `null`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-133">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>

- <span data-ttu-id="d48ad-134">Użyj [operatora łączenia wartości null](../../language-reference/operators/null-coalescing-operator.md), `??`, aby przypisać wartość do zmiennej bazowego typu wartości na podstawie wartości typu nullable: `int? x = null; int y = x ?? -1;`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-134">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to a variable of an underlying value type based on a nullable-type value: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="d48ad-135">W przykładzie, ponieważ `x` jest `null`, wartość wyniku `y` jest `-1`.</span><span class="sxs-lookup"><span data-stu-id="d48ad-135">In the example, since `x` is `null`, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="d48ad-136">Jeśli zdefiniowana przez użytkownika konwersja jest zdefiniowana między dwoma typami wartości, można także użyć tej samej konwersji z odpowiadającymi im typami dopuszczającymi wartość null.</span><span class="sxs-lookup"><span data-stu-id="d48ad-136">If a user-defined conversion is defined between two value types, the same conversion can also be used with the corresponding nullable types.</span></span>

- <span data-ttu-id="d48ad-137">Zagnieżdżone typy wartości null są niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="d48ad-137">Nested nullable value types are not allowed.</span></span> <span data-ttu-id="d48ad-138">Następujący wiersz nie kompiluje się: `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="d48ad-138">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>
