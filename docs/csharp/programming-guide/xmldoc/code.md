# <a name="code-c-programming-guide"></a>> kodu \<(C# Przewodnik programowania)
## <a name="syntax"></a>Składnia  
  
```xml  
<code>content</code>  
```  
  
## <a name="parameters"></a>Parametry  
 `content`  
 Tekst, który ma zostać oznaczony jako kod.  
  
## <a name="remarks"></a>Uwagi  
 Tag \<kod > umożliwia wskazanie wielu wierszy jako kodu. Użyj [\<c >](./code-inline.md) , aby wskazać, że tekst w opisie powinien być oznaczony jako kod.  
  
 Kompiluj z [-doc](../../language-reference/compiler-options/doc-compiler-option.md) , aby przetwarzać komentarze dokumentacji do pliku.  
  
## <a name="example"></a>Przykład
