# <a name="code-c-programming-guide"></a>\<kod> (przewodnik programowania C#)

## <a name="syntax"></a>Składnia

```xml
<code>content</code>
```

## <a name="parameters"></a>Parametry

- `content`

  Tekst, który ma być oznaczony jako kod.

## <a name="remarks"></a>Uwagi

Kod \<> tag umożliwia wskazanie wielu wierszy jako kodu. Użyj [ \<c>,](./code-inline.md) aby wskazać, że tekst w opisie powinien być oznaczony jako kod.

Skompiluj za pomocą [-doc,](../../language-reference/compiler-options/doc-compiler-option.md) aby przetworzyć komentarze dokumentacji do pliku.
