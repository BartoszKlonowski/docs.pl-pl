# <a name="code-c-programming-guide"></a>\<code>(Przewodnik programowania w języku C#)

## <a name="syntax"></a>Składnia

```xml
<code>content</code>
```

## <a name="parameters"></a>Parametry

- `content`

  Tekst, który ma zostać oznaczony jako kod.

## <a name="remarks"></a>Uwagi

`<code>`Tag jest używany do wskazania wielu wierszy kodu. Użyj, [\<c>](./code-inline.md) Aby wskazać, że tekst jednowierszowy w opisie powinien być oznaczony jako kod.

Kompiluj z [-doc](../../language-reference/compiler-options/doc-compiler-option.md) , aby przetwarzać komentarze dokumentacji do pliku.
