# <a name="ltcodegt-c-programming-guide"></a>&lt;Kod&gt; (C# przewodnik programowania w języku)
## <a name="syntax"></a>Składnia  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parametry  
 `content`  
 Tekst, który ma być oznaczona jako kod.  
  
## <a name="remarks"></a>Uwagi  
 \<Kod > tagu umożliwia wskazująca wiele wierszy, jak kod. Użyj [ \<c >](../../../csharp/programming-guide/xmldoc/code-inline.md) aby wskazać, że tekst opisu powinien być oznaczony jako kod.  
  
 Kompiluj z użyciem [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) na przetwarzanie komentarzy dokumentacji do pliku.
