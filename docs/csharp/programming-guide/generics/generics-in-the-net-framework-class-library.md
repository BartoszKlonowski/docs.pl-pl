# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Typy ogólne w bibliotece klas programu .NET Framework (Przewodnik programowania w języku C#)
Biblioteka klas programu .NET Framework w wersji 2.0 zapewnia nowy obszar nazw <xref:System.Collections.Generic>, która obejmuje kilka gotowych do użycia kolekcji ogólnej klasy i interfejsy skojarzone. Innych przestrzeniach nazw, takich jak <xref:System>, takich jak udostępniają nowe interfejsy ogólne <xref:System.IComparable%601>. Te klasy i interfejsy są bardziej wydajny i bezpieczny niż klasy kolekcja nierodzajową podane we wcześniejszych wersjach programu .NET Framework. Przed projektowanie i wdrażanie własnych klas kolekcji niestandardowej, należy wziąć pod uwagę czy możesz użyć lub wyprowadzenia klasy z jednej z klas w bibliotece klas programu .NET Framework.
