approach to parameterized types without the complexity of C++ templates.</span></span> <span data-ttu-id="90c70-106">Ponadto C# nie próbuje zapewniają wszystkich funkcji, które zapewniają szablonów  <span data-ttu-id="90c70-109">Poniżej przedstawiono podstawowe różnice między typami ogólnymi C# i szablonów języka C++:</span><span class="sxs-lookup"><span data-stu-id="90c70-109">The following are the key differences between C# Generics and C++ templates:</span></span>  
  
-   <span data-ttu-id="90c70-110">Typami ogólnymi C# nie są oferowane tyle samo elastyczność jako szablonów języka C++.</span><span class="sxs-lookup"><span data-stu-id="90c70-110">C# generics do not provide the same amount of flexibility as C++ templates.</span></span> <span data-ttu-id="90c70-111">Na przykład prawdopodobnie nie wywoła operatorów arytmetycznych w języku C# klasy ogólnej, chociaż jest możliwe do wywołania operatory zdefiniowane przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="90c70-111">For example, it is not possible to call arithmetic operators in a C# generic class, although it is possible to call user defined operators.</span></span>  
  
-   <span data-ttu-id="90c70-112">C# nie zezwala na parametrów szablonu bez typu, takie jak `template C<int i> {}`.</span><span class="sxs-lookup"><span data-stu-id="90c70-112">C# does not allow non-type template parameters, such as `template C<int i> {}`.</span></span>  
  
-   <span data-ttu-id="90c70-113">C# nie obsługuje jawna specjalizacja; oznacza to, że implementacja niestandardowa szablonu dla określonego typu.</span><span class="sxs-lookup"><span data-stu-id="90c70-113">C# does not support explicit specialization; that is, a custom implementation of a template for a specific type.</span></span>  
  
-   <span data-ttu-id="90c70-114">C# nie obsługuje częściowej specjalizacji: implementacja niestandardowa dla podzbioru argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="90c70-114">C# does not support partial specialization: a custom implementation for a subset of the type arguments.</span></span>  
  
-   <span data-ttu-id="90c70-115">C# nie zezwala na parametr typu ma być używany jako klasa bazowa dla typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="90c70-115">C# does not allow the type parameter to be used as the base class for the generic type.</span></span>  
  
-   <span data-ttu-id="90c70-116">C# nie zezwala na parametry typu mają domyślne typy.</span><span class="sxs-lookup"><span data-stu-id="90c70-116">C# does not allow type parameters to have default types.</span></span>  
  
-   <span data-ttu-id="90c70-117">W języku C#, parametru typu generycznego nie może sam być ogólnego, chociaż typy utworzone mogą być używane jako ogólne.</span><span class="sxs-lookup"><span data-stu-id="90c70-117">In C#, a generic type parameter cannot itself be a generic, although constructed types can be used as generics.</span></span> <span data-ttu-id="90c70-118">C++ umożliwia parametry szablonu.</span><span class="sxs-lookup"><span data-stu-id="90c70-118">C++ does allow template parameters.</span></span>  
  
-   <span data-ttu-id="90c70-119">C++ umożliwia kod, który nie jest prawidłowe dla wszystkich parametrów typu w szablonie, które następnie są sprawdzane pod kątem określonego typu, używany jako parametr typu.</span><span class="sxs-lookup"><span data-stu-id="90c70-119">C++ allows code that might not be valid for all type parameters in the template, which is then checked for the specific type used as the type parameter.</span></span> <span data-ttu-id="90c70-120">Język C# wymaga kod w klasie do zapisania w taki sposób, że będzie ona współdziałać z dowolnego typu, który spełnia ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="90c70-120">C# requires code in a class to be written in such a way that it will work with any type that satisfies the constraints.</span></span> <span data-ttu-id="90c70-121">Na przykład w języku C++ istnieje możliwość pisania funkcji, która używa operatorów arytmetycznych `+` i `-` obiektów parametr typu, która powoduje wygenerowanie błędu w momencie konkretyzacji szablonu o typie, który nie obsługuje tych operatory.</span><span class="sxs-lookup"><span data-stu-id="90c70-121">For example, in C++ it is possible to write a function that uses the arithmetic operators `+` and `-` on objects of the type parameter, which will produce an error at the time of instantiation of the template with a type that does not support these operators.</span></span> <span data-ttu-id="90c70-122">C# nie zezwalają na to; konstrukcji języka tylko dozwolone są te, które wynikają z ograniczeń.</span><span class="sxs-lookup"><span data-stu-id="90c70-122">C# disallows this; the only language constructs allowed are those that can be deduced from the constraints.</span></span>
