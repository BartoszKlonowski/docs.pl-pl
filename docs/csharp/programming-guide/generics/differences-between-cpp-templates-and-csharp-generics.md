<span data-ttu-id="0823e-103">C# Generics i C++ szablony są zarówno funkcje języka, które zapewniają obsługę typów sparametryzowanych.</span><span class="sxs-lookup"><span data-stu-id="0823e-103">C# Generics and C++ templates are both language features that provide support for parameterized types.</span></span> <span data-ttu-id="0823e-104">Istnieje jednak wiele różnic między nimi.</span><span class="sxs-lookup"><span data-stu-id="0823e-104">However, there are many differences between the two.</span></span> <span data-ttu-id="0823e-105">Na poziomie składni generycznych C# są prostsze podejście do typów sparametryzowanych bez złożoności szablonów C++.</span><span class="sxs-lookup"><span data-stu-id="0823e-105">At the syntax level, C# generics are a simpler approach to parameterized types without the complexity of C++ templates.</span></span> <span data-ttu-id="0823e-106">Ponadto C# nie próbuje zapewnić wszystkie funkcje, które zapewniają szablony Języka C++.</span><span class="sxs-lookup"><span data-stu-id="0823e-106">In addition, C# does not attempt to provide all of the functionality that C++ templates provide.</span></span> <span data-ttu-id="0823e-107">Na poziomie implementacji podstawową różnicą jest to, że substytucje typów ogólnych języka C# są wykonywane w czasie wykonywania, a informacje o typie ogólnym są zachowywane dla obiektów wystąpienia.</span><span class="sxs-lookup"><span data-stu-id="0823e-107">At the implementation level, the primary difference is that C# generic type substitutions are performed at runtime and generic type information is thereby preserved for instantiated objects.</span></span> <span data-ttu-id="0823e-108">Aby uzyskać więcej informacji, zobacz [Ogólne w czasie wykonywania](./generics-in-the-run-time.md).</span><span class="sxs-lookup"><span data-stu-id="0823e-108">For more information, see [Generics in the Run Time](./generics-in-the-run-time.md).</span></span>  
  
 <span data-ttu-id="0823e-109">Poniżej przedstawiono kluczowe różnice między szablonami C# Generics i C++:</span><span class="sxs-lookup"><span data-stu-id="0823e-109">The following are the key differences between C# Generics and C++ templates:</span></span>  
  
- <span data-ttu-id="0823e-110">Generykczna języka C# nie zapewniają taką samą elastyczność jak szablony Języka C++.</span><span class="sxs-lookup"><span data-stu-id="0823e-110">C# generics do not provide the same amount of flexibility as C++ templates.</span></span> <span data-ttu-id="0823e-111">Na przykład nie jest możliwe wywołanie operatorów arytmetycznych w klasie ogólnej Języka C#, chociaż można wywołać operatory zdefiniowane przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="0823e-111">For example, it is not possible to call arithmetic operators in a C# generic class, although it is possible to call user defined operators.</span></span>  
  
- <span data-ttu-id="0823e-112">C# nie zezwala na parametry szablonu `template C<int i> {}`nietypu, takie jak .</span><span class="sxs-lookup"><span data-stu-id="0823e-112">C# does not allow non-type template parameters, such as `template C<int i> {}`.</span></span>  
  
- <span data-ttu-id="0823e-113">C# nie obsługuje jawnej specjalizacji; oznacza to, że implementacja niestandardowa szablonu dla określonego typu.</span><span class="sxs-lookup"><span data-stu-id="0823e-113">C# does not support explicit specialization; that is, a custom implementation of a template for a specific type.</span></span>  
  
- <span data-ttu-id="0823e-114">C# nie obsługuje częściowej specjalizacji: implementacja niestandardowa dla podzbioru argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="0823e-114">C# does not support partial specialization: a custom implementation for a subset of the type arguments.</span></span>  
  
- <span data-ttu-id="0823e-115">C# nie zezwala na parametr typu, który ma być używany jako klasa podstawowa dla typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="0823e-115">C# does not allow the type parameter to be used as the base class for the generic type.</span></span>  
  
- <span data-ttu-id="0823e-116">C# nie zezwala na parametry typu mają typy domyślne.</span><span class="sxs-lookup"><span data-stu-id="0823e-116">C# does not allow type parameters to have default types.</span></span>  
  
- <span data-ttu-id="0823e-117">W języku C#, parametr typu ogólnego sam nie może być rodzajowy, chociaż typy konstruowane mogą służyć jako generycznych.</span><span class="sxs-lookup"><span data-stu-id="0823e-117">In C#, a generic type parameter cannot itself be a generic, although constructed types can be used as generics.</span></span> <span data-ttu-id="0823e-118">C++ zezwala na parametry szablonu.</span><span class="sxs-lookup"><span data-stu-id="0823e-118">C++ does allow template parameters.</span></span>  
  
- <span data-ttu-id="0823e-119">C++ umożliwia kod, który może nie być prawidłowy dla wszystkich parametrów typu w szablonie, który jest następnie sprawdzany dla określonego typu używanego jako parametr typu.</span><span class="sxs-lookup"><span data-stu-id="0823e-119">C++ allows code that might not be valid for all type parameters in the template, which is then checked for the specific type used as the type parameter.</span></span> <span data-ttu-id="0823e-120">C# wymaga kodu w klasie, które mają być napisane w taki sposób, że będzie działać z dowolnego typu, który spełnia ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="0823e-120">C# requires code in a class to be written in such a way that it will work with any type that satisfies the constraints.</span></span> <span data-ttu-id="0823e-121">Na przykład w języku C++ można napisać funkcję, która `+` używa `-` operatorów arytmetycznych i obiektów parametru typu, który spowoduje błąd w momencie wystąpienia szablonu z typem, który nie obsługuje tych operatorów.</span><span class="sxs-lookup"><span data-stu-id="0823e-121">For example, in C++ it is possible to write a function that uses the arithmetic operators `+` and `-` on objects of the type parameter, which will produce an error at the time of instantiation of the template with a type that does not support these operators.</span></span> <span data-ttu-id="0823e-122">C# nie zezwala na to; tylko konstrukcje języka dozwolone są te, które można wywnioskować z ograniczeń.</span><span class="sxs-lookup"><span data-stu-id="0823e-122">C# disallows this; the only language constructs allowed are those that can be deduced from the constraints.</span></span>
