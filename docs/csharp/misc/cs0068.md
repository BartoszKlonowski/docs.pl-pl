 <span data-ttu-id="46e81-105">Zdarzenie w interfejsie nie może mieć inicjatora.</span><span class="sxs-lookup"><span data-stu-id="46e81-105">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="46e81-106">Aby uzyskać więcej informacji, zobacz [interfejsy](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="46e81-106">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="46e81-107">Poniższy przykład generuje CS0068:</span><span class="sxs-lookup"><span data-stu-id="46e81-107">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
