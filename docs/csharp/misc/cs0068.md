 <span data-ttu-id="ef48b-104">Zdarzenie w interfejsie nie może mieć inicjatora.</span><span class="sxs-lookup"><span data-stu-id="ef48b-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="ef48b-105">Więcej informacji znajdziesz w artykule [Interfejsy](../../csharp/programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="ef48b-105">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="ef48b-106">Poniższy przykład spowoduje wygenerowanie CS0068:</span><span class="sxs-lookup"><span data-stu-id="ef48b-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
