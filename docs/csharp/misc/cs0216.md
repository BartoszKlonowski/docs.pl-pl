# <a name="compiler-error-cs0216"></a><span data-ttu-id="2b6b6-103">Błąd kompilatora CS0216</span><span class="sxs-lookup"><span data-stu-id="2b6b6-103">Compiler Error CS0216</span></span>

<span data-ttu-id="2b6b6-104">Operator "operator" wymaga, aby można było zdefiniować pasujący operator "missing_operator"</span><span class="sxs-lookup"><span data-stu-id="2b6b6-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="2b6b6-105">Zdefiniowany przez użytkownika operator [true](../language-reference/operators/true-false-operators.md) wymaga zdefiniowanego przez użytkownika operatora [false](../language-reference/operators/true-false-operators.md) i na odwrót.</span><span class="sxs-lookup"><span data-stu-id="2b6b6-105">A user-defined [true](../language-reference/operators/true-false-operators.md) operator requires a user-defined [false](../language-reference/operators/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="2b6b6-106">Poniższy przykład generuje CS0216:</span><span class="sxs-lookup"><span data-stu-id="2b6b6-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
