# <a name="compiler-error-cs0548"></a>CS0548 błąd kompilatora
"właściwość": właściwość lub indeksator musi mieć co najmniej jedną metodę dostępu  
  
 Właściwość musi mieć co najmniej jedną metodę dostępu (get lub set).  
  
 Aby uzyskać więcej informacji, zobacz i [przy użyciu właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0548.  
  
```csharp  
// CS0548.cs  
// compile with: /target:library  
public class b  
{  
   public int MyProp {}   // CS0548  
  
   public int MyProp2   // OK  
   {  
      get  
      {  
         return 0;  
      }  
      set {}  
   }  
}  
```
