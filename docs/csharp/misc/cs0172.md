# <a name="compiler-error-cs0172"></a><span data-ttu-id="99d42-102">CS0172 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="99d42-102">Compiler Error CS0172</span></span>
<span data-ttu-id="99d42-103">Nie można określić typu wyrażenia warunkowego, ponieważ "type1" i "type2" niejawnie skonwertować wzajemnie</span><span class="sxs-lookup"><span data-stu-id="99d42-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="99d42-104">W instrukcji warunkowej, musi mieć możliwość konwersji typów po obu stronach `:` operatora.</span><span class="sxs-lookup"><span data-stu-id="99d42-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="99d42-105">Ponadto nie może być procedury konwersji wzajemne; wystarczy tylko jeden konwersji.</span><span class="sxs-lookup"><span data-stu-id="99d42-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="99d42-106">Aby uzyskać więcej informacji, zobacz [operatory konwersji](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="99d42-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="99d42-107">Poniższy przykład generuje CS0172:</span><span class="sxs-lookup"><span data-stu-id="99d42-107">The following sample generates CS0172:</span></span>  
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
