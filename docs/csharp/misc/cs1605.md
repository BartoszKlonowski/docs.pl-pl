# <a name="compiler-error-cs1605"></a>CS1605 błąd kompilatora
Nie można przekazać "var" jako wartości ref lub out argumentu, ponieważ jest tylko do odczytu  
  
 Przekazano zmienną jako [ref](../../csharp/language-reference/keywords/ref.md) lub [limit](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr powinien być modyfikowany w nazwie metody. W związku z tym nie jest możliwe do przekazywania tylko do odczytu jako parametru `ref` lub `out`.
