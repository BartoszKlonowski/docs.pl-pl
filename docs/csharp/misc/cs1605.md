# <a name="compiler-error-cs1605"></a>Błąd kompilatora CS1605

Nie można przekazać elementu "var" jako argumentu "ref" lub "out", ponieważ jest on tylko do odczytu  
  
 Zmienna przeniesiona jako parametr [ref](../language-reference/keywords/ref.md) lub [out](../language-reference/keywords/out-parameter-modifier.md) powinna zostać zmodyfikowana w wywołanej metodzie. W związku z tym nie jest możliwe przekazanie parametru tylko do odczytu jako `ref` lub `out` .
