# <a name="compiler-error-cs1605"></a>CS1605 błąd kompilatora
Nie można przekazać "var" jako argumentu ref lub out argumentu, ponieważ jest on tylko do odczytu  
  
 Zmienna jest przekazywana jako [ref](../../csharp/language-reference/keywords/ref.md) lub [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr powinien być modyfikowane w metodzie wywoływanej. W związku z tym, nie jest możliwe do przekazania jako parametru tylko do odczytu `ref` lub `out`.
