# <a name="compiler-error-cs2020"></a>CS2020 błąd kompilatora
Tylko pierwszy zestaw plików wejściowych można zbudować obiekt docelowy inny niż "module"  
  
 W zestawieniu uniwersalny, pierwszy plik wyjściowy muszą zostać skompilowane przy użyciu [/target: exe](../../csharp/language-reference/compiler-options/target-exe-compiler-option.md), [/target: winexe](../../csharp/language-reference/compiler-options/target-winexe-compiler-option.md), lub [/target: Library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md). Wszystkie pliki wyjściowe kolejnych muszą zostać skompilowane przy użyciu [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md).
