# <a name="compiler-error-cs1639"></a>CS1639 błąd kompilatora
Podpis klasy otoki coclass zarządzanego "podpisu" dla interfejsu "interface" nie jest prawidłową sygnaturą nazwy klasy  
  
 Ten błąd występuje, gdy metadane dla interfejsu jest nieprawidłowa. Nigdy nie powinno to nastąpić, jeśli interfejs został wygenerowany za pomocą języka C# lub obsługiwanych języków .NET. Sprawdź u dostawcy interfejsu lub sprawdź, czy zestawu interfejs został pomyślnie wygenerowany.
