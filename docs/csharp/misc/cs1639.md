# <a name="compiler-error-cs1639"></a>Błąd kompilatora CS1639

Sygnatura "Signature" zarządzanej klasy otoki coclass dla interfejsu "Interface" nie jest prawidłową sygnaturą nazwy klasy  
  
 Ten błąd występuje, jeśli metadane interfejsu są nieprawidłowe. Ta wartość nigdy nie powinna wystąpić, jeśli interfejs został wygenerowany przy użyciu języka C# lub obsługiwanego języka .NET. Należy skontaktować się z dostawcą interfejsu lub sprawdzić, czy zestaw interfejsów został wygenerowany pomyślnie.
