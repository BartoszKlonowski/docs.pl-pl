# <a name="compiler-error-cs1639"></a>CS1639 błąd kompilatora
Sygnatura klasy otoki coclass zarządzanego "podpis" dla interfejsu "interface" nie jest prawidłową sygnaturą nazwy klasy  
  
 Ten błąd występuje, jeśli metadane dla interfejsu są nieprawidłowe. To powinno nigdy nie nastąpić, jeśli interfejs został wygenerowany za pomocą C# lub obsługiwanego języka .NET. Skontaktuj się z dostawcą interfejsu lub upewnij się, zestaw interfejs został pomyślnie wygenerowany.
