# <a name="compiler-error-cs1922"></a><span data-ttu-id="a6364-102">Błąd kompilatora CS1922</span><span class="sxs-lookup"><span data-stu-id="a6364-102">Compiler Error CS1922</span></span>

<span data-ttu-id="a6364-103">Inicjator kolekcji wymaga, aby typ "Type" zaimplementował element System. Collections. IEnumerable.</span><span class="sxs-lookup"><span data-stu-id="a6364-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="a6364-104">Aby można było użyć inicjatora kolekcji z typem, typ musi implementować `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="a6364-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="a6364-105">Ten błąd może wystąpić w przypadku przypadkowego użycia składni inicjatora kolekcji, jeśli zamierzasz użyć inicjatora obiektów.</span><span class="sxs-lookup"><span data-stu-id="a6364-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="a6364-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a6364-106">To correct this error</span></span>

- <span data-ttu-id="a6364-107">Jeśli typ nie reprezentuje kolekcji, użyj składni inicjatora obiektów zamiast składni inicjatora kolekcji.</span><span class="sxs-lookup"><span data-stu-id="a6364-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="a6364-108">Jeśli typ reprezentuje kolekcję, zmodyfikuj ją do implementacji `IEnumerable` przed użyciem inicjatorów kolekcji w celu zainicjowania obiektów tego typu.</span><span class="sxs-lookup"><span data-stu-id="a6364-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>

- <span data-ttu-id="a6364-109">Jeśli typ reprezentuje kolekcję i nie masz dostępu do kodu źródłowego, po prostu zainicjuj jego elementy przy użyciu konstruktorów klas lub innych metod inicjacji.</span><span class="sxs-lookup"><span data-stu-id="a6364-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="a6364-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="a6364-110">Example</span></span>

<span data-ttu-id="a6364-111">Poniższy kod generuje CS1922:</span><span class="sxs-lookup"><span data-stu-id="a6364-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
