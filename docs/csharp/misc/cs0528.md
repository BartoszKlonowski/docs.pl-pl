# <a name="compiler-error-cs0528"></a>CS0528 błąd kompilatora
"interface" znajduje się już na liście interfejsów  
  
 Lista dziedziczenia interfejsu zawiera zduplikowany. [Interfejsu](../../csharp/language-reference/keywords/interface.md) można określić tylko jeden raz na liście dziedziczenia.  
  
 Poniższy przykład spowoduje wygenerowanie CS0528:  
  
```csharp  
// CS0528.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public class b : a, a   // CS0528  
   {  
      public void Main()  
      {  
      }  
   }  
}  
```
