# <a name="compiler-error-cs1504"></a>CS1504 błąd kompilatora
Nie można otworzyć pliku źródłowego "file" ("przyczyny")  
  
 Nie można otworzyć pliku źródłowego lub odczytywane przez kompilator. Plik może być zablokowany przez inną aplikację lub mogą występować inne problemy z systemem operacyjnym. Komunikat zawiera system operacyjny Przyczyna Dlaczego kompilator nie może otworzyć lub odczytać pliku.
