# <a name="compiler-error-cs1504"></a>Błąd kompilatora CS1504
Nie można otworzyć pliku źródłowego "File" (Przyczyna:)  
  
 Nie można otworzyć pliku źródłowego lub odczytać go przez kompilator. Plik może być zablokowany przez inną aplikację lub Wystąpił inny problem z systemem operacyjnym. Komunikat zawiera powód systemu operacyjnego, dla którego kompilator nie może otworzyć lub odczytać pliku.
