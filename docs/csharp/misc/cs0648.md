# <a name="compiler-error-cs0648"></a>CS0648 błąd kompilatora
"type" jest typem, nie jest obsługiwany przez język  
  
 Metadane, który został wygenerowany w innym języku, prawdopodobnie C++, zawiera typ, który nie został oznaczony jako zarządzany. Informacje dotyczące tego typu znajduje się w metadanych, ale typ nie jest dostępna dla programów napisanych w C#.
