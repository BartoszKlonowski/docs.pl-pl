# <a name="compiler-error-cs0648"></a>CS0648 błąd kompilatora
"type" jest typem obsługiwanym przez język  
  
 Metadane, który został wygenerowany z innego języka, prawdopodobnie C++ zawiera typ, który nie został oznaczony jako zarządzany. Informacje o tym typie jest zawarte w metadanych, ale typ nie jest dostępny dla programów napisanych w języku C#.
