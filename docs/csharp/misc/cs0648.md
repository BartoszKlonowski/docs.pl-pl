# <a name="compiler-error-cs0648"></a>Błąd kompilatora CS0648

Typ "Type" jest nieobsługiwany przez język  
  
 Metadane, które zostały wygenerowane z innego języka, prawdopodobnie C++, zawierały typ, który nie został oznaczony jako zarządzany. Informacje o tym typie są zawarte w metadanych, ale typ nie jest dostępny dla programów pisanych w języku C#.
