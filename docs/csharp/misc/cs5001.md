 <span data-ttu-id="d6382-104">Ten błąd występuje, gdy nie static [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody o prawidłowej sygnaturze znajduje się w kodzie, który spowoduje utworzenie pliku wykonywalnego.</span><span class="sxs-lookup"><span data-stu-id="d6382-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="d6382-105">Ten błąd występuje także, czy punkt wejścia funkcji `Main`, jest zdefiniowana z niewłaściwego case, takich jak małe `main`.</span><span class="sxs-lookup"><span data-stu-id="d6382-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>
