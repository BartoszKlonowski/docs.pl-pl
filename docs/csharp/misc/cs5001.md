# <a name="compiler-error-cs5001"></a>CS5001 błąd kompilatora

Program "program" nie zawiera statycznej metody "Main" odpowiedniej jako punkt wejścia  
  
Ten błąd występuje, gdy brak statycznego `Main` metody o poprawnej sygnatury znajduje się w kodzie, który tworzy plik wykonywalny. Ten błąd występuje także w przypadku funkcji, punkt wejścia `Main`, jest zdefiniowana z przypadkiem niewłaściwy, takich jak małe `main`. Aby uzyskać informacje o regułach, które są stosowane do `Main` metody, zobacz [Main() i argumenty wiersza polecenia](../programming-guide/main-and-command-args/index.md).

Jeśli `Main` metoda ma `async` modyfikator, upewnij się, że [wybranej wersji języka C#](../language-reference/configure-language-version.md) jest w wersji 7.1 lub nowszej.

## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS5001:  
  
```csharp  
// CS5001.cs  
// CS5001 expected  
public class a  
{  
   // Uncomment the following line to resolve.  
   // static void Main() {}  
}  
```
