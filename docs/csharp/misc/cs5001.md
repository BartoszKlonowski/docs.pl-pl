# <a name="compiler-error-cs5001"></a><span data-ttu-id="25b22-102">CS5001 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="25b22-102">Compiler Error CS5001</span></span>
<span data-ttu-id="25b22-103">Program "program" nie zawiera statycznej metody "Main" odpowiedniej jako punkt wejścia</span><span class="sxs-lookup"><span data-stu-id="25b22-103">Program 'program' does not contain a static 'Main' method suitable for an entry point</span></span>  
  
 <span data-ttu-id="25b22-104">Ten błąd występuje, gdy nie static [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody o prawidłowej sygnaturze znajduje się w kodzie, który spowoduje utworzenie pliku wykonywalnego.</span><span class="sxs-lookup"><span data-stu-id="25b22-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="25b22-105">Ten błąd występuje także, czy punkt wejścia funkcji `Main`, jest zdefiniowana z niewłaściwego case, takich jak małe `main`.</span><span class="sxs-lookup"><span data-stu-id="25b22-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="25b22-106">`Main` musi być deklarowane jako statyczne i zwracać [void](../../csharp/language-reference/keywords/void.md) lub [int](../../csharp/language-reference/keywords/int.md), i musi mieć albo bez parametrów lub też jeden parametr typu `string[]`.</span><span class="sxs-lookup"><span data-stu-id="25b22-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
