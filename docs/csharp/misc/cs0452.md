# <a name="compiler-error-cs0452"></a><span data-ttu-id="016c1-102">CS0452 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="016c1-102">Compiler Error CS0452</span></span>
<span data-ttu-id="016c1-103">Typ 'Nazwa typu' musi należeć typu odwołania aby można było go użyć jako parametru "Nazwa parametru" ogólnego typu lub metody "identyfikator rodzajowy"</span><span class="sxs-lookup"><span data-stu-id="016c1-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="016c1-104">Ten błąd występuje, gdy przekazać typ wartości takich jak `struct` lub `int` jako parametr typu ogólnego lub metody, która ma ograniczenie typu odwołania.</span><span class="sxs-lookup"><span data-stu-id="016c1-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="016c1-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="016c1-105">Example</span></span>  
 <span data-ttu-id="016c1-106">Poniższy kod generuje błąd CS0452.</span><span class="sxs-lookup"><span data-stu-id="016c1-106">The following code generates error CS0452.</span></span>

```csharp  
// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452

```
