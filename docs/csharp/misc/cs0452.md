# <a name="compiler-error-cs0452"></a><span data-ttu-id="45ae3-102">CS0452 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="45ae3-102">Compiler Error CS0452</span></span>
<span data-ttu-id="45ae3-103">Typ "Nazwa typu" musi być typem odwołania do użycia jako parametr "Nazwa parametru" w ogólnym typie lub metodzie "identyfikator rodzajowy"</span><span class="sxs-lookup"><span data-stu-id="45ae3-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="45ae3-104">Ten błąd występuje, gdy przekazujesz takie jak typ wartości `struct` lub `int` jako parametr typu ogólnego lub metody, która ma ograniczenie typu odwołania.</span><span class="sxs-lookup"><span data-stu-id="45ae3-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="45ae3-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="45ae3-105">Example</span></span>  
 <span data-ttu-id="45ae3-106">Poniższy kod generuje błąd CS0452.</span><span class="sxs-lookup"><span data-stu-id="45ae3-106">The following code generates error CS0452.</span></span>  
  
```csharp  
// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452  
public class Derived2<S> : BaseClass<S> where S : struct { } // CS0452  
```
