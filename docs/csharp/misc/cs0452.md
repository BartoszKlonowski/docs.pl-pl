<span data-ttu-id="fe014-103">Typ "Nazwa typu" musi być typem odwołania do użycia jako parametr "Nazwa parametru" w ogólnym typie lub metodzie "identyfikator rodzajowy"</span><span class="sxs-lookup"><span data-stu-id="fe014-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>
