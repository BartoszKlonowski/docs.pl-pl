# <a name="compiler-error-cs0452"></a><span data-ttu-id="f473c-103">Błąd kompilatora CS0452</span><span class="sxs-lookup"><span data-stu-id="f473c-103">Compiler Error CS0452</span></span>
<span data-ttu-id="f473c-104">Typ "type name" musi być typem referencyjnym, aby można go było użyć jako parametru "Nazwa parametru" w typie ogólnym lub metodzie "Generic"</span><span class="sxs-lookup"><span data-stu-id="f473c-104">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="f473c-105">Ten błąd występuje, gdy przekazujesz typ wartości, taki jak `struct` lub `int` jako parametr do typu ogólnego lub metody, która ma ograniczenie typu odwołania.</span><span class="sxs-lookup"><span data-stu-id="f473c-105">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f473c-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="f473c-106">Example</span></span>  
 <span data-ttu-id="f473c-107">Poniższy kod generuje błąd CS0452.</span><span class="sxs-lookup"><span data-stu-id="f473c-107">The following code generates error CS0452.</span></span>

```csharp  
// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452  
public class Derived2<S> : BaseClass<S> where S : struct { } // CS0452  
```
