## <a name="example"></a><span data-ttu-id="70219-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="70219-110">Example</span></span>  
 <span data-ttu-id="70219-111">Poniższy przykład przedstawia sposób tworzenia CS1934:</span><span class="sxs-lookup"><span data-stu-id="70219-111">The following example shows one way to produce CS1934:</span></span>

static class Test  
{  
    public static void Main()  
    {
