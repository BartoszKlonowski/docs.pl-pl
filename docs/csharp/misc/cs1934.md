## <a name="example"></a><span data-ttu-id="a2996-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="a2996-110">Example</span></span>

```csharp

using System.Collections;  
static class Test  
{  
    public static void Main()  
    {  
        var list = new ArrayList { 0, 1, 2, 3, 4, 5 };  
        var q = from x in list // CS1934

    }
```  
