## <a name="example"></a><span data-ttu-id="f8ce0-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="f8ce0-110">Example</span></span>  
 <span data-ttu-id="f8ce0-111">W poniższym przykładzie pokazano jeden ze sposobów tworzenia CS1934:</span><span class="sxs-lookup"><span data-stu-id="f8ce0-111">The following example shows one way to produce CS1934:</span></span>  
  
```csharp

using System.Collections;  
static class Test  
{  
    public static void Main()  
    {  
        var list = new ArrayList { 0, 1, 2, 3, 4, 5 };  
        var q = from x in list // CS1934

    }  
}  
```
