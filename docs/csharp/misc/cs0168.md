# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="62840-102">Kompilator CS0168 ostrzegawcze (poziom 3)</span><span class="sxs-lookup"><span data-stu-id="62840-102">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="62840-103">Zmienna "var" jest zadeklarowana, ale nigdy używane</span><span class="sxs-lookup"><span data-stu-id="62840-103">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="62840-104">Kompilator generuje ostrzeżenie trzech poziom, gdy zadeklarować zmiennej, ale nie należy używać.</span><span class="sxs-lookup"><span data-stu-id="62840-104">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="62840-105">Poniższy przykład generuje ostrzeżenie CS0168:</span><span class="sxs-lookup"><span data-stu-id="62840-105">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
