# <a name="compiler-warning-level-3-cs0168"></a>Kompilatora (poziom 3) ostrzeżenie CS0168
Zmienna "var" jest zadeklarowana, ale nigdy używana

Kompilator generuje ostrzeżenia poziomu 3, gdy zadeklarować zmienną, ale nie należy jej używać.
  
Poniższy przykład generuje ostrzeżenie CS0168:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
