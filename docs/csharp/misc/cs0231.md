<span data-ttu-id="57c8b-103">Parametr params musi być ostatnim parametrem formalnej listy parametrów.</span><span class="sxs-lookup"><span data-stu-id="57c8b-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="57c8b-104">[Params](../../csharp/language-reference/keywords/params.md) parametr obsługuje zmiennej liczby argumentów i musi być późniejsza niż inne parametry.</span><span class="sxs-lookup"><span data-stu-id="57c8b-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="57c8b-105">Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="57c8b-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="57c8b-106">Poniższy przykład generuje CS0231:</span><span class="sxs-lookup"><span data-stu-id="57c8b-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```
