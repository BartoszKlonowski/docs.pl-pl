# <a name="compiler-error-cs0231"></a>CS0231 błąd kompilatora
Parametr params musi być ostatnim parametrem formalnej listy parametrów.  
  
 [Params](../../csharp/language-reference/keywords/params.md) parametr obsługuje zmiennej liczby argumentów i musi być późniejsza niż wszystkie inne parametry. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0231:  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```
