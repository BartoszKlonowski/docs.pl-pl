# <a name="compiler-error-cs0180"></a>CS0180 błąd kompilatora
'składowa' nie może być zewnętrzny i abstrakcyjny  
  
 [Abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i [extern](../../csharp/language-reference/keywords/extern.md) słowa kluczowe są wzajemnie się wykluczają. `extern` — Słowo kluczowe oznacza, że składowa jest zdefiniowana poza plikiem i **abstrakcyjne** oznacza, że implementacja znajduje się w klasie pochodnej. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0180:  
  
```csharp  
// CS0180.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public extern abstract int Foo(int a);   // CS0180  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
