# <a name="compiler-error-cs0729"></a>Błąd kompilatora CS0729
Typ "Type" jest zdefiniowany w tym zestawie, ale dla niego jest określony typ przesyłania dalej  
  
 Nie można użyć usługi przesyłania dalej dla typu zdefiniowanego w tym samym zestawie.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0729.  
  
```csharp  
// CS0729.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:TypeForwardedTo(typeof(TestClass))]   // CS0729  
class TestClass {}  
```
