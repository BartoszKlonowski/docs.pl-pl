# <a name="compiler-error-cs1041"></a>CS1041 błąd kompilatora
Oczekiwano identyfikatora, "słowo kluczowe" jest słowem kluczowym  
  
 Słowo zastrzeżone dla C# odnaleziono języka, których Oczekiwano identyfikatora. Zastąp [— słowo kluczowe](../../csharp/language-reference/keywords/index.md) z identyfikatorem określony przez użytkownika.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1041:  
  
```csharp  
// CS1041a.cs  
class MyClass  
{  
    public void f(int long)   // CS1041  
    // Try the following instead:  
    // public void f(int i)  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```  
  
## <a name="example"></a>Przykład  
 Podczas importowania z innego języka programowania, który ma ten sam zestaw słów zastrzeżonych, można zmodyfikować zarezerwowanych identyfikatora z @ prefiksu, jak pokazano w następującym przykładzie.  
  
 Identyfikator z `@` prefiks jest nazywany identyfikator dosłownego wyrażenia.  
  
```csharp  
// CS1041b.cs  
class MyClass  
{  
    public void f(int long)   // CS1041  
    // Try the following instead:  
    // public void f(int @long)  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
