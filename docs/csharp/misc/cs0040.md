# <a name="compiler-error-cs0040"></a>CS0040 błąd kompilatora
Nieoczekiwany błąd podczas tworzenia pliku informacji o debugowaniu — "Przyczyna"  
  
 Ten błąd może wystąpić w przypadku korzystania z [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) — opcja kompilatora i wskazuje, że kompilator nie mógł zapisać do pliku .pdb. Możliwe rozwiązania tego błędu mogą być ponowna instalacja programu Visual Studio, zapewniając, że kompilator ma dostęp do zapisu pliku lub katalogu lub nie kompilowania przy użyciu opcji/Debug.
