# <a name="compiler-error-cs0040"></a>Błąd kompilatora CS0040
Nieoczekiwany błąd podczas tworzenia pliku informacji o debugowaniu — "Przyczyna"  
  
 Ten błąd może wystąpić podczas korzystania z opcji kompilatora [debugowania](../language-reference/compiler-options/debug-compiler-option.md) i wskazuje, że kompilator nie może zapisać do pliku. pdb. Możliwe rozwiązania tego błędu obejmują ponowną instalację programu Visual Studio, dzięki czemu kompilator ma dostęp do zapisu do pliku lub katalogu lub nie jest kompilowany przy użyciu opcji/Debug.
