# <a name="compiler-error-cs0040"></a>CS0040 błąd kompilatora
Nieoczekiwany błąd tworzenia pliku informacji o debugowaniu — "reason"  
  
 Ten błąd może wystąpić w przypadku korzystania z [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) — opcja kompilatora i wskazuje, że kompilator nie może zapisać do pliku .pdb. Możliwe rozwiązania tego błędu obejmują ponowne zainstalowanie programu Visual Studio, zapewniając, że kompilator ma dostęp do zapisu do pliku lub katalogu lub nie kompilowania za pomocą opcji/Debug.
