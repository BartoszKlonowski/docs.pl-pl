# <a name="compiler-error-cs0711"></a><span data-ttu-id="b893b-103">Błąd kompilatora CS0711</span><span class="sxs-lookup"><span data-stu-id="b893b-103">Compiler Error CS0711</span></span>

<span data-ttu-id="b893b-104">Klasy statyczne nie mogą zawierać destruktorów</span><span class="sxs-lookup"><span data-stu-id="b893b-104">Static classes cannot contain destructors</span></span>  
  
 <span data-ttu-id="b893b-105">Nie można utworzyć wystąpienia klasy statycznej, dlatego nie ma potrzeby stosowania konstruktorów ani destruktorów.</span><span class="sxs-lookup"><span data-stu-id="b893b-105">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="b893b-106">Aby uniknąć tego błędu, Usuń wszystkie destruktory z klas statycznych lub, jeśli naprawdę chcesz skonstruować i zniszczyć wystąpienia, ustaw klasę jako niestatyczną.</span><span class="sxs-lookup"><span data-stu-id="b893b-106">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>  
  
 <span data-ttu-id="b893b-107">Poniższy przykład generuje CS0711:</span><span class="sxs-lookup"><span data-stu-id="b893b-107">The following sample generates CS0711:</span></span>  
  
```csharp  
// CS0711.cs  
public static class C  
{  
   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
