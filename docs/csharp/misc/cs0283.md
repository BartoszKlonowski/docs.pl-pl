# <a name="compiler-error-cs0283"></a>Błąd kompilatora CS0283
Typ "Type" nie może być zadeklarowany jako const  
  
Typ określony w deklaracji [stałej](../language-reference/keywords/const.md) musi mieć wartość `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, typ [wyliczenia](../language-reference/builtin-types/enum.md) lub typ referencyjny, do którego przypisano wartość `null`. Każde wyrażenie stałe musi zwracać wartość typu docelowego lub typu, który jest niejawnie konwertowany na typ docelowy.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0283.  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
