# <a name="compiler-error-cs1542"></a>CS1542 błąd kompilatora
Nie można dodać 'dll' do tego zestawu, ponieważ już jest zestaw; Użyj "/ R" zamiast opcji  
  
 Plik który został wywołany [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md) — opcja kompilatora nie został zbudowany przy użyciu [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md); użyj [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) można odwoływać się do pliku w tej kompilacji.
