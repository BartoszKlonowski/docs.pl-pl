# <a name="compiler-error-cs1542"></a>CS1542 błąd kompilatora
Nie można dodać 'dll' do tego zestawu, ponieważ jest już zestawu; Użyj "/ R" zamiast tego należy  
  
 Plik, który został wywołany [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md) — opcja kompilatora nie został skompilowany z [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md); użyj [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) na odwołanie do pliku w tej kompilacji.
