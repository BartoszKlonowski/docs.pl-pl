# <a name="compiler-error-cs1542"></a>Błąd kompilatora CS1542

nie można dodać biblioteki DLL do tego zestawu, ponieważ jest już zestawem. Zamiast tego użyj opcji "/R"  
  
 Plik, do którego odwołuje się opcja kompilatora [-addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) , nie został skompilowany z [elementem-target: module](../language-reference/compiler-options/target-module-compiler-option.md); Użyj [parametru-reference](../language-reference/compiler-options/reference-compiler-option.md) , aby odwołać się do pliku w tej kompilacji.
