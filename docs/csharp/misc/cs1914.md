## <a name="example"></a><span data-ttu-id="51874-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="51874-109">Example</span></span>  

 <span data-ttu-id="51874-110">Poniższy kod generuje CS1914, ponieważ inicjator próbuje zainicjować `TestClass.Number` pole, czyli `static` :</span><span class="sxs-lookup"><span data-stu-id="51874-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
