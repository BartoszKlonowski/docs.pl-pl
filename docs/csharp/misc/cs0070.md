# <a name="compiler-error-cs0070"></a><span data-ttu-id="85fc3-103">Błąd kompilatora CS0070</span><span class="sxs-lookup"><span data-stu-id="85fc3-103">Compiler Error CS0070</span></span>
<span data-ttu-id="85fc3-104">Zdarzenie "Event" może występować tylko po lewej stronie znaków + = lub-= (z wyjątkiem sytuacji, gdy jest używane z poziomu typu "Type")</span><span class="sxs-lookup"><span data-stu-id="85fc3-104">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="85fc3-105">Poza klasą, w której jest zdefiniowana, [zdarzenie](../language-reference/keywords/event.md) może dodawać lub odejmować tylko odwołania.</span><span class="sxs-lookup"><span data-stu-id="85fc3-105">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="85fc3-106">Aby uzyskać więcej informacji, zobacz [zdarzenia](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="85fc3-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="85fc3-107">Poniższy przykład generuje CS0070:</span><span class="sxs-lookup"><span data-stu-id="85fc3-107">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
