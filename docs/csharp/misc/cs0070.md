# <a name="compiler-error-cs0070"></a>CS0070 błąd kompilatora
Zdarzenie "event" może występować tylko po lewej stronie znaków += lub-= (z wyjątkiem sytuacji, gdy używane jest z wnętrza typu "type")  
  
 Poza klasa jest zdefiniowana, [zdarzeń](../../csharp/language-reference/keywords/event.md) tylko dodawać lub odejmować odwołania. Aby uzyskać więcej informacji, zobacz [zdarzenia](../../csharp/programming-guide/events/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0070:  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
