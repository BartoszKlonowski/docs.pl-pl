# <a name="compiler-error-cs0070"></a><span data-ttu-id="0ce47-102">CS0070 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="0ce47-102">Compiler Error CS0070</span></span>
<span data-ttu-id="0ce47-103">Zdarzenie "event" może występować tylko po lewej stronie znaków += lub-= (z wyjątkiem przypadków, gdy używane jest z wnętrza typu "type")</span><span class="sxs-lookup"><span data-stu-id="0ce47-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="0ce47-104">Poza klasa jest zdefiniowana w [zdarzenie](../../csharp/language-reference/keywords/event.md) można tylko dodać lub odejmowania odwołań.</span><span class="sxs-lookup"><span data-stu-id="0ce47-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="0ce47-105">Aby uzyskać więcej informacji, zobacz [zdarzenia](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="0ce47-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="0ce47-106">Poniższy przykład generuje CS0070:</span><span class="sxs-lookup"><span data-stu-id="0ce47-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
