# <a name="compiler-error-cs0416"></a>CS0416 błąd kompilatora
"parametr typu": argument atrybutu nie może używać parametrów typu  
  
 Parametr typu została użyta jako argument atrybutu, która nie jest dozwolona. Użyj typu nieogólnego.  
  
 Poniższy przykład spowoduje wygenerowanie CS0416:  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
