# <a name="compiler-error-cs0416"></a><span data-ttu-id="4f472-102">CS0416 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="4f472-102">Compiler Error CS0416</span></span>
<span data-ttu-id="4f472-103">"parametr typu": argument atrybutu nie może używać parametrów typu</span><span class="sxs-lookup"><span data-stu-id="4f472-103">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="4f472-104">Parametr typu została użyta jako argument atrybutu, która nie jest dozwolona.</span><span class="sxs-lookup"><span data-stu-id="4f472-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="4f472-105">Użyj typu nieogólnego.</span><span class="sxs-lookup"><span data-stu-id="4f472-105">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="4f472-106">Poniższy przykład spowoduje wygenerowanie CS0416:</span><span class="sxs-lookup"><span data-stu-id="4f472-106">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
