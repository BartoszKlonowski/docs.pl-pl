 <span data-ttu-id="ac2a8-104">Ten błąd występuje, gdy modyfikator typu parametru w metodzie anonimowej nie jest zgodny z użytym w deklaracji delegata, do którego jest stosowana Metoda.</span><span class="sxs-lookup"><span data-stu-id="ac2a8-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ac2a8-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="ac2a8-105">Example</span></span>  

 <span data-ttu-id="ac2a8-106">Poniższy przykład generuje CS1677:</span><span class="sxs-lookup"><span data-stu-id="ac2a8-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
