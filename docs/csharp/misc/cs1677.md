# <a name="compiler-error-cs1677"></a>CS1677 błąd kompilatora
Parametr 'Liczba' nie powinien być deklarowany za pomocą słowa kluczowego "— słowo kluczowe"  
  
 Ten błąd występuje, gdy modyfikator typu parametru w metodę anonimową nie pasuje do użyty w deklaracji delegata, do którego jest rzutowanie metody.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1677:  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
