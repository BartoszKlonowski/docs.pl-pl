 <span data-ttu-id="9a3eb-104">Ten błąd występuje, gdy modyfikator parametru typu w metodzie anonimowej niezgodne używany w deklaracji obiektu delegowanego, do której są rzutowanie metody.</span><span class="sxs-lookup"><span data-stu-id="9a3eb-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9a3eb-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="9a3eb-105">Example</span></span>  
 <span data-ttu-id="9a3eb-106">Poniższy przykład generuje CS1677:</span><span class="sxs-lookup"><span data-stu-id="9a3eb-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs

class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
