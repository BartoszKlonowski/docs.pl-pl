# <a name="compiler-warning-level-1-cs1707"></a>Kompilator CS1707 ostrzegawcze (poziom 1)
Delegat "element DelegateName" powiązany z "MethodName1" zamiast "MethodName2" z powodu nowych reguł języka  
  
 C# 2.0 implementuje nowych zasad dla powiązania delegata do metody. Dodatkowe informacje jest uznawany za który nie został przeglądał w przeszłości. To ostrzeżenie oznacza, że obiekt delegowany teraz jest powiązany z innego przeciążenia metody, nie był wcześniej powiązany z. Możesz sprawdzić, czy delegat naprawdę powinny być powiązane z "MethodName1" zamiast "MethodName2".  
  
 Opis sposobu kompilator określa powiązać obiektu delegowanego z metody, zobacz [przy użyciu wariancji w Delegatach](http://msdn.microsoft.com/library/e6acad03-93e0-4efb-a158-8696d5eb4ecf).
