# <a name="compiler-warning-level-1-cs1707"></a>Kompilatora (poziom 1) ostrzeżenie CS1707
Delegat "element"DelegateName powiązany z "MethodName1" zamiast "MethodName2" z powodu nowych reguł języka  
  
 W wersji 2.0 C# implementuje nowe reguły dla powiązania obiekt delegowany do metody. Dodatkowe informacje jest uznawany za który nie został przyjrzano w przeszłości. To ostrzeżenie wskazuje, że delegat jest teraz powiązany z innego przeciążenia metody, nie był wcześniej powiązany z. Możesz sprawdzić, czy delegata naprawdę ma zostać powiązana z "MethodName1" zamiast "MethodName2".  
  
 Aby uzyskać opis sposób kompilator Określa, którą metodę można powiązać z delegatem, zobacz [przy użyciu wariancje w Delegatach](https://msdn.microsoft.com/library/e6acad03-93e0-4efb-a158-8696d5eb4ecf).
