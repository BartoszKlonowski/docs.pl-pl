## <a name="example"></a><span data-ttu-id="995e9-112">Przykład</span><span class="sxs-lookup"><span data-stu-id="995e9-112">Example</span></span>  

 <span data-ttu-id="995e9-113">Poniższy kod generuje CS0844:</span><span class="sxs-lookup"><span data-stu-id="995e9-113">The following code generates CS0844:</span></span>  
  
```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="995e9-114">Popraw błąd przy użyciu, `this.num` Aby odwołać się do pola class</span><span class="sxs-lookup"><span data-stu-id="995e9-114">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

        {  
            num = 5; // Error fixed.

        }

        }  
    }
