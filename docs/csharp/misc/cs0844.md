## <a name="example"></a><span data-ttu-id="4a7a4-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="4a7a4-111">Example</span></span>

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="4a7a4-113">Popraw błąd przy użyciu `this.num` do odwoływania się do pola klasy</span><span class="sxs-lookup"><span data-stu-id="4a7a4-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
