# <a name="compiler-error-cs1620"></a><span data-ttu-id="d2715-102">Błąd kompilatora CS1620</span><span class="sxs-lookup"><span data-stu-id="d2715-102">Compiler Error CS1620</span></span>
<span data-ttu-id="d2715-103">Argument "number" musi być przesłany ze słowem kluczowym "Keyword"</span><span class="sxs-lookup"><span data-stu-id="d2715-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="d2715-104">Ten błąd występuje, gdy przekazujesz argument do funkcji, która przyjmuje parametr [ref](../language-reference/keywords/ref.md) lub [out](../language-reference/keywords/out-parameter-modifier.md) , i `ref` nie dołączysz słowa kluczowego or `out` w punkcie wywołania lub dołączysz nieprawidłowe słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="d2715-104">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="d2715-105">Tekst błędu wskazuje odpowiednie słowo kluczowe do użycia i argument, który spowodował błąd.</span><span class="sxs-lookup"><span data-stu-id="d2715-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="d2715-106">Poniższy przykład generuje CS1620:</span><span class="sxs-lookup"><span data-stu-id="d2715-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
