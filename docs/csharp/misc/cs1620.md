# <a name="compiler-error-cs1620"></a><span data-ttu-id="2206c-102">CS1620 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="2206c-102">Compiler Error CS1620</span></span>
<span data-ttu-id="2206c-103">Argument "number" musi być przekazywany ze słowem kluczowym "— słowo kluczowe"</span><span class="sxs-lookup"><span data-stu-id="2206c-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="2206c-104">Ten błąd występuje, jeśli przekazujesz argument do funkcji, która przyjmuje [ref](../../csharp/language-reference/keywords/ref.md) lub [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru i nie uwzględniają `ref` lub `out` — słowo kluczowe w punkcie połączenia lub możesz dołączyć Nieprawidłowe słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="2206c-104">This error occurs if you are passing an argument to a function that takes a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="2206c-105">Tekst błędu wskazuje słowo kluczowe odpowiednie do użycia i argument, który spowodował awarię.</span><span class="sxs-lookup"><span data-stu-id="2206c-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="2206c-106">Poniższy przykład spowoduje wygenerowanie CS1620:</span><span class="sxs-lookup"><span data-stu-id="2206c-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
