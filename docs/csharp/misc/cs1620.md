# <a name="compiler-error-cs1620"></a>CS1620 błąd kompilatora
Argument "number" musi być przekazywany ze słowem kluczowym "— słowo kluczowe"  
  
 Ten błąd występuje, jeśli przekazujesz argument do funkcji, która przyjmuje [ref](../../csharp/language-reference/keywords/ref.md) lub [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru i nie uwzględniają `ref` lub `out` — słowo kluczowe w punkcie połączenia lub możesz dołączyć Nieprawidłowe słowo kluczowe. Tekst błędu wskazuje słowo kluczowe odpowiednie do użycia i argument, który spowodował awarię.  
  
 Poniższy przykład spowoduje wygenerowanie CS1620:  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
