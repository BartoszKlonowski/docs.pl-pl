# <a name="compiler-error-cs1637"></a>CS1637 błąd kompilatora
Iteratory nie mogą mieć niebezpiecznych parametrów ani yield typów  
  
 Zapoznaj się z listą argumentów iteratora i typ żadnych instrukcji yield, aby zweryfikować, że nie używasz żadnych niebezpiecznych typów.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1637:  
  
```csharp  
// CS1637.cs  
// compile with: /unsafe  
using System.Collections;  
  
public unsafe class C  
{  
    public IEnumerator Iterator1(int* p)  // CS1637  
    {  
        yield return null;  
    }  
}  
```
