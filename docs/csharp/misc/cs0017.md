# <a name="compiler-error-cs0017"></a>CS0017 błąd kompilatora
Program "Nazwa pliku wyjściowego" ma więcej niż jeden punkt wejścia zdefiniowane. Skompiluj z opcją/Main, aby określić typ zawierający punkt wejścia.  
  
 Program może mieć tylko jedną [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody.  
  
 Aby rozwiązać ten problem, możesz usunąć wszystkie metody Main w kodzie, z wyjątkiem jednego, możesz też [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opcję kompilatora, aby określić które metody Main, którego chcesz użyć.  
  
 Poniższy przykład spowoduje wygenerowanie CS0017:  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
