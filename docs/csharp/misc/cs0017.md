# <a name="compiler-error-cs0017"></a><span data-ttu-id="4341a-103">Błąd kompilatora CS0017</span><span class="sxs-lookup"><span data-stu-id="4341a-103">Compiler Error CS0017</span></span>

<span data-ttu-id="4341a-104">Nazwa pliku wyjściowego programu "ma zdefiniowany więcej niż jeden punkt wejścia.</span><span class="sxs-lookup"><span data-stu-id="4341a-104">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="4341a-105">Kompiluj z/Main, aby określić typ, który zawiera punkt wejścia.</span><span class="sxs-lookup"><span data-stu-id="4341a-105">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="4341a-106">Program może mieć tylko jedną metodę [Main](../programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="4341a-106">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="4341a-107">Aby rozwiązać ten problem, można usunąć wszystkie metody główne w kodzie, z wyjątkiem jednego, lub użyć opcji [-Main](../language-reference/compiler-options/main-compiler-option.md) kompilatora, aby określić, która metoda Main ma być używana.</span><span class="sxs-lookup"><span data-stu-id="4341a-107">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="4341a-108">Poniższy przykład generuje CS0017:</span><span class="sxs-lookup"><span data-stu-id="4341a-108">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
