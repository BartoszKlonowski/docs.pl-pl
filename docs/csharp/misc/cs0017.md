<span data-ttu-id="5803c-103">Nazwa pliku wyjściowego programu "ma zdefiniowany więcej niż jeden punkt wejścia.</span><span class="sxs-lookup"><span data-stu-id="5803c-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="5803c-104">Kompiluj z/Main, aby określić typ, który zawiera punkt wejścia.</span><span class="sxs-lookup"><span data-stu-id="5803c-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="5803c-105">Program może mieć tylko jedną metodę [Main](../programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="5803c-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="5803c-106">Aby rozwiązać ten problem, można usunąć wszystkie metody główne w kodzie, z wyjątkiem jednego, lub użyć opcji [-Main](../language-reference/compiler-options/main-compiler-option.md) kompilatora, aby określić, która metoda Main ma być używana.</span><span class="sxs-lookup"><span data-stu-id="5803c-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="5803c-107">Poniższy przykład generuje CS0017:</span><span class="sxs-lookup"><span data-stu-id="5803c-107">The following sample generates CS0017:</span></span>

```csharp  
{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
