# <a name="compiler-error-cs0568"></a><span data-ttu-id="a7cbc-102">Błąd kompilatora CS0568</span><span class="sxs-lookup"><span data-stu-id="a7cbc-102">Compiler Error CS0568</span></span>
<span data-ttu-id="a7cbc-103">Struktury nie mogą zawierać jawnych konstruktorów bez parametrów</span><span class="sxs-lookup"><span data-stu-id="a7cbc-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="a7cbc-104">Każda [Struktura](../language-reference/keywords/struct.md) ma już Konstruktor bez parametrów, który inicjuje obiekt na zero.</span><span class="sxs-lookup"><span data-stu-id="a7cbc-104">Each [struct](../language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="a7cbc-105">W związku z tym konstruktory, które można utworzyć dla struktury, muszą mieć co najmniej jeden parametr.</span><span class="sxs-lookup"><span data-stu-id="a7cbc-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="a7cbc-106">Poniższy przykład generuje CS0568:</span><span class="sxs-lookup"><span data-stu-id="a7cbc-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
