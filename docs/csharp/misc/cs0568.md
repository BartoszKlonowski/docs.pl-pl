# <a name="compiler-error-cs0568"></a>CS0568 błąd kompilatora
Struktury nie mogą zawierać jawnych konstruktorów bez parametrów  
  
 Każdy [struktury](../../csharp/language-reference/keywords/struct.md) już ma konstruktora bez parametrów, które inicjuje obiekt do zera. W związku z tym konstruktory, które można utworzyć struktury, należy wykonać co najmniej jeden parametr.  
  
 Poniższy przykład spowoduje wygenerowanie CS0568:  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
