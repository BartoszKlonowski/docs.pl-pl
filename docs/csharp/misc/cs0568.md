# <a name="compiler-error-cs0568"></a><span data-ttu-id="f9724-102">CS0568 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="f9724-102">Compiler Error CS0568</span></span>
<span data-ttu-id="f9724-103">Struktury nie mogą zawierać jawnych konstruktorów bez parametrów</span><span class="sxs-lookup"><span data-stu-id="f9724-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="f9724-104">Każdy [struktury](../../csharp/language-reference/keywords/struct.md) ma już domyślnego konstruktora, który inicjuje obiekt od zera.</span><span class="sxs-lookup"><span data-stu-id="f9724-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="f9724-105">W związku z tym konstruktorów, które możesz utworzyć struktury musi mieć co najmniej jeden parametr.</span><span class="sxs-lookup"><span data-stu-id="f9724-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="f9724-106">Poniższy przykład generuje CS0568:</span><span class="sxs-lookup"><span data-stu-id="f9724-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
