# <a name="compiler-error-cs0241"></a>CS0241 błąd kompilatora
Specyfikatory parametrów domyślne nie są dozwolone.  
  
 [Parametry metody](../../csharp/language-reference/keywords/method-parameters.md) nie może mieć wartości domyślne. Użyj przeciążenia metody, aby osiągnąć ten sam efekt. Aby uzyskać więcej informacji, zobacz [przekazanie parametrów](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0241. Ponadto przykład pokazuje, jak zasymulować, za pomocą przeciążenia metody z argumentami domyślnymi.  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
