 <span data-ttu-id="3f0d1-108">Poniższy przykład generuje CS0241.</span><span class="sxs-lookup"><span data-stu-id="3f0d1-108">The following sample generates CS0241.</span></span> <span data-ttu-id="3f0d1-109">Ponadto pokazano symulowanie z przeładowanie metody z argumentami domyślnymi.</span><span class="sxs-lookup"><span data-stu-id="3f0d1-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
