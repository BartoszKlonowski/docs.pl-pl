## <a name="example"></a><span data-ttu-id="7e51a-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="7e51a-107">Example</span></span>  
 <span data-ttu-id="7e51a-108">Poniższy przykład spowoduje wygenerowanie CS0191.</span><span class="sxs-lookup"><span data-stu-id="7e51a-108">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
