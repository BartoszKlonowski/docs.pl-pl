# <a name="compiler-error-cs0191"></a>CS0191 błąd kompilatora
Właściwość lub indeksator "name" nie może być przypisany do — jest tylko do odczytu  
  
 A [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) pola przyjmuje tylko przypisania w konstruktorze lub w deklaracji. Aby uzyskać więcej informacji, zobacz [konstruktory](../../csharp/programming-guide/classes-and-structs/constructors.md).  
  
 CS0191 powoduje również, jeśli `readonly` pole jest [statyczne](../../csharp/language-reference/keywords/static.md) i Konstruktor nie jest oznaczony `static`.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0191.  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
