 <span data-ttu-id="c6553-105">Pole [tylko do odczytu](../language-reference/keywords/readonly.md) może przyjmować tylko przypisanie w konstruktorze lub w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="c6553-105">A [readonly](../language-reference/keywords/readonly.md) field can only take an assignment in a constructor or at declaration.</span></span> <span data-ttu-id="c6553-106">Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="c6553-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>

## <a name="example"></a><span data-ttu-id="c6553-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="c6553-108">Example</span></span>  

 <span data-ttu-id="c6553-109">Poniższy przykład generuje CS0191.</span><span class="sxs-lookup"><span data-stu-id="c6553-109">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
