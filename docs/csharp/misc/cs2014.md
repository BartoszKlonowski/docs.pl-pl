# <a name="compiler-warning-level-1-cs2014"></a>Ostrzeżenie kompilatora (poziom 1) CS2014
Opcja kompilatora "stara opcja" jest przestarzała, Użyj zamiast niej opcji "New Option"  
  
 Forma opcji kompilatora jest przestarzała. Zobacz [ C# opcje kompilatora](../language-reference/compiler-options/index.md) , aby uzyskać więcej informacji.
