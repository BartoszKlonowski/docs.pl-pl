# <a name="compiler-error-cs0214"></a>CS0214 błąd kompilatora
Wskaźniki i bufory o ustalonym rozmiarze mogą być używane tylko w niebezpiecznym kontekście  
  
 Wskaźniki można używać tylko z [niebezpieczne](../../csharp/language-reference/keywords/unsafe.md) — słowo kluczowe. Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Poniższy przykład generuje CS0214:  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
