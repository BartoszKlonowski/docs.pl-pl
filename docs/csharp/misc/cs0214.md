# <a name="compiler-error-cs0214"></a><span data-ttu-id="fd5a1-103">Błąd kompilatora CS0214</span><span class="sxs-lookup"><span data-stu-id="fd5a1-103">Compiler Error CS0214</span></span>
<span data-ttu-id="fd5a1-104">Wskaźniki i bufory o ustalonym rozmiarze mogą być używane tylko w niebezpiecznym kontekście</span><span class="sxs-lookup"><span data-stu-id="fd5a1-104">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="fd5a1-105">Wskaźników można używać tylko z [niebezpiecznym](../language-reference/keywords/unsafe.md) słowem kluczowym.</span><span class="sxs-lookup"><span data-stu-id="fd5a1-105">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="fd5a1-106">Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="fd5a1-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="fd5a1-107">Poniższy przykład generuje CS0214:</span><span class="sxs-lookup"><span data-stu-id="fd5a1-107">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
