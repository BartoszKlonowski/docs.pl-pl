<span data-ttu-id="77604-104">Instrukcja foreach wymaga, aby typ zwracany "Type" typu. GetEnumerator () ' musi mieć odpowiednią metodę publiczną MoveNext i Właściwość publiczną Current</span><span class="sxs-lookup"><span data-stu-id="77604-104">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="77604-105"><xref:System.Collections.IEnumerable.GetEnumerator%2A>Funkcja, używana do włączania instrukcji foreach, nie może zwracać wskaźnika lub tablicy; musi zwrócić wystąpienie klasy, która może pełnić rolę modułu wyliczającego.</span><span class="sxs-lookup"><span data-stu-id="77604-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="77604-106">Odpowiednie wymagania służące jako moduł wyliczający obejmują publiczną właściwość bieżącą i publiczną metodę MoveNext.</span><span class="sxs-lookup"><span data-stu-id="77604-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>  
  
> [!NOTE]

 <span data-ttu-id="77604-109">Poniższy przykład generuje CS0202:</span><span class="sxs-lookup"><span data-stu-id="77604-109">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
