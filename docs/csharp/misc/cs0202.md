# <a name="compiler-error-cs0202"></a>CS0202 błąd kompilatora
Instrukcja foreach wymaga aby typ zwracany "type" elementu "typu. GetEnumerator() "musi mieć odpowiednią metodę publiczną MoveNext i właściwość publiczną Current  
  
 A <xref:System.Collections.IEnumerable.GetEnumerator%2A> funkcji, umożliwiają używanie instrukcji foreach nie może zwracać wskaźnik lub tablica; musi zwracać wystąpienia klasy, która może działać jako moduł wyliczający. Odpowiednie wymagania jako moduł wyliczający zawierają właściwość publiczną Current i metodę publiczną MoveNext.  
  
> [!NOTE]
>  W C# 2.0, kompilator automatycznie generuje bieżącą i MoveNext dla Ciebie. Aby uzyskać więcej informacji, zobacz przykład kodu w [interfejsów ogólnych](../../csharp/programming-guide/generics/generic-interfaces.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0202:  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
