<span data-ttu-id="df712-103">Właściwość lub indeksator "właściwości/indeksatora" nie można użyć w tym kontekście, ponieważ niedostępna jest metoda dostępu get</span><span class="sxs-lookup"><span data-stu-id="df712-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="df712-104">Ten błąd występuje, gdy próbują uzyskać dostęp niedostępny `get` metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="df712-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="df712-105">Aby rozwiązać ten problem, Zwiększ dostępność metody dostępu lub zmień lokalizację wywoływania.</span><span class="sxs-lookup"><span data-stu-id="df712-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="df712-106">Aby uzyskać więcej informacji, zobacz [dostępności metody dostępu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) i [właściwości](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="df712-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="df712-107">Poniższy przykład generuje CS0271:</span><span class="sxs-lookup"><span data-stu-id="df712-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
