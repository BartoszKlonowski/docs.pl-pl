# <a name="compiler-error-cs0271"></a><span data-ttu-id="57017-102">Błąd kompilatora CS0271</span><span class="sxs-lookup"><span data-stu-id="57017-102">Compiler Error CS0271</span></span>
<span data-ttu-id="57017-103">Nie można użyć właściwości lub indeksatora "Property/Indexer" w tym kontekście, ponieważ metoda dostępu get jest niedostępna</span><span class="sxs-lookup"><span data-stu-id="57017-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="57017-104">Ten błąd występuje, gdy próbujesz uzyskać dostęp do niedostępnego `get` dostępu.</span><span class="sxs-lookup"><span data-stu-id="57017-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="57017-105">Aby rozwiązać ten problem, zwiększ dostępność metody dostępu lub Zmień lokalizację wywołania.</span><span class="sxs-lookup"><span data-stu-id="57017-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="57017-106">Aby uzyskać więcej informacji, zobacz dostępność i [Właściwości](../programming-guide/classes-and-structs/properties.md) [metody dostępu](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) .</span><span class="sxs-lookup"><span data-stu-id="57017-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="57017-107">Poniższy przykład generuje CS0271:</span><span class="sxs-lookup"><span data-stu-id="57017-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
