<span data-ttu-id="b1ae0-103">Właściwości lub indeksatora "właściwość/indeksator" nie może być używany w tym kontekście, ponieważ get akcesor jest niedostępny</span><span class="sxs-lookup"><span data-stu-id="b1ae0-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="b1ae0-104">Ten błąd występuje podczas próby `get` uzyskania dostępu do niedostępnego akcesora.</span><span class="sxs-lookup"><span data-stu-id="b1ae0-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="b1ae0-105">Aby rozwiązać ten błąd, zwiększ dostępność akcesora lub zmień lokalizację wywołującą.</span><span class="sxs-lookup"><span data-stu-id="b1ae0-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="b1ae0-106">Aby uzyskać więcej informacji, zobacz Dostępność programu [Accessor](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) i [właściwości](../programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="b1ae0-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="b1ae0-107">Poniższy przykład generuje CS0271:</span><span class="sxs-lookup"><span data-stu-id="b1ae0-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
