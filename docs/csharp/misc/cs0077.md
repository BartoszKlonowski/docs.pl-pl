# <a name="compiler-error-cs0077"></a><span data-ttu-id="f6374-102">CS0077 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="f6374-102">Compiler Error CS0077</span></span>
<span data-ttu-id="f6374-103">Operator musi być używany z typem referencyjnym lub typ dopuszczający wartość null ("int" jest typem wartości niedopuszczającym wartości).</span><span class="sxs-lookup"><span data-stu-id="f6374-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="f6374-104">[Jako](../../csharp/language-reference/keywords/as.md) przekazano operator [typu wartości](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="f6374-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="f6374-105">Ponieważ `as` może zwrócić [null](../../csharp/language-reference/keywords/null.md), tylko mogą być przekazywane [typy odwołań](../../csharp/language-reference/keywords/reference-types.md) lub typ dopuszczający wartość null.</span><span class="sxs-lookup"><span data-stu-id="f6374-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="f6374-106">Aby uzyskać więcej informacji na temat typów dopuszczających wartości zerowe zobacz [typów dopuszczających wartości zerowe](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="f6374-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="f6374-107">Poniższy przykład spowoduje wygenerowanie CS0077:</span><span class="sxs-lookup"><span data-stu-id="f6374-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
