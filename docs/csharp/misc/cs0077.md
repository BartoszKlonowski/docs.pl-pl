# <a name="compiler-error-cs0077"></a><span data-ttu-id="d9ec9-102">Błąd kompilatora CS0077</span><span class="sxs-lookup"><span data-stu-id="d9ec9-102">Compiler Error CS0077</span></span>
<span data-ttu-id="d9ec9-103">Operator as musi być używany z typem referencyjnym lub typem dopuszczającym wartość null (wartość "int" to typ wartości niedopuszczający wartości null).</span><span class="sxs-lookup"><span data-stu-id="d9ec9-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="d9ec9-104">Operator [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) został przekazano [Typ wartości](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="d9ec9-104">The [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="d9ec9-105">Ponieważ `as` może zwracać [wartość null](../../csharp/language-reference/keywords/null.md), można tylko przekazywać [typy odwołania](../../csharp/language-reference/keywords/reference-types.md) lub typ dopuszczający wartość null.</span><span class="sxs-lookup"><span data-stu-id="d9ec9-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="d9ec9-106">Aby uzyskać więcej informacji na temat typów dopuszczających wartości null, zobacz [Typy dopuszczające wartości null](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="d9ec9-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="d9ec9-107">Poniższy przykład generuje CS0077:</span><span class="sxs-lookup"><span data-stu-id="d9ec9-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
