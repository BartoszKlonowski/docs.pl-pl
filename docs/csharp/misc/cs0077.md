# <a name="compiler-error-cs0077"></a>CS0077 błąd kompilatora
Operator musi być używany z typem referencyjnym lub typ dopuszczający wartość null ("int" jest typem wartości niedopuszczającym wartości).  
  
 [Jako](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) przekazano operator [typu wartości](../../csharp/language-reference/keywords/value-types.md). Ponieważ `as` może zwrócić [null](../../csharp/language-reference/keywords/null.md), tylko mogą być przekazywane [typy odwołań](../../csharp/language-reference/keywords/reference-types.md) lub typ dopuszczający wartość null. Aby uzyskać więcej informacji na temat typów dopuszczających wartości zerowe zobacz [typów dopuszczających wartości zerowe](../../csharp/programming-guide/nullable-types/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
