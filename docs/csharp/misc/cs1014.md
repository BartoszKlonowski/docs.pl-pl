# <a name="compiler-error-cs1014"></a>CS1014 błąd kompilatora
Get lub set Oczekiwano metody dostępu  
  
 W deklaracji właściwości znaleziono deklaracji metody. Można zadeklarować tylko `get` i `set` metody we właściwości.  
  
 Aby uzyskać więcej informacji na temat właściwości, zobacz [przy użyciu właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1014.  
  
```csharp  
// CS1014.cs  
// compile with: /target:library  
class Sample  
{  
   public int TestProperty  
   {  
      get  
      {  
         return 0;  
      }  
      int z;   // CS1014  not get or set  
   }  
}  
```
