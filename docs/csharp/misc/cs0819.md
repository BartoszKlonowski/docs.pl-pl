## <a name="example-1"></a><span data-ttu-id="b4453-110">Przykład 1</span><span class="sxs-lookup"><span data-stu-id="b4453-110">Example 1</span></span>

```csharp

class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        //var a = 3;
```
