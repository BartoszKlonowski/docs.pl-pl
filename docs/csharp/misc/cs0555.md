# <a name="compiler-error-cs0555"></a><span data-ttu-id="acd40-103">Błąd kompilatora CS0555</span><span class="sxs-lookup"><span data-stu-id="acd40-103">Compiler Error CS0555</span></span>

<span data-ttu-id="acd40-104">Zdefiniowany przez użytkownika operator nie może pobrać obiektu typu otaczającego i dokonać konwersji na obiekt typu otaczającego</span><span class="sxs-lookup"><span data-stu-id="acd40-104">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="acd40-105">Zdefiniowane przez użytkownika konwersje na wartości otaczającej klasy są niedozwolone; Ten operator nie jest potrzebny.</span><span class="sxs-lookup"><span data-stu-id="acd40-105">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="acd40-106">Poniższy przykład generuje CS0555:</span><span class="sxs-lookup"><span data-stu-id="acd40-106">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
