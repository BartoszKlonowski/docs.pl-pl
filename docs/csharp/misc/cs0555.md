# <a name="compiler-error-cs0555"></a><span data-ttu-id="92dbb-102">CS0555 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="92dbb-102">Compiler Error CS0555</span></span>

<span data-ttu-id="92dbb-103">Zdefiniowany przez użytkownika operator nie może pobrać obiektu typu otaczającego i konwersji na obiekt typu otaczającego</span><span class="sxs-lookup"><span data-stu-id="92dbb-103">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="92dbb-104">Konwersje zdefiniowane przez użytkownika wartości otaczającej klasy nie są dozwolone taki operator nie jest konieczne.</span><span class="sxs-lookup"><span data-stu-id="92dbb-104">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="92dbb-105">Poniższy przykład spowoduje wygenerowanie CS0555:</span><span class="sxs-lookup"><span data-stu-id="92dbb-105">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
