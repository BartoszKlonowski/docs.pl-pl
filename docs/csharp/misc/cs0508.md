## <a name="example"></a><span data-ttu-id="b34be-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="b34be-107">Example</span></span>  

 <span data-ttu-id="b34be-108">Poniższy przykład generuje CS0508.</span><span class="sxs-lookup"><span data-stu-id="b34be-108">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
