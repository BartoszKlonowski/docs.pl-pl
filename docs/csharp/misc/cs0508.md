# <a name="compiler-error-cs0508"></a>CS0508 błąd kompilatora
"Type 1": typem zwracanym musi być "Typ 2", aby dopasować zgodnym z przesłoniętą składową "Nazwa elementu członkowskiego"  
  
 Próbowano zmienić typ zwracany w zastąpienie metody. Aby rozwiązać ten problem, upewnij się, że obie metody zadeklarować, że taki sam typ zwracany.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0508.  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
