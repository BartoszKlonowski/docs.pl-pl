# <a name="compiler-error-cs1056"></a>CS1056 błąd kompilatora
Nieoczekiwany znak "character"  
  
 Kompilator języka C# Napotkano nieoczekiwany znak i nie może zidentyfikować token obecnie przetwarzane. Na przykład jeśli kompilator napotka EUR znak w trakcie przetwarzania identyfikator, będzie on mógł klasyfikowania identyfikator, ponieważ znak EUR tylko będzie ważny w ciągu i kompilator wie, że nie przetwarza ciąg.
