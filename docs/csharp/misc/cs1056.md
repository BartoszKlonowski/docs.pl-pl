# <a name="compiler-error-cs1056"></a>CS1056 błąd kompilatora
Nieoczekiwany znak "character"  
  
 C# Kompilator napotkał nieoczekiwany znak i nie może zidentyfikować token w danej chwili przetwarzany. Na przykład gdy kompilator napotka znak Euro w trakcie przetwarzania identyfikatora, będzie mogła sklasyfikować identyfikator, ponieważ znak Euro tylko będzie prawidłowa wewnątrz ciągu, a kompilator wie, że nie przetwarza ciąg.
