# <a name="compiler-error-cs1056"></a>Błąd kompilatora CS1056

Nieoczekiwany znak "Character"  
  
 Kompilator języka C# napotkał nieoczekiwany znak i nie może zidentyfikować aktualnie przetwarzanego tokenu. Na przykład, jeśli kompilator napotyka symbol euro w środku przetwarzania identyfikatora, nie będzie w stanie sklasyfikować identyfikatora, ponieważ znak euro będzie prawidłowy tylko wewnątrz ciągu, a kompilator wie, że nie przetwarza ciągu.
