# <a name="compiler-error-cs2018"></a>Błąd kompilatora CS2018

Nie można znaleźć pliku komunikatów "cscmsgs.dll"  
  
 Nie znaleziono pliku dll, który zawiera błąd kompilatora i komunikaty ostrzegawcze. Ten plik musi znajdować się w tym samym katalogu, w którym znajdują się inne pliki obsługi kompilatora.
