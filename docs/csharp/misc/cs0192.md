# <a name="compiler-error-cs0192"></a><span data-ttu-id="8171f-102">Błąd kompilatora CS0192</span><span class="sxs-lookup"><span data-stu-id="8171f-102">Compiler Error CS0192</span></span>
<span data-ttu-id="8171f-103">Pola statycznego pola tylko do odczytu "nazwa" nie mogą być przekazywane ref lub out (z wyjątkiem konstruktora statycznego)</span><span class="sxs-lookup"><span data-stu-id="8171f-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="8171f-104">Pole (zmienna) oznaczone słowem kluczowym [tylko do odczytu](../language-reference/keywords/readonly.md) nie może być przekazane do parametru [ref](../language-reference/keywords/ref.md) lub [out,](../language-reference/keywords/out-parameter-modifier.md) z wyjątkiem wewnątrz konstruktora.</span><span class="sxs-lookup"><span data-stu-id="8171f-104">A field (variable) marked with the [readonly](../language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="8171f-105">Aby uzyskać więcej informacji, zobacz [Pola](../programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="8171f-105">For more information, see [Fields](../programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="8171f-106">CS0192 również wyniki, jeśli `readonly` pole jest [statyczne](../language-reference/keywords/static.md) `static`i konstruktora nie jest oznaczony .</span><span class="sxs-lookup"><span data-stu-id="8171f-106">CS0192 also results if the `readonly` field is [static](../language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8171f-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="8171f-107">Example</span></span>  
 <span data-ttu-id="8171f-108">Poniższy przykład generuje CS0192.</span><span class="sxs-lookup"><span data-stu-id="8171f-108">The following sample generates CS0192.</span></span>  
  
```csharp
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
