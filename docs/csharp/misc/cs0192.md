<span data-ttu-id="bc8d9-103">Nie można przekazać pól pola statycznego tylko do odczytu "name" ref lub out (z wyjątkiem w konstruktorze statycznym)</span><span class="sxs-lookup"><span data-stu-id="bc8d9-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="bc8d9-104">Pole (zmienna) oznaczone [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) — słowo kluczowe nie mogą być przekazywane do [ref](../../csharp/language-reference/keywords/ref.md) lub [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru, z wyjątkiem w konstruktorze.</span><span class="sxs-lookup"><span data-stu-id="bc8d9-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="bc8d9-105">Aby uzyskać więcej informacji, zobacz [pola](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="bc8d9-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="bc8d9-106">CS0192 powoduje również, jeśli `readonly` pole jest [statyczne](../../csharp/language-reference/keywords/static.md) i Konstruktor nie jest oznaczony `static`.</span><span class="sxs-lookup"><span data-stu-id="bc8d9-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bc8d9-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="bc8d9-107">Example</span></span>  
 <span data-ttu-id="bc8d9-108">Poniższy przykład spowoduje wygenerowanie CS0192.</span><span class="sxs-lookup"><span data-stu-id="bc8d9-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
