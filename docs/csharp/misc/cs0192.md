 <span data-ttu-id="be79c-104">Pola (zmienna) oznaczonych [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) — słowo kluczowe nie mogą być przekazywane do [ref](../../csharp/language-reference/keywords/ref.md) lub [limit](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru z wyjątkiem wewnątrz konstruktora.</span><span class="sxs-lookup"><span data-stu-id="be79c-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="be79c-105">Aby uzyskać więcej informacji, zobacz [pola](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="be79c-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>

## <a name="example"></a><span data-ttu-id="be79c-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="be79c-107">Example</span></span>  
 <span data-ttu-id="be79c-108">Poniższy przykład generuje CS0192.</span><span class="sxs-lookup"><span data-stu-id="be79c-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
