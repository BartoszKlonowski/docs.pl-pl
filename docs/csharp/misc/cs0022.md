# <a name="compiler-error-cs0022"></a>Błąd kompilatora CS0022
Niewłaściwa liczba indeksów wewnątrz [], oczekiwano elementu "number"  
  
 Operacja dostępu do tablicy określiła niepoprawną liczbę wymiarów w nawiasach kwadratowych. Aby uzyskać więcej informacji, zobacz [tablice](../programming-guide/arrays/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0022:  
  
```csharp  
// CS0022.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int[] a = new int[10];  
        a[0] = 0;     // single-dimension array  
        a[0,1] = 9;   // CS0022, the array does not have two dimensions  
    }  
}  
```
