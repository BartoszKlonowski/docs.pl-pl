# <a name="compiler-error-cs0022"></a>CS0022 błąd kompilatora
Niewłaściwa liczba indeksów wewnątrz konstrukcji [], oczekiwana liczba  
  
 Operacja dostęp do tablicy określona niepoprawna liczba wymiarów w nawiasy kwadratowe. Aby uzyskać więcej informacji, zobacz [tablic](../../csharp/programming-guide/arrays/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0022:  
  
```csharp  
// CS0022.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int[] a = new int[10];  
        a[0] = 0;     // single-dimension array  
        a[0,1] = 9;   // CS0022, the array does not have two dimensions  
    }  
}  
```
