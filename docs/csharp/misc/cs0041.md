# <a name="compiler-error-cs0041"></a>Błąd kompilatora CS0041
W pełni kwalifikowana nazwa elementu "Type" jest za długa, aby uzyskać informacje o debugowaniu. Kompiluj bez opcji "/debug".  
  
 Ten błąd może wystąpić, gdy jest używana opcja kompilatora [/Debug](../language-reference/compiler-options/debug-compiler-option.md) . Jeśli wystąpi ten błąd, spróbuj usunąć pliki PDB z katalogu bin i ponownie skompilować. Jeśli ten błąd nadal występuje, może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio.
