# <a name="compiler-error-cs0144"></a>CS0144 błąd kompilatora
Nie można utworzyć wystąpienia abstrakcyjnej klasy lub interfejsu, "interface"  
  
 Nie można utworzyć wystąpienia [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) klasy lub [interfejsu](../../csharp/language-reference/keywords/interface.md). Więcej informacji znajdziesz w artykule [Interfejsy](../../csharp/programming-guide/interfaces/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0144:  
  
```csharp  
// CS0144.cs  
interface MyInterface  
{  
}  
public class MyClass  
{  
   public static void Main()  
   {  
      MyInterface myInterface = new MyInterface ();   // CS0144  
   }  
}  
```
