# <a name="compiler-error-cs1664"></a>CS1664 błąd kompilatora
Bufor o ustalonym rozmiarze o długości "length" i typie "type" jest za duży  
  
 Maksymalny rozmiar buforu ustalonym rozmiarze (określone długości pomnożona przez rozmiar elementu) wynosi 2 ^ 31 = 268435455.
