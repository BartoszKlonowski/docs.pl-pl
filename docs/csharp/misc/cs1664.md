# <a name="compiler-error-cs1664"></a>Błąd kompilatora CS1664
Bufor o ustalonym rozmiarze o długości "Length" i typie "Type" jest zbyt duży  
  
 Maksymalny rozmiar buforu o ustalonym rozmiarze (określony przez długość pomnożoną przez rozmiar elementu) to 2 ^ 31 = 268435455.
