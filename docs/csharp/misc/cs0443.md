# <a name="compiler-error-cs0443"></a>Błąd kompilatora CS0443
Błąd składni, oczekiwano wartości  
  
 Ten błąd występuje, gdy odwołujesz się do tablicy bez określania wartości dla indeksu tablicy.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0443.  
  
```csharp  
// CS0443.cs
using System;
class MyClass
{  
    public static void Main()
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}
    }  
}  
```
