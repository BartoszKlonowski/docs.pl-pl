# <a name="compiler-error-cs1713"></a>CS1713 błąd kompilatora
Nieoczekiwany błąd podczas tworzenia nazwy metadanych dla typu Typename1 — "Reason"  
  
 Wewnętrzny błąd kompilatora często przyczyną tego błędu. Wprowadzamy drobne zmiany do kodu, takich jak skracania nazwę, a następnie ponownego kompilowania może rozwiązać ten problem.
