# <a name="compiler-error-cs1713"></a>CS1713 błąd kompilatora
Nieoczekiwany błąd podczas tworzenia nazwy metadanych dla typu Typename1 — "Przyczyna"  
  
 Wewnętrzny błąd kompilatora często przyczyną tego błędu. Tworzenie niewielkimi zmianami w kodzie, takich jak skracania nazwy, a następnie ponownego kompilowania, może rozwiązać ten problem.
