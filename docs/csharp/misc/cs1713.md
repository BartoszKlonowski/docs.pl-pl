# <a name="compiler-error-cs1713"></a>Błąd kompilatora CS1713

Nieoczekiwany błąd podczas kompilowania nazwy metadanych dla typu Typename1 — 'Reason '  
  
 Ten błąd jest często spowodowany przez wewnętrzny błąd kompilatora. Wprowadzanie drobnych zmian w kodzie, takich jak skracanie długości nazwy, a następnie ponowne kompilowanie, może rozwiązać ten problem.
