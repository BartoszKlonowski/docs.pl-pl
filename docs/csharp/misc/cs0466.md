# <a name="compiler-error-cs0466"></a><span data-ttu-id="6a8cb-102">CS0466 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="6a8cb-102">Compiler Error CS0466</span></span>
<span data-ttu-id="6a8cb-103">"metoda1" nie powinien mieć parametru params, ponieważ "metoda2" nie ma</span><span class="sxs-lookup"><span data-stu-id="6a8cb-103">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="6a8cb-104">Nie można użyć `params` parametru w elemencie członkowskim klasy, gdy zaimplementowany interfejs nie jest używany.</span><span class="sxs-lookup"><span data-stu-id="6a8cb-104">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6a8cb-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="6a8cb-105">Example</span></span>  
 <span data-ttu-id="6a8cb-106">Poniższy przykład generuje CS0466.</span><span class="sxs-lookup"><span data-stu-id="6a8cb-106">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
