# <a name="compiler-error-cs0410"></a>CS0410 błąd kompilatora
Żadne przeciążenie dla elementu "method" nie ma prawidłowych parametrów i zwracanych typów  
  
 Ten błąd występuje, jeśli zostanie podjęta próba tworzenia wystąpienia delegata z funkcją, która ma nieprawidłowy parametr typy. Typy parametru obiektu delegowanego musi odpowiadać funkcja, która jest przypisywana do delegata.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0410:  
  
```csharp  
// CS0410.cs  
// compile with: /langversion:ISO-1  
  
class Test  
{  
    delegate void D(double d );  
    static void F(int i) { }  
  
    static void Main()  
    {  
        D d = new D(F);  // CS0410  
    }  
}  
```
