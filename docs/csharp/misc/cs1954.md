# <a name="compiler-error-cs1954"></a><span data-ttu-id="4aff2-102">CS1954 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="4aff2-102">Compiler Error CS1954</span></span>
<span data-ttu-id="4aff2-103">Nie można użyć najlepsze dopasowanie użycie metody przeciążonej "metoda" dla elementu inicjatora kolekcji.</span><span class="sxs-lookup"><span data-stu-id="4aff2-103">The best overloaded method match 'method' for the collection initializer element cannot be used.</span></span> <span data-ttu-id="4aff2-104">Metody "Add" inicjatora kolekcji nie mogą mieć ref ani out parametrów.</span><span class="sxs-lookup"><span data-stu-id="4aff2-104">Collection initializer 'Add' methods cannot have ref or out parameters.</span></span>  
  
 <span data-ttu-id="4aff2-105">Klasa kolekcji może być inicjowane przy użyciu inicjatora kolekcji, klasa musi mieć publiczny `Add` metodę, która nie jest `ref` lub `out` parametru.</span><span class="sxs-lookup"><span data-stu-id="4aff2-105">For a collection class to be initialized by using a collection initializer, the class must have a public `Add` method that is not a `ref` or `out` parameter.</span></span>

-   <span data-ttu-id="4aff2-107">Jeśli zmodyfikujesz kod źródłowy klasy kolekcji, należy podać `Add` metodę, która nie przyjmuje `ref` lub `out` parametru.</span><span class="sxs-lookup"><span data-stu-id="4aff2-107">If you can modify the source code of the collection class, provide an `Add` method that does not take a `ref` or `out` parameter.</span></span>

## <a name="example"></a><span data-ttu-id="4aff2-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="4aff2-110">Example</span></span>  
 <span data-ttu-id="4aff2-111">Poniższy przykład tworzy CS1954, ponieważ jedyną dostępną przeciążenia `Add` listy w `MyList` ma `ref` parametru.</span><span class="sxs-lookup"><span data-stu-id="4aff2-111">The following example produces CS1954 because the only available overload of the `Add` list in `MyList` has a `ref` parameter.</span></span>  
  
```csharp  
// cs1954.cs  
using System.Collections.Generic;  
class MyList<T> : IEnumerable<T>  
{  
    List<T> _list;  
    public void Add(ref T item)  
    {  
        _list.Add(item);  
    }  
  
    public System.Collections.Generic.IEnumerator<T> GetEnumerator()  
    {  
        int index = 0;  
        T current = _list[index];  
        while (current != null)  
        {  
            yield return _list[index++];  
        }  
    }  
  
    System.Collections.IEnumerator System.Collections.IEnumerable.GetEnumerator()  
    {  
        return GetEnumerator();  
    }  
}  
  
public class MyClass  
{  
    public string tree { get; set; }  
}  
class Program  
{  
    static void Main(string[] args)  
    {  
        MyList<MyClass> myList = new MyList<MyClass> { new MyClass { tree = "maple" } }; // CS1954  
    }  
}  
```
