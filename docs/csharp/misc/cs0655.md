 <span data-ttu-id="50c96-104">Zobacz [atrybuty](../../csharp/programming-guide/concepts/attributes/index.md) dyskusję na temat typów prawidłowego parametru atrybutu.</span><span class="sxs-lookup"><span data-stu-id="50c96-104">See [Attributes](../../csharp/programming-guide/concepts/attributes/index.md) for a discussion of valid parameter types for an attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="50c96-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="50c96-105">Example</span></span>  
 <span data-ttu-id="50c96-106">Poniższy przykład spowoduje wygenerowanie CS0655:</span><span class="sxs-lookup"><span data-stu-id="50c96-106">The following sample generates CS0655:</span></span>  
  
```csharp  
// CS0655.cs  
using System;  
  
class MyAttribute : Attribute  
{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;  
    public int e = 0;  
}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]  
class C  
{  
    public static void Main()  
    {  
    }  
}  
```
