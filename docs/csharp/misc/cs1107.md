 <span data-ttu-id="9e5cf-105">Jest to błąd dla modyfikatorów parametrów, takich jak `this` , `ref` , `in` , i `out` Aby pojawił się więcej niż jeden raz w definicji parametru.</span><span class="sxs-lookup"><span data-stu-id="9e5cf-105">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9e5cf-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="9e5cf-106">Example</span></span>  
 <span data-ttu-id="9e5cf-107">Poniższy przykład generuje CS1107:</span><span class="sxs-lookup"><span data-stu-id="9e5cf-107">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
