## <a name="example"></a><span data-ttu-id="73e1c-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="73e1c-105">Example</span></span>  
 <span data-ttu-id="73e1c-106">Poniższy przykład generuje CS1909.</span><span class="sxs-lookup"><span data-stu-id="73e1c-106">The following sample generates CS1909.</span></span>  
  
```csharp

using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
