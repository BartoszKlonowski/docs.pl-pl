# <a name="compiler-error-cs0138"></a>CS0138 błąd kompilatora
Using — dyrektywa przestrzeń nazw będzie stosowany tylko do przestrzeni nazw; "type" jest typem nie przestrzeń nazw  
  
 A [przy użyciu](../../csharp/language-reference/keywords/using.md) dyrektywy przyjmuje tylko nazwa przestrzeni nazw jako parametr. Aby uzyskać więcej informacji, zobacz [przestrzenie nazw](../../csharp/programming-guide/namespaces/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0138:  
  
```csharp  
// CS0138.cs  
using System.Object;   // CS0138  
```
