# <a name="compiler-error-cs0516"></a>CS0516 błąd kompilatora
Konstruktor "Konstruktor" nie może wywołać sam siebie  
  
 Program nie można rekursywnie wywołania konstruktorów.  
  
 Poniższy przykład spowoduje wygenerowanie CS0516:  
  
```csharp  
// CS0516.cs  
namespace x  
{  
   public class clx  
   {  
      public clx() : this()   // CS0516, delete "this()"  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
