# <a name="compiler-error-cs0516"></a>Błąd kompilatora CS0516

Konstruktor konstruktora nie może wywoływać samego siebie  
  
 Program nie może rekursywnie wywołać konstruktorów.  
  
 Poniższy przykład generuje CS0516:  
  
```csharp  
// CS0516.cs  
namespace x  
{  
   public class clx  
   {  
      public clx() : this()   // CS0516, delete "this()"  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
