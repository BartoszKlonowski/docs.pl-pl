<span data-ttu-id="9c8e4-103">Lokalna zmienna lub jego składowych nie może mieć adresu pobranego i użyć wewnątrz metody anonimowej lub lambda wyrażenia</span><span class="sxs-lookup"><span data-stu-id="9c8e4-103">Local 'variable' or its members cannot have their address taken and be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="9c8e4-104">Ten błąd jest generowany, użyj zmiennej i spróbuj wykonać jego adres, gdy jeden z tych akcji odbywa się wewnątrz metody anonimowej.</span><span class="sxs-lookup"><span data-stu-id="9c8e4-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c8e4-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="9c8e4-105">Example</span></span>  
 <span data-ttu-id="9c8e4-106">Poniższy przykład spowoduje wygenerowanie CS1686.</span><span class="sxs-lookup"><span data-stu-id="9c8e4-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
