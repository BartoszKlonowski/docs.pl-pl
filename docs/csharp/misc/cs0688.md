## <a name="example"></a><span data-ttu-id="f9023-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="f9023-109">Example</span></span>  
 <span data-ttu-id="f9023-110">Poniższy przykład generuje CS0688.</span><span class="sxs-lookup"><span data-stu-id="f9023-110">The following sample generates CS0688.</span></span> <span data-ttu-id="f9023-111">Aby rozwiązać ostrzeżenie bez modyfikowania klasy bazowej, Usuń atrybut Security z metody zastępującej.</span><span class="sxs-lookup"><span data-stu-id="f9023-111">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="f9023-112">Nie spowoduje to rozwiązania problemu z zabezpieczeniami.</span><span class="sxs-lookup"><span data-stu-id="f9023-112">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
