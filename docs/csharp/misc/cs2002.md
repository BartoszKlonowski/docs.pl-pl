# <a name="compiler-warning-level-1-cs2002"></a>Kompilatora (poziom 1) ostrzeżenie CS2002
Plik źródłowy 'Plik' został określony wiele razy  
  
 Nazwa pliku źródłowego został przekazany do kompilatora więcej niż jeden raz. Można określić tylko raz do kompilatora do tworzenia pliku wyjściowego pliku.  
  
 Nie można pominąć to ostrzeżenie [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opcji.  
  
 Poniższy przykład spowoduje wygenerowanie CS2002:  
  
```csharp  
// CS2002.cs  
// compile with: CS2002.cs  
public class A  
{  
public static void Main(){}  
}  
```  
  
 Aby wygenerować błąd, skompilować przykład za pomocą wiersza polecenia:  
  
```console  
csc CS2002.cs CS2002.cs  
```
