# <a name="compiler-error-cs1007"></a>CS1007 błąd kompilatora
Metoda dostępu właściwości, które już zdefiniowany  
  
 Kiedy Deklarujesz [właściwość](../../csharp/programming-guide/classes-and-structs/using-properties.md), musi również zadeklarować jego metod dostępu. Jednak właściwość nie może mieć więcej niż jeden `get` metody dostępu lub więcej niż jedną `set` metody dostępu.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1007:  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
