# <a name="compiler-error-cs1007"></a><span data-ttu-id="8d076-102">Błąd kompilatora CS1007</span><span class="sxs-lookup"><span data-stu-id="8d076-102">Compiler Error CS1007</span></span>

 <span data-ttu-id="8d076-104">Podczas deklarowania [Właściwości](../programming-guide/classes-and-structs/using-properties.md)należy również zadeklarować metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="8d076-104">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="8d076-105">Jednak właściwość nie może mieć więcej niż jednej `get` metody dostępu lub więcej niż jednej `set` metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="8d076-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8d076-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="8d076-106">Example</span></span>  
 <span data-ttu-id="8d076-107">Poniższy przykład generuje CS1007:</span><span class="sxs-lookup"><span data-stu-id="8d076-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
