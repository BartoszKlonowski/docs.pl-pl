 <span data-ttu-id="21759-105">Podczas deklarowania [Właściwości](../programming-guide/classes-and-structs/using-properties.md)należy również zadeklarować metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="21759-105">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="21759-106">Jednak właściwość nie może mieć więcej niż jednej `get` metody dostępu lub więcej niż jednej `set` metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="21759-106">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="21759-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="21759-107">Example</span></span>  

 <span data-ttu-id="21759-108">Poniższy przykład generuje CS1007:</span><span class="sxs-lookup"><span data-stu-id="21759-108">The following sample generates CS1007:</span></span>

```csharp  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
