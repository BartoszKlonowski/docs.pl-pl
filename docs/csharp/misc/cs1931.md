## <a name="example"></a><span data-ttu-id="28a4e-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="28a4e-107">Example</span></span>  
 <span data-ttu-id="28a4e-108">Poniższy kod generuje CS1931, ponieważ identyfikator `x` jest używany zarówno jako zmienna lokalna w `Main`, jak i jako zmienna zakresu w wyrażeniu zapytania:</span><span class="sxs-lookup"><span data-stu-id="28a4e-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

```csharp  
    {  
        static void Main()

            int x = 1;  
            var y = from x in Enumerable.Range(1, 100) // CS1931

        }  
    }  
```
