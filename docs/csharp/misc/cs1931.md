 <span data-ttu-id="09fc7-104">Deklaracja zmiennej zakresu, podobnie jak każdej innej deklaracji musi mieć identyfikator, który jest unikatowy w obrębie przestrzeni deklaracji zmiennej.</span><span class="sxs-lookup"><span data-stu-id="09fc7-104">The declaration of a range variable, just like every other declaration, must have an identifier which is unique within the variable's declaration space.</span></span>

## <a name="example"></a><span data-ttu-id="09fc7-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="09fc7-107">Example</span></span>  
 <span data-ttu-id="09fc7-108">Poniższy kod generuje CS1931, ponieważ identyfikator `x` jest używany zarówno jako zmienna lokalna w `Main` oraz zmiennej zakresu w wyrażeniu zapytania:</span><span class="sxs-lookup"><span data-stu-id="09fc7-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

```csharp  
class Test  
    {  
        static void Main()  
        {  
            int x = 1;  
            var y = from x in Enumerable.Range(1, 100) // CS1931  
                    select x;  
        }  
    }  
```
