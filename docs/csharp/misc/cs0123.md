<span data-ttu-id="15daf-103">Delegat "delegate" nie pasuje żadne przeciążenie dla "method"</span><span class="sxs-lookup"><span data-stu-id="15daf-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="15daf-104">Próba utworzenia delegata nie powiodła się, ponieważ nie został użyty w poprawnej sygnatury.</span><span class="sxs-lookup"><span data-stu-id="15daf-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="15daf-105">Wystąpienia delegata musi być zadeklarowany z takiego samego podpisu jak deklaracji obiektu delegowanego.</span><span class="sxs-lookup"><span data-stu-id="15daf-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>

 <span data-ttu-id="15daf-108">Poniższy przykład generuje CS0123.</span><span class="sxs-lookup"><span data-stu-id="15daf-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
