# <a name="compiler-error-cs0123"></a>CS0123 błąd kompilatora
Żadne przeciążenie dla elementu "method" pasuje do delegata "delegowanie"  
  
 Próba utworzenia delegata nie powiodła się, ponieważ nie użyto poprawnej sygnatury. Wystąpienia obiektu delegowanego musi być zadeklarowany z taką samą sygnaturę jak deklaracja delegata.  
  
 Można rozwiązać ten problem, dostosowując jedną metodę lub delegować podpisu. Aby uzyskać więcej informacji, zobacz [delegatów](../../csharp/programming-guide/delegates/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0123.  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
