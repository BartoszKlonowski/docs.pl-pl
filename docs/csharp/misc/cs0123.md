# <a name="compiler-error-cs0123"></a><span data-ttu-id="f76b4-102">Błąd kompilatora CS0123</span><span class="sxs-lookup"><span data-stu-id="f76b4-102">Compiler Error CS0123</span></span>
<span data-ttu-id="f76b4-103">Żadne Przeciążenie elementu "Method" nie pasuje do delegata "Delegate"</span><span class="sxs-lookup"><span data-stu-id="f76b4-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="f76b4-104">Próba utworzenia delegata nie powiodła się, ponieważ nie użyto poprawnej sygnatury.</span><span class="sxs-lookup"><span data-stu-id="f76b4-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="f76b4-105">Wystąpienia obiektu delegowanego muszą być zadeklarowane za pomocą tego samego podpisu co deklaracja delegata.</span><span class="sxs-lookup"><span data-stu-id="f76b4-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="f76b4-106">Możesz rozwiązać ten problem, dostosowując metodę lub podpis delegata.</span><span class="sxs-lookup"><span data-stu-id="f76b4-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="f76b4-107">Aby uzyskać więcej informacji, [](../programming-guide/delegates/index.md)Zobacz delegats.</span><span class="sxs-lookup"><span data-stu-id="f76b4-107">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="f76b4-108">Poniższy przykład generuje CS0123.</span><span class="sxs-lookup"><span data-stu-id="f76b4-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
