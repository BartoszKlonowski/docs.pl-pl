<span data-ttu-id="77de3-104">Przekazywanie "argument" jako ref lub out lub pobranie jego adresu może spowodować wyjątek czasu wykonywania, ponieważ jest to pole klasy marshal-by-Reference</span><span class="sxs-lookup"><span data-stu-id="77de3-104">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="77de3-105">Każda klasa, która dziedziczy bezpośrednio lub pośrednio, z <xref:System.MarshalByRefObject> jest klasą marshal-by-Reference.</span><span class="sxs-lookup"><span data-stu-id="77de3-105">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="77de3-106">Takie klasy mogą być organizowane przez odwołanie między granicami procesów i komputerów.</span><span class="sxs-lookup"><span data-stu-id="77de3-106">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="77de3-107">W rezultacie wystąpienia tej klasy mogą być serwerami proxy dla obiektów zdalnych.</span><span class="sxs-lookup"><span data-stu-id="77de3-107">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="77de3-108">Nie można przekazać pola obiektu proxy jako [ref](../language-reference/keywords/ref.md) lub [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="77de3-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="77de3-109">Example</span></span>  

 <span data-ttu-id="77de3-110">Poniższy przykład generuje CS0197.</span><span class="sxs-lookup"><span data-stu-id="77de3-110">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
