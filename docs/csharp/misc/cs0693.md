# <a name="compiler-warning-level-3-cs0693"></a>Kompilatora (poziom 3) ostrzeżenie CS0693
Parametr typu "parametr typu" ma taką samą nazwę jak parametr typu zewnętrznego "type"  
  
 Ten błąd występuje, gdy masz ogólnej składowej, takie jak metody wewnątrz klasy ogólnej. Ponieważ parametr typu metody nie jest zawsze taki sam jak parametr typu klasy, nie można nadać im obu takiej samej nazwie. Aby uzyskać więcej informacji, zobacz [metod ogólnych](../../csharp/programming-guide/generics/generic-methods.md).  
  
 Aby uniknąć tej sytuacji, należy użyć innej nazwy dla jednego z parametrów typu.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0693.  
  
```csharp  
// CS0693.cs  
// compile with: /W:3 /target:library  
class Outer<T>  
{  
   class Inner<T> {}   // CS0693  
   // try the following line instead  
   // class Inner<U> {}  
}  
```
