# <a name="compiler-warning-level-1-cs1589"></a>Kompilatora (poziom 1) ostrzeżenie CS1589
Nie można dołączyć fragmentu XML "fragment" w pliku 'Plik' — przyczyny  
  
 Składnia (*fragmentu*) z [ \<obejmują >](../../csharp/programming-guide/xmldoc/include.md) tagu odwołanie do pliku (`file`), była nieprawidłowa dla określonego ***Przyczyna***.  
  
 Źle sformułowane wiersza zostaną umieszczone w wygenerowanym pliku XML.  
  
 Poniższy przykład spowoduje wygenerowanie CS1589:  
  
```csharp  
// CS1589.cs  
// compile with: /W:1 /doc:CS1589_out.xml  
  
/// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/' />   // CS1589  
// try the following line instead  
// /// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/*' />  
class Test  
{  
   public static void Main()  
   {  
   }  
}  
```
