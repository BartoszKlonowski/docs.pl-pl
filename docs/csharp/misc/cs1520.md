# <a name="compiler-error-cs1520"></a><span data-ttu-id="82354-102">CS1520 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="82354-102">Compiler Error CS1520</span></span>
<span data-ttu-id="82354-103">Metoda musi mieć typ zwracany</span><span class="sxs-lookup"><span data-stu-id="82354-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="82354-104">Metody, która jest zadeklarowana w klasy, struktury lub interfejsu musi mieć jawnego typu zwracanego.</span><span class="sxs-lookup"><span data-stu-id="82354-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="82354-105">W poniższym przykładzie kwadratowy metoda ma zostać zwrócona wartość [ciąg](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="82354-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="82354-106">Poniższy przykład generuje CS1520:</span><span class="sxs-lookup"><span data-stu-id="82354-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="82354-107">Alternatywnie ten błąd może wystąpić, gdy wielkość liter nazwy konstruktora różni się od deklaracji klasy lub struktury, jak w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="82354-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="82354-108">Ponieważ nazwa nie jest dokładnie taka sama jak nazwa klasy, kompilator zinterpretuje ją jako metodę regularne nie konstruktora i tworzy kod błędu:</span><span class="sxs-lookup"><span data-stu-id="82354-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
