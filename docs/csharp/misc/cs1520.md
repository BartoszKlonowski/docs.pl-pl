# <a name="compiler-error-cs1520"></a><span data-ttu-id="8b1fa-102">Błąd kompilatora CS1520</span><span class="sxs-lookup"><span data-stu-id="8b1fa-102">Compiler Error CS1520</span></span>
<span data-ttu-id="8b1fa-103">Metoda musi mieć typ zwracany</span><span class="sxs-lookup"><span data-stu-id="8b1fa-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="8b1fa-104">Metoda, która jest zadeklarowana w klasie, struktury lub interfejsu musi mieć jawny typ zwracany.</span><span class="sxs-lookup"><span data-stu-id="8b1fa-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="8b1fa-105">W poniższym przykładzie Square Metoda ma wartość zwracaną [ciągu:](../language-reference/builtin-types/reference-types.md)</span><span class="sxs-lookup"><span data-stu-id="8b1fa-105">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="8b1fa-106">Poniższy przykład generuje CS1520:</span><span class="sxs-lookup"><span data-stu-id="8b1fa-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="8b1fa-107">Alternatywnie ten błąd może wystąpić, gdy przypadek nazwy konstruktora różni się od nazwy klasy lub struktury deklaracji, jak w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="8b1fa-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="8b1fa-108">Ponieważ nazwa nie jest dokładnie taka sama jak nazwa klasy, kompilator interpretuje go jako metodę zwykłą, a nie konstruktora i generuje błąd:</span><span class="sxs-lookup"><span data-stu-id="8b1fa-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
