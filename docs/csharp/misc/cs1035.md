# <a name="compiler-error-cs1035"></a>Błąd kompilatora CS1035
Znaleziono koniec pliku, oczekiwano znaku "*/"  
  
 Ogranicznik otwierającego komentarza nie został dopasowany do ogranicznika zamykającego.  
  
 Poniższy przykład generuje CS1035:  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
