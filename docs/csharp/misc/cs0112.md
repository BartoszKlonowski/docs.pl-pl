# <a name="compiler-error-cs0112"></a>CS0112 błąd kompilatora
Nie można oznaczyć statycznej składowej "function" jako przesłoniętej, wirtualnej lub abstrakcyjnej  
  
 Wszelkie deklaracji metody, która używa `override`, **wirtualnego**, lub **abstrakcyjne** — słowo kluczowe nie można również użyć **statyczne** — słowo kluczowe.  
  
 Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0112:  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
