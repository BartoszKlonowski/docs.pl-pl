# <a name="compiler-error-cs0112"></a><span data-ttu-id="40793-103">Błąd kompilatora CS0112</span><span class="sxs-lookup"><span data-stu-id="40793-103">Compiler Error CS0112</span></span>

<span data-ttu-id="40793-104">Statyczna składowa "Function" nie może być oznaczona jako override, Virtual ani abstract</span><span class="sxs-lookup"><span data-stu-id="40793-104">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="40793-105">Każda deklaracja metody, która używa `override` słowa kluczowego, **Virtual**lub **abstract** , nie może również używać słowa kluczowego **static** .</span><span class="sxs-lookup"><span data-stu-id="40793-105">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="40793-106">Aby uzyskać więcej informacji, zobacz [metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="40793-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="40793-107">Poniższy przykład generuje CS0112:</span><span class="sxs-lookup"><span data-stu-id="40793-107">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
