# <a name="compiler-error-cs0112"></a><span data-ttu-id="c71c9-102">CS0112 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="c71c9-102">Compiler Error CS0112</span></span>
<span data-ttu-id="c71c9-103">Nie można oznaczyć statycznej składowej "function" jako przesłoniętej, wirtualnej lub abstrakcyjnej</span><span class="sxs-lookup"><span data-stu-id="c71c9-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="c71c9-104">Wszelkie deklaracji metody, która używa `override`, **wirtualnego**, lub **abstrakcyjne** — słowo kluczowe nie można również użyć **statyczne** — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="c71c9-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="c71c9-105">Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="c71c9-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="c71c9-106">Poniższy przykład spowoduje wygenerowanie CS0112:</span><span class="sxs-lookup"><span data-stu-id="c71c9-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
