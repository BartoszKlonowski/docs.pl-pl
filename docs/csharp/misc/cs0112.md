# <a name="compiler-error-cs0112"></a><span data-ttu-id="75aa4-102">CS0112 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="75aa4-102">Compiler Error CS0112</span></span>
<span data-ttu-id="75aa4-103">Nie można oznaczyć statycznego elementu członkowskiego "function" jako przesłonięty, wirtualny lub abstrakcyjny</span><span class="sxs-lookup"><span data-stu-id="75aa4-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="75aa4-104">Wszelkie deklaracji metody, która używa `override`, **wirtualnego**, lub **abstrakcyjny** również nie można użyć słowa kluczowego **statycznych** — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="75aa4-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="75aa4-105">Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="75aa4-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="75aa4-106">Poniższy przykład generuje CS0112:</span><span class="sxs-lookup"><span data-stu-id="75aa4-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
