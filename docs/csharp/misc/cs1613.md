# <a name="compiler-error-cs1613"></a><span data-ttu-id="b2cc4-102">Błąd kompilatora CS1613</span><span class="sxs-lookup"><span data-stu-id="b2cc4-102">Compiler Error CS1613</span></span>
<span data-ttu-id="b2cc4-103">Nie można znaleźć zarządzanej klasy otoki coclass "Class" dla interfejsu "Interface" (czy nie brakuje odwołania do zestawu?)</span><span class="sxs-lookup"><span data-stu-id="b2cc4-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="b2cc4-104">Podjęto próbę utworzenia wystąpienia obiektu COM z interfejsu.</span><span class="sxs-lookup"><span data-stu-id="b2cc4-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="b2cc4-105">Interfejs ma **atrybutem ComImport** i `CoClass` atrybuty, ale kompilator nie może znaleźć `CoClass` typu podanych dla atrybutu.</span><span class="sxs-lookup"><span data-stu-id="b2cc4-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="b2cc4-106">Aby rozwiązać ten problem, możesz wypróbować jedną z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="b2cc4-106">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="b2cc4-107">Dodaj odwołanie do zestawu, który ma klasę coclass (większość czasu interfejs i Klasa coclass powinny znajdować się w tym samym zestawie).</span><span class="sxs-lookup"><span data-stu-id="b2cc4-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="b2cc4-108">Aby uzyskać informacje, zobacz okno dialogowe [/Reference](../language-reference/compiler-options/reference-compiler-option.md) lub [Dodawanie odwołania](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="b2cc4-108">See [/reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="b2cc4-109">`CoClass` Popraw atrybut w interfejsie.</span><span class="sxs-lookup"><span data-stu-id="b2cc4-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="b2cc4-110">W poniższym przykładzie pokazano poprawne użycie **klasy CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="b2cc4-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
