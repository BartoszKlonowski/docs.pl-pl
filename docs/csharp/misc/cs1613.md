# <a name="compiler-error-cs1613"></a><span data-ttu-id="3f3fb-103">Błąd kompilatora CS1613</span><span class="sxs-lookup"><span data-stu-id="3f3fb-103">Compiler Error CS1613</span></span>
<span data-ttu-id="3f3fb-104">Nie można znaleźć zarządzanej klasy otoki coclass "Class" dla interfejsu "Interface" (czy nie brakuje odwołania do zestawu?)</span><span class="sxs-lookup"><span data-stu-id="3f3fb-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="3f3fb-105">Podjęto próbę utworzenia wystąpienia obiektu COM z interfejsu.</span><span class="sxs-lookup"><span data-stu-id="3f3fb-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="3f3fb-106">Interfejs ma **atrybutem ComImport** i `CoClass` atrybuty, ale kompilator nie może znaleźć typu podanych dla `CoClass` atrybutu.</span><span class="sxs-lookup"><span data-stu-id="3f3fb-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="3f3fb-107">Aby rozwiązać ten problem, możesz wypróbować jedną z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="3f3fb-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="3f3fb-108">Dodaj odwołanie do zestawu, który ma klasę coclass (większość czasu interfejs i Klasa coclass powinny znajdować się w tym samym zestawie).</span><span class="sxs-lookup"><span data-stu-id="3f3fb-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="3f3fb-109">Aby uzyskać informacje, zobacz [-Reference](../language-reference/compiler-options/reference-compiler-option.md) lub [Add Reference](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="3f3fb-109">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="3f3fb-110">Popraw `CoClass` atrybut w interfejsie.</span><span class="sxs-lookup"><span data-stu-id="3f3fb-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="3f3fb-111">W poniższym przykładzie pokazano poprawne użycie **klasy CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="3f3fb-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
