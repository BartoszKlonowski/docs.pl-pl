# <a name="compiler-error-cs0111"></a>CS0111 błąd kompilatora
Typ "class" definiuje już składową o nazwie "członek" z tymi samymi typami parametrów  
  
 CS0111 występuje, jeśli klasa zawiera dwie deklaracje element członkowski o takiej samej nazwie i typy parametrów. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0111.  
  
```csharp  
// CS0111.cs  
class A  
{  
   void Test() { }  
   public static void Test(){}   // CS0111  
  
   public static void Main() {}  
}  
```
