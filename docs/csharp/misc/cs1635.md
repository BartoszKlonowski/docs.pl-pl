# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="4cef7-102">Kompilator CS1635 ostrzegawcze (poziom 1)</span><span class="sxs-lookup"><span data-stu-id="4cef7-102">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="4cef7-103">Nie można przywrócić ostrzeżenia "Kod ostrzeżenia", ponieważ zostało globalnie wyłączone</span><span class="sxs-lookup"><span data-stu-id="4cef7-103">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="4cef7-104">To ostrzeżenie występuje, gdy używasz **/nowarn** polecenia wiersz opcji lub ustawienia, aby wyłączyć ostrzeżenia dla jednostki całego kompilacji projektu, ale możesz użyć `#pragma warning restore` próbę przywrócić tego ostrzeżenia.</span><span class="sxs-lookup"><span data-stu-id="4cef7-104">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="4cef7-105">Aby rozwiązać ten problem, Usuń ustawienie projektu lub/nowarn — opcja wiersza polecenia lub usunąć `#pragma warning restore` wszelkie ostrzeżenia wyłączania za pomocą wiersza polecenia lub ustawień projektu.</span><span class="sxs-lookup"><span data-stu-id="4cef7-105">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="4cef7-106">Aby uzyskać więcej informacji, zobacz [ostrzeżenie #pragma](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) tematu.</span><span class="sxs-lookup"><span data-stu-id="4cef7-106">For more information, see the [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="4cef7-107">Poniższy przykład generuje CS1635:</span><span class="sxs-lookup"><span data-stu-id="4cef7-107">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
