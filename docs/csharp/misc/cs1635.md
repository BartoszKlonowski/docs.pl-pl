# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="c5f02-103">Ostrzeżenie kompilatora (poziom 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="c5f02-103">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="c5f02-104">Nie można przywrócić ostrzeżenia "Kod ostrzegawczy", ponieważ został on wyłączony globalnie</span><span class="sxs-lookup"><span data-stu-id="c5f02-104">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="c5f02-105">To ostrzeżenie występuje, jeśli używasz opcji wiersza polecenia **/nowarn** lub ustawienia projektu, aby wyłączyć Ostrzeżenie dla całej jednostki kompilacji, ale używasz, `#pragma warning restore` aby spróbować przywrócić to ostrzeżenie.</span><span class="sxs-lookup"><span data-stu-id="c5f02-105">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="c5f02-106">Aby rozwiązać ten problem, Usuń opcję wiersza polecenia/nowarn lub ustawienia projektu lub Usuń `#pragma warning restore` dla wszystkich ostrzeżeń, które są wyłączane za pomocą ustawienia wiersza polecenia lub projektu.</span><span class="sxs-lookup"><span data-stu-id="c5f02-106">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="c5f02-107">Aby uzyskać więcej informacji, zapoznaj się z tematem [ostrzeżenie #pragma](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="c5f02-107">For more information, see the [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="c5f02-108">Poniższy przykład generuje CS1635:</span><span class="sxs-lookup"><span data-stu-id="c5f02-108">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
