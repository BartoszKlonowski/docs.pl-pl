# <a name="compiler-error-cs0167"></a>Błąd kompilatora CS0167
Delegat "delegat" nie ma metody Invoke  
  
 Został zaimportowany i użyty program zarządzany (taki, który używa środowiska uruchomieniowego języka wspólnego .NET Framework), który został utworzony za pomocą innego kompilatora. Ten kompilator zezwolił źle sformułowanego [delegata](../language-reference/keywords/delegate.md). W związku z `Invoke` tym Metoda była niedostępna. Aby uzyskać więcej informacji, [](../programming-guide/delegates/index.md)Zobacz delegats.
