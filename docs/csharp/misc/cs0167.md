# <a name="compiler-error-cs0167"></a>CS0167 błąd kompilatora
Delegat "delegate" nie ma metody Invoke  
  
 Zaimportowane i używany program zarządzanych (taki, który używa środowiska .NET Framework środowisko uruchomieniowe języka wspólnego) utworzony z innego kompilatora. Dozwolone przez kompilator źle sformułowane [delegować](../../csharp/language-reference/keywords/delegate.md). W związku z tym `Invoke` — metoda nie jest dostępna. Aby uzyskać więcej informacji, zobacz [delegatów](../../csharp/programming-guide/delegates/index.md).
