# <a name="compiler-error-cs0167"></a>Błąd kompilatora CS0167

Delegat "delegat" nie ma metody Invoke  
  
 Został zaimportowany i użyty program zarządzany (jeden, który używa środowiska uruchomieniowego .NET), który został utworzony przy użyciu innego kompilatora. Ten kompilator zezwolił źle sformułowanego [delegata](../language-reference/builtin-types/reference-types.md). W związku z tym `Invoke` Metoda była niedostępna. Aby uzyskać więcej informacji, zobacz [delegats](../programming-guide/delegates/index.md).
