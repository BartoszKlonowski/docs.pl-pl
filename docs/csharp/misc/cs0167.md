# <a name="compiler-error-cs0167"></a>CS0167 błąd kompilatora
Delegat "delegowanie" nie ma metody Invoke  
  
 Importowane i używane zarządzanego programu (jedną, która używa .NET Framework środowisko uruchomieniowe języka wspólnego) który został utworzony za pomocą innego kompilatora. Przez kompilator może źle sformułowane [delegować](../../csharp/language-reference/keywords/delegate.md). W związku z tym `Invoke` metoda nie jest dostępna. Aby uzyskać więcej informacji, zobacz [delegatów](../../csharp/programming-guide/delegates/index.md).
