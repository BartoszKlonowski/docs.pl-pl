# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="add13-103">Ostrzeżenie kompilatora (poziom 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="add13-103">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="add13-104">Metoda, operator lub metoda dostępu "Method" jest oznaczona jako zewnętrzna i nie ma atrybutów.</span><span class="sxs-lookup"><span data-stu-id="add13-104">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="add13-105">Rozważ dodanie atrybutu DllImport w celu określenia implementacji zewnętrznej.</span><span class="sxs-lookup"><span data-stu-id="add13-105">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="add13-106">Metoda oznaczona `extern` również powinna być oznaczona atrybutem, na przykład atrybut [dllimport](xref:System.Runtime.InteropServices.DllImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="add13-106">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="add13-107">Atrybut określa, gdzie Metoda jest zaimplementowana.</span><span class="sxs-lookup"><span data-stu-id="add13-107">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="add13-108">W czasie wykonywania program będzie potrzebować tych informacji.</span><span class="sxs-lookup"><span data-stu-id="add13-108">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="add13-109">Poniższy przykład generuje CS0626:</span><span class="sxs-lookup"><span data-stu-id="add13-109">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
