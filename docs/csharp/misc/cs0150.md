# <a name="compiler-error-cs0150"></a><span data-ttu-id="316e7-102">Błąd kompilatora CS0150</span><span class="sxs-lookup"><span data-stu-id="316e7-102">Compiler Error CS0150</span></span>
<span data-ttu-id="316e7-103">Oczekiwana jest wartość stała</span><span class="sxs-lookup"><span data-stu-id="316e7-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="316e7-104">Znaleziono zmienną, w której Oczekiwano stałej.</span><span class="sxs-lookup"><span data-stu-id="316e7-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="316e7-105">Aby uzyskać więcej informacji, zobacz [przełącznik](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="316e7-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="316e7-106">Poniższy przykład generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="316e7-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="316e7-107">Ten błąd jest tworzony również, gdy rozmiar tablicy jest określony za pomocą wartości zmiennej i inicjowany za pomocą inicjatora tablicy.</span><span class="sxs-lookup"><span data-stu-id="316e7-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="316e7-108">Aby usunąć błąd, zainicjuj tablicę w oddzielnym instrukcji lub instrukcjach.</span><span class="sxs-lookup"><span data-stu-id="316e7-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
