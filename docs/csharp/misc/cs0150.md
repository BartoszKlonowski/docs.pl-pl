<span data-ttu-id="da4bf-103">Oczekiwano wartości stałej</span><span class="sxs-lookup"><span data-stu-id="da4bf-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="da4bf-104">Zmienna został znaleziony, gdy Oczekiwano stałej.</span><span class="sxs-lookup"><span data-stu-id="da4bf-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="da4bf-105">Aby uzyskać więcej informacji, zobacz [przełącznika](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="da4bf-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="da4bf-106">Poniższy przykład generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="da4bf-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="da4bf-107">Ten błąd również jest generowany, gdy rozmiar tablicy jest określony z wartością zmiennej i zainicjować za pomocą inicjatora tablicy.</span><span class="sxs-lookup"><span data-stu-id="da4bf-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="da4bf-108">Aby usunąć ten błąd, należy zainicjować tablicy w osobnych instrukcji lub instrukcjach.</span><span class="sxs-lookup"><span data-stu-id="da4bf-108">To remove the error, initialize the array in a separate statement or statements.</span></span>

```csharp  
            }  
        }  
  
    }  
```
