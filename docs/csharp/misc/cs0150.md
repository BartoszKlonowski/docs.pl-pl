<span data-ttu-id="5ee32-103">Oczekiwano wartości stałej</span><span class="sxs-lookup"><span data-stu-id="5ee32-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="5ee32-104">Zmienna został znaleziony, których Oczekiwano stałej.</span><span class="sxs-lookup"><span data-stu-id="5ee32-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="5ee32-105">Aby uzyskać więcej informacji, zobacz [Przełącz](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="5ee32-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="5ee32-106">Poniższy przykład spowoduje wygenerowanie CS0150:</span><span class="sxs-lookup"><span data-stu-id="5ee32-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="5ee32-107">Ten błąd również jest generowany, gdy rozmiar tablicy jest określony z wartością zmiennej i inicjowany za pomocą inicjatora tablicy.</span><span class="sxs-lookup"><span data-stu-id="5ee32-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="5ee32-108">Aby usunąć ten błąd, należy zainicjować tablicy w osobnych instrukcji lub instrukcji.</span><span class="sxs-lookup"><span data-stu-id="5ee32-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp

    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
