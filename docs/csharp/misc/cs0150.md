# <a name="compiler-error-cs0150"></a>CS0150 błąd kompilatora
Oczekiwano wartości stałej  
  
 Zmienna został znaleziony, których Oczekiwano stałej. Aby uzyskać więcej informacji, zobacz [Przełącz](../../csharp/language-reference/keywords/switch.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0150:  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 Ten błąd również jest generowany, gdy rozmiar tablicy jest określony z wartością zmiennej i inicjowany za pomocą inicjatora tablicy. Aby usunąć ten błąd, należy zainicjować tablicy w osobnych instrukcji lub instrukcji.  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
