# <a name="compiler-error-cs0743"></a><span data-ttu-id="76148-102">CS0743 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="76148-102">Compiler Error CS0743</span></span>
<span data-ttu-id="76148-103">Oczekiwano kontekstowego słowa kluczowego "on"</span><span class="sxs-lookup"><span data-stu-id="76148-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="76148-104">Wzorzec `join` klauzula jest `join`... `in`... `on`... `equals`, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="76148-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="76148-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="76148-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="76148-106">Dodaj `on` słowa kluczowego `join` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="76148-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="76148-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="76148-107">Example</span></span>  
 <span data-ttu-id="76148-108">Poniższy kod generuje CS0743:</span><span class="sxs-lookup"><span data-stu-id="76148-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
