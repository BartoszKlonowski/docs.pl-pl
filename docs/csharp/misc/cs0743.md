# <a name="compiler-error-cs0743"></a><span data-ttu-id="a0f2e-103">Błąd kompilatora CS0743</span><span class="sxs-lookup"><span data-stu-id="a0f2e-103">Compiler Error CS0743</span></span>

<span data-ttu-id="a0f2e-104">Oczekiwano kontekstowego słowa kluczowego "on"</span><span class="sxs-lookup"><span data-stu-id="a0f2e-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="a0f2e-105">Wzorzec dla `join` klauzuli to.. `join` . `in` ...`on` ... `equals` , jak pokazano w tym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="a0f2e-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```

## <a name="example"></a><span data-ttu-id="a0f2e-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="a0f2e-108">Example</span></span>  

 <span data-ttu-id="a0f2e-109">Poniższy kod generuje CS0743:</span><span class="sxs-lookup"><span data-stu-id="a0f2e-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
