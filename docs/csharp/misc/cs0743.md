# <a name="compiler-error-cs0743"></a><span data-ttu-id="388f1-103">Błąd kompilatora CS0743</span><span class="sxs-lookup"><span data-stu-id="388f1-103">Compiler Error CS0743</span></span>
<span data-ttu-id="388f1-104">Oczekiwano kontekstowego słowa kluczowego "on"</span><span class="sxs-lookup"><span data-stu-id="388f1-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="388f1-105">Wzorzec dla `join` klauzuli to.. `join` . `in` ...`on` ... `equals` , jak pokazano w tym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="388f1-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="388f1-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="388f1-106">To correct this error</span></span>  
  
1. <span data-ttu-id="388f1-107">Dodaj `on` słowo kluczowe do `join` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="388f1-107">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="388f1-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="388f1-108">Example</span></span>  
 <span data-ttu-id="388f1-109">Poniższy kod generuje CS0743:</span><span class="sxs-lookup"><span data-stu-id="388f1-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
