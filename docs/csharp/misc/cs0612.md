# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="3eb66-102">Ostrzeżenie kompilatora (poziom 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="3eb66-102">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="3eb66-103">"członek" jest przestarzały</span><span class="sxs-lookup"><span data-stu-id="3eb66-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="3eb66-104">Projektant klasy oznaczył element członkowski [atrybutem Przestarzałe](../language-reference/attributes/general.md#obsolete-attribute).</span><span class="sxs-lookup"><span data-stu-id="3eb66-104">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="3eb66-105">Oznacza to, że element członkowski może nie być obsługiwany w przyszłej wersji klasy.</span><span class="sxs-lookup"><span data-stu-id="3eb66-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="3eb66-106">W poniższym przykładzie pokazano, jak dostęp do przestarzałego elementu członkowskiego generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="3eb66-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
