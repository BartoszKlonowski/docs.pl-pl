# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="a925e-102">Kompilatora (poziom 1) ostrzeżenie CS0612</span><span class="sxs-lookup"><span data-stu-id="a925e-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="a925e-103">"członek" jest przestarzały</span><span class="sxs-lookup"><span data-stu-id="a925e-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="a925e-104">Projektant klas oznaczona jako element członkowski o [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="a925e-104">The class designer marked a member with the [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="a925e-105">Oznacza to, elementu członkowskiego nie mogą być obsługiwane w przyszłych wersjach klasy.</span><span class="sxs-lookup"><span data-stu-id="a925e-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="a925e-106">Poniższy przykład pokazuje, jak uzyskiwanie dostępu do przestarzała składowa generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="a925e-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
