# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="bcf00-102">Kompilator CS0612 ostrzegawcze (poziom 1)</span><span class="sxs-lookup"><span data-stu-id="bcf00-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="bcf00-103">"członek" jest przestarzała</span><span class="sxs-lookup"><span data-stu-id="bcf00-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="bcf00-104">Projektant klas oznaczony jako element członkowski o [przestarzałe](http://msdn.microsoft.com/en-us/05e99cd0-bda6-4f79-a890-1ca093b4b488) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="bcf00-104">The class designer marked a member with the [Obsolete](http://msdn.microsoft.com/en-us/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="bcf00-105">Oznacza to, że element członkowski nie może być obsługiwany w przyszłej wersji klasy.</span><span class="sxs-lookup"><span data-stu-id="bcf00-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="bcf00-106">Poniższy przykład przedstawia sposób uzyskiwania dostępu do przestarzały element członkowski generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="bcf00-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
