# <a name="compiler-error-cs1900"></a>CS1900 błąd kompilatora
Poziom ostrzeżeń musi być z zakresu od 0 do 4  
  
 [/ Warn](../../csharp/language-reference/compiler-options/warn-compiler-option.md) — opcja kompilatora przyjmuje tylko jednej z pięciu możliwych wartości (0, 1, 2, 3 lub 4). Dowolna inna wartość przekazywana do **/ warn** spowoduje CS1900.  
  
 Poniższy przykład spowoduje wygenerowanie CS1900:  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
