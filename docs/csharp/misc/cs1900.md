# <a name="compiler-error-cs1900"></a>Błąd kompilatora CS1900
Poziom ostrzeżeń musi zawierać się w zakresie 0-4  
  
 Opcja kompilatora [/warn](../language-reference/compiler-options/warn-compiler-option.md) może przyjmować jedną z pięciu możliwych wartości (0, 1, 2, 3 lub 4). Wszystkie inne wartości przesłane do **/warn** będą wynikiem CS1900.  
  
 Poniższy przykład generuje CS1900:  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
