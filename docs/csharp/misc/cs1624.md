# <a name="compiler-error-cs1624"></a>CS1624 błąd kompilatora
Treść "metody dostępu" nie może być blokiem iteratora, ponieważ "type" nie jest typem interfejsu iteratora  
  
 Ten błąd występuje, jeśli jest używana metoda iteratora dostępu, ale zwracany typ nie jest jednym z typów interfejsu iteratora: <xref:System.Collections.IEnumerable>, <xref:System.Collections.Generic.IEnumerable%601>, <xref:System.Collections.IEnumerator>, <xref:System.Collections.Generic.IEnumerator%601>. Aby uniknąć tego błędu, należy użyć jednego z typów interfejsu iteratora jako typ zwracany.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1624:  
  
```csharp  
// CS1624.cs  
using System;  
using System.Collections;  
  
class C  
{  
    public int Iterator  
    // Try this instead:  
    // public IEnumerable Iterator  
    {  
        get  // CS1624  
        {  
            yield return 1;  
        }  
    }  
}  
```
