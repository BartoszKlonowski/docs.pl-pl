# <a name="compiler-warning-level-1-cs3014"></a>Kompilator CS3014 ostrzegawcze (poziom 1)
"członek" nie potrzebuje atrybutu CLSCompliant, ponieważ zestaw nie ma atrybutu CLSCompliant  
  
 W pliku kodu źródłowego, jeżeli nie określono zgodności z typowych specyfikacji języka (CLS) konstrukcji w pliku została oznaczona jako zgodne ze specyfikacją CLS. Jest to niedozwolone. Aby usunąć to ostrzeżenie, Dodaj atrybut poziomu zestawu ze specyfikacją CLS zgodne do pliku (w poniższym przykładzie, usuń znaczniki komentarza wiersza, który zawiera atrybut poziomu zestawu). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3014:  
  
```  
// CS3014.cs  
  
using System;  
  
// [assembly:CLSCompliant(true)]  
public class I  
{  
    [CLSCompliant(true)]   // CS3014  
    public void M()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
