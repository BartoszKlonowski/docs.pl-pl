# <a name="compiler-error-cs2001"></a>Błąd kompilatora CS2001

Nie można znaleźć pliku źródłowego "File"  
  
 Nazwa pliku źródłowego została przeniesiona do kompilatora, ale nie można jej zlokalizować. Sprawdź pisownię nazwy pliku i lokalizację pliku.
