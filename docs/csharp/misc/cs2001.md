# <a name="compiler-error-cs2001"></a>CS2001 błąd kompilatora
Nie można odnaleźć pliku źródłowego 'Plik'.  
  
 Nazwa pliku źródłowego do kompilatora został przekazany, ale nie można zlokalizować. Sprawdź pisownię nazwy pliku i lokalizację pliku.
