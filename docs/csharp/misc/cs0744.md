## <a name="example"></a><span data-ttu-id="32e4a-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="32e4a-107">Example</span></span>  
 <span data-ttu-id="32e4a-108">Poniższy kod generuje CS0744:</span><span class="sxs-lookup"><span data-stu-id="32e4a-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
