# <a name="compiler-error-cs0213"></a>Błąd kompilatora CS0213
Nie można użyć instrukcji fixed, aby przyjąć adres już ustalonego wyrażenia  
  
 Zmienna lokalna w niebezpiecznej metodzie lub parametr jest już ustalony (na stosie), dlatego nie można przyjąć adresu żadnej z tych dwóch zmiennych w wyrażeniu [stałym](../language-reference/keywords/fixed-statement.md) . [](../language-reference/keywords/unsafe.md) Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../programming-guide/unsafe-code-pointers/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0213.  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
