# <a name="compiler-error-cs0213"></a>CS0213 błąd kompilatora
Nie można użyć instrukcji fixed można przyjąć adresu już ustalonego wyrażenia  
  
 Zmienna lokalna w [niebezpieczne](../../csharp/language-reference/keywords/unsafe.md) metody lub parametru jest już wyeliminowany (na stosie), nie można przyjąć adresu którejkolwiek z tych dwóch zmiennych w [stałej](../../csharp/language-reference/keywords/fixed-statement.md) wyrażenia. Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0213.  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
