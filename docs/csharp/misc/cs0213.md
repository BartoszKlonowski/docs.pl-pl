<span data-ttu-id="8bf69-104">Nie można użyć instrukcji fixed, aby przyjąć adres już ustalonego wyrażenia</span><span class="sxs-lookup"><span data-stu-id="8bf69-104">You cannot use the fixed statement to take the address of an already fixed expression</span></span>  
  
 <span data-ttu-id="8bf69-105">Zmienna lokalna w [niebezpiecznej](../language-reference/keywords/unsafe.md) metodzie lub parametr jest już ustalony (na stosie), dlatego nie można przyjąć adresu żadnej z tych dwóch zmiennych w wyrażeniu [stałym](../language-reference/keywords/fixed-statement.md) .</span><span class="sxs-lookup"><span data-stu-id="8bf69-105">A local variable in an [unsafe](../language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="8bf69-106">Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="8bf69-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="8bf69-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="8bf69-107">Example</span></span>  

 <span data-ttu-id="8bf69-108">Poniższy przykład generuje CS0213.</span><span class="sxs-lookup"><span data-stu-id="8bf69-108">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
