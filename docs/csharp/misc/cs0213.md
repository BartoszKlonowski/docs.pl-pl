 <span data-ttu-id="a39ce-104">Zmienna lokalna w niebezpiecznej metodzie lub parametr jest już ustalony (na stosie), dlatego nie można przyjąć adresu żadnej z tych dwóch zmiennych w wyrażeniu [stałym](../language-reference/keywords/fixed-statement.md) . ## <a name="example"></a><span data-ttu-id="a39ce-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="a39ce-106">Example</span></span>  
 <span data-ttu-id="a39ce-107">Poniższy przykład generuje CS0213.</span><span class="sxs-lookup"><span data-stu-id="a39ce-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
