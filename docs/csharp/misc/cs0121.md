# <a name="compiler-error-cs0121"></a>Błąd kompilatora CS0121

Wywołanie jest niejednoznaczne między następującymi metodami lub właściwościami: "metoda1" i "method2"

Z powodu niejawnej konwersji kompilator nie może wywołać jeden formularz przeciążonej metody. Można naprawić ten błąd, w następujący sposób:

- Określ parametry metody w taki sposób, że niejawna konwersja nie odbywa się.

- Usuń wszystkie przeciążenia metody.

- Rzutowanie na typ właściwości przed wywołaniem metody.

## <a name="examples"></a>Przykłady

Poniższy przykład generuje błąd kompilatora CS0121:

```csharp
public class C
{
   void f(int i, double d)
   {
   }

   void f(double d, int i)
   {
   }

   public static void Main()
   {
      C c = new C();

      c.f(1, 1);   // CS0121

      // Try the following code instead:
      // c.f(1, 1.0);
      // or
      // c.f(1.0, 1);
      // or
      // c.f(1, (double)1);   // Cast and specify which method to call.
   }
}
```

```csharp
class Program2
{
    static int ol_invoked = 0;

    delegate int D1(int x);
    delegate T D1<T>(T x);
    delegate T D1<T, U>(U u);

    static void F(D1 d1) { ol_invoked = 1; }
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }

    static int Test001()
    {
        F(delegate(int x) { return 1; }); // CS0121
        if (ol_invoked == 1)
            return 0;
        else
            return 1;
    }

    static int Main()
    {
        return Test001();
    }
}
```
