# <a name="compiler-error-cs0415"></a>CS0415 błąd kompilatora
Atrybut "IndexerName" jest prawidłowy tylko w indeksatorze, który nie jest jawną deklaracją składowej interfejsu  
  
 Ten błąd występuje, jeśli używasz atrybutu IndexerName w indeksatorze, który był jawnych implementacji interfejsu. Można uniknąć tego błędu, usuwając nazwę interfejsu z deklaracji indeksatora, jeśli jest to możliwe. Aby uzyskać więcej informacji, zobacz [klasy IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).  
  
 Poniższy przykład spowoduje wygenerowanie CS0415:  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
