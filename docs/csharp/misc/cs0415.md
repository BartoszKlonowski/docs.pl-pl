# <a name="compiler-error-cs0415"></a><span data-ttu-id="8f172-102">CS0415 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="8f172-102">Compiler Error CS0415</span></span>
<span data-ttu-id="8f172-103">Atrybut "IndexerName" jest prawidłowy tylko w indeksatorze, który nie jest jawną deklaracją składowej interfejsu</span><span class="sxs-lookup"><span data-stu-id="8f172-103">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="8f172-104">Ten błąd występuje, jeśli używasz atrybutu IndexerName w indeksatorze, który był jawnych implementacji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8f172-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="8f172-105">Można uniknąć tego błędu, usuwając nazwę interfejsu z deklaracji indeksatora, jeśli jest to możliwe.</span><span class="sxs-lookup"><span data-stu-id="8f172-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="8f172-106">Aby uzyskać więcej informacji, zobacz [klasy IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="8f172-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="8f172-107">Poniższy przykład spowoduje wygenerowanie CS0415:</span><span class="sxs-lookup"><span data-stu-id="8f172-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
