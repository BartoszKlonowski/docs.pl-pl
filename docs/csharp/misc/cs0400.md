<span data-ttu-id="cf915-104">Nie można znaleźć nazwy typu lub przestrzeni nazw "identifier" w globalnej przestrzeni nazw (czy nie brakuje odwołania do zestawu?)</span><span class="sxs-lookup"><span data-stu-id="cf915-104">The type or namespace name 'identifier' could not be found in the global namespace (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="cf915-105">Nie znaleziono identyfikatora z zakresem globalnym operatora ( `::` ) w globalnej przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="cf915-105">The identifier scoped with the global scope operator (`::`) was not found in the global namespace.</span></span> <span data-ttu-id="cf915-106">Brak odwołania do zestawu, który zawiera identyfikator lub identyfikator może być zadeklarowany w klasie lub przestrzeni nazw innej niż globalna przestrzeń nazw.</span><span class="sxs-lookup"><span data-stu-id="cf915-106">You may be missing an assembly reference that contains the identifier, or the identifier may be declared in a class or namespace other than the global namespace.</span></span> <span data-ttu-id="cf915-107">Ten błąd może również wystąpić, jeśli identyfikator o zakresie globalnym nie został zadeklarowany lub jest błędnie napisany.</span><span class="sxs-lookup"><span data-stu-id="cf915-107">This error could also occur if a globally scoped identifier is not declared or is misspelled.</span></span>  
  
 <span data-ttu-id="cf915-108">Aby uniknąć tego błędu, Znajdź deklarację identyfikatora, sprawdź poprawność pisowni i jeśli deklaracja znajduje się w osobnym zestawie, upewnij się, że masz odpowiednie odwołanie do zestawu.</span><span class="sxs-lookup"><span data-stu-id="cf915-108">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="cf915-109">Jeśli identyfikator jest zadeklarowany wewnątrz innego typu lub przestrzeni nazw, należy użyć w pełni kwalifikowanej nazwy po::.</span><span class="sxs-lookup"><span data-stu-id="cf915-109">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="cf915-110">Poniższy przykład generuje CS0400:</span><span class="sxs-lookup"><span data-stu-id="cf915-110">The following sample generates CS0400:</span></span>

        // CS0400 - D could not be found

   }
