# <a name="compiler-error-cs1719"></a>CS1719 błąd kompilatora
Błąd podczas odczytu pliku zasobów Win32 "Nazwa pliku"--"Przyczyna"  
  
 Próba odczytu pliku zasobów Win32 nie powiodło się z powodu podane w dzienniku błędów, zwykle coś takich jak "nie można odnaleźć pliku" lub "odmowa dostępu". Ten błąd został rozwiązany przez rozwiązania problemu opisanego przez przyczyny.
