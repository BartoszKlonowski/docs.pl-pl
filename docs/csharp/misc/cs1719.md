# <a name="compiler-error-cs1719"></a>CS1719 błąd kompilatora
Wystąpił błąd podczas odczytu pliku zasobów Win32 "Nazwa pliku"--"reason"  
  
 Próba odczytu pliku zasobów Win32 nie powiodło się z powodu podane w wyniku błędu, zwykle mielibyśmy mieć czegoś podobnego "nie można odnaleźć pliku" lub "odmowa dostępu". Ten błąd został rozwiązany, rozwiązując problem opisany przez podanie przyczyny.
