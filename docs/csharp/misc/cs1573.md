# <a name="compiler-warning-level-4-cs1573"></a>Kompilatora (poziom 4) ostrzeżenie CS1573
Parametr "parametru" ma nie pasującego tagu param w komentarzu XML dla elementu "parameter" (ale inne parametry mają)  
  
 Korzystając z [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md) — opcja kompilatora, komentarz został określony dla niektórych, ale nie wszystkie parametry w metodzie. Użytkownik zapomniał wprowadzić komentarz dla tych parametrów.  
  
 Poniższy przykład spowoduje wygenerowanie CS1573:  
  
```csharp  
// CS1573.cs  
// compile with: /W:4  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   // enter a comment for Char1?  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   public static void Main ()  
   {  
   }  
}  
```
