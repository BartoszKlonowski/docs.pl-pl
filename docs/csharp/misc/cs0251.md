# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="92bac-102">CS0251 ostrzeżenie (poziom 2) kompilator</span><span class="sxs-lookup"><span data-stu-id="92bac-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="92bac-103">Indeksowanie tablicy z ujemnym indeksem (tablicy wskaźników zawsze rozpoczynają się od zera)</span><span class="sxs-lookup"><span data-stu-id="92bac-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="92bac-104">Nie należy używać liczbą ujemną, indeks do tablicy.</span><span class="sxs-lookup"><span data-stu-id="92bac-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="92bac-105">Poniższy przykład spowoduje wygenerowanie CS0251:</span><span class="sxs-lookup"><span data-stu-id="92bac-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
