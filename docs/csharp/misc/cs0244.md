 <span data-ttu-id="631ad-104">[Jest](../../csharp/language-reference/keywords/is.md) i [jako](../../csharp/language-reference/keywords/as.md) słowa kluczowe nie są prawidłowe w typach wskaźnika do użytku.</span><span class="sxs-lookup"><span data-stu-id="631ad-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="631ad-105">Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="631ad-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="631ad-106">Poniższy przykład spowoduje wygenerowanie CS0244:</span><span class="sxs-lookup"><span data-stu-id="631ad-106">The following sample generates CS0244:</span></span>  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
