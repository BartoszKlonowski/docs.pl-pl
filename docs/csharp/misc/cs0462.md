<span data-ttu-id="7c233-103">Dziedziczone elementy członkowskie "Członek1" i "member2" mają taką samą sygnaturę w typie "type", dlatego nie mogą być przesłaniane</span><span class="sxs-lookup"><span data-stu-id="7c233-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="7c233-104">Ten błąd pojawia się wraz z wprowadzeniem strony Ogólne.</span><span class="sxs-lookup"><span data-stu-id="7c233-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="7c233-105">Zwykle nie może mieć dwie wersje metody w klasie z tym samym podpisie.</span><span class="sxs-lookup"><span data-stu-id="7c233-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="7c233-106">Z ogólnymi, można jednak określić Metoda ogólna, który może być zduplikowana innej metody, jeśli zostanie on uruchomiony z określonym typem.</span><span class="sxs-lookup"><span data-stu-id="7c233-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7c233-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="7c233-107">Example</span></span>  
 <span data-ttu-id="7c233-108">Gdy `C<int>` zostanie uruchomiony, dwie wersje metody `F` są tworzone o tej samej sygnaturze, więc przesłonięcia w klasie `D` nie można zdecydować, które z nich ma dotyczyć zastąpienie do.</span><span class="sxs-lookup"><span data-stu-id="7c233-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="7c233-109">Poniższy przykład generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="7c233-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
