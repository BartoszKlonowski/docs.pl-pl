# <a name="compiler-error-cs0462"></a><span data-ttu-id="55746-102">Błąd kompilatora CS0462</span><span class="sxs-lookup"><span data-stu-id="55746-102">Compiler Error CS0462</span></span>
<span data-ttu-id="55746-103">Dziedziczone elementy członkowskie "member1" i "member2" mają ten sam podpis typu typu, więc nie można ich zastąpić</span><span class="sxs-lookup"><span data-stu-id="55746-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="55746-104">Ten błąd pojawia się wraz z wprowadzeniem leków generycznych.</span><span class="sxs-lookup"><span data-stu-id="55746-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="55746-105">Zwykle nie można mieć dwie wersje metody w klasie o tym samym podpisie.</span><span class="sxs-lookup"><span data-stu-id="55746-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="55746-106">Jednak w przypadku typów ogólnych można określić metodę rodzajową, która może zduplikować inną metodę, jeśli jest ona tworzone z określonego typu.</span><span class="sxs-lookup"><span data-stu-id="55746-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55746-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="55746-107">Example</span></span>  
 <span data-ttu-id="55746-108">Po `C<int>` utworzeniu wystąpienia dwie wersje `F` metody są tworzone z tym samym podpisem, więc zastąpienie w klasie `D` nie może zdecydować, który z nich zastosować zastąpienie.</span><span class="sxs-lookup"><span data-stu-id="55746-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="55746-109">Poniższy przykład generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="55746-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
