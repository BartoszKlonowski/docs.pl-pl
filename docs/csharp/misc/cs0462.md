# <a name="compiler-error-cs0462"></a><span data-ttu-id="dfb3e-103">Błąd kompilatora CS0462</span><span class="sxs-lookup"><span data-stu-id="dfb3e-103">Compiler Error CS0462</span></span>
<span data-ttu-id="dfb3e-104">Dziedziczone składowe "member1" i "member2" mają tę samą sygnaturę w typie "Type", więc nie można ich przesłonić</span><span class="sxs-lookup"><span data-stu-id="dfb3e-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="dfb3e-105">Ten błąd występuje w przypadku wprowadzenia typów ogólnych.</span><span class="sxs-lookup"><span data-stu-id="dfb3e-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="dfb3e-106">Zwykle nie mogą istnieć dwie wersje metody w klasie o tej samej sygnaturze.</span><span class="sxs-lookup"><span data-stu-id="dfb3e-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="dfb3e-107">Jednak przy użyciu typów ogólnych można określić metodę rodzajową, która może duplikować inną metodę, jeśli jest tworzona przy użyciu określonego typu.</span><span class="sxs-lookup"><span data-stu-id="dfb3e-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dfb3e-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="dfb3e-108">Example</span></span>  
 <span data-ttu-id="dfb3e-109">Gdy `C<int>` jest tworzone wystąpienie, dwie wersje metody `F` są tworzone z tą samą sygnaturą, więc przesłonięcie klasy `D` nie może zdecydować, która z nich ma zastosować przesłonięcie.</span><span class="sxs-lookup"><span data-stu-id="dfb3e-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="dfb3e-110">Poniższy przykład generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="dfb3e-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
