# <a name="compiler-warning-level-4-cs0402"></a>Kompilatora (poziom 4) ostrzeżenie CS0402
'Identyfikator': punkt wejścia nie może być elementem ogólnym ani typu ogólnego  
  
 Punkt wejścia został znaleziony w typie ogólnym. Aby usunąć to ostrzeżenie, należy zaimplementować Main w nieogólnej klasie lub strukturze.  
  
```csharp  
// CS0402.cs  
// compile with: /W:4  
class C<T>  
{  
   public static void Main()  // CS0402  
   {  
  
   }  
}  
  
class CMain  
{  
   public static void Main() {}  
}  
```
