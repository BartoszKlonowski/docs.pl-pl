## <a name="example"></a><span data-ttu-id="216fa-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="216fa-105">Example</span></span>  
 <span data-ttu-id="216fa-106">Poniższy przykład spowoduje wygenerowanie CS1722.</span><span class="sxs-lookup"><span data-stu-id="216fa-106">The following sample generates CS1722.</span></span>
