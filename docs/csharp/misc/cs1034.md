# <a name="compiler-error-cs1034"></a>CS1034 błąd kompilatora
Przekroczono ograniczenie kompilatora: Wiersz nie może przekraczać 'Liczba' znaków  
  
 Maksymalną liczbę znaków dozwoloną w wierszu jest 16,777,214.
