<span data-ttu-id="8dab9-104">Ograniczenia dotyczące zastąpień i jawnych metod implementacji interfejsu są dziedziczone z metody podstawowej, dlatego nie można ich określić bezpośrednio</span><span class="sxs-lookup"><span data-stu-id="8dab9-104">Constraints for override and explicit interface implementation methods are inherited from the base method, so they cannot be specified directly</span></span>

## <a name="example"></a><span data-ttu-id="8dab9-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="8dab9-107">Example</span></span>  

 <span data-ttu-id="8dab9-108">Poniższy przykład generuje CS0460.</span><span class="sxs-lookup"><span data-stu-id="8dab9-108">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
