<span data-ttu-id="c515a-103">Ograniczenia dla przesłoniętych i jawnych metod implementacji interfejsu są dziedziczone z metody podstawowej, dlatego nie może być określone bezpośrednio</span><span class="sxs-lookup"><span data-stu-id="c515a-103">Constraints for override and explicit interface implementation methods are inherited from the base method, so they cannot be specified directly</span></span>

## <a name="example"></a><span data-ttu-id="c515a-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="c515a-106">Example</span></span>  
 <span data-ttu-id="c515a-107">Poniższy przykład spowoduje wygenerowanie CS0460.</span><span class="sxs-lookup"><span data-stu-id="c515a-107">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass   
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
