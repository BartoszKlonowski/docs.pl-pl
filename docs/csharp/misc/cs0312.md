## <a name="example"></a><span data-ttu-id="36b22-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="36b22-111">Example</span></span>

    static void MTyVar<T, U>() where T : U { }

        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }
