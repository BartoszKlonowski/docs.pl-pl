# <a name="compiler-error-cs0312"></a>Błąd kompilatora CS0312
Typ "type1" nie może być używany jako parametr typu "name" w typie ogólnym lub metodzie "name". Typ dopuszczający wartość null "type1" nie spełnia ograniczenia elementu "type2".  
  
 Typ dopuszczający wartość null jest różny od jego odpowiednika niedopuszczający wartości null; między nimi nie istnieje niejawna konwersja odwołań ani nie ma identyfikacji żadnej konwersji. Konwersja pakującej do wartości null nie spełnia ograniczenia typu ogólnego. W poniższym przykładzie pierwszy parametr typu jest `Nullable<int>` a i drugi parametr typu `System.Int32`jest.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń ograniczenie.  
  
2. W poniższym przykładzie należy wprowadzić drugi argument `int?` typu lub. `object`  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0312:  
  
```csharp  
// cs0312.cs  
class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```  
  
 Chociaż typ dopuszczający wartość null jest różny od typu niedopuszczający wartości null, różne rodzaje konwersji są dozwolone między wartościami null i niedopuszczające wartości null.
