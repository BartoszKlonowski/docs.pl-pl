# <a name="compiler-error-cs0220"></a><span data-ttu-id="7cce5-102">Błąd kompilatora CS0220</span><span class="sxs-lookup"><span data-stu-id="7cce5-102">Compiler Error CS0220</span></span>
<span data-ttu-id="7cce5-103">Operacja przekroczy przepływ w czasie kompilacji w trybie sprawdzonym</span><span class="sxs-lookup"><span data-stu-id="7cce5-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="7cce5-104">Wykryto, że operacja [](../language-reference/keywords/checked.md)została sprawdzona, co jest ustawieniem domyślnym i utracono utratę danych.</span><span class="sxs-lookup"><span data-stu-id="7cce5-104">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="7cce5-105">Popraw dane wejściowe do przypisania lub użyj opcji [niezaznaczone](../language-reference/keywords/unchecked.md) , aby usunąć ten błąd.</span><span class="sxs-lookup"><span data-stu-id="7cce5-105">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="7cce5-106">Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="7cce5-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="7cce5-107">Poniższy przykład generuje CS0220:</span><span class="sxs-lookup"><span data-stu-id="7cce5-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
