# <a name="compiler-error-cs0220"></a>CS0220 błąd kompilatora
Przepełnienie w operacji w czasie kompilacji w trybie zaznaczone  
  
 Operacja została wykryta przez [zaznaczone](../../csharp/language-reference/keywords/checked.md), co jest ustawieniem domyślnym i spowodowała utratę danych. Albo Popraw dane wejściowe do przypisania lub użyj [unchecked](../../csharp/language-reference/keywords/unchecked.md) Aby rozwiązać ten problem. Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0220:  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
