# <a name="compiler-error-cs0220"></a><span data-ttu-id="d4603-103">Błąd kompilatora CS0220</span><span class="sxs-lookup"><span data-stu-id="d4603-103">Compiler Error CS0220</span></span>

<span data-ttu-id="d4603-104">Operacja przekroczy przepływ w czasie kompilacji w trybie sprawdzonym</span><span class="sxs-lookup"><span data-stu-id="d4603-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="d4603-105">Wykryto, że operacja została [sprawdzona](../language-reference/keywords/checked.md), co jest ustawieniem domyślnym i utracono utratę danych.</span><span class="sxs-lookup"><span data-stu-id="d4603-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="d4603-106">Popraw dane wejściowe do przypisania lub użyj opcji [niezaznaczone](../language-reference/keywords/unchecked.md) , aby usunąć ten błąd.</span><span class="sxs-lookup"><span data-stu-id="d4603-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="d4603-107">Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="d4603-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="d4603-108">Poniższy przykład generuje CS0220:</span><span class="sxs-lookup"><span data-stu-id="d4603-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
