<span data-ttu-id="6fcd9-103">Przepełnienie w operacji w czasie kompilacji w trybie zaznaczone</span><span class="sxs-lookup"><span data-stu-id="6fcd9-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="6fcd9-104">Operacja została wykryta przez [zaznaczone](../../csharp/language-reference/keywords/checked.md), co jest ustawieniem domyślnym i spowodowała utratę danych.</span><span class="sxs-lookup"><span data-stu-id="6fcd9-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="6fcd9-105">Albo Popraw dane wejściowe do przypisania lub użyj [unchecked](../../csharp/language-reference/keywords/unchecked.md) Aby rozwiązać ten problem.</span><span class="sxs-lookup"><span data-stu-id="6fcd9-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="6fcd9-106">Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="6fcd9-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="6fcd9-107">Poniższy przykład spowoduje wygenerowanie CS0220:</span><span class="sxs-lookup"><span data-stu-id="6fcd9-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
