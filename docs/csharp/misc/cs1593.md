# <a name="compiler-error-cs1593"></a>Błąd kompilatora CS1593
Obiekt delegowany "del" nie przyjmuje argumentów "number"  
  
 Liczba argumentów przesłanych do wywołania [delegata](../language-reference/keywords/delegate.md) nie zgadza się z liczbą parametrów w deklaracji delegata.  
  
 Poniższy przykład generuje CS1593:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
