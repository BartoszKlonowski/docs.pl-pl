# <a name="compiler-error-cs1593"></a><span data-ttu-id="dd3ad-103">Błąd kompilatora CS1593</span><span class="sxs-lookup"><span data-stu-id="dd3ad-103">Compiler Error CS1593</span></span>

<span data-ttu-id="dd3ad-104">Obiekt delegowany "del" nie przyjmuje argumentów "number"</span><span class="sxs-lookup"><span data-stu-id="dd3ad-104">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="dd3ad-105">Liczba argumentów przesłanych do wywołania [delegata](../language-reference/builtin-types/reference-types.md) nie zgadza się z liczbą parametrów w deklaracji delegata.</span><span class="sxs-lookup"><span data-stu-id="dd3ad-105">The number of arguments passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="dd3ad-106">Poniższy przykład generuje CS1593:</span><span class="sxs-lookup"><span data-stu-id="dd3ad-106">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
