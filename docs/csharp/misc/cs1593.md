# <a name="compiler-error-cs1593"></a><span data-ttu-id="31971-102">CS1593 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="31971-102">Compiler Error CS1593</span></span>
<span data-ttu-id="31971-103">Delegat "del" nie przyjmuje argumentów "liczba"</span><span class="sxs-lookup"><span data-stu-id="31971-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="31971-104">Liczba argumentów przekazanych do [delegować](../../csharp/language-reference/keywords/delegate.md) wywołania nie zgadza się z liczbą parametrów w deklaracja delegata.</span><span class="sxs-lookup"><span data-stu-id="31971-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="31971-105">Poniższy przykład spowoduje wygenerowanie CS1593:</span><span class="sxs-lookup"><span data-stu-id="31971-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
