# <a name="compiler-error-cs1022"></a>CS1022 błąd kompilatora
Definicja typu lub przestrzeni nazw lub Oczekiwano końca z pliku  
  
 Plik kodu źródłowego nie ma pasującego zestaw nawiasów klamrowych.  
  
 Poniższy przykład spowoduje wygenerowanie CS1022:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
