# <a name="compiler-error-cs1022"></a>Błąd kompilatora CS1022
Oczekiwano definicji typu lub przestrzeni nazw albo znacznika końca pliku  
  
 Plik kodu źródłowego nie ma zgodnego zestawu nawiasów klamrowych.  
  
 Poniższy przykład generuje CS1022:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
