# <a name="compiler-error-cs0245"></a>CS0245 błąd kompilatora
Destruktory i obiekt. Finalizowanie nie można wywołać bezpośrednio. Należy wziąć pod uwagę, wywołanie metody IDisposable.Dispose, jeśli jest dostępny.  
  
 Aby uzyskać więcej informacji, zobacz [— podstawy programowania dla wyrzucania elementów bezużytecznych](../../standard/garbage-collection/index.md) i [destruktory](../../csharp/programming-guide/classes-and-structs/destructors.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0245:  
  
```csharp  
// CS0245.cs  
using System;  
using System.Collections;  
  
class MyClass // : IDisposable  
{  
   /*  
   public void Dispose()  
   {  
      // cleanup code goes here  
   }  
   */  
  
   void m()  
   {  
      this.Finalize();   // CS0245  
      // this.Dispose();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
