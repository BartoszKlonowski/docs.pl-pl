<span data-ttu-id="dbabf-103">Metoda ani delegat nie może zwracać typu "type"</span><span class="sxs-lookup"><span data-stu-id="dbabf-103">Method or delegate cannot return type 'type'</span></span>  
  
 <span data-ttu-id="dbabf-104">Niektóre typy w bibliotece klas .NET Framework, na przykład <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> i <xref:System.ArgIterator> nie można użyć jako typy zwracane ponieważ potencjalnie może być używane do wykonywania operacji niebezpieczny.</span><span class="sxs-lookup"><span data-stu-id="dbabf-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as return types because they can potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="dbabf-105">Poniższy przykład generuje CS1599:</span><span class="sxs-lookup"><span data-stu-id="dbabf-105">The following sample generates CS1599:</span></span>  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
