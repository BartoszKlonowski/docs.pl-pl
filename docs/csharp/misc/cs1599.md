# <a name="compiler-error-cs1599"></a>Błąd kompilatora CS1599
Metoda lub obiekt delegowany nie może zwracać typu "Type"  
  
 Niektórych typów w bibliotece klas .NET Framework, na przykład, <xref:System.TypedReference> <xref:System.RuntimeArgumentHandle> i <xref:System.ArgIterator> nie można używać jako typów zwracanych, ponieważ mogą one być używane do wykonywania niebezpiecznych operacji.  
  
 Poniższy przykład generuje CS1599:  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
