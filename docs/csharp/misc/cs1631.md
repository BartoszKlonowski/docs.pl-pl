# <a name="compiler-error-cs1631"></a><span data-ttu-id="46c59-103">Błąd kompilatora CS1631</span><span class="sxs-lookup"><span data-stu-id="46c59-103">Compiler Error CS1631</span></span>
<span data-ttu-id="46c59-104">Nie można przekazać wartości w treści klauzuli catch</span><span class="sxs-lookup"><span data-stu-id="46c59-104">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="46c59-105">Instrukcja Yield nie jest dozwolona w treści klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="46c59-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="46c59-106">Aby uniknąć tego błędu, Przenieś instrukcję Yield poza treść klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="46c59-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="46c59-107">Poniższy przykład generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="46c59-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
