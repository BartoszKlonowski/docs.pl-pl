# <a name="compiler-error-cs1631"></a>CS1631 błąd kompilatora
Nie można przekazać wartości w treści klauzuli catch  
  
 Instrukcji yield w treści klauzuli "catch" nie jest dozwolone. Aby uniknąć tego błędu, należy przenieść instrukcji yield poza treścią klauzuli "catch".  
  
 Poniższy przykład spowoduje wygenerowanie CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
