# <a name="compiler-error-cs1631"></a><span data-ttu-id="9d963-102">CS1631 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="9d963-102">Compiler Error CS1631</span></span>
<span data-ttu-id="9d963-103">Nie można użyć instrukcji yield z wartością w treści klauzuli catch</span><span class="sxs-lookup"><span data-stu-id="9d963-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="9d963-104">Instrukcji yield w treści klauzuli catch nie jest dozwolone.</span><span class="sxs-lookup"><span data-stu-id="9d963-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="9d963-105">Aby uniknąć tego błędu, należy przenieść poza treści klauzuli catch instrukcji yield.</span><span class="sxs-lookup"><span data-stu-id="9d963-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="9d963-106">Poniższy przykład generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="9d963-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
