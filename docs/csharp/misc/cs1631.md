# <a name="compiler-error-cs1631"></a><span data-ttu-id="59ffb-102">Błąd kompilatora CS1631</span><span class="sxs-lookup"><span data-stu-id="59ffb-102">Compiler Error CS1631</span></span>
<span data-ttu-id="59ffb-103">Nie można uzyskać wartości w treści klauzuli catch</span><span class="sxs-lookup"><span data-stu-id="59ffb-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="59ffb-104">Yield Instrukcji nie jest dozwolone z treści catch klauzuli.</span><span class="sxs-lookup"><span data-stu-id="59ffb-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="59ffb-105">Aby uniknąć tego błędu, przenieś yield instrukcji poza treści catch klauzuli.</span><span class="sxs-lookup"><span data-stu-id="59ffb-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="59ffb-106">Poniższy przykład generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="59ffb-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
