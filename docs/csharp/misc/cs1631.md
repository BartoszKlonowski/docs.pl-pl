# <a name="compiler-error-cs1631"></a>Błąd kompilatora CS1631
Nie można przekazać wartości w treści klauzuli catch  
  
 Instrukcja Yield nie jest dozwolona w treści klauzuli catch. Aby uniknąć tego błędu, Przenieś instrukcję Yield poza treść klauzuli catch.  
  
 Poniższy przykład generuje CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
