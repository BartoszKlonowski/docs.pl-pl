 <span data-ttu-id="61e69-104">[To](../language-reference/keywords/this.md) słowo kluczowe odwołuje się do obiektu, który jest wystąpieniem typu.</span><span class="sxs-lookup"><span data-stu-id="61e69-104">The [this](../language-reference/keywords/this.md) keyword refers to an object, which is an instance of a type.</span></span> <span data-ttu-id="61e69-105">Ponieważ metody statyczne są niezależne od dowolnego wystąpienia klasy zawierającej, słowo kluczowe "This" jest bez znaczenia i dlatego jest niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="61e69-105">Since static methods are independent of any instance of the containing class, the "this" keyword is meaningless and is therefore not allowed.</span></span> <span data-ttu-id="61e69-106">Aby uzyskać więcej informacji, zobacz [klasy statyczne i statyczne elementy członkowskie klas](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md) i [obiekty](../programming-guide/classes-and-structs/objects.md).</span><span class="sxs-lookup"><span data-stu-id="61e69-106">For more information, see [Static Classes and Static Class Members](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md) and [Objects](../programming-guide/classes-and-structs/objects.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="61e69-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="61e69-107">Example</span></span>  
 <span data-ttu-id="61e69-108">Poniższy przykład generuje CS0026:</span><span class="sxs-lookup"><span data-stu-id="61e69-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
