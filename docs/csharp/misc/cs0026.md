## <a name="example"></a><span data-ttu-id="f6c3c-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="f6c3c-107">Example</span></span>  
 <span data-ttu-id="f6c3c-108">Poniższy przykład generuje CS0026:</span><span class="sxs-lookup"><span data-stu-id="f6c3c-108">The following example generates CS0026:</span></span>

```csharp  
   public static int i = 0;  
  
   public static void Main()  
   {

   }  
}  
```
