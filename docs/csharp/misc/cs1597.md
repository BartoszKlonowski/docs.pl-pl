# <a name="compiler-error-cs1597"></a>Błąd kompilatora CS1597
Użycie średnika po bloku metody lub akcesora jest nieprawidłowe  
  
 Średniki nie są konieczne (lub są dozwolone), aby zakończyć blok metody lub akcesora.  
  
 Poniższy przykład generuje CS1597:  
  
```csharp  
// CS1597.cs  
class TestClass  
{  
   public static void Main()  
   {  
   };   // CS1597, remove semicolon  
}  
```
