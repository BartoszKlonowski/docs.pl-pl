# <a name="compiler-warning-level-1-cs1570"></a>Ostrzeżenie kompilatora (poziom 1) CS1570
Komentarz XML w konstrukcji "konstrukcja" ma nieprawidłowo sformułowany kod XML — "Przyczyna"  
  
 W przypadku korzystania z [/doc](../language-reference/compiler-options/doc-compiler-option.md)wszystkie komentarze w kodzie źródłowym muszą być w formacie XML. Każdy błąd ze znacznikiem XML spowoduje wygenerowanie CS1570. Na przykład:  
  
- W przypadku przekazywania ciągu do elementu **cref**, takiego jak w [ \<tagu > wyjątku](../programming-guide/xmldoc/exception.md) , ciąg musi być ujęty w znaki podwójnego cudzysłowu.  
  
- Jeśli używasz znacznika, takiego jak [ \<seealso — >](../programming-guide/xmldoc/seealso.md), który nie ma taga zamykającego, musisz określić ukośnik do przodu przed zamykającym nawiasem ostrym.  
  
- Jeśli musisz użyć symbolu o wartości większej niż lub mniejszej niż ", musisz przedstawić je za pomocą **&gt;** lub. **&lt;**  
  
- Brak atrybutu pliku lub ścieżki w [ \<tagu include >](../programming-guide/xmldoc/include.md) lub nie został on poprawnie sformułowany.  
  
 Poniższy przykład generuje CS1570:  
  
```csharp  
// CS1570.cs  
// compile with: /W:1  
namespace ns  
{  
   // the following line generates CS1570  
   /// <summary> returns true if < 5 </summary>  
   // try this instead  
   // /// <summary> returns true if <5 </summary>  
  
   public class MyClass  
   {  
      public static void Main ()  
      {  
      }  
   }  
}  
```
