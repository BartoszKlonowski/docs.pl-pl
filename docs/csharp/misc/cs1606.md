# <a name="compiler-error-cs1606"></a>CS1606 błąd kompilatora
Podpisywanie zestawu nie powiodła się; dane wyjściowe nie mogą być podpisane — przyczyny  
  
 Zestaw został utworzony, ale próba kompilator na zakończenie podpisaniem wystąpił błąd.
