# <a name="compiler-error-cs1606"></a>CS1606 błąd kompilatora
Podpisywanie zestawu nie powiodła się; wynik może nie być podpisany — przyczyny  
  
 Zestaw został utworzony, ale gdy kompilator próbował zakończyć rejestrowania jej, wystąpił błąd.
