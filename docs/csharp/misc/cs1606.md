# <a name="compiler-error-cs1606"></a>Błąd kompilatora CS1606

Podpisywanie zestawu nie powiodło się; dane wyjściowe mogą nie być podpisane — Przyczyna  
  
 Zestaw został utworzony, ale gdy kompilator próbował zakończyć podpisywanie, wystąpił błąd.
