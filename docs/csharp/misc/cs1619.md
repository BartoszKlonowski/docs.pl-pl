# <a name="compiler-error-cs1619"></a>Błąd kompilatora CS1619

Nie można utworzyć pliku tymczasowego "filename" — Przyczyna  
  
 Kompilator nie może utworzyć pliku tymczasowego dla danego powodu (na przykład dysk jest pełny).
