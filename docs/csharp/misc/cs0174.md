# <a name="compiler-error-cs0174"></a>CS0174 błąd kompilatora
Klasa podstawowa jest wymagana dla odwołania "base"  
  
 Ten błąd występuje tylko gdy .NET Framework środowisko uruchomieniowe języka wspólnego kompiluje kod źródłowy `System.Object` klasy, która jest to jedyna klasa, która ma klasy podstawowej.
