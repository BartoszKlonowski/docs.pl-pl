# <a name="compiler-error-cs0174"></a>Błąd kompilatora CS0174
Klasa bazowa jest wymagana dla odwołania "Base"  
  
 Ten błąd występuje tylko wtedy, gdy .NET Framework środowisko uruchomieniowe języka wspólnego kompiluje kod źródłowy dla `System.Object` klasy, która jest jedyną klasą, która nie ma klasy bazowej.
