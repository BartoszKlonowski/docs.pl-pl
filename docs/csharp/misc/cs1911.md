# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="6e392-102">Kompilatora (poziom 1) ostrzeżenie CS1911</span><span class="sxs-lookup"><span data-stu-id="6e392-102">Compiler Warning (level 1) CS1911</span></span>
<span data-ttu-id="6e392-103">Dostęp do elementu członkowskiego "name", za pomocą słowa kluczowego "base" z metody anonimowej, wyrażenia lambda, wyrażenia zapytania lub iteratora powoduje zweryfikowanie kodu.</span><span class="sxs-lookup"><span data-stu-id="6e392-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="6e392-104">Przenieś dostęp do metody pomocnika dla typu zawierającego.</span><span class="sxs-lookup"><span data-stu-id="6e392-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="6e392-105">Podczas wywoływania funkcji wirtualnych za pomocą `base` — słowo kluczowe wewnątrz treści metody iteratora lub anonimowymi spowodują zweryfikowanie kodu.</span><span class="sxs-lookup"><span data-stu-id="6e392-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="6e392-106">Zweryfikowanie kodu nie będzie można uruchamiać w środowisku częściowej relacji zaufania.</span><span class="sxs-lookup"><span data-stu-id="6e392-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="6e392-107">Jeden dla CS1911 jest przenoszenie funkcji wirtualnej wywołanie funkcji pomocnika.</span><span class="sxs-lookup"><span data-stu-id="6e392-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6e392-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="6e392-108">Example</span></span>  
 <span data-ttu-id="6e392-109">Poniższy przykład spowoduje wygenerowanie CS1911.</span><span class="sxs-lookup"><span data-stu-id="6e392-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
