# <a name="compiler-error-cs0066"></a><span data-ttu-id="ff19c-103">Błąd kompilatora CS0066</span><span class="sxs-lookup"><span data-stu-id="ff19c-103">Compiler Error CS0066</span></span>
<span data-ttu-id="ff19c-104">"Event": zdarzenie musi być typu delegowanego</span><span class="sxs-lookup"><span data-stu-id="ff19c-104">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="ff19c-105">Słowo kluczowe zdarzenia wymaga typu [delegata](../language-reference/builtin-types/reference-types.md) .</span><span class="sxs-lookup"><span data-stu-id="ff19c-105">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="ff19c-106">Aby uzyskać więcej informacji, zobacz [zdarzenia](../programming-guide/events/index.md) i [Delegaty](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="ff19c-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="ff19c-107">Poniższy przykład generuje CS0066:</span><span class="sxs-lookup"><span data-stu-id="ff19c-107">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
