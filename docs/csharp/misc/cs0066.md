<span data-ttu-id="208d1-103">"event": zdarzenie musi być typu delegata</span><span class="sxs-lookup"><span data-stu-id="208d1-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="208d1-104">Event, słowo kluczowe wymaga [delegować](../../csharp/language-reference/keywords/delegate.md) typu.</span><span class="sxs-lookup"><span data-stu-id="208d1-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="208d1-105">Aby uzyskać więcej informacji, zobacz [zdarzenia](../../csharp/programming-guide/events/index.md) i [delegatów](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="208d1-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="208d1-106">Poniższy przykład generuje CS0066:</span><span class="sxs-lookup"><span data-stu-id="208d1-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
