# <a name="compiler-error-cs0439"></a>CS0439 błąd kompilatora
Deklaracja aliasu zewnętrznego musi poprzedzać wszystkie inne elementy zdefiniowane w przestrzeni nazw  
  
 Ten błąd występuje, gdy `extern` deklaracji przychodzi po czymś innym, takich jak `using` deklaracji w tej samej przestrzeni nazw. `extern` Deklaracji musi występować przed wszystkimi innymi elementami w przestrzeni nazw.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0439:  
  
```csharp  
// CS0439.cs  
using System;  
  
extern alias MyType;   // CS0439  
// To resolve the error, make the extern alias the first line in the file.  
  
public class Test   
{  
    public static void Main()   
    {  
    }  
}  
```
