# <a name="compiler-error-cs0061"></a>CS0061 błąd kompilatora
Niespójność dostępności: podstawowy interfejs "interfejs 1" jest mniej dostępny niż interfejs "interface 2"  
  
 A [publicznych](../../csharp/language-reference/keywords/public.md) konstrukcja musi zwracać obiekt dostępny publicznie.  
  
 Dostępność interfejsu nie zawężony interfejsu pochodnego. Aby uzyskać więcej informacji, zobacz [interfejsów](../../csharp/programming-guide/interfaces/index.md) i [modyfikatory dostępu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0061.  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
