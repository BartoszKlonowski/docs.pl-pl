 <span data-ttu-id="9f7d0-104">A [publicznych](../../csharp/language-reference/keywords/public.md) konstrukcja musi zwracać obiekt dostępny publicznie.</span><span class="sxs-lookup"><span data-stu-id="9f7d0-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="9f7d0-107">Poniższy przykład spowoduje wygenerowanie CS0061.</span><span class="sxs-lookup"><span data-stu-id="9f7d0-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
