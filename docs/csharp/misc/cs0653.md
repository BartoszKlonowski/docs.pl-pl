# <a name="compiler-error-cs0653"></a><span data-ttu-id="5c5bb-103">Błąd kompilatora CS0653</span><span class="sxs-lookup"><span data-stu-id="5c5bb-103">Compiler Error CS0653</span></span>

 <span data-ttu-id="5c5bb-105">Nie można użyć [abstrakcyjnej](../language-reference/keywords/abstract.md) klasy atrybutów niestandardowych jako atrybutu.</span><span class="sxs-lookup"><span data-stu-id="5c5bb-105">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>  
  
 <span data-ttu-id="5c5bb-106">Poniższy przykład generuje CS0653:</span><span class="sxs-lookup"><span data-stu-id="5c5bb-106">The following sample generates CS0653:</span></span>  
  
```csharp  
// CS0653.cs  
using System;  
  
public abstract class MyAttribute : Attribute  
{  
}  
  
public class My2Attribute : MyAttribute  
{  
}  
  
[My]   // CS0653  
// try the following line instead  
// [My2]

{  
   public static void Main()  
   {  
   }  
}  
```
