 <span data-ttu-id="0124e-104">[Abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) klasy atrybutów niestandardowych nie można użyć jako atrybut.</span><span class="sxs-lookup"><span data-stu-id="0124e-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>

```csharp

{  
}
```
