# <a name="compiler-error-cs0761"></a><span data-ttu-id="5144f-103">Błąd kompilatora CS0761</span><span class="sxs-lookup"><span data-stu-id="5144f-103">Compiler Error CS0761</span></span>
<span data-ttu-id="5144f-104">Częściowe deklaracje metody "Method \<T> " mają niespójne ograniczenia parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="5144f-104">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="5144f-105">Jeśli metoda częściowa ma implementację, ograniczenie typu ogólnego musi być takie samo jak w przypadku ograniczenia zdefiniowanego w podpisie metody.</span><span class="sxs-lookup"><span data-stu-id="5144f-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5144f-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="5144f-106">To correct this error</span></span>  
  
1. <span data-ttu-id="5144f-107">Ustaw ograniczenia typu ogólnego identyczne dla każdej części metody częściowej.</span><span class="sxs-lookup"><span data-stu-id="5144f-107">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5144f-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5144f-108">Example</span></span>  
 <span data-ttu-id="5144f-109">Poniższy kod generuje CS0761:</span><span class="sxs-lookup"><span data-stu-id="5144f-109">The following code generates CS0761:</span></span>  
  
```csharp  
// cs0761.cs  
using System;  
  
public partial class C  
{  
    partial void Part<T>() where T : class;  
    partial void Part<T>() where T : struct // CS0761

    }

    {  
        return 1;  
    }  
}  
```
