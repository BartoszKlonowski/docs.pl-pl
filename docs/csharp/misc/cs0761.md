 <span data-ttu-id="b1de5-104">Jeśli metoda częściowa ma implementację, ograniczenie typu ogólnego musi być takie samo jak w przypadku ograniczenia zdefiniowanego w podpisie metody.</span><span class="sxs-lookup"><span data-stu-id="b1de5-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
