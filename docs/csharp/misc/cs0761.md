# <a name="compiler-error-cs0761"></a><span data-ttu-id="b724d-103">Błąd kompilatora CS0761</span><span class="sxs-lookup"><span data-stu-id="b724d-103">Compiler Error CS0761</span></span>

<span data-ttu-id="b724d-104">Częściowe deklaracje metody "Method \<T> " mają niespójne ograniczenia parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="b724d-104">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="b724d-105">Jeśli metoda częściowa ma implementację, ograniczenie typu ogólnego musi być takie samo jak w przypadku ograniczenia zdefiniowanego w podpisie metody.</span><span class="sxs-lookup"><span data-stu-id="b724d-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
