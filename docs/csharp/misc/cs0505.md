<span data-ttu-id="ef0fd-103">'Członek1': nie można przesłonić, ponieważ 'członek2' nie jest funkcją</span><span class="sxs-lookup"><span data-stu-id="ef0fd-103">'member1': cannot override because 'member2' is not a function</span></span>  
  
 <span data-ttu-id="ef0fd-104">Deklaracja klasy, które próbowano zastąpić inne niż metodę w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="ef0fd-104">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="ef0fd-105">Zastąpienia musi odpowiadać typowi elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="ef0fd-105">Overrides must match the member type.</span></span> <span data-ttu-id="ef0fd-106">Jeśli metoda o takiej samej nazwie jak pożądana jest metodą w klasie bazowej, użyj [nowe](../../csharp/language-reference/keywords/new-modifier.md) (a nie [zastąpienia](../../csharp/language-reference/keywords/override.md)) w deklaracji metody w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="ef0fd-106">If a method with the same name as a method in a base class is desired, use [new](../../csharp/language-reference/keywords/new-modifier.md) (and not [override](../../csharp/language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="ef0fd-107">Poniższy przykład spowoduje wygenerowanie CS0505:</span><span class="sxs-lookup"><span data-stu-id="ef0fd-107">The following sample generates CS0505:</span></span>  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
