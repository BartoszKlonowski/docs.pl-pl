 <span data-ttu-id="0a73b-105">Deklaracja klasy podjęła próbę zastąpienia metody niebędącej klasą bazową.</span><span class="sxs-lookup"><span data-stu-id="0a73b-105">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="0a73b-106">Zastąpienia muszą być zgodne z typem elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="0a73b-106">Overrides must match the member type.</span></span> <span data-ttu-id="0a73b-107">Jeśli wymagana jest metoda o takiej samej nazwie jak metoda w klasie bazowej, należy użyć [nowej](../language-reference/keywords/new-modifier.md) (i nie [przesłonięcia](../language-reference/keywords/override.md)) dla deklaracji metody w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="0a73b-107">If a method with the same name as a method in a base class is desired, use [new](../language-reference/keywords/new-modifier.md) (and not [override](../language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="0a73b-108">Poniższy przykład generuje CS0505:</span><span class="sxs-lookup"><span data-stu-id="0a73b-108">The following sample generates CS0505:</span></span>  
  
```csharp

{  
   public int i;  
}

   public override int i() { return 0; }   // CS0505

```
