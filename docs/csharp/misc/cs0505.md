# <a name="compiler-error-cs0505"></a>CS0505 błąd kompilatora
'Członek1': nie można przesłonić, ponieważ 'członek2' nie jest funkcją  
  
 Deklaracja klasy, które próbowano zastąpić inne niż metodę w klasie bazowej. Zastąpienia musi odpowiadać typowi elementu członkowskiego. Jeśli metoda o takiej samej nazwie jak pożądana jest metodą w klasie bazowej, użyj [nowe](../../csharp/language-reference/keywords/new.md) (a nie [zastąpienia](../../csharp/language-reference/keywords/override.md)) w deklaracji metody w klasie bazowej.  
  
 Poniższy przykład spowoduje wygenerowanie CS0505:  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
