# <a name="compiler-error-cs1673"></a><span data-ttu-id="29c22-102">CS1673 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="29c22-102">Compiler Error CS1673</span></span>
<span data-ttu-id="29c22-103">Metod anonimowych, wyrażeń lambda i wyrażenia zapytań wewnątrz struktury nie mogą uzyskać dostęp do składowych wystąpienia obiektu "this".</span><span class="sxs-lookup"><span data-stu-id="29c22-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="29c22-104">Należy wziąć pod uwagę kopiowanie "this" do zmiennej lokalnej poza metody anonimowej, wyrażenia lambda ani wyrażenia zapytania i zamiast tego za pomocą lokalnego.</span><span class="sxs-lookup"><span data-stu-id="29c22-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="29c22-105">Poniższy przykład spowoduje wygenerowanie CS1673:</span><span class="sxs-lookup"><span data-stu-id="29c22-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
