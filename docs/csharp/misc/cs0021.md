## <a name="example"></a><span data-ttu-id="dfad8-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="dfad8-108">Example</span></span>  
 <span data-ttu-id="dfad8-109">Ten plik jest kompilowany do pliku DLL — z atrybutem oznaczonym jako `DefaultMember` komentarz — w celu wygenerowania błędu.</span><span class="sxs-lookup"><span data-stu-id="dfad8-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
