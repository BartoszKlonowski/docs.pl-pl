 <span data-ttu-id="16fa1-104">Nastąpiła próba dostępu do wartości za pomocą indeksatora w typie danych, która nie obsługuje [indeksatory](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="16fa1-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="16fa1-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="16fa1-108">Example</span></span>  
 <span data-ttu-id="16fa1-109">Ten plik kompiluje się do pliku dll — z `DefaultMember` atrybutu oznaczone jako komentarz — aby wygenerować kod błędu.</span><span class="sxs-lookup"><span data-stu-id="16fa1-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```

// compile with: /reference:CPP0021.dll

        MyClassMC myMC = new MyClassMC();
