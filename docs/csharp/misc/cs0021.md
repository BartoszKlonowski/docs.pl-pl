# <a name="compiler-error-cs0021"></a>Błąd kompilatora CS0021
Nie można zastosować indeksowania z [] do wyrażenia typu "Type".  
  
 Podjęto próbę uzyskania dostępu do wartości za pomocą indeksatora w przypadku typu danych, który nie obsługuje [indeksatorów](../programming-guide/indexers/index.md).  
  
 Możesz uzyskać CS0021 podczas próby użycia indeksatora w C++ zestawie. W takim przypadku dekorować C++ klasę z `DefaultMember` atrybutem, dzięki czemu C# kompilator wie, który indeksator jest wartością domyślną. Poniższy przykład generuje CS0021.  
  
## <a name="example"></a>Przykład  
 Ten plik jest kompilowany do pliku DLL — z atrybutem oznaczonym jako `DefaultMember` komentarz — w celu wygenerowania błędu.  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a>Przykład  
 Poniżej znajduje się C# plik, który wywołuje plik. dll. Ten plik próbuje uzyskać dostęp do klasy za pośrednictwem indeksatora, ale ponieważ żaden element członkowski nie został zadeklarowany jako domyślny indeksator do użycia, zostanie wygenerowany błąd.  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
