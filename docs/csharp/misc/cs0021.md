# <a name="compiler-error-cs0021"></a>CS0021 błąd kompilatora
Nie można zastosować indeksowania przy użyciu konstrukcji [] do wyrażenia typu "type"  
  
 Nastąpiła próba dostępu do wartości za pomocą indeksatora w typie danych, który nie obsługuje [indeksatory](../../csharp/programming-guide/indexers/index.md).  
  
 CS0021 mogą wystąpić podczas próby użycia indeksatora w C++ zestawu. W tym przypadku dekoracji C++ klasy `DefaultMember` atrybut więc C# kompilator wie, których indeksator jest ustawieniem domyślnym. Poniższy przykład spowoduje wygenerowanie CS0021.  
  
## <a name="example"></a>Przykład  
 Ten plik jest kompilowany do pliku .dll — za pomocą `DefaultMember` atrybut oznaczone jako komentarz — aby wygenerować błąd.  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a>Przykład  
 Poniżej przedstawiono C# pliku, który wywołuje plik .dll. Ten plik próby dostępu do tej klasy za pomocą indeksatora, ale ponieważ żaden członek został zadeklarowany jako domyślny indeksator ma być używany, zostanie wygenerowany błąd.  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
