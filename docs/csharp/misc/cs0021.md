# <a name="compiler-error-cs0021"></a><span data-ttu-id="5a264-102">CS0021 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="5a264-102">Compiler Error CS0021</span></span>

 <span data-ttu-id="5a264-104">Nastąpiła próba dostępu do wartości za pomocą indeksatora w typie danych, który nie obsługuje [indeksatory](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="5a264-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="5a264-105">CS0021 mogą wystąpić podczas próby użycia indeksatora w C++ zestawu.</span><span class="sxs-lookup"><span data-stu-id="5a264-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="5a264-106">W tym przypadku dekoracji C++ klasy `DefaultMember` atrybut więc C# kompilator wie, których indeksator jest ustawieniem domyślnym.</span><span class="sxs-lookup"><span data-stu-id="5a264-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="5a264-107">Poniższy przykład spowoduje wygenerowanie CS0021.</span><span class="sxs-lookup"><span data-stu-id="5a264-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5a264-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5a264-108">Example</span></span>  
 <span data-ttu-id="5a264-109">Ten plik jest kompilowany do pliku .dll — za pomocą `DefaultMember` atrybut oznaczone jako komentarz — aby wygenerować błąd.</span><span class="sxs-lookup"><span data-stu-id="5a264-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="5a264-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="5a264-110">Example</span></span>  
 <span data-ttu-id="5a264-111">Poniżej przedstawiono C# pliku, który wywołuje plik .dll.</span><span class="sxs-lookup"><span data-stu-id="5a264-111">The following is the C# file that calls the .dll file.</span></span> <span data-ttu-id="5a264-112">Ten plik próby dostępu do tej klasy za pomocą indeksatora, ale ponieważ żaden członek został zadeklarowany jako domyślny indeksator ma być używany, zostanie wygenerowany błąd.</span><span class="sxs-lookup"><span data-stu-id="5a264-112">This file attempts to access the class via an indexer, but because no member was declared as the default indexer to be used, the error is generated.</span></span>  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
