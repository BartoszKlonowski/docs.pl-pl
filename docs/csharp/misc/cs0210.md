<span data-ttu-id="88671-103">Należy podać inicjatora w deklaracji instrukcji fixed lub using</span><span class="sxs-lookup"><span data-stu-id="88671-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="88671-104">Należy zadeklarować i zainicjować zmiennej w [stałej instrukcji](../../csharp/language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="88671-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="88671-105">Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="88671-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="88671-106">Poniższy przykład generuje CS0210:</span><span class="sxs-lookup"><span data-stu-id="88671-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="88671-107">Poniższy przykład generuje również CS0210 ponieważ [za pomocą instrukcji](../../csharp/language-reference/keywords/using-statement.md) ma żadnego inicjatora.</span><span class="sxs-lookup"><span data-stu-id="88671-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>

```csharp  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
