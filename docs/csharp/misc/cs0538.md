# <a name="compiler-error-cs0538"></a><span data-ttu-id="5bbb7-102">CS0538 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="5bbb7-102">Compiler Error CS0538</span></span>
<span data-ttu-id="5bbb7-103">"Nazwa" w jawnej deklaracji interfejsu nie jest interfejsem</span><span class="sxs-lookup"><span data-stu-id="5bbb7-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="5bbb7-104">Podjęto próbę jawnie deklarować [interfejsu](../../csharp/language-reference/keywords/interface.md), ale nie określono interfejsu.</span><span class="sxs-lookup"><span data-stu-id="5bbb7-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="5bbb7-105">Poniższy przykład generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="5bbb7-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
