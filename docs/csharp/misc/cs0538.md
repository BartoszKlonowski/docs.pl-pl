# <a name="compiler-error-cs0538"></a>CS0538 błąd kompilatora
"name" w jawnej deklaracji interfejsu nie jest interfejsem  
  
 Nastąpiła próba można jawnie deklarować [interfejsu](../../csharp/language-reference/keywords/interface.md), ale nie określono interfejsu.  
  
 Poniższy przykład spowoduje wygenerowanie CS0538:  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
