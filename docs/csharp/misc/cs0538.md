# <a name="compiler-error-cs0538"></a><span data-ttu-id="35159-102">Błąd kompilatora CS0538</span><span class="sxs-lookup"><span data-stu-id="35159-102">Compiler Error CS0538</span></span>
<span data-ttu-id="35159-103">element "name" w jawnej deklaracji interfejsu nie jest interfejsem</span><span class="sxs-lookup"><span data-stu-id="35159-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="35159-104">Podjęto próbę jawnego deklarowania [interfejsu](../language-reference/keywords/interface.md), ale nie określono interfejsu.</span><span class="sxs-lookup"><span data-stu-id="35159-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="35159-105">Poniższy przykład generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="35159-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
