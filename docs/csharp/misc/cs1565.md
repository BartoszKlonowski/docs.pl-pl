# <a name="compiler-error-cs1565"></a>Błąd kompilatora CS1565

Określono opcje powodujące konflikt: plik zasobów Win32; Ikona Win32  
  
 Nie można określić jednocześnie opcji kompilatora [-win32res —](../language-reference/compiler-options/win32res-compiler-option.md) i [-win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) kompilatora w tej samej kompilacji.
