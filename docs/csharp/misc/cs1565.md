# <a name="compiler-error-cs1565"></a>CS1565 błąd kompilatora
Określono opcje powodujące konflikt: Plik zasobów Win32; Ikona Win32  
  
 Nie jest prawidłową podać obydwie wartości [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) — opcja kompilatora i [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) — opcja kompilatora w tej samej kompilacji.
