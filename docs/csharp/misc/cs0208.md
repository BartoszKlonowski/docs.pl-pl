# <a name="compiler-error-cs0208"></a>Błąd kompilatora CS0208
Nie można przyjąć adresu, pobrać rozmiaru lub zadeklarować wskaźnika do typu zarządzanego ("Type")  
  
 Nawet w przypadku użycia ze [](../../csharp/language-reference/keywords/unsafe.md) słowem kluczowym UNSAFE, pobierając adres obiektu zarządzanego, pobieranie rozmiaru obiektu zarządzanego lub deklarowanie wskaźnika do typu zarządzanego jest niedozwolone. Typem zarządzanym jest:  
  
- dowolny typ referencyjny  
  
- Każda struktura, która zawiera typ referencyjny jako pole lub właściwość  
  
 Aby uzyskać więcej informacji, zobacz [typy niezarządzane](../../csharp/language-reference/builtin-types/unmanaged-types.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0208:  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
