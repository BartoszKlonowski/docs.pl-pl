# <a name="compiler-error-cs0678"></a>CS0678 błąd kompilatora
'Zmienna': pole nie może być jednocześnie nietrwałe i tylko do odczytu  
  
 Korzystanie z [volatile](../../csharp/language-reference/keywords/volatile.md) i [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) słów kluczowych jest wzajemnie się wykluczają.  
  
 Poniższy przykład spowoduje wygenerowanie CS0678:  
  
```csharp  
// CS0678.cs  
using System;  
  
class TestClass  
{  
   private readonly volatile int i;   // CS0678  
   // try the following line instead  
   // private volatile int i;  
  
   public static void Main()  
   {  
   }  
}  
```
