# <a name="compiler-error-cs0221"></a>CS0221 błąd kompilatora
Nie można przekonwertować wartości stałej "wartość" do "type" (Użyj składni "unchecked" Aby zastąpić)  
  
 Operatora przypisania, które mogłyby spowodować utratę danych zostało wykryte przez [zaznaczone](../../csharp/language-reference/keywords/checked.md), która jest domyślnie włączone. Popraw przypisanie albo użyj [unchecked](../../csharp/language-reference/keywords/unchecked.md) Aby rozwiązać ten problem. Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0221:  
  
```csharp  
// CS0221.cs  
public class MyClass  
{  
   public static void Main()  
   {  
      // unchecked  
      // {  
         int a = (int)0xFFFFFFFF;   // CS0221  
         a++;  
      // }  
   }  
}  
```
