# <a name="compiler-error-cs1630"></a>CS1630 błąd kompilatora
Nieprawidłowa opcja "opcja" / errorreport; musi to być wiersz, wysyłania, kolejki lub Brak  
  
 Opcja wiersza polecenia [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) musi następować **wiersza**, **wysyłania**, **kolejki**, lub **Brak**, określania, jakie działanie ma podjęte po wystąpieniu błędu wewnętrznego kompilatora.
