# <a name="compiler-error-cs1630"></a>Błąd kompilatora CS1630

Nieprawidłowa opcja "Option" dla/errorreport; musi być Prompt, Send, Queue lub None  
  
 Po wystąpieniu błędu kompilatora wewnętrznego należy użyć opcji wiersza polecenia [-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) , a następnie **monitu**, **wysyłania**, **kolejki**lub **braku**.
