# <a name="compiler-error-cs0450"></a><span data-ttu-id="90e5e-103">Błąd kompilatora CS0450</span><span class="sxs-lookup"><span data-stu-id="90e5e-103">Compiler Error CS0450</span></span>
<span data-ttu-id="90e5e-104">"Nazwa parametru typu": nie można jednocześnie określić klasy ograniczenia i ograniczenia "Class" lub "struct"</span><span class="sxs-lookup"><span data-stu-id="90e5e-104">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="90e5e-105">Jeśli parametr typu jest ograniczony przez ograniczenie typu struktury, jest on logicznie sprzeczny z tym, że jest również ograniczony przez określony typ klasy, ponieważ struktura i Klasa są wzajemnie wykluczające się kategorie.</span><span class="sxs-lookup"><span data-stu-id="90e5e-105">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="90e5e-106">Jeśli parametr typu jest ograniczony przez określone ograniczenie typu klasy, jest on według definicji ograniczonej przez ograniczenie typu klasy, a więc określenie ograniczenia typu klasy jest nadmiarowe.</span><span class="sxs-lookup"><span data-stu-id="90e5e-106">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="90e5e-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="90e5e-107">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
