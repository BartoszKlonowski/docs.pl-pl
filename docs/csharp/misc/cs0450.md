# <a name="compiler-error-cs0450"></a><span data-ttu-id="e7e03-102">CS0450 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e7e03-102">Compiler Error CS0450</span></span>
<span data-ttu-id="e7e03-103">"Type Nazwa parametru": nie można określić klasy ograniczenia i ograniczenia "class" lub "struct"</span><span class="sxs-lookup"><span data-stu-id="e7e03-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="e7e03-104">Jeśli parametr typu jest ograniczony przez ograniczenia typu struktury, jest logicznie sprzeczne dla niej również będą ograniczane przez typ określonej klasy, ponieważ struktury i klasy są wzajemnie wykluczających się kategorii.</span><span class="sxs-lookup"><span data-stu-id="e7e03-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="e7e03-105">Jeśli parametr typu jest ograniczony przez ograniczenia typu określonej klasy, będzie ona wówczas zgodnie z definicją ograniczonych przez ograniczenie typu klasy, a więc określenie ograniczenie typu klasy jest nadmiarowy.</span><span class="sxs-lookup"><span data-stu-id="e7e03-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e7e03-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="e7e03-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
