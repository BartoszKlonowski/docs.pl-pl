# <a name="compiler-error-cs0522"></a><span data-ttu-id="5145c-102">CS0522 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="5145c-102">Compiler Error CS0522</span></span>
<span data-ttu-id="5145c-103">"konstruktora": struktury nie mogą wywoływać konstruktorów klasy podstawowej</span><span class="sxs-lookup"><span data-stu-id="5145c-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="5145c-104">A [struktury](../../csharp/language-reference/keywords/struct.md) nie można wywołać konstruktora klasy podstawowej; Usuń wywołanie konstruktora klasy podstawowej.</span><span class="sxs-lookup"><span data-stu-id="5145c-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="5145c-105">Poniższy przykład generuje CS0522:</span><span class="sxs-lookup"><span data-stu-id="5145c-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
