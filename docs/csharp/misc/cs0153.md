# <a name="compiler-error-cs0153"></a>CS0153 błąd kompilatora
Instrukcja goto case jest prawidłowa tylko wewnątrz instrukcji switch  
  
 Próbowano użyć [Przełącz](../../csharp/language-reference/keywords/switch.md) składni poza **Przełącz** instrukcji. Aby uzyskać więcej informacji, zobacz [Przełącz](../../csharp/language-reference/keywords/switch.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0153:  
  
```csharp  
// CS0153.cs  
public class a  
{  
   public static void Main()  
   {  
      goto case 5;   // CS0153  
   }  
}  
```
