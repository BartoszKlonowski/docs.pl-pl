## <a name="example"></a><span data-ttu-id="d6bc6-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d6bc6-107">Example</span></span>  
 <span data-ttu-id="d6bc6-108">Poniższy przykład generuje CS0411:</span><span class="sxs-lookup"><span data-stu-id="d6bc6-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
