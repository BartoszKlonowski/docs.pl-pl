# <a name="compiler-error-cs0411"></a><span data-ttu-id="49f49-102">CS0411 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="49f49-102">Compiler Error CS0411</span></span>
<span data-ttu-id="49f49-103">Nie można wywnioskować argumentów typu dla metody "method" z użycia.</span><span class="sxs-lookup"><span data-stu-id="49f49-103">The type arguments for method 'method' cannot be inferred from the usage.</span></span> <span data-ttu-id="49f49-104">Spróbuj jawne określenie argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="49f49-104">Try specifying the type arguments explicitly.</span></span>  
  
 <span data-ttu-id="49f49-105">Ten błąd występuje, jeśli wywołanie metody ogólnej bez podawania jawnie argumentów typu, a kompilator nie można wywnioskować argumentów typu, które są przeznaczone.</span><span class="sxs-lookup"><span data-stu-id="49f49-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="49f49-106">Aby uniknąć tego błędu, należy dodać argumenty zamierzonego typu w nawiasy ostre.</span><span class="sxs-lookup"><span data-stu-id="49f49-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="49f49-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="49f49-107">Example</span></span>  
 <span data-ttu-id="49f49-108">Poniższy przykład spowoduje wygenerowanie CS0411:</span><span class="sxs-lookup"><span data-stu-id="49f49-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
