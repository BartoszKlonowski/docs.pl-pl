 <span data-ttu-id="e049e-105">Ten błąd występuje, gdy wywołanie metody rodzajowej bez podawania jawne argumenty typu i kompilator nie można wywnioskować argumentów typu, które są przeznaczone.</span><span class="sxs-lookup"><span data-stu-id="e049e-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="e049e-106">Aby uniknąć tego błędu, należy dodać argumentów danego typu w nawiasach ostrych.</span><span class="sxs-lookup"><span data-stu-id="e049e-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e049e-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="e049e-107">Example</span></span>  
 <span data-ttu-id="e049e-108">Poniższy przykład generuje CS0411:</span><span class="sxs-lookup"><span data-stu-id="e049e-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
