## <a name="example"></a><span data-ttu-id="bbd21-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="bbd21-108">Example</span></span>  
 <span data-ttu-id="bbd21-109">Poniższy przykład generuje CS0411:</span><span class="sxs-lookup"><span data-stu-id="bbd21-109">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
