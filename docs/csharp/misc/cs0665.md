# <a name="compiler-warning-level-3-cs0665"></a>Kompilatora (poziom 3) ostrzeżenie CS0665
Przypisanie w wyrażeniu warunkowym jest zawsze stałe; czy zamierzałeś użyć == zamiast =?  
  
 Wyrażenie warunkowe używane [= — operator](../../csharp/language-reference/operators/assignment-operator.md) i nie [== — operator](../../csharp/language-reference/operators/equality-operators.md#equality-operator-).
  
 Poniższy przykład spowoduje wygenerowanie CS0665:  
  
```csharp  
// CS0665.cs  
// compile with: /W:3  
class Test  
{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
