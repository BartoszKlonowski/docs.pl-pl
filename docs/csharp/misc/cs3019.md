# <a name="compiler-warning-level-2-cs3019"></a>CS3019 ostrzeżenie (poziom 2) kompilator
Sprawdzanie zgodności ze specyfikacją CLS będzie nie można wykonać na "type", ponieważ nie jest on widoczny spoza tego zestawu.  
  
 To ostrzeżenie występuje, gdy typ lub element członkowski, który ma <xref:System.CLSCompliantAttribute> atrybut nie jest widoczny z innego zestawu. Aby rozwiązać ten problem, usuń atrybut na dowolnej klasy lub elementów członkowskich, które nie są widoczne z innego zestawu lub widoczności typów lub elementów członkowskich. Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS3019:  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019  
class C  
{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
