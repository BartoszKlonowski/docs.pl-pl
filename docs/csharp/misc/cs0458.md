# <a name="compiler-warning-level-2-cs0458"></a>Ostrzeżenie o kompilatorze (poziom 2) CS0458
Wynikiem wyrażenia jest zawsze "null" typu "nazwa typu"  
  
 To ostrzeżenie jest `nullable` spowodowane przez wyrażenie, które zawsze powoduje . `null`  
  
 Poniższy kod generuje ostrzeżenie CS0458.  
  
## <a name="example"></a>Przykład  
 W tym przykładzie przedstawiono szereg różnych `nullable` operacji z typami, które spowoduje ten błąd.  
  
```csharp  
// CS0458.cs  
using System;  
public  class Test
{  
    public static void Main()  
    {  
        int a = 5;  
        int? b = a + null;    // CS0458  
        int? qa = 15;  
        b = qa + null;        // CS0458  
        b -= null;            // CS0458  
        int? qa2 = null;  
        b = qa2 + null;       // CS0458  
        qa2 -= null;          // CS0458  
    }  
}  
```
