## <a name="example"></a><span data-ttu-id="7dab4-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="7dab4-106">Example</span></span>  
 <span data-ttu-id="7dab4-107">W tym przykładzie przedstawiono szereg różnych operacji z nullable typów wartości, które spowoduje ten błąd.</span><span class="sxs-lookup"><span data-stu-id="7dab4-107">This example illustrates a number of the different operations with nullable value types that will cause this error.</span></span>  
  
```csharp

using System;  
public  class Test
{  
    public static void Main()  
    {  
        int a = 5;  
        int? b = a + null;    // CS0458  
        int? qa = 15;  
        b = qa + null;        // CS0458  
        b -= null;            // CS0458  
        int? qa2 = null;  
        b = qa2 + null;       // CS0458  
        qa2 -= null;          // CS0458  
    }  
}  
```
