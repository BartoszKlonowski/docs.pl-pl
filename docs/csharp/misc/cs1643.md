# <a name="compiler-error-cs1643"></a><span data-ttu-id="7c903-103">Błąd kompilatora CS1643</span><span class="sxs-lookup"><span data-stu-id="7c903-103">Compiler Error CS1643</span></span>

 <span data-ttu-id="7c903-105">Ten błąd występuje, gdy treść delegata nie zawiera instrukcji return lub zawiera instrukcję return, która nie może zostać zweryfikowana, zostanie osiągnięty.</span><span class="sxs-lookup"><span data-stu-id="7c903-105">This error occurs if a delegate body does not have a return statement, or has a return statement that the compiler is unable to verify will be reached.</span></span> <span data-ttu-id="7c903-106">W poniższym przykładzie kompilator nie podejmuje próby przewidywania wyniku odgałęzienia w celu sprawdzenia, czy blok metody anonimowej zawsze zwraca wartość.</span><span class="sxs-lookup"><span data-stu-id="7c903-106">In the example below, the compiler does not attempt to predict the result of the branching condition in order to verify that the anonymous method block always returns a value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7c903-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="7c903-107">Example</span></span>  

 <span data-ttu-id="7c903-108">Poniższy przykład generuje CS1643:</span><span class="sxs-lookup"><span data-stu-id="7c903-108">The following sample generates CS1643:</span></span>  
  
```csharp  
// CS1643.cs  
delegate int MyDelegate();  
  
class C  
{  
    static void Main()  
    {  
        MyDelegate d = delegate  
        {                 // CS1643  
            int i = 0;  
            if (i == 0)  
                return 1;  
        };  
    }  
}  
```
