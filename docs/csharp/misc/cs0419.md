# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="518f3-102">Kompilator CS0419 ostrzegawcze (poziom 3)</span><span class="sxs-lookup"><span data-stu-id="518f3-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="518f3-103">Niejednoznaczne odwołanie w atrybucie cref: "Nazwa1 metody".</span><span class="sxs-lookup"><span data-stu-id="518f3-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="518f3-104">Przy założeniu "Nazwa2 metody", ale mogły również zostać dopasowane inne przeciążenia, w tym "Nazwa3 metody".</span><span class="sxs-lookup"><span data-stu-id="518f3-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="518f3-105">W komentarzu XML dokumentacji w kodzie nie można rozpoznać odwołania.</span><span class="sxs-lookup"><span data-stu-id="518f3-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="518f3-106">Może mieć to miejsce, jeśli metoda jest przeciążony lub znajdują się dwie różne identyfikatory o takiej samej nazwie.</span><span class="sxs-lookup"><span data-stu-id="518f3-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="518f3-107">Aby rozwiązać ostrzeżenia, odróżniania odwołanie za pomocą nazwy kwalifikowanej, lub Dołącz przeciążenie w nawiasach.</span><span class="sxs-lookup"><span data-stu-id="518f3-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="518f3-108">Poniższy przykład generuje CS0419.</span><span class="sxs-lookup"><span data-stu-id="518f3-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
