# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="c469c-103">Ostrzeżenie kompilatora (poziom 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="c469c-103">Compiler Warning (level 3) CS0419</span></span>

<span data-ttu-id="c469c-104">Niejednoznaczne odwołanie w atrybucie cref: "Method Name1".</span><span class="sxs-lookup"><span data-stu-id="c469c-104">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="c469c-105">Przyjęto założenie metody NAME2 ', ale mogło również dopasować inne przeciążenia, w tym metodę Nazwa3 '.</span><span class="sxs-lookup"><span data-stu-id="c469c-105">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="c469c-106">W komentarzu dokumentacji XML w kodzie nie można rozpoznać odwołania.</span><span class="sxs-lookup"><span data-stu-id="c469c-106">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="c469c-107">Taka sytuacja może wystąpić, jeśli metoda jest przeciążona lub zostaną znalezione dwa różne identyfikatory o tej samej nazwie.</span><span class="sxs-lookup"><span data-stu-id="c469c-107">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="c469c-108">Aby rozwiązać ten problem, użyj kwalifikowanej nazwy, aby odróżnić odwołanie, lub Dołącz określone Przeciążenie w nawiasach.</span><span class="sxs-lookup"><span data-stu-id="c469c-108">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="c469c-109">Poniższy przykład generuje CS0419.</span><span class="sxs-lookup"><span data-stu-id="c469c-109">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
