# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="4f495-102">Kompilatora (poziom 3) ostrzeżenie CS0419</span><span class="sxs-lookup"><span data-stu-id="4f495-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="4f495-103">Niejednoznaczne odwołanie w atrybucie cref: "Metoda Nazwa1".</span><span class="sxs-lookup"><span data-stu-id="4f495-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="4f495-104">Zakładając, że "Metoda Nazwa2", ale mogły również zostać dopasowane inne przeciążenia, w tym "Metoda nazwa3".</span><span class="sxs-lookup"><span data-stu-id="4f495-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="4f495-105">W komentarzu dokumentacji XML w kodzie nie można rozpoznać odwołania.</span><span class="sxs-lookup"><span data-stu-id="4f495-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="4f495-106">Taka sytuacja może wystąpić, czy metoda jest przeciążona, czy znajdują się dwa różne identyfikatory o takiej samej nazwie.</span><span class="sxs-lookup"><span data-stu-id="4f495-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="4f495-107">Aby rozwiązać ostrzeżenia, odróżnić odwołanie za pomocą nazwy kwalifikowanej, lub Dołącz określonego przeciążenia w nawiasach.</span><span class="sxs-lookup"><span data-stu-id="4f495-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="4f495-108">Poniższy przykład spowoduje wygenerowanie CS0419.</span><span class="sxs-lookup"><span data-stu-id="4f495-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
