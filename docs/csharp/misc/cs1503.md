# <a name="compiler-error-cs1503"></a>CS1503 błąd kompilatora
Argument "number" nie można przekonwertować z TypeA TypeB  
  
 Typ jednego z argumentów w metodzie niezgodny typ, który został przekazany, po utworzeniu wystąpienia klasy. Zazwyczaj ten błąd pojawia się wraz z CS1502. Zobacz [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) dyskusję na temat sposobu rozwiązania tego błędu.
