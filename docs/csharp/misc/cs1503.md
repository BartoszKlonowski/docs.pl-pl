# <a name="compiler-error-cs1503"></a>Błąd kompilatora CS1503
Nie można przekonwertować argumentu "number" z typu "TypeB"  
  
 Typ jednego argumentu w metodzie jest niezgodny z typem, który został przesłany podczas tworzenia wystąpienia klasy. Ten błąd występuje zazwyczaj wraz z CS1502. Zobacz [CS1502](../language-reference/compiler-messages/cs1502.md) , aby zapoznać się z omówieniem sposobu rozwiązania tego błędu.
