# <a name="compiler-error-cs0239"></a><span data-ttu-id="a01f5-102">Błąd kompilatora CS0239</span><span class="sxs-lookup"><span data-stu-id="a01f5-102">Compiler Error CS0239</span></span>
<span data-ttu-id="a01f5-103">"member": nie można przesłonić dziedziczonej składowej dziedziczonej ", ponieważ jest zapieczętowana</span><span class="sxs-lookup"><span data-stu-id="a01f5-103">'member' : cannot override inherited member 'inherited member' because it is sealed</span></span>  
  
 <span data-ttu-id="a01f5-104">Składowa nie może [](../language-reference/keywords/override.md) przesłonić [zapieczętowanego](../language-reference/keywords/sealed.md) dziedziczonego elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="a01f5-104">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="a01f5-105">Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="a01f5-105">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="a01f5-106">Poniższy przykład generuje CS0239:</span><span class="sxs-lookup"><span data-stu-id="a01f5-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
