# <a name="compiler-error-cs0239"></a><span data-ttu-id="9026e-103">Błąd kompilatora CS0239</span><span class="sxs-lookup"><span data-stu-id="9026e-103">Compiler Error CS0239</span></span>

 <span data-ttu-id="9026e-105">Składowa nie może [przesłonić](../language-reference/keywords/override.md) [zapieczętowanego](../language-reference/keywords/sealed.md) dziedziczonego elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="9026e-105">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="9026e-106">Aby uzyskać więcej informacji, zobacz [zaznaczone i niezaznaczone](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="9026e-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="9026e-107">Poniższy przykład generuje CS0239:</span><span class="sxs-lookup"><span data-stu-id="9026e-107">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
