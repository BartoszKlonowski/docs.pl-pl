# <a name="compiler-error-cs1507"></a>Błąd kompilatora CS1507
Nie można połączyć pliku zasobów "File" podczas kompilowania modułu  
  
 [-linkresource —](../language-reference/compiler-options/linkresource-compiler-option.md) została użyta w tej samej kompilacji z elementem [/target: module](../language-reference/compiler-options/target-module-compiler-option.md), co jest niedozwolone. Na przykład następujące opcje generują CS1507:  
  
```console  
csc /linkresource:rf.resource /target:module in.cs  
```  
  
 Zasoby osadzania ([/Resource](../language-reference/compiler-options/resource-compiler-option.md)) są jednak dozwolone.
