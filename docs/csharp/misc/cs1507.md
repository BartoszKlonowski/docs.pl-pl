# <a name="compiler-error-cs1507"></a>CS1507 błąd kompilatora
Nie można połączyć zasobu w pliku 'Plik', podczas kompilowania modułu  
  
 [/ linkresource](../../csharp/language-reference/compiler-options/linkresource-compiler-option.md) została użyta w tej samej kompilacji za pomocą [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md), co jest niedozwolone. Na przykład następujące opcje wygeneruje CS1507:  
  
```console  
csc /linkresource:rf.resource /target:module in.cs  
```  
  
 Osadzanie zasobów ([/Resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md)), jest jednak dozwolony.
