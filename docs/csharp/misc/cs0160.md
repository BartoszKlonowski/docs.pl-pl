# <a name="compiler-error-cs0160"></a><span data-ttu-id="7d2b2-102">Błąd kompilatora CS0160</span><span class="sxs-lookup"><span data-stu-id="7d2b2-102">Compiler Error CS0160</span></span>
<span data-ttu-id="7d2b2-103">Poprzednia klauzula catch przechwytuje już wszystkie wyjątki tego typu lub klasy Super ("Type")</span><span class="sxs-lookup"><span data-stu-id="7d2b2-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="7d2b2-104">Seria instrukcji [catch](../language-reference/keywords/try-catch.md) musi być w kolejności malejącej.</span><span class="sxs-lookup"><span data-stu-id="7d2b2-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="7d2b2-105">Na przykład najbardziej pochodne obiekty muszą być wyświetlane jako pierwsze.</span><span class="sxs-lookup"><span data-stu-id="7d2b2-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="7d2b2-106">Aby uzyskać więcej informacji, zobacz [wyjątki i obsługa wyjątków](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="7d2b2-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="7d2b2-107">Poniższy przykład generuje CS0160:</span><span class="sxs-lookup"><span data-stu-id="7d2b2-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
