<span data-ttu-id="b83d9-103">Poprzednia klauzula catch przechwytuje już wszystkie wyjątki tego typu lub nadtypu ("type")</span><span class="sxs-lookup"><span data-stu-id="b83d9-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
 <span data-ttu-id="b83d9-104">Szereg **catch** instrukcje musi być według kolejności pochodnym.</span><span class="sxs-lookup"><span data-stu-id="b83d9-104">A series of **catch** statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="b83d9-105">Na przykład obiektów najdalszych pochodnych musi występować jako pierwszy.</span><span class="sxs-lookup"><span data-stu-id="b83d9-105">For example, the most derived objects must appear first.</span></span>  
  
 <span data-ttu-id="b83d9-106">Aby uzyskać więcej informacji, zobacz [instrukcje obsługi wyjątków](../../csharp/language-reference/keywords/exception-handling-statements.md) i [wyjątki i obsługa wyjątków](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="b83d9-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="b83d9-107">Poniższy przykład generuje CS0160:</span><span class="sxs-lookup"><span data-stu-id="b83d9-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
