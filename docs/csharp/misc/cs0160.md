# <a name="compiler-error-cs0160"></a>CS0160 błąd kompilatora
Poprzednia klauzula catch przechwytuje już wszystkie wyjątki tego typu lub nadtypu ("type")  
  
Szereg [catch](../../csharp/language-reference/keywords/try-catch.md) instrukcji musi być malejąco według pochodnym. Na przykład najbardziej pochodnej obiektów musi występować jako pierwszy.
  
 Aby uzyskać więcej informacji, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0160:  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
