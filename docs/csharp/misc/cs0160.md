# <a name="compiler-error-cs0160"></a><span data-ttu-id="a89d5-102">CS0160 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="a89d5-102">Compiler Error CS0160</span></span>
<span data-ttu-id="a89d5-103">Poprzednia klauzula catch przechwytuje już wszystkie wyjątki tego typu lub nadtypu ("type")</span><span class="sxs-lookup"><span data-stu-id="a89d5-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="a89d5-104">Szereg [catch](../../csharp/language-reference/keywords/try-catch.md) instrukcji musi być malejąco według pochodnym.</span><span class="sxs-lookup"><span data-stu-id="a89d5-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="a89d5-105">Na przykład najbardziej pochodnej obiektów musi występować jako pierwszy.</span><span class="sxs-lookup"><span data-stu-id="a89d5-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="a89d5-106">Aby uzyskać więcej informacji, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="a89d5-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="a89d5-107">Poniższy przykład spowoduje wygenerowanie CS0160:</span><span class="sxs-lookup"><span data-stu-id="a89d5-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
