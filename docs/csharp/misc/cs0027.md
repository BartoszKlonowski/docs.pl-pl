# <a name="compiler-error-cs0027"></a><span data-ttu-id="1ffdf-102">Błąd kompilatora CS0027</span><span class="sxs-lookup"><span data-stu-id="1ffdf-102">Compiler Error CS0027</span></span>
<span data-ttu-id="1ffdf-103">Słowo kluczowe "to" nie jest dostępne w bieżącym kontekście</span><span class="sxs-lookup"><span data-stu-id="1ffdf-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="1ffdf-104">[To](../language-reference/keywords/this.md) słowo kluczowe zostało znalezione poza właściwości, metody lub konstruktora.</span><span class="sxs-lookup"><span data-stu-id="1ffdf-104">The [this](../language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="1ffdf-105">Aby naprawić ten błąd, zmodyfikuj `this` instrukcję, aby wyeliminować użycie słowa kluczowego i/lub przenieść część lub wszystkie instrukcji wewnątrz właściwości, metody lub konstruktora.</span><span class="sxs-lookup"><span data-stu-id="1ffdf-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="1ffdf-106">Poniższy przykład generuje CS0027:</span><span class="sxs-lookup"><span data-stu-id="1ffdf-106">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
