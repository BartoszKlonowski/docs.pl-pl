# <a name="compiler-warning-level-2-cs1668"></a>CS1668 ostrzeżenie (poziom 2) kompilator
Nieprawidłową ścieżkę wyszukiwania "path" określona w "— ciąg ścieżki"--"komunikat o błędzie systemu"  
  
 Ścieżka dostarczane do [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) polecenia w wierszu nie jest prawidłowy lub ścieżka w zmiennej środowiskowej LIB jest nieprawidłowa. Sprawdź ścieżkę używany do sprawdzenia, czy istnieje i jest możliwy. Komunikat o błędzie w pojedynczych cudzysłowach jest błąd zwrócony przez system operacyjny.
