# <a name="compiler-warning-level-2-cs1668"></a>Kompilator CS1668 ostrzegawcze (poziom 2)
Nieprawidłową ścieżkę wyszukiwania "path" określony w ścieżce ciągu--"komunikat o błędzie systemu"  
  
 Podana ścieżka do [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) w wierszu polecenia wiersz jest nieprawidłowy lub ścieżka w zmiennej środowiskowej LIB jest nieprawidłowa. Sprawdź ścieżkę używany do sprawdzenia, czy istnieje i jest dostępny. Komunikat o błędzie w pojedynczym cudzysłowie to kod błędu zwrócony przez system operacyjny.
