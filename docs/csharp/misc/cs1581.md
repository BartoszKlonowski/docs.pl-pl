# <a name="compiler-warning-level-1-cs1581"></a>Kompilatora (poziom 1) ostrzeżenie CS1581
Nieprawidłowy zwracany typ w atrybucie cref komentarza XML  
  
 Podczas próby odwoływać się do metody, kompilator wykrył błąd z powodu nieprawidłowego typu zwracanego.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1581:  
  
```csharp  
// CS1581.cs  
// compile with: /W:1 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
    /// <summary>help text</summary>  
    public static void Main()  
    {  
    }  
  
    /// <summary>help text</summary>  
    public static explicit operator int(MyClass f)  
    {  
        return 0;  
    }  
}  
  
/// <seealso cref="MyClass.explicit operator intt(MyClass)"/>  // CS1581  
// try the following line instead  
// /// <seealso cref="MyClass.explicit operator int(MyClass)"/>  
public class MyClass2  
{  
}  
```
