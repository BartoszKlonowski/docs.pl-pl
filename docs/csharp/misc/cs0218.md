# <a name="compiler-error-cs0218"></a><span data-ttu-id="f5e52-102">CS0218 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="f5e52-102">Compiler Error CS0218</span></span>
<span data-ttu-id="f5e52-103">Typ ("type") musi zawierać deklaracje operatora true i operatora false</span><span class="sxs-lookup"><span data-stu-id="f5e52-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="f5e52-104">Jeśli typ zdefiniowany przez użytkownika przeciążenia [& — operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) lub [ &#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), musi także definiować [true i false](../language-reference/keywords/true-false-operators.md) operatorów, aby upewnić się, zwarcie [& & — operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) lub [ &#124; &#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) zdefiniowane.</span><span class="sxs-lookup"><span data-stu-id="f5e52-104">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/keywords/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="f5e52-105">Poniższy przykład spowoduje wygenerowanie CS0218:</span><span class="sxs-lookup"><span data-stu-id="f5e52-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
