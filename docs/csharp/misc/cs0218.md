# <a name="compiler-error-cs0218"></a>CS0218 błąd kompilatora
Typ ("type") musi zawierać deklaracje operatora true i operatora false  
  
Jeśli typ zdefiniowany przez użytkownika przeciążenia [& — operator](../language-reference/operators/and-operator.md) lub [ &#124; operator](../language-reference/operators/or-operator.md), musi także definiować [true i false](../language-reference/keywords/true-false-operators.md) operatorów, aby upewnić się, zwarcie [& & — operator](../language-reference/operators/conditional-and-operator.md) lub [ &#124; &#124; operator](../language-reference/operators/conditional-or-operator.md) zdefiniowane.
  
 Poniższy przykład spowoduje wygenerowanie CS0218:  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
