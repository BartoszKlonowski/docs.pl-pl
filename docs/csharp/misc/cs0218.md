# <a name="compiler-error-cs0218"></a><span data-ttu-id="af53f-103">Błąd kompilatora CS0218</span><span class="sxs-lookup"><span data-stu-id="af53f-103">Compiler Error CS0218</span></span>

<span data-ttu-id="af53f-104">Typ ("Type") musi zawierać deklaracje operatora true i operatora false</span><span class="sxs-lookup"><span data-stu-id="af53f-104">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="af53f-105">Jeśli typ zdefiniowany przez użytkownika jest przeciążony [ operatorem&](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) lub [ operatorem&#124;](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), musi także definiować operatory [true i false](../language-reference/operators/true-false-operators.md) , aby można było zdefiniować operator [&& ](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) krótkich obwodów lub operator [&#124;&#124; ](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) .</span><span class="sxs-lookup"><span data-stu-id="af53f-105">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="af53f-106">Poniższy przykład generuje CS0218:</span><span class="sxs-lookup"><span data-stu-id="af53f-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
