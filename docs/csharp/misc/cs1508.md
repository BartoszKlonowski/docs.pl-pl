# <a name="compiler-error-cs1508"></a>Błąd kompilatora CS1508
Identyfikator zasobu "identifier" został już użyty w tym zestawie  
  
 W kompilacji ten sam identyfikator (***Identyfikator***) został przekazano do co najmniej dwóch opcji kompilatora [/Resource](../language-reference/compiler-options/resource-compiler-option.md) lub [/linkresource](../language-reference/compiler-options/linkresource-compiler-option.md) .  
  
 Na przykład następujące opcje generują CS1508:  
  
```  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
