# <a name="compiler-error-cs0526"></a>CS0526 błąd kompilatora
Interfejsy nie mogą zawierać konstruktorów  
  
 Nejde definovat konstruktory [interfejsów](../../csharp/language-reference/keywords/interface.md). Metoda jest uznawana za konstruktora, jeśli ma taką samą nazwę jak klasa i bez zwrotu typu.  
  
 Poniższy przykład spowoduje wygenerowanie CS0526:  
  
```csharp  
// CS0526.cs  
namespace x  
{  
   public interface clx  
   {  
      public clx()   // CS0526  
      {  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
