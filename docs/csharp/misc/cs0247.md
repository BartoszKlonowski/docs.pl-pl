# <a name="compiler-error-cs0247"></a>CS0247 błąd kompilatora
Nie można użyć ujemnego rozmiaru w przypadku słowa kluczowego stackalloc  
  
 Wartość ujemną został przekazany do [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) instrukcji.  
  
 Poniższy przykład generuje CS0247:  
  
```csharp  
// CS0247.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int [-30];   // CS0247  
   }  
}  
```
