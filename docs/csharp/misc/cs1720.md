# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="a9353-102">Kompilator CS1720 ostrzegawcze (poziom 1)</span><span class="sxs-lookup"><span data-stu-id="a9353-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="a9353-103">Wyrażenie będzie zawsze powodować wystąpienie wyjątku System.NullReferenceException, ponieważ wartość domyślna "typu ogólnego" ma wartość null</span><span class="sxs-lookup"><span data-stu-id="a9353-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="a9353-104">Jeśli piszesz wyrażenie obejmujące wartość domyślna zmiennej typu ogólnego będący typem odwołania (na przykład klasa), ten błąd wystąpi.</span><span class="sxs-lookup"><span data-stu-id="a9353-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="a9353-105">Należy wziąć pod uwagę następujące wyrażenie:</span><span class="sxs-lookup"><span data-stu-id="a9353-105">Consider the following expression:</span></span>  
  

 <span data-ttu-id="a9353-106">Ponieważ `T` jest typem referencyjnym, jego wartość domyślna to null, a więc próby zastosowania <xref:System.Object.ToString%2A> zgłosi ona metodę <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="a9353-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>
