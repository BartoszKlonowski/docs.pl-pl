# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="859b2-102">Ostrzeżenie o kompilatorze (poziom 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="859b2-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="859b2-103">Wyrażenie zawsze spowoduje System.NullReferenceException, ponieważ domyślna wartość "typ ogólny" jest null</span><span class="sxs-lookup"><span data-stu-id="859b2-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="859b2-104">Jeśli zapiszesz wyrażenie dotyczące domyślnej zmiennej typu ogólnego, która jest typem odwołania (na przykład klasy), ten błąd wystąpi.</span><span class="sxs-lookup"><span data-stu-id="859b2-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="859b2-105">Należy wziąć pod uwagę następujące wyrażenie:</span><span class="sxs-lookup"><span data-stu-id="859b2-105">Consider the following expression:</span></span>  
  
```csharp  
default(T).ToString()  
```  
  
 <span data-ttu-id="859b2-106">Ponieważ `T` jest typem odwołania, jego wartością domyślną jest <xref:System.Object.ToString%2A> null, a <xref:System.NullReferenceException>więc próba zastosowania metody do niego spowoduje wyrzucając .</span><span class="sxs-lookup"><span data-stu-id="859b2-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="859b2-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="859b2-107">Example</span></span>  
 <span data-ttu-id="859b2-108">Ograniczenie odwołania do klasy `T` dla typu `T` zapewnia, że jest typem odwołania.</span><span class="sxs-lookup"><span data-stu-id="859b2-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="859b2-109">Poniższy przykład generuje CS1720.</span><span class="sxs-lookup"><span data-stu-id="859b2-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester
{  
    public static void GenericClass<T>(T t1) where T : class
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
