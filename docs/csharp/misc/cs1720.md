# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="338ab-102">Kompilatora (poziom 1) ostrzeżenie CS1720</span><span class="sxs-lookup"><span data-stu-id="338ab-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="338ab-103">Wyrażenie będzie zawsze powodować wystąpienie wyjątku System.NullReferenceException, ponieważ wartość domyślna typu"Ogólne" ma wartość null</span><span class="sxs-lookup"><span data-stu-id="338ab-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="338ab-104">Jeśli piszesz wyrażenie obejmujące domyślną zmienną typu rodzajowego, który jest typem odwołania (na przykład klasa), ten błąd wystąpi.</span><span class="sxs-lookup"><span data-stu-id="338ab-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="338ab-105">Należy wziąć pod uwagę następujące wyrażenie:</span><span class="sxs-lookup"><span data-stu-id="338ab-105">Consider the following expression:</span></span>  
  
```csharp

```  
  
 <span data-ttu-id="338ab-106">Ponieważ `T` jest typem referencyjnym, jego wartość domyślna to null, a więc próby zastosowania <xref:System.Object.ToString%2A> zgłosi metodę, aby go <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="338ab-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="338ab-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="338ab-107">Example</span></span>

 <span data-ttu-id="338ab-109">Poniższy przykład spowoduje wygenerowanie CS1720.</span><span class="sxs-lookup"><span data-stu-id="338ab-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
