# <a name="compiler-error-cs0543"></a>CS0543 błąd kompilatora
'wyliczenie': wartość modułu wyliczającego jest zbyt duża dla swojego typu  
  
 Wartość, która została przypisana do elementu w [wyliczenie](../../csharp/language-reference/keywords/enum.md) znajduje się poza zakresem typu danych.  
  
 Poniższy przykład spowoduje wygenerowanie CS0543:  
  
```csharp  
// CS0543.cs  
namespace x  
{  
   enum I : byte  
   {a = 255, b, c}   // CS0543  
   public class clx  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
