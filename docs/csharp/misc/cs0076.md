# <a name="compiler-error-cs0076"></a>Błąd kompilatora CS0076
Nazwa modułu wyliczającego "value__" jest zarezerwowana i nie można jej użyć  
  
 Element w [wyliczeniu](../language-reference/builtin-types/enum.md) nie może mieć identyfikatora o nazwie **value__**.
