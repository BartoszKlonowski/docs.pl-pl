# <a name="compiler-error-cs1654"></a>CS1654 błąd kompilatora
Nie można zmodyfikować składowych "Zmienna", ponieważ jest "read-only typu zmiennej"  
  
 Ten błąd występuje, gdy spróbujesz zmodyfikować składowych elementu Zmienna, która jest tylko do odczytu, ponieważ jest on specjalnych konstrukcji.  
  
 Znajduje się w jednej wspólnej obszar, który dzieje się tak [foreach](../../csharp/language-reference/keywords/foreach-in.md) pętli. Jest to błąd czasu kompilacji do modyfikowania wartości elementów kolekcji. W związku z tym, nie może wprowadzać żadnych zmian elementów, które są [typy wartości](../../csharp/language-reference/keywords/value-types.md), w tym [struktury](../../csharp/programming-guide/classes-and-structs/structs.md). W kolekcji, której elementy są [typy odwołań](../../csharp/language-reference/keywords/reference-types.md), można zmodyfikować dostępne elementy członkowskie każdego elementu, ale wygeneruje wszelkie próby, aby dodać lub usunąć lub zamienić ukończone elementy [błąd kompilatora CS1656](../../csharp/language-reference/compiler-messages/cs1656.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje błąd CS1654, ponieważ `Book` jest `struct`. Aby naprawić błąd, zmień `struct` do [klasy](../../csharp/language-reference/keywords/class.md).  
  
```csharp  
using System.Collections.Generic;  
using System.Text;  
  
namespace CS1654  
{  
  
    struct Book  
    {  
        public string Title;  
        public string Author;  
        public double Price;  
        public Book(string t, string a, double p)  
        {  
            Title=t;  
            Author=a;  
            Price=p;  
  
        }  
    }  
  
    class Program  
    {  
        List<Book> list;  
        static void Main(string[] args)  
        {  
             //Use a collection initializer to initialize the list  
            Program prog = new Program();  
            prog.list = new List<Book>();  
            prog.list.Add(new Book ("The C# Programming Language",  
                                    "Hejlsberg, Wiltamuth, Golde",  
                                     29.95));  
            prog.list.Add(new Book ("The C++ Programming Language",  
                                    "Stroustrup",  
                                     29.95));  
            prog.list.Add(new Book ("The C Programming Language",  
                                    "Kernighan, Ritchie",  
                                    29.95));  
            foreach(Book b in prog.list)  
            {  
                //Compile error if Book is a struct  
                //Make Book a class to modify its members  
                b.Price +=9.95; // CS1654  
            }  
  
        }  
    }  
}  
```
