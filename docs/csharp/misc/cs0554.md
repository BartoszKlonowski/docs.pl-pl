 <span data-ttu-id="92136-105">Zdefiniowane przez użytkownika konwersje na wartości klasy pochodnej nie są dozwolone; Ten operator nie jest potrzebny.</span><span class="sxs-lookup"><span data-stu-id="92136-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="92136-106">Zobacz rozdział 6 w specyfikacji języka C#, aby uzyskać więcej informacji na temat konwersji zdefiniowanych przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="92136-106">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="92136-107">Poniższy przykład generuje CS0554:</span><span class="sxs-lookup"><span data-stu-id="92136-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
