 <span data-ttu-id="fa0bf-104">Konwersje zdefiniowane przez użytkownika wartości klasy pochodnej nie są dozwolone taki operator nie jest konieczne.</span><span class="sxs-lookup"><span data-stu-id="fa0bf-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>

 <span data-ttu-id="fa0bf-106">Poniższy przykład spowoduje wygenerowanie CS0554:</span><span class="sxs-lookup"><span data-stu-id="fa0bf-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
