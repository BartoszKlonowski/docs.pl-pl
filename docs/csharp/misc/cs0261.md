# <a name="compiler-error-cs0261"></a>CS0261 błąd kompilatora
Częściowe deklaracje elementu "type" musi być wszystkie klasami, strukturami lub wszystkie interfejsy  
  
 Ten błąd występuje, jeśli typ częściowej jest zadeklarowany jako inny rodzaj konstrukcji w różnych miejscach. Aby uzyskać więcej informacji, zobacz [klasy częściowe i metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0261:  
  
```csharp  
// CS0261.cs  
partial class A  // CS0261 – A declared as a class here, but as a struct below  
{  
}  
  
partial struct A  
{  
}  
```
