# <a name="compiler-error-cs1575"></a><span data-ttu-id="0aa75-102">CS1575 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="0aa75-102">Compiler Error CS1575</span></span>
<span data-ttu-id="0aa75-103">Wymaga wyrażeniu stackalloc po typie,]</span><span class="sxs-lookup"><span data-stu-id="0aa75-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="0aa75-104">Rozmiar żądanej alokacji z [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), należy określić w nawiasach kwadratowych.</span><span class="sxs-lookup"><span data-stu-id="0aa75-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="0aa75-105">Poniższy przykład generuje CS1575:</span><span class="sxs-lookup"><span data-stu-id="0aa75-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
