# <a name="compiler-error-cs1575"></a><span data-ttu-id="117c5-103">Błąd kompilatora CS1575</span><span class="sxs-lookup"><span data-stu-id="117c5-103">Compiler Error CS1575</span></span>
<span data-ttu-id="117c5-104">Wyrażenie stackalloc wymaga [] po typie</span><span class="sxs-lookup"><span data-stu-id="117c5-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="117c5-105">Rozmiar żądanego przydziału z [stackalloc](../language-reference/operators/stackalloc.md)musi być określony w nawiasach kwadratowych.</span><span class="sxs-lookup"><span data-stu-id="117c5-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="117c5-106">Poniższy przykład generuje CS1575:</span><span class="sxs-lookup"><span data-stu-id="117c5-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
