# <a name="compiler-error-cs0043"></a>CS0043 błąd kompilatora
Plik PDB 'Plik' ma format niepoprawny lub nieaktualny. Usuń go i ponownie skompilować.  
  
 Usuń plik .pdb dla tej kompilacji i ponownej kompilacji.
