# <a name="compiler-error-cs0043"></a>Błąd kompilatora CS0043

Plik PDB "File" ma nieprawidłowy format lub jest nieaktualny. Usuń je i Skompiluj ponownie.

 Usuń plik. pdb dla tej kompilacji i ponownie skompiluj.

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
