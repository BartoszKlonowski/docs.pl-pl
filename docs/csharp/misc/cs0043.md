# <a name="compiler-error-cs0043"></a>CS0043 błąd kompilatora
Plik PDB 'Plik' ma format niepoprawny lub nieaktualny. Usuń go i skompiluj ponownie.  
  
 Usuń plik PDB dla tej kompilacji i skompilować ją ponownie.
