# <a name="compiler-warning-level-2-cs0114"></a>CS0114 ostrzeżenie (poziom 2) kompilator
"function1" ukrywa dziedziczoną składową "function2". Aby bieżąca metoda przesłoniła tę implementację, Dodaj słowo kluczowe override. W przeciwnym razie Dodaj słowo kluczowe new.  
  
 Deklaracji w klasie jest w konflikcie z deklaracją w klasie bazowej, będą ukryte składowej klasy bazowej.  
  
 Aby uzyskać więcej informacji, zobacz [podstawowy](../../csharp/language-reference/keywords/base.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0114:  
  
```csharp  
// CS0114.cs  
// compile with: /W:2 /warnaserror  
abstract public class clx  
{  
   public abstract void f();  
}  
  
public class cly : clx  
{  
   public void f() // CS0114, hides base class member  
   // try the following line instead  
   // override public void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
