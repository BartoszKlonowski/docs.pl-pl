# <a name="compiler-error-cs0171"></a><span data-ttu-id="20670-102">Błąd kompilatora CS0171</span><span class="sxs-lookup"><span data-stu-id="20670-102">Compiler Error CS0171</span></span>
<span data-ttu-id="20670-103">Pole zapasowe dla automatycznie implementowanej właściwości „name” musi być w pełni przypisane przed zwróceniem sterowania obiektowi wywołującemu.</span><span class="sxs-lookup"><span data-stu-id="20670-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="20670-104">Rozważ wywołanie konstruktora bez parametrów z inicjatora konstruktora.</span><span class="sxs-lookup"><span data-stu-id="20670-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="20670-105">Konstruktor w [strukturze](../language-reference/builtin-types/struct.md) musi inicjować wszystkie pola w strukturze.</span><span class="sxs-lookup"><span data-stu-id="20670-105">A constructor in a [struct](../language-reference/builtin-types/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="20670-106">Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="20670-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="20670-107">Poniższy przykład generuje CS0171:</span><span class="sxs-lookup"><span data-stu-id="20670-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
