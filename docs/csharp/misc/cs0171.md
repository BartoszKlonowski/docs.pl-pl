<span data-ttu-id="f2d4c-103">Pole zapasowe dla automatycznie implementowanej właściwości "name" musi pełni przypisane przed zwróceniem sterowania do obiektu wywołującego.</span><span class="sxs-lookup"><span data-stu-id="f2d4c-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="f2d4c-104">Należy wziąć pod uwagę podczas wywoływania konstruktora domyślnego z inicjatora konstruktora.</span><span class="sxs-lookup"><span data-stu-id="f2d4c-104">Consider calling the default constructor from a constructor initializer.</span></span>

```csharp  
   MyStruct(int initField)   // CS0171

      // i = initField;      // uncomment this line to resolve this error

   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
