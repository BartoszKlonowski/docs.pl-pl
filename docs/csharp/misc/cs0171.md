# <a name="compiler-error-cs0171"></a>CS0171 błąd kompilatora
Pole zapasowe dla automatycznie implementowanej właściwości „name” musi być w pełni przypisane przed zwróceniem sterowania obiektowi wywołującemu. Należy wziąć pod uwagę podczas wywoływania konstruktora bez parametrów z inicjatora konstruktora.  
  
 Konstruktor w [struktury](../../csharp/language-reference/keywords/struct.md) musi inicjować wszystkich pól w strukturze. Aby uzyskać więcej informacji, zobacz [konstruktory](../../csharp/programming-guide/classes-and-structs/constructors.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0171:  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
