# <a name="compiler-error-cs0171"></a><span data-ttu-id="19f8c-103">Błąd kompilatora CS0171</span><span class="sxs-lookup"><span data-stu-id="19f8c-103">Compiler Error CS0171</span></span>
<span data-ttu-id="19f8c-104">Pole zapasowe dla automatycznie zaimplementowanej właściwości "name" musi być w pełni przypisane przed zwróceniem sterowania do obiektu wywołującego.</span><span class="sxs-lookup"><span data-stu-id="19f8c-104">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="19f8c-105">Rozważ wywołanie konstruktora bez parametrów z inicjatora konstruktora.</span><span class="sxs-lookup"><span data-stu-id="19f8c-105">Consider calling the parameterless constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="19f8c-106">Konstruktor w [strukturze](../language-reference/builtin-types/struct.md) musi inicjować wszystkie pola w strukturze.</span><span class="sxs-lookup"><span data-stu-id="19f8c-106">A constructor in a [struct](../language-reference/builtin-types/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="19f8c-107">Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="19f8c-107">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="19f8c-108">Poniższy przykład generuje CS0171:</span><span class="sxs-lookup"><span data-stu-id="19f8c-108">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
