# <a name="compiler-error-cs1648"></a>CS1648 błąd kompilatora
Nie można modyfikować składowych pola tylko do odczytu 'Identyfikator' (z wyjątkiem w konstruktorze lub inicjatorze zmiennych)  
  
 Ten błąd występuje, gdy spróbujesz zmodyfikować członkiem pola, które jest tylko do odczytu, według których nie jest dozwolone do zmodyfikowania. Aby rozwiązać ten problem, należy ograniczyć przypisania do pól tylko do odczytu w konstruktorze lub inicjatorze zmiennych, lub usuń readonly — słowo kluczowe z deklarację pola.  
  
 Poniższy przykład spowoduje wygenerowanie CS1648:  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
