<span data-ttu-id="45e2b-103">Nie można modyfikować składowych pola tylko do odczytu (z wyjątkiem w konstruktorze lub inicjatorze zmiennych).</span><span class="sxs-lookup"><span data-stu-id="45e2b-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>

 <span data-ttu-id="45e2b-104">Ten błąd występuje podczas próby zmodyfikowania elementu członkowskiego pola, które jest tylko do odczytu, jeśli nie można go modyfikować.</span><span class="sxs-lookup"><span data-stu-id="45e2b-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="45e2b-105">Aby rozwiązać ten problem, Ogranicz przydziały do pól tylko do odczytu do konstruktora lub inicjatora zmiennych lub Usuń słowo kluczowe readonly z deklaracji pola.</span><span class="sxs-lookup"><span data-stu-id="45e2b-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>

## <a name="example"></a><span data-ttu-id="45e2b-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="45e2b-106">Example</span></span>

 <span data-ttu-id="45e2b-107">Poniższy przykład generuje CS1648:</span><span class="sxs-lookup"><span data-stu-id="45e2b-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
