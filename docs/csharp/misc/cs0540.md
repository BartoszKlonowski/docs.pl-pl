## <a name="example"></a><span data-ttu-id="8892f-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="8892f-107">Example</span></span>  

 <span data-ttu-id="8892f-108">Poniższy przykład generuje CS0540.</span><span class="sxs-lookup"><span data-stu-id="8892f-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
