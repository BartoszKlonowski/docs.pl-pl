# <a name="compiler-error-cs0540"></a>CS0540 błąd kompilatora
"składowej interfejsu": typ zawierający nie implementuje interfejsu "interface"  
  
 Podjęto próbę implementować składowej interfejsu w [klasy](../../csharp/language-reference/keywords/class.md) nie jest pochodny od [interfejsu](../../csharp/language-reference/keywords/interface.md). Należy usunąć implementacji elementu członkowskiego interfejsu lub dodać interfejs do listy klas podstawowych, klasy.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0540.  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0540.  
  
```csharp  
// CS0540_b.cs  
using System;  
class C {  
   void IDisposable.Dispose() {}   // CS0540  
}  
  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
