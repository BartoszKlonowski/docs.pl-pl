## <a name="example"></a><span data-ttu-id="24666-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="24666-106">Example</span></span>  
 <span data-ttu-id="24666-107">Poniższy przykład generuje CS0540.</span><span class="sxs-lookup"><span data-stu-id="24666-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
