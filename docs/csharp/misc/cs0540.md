## <a name="example"></a><span data-ttu-id="e23cd-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="e23cd-106">Example</span></span>  
 <span data-ttu-id="e23cd-107">Poniższy przykład spowoduje wygenerowanie CS0540.</span><span class="sxs-lookup"><span data-stu-id="e23cd-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
