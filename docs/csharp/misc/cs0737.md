## <a name="example"></a><span data-ttu-id="b2ab7-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="b2ab7-110">Example</span></span>  

 <span data-ttu-id="b2ab7-111">Poniższy kod generuje CS0737:</span><span class="sxs-lookup"><span data-stu-id="b2ab7-111">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
