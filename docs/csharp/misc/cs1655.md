# <a name="compiler-error-cs1655"></a>CS1655 błąd kompilatora
Nie można przekazać pól 'Zmienna' jako ref lub out argument jest "typ zmiennej tylko do odczytu"  
  
 Ten błąd występuje, jeśli próbujesz przekazać, jest członkiem [foreach](../../csharp/language-reference/keywords/foreach-in.md) zmiennej [przy użyciu](../../csharp/language-reference/keywords/using-statement.md) zmiennej lub [stałej](../../csharp/language-reference/keywords/fixed-statement.md) zmienną do funkcji ze specyfikatorem ref ani out argumentu. Ponieważ te zmienne są traktowane jako tylko do odczytu w tych kontekstach, jest to niedozwolone.  
  
 Poniższy przykład spowoduje wygenerowanie CS1655:  
  
```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
