# <a name="compiler-error-cs1655"></a><span data-ttu-id="3b651-102">CS1655 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="3b651-102">Compiler Error CS1655</span></span>
<span data-ttu-id="3b651-103">Nie można przekazać pól 'Zmienna' jako ref lub out argument jest "typ zmiennej tylko do odczytu"</span><span class="sxs-lookup"><span data-stu-id="3b651-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="3b651-104">Ten błąd występuje, jeśli próbujesz przekazać, jest członkiem [foreach](../../csharp/language-reference/keywords/foreach-in.md) zmiennej [przy użyciu](../../csharp/language-reference/keywords/using-statement.md) zmiennej lub [stałej](../../csharp/language-reference/keywords/fixed-statement.md) zmienną do funkcji ze specyfikatorem ref ani out argumentu.</span><span class="sxs-lookup"><span data-stu-id="3b651-104">This error occurs if you are attempting to pass a member of a [foreach](../../csharp/language-reference/keywords/foreach-in.md) variable, a [using](../../csharp/language-reference/keywords/using-statement.md) variable, or a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="3b651-105">Ponieważ te zmienne są traktowane jako tylko do odczytu w tych kontekstach, jest to niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="3b651-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="3b651-106">Poniższy przykład spowoduje wygenerowanie CS1655:</span><span class="sxs-lookup"><span data-stu-id="3b651-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
