# <a name="compiler-error-cs1655"></a><span data-ttu-id="f8f33-102">CS1655 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="f8f33-102">Compiler Error CS1655</span></span>
<span data-ttu-id="f8f33-103">Nie można przekazać pól elementu "Zmienna" jako wartości ref lub out argumentu, ponieważ jest "typ zmiennej tylko do odczytu"</span><span class="sxs-lookup"><span data-stu-id="f8f33-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="f8f33-104">Ten błąd występuje podczas próby przekazania członkiem [foreach](../../csharp/language-reference/keywords/foreach-in.md) zmiennej [przy użyciu](../../csharp/language-reference/keywords/using-statement.md) zmiennej lub [stałej](../../csharp/language-reference/keywords/fixed-statement.md) zmiennej jako wartości ref lub out argumentu funkcji.</span><span class="sxs-lookup"><span data-stu-id="f8f33-104">This error occurs if you are attempting to pass a member of a [foreach](../../csharp/language-reference/keywords/foreach-in.md) variable, a [using](../../csharp/language-reference/keywords/using-statement.md) variable, or a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="f8f33-105">Ponieważ te zmienne są traktowane jako tylko do odczytu w tych kontekstów, co nie jest dozwolone.</span><span class="sxs-lookup"><span data-stu-id="f8f33-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="f8f33-106">Poniższy przykład generuje CS1655:</span><span class="sxs-lookup"><span data-stu-id="f8f33-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
