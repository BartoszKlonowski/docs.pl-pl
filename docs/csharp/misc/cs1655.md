# <a name="compiler-error-cs1655"></a><span data-ttu-id="d9939-102">Błąd kompilatora CS1655</span><span class="sxs-lookup"><span data-stu-id="d9939-102">Compiler Error CS1655</span></span>
<span data-ttu-id="d9939-103">Nie można przekazać pól "zmiennej" jako argumentu ref lub out, ponieważ jest to "typ zmiennej tylko do odczytu"</span><span class="sxs-lookup"><span data-stu-id="d9939-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="d9939-104">Ten błąd występuje, jeśli próbujesz przekazać element członkowski [foreach](../language-reference/keywords/foreach-in.md) zmiennej, [przy użyciu](../language-reference/keywords/using-statement.md) zmiennej lub [stałej](../language-reference/keywords/fixed-statement.md) zmiennej do funkcji jako argument ref lub out.</span><span class="sxs-lookup"><span data-stu-id="d9939-104">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="d9939-105">Ponieważ te zmienne są uważane za tylko do odczytu w tych kontekstach, nie jest to dozwolone.</span><span class="sxs-lookup"><span data-stu-id="d9939-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="d9939-106">Poniższy przykład generuje CS1655:</span><span class="sxs-lookup"><span data-stu-id="d9939-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
