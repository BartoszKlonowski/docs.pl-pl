# <a name="compiler-error-cs1600"></a>Błąd kompilatora CS1600
Kompilacja anulowana przez użytkownika  
  
 Kompilacja z kompilatorem C# została anulowana podczas korzystania ze środowiska IDE programu Visual Studio.
