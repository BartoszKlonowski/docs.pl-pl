# <a name="compiler-error-cs0704"></a>CS0704 błąd kompilatora
Nie można odszukać elementów członkowskich w 'type' ponieważ jest to parametr typu  
  
 Nie można określić typu wewnętrzny za pomocą parametru typu. Spróbuj jawnie przy użyciu odpowiedniego typu.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0704:  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
