# <a name="compiler-warning-level-4-cs0422"></a>Kompilatora (poziom 4) ostrzeżenie CS0422
/ Opcja przyrostowe nie jest już obsługiwana  
  
 Kompilacja przyrostowa (**/incr** lub **/ incremental**) nie jest obsługiwana w elemencie wizualnym C# 2005.
