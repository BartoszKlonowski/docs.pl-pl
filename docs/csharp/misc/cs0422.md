# <a name="compiler-warning-level-4-cs0422"></a>Ostrzeżenie kompilatora (poziom 4) CS0422

Opcja/Incremental nie jest już obsługiwana

 Kompilacja przyrostowa ( **/incr e** lub **/Incremental**) nie jest obsługiwana C# w 2,0.
 
> [!NOTE]
> To ostrzeżenie kompilatora nie jest już używane w Roslyn.
