# <a name="compiler-warning-level-4-cs0422"></a>Ostrzeżenie o kompilatorze (poziom 4) CS0422

Opcja /przyrostowa nie jest już obsługiwana

 Kompilacja przyrostowa (**/incr** lub **/przyrostowa)** nie jest obsługiwana w języku C# 2.0.

> [!NOTE]
> To ostrzeżenie kompilatora nie jest już używany w Roslyn.
