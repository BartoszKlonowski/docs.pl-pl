# <a name="compiler-error-cs0635"></a>CS0635 błąd kompilatora
'attribute': element System.Interop.UnmanagedType.CustomMarshaller wymaga nazwanych argumentów ComType oraz Marshal  
  
 **ComType** i **kierowanie** muszą być określone argumenty, gdy jest w formacie kierowanie **element System.Interop.UnmanagedType.CustomMarshaller**.
