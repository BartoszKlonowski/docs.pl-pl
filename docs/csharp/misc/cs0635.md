# <a name="compiler-error-cs0635"></a>Błąd kompilatora CS0635
"Attribute": System. Interop. UnmanagedType. CustomMarshaller wymaga nazwanych argumentów ComType i Marshal  
  
 Argumenty **ComType** i **Marshal** muszą być określone, jeśli format Marshal ma wartość **System. Interop. UnmanagedType. CustomMarshaller**.
