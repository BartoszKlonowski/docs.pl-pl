# <a name="compiler-warning-level-2-cs1572"></a>CS1572 ostrzeżenie (poziom 2) kompilator
Komentarz XML w elemencie "konstrukcji" ma tag param dla "parametru", ale nie ma parametru o takiej nazwie  
  
 Korzystając z [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md) — opcja kompilatora, komentarz został określony dla parametru, który nie istnieje w metodzie. Zmień wartość przekazana do atrybutu nazwy lub usuń jeden z wierszy komentarz.  
  
 Poniższy przykład spowoduje wygenerowanie CS1572:  
  
```csharp  
// CS1572.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>Used to indicate status.</param>  
   /// <param name='Char2'>???</param> // CS1572  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
