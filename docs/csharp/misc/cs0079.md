# <a name="compiler-error-cs0079"></a><span data-ttu-id="eb2ca-103">Błąd kompilatora CS0079</span><span class="sxs-lookup"><span data-stu-id="eb2ca-103">Compiler Error CS0079</span></span>

<span data-ttu-id="eb2ca-104">Zdarzenie "Event" może występować tylko po lewej stronie znaków + = lub-=</span><span class="sxs-lookup"><span data-stu-id="eb2ca-104">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="eb2ca-105">[Zdarzenie](../language-reference/keywords/event.md) zostało wywołane nieprawidłowo.</span><span class="sxs-lookup"><span data-stu-id="eb2ca-105">An [event](../language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="eb2ca-106">Aby uzyskać więcej informacji, zobacz [zdarzenia](../programming-guide/events/index.md) i [Delegaty](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="eb2ca-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="eb2ca-107">Poniższy przykład generuje CS0079:</span><span class="sxs-lookup"><span data-stu-id="eb2ca-107">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
