# <a name="compiler-error-cs0447"></a><span data-ttu-id="777ea-102">CS0447 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="777ea-102">Compiler Error CS0447</span></span>
<span data-ttu-id="777ea-103">Nie można użyć atrybutów w argumentach typu, tylko w parametrach typu</span><span class="sxs-lookup"><span data-stu-id="777ea-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="777ea-104">Ten błąd występuje w przypadku zastosowania atrybutu do argumentu typu, który występuje w instrukcji wywołania.</span><span class="sxs-lookup"><span data-stu-id="777ea-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="777ea-105">Jest to dopuszczalne zastosować atrybut do parametru typu w instrukcji deklaracji klasy lub metody takie jak następujące:</span><span class="sxs-lookup"><span data-stu-id="777ea-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```  
  
 <span data-ttu-id="777ea-106">Następujący wiersz kodu spowoduje wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="777ea-106">The following line of code will generate this error.</span></span> <span data-ttu-id="777ea-107">Zakłada ona, że klasa `C`zdefiniowanej w poprzednim wierszu kodu, ma statycznej metody o nazwie `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="777ea-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```
