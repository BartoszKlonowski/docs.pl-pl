# <a name="compiler-error-cs0447"></a><span data-ttu-id="3e483-103">Błąd kompilatora CS0447</span><span class="sxs-lookup"><span data-stu-id="3e483-103">Compiler Error CS0447</span></span>

<span data-ttu-id="3e483-104">Atrybutów nie można używać w argumentach typu, tylko w parametrach typu</span><span class="sxs-lookup"><span data-stu-id="3e483-104">Attributes cannot be used on type arguments, only on type parameters</span></span>

<span data-ttu-id="3e483-105">Ten błąd występuje w przypadku zastosowania atrybutu do argumentu typu, który występuje w instrukcji wywołania.</span><span class="sxs-lookup"><span data-stu-id="3e483-105">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="3e483-106">Możliwe jest stosowanie atrybutu do parametru typu w instrukcji klasy lub deklaracji metody, takiej jak:</span><span class="sxs-lookup"><span data-stu-id="3e483-106">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```csharp
class C<[some attribute] T> {…}
```

<span data-ttu-id="3e483-107">Następujący wiersz kodu spowoduje wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="3e483-107">The following line of code will generate this error.</span></span> <span data-ttu-id="3e483-108">Przyjęto założenie, że Klasa `C` , zdefiniowana w poprzednim wierszu kodu, ma metodę statyczną o nazwie `MyStaticMethod` .</span><span class="sxs-lookup"><span data-stu-id="3e483-108">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```csharp
C<[some attribute] T>.MyStaticMethod();
```

## <a name="example"></a><span data-ttu-id="3e483-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="3e483-109">Example</span></span>

<span data-ttu-id="3e483-110">Poniższy kod generuje błąd CS0447:</span><span class="sxs-lookup"><span data-stu-id="3e483-110">The following code generates error CS0447:</span></span>

```csharp
// CS0447.cs
using System;

namespace Test41
{
    public interface I<A>
    {
        void Meth<B>();
    }
    public class B : I<int>
    {
        void I<[Test] int>.Meth<X>() { }  // CS0447
    }
}
```
