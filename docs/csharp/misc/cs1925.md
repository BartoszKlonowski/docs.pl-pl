# <a name="compiler-error-cs1925"></a>CS1925 błąd kompilatora
Nie można zainicjować obiektu typu "type", przy użyciu inicjatora kolekcji.  
  
 Inicjatory kolekcji są dozwolone tylko w przypadku klas kolekcji, które spełniają określone kryteria. Aby uzyskać więcej informacji, zobacz [inicjatory obiektów i kolekcji](../../csharp/programming-guide/classes-and-structs/object-and-collection-initializers.md). Ten błąd również jest generowany, gdy użytkownik próbuje użyć krótkiej formy inicjatora tablicy zagnieżdżone wewnątrz inicjatora kolekcji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Inicjowanie obiektu przez wywołanie jego konstruktorów i metod.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1925:  
  
```csharp  
// cs1925.cs  
public class Student  
{  
    public int[] Scores;  
}  
  
class Test  
{  
    static void Main(string[] args)  
    {  
        Student student = new Student { Scores = { 1, 2, 3 } }; // CS1925  
    }  
}  
```
