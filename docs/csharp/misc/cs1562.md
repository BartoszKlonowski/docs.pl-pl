# <a name="compiler-error-cs1562"></a>Błąd kompilatora CS1562

Dla wyjść bez źródła musi być określona opcja/out  
  
 Kompilacja może utworzyć plik wyjściowy, ale nie istniał plik kodu źródłowego jako dane wejściowe, z którego może być implikowana nazwa pliku wyjściowego. Na przykład może być podejmowana próba skompilowania pliku metadanych lub samego zasobu.  
  
 Użyj opcji [kompilatora,](../language-reference/compiler-options/out-compiler-option.md) aby określić nazwę pliku wyjściowego.
