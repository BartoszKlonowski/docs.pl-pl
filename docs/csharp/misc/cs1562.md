# <a name="compiler-error-cs1562"></a>CS1562 błąd kompilatora
Dla wyjść bez źródła muszą mieć określono opcję/out.  
  
 Kompilacja, można utworzyć pliku wyjściowego, ale nie było żadnych pliku kodu źródłowego jako dane wejściowe, z którego może wynikać z nazw plików wyjściowych. Na przykład możesz spróbować skompilować pliku lub zasobu — tylko metadane.  
  
 Użyj [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) opcję kompilatora, aby określić nazwę pliku wyjściowego.
