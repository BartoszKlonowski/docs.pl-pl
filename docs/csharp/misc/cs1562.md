# <a name="compiler-error-cs1562"></a>CS1562 błąd kompilatora
Dla wyjść bez źródła musi być określona opcja/out  
  
 Kompilacja można utworzyć pliku wyjściowego, ale nie znaleziono żadnych pliku kodu źródłowego jako dane wejściowe, z którego można niejawnego nazwę pliku wyjściowego. Można na przykład próby skompiluj plik lub zasób — tylko metadane.  
  
 Użyj [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) opcję kompilatora, aby określić nazwę pliku wyjściowego.
