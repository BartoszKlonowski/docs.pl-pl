 <span data-ttu-id="a0466-105">Nie można zainicjować tablic wielowymiarowych w drzewach wyrażeń przy użyciu inicjatora tablicy.</span><span class="sxs-lookup"><span data-stu-id="a0466-105">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>

1. <span data-ttu-id="a0466-107">Utwórz i zainicjuj tablicę przed utworzeniem drzewa wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="a0466-107">Create and initialize the array before creating the expression tree.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0466-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="a0466-108">Example</span></span>  

 <span data-ttu-id="a0466-109">Poniższy przykład generuje CS0838:</span><span class="sxs-lookup"><span data-stu-id="a0466-109">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
