# <a name="compiler-error-cs0750"></a>CS0750 błąd kompilatora
Metoda częściowa nie może mieć modyfikatorów dostępu lub virtual, abstract, override, new, sealed lub extern modyfikatorów.  
  
 Ze względu na ich specjalnego zachowania metody częściowe obowiązują ograniczenia dotyczące modyfikatory, które mogą akceptować.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nieautoryzowanego modyfikatora w deklaracji metody.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0750:  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
