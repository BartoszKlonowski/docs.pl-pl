# <a name="compiler-error-cs0750"></a><span data-ttu-id="04576-103">Błąd kompilatora CS0750</span><span class="sxs-lookup"><span data-stu-id="04576-103">Compiler Error CS0750</span></span>

<span data-ttu-id="04576-104">Metoda częściowa nie może mieć modyfikatorów dostępu ani modyfikatorów Virtual, abstract, override, New, Sealed lub extern.</span><span class="sxs-lookup"><span data-stu-id="04576-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="04576-105">Ze względu na zachowanie specjalne metody częściowe podlegają ograniczeniom dotyczącym modyfikatorów, które mogą być akceptowane.</span><span class="sxs-lookup"><span data-stu-id="04576-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04576-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="04576-106">To correct this error</span></span>  
  
1. <span data-ttu-id="04576-107">Usuń nieautoryzowany modyfikator z deklaracji metody.</span><span class="sxs-lookup"><span data-stu-id="04576-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04576-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="04576-108">Example</span></span>  

 <span data-ttu-id="04576-109">Poniższy przykład generuje CS0750:</span><span class="sxs-lookup"><span data-stu-id="04576-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
