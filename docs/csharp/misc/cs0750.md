# <a name="compiler-error-cs0750"></a><span data-ttu-id="729ad-102">Błąd kompilatora CS0750</span><span class="sxs-lookup"><span data-stu-id="729ad-102">Compiler Error CS0750</span></span>
<span data-ttu-id="729ad-103">Metoda częściowa nie może mieć modyfikatorów dostępu ani modyfikatorów Virtual, abstract, override, New, Sealed lub extern.</span><span class="sxs-lookup"><span data-stu-id="729ad-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="729ad-104">Ze względu na zachowanie specjalne metody częściowe podlegają ograniczeniom dotyczącym modyfikatorów, które mogą być akceptowane.</span><span class="sxs-lookup"><span data-stu-id="729ad-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="729ad-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="729ad-105">To correct this error</span></span>  
  
1. <span data-ttu-id="729ad-106">Usuń nieautoryzowany modyfikator z deklaracji metody.</span><span class="sxs-lookup"><span data-stu-id="729ad-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="729ad-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="729ad-107">Example</span></span>  
 <span data-ttu-id="729ad-108">Poniższy przykład generuje CS0750:</span><span class="sxs-lookup"><span data-stu-id="729ad-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
