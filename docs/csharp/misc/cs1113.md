## <a name="example"></a><span data-ttu-id="c24df-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="c24df-109">Example</span></span>  
 <span data-ttu-id="c24df-110">Poniższy przykład generuje CS1113:</span><span class="sxs-lookup"><span data-stu-id="c24df-110">The following example generates CS1113:</span></span>  
  
```csharp  
// cs1113.cs  
using System;  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }  
}  
  
public struct S  
{  
}  
  
public class Test  
{  
    static int Main()  
    {  
        Func<S> f = new S().ExtMethod; // CS1113  
        return 1;  
    }  
}  
```
