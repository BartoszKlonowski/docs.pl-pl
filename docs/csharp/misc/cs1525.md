# <a name="compiler-error-cs1525"></a><span data-ttu-id="356cf-102">CS1525 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="356cf-102">Compiler Error CS1525</span></span>
<span data-ttu-id="356cf-103">Nieprawidłowe wyrażenie termin "character"</span><span class="sxs-lookup"><span data-stu-id="356cf-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="356cf-104">Kompilator wykrył nieprawidłowy znak w wyrażeniu.</span><span class="sxs-lookup"><span data-stu-id="356cf-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="356cf-105">Poniższy przykład generuje CS1525:</span><span class="sxs-lookup"><span data-stu-id="356cf-105">The following sample generates CS1525:</span></span>  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="356cf-106">Puste etykiety można również generować CS1525, jak w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="356cf-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
