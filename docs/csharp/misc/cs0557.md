# <a name="compiler-error-cs0557"></a><span data-ttu-id="87c6e-102">CS0557 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="87c6e-102">Compiler Error CS0557</span></span>
<span data-ttu-id="87c6e-103">Zduplikowana konwersja zdefiniowana przez użytkownika w typie "class"</span><span class="sxs-lookup"><span data-stu-id="87c6e-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="87c6e-104">Procedury konwersji zduplikowane są niedozwolone w klasie.</span><span class="sxs-lookup"><span data-stu-id="87c6e-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="87c6e-105">Poniższy przykład generuje CS0557:</span><span class="sxs-lookup"><span data-stu-id="87c6e-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
