## <a name="example"></a><span data-ttu-id="d707e-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d707e-107">Example</span></span>  
 <span data-ttu-id="d707e-108">Poniższy kod generuje CS1952:</span><span class="sxs-lookup"><span data-stu-id="d707e-108">The following code produces CS1952:</span></span>  
  
```csharp  
// cs1952.cs  
using System;  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```
