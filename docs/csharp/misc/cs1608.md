# <a name="compiler-error-cs1608"></a>Błąd kompilatora CS1608
Wymagany atrybut jest niedozwolony w typach C#  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> nie jest dozwolone dla typów zdefiniowanych w kodzie C#.
