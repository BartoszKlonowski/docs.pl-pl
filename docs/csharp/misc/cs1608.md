# <a name="compiler-error-cs1608"></a>CS1608 błąd kompilatora
Wymagany atrybut nie jest dozwolone w typach C#  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> nie jest dozwolona dla typów zdefiniowanych w kodzie języka C#.
