# <a name="compiler-error-cs2011"></a>CS2011 błąd kompilatora
Nie można otworzyć pliku 'Plik' odpowiedzi  
  
 Plik odpowiedzi została określona w kompilacji, ale kompilator nie może znaleźć i otworzyć plik.  
  
 Aby uzyskać więcej informacji na pliki odpowiedzi, zobacz [@ (Określ plik odpowiedzi)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
