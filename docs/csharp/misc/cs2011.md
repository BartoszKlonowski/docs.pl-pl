# <a name="compiler-error-cs2011"></a>CS2011 błąd kompilatora
Nie można otworzyć pliku plik odpowiedzi  
  
 Plik odpowiedzi został określony w kompilacji, ale kompilator nie może znaleźć i otworzyć plik.  
  
 Aby uzyskać więcej informacji na temat plików odpowiedzi, zobacz [@ (Określ plik odpowiedzi)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
