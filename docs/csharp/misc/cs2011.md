# <a name="compiler-error-cs2011"></a>Błąd kompilatora CS2011

Nie można otworzyć pliku odpowiedzi "File"  
  
 Plik odpowiedzi został określony w kompilacji, ale kompilator nie mógł zlokalizować i otworzyć tego pliku.  
  
 Aby uzyskać więcej informacji na temat plików odpowiedzi, zobacz [@ (Określ plik odpowiedzi)](../language-reference/compiler-options/response-file-compiler-option.md).
