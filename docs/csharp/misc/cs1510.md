## <a name="example"></a><span data-ttu-id="bd491-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="bd491-106">Example</span></span>  
 <span data-ttu-id="bd491-107">Poniższy przykład generuje CS1510:</span><span class="sxs-lookup"><span data-stu-id="bd491-107">The following sample generates CS1510:</span></span>  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
