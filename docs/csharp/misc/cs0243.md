 <span data-ttu-id="1c46c-106">Kompilator nigdy nie jest powiązany z Przesłaniaj metody; jest on tylko powiązany z metody podstawowej i środowisko uruchomieniowe języka wspólnego wywołuje override, zależnie od potrzeb.</span><span class="sxs-lookup"><span data-stu-id="1c46c-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="1c46c-107">Poniższy przykład generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="1c46c-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
