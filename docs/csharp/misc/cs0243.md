 <span data-ttu-id="feaa2-104">[Warunkowego](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atrybut nie jest dozwolony dla metody oznaczonej za pomocą [zastąpienia](../../csharp/language-reference/keywords/override.md) — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="feaa2-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="feaa2-105">Aby uzyskać więcej informacji, zobacz [wiedząc, gdy użycie zastępowania i nowych słów kluczowych](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="feaa2-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="feaa2-106">Kompilator nigdy nie jest powiązany z Przesłaniaj metody; jest on tylko powiązany z metody podstawowej i środowisko uruchomieniowe języka wspólnego wywołuje override, zależnie od potrzeb.</span><span class="sxs-lookup"><span data-stu-id="feaa2-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="feaa2-107">Poniższy przykład generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="feaa2-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
