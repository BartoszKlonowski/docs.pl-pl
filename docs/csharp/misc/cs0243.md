# <a name="compiler-error-cs0243"></a><span data-ttu-id="e7352-102">Błąd kompilatora CS0243</span><span class="sxs-lookup"><span data-stu-id="e7352-102">Compiler error CS0243</span></span>

<span data-ttu-id="e7352-103">Atrybut Warunkowy nie jest prawidłowy w przypadku "metody", ponieważ jest to metoda zastępowania</span><span class="sxs-lookup"><span data-stu-id="e7352-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="e7352-104">Atrybut <xref:System.Diagnostics.ConditionalAttribute> nie jest dozwolony w metodzie oznaczonej słowem kluczowym [override.](../language-reference/keywords/override.md)</span><span class="sxs-lookup"><span data-stu-id="e7352-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="e7352-105">Aby uzyskać więcej informacji, zobacz [Wiedza kiedy używać zastępowania i nowe słowa kluczowe](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="e7352-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="e7352-106">Kompilator nigdy nie wiąże się zastąpić metody.</span><span class="sxs-lookup"><span data-stu-id="e7352-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="e7352-107">Wiąże się tylko z podstawowych metod, a środowisko uruchomieniowe języka wspólnego wywołuje zastąpienie, odpowiednio.</span><span class="sxs-lookup"><span data-stu-id="e7352-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="e7352-108">Następujący kod generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="e7352-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
