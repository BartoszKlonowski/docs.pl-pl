# <a name="compiler-error-cs0243"></a><span data-ttu-id="7486b-103">Błąd kompilatora CS0243</span><span class="sxs-lookup"><span data-stu-id="7486b-103">Compiler error CS0243</span></span>

<span data-ttu-id="7486b-104">Atrybut Conditional jest nieprawidłowy w metodzie "Method", ponieważ jest to metoda przesłonięcia</span><span class="sxs-lookup"><span data-stu-id="7486b-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="7486b-105"><xref:System.Diagnostics.ConditionalAttribute>Atrybut jest niedozwolony w metodzie, która jest oznaczona za pomocą słowa kluczowego [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="7486b-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="7486b-106">Aby uzyskać więcej informacji, zobacz informacje o tym, [Kiedy używać przesłonięć i nowych słów kluczowych](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="7486b-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="7486b-107">Kompilator nigdy nie wiąże się z metodami przesłonięcia.</span><span class="sxs-lookup"><span data-stu-id="7486b-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="7486b-108">Jest on powiązany tylko z podstawową metodą, a środowisko uruchomieniowe języka wspólnego wywołuje przesłonięcie, zgodnie z potrzebami.</span><span class="sxs-lookup"><span data-stu-id="7486b-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="7486b-109">Poniższy kod generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="7486b-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
