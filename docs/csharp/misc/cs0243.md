# <a name="compiler-error-cs0243"></a>CS0243 błąd kompilatora
Atrybut Conditional jest nieprawidłowy w "method", ponieważ jest to metoda przesłonięcia  
  
 [Warunkowego](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atrybut nie jest dozwolony dla metody oznaczonej za pomocą [zastąpienia](../../csharp/language-reference/keywords/override.md) — słowo kluczowe. Aby uzyskać więcej informacji, zobacz [wiedząc, gdy użycie zastępowania i nowych słów kluczowych](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).  
  
 Kompilator nigdy nie jest powiązany z Przesłaniaj metody; jest on tylko powiązany z metody podstawowej i środowisko uruchomieniowe języka wspólnego wywołuje override, zależnie od potrzeb.  
  
 Poniższy przykład generuje CS0243:  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
