 <span data-ttu-id="f7d71-104">[Warunkowego](https://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atrybut nie jest dozwolony na metodzie, która jest oznaczona za pomocą [zastąpienia](../../csharp/language-reference/keywords/override.md) — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="f7d71-104">The [Conditional](https://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="f7d71-105">Aby uzyskać więcej informacji, zobacz [, wiedząc, gdy użycie zastępowania i nowych słów kluczowych](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="f7d71-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="f7d71-106">Kompilator nigdy nie wiąże się Przesłaniaj metody; tylko tworzy powiązanie z metody podstawowej, a środowisko uruchomieniowe języka wspólnego wywołuje override, zgodnie z potrzebami.</span><span class="sxs-lookup"><span data-stu-id="f7d71-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="f7d71-107">Poniższy przykład spowoduje wygenerowanie CS0243:</span><span class="sxs-lookup"><span data-stu-id="f7d71-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
