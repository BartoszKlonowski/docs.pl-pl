# <a name="compiler-error-cs1583"></a>CS1583 błąd kompilatora
'Plik' nie jest prawidłowym plikiem zasobów Win32  
  
 Ten błąd występuje, gdy należy określić nazwę pliku [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opcję kompilatora, która nie jest prawidłowy lub jest poprawnie sformatowany plik zasobów Win32. W programie Visual Studio w okienku aplikacji projektanta projektu określono nazwę pliku.
