# <a name="compiler-error-cs1583"></a>Błąd kompilatora CS1583
"plik" nie jest prawidłowym plikiem zasobów Win32  
  
 Ten błąd występuje w przypadku określenia nazwy pliku z opcją kompilatora [-win32res —](../language-reference/compiler-options/win32res-compiler-option.md) , która nie jest prawidłowym lub poprawnie sformatowanym plikiem zasobów Win32. W programie Visual Studio nazwa pliku jest określona w okienku aplikacji projektanta projektu.
