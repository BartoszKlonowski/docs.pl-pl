# <a name="compiler-error-cs1583"></a>CS1583 błąd kompilatora
'Plik' nie jest prawidłowym plikiem zasobów Win32  
  
 Ten błąd występuje, gdy należy określić nazwę pliku z [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opcję kompilatora, która nie jest prawidłowy lub poprawnie sformatowany plik zasobów Win32. W programie Visual Studio nazwa pliku jest określona w okienku aplikacji projektanta projektu.
