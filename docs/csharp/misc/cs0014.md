# <a name="compiler-error-cs0014"></a>CS0014 błąd kompilatora
Nie można odnaleźć wymaganego pliku 'Plik'.  
  
 Plik jest wymagane przez kompilator, ale nie w systemie. Upewnij się, że ścieżka jest poprawna. Jeśli plik jest [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] systemu plików, a następnie możesz może wymagać naprawy instalacji else Usuń i ponownie zainstalować [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] całkowicie.
