# <a name="compiler-error-cs0014"></a>CS0014 błąd kompilatora
Nie można odnaleźć wymaganego pliku 'Plik'  
  
 Plik jest wymagane przez kompilator, ale nie w systemie. Upewnij się, że ścieżka jest poprawna. Jeśli plik jest plikiem systemu Visual Studio, następnie konieczne może być napraw instalację lub — w przeciwnym razie Usuń i zainstaluj pakiet Visual Studio całkowicie.
