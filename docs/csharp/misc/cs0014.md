# <a name="compiler-error-cs0014"></a>Błąd kompilatora CS0014
Nie można odnaleźć wymaganego pliku "File"  
  
 Kompilator wymaga pliku, ale nie znajduje się w systemie. Upewnij się, że ścieżka jest poprawna. Jeśli plik jest plikiem systemowym programu Visual Studio, może być konieczne naprawienie instalacji lub całkowite usunięcie i ponowne zainstalowanie programu Visual Studio.
