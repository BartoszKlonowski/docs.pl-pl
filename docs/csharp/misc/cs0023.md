# <a name="compiler-error-cs0023"></a>CS0023 błąd kompilatora
Nie można zastosować operatora "operator" do argumentu operacji typu "type"  
  
 Nastąpiła próba do zastosowania operatora do zmiennej o typie nie był projektowany do pracy za pomocą operatora. Aby uzyskać więcej informacji, zobacz [typy](../../csharp/programming-guide/types/index.md) i [ C# operatory](../../csharp/language-reference/operators/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0023:  
  
```csharp  
// CS0023.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         string s = "hello";  
         s = -s;   // CS0023, minus operator not allowed on strings  
      }  
   }  
}  
```
