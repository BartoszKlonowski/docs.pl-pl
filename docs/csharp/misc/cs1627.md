# <a name="compiler-error-cs1627"></a><span data-ttu-id="71aa6-102">CS1627 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="71aa6-102">Compiler Error CS1627</span></span>
<span data-ttu-id="71aa6-103">Oczekiwano wyrażenia po zwróceniu przekazu</span><span class="sxs-lookup"><span data-stu-id="71aa6-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="71aa6-104">Ten błąd występuje, gdy `yield` jest używany bez wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="71aa6-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="71aa6-105">Aby uniknąć tego błędu, należy wstawić odpowiednie wyrażenie w instrukcji.</span><span class="sxs-lookup"><span data-stu-id="71aa6-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="71aa6-106">Poniższy przykład spowoduje wygenerowanie CS1627:</span><span class="sxs-lookup"><span data-stu-id="71aa6-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
