# <a name="compiler-error-cs1627"></a><span data-ttu-id="7b489-103">Błąd kompilatora CS1627</span><span class="sxs-lookup"><span data-stu-id="7b489-103">Compiler Error CS1627</span></span>

<span data-ttu-id="7b489-104">Oczekiwano wyrażenia po zwrocie Yield</span><span class="sxs-lookup"><span data-stu-id="7b489-104">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="7b489-105">Ten błąd występuje, gdy `yield` jest używany bez wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="7b489-105">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="7b489-106">Aby uniknąć tego błędu, Wstaw odpowiednie wyrażenie do instrukcji.</span><span class="sxs-lookup"><span data-stu-id="7b489-106">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="7b489-107">Poniższy przykład generuje CS1627:</span><span class="sxs-lookup"><span data-stu-id="7b489-107">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
