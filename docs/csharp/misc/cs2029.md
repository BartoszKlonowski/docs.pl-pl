# <a name="compiler-warning-level-1-cs2029"></a>Kompilatora (poziom 1) ostrzeżenie CS2029
Nieprawidłowa wartość "/ define"; 'Identyfikator' nie jest prawidłowym identyfikatorem  
  
 To ostrzeżenie występuje, gdy wartość, która jest używana w [/ define](../../csharp/language-reference/compiler-options/define-compiler-option.md) opcja zawiera nieprawidłowe znaki.  
  
 Nie można pominąć to ostrzeżenie [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opcji.
