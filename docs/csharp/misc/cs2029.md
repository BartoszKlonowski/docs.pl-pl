# <a name="compiler-warning-level-1-cs2029"></a>Ostrzeżenie kompilatora (poziom 1) CS2029
Nieprawidłowa wartość parametru "/define"; "Identyfikator" nie jest prawidłowym identyfikatorem  
  
 To ostrzeżenie występuje, gdy wartość użyta w opcji [/define](../language-reference/compiler-options/define-compiler-option.md) zawiera nieprawidłowe znaki.  
  
 To ostrzeżenie nie może być pomijane przez opcję [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
