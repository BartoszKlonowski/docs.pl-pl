# <a name="compiler-error-cs0146"></a>CS0146 błąd kompilatora
Cykliczna zależność klasy bazowej obejmująca 'klasa1' i 'klasa2'  
  
 Lista dziedziczenia klasy zawiera bezpośrednie lub pośrednie odwołanie do samego siebie. Klasa nie może dziedziczyć po sobie samym. Aby uzyskać więcej informacji, zobacz [dziedziczenia](../../csharp/programming-guide/classes-and-structs/inheritance.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0146:  
  
```csharp  
// CS0146.cs  
namespace MyNamespace  
{  
   public interface InterfaceA  
   {  
   }  
  
   public class MyClass : InterfaceA, MyClass2  
   {  
      public void Main()  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass   // CS0146  
   {  
   }  
}  
```
