 <span data-ttu-id="ed17a-106">Poniższy przykład spowoduje wygenerowanie CS0169:</span><span class="sxs-lookup"><span data-stu-id="ed17a-106">The following sample generates CS0169:</span></span>

{  
   int i;   // CS0169, i is not used anywhere
