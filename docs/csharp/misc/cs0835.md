## <a name="example"></a><span data-ttu-id="f8b72-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="f8b72-109">Example</span></span>  

 <span data-ttu-id="f8b72-110">Poniższy przykład generuje CS0835:</span><span class="sxs-lookup"><span data-stu-id="f8b72-110">The following example generates CS0835:</span></span>  
  
```csharp

using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
