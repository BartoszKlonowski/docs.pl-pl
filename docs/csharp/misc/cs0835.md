 <span data-ttu-id="40189-104">Jeśli wyrażenie lambda jest konwertowana na drzewo wyrażenia, drzewa wyrażenia musi mieć typ obiektu delegowanego dla jego argumentu.</span><span class="sxs-lookup"><span data-stu-id="40189-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="40189-105">Ponadto wyrażenia lambda musi być możliwe do przekonwertowania na typ delegowany.</span><span class="sxs-lookup"><span data-stu-id="40189-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="40189-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="40189-108">Example</span></span>  
 <span data-ttu-id="40189-109">Poniższy przykład generuje CS0835:</span><span class="sxs-lookup"><span data-stu-id="40189-109">The following example generates CS0835:</span></span>  
  
```csharp

using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
