 <span data-ttu-id="5e367-105">Jeśli wyrażenie lambda jest konwertowane na drzewo wyrażenia, drzewo wyrażenia musi mieć typ delegata dla tego argumentu.</span><span class="sxs-lookup"><span data-stu-id="5e367-105">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="5e367-106">Ponadto wyrażenie lambda musi być konwertowane na typ delegata.</span><span class="sxs-lookup"><span data-stu-id="5e367-106">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5e367-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="5e367-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="5e367-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="5e367-109">Example</span></span>  
 <span data-ttu-id="5e367-110">Poniższy przykład generuje CS0835:</span><span class="sxs-lookup"><span data-stu-id="5e367-110">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
