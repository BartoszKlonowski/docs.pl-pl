## <a name="example"></a><span data-ttu-id="84224-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="84224-105">Example</span></span>  
 <span data-ttu-id="84224-106">Poniższy przykład generuje CS0448.</span><span class="sxs-lookup"><span data-stu-id="84224-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```

   public static T? operator ++(T? t) { return new T(); }

}
