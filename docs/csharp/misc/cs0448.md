# <a name="compiler-error-cs0448"></a>Błąd kompilatora CS0448

Typ zwracany dla operatora + + lub--musi być typem zawierającym lub pochodnym z typu zawierającego  
  
 Podczas przesłonięcia `++` `--` operatorów or muszą zwracać ten sam typ co typ zawierający lub zwracać typ pochodzący od typu zawierającego.  
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS0448.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS0448.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
