# <a name="compiler-error-cs0113"></a>Błąd kompilatora CS0113
Składowa "Function" oznaczona jako override nie może być oznaczona jako New ani Virtual  
  
 Oznaczenie metody za pomocą słowa kluczowego [New](../language-reference/keywords/new-modifier.md) i [override](../language-reference/keywords/override.md) nie wykluczają się wzajemnie.  
  
 Poniższy przykład generuje CS0113:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
