# <a name="compiler-error-cs0113"></a><span data-ttu-id="27b1e-102">Błąd kompilatora CS0113</span><span class="sxs-lookup"><span data-stu-id="27b1e-102">Compiler Error CS0113</span></span>
<span data-ttu-id="27b1e-103">Składowa "Function" oznaczona jako override nie może być oznaczona jako New ani Virtual</span><span class="sxs-lookup"><span data-stu-id="27b1e-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="27b1e-104">Oznaczenie metody za pomocą słowa kluczowego [New](../language-reference/keywords/new-modifier.md) i [override](../language-reference/keywords/override.md) nie wykluczają się wzajemnie.</span><span class="sxs-lookup"><span data-stu-id="27b1e-104">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="27b1e-105">Poniższy przykład generuje CS0113:</span><span class="sxs-lookup"><span data-stu-id="27b1e-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
