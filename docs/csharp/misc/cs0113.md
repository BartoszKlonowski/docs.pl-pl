# <a name="compiler-error-cs0113"></a>CS0113 błąd kompilatora
Składowej "function" ze specyfikatorem override nie można oznaczyć jako nowej lub wirtualnej  
  
 Jest wzajemnie wykluczających się do oznaczania metody z [nowe](../../csharp/language-reference/keywords/new-modifier.md) i [zastąpienia](../../csharp/language-reference/keywords/override.md) słów kluczowych.  
  
 Poniższy przykład spowoduje wygenerowanie CS0113:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
