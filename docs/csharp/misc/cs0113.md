# <a name="compiler-error-cs0113"></a><span data-ttu-id="f3a42-102">CS0113 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="f3a42-102">Compiler Error CS0113</span></span>
<span data-ttu-id="f3a42-103">Element członkowski "function" oznaczona jako przesłonięty nie można oznaczyć jako nowy lub wirtualny</span><span class="sxs-lookup"><span data-stu-id="f3a42-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="f3a42-104">Jest wykluczają się wzajemnie, aby oznaczyć metodę o [nowe](../../csharp/language-reference/keywords/new.md) i [zastąpienia](../../csharp/language-reference/keywords/override.md) słów kluczowych.</span><span class="sxs-lookup"><span data-stu-id="f3a42-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="f3a42-105">Poniższy przykład generuje CS0113:</span><span class="sxs-lookup"><span data-stu-id="f3a42-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
