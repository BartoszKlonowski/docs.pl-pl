# <a name="compiler-error-cs1560"></a>CS1560 błąd kompilatora
Nieprawidłowa nazwa pliku określona dla dyrektywy preprocesora. Nazwa pliku jest zbyt długa lub nie prawidłowej nazwy pliku  
  
 Nazwa pliku, który został określony za pomocą [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) przekroczyła _MAX_PATH (256 znaków) lub wiersz, w którym `#line` znaleziono Przekroczono 2000 znaków.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1560.  
  
```csharp  
// cs1560.cs   
using System;   
class MyClass   
{     
   public static void Main()      
   {        
      Console.WriteLine("Normal line #1.");     
      #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560  
    }  
}  
```
