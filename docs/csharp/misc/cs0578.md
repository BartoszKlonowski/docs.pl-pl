## <a name="example"></a><span data-ttu-id="9df4d-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="9df4d-106">Example</span></span>

<span data-ttu-id="9df4d-107">Poniższy przykład generuje CS0578.</span><span class="sxs-lookup"><span data-stu-id="9df4d-107">The following sample generates CS0578.</span></span> <span data-ttu-id="9df4d-108">Aby rozwiązać ten problem, należy usunąć <xref:System.Diagnostics.ConditionalAttribute> lub zmienić wartość zwracaną metody na `void` .</span><span class="sxs-lookup"><span data-stu-id="9df4d-108">To resolve this error, you must either delete <xref:System.Diagnostics.ConditionalAttribute>, or you must change the return value of the method to `void`.</span></span>

```csharp

// compile with: /target:library
public class MyClass
{
    [System.Diagnostics.ConditionalAttribute("a")]   // CS0578
    public int TestMethod()
    {
        return 0;
    }
}
```
