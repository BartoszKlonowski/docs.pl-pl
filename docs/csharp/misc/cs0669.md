<span data-ttu-id="8369e-103">Klasa z atrybutem ComImport nie może mieć Konstruktor zdefiniowany przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="8369e-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="8369e-104">Warstwa międzyoperacyjności modelu COM w środowisku uruchomieniowym języka dostarcza Konstruktor [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) klasy.</span><span class="sxs-lookup"><span data-stu-id="8369e-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="8369e-105">W rezultacie obiekt COM może służyć jako zarządzanego obiektu w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="8369e-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

```csharp
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
