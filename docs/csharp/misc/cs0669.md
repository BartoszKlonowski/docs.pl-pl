# <a name="compiler-error-cs0669"></a><span data-ttu-id="7a1d3-103">Błąd kompilatora CS0669</span><span class="sxs-lookup"><span data-stu-id="7a1d3-103">Compiler Error CS0669</span></span>

<span data-ttu-id="7a1d3-104">Klasa o atrybucie atrybutem ComImport nie może mieć konstruktora zdefiniowanego przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="7a1d3-104">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="7a1d3-105">Warstwa międzyoperacyjna modelu COM w środowisku uruchomieniowym języka wspólnego udostępnia Konstruktor dla klas [atrybutem ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="7a1d3-105">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="7a1d3-106">W związku z tym obiekt COM może być używany jako obiekt zarządzany w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="7a1d3-106">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="7a1d3-107">Poniższy przykład generuje CS0669:</span><span class="sxs-lookup"><span data-stu-id="7a1d3-107">The following sample generates CS0669:</span></span>

```csharp

[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
