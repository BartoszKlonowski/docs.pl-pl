# <a name="compiler-error-cs0669"></a><span data-ttu-id="3894e-102">CS0669 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="3894e-102">Compiler Error CS0669</span></span>

<span data-ttu-id="3894e-103">Klasa z atrybutem ComImport nie może mieć Konstruktor zdefiniowany przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="3894e-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="3894e-104">Warstwa międzyoperacyjności modelu COM w środowisko uruchomieniowe języka wspólnego dostarcza Konstruktor [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) klasy.</span><span class="sxs-lookup"><span data-stu-id="3894e-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="3894e-105">W związku z tym obiekt COM może służyć jako zarządzanego obiektu w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="3894e-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="3894e-106">Poniższy przykład spowoduje wygenerowanie CS0669:</span><span class="sxs-lookup"><span data-stu-id="3894e-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
