<span data-ttu-id="dcfe4-103">'typ_1' nie jest zgodny ze specyfikacją CLS, ponieważ interfejs podstawowy 'typ_2' nie jest zgodny ze specyfikacją CLS</span><span class="sxs-lookup"><span data-stu-id="dcfe4-103">'type_1' is not CLS-compliant because base interface 'type_2' is not CLS-compliant</span></span>

## <a name="example"></a><span data-ttu-id="dcfe4-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="dcfe4-105">Example</span></span>  
 <span data-ttu-id="dcfe4-106">Poniższy przykład zawiera interfejs z metodą, która używa typie zgodnym ze specyfikacją CLS w jeho signatura wprowadzania typu niezgodnych ze specyfikacją CLS są zgodne.</span><span class="sxs-lookup"><span data-stu-id="dcfe4-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>

```csharp  
// compile with: /target:library

{  
   void IMethod(uint i);  
}  
```
