# <a name="compiler-error-cs2008"></a>CS2008 błąd kompilatora
Nie określono danych wejściowych  
  
 Kompilator została wywołana i opcje kompilatora zostały określone, ale nie pliki kodu źródłowego nie zostały przekazane.
