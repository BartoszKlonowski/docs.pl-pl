# <a name="compiler-error-cs1040"></a>CS1040 błąd kompilatora
Dyrektywy preprocesora muszą wystąpić jako pierwszy znak inny niż biały na linię  
  
 A [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md) został znaleziony w wierszu, a nie była pierwszym tokenie w wierszu. Dyrektywa musi być pierwszym tokenie w wierszu.  
  
 Poniższy przykład spowoduje wygenerowanie CS1040:  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
