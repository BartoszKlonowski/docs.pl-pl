# <a name="compiler-warning-level-4-cs0109"></a>Kompilatora (poziom 4) ostrzeżenie CS0109
Składowej 'składowa' nie ukrywa dziedziczonej składowej. Słowo kluczowe new nie jest wymagana  
  
 Deklarację klasy uwzględnione [nowe](../../csharp/language-reference/keywords/new.md) słowa kluczowego, nawet jeśli deklaracja nie powoduje zastąpienia istniejącej deklaracji w klasie bazowej. Możesz usunąć **nowe** — słowo kluczowe.  
  
 Poniższy przykład spowoduje wygenerowanie CS0109:  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
