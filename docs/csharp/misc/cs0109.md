<span data-ttu-id="a9a6f-104">Członek "member" nie ukrywa dziedziczonego elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="a9a6f-104">The member 'member' does not hide an inherited member.</span></span> <span data-ttu-id="a9a6f-105">Słowo kluczowe New nie jest wymagane</span><span class="sxs-lookup"><span data-stu-id="a9a6f-105">The new keyword is not required</span></span>  
  
 <span data-ttu-id="a9a6f-106">Deklaracja klasy zawiera słowo kluczowe [New](../language-reference/keywords/new-modifier.md) , mimo że deklaracja nie przesłania istniejącej deklaracji w klasie bazowej.</span><span class="sxs-lookup"><span data-stu-id="a9a6f-106">A class declaration included the [new](../language-reference/keywords/new-modifier.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="a9a6f-107">Można usunąć słowo kluczowe **New** .</span><span class="sxs-lookup"><span data-stu-id="a9a6f-107">You can delete the **new** keyword.</span></span>  
  
 <span data-ttu-id="a9a6f-108">Poniższy przykład generuje CS0109:</span><span class="sxs-lookup"><span data-stu-id="a9a6f-108">The following sample generates CS0109:</span></span>  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
