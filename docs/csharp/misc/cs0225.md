# <a name="compiler-error-cs0225"></a>CS0225 błąd kompilatora
Parametr params musi być tablicą jednowymiarową  
  
 Korzystając z [params](../../csharp/language-reference/keywords/params.md) — słowo kluczowe, należy określić Jednowymiarowa tablica typu danych. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0225:  
  
```csharp  
// CS0225.cs  
public class MyClass  
{  
    public static void TestParams(params int a)   // CS0225  
    // try the following line instead  
    // public static void TestParams(params int[] a)  
    {  
    }  
  
    public static void Main()  
    {  
        TestParams(1);  
    }  
}  
```
