# <a name="compiler-error-cs0100"></a>CS0100 błąd kompilatora
Nazwa parametru "parametr name" jest duplikatem  
  
 Tej samej nazwie parametr jest używany w deklaracji metody więcej niż raz. Nazwy parametrów muszą być unikatowe w deklaracji metody. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0100:  
  
```csharp  
// CS0100.cs  
namespace x  
{  
   public class a  
   {  
      public static void f(int i, char i)   // CS0100  
      // try the following line instead  
      // public static void f(int i, char j)  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
