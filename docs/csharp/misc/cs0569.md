# <a name="compiler-error-cs0569"></a>CS0569 błąd kompilatora
"metoda2": nie można zastąpić "metoda1", ponieważ nie jest obsługiwany przez język  
  
 Ten błąd występuje, gdy pochodzi z klasy podstawowej został zapisany w innym języku i kompilator nie obsługuje metody, które chcesz zastąpić.
