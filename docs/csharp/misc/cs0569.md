# <a name="compiler-error-cs0569"></a>CS0569 błąd kompilatora
"method2": nie może przesłonić "metoda1", ponieważ nie jest obsługiwany przez język  
  
 Ten błąd występuje, gdy pochodzi z klasy bazowej, który został zapisany w innym języku i w przypadku, gdy kompilator nie obsługuje metody, który chcesz zastąpić.
