# <a name="compiler-error-cs0569"></a>Błąd kompilatora CS0569

"method2": nie można przesłonić elementu "Metoda1", ponieważ nie jest on obsługiwany przez język  
  
 Ten błąd występuje, gdy pochodzi od klasy bazowej, która została zapisywana w innym języku i gdy kompilator nie obsługuje metody, która ma zostać przesłonięta.
