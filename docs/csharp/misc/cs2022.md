# <a name="compiler-error-cs2022"></a>CS2022 błąd kompilatora
Opcje "/ out" i "/ target" musi występować przed nazwami plików źródłowych  
  
 [/Out (ustawiona nazwa pliku wyjściowego)](../../csharp/language-reference/compiler-options/out-compiler-option.md) i [/TARGET (Określ dane wyjściowe pliku Format)](../../csharp/language-reference/compiler-options/target-compiler-option.md) — opcje kompilatora, gdy określona w wierszu polecenia musi poprzedzać plików kodu źródłowego.
