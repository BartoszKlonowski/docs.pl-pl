# <a name="compiler-error-cs2022"></a>Błąd kompilatora CS2022
Opcje "/out" i "/target" muszą występować przed nazwami plików źródłowych  
  
 Opcje kompilatora [/out (Ustaw nazwę pliku wyjściowego)](../language-reference/compiler-options/out-compiler-option.md) i [/Target (Określ format pliku wyjściowego)](../language-reference/compiler-options/target-compiler-option.md) , gdy są określone w wierszu polecenia, muszą poprzedzać pliki kodu źródłowego.
