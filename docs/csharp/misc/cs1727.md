# <a name="compiler-error-cs1727"></a>CS1727 błąd kompilatora
Nie można wysłać raport o błędach automatycznie bez autoryzacji. Odwiedź stronę "Aby autoryzować wysłanie raportu o błędzie.  
  
 Witryny sieci Web, na liście tekst błędu wyjaśnia, jak włączyć automatycznego raportowania błędów dotyczących [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)] narzędzi wiersza polecenia.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1727.  
  
```csharp  
// CS1727.cs  
// compile with: /errorreport:send  
// CS1727 expected  
class Test  
{  
    static void Main(){}  
}  
```
