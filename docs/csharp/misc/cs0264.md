# <a name="compiler-error-cs0264"></a>CS0264 błąd kompilatora
Częściowe deklaracje elementu "type" musi mieć takie same nazwy parametrów typu w identycznej kolejności  
  
 Ten błąd występuje podczas definiowania typu rodzajowego w częściowe deklaracje i parametry typu nie są zgodne w nazwie lub kolejności w całym wszystkie częściowe deklaracje. Aby pozbyć się ten błąd, sprawdź parametry typu dla każdego częściowa deklaracja i upewnij się taką samą nazwę i kolejność parametrów jest używany. Aby uzyskać więcej informacji, zobacz [klasy częściowe i metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).a [parametrów typu ogólnego](../../csharp/programming-guide/generics/generic-type-parameters.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0264.  
  
```csharp  
// CS0264.cs  
  
partial class MyClass<T>  // CS0264  
{  
}  
  
partial class MyClass <MyType>  
{  
}  
```
