# <a name="compiler-error-cs0742"></a>Błąd kompilatora CS0742
Treść zapytania musi kończyć się klauzulą SELECT lub klauzulą grupy  
  
 Wyrażenie zapytania musi kończyć się `select` klauzulą `group` lub klauzulą bez kontynuacji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj [klauzulę SELECT](../language-reference/keywords/select-clause.md) lub [](../language-reference/keywords/group-clause.md) Group do zapytania.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0742:  
  
```csharp  
// cs0742.cs  
using System.Linq;  
public class Test  
{  
    public static int Main()  
    {  
        int[] array = { 1, 2, 3 };  
        var query = from num in array; // CS0742  
        return 1;  
    }  
}  
```  
  
 Jeśli klauzula używa słowa kluczowego INTO do przechowywania wyników grupowania w tymczasowym identyfikatorze, nie może być ostatnią klauzulą w zapytaniu. [](../language-reference/keywords/into.md) `group` Nadal wymagana jest `group` klauzula lubdruga.`select`
