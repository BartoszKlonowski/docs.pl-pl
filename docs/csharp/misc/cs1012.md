# <a name="compiler-error-cs1012"></a>CS1012 błąd kompilatora
Zbyt wiele znaków w literale znakowym  
  
 Próbowano zainicjować [char](../../csharp/language-reference/keywords/char.md) stałej z więcej niż jeden znak.  
  
 CS1012 może również wystąpić podczas ustalania powiązanie danych. Na przykład następujący wiersz błąd uniemożliwiający:  
  
 `<%# DataBinder.Eval(Container.DataItem, 'doctitle') %>`  
  
 Zamiast tego spróbuj wykonać następujące polecenie:  
  
 `<%# DataBinder.Eval(Container.DataItem, "doctitle") %>`  
  
 Poniższy przykład spowoduje wygenerowanie CS1012:  
  
```csharp  
// CS1012.cs  
class Sample  
{  
   static void Main()  
   {  
      char a = 'xx';   // CS1012  
      char a2 = 'x';   // OK  
      System.Console.WriteLine(a2);  
   }  
}  
```
