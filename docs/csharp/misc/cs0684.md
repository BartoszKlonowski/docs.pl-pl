# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="81ccc-102">Kompilatora (poziom 1) ostrzeżenie CS0684</span><span class="sxs-lookup"><span data-stu-id="81ccc-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="81ccc-103">interfejs "interface" z oznaczeniem "CoClassAttribute" nie ma oznaczenia "ComImportAttribute"</span><span class="sxs-lookup"><span data-stu-id="81ccc-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="81ccc-104">Jeśli określisz **CoClassAttribute** w interfejsie, należy także określić **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="81ccc-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="81ccc-105">Poniższy przykład spowoduje wygenerowanie CS0684:</span><span class="sxs-lookup"><span data-stu-id="81ccc-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
