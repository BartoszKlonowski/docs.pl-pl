# <a name="compiler-warning-level-1-cs0684"></a>Kompilator CS0684 ostrzegawcze (poziom 1)
interfejs "interface" z oznaczeniem "CoClassAttribute" nie ma oznaczenia "ComImportAttribute"  
  
 Jeśli określisz **CoClassAttribute** w interfejsie, należy także określić **ComImportAttribute**.  
  
 Poniższy przykład generuje CS0684:  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
