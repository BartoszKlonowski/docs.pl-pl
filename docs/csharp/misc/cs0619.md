# <a name="compiler-error-cs0619"></a><span data-ttu-id="e2d87-102">Błąd kompilatora CS0619</span><span class="sxs-lookup"><span data-stu-id="e2d87-102">Compiler Error CS0619</span></span>
<span data-ttu-id="e2d87-103">element "member" jest przestarzały: "text"</span><span class="sxs-lookup"><span data-stu-id="e2d87-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="e2d87-104">Element członkowski klasy został oznaczony przy użyciu [przestarzały] ((... /Programming-Guide/Concepts/Attributes/Common-Attributes.MD # przestarzały), taki jak błąd zostanie wystawiony, gdy występuje odwołanie do składowej klasy.</span><span class="sxs-lookup"><span data-stu-id="e2d87-104">A class member was marked with the [Obsolete]((../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="e2d87-105">Poniższy przykład generuje CS0619:</span><span class="sxs-lookup"><span data-stu-id="e2d87-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
