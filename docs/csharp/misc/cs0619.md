# <a name="compiler-error-cs0619"></a><span data-ttu-id="acb90-102">CS0619 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="acb90-102">Compiler Error CS0619</span></span>
<span data-ttu-id="acb90-103">"członek" jest przestarzały: "tekst"</span><span class="sxs-lookup"><span data-stu-id="acb90-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="acb90-104">Element członkowski klasy została oznaczona atrybutem [przestarzałe](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atrybutu w taki sposób, że gdy odwołuje się do elementu członkowskiego klasy zostanie wygenerowany błąd.</span><span class="sxs-lookup"><span data-stu-id="acb90-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="acb90-105">Poniższy przykład generuje CS0619:</span><span class="sxs-lookup"><span data-stu-id="acb90-105">The following sample generates CS0619:</span></span>  
  
```  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
