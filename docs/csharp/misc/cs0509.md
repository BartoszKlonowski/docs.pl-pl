# <a name="compiler-error-cs0509"></a>CS0509 błąd kompilatora
'klasa1': nie może pochodzić od typu zapieczętowanego 'klasa2'  
  
 A [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md) klasy nie może działać jako [podstawowy](../../csharp/language-reference/keywords/base.md) klasy. Domyślnie są zapieczętowane struktury.  
  
 Poniższy przykład spowoduje wygenerowanie CS0509:  
  
```csharp  
// CS0509.cs  
// compile with: /target:library  
sealed public class clx {}  
public class cly : clx {}   // CS0509  
```
