# <a name="compiler-error-cs1672"></a>Błąd kompilatora CS1672

Nieprawidłowa opcja "Option" dla/platform; musi to być AnyCPU, x86, Itanium lub x64  
  
 Opcje określają typ procesora; Użyj jednej z wymienionych formularzy.
