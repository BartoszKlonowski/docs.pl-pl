# <a name="compiler-error-cs0637"></a><span data-ttu-id="d214e-102">Błąd kompilatora CS0637</span><span class="sxs-lookup"><span data-stu-id="d214e-102">Compiler Error CS0637</span></span>

<span data-ttu-id="d214e-103">Atrybut FieldOffset jest niedozwolony w polach static i const.</span><span class="sxs-lookup"><span data-stu-id="d214e-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="d214e-104">Atrybutu [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) nie można używać dla pól oznaczonych jako [static](../language-reference/keywords/static.md) ani [const](../language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="d214e-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="d214e-105">Poniższy przykład generuje CS0637:</span><span class="sxs-lookup"><span data-stu-id="d214e-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
