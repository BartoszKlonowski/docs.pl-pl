# <a name="compiler-error-cs0637"></a><span data-ttu-id="cea6a-102">CS0637 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="cea6a-102">Compiler Error CS0637</span></span>

<span data-ttu-id="cea6a-103">Atrybut FieldOffset jest niedozwolony w polach typu static lub const.</span><span class="sxs-lookup"><span data-stu-id="cea6a-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="cea6a-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) nie można używać atrybutu pola oznaczone [statycznych](../../csharp/language-reference/keywords/static.md) lub [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="cea6a-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="cea6a-105">Poniższy przykład generuje CS0637:</span><span class="sxs-lookup"><span data-stu-id="cea6a-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
