# <a name="compiler-error-cs1641"></a><span data-ttu-id="d592d-103">Błąd kompilatora CS1641</span><span class="sxs-lookup"><span data-stu-id="d592d-103">Compiler Error CS1641</span></span>
<span data-ttu-id="d592d-104">Pole buforu o ustalonym rozmiarze musi mieć specyfikator rozmiaru tablicy po nazwie pola</span><span class="sxs-lookup"><span data-stu-id="d592d-104">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="d592d-105">W przeciwieństwie do zwykłych tablic, bufory o ustalonym rozmiarze wymagają określenia stałego rozmiaru w punkcie deklaracji.</span><span class="sxs-lookup"><span data-stu-id="d592d-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="d592d-106">Aby rozwiązać ten problem, Dodaj literał dodatniej liczby całkowitej lub stałą dodatnią liczbę całkowitą i Umieść nawiasy kwadratowe po identyfikatorze.</span><span class="sxs-lookup"><span data-stu-id="d592d-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="d592d-107">Poniższy przykład generuje CS1641:</span><span class="sxs-lookup"><span data-stu-id="d592d-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
