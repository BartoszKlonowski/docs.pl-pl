# <a name="compiler-error-cs1626"></a><span data-ttu-id="db6f7-102">CS1626 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="db6f7-102">Compiler Error CS1626</span></span>
<span data-ttu-id="db6f7-103">Nie można przekazać wartości w treści bloku try z klauzulą catch</span><span class="sxs-lookup"><span data-stu-id="db6f7-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="db6f7-104">Instrukcja yield nie jest dozwolona w bloku try, jeśli klauzuli catch skojarzone z bloku try.</span><span class="sxs-lookup"><span data-stu-id="db6f7-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="db6f7-105">Aby uniknąć tego błędu, Przenieś instrukcji yield poza blok try/catch/finally lub Usuń blok catch.</span><span class="sxs-lookup"><span data-stu-id="db6f7-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="db6f7-106">Poniższy przykład spowoduje wygenerowanie CS1626:</span><span class="sxs-lookup"><span data-stu-id="db6f7-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {
      
      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
