# <a name="compiler-error-cs1626"></a><span data-ttu-id="8bf88-102">Błąd kompilatora CS1626</span><span class="sxs-lookup"><span data-stu-id="8bf88-102">Compiler Error CS1626</span></span>
<span data-ttu-id="8bf88-103">Nie można uzyskać wartości w treści bloku try z klauzulą catch</span><span class="sxs-lookup"><span data-stu-id="8bf88-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="8bf88-104">Yield Instrukcji nie jest dozwolone w try bloku, jeśli istnieje catch klauzuli skojarzone z try bloku.</span><span class="sxs-lookup"><span data-stu-id="8bf88-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="8bf88-105">Aby uniknąć tego błędu, przenieś yield instrukcji z try/catch/finally bloku lub usunąć catch bloku.</span><span class="sxs-lookup"><span data-stu-id="8bf88-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="8bf88-106">Poniższy przykład generuje CS1626:</span><span class="sxs-lookup"><span data-stu-id="8bf88-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
