# <a name="compiler-error-cs1626"></a><span data-ttu-id="477e5-103">Błąd kompilatora CS1626</span><span class="sxs-lookup"><span data-stu-id="477e5-103">Compiler Error CS1626</span></span>

<span data-ttu-id="477e5-104">Nie można przekazać wartości w treści bloku try z klauzulą catch</span><span class="sxs-lookup"><span data-stu-id="477e5-104">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="477e5-105">Instrukcja Yield jest niedozwolona w bloku try, jeśli istnieje klauzula catch skojarzona z blokiem try.</span><span class="sxs-lookup"><span data-stu-id="477e5-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="477e5-106">Aby uniknąć tego błędu, Przenieś instrukcję Yield z bloku try/catch/finally lub Usuń blok catch.</span><span class="sxs-lookup"><span data-stu-id="477e5-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="477e5-107">Poniższy przykład generuje CS1626:</span><span class="sxs-lookup"><span data-stu-id="477e5-107">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
