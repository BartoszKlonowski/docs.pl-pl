# <a name="compiler-warning-level-1-cs3012"></a>Kompilatora (poziom 1) ostrzeżenie CS3012

Nie można określić atrybutu CLSCompliant w module, który różni się od atrybutu CLSCompliant w zestawie

W kolejności dla modułu były zgodne z Common Language Specification (CLS) za pośrednictwem `[module:System.CLSCompliant(true)]`, muszą zostać skompilowane przy użyciu [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) — opcja kompilatora. Aby uzyskać więcej informacji na temat specyfikację CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).

## <a name="example"></a>Przykład

Poniższy przykład podczas kompilowania bez `/target:module`, generuje CS3012:

```csharp
// CS3012.cs
// compile with: /W:1

[module:System.CLSCompliant(true)]   // CS3012
public class C
{
    public static void Main()
    {
    }
}
```
