# <a name="compiler-error-cs0157"></a><span data-ttu-id="f93dc-103">Błąd kompilatora CS0157</span><span class="sxs-lookup"><span data-stu-id="f93dc-103">Compiler Error CS0157</span></span>
<span data-ttu-id="f93dc-104">Kontrolka nie może opuścić treści klauzuli finally</span><span class="sxs-lookup"><span data-stu-id="f93dc-104">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="f93dc-105">Wszystkie instrukcje w klauzuli [finally](../language-reference/keywords/try-catch-finally.md) muszą zostać wykonane.</span><span class="sxs-lookup"><span data-stu-id="f93dc-105">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="f93dc-106">Aby uzyskać więcej informacji, zobacz [wyjątki i obsługa wyjątków](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="f93dc-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="f93dc-107">Poniższy przykład generuje CS0157:</span><span class="sxs-lookup"><span data-stu-id="f93dc-107">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
