 <span data-ttu-id="d6fa7-104">[Zdarzenie](../language-reference/keywords/event.md) zostało zadeklarowane, ale nigdy nie użyte w klasie, w której została zadeklarowana.</span><span class="sxs-lookup"><span data-stu-id="d6fa7-104">An [event](../language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>  
  
 <span data-ttu-id="d6fa7-105">Poniższy przykład generuje CS0067:</span><span class="sxs-lookup"><span data-stu-id="d6fa7-105">The following sample generates CS0067:</span></span>

```csharp  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067

   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
