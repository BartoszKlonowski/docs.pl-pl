 <span data-ttu-id="17bce-104">[Zdarzeń](../../csharp/language-reference/keywords/event.md) została zadeklarowana, ale nigdy nie jest używana w klasie, w którym został zadeklarowany.</span><span class="sxs-lookup"><span data-stu-id="17bce-104">An [event](../../csharp/language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>

```csharp  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067

   {  
      if (evt != null)  
         evt();  
   }  
*/

   {  
   }  
}  
```
