# <a name="compiler-warning-level-3-cs0067"></a>Kompilator CS0067 ostrzegawcze (poziom 3)
Zdarzenie "event" nie jest nigdy używana  
  
 [Zdarzeń](../../csharp/language-reference/keywords/event.md) został zadeklarowany, ale nigdy nie są używane w klasie, w którym został zadeklarowany.  
  
 Poniższy przykład generuje CS0067:  
  
```csharp  
// CS0067.cs  
// compile with: /W:3  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067  
/*  
   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
