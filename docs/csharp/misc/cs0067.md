# <a name="compiler-warning-level-3-cs0067"></a><span data-ttu-id="e923e-103">Ostrzeżenie kompilatora (poziom 3) CS0067</span><span class="sxs-lookup"><span data-stu-id="e923e-103">Compiler Warning (level 3) CS0067</span></span>
<span data-ttu-id="e923e-104">Zdarzenie "Event" nie jest nigdy używane</span><span class="sxs-lookup"><span data-stu-id="e923e-104">The event 'event' is never used</span></span>  
  
 <span data-ttu-id="e923e-105">[Zdarzenie](../language-reference/keywords/event.md) zostało zadeklarowane, ale nigdy nie użyte w klasie, w której została zadeklarowana.</span><span class="sxs-lookup"><span data-stu-id="e923e-105">An [event](../language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>  
  
 <span data-ttu-id="e923e-106">Poniższy przykład generuje CS0067:</span><span class="sxs-lookup"><span data-stu-id="e923e-106">The following sample generates CS0067:</span></span>  
  
```csharp  
// CS0067.cs  
// compile with: /W:3  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067  
/*  
   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
