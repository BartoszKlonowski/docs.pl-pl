# <a name="compiler-error-cs0572"></a><span data-ttu-id="55193-103">Błąd kompilatora CS0572</span><span class="sxs-lookup"><span data-stu-id="55193-103">Compiler Error CS0572</span></span>
<span data-ttu-id="55193-104">"Type": nie można odwoływać się do typu za pomocą wyrażenia; Zamiast tego spróbuj użyć "path_to_type"</span><span class="sxs-lookup"><span data-stu-id="55193-104">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="55193-105">Podjęto próbę uzyskania dostępu do elementu członkowskiego klasy za pomocą identyfikatora, który jest niedozwolony w tej sytuacji.</span><span class="sxs-lookup"><span data-stu-id="55193-105">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="55193-106">Poniższy przykład generuje CS0572:</span><span class="sxs-lookup"><span data-stu-id="55193-106">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
