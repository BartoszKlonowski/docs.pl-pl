# <a name="compiler-error-cs0572"></a><span data-ttu-id="efbfe-102">CS0572 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="efbfe-102">Compiler Error CS0572</span></span>
<span data-ttu-id="efbfe-103">"type": nie może odwoływać się do typu przy użyciu wyrażenia. Zamiast tego spróbuj wykonać "path_to_type"</span><span class="sxs-lookup"><span data-stu-id="efbfe-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="efbfe-104">Nastąpiła próba dostępu do składowej klasy za pomocą identyfikatora, który nie jest dozwolona w tej sytuacji.</span><span class="sxs-lookup"><span data-stu-id="efbfe-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="efbfe-105">Poniższy przykład spowoduje wygenerowanie CS0572:</span><span class="sxs-lookup"><span data-stu-id="efbfe-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
