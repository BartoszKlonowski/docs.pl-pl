# <a name="compiler-error-cs0572"></a>CS0572 błąd kompilatora
'type': nie można odwoływać się do typu przy użyciu wyrażenia. Zamiast tego spróbuj "path_to_type"  
  
 Nastąpiła próba dostępu do elementu członkowskiego klasy za pomocą identyfikatora, który nie jest dozwolony w tej sytuacji.  
  
 Poniższy przykład generuje CS0572:  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
