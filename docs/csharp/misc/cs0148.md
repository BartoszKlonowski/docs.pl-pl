# <a name="compiler-error-cs0148"></a>CS0148 błąd kompilatora
Delegat "delegowanie" nie ma prawidłowego konstruktora  
  
 Importowane i używane zarządzanego programu (jedną, która używa .NET Framework środowisko uruchomieniowe języka wspólnego) który został utworzony za pomocą innego kompilatora. Przez kompilator może źle sformułowane [delegować](../../csharp/language-reference/keywords/delegate.md) konstruktora. Aby uzyskać więcej informacji, zobacz [delegatów](../../csharp/programming-guide/delegates/index.md).
