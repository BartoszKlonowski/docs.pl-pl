# <a name="compiler-error-cs0148"></a>Błąd kompilatora CS0148
Delegat delegata nie ma prawidłowego konstruktora  
  
 Został zaimportowany i użyty program zarządzany (taki, który używa środowiska uruchomieniowego języka wspólnego .NET Framework), który został utworzony za pomocą innego kompilatora. Ten kompilator umożliwił niewłaściwie sformułowany Konstruktor [delegatów](../language-reference/builtin-types/reference-types.md) . Aby uzyskać więcej informacji, zobacz [delegats](../programming-guide/delegates/index.md).
