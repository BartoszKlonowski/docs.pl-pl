# <a name="compiler-error-cs0148"></a>Błąd kompilatora CS0148
Delegat delegata nie ma prawidłowego konstruktora  
  
 Został zaimportowany i użyty program zarządzany (taki, który używa środowiska uruchomieniowego języka wspólnego .NET Framework), który został utworzony za pomocą innego kompilatora. Ten kompilator umożliwił niewłaściwie sformułowany [](../language-reference/keywords/delegate.md) Konstruktor delegatów. Aby uzyskać więcej informacji, [](../programming-guide/delegates/index.md)Zobacz delegats.
