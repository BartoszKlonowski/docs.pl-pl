# <a name="compiler-error-cs0148"></a>CS0148 błąd kompilatora
Delegat "delegate" nie ma prawidłowego konstruktora  
  
 Zaimportowane i używany program zarządzanych (taki, który używa środowiska .NET Framework środowisko uruchomieniowe języka wspólnego) utworzony z innego kompilatora. Dozwolone przez kompilator źle sformułowane [delegować](../../csharp/language-reference/keywords/delegate.md) konstruktora. Aby uzyskać więcej informacji, zobacz [delegatów](../../csharp/programming-guide/delegates/index.md).
