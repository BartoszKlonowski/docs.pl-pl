 <span data-ttu-id="b7596-104">Klasa, która jawnie implementuje Właściwość interfejsu, musi implementować wszystkie metody dostępu zdefiniowane przez interfejs.</span><span class="sxs-lookup"><span data-stu-id="b7596-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="b7596-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="b7596-106">Example</span></span>  
 <span data-ttu-id="b7596-107">Poniższy przykład generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="b7596-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
