# <a name="compiler-error-cs0551"></a>CS0551 błąd kompilatora
Jawna implementacja interfejsu "wdrożenia" brakuje metody dostępu "metody dostępu"  
  
 Klasa, która jawnie implementuje właściwość interfejsu musi implementować wszystkie metody dostępu, które definiuje interfejs.  
  
 Aby uzyskać więcej informacji, zobacz [przy użyciu właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0551.  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
