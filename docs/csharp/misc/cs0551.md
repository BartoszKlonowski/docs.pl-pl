# <a name="compiler-error-cs0551"></a><span data-ttu-id="1fde7-103">Błąd kompilatora CS0551</span><span class="sxs-lookup"><span data-stu-id="1fde7-103">Compiler Error CS0551</span></span>

<span data-ttu-id="1fde7-104">W jawnej implementacji interfejsu "implementacja" brakuje metody dostępu "akcesor"</span><span class="sxs-lookup"><span data-stu-id="1fde7-104">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="1fde7-105">Klasa, która jawnie implementuje Właściwość interfejsu, musi implementować wszystkie metody dostępu zdefiniowane przez interfejs.</span><span class="sxs-lookup"><span data-stu-id="1fde7-105">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="1fde7-106">Aby uzyskać więcej informacji, zobacz [Używanie właściwości](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="1fde7-106">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1fde7-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="1fde7-107">Example</span></span>  

 <span data-ttu-id="1fde7-108">Poniższy przykład generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="1fde7-108">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
