## <a name="example"></a><span data-ttu-id="cdad7-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="cdad7-106">Example</span></span>  
 <span data-ttu-id="cdad7-107">Poniższy przykład generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="cdad7-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
