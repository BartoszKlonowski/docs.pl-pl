## <a name="example"></a><span data-ttu-id="660ee-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="660ee-107">Example</span></span>

```csharp  
// CS0305.cs

{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
