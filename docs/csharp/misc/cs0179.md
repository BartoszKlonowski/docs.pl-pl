# <a name="compiler-error-cs0179"></a>CS0179 błąd kompilatora
'składowa' nie może być zewnętrzny ani deklarować treści  
  
 Gdy członek klasy jest oznaczony [extern](../../csharp/language-reference/keywords/extern.md), oznacza to, że definicja składowej znajduje się w innym pliku. W związku z tym, element członkowski klasy oznaczona jako **extern** nie może być zdefiniowana w klasie. Usunąć `extern` — słowo kluczowe lub usuń definicję. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0179:  
  
```csharp  
// CS0179.cs  
public class MyClass  
{  
   public extern int ExternMethod(int aa)   // CS0179  
   {  
      return 0;  
   }  
   // try the following line instead  
   // public extern int ExternMethod(int aa);  
  
   public static void Main()  
   {  
   }  
}  
```
