# <a name="compiler-error-cs0531"></a><span data-ttu-id="0d222-103">Błąd kompilatora CS0531</span><span class="sxs-lookup"><span data-stu-id="0d222-103">Compiler Error CS0531</span></span>
<span data-ttu-id="0d222-104">"member": składowe interfejsu nie mogą mieć definicji</span><span class="sxs-lookup"><span data-stu-id="0d222-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="0d222-105">Metody, które są zadeklarowane w [interfejsie](../language-reference/keywords/interface.md) , muszą być zaimplementowane w klasie, która dziedziczy z niej, a nie w samym interfejsie.</span><span class="sxs-lookup"><span data-stu-id="0d222-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="0d222-106">Poniższy przykład generuje CS0531:</span><span class="sxs-lookup"><span data-stu-id="0d222-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
