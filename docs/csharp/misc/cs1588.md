# <a name="compiler-error-cs1588"></a>CS1588 błąd kompilatora
Nie można określić katalogu środowiska CLR — "reason"  
  
 Kompilator nie może pobrać wykonywalnych języka wspólnego .NET Framework jest zainstalowana w katalogu. Ten błąd wskazuje, nieprawidłowo zainstalowanie środowiska uruchomieniowego języka wspólnego.
