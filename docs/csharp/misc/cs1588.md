# <a name="compiler-error-cs1588"></a>CS1588 błąd kompilatora
Nie można określić katalogu środowiska CLR--"Przyczyna"  
  
 Kompilator nie można pobrać zainstalowanej .NET Framework środowisko uruchomieniowe języka wspólnego w katalogu. Ten błąd oznacza, że nieprawidłowo zainstalowano środowisko uruchomieniowe języka wspólnego.
