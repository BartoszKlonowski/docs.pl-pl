# <a name="compiler-error-cs1588"></a>Błąd kompilatora CS1588
Nie można określić katalogu środowiska uruchomieniowego języka wspólnego — "Przyczyna"  
  
 Kompilator nie może pobrać katalogu, w którym jest zainstalowany środowisko uruchomieniowe platformy .NET. Ten błąd wskazuje, że środowisko uruchomieniowe języka wspólnego nie jest prawidłowo zainstalowane.
