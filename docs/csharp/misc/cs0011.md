## <a name="example"></a><span data-ttu-id="87b25-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="87b25-107">Example</span></span>

## <a name="example"></a><span data-ttu-id="87b25-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="87b25-108">Example</span></span>  
 <span data-ttu-id="87b25-109">Drugi plik tworzy bibliotekę DLL, która definiuje klasę `C`, która pochodzi od klasy `B`, która została utworzona w poprzednim przykładzie.</span><span class="sxs-lookup"><span data-stu-id="87b25-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="87b25-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="87b25-110">Example</span></span>
