# <a name="compiler-error-cs0010"></a>Błąd kompilatora CS0010
Nieoczekiwany błąd krytyczny--"Error".  
  
 Ten błąd jest generowany, gdy coś całkowicie nieoczekiwanego nie można zatrzymać kompilacji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Skompiluj ponownie inny projekt. Jeśli wystąpi ten sam błąd, spróbuj zainstalować ponownie program Visual Studio. Jeśli zostanie wyświetlony ten błąd, Wyślij raport o błędach do firmy Microsoft.
