# <a name="compiler-error-cs0010"></a>CS0010 błąd kompilatora
Nieoczekiwany błąd krytyczny — "error".  
  
 Ten błąd jest generowany, gdy wystąpi coś zupełnie nieoczekiwane można zatrzymać kompilacji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Skompiluj ponownie innego projektu. Jeśli pojawi się ten sam błąd, spróbuj ponownie zainstalować program Visual Studio. Jeśli ten błąd jest wyświetlany, wysyłania raportów o błędach do firmy Microsoft.
