<span data-ttu-id="a007e-103">Przy użyciu klauzuli musi występować przed wszystkimi innymi elementami zdefiniowanymi w przestrzeni nazw poza deklaracjami aliasów zewnętrznych</span><span class="sxs-lookup"><span data-stu-id="a007e-103">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>  
  
 <span data-ttu-id="a007e-104">A [przy użyciu](../../csharp/language-reference/keywords/using.md) klauzuli musi występować jako pierwszy w przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="a007e-104">A [using](../../csharp/language-reference/keywords/using.md) clause must appear first in a namespace.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a007e-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="a007e-105">Example</span></span>  
 <span data-ttu-id="a007e-106">Poniższy przykład spowoduje wygenerowanie CS1529:</span><span class="sxs-lookup"><span data-stu-id="a007e-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
