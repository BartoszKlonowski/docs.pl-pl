## <a name="example"></a><span data-ttu-id="5d0f4-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="5d0f4-105">Example</span></span>  
 <span data-ttu-id="5d0f4-106">Poniższy przykład generuje CS1529:</span><span class="sxs-lookup"><span data-stu-id="5d0f4-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}

```
