# <a name="compiler-error-cs1529"></a><span data-ttu-id="33966-103">Błąd kompilatora CS1529</span><span class="sxs-lookup"><span data-stu-id="33966-103">Compiler Error CS1529</span></span>
<span data-ttu-id="33966-104">Klauzula using musi poprzedzać wszystkie inne elementy zdefiniowane w przestrzeni nazw poza deklaracjami aliasów zewnętrznych</span><span class="sxs-lookup"><span data-stu-id="33966-104">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>  
  
 <span data-ttu-id="33966-105">Klauzula [using](../language-reference/keywords/using.md) musi występować jako pierwsza w przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="33966-105">A [using](../language-reference/keywords/using.md) clause must appear first in a namespace.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33966-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="33966-106">Example</span></span>  
 <span data-ttu-id="33966-107">Poniższy przykład generuje CS1529:</span><span class="sxs-lookup"><span data-stu-id="33966-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
