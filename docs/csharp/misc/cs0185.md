# <a name="compiler-error-cs0185"></a>CS0185 błąd kompilatora
"type" nie jest typem referencyjnym, wymagane przez instrukcję lock  
  
 [Blokady](../../csharp/language-reference/keywords/lock-statement.md) oświadczenie tylko w stanie ocenić typy odwołań. Aby uzyskać więcej informacji, zobacz [synchronizacji wątków](https://msdn.microsoft.com/library/413e1f28-a2c5-4eec-8338-aa43e7982ff4) i [typów referencyjnych](../../csharp/language-reference/keywords/reference-types.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0185:  
  
```csharp  
// CS0185.cs  
public class MainClass  
{  
    public static void Main ()  
    {  
        lock (1)   // CS0185  
        // try the following lines instead  
        // MainClass x = new MainClass();  
        // lock(x)  
        {  
        }  
    }  
}  
```
