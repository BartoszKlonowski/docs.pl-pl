# <a name="compiler-error-cs1621"></a><span data-ttu-id="37fe1-102">CS1621 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="37fe1-102">Compiler Error CS1621</span></span>
<span data-ttu-id="37fe1-103">Nie można użyć instrukcji yield wewnątrz metody anonimowej lub wyrażenia lambda</span><span class="sxs-lookup"><span data-stu-id="37fe1-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="37fe1-104">Instrukcja yield nie może być w bloku metody anonimowej w iterator.</span><span class="sxs-lookup"><span data-stu-id="37fe1-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="37fe1-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="37fe1-105">Example</span></span>  
 <span data-ttu-id="37fe1-106">Poniższy przykład spowoduje wygenerowanie CS1621:</span><span class="sxs-lookup"><span data-stu-id="37fe1-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
