# <a name="compiler-error-cs1621"></a><span data-ttu-id="3697c-103">Błąd kompilatora CS1621</span><span class="sxs-lookup"><span data-stu-id="3697c-103">Compiler Error CS1621</span></span>
<span data-ttu-id="3697c-104">Nie można użyć instrukcji Yield wewnątrz metody anonimowej lub wyrażenia lambda</span><span class="sxs-lookup"><span data-stu-id="3697c-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="3697c-105">Nie można użyć instrukcji [Yield](../language-reference/keywords/yield.md) w [metodzie anonimowej](../language-reference/operators/delegate-operator.md) ani [wyrażeniu lambda](../language-reference/operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="3697c-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="3697c-106">Poniższy przykład generuje CS1621:</span><span class="sxs-lookup"><span data-stu-id="3697c-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
