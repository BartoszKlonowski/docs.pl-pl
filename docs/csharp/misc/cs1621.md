# <a name="compiler-error-cs1621"></a>Błąd kompilatora CS1621
Nie można użyć instrukcji Yield wewnątrz metody anonimowej lub wyrażenia lambda  
  
Nie można użyć instrukcji [Yield](../../csharp/language-reference/keywords/yield.md) w metodzie [anonimowej](../language-reference/operators/delegate-operator.md) ani [wyrażeniu lambda](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).
  
Poniższy przykład generuje CS1621:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
