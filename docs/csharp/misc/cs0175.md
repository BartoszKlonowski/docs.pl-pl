# <a name="compiler-error-cs0175"></a><span data-ttu-id="9fe2f-103">Błąd kompilatora CS0175</span><span class="sxs-lookup"><span data-stu-id="9fe2f-103">Compiler Error CS0175</span></span>

<span data-ttu-id="9fe2f-104">Użycie słowa kluczowego "Base" jest nieprawidłowe w tym kontekście</span><span class="sxs-lookup"><span data-stu-id="9fe2f-104">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="9fe2f-105">[Podstawowe](../language-reference/keywords/base.md) słowo kluczowe musi być używane do określania określonego elementu członkowskiego klasy podstawowej.</span><span class="sxs-lookup"><span data-stu-id="9fe2f-105">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="9fe2f-106">Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="9fe2f-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="9fe2f-107">Poniższy przykład generuje CS0175:</span><span class="sxs-lookup"><span data-stu-id="9fe2f-107">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
