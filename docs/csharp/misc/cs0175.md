# <a name="compiler-error-cs0175"></a>CS0175 błąd kompilatora
Użycie słowa kluczowego "base" jest nieprawidłowy w tym kontekście  
  
 [Podstawowy](../../csharp/language-reference/keywords/base.md) — słowo kluczowe może służyć do określania określonego elementu członkowskiego klasy bazowej. Aby uzyskać więcej informacji, zobacz [konstruktory](../../csharp/programming-guide/classes-and-structs/constructors.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0175:  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
