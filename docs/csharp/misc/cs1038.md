# <a name="compiler-error-cs1038"></a>CS1038 błąd kompilatora
\#Oczekiwano dyrektywy endregion  
  
 A [#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) dyrektywy nie ma pasującego [#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) dyrektywy.  
  
 Poniższy przykład spowoduje wygenerowanie CS1038:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
