# <a name="compiler-error-cs0582"></a>CS0582 błąd kompilatora
Warunkowy nie jest prawidłowy w składowych interfejsu  
  
 **ConditionalAttribute** nie jest prawidłowy w składowej interfejsu.  
  
 Poniższy przykład spowoduje wygenerowanie CS0582:  
  
```csharp  
// CS0582.cs  
// compile with: /target:library  
using System.Diagnostics;  
interface MyIFace  
{  
   [ConditionalAttribute("DEBUG")]   // CS0582  
   void zz();  
}  
```
