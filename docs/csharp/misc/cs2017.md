# <a name="compiler-error-cs2017"></a>CS2017 błąd kompilatora
Nie można określić opcji/main kompilowania modułu lub biblioteki  
  
 Nie można określić główny punkt wejścia, gdy kompilujesz [/target: Library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS2017:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
