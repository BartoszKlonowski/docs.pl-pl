# <a name="compiler-error-cs2017"></a>Błąd kompilatora CS2017
Nie można określić/Main w przypadku kompilowania modułu lub biblioteki  
  
 Podczas kompilowania [biblioteki/target:](../language-reference/compiler-options/target-library-compiler-option.md)nie można określić głównego punktu wejścia.  
  
 Poniższy przykład generuje CS2017:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
