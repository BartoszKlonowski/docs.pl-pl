# <a name="compiler-error-cs0263"></a>CS0263 błąd kompilatora
Częściowe deklaracje elementu "type" nie może określać różnych klas bazowych  
  
 Podczas definiowania typów w deklaracjach częściowej, należy określić ten sam typ podstawowy, w każdym częściowe deklaracje. Aby uzyskać więcej informacji, zobacz [klasy częściowe i metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0263:  
  
```csharp  
// CS0263.cs  
// compile with: /target:library  
class B1  
{  
}  
  
class B2  
{  
}  
partial class C : B1  // CS0263 – is the base class B1 or B2?  
{  
}  
  
partial class C : B2  
{  
}  
```
