# <a name="compiler-error-cs1527"></a><span data-ttu-id="59723-102">CS1527 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="59723-102">Compiler Error CS1527</span></span>
<span data-ttu-id="59723-103">Elementów definiowanych w przestrzeni nazw nie można jawnie deklarować jako prywatnych, chronionych, chronionych wewnętrznych lub prywatnych chronionych.</span><span class="sxs-lookup"><span data-stu-id="59723-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="59723-104">Deklaracje typów w przestrzeni nazw może mieć [publicznych](../../csharp/language-reference/keywords/public.md) lub [wewnętrzny](../../csharp/language-reference/keywords/internal.md) dostępu.</span><span class="sxs-lookup"><span data-stu-id="59723-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="59723-105">Jeśli nie ułatwień dostępu jest określony, **wewnętrzny** jest ustawieniem domyślnym.</span><span class="sxs-lookup"><span data-stu-id="59723-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="59723-106">Poniższy przykład spowoduje wygenerowanie CS1527:</span><span class="sxs-lookup"><span data-stu-id="59723-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="59723-107">Poniższy przykład generuje CS1527, ponieważ gdy żadnej przestrzeni nazw jest jawnie zadeklarowana w kodzie programu, wszystkie deklaracje typu znajdują się niejawnie w globalnej przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="59723-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
