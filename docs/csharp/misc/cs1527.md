# <a name="compiler-error-cs1527"></a><span data-ttu-id="fdb7e-102">Błąd kompilatora CS1527</span><span class="sxs-lookup"><span data-stu-id="fdb7e-102">Compiler Error CS1527</span></span>

<span data-ttu-id="fdb7e-103">Elementów zdefiniowanych w przestrzeni nazw nie można jawnie deklarować jako prywatnych, chronionych, chronionych wewnętrznie lub prywatnych chronionych.</span><span class="sxs-lookup"><span data-stu-id="fdb7e-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="fdb7e-104">Deklaracje typów w przestrzeni nazw mogą mieć dostęp [publiczny](../language-reference/keywords/public.md) lub [wewnętrzny](../language-reference/keywords/internal.md) .</span><span class="sxs-lookup"><span data-stu-id="fdb7e-104">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="fdb7e-105">Jeśli nie określono dostępności, wartość domyślna to **Internal** .</span><span class="sxs-lookup"><span data-stu-id="fdb7e-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="fdb7e-106">Poniższy przykład generuje CS1527:</span><span class="sxs-lookup"><span data-stu-id="fdb7e-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="fdb7e-107">Poniższy przykład generuje CS1527, ponieważ jeśli żadna przestrzeń nazw nie jest jawnie zadeklarowana w kodzie programu, wszystkie deklaracje typów są zlokalizowane niejawnie w globalnej przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="fdb7e-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
