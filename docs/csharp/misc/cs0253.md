# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="97d81-102">Ostrzeżenie o kompilatorze (poziom 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="97d81-102">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="97d81-103">Możliwe niezamierzone porównanie referencyjne; aby uzyskać porównanie wartości, oddaj prawą stronę, aby wpisać "typ"</span><span class="sxs-lookup"><span data-stu-id="97d81-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="97d81-104">Kompilator wykonuje porównanie odwołania.</span><span class="sxs-lookup"><span data-stu-id="97d81-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="97d81-105">Jeśli chcesz porównać wartość ciągów, rzutuj prawą `type`stronę wyrażenia na .</span><span class="sxs-lookup"><span data-stu-id="97d81-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="97d81-106">Poniższy przykład generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="97d81-106">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
