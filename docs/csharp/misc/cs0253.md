# <a name="compiler-warning-level-2-cs0253"></a>Ostrzeżenie kompilatora (poziom 2) CS0253
Możliwe niezamierzone porównanie odwołań; Aby uzyskać porównanie wartości, Rzutowanie prawej strony na typ "typ"  
  
 Kompilator wykonuje porównanie odwołań. Jeśli chcesz porównać wartość ciągów, Rzutowanie prawej strony wyrażenia na `type` .  
  
 Poniższy przykład generuje CS0253:  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
