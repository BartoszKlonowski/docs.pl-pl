# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="150c7-103">Ostrzeżenie kompilatora (poziom 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="150c7-103">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="150c7-104">Możliwe niezamierzone porównanie odwołań; Aby uzyskać porównanie wartości, Rzutowanie prawej strony na typ "typ"</span><span class="sxs-lookup"><span data-stu-id="150c7-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="150c7-105">Kompilator wykonuje porównanie odwołań.</span><span class="sxs-lookup"><span data-stu-id="150c7-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="150c7-106">Jeśli chcesz porównać wartość ciągów, Rzutowanie prawej strony wyrażenia na `type` .</span><span class="sxs-lookup"><span data-stu-id="150c7-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="150c7-107">Poniższy przykład generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="150c7-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
