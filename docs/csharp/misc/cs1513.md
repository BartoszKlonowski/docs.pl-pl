# <a name="compiler-error-cs1513"></a>Błąd kompilatora CS1513

} oczekiwano  
  
 Kompilator oczekiwał zamykającego nawiasu klamrowego ( `}` ), który nie został odnaleziony.  
  
 Poniższy przykład generuje CS1513:  
  
```csharp  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
