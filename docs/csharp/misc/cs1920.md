# <a name="compiler-error-cs1920"></a><span data-ttu-id="edccd-102">Błąd kompilatora CS1920</span><span class="sxs-lookup"><span data-stu-id="edccd-102">Compiler Error CS1920</span></span>
<span data-ttu-id="edccd-103">Inicjator elementu nie może być pusty.</span><span class="sxs-lookup"><span data-stu-id="edccd-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="edccd-104">Inicjator kolekcji składa się z sekwencji inicjatorów elementów.</span><span class="sxs-lookup"><span data-stu-id="edccd-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="edccd-105">Inicjatory elementów nie muszą być ujęte w nawiasy klamrowe, chyba że zawierają wyrażenie przypisania.</span><span class="sxs-lookup"><span data-stu-id="edccd-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="edccd-106">Jednak w przypadku podawania nawiasów klamrowych nie mogą być puste.</span><span class="sxs-lookup"><span data-stu-id="edccd-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="edccd-107">Jeśli inicjator elementu jest inicjatorem obiektu, nawiasy klamrowe mogą być puste, o ile inicjator zawiera nowe wyrażenie tworzenia obiektu.</span><span class="sxs-lookup"><span data-stu-id="edccd-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="edccd-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="edccd-108">To correct this error</span></span>

- <span data-ttu-id="edccd-110">Jeśli wyrażenie ma być inicjatorem obiektów, Dodaj nowe wyrażenie tworzenia obiektu przed nawiasami klamrowymi.</span><span class="sxs-lookup"><span data-stu-id="edccd-110">If the expression is intended to be an object initializer, add the new object creation expression in front of the braces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="edccd-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="edccd-111">Example</span></span>  
 <span data-ttu-id="edccd-112">Poniższy przykład generuje CS1920:</span><span class="sxs-lookup"><span data-stu-id="edccd-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer   
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized   
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
