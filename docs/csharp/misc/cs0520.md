# <a name="compiler-error-cs0520"></a>Błąd kompilatora CS0520
Wstępnie zdefiniowany typ "name" jest niepoprawnie zadeklarowany  
  
 Nie znaleziono co najmniej jednego pliku wymaganego przez kompilator. Ponownie zainstaluj .NET Framework środowiska uruchomieniowego języka wspólnego.
