# <a name="compiler-error-cs1715"></a><span data-ttu-id="1f38d-102">Błąd kompilatora CS1715</span><span class="sxs-lookup"><span data-stu-id="1f38d-102">Compiler Error CS1715</span></span>
<span data-ttu-id="1f38d-103">"Type1": typ musi być "type2", aby pasować do przesłoniętego elementu członkowskiego "MemberName"</span><span class="sxs-lookup"><span data-stu-id="1f38d-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="1f38d-104">Ten błąd jest taki sam jak [błąd kompilatora CS0508](./cs0508.md), z wyjątkiem tego, że CS0508 jest teraz stosowana tylko do metod, które mają typy zwracane, natomiast CS1715 ma zastosowanie do właściwości i indeksatorów, które mają tylko typ "Types", a nie "typy zwracane".</span><span class="sxs-lookup"><span data-stu-id="1f38d-104">This error is the same as [Compiler Error CS0508](./cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1f38d-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="1f38d-105">Example</span></span>  
 <span data-ttu-id="1f38d-106">Poniższy kod generuje CS1715.</span><span class="sxs-lookup"><span data-stu-id="1f38d-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
