# <a name="compiler-error-cs1715"></a><span data-ttu-id="03e36-102">CS1715 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="03e36-102">Compiler Error CS1715</span></span>
<span data-ttu-id="03e36-103">"Type1": typ musi być "Type2" odpowiadające zastąpionym elementem członkowskim "MemberName"</span><span class="sxs-lookup"><span data-stu-id="03e36-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="03e36-104">Ten błąd jest taka sama jak [CS0508 błąd kompilatora](../../csharp/misc/cs0508.md), ale CS0508 teraz ma zastosowanie tylko do metod, które mają zwracanych typów, gdy CS1715 ma zastosowanie do właściwości i indeksatorów, w których tylko "typy" zamiast "zwracane typy".</span><span class="sxs-lookup"><span data-stu-id="03e36-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03e36-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="03e36-105">Example</span></span>  
 <span data-ttu-id="03e36-106">Poniższy kod generuje CS1715.</span><span class="sxs-lookup"><span data-stu-id="03e36-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
