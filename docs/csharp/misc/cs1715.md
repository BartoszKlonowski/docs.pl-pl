# <a name="compiler-error-cs1715"></a><span data-ttu-id="583d2-102">CS1715 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="583d2-102">Compiler Error CS1715</span></span>
<span data-ttu-id="583d2-103">'Typ1': typ musi być 'Typ2' do dopasowania zgodnym z przesłoniętą składową "MemberName"</span><span class="sxs-lookup"><span data-stu-id="583d2-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="583d2-104">Ten błąd jest taka sama jak [CS0508 błąd kompilatora](../../csharp/misc/cs0508.md), chyba że CS0508 teraz ma zastosowanie tylko do metody, które mają typy zwracane, gdy CS1715 ma zastosowanie do właściwości i indeksatorów, którzy mają tylko "typy" zamiast "typy zwracane".</span><span class="sxs-lookup"><span data-stu-id="583d2-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="583d2-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="583d2-105">Example</span></span>  
 <span data-ttu-id="583d2-106">Poniższy kod generuje CS1715.</span><span class="sxs-lookup"><span data-stu-id="583d2-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
