# <a name="compiler-error-cs0159"></a><span data-ttu-id="5000a-102">Błąd kompilatora CS0159</span><span class="sxs-lookup"><span data-stu-id="5000a-102">Compiler Error CS0159</span></span>
<span data-ttu-id="5000a-103">Brak etykiety "etykieta" w zakresie instrukcji goto</span><span class="sxs-lookup"><span data-stu-id="5000a-103">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="5000a-104">Etykieta, do której odwołuje się instrukcja [goto](../language-reference/keywords/goto.md) , nie została znaleziona w zakresie `goto` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="5000a-104">The label referenced by the [goto](../language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="5000a-105">Poniższy przykład generuje CS0159:</span><span class="sxs-lookup"><span data-stu-id="5000a-105">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
