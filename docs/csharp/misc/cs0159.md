# <a name="compiler-error-cs0159"></a><span data-ttu-id="c56a3-102">CS0159 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="c56a3-102">Compiler Error CS0159</span></span>
<span data-ttu-id="c56a3-103">Brak etykiety "etykieta" w zakresie instrukcji goto</span><span class="sxs-lookup"><span data-stu-id="c56a3-103">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="c56a3-104">Etykieta odwołuje się [goto](../../csharp/language-reference/keywords/goto.md) nie można odnaleźć instrukcji w zakresie `goto` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="c56a3-104">The label referenced by the [goto](../../csharp/language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="c56a3-105">Poniższy przykład generuje CS0159:</span><span class="sxs-lookup"><span data-stu-id="c56a3-105">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
