Explicit)</span><span class="sxs-lookup"><span data-stu-id="d9e80-104">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="d9e80-105">Należy użyć atrybutu **StructLayout (LayoutKind. Explicit)** w deklaracji struktury, jeśli zawiera ona elementy członkowskie oznaczone atrybutem **FieldOffset** .</span><span class="sxs-lookup"><span data-stu-id="d9e80-105">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="d9e80-106">Aby uzyskać więcej informacji, zobacz [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="d9e80-106">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="d9e80-107">Poniższy przykład generuje CS0636:</span><span class="sxs-lookup"><span data-stu-id="d9e80-107">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
