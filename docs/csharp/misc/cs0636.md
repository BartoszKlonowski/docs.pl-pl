# <a name="compiler-error-cs0636"></a><span data-ttu-id="be9c8-102">CS0636 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="be9c8-102">Compiler Error CS0636</span></span>
<span data-ttu-id="be9c8-103">Atrybut FieldOffset można umieścić wyłącznie w elementach członkowskich o typie oznaczonym przy użyciu atrybutu StructLayout(LayoutKind.Explicit).</span><span class="sxs-lookup"><span data-stu-id="be9c8-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="be9c8-104">Należy użyć **użyciu klasy StructLayout(LayoutKind.Explicit)** atrybutu w deklaracji struktury, jeśli zawiera ona żadnych elementów członkowskich oznaczonych **FieldOffset** atrybutu.</span><span class="sxs-lookup"><span data-stu-id="be9c8-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="be9c8-105">Aby uzyskać więcej informacji, zobacz [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="be9c8-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="be9c8-106">Poniższy przykład generuje CS0636:</span><span class="sxs-lookup"><span data-stu-id="be9c8-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
