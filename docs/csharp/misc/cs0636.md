# <a name="compiler-error-cs0636"></a><span data-ttu-id="16b87-102">CS0636 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="16b87-102">Compiler Error CS0636</span></span>
<span data-ttu-id="16b87-103">Atrybut FieldOffset tylko można umieścić na elementach członkowskich w typach oznaczone elementem StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="16b87-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="16b87-104">Należy użyć **StructLayout(LayoutKind.Explicit)** atrybutu w deklaracji struktury, jeśli zawiera on żadnych elementów członkowskich oznaczone **FieldOffset** atrybutu.</span><span class="sxs-lookup"><span data-stu-id="16b87-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="16b87-105">Aby uzyskać więcej informacji, zobacz [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="16b87-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="16b87-106">Poniższy przykład spowoduje wygenerowanie CS0636:</span><span class="sxs-lookup"><span data-stu-id="16b87-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
