# <a name="compiler-error-cs0690"></a>CS0690 błąd kompilatora
Plik wejściowy 'Plik' zawiera nieprawidłowe metadane.  
  
 Można otworzyć pliku metadanych, ale z powodu problemu z niektórych lokalizacji metadanych jest uszkodzony. Jest to podobne do błędu [CS0009](../../csharp/misc/cs0009.md), z tą różnicą, że jesteś w stanie otworzyć pliku metadanych.  
  
 PE (proces pliku wykonywalnego) jest plikiem wykonywalnym.
