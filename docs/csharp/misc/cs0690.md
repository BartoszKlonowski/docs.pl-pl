# <a name="compiler-error-cs0690"></a>Błąd kompilatora CS0690
Plik wejściowy "File" zawiera nieprawidłowe metadane.  
  
 Możesz otworzyć plik metadanych, ale z powodu problemu z lokalizacją, metadane uległy uszkodzeniu. Jest to podobne do błędu [CS0009](./cs0009.md), z tą różnicą, że można otworzyć plik metadanych.  
  
 PE (proces plików wykonywalnych) jest plikiem wykonywalnym.
