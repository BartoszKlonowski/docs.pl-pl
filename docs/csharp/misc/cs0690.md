# <a name="compiler-error-cs0690"></a>CS0690 błąd kompilatora
Plik wejściowy 'Plik' zawiera nieprawidłowe metadane.  
  
 Można otworzyć pliku metadanych, ale z powodu jakiś problem lokalizacja metadanych był uszkodzony. Jest to podobne do błąd [CS0009](../../csharp/misc/cs0009.md), z wyjątkiem tego, że jesteś w stanie otworzyć pliku metadanych.  
  
 PE (proces wykonywalny) jest plikiem wykonywalnym.
