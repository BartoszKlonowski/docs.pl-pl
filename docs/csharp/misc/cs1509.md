# <a name="compiler-error-cs1509"></a>CS1509 błąd kompilatora
Odwołania pliku 'Plik' nie jest zestawem; Użyj "/ addmodule" zamiast opcji  
  
 Plik wyjściowy (plik wyjściowy 1), produkowanych w kompilacji, która jest używana [/target: module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) (nie ma w manifeście zestawu), został określony w [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md). Dlatego zamiast dołączać zestawu do zestawu dla bieżącego programu, informacje o metadanych w pliku danych wyjściowych 1 zostanie dodany do zestawu dla bieżącego programu.
