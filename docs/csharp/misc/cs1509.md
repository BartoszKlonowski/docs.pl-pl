# <a name="compiler-error-cs1509"></a>Błąd kompilatora CS1509
Plik "File", do którego istnieje odwołanie, nie jest zestawem; Zamiast tego użyj opcji "/addmodule"  
  
 Plik wyjściowy (plik wyjściowy 1) utworzony w kompilacji, która użyła opcji [/target: module](../language-reference/compiler-options/target-module-compiler-option.md) (nie ma manifestu zestawu), został określony jako [/Reference](../language-reference/compiler-options/reference-compiler-option.md). Dlatego zamiast dołączania zestawu do zestawu bieżącego programu informacje o metadanych w pliku wyjściowym 1 zostaną dodane do zestawu dla bieżącego programu.
