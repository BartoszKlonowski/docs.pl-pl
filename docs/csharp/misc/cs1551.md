# <a name="compiler-error-cs1551"></a>CS1551 błąd kompilatora
Indeksatory musi mieć co najmniej jeden parametr  
  
 [Indeksatora](../../csharp/programming-guide/indexers/index.md) , przyjmuje żadnych argumentów został zadeklarowany.  
  
 Poniższy przykład spowoduje wygenerowanie CS1551:  
  
```csharp  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
