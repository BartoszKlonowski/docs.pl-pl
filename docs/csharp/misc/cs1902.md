# <a name="compiler-error-cs1902"></a>Błąd kompilatora CS1902
Nieprawidłowa opcja "Option" dla przełącznika/Debug; musi być pełny lub pdbonly  
  
 Nieprawidłowa opcja została przeniesiona do opcji kompilatora [/Debug](../language-reference/compiler-options/debug-compiler-option.md) .  
  
 Poniższy przykład generuje CS1902:  
  
```csharp  
// CS1902.cs  
// compile with: /debug:x  
// CS1902 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
