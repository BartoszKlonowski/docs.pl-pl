## <a name="example"></a><span data-ttu-id="d2045-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="d2045-106">Example</span></span>  
 <span data-ttu-id="d2045-107">Poniższy przykład generuje CS1642.</span><span class="sxs-lookup"><span data-stu-id="d2045-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs

unsafe class C  
{  
   fixed int a[10];   // CS1642

{  
    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```
