## <a name="example"></a><span data-ttu-id="d495a-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d495a-107">Example</span></span>  

 <span data-ttu-id="d495a-108">Poniższy przykład generuje CS1642.</span><span class="sxs-lookup"><span data-stu-id="d495a-108">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
