# <a name="compiler-error-cs0249"></a>CS0249 błąd kompilatora
Nie zastępuje obiekt. Finalize. Zamiast tego Udostępnij destruktor.  
  
 Użyj składni destruktora, aby określić instrukcji do wykonania, kiedy niszczony jest obiekt.  

 Poniższy przykład spowoduje wygenerowanie CS0249:  
  
```csharp  
// CS0249.cs  
class MyClass  
{  
   protected override void Finalize()   // CS0249  
   // try the following line instead  
   // ~MyClass()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
