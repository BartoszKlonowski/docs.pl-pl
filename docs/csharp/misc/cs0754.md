## <a name="example"></a><span data-ttu-id="46df7-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="46df7-108">Example</span></span>  
 <span data-ttu-id="46df7-109">Poniższy kod generuje CS0754:</span><span class="sxs-lookup"><span data-stu-id="46df7-109">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
