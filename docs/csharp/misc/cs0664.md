<span data-ttu-id="c17d2-103">Nie można niejawnie przekonwertować literału typu double na typ 'type'; Użyj sufiksu "sufiks" w celu utworzenia literału tego typu</span><span class="sxs-lookup"><span data-stu-id="c17d2-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="c17d2-104">Nie można wykonać przypisania; Użyj sufiksu, aby naprawić tę instrukcję.</span><span class="sxs-lookup"><span data-stu-id="c17d2-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="c17d2-105">Dokumentacja dla każdego typu identyfikuje sufiksem odpowiedniego dla typu.</span><span class="sxs-lookup"><span data-stu-id="c17d2-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="c17d2-106">Aby uzyskać więcej informacji o konwersji, zobacz [rzutowanie i konwersje typów](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="c17d2-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="c17d2-107">Poniższy przykład generuje CS0664:</span><span class="sxs-lookup"><span data-stu-id="c17d2-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
