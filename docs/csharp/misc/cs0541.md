# <a name="compiler-error-cs0541"></a><span data-ttu-id="30e16-102">Błąd kompilatora CS0541</span><span class="sxs-lookup"><span data-stu-id="30e16-102">Compiler Error CS0541</span></span>
<span data-ttu-id="30e16-103">"Deklaracja": jawna deklaracja interfejsu może być deklarowana tylko w klasie lub strukturze</span><span class="sxs-lookup"><span data-stu-id="30e16-103">'declaration' : explicit interface declaration can only be declared in a class or struct</span></span>  
  
 <span data-ttu-id="30e16-104">Znaleziono jawną deklarację [interfejsu](../language-reference/keywords/interface.md) poza [klasą](../language-reference/keywords/class.md) lub [strukturą](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="30e16-104">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="30e16-105">Poniższy przykład generuje CS0541:</span><span class="sxs-lookup"><span data-stu-id="30e16-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
