# <a name="compiler-error-cs0541"></a>CS0541 błąd kompilatora
"deklaracją": jawnej deklaracji interfejsu mogą być deklarowane tylko w klasie lub strukturze  
  
 Jawnie [interfejsu](../../csharp/language-reference/keywords/interface.md) deklaracji znaleziono poza [klasy](../../csharp/language-reference/keywords/class.md) lub [struktury](../../csharp/language-reference/keywords/struct.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0541:  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
