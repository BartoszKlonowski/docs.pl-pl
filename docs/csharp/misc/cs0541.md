# <a name="compiler-error-cs0541"></a>Błąd kompilatora CS0541
"Deklaracja": jawna deklaracja interfejsu może być deklarowana tylko w klasie lub strukturze  
  
 Znaleziono jawną deklarację [interfejsu](../language-reference/keywords/interface.md) poza [klasą](../language-reference/keywords/class.md) lub [strukturą](../language-reference/keywords/struct.md).  
  
 Poniższy przykład generuje CS0541:  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
