# <a name="compiler-error-cs0470"></a>Błąd kompilatora CS0470
Metoda "Method" nie może implementować metody dostępu interfejsu "akcesor" dla typu "Type". Użyj jawnej implementacji interfejsu.  
  
 Ten błąd jest generowany, gdy metoda dostępu próbuje zaimplementować interfejs. Należy użyć jawnej implementacji interfejsu.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0470.  
  
```csharp  
// CS0470.cs  
// compile with: /target:library  
  
interface I  
{  
   int P { get; }  
}  
  
class MyClass : I  
{  
   public int get_P() { return 0; }   // CS0470  
   public int P2 { get { return 0;} }   // OK  
}  
```
