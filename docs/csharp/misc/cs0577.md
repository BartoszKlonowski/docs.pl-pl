# <a name="compiler-error-cs0577"></a><span data-ttu-id="84be9-103">Błąd kompilatora CS0577</span><span class="sxs-lookup"><span data-stu-id="84be9-103">Compiler Error CS0577</span></span>

<span data-ttu-id="84be9-104">Atrybut Conditional jest nieprawidłowy w funkcji "Function", ponieważ jest to Konstruktor, destruktor, operator lub Jawna implementacja interfejsu</span><span class="sxs-lookup"><span data-stu-id="84be9-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="84be9-105">`Conditional` nie można zastosować do określonych metod.</span><span class="sxs-lookup"><span data-stu-id="84be9-105">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="84be9-106">Na przykład nie można użyć niektórych atrybutów w jawnej definicji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="84be9-106">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="84be9-107">Poniższy przykład generuje CS0577:</span><span class="sxs-lookup"><span data-stu-id="84be9-107">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
