<span data-ttu-id="930ae-104">"Konstruktor": Konstruktor statyczny nie może mieć parametrów</span><span class="sxs-lookup"><span data-stu-id="930ae-104">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="930ae-105">Nie można zadeklarować konstruktora [statycznego](../language-reference/keywords/static.md) z co najmniej jednym parametrem.</span><span class="sxs-lookup"><span data-stu-id="930ae-105">A [static](../language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="930ae-106">Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="930ae-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="930ae-107">Poniższy przykład generuje CS0132:</span><span class="sxs-lookup"><span data-stu-id="930ae-107">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
