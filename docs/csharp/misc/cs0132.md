# <a name="compiler-error-cs0132"></a>Błąd kompilatora CS0132
"Konstruktor": Konstruktor statyczny nie może mieć parametrów  
  
 Nie można zadeklarować konstruktora [statycznego](../language-reference/keywords/static.md) z co najmniej jednym parametrem. Aby uzyskać więcej informacji, zobacz [konstruktory](../programming-guide/classes-and-structs/constructors.md).  
  
 Poniższy przykład generuje CS0132:  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
