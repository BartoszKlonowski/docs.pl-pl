## <a name="example"></a><span data-ttu-id="1a916-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="1a916-105">Example</span></span>  
 <span data-ttu-id="1a916-106">Poniższy przykład spowoduje wygenerowanie CS0730.</span><span class="sxs-lookup"><span data-stu-id="1a916-106">The following sample generates CS0730.</span></span> <span data-ttu-id="1a916-107">Składa się z dwóch plików źródłowych.</span><span class="sxs-lookup"><span data-stu-id="1a916-107">It consists of two source files.</span></span> <span data-ttu-id="1a916-108">Najpierw skompiluj plik biblioteki `CS0730a.cs`i kompilacji pliku `CS0730.cs` odwołuje się do pliku biblioteki.</span><span class="sxs-lookup"><span data-stu-id="1a916-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
