 <span data-ttu-id="685a8-107">Poniższy przykład generuje CS0730.</span><span class="sxs-lookup"><span data-stu-id="685a8-107">The following sample generates CS0730.</span></span> <span data-ttu-id="685a8-108">Składa się z dwóch plików źródłowych.</span><span class="sxs-lookup"><span data-stu-id="685a8-108">It consists of two source files.</span></span> <span data-ttu-id="685a8-109">Najpierw Skompiluj plik biblioteki `CS0730a.cs` i skompiluj plik `CS0730.cs` odwołujący się do pliku biblioteki.</span><span class="sxs-lookup"><span data-stu-id="685a8-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
