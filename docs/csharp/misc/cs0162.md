# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="53a79-102">CS0162 ostrzeżenie (poziom 2) kompilator</span><span class="sxs-lookup"><span data-stu-id="53a79-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="53a79-103">Wykryto nieosiągalny kod</span><span class="sxs-lookup"><span data-stu-id="53a79-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="53a79-104">Kompilator wykrył kod, który nigdy nie zostanie wykonana.</span><span class="sxs-lookup"><span data-stu-id="53a79-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="53a79-105">Poniższy przykład spowoduje wygenerowanie CS0162:</span><span class="sxs-lookup"><span data-stu-id="53a79-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
