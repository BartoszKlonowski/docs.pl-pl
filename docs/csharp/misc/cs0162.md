# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="8000e-102">Kompilator CS0162 ostrzegawcze (poziom 2)</span><span class="sxs-lookup"><span data-stu-id="8000e-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="8000e-103">Wykryto nieosiągalny kod</span><span class="sxs-lookup"><span data-stu-id="8000e-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="8000e-104">Kompilator wykryto kod, który nigdy nie zostanie wykonana.</span><span class="sxs-lookup"><span data-stu-id="8000e-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="8000e-105">Poniższy przykład generuje CS0162:</span><span class="sxs-lookup"><span data-stu-id="8000e-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
