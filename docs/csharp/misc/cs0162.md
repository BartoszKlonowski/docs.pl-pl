# <a name="compiler-warning-level-2-cs0162"></a>CS0162 ostrzeżenie (poziom 2) kompilator
Wykryto nieosiągalny kod  
  
 Kompilator wykrył kod, który nigdy nie zostanie wykonana.  
  
 Poniższy przykład spowoduje wygenerowanie CS0162:  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
