## <a name="example"></a><span data-ttu-id="2b8bb-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="2b8bb-106">Example</span></span>

```csharp
// CS0162.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        goto lab1;
        {
            // The following statements cannot be reached:
            int i = 9;   // CS0162
            i++;
        }
    lab1:
        {
        }
    }
}
```
