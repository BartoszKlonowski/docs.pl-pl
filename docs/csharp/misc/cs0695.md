# <a name="compiler-error-cs0695"></a><span data-ttu-id="e487a-103">Błąd kompilatora CS0695</span><span class="sxs-lookup"><span data-stu-id="e487a-103">Compiler Error CS0695</span></span>
<span data-ttu-id="e487a-104">element "typ ogólny" nie może implementować obu metod "Generic Interface" i "Generic Interface", ponieważ mogą one mieć charakter w przypadku niektórych podstawiania parametrów typu</span><span class="sxs-lookup"><span data-stu-id="e487a-104">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="e487a-105">Ten błąd występuje, gdy Klasa ogólna implementuje więcej niż jeden parametryzacja tego samego interfejsu ogólnego i istnieje podstawianie parametrów typu, które spowodują, że te dwa interfejsy są identyczne.</span><span class="sxs-lookup"><span data-stu-id="e487a-105">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="e487a-106">Aby uniknąć tego błędu, zaimplementuj tylko jeden z interfejsów lub Zmień parametry typu, aby uniknąć konfliktu.</span><span class="sxs-lookup"><span data-stu-id="e487a-106">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="e487a-107">Poniższy przykład generuje CS0695:</span><span class="sxs-lookup"><span data-stu-id="e487a-107">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
