# <a name="compiler-error-cs1535"></a>CS1535 błąd kompilatora
Przeciążony operator jednoargumentowy "operator" przyjmuje jeden parametr  
  
 Definicja jednoargumentowy [operator](../../csharp/language-reference/operators/operator-overloading.md) musi mieć jeden parametr.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1535:  
  
```csharp  
// CS1535.cs  
class MyClass  
{  
    // uncomment the method parameter to resolve CS1535  
    public static MyClass operator ++ (/*MyClass MC1*/)   // CS1535  
    {  
        return new MyClass();  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
