# <a name="compiler-error-cs0534"></a><span data-ttu-id="e6080-102">CS0534 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e6080-102">Compiler Error CS0534</span></span>
<span data-ttu-id="e6080-103">"function1" nie implementuje odziedziczonego abstrakcyjnego elementu członkowskiego "function2"</span><span class="sxs-lookup"><span data-stu-id="e6080-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="e6080-104">Klasa jest wymagane do wdrożenia całości [abstrakcyjny](../../csharp/language-reference/keywords/abstract.md) elementów członkowskich w klasie podstawowej, chyba że klasa jest również jako abstract.</span><span class="sxs-lookup"><span data-stu-id="e6080-104">A class is required to implement all the [abstract](../../csharp/language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="e6080-105">Poniższy przykład generuje CS0534:</span><span class="sxs-lookup"><span data-stu-id="e6080-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
