# <a name="compiler-error-cs0534"></a><span data-ttu-id="4167a-102">Błąd kompilatora CS0534</span><span class="sxs-lookup"><span data-stu-id="4167a-102">Compiler Error CS0534</span></span>
<span data-ttu-id="4167a-103">element "function1" nie implementuje dziedziczonej abstrakcyjnej składowej "function2"</span><span class="sxs-lookup"><span data-stu-id="4167a-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="4167a-104">Klasa jest wymagana do zaimplementowania wszystkich [abstrakcyjnych](../language-reference/keywords/abstract.md) elementów członkowskich w klasie bazowej, chyba że Klasa jest również abstrakcyjna.</span><span class="sxs-lookup"><span data-stu-id="4167a-104">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="4167a-105">Poniższy przykład generuje CS0534:</span><span class="sxs-lookup"><span data-stu-id="4167a-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
