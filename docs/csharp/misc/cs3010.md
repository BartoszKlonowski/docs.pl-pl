# <a name="compiler-warning-level-1-cs3010"></a>Kompilatora (poziom 1) ostrzeżenie CS3010
'składowa': Interfejsy zgodne ze specyfikacją CLS muszą mieć tylko składowe zgodne ze specyfikacją CLS  
  
 W zestawie oznaczone `[assembly:CLCSompliant(true)]`, interfejs zawiera członka oznaczone `[CLCSompliant(false)]`. Usuń jeden z atrybutów zgodności Common Language Specification (CLS). Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3010:  
  
```csharp  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}  
  
public class C : I  
{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
