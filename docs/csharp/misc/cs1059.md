## <a name="example"></a><span data-ttu-id="ea7e3-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="ea7e3-110">Example</span></span>  
 <span data-ttu-id="ea7e3-111">Poniższy przykład generuje CS1059, ponieważ `i` jest stałą, a nie zmienną, i `E` jest `Enum` typem, którego elementy są również wartościami stałymi.</span><span class="sxs-lookup"><span data-stu-id="ea7e3-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
