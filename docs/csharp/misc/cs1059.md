## <a name="example"></a><span data-ttu-id="b1b06-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="b1b06-111">Example</span></span>  

 <span data-ttu-id="b1b06-112">Poniższy przykład generuje CS1059, ponieważ `i` jest stałą, a nie zmienną, i `E` jest `Enum` typem, którego elementy są również wartościami stałymi.</span><span class="sxs-lookup"><span data-stu-id="b1b06-112">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
