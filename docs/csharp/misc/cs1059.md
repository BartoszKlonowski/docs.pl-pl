## <a name="example"></a><span data-ttu-id="9a544-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="9a544-110">Example</span></span>  
 <span data-ttu-id="9a544-111">Poniższy przykład generuje CS1059, ponieważ `i` jest stałą, nie w zmiennej i `E` jest `Enum` typu, w której elementy są również wartości stałych.</span><span class="sxs-lookup"><span data-stu-id="9a544-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
