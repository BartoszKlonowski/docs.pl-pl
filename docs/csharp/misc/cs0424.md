# <a name="compiler-error-cs0424"></a>CS0424 błąd kompilatora
"class": klasa z atrybutem ComImport nie może określać klasy bazowej  
  
 Określanie <xref:System.Runtime.InteropServices.ComImportAttribute> atrybut wskazuje na to, że implementacja klasy jest mają być importowane z modułu COM. Dodatkowe metody lub pola dziedziczone z klasy bazowej nie są dozwolone do dodania do implementacji zdefiniowanego w COM module.  
  
 Poniższy przykład spowoduje wygenerowanie CS0424:  
  
```csharp  
// CS0424.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
public class A {}  
  
[ ComImport, Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829") ]  
class B : A {}   // CS0424 error  
```
