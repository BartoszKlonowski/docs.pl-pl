# <a name="compiler-error-cs0724"></a><span data-ttu-id="7fb13-103">Błąd kompilatora CS0724</span><span class="sxs-lookup"><span data-stu-id="7fb13-103">Compiler Error CS0724</span></span>

<span data-ttu-id="7fb13-104">Instrukcja throw bez argumentów jest niedozwolona w klauzuli finally, która jest zagnieżdżona w najbliższej otaczającej klauzuli catch</span><span class="sxs-lookup"><span data-stu-id="7fb13-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>

## <a name="example"></a><span data-ttu-id="7fb13-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="7fb13-105">Example</span></span>

<span data-ttu-id="7fb13-106">Poniższy przykład generuje CS0724 z powodu `throw` instrukcji wewnątrz `finally` bloku klauzuli:</span><span class="sxs-lookup"><span data-stu-id="7fb13-106">The following example generates CS0724 because of the `throw` statement inside the `finally` clause block:</span></span>

```csharp  
// CS0724.cs  
using System;  
  
class Program
{
    static void Test()
    {
        try
        {
            throw new Exception();
        }
        catch
        {
            try
            {
            }
            finally
            {
                throw; // CS0724
            }
        }
    }

    static void Main()
    {
    }
}
```
