# <a name="compiler-error-cs1623"></a><span data-ttu-id="c2ccd-102">CS1623 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="c2ccd-102">Compiler Error CS1623</span></span>
<span data-ttu-id="c2ccd-103">Iteratory nie mogą mieć parametrów out lub ref</span><span class="sxs-lookup"><span data-stu-id="c2ccd-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="c2ccd-104">Ten błąd występuje, jeśli metoda iteratora jest przyjmuje `in`, `ref`, lub `out` parametru.</span><span class="sxs-lookup"><span data-stu-id="c2ccd-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="c2ccd-105">Aby uniknąć tego błędu, Usuń `in`, `ref`, lub `out` słowa kluczowego w podpisie metody.</span><span class="sxs-lookup"><span data-stu-id="c2ccd-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c2ccd-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="c2ccd-106">Example</span></span>  
 <span data-ttu-id="c2ccd-107">Poniższy przykład spowoduje wygenerowanie CS1623:</span><span class="sxs-lookup"><span data-stu-id="c2ccd-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
