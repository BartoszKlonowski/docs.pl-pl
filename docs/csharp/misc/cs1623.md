# <a name="compiler-error-cs1623"></a>Błąd kompilatora CS1623
Iteratory nie mogą mieć parametrów ref ani out  
  
 Ten błąd występuje, gdy metoda iteratora przyjmuje `in` `ref` parametr,, lub `out` . Aby uniknąć tego błędu, Usuń `in` `ref` słowo kluczowe,, lub `out` z sygnatury metody.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1623:  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
