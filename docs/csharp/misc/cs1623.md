# <a name="compiler-error-cs1623"></a><span data-ttu-id="7f0eb-103">Błąd kompilatora CS1623</span><span class="sxs-lookup"><span data-stu-id="7f0eb-103">Compiler Error CS1623</span></span>

<span data-ttu-id="7f0eb-104">Iteratory nie mogą mieć parametrów ref ani out</span><span class="sxs-lookup"><span data-stu-id="7f0eb-104">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="7f0eb-105">Ten błąd występuje, gdy metoda iteratora przyjmuje `in` `ref` parametr,, lub `out` .</span><span class="sxs-lookup"><span data-stu-id="7f0eb-105">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="7f0eb-106">Aby uniknąć tego błędu, Usuń `in` `ref` słowo kluczowe,, lub `out` z sygnatury metody.</span><span class="sxs-lookup"><span data-stu-id="7f0eb-106">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7f0eb-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="7f0eb-107">Example</span></span>  

 <span data-ttu-id="7f0eb-108">Poniższy przykład generuje CS1623:</span><span class="sxs-lookup"><span data-stu-id="7f0eb-108">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
