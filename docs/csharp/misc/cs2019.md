# <a name="compiler-error-cs2019"></a>CS2019 błąd kompilatora
Nieprawidłowy typ docelowy dla opcji/target: należy określić "exe", "winexe", "library" lub "module"  
  
 [/Target](../../csharp/language-reference/compiler-options/target-compiler-option.md) użyto opcji kompilatora, ale przekazano nieprawidłowy parametr. Aby rozwiązać ten problem, należy ponownie skompilować program za pomocą formularza z **/target** opcja, która jest odpowiednia do pliku wyjściowego.  
  
 Poniższy przykład spowoduje wygenerowanie CS2017:  
  
```csharp  
// CS2019.cs  
// compile with: /target:libra  
// CS2019 expected  
class MyClass  
{  
}  
```
