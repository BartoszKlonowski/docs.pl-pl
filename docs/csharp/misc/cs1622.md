 <span data-ttu-id="2bcc6-107">Poniższy przykład generuje CS1622:</span><span class="sxs-lookup"><span data-stu-id="2bcc6-107">The following sample generates CS1622:</span></span>  
  
```csharp

class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()

      return (IEnumerator) this;  // CS1622

   }  
}
```
