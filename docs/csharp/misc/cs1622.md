 <span data-ttu-id="c3cea-108">Poniższy przykład generuje CS1622:</span><span class="sxs-lookup"><span data-stu-id="c3cea-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
