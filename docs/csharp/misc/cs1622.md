# <a name="compiler-error-cs1622"></a><span data-ttu-id="0057e-103">Błąd kompilatora CS1622</span><span class="sxs-lookup"><span data-stu-id="0057e-103">Compiler Error CS1622</span></span>

<span data-ttu-id="0057e-104">Nie można zwrócić wartości z iteratora.</span><span class="sxs-lookup"><span data-stu-id="0057e-104">Cannot return a value from an iterator.</span></span> <span data-ttu-id="0057e-105">Użyj instrukcji yield return, aby zwrócić wartość lub uzyskać przerwanie, aby zakończyć iterację.</span><span class="sxs-lookup"><span data-stu-id="0057e-105">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="0057e-106">Iterator jest specjalną funkcją, która zwraca wartość za pośrednictwem instrukcji Yield, a nie instrukcji return.</span><span class="sxs-lookup"><span data-stu-id="0057e-106">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="0057e-107">Aby uzyskać więcej informacji, zobacz **Iteratory**.</span><span class="sxs-lookup"><span data-stu-id="0057e-107">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="0057e-108">Poniższy przykład generuje CS1622:</span><span class="sxs-lookup"><span data-stu-id="0057e-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
