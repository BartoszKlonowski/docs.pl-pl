# <a name="compiler-error-cs1676"></a><span data-ttu-id="cdb2f-102">CS1676 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="cdb2f-102">Compiler Error CS1676</span></span>
<span data-ttu-id="cdb2f-103">Parametr 'Liczba' musi być zadeklarowany ze słowem kluczowym "— słowo kluczowe"</span><span class="sxs-lookup"><span data-stu-id="cdb2f-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="cdb2f-104">Ten błąd występuje, gdy modyfikator typu parametru w metodę anonimową różni się od używanej w deklaracji delegata, który jest rzutowanie metody do.</span><span class="sxs-lookup"><span data-stu-id="cdb2f-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="cdb2f-105">Poniższy przykład spowoduje wygenerowanie CS1676:</span><span class="sxs-lookup"><span data-stu-id="cdb2f-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
