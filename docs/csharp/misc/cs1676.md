# <a name="compiler-error-cs1676"></a><span data-ttu-id="e25f5-102">CS1676 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e25f5-102">Compiler Error CS1676</span></span>
<span data-ttu-id="e25f5-103">Parametr 'Liczba' musi być zadeklarowany ze słowem kluczowym "— słowo kluczowe"</span><span class="sxs-lookup"><span data-stu-id="e25f5-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="e25f5-104">Ten błąd występuje, gdy parametr modyfikator typu w metodzie anonimowej różni się od używanego w deklaracji obiektu delegowanego, które są rzutowanie metody do.</span><span class="sxs-lookup"><span data-stu-id="e25f5-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="e25f5-105">Poniższy przykład generuje CS1676:</span><span class="sxs-lookup"><span data-stu-id="e25f5-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
