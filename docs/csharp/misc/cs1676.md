# <a name="compiler-error-cs1676"></a>CS1676 błąd kompilatora
Parametr 'Liczba' musi być zadeklarowany ze słowem kluczowym "— słowo kluczowe"  
  
 Ten błąd występuje, gdy modyfikator typu parametru w metodę anonimową różni się od używanej w deklaracji delegata, który jest rzutowanie metody do.  
  
 Poniższy przykład spowoduje wygenerowanie CS1676:  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
