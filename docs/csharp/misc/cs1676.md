# <a name="compiler-error-cs1676"></a><span data-ttu-id="a5b88-102">Błąd kompilatora CS1676</span><span class="sxs-lookup"><span data-stu-id="a5b88-102">Compiler Error CS1676</span></span>

<span data-ttu-id="a5b88-103">Parametr "number" musi być zadeklarowany ze słowem kluczowym "Keyword"</span><span class="sxs-lookup"><span data-stu-id="a5b88-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="a5b88-104">Ten błąd występuje, gdy modyfikator typu parametru w metodzie anonimowej różni się od wartości użytej w deklaracji delegata, do którego jest stosowana Metoda.</span><span class="sxs-lookup"><span data-stu-id="a5b88-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="a5b88-105">Poniższy przykład generuje CS1676:</span><span class="sxs-lookup"><span data-stu-id="a5b88-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
