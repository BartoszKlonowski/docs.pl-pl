# <a name="compiler-error-cs0274"></a>CS0274 błąd kompilatora
Nie można określić modyfikatorów dostępności dla obu metod dostępu właściwości lub indeksatora "właściwości/indeksatora"  
  
 Ten błąd występuje, gdy Deklarujesz na obie metody dostępu właściwość lub indeksator o modyfikatory dostępu. Aby rozwiązać ten problem, należy użyć modyfikatora dostępu tylko na jednej z dwóch metod dostępu. Aby uzyskać więcej informacji, zobacz [dostępności metody dostępu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).  
  
 Poniższy przykład generuje CS0274:  
  
```csharp  
// CS0274.cs  
public class MyClass  
{  
    public int Property   // CS0274  
    {  
        public get { return 0; }  
        protected set { }  
    }  
}  
```
