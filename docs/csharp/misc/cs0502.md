 <span data-ttu-id="eeafe-104">Element członkowski klasy nie może być jednocześnie [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="eeafe-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="eeafe-105">Poniższy przykład spowoduje wygenerowanie CS0502:</span><span class="sxs-lookup"><span data-stu-id="eeafe-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
