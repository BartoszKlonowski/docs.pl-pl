# <a name="compiler-error-cs0502"></a><span data-ttu-id="2fceb-102">Błąd kompilatora CS0502</span><span class="sxs-lookup"><span data-stu-id="2fceb-102">Compiler Error CS0502</span></span>
<span data-ttu-id="2fceb-103">element "member" nie może być abstrakcyjny i zapieczętowany</span><span class="sxs-lookup"><span data-stu-id="2fceb-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="2fceb-104">Element członkowski klasy nie może być [abstrakcyjny](../language-reference/keywords/abstract.md) i [zapieczętowany](../language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="2fceb-104">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="2fceb-105">Poniższy przykład generuje CS0502:</span><span class="sxs-lookup"><span data-stu-id="2fceb-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
