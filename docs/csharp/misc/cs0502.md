# <a name="compiler-error-cs0502"></a>CS0502 błąd kompilatora
'składowa' nie może być abstrakcyjny i zapieczętowany  
  
 Element członkowski klasy nie może być jednocześnie [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0502:  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
