# <a name="compiler-error-cs0128"></a>CS0128 błąd kompilatora
Zmienna lokalna o nazwie "zmiennej" jest już zdefiniowany w tym zakresie  
  
 Kompilator wykrył deklaracje dwie zmienne lokalne o takiej samej nazwie. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0128:  
  
```csharp  
// CS0128.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         char i;  
         int i;   // CS0128  
      }  
   }  
}  
```
