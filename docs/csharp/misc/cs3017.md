# <a name="compiler-warning-level-1-cs3017"></a>Ostrzeżenie kompilatora (poziom 1) CS3017

Nie można określić atrybutu CLSCompliant w module, który różni się od atrybutu CLSCompliant w zestawie  
  
 To ostrzeżenie występuje, jeśli istnieje atrybut CLSCompliant zestawu, który powoduje konflikt z atrybutem CLSCompliant modułu. Zestaw, który jest zgodny ze specyfikacją CLS nie może zawierać modułów, które nie są zgodne ze specyfikacją CLS. Aby rozwiązać ten problem, upewnij się, że atrybuty zestawu i modułu CLSCompliant są zarówno true, jak i/lub powodują usunięcie jednego z atrybutów. Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS3017:  
  
```csharp  
// CS3017.cs  
// compile with: /target:module  
  
using System;  
  
[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]  
class C  
{  
    static void Main() {}  
}  
```
