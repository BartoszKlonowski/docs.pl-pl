## <a name="example"></a><span data-ttu-id="e00ec-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="e00ec-108">Example</span></span>  
 <span data-ttu-id="e00ec-109">Poniższy przykład generuje CS3017:</span><span class="sxs-lookup"><span data-stu-id="e00ec-109">The following example generates CS3017:</span></span>

```  
// compile with: /target:module

[module: CLSCompliant(true)]

// Try this line instead:

}  
```
