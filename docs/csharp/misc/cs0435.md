# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="17d6d-103">Ostrzeżenie kompilatora (poziom 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="17d6d-103">Compiler Warning (level 2) CS0435</span></span>
<span data-ttu-id="17d6d-104">Przestrzeń nazw "namespace" w elemencie "Assembly" powoduje konflikt z zaimportowanym typem "Type" w elemencie "Assembly".</span><span class="sxs-lookup"><span data-stu-id="17d6d-104">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="17d6d-105">Użycie przestrzeni nazw zdefiniowanej w elemencie "Assembly".</span><span class="sxs-lookup"><span data-stu-id="17d6d-105">Using the namespace defined in 'assembly'.</span></span>  
  
 <span data-ttu-id="17d6d-106">To ostrzeżenie jest generowane, gdy przestrzeń nazw w pliku źródłowym (file_2) powoduje konflikt z zaimportowanym typem w file_1.</span><span class="sxs-lookup"><span data-stu-id="17d6d-106">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="17d6d-107">Kompilator używa tego samego w pliku źródłowym.</span><span class="sxs-lookup"><span data-stu-id="17d6d-107">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="17d6d-108">Poniższy przykład generuje CS0435:</span><span class="sxs-lookup"><span data-stu-id="17d6d-108">The following example generates CS0435:</span></span>  
  
 <span data-ttu-id="17d6d-109">Najpierw Skompiluj ten plik:</span><span class="sxs-lookup"><span data-stu-id="17d6d-109">Compile this file first:</span></span>  
  
```csharp  
// CS0435_1.cs  
// compile with: /t:library  
public class Util
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="17d6d-110">Następnie Skompiluj ten plik:</span><span class="sxs-lookup"><span data-stu-id="17d6d-110">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util
{  
   public class A { }  
}  
  
public class Test
{  
   public static void Main()
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
