 <span data-ttu-id="b0bc6-105">To ostrzeżenie zostanie wyświetlone, jeśli przestrzeń nazw w pliku źródłowym (file_2) powoduje konflikt z zaimportowanym typem w elemencie file_1.</span><span class="sxs-lookup"><span data-stu-id="b0bc6-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="b0bc6-106">Kompilator używa jednego w pliku źródłowym.</span><span class="sxs-lookup"><span data-stu-id="b0bc6-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="b0bc6-107">Poniższy przykład generuje CS0435:</span><span class="sxs-lookup"><span data-stu-id="b0bc6-107">The following example generates CS0435:</span></span>

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="b0bc6-109">Następnie kompilacji tego pliku:</span><span class="sxs-lookup"><span data-stu-id="b0bc6-109">Then, compile this file:</span></span>

namespace Util   
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
