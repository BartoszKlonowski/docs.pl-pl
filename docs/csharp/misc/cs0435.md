# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="e2e55-102">CS0435 ostrzeżenie (poziom 2) kompilator</span><span class="sxs-lookup"><span data-stu-id="e2e55-102">Compiler Warning (level 2) CS0435</span></span>
<span data-ttu-id="e2e55-103">Przestrzeń nazw "namespace", "assembly" konflikt z zaimportowanym typem "type" w "assembly".</span><span class="sxs-lookup"><span data-stu-id="e2e55-103">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="e2e55-104">Za pomocą przestrzeni nazw, zdefiniowane w "assembly".</span><span class="sxs-lookup"><span data-stu-id="e2e55-104">Using the namespace defined in 'assembly'.</span></span>  
  
 <span data-ttu-id="e2e55-105">To ostrzeżenie zostanie wyświetlone, jeśli przestrzeń nazw w pliku źródłowym (file_2) powoduje konflikt z zaimportowanym typem w elemencie file_1.</span><span class="sxs-lookup"><span data-stu-id="e2e55-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="e2e55-106">Kompilator używa jednego w pliku źródłowym.</span><span class="sxs-lookup"><span data-stu-id="e2e55-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="e2e55-107">Poniższy przykład generuje CS0435:</span><span class="sxs-lookup"><span data-stu-id="e2e55-107">The following example generates CS0435:</span></span>  
  
 <span data-ttu-id="e2e55-108">Najpierw skompiluj ten plik:</span><span class="sxs-lookup"><span data-stu-id="e2e55-108">Compile this file first:</span></span>  
  
```csharp  
// CS0435_1.cs  
// compile with: /t:library  
public class Util   
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="e2e55-109">Następnie kompilacji tego pliku:</span><span class="sxs-lookup"><span data-stu-id="e2e55-109">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util   
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
