# <a name="compiler-error-cs1512"></a><span data-ttu-id="1060d-102">CS1512 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="1060d-102">Compiler Error CS1512</span></span>
<span data-ttu-id="1060d-103">Słowo kluczowe "base" nie jest dostępny w bieżącym kontekście</span><span class="sxs-lookup"><span data-stu-id="1060d-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="1060d-104">[Podstawowy](../../csharp/language-reference/keywords/base.md) użyto słowa kluczowego poza metody, właściwości lub konstruktora.</span><span class="sxs-lookup"><span data-stu-id="1060d-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="1060d-105">Poniższy przykład generuje CS1512:</span><span class="sxs-lookup"><span data-stu-id="1060d-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
