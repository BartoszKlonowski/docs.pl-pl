<span data-ttu-id="d49d0-103">Słowo kluczowe "Base" jest niedostępne w bieżącym kontekście</span><span class="sxs-lookup"><span data-stu-id="d49d0-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="d49d0-104">[Podstawowe](../language-reference/keywords/base.md) słowo kluczowe zostało użyte poza metodą, właściwością lub konstruktorem.</span><span class="sxs-lookup"><span data-stu-id="d49d0-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="d49d0-105">Poniższy przykład generuje CS1512:</span><span class="sxs-lookup"><span data-stu-id="d49d0-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
