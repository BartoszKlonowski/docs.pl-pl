# <a name="compiler-error-cs1559"></a>CS1559 błąd kompilatora
Nie można użyć "object" dla metody Main, ponieważ jest on zaimportowany  
  
 Określono nieprawidłową klasę do [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora; klasy nie można użyć jako lokalizację [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody.
