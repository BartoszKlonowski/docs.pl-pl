# <a name="compiler-error-cs1559"></a>Błąd kompilatora CS1559

Nie można użyć elementu "Object" dla metody Main, ponieważ jest on zaimportowany  
  
 Określono nieprawidłową klasę jako opcję kompilatora [głównego](../language-reference/compiler-options/main-compiler-option.md) ; nie można użyć klasy jako lokalizacji metody [Main](../programming-guide/main-and-command-args/index.md) .
