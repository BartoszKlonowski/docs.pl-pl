# <a name="compiler-error-cs1534"></a>CS1534 błąd kompilatora
Przeciążony operator binarny "operator" przyjmuje dwa parametry  
  
 Definicja plik binarny [operator](../../csharp/language-reference/keywords/operator.md) należy wykonać dwa parametry.  
  
 Poniższy przykład spowoduje wygenerowanie CS1534:  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
