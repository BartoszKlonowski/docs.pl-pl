# <a name="compiler-error-cs0008"></a>CS0008 błąd kompilatora
Nieoczekiwany błąd podczas odczytywania metadanych z pliku 'Plik'-'Opis'  
  
 Biblioteki DLL został otwarty pomyślnie pobierania metadanych, ale jest uszkodzony, w taki sposób, że nie można odczytać danych z niego. Aby uzyskać więcej informacji, zobacz [/Reference (opcje kompilatora C#)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
