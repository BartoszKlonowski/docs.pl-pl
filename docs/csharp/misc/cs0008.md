# <a name="compiler-error-cs0008"></a>Błąd kompilatora CS0008

Nieoczekiwany błąd podczas odczytywania metadanych z pliku "File" — "Description"  
  
 Pomyślnie otwarto bibliotekę DLL na potrzeby pobierania metadanych, ale jest ona uszkodzona, ponieważ nie można było odczytać z niej danych. Aby uzyskać więcej informacji, zobacz [-Reference (opcje kompilatora C#)](../language-reference/compiler-options/reference-compiler-option.md).
