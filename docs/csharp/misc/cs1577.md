# <a name="compiler-error-cs1577"></a>Błąd kompilatora CS1577

Generowanie zestawu nie powiodło się — Przyczyna  
  
 Część kompilacji zestawu nie powiodła się po pomyślnym skompilowaniu kodu źródłowego przez kompilator języka C#. Aby uzyskać więcej informacji, zobacz dokumentację dotyczącą błędów narzędzia ALink ([ błędy i ostrzeżenia narzędziaAl.exe](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)).
