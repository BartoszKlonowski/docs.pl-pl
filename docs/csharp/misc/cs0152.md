# <a name="compiler-error-cs0152"></a>CS0152 błąd kompilatora
Etykieta Etykieta występuje już w tej instrukcji switch  
  
 Etykiety został powtórzony podczas [Przełącz](../../csharp/language-reference/keywords/switch.md) instrukcji. Aby uzyskać więcej informacji, zobacz [Przełącz](../../csharp/language-reference/keywords/switch.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0152:  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
