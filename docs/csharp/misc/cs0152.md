<span data-ttu-id="a5c57-103">Etykieta "etykieta" występuje już w tej instrukcji switch</span><span class="sxs-lookup"><span data-stu-id="a5c57-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="a5c57-104">Etykieta została powtórzona w instrukcji [Switch](../language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="a5c57-104">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="a5c57-105">Aby uzyskać więcej informacji, zobacz [przełącznik](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="a5c57-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="a5c57-106">Poniższy przykład generuje CS0152:</span><span class="sxs-lookup"><span data-stu-id="a5c57-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
