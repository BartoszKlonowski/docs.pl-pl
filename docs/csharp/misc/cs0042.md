# <a name="compiler-error-cs0042"></a>Błąd kompilatora CS0042

Nieoczekiwany błąd podczas tworzenia pliku informacji o debugowaniu "plik" — Przyczyna:  
  
 Kompilator nie może utworzyć informacji debugowania; *Przyczyna* zawiera dodatkowe informacje dotyczące warunku błędu.
