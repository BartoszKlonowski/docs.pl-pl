# <a name="compiler-error-cs0042"></a>CS0042 błąd kompilatora
Nieoczekiwany błąd tworzenia pliku 'Plik' informacje o debugowaniu — "reason"  
  
 Kompilator nie może utworzyć informacje debugowania; *Przyczyna* zawiera dodatkowe informacje na temat warunku błędu.
