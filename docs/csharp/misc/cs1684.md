 <span data-ttu-id="5797b-104">Ten błąd może być spowodowany przez odwołanie w jednej przestrzeni nazw odwołujące się do typu, który jest wyświetlany jest tekst istnieje w drugim przestrzeni nazw, ale typ nie istnieje.</span><span class="sxs-lookup"><span data-stu-id="5797b-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="5797b-105">Na przykład mydll.dll wynika z tego typu `A` istnieje wewnątrz yourdll.dll, ale żaden taki typ nie istnieje wewnątrz yourdll.dll.</span><span class="sxs-lookup"><span data-stu-id="5797b-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="5797b-106">Jedną z możliwych przyczyn tego błędu jest wersja yourdll.dll używasz zbyt stara i `A` jeszcze nie został zdefiniowany.</span><span class="sxs-lookup"><span data-stu-id="5797b-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="5797b-107">Poniższy przykład spowoduje wygenerowanie CS1684.</span><span class="sxs-lookup"><span data-stu-id="5797b-107">The following sample generates CS1684.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5797b-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5797b-108">Example</span></span>  
  
```csharp  
// CS1684_a.cs  
// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="5797b-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="5797b-109">Example</span></span>  
  
```csharp  
// CS1684_b.cs  
// compile with: /target:library /r:cs1684_a.dll  
// post-build command: del /f CS1684_a.dll  
using System;  
public class Ref   
{  
   public static A GetA() { return new A(); }  
   public static C2 GetC() { return new C2(); }  
}  
```
