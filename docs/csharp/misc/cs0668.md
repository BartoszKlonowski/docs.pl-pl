# <a name="compiler-error-cs0668"></a><span data-ttu-id="95fff-102">CS0668 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="95fff-102">Compiler Error CS0668</span></span>
<span data-ttu-id="95fff-103">Istnieją dwa indeksatory o różnych nazwach; należy użyć atrybutu IndexerName o takiej samej nazwie, dla każdego indeksatora w typie</span><span class="sxs-lookup"><span data-stu-id="95fff-103">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="95fff-104">Wartości przekazanych do **IndexerName** atrybut musi być taka sama dla wszystkich indeksatorów w typie.</span><span class="sxs-lookup"><span data-stu-id="95fff-104">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="95fff-105">Aby uzyskać więcej informacji na temat **IndexerName** atrybutów, zobacz [IndexerNameAttribute klasy](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="95fff-105">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="95fff-106">Poniższy przykład generuje CS0668:</span><span class="sxs-lookup"><span data-stu-id="95fff-106">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
