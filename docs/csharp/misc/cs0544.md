 <span data-ttu-id="3066c-105">Podjęto próbę zastąpienia niewłaściwościowego typu danych jako [Właściwości](../programming-guide/classes-and-structs/properties.md), co jest niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="3066c-105">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

   public int i;  
}

      {

      }  
   }
