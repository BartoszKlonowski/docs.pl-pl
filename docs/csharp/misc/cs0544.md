 <span data-ttu-id="87af6-104">Podjęto próbę zastąpienia niewłaściwościowego typu danych jako [Właściwości](../programming-guide/classes-and-structs/properties.md), co jest niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="87af6-104">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp

{  
   public int i;  
}

   // public new int i {

      {  
         return 0;  
      }  
   }

```
