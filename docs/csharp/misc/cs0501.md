# <a name="compiler-error-cs0501"></a>Błąd kompilatora CS0501
Funkcja składowa musi deklarować treść, ponieważ nie jest oznaczona jako abstrakcyjna, extern lub częściowa  
  
 Metody nieabstrakcyjne muszą mieć implementacje.  
  
 Poniższy przykład generuje CS0501:  
  
```csharp  
// CS0501.cs  
// compile with: /target:library  
public class clx  
{  
   public void f();   // CS0501 declared but not defined  
   public void g() {}   // OK  
}  
```
