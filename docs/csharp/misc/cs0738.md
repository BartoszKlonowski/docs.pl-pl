## <a name="example"></a><span data-ttu-id="92bc6-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="92bc6-108">Example</span></span>  
 <span data-ttu-id="92bc6-109">Poniższy kod generuje CS0738, ponieważ metoda klasy zwraca `void` i zwraca składowej interfejsu o takiej samej nazwie `int`:</span><span class="sxs-lookup"><span data-stu-id="92bc6-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

{  
    int TestMethod();  
}
