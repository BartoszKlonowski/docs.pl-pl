## <a name="example"></a><span data-ttu-id="c08fa-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="c08fa-109">Example</span></span>  

 <span data-ttu-id="c08fa-110">Poniższy kod generuje CS0738, ponieważ metoda klasy zwraca `void` i element członkowski interfejsu o tej samej nazwie zwraca `int` :</span><span class="sxs-lookup"><span data-stu-id="c08fa-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

    int TestMethod();
