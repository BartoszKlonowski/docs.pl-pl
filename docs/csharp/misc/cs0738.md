## <a name="example"></a><span data-ttu-id="6de21-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="6de21-109">Example</span></span>  
 <span data-ttu-id="6de21-110">Poniższy kod generuje CS0738, ponieważ metoda klasy zwraca `void` i element członkowski interfejsu o tej samej nazwie zwraca `int` :</span><span class="sxs-lookup"><span data-stu-id="6de21-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
