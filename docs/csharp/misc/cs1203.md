# <a name="compiler-warning-level-1-cs1203"></a>Kompilatora (poziom 1) ostrzeżenie CS1203
Funkcja "funkcji" jest przestarzała. Użyj "funkcji" zamiast ".  
  
 Funkcja "nieprawidłową funkcję" jest przestarzała. Zamiast tego użyj "Nieprawidłowa funkcja" ".  
  
 Funkcja, której próbujesz użyć, jest już przestarzały. Zaktualizuj swój kod, aby zamiast tego użyj nieprawidłowa funkcja.
