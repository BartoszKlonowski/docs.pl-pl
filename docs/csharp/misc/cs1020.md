# <a name="compiler-error-cs1020"></a>CS1020 błąd kompilatora
Oczekiwano operatora binarnego  
  
 Nastąpiła próba do definiowania Przeciążony operator, ale operator nie Oczekiwano operatora binarnego, która przyjmuje dwa parametry. Aby uzyskać listę operatory z możliwością przeciążenia, zobacz [operatory z możliwością przeciążenia](../../csharp/language-reference/operators/operator-overloading.md#overloadable-operators) części [przeciążania operatora](../../csharp/language-reference/operators/operator-overloading.md) artykułu.
  
 Poniższy przykład spowoduje wygenerowanie CS1020:  
  
```csharp  
// CS1020.cs  
public class iii  
{  
   public static int operator ++(iii aa, int bb)   // CS1020, change ++ to +  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
