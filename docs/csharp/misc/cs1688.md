<span data-ttu-id="7a73d-103">Nie można przekonwertować bloku metody anonimowej bez listy parametrów w celu delegowania typu "delegate", ponieważ ma jeden lub więcej parametrów out</span><span class="sxs-lookup"><span data-stu-id="7a73d-103">Cannot convert anonymous method block without a parameter list to delegate type 'delegate' because it has one or more out parameters</span></span>  
  
 <span data-ttu-id="7a73d-104">Kompilator umożliwia parametry, które mają zostać pominięte z bloku metody anonimowej w większości przypadków.</span><span class="sxs-lookup"><span data-stu-id="7a73d-104">The compiler allows parameters to be omitted from an anonymous method block in most cases.</span></span> <span data-ttu-id="7a73d-105">Ten błąd pojawia się, gdy blok metody anonimowej nie ma `out` listy parametrów, ale pełnomocnik ma parametr.</span><span class="sxs-lookup"><span data-stu-id="7a73d-105">This error arises when the anonymous method block does not have a parameter list, but the delegate has an `out` parameter.</span></span> <span data-ttu-id="7a73d-106">Kompilator nie zezwala na tę sytuację, ponieważ `out` musiałby zignorować obecność parametru, który jest mało prawdopodobne, aby być poprawne zachowanie.</span><span class="sxs-lookup"><span data-stu-id="7a73d-106">The compiler does not allow this situation because it would need to ignore the presence of the `out` parameter, which is unlikely to be the correct behavior.</span></span>
