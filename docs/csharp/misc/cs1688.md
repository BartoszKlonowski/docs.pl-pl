# <a name="compiler-error-cs1688"></a>CS1688 błąd kompilatora
Nie można przekonwertować bloku metody anonimowej bez listy parametrów na typ "delegowanie" delegata, ponieważ ma on jeden lub więcej parametrów out  
  
 Kompilator umożliwi parametry można pominąć bloku metody anonimowej w większości przypadków. Ten błąd pojawia się po bloku metody anonimowej bez listy parametrów, ale pełnomocnik ma `out` parametru. Kompilator nie zezwala na tę sytuację, ponieważ musiałaby Ignoruj obecności `out` parametr, który nie powinny być poprawne zachowanie.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje błąd CS1688.  
  
```csharp  
// CS1688.cs  
using System;  
delegate void OutParam(out int i);  
class ErrorCS1676  
{  
    static void Main()   
    {  
        OutParam o;  
        o = delegate  // CS1688  
        // Try this instead:  
        // o = delegate(out int i)  
        {   
            Console.WriteLine("");  
        };  
    }  
}  
```
