# <a name="compiler-error-cs0149"></a><span data-ttu-id="7750a-102">Błąd kompilatora CS0149</span><span class="sxs-lookup"><span data-stu-id="7750a-102">Compiler Error CS0149</span></span>
<span data-ttu-id="7750a-103">Oczekiwana nazwa metody</span><span class="sxs-lookup"><span data-stu-id="7750a-103">Method name expected</span></span>  
  
 <span data-ttu-id="7750a-104">Podczas tworzenia [pełnomocnika](../language-reference/builtin-types/reference-types.md)określ metodę.</span><span class="sxs-lookup"><span data-stu-id="7750a-104">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="7750a-105">Aby uzyskać więcej informacji, zobacz [Delegatów](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="7750a-105">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="7750a-106">Poniższy przykład generuje CS0149:</span><span class="sxs-lookup"><span data-stu-id="7750a-106">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
