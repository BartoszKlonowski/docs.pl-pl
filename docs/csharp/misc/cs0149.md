# <a name="compiler-error-cs0149"></a><span data-ttu-id="c985c-103">Błąd kompilatora CS0149</span><span class="sxs-lookup"><span data-stu-id="c985c-103">Compiler Error CS0149</span></span>
<span data-ttu-id="c985c-104">Oczekiwano nazwy metody</span><span class="sxs-lookup"><span data-stu-id="c985c-104">Method name expected</span></span>  
  
 <span data-ttu-id="c985c-105">Podczas tworzenia [delegata](../language-reference/builtin-types/reference-types.md)należy określić metodę.</span><span class="sxs-lookup"><span data-stu-id="c985c-105">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="c985c-106">Aby uzyskać więcej informacji, zobacz [delegats](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="c985c-106">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="c985c-107">Poniższy przykład generuje CS0149:</span><span class="sxs-lookup"><span data-stu-id="c985c-107">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
