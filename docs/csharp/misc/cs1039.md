# <a name="compiler-error-cs1039"></a>CS1039 błąd kompilatora
Niezakończony literał ciągu  
  
 Kompilator wykrył źle sformułowane [ciąg](../../csharp/language-reference/keywords/string.md) literału.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1039. Aby naprawić błąd, należy dodać kończącego znaku cudzysłowu.  
  
```csharp  
// CS1039.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        string b = @"hello, world;   // CS1039  
    }  
}  
```
