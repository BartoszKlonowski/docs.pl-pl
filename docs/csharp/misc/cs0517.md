# <a name="compiler-error-cs0517"></a>CS0517 błąd kompilatora
"class" nie ma klasy bazowej i nie można wywołać konstruktora bazowego  
  
 CS0517 może wystąpić tylko wtedy, gdy .NET Framework środowisko uruchomieniowe języka wspólnego kompiluje kod źródłowy dla klasy obiektów jest to jedyna klasa, która nie ma klasy bazowej.
