# <a name="compiler-error-cs0517"></a>Błąd kompilatora CS0517
Klasa "Class" nie ma klasy bazowej i nie może wywołać konstruktora podstawowego  
  
 CS0517 może wystąpić tylko wtedy, gdy .NET Framework środowisko uruchomieniowe języka wspólnego kompiluje kod źródłowy klasy Object, który jest jedyną klasą, która nie ma klasy bazowej.
