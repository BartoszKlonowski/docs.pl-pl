# <a name="compiler-error-cs0308"></a><span data-ttu-id="680a8-102">CS0308 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="680a8-102">Compiler Error CS0308</span></span>
<span data-ttu-id="680a8-103">Typ lub metodę nierodzajową 'Identyfikator' nie można używać z argumentami typu.</span><span class="sxs-lookup"><span data-stu-id="680a8-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="680a8-104">Ta metoda lub typ nie jest rodzajowa, ale zostało użyte z argumentami typu.</span><span class="sxs-lookup"><span data-stu-id="680a8-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="680a8-105">Aby uniknąć tego błędu, Usuń nawiasach i argumentów typu lub ponownie zadeklarować metody lub typu jako typu lub metody ogólnej.</span><span class="sxs-lookup"><span data-stu-id="680a8-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="680a8-106">Poniższy przykład generuje CS0308:</span><span class="sxs-lookup"><span data-stu-id="680a8-106">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="680a8-107">Poniższy przykład generuje również CS0308.</span><span class="sxs-lookup"><span data-stu-id="680a8-107">The following example also generates CS0308.</span></span> <span data-ttu-id="680a8-108">Aby rozwiązać problem, należy użyć dyrektywy "przy użyciu system.Collections.Generic —".</span><span class="sxs-lookup"><span data-stu-id="680a8-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
