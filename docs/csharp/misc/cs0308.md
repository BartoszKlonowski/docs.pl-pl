# <a name="compiler-error-cs0308"></a><span data-ttu-id="ae86a-102">CS0308 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="ae86a-102">Compiler Error CS0308</span></span>
<span data-ttu-id="ae86a-103">Nie można używać nieogólnego lub metody typu 'Identyfikator' z argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="ae86a-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="ae86a-104">Metody lub typu nie jest ogólna, ale zostało użyte z argumentami typu.</span><span class="sxs-lookup"><span data-stu-id="ae86a-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="ae86a-105">Aby uniknąć tego błędu, Usuń pod kątem nawiasy i argumenty typu lub ponownie zadeklarować metody lub typu jako typu lub metody rodzajowej.</span><span class="sxs-lookup"><span data-stu-id="ae86a-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="ae86a-106">Poniższy przykład generuje CS0308:</span><span class="sxs-lookup"><span data-stu-id="ae86a-106">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="ae86a-107">Poniższy przykład generuje również CS0308.</span><span class="sxs-lookup"><span data-stu-id="ae86a-107">The following example also generates CS0308.</span></span> <span data-ttu-id="ae86a-108">Aby naprawić błąd, należy użyć dyrektywy "using System.Collections.Generic".</span><span class="sxs-lookup"><span data-stu-id="ae86a-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
