# <a name="compiler-error-cs0308"></a><span data-ttu-id="00f7f-103">Błąd kompilatora CS0308</span><span class="sxs-lookup"><span data-stu-id="00f7f-103">Compiler Error CS0308</span></span>
<span data-ttu-id="00f7f-104">Nie można użyć typu "Identyfikator" nieogólnym ani metody z argumentami typu.</span><span class="sxs-lookup"><span data-stu-id="00f7f-104">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="00f7f-105">Metoda lub typ nie jest rodzajowy, ale został użyty z argumentami typu.</span><span class="sxs-lookup"><span data-stu-id="00f7f-105">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="00f7f-106">Aby uniknąć tego błędu, Usuń nawiasy ukośne i argumenty typu lub Zadeklaruj metodę lub typ jako metodę rodzajową lub typ.</span><span class="sxs-lookup"><span data-stu-id="00f7f-106">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="00f7f-107">Poniższy przykład generuje CS0308:</span><span class="sxs-lookup"><span data-stu-id="00f7f-107">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="00f7f-108">Poniższy przykład generuje również CS0308.</span><span class="sxs-lookup"><span data-stu-id="00f7f-108">The following example also generates CS0308.</span></span> <span data-ttu-id="00f7f-109">Aby rozwiązać ten problem, użyj dyrektywy "using System. Collections. Generic".</span><span class="sxs-lookup"><span data-stu-id="00f7f-109">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
