# <a name="compiler-error-cs1918"></a><span data-ttu-id="2b7a1-102">CS1918 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="2b7a1-102">Compiler Error CS1918</span></span>
<span data-ttu-id="2b7a1-103">Typ elementów członkowskich właściwości "name", "type" nie można przypisać inicjatora obiektu, ponieważ jest wartość typu.</span><span class="sxs-lookup"><span data-stu-id="2b7a1-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="2b7a1-104">Ten błąd występuje, gdy spróbujesz użyć inicjatora obiektów do zainicjowania właściwości typu struktury, która jest właściwością klasy, która jest inicjowany.</span><span class="sxs-lookup"><span data-stu-id="2b7a1-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2b7a1-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="2b7a1-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="2b7a1-106">Pola właściwości w inicjatorze obiektów musi pełne zainicjowanie, zmiana struktury do typu klasy.</span><span class="sxs-lookup"><span data-stu-id="2b7a1-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="2b7a1-107">W przeciwnym razie zainicjowania elementy członkowskie struktury w wywołaniu metody oddzielne po utworzeniu obiektu przy użyciu inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="2b7a1-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2b7a1-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="2b7a1-108">Example</span></span>  
 <span data-ttu-id="2b7a1-109">Poniższy przykład generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="2b7a1-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
