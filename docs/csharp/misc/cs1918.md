# <a name="compiler-error-cs1918"></a><span data-ttu-id="a687c-102">Błąd kompilatora CS1918</span><span class="sxs-lookup"><span data-stu-id="a687c-102">Compiler Error CS1918</span></span>
<span data-ttu-id="a687c-103">Składowych właściwości "name" typu "Type" nie można przypisać za pomocą inicjatora obiektów, ponieważ jest on typu wartości.</span><span class="sxs-lookup"><span data-stu-id="a687c-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="a687c-104">Ten błąd występuje podczas próby użycia inicjatora obiektów w celu zainicjowania właściwości typu struktury, który jest samą właściwością inicjowanej klasy.</span><span class="sxs-lookup"><span data-stu-id="a687c-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a687c-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a687c-105">To correct this error</span></span>  
  
1. <span data-ttu-id="a687c-106">Jeśli musisz w pełni zainicjować pola właściwości w inicjatorze obiektów, Zmień strukturę na typ klasy.</span><span class="sxs-lookup"><span data-stu-id="a687c-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="a687c-107">W przeciwnym razie zainicjuj elementy członkowskie struktury w osobnym wywołaniu metody po utworzeniu obiektu za pomocą inicjatora obiektów.</span><span class="sxs-lookup"><span data-stu-id="a687c-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a687c-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="a687c-108">Example</span></span>  
 <span data-ttu-id="a687c-109">Poniższy przykład generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="a687c-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
