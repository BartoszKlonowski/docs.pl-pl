# <a name="compiler-error-cs1918"></a><span data-ttu-id="a04d9-102">CS1918 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="a04d9-102">Compiler Error CS1918</span></span>
<span data-ttu-id="a04d9-103">Składowych właściwości "name", "type" nie można przypisać za pomocą inicjatora obiektu, ponieważ jest on wartości typu typu.</span><span class="sxs-lookup"><span data-stu-id="a04d9-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="a04d9-104">Ten błąd występuje, gdy użytkownik próbuje użyć inicjatora obiektów, aby zainicjować właściwości typu struktury, która sama jest właściwością klasy, który jest inicjowany.</span><span class="sxs-lookup"><span data-stu-id="a04d9-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a04d9-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a04d9-105">To correct this error</span></span>  
  
1. <span data-ttu-id="a04d9-106">Jeśli należy całkowicie zainicjować pola, właściwości w inicjatorze obiektu, należy zmienić struktury do typu klasy.</span><span class="sxs-lookup"><span data-stu-id="a04d9-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="a04d9-107">Po utworzeniu obiektu za pomocą inicjatora obiektów, w przeciwnym razie zainicjować elementy członkowskie struktury w wywołaniu metody oddzielne.</span><span class="sxs-lookup"><span data-stu-id="a04d9-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a04d9-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="a04d9-108">Example</span></span>  
 <span data-ttu-id="a04d9-109">Poniższy przykład generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="a04d9-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
