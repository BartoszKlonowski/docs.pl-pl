# <a name="compiler-error-cs0549"></a><span data-ttu-id="1908a-103">Błąd kompilatora CS0549</span><span class="sxs-lookup"><span data-stu-id="1908a-103">Compiler Error CS0549</span></span>

<span data-ttu-id="1908a-104">"Function" to nowa wirtualna składowa w klasie "Class" klasy zapieczętowanej</span><span class="sxs-lookup"><span data-stu-id="1908a-104">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="1908a-105">Klasa [zapieczętowana](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) nie może być używana jako klasa bazowa.</span><span class="sxs-lookup"><span data-stu-id="1908a-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="1908a-106">W związku z tym nie jest możliwe posiadanie metody wirtualnej w klasie zapieczętowanej.</span><span class="sxs-lookup"><span data-stu-id="1908a-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="1908a-107">Poniższy przykład generuje CS0549:</span><span class="sxs-lookup"><span data-stu-id="1908a-107">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
