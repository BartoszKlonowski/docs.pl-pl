# <a name="compiler-error-cs0549"></a>Błąd kompilatora CS0549
"Function" to nowa wirtualna składowa w klasie "Class" klasy zapieczętowanej  
  
 Klasa [zapieczętowana](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) nie może być używana jako klasa bazowa.  W związku z tym nie jest możliwe posiadanie metody wirtualnej w klasie zapieczętowanej.  
  
 Poniższy przykład generuje CS0549:  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
