 <span data-ttu-id="07986-105">Klasa [zapieczętowana](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) nie może być używana jako klasa bazowa.</span><span class="sxs-lookup"><span data-stu-id="07986-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="07986-106">W związku z tym nie jest możliwe posiadanie metody wirtualnej w klasie zapieczętowanej.</span><span class="sxs-lookup"><span data-stu-id="07986-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>

```csharp  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
