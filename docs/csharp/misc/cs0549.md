# <a name="compiler-error-cs0549"></a>CS0549 błąd kompilatora
'Funkcja' to nowa wirtualna składowa w klasie zapieczętowanej "class"  
  
 A [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md)[klasy](../../csharp/language-reference/keywords/class.md) nie można użyć jako klasę bazową.  Dlatego jest bezcelowe mieć wirtualnej metody w klasie zapieczętowanej.  
  
 Poniższy przykład spowoduje wygenerowanie CS0549:  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
