# <a name="compiler-error-cs1028"></a>CS1028 błąd kompilatora
Nieoczekiwana dyrektywa preprocesora  
  
 A [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md) został znaleziony, ale nie oczekuje.  
  
 Na przykład `#endif` znaleziono bez poprzedzających `#if`.  
  
 Poniższy przykład spowoduje wygenerowanie CS1028:  
  
```csharp  
// CS1028.cs  
#endif   // CS1028, no matching #if  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
