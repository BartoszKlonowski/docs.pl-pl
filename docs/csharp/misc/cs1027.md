# <a name="compiler-error-cs1027"></a>Błąd kompilatora CS1027
\#Dyrektywa endif oczekiwana  
  
 Nie znaleziono zgodnej `#endif` [dyrektywy preprocesora](../language-reference/preprocessor-directives/index.md) dla określonej `#if` dyrektywy. Lub kompilator mógł znaleźć `#endregion` dyrektywę, gdy w bloku nie ma żadnej pasującej `#region` dyrektywy `#if` .  
  
 Poniższy przykład generuje CS1027:  
  
```csharp  
// CS1027.cs  
#if true   // CS1027, uncomment next line to resolve  
// #endif  
  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
