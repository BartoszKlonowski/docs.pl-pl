 <span data-ttu-id="fb9f3-105">Metoda nie była zgodna z typem delegata.</span><span class="sxs-lookup"><span data-stu-id="fb9f3-105">The method was not compatible with the delegate type.</span></span> <span data-ttu-id="fb9f3-106">Pasujące typy argumentów, ale zwracany typ nie jest prawidłowym typem zwracanym dla tego delegata.</span><span class="sxs-lookup"><span data-stu-id="fb9f3-106">The argument types matched, but the return type was not the correct return type for that delegate.</span></span> <span data-ttu-id="fb9f3-107">Aby uniknąć tego błędu, należy użyć innej metody, zmienić zwracany typ metody lub zmienić zwracany typ delegata.</span><span class="sxs-lookup"><span data-stu-id="fb9f3-107">To avoid this error, use a different method, change the method's return type, or change the delegate's return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fb9f3-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="fb9f3-108">Example</span></span>  
 <span data-ttu-id="fb9f3-109">Poniższy przykład generuje CS0407:</span><span class="sxs-lookup"><span data-stu-id="fb9f3-109">The following sample generates CS0407:</span></span>  
  
```csharp  
// CS0407.cs  
public delegate int MyDelegate();  
  
class C  
{  
    MyDelegate d;  
  
    public C()  
    {  
        d = new MyDelegate(F);  // OK: F returns int  
        d = new MyDelegate(G);  // CS0407 – G doesn't return int  
    }  
  
    public int F()  
    {  
        return 1;  
    }  
  
    public void G()  
    {  
    }  
  
    public static void Main()  
    {  
        C c1 = new C();  
    }  
}  
```
