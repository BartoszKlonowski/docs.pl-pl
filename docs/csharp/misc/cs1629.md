# <a name="compiler-error-cs1629"></a>CS1629 błąd kompilatora
Niebezpieczny kod nie może występować w iteratorach.  
  
 C# Specyfikacji języka nie zezwalaj na niebezpieczny kod w iteratorach.  
  
 Poniższy przykład spowoduje wygenerowanie CS1629:  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
