 <span data-ttu-id="f7884-105">Specyfikacja języka C# nie zezwala na użycie niebezpiecznego kodu w iteratorach.</span><span class="sxs-lookup"><span data-stu-id="f7884-105">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="f7884-106">Poniższy przykład generuje CS1629:</span><span class="sxs-lookup"><span data-stu-id="f7884-106">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe
using System.Collections.Generic;  
class C
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
