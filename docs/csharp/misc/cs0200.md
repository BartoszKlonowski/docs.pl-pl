<span data-ttu-id="067f8-104">Podjęto próbę przypisania wartości do [właściwości,](../programming-guide/classes-and-structs/using-properties.md)ale właściwość nie ma zestawu dostępu lub przypisanie było poza konstruktorem.</span><span class="sxs-lookup"><span data-stu-id="067f8-104">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="067f8-105">Rozwiąż ten błąd, dodając zestaw akcesor.</span><span class="sxs-lookup"><span data-stu-id="067f8-105">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="067f8-106">Aby uzyskać więcej informacji, zobacz [Jak zadeklarować i używać odczytu właściwości zapisu](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span><span class="sxs-lookup"><span data-stu-id="067f8-106">For more information, see [How to declare and use read write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="067f8-107">Przykłady</span><span class="sxs-lookup"><span data-stu-id="067f8-107">Examples</span></span>
<span data-ttu-id="067f8-108">Następujący przykład generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="067f8-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="067f8-109">W poniższym przykładzie użyto [właściwości automatycznie implementowanych](../programming-guide/classes-and-structs/auto-implemented-properties.md), [inicjatorów obiektów](../programming-guide/classes-and-structs/object-and-collection-initializers.md)i nadal generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="067f8-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="067f8-110">Przypisanie do właściwości lub indeksatora "właściwość", która jest tylko do odczytu, można osiągnąć poprzez dodanie akcesora zestawu lub przypisując do właściwości w konstruktorze obiektu.</span><span class="sxs-lookup"><span data-stu-id="067f8-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
