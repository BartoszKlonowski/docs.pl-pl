# <a name="compiler-error-cs0200"></a>Błąd kompilatora CS0200
Nie można przypisać właściwości lub "właściwości" do właściwości lub indeksatora — jest tylko do odczytu

Podjęto próbę przypisania wartości do [właściwości,](../programming-guide/classes-and-structs/using-properties.md)ale właściwość nie ma zestawu dostępu lub przypisanie było poza konstruktorem. Rozwiąż ten błąd, dodając zestaw akcesor. Aby uzyskać więcej informacji, zobacz [Jak zadeklarować i używać odczytu właściwości zapisu](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).

## <a name="examples"></a>Przykłady
Następujący przykład generuje CS0200:

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

W poniższym przykładzie użyto [właściwości automatycznie implementowanych](../programming-guide/classes-and-structs/auto-implemented-properties.md), [inicjatorów obiektów](../programming-guide/classes-and-structs/object-and-collection-initializers.md)i nadal generuje CS0200:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

Przypisanie do właściwości lub indeksatora "właściwość", która jest tylko do odczytu, można osiągnąć poprzez dodanie akcesora zestawu lub przypisując do właściwości w konstruktorze obiektu.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
