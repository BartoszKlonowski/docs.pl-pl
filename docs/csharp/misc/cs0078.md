# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="111a2-102">Kompilator CS0078 ostrzegawcze (poziom 4)</span><span class="sxs-lookup"><span data-stu-id="111a2-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="111a2-103">Sufiks "l" można łatwo pomylić z cyfrą "1" — Użyj "L" dla jasności</span><span class="sxs-lookup"><span data-stu-id="111a2-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="111a2-104">Kompilator wyświetli ostrzeżenie, gdy wykryje Rzutowanie na długo przy użyciu małych l zamiast L. wielkie litery</span><span class="sxs-lookup"><span data-stu-id="111a2-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="111a2-105">Poniższy przykład generuje CS0078:</span><span class="sxs-lookup"><span data-stu-id="111a2-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
