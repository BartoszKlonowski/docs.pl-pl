# <a name="compiler-error-cs0401"></a>Błąd kompilatora CS0401
Ograniczenie New () musi być ostatnim określonym ograniczeniem  
  
 W przypadku korzystania z wielu ograniczeń należy wyświetlić wszystkie pozostałe ograniczenia przed utworzeniem ograniczenia New ().  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0401.  
  
```csharp  
// CS0401.cs  
// compile with: /target:library  
using System;  
 class C<T> where T : new(), IDisposable {}  // CS0401  
  
class D<T> where T : IDisposable  
{  
   static void F<U>() where U : new(), IDisposable{}   // CS0401  
}  
```
