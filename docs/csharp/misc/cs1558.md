# <a name="compiler-error-cs1558"></a>Błąd kompilatora CS1558
element "Class" nie ma odpowiedniej statycznej metody Main  
  
 Opcja kompilatora [/Main](../language-reference/compiler-options/main-compiler-option.md) określa klasę, w której ma zostać wyszukana Metoda **Main** . Jednak Metoda [Main](../programming-guide/main-and-command-args/index.md) nie została prawidłowo zdefiniowana.  
  
 Poniższy przykład generuje CS1558 z powodu nieprawidłowego typu zwracanego.  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
