# <a name="compiler-error-cs1558"></a><span data-ttu-id="0d17a-102">CS1558 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="0d17a-102">Compiler Error CS1558</span></span>
<span data-ttu-id="0d17a-103">"class" nie ma odpowiedniej statycznej metody Main</span><span class="sxs-lookup"><span data-stu-id="0d17a-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="0d17a-104">[/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora określonej klasy, w którym znajdują się **Main** metody.</span><span class="sxs-lookup"><span data-stu-id="0d17a-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="0d17a-105">Jednak [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody nie został poprawnie zdefiniowany.</span><span class="sxs-lookup"><span data-stu-id="0d17a-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="0d17a-106">Poniższy przykład generuje CS1558 ze względu na nieprawidłowy typ zwracany.</span><span class="sxs-lookup"><span data-stu-id="0d17a-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
