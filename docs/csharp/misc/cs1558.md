# <a name="compiler-error-cs1558"></a>CS1558 błąd kompilatora
"class" nie ma odpowiedniej statycznej metody Main  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora określonej klasy, w którym znajdują się **Main** metody. Jednak [Main](../../csharp/programming-guide/main-and-command-args/index.md) metody nie został poprawnie zdefiniowany.  
  
 Poniższy przykład generuje CS1558 ze względu na nieprawidłowy typ zwracany.  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
