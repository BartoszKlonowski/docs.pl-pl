# <a name="compiler-error-cs1558"></a><span data-ttu-id="a4396-102">Błąd kompilatora CS1558</span><span class="sxs-lookup"><span data-stu-id="a4396-102">Compiler Error CS1558</span></span>
<span data-ttu-id="a4396-103">"klasa" nie posiada odpowiedniej statycznej metody głównej</span><span class="sxs-lookup"><span data-stu-id="a4396-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="a4396-104">[Opcja -main](../language-reference/compiler-options/main-compiler-option.md) kompilator określony klasy, w którym szukać **Main** metody.</span><span class="sxs-lookup"><span data-stu-id="a4396-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="a4396-105">Jednak [Main](../programming-guide/main-and-command-args/index.md) metoda nie została zdefiniowana poprawnie.</span><span class="sxs-lookup"><span data-stu-id="a4396-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="a4396-106">Poniższy przykład generuje CS1558 z powodu nieprawidłowego typu zwracanego.</span><span class="sxs-lookup"><span data-stu-id="a4396-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
