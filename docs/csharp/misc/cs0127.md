# <a name="compiler-error-cs0127"></a>Błąd kompilatora CS0127
Ponieważ funkcja "Function" zwraca wartość void, po słowie kluczowym return nie może następować wyrażenie obiektu  
  
 Metoda z typem zwracanym [void](../language-reference/keywords/void.md) nie może zwracać wartości. Aby uzyskać więcej informacji, zobacz [metody](../programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład generuje CS0127:  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
