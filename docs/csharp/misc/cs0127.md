 <span data-ttu-id="5996f-104">Metoda z [void](../../csharp/language-reference/keywords/void.md) zwracać typ nie może zwracać wartości.</span><span class="sxs-lookup"><span data-stu-id="5996f-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="5996f-105">Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="5996f-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="5996f-106">Poniższy przykład spowoduje wygenerowanie CS0127:</span><span class="sxs-lookup"><span data-stu-id="5996f-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs

   public class MyClass  
   {  
      public int hiddenMember2  
      {

            return 0;  
         }

            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
