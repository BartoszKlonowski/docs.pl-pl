# <a name="compiler-warning-level-3-cs0105"></a>Kompilatora (poziom 3) ostrzeżenie CS0105
Za pomocą dyrektywy "namespace" pojawiła się poprzednio w tej przestrzeni nazw  
  
 A [przestrzeni nazw](../../csharp/language-reference/keywords/namespace.md), które powinny być deklarowane tylko po zadeklarowano więcej niż jeden raz; Usuń wszystkie deklaracje zduplikowaną przestrzeń nazw.  
  
 Poniższy przykład spowoduje wygenerowanie CS0105:  
  
```csharp  
// CS0105.cs  
// compile with: /W:3  
using System;  
using System;   // CS0105  
  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
```
