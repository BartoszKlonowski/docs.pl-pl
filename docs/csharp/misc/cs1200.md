# <a name="compiler-warning-level-1-cs1200"></a>Kompilator CS1200 ostrzegawcze (poziom 1)
Funkcja "nieprawidłową funkcję" jest przestarzała. Zamiast tego użyj 'nieprawidłowa funkcja' ".  
  
 Funkcja, do której próbujesz użyć, jest teraz przestarzałe. Zaktualizuj swój kod, aby zamiast tego użyj nieprawidłowa funkcja.
