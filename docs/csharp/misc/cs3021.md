 <span data-ttu-id="0f31a-104">To ostrzeżenie występuje, gdy `[CLSCompliant(false)]` pojawia się na klasie w zestawie, która nie ma ustawiony atrybut CLSCompliant zestawu poziom na wartość true (to znaczy wiersza `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="0f31a-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="0f31a-105">Ponieważ zestaw jest nie deklarowanie się ze specyfikacją CLS zgodne, nie jest nie trzeba zadeklarować się niezgodne, ponieważ zakłada się, jest niezgodny z dla wszystkich elementów w zestawie.</span><span class="sxs-lookup"><span data-stu-id="0f31a-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="0f31a-106">Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="0f31a-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>

## <a name="example"></a><span data-ttu-id="0f31a-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="0f31a-108">Example</span></span>  
 <span data-ttu-id="0f31a-109">Poniższy przykład generuje CS3021:</span><span class="sxs-lookup"><span data-stu-id="0f31a-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
