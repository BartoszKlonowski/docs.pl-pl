# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="e0c9c-103">Ostrzeżenie kompilatora (poziom 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="e0c9c-103">Compiler Warning (level 1) CS0672</span></span>

<span data-ttu-id="e0c9c-104">Składowa "member1" przesłania przestarzałą składową "member2.</span><span class="sxs-lookup"><span data-stu-id="e0c9c-104">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="e0c9c-105">Dodaj przestarzały atrybut do "member1"</span><span class="sxs-lookup"><span data-stu-id="e0c9c-105">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="e0c9c-106">Kompilator znalazł metodę, która `override` jest oznaczona jako `obsolete` .</span><span class="sxs-lookup"><span data-stu-id="e0c9c-106">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="e0c9c-107">Jednak Metoda przesłaniania nie została wcześniej oznaczona jako przestarzała.</span><span class="sxs-lookup"><span data-stu-id="e0c9c-107">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="e0c9c-108">Metoda zastępująca nadal generuje [CS0612](./cs0612.md), jeśli zostanie wywołana.</span><span class="sxs-lookup"><span data-stu-id="e0c9c-108">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="e0c9c-109">Przejrzyj deklaracje metody i jawnie wskaż, czy Metoda (i wszystkie jej zastąpienia) powinna być oznaczona `obsolete` .</span><span class="sxs-lookup"><span data-stu-id="e0c9c-109">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="e0c9c-110">Poniższy przykład generuje CS0672:</span><span class="sxs-lookup"><span data-stu-id="e0c9c-110">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
