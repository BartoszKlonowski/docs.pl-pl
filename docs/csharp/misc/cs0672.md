# <a name="compiler-warning-level-1-cs0672"></a>Kompilatora (poziom 1) ostrzeżenie CS0672
Element członkowski "Członek1" przesłania przestarzałą składową "członek2. Dodaj Obsolete atrybutu 'Członek1'  
  
 Znaleziono kompilatora `override` do metody oznaczone jako `obsolete`. Jednak metoda przesłaniania nie sam jest oznaczony jako przestarzały. Metoda przesłaniania nadal wygeneruje [CS0612](../../csharp/misc/cs0612.md), jeśli wywołano.  
  
 Przejrzyj swoje deklaracje metody, a następnie jawnie wskazać, czy metody (i wszystkich jego zastąpienia) powinna być oznaczona jako `obsolete`.  
  
 Poniższy przykład spowoduje wygenerowanie CS0672:  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
