# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="def03-102">Kompilatora (poziom 1) ostrzeżenie CS0672</span><span class="sxs-lookup"><span data-stu-id="def03-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="def03-103">Element członkowski "Członek1" przesłania przestarzałą składową "członek2.</span><span class="sxs-lookup"><span data-stu-id="def03-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="def03-104">Dodaj Obsolete atrybutu 'Członek1'</span><span class="sxs-lookup"><span data-stu-id="def03-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="def03-105">Znaleziono kompilatora `override` do metody oznaczone jako `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="def03-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="def03-106">Jednak metoda przesłaniania nie sam jest oznaczony jako przestarzały.</span><span class="sxs-lookup"><span data-stu-id="def03-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="def03-107">Metoda przesłaniania nadal wygeneruje [CS0612](../../csharp/misc/cs0612.md), jeśli wywołano.</span><span class="sxs-lookup"><span data-stu-id="def03-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="def03-108">Przejrzyj swoje deklaracje metody, a następnie jawnie wskazać, czy metody (i wszystkich jego zastąpienia) powinna być oznaczona jako `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="def03-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="def03-109">Poniższy przykład spowoduje wygenerowanie CS0672:</span><span class="sxs-lookup"><span data-stu-id="def03-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
