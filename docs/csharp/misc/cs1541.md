# <a name="compiler-error-cs1541"></a>CS1541 błąd kompilatora
Nieprawidłowa opcja odwołania: "symbol" — nie można odwoływać się do katalogów  
  
 Kompilator wykrył próbę określ katalog, a nie do określonego pliku. Na przykład, kiedy używasz [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) — opcja kompilatora, musisz określić plik; nie jest możliwe określić katalog.  
  
 Na przykład przekazanie `/reference:c:\` kompilatorowi wygenerowanie CS1541.
