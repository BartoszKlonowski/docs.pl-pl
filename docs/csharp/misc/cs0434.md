# <a name="compiler-error-cs0434"></a><span data-ttu-id="07487-103">Błąd kompilatora CS0434</span><span class="sxs-lookup"><span data-stu-id="07487-103">Compiler Error CS0434</span></span>

<span data-ttu-id="07487-104">Przestrzeń nazw NamespaceName1 w NamespaceName2 powoduje konflikt z typem TypeName1 w NamespaceName3</span><span class="sxs-lookup"><span data-stu-id="07487-104">The namespace NamespaceName1 in NamespaceName2 conflicts with the type TypeName1 in NamespaceName3</span></span>  
  
 <span data-ttu-id="07487-105">Ten błąd występuje, gdy importowany typ i zaimportowana zagnieżdżona przestrzeń nazw mają tę samą w pełni kwalifikowaną nazwę.</span><span class="sxs-lookup"><span data-stu-id="07487-105">This error occurs when an imported type and an imported nested namespace have the same fully qualified name.</span></span> <span data-ttu-id="07487-106">W przypadku odwołania do tej nazwy kompilator nie może rozróżnić obu.</span><span class="sxs-lookup"><span data-stu-id="07487-106">When that name is referenced, the compiler is unable to distinguish between the two.</span></span> <span data-ttu-id="07487-107">Jeśli można zmienić zaimportowany kod źródłowy, można rozwiązać błąd, zmieniając nazwę typu lub przestrzeni nazw, tak aby oba elementy były unikatowe w obrębie zestawu.</span><span class="sxs-lookup"><span data-stu-id="07487-107">If you can change the imported source code, you can resolve the error by changing the name of either the type or the namespace so that both are unique within the assembly.</span></span>  
  
 <span data-ttu-id="07487-108">Poniższy kod generuje błąd CS0434.</span><span class="sxs-lookup"><span data-stu-id="07487-108">The following code generates error CS0434.</span></span>  
  
## <a name="example"></a><span data-ttu-id="07487-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="07487-109">Example</span></span>  

 <span data-ttu-id="07487-110">Ten kod tworzy pierwszą kopię typu przy użyciu identycznej w pełni kwalifikowanej nazwy.</span><span class="sxs-lookup"><span data-stu-id="07487-110">This code creates the first copy of the type with the identical fully qualified name.</span></span>  
  
```csharp  
// CS0434_1.cs  
// compile with: /t:library  
namespace TypeBindConflicts
{  
    namespace NsImpAggPubImp
    {  
        public class X { }  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="07487-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="07487-111">Example</span></span>
