## <a name="example"></a><span data-ttu-id="ba747-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="ba747-108">Example</span></span>  
 <span data-ttu-id="ba747-109">Poniższy przykład generuje CS1930, ponieważ `num` identyfikator jest używany dla `from` zmiennej zakresu w klauzuli i zmiennej zakresu wprowadzone przez klauzulę. `let`</span><span class="sxs-lookup"><span data-stu-id="ba747-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;
    }  
}  
```
