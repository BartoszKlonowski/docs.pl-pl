# <a name="compiler-error-cs2006"></a>Błąd kompilatora CS2006
Błąd składni wiersza polecenia: Brak opcji "text" elementu "Option"  
  
 Składnia dla *opcji* wymaga dodatkowego tekstu. Aby uzyskać więcej informacji, zobacz [Opcje kompilatora](../language-reference/compiler-options/index.md).
