# <a name="compiler-error-cs2006"></a>CS2006 błąd kompilatora
Błąd składni wiersza polecenia: Brak 'text' dla opcji "option"  
  
 Składnia *opcji* wymaga dodatkowego tekstu. Aby uzyskać informacje, zobacz [— opcje kompilatora](../../csharp/language-reference/compiler-options/index.md).
