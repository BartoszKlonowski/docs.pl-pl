# <a name="compiler-error-cs1101"></a><span data-ttu-id="9a1d5-102">Błąd kompilatora CS1101</span><span class="sxs-lookup"><span data-stu-id="9a1d5-102">Compiler Error CS1101</span></span>
<span data-ttu-id="9a1d5-103">Modyfikatora parametru "ref" nie można używać z elementem "This".</span><span class="sxs-lookup"><span data-stu-id="9a1d5-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="9a1d5-104">`this` Gdy słowo kluczowe modyfikuje pierwszy parametr metody statycznej, sygnalizuje kompilatorowi, że metoda jest metodą rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="9a1d5-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="9a1d5-105">W pierwszym parametrze metody rozszerzenia nie są wymagana żadne inne Modyfikatory ani nie są dozwolone.</span><span class="sxs-lookup"><span data-stu-id="9a1d5-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9a1d5-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="9a1d5-106">Example</span></span>  
 <span data-ttu-id="9a1d5-107">Poniższy przykład generuje CS1101:</span><span class="sxs-lookup"><span data-stu-id="9a1d5-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
