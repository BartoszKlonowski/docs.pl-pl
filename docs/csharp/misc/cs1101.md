# <a name="compiler-error-cs1101"></a><span data-ttu-id="174c5-103">Błąd kompilatora CS1101</span><span class="sxs-lookup"><span data-stu-id="174c5-103">Compiler Error CS1101</span></span>

<span data-ttu-id="174c5-104">Modyfikatora parametru "ref" nie można używać z elementem "This".</span><span class="sxs-lookup"><span data-stu-id="174c5-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="174c5-105">Gdy `this` słowo kluczowe modyfikuje pierwszy parametr metody statycznej, sygnalizuje kompilatorowi, że metoda jest metodą rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="174c5-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="174c5-106">W pierwszym parametrze metody rozszerzenia nie są wymagana żadne inne Modyfikatory ani nie są dozwolone.</span><span class="sxs-lookup"><span data-stu-id="174c5-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="174c5-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="174c5-107">Example</span></span>  

 <span data-ttu-id="174c5-108">Poniższy przykład generuje CS1101:</span><span class="sxs-lookup"><span data-stu-id="174c5-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
