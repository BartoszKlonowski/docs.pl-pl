# <a name="compiler-error-cs1101"></a><span data-ttu-id="66e5f-102">CS1101 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="66e5f-102">Compiler Error CS1101</span></span>
<span data-ttu-id="66e5f-103">Modyfikatora parametru "ref" nie można używać z "this".</span><span class="sxs-lookup"><span data-stu-id="66e5f-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="66e5f-104">Gdy `this` pierwszy parametr metody statycznej modyfikuje słowo kluczowe, sygnalizuje kompilatorowi, metoda jest metodą rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="66e5f-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="66e5f-105">Brak innych modyfikatorów są potrzebne lub je povolen pierwszy parametr metody rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="66e5f-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="66e5f-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="66e5f-106">Example</span></span>  
 <span data-ttu-id="66e5f-107">Poniższy przykład generuje CS1101:</span><span class="sxs-lookup"><span data-stu-id="66e5f-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
