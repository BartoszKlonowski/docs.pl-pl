# <a name="compiler-error-cs0535"></a><span data-ttu-id="55607-102">Błąd kompilatora CS0535</span><span class="sxs-lookup"><span data-stu-id="55607-102">Compiler Error CS0535</span></span>
<span data-ttu-id="55607-103">element "Class" nie implementuje składowej interfejsu "member"</span><span class="sxs-lookup"><span data-stu-id="55607-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="55607-104">[Klasa](../language-reference/keywords/class.md) pochodna z [interfejsu](../language-reference/keywords/interface.md), ale Klasa nie implementuje co najmniej jednego elementu członkowskiego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="55607-104">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="55607-105">Klasa musi implementować wszystkie elementy członkowskie interfejsów, z których dziedziczy lub w przeciwnym razie `abstract`.</span><span class="sxs-lookup"><span data-stu-id="55607-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55607-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="55607-106">Example</span></span>  
 <span data-ttu-id="55607-107">Poniższy przykład generuje CS0535.</span><span class="sxs-lookup"><span data-stu-id="55607-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
