 <span data-ttu-id="ed7cd-104">A [klasy](../../csharp/language-reference/keywords/class.md) pochodną [interfejsu](../../csharp/language-reference/keywords/interface.md), ale klasa nie implementuje jedną lub więcej członków interfejsu.</span><span class="sxs-lookup"><span data-stu-id="ed7cd-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="ed7cd-105">Klasa musi implementować wszystkie elementy członkowskie interfejsów, z których ona pochodzi lub — w przeciwnym razie można zadeklarować `abstract`.</span><span class="sxs-lookup"><span data-stu-id="ed7cd-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ed7cd-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="ed7cd-106">Example</span></span>  
 <span data-ttu-id="ed7cd-107">Poniższy przykład spowoduje wygenerowanie CS0535.</span><span class="sxs-lookup"><span data-stu-id="ed7cd-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
