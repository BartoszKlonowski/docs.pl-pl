# <a name="compiler-error-cs1615"></a><span data-ttu-id="fc4cb-103">Błąd kompilatora CS1615</span><span class="sxs-lookup"><span data-stu-id="fc4cb-103">Compiler Error CS1615</span></span>
<span data-ttu-id="fc4cb-104">Argumentu "number" nie należy przekazywać ze słowem kluczowym "Keyword"</span><span class="sxs-lookup"><span data-stu-id="fc4cb-104">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="fc4cb-105">Jeden z słów kluczowych `ref` lub **out** został użyty, gdy funkcja nie przyjmuje `ref` parametru lub **out** dla tego argumentu.</span><span class="sxs-lookup"><span data-stu-id="fc4cb-105">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="fc4cb-106">Aby rozwiązać ten problem, Usuń nieprawidłowe słowo kluczowe i użyj odpowiedniego słowa kluczowego, które jest zgodne z deklaracją funkcji, jeśli istnieje.</span><span class="sxs-lookup"><span data-stu-id="fc4cb-106">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="fc4cb-107">Poniższy przykład generuje CS1615:</span><span class="sxs-lookup"><span data-stu-id="fc4cb-107">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
