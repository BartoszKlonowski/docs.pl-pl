# <a name="compiler-error-cs0662"></a>CS0662 błąd kompilatora

"method" nie można określić tylko atrybutu Out w parametrze ref. Używać zarówno w i atrybuty lub nie.  
  
 Metoda interfejsu ma parametr, który używa [ref](../../csharp/language-reference/keywords/ref.md) przy użyciu tylko [się](xref:System.Runtime.InteropServices.OutAttribute) atrybutu. A `ref` parametr, który używa **się** atrybutu należy również użyć [w](xref:System.Runtime.InteropServices.InAttribute) atrybutu.  
  
 Poniższy przykład spowoduje wygenerowanie CS0662:  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
