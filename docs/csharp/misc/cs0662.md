<span data-ttu-id="2d4b0-103">"method" nie można określić tylko atrybutu Out w parametrze ref.</span><span class="sxs-lookup"><span data-stu-id="2d4b0-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="2d4b0-104">Używać zarówno w i atrybuty lub nie.</span><span class="sxs-lookup"><span data-stu-id="2d4b0-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="2d4b0-105">Metoda interfejsu ma parametr, który używa [ref](../../csharp/language-reference/keywords/ref.md) przy użyciu tylko [się](xref:System.Runtime.InteropServices.OutAttribute) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="2d4b0-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="2d4b0-106">A `ref` parametr, który używa **się** atrybutu należy również użyć [w](xref:System.Runtime.InteropServices.InAttribute) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="2d4b0-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="2d4b0-107">Poniższy przykład spowoduje wygenerowanie CS0662:</span><span class="sxs-lookup"><span data-stu-id="2d4b0-107">The following sample generates CS0662:</span></span>  
  
```csharp

interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
