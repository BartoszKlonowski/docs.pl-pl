# <a name="compiler-error-cs0662"></a><span data-ttu-id="847c7-102">Błąd kompilatora CS0662</span><span class="sxs-lookup"><span data-stu-id="847c7-102">Compiler Error CS0662</span></span>

<span data-ttu-id="847c7-103">Metoda "Method" nie może określać tylko atrybutu out dla parametru ref.</span><span class="sxs-lookup"><span data-stu-id="847c7-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="847c7-104">Użyj zarówno atrybutów in, jak i out lub żadnego z nich.</span><span class="sxs-lookup"><span data-stu-id="847c7-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="847c7-105">Metoda interfejsu ma parametr, który używa elementu [ref](../language-reference/keywords/ref.md) z tylko atrybutem [out](xref:System.Runtime.InteropServices.OutAttribute) .</span><span class="sxs-lookup"><span data-stu-id="847c7-105">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="847c7-106">Parametr, który używa atrybutu **out** , musi również używać atrybutu [in.](xref:System.Runtime.InteropServices.InAttribute) `ref`</span><span class="sxs-lookup"><span data-stu-id="847c7-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="847c7-107">Poniższy przykład generuje CS0662:</span><span class="sxs-lookup"><span data-stu-id="847c7-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
