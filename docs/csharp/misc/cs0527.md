# <a name="compiler-error-cs0527"></a>CS0527 błąd kompilatora
Typ 'typ' na liście interfejsów nie jest interfejsem  
  
 Możliwe jest [struktury](../../csharp/language-reference/keywords/struct.md) lub [interfejsu](../../csharp/language-reference/keywords/interface.md) dziedziczyć z innego interfejsu, ale nie z innego typu.  
  
 Poniższy przykład spowoduje wygenerowanie CS0527:  
  
```csharp  
// CS0527.cs  
// compile with: /target:library  
public struct clx : int {}   // CS0527 int not an interface  
```
