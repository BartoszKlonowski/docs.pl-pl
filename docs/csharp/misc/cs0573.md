# <a name="compiler-error-cs0573"></a>CS0573 błąd kompilatora
"w polu deklaracji": nie można umieścić inicjatorów pola wystąpienia w strukturach  
  
 Nie można zainicjować pola wystąpienia [struktury](../../csharp/language-reference/keywords/struct.md). Pola typu wartości zostanie zainicjowana w celu ich wartości domyślne, a typ odwołania pola zostanie zainicjowana w celu `null`.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0573:  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```
