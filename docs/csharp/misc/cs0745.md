# <a name="compiler-error-cs0745"></a><span data-ttu-id="ed125-103">Błąd kompilatora CS0745</span><span class="sxs-lookup"><span data-stu-id="ed125-103">Compiler Error CS0745</span></span>
<span data-ttu-id="ed125-104">Oczekiwano kontekstowego słowa kluczowego "by"</span><span class="sxs-lookup"><span data-stu-id="ed125-104">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="ed125-105">Po wzorcu dla `group` klauzuli `group...by` następuje opcjonalne `into` , jak pokazano w następującym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="ed125-105">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>  
  
```csharp

var query = from name in names  
            group name by name[0];  
```  
  
 <span data-ttu-id="ed125-106">lub</span><span class="sxs-lookup"><span data-stu-id="ed125-106">or</span></span>  
  
```csharp  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ed125-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="ed125-107">To correct this error</span></span>  
  
1. <span data-ttu-id="ed125-108">Dodaj `by` słowo kluczowe do `group` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="ed125-108">Add the `by` keyword to the `group` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ed125-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="ed125-109">Example</span></span>  
 <span data-ttu-id="ed125-110">Poniższy kod generuje CS0745:</span><span class="sxs-lookup"><span data-stu-id="ed125-110">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
