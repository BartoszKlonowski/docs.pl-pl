# <a name="compiler-error-cs0745"></a>Błąd kompilatora CS0745
Oczekiwano kontekstowego słowa kluczowego "by"  
  
 `group...by` Po wzorcu dla `group` klauzuli następuje opcjonalne `into`, jak pokazano w następującym przykładzie:  
  
```csharp  
string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
var query = from name in names  
            group name by name[0];  
```  
  
 lub  
  
```csharp  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj słowo kluczowe do `group`klauzuli. `by`  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0745:  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
