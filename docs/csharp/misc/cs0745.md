# <a name="compiler-error-cs0745"></a><span data-ttu-id="720ba-102">Błąd kompilatora CS0745</span><span class="sxs-lookup"><span data-stu-id="720ba-102">Compiler Error CS0745</span></span>

 <span data-ttu-id="720ba-104">Wzorzec dla klauzuli `group` jest `group...by` po którym następuje opcjonalny `into`, jak pokazano w następującym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="720ba-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

```csharp  
var query = from name in names  
            group name by name[0];  
```

```csharp

             //...additional query clauses  
```

## <a name="example"></a><span data-ttu-id="720ba-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="720ba-108">Example</span></span>  
 <span data-ttu-id="720ba-109">Poniższy kod generuje CS0745:</span><span class="sxs-lookup"><span data-stu-id="720ba-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
