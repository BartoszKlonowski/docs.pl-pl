## <a name="example"></a><span data-ttu-id="eb3f2-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="eb3f2-108">Example</span></span>  
 <span data-ttu-id="eb3f2-109">Poniższy kod generuje CS0745:</span><span class="sxs-lookup"><span data-stu-id="eb3f2-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
