# <a name="compiler-error-cs0682"></a>CS0682 błąd kompilatora
"type1" nie może implementować "type2", ponieważ nie jest obsługiwany przez język  
  
 Ten błąd występuje, gdy użytkownik próbuje zaimplementować interfejs w innym języku i kompilator nie obsługuje interfejsu.
