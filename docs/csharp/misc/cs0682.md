# <a name="compiler-error-cs0682"></a>CS0682 błąd kompilatora
"type1" nie może implementować 'Typ2', ponieważ nie jest obsługiwany przez język  
  
 Ten błąd występuje, gdy próbuje zaimplementować interfejs napisany w innym języku, a kompilator nie obsługuje interfejsu.
