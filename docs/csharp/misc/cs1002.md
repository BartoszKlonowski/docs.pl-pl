# <a name="compiler-error-cs1002"></a>Błąd kompilatora CS1002

; oczekuj  
  
 Kompilator wykrył brak średnika. Średnik jest wymagany na końcu każdej instrukcji w języku C#. Instrukcja może obejmować więcej niż jeden wiersz.  
  
 Poniższy przykład generuje CS1002:  
  
```csharp  
// CS1002.cs  
namespace x  
{  
   abstract public class clx  
   {  
      int i   // CS1002, missing semicolon  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
