<span data-ttu-id="400db-104">Nieprawidłowy typ parametru void</span><span class="sxs-lookup"><span data-stu-id="400db-104">Invalid parameter type void</span></span>  
  
 <span data-ttu-id="400db-105">Nie jest konieczne lub nie jest prawidłowy, aby określić parametr [void](../language-reference/builtin-types/void.md) inny niż wskaźnik typu void.</span><span class="sxs-lookup"><span data-stu-id="400db-105">It is not necessary or valid to specify a [void](../language-reference/builtin-types/void.md) parameter other than a void pointer.</span></span>

   {  
   }
