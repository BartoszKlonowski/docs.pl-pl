# <a name="compiler-error-cs1055"></a><span data-ttu-id="e986d-103">Błąd kompilatora CS1055</span><span class="sxs-lookup"><span data-stu-id="e986d-103">Compiler Error CS1055</span></span>

<span data-ttu-id="e986d-104">Oczekiwano metody dostępu Add lub Remove</span><span class="sxs-lookup"><span data-stu-id="e986d-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="e986d-105">Jeśli [zdarzenie](../language-reference/keywords/event.md) nie jest zadeklarowane jako pole, musi definiować zarówno funkcje dostępu **Dodaj** , jak i **Usuń** .</span><span class="sxs-lookup"><span data-stu-id="e986d-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="e986d-106">Poniższy przykład generuje CS1055:</span><span class="sxs-lookup"><span data-stu-id="e986d-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
