# <a name="compiler-error-cs1724"></a>Błąd kompilatora CS1724

Wartość określona dla argumentu dla elementu "System. Runtime. InteropServices. DefaultCharSetAttribute" jest nieprawidłowa  
  
 Ten błąd jest generowany przez nieprawidłowy argument <xref:System.Runtime.InteropServices.DefaultCharSetAttribute> klasy.  
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS1724.  
  
```csharp  
// CS1724.cs  
using System.Runtime.InteropServices;  
// To resolve, replace 42 with a valid CharSet value.  
 [module:DefaultCharSetAttribute((CharSet)42)]   // CS1724  
class C {
  
[DllImport("F.Dll")]  
extern static void FW1Named();  
  
static void Main() {}  
}  
```
