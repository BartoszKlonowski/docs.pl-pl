# <a name="compiler-error-cs0533"></a>CS0533 błąd kompilatora
"Klasa pochodna członek" ukrywa odziedziczonej abstrakcyjnej składowej 'składowa klasy podstawowej'  
  
 Podstawa [klasy](../../csharp/language-reference/keywords/class.md) metoda jest ukryta. Sprawdź składnię deklaracji, aby zobaczyć, czy jest poprawny.  
  
 Aby uzyskać więcej informacji, zobacz [podstawowy](../../csharp/language-reference/keywords/base.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0533:  
  
```csharp  
// CS0533.cs  
namespace x  
{  
   abstract public class a  
   {  
      abstract public void f();  
   }  
  
   abstract public class b : a  
   {  
      new abstract public void f();   // CS0533  
      // try the following lines instead  
      // override public void f()  
      // {  
      // }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
