# <a name="compiler-error-cs0663"></a><span data-ttu-id="cccbb-103">Błąd kompilatora CS0663</span><span class="sxs-lookup"><span data-stu-id="cccbb-103">Compiler Error CS0663</span></span>
<span data-ttu-id="cccbb-104">Nie można definiować przeciążonych metod, które różnią się tylko parametrami ref i out.</span><span class="sxs-lookup"><span data-stu-id="cccbb-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="cccbb-105">Metody, które różnią się jedynie użyciem parametrów [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) i [out](../language-reference/keywords/out-parameter-modifier.md) na parametrze, są niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="cccbb-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="cccbb-106">Poniższy przykład generuje CS0663:</span><span class="sxs-lookup"><span data-stu-id="cccbb-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
