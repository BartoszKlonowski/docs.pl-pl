<span data-ttu-id="cb83f-103">Nie można definiować przeciążonych metod, które różnią się jedynie parametrami ref i out.</span><span class="sxs-lookup"><span data-stu-id="cb83f-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="cb83f-104">Metody, które różnią się tylko na ich użycie [w](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) i [limit](../../csharp/language-reference/keywords/out-parameter-modifier.md) nie są dozwolone w parametrze.</span><span class="sxs-lookup"><span data-stu-id="cb83f-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="cb83f-105">Poniższy przykład generuje CS0663:</span><span class="sxs-lookup"><span data-stu-id="cb83f-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
