# <a name="compiler-error-cs1662"></a><span data-ttu-id="e8318-103">Błąd kompilatora CS1662</span><span class="sxs-lookup"><span data-stu-id="e8318-103">Compiler Error CS1662</span></span>

<span data-ttu-id="e8318-104">Nie można przekonwertować bloku metody anonimowej na typ delegata "typ delegata", ponieważ niektóre typy zwracane w bloku nie są niejawnie konwertowane na typ zwracany delegata</span><span class="sxs-lookup"><span data-stu-id="e8318-104">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="e8318-105">Ten błąd występuje, gdy instrukcja return bloku metody anonimowej ma typ, który nie został niejawnie przekonwertowany na zwracany typ delegata.</span><span class="sxs-lookup"><span data-stu-id="e8318-105">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="e8318-106">Poniższy przykład generuje CS1662:</span><span class="sxs-lookup"><span data-stu-id="e8318-106">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
