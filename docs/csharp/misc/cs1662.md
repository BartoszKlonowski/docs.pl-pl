# <a name="compiler-error-cs1662"></a><span data-ttu-id="d77a3-102">CS1662 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="d77a3-102">Compiler Error CS1662</span></span>
<span data-ttu-id="d77a3-103">Nie można przekonwertować bloku metody anonimowej, aby delegować typu "typ delegata", ponieważ niektóre typy zwracane występujące w bloku nie umożliwiają niejawnej konwersji na typ zwracany delegata</span><span class="sxs-lookup"><span data-stu-id="d77a3-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>  
  
 <span data-ttu-id="d77a3-104">Ten błąd występuje, jeśli instrukcja return w bloku metody anonimowej typu, które nie umożliwiają niejawnej konwersji na zwracany typ delegata.</span><span class="sxs-lookup"><span data-stu-id="d77a3-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>  
  
 <span data-ttu-id="d77a3-105">Poniższy przykład generuje CS1662:</span><span class="sxs-lookup"><span data-stu-id="d77a3-105">The following sample generates CS1662:</span></span>  
  
```csharp  
// CS1662.cs  
  
delegate int MyDelegate(int i);  
  
class C  
{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662  
     // Try this instead:  
     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}  
```
