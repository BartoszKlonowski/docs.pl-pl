# <a name="compiler-error-cs0629"></a><span data-ttu-id="d5bae-102">CS0629 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="d5bae-102">Compiler Error CS0629</span></span>
<span data-ttu-id="d5bae-103">Warunkowy element członkowski "członek" nie może implementować elementu członkowskiego interfejsu "elementu członkowskiego klasy podstawowej" w typie "Nazwa typu"</span><span class="sxs-lookup"><span data-stu-id="d5bae-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="d5bae-104">[Warunkowego](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atrybutu nie można używać w implementacji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="d5bae-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="d5bae-105">Poniższy przykład generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="d5bae-105">The following sample generates CS0629:</span></span>  
  
```  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
