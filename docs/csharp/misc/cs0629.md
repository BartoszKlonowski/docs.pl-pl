# <a name="compiler-error-cs0629"></a><span data-ttu-id="220be-102">Błąd kompilatora CS0629</span><span class="sxs-lookup"><span data-stu-id="220be-102">Compiler error CS0629</span></span>

<span data-ttu-id="220be-103">Element warunkowy "element członkowski" nie może zaimplementować elementu członkowskiego interfejsu "element członkowski klasy podstawowej" w typie "Nazwa typu"</span><span class="sxs-lookup"><span data-stu-id="220be-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="220be-104">Atrybut [warunkowy](../language-reference/attributes/general.md#conditional-attribute) nie może być używany w implementacji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="220be-104">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="220be-105">Następujący przykład generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="220be-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
