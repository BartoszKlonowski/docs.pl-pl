# <a name="compiler-error-cs0009"></a>Błąd kompilatora CS0009

Nie można otworzyć pliku metadanych "File" — "Description"  
  
 Plik określony za pomocą opcji kompilatora [odwołania](../language-reference/compiler-options/reference-compiler-option.md) nie zawiera prawidłowych metadanych.
