# <a name="compiler-error-cs0101"></a>CS0101 błąd kompilatora
Przestrzeń nazw "namespace" zawiera już definicję elementu "type"  
  
 A [przestrzeni nazw](../../csharp/language-reference/keywords/namespace.md) ma zduplikowane identyfikatory. Zmień nazwę lub usunąć jedno ze zduplikowanych identyfikatorów. Aby uzyskać więcej informacji, zobacz [przestrzeni nazw](../../csharp/programming-guide/namespaces/index.md)  
  
 Poniższy przykład spowoduje wygenerowanie CS0101:  
  
```csharp  
// CS0101.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      static public void Main()  
      {  
      }  
   }  
  
   public class MyClass   // CS0101  
   {  
   }  
}  
```
