# <a name="compiler-warning-level-1-cs1202"></a>Ostrzeżenie kompilatora (poziom 1) CS1202
Funkcja "Nieprawidłowa funkcja" jest przestarzała. Użyj "prawidłowej funkcji" zamiast ".  
  
 Funkcja, której próbujesz użyć, jest obecnie przestarzała. Zaktualizuj swój kod, aby zamiast tego użyć prawidłowej funkcji.
