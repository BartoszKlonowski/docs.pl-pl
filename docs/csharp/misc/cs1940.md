<span data-ttu-id="7fcb9-103">Znaleziono wiele implementacji wzorca zapytania dla typu źródła "type".</span><span class="sxs-lookup"><span data-stu-id="7fcb9-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="7fcb9-104">Niejednoznaczne wywołanie "method".</span><span class="sxs-lookup"><span data-stu-id="7fcb9-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="7fcb9-105">Ten błąd jest generowany, gdy wiele implementacji metody zapytania są definiowane i kompilator nie może usunąć niejednoznaczność, która z nich najlepiej jest stosować dla zapytania.</span><span class="sxs-lookup"><span data-stu-id="7fcb9-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="7fcb9-106">W poniższym przykładzie obie wersje `Select` mieć taki sam podpis, ponieważ obie przyjąć jedną `int` jako parametr wejściowy i mieć `int` jako do wartości zwracanej.</span><span class="sxs-lookup"><span data-stu-id="7fcb9-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="7fcb9-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="7fcb9-109">Example</span></span>  
 <span data-ttu-id="7fcb9-110">Poniższy kod generuje CS1940:</span><span class="sxs-lookup"><span data-stu-id="7fcb9-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
