## <a name="example"></a><span data-ttu-id="d07d6-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="d07d6-109">Example</span></span>  
 <span data-ttu-id="d07d6-110">Poniższy kod generuje CS1940:</span><span class="sxs-lookup"><span data-stu-id="d07d6-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
