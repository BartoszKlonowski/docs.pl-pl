 <span data-ttu-id="aa49f-105">Ten błąd jest generowany, gdy wiele implementacji metody zapytania są definiowane i kompilator nie można odróżnić, który z nich najlepiej jest używać dla zapytania.</span><span class="sxs-lookup"><span data-stu-id="aa49f-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="aa49f-106">W poniższym przykładzie, obie wersje `Select` ma taki sam podpis, ponieważ oba akceptują jeden `int` jako parametr wejściowy i `int` jako wartości zwracanej.</span><span class="sxs-lookup"><span data-stu-id="aa49f-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="aa49f-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="aa49f-109">Example</span></span>  
 <span data-ttu-id="aa49f-110">Poniższy kod generuje CS1940:</span><span class="sxs-lookup"><span data-stu-id="aa49f-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
