<span data-ttu-id="32eb8-103">Znaleziono wiele implementacji wzorca zapytania dla typu źródła "Type".</span><span class="sxs-lookup"><span data-stu-id="32eb8-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="32eb8-104">Niejednoznaczne wywołanie metody "Method".</span><span class="sxs-lookup"><span data-stu-id="32eb8-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="32eb8-105">Ten błąd jest generowany, gdy zdefiniowano wiele implementacji metody zapytania, a kompilator nie może odróżnić, które z nich najlepiej użyć dla zapytania.</span><span class="sxs-lookup"><span data-stu-id="32eb8-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="32eb8-106">W poniższym przykładzie obie wersje `Select` mają tę samą sygnaturę, ponieważ oba te elementy akceptują jeden `int` jako parametr wejściowy i mają `int` jako wartość zwracaną.</span><span class="sxs-lookup"><span data-stu-id="32eb8-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="32eb8-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="32eb8-109">Example</span></span>  
 <span data-ttu-id="32eb8-110">Poniższy kod generuje CS1940:</span><span class="sxs-lookup"><span data-stu-id="32eb8-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
