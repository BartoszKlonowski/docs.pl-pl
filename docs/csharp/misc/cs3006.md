# <a name="compiler-warning-level-1-cs3006"></a>Kompilator CS3006 ostrzegawcze (poziom 1)
Przeciążona metoda "method" różniąca się tylko parametrem ref lub out albo rangą tablicy nie jest zgodne ze specyfikacją CLS  
  
 Metoda nie nie może być przeciążane na podstawie [ref](../../csharp/language-reference/keywords/ref.md) lub [limit](../../csharp/language-reference/keywords/out.md) parametru nadal są zgodne z typowych specyfikacji języka (CLS). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3006. Aby usunąć to ostrzeżenie, komentarz atrybut poziomu zestawu, albo usuń jeden z definicjami metod.  
  
```  
// CS3006.cs  
  
using System;  
  
[assembly: CLSCompliant(true)]  
public class MyClass  
{  
    public void f(int i)  
    {  
    }  
  
    public void f(ref int i)   // CS3006  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
