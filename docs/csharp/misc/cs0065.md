# <a name="compiler-error-cs0065"></a><span data-ttu-id="bf58d-102">Błąd kompilatora CS0065</span><span class="sxs-lookup"><span data-stu-id="bf58d-102">Compiler Error CS0065</span></span>

<span data-ttu-id="bf58d-103">"Event": Właściwość zdarzenia musi mieć metody dostępu Add i Remove</span><span class="sxs-lookup"><span data-stu-id="bf58d-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="bf58d-104">Zdarzenie, które nie jest polem, musi mieć obie metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="bf58d-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="bf58d-105">Poniższy przykład generuje CS0065:</span><span class="sxs-lookup"><span data-stu-id="bf58d-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
