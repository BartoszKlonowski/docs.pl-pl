# <a name="compiler-error-cs0065"></a><span data-ttu-id="34385-102">CS0065 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="34385-102">Compiler Error CS0065</span></span>
<span data-ttu-id="34385-103">"event": właściwość zdarzenia musi mieć metody dostępu add i remove</span><span class="sxs-lookup"><span data-stu-id="34385-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="34385-104">Zdarzenie, które nie jest to pole musi mieć obie metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="34385-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="34385-105">Poniższy przykład spowoduje wygenerowanie CS0065:</span><span class="sxs-lookup"><span data-stu-id="34385-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
