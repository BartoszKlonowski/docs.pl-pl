# <a name="compiler-error-cs1511"></a><span data-ttu-id="ac45d-102">CS1511 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="ac45d-102">Compiler Error CS1511</span></span>

 <span data-ttu-id="ac45d-104">[Podstawowy](../../csharp/language-reference/keywords/base.md) użyto słowa kluczowego w [statyczne](../../csharp/language-reference/keywords/static.md) metody.</span><span class="sxs-lookup"><span data-stu-id="ac45d-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="ac45d-105">`base` można wywołać tylko w konstruktora wystąpienia, metoda wystąpienia lub metoda dostępu do wystąpienia.</span><span class="sxs-lookup"><span data-stu-id="ac45d-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ac45d-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="ac45d-106">Example</span></span>  
 <span data-ttu-id="ac45d-107">Poniższy przykład spowoduje wygenerowanie CS1511.</span><span class="sxs-lookup"><span data-stu-id="ac45d-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
