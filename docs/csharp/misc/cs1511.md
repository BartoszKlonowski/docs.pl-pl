# <a name="compiler-error-cs1511"></a>CS1511 błąd kompilatora
Słowo kluczowe "base" nie jest dostępna w statycznej metody  
  
 [Podstawowy](../../csharp/language-reference/keywords/base.md) użyto słowa kluczowego w [statyczne](../../csharp/language-reference/keywords/static.md) metody. `base` można wywołać tylko w konstruktora wystąpienia, metoda wystąpienia lub metoda dostępu do wystąpienia.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1511.  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
