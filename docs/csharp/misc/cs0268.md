# <a name="compiler-error-cs0268"></a>CS0268 błąd kompilatora
Importowany typ "type" jest nieprawidłowy. Zawiera on cykliczną zależność klasy bazowej.  
  
 Ten błąd występuje, jeśli typ zaimportowany z innego języka ma cykliczną zależność klasy bazowej. Taki typ, nie można używać w C# program. Aby rozwiązać ten problem, Sprawdź typy zaimportowane z innych języków w zestawach lub moduły.
