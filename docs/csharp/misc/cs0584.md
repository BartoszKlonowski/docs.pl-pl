# <a name="compiler-error-cs0584"></a>Compiler Error CS0584
Błąd wewnętrzny kompilatora: etap "etap" symbol "symbol"  
  
 Spróbuj określić, czy kompilacja nie kończy się niepowodzeniem, ponieważ nie można przeanalizować nieoczekiwanej składni. Jeśli nie jest tak, spróbuj [Porozmawiaj z nami](/visualstudio/ide/talk-to-us).
