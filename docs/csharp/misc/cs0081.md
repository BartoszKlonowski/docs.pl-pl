# <a name="compiler-error-cs0081"></a><span data-ttu-id="a8bd8-102">CS0081 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="a8bd8-102">Compiler Error CS0081</span></span>
<span data-ttu-id="a8bd8-103">Deklaracja parametru typu musi być identyfikatorem, nie typu</span><span class="sxs-lookup"><span data-stu-id="a8bd8-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="a8bd8-104">Kiedy Deklarujesz metody ogólnej lub typu, należy określić parametr typu jako identyfikator, na przykład "T" lub "inputType".</span><span class="sxs-lookup"><span data-stu-id="a8bd8-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="a8bd8-105">Gdy klient kod wywołuje metodę, dostarcza mu typ, który zamienia każde wystąpienie identyfikatora w metodzie lub klasie treści.</span><span class="sxs-lookup"><span data-stu-id="a8bd8-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="a8bd8-106">Aby uzyskać więcej informacji, zobacz [parametrów typu ogólnego](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="a8bd8-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
