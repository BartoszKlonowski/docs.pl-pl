# <a name="compiler-error-cs0081"></a><span data-ttu-id="94b8c-102">CS0081 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="94b8c-102">Compiler Error CS0081</span></span>
<span data-ttu-id="94b8c-103">Deklaracja parametru typu musi być identyfikatorem nie jest typem</span><span class="sxs-lookup"><span data-stu-id="94b8c-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="94b8c-104">W deklaracji metody ogólnej lub typu, należy określić parametr typu jako identyfikator, na przykład "T" lub "inputType".</span><span class="sxs-lookup"><span data-stu-id="94b8c-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="94b8c-105">Gdy kod klienta wywołuje metodę, dostarcza mu typ, który zamienia każde jej wystąpienie identyfikatora w metodzie lub klasie treści.</span><span class="sxs-lookup"><span data-stu-id="94b8c-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="94b8c-106">Aby uzyskać więcej informacji, zobacz [ogólnych parametrów typu](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="94b8c-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
