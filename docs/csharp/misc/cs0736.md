<span data-ttu-id="abef1-103">element "type name" nie implementuje składowej interfejsu "nazwa elementu członkowskiego".</span><span class="sxs-lookup"><span data-stu-id="abef1-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="abef1-104">Metoda "name" nie może implementować składowej interfejsu, ponieważ jest statyczna.</span><span class="sxs-lookup"><span data-stu-id="abef1-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="abef1-105">Ten błąd jest generowany, gdy metoda statyczna jest niejawnie lub jawnie zadeklarowana jako implementacja elementu członkowskiego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="abef1-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>

## <a name="example"></a><span data-ttu-id="abef1-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="abef1-110">Example</span></span>  
 <span data-ttu-id="abef1-111">Poniższy kod generuje CS0736, ponieważ `Program.testMethod` jest zadeklarowany jako statyczny:</span><span class="sxs-lookup"><span data-stu-id="abef1-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
