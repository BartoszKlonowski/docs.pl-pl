## <a name="example"></a><span data-ttu-id="7b17c-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="7b17c-111">Example</span></span>  

 <span data-ttu-id="7b17c-112">Poniższy kod generuje CS0736, ponieważ `Program.testMethod` jest zadeklarowany jako statyczny:</span><span class="sxs-lookup"><span data-stu-id="7b17c-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
