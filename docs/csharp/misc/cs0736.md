<span data-ttu-id="e20a9-103">'Nazwa typu' nie implementuje składowej interfejsu "elementu członkowskiego name".</span><span class="sxs-lookup"><span data-stu-id="e20a9-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="e20a9-104">"Metoda name" nie może implementować składowej interfejsu ponieważ jest statyczna.</span><span class="sxs-lookup"><span data-stu-id="e20a9-104">'method name' cannot implement an interface member because it is static.</span></span>

## <a name="example"></a><span data-ttu-id="e20a9-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="e20a9-110">Example</span></span>  
 <span data-ttu-id="e20a9-111">Poniższy kod generuje CS0736, ponieważ `Program.testMethod` jest zadeklarowane jako statyczne:</span><span class="sxs-lookup"><span data-stu-id="e20a9-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
