## <a name="example"></a><span data-ttu-id="dab5f-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="dab5f-110">Example</span></span>  
 <span data-ttu-id="dab5f-111">Poniższy kod generuje CS0736, ponieważ `Program.testMethod` jest zadeklarowany jako statyczny:</span><span class="sxs-lookup"><span data-stu-id="dab5f-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
