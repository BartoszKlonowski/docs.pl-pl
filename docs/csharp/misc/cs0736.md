<span data-ttu-id="15947-103">'Nazwa typu' nie implementuje składowej interfejsu "elementu członkowskiego name".</span><span class="sxs-lookup"><span data-stu-id="15947-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="15947-104">"Metoda name" nie może implementować składowej interfejsu ponieważ jest statyczna.</span><span class="sxs-lookup"><span data-stu-id="15947-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="15947-105">Ten błąd jest generowany, gdy metoda statyczna jest jawnie lub niejawnie zadeklarowany jako implementację elementu członkowskiego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="15947-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>

## <a name="example"></a><span data-ttu-id="15947-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="15947-110">Example</span></span>  
 <span data-ttu-id="15947-111">Poniższy kod generuje CS0736, ponieważ `Program.testMethod` jest zadeklarowane jako statyczne:</span><span class="sxs-lookup"><span data-stu-id="15947-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
