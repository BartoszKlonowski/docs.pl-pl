# <a name="compiler-error-cs1044"></a>CS1044 błąd kompilatora
Nie można użyć więcej niż jeden typ w, używając, stałe, lub instrukcji deklaracji  
  
 Nieprawidłowa instrukcja można znaleźć kompilatora.  
  
 Poniższy przykład spowoduje wygenerowanie CS1044:  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
