# <a name="compiler-error-cs0143"></a>CS0143 błąd kompilatora
Typ "class" nie ma zdefiniowanych konstruktorów  
  
 Nie odpowiedniego konstruktora jest dostępna. Jest to w przypadku wartości liczbowej wbudowanych typów, które są inicjowane poprzez przypisywanie wartości do nich.  
  
 Poniższy przykład spowoduje wygenerowanie CS0143:  
  
```csharp  
// CS0143.cs  
class MyClass  
{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
