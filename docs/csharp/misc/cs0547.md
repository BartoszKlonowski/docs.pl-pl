# <a name="compiler-error-cs0547"></a>CS0547 błąd kompilatora
"właściwość": właściwość lub indeksator nie może mieć typu void  
  
 [void](../../csharp/language-reference/keywords/void.md) jest nieprawidłowy, ponieważ wartość zwracana dla właściwości.  
  
 Aby uzyskać więcej informacji, zobacz [właściwości](../../csharp/programming-guide/classes-and-structs/properties.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0547:  
  
```csharp  
// CS0547.cs  
public class a  
{  
   public void i   // CS0547  
   // Try the following declaration instead:  
   // public int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class b : a  
{  
   public static void Main()  
   {  
   }  
}  
```
