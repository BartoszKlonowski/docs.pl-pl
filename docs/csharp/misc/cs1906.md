# <a name="compiler-error-cs1906"></a>CS1906 błąd kompilatora
Nieprawidłowa opcja "opcji"; Widoczność zasobu musi mieć wartość "public" lub "private"  
  
 Ten błąd wskazuje nieprawidłową [/Resource (osadzanie pliku zasobów do danych wyjściowych)](../../csharp/language-reference/compiler-options/resource-compiler-option.md) lub [/linkresource (łącze do zasobów .NET Framework)](../../csharp/language-reference/compiler-options/linkresource-compiler-option.md) opcji wiersza polecenia. Sprawdź składnię **/Resource** lub **/linkresource** wiersza polecenia i upewnij się, że modyfikator dostępności używana jest **publicznych** lub `private`.
