# <a name="compiler-error-cs1906"></a>Błąd kompilatora CS1906
Nieprawidłowa opcja "Option"; Widoczność zasobu musi mieć wartość "Public" lub "Private"  
  
 Ten błąd wskazuje nieprawidłową [/Resource (osadzony plik zasobów do danych wyjściowych)](../language-reference/compiler-options/resource-compiler-option.md) lub [/linkresource (Link do zasobu .NET Framework)](../language-reference/compiler-options/linkresource-compiler-option.md) opcji wiersza polecenia. Sprawdź składnię opcji wiersza polecenia **/Resource** lub **/linkresource** i upewnij się, że używany modyfikator dostępności jest **publiczny** lub `private`.
