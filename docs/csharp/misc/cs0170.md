 <span data-ttu-id="30cd8-104">Pole w strukturze zostało użyte bez uprzedniego zainicjowania.</span><span class="sxs-lookup"><span data-stu-id="30cd8-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="30cd8-105">Aby rozwiązać ten problem, najpierw określ, które pole zostało niezainicjowane, a następnie inicjuj je przed próbą uzyskania do niego dostępu.</span><span class="sxs-lookup"><span data-stu-id="30cd8-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="30cd8-106">Aby uzyskać więcej informacji na temat inicjowania struktur, zobacz [Typy struktury](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="30cd8-106">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="30cd8-107">Poniższy przykład generuje CS0170:</span><span class="sxs-lookup"><span data-stu-id="30cd8-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```
