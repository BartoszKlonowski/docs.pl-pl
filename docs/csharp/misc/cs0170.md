# <a name="compiler-error-cs0170"></a>CS0170 błąd kompilatora
Użycie prawdopodobnie nieprzypisanego pola "field"  
  
 Pole w strukturze został użyty bez uprzedniego inicjowany. Aby rozwiązać ten problem, najpierw określić pole, które zostało niezainicjowany i następnie zainicjowanie jej, zanim spróbujesz uzyskać do niego dostęp. Aby uzyskać więcej informacji na temat inicjowania struktury, zobacz [struktury](../../csharp/programming-guide/classes-and-structs/structs.md) i [przy użyciu struktury](../../csharp/programming-guide/classes-and-structs/using-structs.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0170:  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
