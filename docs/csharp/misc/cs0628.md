# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="46e13-103">Ostrzeżenie kompilatora (poziom 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="46e13-103">Compiler Warning (level 4) CS0628</span></span>

<span data-ttu-id="46e13-104">"member": w klasie zapieczętowanej zadeklarowano nową chronioną składową</span><span class="sxs-lookup"><span data-stu-id="46e13-104">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="46e13-105">Klasa [zapieczętowana](../language-reference/keywords/sealed.md) nie może wprowadzić [chronionego](../language-reference/keywords/protected.md) elementu członkowskiego, ponieważ żadna inna Klasa nie będzie mogła dziedziczyć z `sealed` klasy i korzystać z `protected` elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="46e13-105">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="46e13-106">Poniższy przykład generuje CS0628:</span><span class="sxs-lookup"><span data-stu-id="46e13-106">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
