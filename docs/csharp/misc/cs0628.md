# <a name="compiler-warning-level-4-cs0628"></a>Kompilatora (poziom 4) ostrzeżenie CS0628
'składowa': w klasie zapieczętowanej zadeklarowano nową chronioną składową  
  
 A [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md) klasy nie można przedstawić [chronione](../../csharp/language-reference/keywords/protected.md) elementu członkowskiego, ponieważ klasa nie będzie można dziedziczyć `sealed` klasy i użyć `protected` elementu członkowskiego.  
  
 Poniższy przykład spowoduje wygenerowanie CS0628:  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
