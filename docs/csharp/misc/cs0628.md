# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="8720b-102">Ostrzeżenie kompilatora (poziom 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="8720b-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="8720b-103">"member": w klasie zapieczętowanej zadeklarowano nową chronioną składową</span><span class="sxs-lookup"><span data-stu-id="8720b-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="8720b-104">Klasa [zapieczętowana](../language-reference/keywords/sealed.md) nie może wprowadzić chronionego elementu członkowskiego, ponieważ żadna inna Klasa nie będzie mogła `sealed` dziedziczyć z klasy `protected` i korzystać z elementu członkowskiego. [](../language-reference/keywords/protected.md)</span><span class="sxs-lookup"><span data-stu-id="8720b-104">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="8720b-105">Poniższy przykład generuje CS0628:</span><span class="sxs-lookup"><span data-stu-id="8720b-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
