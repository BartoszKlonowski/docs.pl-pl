# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="7ed72-102">Kompilator CS0628 ostrzegawcze (poziom 4)</span><span class="sxs-lookup"><span data-stu-id="7ed72-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="7ed72-103">"członek": w klasie zapieczętowanej został zadeklarowany nowy chroniony element członkowski</span><span class="sxs-lookup"><span data-stu-id="7ed72-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="7ed72-104">A [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md) klasy nie można przedstawić [chronione](../../csharp/language-reference/keywords/protected.md) elementu członkowskiego, ponieważ klasa nie będzie można dziedziczyć `sealed` klasy i użyj `protected` elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="7ed72-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="7ed72-105">Poniższy przykład generuje CS0628:</span><span class="sxs-lookup"><span data-stu-id="7ed72-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
