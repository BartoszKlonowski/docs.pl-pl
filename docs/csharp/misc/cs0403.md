# <a name="compiler-error-cs0403"></a><span data-ttu-id="62272-102">CS0403 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="62272-102">Compiler Error CS0403</span></span>

 <span data-ttu-id="62272-105">Nie można przypisać wartości null na nieznany typ o nazwie, ponieważ może być typem wartości, która nie zezwala na przypisanie wartości null.</span><span class="sxs-lookup"><span data-stu-id="62272-105">You cannot assign null to the unknown type named because it might be a value type, which does not allow null assignment.</span></span> <span data-ttu-id="62272-106">Jeśli ogólne klasy nie jest przeznaczona do akceptowania typów wartości, należy użyć ograniczenie typu klasy.</span><span class="sxs-lookup"><span data-stu-id="62272-106">If your generic class is not intended to accept value types, use the class type constraint.</span></span> <span data-ttu-id="62272-107">Jeśli może akceptować typy wartości, takie jak typy wbudowane, można zamienić przypisania do wartości null na wyrażenie `default(T)`, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="62272-107">If it can accept value types, such as the built-in types, you may be able to replace the assignment to null with the expression `default(T)`, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="62272-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="62272-108">Example</span></span>  
 <span data-ttu-id="62272-109">Poniższy przykład spowoduje wygenerowanie CS0403.</span><span class="sxs-lookup"><span data-stu-id="62272-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
