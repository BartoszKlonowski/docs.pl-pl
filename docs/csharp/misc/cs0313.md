## <a name="to-correct-this-error"></a><span data-ttu-id="4024c-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4024c-108">To correct this error</span></span>  
  
1. <span data-ttu-id="4024c-109">Przy użyciu kodu, który następuje na przykład, jedno rozwiązanie polega na określeniu zwykłego `ImplStruct` jako argumentu pierwszego typu w `TestMethod`wywołaniu.</span><span class="sxs-lookup"><span data-stu-id="4024c-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="4024c-110">Następnie zmodyfikuj `TestMethod` , aby utworzyć `Implstruct` w instrukcji return wersję dopuszczającą wartość null:</span><span class="sxs-lookup"><span data-stu-id="4024c-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>

  
## <a name="example"></a><span data-ttu-id="4024c-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="4024c-111">Example</span></span>  
 <span data-ttu-id="4024c-112">Poniższy kod generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="4024c-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
