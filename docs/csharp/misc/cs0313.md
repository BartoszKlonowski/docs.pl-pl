## <a name="to-correct-this-error"></a><span data-ttu-id="c2d69-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="c2d69-108">To correct this error</span></span>  
  
1. <span data-ttu-id="c2d69-109">Przy użyciu kodu, który następuje na przykład, jedno rozwiązanie ma określić zwyczajową `ImplStruct` jako pierwszy argument typu w wywołaniu do `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="c2d69-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="c2d69-110">Następnie zmodyfikuj `TestMethod` w celu utworzenia wartości null `Implstruct` w instrukcji return:</span><span class="sxs-lookup"><span data-stu-id="c2d69-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>

  
## <a name="example"></a><span data-ttu-id="c2d69-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="c2d69-111">Example</span></span>

<span data-ttu-id="c2d69-112">Poniższy kod generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="c2d69-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
