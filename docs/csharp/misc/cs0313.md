## <a name="example"></a><span data-ttu-id="9cd87-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="9cd87-111">Example</span></span>  
 <span data-ttu-id="9cd87-112">Poniższy kod generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="9cd87-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
