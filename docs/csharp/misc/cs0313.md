# <a name="compiler-error-cs0313"></a>Błąd kompilatora CS0313
Typ "type1" nie może być używany jako parametr typu "Parameter Name" w typie ogólnym lub metodzie "type2". Typ dopuszczający wartość null "type1" nie spełnia ograniczenia elementu "type2". Typy dopuszczające wartości null nie mogą spełniać żadnych ograniczeń interfejsu.  
  
 Typ dopuszczający wartość null nie jest równoważny z jego odpowiednikiem niedopuszczanym do wartości null. `ImplStruct` W poniższym przykładzie `BaseInterface` spełnia `ImplStruct?` warunekograniczenia`BaseInterface`, ale nie jest to implementowane. `Nullable<ImplStruct>`  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Przy użyciu kodu, który następuje na przykład, jedno rozwiązanie polega na określeniu zwykłego `ImplStruct` jako argumentu pierwszego typu w `TestMethod`wywołaniu. Następnie zmodyfikuj `TestMethod` , aby utworzyć `Implstruct` w instrukcji return wersję dopuszczającą wartość null:  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0313:  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
