# <a name="compiler-warning-level-1-cs2023"></a>Kompilator CS2023 ostrzegawcze (poziom 1)
Opcja/noconfig zostanie zignorowana, ponieważ została ona określona w pliku odpowiedzi  
  
 [/Noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) — opcja kompilatora została określona w pliku odpowiedzi, co jest niedozwolone.  
  
 Nie można pominąć to ostrzeżenie przez [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opcji.
