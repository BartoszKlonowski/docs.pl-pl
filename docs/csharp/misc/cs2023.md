# <a name="compiler-warning-level-1-cs2023"></a>Ostrzeżenie kompilatora (poziom 1) CS2023
Ignorowanie opcji/noconfig, ponieważ została ona określona w pliku odpowiedzi  
  
 W pliku odpowiedzi określono opcję kompilatora [-noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) , co jest niedozwolone.  
  
 To ostrzeżenie nie może być pomijane przez opcję [-nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
