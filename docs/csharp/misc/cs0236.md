# <a name="compiler-error-cs0236"></a>CS0236 błąd kompilatora
Inicjator pola nie może odwoływać się do niestatycznego pola, metody lub właściwości "field"  
  
 Pola wystąpienia nie można zainicjować innych pól wystąpień poza metodą. Jeśli chcesz zainicjować zmienną poza metodą, należy wziąć pod uwagę przeprowadzenia inicjalizacji w konstruktorze klasy. Aby uzyskać więcej informacji, zobacz [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0236:  
  
```csharp  
// CS0236.cs  
public class MyClass  
{  
   public int i = 5;  
   public int j = i;  // CS0236  
   public int k;      // initialize in constructor  
  
   MyClass()  
   {  
      k = i;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
