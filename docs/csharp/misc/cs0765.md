# <a name="compiler-error-cs0765"></a><span data-ttu-id="08d39-102">Błąd kompilatora CS0765</span><span class="sxs-lookup"><span data-stu-id="08d39-102">Compiler Error CS0765</span></span>
<span data-ttu-id="08d39-103">W drzewach wyrażeń nie można używać metod częściowych zawierających tylko deklarację definiującą ani usuniętych metod warunkowych.</span><span class="sxs-lookup"><span data-stu-id="08d39-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="08d39-104">Mimo że wywołanie usuniętej metody częściowej jest wyrażeniem, nie jest to akceptowalne wyrażenie w drzewie wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="08d39-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="08d39-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="08d39-105">To correct this error</span></span>  
  
1. <span data-ttu-id="08d39-106">Dodaj deklarację implementującą dla metody częściowej lub Usuń kod, który powoduje, że metoda warunkowa ma zostać wykluczona z kompilacji.</span><span class="sxs-lookup"><span data-stu-id="08d39-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="08d39-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="08d39-107">Example</span></span>  
 <span data-ttu-id="08d39-108">Poniższy kod generuje CS0765 w dwóch lokalizacjach:</span><span class="sxs-lookup"><span data-stu-id="08d39-108">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
