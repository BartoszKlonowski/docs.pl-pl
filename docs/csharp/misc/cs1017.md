# <a name="compiler-error-cs1017"></a>CS1017 błąd kompilatora
Klauzule catch nie mogą następować po ogólnej klauzuli catch instrukcji try.  
  
 A `catch` blok, który nie przyjmuje żadnych parametrów musi być ostatni szereg `catch` bloków. Aby uzyskać więcej informacji na temat wyjątków, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1017:  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
