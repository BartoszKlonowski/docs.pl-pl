# <a name="compiler-error-cs0060"></a>CS0060 błąd kompilatora
Niespójność dostępności: klasa bazowa 'klasa1' jest mniej dostępna niż klasa 'klasa2'  
  
 Dostępność klas powinny być zgodne między klasy bazowej i odziedziczoną klasę.  
  
 Poniższy przykład spowoduje wygenerowanie CS0060:  
  
```csharp  
// CS0060.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2 : MyClass   // CS0060  
{  
   public static void Main()  
   {  
   }  
}  
```
