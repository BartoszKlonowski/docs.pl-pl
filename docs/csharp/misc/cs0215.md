 <span data-ttu-id="94862-104">Zdefiniowane przez użytkownika [true](../../csharp/language-reference/keywords/true-operator.md) i [false](../../csharp/language-reference/keywords/false-operator.md) operatorów musi mieć typ zwracany [bool](../../csharp/language-reference/keywords/bool.md).</span><span class="sxs-lookup"><span data-stu-id="94862-104">User-defined [true](../../csharp/language-reference/keywords/true-operator.md) and [false](../../csharp/language-reference/keywords/false-operator.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="94862-105">Aby uzyskać więcej informacji, zobacz [operatory z możliwością przeciążenia](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) i [operator](../../csharp/language-reference/keywords/operator.md) artykułu — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="94862-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) and the [operator](../../csharp/language-reference/keywords/operator.md) keyword article.</span></span>  
  
 <span data-ttu-id="94862-106">Poniższy przykład spowoduje wygenerowanie CS0215:</span><span class="sxs-lookup"><span data-stu-id="94862-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
