# <a name="compiler-error-cs0215"></a>CS0215 błąd kompilatora
Typ zwracany operatora True lub False musi być logiczny  
  
 Zdefiniowane przez użytkownika [true i false](../language-reference/operators/true-false-operators.md) operatorów musi mieć typ zwracany [bool](../language-reference/keywords/bool.md). Aby uzyskać więcej informacji, zobacz [operatory z możliwością przeciążenia](../programming-guide/statements-expressions-operators/overloadable-operators.md) i [operator](../language-reference/keywords/operator.md) artykułu — słowo kluczowe.  
  
 Poniższy przykład spowoduje wygenerowanie CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
