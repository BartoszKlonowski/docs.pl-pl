# <a name="compiler-error-cs0215"></a><span data-ttu-id="5df1e-103">Błąd kompilatora CS0215</span><span class="sxs-lookup"><span data-stu-id="5df1e-103">Compiler Error CS0215</span></span>

<span data-ttu-id="5df1e-104">Typ zwracany operatora true lub false musi być typem bool</span><span class="sxs-lookup"><span data-stu-id="5df1e-104">The return type of operator True or False must be bool</span></span>  
  
<span data-ttu-id="5df1e-105">Operatory [true i false](../language-reference/operators/true-false-operators.md) zdefiniowane przez użytkownika muszą mieć zwracany typ [bool](../language-reference/builtin-types/bool.md).</span><span class="sxs-lookup"><span data-stu-id="5df1e-105">User-defined [true and false](../language-reference/operators/true-false-operators.md) operators must have a return type of [bool](../language-reference/builtin-types/bool.md).</span></span>
  
<span data-ttu-id="5df1e-106">Poniższy przykład generuje CS0215:</span><span class="sxs-lookup"><span data-stu-id="5df1e-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
