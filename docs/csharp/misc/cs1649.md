# <a name="compiler-error-cs1649"></a><span data-ttu-id="6efc0-102">CS1649 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="6efc0-102">Compiler Error CS1649</span></span>
<span data-ttu-id="6efc0-103">Składowych pola tylko do odczytu 'Identyfikator' nie mogą być przekazywane ref lub out (z wyjątkiem w konstruktorze)</span><span class="sxs-lookup"><span data-stu-id="6efc0-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="6efc0-104">Ten błąd występuje, jeśli przekażesz zmienną do funkcji, która jest elementem członkowskim `readonly` pola jako `ref` lub `out` argumentu.</span><span class="sxs-lookup"><span data-stu-id="6efc0-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="6efc0-105">Ponieważ `ref` i `out` parametry mogą zostać zmodyfikowane przez funkcję, jest to niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="6efc0-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="6efc0-106">Aby rozwiązać ten problem, Usuń `readonly` słowo kluczowe w polu lub nie przekazuj jej członkowie `readonly` pola funkcji.</span><span class="sxs-lookup"><span data-stu-id="6efc0-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="6efc0-107">Na przykład, możesz spróbować tworzenia zmiennej tymczasowej, które można modyfikować oraz przekazywania tymczasowych jako `ref` argumentu, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="6efc0-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6efc0-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="6efc0-108">Example</span></span>  
 <span data-ttu-id="6efc0-109">Poniższy przykład spowoduje wygenerowanie CS1649:</span><span class="sxs-lookup"><span data-stu-id="6efc0-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
