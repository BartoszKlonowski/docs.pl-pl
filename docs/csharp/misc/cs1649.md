# <a name="compiler-error-cs1649"></a><span data-ttu-id="56f3b-102">Błąd kompilatora CS1649</span><span class="sxs-lookup"><span data-stu-id="56f3b-102">Compiler Error CS1649</span></span>
<span data-ttu-id="56f3b-103">Elementy członkowskie pola tylko do odczytu "identyfikator" nie mogą być przekazywane ref lub out (z wyjątkiem konstruktora)</span><span class="sxs-lookup"><span data-stu-id="56f3b-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="56f3b-104">Ten błąd występuje, jeśli przekazać zmienną do `readonly` funkcji, `ref` która `out` jest członkiem pola jako lub argumentu.</span><span class="sxs-lookup"><span data-stu-id="56f3b-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="56f3b-105">Ponieważ `ref` `out` i parametry mogą być modyfikowane przez funkcję, nie jest to dozwolone.</span><span class="sxs-lookup"><span data-stu-id="56f3b-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="56f3b-106">Aby rozwiązać ten błąd, usuń `readonly` słowo kluczowe w polu lub `readonly` nie należy przekazywać elementów członkowskich pola do funkcji.</span><span class="sxs-lookup"><span data-stu-id="56f3b-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="56f3b-107">Na przykład można spróbować utworzyć zmienną tymczasową, która może `ref` być modyfikowana i przekazywanie tymczasowe jako argument, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="56f3b-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="56f3b-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="56f3b-108">Example</span></span>  
 <span data-ttu-id="56f3b-109">Poniższy przykład generuje CS1649:</span><span class="sxs-lookup"><span data-stu-id="56f3b-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
