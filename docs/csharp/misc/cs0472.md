# <a name="compiler-warning-level-2-cs0472"></a><span data-ttu-id="a4a58-103">Ostrzeżenie kompilatora (poziom 2) CS0472</span><span class="sxs-lookup"><span data-stu-id="a4a58-103">Compiler Warning (level 2) CS0472</span></span>

<span data-ttu-id="a4a58-104">Wynikiem wyrażenia jest zawsze "wartość1", ponieważ wartość typu "wartość2" nigdy nie jest równa wartości "null" typu "wartość3"</span><span class="sxs-lookup"><span data-stu-id="a4a58-104">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>  
  
 <span data-ttu-id="a4a58-105">Kompilator powinien ostrzegać, jeśli używasz operatora ze stałą wartością null.</span><span class="sxs-lookup"><span data-stu-id="a4a58-105">The compiler should warn if you use an operator with a constant null value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a4a58-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="a4a58-106">Example</span></span>  

 <span data-ttu-id="a4a58-107">Poniższy przykład generuje CS0472.</span><span class="sxs-lookup"><span data-stu-id="a4a58-107">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
