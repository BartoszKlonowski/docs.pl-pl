# <a name="compiler-error-cs1617"></a>CS1617 błąd kompilatora
Nieprawidłowa opcja "opcji" dla przełącznika/langversion; musi być ISO-1, ISO-2 lub domyślny  
  
 Ten błąd występuje, jeśli użyto [/langversion](../../csharp/language-reference/compiler-options/langversion-compiler-option.md) ustawienie przełącznika lub projekt wiersza polecenia, ale nie określono opcji obowiązujący język. Aby rozwiązać ten problem, sprawdź ustawienia projektu lub składnia wiersza polecenia i zmień go na jeden z wyświetlonych opcji.  
  
 Na przykład kompilowania za pomocą `csc /langversion:ISO` wygeneruje błąd CS1617.
