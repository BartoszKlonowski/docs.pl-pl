 <span data-ttu-id="7c871-104">Na liście dziedziczenia [interfejsu](../../csharp/language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samej siebie.</span><span class="sxs-lookup"><span data-stu-id="7c871-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="7c871-105">Interfejs nie może dziedziczyć po sobie samej.</span><span class="sxs-lookup"><span data-stu-id="7c871-105">An interface cannot inherit from itself.</span></span>

   public interface a  
   {  
   }  
  
   public interface b : a, c

   }
