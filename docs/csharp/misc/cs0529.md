# <a name="compiler-error-cs0529"></a>CS0529 błąd kompilatora
Dziedziczony interfejs "interface1" powoduje cykl w hierarchii interfejsów "interface2"  
  
 Na liście dziedziczenia dla [interfejsu](../../csharp/language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samego siebie. Interfejs nie może dziedziczyć po sobie samym.  
  
 Poniższy przykład spowoduje wygenerowanie CS0529:  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
