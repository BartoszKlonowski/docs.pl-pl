<span data-ttu-id="194d3-103">Dziedziczony interfejs "interface1" powoduje cykl w hierarchii interfejsów "interface2"</span><span class="sxs-lookup"><span data-stu-id="194d3-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="194d3-104">Na liście dziedziczenia [interfejsu](../../csharp/language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samej siebie.</span><span class="sxs-lookup"><span data-stu-id="194d3-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="194d3-105">Interfejs nie może dziedziczyć po sobie samej.</span><span class="sxs-lookup"><span data-stu-id="194d3-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="194d3-106">Poniższy przykład generuje CS0529:</span><span class="sxs-lookup"><span data-stu-id="194d3-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
