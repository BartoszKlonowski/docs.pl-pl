 <span data-ttu-id="cf03c-104">Lista dziedziczenia dla [interfejsu](../language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samego siebie.</span><span class="sxs-lookup"><span data-stu-id="cf03c-104">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="cf03c-105">Interfejs nie może dziedziczyć po samym sobie.</span><span class="sxs-lookup"><span data-stu-id="cf03c-105">An interface cannot inherit from itself.</span></span>

```csharp  
// CS0529.cs

{  
   public interface a  
   {  
   }  
  
   public interface b : a, c

   {  
   }  
}  
```
