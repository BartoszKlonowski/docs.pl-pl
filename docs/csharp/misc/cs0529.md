# <a name="compiler-error-cs0529"></a><span data-ttu-id="ad3d3-102">CS0529 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="ad3d3-102">Compiler Error CS0529</span></span>
<span data-ttu-id="ad3d3-103">Dziedziczony interfejs "interface1" powoduje cykl w hierarchii interfejsów "interface2"</span><span class="sxs-lookup"><span data-stu-id="ad3d3-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="ad3d3-104">Na liście dziedziczenia dla [interfejsu](../../csharp/language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samego siebie.</span><span class="sxs-lookup"><span data-stu-id="ad3d3-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="ad3d3-105">Interfejs nie może dziedziczyć po sobie samym.</span><span class="sxs-lookup"><span data-stu-id="ad3d3-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="ad3d3-106">Poniższy przykład spowoduje wygenerowanie CS0529:</span><span class="sxs-lookup"><span data-stu-id="ad3d3-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
