<span data-ttu-id="33aa7-104">Dziedziczony interfejs ", interfejs1" powoduje cykl w hierarchii interfejsów elementu "interface2"</span><span class="sxs-lookup"><span data-stu-id="33aa7-104">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="33aa7-105">Lista dziedziczenia dla [interfejsu](../language-reference/keywords/interface.md) zawiera bezpośrednie lub pośrednie odwołanie do samego siebie.</span><span class="sxs-lookup"><span data-stu-id="33aa7-105">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="33aa7-106">Interfejs nie może dziedziczyć po samym sobie.</span><span class="sxs-lookup"><span data-stu-id="33aa7-106">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="33aa7-107">Poniższy przykład generuje CS0529:</span><span class="sxs-lookup"><span data-stu-id="33aa7-107">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
