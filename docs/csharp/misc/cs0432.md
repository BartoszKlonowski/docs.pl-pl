# <a name="compiler-error-cs0432"></a>CS0432 błąd kompilatora
'Identyfikator' nie znaleziono aliasu  
  
 Ten błąd występuje, gdy używasz "::" po prawej stronie identyfikator, który nie jest aliasem. Aby naprawić błąd, należy użyć "." zamiast niego.  
  
 Poniższy przykład generuje CS0432:  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
