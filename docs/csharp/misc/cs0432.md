# <a name="compiler-error-cs0432"></a>Błąd kompilatora CS0432
Nie znaleziono aliasu "identifier"  
  
 Ten błąd występuje, gdy używasz "::" z prawej strony identyfikatora, który nie jest aliasem. Aby rozwiązać ten problem, użyj "." zamiast tego.  
  
 Poniższy przykład generuje CS0432:  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
