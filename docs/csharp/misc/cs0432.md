# <a name="compiler-error-cs0432"></a><span data-ttu-id="131d8-102">CS0432 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="131d8-102">Compiler Error CS0432</span></span>
<span data-ttu-id="131d8-103">Alias 'Identyfikator' nie znaleziono</span><span class="sxs-lookup"><span data-stu-id="131d8-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="131d8-104">Ten błąd występuje, gdy używasz "::" z prawej strony identyfikator, który nie jest alias.</span><span class="sxs-lookup"><span data-stu-id="131d8-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="131d8-105">Aby rozwiązać problem, należy użyć "." zamiast niego.</span><span class="sxs-lookup"><span data-stu-id="131d8-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="131d8-106">Poniższy przykład generuje CS0432:</span><span class="sxs-lookup"><span data-stu-id="131d8-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
