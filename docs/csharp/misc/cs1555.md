# <a name="compiler-error-cs1555"></a>CS1555 błąd kompilatora
Nie można odnaleźć określonego dla metody Main "class"  
  
 Klasa została określona w celu [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora, ale nazwa klasy nie został znaleziony w kodzie źródłowym.
