# <a name="compiler-error-cs1555"></a>CS1555 błąd kompilatora
Nie można odnaleźć "class" określonego dla metody Main  
  
 Klasa został określony w [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora, ale nazwa klasy nie można odnaleźć w kodzie źródłowym.
