# <a name="compiler-error-cs1555"></a>Błąd kompilatora CS1555

Nie można znaleźć elementu "Class" określonego dla metody Main  
  
 Klasa została określona jako opcja kompilatora [-Main](../language-reference/compiler-options/main-compiler-option.md) , ale w kodzie źródłowym nie znaleziono nazwy klasy.
