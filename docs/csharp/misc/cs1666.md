# <a name="compiler-error-cs1666"></a><span data-ttu-id="84fc0-102">CS1666 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="84fc0-102">Compiler Error CS1666</span></span>
<span data-ttu-id="84fc0-103">Nie można użyć buforów o ustalonym rozmiarze zawartych w wyrażeniach unfixed.</span><span class="sxs-lookup"><span data-stu-id="84fc0-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="84fc0-104">Spróbuj użyć instrukcji fixed.</span><span class="sxs-lookup"><span data-stu-id="84fc0-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="84fc0-105">Ten błąd występuje, jeśli używasz stały rozmiar buforu w wyrażeniu dotyczące klasy, który nie jest stała.</span><span class="sxs-lookup"><span data-stu-id="84fc0-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="84fc0-106">Środowisko uruchomieniowe jest bezpłatna do przenoszenia Niepoprawione klasy do optymalizacji dostępu do pamięci, co może prowadzić do błędów, gdy za pomocą stały rozmiar buforu.</span><span class="sxs-lookup"><span data-stu-id="84fc0-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="84fc0-107">Aby uniknąć tego błędu, należy użyć `fixed` — słowo kluczowe w instrukcji.</span><span class="sxs-lookup"><span data-stu-id="84fc0-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="84fc0-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="84fc0-108">Example</span></span>  
 <span data-ttu-id="84fc0-109">Poniższy przykład spowoduje wygenerowanie CS1666.</span><span class="sxs-lookup"><span data-stu-id="84fc0-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
