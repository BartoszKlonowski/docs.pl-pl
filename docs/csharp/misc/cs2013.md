# <a name="compiler-error-cs2013"></a>CS2013 błąd kompilatora
Nieprawidłowy numer podstawowy obrazu "value"  
  
 Nieprawidłowa wartość (nie liczba) został przekazany do [/baseAddress](../../csharp/language-reference/compiler-options/baseaddress-compiler-option.md) — opcja kompilatora.  
  
 Poniższy przykład spowoduje wygenerowanie CS2013:  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
