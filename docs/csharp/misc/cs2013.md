# <a name="compiler-error-cs2013"></a>CS2013 błąd kompilatora
Nieprawidłowy numer podstawowy obrazu "wartość"  
  
 Nieprawidłowa wartość (nie liczba) została przekazana do [/baseAddress](../../csharp/language-reference/compiler-options/baseaddress-compiler-option.md) — opcja kompilatora.  
  
 Poniższy przykład generuje CS2013:  
  
```  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
