# <a name="compiler-error-cs1015"></a>CS1015 błąd kompilatora
Obiekt, oczekiwano typu class lub ciągu  
  
 Próbowano przekazać typu wstępnie zdefiniowanymi danymi na [catch](../../csharp/language-reference/keywords/try-catch.md) bloku. Tylko typy danych, które wynikają z <xref:System.Exception?displayProperty=nameWithType> mogą być przekazywane do `catch` bloku. Aby uzyskać więcej informacji na temat wyjątków, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1015:  
  
```csharp  
// CS1015.cs  
class Sample  
{  
    static void Main()  
    {  
        try   
        {  
        }  
        catch(int)   // CS1015, int is not derived from System.Exception  
        {  
        }  
    }  
}  
```
