# <a name="compiler-error-cs0205"></a>CS0205 błąd kompilatora
Nie można wywołać abstrakcyjnej, bazowej składowej: "method"  
  
 Nie można wywołać [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) metody, ponieważ nie ma treści metody. Aby uzyskać więcej informacji, zobacz [abstrakcyjnych i zapieczętowanych klas i składowych klasy](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0205:  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
