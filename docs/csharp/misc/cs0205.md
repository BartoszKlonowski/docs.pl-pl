 <span data-ttu-id="00eb0-106">Poniższy przykład generuje CS0205:</span><span class="sxs-lookup"><span data-stu-id="00eb0-106">The following sample generates CS0205:</span></span>

```csharp  
   abstract public void M();  
}

   {  
      base.M();   // CS0205, delete this line  
   }

   {  
   }  
}  
```
