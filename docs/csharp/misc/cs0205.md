 <span data-ttu-id="ef89e-106">Poniższy przykład generuje CS0205:</span><span class="sxs-lookup"><span data-stu-id="ef89e-106">The following sample generates CS0205:</span></span>  
  
```csharp

abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
