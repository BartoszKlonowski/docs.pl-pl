# <a name="compiler-warning-level-3-cs0219"></a>Kompilatora (poziom 3) ostrzeżenie CS0219
Przypisano zmienną "Zmienna", ale jego wartość nie jest nigdy używana  
  
 Kompilator generuje ostrzeżenia poziomu 3, jeśli zadeklarować i przypisać do zmiennej, ale nie należy jej używać.  
  
 Poniższy przykład spowoduje wygenerowanie CS0219:  
  
```csharp  
// CS0219.cs  
// compile with: /W:3  
public class MyClass  
{  
   public static void Main()  
   {  
      int a = 0;   // CS0219  
   }  
}  
```
