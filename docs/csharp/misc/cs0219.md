# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="7a9d1-103">Ostrzeżenie kompilatora (poziom 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="7a9d1-103">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="7a9d1-104">Zmienna "zmienna" jest przypisana, ale jej wartość nigdy nie jest używana</span><span class="sxs-lookup"><span data-stu-id="7a9d1-104">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="7a9d1-105">Kompilator generuje ostrzeżenie poziomu-3, gdy deklarujesz i przypisujesz zmienną, ale nie korzystasz z niej.</span><span class="sxs-lookup"><span data-stu-id="7a9d1-105">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="7a9d1-106">Kompilator generuje to ostrzeżenie tylko wtedy, gdy wartość zmiennej jest stałą czasu kompilacji.</span><span class="sxs-lookup"><span data-stu-id="7a9d1-106">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="7a9d1-107">Przypisanie niestałego wyrażenia lub wyniku metody do zmiennej lokalnej ułatwia obserwowanie tych wyrażeń w debugerze.</span><span class="sxs-lookup"><span data-stu-id="7a9d1-107">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="7a9d1-108">Powoduje również, że wynik jest osiągalny, uniemożliwiając wyrzucanie elementów bezużytecznych, gdy ta zmienna jest osiągalna.</span><span class="sxs-lookup"><span data-stu-id="7a9d1-108">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="7a9d1-109">W poniższym przykładzie pokazano przypadki, kiedy i kiedy nie jest generowane ostrzeżenie:</span><span class="sxs-lookup"><span data-stu-id="7a9d1-109">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
