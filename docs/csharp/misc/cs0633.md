 <span data-ttu-id="d43a0-105">Każdy argument przekazywany do <xref:System.Diagnostics.ConditionalAttribute> lub <xref:System.Runtime.CompilerServices.IndexerNameAttribute> atrybuty musi być prawidłowym identyfikatorem.</span><span class="sxs-lookup"><span data-stu-id="d43a0-105">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="d43a0-106">Oznacza to, że nie może zawierać znaków, takich jak "+", które są niedozwolone w przypadku wystąpienia w identyfikatorach.</span><span class="sxs-lookup"><span data-stu-id="d43a0-106">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d43a0-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d43a0-107">Example</span></span>  

 <span data-ttu-id="d43a0-108">Ten przykład ilustruje CS0633 przy użyciu <xref:System.Diagnostics.ConditionalAttribute> .</span><span class="sxs-lookup"><span data-stu-id="d43a0-108">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="d43a0-109">Poniższy przykład generuje CS0633.</span><span class="sxs-lookup"><span data-stu-id="d43a0-109">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="d43a0-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="d43a0-110">Example</span></span>  

 <span data-ttu-id="d43a0-111">Ten przykład ilustruje CS0633 przy użyciu <xref:System.Runtime.CompilerServices.IndexerNameAttribute> .</span><span class="sxs-lookup"><span data-stu-id="d43a0-111">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
