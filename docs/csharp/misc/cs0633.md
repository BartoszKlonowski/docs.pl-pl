## <a name="example"></a><span data-ttu-id="a2d19-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="a2d19-106">Example</span></span>  
 <span data-ttu-id="a2d19-107">W tym przykładzie przedstawiono przy użyciu CS0633 <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="a2d19-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="a2d19-108">Poniższy przykład generuje CS0633.</span><span class="sxs-lookup"><span data-stu-id="a2d19-108">The following sample generates CS0633.</span></span>  
  
```csharp

using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="a2d19-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="a2d19-109">Example</span></span>  
 <span data-ttu-id="a2d19-110">W tym przykładzie przedstawiono przy użyciu CS0633 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="a2d19-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
