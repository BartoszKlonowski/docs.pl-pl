# <a name="compiler-error-cs0633"></a>CS0633 błąd kompilatora
Argument atrybutu "attribute" musi być prawidłowym identyfikatorem  
  
 Dowolnego argumentu, który jest przekazywany do <xref:System.Diagnostics.ConditionalAttribute> lub <xref:System.Runtime.CompilerServices.IndexerNameAttribute> atrybutów musi być prawidłowym identyfikatorem. Oznacza to, że nie może zawierać znaków takich jak "+" są niedozwolone, jeśli występują one w identyfikatorach.  
  
## <a name="example"></a>Przykład  
 Ten przykład ilustruje użycie CS0633 <xref:System.Diagnostics.ConditionalAttribute>. Poniższy przykład spowoduje wygenerowanie CS0633.  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a>Przykład  
 Ten przykład ilustruje użycie CS0633 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
