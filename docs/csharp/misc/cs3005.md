# <a name="compiler-warning-level-1-cs3005"></a>Kompilator CS3005 ostrzegawcze (poziom 1)
Identyfikator 'Identyfikator' różni się tylko wielkością liter nie jest zgodne ze specyfikacją CLS  
  
 A [publicznego](../../csharp/language-reference/keywords/public.md), [chronione](../../csharp/language-reference/keywords/protected.md), lub `protected internal` identyfikator, który różni się od innego `public`, `protected`, lub `protected internal` jest identyfikator tylko w przypadku co najmniej jednej litery nie są zgodne z typowych specyfikacji języka (CLS). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3003:  
  
```csharp  
// CS3005.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
