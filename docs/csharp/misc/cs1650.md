# <a name="compiler-error-cs1650"></a><span data-ttu-id="7ce04-102">CS1650 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="7ce04-102">Compiler Error CS1650</span></span>
<span data-ttu-id="7ce04-103">Pól pola statycznego tylko do odczytu "identyfikator" nie można przypisać do (z wyjątkiem w konstruktorze statycznym lub inicjatorze zmiennych)</span><span class="sxs-lookup"><span data-stu-id="7ce04-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="7ce04-104">Ten błąd występuje, gdy spróbujesz zmodyfikować to element członkowski pola, która jest tylko do odczytu i static którym nie jest dozwolone do zmodyfikowania.</span><span class="sxs-lookup"><span data-stu-id="7ce04-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="7ce04-105">Aby rozwiązać ten problem, należy ograniczyć przypisania do pól tylko do odczytu w konstruktorze lub inicjatorze zmiennych lub usuń `readonly` słowo kluczowe z deklarację pola.</span><span class="sxs-lookup"><span data-stu-id="7ce04-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
