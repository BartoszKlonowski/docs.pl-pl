# <a name="compiler-error-cs0589"></a>Błąd kompilatora CS0589

Wewnętrzny błąd kompilatora: etap "PARSE"

 Spróbuj określić, czy kompilator kończy się niepowodzeniem z powodu niemożności przeanalizowania nieoczekiwanej składni. Jeśli tak się nie dzieje, spróbuj [skontaktować się z nami](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
