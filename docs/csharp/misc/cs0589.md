# <a name="compiler-error-cs0589"></a>Compiler Error CS0589
Błąd wewnętrzny kompilatora: etap "PARSE"  
  
 Spróbuj określić, jeśli kończy się niepowodzeniem z powodu jego brakiem, aby przeanalizować Nieoczekiwana składnia kompilator. Jeśli nie jest tak, spróbuj [Porozmawiaj z nami](/visualstudio/ide/talk-to-us).
