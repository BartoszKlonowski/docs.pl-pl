# <a name="compiler-error-cs0589"></a>Błąd kompilatora CS0589

Błąd kompilatora wewnętrznego: etap "PARSE"

 Spróbuj ustalić, czy kompilator nie działa z powodu jego niemożności przeanalizowania nieoczekiwanej składni. Jeśli tak nie jest, spróbuj [porozmawiać z nami](/visualstudio/ide/feedback-options).

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
