# <a name="compiler-error-cs1025"></a>CS1025 błąd kompilatora
Jednowierszowego komentarza lub oczekiwano końcowego elementu wiersza  
  
 Wiersz z [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md) nie może mieć komentarz wielowierszowy.  
  
 Poniższy przykład spowoduje wygenerowanie CS1025:  
  
```csharp  
#if true /* hello  
*/   // CS1025  
#endif   // this is a good comment  
```  
  
 CS1025 mógł także wystąpić, Jeśli spróbujesz nieprawidłowy dyrektywy preprocesora, w następujący sposób:  
  
```csharp  
// CS1025.cs  
#define a  
  
class Sample  
{  
   static void Main()  
   {  
      #if a 1   // CS1025, invalid syntax  
         System.Console.WriteLine("Hello, World!");  
      #endif  
   }  
}  
```
