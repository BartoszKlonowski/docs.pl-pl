# <a name="compiler-error-cs1947"></a><span data-ttu-id="6924f-102">CS1947 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="6924f-102">Compiler Error CS1947</span></span>

 <span data-ttu-id="6924f-104">Zmienna zakresu jest podobna do zmiennej iteracji w `foreach` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="6924f-104">A range variable is like an iteration variable in a `foreach` statement.</span></span> <span data-ttu-id="6924f-105">Nie można przypisać do w wyrażeniu zapytania.</span><span class="sxs-lookup"><span data-stu-id="6924f-105">It cannot be assigned to in a query expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6924f-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="6924f-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="6924f-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="6924f-109">Example</span></span>  
 <span data-ttu-id="6924f-110">Poniższy kod generuje CS1947:</span><span class="sxs-lookup"><span data-stu-id="6924f-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
