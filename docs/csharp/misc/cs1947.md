## <a name="example"></a><span data-ttu-id="b9e08-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="b9e08-109">Example</span></span>  
 <span data-ttu-id="b9e08-110">Poniższy kod generuje CS1947:</span><span class="sxs-lookup"><span data-stu-id="b9e08-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
