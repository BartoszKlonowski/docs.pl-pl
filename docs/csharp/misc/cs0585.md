# <a name="compiler-error-cs0585"></a>Błąd kompilatora CS0585

Wewnętrzny błąd kompilatora: etap "etap"

 Spróbuj określić, czy kompilator kończy się niepowodzeniem z powodu niemożności przeanalizowania nieoczekiwanej składni. Jeśli tak się nie dzieje, spróbuj [skontaktować się z nami](/visualstudio/ide/feedback-options).

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
