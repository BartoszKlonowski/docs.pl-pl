 <span data-ttu-id="a73ee-104">Typ zwracany dla zdefiniowanej przez użytkownika [operator](../../csharp/language-reference/keywords/operator.md) musi występować przed operator — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="a73ee-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>
