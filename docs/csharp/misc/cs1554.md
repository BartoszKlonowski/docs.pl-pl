<span data-ttu-id="3dee8-105">Zwracany typ [operatora przeciążonego](../language-reference/operators/operator-overloading.md) musi występować przed `operator` słowem kluczowym.</span><span class="sxs-lookup"><span data-stu-id="3dee8-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
