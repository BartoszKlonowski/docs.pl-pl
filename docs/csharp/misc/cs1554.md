<span data-ttu-id="ac96d-104">Deklaracja jest nieprawidłowa; Użyj \<type> operatora "operator operacji (..." INSTEAD</span><span class="sxs-lookup"><span data-stu-id="ac96d-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="ac96d-105">Zwracany typ [operatora przeciążonego](../language-reference/operators/operator-overloading.md) musi występować przed `operator` słowem kluczowym.</span><span class="sxs-lookup"><span data-stu-id="ac96d-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
