# <a name="compiler-error-cs1554"></a>CS1554 błąd kompilatora
Deklaracja nie jest prawidłowa; Użyj "\<typ > operator op (..." Zamiast tego  
  
 Typ zwracany dla zdefiniowanej przez użytkownika [operator](../../csharp/language-reference/keywords/operator.md) musi występować przed operator — słowo kluczowe.  
  
 Poniższy przykład spowoduje wygenerowanie CS1554:  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
