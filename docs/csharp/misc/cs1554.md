# <a name="compiler-error-cs1554"></a><span data-ttu-id="409f4-102">CS1554 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="409f4-102">Compiler Error CS1554</span></span>
<span data-ttu-id="409f4-103">Deklaracja nie jest prawidłowa; Użyj "\<typ > operator op (..." Zamiast tego</span><span class="sxs-lookup"><span data-stu-id="409f4-103">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="409f4-104">Zwracany typ [Przeciążony operator](../../csharp/language-reference/operators/operator-overloading.md) musi występować przed `operator` — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="409f4-104">The return type of an [overloaded operator](../../csharp/language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>
  
<span data-ttu-id="409f4-105">Poniższy przykład spowoduje wygenerowanie CS1554:</span><span class="sxs-lookup"><span data-stu-id="409f4-105">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
