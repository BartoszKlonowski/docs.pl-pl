# <a name="compiler-error-cs0072"></a><span data-ttu-id="bd8ed-102">Błąd kompilatora CS0072</span><span class="sxs-lookup"><span data-stu-id="bd8ed-102">Compiler Error CS0072</span></span>
<span data-ttu-id="bd8ed-103">"Event": nie można przesłonić; "Metoda" nie jest zdarzeniem</span><span class="sxs-lookup"><span data-stu-id="bd8ed-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="bd8ed-104">[Zdarzenie](../language-reference/keywords/event.md) może zastąpić tylko inne zdarzenie.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-104">An [event](../language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="bd8ed-105">Aby uzyskać więcej informacji, zobacz [zdarzenia](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="bd8ed-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="bd8ed-106">Poniższy przykład generuje CS0072:</span><span class="sxs-lookup"><span data-stu-id="bd8ed-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
