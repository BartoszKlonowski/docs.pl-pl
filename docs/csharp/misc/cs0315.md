## <a name="example"></a><span data-ttu-id="05825-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="05825-108">Example</span></span>  
 <span data-ttu-id="05825-109">Poniższy przykład generuje CS0315:</span><span class="sxs-lookup"><span data-stu-id="05825-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}

{  
    public static int Main()

    }
```  
