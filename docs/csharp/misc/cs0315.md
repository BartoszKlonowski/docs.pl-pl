# <a name="compiler-error-cs0315"></a><span data-ttu-id="707e1-102">Błąd kompilatora CS0315</span><span class="sxs-lookup"><span data-stu-id="707e1-102">Compiler Error CS0315</span></span>
<span data-ttu-id="707e1-103">Typ "ValueType" nie może być używany jako parametr typu "t" w typie ogólnym lub metodzie\<"TypeorMethod T >".</span><span class="sxs-lookup"><span data-stu-id="707e1-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="707e1-104">Nie istnieje konwersja z opakowania "valueType" na "ReferenceType".</span><span class="sxs-lookup"><span data-stu-id="707e1-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="707e1-105">Ten błąd występuje w przypadku ograniczenia typu ogólnego do określonej klasy i próby konstruowania wystąpienia tej klasy przy użyciu typu wartości, który nie może być niejawnie opakowany do niego.</span><span class="sxs-lookup"><span data-stu-id="707e1-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="707e1-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="707e1-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="707e1-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="707e1-108">Example</span></span>  
 <span data-ttu-id="707e1-109">Poniższy przykład generuje CS0315:</span><span class="sxs-lookup"><span data-stu-id="707e1-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
