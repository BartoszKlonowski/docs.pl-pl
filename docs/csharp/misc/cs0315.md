# <a name="compiler-error-cs0315"></a><span data-ttu-id="ba9c9-102">Błąd kompilatora CS0315</span><span class="sxs-lookup"><span data-stu-id="ba9c9-102">Compiler Error CS0315</span></span>

 <span data-ttu-id="ba9c9-105">Ten błąd występuje podczas ograniczania typu ogólnego do określonej klasy i spróbuj skonstruować wystąpienie tej klasy przy użyciu typu wartości, który nie może być niejawnie zapakowane do niego.</span><span class="sxs-lookup"><span data-stu-id="ba9c9-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>

## <a name="example"></a><span data-ttu-id="ba9c9-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="ba9c9-108">Example</span></span>  
 <span data-ttu-id="ba9c9-109">Poniższy przykład generuje CS0315:</span><span class="sxs-lookup"><span data-stu-id="ba9c9-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
