# <a name="compiler-error-cs1530"></a>CS1530 błąd kompilatora
Słowo kluczowe "new" nie jest dozwolona w elementach zdefiniowanych w przestrzeni nazw  
  
 Nie jest konieczne określić [nowe](../../csharp/language-reference/keywords/new.md) słowo kluczowe w konstrukcji, która znajduje się w [przestrzeni nazw](../../csharp/language-reference/keywords/namespace.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS1530:  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
