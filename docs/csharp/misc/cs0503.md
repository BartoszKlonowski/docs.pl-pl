# <a name="compiler-error-cs0503"></a>CS0503 błąd kompilatora
Metoda abstrakcyjna "method" nie może być oznaczona jako wirtualna  
  
 Jest nadmiarowe do oznaczania metody elementu członkowskiego jako [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i [wirtualnego](../../csharp/language-reference/keywords/virtual.md) ponieważ **abstrakcyjne** oznacza **wirtualnego**.  
  
 Poniższy przykład spowoduje wygenerowanie CS0503:  
  
```csharp  
// CS0503.cs  
namespace x  
{  
   abstract public class clx  
   {  
      abstract virtual public void f();   // CS0503  
   }  
}  
```
