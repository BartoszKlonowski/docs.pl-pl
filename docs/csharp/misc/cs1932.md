# <a name="compiler-error-cs1932"></a>Błąd kompilatora CS1932
Nie można przypisać wyrażenia "Expression" do zmiennej zakresu.  
  
 Kompilator musi mieć możliwość wywnioskowania typu zmiennej zakresu, niezależnie od tego, czy jest on wprowadzony w `from` klauzuli `let` , czy w klauzuli. Nie może mieć wartości null, ponieważ wartość null nie jest typem i nie można jej przypisać za pomocą wyrażenia typu niebezpiecznego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe przypisanie.  
  
- Jawne rzutowanie wyrażenia na dozwolony typ  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1932, ponieważ nie można wywnioskować typu zmiennej zakresu. Rzutowanie wartości na zamierzony typ, aby naprawić błąd, jak pokazano w poniższym przykładzie.  
  
```csharp  
// CS1932.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
  
        var x = from i in Enumerable.Range(1, 100)  
                let k = null // CS1932  
                // Try the following line instead.  
                let k = (string) null  
                select i;  
    }  
}  
```
