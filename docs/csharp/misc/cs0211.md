# <a name="compiler-error-cs0211"></a><span data-ttu-id="ebf5e-103">Błąd kompilatora CS0211</span><span class="sxs-lookup"><span data-stu-id="ebf5e-103">Compiler Error CS0211</span></span>
<span data-ttu-id="ebf5e-104">Nie można pobrać adresu danego wyrażenia</span><span class="sxs-lookup"><span data-stu-id="ebf5e-104">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="ebf5e-105">Można przyjąć adres pól, zmienne lokalne i pośrednika wskaźników, ale nie można wykonać na przykład adresu sumy dwóch zmiennych lokalnych.</span><span class="sxs-lookup"><span data-stu-id="ebf5e-105">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="ebf5e-106">Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="ebf5e-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="ebf5e-107">Poniższy przykład generuje CS0211:</span><span class="sxs-lookup"><span data-stu-id="ebf5e-107">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
