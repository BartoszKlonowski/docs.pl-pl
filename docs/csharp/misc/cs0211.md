<span data-ttu-id="cea96-103">Nie można pobrać adresu podanego wyrażenia</span><span class="sxs-lookup"><span data-stu-id="cea96-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="cea96-104">Można przyjąć adresu pola, zmienne lokalne i pośrednie wskaźników, ale nie możesz wykonać, na przykład adres sumę dwóch zmiennych lokalnych.</span><span class="sxs-lookup"><span data-stu-id="cea96-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="cea96-105">Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="cea96-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="cea96-106">Poniższy przykład generuje CS0211:</span><span class="sxs-lookup"><span data-stu-id="cea96-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
