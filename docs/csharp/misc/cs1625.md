# <a name="compiler-error-cs1625"></a><span data-ttu-id="38a51-103">Błąd kompilatora CS1625</span><span class="sxs-lookup"><span data-stu-id="38a51-103">Compiler Error CS1625</span></span>

<span data-ttu-id="38a51-104">Nie można wykonać Yield w treści klauzuli finally</span><span class="sxs-lookup"><span data-stu-id="38a51-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="38a51-105">Instrukcja Yield nie jest dozwolona w treści klauzuli finally.</span><span class="sxs-lookup"><span data-stu-id="38a51-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="38a51-106">Aby uniknąć tego błędu, Przenieś instrukcję Yield z klauzuli finally.</span><span class="sxs-lookup"><span data-stu-id="38a51-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="38a51-107">Poniższy przykład generuje CS1625:</span><span class="sxs-lookup"><span data-stu-id="38a51-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
