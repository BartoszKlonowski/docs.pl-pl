# <a name="compiler-error-cs1625"></a><span data-ttu-id="5a143-102">CS1625 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="5a143-102">Compiler Error CS1625</span></span>
<span data-ttu-id="5a143-103">Nie można użyć instrukcji yield w treści klauzuli finally</span><span class="sxs-lookup"><span data-stu-id="5a143-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="5a143-104">Instrukcja yield nie jest dozwolona w treści klauzuli finally.</span><span class="sxs-lookup"><span data-stu-id="5a143-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="5a143-105">Aby uniknąć tego błędu, należy przenieść instrukcji yield z klauzuli finally.</span><span class="sxs-lookup"><span data-stu-id="5a143-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="5a143-106">Poniższy przykład generuje CS1625:</span><span class="sxs-lookup"><span data-stu-id="5a143-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
