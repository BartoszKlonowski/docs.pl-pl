# <a name="compiler-warning-level-3-cs1702"></a>Kompilatora (poziom 3) ostrzeżenie CS1702
Przy założeniu, że odwołanie do zestawu "#1. Nazwa zestawu" pasuje do "#2. Nazwa zestawu", może być konieczne podanie zasad środowiska uruchomieniowego  
  
 Odwołania do dwóch zestawów mają różne kompilacji i/lub numery wersji, dzięki czemu nie będzie automatycznie ujednolicenie. Może być konieczne podanie zasady czasu wykonywania, aby wymusić ujednolicenie przy użyciu dyrektywy w pliku config aplikacji.
