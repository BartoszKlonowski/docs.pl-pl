# <a name="compiler-warning-level-3-cs1702"></a>Ostrzeżenie kompilatora (poziom 3) CS1702

Zakładając, że odwołanie do zestawu "Nazwa zestawu #1" pasuje do "nazwy zestawu #2", może być konieczne podanie zasad środowiska uruchomieniowego  
  
 Dwa odwołania do zestawów mają różne numery kompilacji i/lub poprawki, więc nie będą automatycznie ujednolicania. Może być konieczne dostarczenie zasad czasu wykonywania w celu wymuszenia zjednoczenia przy użyciu dyrektyw w pliku Application. config.
