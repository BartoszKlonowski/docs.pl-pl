# <a name="compiler-warning-level-3-cs1702"></a>Kompilator CS1702 ostrzegawcze (poziom 3)
Zakładając, że odwołanie do zestawu "#1. Nazwa zestawu" odpowiada "Nazwa zestawu #2", może być konieczne podanie zasad środowiska uruchomieniowego  
  
 Odwołania do dwóch zestawów mają różne kompilacji i/lub numeru wydania, więc nie zostanie automatycznie ujednolicenie. Może być konieczne podanie zasad środowiska wykonawczego, aby wymusić ujednolicenie przy użyciu dyrektywy w pliku config aplikacji.
