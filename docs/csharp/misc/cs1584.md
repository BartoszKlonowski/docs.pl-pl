# <a name="compiler-warning-level-1-cs1584"></a>Kompilatora (poziom 1) ostrzeżenie CS1584
Komentarz XML w elemencie "członek" ma atrybut cref składniowo niepoprawny "invalid_syntax"  
  
 Jeden z parametrów przekazanych do tagu dla komentarzy do dokumentacji ma nieprawidłową składnię. Aby uzyskać więcej informacji, zobacz [tagi zalecane dla komentarzy do dokumentacji](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1584.  
  
```csharp  
// CS1584.cs  
// compile with: /W:1 /doc:CS1584.xml  
/// <remarks>Test class</remarks>  
public class Test  
{  
   /// <remarks>Called in <see cref="Test.Mai()n"/>.</remarks>   // CS1584  
   // try the following line instead  
   // /// <remarks>Called in <see cref="Test.Main()"/>.</remarks>  
   public static void Test2() {}  
  
   /// <remarks>Main method</remarks>  
   public static void Main() {}  
}  
```
