# <a name="compiler-error-cs1725"></a>Błąd kompilatora CS1725
Odwołanie do zestawu znajomych "odwołanie" jest nieprawidłowe. InternalsVisibleTo deklaracje nie mogą mieć wersji, kultury, tokenu klucza publicznego lub architektury procesora określony.  
  
 Nie można dodać kultury wersji w odwołaniu do zestawu znajomego. Klasy częściowe powinny być widoczne dla zestawów znajomych.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1725.  
  
```csharp  
// CS1725.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:InternalsVisibleTo("partial01,version=1.1.0.0")]   // CS1725  
// try the following line instead  
// [assembly:InternalsVisibleTo("partial01")]  
  
partial class TestClass
{  
   public static string strBar = "my string";  
}  
```
