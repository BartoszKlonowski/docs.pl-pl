# <a name="compiler-error-cs0254"></a><span data-ttu-id="20e50-103">Błąd kompilatora CS0254</span><span class="sxs-lookup"><span data-stu-id="20e50-103">Compiler Error CS0254</span></span>
<span data-ttu-id="20e50-104">Prawa strona przypisania instrukcji "Fixed" nie może być wyrażeniem rzutowania</span><span class="sxs-lookup"><span data-stu-id="20e50-104">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="20e50-105">Po prawej stronie [stałego](../language-reference/keywords/fixed-statement.md) wyrażenia nie można użyć rzutowania.</span><span class="sxs-lookup"><span data-stu-id="20e50-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="20e50-106">Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="20e50-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="20e50-107">Poniższy przykład generuje CS0254:</span><span class="sxs-lookup"><span data-stu-id="20e50-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
