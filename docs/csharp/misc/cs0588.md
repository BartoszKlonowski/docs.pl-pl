# <a name="compiler-error-cs0588"></a>Błąd kompilatora CS0588

Wewnętrzny błąd kompilatora: etap "LEX"

 Spróbuj określić, czy kompilator kończy się niepowodzeniem z powodu niemożności przeanalizowania nieoczekiwanej składni. Jeśli tak nie jest, spróbuj [skontaktować się z nami](/visualstudio/ide/feedback-options).

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
