# <a name="compiler-error-cs2012"></a>CS2012 błąd kompilatora
Nie można otworzyć pliku do zapisu  
  
 Podczas używania [/bugreport:](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) `file` — opcja kompilatora, nie można otworzyć pliku do zapisu. Upewnij się, że określono prawidłową nazwę pliku oraz że plik nie jest tylko do odczytu.
