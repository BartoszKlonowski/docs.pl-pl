# <a name="compiler-error-cs2012"></a>Błąd kompilatora CS2012
Nie można otworzyć pliku "File" do zapisu  
  
 Przy użyciu opcji kompilatora [-bugreport:](../language-reference/compiler-options/bugreport-compiler-option.md) `file` nie można otworzyć pliku do zapisu. Upewnij się, że określono prawidłową nazwę pliku i że plik nie jest tylko do odczytu.
