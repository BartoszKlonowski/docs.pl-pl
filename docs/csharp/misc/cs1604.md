# <a name="compiler-error-cs1604"></a>Błąd kompilatora CS1604
Nie można przypisać do elementu "variable", ponieważ jest on tylko do odczytu  
  
 Wykonano przypisanie do zmiennej tylko do odczytu. Aby uniknąć tego błędu, nie przypisuj ani nie próbuj modyfikować tej zmiennej w tym kontekście.
