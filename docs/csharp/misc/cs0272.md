## <a name="example"></a><span data-ttu-id="d0e38-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="d0e38-108">Example</span></span>

```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}

{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
