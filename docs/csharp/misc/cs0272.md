## <a name="example"></a><span data-ttu-id="a2caf-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="a2caf-107">Example</span></span>

```csharp

{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}

{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
