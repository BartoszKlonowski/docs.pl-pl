## <a name="example"></a><span data-ttu-id="f3abc-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="f3abc-107">Example</span></span>

```csharp

{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}

{  
    static void Main()  
    {  
        MyClass c = new MyClass();

        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
