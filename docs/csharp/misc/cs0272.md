# <a name="compiler-error-cs0272"></a>CS0272 błąd kompilatora
Właściwość lub indeksator "właściwości/indeksatora" nie można użyć w tym kontekście, ponieważ niedostępna jest metoda dostępu set  
  
 Ten błąd występuje, gdy `set` dostępu nie jest dostępny do kodu programu. Aby rozwiązać ten problem, Zwiększ dostępność metody dostępu lub zmień lokalizację wywoływania. Aby uzyskać więcej informacji, zobacz [Ograniczanie dostępności metody dostępu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0272:  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
