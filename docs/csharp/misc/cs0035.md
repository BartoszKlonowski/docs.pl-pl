# <a name="compiler-error-cs0035"></a><span data-ttu-id="76d35-102">CS0035 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="76d35-102">Compiler Error CS0035</span></span>
<span data-ttu-id="76d35-103">Operator 'operator' jest niejednoznaczny dla argumentu operacji typu "type"</span><span class="sxs-lookup"><span data-stu-id="76d35-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="76d35-104">Kompilator ma więcej niż jeden dostępnej konwersji i nie może określić, do wyboru przed zastosowaniem operatora.</span><span class="sxs-lookup"><span data-stu-id="76d35-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="76d35-105">Aby uzyskać więcej informacji, zobacz [konwersje z szablonem zdefiniowane użytkownika](/cpp/misc/templated-user-defined-conversions) i [operatory konwersji](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="76d35-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="76d35-106">Poniższy przykład spowoduje wygenerowanie CS0035:</span><span class="sxs-lookup"><span data-stu-id="76d35-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
