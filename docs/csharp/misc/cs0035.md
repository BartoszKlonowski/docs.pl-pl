# <a name="compiler-error-cs0035"></a>CS0035 błąd kompilatora
Operator 'operator' jest niejednoznaczny dla argumentu operacji typu "type"  
  
Kompilator ma więcej niż jeden dostępnej konwersji i nie może określić, do wyboru przed zastosowaniem operatora.
  
Poniższy przykład spowoduje wygenerowanie CS0035:  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
