# <a name="compiler-error-cs1557"></a>Błąd kompilatora CS1557
Nie można użyć elementu "Class" dla metody Main, ponieważ znajduje się on w innym pliku wyjściowym  
  
 Opcja kompilatora [/Main](../language-reference/compiler-options/main-compiler-option.md) została określona dla jednego pliku wyjściowego w kompilacji pliku z wielu wyników. Jednak nie znaleziono klasy w kodzie źródłowym dla kompilacji/Main; został on odnaleziony w pliku kodu źródłowego dla jednego z pozostałych plików wyjściowych w kompilacji.
