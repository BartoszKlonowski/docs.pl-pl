# <a name="compiler-error-cs1557"></a>CS1557 błąd kompilatora
Nie można użyć "class" dla metody Main, ponieważ znajduje się w innym pliku wyjściowym.  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) określono opcję kompilatora dla jednego pliku w kompilacji wielokrotnego wyjściowego pliku wyjściowego. Jednak nie znaleziono klasy w kodzie źródłowym dla / main kompilacji; znaleziono w pliku kodu źródłowego dla jednej z pozostałych plików danych wyjściowych kompilacji.
