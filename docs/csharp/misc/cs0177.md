# <a name="compiler-error-cs0177"></a>CS0177 błąd kompilatora
Parametr out "parametru" muszą być przypisane do, zanim sterowanie wyjdzie z bieżącej metody  
  
 Parametr oznaczony przy użyciu [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) — słowo kluczowe nie zostało przypisane wartości w treści metody. Aby uzyskać więcej informacji, zobacz [przekazywanie parametrów](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 Poniższy przykład spowoduje wygenerowanie CS0177:  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
