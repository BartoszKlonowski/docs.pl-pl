 <span data-ttu-id="f1601-104">Parametr oznaczony za pomocą słowa kluczowego [out](../language-reference/keywords/out-parameter-modifier.md) nie ma przypisanej wartości w treści metody.</span><span class="sxs-lookup"><span data-stu-id="f1601-104">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="f1601-105">Aby uzyskać więcej informacji, zobacz [przekazywanie parametrów](../programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="f1601-105">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>

   public static void Foo(out int i)   // CS0177

       Foo(out x);  
   }  
}
