# <a name="compiler-error-cs0811"></a>CS0811 błąd kompilatora
W pełni kwalifikowana nazwa "name" jest za długa dla informacji debugowania. Skompiluj bez opcji "/ debug".  
  
 Istnieją ograniczenia wielkości w zmiennej i typ nazwy w informacjach debugowania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Jeśli nie jest możliwe zmodyfikowanie nazwy, jedyną alternatywą jest skompilować bez [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) opcji.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS0811:  
  
```csharp  
// cs0811.cs  
//Compile with: /debug  
using System;  
using System.Collections.Generic;  
  
namespace TestNamespace  
{  
    using Long = List<List<List<List<List<List<List<List<List<List<List<List<List  
   <List<List<List<List<List<List<List<List<List<List<List<List<List<List<List<int>>>>>>>>>>>>>>>>>>>>>>>>>>>>; // CS0811  
  
    class Test  
    {  
        static int Main()  
        {  
            return 1;  
        }  
    }  
}  
```
