# <a name="compiler-error-cs0025"></a>Błąd kompilatora CS0025

Nie można odnaleźć pliku biblioteki standardowej "File"  
  
 Nie znaleziono pliku, który jest wymagany przez kompilator. Sprawdź, czy ścieżka jest poprawna i czy plik istnieje.  
  
 Jeśli plik jest plikiem systemowym programu Visual Studio, może być konieczne naprawienie instalacji programu Visual Studio lub jej całkowite zainstalowanie.
