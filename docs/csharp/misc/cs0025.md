# <a name="compiler-error-cs0025"></a>CS0025 błąd kompilatora
Nie można odnaleźć biblioteki standardowej "pliku"  
  
 Nie znaleziono pliku wymaganego przez kompilator. Sprawdź, czy ścieżka jest prawidłowa i czy plik istnieje.  
  
 Jeśli plik jest [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] systemu plików, należy naprawić Twojej [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] instalacji, lub całkowicie zainstaluj go ponownie.
