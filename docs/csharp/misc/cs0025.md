# <a name="compiler-error-cs0025"></a>CS0025 błąd kompilatora
Nie można odnaleźć biblioteki standardowej pliku 'Plik'  
  
 Nie znaleziono pliku, które są wymagane przez kompilator. Sprawdź, czy ścieżka jest prawidłowa i czy plik istnieje.  
  
 Jeśli plik jest plikiem systemu Visual Studio, może być konieczne napraw instalację programu Visual Studio lub jego ponowną instalację całkowicie.
