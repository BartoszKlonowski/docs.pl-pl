# <a name="compiler-error-cs1609"></a>CS1609 błąd kompilatora
Modyfikatory nie może być umieszczona w deklaracjach metod dostępu zdarzeń  
  
 Modyfikatory można umieścić tylko w deklaracjach zdarzeń, a nie w deklaracjach metod dostępu zdarzeń. Aby uzyskać więcej informacji, zobacz [przy użyciu właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1609.  
  
```csharp  
// CS1609.cs  
// compile with: /target:library  
delegate int Del();  
class A  
{  
   public event Del MyEvent   
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```
