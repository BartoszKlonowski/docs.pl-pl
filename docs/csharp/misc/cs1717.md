 <span data-ttu-id="e13a2-104">To ostrzeżenie występuje, gdy przypisujesz zmienną do samej siebie, takich jak `a = a`.</span><span class="sxs-lookup"><span data-stu-id="e13a2-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="e13a2-105">Kilka typowych pomyłek może wygenerować tego ostrzeżenia:</span><span class="sxs-lookup"><span data-stu-id="e13a2-105">Several common mistakes can generate this warning:</span></span>
