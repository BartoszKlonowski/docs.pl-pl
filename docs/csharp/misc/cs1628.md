# <a name="compiler-error-cs1628"></a>CS1628 błąd kompilatora
Nie można użyć w ref lub out parametru "parameter" wewnątrz metody anonimowej, wyrażenia lambda ani wyrażenia zapytania  
  
 Ten błąd występuje, jeśli używasz `in`, `ref`, lub `out` parametru wewnątrz bloku metody anonimowej. Aby uniknąć tego błędu, należy użyć zmiennej lokalnej lub niektórych innych konstrukcji.  
  
 Poniższy przykład generuje CS1628:  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
