 <span data-ttu-id="23508-104">Ten błąd występuje, jeśli używasz `in`, `ref`, lub `out` parametru wewnątrz bloku metody anonimowej.</span><span class="sxs-lookup"><span data-stu-id="23508-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="23508-105">Aby uniknąć tego błędu, należy użyć zmiennej lokalnej lub niektórych innych konstrukcji.</span><span class="sxs-lookup"><span data-stu-id="23508-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="23508-106">Poniższy przykład generuje CS1628:</span><span class="sxs-lookup"><span data-stu-id="23508-106">The following sample generates CS1628:</span></span>

```csharp  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
