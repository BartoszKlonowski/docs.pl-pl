# <a name="compiler-error-cs1628"></a><span data-ttu-id="d57ca-103">Błąd kompilatora CS1628</span><span class="sxs-lookup"><span data-stu-id="d57ca-103">Compiler Error CS1628</span></span>

<span data-ttu-id="d57ca-104">Nie można użyć parametru ref ani out parametru "parameter" w metodzie anonimowej, wyrażeniu lambda ani wyrażeniu zapytania</span><span class="sxs-lookup"><span data-stu-id="d57ca-104">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="d57ca-105">Ten błąd występuje, jeśli używasz `in` parametru, `ref` lub `out` w bloku metody anonimowej.</span><span class="sxs-lookup"><span data-stu-id="d57ca-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="d57ca-106">Aby uniknąć tego błędu, należy użyć zmiennej lokalnej lub innej konstrukcji.</span><span class="sxs-lookup"><span data-stu-id="d57ca-106">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="d57ca-107">Poniższy przykład generuje CS1628:</span><span class="sxs-lookup"><span data-stu-id="d57ca-107">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
