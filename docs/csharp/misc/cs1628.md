<span data-ttu-id="1c5b3-103">Nie można używać w ref lub out parametru "parametru" wewnątrz metody anonimowej, wyrażenia lambda lub wyrażeniu zapytania</span><span class="sxs-lookup"><span data-stu-id="1c5b3-103">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="1c5b3-104">Ten błąd występuje, jeśli używasz `in`, `ref`, lub `out` parametru wewnątrz bloku metody anonimowej.</span><span class="sxs-lookup"><span data-stu-id="1c5b3-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="1c5b3-105">Aby uniknąć tego błędu, należy użyć zmiennej lokalnej lub niektórych innych konstrukcji.</span><span class="sxs-lookup"><span data-stu-id="1c5b3-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="1c5b3-106">Poniższy przykład spowoduje wygenerowanie CS1628:</span><span class="sxs-lookup"><span data-stu-id="1c5b3-106">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs

class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
