 <span data-ttu-id="4c4ee-104">Podjęto próbę jawnego deklarowania elementu członkowskiego [interfejsu](../language-reference/keywords/interface.md) , który nie istnieje.</span><span class="sxs-lookup"><span data-stu-id="4c4ee-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="4c4ee-105">Należy usunąć deklarację lub ją zmienić tak, aby odnosiła się do prawidłowego elementu członkowskiego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="4c4ee-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>

      void m();

         return 0;  
      }  
   }
