# <a name="compiler-error-cs0539"></a>CS0539 błąd kompilatora
"członek" w jawnej deklaracji interfejsu nie jest składową interfejsu  
  
 Nastąpiła próba można jawnie deklarować [interfejsu](../../csharp/language-reference/keywords/interface.md) elementu członkowskiego, który nie istnieje. Należy usunąć deklaracji lub go zmienić, aby odnosi się do elementu członkowskiego interfejsu prawidłowe.  
  
 Poniższy przykład spowoduje wygenerowanie CS0539:  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
