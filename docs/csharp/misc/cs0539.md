<span data-ttu-id="48d28-104">element "member" w jawnej deklaracji interfejsu nie jest elementem członkowskim interfejsu</span><span class="sxs-lookup"><span data-stu-id="48d28-104">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="48d28-105">Podjęto próbę jawnego deklarowania elementu członkowskiego [interfejsu](../language-reference/keywords/interface.md) , który nie istnieje.</span><span class="sxs-lookup"><span data-stu-id="48d28-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="48d28-106">Należy usunąć deklarację lub ją zmienić tak, aby odnosiła się do prawidłowego elementu członkowskiego interfejsu.</span><span class="sxs-lookup"><span data-stu-id="48d28-106">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="48d28-107">Poniższy przykład generuje CS0539:</span><span class="sxs-lookup"><span data-stu-id="48d28-107">The following sample generates CS0539:</span></span>  
  
```csharp  
// CS0539.cs

   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
