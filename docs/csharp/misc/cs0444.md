# <a name="compiler-warning-level-2-cs0444"></a>CS0444 ostrzeżenie (poziom 2) kompilator
Wstępnie zdefiniowane typu "nie można odnaleźć nazwy typu 1" w "systemową przestrzenią nazw 1", ale został znaleziony w "systemową przestrzenią nazw 2  
  
 Wstępnie zdefiniowanych obiektów, takich jak <xref:System.Int32> nie został znaleziony, gdzie kompilator powinien go znaleźć, ale zamiast tego znaleziono w "przestrzeń nazw System 2'.  
  
 Ten błąd może wskazywać, czy niepoprawnie zainstalowany .NET Framework. Aby rozwiązać ten problem, zainstaluj ponownie .NET Framework.  
  
 Jeśli piszesz własnego bibliotek klas bazowych, również może wystąpić ten błąd. W takim przypadku aby naprawić błąd, należy przebudować mscorlib.
