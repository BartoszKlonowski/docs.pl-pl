# <a name="compiler-error-cs2015"></a>CS2015 błąd kompilatora
'Plik' jest plikiem binarnym, nie plikiem tekstowym  
  
 Plik został przekazany do kompilatora, który był pliku binarnego. Kompilator oczekuje pliku kodu źródłowego.
