# <a name="compiler-error-cs2015"></a>Błąd kompilatora CS2015

plik "File" jest plikiem binarnym, a nie plikiem tekstowym  
  
 Plik został przekazano do kompilatora, który był plikiem binarnym. Kompilator oczekuje pliku kodu źródłowego.
