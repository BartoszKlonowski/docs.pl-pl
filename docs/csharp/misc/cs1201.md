# <a name="compiler-warning-level-1-cs1201"></a>Ostrzeżenie kompilatora (poziom 1) CS1201

Funkcja "Nieprawidłowa funkcja" jest przestarzała. Użyj zamiast tego "prawidłowej funkcji".  
  
 Funkcja, której próbujesz użyć, jest obecnie przestarzała. Zaktualizuj swój kod, aby zamiast tego użyć prawidłowej funkcji.
