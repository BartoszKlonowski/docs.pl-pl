# <a name="compiler-warning-level-2-cs1571"></a>CS1571 ostrzeżenie (poziom 2) kompilator
Komentarz XML w elemencie "konstrukcji" ma zduplikowany tag param dla "parametru"  
  
 Korzystając z [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md) — opcja kompilatora, znaleziono wielu komentarzy dla tego samego parametru metody. Usuń jeden z zduplikowane wiersze.  
  
 Poniższy przykład spowoduje wygenerowanie CS1571:  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
