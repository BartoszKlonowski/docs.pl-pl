# <a name="compiler-error-cs0228"></a>Błąd kompilatora CS0228

element "Type" nie zawiera definicji elementu "member" lub jest niedostępny  
  
 Być może zastąpisz systemową wersję **System. Hashtable**, **System. String**lub **System. Array**, a zamiennik nie zawiera `member` .  
  
 W przeciwnym razie może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio.
