# <a name="compiler-error-cs0228"></a>CS0228 błąd kompilatora
"type" nie zawiera definicji "członek" lub jest niedostępny  
  
 Zastąpiono wersję systemu **System.Hashtable**, **System.String**, lub **System.Array**, i nie zawiera zastąpienia `member`.  
  
 W przeciwnym razie może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio.
