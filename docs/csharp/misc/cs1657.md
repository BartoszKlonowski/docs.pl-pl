 <span data-ttu-id="5186c-105">Ten błąd występuje, gdy zmienna jest przenoszona jako argument [ref](../language-reference/keywords/ref.md) lub [out](../language-reference/keywords/out-parameter-modifier.md) w kontekście, w którym ta zmienna jest tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="5186c-105">This error occurs when a variable is passed as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="5186c-106">Konteksty ReadOnly obejmują zmienne iteracji [foreach](../language-reference/keywords/foreach-in.md) , [przy użyciu](../language-reference/keywords/using-statement.md) zmiennych i `fixed` zmiennych.</span><span class="sxs-lookup"><span data-stu-id="5186c-106">Readonly contexts include [foreach](../language-reference/keywords/foreach-in.md) iteration variables, [using](../language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="5186c-107">Aby rozwiązać ten problem, nie wywołuj funkcji przyjmujących lub `foreach` `using` `fixed` zmiennej jako `ref` `out` parametru lub w `using` blokach, `foreach` instrukcjach i `fixed` instrukcjach.</span><span class="sxs-lookup"><span data-stu-id="5186c-107">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5186c-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5186c-108">Example</span></span>  
 <span data-ttu-id="5186c-109">Poniższy przykład generuje CS1657:</span><span class="sxs-lookup"><span data-stu-id="5186c-109">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
