 <span data-ttu-id="603e2-104">Występuje konflikt nazw między parametr typu metody rodzajowej i zmienną lokalną w metodzie lub jeden z parametrów metody.</span><span class="sxs-lookup"><span data-stu-id="603e2-104">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="603e2-105">Aby uniknąć tego błędu, należy zmienić wszystkie parametry powodujące konflikt lub zmiennych lokalnych.</span><span class="sxs-lookup"><span data-stu-id="603e2-105">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="603e2-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="603e2-106">Example</span></span>  
 <span data-ttu-id="603e2-107">Poniższy przykład generuje CS0412:</span><span class="sxs-lookup"><span data-stu-id="603e2-107">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
