# <a name="compiler-error-cs0412"></a><span data-ttu-id="e2e42-102">CS0412 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e2e42-102">Compiler Error CS0412</span></span>
<span data-ttu-id="e2e42-103">"Ogólne": parametr lub zmienna lokalna nie może mieć taką samą nazwę jak parametr typu metody</span><span class="sxs-lookup"><span data-stu-id="e2e42-103">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="e2e42-104">Występuje konflikt nazw między parametr typu metody ogólnej i zmienną lokalną w metodzie lub jeden z parametrów metody.</span><span class="sxs-lookup"><span data-stu-id="e2e42-104">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="e2e42-105">Aby uniknąć tego błędu, Zmień nazwę wszelkie parametry powodujące konflikt lub zmiennych lokalnych.</span><span class="sxs-lookup"><span data-stu-id="e2e42-105">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e2e42-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="e2e42-106">Example</span></span>  
 <span data-ttu-id="e2e42-107">Poniższy przykład spowoduje wygenerowanie CS0412:</span><span class="sxs-lookup"><span data-stu-id="e2e42-107">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
