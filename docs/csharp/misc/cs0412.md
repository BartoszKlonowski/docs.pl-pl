 <span data-ttu-id="7e1d3-105">Wystąpił konflikt nazw między parametrem typu metody generycznej i zmienną lokalną w metodzie lub jednym z parametrów metody.</span><span class="sxs-lookup"><span data-stu-id="7e1d3-105">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="7e1d3-106">Aby uniknąć tego błędu, należy zmienić nazwy wszystkich parametrów powodujących konflikt lub zmiennych lokalnych.</span><span class="sxs-lookup"><span data-stu-id="7e1d3-106">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7e1d3-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="7e1d3-107">Example</span></span>  
 <span data-ttu-id="7e1d3-108">Poniższy przykład generuje CS0412:</span><span class="sxs-lookup"><span data-stu-id="7e1d3-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
