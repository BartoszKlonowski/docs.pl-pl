# <a name="compiler-warning-level-2-cs0440"></a><span data-ttu-id="1c124-102">Kompilator CS0440 ostrzegawcze (poziom 2)</span><span class="sxs-lookup"><span data-stu-id="1c124-102">Compiler Warning (level 2) CS0440</span></span>
<span data-ttu-id="1c124-103">Definiowanie aliasu o nazwie "global" jest niezalecane, ponieważ łańcuch "globalne::" zawsze odwołuje się do globalnej przestrzeni nazw, a nie do aliasu</span><span class="sxs-lookup"><span data-stu-id="1c124-103">Defining an alias named 'global' is ill-advised since 'global::' always references the global namespace and not an alias</span></span>  
  
 <span data-ttu-id="1c124-104">To ostrzeżenie podczas definiowania aliasu o nazwie globalnego.</span><span class="sxs-lookup"><span data-stu-id="1c124-104">This warning is issued when you define an alias named global.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1c124-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="1c124-105">Example</span></span>  
 <span data-ttu-id="1c124-106">Poniższy przykład generuje CS0440:</span><span class="sxs-lookup"><span data-stu-id="1c124-106">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
