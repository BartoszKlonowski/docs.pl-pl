<span data-ttu-id="d99ee-104">Nie zaleca się definiowania aliasu o nazwie "Global", ponieważ element "Global::" zawsze odwołuje się do globalnej przestrzeni nazw, a nie do aliasu</span><span class="sxs-lookup"><span data-stu-id="d99ee-104">Defining an alias named 'global' is ill-advised since 'global::' always references the global namespace and not an alias</span></span>

## <a name="example"></a><span data-ttu-id="d99ee-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="d99ee-106">Example</span></span>  
 <span data-ttu-id="d99ee-107">Poniższy przykład generuje CS0440:</span><span class="sxs-lookup"><span data-stu-id="d99ee-107">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
