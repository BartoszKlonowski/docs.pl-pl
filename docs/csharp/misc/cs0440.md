# <a name="compiler-warning-level-2-cs0440"></a>CS0440 ostrzeżenie (poziom 2) kompilator
Definiowanie aliasu o nazwie "global" jest niezalecane, ponieważ łańcuch "globalne::" zawsze odwołuje się do globalnej przestrzeni nazw, a nie do aliasu  
  
 To ostrzeżenie zostanie wyświetlone podczas definiowania aliasu o nazwie globalnego.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0440:  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
