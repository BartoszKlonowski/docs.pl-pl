# <a name="compiler-error-cs1528"></a><span data-ttu-id="93e3a-103">Błąd kompilatora CS1528</span><span class="sxs-lookup"><span data-stu-id="93e3a-103">Compiler Error CS1528</span></span>

<span data-ttu-id="93e3a-104">Oczekuj or = (nie można określić argumentów konstruktora w deklaracji)</span><span class="sxs-lookup"><span data-stu-id="93e3a-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="93e3a-105">Odwołanie do klasy zostało sformułowane tak, jakby obiekt klasy był tworzony.</span><span class="sxs-lookup"><span data-stu-id="93e3a-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="93e3a-106">Na przykład podjęto próbę przekazania zmiennej do konstruktora.</span><span class="sxs-lookup"><span data-stu-id="93e3a-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="93e3a-107">Użyj operatora [New](../language-reference/operators/new-operator.md) , aby utworzyć obiekt klasy.</span><span class="sxs-lookup"><span data-stu-id="93e3a-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="93e3a-108">Poniższy przykład generuje CS1528:</span><span class="sxs-lookup"><span data-stu-id="93e3a-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
