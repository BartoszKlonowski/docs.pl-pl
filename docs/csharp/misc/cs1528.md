# <a name="compiler-error-cs1528"></a>Błąd kompilatora CS1528
Oczekuj or = (nie można określić argumentów konstruktora w deklaracji)  
  
 Odwołanie do klasy zostało sformułowane tak, jakby obiekt klasy był tworzony. Na przykład podjęto próbę przekazania zmiennej do konstruktora. Użyj operatora [New](../language-reference/operators/new-operator.md) , aby utworzyć obiekt klasy.  
  
 Poniższy przykład generuje CS1528:  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
