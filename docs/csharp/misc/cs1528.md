# <a name="compiler-error-cs1528"></a>CS1528 błąd kompilatora
Oczekiwano znaku; lub = (nie można określić argumentów konstruktora w deklaracji)  
  
 Odwołanie do klasy powstał tak, jakby utworzono obiekt do klasy. Na przykład została podjęta próba przekazać zmienną do konstruktora. Użyj [nowe](../../csharp/language-reference/keywords/new.md) operatora do utworzenia obiektu klasy.  
  
 Poniższy przykład spowoduje wygenerowanie CS1528:  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
