# <a name="compiler-error-cs0005"></a>Błąd kompilatora CS0005
Po opcji kompilatora "compiler_option" musi występować argument  
  
 Niektóre opcje kompilatora wymagają parametrów. Jeśli nie przekażesz argumentów wymaganych przez opcję kompilatora, CS0005 jest generowany.  
  
 Aby uzyskać więcej informacji, zobacz [Opcje kompilatora języka C#](../language-reference/compiler-options/index.md).
