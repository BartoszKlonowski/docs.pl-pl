# <a name="compiler-error-cs0005"></a>CS0005 błąd kompilatora
— Opcja kompilatora "compiler_option" musi występować argument  
  
 Niektóre opcje kompilatora wymagane parametry. Jeśli wymagane przez opcję kompilatora argumenty nie są przekazywane, generowany jest CS0005.  
  
 Aby uzyskać więcej informacji, zobacz [opcje kompilatora C#](../../csharp/language-reference/compiler-options/index.md).
