# <a name="compiler-error-cs2035"></a>Błąd kompilatora CS2035

Błąd składni wiersza polecenia: brak elementu ": \<number> " dla opcji "compiler_option"  
  
 Niektóre opcje kompilatora wymagają wartości.  
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS2035.  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
