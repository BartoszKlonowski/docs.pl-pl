# <a name="compiler-error-cs2035"></a>CS2035 błąd kompilatora
Błąd składni wiersza polecenia:  Brak ':\<liczba > "dla opcji"compiler_option"  
  
 Niektóre opcje kompilatora wymaga wartości.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS2035.  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
