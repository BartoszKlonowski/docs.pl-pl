# <a name="compiler-error-cs0158"></a>CS0158 błąd kompilatora
Etykieta "etykieta" zasłania inną etykietę o takiej samej nazwie w zawartym zakresie  
  
 Etykieta w zakresie wewnętrznym ukrywa etykiety o tej samej nazwie w zewnętrznym zakresie. Aby uzyskać więcej informacji, zobacz [goto](../../csharp/language-reference/keywords/goto.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0158:  
  
```csharp  
// CS0158.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         goto lab1;  
         lab1:  
         {  
            lab1:  
            goto lab1;   // CS0158  
         }  
      }  
   }  
}  
```
