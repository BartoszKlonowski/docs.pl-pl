# <a name="compiler-error-cs0625"></a><span data-ttu-id="538e7-103">Błąd kompilatora CS0625</span><span class="sxs-lookup"><span data-stu-id="538e7-103">Compiler Error CS0625</span></span>
<span data-ttu-id="538e7-104">"pole": typy pól wystąpienia oznaczone elementem StructLayout (LayoutKind. Explicit) muszą mieć atrybut FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="538e7-104">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="538e7-105">Gdy struktura jest oznaczona jawnym atrybutem **StructLayout** , wszystkie pola w strukturze muszą mieć atrybut [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) .</span><span class="sxs-lookup"><span data-stu-id="538e7-105">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="538e7-106">Aby uzyskać więcej informacji, zobacz [Klasa StructLayoutAttribute](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="538e7-106">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="538e7-107">Poniższy przykład generuje CS0625:</span><span class="sxs-lookup"><span data-stu-id="538e7-107">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
