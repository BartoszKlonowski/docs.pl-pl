# <a name="compiler-error-cs0625"></a><span data-ttu-id="dbfcf-102">CS0625 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="dbfcf-102">Compiler Error CS0625</span></span>
<span data-ttu-id="dbfcf-103">"field": typy pól wystąpienia oznaczone elementem StructLayout(LayoutKind.Explicit) muszą mieć atrybut FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="dbfcf-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="dbfcf-104">Gdy struktura jest oznaczony za pomocą jawnego **StructLayout** atrybut, musi mieć wszystkie pola w strukturze [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="dbfcf-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="dbfcf-105">Aby uzyskać więcej informacji, zobacz [structlayoutattribute — klasa](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="dbfcf-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="dbfcf-106">Poniższy przykład spowoduje wygenerowanie CS0625:</span><span class="sxs-lookup"><span data-stu-id="dbfcf-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
