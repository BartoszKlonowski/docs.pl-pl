# <a name="compiler-error-cs0625"></a><span data-ttu-id="85a72-102">CS0625 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="85a72-102">Compiler Error CS0625</span></span>
<span data-ttu-id="85a72-103">"field": typy pól wystąpienia oznaczone przy użyciu klasy StructLayout(LayoutKind.Explicit) muszą mieć atrybut FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="85a72-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="85a72-104">Gdy struktury jest oznaczony atrybutem jawnego **StructLayout** musi mieć atrybut, wszystkie pola w strukturze [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atrybutu.</span><span class="sxs-lookup"><span data-stu-id="85a72-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="85a72-105">Aby uzyskać więcej informacji, zobacz [structlayoutattribute — klasa](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="85a72-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="85a72-106">Poniższy przykład generuje CS0625:</span><span class="sxs-lookup"><span data-stu-id="85a72-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
