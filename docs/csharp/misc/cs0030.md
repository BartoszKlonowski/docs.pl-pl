<span data-ttu-id="21ac5-103">Nie można przekonwertować na typ 'typ' do 'typ'</span><span class="sxs-lookup"><span data-stu-id="21ac5-103">Cannot convert type 'type' to 'type'</span></span>  
  
 <span data-ttu-id="21ac5-104">Należy podać procedur konwersji do obsługi niektóre przeciążenia operatora.</span><span class="sxs-lookup"><span data-stu-id="21ac5-104">You must provide conversion routines to support certain operator overloads.</span></span> <span data-ttu-id="21ac5-105">Aby uzyskać więcej informacji, zobacz [operatory konwersji](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="21ac5-105">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="21ac5-106">Poniższy przykład spowoduje wygenerowanie CS0030:</span><span class="sxs-lookup"><span data-stu-id="21ac5-106">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
