# <a name="compiler-error-cs1714"></a>Błąd kompilatora CS1714

Nie można rozpoznać klasy bazowej lub interfejsu TypeName1 lub jest ono nieprawidłowe  
  
 Wdrażasz TypeName1 z klasy lub interfejsu, którego nie można rozpoznać, na przykład kompilator nie mógł go zlokalizować lub jest nieprawidłowy. Rozwiązaniem jest określenie, które z tych dwóch przypadków jest, i dokładniej określić lokalizację typu lub naprawić wszelkie błędy kompilatora w klasie bazowej.
