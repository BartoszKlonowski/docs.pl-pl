# <a name="compiler-error-cs1714"></a>CS1714 błąd kompilatora
Klasy podstawowej lub interfejsu TypeName1 nie może zostać rozpoznane lub jest nieprawidłowy  
  
 W przypadku implementowania TypeName1 z klasy lub interfejsu, że albo nie można rozpoznać, na przykład kompilator mógł zlokalizować go lub jest nieprawidłowy. Rozwiązanie to określić, która z tych dwóch przypadków jest i właściwie Określ lokalizację typu, lub napraw wszystkie błędy kompilatora w klasie podstawowej.
