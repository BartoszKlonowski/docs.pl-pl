# <a name="compiler-error-cs1714"></a>CS1714 błąd kompilatora
Klasy bazowej lub interfejsu TypeName1 nie może zostać rozpoznane lub jest nieprawidłowy  
  
 W przypadku wdrażania TypeName1 z klasy lub interfejsu, albo nie można rozpoznać, na przykład, kompilator mógł go zlokalizować lub jest nieprawidłowy. To rozwiązanie ma na celu określenie, które z tych dwóch przypadkach jest i bardziej poprawnie Określ lokalizację tego typu, lub napraw wszystkie błędy kompilatora w klasie bazowej.
