# <a name="compiler-warning-level-1-cs0809"></a>Kompilatora (poziom 1) ostrzeżenie CS0809
Przestarzała składowa "memberA" przesłania nieprzestarzałą składową "memberB".  
  
 Zazwyczaj element członkowski, który jest oznaczony jako przestarzały, nie powinny przesłaniać składowej, która nie jest oznaczony jako przestarzały. To ostrzeżenie jest generowana w [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] , ale nie [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Obsolete` atrybut z nadrzędnych elementu członkowskiego albo dodaj go do składowej klasy bazowej.  
  
## <a name="example"></a>Przykład  
  
```csharp  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}  
public class C : Base  
{  
    [System.Obsolete()]  
    public override void Test1() // CS0809  
    {  
    }  
}  
```
