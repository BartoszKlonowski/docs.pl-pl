<span data-ttu-id="2ea7e-103">"type" nie można oznaczyć jako zgodny ze specyfikacją CLS, ponieważ jest on członkiem niezgodna ze specyfikacją CLS typu "type"</span><span class="sxs-lookup"><span data-stu-id="2ea7e-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="2ea7e-104">To ostrzeżenie występuje, jeśli zagnieżdżona klasa z atrybutem CLSCompliant równa `true` został zadeklarowany jako składowa klasy typu zadeklarowana z atrybutem CLSCompliant równa `false`.</span><span class="sxs-lookup"><span data-stu-id="2ea7e-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="2ea7e-105">Jest to niedozwolone, ponieważ zagnieżdżona klasa nie może być zgodne ze specyfikacją CLS, jeśli jest to składowa klasy zewnętrznego, który nie jest zgodny ze specyfikacją CLS.</span><span class="sxs-lookup"><span data-stu-id="2ea7e-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="2ea7e-106">Aby rozwiązać to ostrzeżenie, usuń atrybut CLSCompliant z klasy zagnieżdżonej, lub zmienić go z `true` do `false`.</span><span class="sxs-lookup"><span data-stu-id="2ea7e-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="2ea7e-107">Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](~/docs/standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="2ea7e-107">For more information on CLS compliance, see [Language independence and language-independent components](~/docs/standard/language-independence.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2ea7e-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="2ea7e-108">Example</span></span>  
 <span data-ttu-id="2ea7e-109">Poniższy przykład spowoduje wygenerowanie CS3018.</span><span class="sxs-lookup"><span data-stu-id="2ea7e-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
