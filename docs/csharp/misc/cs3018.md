# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="3e96a-102">Ostrzeżenie kompilatora (poziom 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="3e96a-102">Compiler Warning (level 1) CS3018</span></span>

<span data-ttu-id="3e96a-103">nie można oznaczyć elementu "Type" jako zgodnego ze specyfikacją CLS, ponieważ jest to element członkowski typu "Type", który nie jest zgodny ze specyfikacją CLS</span><span class="sxs-lookup"><span data-stu-id="3e96a-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="3e96a-104">To ostrzeżenie występuje, jeśli Klasa zagnieżdżona z atrybutem CLSCompliant ustawionym na `true` jest zadeklarowana jako element członkowski klasy zadeklarowanej z atrybutem CLSCompliant ustawionym na `false` .</span><span class="sxs-lookup"><span data-stu-id="3e96a-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="3e96a-105">Nie jest to dozwolone, ponieważ Klasa zagnieżdżona nie może być zgodna ze specyfikacją CLS, jeśli jest elementem członkowskim klasy zewnętrznej, która nie jest zgodna ze specyfikacją CLS.</span><span class="sxs-lookup"><span data-stu-id="3e96a-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="3e96a-106">Aby usunąć to ostrzeżenie, Usuń atrybut CLSCompliant z klasy zagnieżdżonej lub zmień go z `true` na `false` .</span><span class="sxs-lookup"><span data-stu-id="3e96a-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="3e96a-107">Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="3e96a-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="3e96a-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="3e96a-108">Example</span></span>  

 <span data-ttu-id="3e96a-109">Poniższy przykład generuje CS3018.</span><span class="sxs-lookup"><span data-stu-id="3e96a-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
