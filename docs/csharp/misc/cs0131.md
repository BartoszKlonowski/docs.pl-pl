## <a name="example"></a><span data-ttu-id="2a18f-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="2a18f-108">Example</span></span>  
 <span data-ttu-id="2a18f-109">Poniższy przykład generuje CS0131.</span><span class="sxs-lookup"><span data-stu-id="2a18f-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```

 <span data-ttu-id="2a18f-111">Ten błąd może również wystąpić, jeśli podjęto próbę wykonania operacji arytmetycznych na lewej strony operatora przypisania, jak w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="2a18f-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

    public static int Main()

    }
