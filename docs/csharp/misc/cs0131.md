# <a name="compiler-error-cs0131"></a>CS0131 błąd kompilatora
Lewa strona przypisania musi być zmienną, właściwość lub indeksator  
  
 W instrukcji przypisania wartość po prawej stronie jest przypisywana do lewej. Po lewej stronie musi być zmienną, właściwość lub indeksator.  
  
 Aby naprawić ten błąd, upewnij się, że wszystkie operatory są po prawej stronie i że lewa strona zmiennej, właściwość lub indeksator. Aby uzyskać więcej informacji, zobacz [instrukcje, wyrażenia i operatory](../../csharp/programming-guide/statements-expressions-operators/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0131.  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a>Przykład  
 Ten błąd może również wystąpić, jeśli użytkownik podejmie próbę wykonania operacji arytmetycznych po lewej stronie operatora przypisania, jak w poniższym przykładzie.  
  
```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
