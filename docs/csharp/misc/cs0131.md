## <a name="example"></a><span data-ttu-id="c13f0-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="c13f0-108">Example</span></span>  
 <span data-ttu-id="c13f0-109">Poniższy przykład generuje CS0131.</span><span class="sxs-lookup"><span data-stu-id="c13f0-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c13f0-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="c13f0-110">Example</span></span>  
 <span data-ttu-id="c13f0-111">Ten błąd może również wystąpić, Jeśli podjęto próbę wykonania operacji arytmetycznych po lewej stronie operatora przypisania, jak w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="c13f0-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
