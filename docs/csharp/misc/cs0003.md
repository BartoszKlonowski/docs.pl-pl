# <a name="compiler-error-cs0003"></a>CS0003 błąd kompilatora
Za mało pamięci  
  
 Kompilator nie może przydzielić wystarczającej ilości pamięci wirtualnej do ukończenia kompilacji. Zamknij wszystkie zbędne aplikacje i skompiluj ponownie.  
  
 Można również zwiększyć rozmiar pliku stronicowania i upewnij się, że wolne miejsce na dysku.  
  
 Ten błąd może również wystąpić w wyniku o niezgodne wersje Windows Software Development Kit (SDK) i C# kompilatora, jeden lub więcej uszkodzone pliki obsługujące C# kompilatora; Zainstaluj ponownie program Visual Studio.
