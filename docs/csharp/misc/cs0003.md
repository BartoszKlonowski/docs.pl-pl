# <a name="compiler-error-cs0003"></a>Błąd kompilatora CS0003
Za mało pamięci  
  
 Kompilator nie może przydzielić wystarczającej ilości pamięci wirtualnej, aby ukończyć kompilację. Zamknij wszystkie niepotrzebne aplikacje i Skompiluj ponownie.  
  
 Warto również zwiększyć rozmiar pliku stronicowania i upewnić się, że masz wolne miejsce na dysku.  
  
 Ten błąd może również pojawić się w wyniku wystąpienia niezgodnych wersji Windows SDK oraz C# kompilatora lub jednego lub więcej uszkodzonych plików, które obsługują C# kompilator; Zainstaluj ponownie program Visual Studio.
