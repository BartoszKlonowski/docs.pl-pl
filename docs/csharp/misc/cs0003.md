# <a name="compiler-error-cs0003"></a>CS0003 błąd kompilatora
Za mało pamięci  
  
 Kompilator nie może przydzielić wystarczającej ilości pamięci wirtualnej, aby ukończyć kompilacji. Zamknij wszystkie zbędne aplikacje i skompiluj ponownie.  
  
 Można również zwiększenie rozmiaru pliku stronicowania i upewnij się, że masz wolnego miejsca na dysku.  
  
 Ten błąd może również zostać wyświetlony w wyniku o niezgodne wersje [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] i kompilator języka C# lub co najmniej jeden uszkodzonych plików, które obsługują kompilatora C#; ponownie zainstaluj program Visual Studio.
