# <a name="compiler-warning-level-1-cs0602"></a>Ostrzeżenie o kompilatorze (poziom 1) CS0602

Funkcja "old_feature" jest przestarzała. Zamiast tego użyj "new_feature"

 Funkcja języka używana w kodzie *(old_feature)* jest nadal obsługiwana, ale ta obsługa może zostać usunięta w przyszłej wersji. Zamiast tego należy użyć zalecanej składni *(new_feature).*

> [!NOTE]
> To ostrzeżenie kompilatora nie jest już używany w Roslyn.
