# <a name="compiler-warning-level-1-cs0602"></a>Ostrzeżenie kompilatora (poziom 1) CS0602

Funkcja "old_feature" jest przestarzała. Zamiast tego użyj "new_feature"

 Funkcja języka używana w kodzie (*old_feature*) jest nadal obsługiwana, ale pomoc techniczna może zostać usunięta w przyszłej wersji. Zamiast tego należy użyć zalecanej składni (*new_feature*).
 
> [!NOTE]
> To ostrzeżenie kompilatora nie jest już używane w Roslyn.
