# <a name="compiler-warning-level-1-cs0602"></a>Kompilatora (poziom 1) ostrzeżenie CS0602
Funkcja "old_feature" jest przestarzała. Zamiast tego użyj "new_feature"  
  
 Funkcja języka używana w kodzie (*old_feature*) jest nadal obsługiwany, ale że pomocy technicznej mogą zostać usunięte w przyszłej wersji. Zamiast tego należy użyć składni zalecane (*new_feature*).
