# <a name="compiler-error-cs0558"></a><span data-ttu-id="beb52-102">Błąd kompilatora CS0558</span><span class="sxs-lookup"><span data-stu-id="beb52-102">Compiler Error CS0558</span></span>

 <span data-ttu-id="beb52-104">[Modyfikatory](/dotnet/csharp/language-reference/keywords) dostępu **static** i **Public** muszą być określone w operatorach zdefiniowanych przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="beb52-104">Both the **static** and **public** access [modifiers](/dotnet/csharp/language-reference/keywords) must be specified on user-defined operators.</span></span>

```csharp  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
