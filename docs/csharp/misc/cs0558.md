# <a name="compiler-error-cs0558"></a><span data-ttu-id="311f2-103">Błąd kompilatora CS0558</span><span class="sxs-lookup"><span data-stu-id="311f2-103">Compiler Error CS0558</span></span>
<span data-ttu-id="311f2-104">Zdefiniowany przez użytkownika operator "operator" musi być zadeklarowany jako statyczny i publiczny</span><span class="sxs-lookup"><span data-stu-id="311f2-104">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="311f2-105">[Modyfikatory](../language-reference/keywords/index.md) dostępu **static** i **Public** muszą być określone w operatorach zdefiniowanych przez użytkownika.</span><span class="sxs-lookup"><span data-stu-id="311f2-105">Both the **static** and **public** access [modifiers](../language-reference/keywords/index.md) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="311f2-106">Poniższy przykład generuje CS0558:</span><span class="sxs-lookup"><span data-stu-id="311f2-106">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
