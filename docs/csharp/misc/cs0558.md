# <a name="compiler-error-cs0558"></a>Błąd kompilatora CS0558
Zdefiniowany przez użytkownika operator "operator" musi być zadeklarowany jako statyczny i publiczny  
  
 [Modyfikatory](../language-reference/keywords/modifiers.md) dostępu **static** i **Public** muszą być określone w operatorach zdefiniowanych przez użytkownika.  
  
 Poniższy przykład generuje CS0558:  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
