# <a name="compiler-error-cs0537"></a>Błąd kompilatora CS0537
Klasa System. Object nie może mieć klasy bazowej ani implementować interfejsu  
  
 CS0537 występuje podczas odbudowywania <xref:System> bibliotek klas i skąd <xref:System.Object> pochodzą z innej klasy. Nie można napotkać tego błędu. Jeśli to zrobisz, nie należy dziedziczyć <xref:System.Object> z klasy lub interfejsu: jest to katalog główny całej hierarchii klas .NET Framework, a w związku z tym nie pochodzi on z żadnych innych elementów.
