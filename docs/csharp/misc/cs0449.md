# <a name="compiler-error-cs0449"></a><span data-ttu-id="5d798-102">CS0449 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="5d798-102">Compiler Error CS0449</span></span>
<span data-ttu-id="5d798-103">Ograniczenie "class" lub "struct" musi występować przed wszystkimi innymi ograniczeniami</span><span class="sxs-lookup"><span data-stu-id="5d798-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="5d798-104">Ograniczenia parametru typu w typie ogólnym lub metodzie musi nastąpić w dowolnej kolejności: `class` lub `struct` musi być pierwszym, jeśli nie istnieje, a następnie żadnych ograniczeń interfejsu i finally żadnych ograniczeń konstruktora.</span><span class="sxs-lookup"><span data-stu-id="5d798-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="5d798-105">Ten błąd jest spowodowany przez `class` lub `struct` ograniczenia nie pojawia się pierwszy.</span><span class="sxs-lookup"><span data-stu-id="5d798-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="5d798-106">Aby rozwiązać ten problem, Zmień kolejność klauzul ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="5d798-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5d798-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="5d798-107">Example</span></span>  
 <span data-ttu-id="5d798-108">Poniższy przykład generuje CS0449.</span><span class="sxs-lookup"><span data-stu-id="5d798-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
