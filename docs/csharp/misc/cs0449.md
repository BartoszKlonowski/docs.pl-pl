# <a name="compiler-error-cs0449"></a><span data-ttu-id="08235-103">Błąd kompilatora CS0449</span><span class="sxs-lookup"><span data-stu-id="08235-103">Compiler Error CS0449</span></span>
<span data-ttu-id="08235-104">Ograniczenie "Class" lub "struct" musi występować przed wszystkimi innymi ograniczeniami</span><span class="sxs-lookup"><span data-stu-id="08235-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="08235-105">Ograniczenia dla parametru typu ogólnego lub metody muszą wystąpić w określonej kolejności: `class` lub `struct` muszą być pierwsze, jeśli istnieją, a następnie wszelkie ograniczenia interfejsu i na końcu wszelkich ograniczeń konstruktora.</span><span class="sxs-lookup"><span data-stu-id="08235-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="08235-106">Ten błąd jest spowodowany przez `class` lub `struct` nie pojawia się jako pierwszy.</span><span class="sxs-lookup"><span data-stu-id="08235-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="08235-107">Aby rozwiązać ten problem, Zmień kolejność klauzul ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="08235-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="08235-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="08235-108">Example</span></span>  
 <span data-ttu-id="08235-109">Poniższy przykład generuje CS0449.</span><span class="sxs-lookup"><span data-stu-id="08235-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
