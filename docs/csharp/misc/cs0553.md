# <a name="compiler-error-cs0553"></a>Błąd kompilatora CS0553
"Procedura konwersji": zdefiniowana przez użytkownika konwersja na/z klasy bazowej  
  
 Zdefiniowane przez użytkownika konwersje na wartości klasy bazowej nie są dozwolone; Ten operator nie jest potrzebny.  
  
 Poniższy przykład generuje CS0553:  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
