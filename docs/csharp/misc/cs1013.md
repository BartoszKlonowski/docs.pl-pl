# <a name="compiler-error-cs1013"></a>CS1013 błąd kompilatora
Nieprawidłowy numer  
  
 Kompilator wykrył liczbą źle sformułowane. Aby uzyskać więcej informacji na temat typów całkowitych, zobacz [Tabela typów całkowitych](../../csharp/language-reference/keywords/integral-types-table.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1013:  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
