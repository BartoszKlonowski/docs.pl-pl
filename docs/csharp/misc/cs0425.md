## <a name="example"></a><span data-ttu-id="b3016-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="b3016-107">Example</span></span>  
 <span data-ttu-id="b3016-108">Poniższy przykład generuje CS0425:</span><span class="sxs-lookup"><span data-stu-id="b3016-108">The following example generates CS0425:</span></span>  
  
```csharp

{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }

{  
    public static void Main()  
    {  
    }  
}  
```
