## <a name="example"></a><span data-ttu-id="d03a1-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d03a1-107">Example</span></span>  
 <span data-ttu-id="d03a1-108">Poniższy przykład generuje CS0425:</span><span class="sxs-lookup"><span data-stu-id="d03a1-108">The following example generates CS0425:</span></span>  
  
```csharp  
// CS0425.cs  
  
class C1  
{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}  
  
class Derived : IBase  
{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
