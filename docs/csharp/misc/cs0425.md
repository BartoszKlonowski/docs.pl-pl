<span data-ttu-id="14254-104">Ograniczenia dla parametru typu "Method" metody "Method" muszą być zgodne z ograniczeniami dla parametru typu "Parameter Type" metody interfejsu "Method".</span><span class="sxs-lookup"><span data-stu-id="14254-104">The constraints for type parameter 'type parameter' of method 'method' must match the constraints for type parameter 'type parameter' of interface method 'method'.</span></span> <span data-ttu-id="14254-105">Rozważ użycie jawnej implementacji interfejsu.</span><span class="sxs-lookup"><span data-stu-id="14254-105">Consider using an explicit interface implementation instead.</span></span>

## <a name="example"></a><span data-ttu-id="14254-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="14254-108">Example</span></span>  

 <span data-ttu-id="14254-109">Poniższy przykład generuje CS0425:</span><span class="sxs-lookup"><span data-stu-id="14254-109">The following example generates CS0425:</span></span>  
  
```csharp  
// CS0425.cs  
  
class C1  
{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }  
}

{  
    public static void Main()  
    {  
    }  
}  
```
