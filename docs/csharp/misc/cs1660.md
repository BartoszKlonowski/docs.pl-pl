# <a name="compiler-error-cs1660"></a><span data-ttu-id="7bb20-102">CS1660 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="7bb20-102">Compiler Error CS1660</span></span>
<span data-ttu-id="7bb20-103">Nie można przekonwertować bloku metody anonimowej na typ "type", ponieważ nie jest typem delegowanym</span><span class="sxs-lookup"><span data-stu-id="7bb20-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="7bb20-104">Ten błąd występuje, Jeśli spróbujesz przypisać lub w przeciwnym razie przekonwertować bloku metody anonimowej na typ, który nie jest typem delegowanym.</span><span class="sxs-lookup"><span data-stu-id="7bb20-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="7bb20-105">Poniższy przykład generuje CS1660:</span><span class="sxs-lookup"><span data-stu-id="7bb20-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
