# <a name="compiler-error-cs1660"></a><span data-ttu-id="cb456-103">Błąd kompilatora CS1660</span><span class="sxs-lookup"><span data-stu-id="cb456-103">Compiler Error CS1660</span></span>

<span data-ttu-id="cb456-104">Nie można przekonwertować bloku metody anonimowej na typ "Type", ponieważ nie jest to typ delegata</span><span class="sxs-lookup"><span data-stu-id="cb456-104">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="cb456-105">Ten błąd występuje w przypadku próby przypisania lub w inny sposób konwersji bloku metody anonimowej na typ, który nie jest typem delegata.</span><span class="sxs-lookup"><span data-stu-id="cb456-105">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="cb456-106">Poniższy przykład generuje CS1660:</span><span class="sxs-lookup"><span data-stu-id="cb456-106">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
