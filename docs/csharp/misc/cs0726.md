# <a name="compiler-error-cs0726"></a>CS0726 błąd kompilatora
"specyfikator formatu" nie jest prawidłowym specyfikatorem formatu  
  
 Ten błąd występuje w debugerze. Po wpisaniu nazwy zmiennej do jednego z okien debugera można to wykonać za pomocą przecinka i specyfikator formatu. Przykłady: `myInt, h` lub `myString,nq`. Ten błąd pojawia się, gdy kompilator nie może rozpoznać [specyfikatory formatu w C# ](/visualstudio/debugger/format-specifiers-in-csharp).
