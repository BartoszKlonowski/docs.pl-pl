# <a name="compiler-error-cs0726"></a>Błąd kompilatora CS0726

Specyfikator formatu nie jest prawidłowym specyfikatorem formatu  
  
 Ten błąd występuje w debugerze. Po wpisaniu nazwy zmiennej w jednym z okien debugera można wykonać ją przy użyciu przecinka, a następnie specyfikatora formatu. Przykłady: `myInt, h` lub `myString,nq` . Ten błąd występuje, gdy kompilator nie rozpoznaje [specyfikatorów formatu w języku C#](/visualstudio/debugger/format-specifiers-in-csharp).
