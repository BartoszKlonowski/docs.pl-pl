 <span data-ttu-id="bb0c8-104">`#define` i `#undef` [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md) musi być używana na początku programu, przed żadnych innych słów kluczowych, takich jak używane w deklaracji przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="bb0c8-104">The `#define` and `#undef` [preprocessor directives](../../csharp/language-reference/preprocessor-directives/index.md) must be used at the beginning of a program, before any other keywords, such as those used in the namespace declaration.</span></span>  
  
 <span data-ttu-id="bb0c8-105">Poniższy przykład spowoduje wygenerowanie CS1032:</span><span class="sxs-lookup"><span data-stu-id="bb0c8-105">The following sample generates CS1032:</span></span>  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
