# <a name="compiler-error-cs1032"></a>CS1032 błąd kompilatora
Nie można zdefiniować/usunąć definicji symboli preprocesora po pierwszym tokenie w pliku  
  
 `#define` i `#undef` [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md) musi być używana na początku programu, przed żadnych innych słów kluczowych, takich jak używane w deklaracji przestrzeni nazw.  
  
 Poniższy przykład spowoduje wygenerowanie CS1032:  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
