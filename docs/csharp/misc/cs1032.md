# <a name="compiler-error-cs1032"></a>Błąd kompilatora CS1032
Nie można zdefiniować/rozdefiniować symboli preprocesora po pierwszym tokenie w pliku  
  
 `#define` `#undef` [Dyrektywy preprocesora](../language-reference/preprocessor-directives/index.md) i należy używać na początku programu, przed wszelkimi innymi słowami kluczowymi, takimi jak te używane w deklaracji przestrzeni nazw.  
  
 Poniższy przykład generuje CS1032:  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
