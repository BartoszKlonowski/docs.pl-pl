# <a name="compiler-error-cs0176"></a><span data-ttu-id="acb00-103">Błąd kompilatora CS0176</span><span class="sxs-lookup"><span data-stu-id="acb00-103">Compiler Error CS0176</span></span>
<span data-ttu-id="acb00-104">Nie można uzyskać dostępu do statycznej składowej "member" z odwołaniem do wystąpienia; Zakwalifikuj go zamiast nazwy typu</span><span class="sxs-lookup"><span data-stu-id="acb00-104">Static member 'member' cannot be accessed with an instance reference; qualify it with a type name instead</span></span>  
  
 <span data-ttu-id="acb00-105">Tylko nazwa klasy może być używana do kwalifikowania zmiennej [statycznej](../language-reference/keywords/static.md) ; Nazwa wystąpienia nie może być kwalifikatorem.</span><span class="sxs-lookup"><span data-stu-id="acb00-105">Only a class name can be used to qualify a [static](../language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="acb00-106">Aby uzyskać więcej informacji, zobacz [klasy statyczne i statyczne elementy członkowskie klas](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="acb00-106">For more information, see [Static Classes and Static Class Members](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="acb00-107">Poniższy przykład generuje CS0176:</span><span class="sxs-lookup"><span data-stu-id="acb00-107">The following sample generates CS0176:</span></span>  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
