# <a name="compiler-error-cs0176"></a>CS0176 błąd kompilatora
Za pomocą odwołania do wystąpienia; nie można uzyskać dostępu do statycznej składowej 'składowa' zakwalifikować go z nazwą typu  
  
 Nazwa klasy może służyć do kwalifikowania [statyczne](../../csharp/language-reference/keywords/static.md) zmiennej; Nazwa wystąpienia nie może mieć kwalifikator. Aby uzyskać więcej informacji, zobacz [klasy statyczne i statyczne elementy członkowskie klasy](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0176:  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
