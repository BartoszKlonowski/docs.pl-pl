# <a name="compiler-error-cs1594"></a><span data-ttu-id="60197-103">Błąd kompilatora CS1594</span><span class="sxs-lookup"><span data-stu-id="60197-103">Compiler Error CS1594</span></span>
<span data-ttu-id="60197-104">Delegat "delegat" ma kilka nieprawidłowych argumentów</span><span class="sxs-lookup"><span data-stu-id="60197-104">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="60197-105">Typ argumentu przesłanego do wywołania [delegata](../language-reference/builtin-types/reference-types.md) nie zgadza się z typem parametru w deklaracji delegata.</span><span class="sxs-lookup"><span data-stu-id="60197-105">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="60197-106">Poniższy przykład generuje CS1594:</span><span class="sxs-lookup"><span data-stu-id="60197-106">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
