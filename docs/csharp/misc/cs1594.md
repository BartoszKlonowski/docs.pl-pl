# <a name="compiler-error-cs1594"></a><span data-ttu-id="a3402-102">CS1594 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="a3402-102">Compiler Error CS1594</span></span>
<span data-ttu-id="a3402-103">Delegat "delegowanie" ma nieprawidłowe argumenty.</span><span class="sxs-lookup"><span data-stu-id="a3402-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="a3402-104">Typ argumentu przekazanego [delegować](../../csharp/language-reference/keywords/delegate.md) wywołania nie zgadza się z typem parametru w deklaracja delegata.</span><span class="sxs-lookup"><span data-stu-id="a3402-104">The type of an argument passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="a3402-105">Poniższy przykład spowoduje wygenerowanie CS1594:</span><span class="sxs-lookup"><span data-stu-id="a3402-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
