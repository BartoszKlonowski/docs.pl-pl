# <a name="compiler-error-cs1594"></a>CS1594 błąd kompilatora
Delegat "delegowanie" ma nieprawidłowe argumenty.  
  
 Typ argumentu przekazanego [delegować](../../csharp/language-reference/keywords/delegate.md) wywołania nie zgadza się z typem parametru w deklaracja delegata.  
  
 Poniższy przykład spowoduje wygenerowanie CS1594:  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
