# <a name="compiler-error-cs1556"></a>Błąd kompilatora CS1556

element "konstrukcja" określony dla metody Main musi być prawidłową klasą lub strukturą  
  
 [-Main —](../language-reference/compiler-options/main-compiler-option.md) opcja kompilatora została przeniesiona jako identyfikator, który nie jest nazwą klasy.
