# <a name="compiler-error-cs1556"></a>CS1556 błąd kompilatora
"konstruowania" określona dla metody Main musi być prawidłową klasą lub strukturą  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) — opcja kompilatora został przekazany identyfikator, który nie jest nazwa klasy.
