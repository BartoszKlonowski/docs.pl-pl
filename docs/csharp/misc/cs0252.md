# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="19e6e-103">Ostrzeżenie kompilatora (poziom 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="19e6e-103">Compiler Warning (level 2) CS0252</span></span>

<span data-ttu-id="19e6e-104">Możliwe niezamierzone porównanie odwołań; Aby uzyskać porównanie wartości, Rzutowanie lewej strony na typ "typ"</span><span class="sxs-lookup"><span data-stu-id="19e6e-104">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="19e6e-105">Kompilator wykonuje porównanie odwołań.</span><span class="sxs-lookup"><span data-stu-id="19e6e-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="19e6e-106">Jeśli chcesz porównać wartość ciągów, Rzutowanie lewej strony wyrażenia na `type` .</span><span class="sxs-lookup"><span data-stu-id="19e6e-106">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="19e6e-107">Poniższy przykład generuje CS0252:</span><span class="sxs-lookup"><span data-stu-id="19e6e-107">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
