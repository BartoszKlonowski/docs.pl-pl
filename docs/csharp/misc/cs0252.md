# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="a630e-102">CS0252 ostrzeżenie (poziom 2) kompilator</span><span class="sxs-lookup"><span data-stu-id="a630e-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="a630e-103">Możliwe niezamierzone porównanie odwołań. Aby porównać wartości, wykonaj rzutowanie lewej na typ "type"</span><span class="sxs-lookup"><span data-stu-id="a630e-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="a630e-104">Kompilator robi porównanie odwołań.</span><span class="sxs-lookup"><span data-stu-id="a630e-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="a630e-105">Jeśli chcesz porównania wartości ciągów rzutowanie po lewej stronie wyrażenia, aby `type`.</span><span class="sxs-lookup"><span data-stu-id="a630e-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="a630e-106">Poniższy przykład spowoduje wygenerowanie CS0252:</span><span class="sxs-lookup"><span data-stu-id="a630e-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
