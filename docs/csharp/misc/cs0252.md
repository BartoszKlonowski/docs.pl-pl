# <a name="compiler-warning-level-2-cs0252"></a>Ostrzeżenie kompilatora (poziom 2) CS0252
Możliwe niezamierzone porównanie odwołań; Aby uzyskać porównanie wartości, Rzutowanie lewej strony na typ "typ"  
  
 Kompilator wykonuje porównanie odwołań. Jeśli chcesz porównać wartość ciągów, Rzutowanie lewej strony wyrażenia na `type` .  
  
 Poniższy przykład generuje CS0252:  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
