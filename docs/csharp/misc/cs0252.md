# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="47c46-102">Kompilator CS0252 ostrzegawcze (poziom 2)</span><span class="sxs-lookup"><span data-stu-id="47c46-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="47c46-103">Możliwe niezamierzone porównanie odwołań. Aby uzyskać porównanie wartości, rzutowania lewej strony na typ 'type'</span><span class="sxs-lookup"><span data-stu-id="47c46-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="47c46-104">Kompilator jest wykonywanie porównanie odwołań.</span><span class="sxs-lookup"><span data-stu-id="47c46-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="47c46-105">Jeśli chcesz porównać wartości ciągów rzutowania lewej strony wyrażenia do `type`.</span><span class="sxs-lookup"><span data-stu-id="47c46-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="47c46-106">Poniższy przykład generuje CS0252:</span><span class="sxs-lookup"><span data-stu-id="47c46-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
