# <a name="compiler-error-cs1514"></a>CS1514 błąd kompilatora
{Oczekiwano  
  
 Kompilator oczekiwano otwierający nawias klamrowy (`{`) nie został znaleziony.  
  
 Poniższy przykład generuje CS1514:  
  
```  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
