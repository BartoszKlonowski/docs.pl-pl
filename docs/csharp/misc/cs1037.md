# <a name="compiler-error-cs1037"></a>Błąd kompilatora CS1037
Oczekiwano operatora z możliwością przeciążenia  
  
 Podczas określania komentarza z [/doc](../language-reference/compiler-options/doc-compiler-option.md), kompilator napotkał nieprawidłowe łącze.
