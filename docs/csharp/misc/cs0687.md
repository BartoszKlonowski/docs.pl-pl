## <a name="example"></a><span data-ttu-id="f36a2-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="f36a2-109">Example</span></span>  

 <span data-ttu-id="f36a2-110">Poniższy przykład generuje CS0687:</span><span class="sxs-lookup"><span data-stu-id="f36a2-110">The following sample generates CS0687:</span></span>

public class Test
{  
    public static int x = 77;  
  
    public static void Main()
    {
