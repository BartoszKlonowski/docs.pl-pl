# <a name="compiler-error-cs0155"></a>CS0155 błąd kompilatora
Przechwycony lub zgłoszony typ musi pochodzić od klasy System.Exception  
  
 Próbowano przekazać typ danych, który nie pochodzi od **System.Exception** do [catch](../../csharp/language-reference/keywords/try-catch.md) bloku. Tylko typy danych, które wynikają z **System.Exception** mogą być przekazywane do **catch** bloku. Aby uzyskać więcej informacji, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0155:  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
