<span data-ttu-id="3dbd9-103">Przechwycony lub zgłoszony typ musi pochodzić od klasy System.Exception</span><span class="sxs-lookup"><span data-stu-id="3dbd9-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="3dbd9-104">Podjęto próbę przekazania typu danych, który nie pochodzi od **System.Exception** do [catch](../../csharp/language-reference/keywords/try-catch.md) bloku.</span><span class="sxs-lookup"><span data-stu-id="3dbd9-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="3dbd9-105">Tylko typy danych, które pochodzą z **System.Exception** mogą zostać przekazane do **catch** bloku.</span><span class="sxs-lookup"><span data-stu-id="3dbd9-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="3dbd9-106">Aby uzyskać więcej informacji, zobacz [instrukcje obsługi wyjątków](../../csharp/language-reference/keywords/exception-handling-statements.md) i [wyjątki i obsługa wyjątków](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="3dbd9-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="3dbd9-107">Poniższy przykład generuje CS0155:</span><span class="sxs-lookup"><span data-stu-id="3dbd9-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
