<span data-ttu-id="7ac76-104">Przechwycony lub zgłoszony typ musi pochodzić od klasy System. Exception</span><span class="sxs-lookup"><span data-stu-id="7ac76-104">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="7ac76-105">Podjęto próbę przekazania typu danych, który nie pochodzi od elementu **System. Exception** do bloku [catch](../language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="7ac76-105">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="7ac76-106">Tylko typy danych pochodzące z elementu **System. Exception** mogą być przesyłane do bloku **catch** .</span><span class="sxs-lookup"><span data-stu-id="7ac76-106">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="7ac76-107">Aby uzyskać więcej informacji, zobacz [wyjątki i obsługa wyjątków](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="7ac76-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="7ac76-108">Poniższy przykład generuje CS0155:</span><span class="sxs-lookup"><span data-stu-id="7ac76-108">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
