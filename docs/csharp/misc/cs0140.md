# <a name="compiler-error-cs0140"></a>CS0140 błąd kompilatora
Etykieta "etykieta" jest duplikatem  
  
 Etykieta o takiej samej nazwie pojawiły się dwa razy. Aby uzyskać więcej informacji, zobacz [goto](../../csharp/language-reference/keywords/goto.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0140:  
  
```csharp  
// CS0140.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         label1: int i = 0;  
         label1: int j = 0;   // CS0140, comment this line to resolve  
         goto label1;  
      }  
   }  
}  
```
