<span data-ttu-id="18d71-104">Typ "type1" nie może być używany jako parametr typu "name" w typie ogólnym lub metodzie "name".</span><span class="sxs-lookup"><span data-stu-id="18d71-104">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="18d71-105">Nie istnieje konwersja ani konwersja parametrów typu z "type1" na "type2".</span><span class="sxs-lookup"><span data-stu-id="18d71-105">There is no boxing conversion or type parameter conversion from 'type1' to 'type2'.</span></span>

## <a name="example"></a><span data-ttu-id="18d71-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="18d71-109">Example</span></span>  

 <span data-ttu-id="18d71-110">Poniższy kod generuje CS0314:</span><span class="sxs-lookup"><span data-stu-id="18d71-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
