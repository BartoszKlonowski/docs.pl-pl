 <span data-ttu-id="137ef-105">Gdy typem ogólnym używa parametr typu, który jest ograniczony, nową klasę również musi spełniać te tych samych ograniczeń.</span><span class="sxs-lookup"><span data-stu-id="137ef-105">When a generic type uses a type parameter that is constrained, the new class must also satisfy those same constraints.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="137ef-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="137ef-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="137ef-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="137ef-108">Example</span></span>  
 <span data-ttu-id="137ef-109">Poniższy kod generuje CS0314:</span><span class="sxs-lookup"><span data-stu-id="137ef-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
