<span data-ttu-id="3885b-104">Metoda "name" nie jest metodą rodzajową.</span><span class="sxs-lookup"><span data-stu-id="3885b-104">The method 'name' is not a generic method.</span></span> <span data-ttu-id="3885b-105">Jeśli zamierzam listę wyrażeń, użyj nawiasów wokół wyrażenia <.</span><span class="sxs-lookup"><span data-stu-id="3885b-105">If you intended an expression list, use parentheses around the < expression.</span></span>

## <a name="example"></a><span data-ttu-id="3885b-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="3885b-107">Example</span></span>
 <span data-ttu-id="3885b-108">Poniższy przykład generuje CS0471:</span><span class="sxs-lookup"><span data-stu-id="3885b-108">The following example generates CS0471:</span></span>

```csharp
// CS0471.cs
// compile with: /t:library
class Test
{
    public void F(bool x, bool y) {}
    public void F1()
    {
        int a = 1, b = 2, c = 3;
        F(a<b, c>(3));    // CS0471
        // To resolve, try the following instead:
        // F((a<b), c>(3));
    }
}
```
