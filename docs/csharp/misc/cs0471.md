<span data-ttu-id="52801-103">Metoda "name" nie jest metodą rodzajową.</span><span class="sxs-lookup"><span data-stu-id="52801-103">The method 'name' is not a generic method.</span></span> <span data-ttu-id="52801-104">Jeśli zamierzam listę wyrażeń, użyj nawiasów wokół wyrażenia <.</span><span class="sxs-lookup"><span data-stu-id="52801-104">If you intended an expression list, use parentheses around the < expression.</span></span>

## <a name="example"></a><span data-ttu-id="52801-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="52801-106">Example</span></span>
 <span data-ttu-id="52801-107">Poniższy przykład generuje CS0471:</span><span class="sxs-lookup"><span data-stu-id="52801-107">The following example generates CS0471:</span></span>

```csharp
// CS0471.cs
// compile with: /t:library
class Test
{
    public void F(bool x, bool y) {}
    public void F1()
    {
        int a = 1, b = 2, c = 3;
        F(a<b, c>(3));    // CS0471
        // To resolve, try the following instead:
        // F((a<b), c>(3));
    }
}
```
