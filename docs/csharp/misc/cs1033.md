# <a name="compiler-error-cs1033"></a>CS1033 błąd kompilatora
Plik źródłowy przekroczył limit 16 707 565 wierszy reprezentowanych w pliku PDB; informacje o debugowaniu będą niepoprawne  
  
 Plik kodu źródłowego przekracza maksymalną dozwoloną liczbę wierszy, które kompilator może przetwarzać. Aby rozwiązać ten problem, należy utworzyć dwa lub więcej plików kodu źródłowego z oryginalnego pliku. Maksymalna liczba wierszy jest 268,435,454 wierszy. Jeśli używasz **/debug**, przy użyciu więcej niż 16,707,566 spowoduje informacje debugowania uszkodzone.
