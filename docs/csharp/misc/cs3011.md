# <a name="compiler-warning-level-1-cs3011"></a>Kompilatora (poziom 1) ostrzeżenie CS3011
"członek": tylko składowe zgodne ze specyfikacją CLS mogą być abstrakcyjne  
  
 Element członkowski klasy nie może być jednocześnie [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i jest to niezgodne z Common Language Specification (CLS). Specyfikacja CLS określa wszystkich elementach członkowskich klasy są wykonywane. Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3011:  
  
```csharp  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
