## <a name="example"></a><span data-ttu-id="c7e6b-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="c7e6b-106">Example</span></span>

```csharp

public class Test  
{  
    [return:System.CLSCompliant(true)]  // CS3023

    // [method:System.CLSCompliant(true)]

    {  
        return 0;  
    }  
}  
```
