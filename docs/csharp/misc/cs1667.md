# <a name="compiler-error-cs1667"></a>CS1667 błąd kompilatora
Atrybut "attribute" nie jest prawidłowy w metodach dostępu właściwości lub zdarzenia. Jest on prawidłowy w deklaracji "deklaracji typu" tylko.  
  
 Ten błąd występuje, użycie atrybutu dla metody dostępu właściwości lub zdarzenia, gdy powinna znajdować się na właściwość lub zdarzenie, sam. Ten błąd może wystąpić z atrybutami <xref:System.CLSCompliantAttribute>, <xref:System.Diagnostics.ConditionalAttribute>, i <xref:System.ObsoleteAttribute>.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1670:  
  
```csharp  
// CS1667.cs  
using System;  
  
public class C  
{  
    private int i;  
  
    //Try this instead:  
    //[Obsolete]  
    public int ObsoleteProperty  
    {  
        [Obsolete]  // CS1667  
        get { return i; }  
        set { i = value; }  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
