# <a name="compiler-error-cs1563"></a>CS1563 błąd kompilatora
Wyjściowy 'Plik wyjściowy' nie ma żadnych plików źródłowych  
  
 [/Out](../../csharp/language-reference/compiler-options/out-compiler-option.md) określono opcję kompilatora, ale wykonać nie plików kodu źródłowego. Albo nie powinno się używać **/out**, lub należy określić w plikach kodu źródłowego, zgodnie z **/out**.
