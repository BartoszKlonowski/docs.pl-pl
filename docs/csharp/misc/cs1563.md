# <a name="compiler-error-cs1563"></a>Błąd kompilatora CS1563
Wyjściowy plik wyjściowy nie zawiera żadnych plików źródłowych  
  
 Określono opcję kompilatora [/out](../language-reference/compiler-options/out-compiler-option.md) , ale nie są one zgodne z plikami kodu źródłowego. Nie należy używać **/out**lub należy określić pliki kodu źródłowego po **/out**.
