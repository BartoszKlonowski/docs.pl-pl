# <a name="compiler-warning-level-1-cs1592"></a>Kompilator CS1592 ostrzegawcze (poziom 1)
Nieprawidłowo sformułowany kod XML w dołączonym pliku komentarzy — "Przyczyna"  
  
 Problem zgłoszony jako ***Przyczyna***, można odnaleźć pliku określonego przez [ \<obejmują >](../../csharp/programming-guide/xmldoc/include.md) tagu.
