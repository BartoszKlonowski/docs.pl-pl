# <a name="compiler-error-cs0013"></a>CS0013 błąd kompilatora
Nieoczekiwany błąd podczas zapisywania metadanych w pliku "ModelStore.dll"--"nie pozostawiono logicznego miejsca do utworzenia większej liczby ciągów użytkownika.  
  
 .NET Framework środowiska CLR nie Emituj metadanych. Sprawdź, upewnij się, że ścieżka jest poprawna i że dysk nie jest zapełniony. Jeśli problem będzie nadal występował, może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio i/lub .NET Framework.
