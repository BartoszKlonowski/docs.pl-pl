# <a name="compiler-error-cs0013"></a>Błąd kompilatora CS0013
Nieoczekiwany błąd podczas zapisywania metadanych do pliku "ModelStore.dll" — "nie ma pustej przestrzeni logicznej, aby utworzyć więcej ciągów użytkownika.  
  
 Środowisko uruchomieniowe platformy .NET nie emituje metadanych. Upewnij się, że ścieżka jest poprawna i że dysk nie jest zapełniony. Jeśli problem będzie nadal występował, może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio i/lub .NET.
