# <a name="compiler-error-cs0013"></a>CS0013 błąd kompilatora
Nieoczekiwany błąd podczas zapisywania metadanych w pliku "ModelStore.dll" — "nie ma logiczny wolnego miejsca do utworzenia więcej ciągów użytkownika.  
  
 .NET Framework środowisko uruchomieniowe języka wspólnego nie emitować metadane. Zaznacz, aby upewnić się, że ścieżka jest poprawna i że dysk nie jest zapełniony. Jeśli problem będzie się powtarzać, może być konieczne naprawienie lub ponowne zainstalowanie programu Visual Studio i/lub .NET Framework.
