# <a name="compiler-error-cs0206"></a>CS0206 błąd kompilatora
Właściwość lub indeksator nie można przekazać jako out lub ref parametru  
  
 A [właściwość](../../csharp/programming-guide/classes-and-structs/properties.md) nie jest dostępna do przekazania jako [ref](../../csharp/language-reference/keywords/ref.md) lub [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru. Aby uzyskać więcej informacji, zobacz [przekazanie parametrów](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0206:  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
