<span data-ttu-id="94914-103">Prawdopodobnie omyłkowo wystąpiła pusta instrukcja</span><span class="sxs-lookup"><span data-stu-id="94914-103">Possible mistaken empty statement</span></span>  
  
 <span data-ttu-id="94914-104">Średnik po instrukcji warunkowej może spowodować, że kod będzie wykonywany inaczej niż zamierzone.</span><span class="sxs-lookup"><span data-stu-id="94914-104">A semicolon after a conditional statement may cause your code to execute differently than intended.</span></span>

 <span data-ttu-id="94914-106">Poniższy przykład generuje CS0642:</span><span class="sxs-lookup"><span data-stu-id="94914-106">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
