# <a name="compiler-error-cs0574"></a>CS0574 błąd kompilatora
Nazwa destruktora musi odpowiadać Nazwa klasy  
  
 Nazwa destruktora musi być nazwę klasy, poprzedzoną znakiem tyldy (~).  
  
 Poniższy przykład spowoduje wygenerowanie CS0574:  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
