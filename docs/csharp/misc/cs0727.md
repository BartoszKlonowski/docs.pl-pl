# <a name="compiler-error-cs0727"></a>CS0727 błąd kompilatora
Nieprawidłowy format Specyfikator  
  
 Ten błąd występuje w debugerze. Po wpisaniu nazwy zmiennej do jednego z okien debugera, można to wykonać przy przecinkami, a następnie specyfikator formatu. Przykłady: myInt h; lub mójCiąg, nq. Ten błąd pojawia się, gdy kompilator jest całkowicie nie można przeanalizować wpisane w. Aby rozwiązać ten problem, ponownie nazwę zmiennej, i opcjonalnie przecinek i [Nieprawidłowy specyfikator formatu](/visualstudio/debugger/format-specifiers-in-csharp).
