# <a name="compiler-error-cs0727"></a>Błąd kompilatora CS0727
Nieprawidłowy specyfikator formatu  
  
 Ten błąd występuje w debugerze. Po wpisaniu nazwy zmiennej w jednym z okien debugera można wykonać ją przy użyciu przecinka, a następnie specyfikatora formatu. Przykłady: myInt, h; lub parametr NQ. Ten błąd występuje, gdy kompilator nie może przeanalizować wpisanych informacji. Aby rozwiązać ten problem, wpisz ponownie nazwę zmiennej, opcjonalnie po której następuje przecinek i [prawidłowy specyfikator formatu](/visualstudio/debugger/format-specifiers-in-csharp).
