# <a name="compiler-error-cs0037"></a>Błąd kompilatora CS0037
Nie można przekonwertować wartości null na typ "Type", ponieważ jest to typ wartości niedopuszczający wartości null.  
  
 Kompilator nie może przypisywać wartości null do typu wartościowego; wartość null można przypisać tylko do [typu odwołania](../language-reference/keywords/reference-types.md) lub do typu dopuszczającego wartość null. [Struktura](../language-reference/keywords/struct.md) jest typem wartości. Aby uzyskać więcej informacji, zobacz [Typy dopuszczające wartości null](../programming-guide/nullable-types/index.md).  
  
 Poniższy przykład generuje CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
