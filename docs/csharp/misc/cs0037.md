# <a name="compiler-error-cs0037"></a>Błąd kompilatora CS0037
Nie można przekonwertować wartości null na typ "Type", ponieważ jest to typ wartości niedopuszczający wartości null.  
  
 Kompilator nie może przypisać `null` do typu wartości; `null` można przypisać tylko do [typu odwołania](../language-reference/keywords/reference-types.md) lub do [typu wartości null](../programming-guide/nullable-types/index.md). [Struktura](../language-reference/keywords/struct.md) jest typem wartości.
  
 Poniższy przykład generuje CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
