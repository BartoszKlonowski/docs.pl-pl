# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="feb76-102">Kompilatora (poziom 3) ostrzeżenie CS0659</span><span class="sxs-lookup"><span data-stu-id="feb76-102">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="feb76-103">"class" przesłania metodę Object.Equals (object o), ale nie przesłania metody Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="feb76-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="feb76-104">Kompilator wykrył nadpisanie <xref:System.Object.Equals%2A?displayProperty=nameWithType> metoda, ale nie zastępowania <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> metody.</span><span class="sxs-lookup"><span data-stu-id="feb76-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="feb76-105">Zastępowanie <xref:System.Object.Equals%2A> wskazuje, czy też chcesz zastąpić <xref:System.Object.GetHashCode%2A>.</span><span class="sxs-lookup"><span data-stu-id="feb76-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="feb76-106">Poniższy kod generuje CS0659:</span><span class="sxs-lookup"><span data-stu-id="feb76-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
