# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="df6f9-103">Ostrzeżenie kompilatora (poziom 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="df6f9-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="df6f9-104">element "Class" przesłania metodę Object. Equals (Object o), ale nie przesłania obiektu Object. GetHashCode ()</span><span class="sxs-lookup"><span data-stu-id="df6f9-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="df6f9-105">Kompilator wykrył przesłonięcie metody, <xref:System.Object.Equals%2A?displayProperty=nameWithType> ale nie przesłania <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> metody.</span><span class="sxs-lookup"><span data-stu-id="df6f9-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="df6f9-106">Przesłonięcie <xref:System.Object.Equals%2A> oznacza, że chcesz również przesłonić <xref:System.Object.GetHashCode%2A> .</span><span class="sxs-lookup"><span data-stu-id="df6f9-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="df6f9-107">Poniższy kod generuje CS0659:</span><span class="sxs-lookup"><span data-stu-id="df6f9-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
