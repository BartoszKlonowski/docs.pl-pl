# <a name="compiler-warning-level-1-cs3015"></a>Ostrzeżenie kompilatora (poziom 1) CS3015

"sygnatura metody" nie ma dostępnych konstruktorów używających tylko typów zgodnych ze specyfikacją CLS  
  
 Aby była zgodna z Common Language Specification (CLS), lista argumentów klasy atrybutu nie może zawierać tablicy. Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje C3015.  
  
```csharp  
// CS3015.cs  
// compile with: /target:library  
using System;  
  
[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
