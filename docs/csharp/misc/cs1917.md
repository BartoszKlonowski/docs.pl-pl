<span data-ttu-id="f2323-103">Składowych pola tylko do odczytu "name" typu "struct Name" nie można przypisać za pomocą inicjatora obiektów, ponieważ jest on typu wartości.</span><span class="sxs-lookup"><span data-stu-id="f2323-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>

- <span data-ttu-id="f2323-107">Zainicjuj strukturę z konstruktorem.</span><span class="sxs-lookup"><span data-stu-id="f2323-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f2323-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="f2323-108">Example</span></span>  

 <span data-ttu-id="f2323-109">Poniższy kod generuje CS1917:</span><span class="sxs-lookup"><span data-stu-id="f2323-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
