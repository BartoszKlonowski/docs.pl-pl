ParamArrayattribute".</span><span class="sxs-lookup"><span data-stu-id="2630c-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="2630c-105">Użyj zamiast tego słowa kluczowego "params".</span><span class="sxs-lookup"><span data-stu-id="2630c-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="2630c-106">Kompilator języka C# nie zezwala na użycie polecenia <xref:System.ParamArrayAttribute?displayProperty=nameWithType> ; zamiast tego użyj [parametrów](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="2630c-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="2630c-107">Poniższy przykład generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="2630c-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
