# <a name="compiler-error-cs0674"></a><span data-ttu-id="cbe6b-102">Błąd kompilatora CS0674</span><span class="sxs-lookup"><span data-stu-id="cbe6b-102">Compiler Error CS0674</span></span>
<span data-ttu-id="cbe6b-103">Nie należy używać atrybutu "System.ParamArrayAttribute".</span><span class="sxs-lookup"><span data-stu-id="cbe6b-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="cbe6b-104">Zamiast tego użyj słowa kluczowego "params".</span><span class="sxs-lookup"><span data-stu-id="cbe6b-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="cbe6b-105">Kompilator Języka C# nie <xref:System.ParamArrayAttribute?displayProperty=nameWithType>zezwala na użycie ; zamiast tego użyj [paramów.](../language-reference/keywords/params.md)</span><span class="sxs-lookup"><span data-stu-id="cbe6b-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="cbe6b-106">Poniższy przykład generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="cbe6b-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
