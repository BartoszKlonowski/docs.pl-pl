# <a name="compiler-error-cs0674"></a><span data-ttu-id="4e59a-102">CS0674 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="4e59a-102">Compiler Error CS0674</span></span>
<span data-ttu-id="4e59a-103">Nie używaj atrybutu "System.ParamArrayAttribute".</span><span class="sxs-lookup"><span data-stu-id="4e59a-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="4e59a-104">Zamiast tego użyj słowa kluczowego "params".</span><span class="sxs-lookup"><span data-stu-id="4e59a-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="4e59a-105">Kompilator języka C# nie zezwala na korzystanie z <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; użyj [params](../../csharp/language-reference/keywords/params.md) zamiast tego.</span><span class="sxs-lookup"><span data-stu-id="4e59a-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="4e59a-106">Poniższy przykład generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="4e59a-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
