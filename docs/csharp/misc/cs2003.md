# <a name="compiler-error-cs2003"></a>Błąd kompilatora CS2003
Plik odpowiedzi "File" został uwzględniony wiele razy  
  
 Plik odpowiedzi został przekazano do kompilatora więcej niż raz. Plik odpowiedzi można przesłać do kompilatora tylko raz dla pliku wyjściowego.  
  
 Aby uzyskać więcej informacji na temat plików odpowiedzi, zobacz [@ (Określ plik odpowiedzi)](../language-reference/compiler-options/response-file-compiler-option.md).
