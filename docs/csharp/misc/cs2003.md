# <a name="compiler-error-cs2003"></a>CS2003 błąd kompilatora
Plik odpowiedzi 'Plik' został dołączony wielokrotnie  
  
 Plik odpowiedzi został przekazany do kompilatora więcej niż raz. Plik odpowiedzi mogą być przekazywane do kompilatora raz w pliku wyjściowym.  
  
 Aby uzyskać więcej informacji na temat plików odpowiedzi, zobacz [@ (Określ plik odpowiedzi)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
