# <a name="compiler-warning-level-2-cs1587"></a>CS1587 ostrzeżenie (poziom 2) kompilator
Komentarz XML nie został umieszczony w prawidłowym elemencie języka  
  
 Tagi zalecane dla komentarzy do dokumentacji nie są dozwolone w przypadku wszystkich elementów języka. Na przykład tag nie jest dozwolona w przestrzeni nazw. Aby uzyskać więcej informacji na temat komentarzy XML, zobacz [tagi zalecane dla komentarzy do dokumentacji](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1587:  
  
```csharp  
// CS1587.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>test</summary>   // CS1587, tag not allowed on namespace  
namespace MySpace  
{  
    class MyClass  
    {  
        public static void Main()  
        {  
        }  
    }  
}  
```
