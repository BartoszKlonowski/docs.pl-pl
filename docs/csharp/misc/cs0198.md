# <a name="compiler-error-cs0198"></a><span data-ttu-id="3a826-102">Błąd kompilatora CS0198</span><span class="sxs-lookup"><span data-stu-id="3a826-102">Compiler Error CS0198</span></span>
<span data-ttu-id="3a826-103">Nie można przypisać pól statycznego pola tylko do odczytu "name" (z wyjątkiem: w konstruktorze statycznym lub inicjatorze zmiennych).</span><span class="sxs-lookup"><span data-stu-id="3a826-103">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="3a826-104">Zmienna [ReadOnly](../language-reference/keywords/readonly.md) musi mieć takie samo użycie [statyczne](../language-reference/keywords/static.md) jak Konstruktor, w którym ma zostać zainicjowany.</span><span class="sxs-lookup"><span data-stu-id="3a826-104">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="3a826-105">Aby uzyskać więcej informacji, zobacz [statyczne konstruktory](../programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="3a826-105">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="3a826-106">Poniższy przykład generuje CS0198:</span><span class="sxs-lookup"><span data-stu-id="3a826-106">The following sample generates CS0198:</span></span>  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
