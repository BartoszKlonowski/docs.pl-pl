# <a name="compiler-error-cs0198"></a>CS0198 błąd kompilatora
Nie można przypisać pól pola statycznego tylko do odczytu "name" (z wyjątkiem w konstruktorze statycznym lub inicjatorze zmiennych)  
  
 A [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) zmienna musi mieć takie same [statyczne](../../csharp/language-reference/keywords/static.md) użycia jako konstruktor, w którym chcesz go zainicjować. Aby uzyskać więcej informacji, zobacz [konstruktorów statycznych](../../csharp/programming-guide/classes-and-structs/static-constructors.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0198:  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
