# <a name="compiler-error-cs0198"></a><span data-ttu-id="67125-102">CS0198 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="67125-102">Compiler Error CS0198</span></span>
<span data-ttu-id="67125-103">Nie można przypisać pól pola statycznego tylko do odczytu "name" (z wyjątkiem w konstruktorze statycznym lub inicjatorze zmiennych)</span><span class="sxs-lookup"><span data-stu-id="67125-103">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="67125-104">A [tylko do odczytu](../../csharp/language-reference/keywords/readonly.md) zmienna musi mieć takie same [statyczne](../../csharp/language-reference/keywords/static.md) użycia jako konstruktor, w którym chcesz go zainicjować.</span><span class="sxs-lookup"><span data-stu-id="67125-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="67125-105">Aby uzyskać więcej informacji, zobacz [konstruktorów statycznych](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="67125-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="67125-106">Poniższy przykład spowoduje wygenerowanie CS0198:</span><span class="sxs-lookup"><span data-stu-id="67125-106">The following sample generates CS0198:</span></span>  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
