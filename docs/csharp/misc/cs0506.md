# <a name="compiler-error-cs0506"></a><span data-ttu-id="92c18-102">CS0506 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="92c18-102">Compiler Error CS0506</span></span>
<span data-ttu-id="92c18-103">"function1": nie można przesłonić dziedziczonego elementu członkowskiego "function2" z ponieważ nie jest oznaczony "virtual", "abstract" ani "override"</span><span class="sxs-lookup"><span data-stu-id="92c18-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="92c18-104">Metoda została zastąpiona, który nie został jawnie oznaczone jako **wirtualnego**, **abstrakcyjny**, lub `override`.</span><span class="sxs-lookup"><span data-stu-id="92c18-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="92c18-105">Poniższy przykład generuje CS0506:</span><span class="sxs-lookup"><span data-stu-id="92c18-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
