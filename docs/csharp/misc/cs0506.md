# <a name="compiler-error-cs0506"></a><span data-ttu-id="45d1c-102">CS0506 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="45d1c-102">Compiler Error CS0506</span></span>
<span data-ttu-id="45d1c-103">"function1": nie można przesłonić dziedziczonej składowej "function2" z ponieważ nie jest ona oznaczona "wirtualne", "abstract" lub "override"</span><span class="sxs-lookup"><span data-stu-id="45d1c-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="45d1c-104">Metoda została zastąpiona, która nie została jawnie oznaczona jako **wirtualnego**, **abstrakcyjne**, lub `override`.</span><span class="sxs-lookup"><span data-stu-id="45d1c-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="45d1c-105">Poniższy przykład spowoduje wygenerowanie CS0506:</span><span class="sxs-lookup"><span data-stu-id="45d1c-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
