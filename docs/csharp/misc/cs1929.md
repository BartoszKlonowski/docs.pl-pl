 <span data-ttu-id="97434-104">Ten błąd jest generowany przy próbie wywołania metody rozszerzenia z klasy, która nie obejmuje.</span><span class="sxs-lookup"><span data-stu-id="97434-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="97434-105">W tym przykładzie, metoda rozszerzenia jest zdefiniowana w klasie pochodnej `A`, ale nie dla klasy podstawowej `B`.</span><span class="sxs-lookup"><span data-stu-id="97434-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>

1.  <span data-ttu-id="97434-107">Utworzenie nowej metody rozszerzenia dla typu, gdy trzeba go wywołać, w przeciwnym razie Przenieś wywołanie do obiektu typu, które rozszerza istniejącą metodę.</span><span class="sxs-lookup"><span data-stu-id="97434-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="97434-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="97434-108">Example</span></span>  
 <span data-ttu-id="97434-109">Poniższy kod generuje CS1928 i CS1929:</span><span class="sxs-lookup"><span data-stu-id="97434-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
