# <a name="compiler-error-cs1929"></a><span data-ttu-id="51d98-102">Błąd kompilatora CS1929</span><span class="sxs-lookup"><span data-stu-id="51d98-102">Compiler Error CS1929</span></span>

<span data-ttu-id="51d98-103">Argument wystąpienia: nie można konwertować z "Type" na "typeB".</span><span class="sxs-lookup"><span data-stu-id="51d98-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="51d98-104">Ten błąd jest generowany podczas próby wywołania metody rozszerzenia z klasy, której nie rozszerza.</span><span class="sxs-lookup"><span data-stu-id="51d98-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="51d98-105">W przykładzie pokazano, Metoda rozszerzenia jest zdefiniowana dla klasy pochodnej `A` , ale nie dla klasy bazowej `B` .</span><span class="sxs-lookup"><span data-stu-id="51d98-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="51d98-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="51d98-106">To correct this error</span></span>  
  
1. <span data-ttu-id="51d98-107">Utwórz nową metodę rozszerzenia dla typu, który ma zostać wywołany, lub Przenieś wywołanie do obiektu typu, który ma zostać wywołana przez istniejącą metodę.</span><span class="sxs-lookup"><span data-stu-id="51d98-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="51d98-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="51d98-108">Example</span></span>  

 <span data-ttu-id="51d98-109">Poniższy kod generuje CS1928 i CS1929:</span><span class="sxs-lookup"><span data-stu-id="51d98-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
