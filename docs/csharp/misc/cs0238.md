# <a name="compiler-error-cs0238"></a>CS0238 błąd kompilatora
Nie można zapieczętować "członek", ponieważ nie jest przesłonięciem  
  
 [zapieczętowane](../../csharp/language-reference/keywords/sealed.md) została użyta wobec składowej, która nie została również oznaczona atrybutem [zastąpienia](../../csharp/language-reference/keywords/override.md). Aby uzyskać więcej informacji, zobacz [dziedziczenia](../../csharp/programming-guide/classes-and-structs/inheritance.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0238:  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
