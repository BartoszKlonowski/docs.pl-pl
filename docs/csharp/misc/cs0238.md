# <a name="compiler-error-cs0238"></a><span data-ttu-id="af2a0-102">Błąd kompilatora CS0238</span><span class="sxs-lookup"><span data-stu-id="af2a0-102">Compiler Error CS0238</span></span>
<span data-ttu-id="af2a0-103">element "member" nie może być zapieczętowany, ponieważ nie jest przesłonięciem</span><span class="sxs-lookup"><span data-stu-id="af2a0-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="af2a0-104">[zapieczętowany](../language-reference/keywords/sealed.md) został użyty dla elementu członkowskiego, który nie został również [](../language-reference/keywords/override.md)oznaczony jako przesłonięcie.</span><span class="sxs-lookup"><span data-stu-id="af2a0-104">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="af2a0-105">Aby uzyskać więcej informacji, [](../programming-guide/classes-and-structs/inheritance.md)Zobacz dziedziczenie.</span><span class="sxs-lookup"><span data-stu-id="af2a0-105">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="af2a0-106">Poniższy przykład generuje CS0238:</span><span class="sxs-lookup"><span data-stu-id="af2a0-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
