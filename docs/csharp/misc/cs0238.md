# <a name="compiler-error-cs0238"></a><span data-ttu-id="67edb-102">CS0238 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="67edb-102">Compiler Error CS0238</span></span>

 <span data-ttu-id="67edb-104">[zapieczętowane](../../csharp/language-reference/keywords/sealed.md) użyto elementu członkowskiego, który również nie została oznaczona atrybutem [zastąpienia](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="67edb-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="67edb-105">Aby uzyskać więcej informacji, zobacz [dziedziczenia](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="67edb-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="67edb-106">Poniższy przykład generuje CS0238:</span><span class="sxs-lookup"><span data-stu-id="67edb-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
