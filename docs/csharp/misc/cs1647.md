# <a name="compiler-error-cs1647"></a>CS1647 błąd kompilatora
Wyrażenie jest zbyt długie lub zbyt złożone do skompilowania w pobliżu "code"  
  
 Wystąpiło przepełnienie stosu kompilatora przetwarzania kodu. Aby rozwiązać ten problem, należy uprościć kod. Jeśli kod jest prawidłowa, skontaktuj się z [Centrum pomocy technicznej](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
