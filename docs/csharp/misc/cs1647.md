# <a name="compiler-error-cs1647"></a>Błąd kompilatora CS1647

Wyrażenie jest zbyt długie lub złożone do skompilowania w prawie "kodzie"  
  
 Wystąpiło przepełnienie stosu w kompilatorze przetwarzania kodu. Aby rozwiązać ten problem, Uprość kod. Jeśli Twój kod jest prawidłowy, uzyskaj pomoc w witrynie [społeczności deweloperów](https://developercommunity.visualstudio.com/spaces/61/index.html) lub [StackOverflow](https://stackoverflow.com/).
