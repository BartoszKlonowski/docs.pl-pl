# <a name="compiler-error-cs1647"></a>CS1647 błąd kompilatora
Wyrażenie jest zbyt długie lub zbyt złożone do skompilowania w pobliżu "code"  
  
 Wystąpił przepełnienie stosu kompilatora przetwarzania kodu. Aby rozwiązać ten problem, należy uprościć kod. Jeśli Twój kod jest prawidłowa, skontaktuj się z pomocą [Centrum pomocy technicznej](https://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
