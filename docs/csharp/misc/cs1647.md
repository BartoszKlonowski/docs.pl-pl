# <a name="compiler-error-cs1647"></a>CS1647 błąd kompilatora
Wyrażenie jest zbyt długie lub zbyt złożone do skompilowania w pobliżu "code"  
  
 Wystąpił przepełnienie stosu kompilatora przetwarzania kodu. Aby rozwiązać ten problem, należy uprościć kod. Jeśli Twój kod jest prawidłowy, Uzyskaj pomoc na stronie [społeczności deweloperów](https://developercommunity.visualstudio.com/spaces/61/index.html) lokacji lub [StackOverflow](https://stackoverflow.com/).
