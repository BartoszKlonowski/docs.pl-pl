# <a name="compiler-error-cs0656"></a>Błąd kompilatora CS0656
Brak wymaganego przez kompilator elementu członkowskiego "Object. member"  
  
 Jeden z następujących problemów istnieje:  
  
- Instalacja środowiska uruchomieniowego języka wspólnego jest uszkodzona.  
  
- Istnieje odwołanie do zestawu, który definiuje typ, który jest również znaleziony w środowisku uruchomieniowym języka wspólnego. Jednak typ zestawu nie jest zdefiniowany w sposób oczekiwany przez kompilator języka C#.  
  
 Sprawdź odwołania, aby upewnić się, że używasz prawidłowej wersji środowiska uruchomieniowego języka wspólnego.
