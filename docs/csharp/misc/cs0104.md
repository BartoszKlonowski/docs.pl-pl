# <a name="compiler-error-cs0104"></a><span data-ttu-id="81385-102">Błąd kompilatora CS0104</span><span class="sxs-lookup"><span data-stu-id="81385-102">Compiler Error CS0104</span></span>
<span data-ttu-id="81385-103">element "Reference" to niejednoznaczne odwołanie między elementami "identifier" i "identifier"</span><span class="sxs-lookup"><span data-stu-id="81385-103">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="81385-104">Program zawiera dyrektywy [using](../language-reference/keywords/using.md) dla dwóch przestrzeni nazw, a kod odwołuje się do nazwy, która pojawia się w obu obszarach nazw.</span><span class="sxs-lookup"><span data-stu-id="81385-104">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="81385-105">Poniższy przykład generuje CS0104:</span><span class="sxs-lookup"><span data-stu-id="81385-105">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
