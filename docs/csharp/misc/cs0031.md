# <a name="compiler-error-cs0031"></a><span data-ttu-id="e2dfb-102">CS0031 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e2dfb-102">Compiler Error CS0031</span></span>
<span data-ttu-id="e2dfb-103">Nie można przekonwertować wartości stałej "value" na "type".</span><span class="sxs-lookup"><span data-stu-id="e2dfb-103">Constant value 'value' cannot be converted to a 'type'.</span></span> <span data-ttu-id="e2dfb-104">(Aby przesłonić użyj składni "unchecked")</span><span class="sxs-lookup"><span data-stu-id="e2dfb-104">(use 'unchecked' syntax to override)</span></span>  
  
 <span data-ttu-id="e2dfb-105">Nastąpiła próba do przypisania wartości do zmiennej o typie nie można zapisać wartości.</span><span class="sxs-lookup"><span data-stu-id="e2dfb-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="e2dfb-106">Aby uzyskać więcej informacji, zobacz [Typy](../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="e2dfb-106">For more information, see [Types](../../csharp/programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="e2dfb-107">Poniższy przykład generuje CS0031 w kontekstach zaznaczone i niezaznaczone:</span><span class="sxs-lookup"><span data-stu-id="e2dfb-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>  
  
```csharp  
// CS0031.cs  
namespace CS0031  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031  
  
         const decimal d = -10M; // Decimal literal  
            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```
