# <a name="compiler-error-cs0031"></a><span data-ttu-id="7add8-102">Błąd kompilatora CS0031</span><span class="sxs-lookup"><span data-stu-id="7add8-102">Compiler Error CS0031</span></span>

<span data-ttu-id="7add8-103">Nie można przekonwertować wartości stałej "value" na typ "Type".</span><span class="sxs-lookup"><span data-stu-id="7add8-103">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="7add8-104">Podjęto próbę przypisania wartości do zmiennej, której typ nie może przechowywać wartości.</span><span class="sxs-lookup"><span data-stu-id="7add8-104">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="7add8-105">Aby uzyskać więcej informacji, zobacz [typy](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="7add8-105">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="7add8-106">Poniższy przykład generuje CS0031 w kontekstach zaznaczonych i niezaznaczonych:</span><span class="sxs-lookup"><span data-stu-id="7add8-106">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
