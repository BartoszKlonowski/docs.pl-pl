# <a name="compiler-error-cs0666"></a>CS0666 błąd kompilatora
'składowa': Nowa chroniona składowa zadeklarowana w strukturze  
  
 A [struktury](../../csharp/language-reference/keywords/struct.md) nie może być [abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) i zawsze jest niejawnie [zapieczętowanego](../../csharp/language-reference/keywords/sealed.md). Ponieważ struktury nie obsługują dziedziczenie, koncepcji [chronione](../../csharp/language-reference/keywords/protected.md) elementu członkowskiego w strukturze sprawia, że nie ma sensu. Aby uzyskać więcej informacji, zobacz [dziedziczenia](../../csharp/programming-guide/classes-and-structs/inheritance.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0666:  
  
```csharp  
// CS0666.cs  
class M  
{  
    static void Main()  
    {  
    }  
}  
  
struct S  
{  
    protected int x;   // CS0666  
}  
```
