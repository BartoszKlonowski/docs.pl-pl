# <a name="compiler-warning-level-1-cs1522"></a>Kompilatora (poziom 1) ostrzeżenie CS1522
Blok "switch" pusty  
  
 Kompilator wykrył [Przełącz](../../csharp/language-reference/keywords/switch.md) bloku bez **przypadek** lub **domyślne** instrukcji. A `switch` blok musi zawierać co najmniej jeden **przypadek** lub **domyślne** instrukcji.  
  
 Poniższy przykład spowoduje wygenerowanie CS1522:  
  
```csharp  
// CS1522.cs  
// compile with: /W:1  
using System;  
class x  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
