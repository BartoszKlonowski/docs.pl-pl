<span data-ttu-id="eaedd-103">Nazwa typu 'Identyfikator' nie istnieje w typie "type"</span><span class="sxs-lookup"><span data-stu-id="eaedd-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="eaedd-104">Ten błąd występuje, gdy możesz odwołać się do typu, zagnieżdżone w obrębie innego typu, ale nie ma takiej zagnieżdżony typ istnieje.</span><span class="sxs-lookup"><span data-stu-id="eaedd-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="eaedd-105">Może to występować, jeśli wpisujesz nazwę typu zagnieżdżonego.</span><span class="sxs-lookup"><span data-stu-id="eaedd-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="eaedd-106">Sprawdź pisownię nazwy używane i sprawdzić, czy typ otaczający oczekiwanego elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="eaedd-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="eaedd-107">Poniższy przykład generuje CS0426, ponieważ klasa C nie ma zagnieżdżonych typu Odp.:</span><span class="sxs-lookup"><span data-stu-id="eaedd-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
