# <a name="compiler-warning-level-4-cs0028"></a>Kompilatora (poziom 4) ostrzeżenie CS0028
"w deklaracji funkcji" ma nieprawidłową sygnaturę i nie może być punktem wejścia  
  
 Deklaracja metody `Main` był nieprawidłowy: została zadeklarowana z nieprawidłowym podpisem. `Main` musi być zadeklarowany jako statyczny i zwracać albo [int](../../csharp/language-reference/builtin-types/integral-numeric-types.md) lub [void](../../csharp/language-reference/keywords/void.md). Aby uzyskać więcej informacji, zobacz [Main() i argumenty wiersza polecenia](../../csharp/programming-guide/main-and-command-args/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0028:  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
