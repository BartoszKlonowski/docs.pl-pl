# <a name="compiler-error-cs1678"></a><span data-ttu-id="ccee7-102">Błąd kompilatora CS1678</span><span class="sxs-lookup"><span data-stu-id="ccee7-102">Compiler Error CS1678</span></span>
<span data-ttu-id="ccee7-103">Parametr "number" jest zadeklarowany jako typ "type1", ale powinien być "type2"</span><span class="sxs-lookup"><span data-stu-id="ccee7-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="ccee7-104">Ten błąd występuje, gdy typ parametru w metodzie anonimowej różni się od deklaracji pełnomocnika, do którego jest rzutowanie metody.</span><span class="sxs-lookup"><span data-stu-id="ccee7-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="ccee7-105">Poniższy przykład generuje CS1678:</span><span class="sxs-lookup"><span data-stu-id="ccee7-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
