# <a name="compiler-error-cs1678"></a>CS1678 błąd kompilatora
Parametr "number" jest zadeklarowany jako typu "type1", ale powinien być 'type2'  
  
 Ten błąd występuje, gdy typ parametru w metodę anonimową różni się od deklaracji delegata, który jest rzutowanie metody do.  
  
 Poniższy przykład spowoduje wygenerowanie CS1678:  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
