# <a name="compiler-error-cs0073"></a>CS0073 błąd kompilatora
Metody dostępu add lub remove musi mieć treść  
  
 **Dodaj** lub **Usuń** — słowo kluczowe w [zdarzeń](../../csharp/language-reference/keywords/event.md) definicja musi mieć treść. Aby uzyskać więcej informacji, zobacz [zdarzenia](../../csharp/programming-guide/events/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0073:  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
