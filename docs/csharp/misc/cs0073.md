# <a name="compiler-error-cs0073"></a><span data-ttu-id="bd207-103">Błąd kompilatora CS0073</span><span class="sxs-lookup"><span data-stu-id="bd207-103">Compiler Error CS0073</span></span>
<span data-ttu-id="bd207-104">Metoda dostępu Add lub Remove musi mieć treść</span><span class="sxs-lookup"><span data-stu-id="bd207-104">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="bd207-105">**Dodanie** lub **usunięcie** słowa kluczowego w definicji [zdarzenia](../language-reference/keywords/event.md) musi mieć treść.</span><span class="sxs-lookup"><span data-stu-id="bd207-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="bd207-106">Aby uzyskać więcej informacji, zobacz [zdarzenia](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="bd207-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="bd207-107">Poniższy przykład generuje CS0073:</span><span class="sxs-lookup"><span data-stu-id="bd207-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
