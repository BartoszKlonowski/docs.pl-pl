# <a name="compiler-error-cs0073"></a><span data-ttu-id="91b35-102">CS0073 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="91b35-102">Compiler Error CS0073</span></span>
<span data-ttu-id="91b35-103">Metody dostępu add lub remove musi mieć treść</span><span class="sxs-lookup"><span data-stu-id="91b35-103">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="91b35-104">**Dodaj** lub **Usuń** — słowo kluczowe w [zdarzeń](../../csharp/language-reference/keywords/event.md) definicja musi mieć treść.</span><span class="sxs-lookup"><span data-stu-id="91b35-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="91b35-105">Aby uzyskać więcej informacji, zobacz [zdarzenia](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="91b35-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="91b35-106">Poniższy przykład spowoduje wygenerowanie CS0073:</span><span class="sxs-lookup"><span data-stu-id="91b35-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
