# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="70a1a-103">Ostrzeżenie kompilatora (poziom 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="70a1a-103">Compiler Warning (level 2) CS0469</span></span>

<span data-ttu-id="70a1a-104">Wartość "goto case" nie jest możliwa do niejawnego przekonwertowania na typ "Type"</span><span class="sxs-lookup"><span data-stu-id="70a1a-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="70a1a-105">Gdy używasz `goto case` , musi istnieć niejawna konwersja z wartości instrukcji goto case na typ przełącznika.</span><span class="sxs-lookup"><span data-stu-id="70a1a-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="70a1a-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="70a1a-106">Example</span></span>  

 <span data-ttu-id="70a1a-107">Poniższy przykład generuje CS0469.</span><span class="sxs-lookup"><span data-stu-id="70a1a-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
