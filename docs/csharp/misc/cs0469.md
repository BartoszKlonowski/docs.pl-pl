# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="36eea-102">Ostrzeżenie kompilatora (poziom 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="36eea-102">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="36eea-103">Wartość "goto case" nie jest niejawnie konwertowalna na typ "typ"</span><span class="sxs-lookup"><span data-stu-id="36eea-103">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="36eea-104">W przypadku `goto case`korzystania z niego musi istnieć niejawna konwersja z wartości przypadku goto do typu przełącznika.</span><span class="sxs-lookup"><span data-stu-id="36eea-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="36eea-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="36eea-105">Example</span></span>  
 <span data-ttu-id="36eea-106">Poniższy przykład generuje CS0469.</span><span class="sxs-lookup"><span data-stu-id="36eea-106">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
