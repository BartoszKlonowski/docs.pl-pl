# <a name="compiler-error-cs0468"></a>CS0468 błąd kompilatora
Niejednoznaczności między typu "type1" i "type2".  
  
 Ten błąd jest generowany, gdy istnieją dwa typy w zestawie jest kompilowane, które mają taką samą nazwę FQDN. Może to wystąpić, jeśli są w dodanych modułów lub jeden znajduje się w dodanym module i jeden w źródle.
