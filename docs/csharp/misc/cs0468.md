# <a name="compiler-error-cs0468"></a>CS0468 błąd kompilatora
Niejednoznaczność między typu "type1" i type "type2"  
  
 Ten błąd jest generowany, gdy istnieją dwa typy w zestawie jest kompilowane, które mają taką samą nazwę FQDN. To może wystąpić, jeśli są one w dodanych modułów lub jeden znajduje się w dodanym module i jeden w źródle.
