# <a name="compiler-error-cs0468"></a>Błąd kompilatora CS0468

Niejednoznaczność między typem "type1" a typem "type2"

 Ten błąd jest generowany, gdy istnieją dwa typy w zestawie są kompilowane, które mają taką samą nazwę w pełni kwalifikowane. Taka sytuacja może wystąpić, jeśli są one zarówno w dodanych modułów lub jeden jest w dodanym module i jeden w źródle.

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
