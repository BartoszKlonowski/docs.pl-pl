# <a name="compiler-warning-level-1-cs3004"></a>Kompilator CS3004 ostrzegawcze (poziom 1)
Mieszane i rozłożone znaki standardu Unicode nie są zgodne ze specyfikacją CLS  
  
 Złożony tylko znaki UNICODE są dozwolone w [publicznego](../../csharp/language-reference/keywords/public.md), [chronione](../../csharp/language-reference/keywords/protected.md), lub `protected internal` identyfikatorów w celu zapewnienia zgodności z typowych specyfikacji języka (CLS). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](https://msdn.microsoft.com/library/12a7a7h3).
