# <a name="compiler-warning-level-1-cs3004"></a>Ostrzeżenie kompilatora (poziom 1) CS3004
Mieszane i rozłożone znaki Unicode nie są zgodne ze specyfikacją CLS  
  
 W [publicznych](../language-reference/keywords/public.md), [chronionych](../language-reference/keywords/protected.md)lub `protected internal` identyfikatorach można używać tylko znaków Unicode, aby były zgodne z Common Language Specification (CLS). Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz niezależność [od języka i składniki niezależne od języka](../../standard/language-independence.md).
