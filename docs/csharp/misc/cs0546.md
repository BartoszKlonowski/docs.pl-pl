## <a name="example"></a><span data-ttu-id="1d4ea-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="1d4ea-110">Example</span></span>  
 <span data-ttu-id="1d4ea-111">Poniższy przykład generuje CS0546, ponieważ klasa podstawowa nie deklaruje metodą dostępu Set dla właściwości.</span><span class="sxs-lookup"><span data-stu-id="1d4ea-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
