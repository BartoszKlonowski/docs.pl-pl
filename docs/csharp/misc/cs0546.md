# <a name="compiler-error-cs0546"></a><span data-ttu-id="9863c-103">Błąd kompilatora CS0546</span><span class="sxs-lookup"><span data-stu-id="9863c-103">Compiler Error CS0546</span></span>

<span data-ttu-id="9863c-104">"akcesor": nie można przesłonić, ponieważ właściwość "Property" nie ma metody dostępu set z możliwością zastąpienia</span><span class="sxs-lookup"><span data-stu-id="9863c-104">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="9863c-105">Próba przesłonięcia jednej z metod dostępu do właściwości nie powiodła się, ponieważ nie można zastąpić metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="9863c-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="9863c-106">Ten błąd może wystąpić, jeśli:</span><span class="sxs-lookup"><span data-stu-id="9863c-106">This error can occur if:</span></span>  
  
- <span data-ttu-id="9863c-107">Właściwość klasy bazowej nie jest zadeklarowana jako [wirtualna](../language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="9863c-107">the base class property is not declared as [virtual](../language-reference/keywords/virtual.md).</span></span>

 <span data-ttu-id="9863c-109">Jeśli nie chcesz przesłonić właściwości klasy bazowej, możesz użyć słowa kluczowego [New](../language-reference/keywords/new-modifier.md) przed właściwością w klasie pochodnej.</span><span class="sxs-lookup"><span data-stu-id="9863c-109">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="9863c-110">Aby uzyskać więcej informacji, zobacz [Używanie właściwości](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="9863c-110">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="9863c-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="9863c-111">Example</span></span>  

 <span data-ttu-id="9863c-112">Poniższy przykład generuje CS0546, ponieważ klasa bazowa nie deklaruje metody dostępu set dla właściwości.</span><span class="sxs-lookup"><span data-stu-id="9863c-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
