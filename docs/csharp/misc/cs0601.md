# <a name="compiler-error-cs0601"></a><span data-ttu-id="ab6f5-102">CS0601 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="ab6f5-102">Compiler Error CS0601</span></span>
<span data-ttu-id="ab6f5-103">Musi być określony atrybut DllImport metoda oznaczona jako "static" i "extern"</span><span class="sxs-lookup"><span data-stu-id="ab6f5-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="ab6f5-104">`DllImport` Atrybut został użyty na metodzie, która nie ma słowa kluczowe prawidłowy dostęp.</span><span class="sxs-lookup"><span data-stu-id="ab6f5-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="ab6f5-105">Poniższy przykład spowoduje wygenerowanie CS0601:</span><span class="sxs-lookup"><span data-stu-id="ab6f5-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
