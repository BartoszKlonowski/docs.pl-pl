# <a name="compiler-error-cs0601"></a>CS0601 błąd kompilatora
Musi być określony atrybut DllImport metoda oznaczona jako "static" i "extern"  
  
 `DllImport` Atrybut został użyty na metodzie, która nie ma słowa kluczowe prawidłowy dostęp.  
  
 Poniższy przykład spowoduje wygenerowanie CS0601:  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
