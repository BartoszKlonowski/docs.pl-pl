# <a name="compiler-error-cs0004"></a>Błąd kompilatora CS0004
Ostrzeżenie traktowane jako błąd  
  
 Skompilowano z `/warnaserror` opcją kompilatora, która powoduje, że kompilator generuje ostrzeżenia jako błędy.
