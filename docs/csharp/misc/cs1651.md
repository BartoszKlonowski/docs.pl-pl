# <a name="compiler-error-cs1651"></a><span data-ttu-id="479de-102">CS1651 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="479de-102">Compiler Error CS1651</span></span>
<span data-ttu-id="479de-103">Nie można przekazać pól pola statycznego tylko do odczytu 'Identyfikator' ref lub out (z wyjątkiem w konstruktorze statycznym)</span><span class="sxs-lookup"><span data-stu-id="479de-103">Fields of static readonly field 'identifier' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="479de-104">Ten błąd występuje w przypadku przekazania zmienną do funkcji, która jest elementem członkowskim pola statycznego tylko do odczytu jako argumentu ref.</span><span class="sxs-lookup"><span data-stu-id="479de-104">This error occurs if you pass a variable to a function that is a member of a static readonly field as a ref argument.</span></span> <span data-ttu-id="479de-105">Ponieważ parametry ref mogą zostać zmodyfikowane przez funkcję, jest to niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="479de-105">Since ref parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="479de-106">Aby rozwiązać ten problem, Usuń **tylko do odczytu** — słowo kluczowe w polu, lub czy nie — dostęp próbny pola członków tylko do odczytu do funkcji.</span><span class="sxs-lookup"><span data-stu-id="479de-106">To resolve this error, remove the **readonly** keyword on the field, or do not pass the members of the readonly field to the function.</span></span> <span data-ttu-id="479de-107">Na przykład, możesz spróbować tworzenia zmiennej tymczasowej, które można modyfikować oraz przekazywania tymczasowy jako argumentu ref, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="479de-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a ref argument, as shown in the following example.</span></span>  
  
 <span data-ttu-id="479de-108">Poniższy przykład spowoduje wygenerowanie CS1651:</span><span class="sxs-lookup"><span data-stu-id="479de-108">The following sample generates CS1651:</span></span>  
  
```csharp  
// CS1651.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void f(ref int iref)  
   {  
   }  
  
   static void Main()  
   {  
      f(ref Outer.inner.i);  // CS1651  
      // Try this instead:  
      // int tmp = Outer.inner.i;  
      // f(ref tmp);  
   }  
}  
```
