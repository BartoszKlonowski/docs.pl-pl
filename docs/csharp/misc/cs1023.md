# <a name="compiler-error-cs1023"></a><span data-ttu-id="e4cba-103">Błąd kompilatora CS1023</span><span class="sxs-lookup"><span data-stu-id="e4cba-103">Compiler Error CS1023</span></span>

<span data-ttu-id="e4cba-104">Osadzona instrukcja nie może być instrukcją deklaracji ani etykietą</span><span class="sxs-lookup"><span data-stu-id="e4cba-104">Embedded statement cannot be a declaration or labeled statement</span></span>  
  
 <span data-ttu-id="e4cba-105">Osadzona instrukcja, taka jak instrukcje po instrukcji **if** , nie może zawierać deklaracji ani instrukcji oznaczonych etykietami.</span><span class="sxs-lookup"><span data-stu-id="e4cba-105">An embedded statement, such as the statements following an **if** statement, can contain neither declarations nor labeled statements.</span></span>  
  
 <span data-ttu-id="e4cba-106">Poniższy przykład generuje CS1023 dwa razy:</span><span class="sxs-lookup"><span data-stu-id="e4cba-106">The following sample generates CS1023 twice:</span></span>  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
