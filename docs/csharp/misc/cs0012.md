 <span data-ttu-id="31ed6-109">Następująca sekwencja kompilacji spowoduje CS0012:</span><span class="sxs-lookup"><span data-stu-id="31ed6-109">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="31ed6-110">Następnie:</span><span class="sxs-lookup"><span data-stu-id="31ed6-110">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="31ed6-111">Następnie:</span><span class="sxs-lookup"><span data-stu-id="31ed6-111">Then:</span></span>  
  
```csharp

// compile with: /reference:cs0012b.dll

{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
