 <span data-ttu-id="202ed-108">Następująca sekwencja kompilacji spowoduje CS0012:</span><span class="sxs-lookup"><span data-stu-id="202ed-108">The following sequence of compilations will result in CS0012:</span></span>

  
 <span data-ttu-id="202ed-109">Następnie</span><span class="sxs-lookup"><span data-stu-id="202ed-109">Then:</span></span>  
  
```csharp

   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="202ed-110">Następnie</span><span class="sxs-lookup"><span data-stu-id="202ed-110">Then:</span></span>

```csharp  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
