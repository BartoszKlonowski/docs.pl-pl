 <span data-ttu-id="5eb6a-109">Następnie:</span><span class="sxs-lookup"><span data-stu-id="5eb6a-109">Then:</span></span>

{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="5eb6a-110">Następnie:</span><span class="sxs-lookup"><span data-stu-id="5eb6a-110">Then:</span></span>

   }

```
