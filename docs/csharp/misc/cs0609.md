# <a name="compiler-error-cs0609"></a>CS0609 błąd kompilatora
Nie można ustawić IndexerName atrybutu dla indeksatora oznaczonego zastąpienia  
  
 Atrybut name (**IndexerNameAttribute**) nie można zastosować do właściwości indeksowanej, który jest przesłonięciem. Aby uzyskać więcej informacji, zobacz [indeksatory](../../csharp/programming-guide/indexers/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0609:  
  
```csharp  
// CS0609.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public class idx  
{  
   public virtual int this[int iPropIndex]  
   {  
      get  
      {  
         return 0;  
      }  
      set  
      {  
      }  
   }  
}  
  
public class MonthDays : idx  
{  
   [IndexerName("MonthInfoIndexer")]   // CS0609, delete to resolve this CS0609  
   public override int this[int iPropIndex]  
   {  
      get  
      {  
         return 0;  
      }  
      set  
      {  
      }  
   }  
}  
  
public class test  
{  
   public static void Main( string[] args )  
   {  
   }  
}  
```
