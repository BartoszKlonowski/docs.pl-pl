# <a name="compiler-error-cs0609"></a>Błąd kompilatora CS0609

Nie można ustawić IndexerName atrybutu dla indeksatora oznaczonego zastępowaniem

 Nie można zastosować atrybutu Name (**IndexerNameAttribute**) do właściwości indeksowanej, która jest przesłonięciem. Aby uzyskać więcej informacji, zobacz [indeksatory](../programming-guide/indexers/index.md).

 Poniższy przykład generuje CS0609:

```csharp
// CS0609.cs
using System;
using System.Runtime.CompilerServices;

public class idx
{
   public virtual int this[int iPropIndex]
   {
      get
      {
         return 0;
      }
      set
      {
      }
   }
}

public class MonthDays : idx
{
   [IndexerName("MonthInfoIndexer")]   // CS0609, delete to resolve this CS0609
   public override int this[int iPropIndex]
   {
      get
      {
         return 0;
      }
      set
      {
      }
   }
}

public class test
{
   public static void Main(string[] args)
   {
   }
}
```

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn. Poprzedni kod powinien zostać skompilowany pomyślnie.
