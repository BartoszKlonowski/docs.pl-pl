 <span data-ttu-id="3f97e-104">Metoda częściowa nie może mieć [się](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru.</span><span class="sxs-lookup"><span data-stu-id="3f97e-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="3f97e-105">Parametry wyjściowe są niedozwolone, ponieważ jeśli metoda częściowa jest usuwana przez kompilator wówczas nie ma żadnej gwarancji, które nigdy nie przypisano parametru out.</span><span class="sxs-lookup"><span data-stu-id="3f97e-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>
