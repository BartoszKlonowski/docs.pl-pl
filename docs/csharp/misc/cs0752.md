# <a name="compiler-error-cs0752"></a><span data-ttu-id="67e2e-103">Błąd kompilatora CS0752</span><span class="sxs-lookup"><span data-stu-id="67e2e-103">Compiler Error CS0752</span></span>

<span data-ttu-id="67e2e-104">Metoda częściowa nie może mieć parametrów out</span><span class="sxs-lookup"><span data-stu-id="67e2e-104">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="67e2e-105">Metoda częściowa nie może mieć parametru [out](../language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="67e2e-105">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="67e2e-106">Parametry out są niedozwolone, ponieważ jeśli metoda częściowa zostanie usunięta przez kompilator, nie ma gwarancji, że parametr out jest kiedykolwiek przypisany.</span><span class="sxs-lookup"><span data-stu-id="67e2e-106">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="67e2e-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="67e2e-107">To correct this error</span></span>  
  
1. <span data-ttu-id="67e2e-108">Usuń modyfikator out z parametru i zamiast tego użyj wartości zwracanej metody lub Usuń modyfikator częściowy z deklaracji metody.</span><span class="sxs-lookup"><span data-stu-id="67e2e-108">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>
