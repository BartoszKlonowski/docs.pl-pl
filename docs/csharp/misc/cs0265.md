# <a name="compiler-error-cs0265"></a><span data-ttu-id="155a4-102">Błąd kompilatora CS0265</span><span class="sxs-lookup"><span data-stu-id="155a4-102">Compiler Error CS0265</span></span>
<span data-ttu-id="155a4-103">Częściowe deklaracje "typu" mają niespójne ograniczenia dla parametru typu "parametr typu"</span><span class="sxs-lookup"><span data-stu-id="155a4-103">Partial declarations of 'type' have inconsistent constraints for type parameter 'type parameter'</span></span>  
  
 <span data-ttu-id="155a4-104">Ten błąd występuje podczas definiowania klasy ogólnej jako klasy częściowej, tak aby jej częściowe definicje występują w więcej niż jednym miejscu, a ograniczenia typu ogólnego są niespójne lub różne w dwóch lub więcej miejscach.</span><span class="sxs-lookup"><span data-stu-id="155a4-104">This error happens when you define a generic class as a partial class, so that its partial definitions occur in more than one place, and the constraints on the generic type are inconsistent or different in two or more places.</span></span> <span data-ttu-id="155a4-105">Jeśli określisz ograniczenia w więcej niż jednym miejscu, wszystkie muszą być identyczne.</span><span class="sxs-lookup"><span data-stu-id="155a4-105">If you specify the constraints in more than one place, they must all be identical.</span></span> <span data-ttu-id="155a4-106">Najprostszym rozwiązaniem jest określenie ograniczeń w jednym miejscu i pominięcie ich wszędzie indziej.</span><span class="sxs-lookup"><span data-stu-id="155a4-106">The easiest solution is to specify the constraints in one place, and omit them everywhere else.</span></span> <span data-ttu-id="155a4-107">Aby uzyskać więcej informacji, zobacz [Częściowe klasy i metody](../programming-guide/classes-and-structs/partial-classes-and-methods.md) i ograniczenia parametrów [typu](../programming-guide/generics/constraints-on-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="155a4-107">For more information, see [Partial Classes and Methods](../programming-guide/classes-and-structs/partial-classes-and-methods.md) and [Constraints on Type Parameters](../programming-guide/generics/constraints-on-type-parameters.md).</span></span>  
  
 <span data-ttu-id="155a4-108">Poniższy kod generuje błąd CS0265.</span><span class="sxs-lookup"><span data-stu-id="155a4-108">The following code generates error CS0265.</span></span>  
  
## <a name="example"></a><span data-ttu-id="155a4-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="155a4-109">Example</span></span>  
 <span data-ttu-id="155a4-110">W tym kodzie definicje klas częściowych są w jednym pliku, ale mogą być również rozłożone na wiele plików.</span><span class="sxs-lookup"><span data-stu-id="155a4-110">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>  
  
```csharp  
// CS0265.cs  
public class GenericsErrors
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265  
    partial class PartialBadBounds<T> where T : IFace2 { }
}  
```
