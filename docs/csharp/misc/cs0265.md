## <a name="example"></a><span data-ttu-id="78002-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="78002-110">Example</span></span>  

 <span data-ttu-id="78002-111">W tym kodzie definicje klas częściowych są wszystkie w jednym pliku, ale mogą również być rozłożone na wiele plików.</span><span class="sxs-lookup"><span data-stu-id="78002-111">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>

{  
    interface IFace1 { }  
    interface IFace2 { }
