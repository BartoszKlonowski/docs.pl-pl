# <a name="compiler-error-cs0265"></a>CS0265 błąd kompilatora
Częściowe deklaracje elementu "type" mają niespójne ograniczenia dla typu parametru "parametr typu"  
  
 Ten błąd występuje, gdy zdefiniowania klasy ogólnej jako klasę częściową, tak aby jego definicje częściowe występują w więcej niż jednym miejscu, a ograniczenia typu ogólnego są niespójne lub inną w co najmniej dwóch miejscach. Jeśli określisz ograniczenia w więcej niż jednym miejscu, są wszystkie muszą być identyczne. Najprostszym rozwiązaniem jest określić ograniczenia w jednym miejscu i pominięto je wszędzie, gdzie else. Aby uzyskać więcej informacji, zobacz [klasy częściowe i metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md) i [ograniczenia dotyczące parametrów typu](../../csharp/programming-guide/generics/constraints-on-type-parameters.md).  
  
 Poniższy kod generuje błąd CS0265.  
  
## <a name="example"></a>Przykład  
 W tym kodzie są definicje częściowe klasy wszystkie w jednym pliku, ale one również może być rozkładane na wiele plików.  
  
```csharp  
// CS0265.cs  
public class GenericsErrors   
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265  
    partial class PartialBadBounds<T> where T : IFace2 { }   
}  
```
