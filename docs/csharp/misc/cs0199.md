## <a name="example"></a><span data-ttu-id="74369-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="74369-107">Example</span></span>  
 <span data-ttu-id="74369-108">Poniższy przykład generuje CS0199:</span><span class="sxs-lookup"><span data-stu-id="74369-108">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
