<span data-ttu-id="38b22-104">Pól statycznego pola tylko do odczytu nie można przekazywać ref ani out (z wyjątkiem w konstruktorze statycznym)</span><span class="sxs-lookup"><span data-stu-id="38b22-104">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="38b22-105">Zmienna [ReadOnly](../language-reference/keywords/readonly.md) musi mieć takie samo [statyczne](../language-reference/keywords/static.md) użycie jak Konstruktor, w którym chcesz przekazać go jako parametr [ref](../language-reference/keywords/ref.md) lub [out](../language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="38b22-105">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="38b22-106">Aby uzyskać więcej informacji, zobacz [przekazywanie parametrów](../programming-guide/classes-and-structs/passing-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="38b22-106">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="38b22-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="38b22-107">Example</span></span>  

 <span data-ttu-id="38b22-108">Poniższy przykład generuje CS0199:</span><span class="sxs-lookup"><span data-stu-id="38b22-108">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
