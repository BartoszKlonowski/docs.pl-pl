# <a name="compiler-warning-level-4-cs0649"></a><span data-ttu-id="6786c-103">Ostrzeżenie kompilatora (poziom 4) CS0649</span><span class="sxs-lookup"><span data-stu-id="6786c-103">Compiler Warning (level 4) CS0649</span></span>

<span data-ttu-id="6786c-104">Pole "Field" nigdy nie jest przypisane do i zawsze będzie miało wartość domyślną "value"</span><span class="sxs-lookup"><span data-stu-id="6786c-104">Field 'field' is never assigned to, and will always have its default value 'value'</span></span>  
  
 <span data-ttu-id="6786c-105">Kompilator wykrył niezainicjowaną deklarację pola prywatnego lub wewnętrznego, do której nigdy nie przypisano wartości.</span><span class="sxs-lookup"><span data-stu-id="6786c-105">The compiler detected an uninitialized private or internal field declaration that is never assigned a value.</span></span>  
  
 <span data-ttu-id="6786c-106">Poniższy przykład generuje CS0649:</span><span class="sxs-lookup"><span data-stu-id="6786c-106">The following sample generates CS0649:</span></span>  
  
```csharp  
// CS0649.cs  
// compile with: /W:4  
using System.Collections;  
  
class MyClass  
{  
   Hashtable table;  // CS0649  
   // You may have intended to initialize the variable to null  
   // Hashtable table = null;  
  
   // Or you may have meant to create an object here  
   // Hashtable table = new Hashtable();  
  
   public void Func(object o, string p)  
   {  
      // Or here  
      // table = new Hashtable();  
      table[p] = o;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
