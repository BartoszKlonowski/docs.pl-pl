# <a name="compiler-error-cs1578"></a>CS1578 błąd kompilatora
Nazwa pliku, jednowierszowego komentarza lub oczekiwano końcowego elementu wiersza  
  
 Po [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) dyrektywy, tylko nazwę pliku (w znaki podwójnego cudzysłowu) lub komentarz jednowierszowy jest dozwolone.  
  
 Poniższy przykład spowoduje wygenerowanie CS1578:  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
