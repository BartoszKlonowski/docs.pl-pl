<span data-ttu-id="4043f-104">Modyfikatora parametru "out" nie można używać z elementem "This".</span><span class="sxs-lookup"><span data-stu-id="4043f-104">The parameter modifier 'out' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="4043f-105">Gdy `this` słowo kluczowe modyfikuje pierwszy parametr metody statycznej, sygnalizuje kompilatorowi, że metoda jest metodą rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="4043f-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="4043f-106">W pierwszym parametrze metody rozszerzenia nie są wymagana żadne inne Modyfikatory ani nie są dozwolone.</span><span class="sxs-lookup"><span data-stu-id="4043f-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4043f-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4043f-107">To correct this error</span></span>  
  
1. <span data-ttu-id="4043f-108">Usuń nieautoryzowane Modyfikatory z pierwszego parametru.</span><span class="sxs-lookup"><span data-stu-id="4043f-108">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4043f-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="4043f-109">Example</span></span>  

 <span data-ttu-id="4043f-110">Poniższy przykład generuje CS1102:</span><span class="sxs-lookup"><span data-stu-id="4043f-110">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
