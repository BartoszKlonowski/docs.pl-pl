# <a name="compiler-error-cs0583"></a>CS0583 błąd kompilatora
Wewnętrzny błąd kompilatora. Wystąpił błąd wewnętrzny w kompilatorze. Aby obejść ten problem, spróbuj uprościć lub zmienić program w pobliżu miejsc wymienionych poniżej. Lokalizacje, w górnej części listy są bliższe do punktu, w którym wystąpił błąd wewnętrzny. Błędy takie, jak to mogą zostać zgłoszone do firmy Microsoft przy użyciu opcji/errorreport.  
  
 Utwórz plik raportu o błędach z "/ bugreport" opcji i przesłać raport problemu przypisanego raportowania skontaktuj się z pomocą.  
  
 Użyj [/bugreport](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) opcję kompilatora, aby przechwycić problem.
