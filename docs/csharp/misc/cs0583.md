# <a name="compiler-error-cs0583"></a>Błąd kompilatora CS0583

Błąd kompilatora wewnętrznego. Wystąpił wewnętrzny błąd kompilatora. Aby obejść ten problem, spróbuj uprościć lub zmienić program w pobliżu lokalizacji wymienionych poniżej. Lokalizacje u góry listy są bliżej punktu, w którym wystąpił błąd wewnętrzny. Błędy takie jak ten mogą być zgłaszane do firmy Microsoft przy użyciu /errorreport opcji.

 Utwórz plik raportu o błędach z opcją '/bugreport' i prześlij raport do przypisanego kontaktu z raportem o problemach.

 Użyj opcji [kompilatora -bugreport,](../language-reference/compiler-options/bugreport-compiler-option.md) aby przechwycić problem.

> [!NOTE]
> Ten błąd kompilatora nie jest już używany w Roslyn.
