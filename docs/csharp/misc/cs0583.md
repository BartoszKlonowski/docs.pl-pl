# <a name="compiler-error-cs0583"></a>CS0583 błąd kompilatora
Wewnętrzny błąd kompilatora. Wystąpił wewnętrzny błąd kompilatora. Aby obejść ten problem, spróbuj uprościć lub zmienić program w pobliżu miejsc wymienionych poniżej. Lokalizacje na początku listy są bliższe do punktu, w którym wystąpił błąd wewnętrzny. Błędy takie, jak to mogą być zgłaszane do firmy Microsoft przy użyciu opcji/errorreport.  
  
 Utwórz plik raportu o błędach z "/ bugreport" opcji i przesłać raport w celu raportowania skontaktuj się z przypisaną problemów.  
  
 Użyj [/bugreport](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) opcję kompilatora, aby przechwycić problem.
