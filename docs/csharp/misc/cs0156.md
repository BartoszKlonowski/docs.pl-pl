# <a name="compiler-error-cs0156"></a><span data-ttu-id="40075-102">CS0156 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="40075-102">Compiler Error CS0156</span></span>
<span data-ttu-id="40075-103">Instrukcja throw bez argumentów nie jest dozwolona w finally zagnieżdżonej w najbliższej otaczającej klauzuli catch — klauzula</span><span class="sxs-lookup"><span data-stu-id="40075-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="40075-104">A [throw](../../csharp/language-reference/keywords/throw.md) instrukcji bez parametrów, może wystąpić tylko w **catch** klauzula, która nie przyjmuje żadnych parametrów.</span><span class="sxs-lookup"><span data-stu-id="40075-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="40075-105">Aby uzyskać więcej informacji, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="40075-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="40075-106">Poniższy przykład spowoduje wygenerowanie CS0156:</span><span class="sxs-lookup"><span data-stu-id="40075-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
