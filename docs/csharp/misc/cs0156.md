# <a name="compiler-error-cs0156"></a><span data-ttu-id="8e142-103">Błąd kompilatora CS0156</span><span class="sxs-lookup"><span data-stu-id="8e142-103">Compiler Error CS0156</span></span>
<span data-ttu-id="8e142-104">Instrukcja throw bez argumentów jest niedozwolona w klauzuli finally, która jest zagnieżdżona w najbliższej otaczającej klauzuli catch</span><span class="sxs-lookup"><span data-stu-id="8e142-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="8e142-105">Instrukcja [throw](../language-reference/keywords/throw.md) bez parametrów może występować tylko w klauzuli **catch** , która nie przyjmuje żadnych parametrów.</span><span class="sxs-lookup"><span data-stu-id="8e142-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="8e142-106">Aby uzyskać więcej informacji, zobacz [wyjątki i obsługa wyjątków](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="8e142-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="8e142-107">Poniższy przykład generuje CS0156:</span><span class="sxs-lookup"><span data-stu-id="8e142-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
