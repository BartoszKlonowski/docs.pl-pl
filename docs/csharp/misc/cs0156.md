# <a name="compiler-error-cs0156"></a>CS0156 błąd kompilatora
Instrukcja throw bez argumentów nie jest dozwolona w finally zagnieżdżonej w najbliższej otaczającej klauzuli catch — klauzula  
  
 A [throw](../../csharp/language-reference/keywords/throw.md) instrukcji bez parametrów, może wystąpić tylko w **catch** klauzula, która nie przyjmuje żadnych parametrów.  
  
 Aby uzyskać więcej informacji, zobacz [wyjątków i wyjątków](../../csharp/programming-guide/exceptions/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0156:  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
