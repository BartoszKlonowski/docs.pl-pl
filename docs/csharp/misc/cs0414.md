<span data-ttu-id="672ec-105">To ostrzeżenie może wystąpić w kilku scenariuszach, w których kompilator może sprawdzić, czy zmienna nie jest nigdy przywoływana:</span><span class="sxs-lookup"><span data-stu-id="672ec-105">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>

- <span data-ttu-id="672ec-113">Do pola prywatnego lub wewnętrznego są przypisywane wartości stałe w wielu lokalizacjach, ale nie są używane w inny sposób.</span><span class="sxs-lookup"><span data-stu-id="672ec-113">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="672ec-114">Jeśli pole nie jest potrzebne, rozważ jego usunięcie.</span><span class="sxs-lookup"><span data-stu-id="672ec-114">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="672ec-115">W przeciwnym razie użyj go w odpowiedni sposób.</span><span class="sxs-lookup"><span data-stu-id="672ec-115">Otherwise, use it in some appropriate way.</span></span>
