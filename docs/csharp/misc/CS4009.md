## <a name="example"></a><span data-ttu-id="0789e-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="0789e-108">Example</span></span>

<span data-ttu-id="0789e-109">Poniższy przykład generuje CS4009:</span><span class="sxs-lookup"><span data-stu-id="0789e-109">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
    public static async void Main()
    {
        Console.WriteLine("About to wait two seconds");
        await WaitTwoSeconds();
        Console.WriteLine("About to exit the program");
    }

    private static async Task WaitTwoSeconds()
    {
        await Task.Delay(2000);
        Console.WriteLine("Returning from an asynchronous method");
    }
}
```
