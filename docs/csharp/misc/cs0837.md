# <a name="compiler-error-cs0837"></a><span data-ttu-id="884aa-102">CS0837 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="884aa-102">Compiler Error CS0837</span></span>
<span data-ttu-id="884aa-103">Pierwszy argument operacji "is" lub "as" operator nie może być wyrażenie lambda lub metodę anonimową.</span><span class="sxs-lookup"><span data-stu-id="884aa-103">The first operand of an "is" or "as" operator may not be a lambda expression or anonymous method.</span></span>  
  
 <span data-ttu-id="884aa-104">Wyrażenia lambda i metod anonimowych, nie mogą zostać użyte w lewej części [jest](../../csharp/language-reference/keywords/is.md) lub [jako](../../csharp/language-reference/keywords/as.md).</span><span class="sxs-lookup"><span data-stu-id="884aa-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../../csharp/language-reference/keywords/is.md) or [as](../../csharp/language-reference/keywords/as.md).</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="884aa-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="884aa-105">To correct this error</span></span>  
  
- <span data-ttu-id="884aa-106">Jeśli ten błąd obejmuje `is` operatora, należy pamiętać, że `is` przyjmuje wartość i typ i informuje, czy wartość może się do tego typu przez odwołanie, pakowanie i rozpakowywanie konwersji.</span><span class="sxs-lookup"><span data-stu-id="884aa-106">If the error involves the `is` operator, remember that `is` takes a value and a type and tells you whether the value can be made into that type by a reference, boxing, or unboxing conversion.</span></span> <span data-ttu-id="884aa-107">Ponieważ wyrażenia lambda nie są wartościami i mają żadne odwołanie, pakowanie i rozpakowywanie konwersji, wyrażenia lambda nie są kandydatami do `is`.</span><span class="sxs-lookup"><span data-stu-id="884aa-107">Because lambdas are not values and have no reference, boxing, or unboxing conversions, lambdas are not candidates for `is`.</span></span>  
  
- <span data-ttu-id="884aa-108">Jeśli niewłaściwym kodu `as`, naprawianie jest prawdopodobnie można go zmienić na rzutowania.</span><span class="sxs-lookup"><span data-stu-id="884aa-108">If the code misuses `as`, the correction is probably to change it to a cast.</span></span>  
  
## <a name="example"></a><span data-ttu-id="884aa-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="884aa-109">Example</span></span>  
 <span data-ttu-id="884aa-110">Poniższy przykład generuje CS0837:</span><span class="sxs-lookup"><span data-stu-id="884aa-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
