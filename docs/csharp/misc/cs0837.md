 <span data-ttu-id="ebdc7-104">Wyrażenia lambda, anonimowe metody i grupy metod nie mogą być używane po lewej stronie elementu [is](../language-reference/operators/type-testing-and-cast.md#is-operator) lub [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span><span class="sxs-lookup"><span data-stu-id="ebdc7-104">Lambda expressions, anonymous methods and method groups may not be used on the left side of [is](../language-reference/operators/type-testing-and-cast.md#is-operator) or [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span></span>

## <a name="example"></a><span data-ttu-id="ebdc7-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="ebdc7-109">Example</span></span>

<span data-ttu-id="ebdc7-110">Poniższy przykład generuje CS0837:</span><span class="sxs-lookup"><span data-stu-id="ebdc7-110">The following example generates CS0837:</span></span>

```csharp
// cs0837.cs
namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837
            return 1;
        }
    }
}
```
