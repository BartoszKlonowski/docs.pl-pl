# <a name="compiler-warning-level-1-cs3002"></a>Kompilator CS3002 ostrzegawcze (poziom 1)
Typ zwracany "metody" nie jest zgodne ze specyfikacją CLS  
  
 A [publicznego](../../csharp/language-reference/keywords/public.md), [chronione](../../csharp/language-reference/keywords/protected.md), lub `protected internal` metoda musi zwracać wartość, której typ jest zgodne z typowych specyfikacji języka (CLS). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3002:  
  
```  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
