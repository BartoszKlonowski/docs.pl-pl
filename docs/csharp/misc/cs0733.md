# <a name="compiler-error-cs0733"></a>CS0733 błąd kompilatora
Nie można przesłać dalej typu ogólnego "GenericType <>"  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0733. Skompiluj pierwszy plik jako biblioteki i odwoływanie się podczas kompilowania drugiego pliku.  
  
```csharp  
// CS0733a.cs  
// compile with: /target:library  
public class GenericType<T>   
{  
}  
```  
  
```csharp  
// CS0733.cs  
// compile with: /target:library /r:CS0733a.dll  
[assembly: System.Runtime.CompilerServices.TypeForwardedTo(typeof(GenericType<int>))]   // CS0733  
```
