# <a name="compiler-error-cs0733"></a>Błąd kompilatora CS0733
Nie można przekazać typu ogólnego "Generictype<>"  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0733. Skompiluj pierwszy plik jako bibliotekę, a następnie odwołuje się do niego podczas kompilowania drugiego pliku.  
  
```csharp  
// CS0733a.cs  
// compile with: /target:library  
public class GenericType<T>
{  
}  
```  
  
```csharp  
// CS0733.cs  
// compile with: /target:library /r:CS0733a.dll  
[assembly: System.Runtime.CompilerServices.TypeForwardedTo(typeof(GenericType<int>))]   // CS0733  
```
