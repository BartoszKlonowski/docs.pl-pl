# <a name="compiler-error-cs0459"></a><span data-ttu-id="d8a62-103">Błąd kompilatora CS0459</span><span class="sxs-lookup"><span data-stu-id="d8a62-103">Compiler Error CS0459</span></span>

<span data-ttu-id="d8a62-104">Nie można pobrać adresu zmiennej lokalnej tylko do odczytu</span><span class="sxs-lookup"><span data-stu-id="d8a62-104">Cannot take the address of a read-only local variable</span></span>

 <span data-ttu-id="d8a62-105">W języku C# istnieją trzy typowe scenariusze, które generują zmienne lokalne tylko do odczytu: `foreach` , `using` i `fixed` .</span><span class="sxs-lookup"><span data-stu-id="d8a62-105">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="d8a62-106">W każdym z tych przypadków nie wolno pisać do zmiennej lokalnej tylko do odczytu ani do jej adresu.</span><span class="sxs-lookup"><span data-stu-id="d8a62-106">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="d8a62-107">Ten błąd jest generowany, gdy kompilator realizuje próbę uzyskania adresu zmiennej lokalnej tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="d8a62-107">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>

## <a name="example"></a><span data-ttu-id="d8a62-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="d8a62-108">Example</span></span>

 <span data-ttu-id="d8a62-109">Poniższy przykład generuje CS0459, gdy podejmowana jest próba podjęcia adresu zmiennej lokalnej tylko do odczytu w `foreach` pętli i `fixed` bloku instrukcji:</span><span class="sxs-lookup"><span data-stu-id="d8a62-109">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
