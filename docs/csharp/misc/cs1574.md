<span data-ttu-id="41706-104">Komentarz XML w konstrukcji "konstrukcja" zawiera składniowo niepoprawny atrybut cref "name"</span><span class="sxs-lookup"><span data-stu-id="41706-104">XML comment on 'construct' has syntactically incorrect cref attribute 'name'</span></span>  
  
 <span data-ttu-id="41706-105">Ciąg przeszedł do znacznika cref, na przykład w \<exception> tagu, do elementu członkowskiego, który nie jest dostępny w bieżącym środowisku kompilacji.</span><span class="sxs-lookup"><span data-stu-id="41706-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="41706-106">Ciąg przekazywany do tagu cref musi być syntaktycznie poprawną nazwą elementu członkowskiego lub pola.</span><span class="sxs-lookup"><span data-stu-id="41706-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="41706-107">Aby uzyskać więcej informacji, zobacz [zalecane Tagi dla komentarzy do dokumentacji](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span><span class="sxs-lookup"><span data-stu-id="41706-107">For more information, see [Recommended Tags for Documentation Comments](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>  
  
 <span data-ttu-id="41706-108">Poniższy przykład generuje CS1574:</span><span class="sxs-lookup"><span data-stu-id="41706-108">The following sample generates CS1574:</span></span>  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>  
class EClass : Exception  
{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
