# <a name="compiler-error-cs1561"></a>Błąd kompilatora CS1561

Nazwa pliku wyjściowego jest za długa lub nieprawidłowa  
  
 Nazwa pliku wyjściowego nie może być dłuższa niż 256 znaków i może nie zawierać nieprawidłowych znaków, takich jak symbol euro, znak zapytania lub ukośnik odwrotny, między innymi.
