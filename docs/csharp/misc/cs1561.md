# <a name="compiler-error-cs1561"></a>CS1561 błąd kompilatora
Nazwa pliku wyjściowego jest za długa lub nieprawidłowa  
  
 Nazwa pliku wyjściowego nie może być dłuższa niż 256 znaków i nie może zawierać nieprawidłowych znaków, takich jak symbol EUR, znak zapytania lub ukośnik odwrotny, między innymi.
