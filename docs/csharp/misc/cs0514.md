# <a name="compiler-error-cs0514"></a>CS0514 błąd kompilatora
"Konstruktor": Konstruktor statyczny nie może mieć jawnego, "this" lub "base" wywołanie konstruktora  
  
 Wywoływanie `this` w konstruktorze statycznym jest niedozwolone, ponieważ automatycznie wywołuje statyczny Konstruktor przed utworzeniem dowolnego wystąpienia klasy. Ponadto konstruktory statyczne nie są dziedziczone i nie można wywołać bezpośrednio.  
  
 Aby uzyskać więcej informacji, zobacz [to](../../csharp/language-reference/keywords/this.md) i [podstawowy](../../csharp/language-reference/keywords/base.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0514:  
  
```csharp  
// CS0514.cs  
class A  
{  
    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}  
  
class B  
{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
