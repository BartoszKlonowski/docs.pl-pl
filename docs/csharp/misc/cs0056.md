# <a name="compiler-error-cs0056"></a>Błąd kompilatora CS0056
Niespójność dostępności: typ zwracany "Type" jest mniej dostępny niż operatora "operator"  
  
 Publiczna konstrukcja musi zwracać publicznie dostępny obiekt. Aby uzyskać więcej informacji, zobacz [Modyfikatory dostępu](../programming-guide/classes-and-structs/access-modifiers.md).  
  
 Poniższy przykład generuje CS0056:  
  
```csharp  
// CS0056.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class A  
{  
   public static implicit operator MyClass(A a)   // CS0056  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
