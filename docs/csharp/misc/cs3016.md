# <a name="compiler-warning-level-1-cs3016"></a>Ostrzeżenie kompilatora (poziom 1) CS3016

Tablice jako argumenty atrybutów nie są zgodne ze specyfikacją CLS  
  
 Nie jest on zgodny z Common Language Specification (CLS), aby przekazać tablicę do atrybutu. Aby uzyskać więcej informacji na temat zgodności ze specyfikacją CLS, zobacz [niezależność od języka i składniki niezależne od języka](../../standard/language-independence.md).
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS3016:  
  
```csharp  
// CS3016.cs  
  
using System;  
  
[assembly : CLSCompliant(true)]  
[C(new int[] {1, 2})]   // CS3016  
// try the following line instead  
// [C()]  
class C : Attribute  
{  
    public C()  
    {  
    }  
  
    public C(int[] a)  
    {  
    }  
  
    public static void Main ()  
    {  
    }  
}  
```
