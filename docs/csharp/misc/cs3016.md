# <a name="compiler-warning-level-1-cs3016"></a>Kompilator CS3016 ostrzegawcze (poziom 1)
Użycie tablic jako argumentów atrybutów nie jest zgodne ze specyfikacją CLS  
  
 Nie jest zgodne z typowych specyfikacji języka (CLS) do przekazania tablicy do atrybutu. Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3016:  
  
```  
// CS3016.cs  
  
using System;  
  
[assembly : CLSCompliant(true)]  
[C(new int[] {1, 2})]   // CS3016  
// try the following line instead  
// [C()]  
class C : Attribute  
{  
    public C()  
    {  
    }  
  
    public C(int[] a)  
    {  
    }  
  
    public static void Main ()  
    {  
    }  
}  
```
