 <span data-ttu-id="27d24-106">Istnieje kilka scenariuszy, `using` w których lub `lock` bloki spowodują tymczasowy przeciek zasobów.</span><span class="sxs-lookup"><span data-stu-id="27d24-106">There are several scenarios where `using` or `lock` blocks will result in a temporary leak of resources.</span></span> <span data-ttu-id="27d24-107">Oto przykład:</span><span class="sxs-lookup"><span data-stu-id="27d24-107">Here is one example:</span></span>

## <a name="example"></a><span data-ttu-id="27d24-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="27d24-111">Example</span></span>  

 <span data-ttu-id="27d24-112">Poniższy kod generuje ostrzeżenie CS0728.</span><span class="sxs-lookup"><span data-stu-id="27d24-112">The following code will generate warning CS0728.</span></span>  
  
```csharp
// CS0728.cs  
  
using System;  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```
