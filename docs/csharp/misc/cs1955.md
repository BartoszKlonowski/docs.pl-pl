## <a name="example"></a><span data-ttu-id="30fc0-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="30fc0-108">Example</span></span>

<span data-ttu-id="30fc0-109">Poniższy kod generuje CS1955, ponieważ kod próbuje wywołać pola i właściwości, za pomocą [operatora wywołania `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="30fc0-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="30fc0-110">Nie można wywołać pole lub właściwość.</span><span class="sxs-lookup"><span data-stu-id="30fc0-110">You cannot call a field or a property.</span></span> <span data-ttu-id="30fc0-111">Użyj [operator dostępu do elementu członkowskiego `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) dostępu do wartości są przechowywane.</span><span class="sxs-lookup"><span data-stu-id="30fc0-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
