# <a name="compiler-error-cs1955"></a>CS1955 błąd kompilatora
Składowej "name" nie można używać jak metody.  
  
 Może być wywoływany tylko metody i delegatów. Ten błąd jest generowany, gdy spróbujesz coś innego niż metody wywołania lub delegować za pomocą pustych nawiasów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń nawiasy z wyrażenia.  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1955, ponieważ kod próbuje wywołać pola i właściwości, za pomocą operatora wywołania metody [()](../../csharp/language-reference/operators/invocation-operator.md). Nie można wywołać pola lub właściwości, ale dostęp można uzyskać wartości są przechowywane przy użyciu operatora dostępu do elementu członkowskiego ( [.](../../csharp/language-reference/operators/member-access-operator.md) ).  
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
