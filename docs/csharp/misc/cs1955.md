## <a name="example"></a><span data-ttu-id="9eebe-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="9eebe-108">Example</span></span>

<span data-ttu-id="9eebe-109">Poniższy kod generuje CS1955, ponieważ kod próbuje wywołać pole i właściwość przy użyciu [ `()`wyrażenia wywołania ](../language-reference/operators/member-access-operators.md#invocation-expression-).</span><span class="sxs-lookup"><span data-stu-id="9eebe-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="9eebe-110">Nie można wywołać pola lub właściwości.</span><span class="sxs-lookup"><span data-stu-id="9eebe-110">You cannot call a field or a property.</span></span> <span data-ttu-id="9eebe-111">Użyj [wyrażenia `.` dostępu elementu członkowskiego,](../language-reference/operators/member-access-operators.md#member-access-expression-) aby uzyskać dostęp do wartości, która przechowuje.</span><span class="sxs-lookup"><span data-stu-id="9eebe-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
