## <a name="example"></a><span data-ttu-id="24f30-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="24f30-107">Example</span></span>  
 <span data-ttu-id="24f30-108">Poniższy kod generuje CS0825, ponieważ `var` jest używana na pola klasy:</span><span class="sxs-lookup"><span data-stu-id="24f30-108">The following code generates CS0825 because `var` is used on a class field:</span></span>  
  
```csharp

class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;   
    private var bookTitles = new List<string>();

    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
