# <a name="compiler-error-cs0451"></a><span data-ttu-id="e3feb-102">CS0451 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="e3feb-102">Compiler Error CS0451</span></span>
<span data-ttu-id="e3feb-103">Ograniczenie "new()" nie może być używane z ograniczeniem "struct"</span><span class="sxs-lookup"><span data-stu-id="e3feb-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="e3feb-104">Podczas określania ograniczenia typu ogólnego, `new()` ograniczenie może być używana tylko z ograniczenia typu klasy, ograniczenia typu interfejsu, ograniczenia typu odwołania i ograniczeniami parametrów typu, ale nie z ograniczeniami typu wartości.</span><span class="sxs-lookup"><span data-stu-id="e3feb-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e3feb-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="e3feb-105">Example</span></span>  
 <span data-ttu-id="e3feb-106">Poniższy przykład spowoduje wygenerowanie CS0451.</span><span class="sxs-lookup"><span data-stu-id="e3feb-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4   
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
