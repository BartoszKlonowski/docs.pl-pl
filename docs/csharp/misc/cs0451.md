# <a name="compiler-error-cs0451"></a><span data-ttu-id="59287-103">Błąd kompilatora CS0451</span><span class="sxs-lookup"><span data-stu-id="59287-103">Compiler Error CS0451</span></span>
<span data-ttu-id="59287-104">Ograniczenie "New ()" nie może być używane z ograniczeniem "struct"</span><span class="sxs-lookup"><span data-stu-id="59287-104">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="59287-105">Podczas określania ograniczeń typu ogólnego, `new()` ograniczenie może być używane tylko z ograniczeniami typu klasy, ograniczeniami typu interfejsu, ograniczeniami typu odwołania i ograniczeniami parametrów typu, ale nie z ograniczeniami typu wartości.</span><span class="sxs-lookup"><span data-stu-id="59287-105">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="59287-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="59287-106">Example</span></span>  
 <span data-ttu-id="59287-107">Poniższy przykład generuje CS0451.</span><span class="sxs-lookup"><span data-stu-id="59287-107">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
