# <a name="compiler-error-cs0451"></a>Błąd kompilatora CS0451

Ograniczenie "New ()" nie może być używane z ograniczeniem "struct"  
  
 Podczas określania ograniczeń typu ogólnego, `new()` ograniczenie może być używane tylko z ograniczeniami typu klasy, ograniczeniami typu interfejsu, ograniczeniami typu odwołania i ograniczeniami parametrów typu, ale nie z ograniczeniami typu wartości.  
  
## <a name="example"></a>Przykład  

 Poniższy przykład generuje CS0451.  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
