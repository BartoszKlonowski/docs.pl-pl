# <a name="get-started-with-c"></a>Wprowadzenie do C\#

Ta sekcja zawiera krótkie, proste samouczki, które umożliwiają szybkie tworzenie aplikacji przy użyciu języka C# i .NET Core. Istnieją tematy wprowadzenie dla programu Visual Studio i Visual Studio Code. Artykuły te zakładają pewne doświadczenie programowania. Jeśli jesteś nowym użytkownikiem programowania, wypróbuj nasze wprowadzenie do interaktywnych samouczków [języka C#.](../tutorials/intro-to-csharp/index.md)

Dostępne są następujące tematy:

- [Wprowadzenie do języka Języka C# i platformy .NET Framework](introduction-to-the-csharp-language-and-the-net-framework.md)

     Zawiera omówienie języka Języka C# i .NET.

- [Tworzenie aplikacji C# Hello World z usługą .NET Core w programie Visual Studio](../../core/tutorials/with-visual-studio.md)

   Program Visual Studio umożliwia tworzenie kodu, kompilowanie, uruchamianie, debugowanie, profilowanie i publikowanie aplikacji ze zintegrowanego środowiska programistycznego dla systemu Windows lub Mac.

   Ten temat umożliwia utworzenie i uruchomienie prostej aplikacji Hello World, a następnie zmodyfikowanie jej w celu uruchomienia nieco bardziej interaktywnej aplikacji Hello World. Po zakończeniu tworzenia i uruchamiania aplikacji, można również dowiedzieć się, jak [go debugować](../../core/tutorials/debugging-with-visual-studio.md) i jak [ją opublikować,](../../core/tutorials/publishing-with-visual-studio.md) aby można było ją uruchomić na dowolnej platformie obsługiwanej przez .NET Core.

- [Tworzenie biblioteki klas z c# i .NET Standard w programie Visual Studio](../../core/tutorials/library-with-visual-studio.md)

   Biblioteka klas umożliwia definiowanie typów i elementów członkowskich typu, które mogą być wywoływane z innej aplikacji. W tym temacie można utworzyć bibliotekę klas za pomocą jednej metody, która określa, czy ciąg zaczyna się wielkimi literami. Po zakończeniu tworzenia biblioteki można opracować [test jednostkowy,](../../core/tutorials/testing-library-with-visual-studio.md) aby upewnić się, że działa zgodnie z oczekiwaniami, a następnie można udostępnić go [aplikacjom, które chcą z niej korzystać.](../../core/tutorials/consuming-library-with-visual-studio.md)

- [Wprowadzenie do języka C# i programu Visual Studio Code](../../core/tutorials/with-visual-studio-code.md)

   Visual Studio Code to darmowy edytor kodu zoptymalizowany do tworzenia i debugowania nowoczesnych aplikacji internetowych i w chmurze. Obsługuje IntelliSense i jest dostępny dla systemów Linux, macOS i Windows.

   W tym temacie przedstawiono sposób tworzenia i uruchamiania prostej aplikacji Hello World z kodem programu Visual Studio i programiem .NET Core.
