# <a name="get-started-with-c"></a>Wprowadzenie do języka C# #

Ta sekcja zawiera krótki, prosty samouczek, dzięki któremu szybko stworzysz aplikację przy użyciu języka C# i .NET Core. Znajdziesz tu tematy wprowadzające do Visual Studio 2017 i Visual Studio Code. Możesz stworzyć prostą aplikację Hello World, albo, jeśli masz Visual Studio 2017, prostą bibliotekę klas, która może być używana przez inne aplikacje.

Dostępne są następujące tematy:

* [Wprowadzenie do języka C# i .NET Framework](introduction-to-the-csharp-language-and-the-net-framework.md)

     Zawiera omówienie języka C# i platformy .NET.

* [Tworzenie aplikacji Hello world w języku C# na platformie .NET Core w programie Visual Studio 2017](../../core/tutorials/with-visual-studio.md)

   Visual Studio 2017, czyli najnowsza wersja Visual Studio, umożliwia pisanie kodu, kompilację, uruchamianie, debugowanie, profilowanie oraz publikowanie własnych aplikacji ze zintegrowanego środowiska deweloperskiego dla systemu Windows.

   Temat ten umożliwi Ci stworzenie i uruchamienie prostej aplikacji Hello World, a następnie zmodyfikowanie jej do nieco bardziej interaktywnej. Jak już stworzysz i uruchomisz swoją aplikację, możesz też dowiedzieć się, jak [ją zdebugować](../../core/tutorials/debugging-with-visual-studio.md) oraz [opublikować](../../core/tutorials/publishing-with-visual-studio.md), tak aby można było ją uruchomić na dowolnej platformie wspieranej przez .NET Core.

* [Tworzenie biblioteki klas w języku C# i platformie .NET Core w Visual Studio 2017](../../core/tutorials/library-with-visual-studio.md)

   Biblioteka klas pozwala definiować typy i składniki typów, które mogą być wywoływane z innej aplikacji. Ten temat pokaże Ci jak utworzyć bibliotekę z jedną metodą, która sprawdza, czy ciąg znaków rozpoczyna się wielką literą. Po utworzeniu biblioteki, możesz stworzyć [test jednostkowy](../../core/tutorials/testing-library-with-visual-studio.md), aby upewnić się, że metoda działa zgodnie z oczekiwaniami, a następnie możesz udostępnić ją dla [aplikacji, które chcą z niej korzystać](../../core/tutorials/consuming-library-with-visual-studio.md).

* [Wprowadzenie do języka C# i programu Visual Studio Code](../../core/tutorials/with-visual-studio-code.md)

   Visual Studio Code to darmowy edytor kodu zoptymalizowany pod kątem tworzenia i debugowania nowoczesnych aplikacji webowych i chmurowych. Wspiera IntelliSense i jest dostępny na systemach Linux, macOS i Windows.

   W tym temacie przedstawiono sposób tworzenia i uruchamiania prostej aplikacji Hello World za pomocą Visual Studio i .NET Core.
