# <a name="get-started-with-c"></a>Wprowadzenie do języka C# #

Ta sekcja zawiera krótki, prosty samouczki, które pozwalają na szybkie tworzenie aplikacji przy użyciu języka C# i .NET Core. Dla programu Visual Studio 2017 i Visual Studio Code są wprowadzenie — tematy. Można utworzyć prostą aplikację Hello World lub, jeśli masz program Visual Studio 2017, biblioteki klas proste, które mogą być używane przez inne aplikacje.

Dostępne są następujące tematy:

* [Wprowadzenie do języka C# i .NET Framework](introduction-to-the-csharp-language-and-the-net-framework.md)

     Zawiera omówienie języka C# i platformy .NET.

* [Tworzenie aplikacji Hello world w języku C# na platformie .NET Core w programie Visual Studio 2017](../../core/tutorials/with-visual-studio.md)

   Visual Studio 2017 r najnowszej wersji programu Visual Studio umożliwia kodu, Kompiluj, uruchamiać, debugować profilu i publikowanie własnych aplikacji z zintegrowane środowisko deweloperskie dla systemu Windows.

   Temat umożliwia tworzenie i uruchamianie prostej aplikacji Hello World, a następnie zmodyfikować go do uruchomienia nieco większej liczby interaktywnych aplikacji Hello World. Po zakończeniu tworzenia i uruchamiania aplikacji, możesz też dowiedzieć się, jak [debugowania go](../../core/tutorials/debugging-with-visual-studio.md) oraz sposób [ją opublikować,](../../core/tutorials/publishing-with-visual-studio.md) , aby uruchomić na dowolnej platformie obsługiwany przez oprogramowanie .NET Core.

* [Tworzenie biblioteki klas w języku C# i na platformie .NET Core w programie Visual Studio 2017](../../core/tutorials/library-with-visual-studio.md)

   Biblioteka klas pozwala zdefiniować typy i elementy członkowskie typu, które mogą być wywoływane z innej aplikacji. W tym temacie umożliwia utworzenie biblioteki klas z pojedynczej metody, która określa, czy ciąg rozpoczyna się wielką literę. Po zakończeniu tworzenia biblioteki, można opracować [testu jednostkowego](../../core/tutorials/testing-library-with-visual-studio.md) aby upewnić się, że działa on zgodnie z oczekiwaniami, a następnie można udostępnić go do [aplikacji, które chcą korzystać z jego](../../core/tutorials/consuming-library-with-visual-studio.md).

* [Wprowadzenie do języka C# i programu Visual Studio Code](../../core/tutorials/with-visual-studio-code.md)

   Visual Studio Code to edytor kodu wolnego zoptymalizowane pod kątem kompilowanie i debugowanie nowoczesnych witryn sieci web i aplikacji w chmurze. Obsługuje funkcję IntelliSense, a jest dostępna dla systemów Linux, macOS i systemu Windows.

   W tym temacie przedstawiono sposób tworzenia i uruchamiania prostej aplikacji Hello World z kodem Visual Studio i .NET Core.
