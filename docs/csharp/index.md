# <a name="c-guide"></a>Przewodnik C#

Przewodnik C# zawiera wiele informacji na temat języka C#. Ta witryna ma wiele różnych odbiorców. W zależności od środowiska programowania lub przy użyciu języka C# i .NET możesz Poznaj różne sekcje w tym przewodniku.

* Dla deweloperów zupełnie nowym:
    - Uruchom z naszych [Szybki Start](quick-starts/index.md) sekcji. Pozwalają one Eksploruj języka C# interakcyjnego w przeglądarce. Z tego miejsca, można przenieść na naszych [samouczki](tutorials/index.md) sekcji. Te samouczki wyjaśniają, jak utworzyć od podstaw programów C#. Samouczki zawierają krok po kroku proces tworzenie programów. Dowiesz się pojęcia dotyczące języka oraz jak tworzyć własne programów C#. Jeśli wolisz najpierw odczytywania informacje ogólne, spróbuj naszych [samouczek języka C#](tour-of-csharp/index.md). Wyjaśniono pojęcia dotyczące języka C#. Po przeczytaniu tego, będziesz mieć podstawową wiedzę na temat języka i można wypróbować samouczków lub utworzyć coś samodzielnie.

* Dla deweloperów jesteś nowym użytkownikiem C#: 
    - Jeśli wykonane programowanie przed, ale dopiero zaczynasz korzystać z języka C#, przeczytaj [samouczek języka C#](tour-of-csharp/index.md). Dowiesz się podstawowa składnia i struktury języka i samouczek języka aby kontrast C# można użyć z innymi językami, który był używany. Możesz również przeglądać [samouczki](tutorials/index.md) próby podstawowe programów C#.

* Doświadczony C# deweloperów:
    - Jeśli używano C# przed, należy zacząć od odczytywanie w najnowszej wersji języka. Zapoznaj się z [nowości w języku C#](whats-new/index.md) nowych funkcji w bieżącej wersji.
