# <a name="tools-for-net-and-net-core-azure-developers"></a>Narzędzia dla deweloperów platformy Azure platformy .NET i core core

## <a name="sdk-downloads"></a>Pliki do pobrania sdk

Biblioteki platformy Azure dla platformy .NET są implementowane jako [pakiety NuGet](https://www.nuget.org/packages?q=windowsazureofficial). Zapoznaj się z [dokumentacją interfejsu API,](/dotnet/api/overview/azure/?view=azure-dotnet) aby uzyskać instrukcje instalacji uporządkowane przez usługę platformy Azure.

## <a name="development-tools"></a>Narzędzia programistyczne

Visual Studio ma narzędzia dla wielu usług platformy Azure wbudowanych. Niektóre usługi platformy Azure mają dostępne dodatkowe narzędzia lub emulatory, takie jak [Eksplorator usługi Azure Storage.](https://azure.microsoft.com/features/storage-explorer/) W razie potrzeby zapoznaj się z dokumentacją usługi platformy Azure, aby uzyskać dodatkowe narzędzia.

Te instrukcje instalują zalecane środowisko programistyczne dla systemu operacyjnego.

## <a name="windows"></a>[Windows](#tab/windows)

Visual Studio w wersjach 2017 i nowszych mają wbudowaną obsługę tworzenia platformy Azure. W poniższych krokach opisano włączenie obsługi programistycznej platformy Azure w programie Visual Studio.

W programie Visual Studio 2015 i wcześniejszych <a href="vs2015-install.md">należy postępować zgodnie z tymi instrukcjami.</a>

### <a name="step-1-download-visual-studio-2019"></a>Krok 1: Pobierz program Visual Studio 2019

Ten krok można pominąć, jeśli masz już zainstalowany program Visual Studio 2019.

> [!div class="nextstepaction"]
> [Pobierz program Visual Studio 2019](https://www.visualstudio.com/downloads/)

### <a name="step-2-install-the-two-azure-workloads"></a>Krok 2: Instalowanie dwóch obciążeń platformy Azure

W instalatorze programu Visual Studio zainstaluj program Visual Studio (lub zmodyfikuj istniejącą instalację). Upewnij się, że są wybrane obciążenia *deweloperów* i ASP.NET platformy Azure *i tworzenia sieci Web.*

### <a name="step-3-develop-with-net-on-azure"></a>Krok 3: Rozwijaj się za pomocą platformy .NET na platformie Azure

Rozpocznij [wdrażanie pierwszej ASP.NET aplikacji sieci Web w usłudze Azure App Service.](https://docs.microsoft.com/azure/app-service-web/app-service-web-get-started-dotnet)

## <a name="macos"></a>[macOS](#tab/macos)

Program Visual Studio dla komputerów Mac ma wszystko, czego potrzebujesz do tworzenia platformy Azure.

### <a name="step-1-download-visual-studio-for-mac"></a>Krok 1: Pobierz program Visual Studio dla komputerów Mac

> [!div class="nextstepaction"]
> [Pobieranie programu Visual Studio dla komputerów Mac](https://www.visualstudio.com/vs/visual-studio-mac/)

Podczas instalacji narzędzia platformy Azure są wybierane domyślnie.

## <a name="linux"></a>[Linux](#tab/linux)

Visual Studio Code ma wszystko, czego potrzebujesz do tworzenia platformy Azure w systemie Linux.

### <a name="step-1-download-the-net-core-sdk"></a>Krok 1: Pobierz podstawowy sdk .NET

Zestaw SDK i narzędzia wiersza polecenia dla aplikacji .NET Core.

> [!div class="nextstepaction"]
> [Pobierz core sdk .NET](https://dotnet.microsoft.com/download)

### <a name="step-2-visual-studio-code"></a>Krok 2: Kod programu Visual Studio

Edytuj i debuguj aplikacje .NET Core w dowolnym systemie operacyjnym: Windows, Mac i Linux.

> [!div class="nextstepaction"]
> [Pobierz kod programu Visual Studio](https://code.visualstudio.com)
