NET 5 +).</span><span class="sxs-lookup"><span data-stu-id="a6dfb-107">In this case, the runtime in question is called the **Common Language Runtime** or CLR, regardless of the implementation (for example, [Mono](https://www.mono-project.com/), .NET Framework, or .NET Core/.NET 5+).</span></span> <span data-ttu-id="a6dfb-108">Środowisko CLR jest odpowiedzialne za pobieranie kodu zarządzanego, kompilowanie go do kodu maszynowego i wykonywanie go.</span><span class="sxs-lookup"><span data-stu-id="a6dfb-108">CLR is in charge of taking the managed code, compiling it into machine code and then executing it.</span></span> <span data-ttu-id="a6dfb-109">Na początku środowisko uruchomieniowe zapewnia kilka ważnych usług, takich jak automatyczne zarządzanie pamięcią, granice zabezpieczeń, bezpieczeństwo typów itp.</span><span class="sxs-lookup"><span data-stu-id="a6dfb-109">On top of that, runtime provides several important services such as automatic memory management, security boundaries, type safety etc.</span></span>
