# <a name="usage-guidelines"></a>Wskazówki dotyczące użycia
Ta sekcja zawiera wskazówki dotyczące używania popularne typy w publicznie interfejsów API. Dotyczy on bezpośredniego użycia wbudowanych typów Framework (np. serializacji atrybutów) i przeładowanie operatorów wspólnej.  
  
 <xref:System.IDisposable?displayProperty=nameWithType> Interfejsu nie jest uwzględnione w tej sekcji, ale została szczegółowo opisana w [wzorzec Dispose](../../../docs/standard/design-guidelines/dispose-pattern.md) sekcji.  
  
> [!NOTE]
>  Wskazówki i dodatkowe informacje o innych typowe wbudowanych typów .NET Framework, zobacz Tematy odwołania dla następujących: <xref:System.DateTime?displayProperty=nameWithType>, <xref:System.DateTimeOffset?displayProperty=nameWithType>, <xref:System.ICloneable?displayProperty=nameWithType>, <xref:System.IComparable%601?displayProperty=nameWithType>, <xref:System.IEquatable%601?displayProperty=nameWithType>, <xref:System.Nullable%601?displayProperty=nameWithType>, <xref:System.Object?displayProperty=nameWithType> , <xref:System.Uri?displayProperty=nameWithType>.
