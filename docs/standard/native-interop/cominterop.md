# <a name="com-interop-in-net"></a>Współdziałanie modelu COM w programie .NET

Component Object Model (COM) umożliwia obiektowi uwidocznienie jego funkcjonalności innym składnikom i hostowanie aplikacji na platformach Windows. Aby ułatwić użytkownikom współdziałanie z istniejącymi bazami kodu, .NET Framework zawsze zapewniał silną pomoc techniczną dotyczącą współpracy z bibliotekami COM. W przypadku platformy .NET Core 3,0 w systemie Windows dodano dużą część tego wsparcia do programu .NET Core. W tej dokumentacji wyjaśniono, jak wspólna współdziałanie modelu COM Techonologies i jak można je wykorzystać do współpracy z istniejącymi bibliotekami COM.

- [Wywoływane otoki środowiska uruchomieniowego](./runtime-callable-wrapper.md)
