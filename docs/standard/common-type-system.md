## <a name="common-language-specification"></a><span data-ttu-id="21625-132">Specyfikacja języka wspólnego</span><span class="sxs-lookup"><span data-stu-id="21625-132">Common Language Specification</span></span>

<span data-ttu-id="21625-133">Aby włączyć pełne współdziałanie scenariuszy, wszystkie obiekty, które są tworzone w kodzie muszą polegać na wspólne właściwości w językach, które wykorzystują je (są ich _wywołań_).</span><span class="sxs-lookup"><span data-stu-id="21625-133">To enable full interoperability scenarios, all objects that are created in code must rely on some commonality in the languages that are consuming them (are their _callers_).</span></span> <span data-ttu-id="21625-134">Ponieważ istnieje wiele różnych języków, .NET została określona tych commonalities w coś o nazwie **specyfikacja języka wspólnego** (ze specyfikacją CLS).</span><span class="sxs-lookup"><span data-stu-id="21625-134">Since there are numerous different languages, .NET has specified those commonalities in something called the **Common Language Specification** (CLS).</span></span> <span data-ttu-id="21625-135">Ze specyfikacją CLS definiuje zestaw funkcji, które są wymagane przez wiele typowych zastosowań.</span><span class="sxs-lookup"><span data-stu-id="21625-135">CLS defines a set of features that are needed by many common applications.</span></span> <span data-ttu-id="21625-136">Umożliwia także sortowania przepisu dla dowolnego języka, który jest wdrażany w oparciu o .NET na potrzebne do obsługi.</span><span class="sxs-lookup"><span data-stu-id="21625-136">It also provides a sort of recipe for any language that is implemented on top of .NET on what it needs to support.</span></span>
