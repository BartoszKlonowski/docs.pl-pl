# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizacji istniejących aplikacji .NET dla aplikacji zoptymalizowanych pod kątem chmury

> Wizja: Modernizacji istniejących aplikacji .NET Framework aplikacjom zoptymalizowanych pod kątem chmury można zdecydowanie zwiększyć aktywności wdrożenia, aby można było wysłać szybciej i obniżyć koszty dostarczania aplikacji.

Aby skorzystać z zalet chmury i nowych technologii, takich jak kontenery, należy co najmniej częściowo modernizacji istniejących aplikacji .NET. Ostatecznie modernizacji aplikacje przedsiębiorstwa będzie obniżyć całkowity koszt posiadania.

Częściowo modernizacji aplikacji nie musi oznaczać pełnej migracji i rearchitecture. Można początkowo modernizacji istniejących aplikacji z ważne, ale łatwo zrobić modernizacji. Możesz zachować bieżącego kodu podstawowej, napisana istniejące wersje .NET Framework, wszelkie zależności systemu Windows i usług IIS. Rysunek 4-1 prezentuje sposób chmury zoptymalizowanych pod kątem aplikacji będą umieszczane w modelach dojrzałości modernizacji aplikacji Azure.
