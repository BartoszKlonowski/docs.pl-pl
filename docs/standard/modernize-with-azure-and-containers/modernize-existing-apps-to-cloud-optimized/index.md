# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizowanie istniejących aplikacji .NET do aplikacji zoptymalizowane pod kątem chmury

> Wizja: Modernizacja istniejących aplikacji .NET Framework do zoptymalizowane pod kątem chmury aplikacji, aby znacząco zwiększenie elastyczności wdrożenia, co umożliwia szybsze dostarczanie i niższe koszty dostarczania aplikacji.

Aby móc korzystać z zalet chmury i nowych technologii, takich jak kontenery, należy co najmniej częściowo modernizacji istniejących aplikacji .NET. Ostatecznie modernizujesz swoje aplikacje dla przedsiębiorstw będzie niższy całkowity koszt posiadania.

Częściowo modernizowanie aplikacji nie musi oznaczać pełnej migracji i rearchitecture. Można początkowo modernizacji istniejących aplikacji przy użyciu ważne, ale można łatwo zrobić modernizacji. Można zachować bieżącego kodu podstawowego, napisany w istniejącej wersji programu .NET Framework za pomocą jakieś zależności, Windows i usług IIS. Rysunek 4 — 1 pokazuje, jak zoptymalizowane pod kątem chmury aplikacji są umieszczone w modelach dojrzałości modernizacja aplikacji platformy Azure.
