- <span data-ttu-id="19c08-109">**Skalowalność w małych i średnich musi**: W scenariuszach, w których możesz potrzebować kilka maszyn wirtualnych w środowisku produkcyjnym zarządzanie niewielką liczbę maszyn wirtualnych może być przystępne cenowo, dopóki nie można przenieść do bardziej zaawansowanych środowisk PaaS, takich jak koordynatorów.</span><span class="sxs-lookup"><span data-stu-id="19c08-109">**Small and medium scalability needs**: In scenarios where you might need just a couple of VMs for your production environment, managing a small number of VMs might be affordable until you can move to more advanced PaaS environments, like orchestrators.</span></span>
