## <a name="example-project-an-anonymous-type-by-creating-objects-in-the-select-clause"></a><span data-ttu-id="791da-115">Przykład: projekt typu anonimowego przez utworzenie obiektów w `select` klauzuli</span><span class="sxs-lookup"><span data-stu-id="791da-115">Example: Project an anonymous type by creating objects in the `select` clause</span></span>

<span data-ttu-id="791da-116">W tym przykładzie `select` klauzula projektuje typ anonimowy.</span><span class="sxs-lookup"><span data-stu-id="791da-116">In this example, the `select` clause projects an anonymous type.</span></span> <span data-ttu-id="791da-117">Przykład używa `var` do tworzenia `IEnumerable` obiektu.</span><span class="sxs-lookup"><span data-stu-id="791da-117">The example then uses `var` to create the `IEnumerable` object.</span></span> <span data-ttu-id="791da-118">W `foreach` pętli Zmienna iteracji zostaje wystąpieniem typu anonimowego utworzonego w wyrażeniu zapytania.</span><span class="sxs-lookup"><span data-stu-id="791da-118">Within the `foreach` loop, the iteration variable becomes an instance of the anonymous type created in the query expression.</span></span>

<span data-ttu-id="791da-119">Ten przykład używa [przykładowego pliku XML dokumentu XML: klienci i zamówienia](sample-xml-file-customers-orders.md).</span><span class="sxs-lookup"><span data-stu-id="791da-119">This example uses XML document [Sample XML file: Customers and orders](sample-xml-file-customers-orders.md).</span></span>

```csharp
XElement custOrd = XElement.Load("CustomersOrders.xml");
var custList =
    from el in custOrd.Element("Customers").Elements("Customer")
    select new {
        CustomerID = (string)el.Attribute("CustomerID"),
        CompanyName = (string)el.Element("CompanyName"),
        ContactName = (string)el.Element("ContactName")
    };
foreach (var cust in custList)
    Console.WriteLine("{0}:{1}:{2}", cust.CustomerID, cust.CompanyName, cust.ContactName);
```

```vb
Dim custOrd As XElement = XElement.Load("CustomersOrders.xml")
Dim custList = _
    From el In custOrd.<Customers>.<Customer> _
    Select New With { _
        .CustomerID = el.@<CustomerID>, _
        .CompanyName = el.<CompanyName>.Value, _
        .ContactName = el.<ContactName>.Value _
    }
For Each cust In custList
    Console.WriteLine("{0}:{1}:{2}", cust.CustomerID, cust.CompanyName, cust.ContactName)
Next
```
