# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Wybór między .NET Core i .NET Framework dla kontenerów Docker

Istnieją dwa implementacje obsługiwanych dla tworzenia po stronie serwera konteneryzowanych Docker aplikacji z platformą .NET: [.NET Framework](https://www.microsoft.com/net/download/framework) i [.NET Core](https://www.microsoft.com/net/download/core). Mają wiele składników platformy .NET Standard, a kod mogą udostępniać między nimi. Istnieją jednak podstawowe różnice między nimi i implementacji, którego używasz będzie zależeć od co chcesz zrobić. Ta sekcja zawiera wskazówki dotyczące sytuacji wybrać każdego wdrożenia.
