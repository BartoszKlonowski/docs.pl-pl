<span data-ttu-id="964a0-164">Metadane w zestawach referencyjnych nadal przechowują następujące informacje:</span><span class="sxs-lookup"><span data-stu-id="964a0-164">The metadata in reference assemblies continues to keep the following information:</span></span>

- <span data-ttu-id="964a0-168">Jawne implementacje interfejsu.</span><span class="sxs-lookup"><span data-stu-id="964a0-168">Explicit interface implementations.</span></span>
- <span data-ttu-id="964a0-169">Jawnie zaimplementowane właściwości i zdarzenia, ponieważ ich metody dostępu są wirtualne.</span><span class="sxs-lookup"><span data-stu-id="964a0-169">Explicitly implemented properties and events, because their accessors are virtual.</span></span>
- <span data-ttu-id="964a0-170">Wszystkie pola struktur.</span><span class="sxs-lookup"><span data-stu-id="964a0-170">All fields of structures.</span></span>

<span data-ttu-id="964a0-171">Zestawy referencyjne zawierają atrybut [ReferenceAssembly](xref:System.Runtime.CompilerServices.ReferenceAssemblyAttribute) na poziomie zestawu.</span><span class="sxs-lookup"><span data-stu-id="964a0-171">Reference assemblies include an assembly-level [ReferenceAssembly](xref:System.Runtime.CompilerServices.ReferenceAssemblyAttribute) attribute.</span></span> <span data-ttu-id="964a0-172">Ten atrybut może być określony w źródle; Następnie kompilator nie będzie musiał go przeprowadzić.</span><span class="sxs-lookup"><span data-stu-id="964a0-172">This attribute may be specified in source; then the compiler won't need to synthesize it.</span></span> <span data-ttu-id="964a0-173">Z powodu tego atrybutu środowiska uruchomieniowe będą odrzucać ładowanie zestawów odwołań do wykonania (ale mogą być ładowane w trybie tylko odbicie).</span><span class="sxs-lookup"><span data-stu-id="964a0-173">Because of this attribute, runtimes will refuse to load reference assemblies for execution (but they can be loaded in reflection-only mode).</span></span>
