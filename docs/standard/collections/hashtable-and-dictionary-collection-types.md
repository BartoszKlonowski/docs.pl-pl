 <span data-ttu-id="c6c7b-118">Na przykład funkcję mieszania dla ciągu może potrwać kodów ASCII każdego znaku w ciągu i dodać je ze sobą, aby wygenerować wartość skrótu.</span><span class="sxs-lookup"><span data-stu-id="c6c7b-118">For example, a hash function for a string might take the ASCII codes of each character in the string and add them together to generate a hash code.</span></span> <span data-ttu-id="c6c7b-119">Ciąg "piknik" miałyby kod skrótu, który jest inny niż wartość skrótu dla ciągów "koszyka"; w związku z tym ciągi "piknik" i "koszyka" będzie należeć do różnych przedziałów.</span><span class="sxs-lookup"><span data-stu-id="c6c7b-119">The string "picnic" would have a hash code that is different from the hash code for the string "basket"; therefore, the strings "picnic" and "basket" would be in different buckets.</span></span> <span data-ttu-id="c6c7b-120">Z kolei "podkreślić" i "desery" miałyby tę samą wartość skrótu i będzie mieć tego samego zasobnika.</span><span class="sxs-lookup"><span data-stu-id="c6c7b-120">In contrast, "stressed" and "desserts" would have the same hash code and would be in the same bucket.</span></span>
