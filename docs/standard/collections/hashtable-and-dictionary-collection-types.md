 <span data-ttu-id="8d736-118">Na przykład funkcja skrótu dla ciągu może przyjmować kody ASCII każdego znaku w ciągu i dodawać je w celu wygenerowania kodu skrótu.</span><span class="sxs-lookup"><span data-stu-id="8d736-118">For example, a hash function for a string might take the ASCII codes of each character in the string and add them together to generate a hash code.</span></span> <span data-ttu-id="8d736-119">Ciąg "piknik" będzie miał kod skrótu, który różni się od kodu skrótu dla ciągu "koszyk"; w związku z tym ciągi "piknik" i "koszyk" byłyby w różnych zasobnikach.</span><span class="sxs-lookup"><span data-stu-id="8d736-119">The string "picnic" would have a hash code that is different from the hash code for the string "basket"; therefore, the strings "picnic" and "basket" would be in different buckets.</span></span> <span data-ttu-id="8d736-120">Z kolei "stresd" i "Desserts" będą mieć ten sam kod skrótu i znajdować się w tym samym przedziale.</span><span class="sxs-lookup"><span data-stu-id="8d736-120">In contrast, "stressed" and "desserts" would have the same hash code and would be in the same bucket.</span></span>
