# <a name="for-further-reading-parallel-programming"></a>Dalsze informacje (Programowanie równoległe)

Następujące zasoby zawierają dodatkowe informacje dotyczące programowania równoległego w programie .NET:

- [Wzorce programowania równoległego: Opis i zastosowanie wzorców równoległych z dokumentem .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) opisuje typowe wzorce równoległe i najlepsze rozwiązania dotyczące opracowywania składników równoległych z wykorzystaniem tych wzorców.

- [Wzorce projektowania dla dekompozycji i koordynacji na architektury wielordzeniowe](https://docs.microsoft.com/previous-versions/msp-n-p/ff963553(v=pandp.10)) książki opisano wzorce do programowania równoległego, które używają równoległej obsługi programowania wprowadzone w .NET Framework 4.

- [Programowanie równoległe z](https://devblogs.microsoft.com/pfxteam/) blogiem .NET zawiera wiele szczegółowych artykułów na temat programowania równoległego w programie .NET.

- [Przykłady do programowania równoległego ze stroną .NET Framework](https://code.msdn.microsoft.com/ParExtSamples) zawiera wiele przykładów, które demonstrują pośrednie i zaawansowane techniki programowania równoległego.
