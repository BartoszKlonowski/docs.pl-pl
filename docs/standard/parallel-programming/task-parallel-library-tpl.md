 <span data-ttu-id="abd1a-108">Począwszy od [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)], TPL jest preferowany sposób pisania kodu wielowątkowe i równolegle.</span><span class="sxs-lookup"><span data-stu-id="abd1a-108">Starting with the [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)], the TPL is the preferred way to write multithreaded and parallel code.</span></span> <span data-ttu-id="abd1a-109">Jednak nie cały kod nadaje się do przetwarzania równoległego.
