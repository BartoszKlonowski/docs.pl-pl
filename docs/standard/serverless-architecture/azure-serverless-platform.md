# <a name="azure-serverless-platform"></a>Platforma Azure bez użycia serwera

Bezserwerowa platforma Azure obejmuje usługi Azure Functions, Logic Apps i Event Grid. Usługi te współpracują ze sobą i połącz się z większą liczbą innych zasobów. Platformą bez użycia serwera współpracuje ze wszystkim z baz danych i magazynu w celu analytics i machine learning/sztucznej inteligencji.

Można również użyć usługi Application Insights, platformą bez użycia serwera do przechwytywania śladów diagnostyczne i dane telemetryczne. Usługa Application Insights są dostępne dla aplikacji wszystkich typów (komputerowe, aplikacje mobilne, lub w sieci web) oraz implementacji bez użycia serwera.
