 <span data-ttu-id="37266-108">Zabezpieczenia oparte na rolach platformy .NET obsługują autoryzację, tworząc informacje o podmiotu zabezpieczeń, który jest zbudowany ze skojarzonej tożsamości,  <span data-ttu-id="37266-117">Platforma .NET zapewnia obsługę zabezpieczeń opartą na rolach, która jest elastyczna i rozszerzalna, aby zaspokoić potrzeby szerokiego spektrum aplikacji.</span><span class="sxs-lookup"><span data-stu-id="37266-117">.NET provides role-based security support that is flexible and extensible enough to meet the needs of a wide spectrum of applications.</span></span> <span data-ttu-id="37266-118">Możesz zdecydować się na współdziałanie z istniejącymi infrastrukturami uwierzytelniania, takimi jak usługi COM+ 1,0, lub utworzyć niestandardowy system uwierzytelniania.</span><span class="sxs-lookup"><span data-stu-id="37266-118">You can choose to interoperate with existing authentication infrastructures, such as COM+ 1.0 Services, or to create a custom authentication system.</span></span> <span data-ttu-id="37266-119">Zabezpieczenia oparte na rolach są szczególnie odpowiednie do użycia w aplikacjach sieci Web ASP.NET, które są przetwarzane głównie na serwerze programu.</span><span class="sxs-lookup"><span data-stu-id="37266-119">Role-based security is particularly well-suited for use in ASP.NET Web applications, which are processed primarily on the server.</span></span> <span data-ttu-id="37266-120">Jednak zabezpieczeń opartych na rolach platformy .NET można używać zarówno na kliencie, jak i na serwerze.</span><span class="sxs-lookup"><span data-stu-id="37266-120">However, .NET role-based security can be used on either the client or the server.</span></span>
