# <a name="net-guide"></a>.NET — przewodnik

Przewodnik .NET zawiera informacje o platformie .NET. W zależności od znajomości programu .NET można zapoznać się z różnymi sekcjami tego przewodnika i innymi przewodnikami, takimi jak [.NET Core](../core/index.md) lub [.NET Framework](../framework/index.md).

## <a name="new-to-net"></a>Nowość w .NET

Jeśli chcesz uzyskać ogólny przegląd .NET, zobacz [Co to jest .NET?](https://dotnet.microsoft.com/learn/dotnet/what-is-dotnet).

Jeśli dopiero zaczynasz korzystać z programu .NET, zobacz [Wprowadzenie](get-started.md).

Jeśli wolisz wycieczkę z przewodnikiem po głównych funkcjach .NET, zobacz [Przewodnik po .NET](tour.md).

Aby zapoznać się z omówieniem różnych elementów programu .NET i ich dopasowania, zobacz [składniki architektury .NET](components.md).

## <a name="new-to-net-core"></a>Nowość w .NET Core

Jeśli jesteś nowym użytkownikiem programu .NET Core, zapoznaj się [z tematem Wprowadzenie do programu .NET Core](../core/get-started.md).

## <a name="new-to-net-standard"></a>Nowość w standardzie .NET

Jeśli jesteś nowym użytkownikem programu .NET Standard, zapoznaj się z [.NET Standard](net-standard.md).

## <a name="port-net-framework-code-to-net-standard-or-net-core"></a>Kod platformy .NET Framework do standardu .NET lub programu .NET Core

Aby przenieść aplikację, usługę, składnik lub pakiet NuGet do .NET Standard lub .NET Core, zobacz [Port to .NET Core z .NET Framework](../core/porting/index.md). Narzędzia dla .NET Standard i .NET Core jest współużytkowany, więc zawartość jest odpowiednia do przenoszenia do obu struktur.
