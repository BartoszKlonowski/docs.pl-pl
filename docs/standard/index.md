# <a name="net-guide"></a>.NET — przewodnik

Przewodnik .NET zawiera informacje o platformie .NET. W zależności od znajomości platformy .NET warto zapoznać się z różnymi sekcjami tego przewodnika i innymi przewodnikami, takimi jak [.NET Core](../core/index.md) lub [.NET Framework](../framework/index.yml).

## <a name="new-to-net"></a>Nowość w .NET

Jeśli chcesz zapoznać się z omówieniem platformy .NET na wysokim poziomie, zobacz [Co to jest .NET?](https://dotnet.microsoft.com/learn/dotnet/what-is-dotnet).

Jeśli jesteś nowym użytkownikem platformy .NET, zobacz [Wprowadzenie](get-started.md).

Jeśli wolisz, aby z przewodnikiem po głównych funkcjach platformy .NET można przejść do [programu .NET](tour.md).

Aby zapoznać się z omówieniem różnych elementów platformy .NET i ich dopasowania, zobacz [składniki architektoniczne platformy .NET](components.md).

## <a name="new-to-net-core"></a>Nowość w .NET Core

Jeśli dopiero zaczynasz pracę w .NET Core, zapoznaj się [z programem Get started with .NET Core](../core/get-started.md).

## <a name="new-to-net-standard"></a>Nowość w standardzie .NET

Jeśli jesteś nowym użytkownikem platformy .NET Standard, zapoznaj się z programem [.NET Standard](net-standard.md).

## <a name="port-net-framework-code-to-net-standard-or-net-core"></a>Kod programu .NET Framework do platformy .NET Standard lub .NET Core

Aby przenieść pakiet aplikacji, usługi, składnika lub NuGet do platformy .NET Standard lub .NET Core, zobacz [Port do .NET Core z programu .NET Framework](../core/porting/index.md). Narzędzia dla platformy .NET Standard i .NET Core są współużytkowane, więc zawartość jest odpowiednia do przenoszenia do obu struktur.
