# <a name="run-manage-and-monitor-docker-production-environments"></a>Uruchamianie, zarządzanie i monitorowanie środowisk produkcyjnych platformy Docker

Wizja: Aplikacje dla przedsiębiorstw potrzeba uruchomienia o wysokiej dostępności i wysokiej skalowalności; Dział operacji IT muszą mieć możliwość zarządzania i monitorowania tych środowisk i same aplikacje.

Ten filar ostatniego cyklu życia aplikacji konteneryzowanych Docker skupia się na sposób można uruchomić, Zarządzaj i Monitoruj swoje aplikacje w środowiskach produkcyjnych skalowalne o wysokiej dostępności (HA).

Sposób uruchamiania konteneryzowanych aplikacji w środowisku produkcyjnym (technologii architektury i platforma infrastruktury) jest również bardzo powiązane i całkowicie opiera się na wybranych platformach architektury i projektowania, które przyjrzeliśmy się w rozdziale 1 to -e-book. W tym rozdziale sprawdza, czy określonych produktów i technologii firmy Microsoft i innych dostawców, które służą do efektywnej pracy o wysokim stopniu skalowalności, wysokiej dostępności rozproszone aplikacje oraz jak można zarządzać i monitorować je z punktu widzenia działu IT.
