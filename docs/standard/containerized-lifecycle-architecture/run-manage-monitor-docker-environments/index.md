# <a name="run-manage-and-monitor-docker-production-environments"></a>Uruchom, zarządzanie i monitorowanie środowisk produkcyjnych Docker

Wizja: Aplikacje przedsiębiorstwa trzeba uruchamiać z wysoką dostępność i wysoką skalowalność; Dział operacji IT muszą mieć możliwość zarządzania i monitorowania tych środowisk i same aplikacje.

Ten ostatni słupka konteneryzowanych cyklu życia wnioski Docker skupia się na sposób można uruchamiać, zarządzanie i monitorowanie aplikacji w środowisku produkcyjnym skalowalne o wysokiej dostępności (HA).

Jak uruchomić konteneryzowanych aplikacji w środowisku produkcyjnym (technologii platforma i architektury infrastruktury) jest również bardzo powiązane i całkowicie opiera się na wybranych platformach architektury i programowania, które analizujemy w rozdziale 1 tego e-book. W tym rozdziale sprawdza określonych produktów i technologii firmy Microsoft i innych dostawców, które można skutecznie uruchamiać wysoce skalowalna, HA rozproszonych aplikacji oraz sposób zarządzania i monitorować je z punktu widzenia działu IT.
