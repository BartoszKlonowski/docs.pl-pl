# <a name="basic-string-operations-in-net"></a>Operacje ciągu podstawowego w programie .NET
Aplikacje często odpowiadają użytkownikom, tworząc wiadomości na podstawie danych wejściowych użytkownika. Na przykład nie jest rzadkością dla witryn sieci Web, aby odpowiedzieć na nowo zalogowanego użytkownika z wyspecjalizowanym powitaniem, który zawiera nazwę użytkownika. Kilka metod w <xref:System.String?displayProperty=nameWithType> <xref:System.Text.StringBuilder?displayProperty=nameWithType> klasach i umożliwiają dynamiczne konstruowanie ciągów niestandardowych do wyświetlania w interfejsie użytkownika. Metody te pomagają również wykonać szereg podstawowych operacji ciągu, takich jak tworzenie nowych ciągów z tablic bajtów, porównywanie wartości ciągów i modyfikowanie istniejących ciągów.
