# <a name="regular-expression-examples"></a>Przykłady wyrażeń regularnych
Ta sekcja zawiera przykłady kodu, które ilustrują użycie wyrażeń regularnych we wspólnych aplikacjach.  
  
> [!NOTE]
> <xref:System.Web.RegularExpressions> Przestrzeń nazw zawiera wiele obiektów wyrażeń regularnych, które implementują wstępnie zdefiniowane wzorce wyrażeń regularnych na potrzeby analizowania ciągów z dokumentów HTML, XML i ASP.NET. Na przykład <xref:System.Web.RegularExpressions.TagRegex> Klasa identyfikuje początkowe Tagi w ciągu <xref:System.Web.RegularExpressions.CommentRegex> , a Klasa identyfikuje Komentarze ASP.NET w ciągu.
