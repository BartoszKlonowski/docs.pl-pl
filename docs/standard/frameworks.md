## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="af6f9-195">Jak określić Platformy docelowe</span><span class="sxs-lookup"><span data-stu-id="af6f9-195">How to specify target frameworks</span></span>

<span data-ttu-id="af6f9-196">Struktury docelowe są określone w pliku projektu.</span><span class="sxs-lookup"><span data-stu-id="af6f9-196">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="af6f9-197">W przypadku określenia pojedynczej platformy docelowej należy użyć elementu **TargetFramework** .</span><span class="sxs-lookup"><span data-stu-id="af6f9-197">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="af6f9-198">Poniższy plik projektu aplikacji konsolowej pokazuje, jak kierować platformą .NET Core 2,2:</span><span class="sxs-lookup"><span data-stu-id="af6f9-198">The following console app project file demonstrates how to target .NET Core 2.2:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.2</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="af6f9-199">W przypadku określania wielu platform docelowych można warunkowo odwoływać się do zestawów dla każdej platformy docelowej.</span><span class="sxs-lookup"><span data-stu-id="af6f9-199">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="af6f9-200">W kodzie można warunkowo kompilować do tych zestawów przy użyciu symboli preprocesora z logiką *if-then-else* .</span><span class="sxs-lookup"><span data-stu-id="af6f9-200">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="af6f9-201">Następujący plik projektu biblioteki kieruje interfejsy API .NET Standard (`netstandard1.4`) i interfejsów API .NET Framework (`net40` i `net45`).</span><span class="sxs-lookup"><span data-stu-id="af6f9-201">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="af6f9-202">Użyj elementu **TargetFrameworks** w liczbie mnogiej z wieloma platformami docelowymi.</span><span class="sxs-lookup"><span data-stu-id="af6f9-202">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="af6f9-203">Należy pamiętać, `Condition` że atrybuty obejmują pakiety specyficzne dla implementacji, gdy biblioteka jest skompilowana dla dwóch .NET Framework TFMs:</span><span class="sxs-lookup"><span data-stu-id="af6f9-203">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="af6f9-204">W Twojej bibliotece lub aplikacji napiszesz kod warunkowy do skompilowania dla każdej platformy docelowej:</span><span class="sxs-lookup"><span data-stu-id="af6f9-204">Within your library or app, you write conditional code to compile for each target framework:</span></span>

```csharp
public class MyClass
{
    static void Main()
    {
#if NET40
        Console.WriteLine("Target framework: .NET Framework 4.0");
#elif NET45  
        Console.WriteLine("Target framework: .NET Framework 4.5");
#else
        Console.WriteLine("Target framework: .NET Standard 1.4");
#endif
    }
```
