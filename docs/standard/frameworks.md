## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="04436-193">Jak określić platform docelowych</span><span class="sxs-lookup"><span data-stu-id="04436-193">How to specify target frameworks</span></span>

<span data-ttu-id="04436-194">Platformy docelowe są określone w pliku projektu.</span><span class="sxs-lookup"><span data-stu-id="04436-194">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="04436-195">Kiedy struktura pojedynczy element docelowy jest określona, użyj **TargetFramework** elementu.</span><span class="sxs-lookup"><span data-stu-id="04436-195">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="04436-196">Następujący plik projektu aplikacji konsoli Pokazuje, jak i docelowej platformy .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="04436-196">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.0</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="04436-197">Podczas określania wielu platform docelowych możesz warunkowo może odwoływać się zestawów dla każdej platformy docelowej.</span><span class="sxs-lookup"><span data-stu-id="04436-197">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="04436-198">W kodzie, możesz warunkowo skompilować względem tych zestawów przy użyciu symboli preprocesora z *if-then-else* logiki.</span><span class="sxs-lookup"><span data-stu-id="04436-198">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="04436-199">Następujący plik projektu biblioteki jest przeznaczony dla interfejsów API programu .NET Standard (`netstandard1.4`) oraz interfejsów API programu .NET Framework (`net40` i `net45`).</span><span class="sxs-lookup"><span data-stu-id="04436-199">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="04436-200">Użyj liczbę mnogą **TargetFrameworks** elementu z wielu platform docelowych.</span><span class="sxs-lookup"><span data-stu-id="04436-200">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="04436-201">Uwaga jak `Condition` atrybuty zawierają specyficzne dla implementacji pakietów podczas kompilowania biblioteki dla dwóch .NET Framework krótkich nazw platform:</span><span class="sxs-lookup"><span data-stu-id="04436-201">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->

    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```
