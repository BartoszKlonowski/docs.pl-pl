## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="93f47-198">Jak określić ramy docelowe</span><span class="sxs-lookup"><span data-stu-id="93f47-198">How to specify target frameworks</span></span>

<span data-ttu-id="93f47-199">Platformy docelowe są określone w pliku projektu.</span><span class="sxs-lookup"><span data-stu-id="93f47-199">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="93f47-200">Po określeniu pojedynczej platformy docelowej należy użyć **elementu TargetFramework.**</span><span class="sxs-lookup"><span data-stu-id="93f47-200">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="93f47-201">Następujący plik projektu aplikacji konsoli pokazuje, jak kierować.NET Core 3.0:</span><span class="sxs-lookup"><span data-stu-id="93f47-201">The following console app project file demonstrates how to target .NET Core 3.0:</span></span>

```xml

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp3.0</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="93f47-202">Po określeniu wielu platform docelowych, można warunkowo odwoływać się do zestawów dla każdej platformy docelowej.</span><span class="sxs-lookup"><span data-stu-id="93f47-202">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="93f47-203">W kodzie można warunkowo skompilować względem tych zestawów przy użyciu symboli preprocesora z logiką *if-then-else.*</span><span class="sxs-lookup"><span data-stu-id="93f47-203">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="93f47-204">Następujący plik projektu biblioteki jest przeznaczony dla`netstandard1.4`interfejsów API standardu`net40` .NET ( ) i interfejsów API programu .NET Framework ( i `net45`).</span><span class="sxs-lookup"><span data-stu-id="93f47-204">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="93f47-205">Użyj elementu **TargetFrameworks** w liczbie mnogiej z wieloma platformami docelowymi.</span><span class="sxs-lookup"><span data-stu-id="93f47-205">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="93f47-206">Należy zauważyć, `Condition` jak atrybuty obejmują pakiety specyficzne dla implementacji, gdy biblioteka jest kompilowana dla dwóch tfmów .NET Framework:</span><span class="sxs-lookup"><span data-stu-id="93f47-206">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>

    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```
