## <a name="how-to-specify-a-target-framework"></a><span data-ttu-id="fcbd2-220">Jak określić platformę docelową</span><span class="sxs-lookup"><span data-stu-id="fcbd2-220">How to specify a target framework</span></span>

<span data-ttu-id="fcbd2-221">Struktury docelowe są określone w pliku projektu.</span><span class="sxs-lookup"><span data-stu-id="fcbd2-221">Target frameworks are specified in a project file.</span></span> <span data-ttu-id="fcbd2-222">W przypadku określenia pojedynczej platformy docelowej należy użyć elementu **TargetFramework** .</span><span class="sxs-lookup"><span data-stu-id="fcbd2-222">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="fcbd2-223">Poniższy plik projektu aplikacji konsolowej pokazuje, w jaki sposób należy kierować platformą .NET 5,0:</span><span class="sxs-lookup"><span data-stu-id="fcbd2-223">The following console app project file demonstrates how to target .NET 5.0:</span></span>

    <TargetFramework>net5.0</TargetFramework>
  </PropertyGroup>

<span data-ttu-id="fcbd2-224">W przypadku określania wielu platform docelowych można warunkowo odwoływać się do zestawów dla każdej platformy docelowej.</span><span class="sxs-lookup"><span data-stu-id="fcbd2-224">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="fcbd2-225">W kodzie można warunkowo kompilować do tych zestawów przy użyciu symboli preprocesora z logiką *if-then-else* .</span><span class="sxs-lookup"><span data-stu-id="fcbd2-225">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="fcbd2-226">Następujący projekt biblioteki odwołuje się do interfejsów API .NET Standard ( `netstandard1.4` ) i .NET Framework ( `net40` i `net45` ).</span><span class="sxs-lookup"><span data-stu-id="fcbd2-226">The following library project targets APIs of .NET Standard (`netstandard1.4`) and .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="fcbd2-227">Użyj elementu **TargetFrameworks** w liczbie mnogiej z wieloma platformami docelowymi.</span><span class="sxs-lookup"><span data-stu-id="fcbd2-227">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="fcbd2-228">`Condition`Atrybuty obejmują pakiety specyficzne dla implementacji, gdy biblioteka jest skompilowana dla dwóch .NET Framework TFMs:</span><span class="sxs-lookup"><span data-stu-id="fcbd2-228">The `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

<span data-ttu-id="fcbd2-229">W ramach biblioteki lub aplikacji napiszesz kod warunkowy przy użyciu [dyrektyw preprocesora](../csharp/language-reference/preprocessor-directives/preprocessor-if.md) , aby kompilować dla każdej platformy docelowej:</span><span class="sxs-lookup"><span data-stu-id="fcbd2-229">Within your library or app, you write conditional code using [preprocessor directives](../csharp/language-reference/preprocessor-directives/preprocessor-if.md) to compile for each target framework:</span></span>
