    - <span data-ttu-id="37d1e-110">Użyj <xref:System.Xml.XmlAttributeCollection.RemoveAt%2A>, aby usunąć atrybut z kolekcji atrybutów przy użyciu numeru indeksu.</span><span class="sxs-lookup"><span data-stu-id="37d1e-110">Use <xref:System.Xml.XmlAttributeCollection.RemoveAt%2A> to remove an attribute from the attribute collection by using its index number.</span></span>
