# <a name="inferring-schemas-from-xml-documents"></a><span data-ttu-id="3fb1a-102">Wnioskowanie schematów na podstawie dokumentów XML</span><span class="sxs-lookup"><span data-stu-id="3fb1a-102">Inferring Schemas from XML Documents</span></span>
<span data-ttu-id="3fb1a-103">W tym temacie opisano sposób użycia klasy <xref:System.Xml.Schema.XmlSchemaInference> do wywnioskowania schematu języka definicji schematu XML (XSD) ze struktury dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-103">This topic describes how to use the <xref:System.Xml.Schema.XmlSchemaInference> class to infer an XML Schema definition language (XSD) schema from the structure of an XML document.</span></span>  
  
## <a name="the-schema-inference-process"></a><span data-ttu-id="3fb1a-104">Proces wnioskowania schematu</span><span class="sxs-lookup"><span data-stu-id="3fb1a-104">The Schema Inference Process</span></span>  
 <span data-ttu-id="3fb1a-105">Klasa <xref:System.Xml.Schema.XmlSchemaInference> przestrzeni nazw <xref:System.Xml.Schema?displayProperty=nameWithType> służy do generowania co najmniej jednego schematu języka definicji schematu XML (XSD) ze struktury dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-105">The <xref:System.Xml.Schema.XmlSchemaInference> class of the <xref:System.Xml.Schema?displayProperty=nameWithType> namespace is used to generate one or more XML Schema definition language (XSD) schemas from the structure of an XML document.</span></span> <span data-ttu-id="3fb1a-106">Wygenerowane schematy mogą służyć do sprawdzania poprawności oryginalnego dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-106">The generated schemas may be used to validate the original XML document.</span></span>  
  
 <span data-ttu-id="3fb1a-107">Ponieważ dokument XML jest przetwarzany przez klasę <xref:System.Xml.Schema.XmlSchemaInference>, Klasa <xref:System.Xml.Schema.XmlSchemaInference> tworzy założenia dotyczące składników schematu, które opisują elementy i atrybuty w dokumencie XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-107">As an XML document is processed by the <xref:System.Xml.Schema.XmlSchemaInference> class, the <xref:System.Xml.Schema.XmlSchemaInference> class makes assumptions about the schema components that describe the elements and attributes in the XML document.</span></span> <span data-ttu-id="3fb1a-108">Klasa <xref:System.Xml.Schema.XmlSchemaInference> również wnioskuje składniki schematu w ograniczony sposób, wprowadzając najbardziej restrykcyjny typ dla określonego elementu lub atrybutu.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-108">The <xref:System.Xml.Schema.XmlSchemaInference> class also infers schema components in a constrained way by inferring the most restrictive type for a particular element or attribute.</span></span> <span data-ttu-id="3fb1a-109">Po zebraniu większej ilości informacji o dokumencie XML te ograniczenia są zmniejszane przez wnioskowanie mniej restrykcyjnych typów.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-109">As more information about the XML document is gathered, these constraints are loosened by inferring less restrictive types.</span></span> <span data-ttu-id="3fb1a-110">Typ najmniej restrykcyjny, który można wywnioskować, jest `xs:string`.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-110">The least restrictive type that can be inferred is `xs:string`.</span></span>  
  
 <span data-ttu-id="3fb1a-111">Zrób na przykład poniższy fragment dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-111">Take, for example, the following piece of an XML document.</span></span>  
  
```xml  
<parent attribute1="6">  
    <child>One</child>  
    <child>Two</child>  
</parent>  
<parent attribute1="A">  
```  
  
 <span data-ttu-id="3fb1a-112">W powyższym przykładzie, gdy atrybut `attribute1` zostanie napotkany z wartością `6` przez proces <xref:System.Xml.Schema.XmlSchemaInference>, przyjmuje się, że jest on typu `xs:unsignedByte`.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-112">In the example above, when the `attribute1` attribute is encountered with a value of `6` by the <xref:System.Xml.Schema.XmlSchemaInference> process, it is assumed to be of type `xs:unsignedByte`.</span></span> <span data-ttu-id="3fb1a-113">Gdy drugi `parent` element zostanie napotkany przez proces <xref:System.Xml.Schema.XmlSchemaInference>, ograniczenie zostanie oddzielone przez zmodyfikowanie typu na `xs:string`, ponieważ wartość atrybutu `attribute1` jest teraz `A`.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-113">When the second `parent` element is encountered by the <xref:System.Xml.Schema.XmlSchemaInference> process, the constraint is loosened by modifying the type to `xs:string` because the value of the `attribute1` attribute is now `A`.</span></span> <span data-ttu-id="3fb1a-114">Analogicznie, atrybut `minOccurs` dla wszystkich elementów `child` wywnioskowanych w schemacie jest przyznany do `minOccurs="0"`, ponieważ drugi element nadrzędny nie ma elementów podrzędnych.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-114">Similarly, the `minOccurs` attribute for all the `child` elements inferred in the schema are loosened to `minOccurs="0"` because the second parent element has no child elements.</span></span>  
  
## <a name="inferring-schemas-from-xml-documents"></a><span data-ttu-id="3fb1a-115">Wnioskowanie schematów na podstawie dokumentów XML</span><span class="sxs-lookup"><span data-stu-id="3fb1a-115">Inferring Schemas from XML Documents</span></span>  
 <span data-ttu-id="3fb1a-116">Klasa <xref:System.Xml.Schema.XmlSchemaInference> używa dwóch przeciążonych metod <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A> do wywnioskowania schematu z dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-116">The <xref:System.Xml.Schema.XmlSchemaInference> class uses two overloaded <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A> methods to infer a schema from an XML document.</span></span>  
  
 <span data-ttu-id="3fb1a-117">Pierwsza <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> Metoda jest używana do tworzenia schematu na podstawie dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-117">The first <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method is used to create a schema based on an XML document.</span></span> <span data-ttu-id="3fb1a-118">Druga metoda <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> służy do wnioskowania schematu opisującego wiele dokumentów XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-118">The second <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method is used to infer a schema that describes multiple XML documents.</span></span> <span data-ttu-id="3fb1a-119">Na przykład można utworzyć strumieniowo wiele dokumentów XML do metody <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> pojedynczo, aby wygenerować schemat, który opisuje cały zestaw dokumentów XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-119">For example, you can feed multiple XML documents to the <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method one at a time to produce a schema that describes the entire set of XML documents.</span></span>  
  
 <span data-ttu-id="3fb1a-120">Pierwsza <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> Metoda wnioskuje schemat z dokumentu XML zawartego w obiekcie <xref:System.Xml.XmlReader> i zwraca obiekt <xref:System.Xml.Schema.XmlSchemaSet> zawierający wywnioskowany schemat.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-120">The first <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method infers a schema from an XML document contained in an <xref:System.Xml.XmlReader> object, and returns an <xref:System.Xml.Schema.XmlSchemaSet> object containing the inferred schema.</span></span> <span data-ttu-id="3fb1a-121">Druga metoda <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> przeszukuje obiekt <xref:System.Xml.Schema.XmlSchemaSet> dla schematu z tą samą docelową przestrzenią nazw, co dokument XML zawarty w obiekcie <xref:System.Xml.XmlReader>, uściślije istniejący schemat i zwraca obiekt <xref:System.Xml.Schema.XmlSchemaSet> zawierający wywnioskowany schemat.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-121">The second <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method searches an <xref:System.Xml.Schema.XmlSchemaSet> object for a schema with the same target namespace as the XML document contained in the <xref:System.Xml.XmlReader> object, refines the existing schema, and returns an <xref:System.Xml.Schema.XmlSchemaSet> object containing the inferred schema.</span></span>  
  
 <span data-ttu-id="3fb1a-122">Zmiany wprowadzone w rafinowanym schemacie są oparte na nowej strukturze, która znajduje się w dokumencie XML.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-122">The changes made to the refined schema are based on new structure found in the XML document.</span></span> <span data-ttu-id="3fb1a-123">Na przykład w przypadku przechodzenia dokumentu XML założono założenia dotyczące znalezionych typów danych, a schemat jest tworzony na podstawie tych założeń.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-123">For example, as an XML document is traversed, assumptions are made about the data types found, and the schema is created based on those assumptions.</span></span> <span data-ttu-id="3fb1a-124">Jednakże w przypadku wystąpienia danych w drugim przebiegu wnioskowania, który różni się od oryginalnego założeń, schemat zostanie rafinowany.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-124">However, if data is encountered on a second inference pass that differs from the original assumption, the schema is refined.</span></span> <span data-ttu-id="3fb1a-125">Poniższy przykład ilustruje proces uściślania.</span><span class="sxs-lookup"><span data-stu-id="3fb1a-125">The following example illustrates the refinement process.</span></span>
