# <a name="node-collections-in-namednodemaps-and-nodelists"></a>Kolekcje węzłów: NamedNodeMaps i NodeLists
Można pobrać zestawu węzłów i umieść ją w kolekcji uporządkowane i nieuporządkowane. Podczas wprowadzania zestaw węzłów do nieuporządkowanej kolekcji, zestaw jest nazywany NamedNodeMap przez konsorcjum World Wide Web (W3C); można pobierać dane przy użyciu nazwy lub indeksu w kolekcji tego typu. Umieszczając zestaw węzłów w uporządkowaną kolekcję nosi nazwę wstawienia przez konsorcjum W3C, a dane mogą być pobierane według indeksu zaczynającego się od zera. I NodeLists są opisane przez konsorcjum W3C. Implementacje programu Microsoft .NET Framework dla NamedNodeMap jest **XmlNamedNodeMap**, i wstawienia jest implementowany przez **XmlNodeList**.  
  
 Aby uzyskać informacji na temat nieuporządkowanej kolekcji, zobacz [nieuporządkowaną pobierania węzła przez nazwę lub indeks](../../../../docs/standard/data/xml/unordered-node-retrieval-by-name-or-index.md). Informacje na temat uporządkowaną kolekcję można zobaczyć [uporządkowane pobierania węzła za pomocą indeksu](../../../../docs/standard/data/xml/ordered-node-retrieval-by-index.md).
