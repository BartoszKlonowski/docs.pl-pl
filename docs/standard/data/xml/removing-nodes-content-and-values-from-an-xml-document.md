# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Usuwanie węzłów, zawartości i wartości z dokumentu XML
Gdy Document Object Model XML (DOM) znajduje się w pamięci, można usunąć węzły z drzewa lub usunąć zawartość i wartości z określonych typów węzłów. Aby uzyskać informacje na temat usuwania węzła liścia lub całego poddrzewa węzła z dokumentu, zobacz [usuwanie węzłów z modelu dom](removing-nodes-from-the-dom.md). Aby uzyskać informacje na temat usuwania atrybutów elementu, zobacz [usuwanie atrybutów z węzła elementu w modelu dom](removing-attributes-from-an-element-node-in-the-dom.md). Aby uzyskać informacje dotyczące usuwania zawartości węzła, ale pozostawiając węzeł w drzewie, zobacz [usuwanie zawartości węzła w modelu dom](removing-node-content-in-the-dom.md).
