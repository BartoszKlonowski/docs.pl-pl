- <span data-ttu-id="50efd-121">**Wydajność.**</span><span class="sxs-lookup"><span data-stu-id="50efd-121">**Performance.**</span></span> <span data-ttu-id="50efd-122">Lepszą wydajność aplikacji niektóre z klas XML w programie .NET Framework obsługuje model oparty na przesyłania strumieniowego o następującej charakterystyce:</span><span class="sxs-lookup"><span data-stu-id="50efd-122">For better app performance, some of the XML classes in the .NET Framework support a streaming-based model with the following characteristics:</span></span>
