# <a name="inserting-nodes-into-an-xml-document"></a>Wstawianie węzłów do dokumentu XML
Wstawianie nowych węzłów można zmodyfikować dokumentu. Węzły mogą być nowo tworzone istniejący w tym samym dokumencie lub mogą być importowane z innego dokumentu. Aby uzyskać więcej informacji o tworzeniu nowych węzłów, zobacz [utworzyć nowe węzły w modelu DOM](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md). Uzyskać więcej informacji o kopiowaniu istniejących węzłów, zobacz [kopiowania istniejących węzłów](../../../../docs/standard/data/xml/copy-existing-nodes.md), [kopiowanie istniejących węzłów z jednego dokumentu do innego](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md), lub [kopiowanie fragmenty dokumentu](../../../../docs/standard/data/xml/copying-document-fragments.md).
