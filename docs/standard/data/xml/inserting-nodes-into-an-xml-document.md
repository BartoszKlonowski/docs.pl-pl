# <a name="inserting-nodes-into-an-xml-document"></a>Wstawianie węzłów do dokumentu XML
Możesz zmodyfikować dokument, wstawiając nowe węzły. Węzły mogą być nowo tworzone, istniejące w tym samym dokumencie lub mogą być importowane z innego dokumentu. Aby uzyskać więcej informacji na temat tworzenia nowych węzłów, zobacz [Tworzenie nowych węzłów w modelu dom](create-new-nodes-in-the-dom.md). Aby uzyskać więcej informacji na temat kopiowania istniejących węzłów, zobacz [Kopiowanie istniejących węzłów](copy-existing-nodes.md), [Kopiowanie istniejących węzłów z jednego dokumentu do innego](copying-existing-nodes-from-one-document-to-another.md)lub [Kopiowanie fragmentów dokumentów](copying-document-fragments.md).
