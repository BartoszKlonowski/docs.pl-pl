 <span data-ttu-id="f49e6-106">Zapytania w LINQ to XML są zwięzłe i wyraźne, przy użyciu składni bardziej podobnej do SQL niż XPath lub XQuery.</span><span class="sxs-lookup"><span data-stu-id="f49e6-106">Queries in LINQ to XML are succinct and expressive, using syntax more similar to SQL than to XPath or XQuery.</span></span> <span data-ttu-id="f49e6-107">Ponieważ wyniki zapytania mogą być zwracane jako kolekcje elementów lub atrybutów i mogą być używane jako parametry dla obiektów XElement, drzewa XML mogą być łatwo przekształcone z jednego kształtu na drugi.</span><span class="sxs-lookup"><span data-stu-id="f49e6-107">Because query results can be returned as collections of elements or attributes and can be used as parameters for XElement objects, XML trees can be easily transformed from one shape to another.</span></span>
