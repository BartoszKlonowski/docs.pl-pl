# <a name="copying-document-fragments"></a>Kopiowanie fragmentów dokumentu
Można utworzyć węzeł **XmlDocumentFragment** , a następnie dodać do niego węzły. Gdy **XmlDocumentFragment** zostanie wstawiony przy użyciu metody **insertNode** , węzeł **XmlDocumentFragment** nie jest kopiowany, ale jego węzły podrzędne są wstawiane do Document Object Model XML (dom).
