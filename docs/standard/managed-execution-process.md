prawidłowo zdefiniowane.</span><span class="sxs-lookup"><span data-stu-id="98c57-168">During the verification process, MSIL code is examined in an attempt to confirm that the code can access memory ## <a name="running-code"></a><span data-ttu-id="98c57-175">Wykonywanie kodu</span><span class="sxs-lookup"><span data-stu-id="98c57-175">Running Code</span></span>  
 <span data-ttu-id="98c57-176">Środowisko uruchomieniowe języka wspólnego oferuje infrastrukturę, która umożliwia wykonanie zarządzanych odbywa się i usług, które mogą być używane podczas wykonywania.</span><span class="sxs-lookup"><span data-stu-id="98c57-176">The common language runtime provides the infrastructure that enables managed execution to take place and services that can be used during execution.</span></span> <span data-ttu-id="98c57-177">Przed uruchomieniem metody, musi być skompilowany kod specyficznych dla procesora.</span><span class="sxs-lookup"><span data-stu-id="98c57-177">Before a method can be run, it must be compiled to processor-specific code.</span></span> <span data-ttu-id="98c57-178">Każda metoda, dla którego wygenerowano MSIL jest kompilacji JIT, gdy jest wywoływana po raz pierwszy, a następnie uruchom.</span><span class="sxs-lookup"><span data-stu-id="98c57-178">Each method for which MSIL has been generated is JIT-compiled when it is called for the first time, and then run.</span></span> <span data-ttu-id="98c57-179">Przy następnym uruchomieniu metoda uruchamiania istniejącego kodu natywnej kompilacji JIT.</span><span class="sxs-lookup"><span data-stu-id="98c57-179">The next time the method is run, the existing JIT-compiled native code is run.</span></span> <span data-ttu-id="98c57-180">Proces kompilacji JIT, a następnie uruchamiając kod jest powtarzany aż do ukończenia wykonywania.</span><span class="sxs-lookup"><span data-stu-id="98c57-180">The process of JIT-compiling and then running the code is repeated until execution is complete.</span></span>

1.  <span data-ttu-id="98c57-186">Zapewnia, że kod jest prawidłowy kod zarządzany.</span><span class="sxs-lookup"><span data-stu-id="98c57-186">Ensures that the code is valid managed code.</span></span>  
  
2.  <span data-ttu-id="98c57-187">Punkt wejścia w obrazie zmienia się na punkt wejścia w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="98c57-187">Changes the entry point in the image to an entry point in the runtime.</span></span>  
  
 <span data-ttu-id="98c57-188">W 64-bitowej wersji systemu Windows `_CorValidateImage` modyfikuje obrazu, który znajduje się w pamięci przez przekształcania z plikiem PE32 plikiem PE32 + format.</span><span class="sxs-lookup"><span data-stu-id="98c57-188">On 64-bit Windows, `_CorValidateImage` modifies the image that is in memory by transforming it from PE32 to PE32+ format.</span></span>
