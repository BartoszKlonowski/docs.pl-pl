 <span data-ttu-id="2d1f7-115">Wiele metod klas zarządzanych wątków zapewnia przekroczenia limitu czasu, aby pomóc w wykrywaniu zakleszczenii.</span><span class="sxs-lookup"><span data-stu-id="2d1f7-115">Many methods of the managed threading classes provide time-outs to help you detect deadlocks.</span></span> <span data-ttu-id="2d1f7-116">Na przykład poniższy kod próbuje uzyskać blokadę obiektu o nazwie `lockObject` .</span><span class="sxs-lookup"><span data-stu-id="2d1f7-116">For example, the following code attempts to acquire a lock on an object named `lockObject`.</span></span> <span data-ttu-id="2d1f7-117">Jeśli blokada nie zostanie uzyskana w 300 milisekundach, <xref:System.Threading.Monitor.TryEnter%2A?displayProperty=nameWithType> zwraca `false` .</span><span class="sxs-lookup"><span data-stu-id="2d1f7-117">If the lock is not obtained in 300 milliseconds, <xref:System.Threading.Monitor.TryEnter%2A?displayProperty=nameWithType> returns `false`.</span></span>  
  
```vb  
If Monitor.TryEnter(lockObject, 300) Then  
    Try  
        ' Place code protected by the Monitor here.  
    Finally  
        Monitor.Exit(lockObject)  
    End Try  
Else  
    ' Code to execute if the attempt times out.  
End If  
```  
  
```csharp  
if (Monitor.TryEnter(lockObject, 300)) {  
    try {  
        // Place code protected by the Monitor here.  
    }  
    finally {  
        Monitor.Exit(lockObject);  
    }  
}  
else {  
    // Code to execute if the attempt times out.

```  
  
### <a name="race-conditions"></a><span data-ttu-id="2d1f7-118">Warunki wyścigu</span><span class="sxs-lookup"><span data-stu-id="2d1f7-118">Race conditions</span></span>

 <span data-ttu-id="2d1f7-121">Prosty przykład warunku wyścigu powoduje zwiększenie pola.</span><span class="sxs-lookup"><span data-stu-id="2d1f7-121">A simple example of a race condition is incrementing a field.</span></span> <span data-ttu-id="2d1f7-122">Załóżmy, że Klasa ma prywatne pole **statyczne** (**udostępniane** w Visual Basic), które jest zwiększane za każdym razem, gdy tworzone jest wystąpienie klasy, przy użyciu kodu, takiego jak `objCt++;` (C#) lub `objCt += 1` (Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="2d1f7-122">Suppose a class has a private **static** field (**Shared** in Visual Basic) that is incremented every time an instance of the class is created, using code such as `objCt++;` (C#) or `objCt += 1` (Visual Basic).</span></span> <span data-ttu-id="2d1f7-123">Ta operacja wymaga załadowania wartości z `objCt` do rejestru, zwiększenia wartości i zapisania jej w `objCt` .</span><span class="sxs-lookup"><span data-stu-id="2d1f7-123">This operation requires loading the value from `objCt` into a register, incrementing the value, and storing it in `objCt`.</span></span>
