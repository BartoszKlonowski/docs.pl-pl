## <a name="remarks"></a><span data-ttu-id="ed028-145">Uwagi</span><span class="sxs-lookup"><span data-stu-id="ed028-145">Remarks</span></span>  
 <span data-ttu-id="ed028-146">Jeśli Generator serializatora XML nie jest używana, <xref:System.Xml.Serialization.XmlSerializer> generuje kod serializacji i zestawu serializacji dla każdego typu za każdym razem, gdy aplikacja jest uruchamiana.</span><span class="sxs-lookup"><span data-stu-id="ed028-146">When the XML Serializer Generator is not used, a <xref:System.Xml.Serialization.XmlSerializer> generates serialization code and a serialization assembly for each type every time an application is run.</span></span> <span data-ttu-id="ed028-147">Aby zwiększyć wydajność uruchamiania serializacji XML, narzędzie Sgen.exe do generowania te zestawy z wyprzedzeniem.</span><span class="sxs-lookup"><span data-stu-id="ed028-147">To improve the performance of XML serialization startup, use the Sgen.exe tool to generate those assemblies in advance.</span></span> <span data-ttu-id="ed028-148">Następnie można wdrożyć te zestawy z aplikacją.</span><span class="sxs-lookup"><span data-stu-id="ed028-148">These assemblies can then be deployed with the application.</span></span>  
  
 <span data-ttu-id="ed028-149">Generator serializatora XML może również zwiększyć wydajność klientów korzystających z serwerów proxy usług sieci Web XML do komunikowania się z serwerami, ponieważ ten proces serializacji nie będą powodować wydajności trafień, gdy typ jest ładowany po raz pierwszy.</span><span class="sxs-lookup"><span data-stu-id="ed028-149">The XML Serializer Generator can also improve the performance of clients that use XML Web service proxies to communicate with servers because the serialization process will not incur a performance hit when the type is loaded the first time.</span></span>  
  
 <span data-ttu-id="ed028-150">Nie można używać tych wygenerowanych zestawów po stronie serwera usługi sieci Web.</span><span class="sxs-lookup"><span data-stu-id="ed028-150">These generated assemblies cannot be used on the server side of a Web service.</span></span> <span data-ttu-id="ed028-151">Narzędzie to jest tylko dla klientów usługi sieci Web i scenariusze ręczne serializacji.</span><span class="sxs-lookup"><span data-stu-id="ed028-151">This tool is only for Web service clients and manual serialization scenarios.</span></span>  
  
 <span data-ttu-id="ed028-152">Jeśli zestawu zawierającego typ do serializacji nosi MyType.dll, następnie zestawu skojarzonego serializacji będzie miał nazwę MyType.XmlSerializers.dll.</span><span class="sxs-lookup"><span data-stu-id="ed028-152">If the assembly containing the type to serialize is named MyType.dll, then the associated serialization assembly will be named MyType.XmlSerializers.dll.</span></span>
