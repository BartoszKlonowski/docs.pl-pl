# <a name="schemaimporterextension-technology-sample"></a>Przykład technologii SchemaImporterExtension
[Pobierz przykładowe](https://download.microsoft.com/download/4/7/B/47B2164C-E780-4B10-8DE4-2CB5B886E0A6/Technologies/Serialization/Xml%20Serialization/SchemaImporterExtension.zip.exe)  
  
 Ten przykład pokazuje niestandardowy <xref:System.Xml.Serialization.Advanced.SchemaImporterExtension> , umożliwia szczegółową kontrolę generowania kodu po zaimportowaniu schematu XML. Aplikacja przedstawiono sposób tworzenia, zarejestruj się i wywołania to rozszerzenie.  
  
### <a name="to-build-the-sample-using-the-command-prompt"></a>Aby skompilować przykład za pomocą wiersza polecenia  
  
1. Otwórz okno wiersza polecenia i przejdź do jednej z podkatalogi specyficzny dla języka dla próbki.  
  
2. Typ **msbuild.exe OrderSchemaImporterExtension.sln** w wierszu polecenia.  
  
### <a name="to-build-the-sample-using-visual-studio"></a>Aby skompilować przykład za pomocą programu Visual Studio  
  
1. Otwórz Eksplorator plików i przejdź do jednej z podkatalogi specyficzny dla języka dla próbki.  
  
2. Kliknij dwukrotnie ikonę OrderSchemaImporterExtension.sln do otwierania tego PLiku w programie Visual Studio.  
  
3. Na **kompilacji** menu, kliknij przycisk **Kompiluj rozwiązanie**.  
  
 Aplikacja zostanie utworzona w domyślnym katalogu \bin lub \bin\Debug.  
  
### <a name="to-run-the-sample"></a>Aby uruchomić przykład  
  
1. Przejdź do katalogu zawierającego nowy PLik wykonywalny, za pomocą wiersza polecenia.  
  
2. Typ **[nazwa exe]** w wierszu polecenia.
