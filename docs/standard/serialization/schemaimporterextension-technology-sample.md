# <a name="schemaimporterextension-technology-sample"></a>Przykładowy element SchemaImporterExtension technologii
[Pobieranie próbki](https://download.microsoft.com/download/4/7/B/47B2164C-E780-4B10-8DE4-2CB5B886E0A6/Technologies/Serialization/Xml%20Serialization/SchemaImporterExtension.zip.exe)  
  
 W tym przykładzie pokazano niestandardowego <xref:System.Xml.Serialization.Advanced.SchemaImporterExtension> który umożliwia szczegółową kontrolę generowania kodu podczas importowania schematu XML. Aplikacja pokazuje, jak do kompilacji, rejestrowania i wywołanie tego rozszerzenia.  
  
### <a name="to-build-the-sample-using-the-command-prompt"></a>Aby samodzielnie tworzyć przykładowy przy użyciu wiersza polecenia  
  
1.  Otwórz okno wiersza polecenia i przejdź do jednej z przykładowej podkatalogi specyficzny dla języka.  
  
2.  Typ **msbuild.exe OrderSchemaImporterExtension.sln** w wierszu polecenia.  
  
### <a name="to-build-the-sample-using-visual-studio"></a>Aby samodzielnie tworzyć przykładowy przy użyciu programu Visual Studio  
  
1.  Otwórz [!INCLUDE[fileExplorer](../../../includes/fileexplorer-md.md)] i przejdź do jednej z przykładowej podkatalogi specyficzny dla języka.  
  
2.  Kliknij dwukrotnie ikonę OrderSchemaImporterExtension.sln do otwierania tego PLiku w programie Visual Studio.  
  
3.  Na **kompilacji** menu, kliknij przycisk **Kompiluj rozwiązanie**.  
  
 Aplikacja zostanie utworzona w domyślnym katalogu \bin lub \bin\Debug.  
  
### <a name="to-run-the-sample"></a>Aby uruchomić przykładowy  
  
1.  Przejdź do katalogu zawierającego nowy PLik wykonywalny, za pomocą wiersza polecenia.  
  
2.  Typ **[nazwa pliku exe]** w wierszu polecenia.
