# <a name="visual-basic-language-specification"></a>Specyfikacja języka Visual Basic
Specyfikacja języka Visual Basic jest wiarygodne źródło odpowiedzi na wszystkie pytania o gramatyki języka Visual Basic i składni. Zawiera szczegółowe informacje na temat języka, w tym wiele punktów nieuwzględnione w dokumentacji referencyjnej Visual Basic.  
  
 Jest dostępna w specyfikacji [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=188623).  
  
Ta witryna zawiera [specyfikacji VB 11](../../../../_vblang/spec/introduction.md). Jest zbudowany zawartych w plikach Markdown [repozytorium GitHub dotnet/vblang](https://github.com/dotnet/vblang/blob/master/spec/README.md).

Problemy na specyfikacji powinny zostać utworzone w [dotnet/vblang](https://github.com/dotnet/vblang/issues) repozytorium. Lub, jeśli interesuje Cię poprawiania błędów można znaleźć, możesz przesłać [żądanie ściągnięcia](https://github.com/dotnet/vblang/pulls) do tego samego repozytorium.
