# <a name="visual-basic-language-specification"></a>Specyfikacja języka Visual Basic
Specyfikacja języka Visual Basic jest autorytatywne źródło dla odpowiedzi na wszystkie pytania o gramatyki języka Visual Basic i składni. Zawiera szczegółowe informacje na temat języka, łącznie z wieloma punktami nieuwzględnione w dokumentacji języka Visual Basic.  
  
 Specyfikacja jest dostępna w [Microsoft Download Center](https://go.microsoft.com/fwlink/?LinkId=188623).  
  
Ta witryna zawiera [specyfikacji VB 11](../../../../_vblang/spec/introduction.md). Jest zbudowany pliki Markdown znajdujące się w [repozytorium dotnet/vblang GitHub](https://github.com/dotnet/vblang/blob/master/spec/README.md).

Problemy na specyfikacji powinny zostać utworzone w [dotnet/vblang](https://github.com/dotnet/vblang/issues) repozytorium. Lub, jeśli interesują Cię poprawiania błędów można znaleźć, możesz przesłać [żądania ściągnięcia](https://github.com/dotnet/vblang/pulls) na tym samym repozytorium.
