# <a name="accessing-user-data-visual-basic"></a>Uzyskiwanie dostępu do danych użytkownika (Visual Studio)
Ta sekcja zawiera tematy dotyczące `My.User` obiektu i zadań, które można wykonać za pomocą go.  
  
 `My.User` Obiektu zapewnia dostęp do informacji na temat zalogowanego użytkownika, zwracając obiekt, który implementuje <xref:System.Security.Principal.IPrincipal> interfejsu.

|Określenie roli użytkownika|<xref:Microsoft.VisualBasic.ApplicationServices.User.IsInRole%2A>|
