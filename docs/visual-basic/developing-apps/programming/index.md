# <a name="programming-in-visual-basic"></a>Programowanie w Visual Basic
W tej sekcji omówiono zadania programistyczne, które warto dowiedzieć się, jak utworzyć aplikację Visual Basic.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Uzyskiwanie dostępu do zasobów komputera](../../../visual-basic/developing-apps/programming/computer-resources/index.md)  
 Zawiera dokumentację dotyczącą sposobu korzystania z obiektu `My.Computer` w celu uzyskania dostępu do informacji o komputerze, na którym działa aplikacja, oraz o sposobie sterowania komputerem.  
  
 [Rejestrowanie informacji z aplikacji](../../../visual-basic/developing-apps/programming/log-info/index.md)  
 Zawiera dokumentację dotyczącą rejestrowania informacji z aplikacji przy użyciu obiektów `My.Application.Log` i `My.Log` oraz sposobu, w jaki można zwiększyć możliwości rejestrowania aplikacji.  
  
 [Uzyskiwanie dostępu do danych użytkownika](../../../visual-basic/developing-apps/programming/accessing-user-data.md)  
 Zawiera dokumentację zadań, które można wykonać przy użyciu obiektu `My.User`.  
  
 [Uzyskiwanie dostępu do formularzy aplikacji](../../../visual-basic/developing-apps/programming/accessing-application-forms.md)  
 Zawiera dokumentację dotyczącą uzyskiwania dostępu do formularzy aplikacji przy użyciu obiektów `My.Forms` i `My.Application`.  
  
 [Uzyskiwanie dostępu do usług sieci Web aplikacji](../../../visual-basic/developing-apps/programming/accessing-application-web-services.md)  
 Zawiera dokumentację dotyczącą sposobu uzyskiwania dostępu do usług sieci Web, do których odwołuje się aplikacja, przy użyciu obiektu `My.WebServices`.  
  
 [Uzyskiwanie dostępu do ustawień aplikacji](../../../visual-basic/developing-apps/programming/app-settings/index.md)  
 Zawiera dokumentację dotyczącą uzyskiwania dostępu do ustawień aplikacji za pomocą obiektu `My.Settings`.  
  
 [Przetwarzanie dysków, katalogów i plików](drives-directories-files/index.md)  
 Zawiera dokumentację dotyczącą sposobu uzyskiwania dostępu do systemu plików przy użyciu obiektu `My.Computer.FileSystem`.
