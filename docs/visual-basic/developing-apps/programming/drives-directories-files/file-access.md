# <a name="file-access-with-visual-basic"></a>Dostęp do plików za pomocą Visual Basic

Obiekt `My.Computer.FileSystem` zawiera narzędzia do pracy z plikami i folderami. Jego właściwości, metody i zdarzenia umożliwiają tworzenie, kopiowanie, przenoszenie, badanie i usuwanie plików i folderów. `My.Computer.FileSystem` zapewnia lepszą wydajność niż starsze funkcje (`FileOpen`, `FileClose`, `Input`, `InputString`, `LineInput`itp.), które są udostępniane przez Visual Basic w celu zapewnienia zgodności z poprzednimi wersjami.
