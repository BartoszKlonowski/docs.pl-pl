# <a name="file-access-with-visual-basic"></a>Dostęp do plików za pomocą Visual Basic
`My.Computer.FileSystem` Obiektu udostępnia narzędzia do pracy z plikami i folderami. Jego właściwości, metody i zdarzenia umożliwiają tworzenie, skopiować, przenieść, badanie i usuwać pliki i foldery. `My.Computer.FileSystem` zapewnia lepszą wydajność niż starsze funkcje (`FileOpen`, `FileClose`, `Input`, `InputString`, `LineInput`, itp.) są dostarczane przez program Visual Basic dla zgodności z poprzednimi wersjami.
