# <a name="logging-information-from-the-application-visual-basic"></a>Rejestrowanie informacji z aplikacji (Visual Basic)
Ta sekcja zawiera tematy, które opisano, jak rejestrowanie informacji z swojej aplikacji za pomocą `My.Application.Log` lub `My.Log` obiekt i jak rozszerzyć możliwości rejestrowania zdarzeń aplikacji.  
  
 `Log` Obiektu zapewnia metody zapisywanie informacji o aplikacji odbiorniki dzienników i `Log` zaawansowane obiektu `TraceSource` właściwości przedstawia informacje szczegółowe. `Log` Obiekt został skonfigurowany w pliku konfiguracji aplikacji.  
  
 `My.Log` Obiekt jest dostępny tylko dla aplikacji ASP.NET. Dla aplikacji klienckich, należy użyć `My.Application.Log`. Aby uzyskać więcej informacji, zobacz <xref:Microsoft.VisualBasic.Logging.Log>.
