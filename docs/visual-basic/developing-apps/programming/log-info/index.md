# <a name="logging-information-from-the-application-visual-basic"></a>Rejestrowanie informacji z aplikacji (Visual Basic)

Ta sekcja zawiera tematy, które obejmują sposób `My.Application.Log` rejestrowania informacji z aplikacji przy użyciu obiektu lub `My.Log` obiektu i jak rozszerzyć możliwości rejestrowania aplikacji.  
  
 Obiekt `Log` udostępnia metody zapisywania informacji do detektorów dziennika `Log` aplikacji, a `TraceSource` zaawansowana właściwość obiektu zawiera szczegółowe informacje o konfiguracji. Obiekt `Log` jest konfigurowany przez plik konfiguracyjny aplikacji.  
  
 Obiekt `My.Log` jest dostępny tylko dla aplikacji ASP.NET. W przypadku aplikacji `My.Application.Log`klienckich należy użyć pliku . Aby uzyskać więcej informacji, zobacz <xref:Microsoft.VisualBasic.Logging.Log>.
