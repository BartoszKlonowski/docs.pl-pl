# <a name="getting-information-about-the-computer-visual-basic"></a>Pobieranie informacji na temat komputera (Visual Basic)
`My.Computer.Info` Obiekt zawiera właściwości w celu uzyskania informacji na temat pamięci, załadowanych zestawów, nazwa i systemu operacyjnego komputera.  
  
## <a name="remarks"></a>Uwagi  
 Poniższa tabela zawiera listę zadań, które zazwyczaj są realizowane za pośrednictwem `My.Computer.Info` obiektów i punktów do pokazująca, jak wykonać wykonać poszczególne tematy.  
  
|Zadanie|Zobacz|  
|---|---|   
|Określić, ile wirtualnej przestrzeni adresowej jest dostępna dla komputera, na którym jest zainstalowana aplikacja|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.TotalVirtualMemory%2A>|  
|Określanie typu platformy komputera, na którym aplikacja jest uruchomiona|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSPlatform%2A>|  
|Określić systemu operacyjnego komputera, na którym aplikacja jest uruchomiona|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSFullName%2A>|  
|Określić, jakie dodatki service Pack zostały zainstalowane na komputerze, na którym aplikacja jest uruchomiona|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSVersion%2A>|  
|Określić zainstalowanych `UICulture` na komputerze, na którym działa aplikacja.|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.InstalledUICulture%2A>|
