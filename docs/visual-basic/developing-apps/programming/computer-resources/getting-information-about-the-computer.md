# <a name="getting-information-about-the-computer-visual-basic"></a>Pobieranie informacji na temat komputera (Visual Basic)
`My.Computer.Info` Obiektu udostępnia właściwości do pobierania informacji o pamięci, załadowanych zestawów, nazwy i systemu operacyjnego komputera.  
  
## <a name="remarks"></a>Uwagi  
 Ta tabela zawiera listę zadań zwykle osiągnąć za pomocą `My.Computer.Info` obiektów i punktów do tematów pokazująca, jak przeprowadzić każdego.  
  
|Do|Zobacz|  
|---|---|   
|Określić, ile wirtualnej przestrzeni adresowej jest dostępny dla komputera, na którym aplikacja jest zainstalowana|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.TotalVirtualMemory%2A>|  
|Określanie typu platformy komputera, na którym działa aplikacja|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSPlatform%2A>|  
|Określić systemu operacyjnego komputera, na którym działa aplikacja|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSFullName%2A>|  
|Określenia, jakie dodatki service Pack zostały zainstalowane na komputerze, na którym działa aplikacja|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSVersion%2A>|  
|Określić zainstalowana `UICulture` na komputerze, na którym jest uruchomiona aplikacja.|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.InstalledUICulture%2A>|
