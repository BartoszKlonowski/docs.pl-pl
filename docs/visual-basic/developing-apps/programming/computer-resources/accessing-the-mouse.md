# <a name="accessing-the-mouse-visual-basic"></a>Uzyskiwanie dostępu do myszy (Visual Basic)
`My.Computer.Mouse` Obiekt zapewnia sposób, aby znaleźć informacje o myszy komputera, np. w przypadku myszy istnieje, liczba przycisków myszy oraz szczegółowe informacje o kółka myszy.  
  
## <a name="remarks"></a>Uwagi  
 Poniższa tabela zawiera listę zadań skojarzonych z `My.Computer.Mouse` obiektów i punktów do tematów ukazujące sposób wykonania każdego.  
  
|Zadanie|Zobacz|  
|--------|---------|  
|Ustal, czy myszy ma kółka przewijania.|<xref:Microsoft.VisualBasic.Devices.Mouse.WheelExists>|  
|Określić, czy przyciski myszy lewy i prawy zostały zamienione|<xref:Microsoft.VisualBasic.Devices.Mouse.ButtonsSwapped>|  
|Ustaw ile do przewijania, gdy jest obrót kółkiem myszy obracać o jedną pozycję.|<xref:Microsoft.VisualBasic.Devices.Mouse.WheelScrollLines>|
