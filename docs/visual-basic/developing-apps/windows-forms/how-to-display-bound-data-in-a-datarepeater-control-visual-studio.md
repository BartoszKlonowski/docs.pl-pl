### <a name="to-create-a-data-bound-datarepeater"></a><span data-ttu-id="2d730-109">Aby utworzyć DataRepeater powiązane z danymi</span><span class="sxs-lookup"><span data-stu-id="2d730-109">To create a data-bound DataRepeater</span></span>  
  
1.  <span data-ttu-id="2d730-110">Przeciągnij <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> kontrolować z **Visual Basic PowerPacks** karcie **przybornika** formularza lub kontenera formantu.</span><span class="sxs-lookup"><span data-stu-id="2d730-110">Drag a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control from the **Visual Basic PowerPacks** tab in the **Toolbox** to a form or container control.</span></span>  
  
2.  <span data-ttu-id="2d730-111">Przeciągnij uchwyt zmiany rozmiaru i pozycji do rozmiaru i pozycji formantu.</span><span class="sxs-lookup"><span data-stu-id="2d730-111">Drag the sizing and position handles to size and position the control.</span></span>  
  
     <span data-ttu-id="2d730-112">Należy pamiętać, że kontrolka ma dwóch regionach prostokątny.</span><span class="sxs-lookup"><span data-stu-id="2d730-112">Note that the control has two rectangular regions.</span></span> <span data-ttu-id="2d730-113">Górny obszar jest *szablon elementu*; formanty dodawane do szablonu zostanie powtórzona w każdej pozycji w <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> formantu w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="2d730-113">The upper region is the *item template*; controls added to the template will be repeated in each item in the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control at run time.</span></span> <span data-ttu-id="2d730-114">Niższe region jest *okienka ekranu*, której elementy będą wyświetlane.</span><span class="sxs-lookup"><span data-stu-id="2d730-114">The lower region is the *viewport*, where the items will be displayed.</span></span>  
  
     <span data-ttu-id="2d730-115">Można również rozmiaru i pozycji formant lub szablon elementu zmieniając **rozmiar** i **pozycji** właściwości w oknie właściwości.</span><span class="sxs-lookup"><span data-stu-id="2d730-115">You can also size and position the control or the item template by changing the **Size** and **Position** properties in the Properties window.</span></span>  
  
3.  <span data-ttu-id="2d730-116">Na **danych** menu, kliknij przycisk **Pokaż źródeł danych**.</span><span class="sxs-lookup"><span data-stu-id="2d730-116">On the **Data** menu, click **Show Data Sources**.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="2d730-117">Jeśli **źródeł danych** okna jest pusta, Dodaj źródło danych do niej.</span><span class="sxs-lookup"><span data-stu-id="2d730-117">If the **Data Sources** window is empty, add a data source to it.</span></span> <span data-ttu-id="2d730-118">Aby uzyskać więcej informacji, zobacz [Dodawanie nowych źródeł danych](/visualstudio/data-tools/add-new-data-sources).</span><span class="sxs-lookup"><span data-stu-id="2d730-118">For more information, see [Add new data sources](/visualstudio/data-tools/add-new-data-sources).</span></span>  
  
4.  <span data-ttu-id="2d730-119">W **źródeł danych** okna, wybierz węzeł najwyższego poziomu dla tabeli, która zawiera dane, które chcesz powiązać.</span><span class="sxs-lookup"><span data-stu-id="2d730-119">In the **Data Sources** window, select the top-level node for the table that contains the data that you want to bind.</span></span>  
  
5.  <span data-ttu-id="2d730-120">Zmień typ docelowej tabeli `Details` klikając `Details` na liście rozwijanej na węźle tabeli.</span><span class="sxs-lookup"><span data-stu-id="2d730-120">Change the drop type of the table to `Details` by clicking `Details` in the drop-down list on the table node.</span></span>  
  
6.  <span data-ttu-id="2d730-121">Wybierz węzeł tabeli i przeciągnij go na region szablonu elementu <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> formantu.</span><span class="sxs-lookup"><span data-stu-id="2d730-121">Select the table node and drag it onto the item template region of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span>  
  
     <span data-ttu-id="2d730-122">Można określić, jakie typy formantów są wyświetlane dla każdego pola.</span><span class="sxs-lookup"><span data-stu-id="2d730-122">You can specify which types of controls are displayed for each field.</span></span> <span data-ttu-id="2d730-123">Aby uzyskać więcej informacji, zobacz [Ustawianie formantu do utworzenia podczas przeciągania z okna źródeł danych](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span><span class="sxs-lookup"><span data-stu-id="2d730-123">For more information, see [Set the control to be created when dragging from the Data Sources window](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span></span>
