 <span data-ttu-id="9d2ea-110">Mimo że istnieje również możliwość Rysowanie linii i kształtów za pomocą metod grafiki, istnieje kilka zalet za pomocą kontrolki linii i kształtu:</span><span class="sxs-lookup"><span data-stu-id="9d2ea-110">Although it is also possible to draw lines and shapes by using graphics methods, there are several advantages to using the Line and Shape controls:</span></span>

-   <span data-ttu-id="9d2ea-118">Formanty linii i kształtu są kontrolek bez okien; nie dojścia okna mają i w związku z tym zużywać mniej zasobów systemowych.</span><span class="sxs-lookup"><span data-stu-id="9d2ea-118">Line and Shape controls are windowless controls; they have no window handles and therefore use less system resources.</span></span>
