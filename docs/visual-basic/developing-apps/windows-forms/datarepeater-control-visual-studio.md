# <a name="datarepeater-control-visual-studio"></a>DataRepeater Formant (Visual Studio)
Powerpack Visual Basic <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> formant jest kontenerem przewijanego do wielokrotnego formantów, które zawierają dane, na przykład wierszy w tabeli bazy danych. Może służyć jako alternatywę do <xref:System.Windows.Forms.DataGridView> kontrolować, jeśli wymagane jest większa kontrola nad układ danych.  
  
 Formantów PowerPack nie są uwzględnione w programie Visual Studio, ale można pobrać je z [Centrum pobierania](http://www.microsoft.com/en-us/download/details.aspx?id=25169).
