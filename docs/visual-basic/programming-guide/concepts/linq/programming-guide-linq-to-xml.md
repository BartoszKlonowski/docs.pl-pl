# <a name="programming-guide-linq-to-xml-visual-basic"></a>Przewodnik programowania (LINQ to XML) (Visual Basic)
Ta sekcja zawiera informacje koncepcyjne i instrukcje dotyczące programowania przy użyciu [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)].  
  
## <a name="who-should-read-this-documentation"></a>Kto powinien przeczytać tę dokumentację  
 Ta dokumentacja ukierunkowana na deweloperów, którzy już znają Visual Basic i niektóre podstawowe aspekty .NET Framework.  
  
 Celem tej dokumentacji jest ułatwienie [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] do użycia dla wszystkich rodzajów deweloperów. [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] ułatwia programowanie XML. Nie musisz być deweloperem eksperckim, aby go używać.  
  
 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] opiera się na klasach ogólnych. W związku z tym jest bardzo ważne, aby zrozumieć użycie typów ogólnych i klas. Aby uzyskać więcej informacji, zobacz [typy ogólne w Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
