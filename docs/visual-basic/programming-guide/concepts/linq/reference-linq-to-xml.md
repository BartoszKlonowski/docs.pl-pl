# <a name="reference-linq-to-xml"></a>Odwołanie (LINQ do XML)
Ten temat zawiera linki do składnika LINQ to tematy dokumentacji XML.  
  
## <a name="in-this-section"></a>W tej sekcji  
 Dla dokumentacji dla klasy LINQ do XML, zobacz <xref:System.Xml.Linq>.  
  
 Dla odwołania do dokumentacji dla metody rozszerzenia, które pomagają zweryfikować drzew XML względem pliku XSD, zobacz <xref:System.Xml.Schema.Extensions?displayProperty=nameWithType>.
