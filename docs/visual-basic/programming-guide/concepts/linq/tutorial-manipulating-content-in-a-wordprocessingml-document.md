# <a name="tutorial-manipulating-content-in-a-wordprocessingml-document-visual-basic"></a>Samouczek: manipulowanie zawartością w dokumencie WordprocessingML (Visual Basic)
W tym samouczku pokazano, jak zastosować funkcjonalne podejście do transformacji i LINQ to XML do manipulowania dokumentami XML. Visual Basic przykładowe zapytania i manipulowanie informacjami w programie Office Open XML WordprocessingML Documents, które są zapisywane w programie Microsoft Word.  
  
 Aby uzyskać więcej informacji, zobacz [blog Eric](http://www.ericwhite.com).
