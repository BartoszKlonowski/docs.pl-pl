# <a name="serializing-xml-trees-visual-basic"></a>Serializacja drzewa XML (Visual Basic)

 Można kontrolować różne aspekty serializacji. Na przykład można kontrolować, czy zwiększyć wcięcie serializacji XML i czy można zapisać deklaracji XML.
