# <a name="linq-to-xml-for-xpath-users-visual-basic"></a>LINQ to XML dla użytkowników XPath (Visual Basic)

W tym zestawie tematów przedstawiono liczbę wyrażeń XPath i ich [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] odpowiedniki.  
  
 We wszystkich przykładach użyto funkcji XPath w [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)], które są udostępniane przez metody rozszerzające w <xref:System.Xml.XPath.Extensions?displayProperty=nameWithType>. Przykłady wykonują zarówno wyrażenie XPath, jak i wyrażenie [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)]. Każdy przykład porówna wyniki obu zapytań, sprawdzając, czy wyrażenie XPath jest funkcjonalnie równoważne z kwerendą [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)]. Ponieważ oba typy zapytań zwracają węzły z tego samego drzewa XML, porównanie wyników zapytania jest tworzone przy użyciu tożsamości referencyjnej.
