# <a name="pure-functional-transformations-of-xml-visual-basic"></a>Czystej funkcjonalne przekształcenia XML (Visual Basic)
Ta sekcja zawiera samouczek transformacji funkcjonalnej dla języka XML. Obejmuje to wyjaśnienie głównych pojęć i konstrukcji języka, które należy zrozumieć, aby używać transformacji funkcjonalnych, oraz przykłady użycia transformacji funkcjonalnych w celu manipulowania dokumentem XML. Mimo że ten samouczek zawiera LINQ to XML przykłady kodu, wszystkie podstawowe koncepcje dotyczą także innych technologii LINQ.  
  
 [Samouczek: manipulowanie zawartością w samouczku WordprocessingML Document (Visual Basic)](tutorial-manipulating-content-in-a-wordprocessingml-document.md) zawiera serię przykładów, z których każdy kompiluje się w poprzednim. W tych przykładach przedstawiono czyste podejście umożliwiające przekształcenie w celu manipulowania XML.  
  
 W tym samouczku założono praktyczną wiedzę na temat Visual Basic. W tym samouczku nie podano szczegółowej semantyki konstrukcji językowych, ale w razie potrzeby linki są dostępne w dokumentacji językowej.  
  
 Przyjęto również praktyczną wiedzę na temat podstawowych pojęć dotyczących analizy komputerów i XML, w tym przestrzeni nazw XML.
