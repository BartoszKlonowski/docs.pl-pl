# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML udostępnia interfejs programowania XML w pamięci, który wykorzystuje platformę .NET Language-Integrated Query (LINQ). LINQ to XML korzysta z najnowszych funkcji języka .NET Framework i jest porównywalna do zaktualizowanych, przeprojektowana modelu DOM (Document Object) interfejs programowania XML.  
  
 Rodzina LINQ technologii zapewnia spójne zapytania dla obiektów (LINQ to Objects), relacyjne bazy danych (LINQ to SQL) i XML (LINQ to XML).
