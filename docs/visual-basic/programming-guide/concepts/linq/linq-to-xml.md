# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML udostępnia interfejs programowania XML w pamięci, który wykorzystuje architekturę programu .NET Language-Integrated Query (LINQ). LINQ to XML używa najnowszych funkcji języka .NET Framework i jest porównywalna z zaktualizowanym, przeprojektowanym interfejsem programowania XML Document Object Model (DOM).  
  
 Rodzina LINQ technologii zapewnia spójne środowisko zapytań dla obiektów (LINQ to Objects), relacyjnych baz danych (LINQ to SQL) i XML (LINQ to XML).
