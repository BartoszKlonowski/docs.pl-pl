# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>Środowisko IDE i narzędzia programu Visual Studio obsługujące składnik LINQ (Visual Basic)

Zintegrowane środowisko programistyczne (IDE) programu Visual Studio zapewnia następujące funkcje, które obsługują Programowanie aplikacji LINQ:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  

 Object Relational Designer to narzędzie do projektowania wizualnego, którego można użyć w aplikacjach [LINQ to SQL](../../../../framework/data/adonet/sql/linq/index.md) do generowania klas w Visual Basic, które reprezentują dane relacyjne w źródłowej bazie danych. Aby uzyskać więcej informacji, zobacz [LINQ to SQL Tools w programie Visual Studio](/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2).  
  
## <a name="sqlmetal-command-line-tool"></a>Narzędzie wiersza polecenia SQLMetal  

 SQLMetal to narzędzie wiersza polecenia, które może być używane w procesach kompilacji do generowania klas z istniejących baz danych do użycia w aplikacjach LINQ to SQL. Aby uzyskać więcej informacji, zobacz [SqlMetal.exe (Narzędzie generowania kodu)](../../../../framework/tools/sqlmetal-exe-code-generation-tool.md).  
  
## <a name="linq-aware-code-editor"></a>Edytor kodu obsługujący LINQ  

 Edytor kodu Visual Basic obsługuje LINQ w szeroki sposób dzięki możliwościom IntelliSense i formatowaniem.  
  
## <a name="visual-studio-debugger-support"></a>Obsługa debugera programu Visual Studio  

 Debuger programu Visual Studio obsługuje debugowanie wyrażeń zapytań. Aby uzyskać więcej informacji, zobacz [debugowanie LINQ](/visualstudio/debugger/debugging-linq).
