# <a name="how-to-work-with-activex-controls-visual-basic"></a>Porady: praca ze sterownikami ActiveX (Visual Basic)
Formanty ActiveX to składniki COM lub obiekty, które można wstawić do strony sieci Web lub innej aplikacji, aby ponownie wykorzystać spakowane funkcje, które zostały zaprogramowane przez kogoś innego. Aby dodać funkcje do **przybornika** programu Visual Studio, można użyć formantów ActiveX opracowanych dla Visual Basic 6,0 i wcześniejszych wersji.  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>Aby dodać kontrolki ActiveX do przybornika  
  
1. W menu **Narzędzia** kliknij polecenie **Wybierz elementy przybornika**.  
  
     Zostanie wyświetlone okno dialogowe **Wybierz Przybornik** .  
  
2. Kliknij kartę **składniki com** .  
  
3. Zaznacz pole wyboru obok kontrolki ActiveX, której chcesz użyć, a następnie kliknij przycisk **OK**.  
  
     Nowa kontrolka pojawia się z innymi narzędziami w **przyborniku**.  
  
    > [!NOTE]
    > Można użyć narzędzia Aximp do ręcznego utworzenia zestawu międzyoperacyjnego dla formantów ActiveX. Aby uzyskać więcej informacji, zobacz [Aximp.exe (Windows Forms importer formantów ActiveX)](../../../framework/tools/aximp-exe-windows-forms-activex-control-importer.md).
