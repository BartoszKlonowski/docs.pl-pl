# <a name="loop-structures-visual-basic"></a>Struktury pętli (Visual Basic)
Struktury pętli Visual Basic zezwalają na uruchamianie kilkukrotnie jeden lub więcej wierszy kodu. Instrukcje w struktury pętli można powtórzyć, dopóki nie zostanie warunek `True`, dopóki nie zostanie warunek `False`, określona liczba razy lub raz dla każdego elementu w kolekcji.  
  
 Na poniższej ilustracji przedstawiono struktury pętli, który uruchamia zestaw instrukcji, dopóki nie zostanie spełniony warunek.  
  
 ![Schemat blokowy czy... Do pętli](../../../../visual-basic/programming-guide/language-features/control-flow/media/dountilloop.gif "DoUntilLoop")  
Uruchomiona zestaw instrukcji, dopóki nie zostanie spełniony warunek  
  
## <a name="while-loops"></a>Pętli WHILE  
 `While`... `End While` konstrukcji uruchamia zestaw instrukcji tak długo, jak warunek określony w `While` instrukcja jest `True`. Aby uzyskać więcej informacji, zobacz [podczas... End While — instrukcja](../../../../visual-basic/language-reference/statements/while-end-while-statement.md).  
  
## <a name="do-loops"></a>Do loops-pętle  
 `Do`... `Loop` konstrukcji można testować warunek na początku lub końca struktury pętli. Można również określić, czy powtórzeń pętli, gdy warunek jest `True` lub aż przyjmie postać `True`. Aby uzyskać więcej informacji, zobacz [czy... Pętla instrukcji](../../../../visual-basic/language-reference/statements/do-loop-statement.md).  
  
## <a name="for-loops"></a>Dla pętli  
 `For`... `Next` konstrukcja wykonuje pętli określoną liczbę razy. Ta zmienna sterująca pętli, nazywany również *licznika*, aby śledzić powtórzeń. Określ początkową i końcową dla tego licznika, a opcjonalnie możesz określić ilość za pomocą którego wykazuje wzrost wartości wraz z jednego powtarzania do następnego. Aby uzyskać więcej informacji, zobacz [dla... Następna instrukcja](../../../../visual-basic/language-reference/statements/for-next-statement.md).  
  
## <a name="for-each-loops"></a>Dla każdej pętli  
 `For Each`... `Next` konstrukcji uruchamia zestaw instrukcji raz dla każdego elementu w kolekcji. Określ zmienna sterująca pętli, ale nie trzeba określić rozpoczęcia lub zakończenia dla niego wartości. Aby uzyskać więcej informacji, zobacz [For Each... Następna instrukcja](../../../../visual-basic/language-reference/statements/for-each-next-statement.md).
