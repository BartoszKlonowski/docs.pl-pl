# <a name="control-flow-in-visual-basic"></a>Przepływ sterowania w Visual Basic
Pozostało nieuregulowanej, program przechodzi przez jego instrukcji od początku do końca. Niektóre programy, bardzo proste, może być napisana tylko ten przepływ jednokierunkowe. Jednak ilości energii wraz z narzędziem dowolnego języka programowania, pochodzi z możliwość zmiany kolejności wykonywania za pomocą instrukcji sterowania i pętli.  
  
 Struktury sterujące umożliwiają regulują przepływem wykonania programu. Przy użyciu struktur sterujących, można napisać kod języka Visual Basic, która sprawia, że decyzje, lub który powtarza akcje. Inne struktury sterujące pozwalają zagwarantować usuwania zasobu lub uruchomić serię instrukcji na odwołanie do tego samego obiektu.
