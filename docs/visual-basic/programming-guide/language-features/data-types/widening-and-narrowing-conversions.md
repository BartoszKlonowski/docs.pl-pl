 <span data-ttu-id="ec2c1-180">Konwersja z *odwołania do typu* kopiuje tylko wskaźnik do wartości.</span><span class="sxs-lookup"><span data-stu-id="ec2c1-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="ec2c1-181">Sama wartość nie jest kopiowany ani zmieniać w jakikolwiek sposób.</span><span class="sxs-lookup"><span data-stu-id="ec2c1-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="ec2c1-182">Jedyną czynnością, którą mogą zmieniać to typ danych zmiennej Przytrzymanie wskaźnika myszy.</span><span class="sxs-lookup"><span data-stu-id="ec2c1-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="ec2c1-183">W poniższym przykładzie typ danych jest konwertowany z klasy pochodnej do swojej klasy bazowej, ale obiekt, który obu zmiennych wskazują teraz pozostaje niezmieniony.</span><span class="sxs-lookup"><span data-stu-id="ec2c1-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>
