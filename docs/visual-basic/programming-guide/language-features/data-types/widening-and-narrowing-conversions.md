 <span data-ttu-id="306f1-180">Konwersja z *typu referencyjnego* kopiuje tylko wskaźnik do wartości.</span><span class="sxs-lookup"><span data-stu-id="306f1-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="306f1-181">Sama wartość nie jest kopiowana ani zmieniana w żaden sposób.</span><span class="sxs-lookup"><span data-stu-id="306f1-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="306f1-182">Jedyną czynnością, którą można zmienić, jest typ danych zmiennej przechowującej wskaźnik.</span><span class="sxs-lookup"><span data-stu-id="306f1-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="306f1-183">W poniższym przykładzie typ danych jest konwertowany z klasy pochodnej do jej klasy bazowej, ale obiekt, który od razu wskazuje, jest niezmieniony.</span><span class="sxs-lookup"><span data-stu-id="306f1-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>  
  
```vb  
' Assume class cSquare inherits from class cShape.  
Dim shape As cShape  
Dim square As cSquare = New cSquare  
' The following statement performs a widening  
' conversion from a derived class to its base class.  
shape = square  
```
