# <a name="structures-visual-basic"></a>Struktury (Visual Basic)
A *struktury* jest uogólnienie typu zdefiniowanego przez użytkownika (UDT) obsługiwane przez poprzednie wersje programu Visual Basic. Oprócz pól struktury można ujawnić właściwości, metody i zdarzenia. Struktura można zaimplementować jeden lub więcej interfejsów i można zadeklarować poziomów dostępu poszczególnych dla każdego pola.  
  
 Można łączyć elementy danych różnych typów, aby utworzyć strukturę. Struktura kojarzy co najmniej jeden *elementy* ze sobą oraz z samej strukturze. Kiedy Deklarujesz struktury, staje się *złożonego typu danych*, i możesz deklarować zmienne tego typu.  
  
 Struktury są przydatne, gdy chcesz, aby pojedynczej zmiennej do przechowywania kilku powiązanych informacji. Na przykład możesz chcieć Zachowaj nazwę, numer wewnętrzny i wynagrodzenia pracownika. Te informacje można użyć kilku zmiennych lub można zdefiniować strukturę i użyć jej do zmiennej pojedynczego pracownika. Zaletą struktury staje się jasne, jeśli masz wielu pracowników i w związku z tym wiele wystąpień zmiennej.
