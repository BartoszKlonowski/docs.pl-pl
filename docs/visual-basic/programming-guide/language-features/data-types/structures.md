# <a name="structures-visual-basic"></a>Struktury (Visual Basic)
A *struktury* jest generalizacji typu zdefiniowane przez użytkownika (UDT) obsługiwany przez poprzednie wersje programu Visual Basic. Oprócz pól struktury mogą uwidaczniać właściwości, metod i zdarzeń. Struktura można zaimplementować jeden lub więcej interfejsów i mogą zadeklarować poziomy poszczególnych dostępu dla każdego pola.  
  
 Można łączyć elementów danych o różnych typach tworzenia struktury. Struktura kojarzy co najmniej jeden *elementy* ze sobą oraz z samej struktury. Deklarowanie struktury, staje się *złożonego typu danych*, i można zadeklarować zmienne tego typu.  
  
 Struktury są przydatne, gdy chcesz pojedynczą zmienną do przechowywania kilku pokrewnych informacji. Można na przykład, Zachowaj nazwę pracownika, numer wewnętrzny i wynagrodzenia. Można użyć kilku zmiennych dla tych informacji, lub można zdefiniować strukturę i użyć jej do zmiennej pojedynczego pracownika. Zaletą struktury stała się oczywista, jeśli masz wiele pracowników i w związku z tym wiele wystąpień zmiennej.
