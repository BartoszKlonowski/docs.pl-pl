# <a name="how-to-hold-more-than-one-value-in-a-variable-visual-basic"></a>Porady: utrzymywanie więcej niż jednej wartości w zmiennej (Visual Basic)
Zmienna posiada więcej niż jedną wartość zadeklarowana za *złożonego typu danych*.  
  
 [Złożone typy danych](../../../../visual-basic/programming-guide/language-features/data-types/composite-data-types.md) obejmują struktur, tablic i klasy. Zmienna typu złożonego danych może zawierać kombinację podstawowe typy danych i innych typów złożonych. Struktury i klasy może zawierać kod, a także dane.  
  
### <a name="to-hold-more-than-one-value-in-a-variable"></a>Aby przechowywać więcej niż jedną wartość w zmiennej  
  
1.  Wyznaczyć, jakiego typu danych, którego chcesz użyć dla zmiennej użytkownika.  
  
2.  Jeśli typ danych nie jest już zdefiniowana, należy zdefiniować go tak, aby użyć go do zmiennej.  
  
    -   Zdefiniuj struktury z [Structure — instrukcja](../../../../visual-basic/language-reference/statements/structure-statement.md).  
  
    -   Zdefiniuj tablicy o [instrukcji Dim](../../../../visual-basic/language-reference/statements/dim-statement.md).  
  
    -   Definiowanie klasy, z [Class — instrukcja](../../../../visual-basic/language-reference/statements/class-statement.md).  
  
3.  Deklarowanie zmiennej użytkownika z `Dim` instrukcji.  
  
4.  Po nazwie zmiennej `As` klauzuli.  
  
5.  Wykonaj `As` — słowo kluczowe z nazwą typu złożonego danych.
