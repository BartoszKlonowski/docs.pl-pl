# <a name="elementary-data-types-visual-basic"></a>Podstawowe typy danych (Visual Basic)
Visual Basic dostarcza zestaw wstępnie zdefiniowanych danych typy, których można użyć dla wielu elementów programowania. W tej sekcji opisano te typy i sposobu ich używania.  
  
> [!NOTE]
>  Wszystkie typy podstawowe dane w języku Visual Basic jest obsługiwana przez struktury lub klasy, która znajduje się w <xref:System> przestrzeni nazw. Kompilator używa każde słowo kluczowe typu danych jako alias dla struktury lub klasy. Na przykład deklarowanie zmiennej przy użyciu słowo zastrzeżone `Byte` jest taka sama jak deklarowanie go przy użyciu nazwy FQDN struktury <xref:System.Byte?displayProperty=nameWithType>.
