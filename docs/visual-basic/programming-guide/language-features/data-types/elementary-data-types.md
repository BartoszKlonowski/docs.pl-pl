# <a name="elementary-data-types-visual-basic"></a>Podstawowe typy danych (Visual Basic)
Visual Basic dostarcza zestaw typów wstępnie zdefiniowanych danych, których można użyć dla wielu elementów programowania. W tej sekcji opisano te typy i sposobu ich używania.  
  
> [!NOTE]
>  Każdy typ danych podstawowych w języku Visual Basic jest obsługiwana przez strukturę lub klasę, która znajduje się w <xref:System> przestrzeni nazw. Kompilator używa każde słowo kluczowe typ danych jako alias dla struktury lub klasy. Na przykład deklarowanie zmiennej za pomocą słowa zarezerwowanego `Byte` jest taka sama jak deklarując jej za pomocą nazwy FQDN struktury <xref:System.Byte?displayProperty=nameWithType>.
