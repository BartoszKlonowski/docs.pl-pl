# <a name="common-tasks-performed-with-visual-basic-operators"></a>Zadanie typowe przeprowadzane przy użyciu operatorów Visual Basic

 W poniższej tabeli przedstawiono dostępne operacje arytmetyczne i bit-shift.  
  
|Do|Zobacz|  
|---|---|  
|Dodaj jedną wartość liczbową na inny|[+ — Operator](../../../../visual-basic/language-reference/operators/addition-operator.md)|  
|Odejmowanie jedną wartość liczbową z innego|[-— Operator (Visual Basic)](../../../../visual-basic/language-reference/operators/subtraction-operator.md)|  
|Zmienić znak wartości numerycznej|[-— Operator (Visual Basic)](../../../../visual-basic/language-reference/operators/subtraction-operator.md)|  
|Mnożenie jedną wartość liczbową przez inną|[* — Operator](../../../../visual-basic/language-reference/operators/multiplication-operator.md)|  
|Podzielić jedną wartość liczbową na inny|[/ — Operator (Visual Basic)](../../../../visual-basic/language-reference/operators/floating-point-division-operator.md)|  
|Znajdź iloraz jedną wartość liczbową podzielona przez inną (bez pozostałej)|[\ — Operator (Visual Basic)](../../../../visual-basic/language-reference/operators/integer-division-operator.md)|  
|Znajdź w pozostałej części jedną wartość liczbową podzielona przez inną (bez iloraz)|[Mod — Operator](../../../../visual-basic/language-reference/operators/mod-operator.md)|  
|Zgłoś jedną wartość liczbową do potęgi równej innej|[^ — Operator](../../../../visual-basic/language-reference/operators/exponentiation-operator.md)|  
|Przesunięcia wzorca bitowego wartości liczbowe w lewo|[<\<Operator](../../../../visual-basic/language-reference/operators/left-shift-operator.md)|  
|Przesunięcia wzorca bitowego wartości liczbowe w prawo|[>> — Operator](../../../../visual-basic/language-reference/operators/right-shift-operator.md)|
