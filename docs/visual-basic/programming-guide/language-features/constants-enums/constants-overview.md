 <span data-ttu-id="490a0-111">Zakres stałej, który jest zestawem wszystkich kodów, które mogą odwoływać się do niego bez kwalifikowania się jego nazwy, jest taka sama jak zmienna zadeklarowana w tej samej lokalizacji.</span><span class="sxs-lookup"><span data-stu-id="490a0-111">A constant's scope, which is the set of all code that can refer to it without qualifying its name, is the same as that of a variable declared in the same location.</span></span> <span data-ttu-id="490a0-112">Aby utworzyć stałą, która istnieje w zakresie określonej procedury, zadeklaruj ją wewnątrz tej procedury.</span><span class="sxs-lookup"><span data-stu-id="490a0-112">To create a constant that exists within the scope of a particular procedure, declare it inside that procedure.</span></span> <span data-ttu-id="490a0-113">Aby utworzyć stałą dostępną w całej aplikacji, należy ją zadeklarować przy użyciu `Public` słowa kluczowego w sekcji deklaracji klasy.</span><span class="sxs-lookup"><span data-stu-id="490a0-113">To create a constant that is available throughout an application, declare it using the `Public` keyword in the declarations section of the class.</span></span>

 <span data-ttu-id="490a0-117">Stała czasu kompilacji jest obliczana w momencie kompilowania kodu, podczas gdy stała czasu wykonywania może być obliczana tylko wtedy, gdy aplikacja jest uruchomiona.</span><span class="sxs-lookup"><span data-stu-id="490a0-117">A compile-time constant is computed at the time the code is compiled, while a run-time constant can only be computed while the application is running.</span></span> <span data-ttu-id="490a0-118">Stała czasu kompilacji będzie miała taką samą wartość za każdym razem, gdy aplikacja zostanie uruchomiona, a stała czasu wykonywania może się zmieniać za każdym razem.</span><span class="sxs-lookup"><span data-stu-id="490a0-118">A compile-time constant will have the same value each time an application runs, while a run-time constant may change each time.</span></span> <span data-ttu-id="490a0-119">Stałe czasu kompilacji są wymagane dla przypadków takich jak granice tablicy, wyrażenia wielkości liter lub inicjatory modułu wyliczającego.</span><span class="sxs-lookup"><span data-stu-id="490a0-119">Compile-time constants are required for cases such as array bounds, case expressions, or enumerator initializers.</span></span>
