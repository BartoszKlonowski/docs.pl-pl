     <span data-ttu-id="f3bd7-124">Dla typów odwołań tylko wskaźnik do danych jest skopiowany (cztery bajtów na 32-bitowych platform ośmiu bajtów na platformach 64-bitowe).</span><span class="sxs-lookup"><span data-stu-id="f3bd7-124">For reference types, only the pointer to the data is copied (four bytes on 32-bit platforms, eight bytes on 64-bit platforms).</span></span> <span data-ttu-id="f3bd7-125">W związku z tym można przekazywać argumenty typu `String` lub `Object` przez wartość bez szkody wydajności.</span><span class="sxs-lookup"><span data-stu-id="f3bd7-125">Therefore, you can pass arguments of type `String` or `Object` by value without harming performance.</span></span>

-   <span data-ttu-id="f3bd7-138">Jeśli procedura ma oryginalny trzeba zmienić odpowiedniego elementu kodu wywołującego, należy zadeklarować odpowiadającego mu parametru [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span><span class="sxs-lookup"><span data-stu-id="f3bd7-138">If the procedure has a genuine need to change the underlying element in the calling code, declare the corresponding parameter [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span></span>
