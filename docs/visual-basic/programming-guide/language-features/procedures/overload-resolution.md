# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="d59c6-102">Rozpoznanie przeciążenia (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="d59c6-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="d59c6-103">Gdy kompilator Visual Basic napotyka wywołanie procedury, która jest zdefiniowana w kilku przeciążonych wersjach, kompilator musi zdecydować, które z przeciążeń mają być wywoływane.</span><span class="sxs-lookup"><span data-stu-id="d59c6-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="d59c6-104">Robi to, wykonując następujące czynności:</span><span class="sxs-lookup"><span data-stu-id="d59c6-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="d59c6-105">**Ułatwienia dostępu.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-105">**Accessibility.**</span></span> <span data-ttu-id="d59c6-106">Eliminuje wszelkie przeciążenia z poziomem dostępu, który uniemożliwia Wywoływanie kodu wywołującego.</span><span class="sxs-lookup"><span data-stu-id="d59c6-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="d59c6-107">**Liczba parametrów.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-107">**Number of Parameters.**</span></span> <span data-ttu-id="d59c6-108">Eliminuje wszelkie przeciążenia, które definiują inną liczbę parametrów niż podano w wywołaniu.</span><span class="sxs-lookup"><span data-stu-id="d59c6-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="d59c6-109">**Typ danych parametru.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-109">**Parameter Data Types.**</span></span> <span data-ttu-id="d59c6-110">Kompilator zapewnia preferencje metod wystąpień względem metod rozszerzających.</span><span class="sxs-lookup"><span data-stu-id="d59c6-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="d59c6-111">Jeśli zostanie znaleziona jakakolwiek metoda wystąpienia, która wymaga tylko konwersji rozszerzających, aby pasowała do wywołania procedury, wszystkie metody rozszerzające są porzucane, a kompilator kontynuuje tylko z użyciem metody wystąpienia.</span><span class="sxs-lookup"><span data-stu-id="d59c6-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="d59c6-112">Jeśli nie zostanie znaleziona taka metoda wystąpienia, kontynuuje z obu tych metod.</span><span class="sxs-lookup"><span data-stu-id="d59c6-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="d59c6-113">W tym kroku eliminuje wszelkie przeciążenia, dla których typy danych argumentów wywołujących nie mogą być konwertowane na typy parametrów zdefiniowane w przeciążenia.</span><span class="sxs-lookup"><span data-stu-id="d59c6-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="d59c6-114">**Zawężanie konwersji.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="d59c6-115">Eliminuje wszelkie przeciążenia, które wymagają konwersji zawężania z typów argumentów wywołujących na zdefiniowane typy parametrów.</span><span class="sxs-lookup"><span data-stu-id="d59c6-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="d59c6-116">Jest to prawdą, czy przełącznik sprawdzania typu ([instrukcja Option Strict](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) jest `On` lub `Off`.</span><span class="sxs-lookup"><span data-stu-id="d59c6-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="d59c6-117">**Najmniej rozszerzanie.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-117">**Least Widening.**</span></span> <span data-ttu-id="d59c6-118">Kompilator traktuje pozostałe przeciążenia w parach.</span><span class="sxs-lookup"><span data-stu-id="d59c6-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="d59c6-119">Dla każdej pary porównuje typy danych zdefiniowanych parametrów.</span><span class="sxs-lookup"><span data-stu-id="d59c6-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="d59c6-120">Jeśli typy w jednym z przeciążeń wszystkie rozszerzają się do odpowiednich typów w drugim, kompilator eliminuje ten ostatni.</span><span class="sxs-lookup"><span data-stu-id="d59c6-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="d59c6-121">Oznacza to, że zachowuje Przeciążenie, które wymaga najmniejszej ilości rozszerzania.</span><span class="sxs-lookup"><span data-stu-id="d59c6-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="d59c6-122">**Pojedynczy kandydat.**</span><span class="sxs-lookup"><span data-stu-id="d59c6-122">**Single Candidate.**</span></span> <span data-ttu-id="d59c6-123">Kontynuuje rozważanie przeciążenia par do momentu pozostawania tylko jednego przeciążenia i rozwiązuje wywołanie do tego przeciążenia.</span><span class="sxs-lookup"><span data-stu-id="d59c6-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="d59c6-124">Jeśli kompilator nie może zmniejszyć przeciążenia do pojedynczego kandydata, generuje błąd.</span><span class="sxs-lookup"><span data-stu-id="d59c6-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
