<span data-ttu-id="fc9de-103">Gdy kompilator języka Visual Basic napotka wywołanie procedury, która jest zdefiniowana w kilku przeciążonych wersjach, kompilator musi zdecydować, które z przeciążeń do wywołania.</span><span class="sxs-lookup"><span data-stu-id="fc9de-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="fc9de-104">Robi to wykonując następujące kroki:</span><span class="sxs-lookup"><span data-stu-id="fc9de-104">It does this by performing the following steps:</span></span>

5. <span data-ttu-id="fc9de-117">**Najmniej poszerzenie.**</span><span class="sxs-lookup"><span data-stu-id="fc9de-117">**Least Widening.**</span></span> <span data-ttu-id="fc9de-118">Kompilator uwzględnia pozostałe przeciążenia w parach.</span><span class="sxs-lookup"><span data-stu-id="fc9de-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="fc9de-119">Dla każdej pary porównuje typy danych zdefiniowanych parametrów.</span><span class="sxs-lookup"><span data-stu-id="fc9de-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="fc9de-120">Jeśli typy w jednym z przeciążeń wszystkie poszerzyć do odpowiednich typów w innych, kompilator eliminuje ten ostatni.</span><span class="sxs-lookup"><span data-stu-id="fc9de-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="fc9de-121">Oznacza to, że zachowuje przeciążenie, które wymaga najmniejszej ilości poszerzenia.</span><span class="sxs-lookup"><span data-stu-id="fc9de-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="fc9de-122">**Jeden kandydat.**</span><span class="sxs-lookup"><span data-stu-id="fc9de-122">**Single Candidate.**</span></span> <span data-ttu-id="fc9de-123">Kontynuuje biorąc pod uwagę przeciążenia w parach, dopóki nie pozostaje tylko jedno przeciążenie i rozwiązuje wywołanie tego przeciążenia.</span><span class="sxs-lookup"><span data-stu-id="fc9de-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="fc9de-124">Jeśli kompilator nie może zmniejszyć przeciążenia do jednego kandydata, generuje błąd.</span><span class="sxs-lookup"><span data-stu-id="fc9de-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
