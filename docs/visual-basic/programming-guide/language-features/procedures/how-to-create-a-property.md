### <a name="to-create-a-get-procedure-that-retrieves-a-property-value"></a><span data-ttu-id="a243d-116">Aby utworzyć procedurę Get, która pobiera wartość właściwości</span><span class="sxs-lookup"><span data-stu-id="a243d-116">To create a Get procedure that retrieves a property value</span></span>  
  
1. <span data-ttu-id="a243d-117">Między instrukcjami `Property` i `End Property` Napisz [instrukcję Get](../../../../visual-basic/language-reference/statements/get-statement.md), a następnie instrukcję `End Get`.</span><span class="sxs-lookup"><span data-stu-id="a243d-117">Between the `Property` and `End Property` statements, write a [Get Statement](../../../../visual-basic/language-reference/statements/get-statement.md), followed by an `End Get` statement.</span></span> <span data-ttu-id="a243d-118">Nie trzeba definiować żadnych parametrów dla procedury `Get`.</span><span class="sxs-lookup"><span data-stu-id="a243d-118">You do not need to define any parameters for the `Get` procedure.</span></span>  
  
2. <span data-ttu-id="a243d-119">Umieść instrukcje Code, aby pobrać wartość właściwości między instrukcjami `Get` i `End Get`.</span><span class="sxs-lookup"><span data-stu-id="a243d-119">Place the code statements to retrieve the property's value between the `Get` and `End Get` statements.</span></span> <span data-ttu-id="a243d-120">Ten kod może obejmować inne obliczenia i manipulowanie danymi, a także generowanie i zwracanie wartości właściwości.</span><span class="sxs-lookup"><span data-stu-id="a243d-120">This code can include other calculations and data manipulations in addition to generating and returning the property's value.</span></span>

 <span data-ttu-id="a243d-122">Należy napisać `Get` procedury dla właściwości Read-Write i dla właściwości tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="a243d-122">You must write a `Get` procedure for a read-write property and for a read-only property.</span></span> <span data-ttu-id="a243d-123">Nie należy definiować `Get` procedury dla właściwości tylko do zapisu.</span><span class="sxs-lookup"><span data-stu-id="a243d-123">You must not define a `Get` procedure for a write-only property.</span></span>  
  
### <a name="to-create-a-set-procedure-that-writes-a-propertys-value"></a><span data-ttu-id="a243d-124">Aby utworzyć procedurę zestawu, która zapisuje wartość właściwości</span><span class="sxs-lookup"><span data-stu-id="a243d-124">To create a Set procedure that writes a property's value</span></span>  
  
1. <span data-ttu-id="a243d-125">Między instrukcjami `Property` i `End Property` Napisz [instrukcję SET](../../../../visual-basic/language-reference/statements/set-statement.md), po której następuje instrukcja `End Set`.</span><span class="sxs-lookup"><span data-stu-id="a243d-125">Between the `Property` and `End Property` statements, write a [Set Statement](../../../../visual-basic/language-reference/statements/set-statement.md), followed by an `End Set` statement.</span></span>  
  
2. <span data-ttu-id="a243d-126">W instrukcji `Set` postępuj według słowa kluczowego `Set` z listą parametrów w nawiasach.</span><span class="sxs-lookup"><span data-stu-id="a243d-126">In the `Set` statement, follow the `Set` keyword with a parameter list in parentheses.</span></span> <span data-ttu-id="a243d-127">Ta lista parametrów musi zawierać co najmniej parametr wartości dla wartości przesłanej przez wywoływany kod.</span><span class="sxs-lookup"><span data-stu-id="a243d-127">This parameter list must include at least a value parameter for the value passed by the calling code.</span></span> <span data-ttu-id="a243d-128">Domyślna nazwa dla tego parametru wartości to `Value`, ale w razie potrzeby można użyć innej nazwy.</span><span class="sxs-lookup"><span data-stu-id="a243d-128">The default name for this value parameter is `Value`, but you can use a different name if appropriate.</span></span> <span data-ttu-id="a243d-129">Parametr value musi mieć ten sam typ danych co sama właściwość.</span><span class="sxs-lookup"><span data-stu-id="a243d-129">The value parameter must have the same data type as the property itself.</span></span>  
  
3. <span data-ttu-id="a243d-130">Umieść instrukcje Code w celu zapisania wartości we właściwości między instrukcjami `Set` i `End Set`.</span><span class="sxs-lookup"><span data-stu-id="a243d-130">Place the code statements to store a value in the property between the `Set` and `End Set` statements.</span></span> <span data-ttu-id="a243d-131">Ten kod może obejmować inne obliczenia i manipulowanie danymi, a także sprawdzanie poprawności i przechowywanie wartości właściwości.</span><span class="sxs-lookup"><span data-stu-id="a243d-131">This code can include other calculations and data manipulations in addition to validating and storing the property's value.</span></span>  
  
4. <span data-ttu-id="a243d-132">Użyj parametru value, aby zaakceptować wartość podaną przez wywołujący kod.</span><span class="sxs-lookup"><span data-stu-id="a243d-132">Use the value parameter to accept the value supplied by the calling code.</span></span> <span data-ttu-id="a243d-133">Tę wartość można przechowywać bezpośrednio w instrukcji przypisania lub użyć jej w wyrażeniu w celu obliczenia wartości wewnętrznej do zapisania.</span><span class="sxs-lookup"><span data-stu-id="a243d-133">You can either store this value directly in an assignment statement, or use it in an expression to calculate the internal value to be stored.</span></span>  
  
 <span data-ttu-id="a243d-134">Należy napisać `Set` procedury dla właściwości Read-Write i dla właściwości tylko do zapisu.</span><span class="sxs-lookup"><span data-stu-id="a243d-134">You must write a `Set` procedure for a read-write property and for a write-only property.</span></span> <span data-ttu-id="a243d-135">Nie należy definiować `Set` procedury dla właściwości tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="a243d-135">You must not define a `Set` procedure for a read-only property.</span></span>
