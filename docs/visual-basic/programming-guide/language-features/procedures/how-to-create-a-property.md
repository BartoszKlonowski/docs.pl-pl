### <a name="to-create-a-property"></a><span data-ttu-id="3ed56-109">Aby utworzyć właściwość</span><span class="sxs-lookup"><span data-stu-id="3ed56-109">To create a property</span></span>  
  
1.  <span data-ttu-id="3ed56-110">Poza właściwość lub procedurę, należy użyć [Property — instrukcja](../../../../visual-basic/language-reference/statements/property-statement.md), a następnie `End Property` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="3ed56-110">Outside any property or procedure, use a [Property Statement](../../../../visual-basic/language-reference/statements/property-statement.md), followed by an `End Property` statement.</span></span>  
  
2.  <span data-ttu-id="3ed56-111">Jeśli właściwość przyjmuje parametry, postępuj zgodnie z `Property` — słowo kluczowe o nazwie procedury, a następnie lista parametrów w nawiasach.</span><span class="sxs-lookup"><span data-stu-id="3ed56-111">If the property takes parameters, follow the `Property` keyword with the name of the procedure, then the parameter list in parentheses.</span></span>

### <a name="to-create-a-get-procedure-that-retrieves-a-property-value"></a><span data-ttu-id="3ed56-116">Aby utworzyć procedurę Get, który pobiera wartość właściwości</span><span class="sxs-lookup"><span data-stu-id="3ed56-116">To create a Get procedure that retrieves a property value</span></span>  
  
1.  <span data-ttu-id="3ed56-117">Między `Property` i `End Property` pisanie instrukcji [instrukcja Get](../../../../visual-basic/language-reference/statements/get-statement.md), a następnie `End Get` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="3ed56-117">Between the `Property` and `End Property` statements, write a [Get Statement](../../../../visual-basic/language-reference/statements/get-statement.md), followed by an `End Get` statement.</span></span> <span data-ttu-id="3ed56-118">Nie musisz zdefiniować inne parametry `Get` procedury.</span><span class="sxs-lookup"><span data-stu-id="3ed56-118">You do not need to define any parameters for the `Get` procedure.</span></span>  
  
2.  <span data-ttu-id="3ed56-119">Umieść instrukcje kodu, który można pobrać wartości właściwości między `Get` i `End Get` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="3ed56-119">Place the code statements to retrieve the property's value between the `Get` and `End Get` statements.</span></span> <span data-ttu-id="3ed56-120">Ten kod może zawierać innych obliczeń i manipulacji danych oprócz generowania i zwracanie wartości właściwości.</span><span class="sxs-lookup"><span data-stu-id="3ed56-120">This code can include other calculations and data manipulations in addition to generating and returning the property's value.</span></span>

 <span data-ttu-id="3ed56-122">Należy napisać `Get` procedury dla właściwości odczytu / zapisu, a dla właściwości tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="3ed56-122">You must write a `Get` procedure for a read-write property and for a read-only property.</span></span> <span data-ttu-id="3ed56-123">Nie można zdefiniować `Get` procedury dla właściwości tylko do zapisu.</span><span class="sxs-lookup"><span data-stu-id="3ed56-123">You must not define a `Get` procedure for a write-only property.</span></span>  
  
### <a name="to-create-a-set-procedure-that-writes-a-propertys-value"></a><span data-ttu-id="3ed56-124">Aby utworzyć procedurę zestaw, który zapisuje wartości właściwości</span><span class="sxs-lookup"><span data-stu-id="3ed56-124">To create a Set procedure that writes a property's value</span></span>  
  
1.  <span data-ttu-id="3ed56-125">Między `Property` i `End Property` pisanie instrukcji [instrukcji Set](../../../../visual-basic/language-reference/statements/set-statement.md), a następnie `End Set` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="3ed56-125">Between the `Property` and `End Property` statements, write a [Set Statement](../../../../visual-basic/language-reference/statements/set-statement.md), followed by an `End Set` statement.</span></span>

3.  <span data-ttu-id="3ed56-130">Umieść instrukcje kodu, do przechowywania wartości właściwości między `Set` i `End Set` instrukcji.</span><span class="sxs-lookup"><span data-stu-id="3ed56-130">Place the code statements to store a value in the property between the `Set` and `End Set` statements.</span></span> <span data-ttu-id="3ed56-131">Ten kod może zawierać innych obliczeń i manipulacji danych oprócz sprawdzania poprawności i przechowywania wartości właściwości.</span><span class="sxs-lookup"><span data-stu-id="3ed56-131">This code can include other calculations and data manipulations in addition to validating and storing the property's value.</span></span>  
  
4.  <span data-ttu-id="3ed56-132">Aby zaakceptować wartość dostarczona przez kod wywołujący, należy użyć parametru wartości.</span><span class="sxs-lookup"><span data-stu-id="3ed56-132">Use the value parameter to accept the value supplied by the calling code.</span></span> <span data-ttu-id="3ed56-133">Możesz zapisać tę wartość bezpośrednio w instrukcji przypisania lub używany w atakach wyrażenie do obliczenia wartości wewnętrznej, które mają być przechowywane.</span><span class="sxs-lookup"><span data-stu-id="3ed56-133">You can either store this value directly in an assignment statement, or use it in an expression to calculate the internal value to be stored.</span></span>  
  
 <span data-ttu-id="3ed56-134">Należy napisać `Set` procedury dla właściwości odczytu / zapisu, a dla właściwości tylko do zapisu.</span><span class="sxs-lookup"><span data-stu-id="3ed56-134">You must write a `Set` procedure for a read-write property and for a write-only property.</span></span> <span data-ttu-id="3ed56-135">Nie można zdefiniować `Set` procedury dla właściwości tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="3ed56-135">You must not define a `Set` procedure for a read-only property.</span></span>
