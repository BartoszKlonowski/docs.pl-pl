## <a name="passing-by-value"></a><span data-ttu-id="921a4-106">Przekazywanie poprzez wartość</span><span class="sxs-lookup"><span data-stu-id="921a4-106">Passing by Value</span></span>  
 <span data-ttu-id="921a4-107">Przekazując argument *przez wartość* , określając [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) — słowo kluczowe dla odpowiadającego mu parametru w definicji procedury.</span><span class="sxs-lookup"><span data-stu-id="921a4-107">You pass an argument *by value* by specifying the [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="921a4-108">Gdy używasz ten mechanizm przekazywania Visual Basic kopiuje wartość odpowiedniego elementu programowania do zmiennej lokalnej w procedurze.</span><span class="sxs-lookup"><span data-stu-id="921a4-108">When you use this passing mechanism, Visual Basic copies the value of the underlying programming element into a local variable in the procedure.</span></span> <span data-ttu-id="921a4-109">Kod procedury nie ma dostęp do odpowiedniego elementu w wywoływanym kodzie.</span><span class="sxs-lookup"><span data-stu-id="921a4-109">The procedure code does not have any access to the underlying element in the calling code.</span></span>  
  
## <a name="passing-by-reference"></a><span data-ttu-id="921a4-110">Przekazywanie poprzez odwołanie</span><span class="sxs-lookup"><span data-stu-id="921a4-110">Passing by Reference</span></span>  
 <span data-ttu-id="921a4-111">Przekazując argument *przez odwołanie* , określając [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) — słowo kluczowe dla odpowiadającego mu parametru w definicji procedury.</span><span class="sxs-lookup"><span data-stu-id="921a4-111">You pass an argument *by reference* by specifying the [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="921a4-112">Gdy używasz ten mechanizm przekazywania Visual Basic zwraca procedury bezpośrednie odwołanie do odpowiedniego elementu programowania kodu wywołującego.</span><span class="sxs-lookup"><span data-stu-id="921a4-112">When you use this passing mechanism, Visual Basic gives the procedure a direct reference to the underlying programming element in the calling code.</span></span>  
  
## <a name="passing-mechanism-and-element-type"></a><span data-ttu-id="921a4-113">Mechanizm przekazywania i typ elementu</span><span class="sxs-lookup"><span data-stu-id="921a4-113">Passing Mechanism and Element Type</span></span>  
 <span data-ttu-id="921a4-114">Przekazywanie mechanizm wyboru nie jest taka sama jak klasyfikacji element podstawowy typu.</span><span class="sxs-lookup"><span data-stu-id="921a4-114">The choice of passing mechanism is not the same as the classification of the underlying element type.</span></span> <span data-ttu-id="921a4-115">Przekazywanie według wartości lub według odwołania odwołuje się do języka Visual Basic dostarcza kod procedury.</span><span class="sxs-lookup"><span data-stu-id="921a4-115">Passing by value or by reference refers to what Visual Basic supplies to the procedure code.</span></span> <span data-ttu-id="921a4-116">Typ wartości lub typ referencyjny odwołuje się do przechowywania elementu programistycznego w pamięci.</span><span class="sxs-lookup"><span data-stu-id="921a4-116">A value type or reference type refers to how a programming element is stored in memory.</span></span>  
  
 <span data-ttu-id="921a4-117">Jednak powiązane mechanizm przekazywania i typ elementu.</span><span class="sxs-lookup"><span data-stu-id="921a4-117">However, the passing mechanism and element type are interrelated.</span></span> <span data-ttu-id="921a4-118">Wartość typu referencyjnego jest wskaźnikiem do danych w pamięci.</span><span class="sxs-lookup"><span data-stu-id="921a4-118">The value of a reference type is a pointer to the data elsewhere in memory.</span></span> <span data-ttu-id="921a4-119">To oznacza, że jeśli przez wartość typu odwołania, kod procedury ma wskaźnik do odpowiedniego elementu danych, mimo że nie ma dostępu do elementu podstawowego.</span><span class="sxs-lookup"><span data-stu-id="921a4-119">This means that when you pass a reference type by value, the procedure code has a pointer to the underlying element's data, even though it cannot access the underlying element itself.</span></span> <span data-ttu-id="921a4-120">Na przykład jeśli element jest zmienną tablicową, kod procedury nie ma dostępu do samej zmiennej, ale mogą uzyskiwać dostęp do tablicy elementów członkowskich.</span><span class="sxs-lookup"><span data-stu-id="921a4-120">For example, if the element is an array variable, the procedure code does not have access to the variable itself, but it can access the array members.</span></span>
