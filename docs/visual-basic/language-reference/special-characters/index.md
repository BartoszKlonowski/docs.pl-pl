# <a name="special-characters-visual-basic-reference"></a>Znaki specjalne (odwołanie w Visual Basic)

Znaki specjalne są wstępnie zdefiniowane, kontekstowych znaki, które modyfikuje elementu programu, do której jest dołączony. Visual Basic obsługuje następujące znaki specjalne: 

- [$](interpolated.md), znak w ciągu interpolowanym.
