# <a name="option-keyword-statement"></a>Option \<keyword>, instrukcja
Wprowadza instrukcję określającą opcję kompilatora, która jest stosowana do całego pliku źródłowego.  
  
## <a name="remarks"></a>Uwagi  
 Opcje kompilatora mogą określać, czy wszystkie zmienne muszą być jawnie zadeklarowane, czy konwersje typu zawężanego muszą być jawne lub czy ciągi mają być porównywane jako tekst, czy jako liczby binarne.  
  
 `Option`Słowo kluczowe może być używane w tych kontekstach:  
  
 [Option Compare — Instrukcja](option-compare-statement.md)  
  
 [Option Explicit, instrukcja](option-explicit-statement.md)  
  
 [Option Infer — Instrukcja](option-infer-statement.md)  
  
 [Option Strict — Instrukcja](option-strict-statement.md)
