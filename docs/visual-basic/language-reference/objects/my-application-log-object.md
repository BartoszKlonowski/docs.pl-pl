# <a name="myapplicationlog-object"></a>My.Application.Log — Obiekt
Udostępnia właściwości i metody mogła zapisać informacji zdarzeń i wyjątków do odbiorniki dzienników aplikacji.  
  
## <a name="remarks"></a>Uwagi  
 Informacje dotyczące metod i właściwości `My.Application.Log` obiektów, zobacz <xref:Microsoft.VisualBasic.Logging.Log>.  
  
 Aby uzyskać więcej informacji, zobacz [rejestrowanie informacji z aplikacji](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md).  
  
> [!NOTE]
>  Można również używać klas w programie .NET Framework do rejestrowania informacji z aplikacji. Aby uzyskać więcej informacji, zobacz [śledzenie i Instrumentacja aplikacji](../../../framework/debug-trace-profile/tracing-and-instrumenting-applications.md).
