# <a name="mylog-object"></a>My.Log — Obiekt

Zawiera właściwości i metody zapisywania informacji o zdarzeniach i wyjątkach w detektorach dzienników aplikacji.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Log` obiektu, zobacz <xref:Microsoft.VisualBasic.Logging.AspLog> .  
  
 `My.Log`Obiekt jest dostępny tylko dla aplikacji ASP.NET. W przypadku aplikacji klienckich Użyj [obiektu My. Application. log](my-application-log-object.md).
