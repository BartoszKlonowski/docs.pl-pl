# <a name="mycomputerfilesystem-object"></a>My.Computer.FileSystem — Obiekt

Zawiera właściwości i metody pracy z dyskami, plikami i katalogami.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Computer.FileSystem` obiektu, zobacz <xref:Microsoft.VisualBasic.FileIO.FileSystem> .  
  
 Aby uzyskać więcej informacji, zobacz [dostęp do plików przy użyciu Visual Basic](../../developing-apps/programming/drives-directories-files/file-access.md).  
  
> [!NOTE]
> Można również używać klas w <xref:System.IO?displayProperty=nameWithType> przestrzeni nazw do pracy z dyskami, plikami i katalogami.
