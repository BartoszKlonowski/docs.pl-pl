# <a name="mycomputermouse-object"></a>My.Computer.Mouse — Obiekt
Udostępnia właściwości do pobierania informacji o formacie i konfiguracji myszy zainstalowane na komputerze lokalnym.  
  
## <a name="remarks"></a>Uwagi  
 Informacje dotyczące metod i właściwości `My.Computer.Mouse` obiektów, zobacz <xref:Microsoft.VisualBasic.Devices.Mouse>.  
  
 Aby uzyskać więcej informacji, zobacz [podczas uzyskiwania dostępu do myszy](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md).  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Klasy:** <xref:Microsoft.VisualBasic.Devices.Mouse>  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
