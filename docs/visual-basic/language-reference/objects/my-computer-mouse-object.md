# <a name="mycomputermouse-object"></a>My.Computer.Mouse — Obiekt

Zawiera właściwości służące do uzyskiwania informacji o formacie i konfiguracji myszy zainstalowanej na komputerze lokalnym.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Computer.Mouse` obiektu, zobacz <xref:Microsoft.VisualBasic.Devices.Mouse> .  
  
 Aby uzyskać więcej informacji, zobacz [Uzyskiwanie dostępu do myszy](../../developing-apps/programming/computer-resources/accessing-the-mouse.md).
