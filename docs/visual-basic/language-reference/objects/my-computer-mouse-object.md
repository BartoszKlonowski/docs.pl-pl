# <a name="mycomputermouse-object"></a>My.Computer.Mouse — Obiekt
Udostępnia właściwości w celu uzyskania informacji na temat formatu i konfiguracji myszy zainstalowane na komputerze lokalnym.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje dotyczące metod i właściwości `My.Computer.Mouse` obiektu, zobacz <xref:Microsoft.VisualBasic.Devices.Mouse>.  
  
 Aby uzyskać więcej informacji, zobacz [uzyskiwania dostępu do myszy](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md).  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.Devices.Mouse>  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
