# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard — Obiekt
Udostępnia metody do manipulowania Schowka.  
  
## <a name="remarks"></a>Uwagi  
 Informacje dotyczące metod i właściwości `My.Computer.Clipboard` obiektów, zobacz <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.  
  
 Aby uzyskać więcej informacji, zobacz [przechowywania danych i odczytywanie ich ze Schowka](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
>  Można również użyć metody <xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType> klasy do manipulowania Schowka.
