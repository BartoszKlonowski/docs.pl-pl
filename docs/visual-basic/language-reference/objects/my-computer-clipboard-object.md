# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard — Obiekt
Zapewnia metody manipulowania schowkiem.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje o metodach i właściwościach `My.Computer.Clipboard` obiektu, zobacz <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy> .  
  
 Aby uzyskać więcej informacji, zobacz [Zapisywanie danych do i odczytywanie ze schowka](../../developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
> Można również użyć metod <xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType> klasy do manipulowania schowkiem.
