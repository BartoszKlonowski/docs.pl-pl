# <a name="mycomputerkeyboard-object"></a>My.Computer.Keyboard — Obiekt

Zapewnia właściwości do uzyskiwania dostępu do bieżącego stanu klawiatury, takie jak klucze są aktualnie naciśnięte i zapewnia metodę wysyłania naciśnięć klawiszy do aktywnego okna.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Computer.Keyword` obiektu, zobacz <xref:Microsoft.VisualBasic.Devices.Keyboard> .  
  
 Aby uzyskać więcej informacji, zobacz [Uzyskiwanie dostępu do klawiatury](../../developing-apps/programming/computer-resources/accessing-the-keyboard.md).
