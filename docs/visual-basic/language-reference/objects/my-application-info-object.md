# <a name="myapplicationinfo-object"></a>My.Application.Info — Obiekt

Zawiera właściwości służące do uzyskiwania informacji o aplikacji, takich jak numer wersji, opis, ładowane zestawy itd.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Application.Info` obiektu, zobacz <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo> .  
  
> [!NOTE]
> Właściwości klasy można użyć, <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> Aby uzyskać informacje o pliku na dysku.
