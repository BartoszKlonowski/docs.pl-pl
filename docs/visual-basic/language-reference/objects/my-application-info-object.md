# <a name="myapplicationinfo-object"></a>My.Application.Info — Obiekt
Udostępnia właściwości w celu uzyskania informacji o aplikacji, takich jak numer wersji, opis, załadowanych zestawów i tak dalej.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje dotyczące metod i właściwości obiektu `My.Application.Info`, zobacz <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Można użyć właściwości <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> klasy w celu uzyskania informacji na temat pliku na dysku.  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
