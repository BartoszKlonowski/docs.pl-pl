# <a name="myapplicationinfo-object"></a>My.Application.Info — Obiekt
Udostępnia właściwości do pobierania informacji o aplikacji, takich jak numer wersji, opis, załadowanych zestawów i tak dalej.  
  
## <a name="remarks"></a>Uwagi  
 Informacje dotyczące metod i właściwości `My.Application.Info` obiektów, zobacz <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Można użyć właściwości <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> klasy, aby uzyskać informacje o pliku na dysku.  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Klasy:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
