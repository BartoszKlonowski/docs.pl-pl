# <a name="mycomputerfilesystemspecialdirectories-object"></a>My.Computer.FileSystem.SpecialDirectories — Obiekt
Udostępnia właściwości umożliwiające dostęp do często katalogi, do których odwołuje się.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje dotyczące metod i właściwości obiektu `My.Computer.FileSystem.SpecialDirectories`, zobacz <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Aby uzyskać więcej informacji, zobacz [jak: Pobieranie zawartości katalogu Moje dokumenty](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> (zapewnia dostęp do <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>)  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
