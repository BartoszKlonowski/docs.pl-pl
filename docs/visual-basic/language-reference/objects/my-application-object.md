# <a name="myapplication-object"></a>My.Application — Obiekt
Udostępnia właściwości, metody i zdarzenia związane z bieżącej aplikacji.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje dotyczące metod i właściwości `My.Application` obiektów, zobacz następujące zasoby:  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase> w przypadku elementów członkowskich, które są dostępne we wszystkich projektach.  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> w przypadku elementów członkowskich, które są dostępne w aplikacjach Windows Forms.  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase> dla elementów członkowskich, które są dostępne w aplikacji konsoli.  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> (klasę bazową <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase> oferuje elementy członkowskie dostępne w aplikacji konsoli i jej klasy bazowej <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase> oferuje elementy członkowskie, które są dostępne we wszystkich projektach)  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
