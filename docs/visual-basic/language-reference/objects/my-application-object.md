# <a name="myapplication-object"></a>My.Application — Obiekt
Udostępnia właściwości, metody i zdarzenia związane z bieżącej aplikacji.  
  
## <a name="remarks"></a>Uwagi  
 Informacje dotyczące metod i właściwości `My.Application` obiektów, zobacz następujące zasoby:  
  
-   <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase> dla elementów członkowskich, które są dostępne we wszystkich projektach.  
  
-   <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> dla elementów członkowskich, które są dostępne w aplikacjach formularzy systemu Windows.  
  
-   <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase> dla elementów członkowskich, które są dostępne w aplikacji konsoli.  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> (klasa podstawowa <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase> zapewnia to członkom dostępne w aplikacji konsoli, a jej klasa podstawowa <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase> zawiera elementów członkowskich, które są dostępne we wszystkich projektach)  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
