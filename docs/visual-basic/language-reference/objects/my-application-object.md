# <a name="myapplication-object"></a>My.Application — Obiekt
Zawiera właściwości, metody i zdarzenia związane z bieżącą aplikacją.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje o metodach i właściwościach `My.Application` obiektu, zobacz następujące zasoby:  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase>dla elementów członkowskich, które są dostępne we wszystkich projektach.  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase>w przypadku elementów członkowskich dostępnych w aplikacjach Windows Forms.  
  
- <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase>elementy członkowskie, które są dostępne w aplikacjach konsolowych.  
  
## <a name="requirements"></a>Wymagania  
 **Przestrzeń nazw:**<xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> (klasa bazowa <xref:Microsoft.VisualBasic.ApplicationServices.ConsoleApplicationBase> zapewnia składowe dostępne w aplikacjach konsolowych, a jej Klasa bazowa <xref:Microsoft.VisualBasic.ApplicationServices.ApplicationBase> zapewnia składowe, które są dostępne we wszystkich projektach)  
  
 **Zestaw:** Biblioteka środowiska uruchomieniowego Visual Basic (w pliku Microsoft. VisualBasic. dll)
