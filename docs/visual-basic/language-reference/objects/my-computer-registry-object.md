# <a name="mycomputerregistry-object"></a>My.Computer.Registry — Obiekt
Udostępnia właściwości i metody do manipulowania w rejestrze.  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać informacje dotyczące metod i właściwości obiektu `My.Computer.Registry`, zobacz <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
 Aby uzyskać więcej informacji, zobacz [odczytywanie z oraz zapisywanie do rejestru](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md).  
  
> [!NOTE]
>  Można też manipulować rejestru przy użyciu metody <xref:Microsoft.Win32.Registry?displayProperty=nameWithType> klasy.  
  
## <a name="requirements"></a>Wymagania  
 **Namespace:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Klasa:** <xref:Microsoft.VisualBasic.MyServices.RegistryProxy> (zapewnia dostęp do <xref:Microsoft.Win32.Registry>)  
  
 **Zestaw:** Visual Basic Runtime Library (w pliku Microsoft.VisualBasic.dll)
