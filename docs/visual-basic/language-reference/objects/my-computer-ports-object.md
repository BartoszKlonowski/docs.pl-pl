# <a name="mycomputerports-object"></a>My.Computer.Ports — Obiekt

Zawiera właściwość i metodę uzyskiwania dostępu do portów szeregowych komputera.  
  
## <a name="remarks"></a>Uwagi  

 Aby uzyskać informacje o metodach i właściwościach `My.Computer.Ports` obiektu, zobacz <xref:Microsoft.VisualBasic.Devices.Ports> .  
  
 Aby uzyskać więcej informacji, zobacz [Uzyskiwanie dostępu do portów komputera](../../developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
> Można również użyć właściwości i metod <xref:System.IO.Ports.SerialPort?displayProperty=nameWithType> klasy, aby uzyskać dostęp do portów szeregowych komputera.
