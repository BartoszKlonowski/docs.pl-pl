# <a name="unable-to-link-to-resource-file-filename-error-message"></a>Nie można połączyć z plikiem zasobów „\<filename>": \<error message>

Kompilator Visual Basic wywołuje konsolidator zestawu (Al.exe, znany również jako ALink) w celu wygenerowania zestawu z manifestem. Konsolidator zgłosił błąd podczas tworzenia linku do natywnego pliku zasobów modelu COM+ z zestawu.  
  
 **Identyfikator błędu:** BC30144  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź cytowany komunikat o błędzie i zapoznaj się z tematem [Al.exe](../../../framework/tools/al-exe-assembly-linker.md). Dalsze wyjaśnienie i porady.  
  
2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
