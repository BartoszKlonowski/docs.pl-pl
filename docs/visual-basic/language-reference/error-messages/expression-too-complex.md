# <a name="expression-too-complex"></a>Wyrażenie jest zbyt złożone
Zmiennoprzecinkowe wyrażenie zawiera zbyt wiele zagnieżdżonych zakresie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podziel wyrażenie na tyle oddzielne wyrażenia w razie potrzeby, aby zapobiec wystąpieniu błędu.
