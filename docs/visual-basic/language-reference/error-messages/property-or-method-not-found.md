# <a name="property-or-method-not-found"></a>Nie odnaleziono właściwości lub metody
Metoda obiektu lub właściwość obiektu, do którego się odwoływano, nie jest zdefiniowana.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Może być błędnie wpisana nazwa obiektu. Aby zobaczyć, jakie właściwości i metody są zdefiniowane dla obiektu, Wyświetl Przeglądarka obiektów. Wybierz odpowiednią bibliotekę obiektów, aby wyświetlić listę dostępnych właściwości i metod.
