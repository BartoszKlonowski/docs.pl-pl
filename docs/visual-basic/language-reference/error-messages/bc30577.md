# <a name="addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="b7fb0-102">Operand „AddressOf" musi być nazwą metody (bez nawiasów)</span><span class="sxs-lookup"><span data-stu-id="b7fb0-102">'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="b7fb0-103">Operator `AddressOf` tworzy wystąpienie delegata procedury, które odwołuje się do określonej procedury.</span><span class="sxs-lookup"><span data-stu-id="b7fb0-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="b7fb0-104">Składnia jest następująca:</span><span class="sxs-lookup"><span data-stu-id="b7fb0-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="b7fb0-105">Wstawiono nawiasy wokół argumentu następującego `AddressOf`, gdzie nie są potrzebne.</span><span class="sxs-lookup"><span data-stu-id="b7fb0-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="b7fb0-106">**Identyfikator błędu:** BC30577</span><span class="sxs-lookup"><span data-stu-id="b7fb0-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="b7fb0-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="b7fb0-107">Example</span></span>

<span data-ttu-id="b7fb0-108">Poniższy przykład generuje BC30577:</span><span class="sxs-lookup"><span data-stu-id="b7fb0-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="b7fb0-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="b7fb0-109">To correct this error</span></span>

1. <span data-ttu-id="b7fb0-110">Usuń nawiasy wokół poniższego argumentu `AddressOf` jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="b7fb0-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb
    Public Sub CountZeroToTen()
        For i = 0 To 10
            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="b7fb0-111">Upewnij się, że argument jest nazwą metody.</span><span class="sxs-lookup"><span data-stu-id="b7fb0-111">Make sure the argument is a method name.</span></span>
