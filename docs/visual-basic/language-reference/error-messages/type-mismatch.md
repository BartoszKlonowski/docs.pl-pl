# <a name="type-mismatch-visual-basic"></a>Niezgodność typów (Visual Basic)
Próbujesz przekonwertować wartości na inny typ w sposób, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź przypisania, aby upewnić się, że jest on prawidłowy.  
  
2.  Upewnij się, że obiekt nie przeszedł do procedury, która oczekuje jedną właściwość lub wartość.  
  
3.  Upewnij się, że nie używasz nazwę modułu lub projekt gdy Oczekiwano wyrażenia.
