# <a name="type-mismatch-visual-basic"></a>Niezgodność typów (Visual Basic)
Próbujesz konwertować wartość do innego typu w taki sposób, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź przypisanie do upewnij się, że jest ona prawidłowa.  
  
2.  Upewnij się, że obiekt nie przeszedł do procedury, która oczekuje jednej właściwości lub wartości.  
  
3.  Upewnij się, że nie używasz nazwę modułu lub projektu gdzie oczekiwany wyrażenie.
