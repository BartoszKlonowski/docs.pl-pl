# <a name="internal-error-happened-at-location"></a>Wystąpił błąd wewnętrzny w lokalizacji \<
Wystąpił błąd wewnętrzny. Wiersz, w którym nastąpiło, jest zawarty w komunikacie o błędzie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że ten błąd nie został wygenerowany przez instrukcję `Error` lub metodę `Raise`; Jeśli nie, skontaktuj się z pomocą techniczną firmy Microsoft, aby zgłosić warunki, w których pojawił się komunikat.
