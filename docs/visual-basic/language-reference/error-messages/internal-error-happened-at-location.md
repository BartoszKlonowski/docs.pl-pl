# <a name="internal-error-happened-at-location"></a>Wystąpił wewnętrzny błąd w \<location>

Wystąpił błąd wewnętrzny. Wiersz, w którym nastąpiło, jest zawarty w komunikacie o błędzie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że ten błąd nie został wygenerowany przez `Error` instrukcję lub `Raise` metodę. Jeśli nie, skontaktuj się z pomocą techniczną firmy Microsoft w celu zgłoszenia warunków, w których pojawił się komunikat.
