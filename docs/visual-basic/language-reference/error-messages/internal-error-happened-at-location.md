# <a name="internal-error-happened-at-ltlocationgt"></a>Wystąpił błąd wewnętrzny w &lt;lokalizacji&gt;
Wystąpił błąd wewnętrzny. Wiersz, w którym wystąpił znajduje się w komunikacie o błędzie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że ten błąd nie został wygenerowany przez `Error` instrukcji lub `Raise` metody; Jeśli nie, skontaktuj się z pomocą techniczną firmy Microsoft, aby zgłosić warunków, w których pojawił się komunikat.
