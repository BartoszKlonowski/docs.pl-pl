# <a name="subscript-out-of-range-visual-basic"></a>Indeks dolny spoza zakresu (Visual Basic)

Indeks dolny tablicy jest nieprawidłowy, ponieważ znajduje się poza dozwolonym zakresem. Najniższa wartość indeksu dolnego dla wymiaru jest zawsze równa 0, a najwyższa wartość dolnego indeksu jest zwracana przez `GetUpperBound` metodę dla tego wymiaru.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień indeks dolny tak, aby mieścił się w prawidłowym zakresie.
