# <a name="subscript-out-of-range-visual-basic"></a>Indeks dolny spoza zakresu (Visual Basic)
Indeks dolny tablicy jest nieprawidłowa, ponieważ jej znajduje się poza dozwolonym zakresem. Najniższa wartość indeksu dolnego wymiaru to zawsze 0 i najwyższą wartość indeksu dolnego, jest zwracany przez `GetUpperBound` metody dla tego wymiaru.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień indeks dolny, aby go mieści się w prawidłowym zakresie.
