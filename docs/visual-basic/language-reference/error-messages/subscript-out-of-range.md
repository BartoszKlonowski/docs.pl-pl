# <a name="subscript-out-of-range-visual-basic"></a>Indeks dolny spoza zakresu (Visual Basic)
Indeks dolny tablicy jest nieprawidłowa, ponieważ znajduje się poza dozwolonym zakresem. Najniższa wartość indeksu dolnego wymiaru jest zawsze zwraca 0, a największa wartość indeksu dolnego `GetUpperBound` metody dla tego wymiaru.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Indeks dolny należy zmienić, tak aby mieści się w prawidłowym zakresie.
