# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>&#39;&lt;methodname&gt; &#39; ma wiele definicji o identycznych podpisach
A `Function` lub `Sub` deklaracja procedury używa procedury identyczne nazwą i listą argumentów jako poprzedniej deklaracji. Jedną z możliwych przyczyn jest próba przeciążenia oryginalne procedury. Przeciążona procedury musi mieć listy różnych argumentów.  
  
 **Identyfikator błędu:** BC30269  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę procedury lub listy argumentów, lub usuń Zduplikowana deklaracja.
