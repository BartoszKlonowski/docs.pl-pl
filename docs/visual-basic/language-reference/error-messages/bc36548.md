# <a name="bc36548-cannot-convert-anonymous-type-to-an-expression-tree-because-a-property-of-the-type-is-used-to-initialize-another-property"></a><span data-ttu-id="cee19-102">BC36548: nie można przekonwertować typu anonimowego na drzewo wyrażenia, ponieważ właściwość typu jest używana do inicjowania innej właściwości</span><span class="sxs-lookup"><span data-stu-id="cee19-102">BC36548: Cannot convert anonymous type to an expression tree because a property of the type is used to initialize another property</span></span>

<span data-ttu-id="cee19-103">Kompilator nie akceptuje konwersji typu anonimowego na drzewo wyrażenia, gdy jedna właściwość typu anonimowego jest używana do inicjowania innej właściwości typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="cee19-103">The compiler does not accept conversion of an anonymous type to an expression tree when one property of the anonymous type is used to initialize another property of the anonymous type.</span></span> <span data-ttu-id="cee19-104">Na przykład, w poniższym kodzie, `Prop1` jest zadeklarowany na liście inicjalizacji, a następnie używany jako wartość początkowa dla `Prop2` .</span><span class="sxs-lookup"><span data-stu-id="cee19-104">For example, in the following code, `Prop1` is declared in the initialization list and then used as the initial value for `Prop2`.</span></span>

<span data-ttu-id="cee19-105">**Identyfikator błędu:** BC36548</span><span class="sxs-lookup"><span data-stu-id="cee19-105">**Error ID:** BC36548</span></span>

## <a name="example"></a><span data-ttu-id="cee19-106">Przykład</span><span class="sxs-lookup"><span data-stu-id="cee19-106">Example</span></span>

```vb
Module M2

    Sub ExpressionExample(Of T)(x As Expressions.Expression(Of Func(Of T)))
    End Sub

    Sub Main()
        ' The following line causes the error.
        ExpressionExample(Function() New With {.Prop1 = 2, .Prop2 = .Prop1})
    End Sub
End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="cee19-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="cee19-107">To correct this error</span></span>

<span data-ttu-id="cee19-108">Przypisz początkową wartość `Prop1` do zmiennej lokalnej.</span><span class="sxs-lookup"><span data-stu-id="cee19-108">Assign the initial value for `Prop1` to a local variable.</span></span> <span data-ttu-id="cee19-109">Przypisz tę zmienną do obu `Prop1` i `Prop2` , jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="cee19-109">Assign that variable to both `Prop1` and `Prop2`, as shown in the following code:</span></span>

```vb
Sub Main()
    Dim temp = 2
    ExpressionExample(Function() New With {.Prop1 = temp, .Prop2 = temp})
End Sub
```
