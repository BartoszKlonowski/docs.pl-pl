# <a name="cant-open-filename-for-writing"></a>Nie można otworzyć "\<nazwa pliku >" do zapisu
Nie można otworzyć określonego pliku do pisania, prawdopodobnie ponieważ został on już otwarty.  
  
 **Identyfikator błędu:** BC2012  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zamknij plik i otwórz go ponownie.  
  
2. Sprawdź uprawnienia do pliku.
