# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Można&#39;t Otwórz &#39; &lt;filename&gt; &#39; do zapisu
Nie można otworzyć określonego pliku do zapisu, prawdopodobnie ponieważ został już otwarty.  
  
 **Identyfikator błędu:** BC2012  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zamknij plik i otwórz go ponownie.  
  
2.  Sprawdź uprawnienia do pliku.
