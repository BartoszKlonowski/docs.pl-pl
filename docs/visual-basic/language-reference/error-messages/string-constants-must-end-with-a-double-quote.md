# <a name="string-constants-must-end-with-a-double-quote"></a>Stałe typu string muszą być zakończone znakiem podwójnego cudzysłowu
Stałe typu String muszą rozpoczynać i kończyć znaki cudzysłowu.  
  
 **Identyfikator błędu:** BC30648  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że literału ciągu kończy się znakiem cudzysłowu ("). Jeśli możesz wkleić wartości z innych edytory tekstów, upewnij się, wklejonych znak jest nieprawidłowy znak cudzysłowu, a nie jeden znaki, które przypominają, takie jak "inteligentne" lub "nawiasów" znaki cudzysłowu ("lub") lub dwa znaki pojedynczego cudzysłowu (").
