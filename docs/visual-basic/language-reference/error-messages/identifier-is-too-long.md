# <a name="identifier-is-too-long"></a>Identyfikator jest za długi
Nazwa lub identyfikator co element programowania jest ograniczona do 1023 znaków. Ponadto w pełni kwalifikowana nazwa nie może przekraczać 1023 znaków. Oznacza to, że ciągu całego identyfikatora (`<namespace>.<...>.<namespace>.<class>.<element>`) nie może być dłuższa niż 1023 znaków, łącznie z rolą operatora dostępu do elementu członkowskiego (`.`) znaków.  
  
 **Identyfikator błędu:** BC30033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ogranicz długość identyfikatora.
