# <a name="xml-axis-properties-do-not-support-late-binding"></a>Właściwości osi XML nie obsługują późnego powiązania
Właściwości osi XML został wskazany bez typu obiektu.  
  
 **Identyfikator błędu:** BC31168  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że obiekt jest silne typizowanych <xref:System.Xml.Linq.XElement> obiektu przed odwołaniem się do właściwości osi XML.
