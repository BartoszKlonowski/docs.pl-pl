# <a name="out-of-memory-visual-basic-compiler-error"></a>Za mało pamięci (Błąd kompilatora Visual Basic)
Jest wymagana większa ilość pamięci niż jest dostępna.  
  
 **Identyfikator błędu:** BC2004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij niepotrzebne aplikacje, dokumenty i pliki źródłowe.  
  
- Eliminowanie niepotrzebnych kontrolek i formularzy w taki sposób, aby w tym samym czasie załadowano mniej  
  
- Zmniejsz liczbę `Public` zmiennych.  
  
- Sprawdź ilość dostępnego miejsca na dysku.  
  
- Zwiększ ilość dostępnej pamięci RAM, instalując dodatkową pamięć lub ponowną alokację pamięci.  
  
- Upewnij się, że pamięć jest zwalniana, gdy nie jest już potrzebne.
