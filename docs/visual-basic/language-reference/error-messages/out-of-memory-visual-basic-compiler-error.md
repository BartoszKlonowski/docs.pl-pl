# <a name="out-of-memory-visual-basic-compiler-error"></a>Za mało pamięci (Błąd kompilatora Visual Basic)
Większa ilość pamięci nie jest wymagana, niż jest dostępne.  
  
 **Identyfikator błędu:** BC2004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij zbędne aplikacje, dokumenty i pliki źródłowe.  
  
- Wyeliminuj niepotrzebne formanty, a więc mniej formularzy są ładowane w tym samym czasie  
  
- Zmniejsz liczbę `Public` zmiennych.  
  
- Sprawdź ilość dostępnego miejsca.  
  
- Zwiększ dostępnej pamięci RAM, instalowania dodatkowej pamięci lub zmienianie alokacji pamięci.  
  
- Upewnij się, że pamięć jest zwalniana, gdy nie jest już potrzebny.
