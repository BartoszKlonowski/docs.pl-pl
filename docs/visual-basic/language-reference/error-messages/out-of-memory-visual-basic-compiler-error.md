# <a name="out-of-memory-visual-basic-compiler-error"></a>Za mało pamięci (Błąd kompilatora Visual Basic)
Większa ilość pamięci jest wymagana, niż jest dostępne.  
  
 **Identyfikator błędu:** BC2004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamknij zbędnych aplikacji, dokumentów i plików źródłowych.  
  
-   Aby wyeliminować niepotrzebnych kontrole i tak mniej formularze są ładowane w tym samym czasie  
  
-   Zmniejsz liczbę `Public` zmiennych.  
  
-   Sprawdź ilość dostępnego miejsca.  
  
-   Zwiększenia dostępnej pamięci RAM, instalowania dodatkowej pamięci lub zmienianie alokacji pamięci.  
  
-   Upewnij się, że pamięci jest zwolniony, kiedy nie jest już potrzebne.
