# <a name="class-does-not-support-automation-or-does-not-support-expected-interface"></a>Klasa nie obsługuje automatyzacji lub oczekiwanego interfejsu

Klasa określona w `GetObject` `CreateObject` wywołaniu lub funkcji nie ma interfejsu programowania lub zmieniono projekt z. dll na. exe lub odwrotnie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją aplikacji, która utworzyła obiekt, aby uzyskać ograniczenia dotyczące używania automatyzacji z tą klasą obiektów.  
  
2. Jeśli projekt został zmieniony z. dll na. exe lub odwrotnie, należy ręcznie wyrejestrować stary plik. dll lub. exe.
