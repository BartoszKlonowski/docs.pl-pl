# <a name="class-does-not-support-automation-or-does-not-support-expected-interface"></a>Klasa nie obsługuje automatyzacji lub oczekiwanego interfejsu
Klasa określona w `GetObject` lub `CreateObject` wywołanie funkcji nie jest widoczne interfejs programowania lub zmienić projekt z dll do .exe, lub na odwrót.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją aplikacji, który utworzył obiekt ograniczeń dotyczących używania usługi automation za pomocą tej klasy obiektów.  
  
2. Projekt zmieniła się z .dll, .exe lub odwrotnie, należy ręcznie Wyrejestruj stare .dll lub .exe.
