# <a name="bc30306-array-subscript-expression-missing"></a>BC30306: Brak wyrażenia indeksu tablicy

Inicjalizacja tablicy pozostawia jeden lub więcej indeksów dolnych, które definiują granice tablicy. Na przykład instrukcja może zawierać wyrażenie `myArray (5,5,,10)` , które opuszcza trzeci indeks dolny.

 **Identyfikator błędu:** BC30306

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Podaj brakujący indeks dolny.
