# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Podczas operacji automatyzacji nie odnaleziono nazwy pliku lub nazwy klasy (Visual Basic)
Nazwa określona dla nazwy pliku lub klasy w wywołaniu `GetObject` nie można odnaleźć funkcji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nazwy i spróbuj ponownie. Upewnij się, że nazwa używana w przypadku `class` parametru jest zgodny z typem zarejestrowany w systemie.
