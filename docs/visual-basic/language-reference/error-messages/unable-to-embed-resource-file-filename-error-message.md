# <a name="bc30143-unable-to-embed-resource-file-filename-error-message"></a>BC30143: nie można osadzić pliku zasobów " \<filename> ": \<error message>

Kompilator Visual Basic wywołuje konsolidator zestawu (Al.exe, znany również jako ALink) w celu wygenerowania zestawu z manifestem. Konsolidator zgłosił błąd osadzania natywnego pliku zasobów modelu com bezpośrednio w zestawie.

 **Identyfikator błędu:** BC30143

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Sprawdź cytowany komunikat o błędzie i zapoznaj się z tematem [Al.exe](../../../framework/tools/al-exe-assembly-linker.md). Dalsze wyjaśnienie i porady.

2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
