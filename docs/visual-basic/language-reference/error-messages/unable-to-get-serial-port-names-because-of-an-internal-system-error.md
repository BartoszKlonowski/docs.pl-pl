# <a name="unable-to-get-serial-port-names-because-of-an-internal-system-error"></a>Nie można uzyskać nazw portów seryjnych w związku z wystąpieniem błędu
Wystąpił błąd wewnętrzny podczas wywoływania właściwości `My.Computer.Ports.SerialPortNames`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Aby uzyskać więcej informacji dotyczących rozwiązywania problemów, zobacz [podstawy debugera](/visualstudio/debugger/debugger-basics) .  
  
2. Zanotuj sytuacje, w których wystąpił błąd i Wywołaj usługi pomocy technicznej firmy Microsoft.
