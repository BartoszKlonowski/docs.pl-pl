# <a name="path-not-found"></a>Nie można odnaleźć ścieżki
Podczas operacji uzyskiwania dostępu do plików lub dostępu do dysku systemu operacyjnego nie mógł znaleźć określonej ścieżki. Ścieżka do pliku zawiera specyfikację dysku oraz katalogów i podkatalogów, które muszą przejść, aby zlokalizować plik. Ścieżka może być względna lub bezwzględna.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zweryfikuj i respecify ścieżki.
