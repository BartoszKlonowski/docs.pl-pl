# <a name="path-not-found"></a>Nie można odnaleźć ścieżki
Podczas operacji dostępu do plików lub dostępu do dysku systemu operacyjnego nie może znaleźć określonej ścieżki. Ścieżka do pliku zawiera specyfikację dysku oraz katalogów i podkatalogów, które muszą przejść, aby zlokalizować plik. Ścieżka może być względna lub bezwzględna.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź i respecify ścieżki.
