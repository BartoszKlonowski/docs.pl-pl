# <a name="path-not-found"></a>Nie można odnaleźć ścieżki
Podczas operacji dostępu do pliku lub dysku system operacyjny nie mógł znaleźć określonej ścieżki. Ścieżka do pliku zawiera specyfikację dysku oraz katalogi i podkatalogi, które muszą zostać przesunięte, aby zlokalizować plik. Ścieżka może być względna lub bezwzględna.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy ścieżka jest określona.
