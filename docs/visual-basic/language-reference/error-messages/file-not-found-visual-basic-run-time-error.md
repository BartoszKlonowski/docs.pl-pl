# <a name="file-not-found-visual-basic-run-time-error"></a>Nie odnaleziono pliku (błąd czasu wykonywania Visual Basic)
Nie odnaleziono pliku, w przypadku, gdy określony. Błąd ma następujące możliwe przyczyny:  
  
-   Oświadczenie odnosi się do pliku, który nie istnieje.  
  
-   Nastąpiła próba wywołania procedury biblioteki dołączanej dynamicznie (DLL), ale biblioteki określone w `Lib` klauzuli `Declare` instrukcji nie można odnaleźć.  
  
-   Podjęto próbę otwarcia projektu lub załadować plik tekstowy, który nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię nazwy pliku i specyfikacji ścieżki.
