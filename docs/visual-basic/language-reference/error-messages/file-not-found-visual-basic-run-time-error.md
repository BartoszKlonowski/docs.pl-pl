# <a name="file-not-found-visual-basic-run-time-error"></a>Nie odnaleziono pliku (błąd czasu wykonywania Visual Basic)

Nie znaleziono pliku, gdzie został określony. Błąd ma następujące możliwe przyczyny:  
  
- Instrukcja odwołuje się do pliku, który nie istnieje.  
  
- Podjęto próbę wywołania procedury w bibliotece dołączanej dynamicznie (DLL), ale nie można odnaleźć biblioteki określonej w `Lib` klauzuli `Declare` instrukcji.  
  
- Podjęto próbę otwarcia projektu lub załadowania pliku tekstowego, który nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię nazwy pliku i specyfikacji ścieżki.
