# <a name="file-not-found-visual-basic-run-time-error"></a>Nie odnaleziono pliku (błąd czasu wykonywania Visual Basic)
Nie można odnaleźć pliku, gdzie określona. Błąd ma następujące możliwe przyczyny:  
  
-   Oświadczenie odnosi się do pliku, który nie istnieje.  
  
-   Podjęto próbę wywołania procedury biblioteki dołączanej dynamicznie (DLL), ale biblioteki określone w `Lib` klauzuli `Declare` instrukcji nie można odnaleźć.  
  
-   Próbujesz otworzyć projekt lub załadować plik tekstowy, który nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię nazwy pliku i specyfikacja ścieżki.
