# <a name="device-io-error"></a>Błąd We/Wy urządzenia
Wystąpił błąd wejściowych lub wyjściowych, gdy program został przy użyciu urządzenia, takie jak drukarka lub dysk.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że urządzenie działa prawidłowo, a następnie spróbuj ponownie wykonać operację.
