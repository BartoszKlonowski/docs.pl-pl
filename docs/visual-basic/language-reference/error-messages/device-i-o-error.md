# <a name="device-io-error"></a>Błąd We/Wy urządzenia
Wystąpił błąd wejścia lub wyjścia, gdy program używa urządzenia, takiego jak drukarka lub stacja dysków.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że urządzenie działa prawidłowo, a następnie spróbuj ponownie wykonać operację.
