# <a name="property-array-index-is-not-valid"></a>Indeks tablicy właściwości nie jest prawidłowy
Podana wartość nie jest prawidłowa dla indeks tablicy właściwości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź dokumentację składnika, aby upewnić się, że indeksu do prawidłowego zakresu dla określonej właściwości.
