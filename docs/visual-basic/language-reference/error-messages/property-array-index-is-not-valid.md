# <a name="property-array-index-is-not-valid"></a>Indeks tablicy właściwości nie jest prawidłowy

Podana wartość jest nieprawidłowa dla indeksu tablicy właściwości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź dokumentację składnika, aby upewnić się, że indeks znajduje się w prawidłowym zakresie dla określonej właściwości.
