# <a name="unable-to-find-required-file-filename"></a>Nie można odnaleźć wymaganego pliku "\<nazwa pliku >"
Plik, którego wymaga program Visual Studio, jest uszkodzony lub nie istnieje.  
  
 **Identyfikator błędu:** BC30655  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zainstaluj ponownie program Visual Studio.
