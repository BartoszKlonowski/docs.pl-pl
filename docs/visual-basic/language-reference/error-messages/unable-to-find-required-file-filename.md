# <a name="bc30655-unable-to-find-required-file-filename"></a>BC30655: nie można odnaleźć wymaganego pliku " \<filename> "

Plik, którego wymaga program Visual Studio, jest uszkodzony lub nie istnieje.

 **Identyfikator błędu:** BC30655

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Zainstaluj ponownie program Visual Studio.
