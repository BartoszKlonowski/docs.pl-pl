# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Nie można odnaleźć wymaganego pliku &#39; &lt;filename&gt;&#39;
Plik, którego wymaga program Visual Studio, jest uszkodzony lub nie istnieje.  
  
 **Identyfikator błędu:** BC30655  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zainstaluj ponownie program Visual Studio.
