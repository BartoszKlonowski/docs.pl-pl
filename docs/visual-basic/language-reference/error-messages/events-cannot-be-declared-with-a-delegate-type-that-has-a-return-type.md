# <a name="events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>Zdarzenia nie mogą być zadeklarowane z typem delegowanym, który ma typ zwracany
Delegat określono procedurę function.  
  
 **Identyfikator błędu:** BC31084  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ delegatów dla `Sub` procedury.
