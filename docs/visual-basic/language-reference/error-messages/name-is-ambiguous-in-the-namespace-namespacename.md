# <a name="39ltnamegt39-is-ambiguous-in-the-namespace-39ltnamespacenamegt39"></a>&#39; &lt;nazwa&gt;&#39; jest niejednoznaczny w przestrzeni nazw &#39;&lt; Nazwa przestrzeni nazw&gt;&#39;
Podana nazwa jest niejednoznaczny i dlatego powoduje konflikt z inną nazwą. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilatora nie ma żadnych reguł rozwiązywania konfliktów; nazwy musi odróżniania samodzielnie.  
  
 **Identyfikator błędu:** BC30560  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
