# <a name="object-required-visual-basic"></a>Wymagany obiekt (Visual Basic)
Odwołania do właściwości i metody często wymaga kwalifikatora jawne obiektu. Jest to wówczas.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy odwołania do właściwości obiektu lub metoda ma prawidłowy obiekt kwalifikatora. Określ kwalifikator obiektu, jeśli nie podasz.  
  
2.  Sprawdź pisownię kwalifikatora obiektu i upewnij się, że obiekt jest widoczny w części programu, w którym możesz odwołuje się do niego.  
  
3.  Jeśli ścieżki są dostarczane do aplikacji hosta **Otwórz plik** polecenia, sprawdź, czy argumenty są poprawne.  
  
4.  Sprawdź dokumentację obiektu i upewnij się, że akcja jest poprawna.
