# <a name="property-not-found"></a>Nie odnaleziono właściwości
Ten obiekt nie obsługuje określonej właściwości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię nazwy właściwości.  
  
2.  Sprawdź dokumentację obiektu, aby upewnić się, że nie chcesz dostęp coś tak jak właściwość "text", gdy obiekt faktycznie obsługuje "podpis" lub o podobnej nazwie właściwości.
