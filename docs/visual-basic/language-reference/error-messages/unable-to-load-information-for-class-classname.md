# <a name="unable-to-load-information-for-class-39ltclassnamegt39"></a>Nie można załadować informacji dla klasy &#39; &lt;classname&gt;&#39;
Odwołano się do klasy, która nie jest dostępna.  
  
 **Identyfikator błędu:** BC30712  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy klasa jest zdefiniowana i że nazwa został poprawnie wpisany.  
  
2.  Wypróbuj, uzyskiwanie dostępu do jednego z elementów członkowskich zadeklarowanych w module. W niektórych przypadkach środowisko debugowania nie można zlokalizować elementów członkowskich, ponieważ moduły, w której są deklarowane nie został jeszcze załadowany.
