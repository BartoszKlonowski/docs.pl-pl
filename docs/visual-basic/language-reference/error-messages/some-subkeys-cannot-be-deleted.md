# <a name="some-subkeys-cannot-be-deleted"></a>Nie można usunąć niektórych podkluczy

Podjęto próbę usunięcia klucza rejestru, ale operacja nie powiodła się, ponieważ nie można usunąć niektórych podkluczy. Zwykle jest to spowodowane brakiem uprawnień.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że masz wystarczające uprawnienia do usunięcia określonych podkluczy.
