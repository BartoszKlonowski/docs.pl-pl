# <a name="variable-uses-an-automation-type-not-supported-in-visual-basic"></a>Zmienna używa typu automatyzacji, który nie jest obsługiwany przez język Visual Basic
Próbowano użyć zmiennej zdefiniowanej w bibliotece typów lub biblioteki, która ma typ danych, nie jest obsługiwany przez program Visual Basic.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj zmiennej typu rozpoznawane przez program Visual Basic.  
  
     —lub—  
  
-   Jeśli ten błąd wystąpi podczas korzystania z `FileGet` lub `FileGetOBject`, upewnij się, że próbujesz użyć został zapisany plik za pomocą `FilePut` lub `FilePutObject`.
