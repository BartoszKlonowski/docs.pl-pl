# <a name="variable-uses-an-automation-type-not-supported-in-visual-basic"></a>Zmienna używa typu automatyzacji, który nie jest obsługiwany przez język Visual Basic
Próbowano użyć zmiennej zdefiniowanej w biblioteki typów lub biblioteki, który ma typ danych nie jest obsługiwana przez [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj zmiennej typu rozpoznaje [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
     —lub—  
  
-   Jeśli ten błąd wystąpi podczas korzystania z `FileGet` lub `FileGetOBject`, upewnij się, że próbujesz użyć został zapisany plik z `FilePut` lub `FilePutObject`.
