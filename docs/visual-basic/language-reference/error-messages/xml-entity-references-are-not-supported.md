# <a name="xml-entity-references-are-not-supported"></a>Odwołania do jednostek XML nie są obsługiwane
Odwołania do jednostki (na przykład `©`) nie jest zdefiniowany w XML 1.0 specyfikacji jest dołączony jako wartość literał XML. Tylko `&`, `"`, `<`, `>`, i `'` odwołania do jednostek XML są obsługiwane w literałach XML.  
  
 **Identyfikator błędu:** BC31180  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń odwołania do nieobsługiwanych jednostki.
