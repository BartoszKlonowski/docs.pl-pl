# <a name="bc30561-name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>BC30561: element " \<name1> " jest niejednoznaczny, importowany z przestrzeni nazw lub typów " \<name2> "

Podana nazwa jest niejednoznaczna i dlatego powoduje konflikt z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; należy samodzielnie odróżnić nazwy.

 **Identyfikator błędu:** BC30561

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Usuń niejednoznaczność nazwy, usuwając Importy przestrzeni nazw.

- W pełni Zakwalifikuj nazwę.
