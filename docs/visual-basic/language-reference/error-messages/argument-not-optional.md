# <a name="argument-not-optional-visual-basic"></a>Argument nie jest opcjonalny (Visual Basic)
Liczba i typy argumentów musi odpowiadać wartości oczekiwanych. Niepoprawna liczba argumentów jest albo pominięty argument nie jest opcjonalny. Argument tylko można pominąć po wywołaniu procedury zdefiniowane przez użytkownika, jeśli został zadeklarowany jako `Optional` w definicji procedury.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Podaj wszystkie konieczne argumenty.  
  
2.  Upewnij się, pominięcia argumenty są opcjonalne. Jeśli nie, należy dostarczyć argument w wywołaniu lub zadeklarować parametru `Optional` w definicji.
