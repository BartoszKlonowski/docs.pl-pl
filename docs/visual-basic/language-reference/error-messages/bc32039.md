# <a name="array-declared-as-for-loop-control-variable-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="62e6c-102">Tablica zadeklarowana jako zmienna sterująca pętli for nie może być zadeklarowana z rozmiarem początkowym</span><span class="sxs-lookup"><span data-stu-id="62e6c-102">Array declared as for loop control variable cannot be declared with an initial size</span></span>

<span data-ttu-id="62e6c-103">Pętla `For Each` używa tablicy jako zmiennej iteracji, ale inicjuje tę tablicę.</span><span class="sxs-lookup"><span data-stu-id="62e6c-103">A `For Each` loop uses an array as its iteration variable but initializes that array.</span></span>

<span data-ttu-id="62e6c-104">**Identyfikator błędu:** BC32039</span><span class="sxs-lookup"><span data-stu-id="62e6c-104">**Error ID:** BC32039</span></span>

## <a name="example"></a><span data-ttu-id="62e6c-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="62e6c-105">Example</span></span>

<span data-ttu-id="62e6c-106">Poniższy przykład generuje BC32039:</span><span class="sxs-lookup"><span data-stu-id="62e6c-106">The following example generates bc32039:</span></span>

```vb  
Dim arrayList As New List(Of Integer())
For Each listElement(1) As Integer In arrayList

Next
```  

## <a name="to-correct-this-error"></a><span data-ttu-id="62e6c-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="62e6c-107">To correct this error</span></span>

```vb

Next
```

<span data-ttu-id="62e6c-109">lub można użyć wnioskowania o typie:</span><span class="sxs-lookup"><span data-stu-id="62e6c-109">or you can use type inference:</span></span>

```vb

For Each listElement In arrayList

Next
```
