# <a name="overflow-visual-basic-error"></a>Przepełnienie (Błąd w Visual Basic)
Literał reprezentuje wartość poza granicami typu danych, do której jest ona obecnie przypisywana.  
  
 **Identyfikator błędu:** BC30036  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zapoznaj się zakres wartości dla obiekt docelowy typ danych i ponownego zapisywania literał są zgodne z tego zakresu.
