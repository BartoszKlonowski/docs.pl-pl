# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>Nieprawidłowe wywołanie procedury lub argument (Visual Basic)
Nie można wykonać pewną część wywołania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź dozwolonych zakresy dla argumentów upewnić się, że żadne rozmieszczenie nie przekracza dozwolonej wartości.
