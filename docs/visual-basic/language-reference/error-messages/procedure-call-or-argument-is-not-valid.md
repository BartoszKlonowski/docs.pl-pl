# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>Nieprawidłowe wywołanie procedury lub argument (Visual Basic)
Nie można ukończyć część wywołania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź dozwolonych zakresów dla argumentów upewnij się, że nie Porozumienia przekracza dozwolone wartości.
