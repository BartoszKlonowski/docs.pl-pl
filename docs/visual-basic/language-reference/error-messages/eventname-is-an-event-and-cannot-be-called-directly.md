# <a name="39lteventnamegt39-is-an-event-and-cannot-be-called-directly"></a>&#39;&lt;eventName&gt; &#39; jest zdarzeniem i nie można bezpośrednio wywołać
"<`eventname`>" jest zdarzeniem i dlatego nie można wywołać bezpośrednio. Użyj `RaiseEvent` instrukcji, aby wywołać zdarzenie.  
  
 Wywołanie procedury określa zdarzenie dla Nazwa procedury. Program obsługi zdarzeń jest procedurą, ale sam zdarzenia jest sygnalizowania urządzenia, które muszą być wywoływane i obsługi.  
  
 **Identyfikator błędu:** BC32022  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Użyj `RaiseEvent` instrukcji sygnału zdarzenia i wywoływać procedury lub procedury, które ją obsługują.
