# <a name="39lttypenamegt39-is-a-type-and-cannot-be-used-as-an-expression"></a>&#39;&lt;Element TypeName&gt; &#39; jest typem i nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wyrażenie jest wymagana. Wyrażenie musi zawierać kombinację zmienne, stałe, literały, właściwości i `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30108  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłowe.
