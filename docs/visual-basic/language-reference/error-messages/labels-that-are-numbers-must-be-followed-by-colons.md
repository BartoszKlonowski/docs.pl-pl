# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>Po etykietach, które są liczbami, muszą następować dwukropki
Numery wierszy są zgodne z regułami inne rodzaje etykiety i musi zawierać dwukropkiem.  
  
 **Identyfikator błędu:** BC30801  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Umieść kod dwukropkiem na początku wiersza kodu; na przykład:  
  
    ```  
    400:    X += 1  
    ```
