# <a name="identifier-expected"></a>Oczekiwano identyfikatora

Element programistyczny, który nie jest rozpoznawalną zadeklarowaną nazwą elementu, występuje, gdy kontekst wymaga nazwy elementu. Jedną z możliwych przyczyn jest to, że atrybut został określony w innym miejscu niż na początku instrukcji.  
  
 **Identyfikator błędu:** BC30203  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy wszystkie atrybuty instrukcji są umieszczone na początku.  
  
- Sprawdź, czy wszystkie nazwy elementów w instrukcji są wpisane prawidłowo.
