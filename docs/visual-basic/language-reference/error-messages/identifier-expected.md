# <a name="identifier-expected"></a>Oczekiwano identyfikatora
Elementu programistycznego, który nie jest nazwą rozpoznawalnych zadeklarowany element występuje, gdy kontekstu nazwę elementu. Jedną z możliwych przyczyn jest to, że atrybut został określony gdzieś innych niż na początku deklaracji.  
  
 **Identyfikator błędu:** BC30203  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie atrybuty w instrukcji są wszystkie umieszczane na początku.  
  
-   Upewnij się, że wszystkie nazwy elementów w instrukcji są poprawne.
