# <a name="cannot-create-activex-component"></a>Nie można utworzyć składnika ActiveX

Próbowano umieścić kontrolkę ActiveX na formularzu w czasie projektowania lub dodać formularz do projektu z kontrolką ActiveX na nim, ale nie można odnaleźć skojarzonych informacji w rejestrze.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Informacje w rejestrze mogły zostać usunięte lub uszkodzone. Zainstaluj ponownie formant ActiveX lub skontaktuj się z dostawcą kontroli.
