# <a name="cannot-create-activex-component"></a>Nie można utworzyć składnika ActiveX
Nastąpiła próba umieścić formant ActiveX na formularzu w czasie projektowania lub dodać formularz do projektu przy użyciu kontrolki ActiveX na nim, ale nie można odnaleźć skojarzone informacje w rejestrze.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Informacje w rejestrze został usunięty lub uszkodzony. Ponownie zainstaluj formant ActiveX, lub skontaktuj się z dostawcą kontrolki.
