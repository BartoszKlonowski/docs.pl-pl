# <a name="expression-does-not-produce-a-value"></a>Wyrażenie nie zwraca wartości
Podjęto próbę użycia wyrażenia, które nie produkuje wartości w kontekście tworzenia wartości, takich jak wywołanie a `Sub` w kontekście, w którym `Function` jest oczekiwany.  
  
 **Identyfikator błędu:** BC30491  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wyrażenie na takie, które generuje wartość.
