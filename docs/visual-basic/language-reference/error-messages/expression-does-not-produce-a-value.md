# <a name="expression-does-not-produce-a-value"></a>Wyrażenie nie zwraca wartości
Próbowano użyć wyrażenie, które zwraca wartości w kontekście zwracające wartość, takich jak wywoływanie `Sub` w kontekście gdzie `Function` oczekuje.  
  
 **Identyfikator błędu:** BC30491  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wyrażenie który generują wartość.
