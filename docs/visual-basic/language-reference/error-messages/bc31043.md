## <a name="to-correct-this-error"></a><span data-ttu-id="92d4d-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="92d4d-108">To correct this error</span></span>

    ```vb
    End Structure

        Dim struct As DemoStruct
        ReDim struct.demoArray(9)
        Struct.demoArray(2) = 777
    End Sub
    ```
