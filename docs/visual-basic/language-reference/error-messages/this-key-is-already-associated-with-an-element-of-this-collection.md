# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Ten klucz jest już skojarzony z elementem tej kolekcji
Określony klucz, dla członka kolekcji już identyfikuje innego członka kolekcji. Klucz jest ciągiem, który został określony w `Add` metodę, która jednoznacznie identyfikuje określonego członka kolekcji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj innego klucza dla tego elementu członkowskiego.
