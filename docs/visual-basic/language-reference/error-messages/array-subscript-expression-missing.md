# <a name="array-subscript-expression-missing"></a>Brak wyrażenia indeksu tablicy
Inicjowanie tablicy powoduje, że co najmniej jednym z indeksów dolnych, które określają granice tablicy. Na przykład instrukcja może zawierać wyrażenia `myArray (5,5,,10)`, który powoduje, że trzecie indeks dolny.  
  
 **Identyfikator błędu:** BC30306  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj brakujące indeks dolny.
