# <a name="array-subscript-expression-missing"></a>Brak wyrażenia indeksu tablicy
Inicjowanie tablicy powoduje, że jeden lub więcej indeksów dolnych, które definiują granice tablicy. Na przykład instrukcja może zawierać wyrażenia `myArray (5,5,,10)`, który powoduje, że indeks trzecich.  
  
 **Identyfikator błędu:** BC30306  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj brakujące indeks dolny.
