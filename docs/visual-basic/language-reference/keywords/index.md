# <a name="keywords-visual-basic"></a>Słowa kluczowe (Visual Basic)

W poniższej tabeli wymieniono wszystkie Visual Basic słowa kluczowe języka.

## <a name="reserved-keywords"></a>Zastrzeżone słowa kluczowe

Następujące słowa kluczowe są *zastrzeżone*, co oznacza, że nie można ich używać jako nazw dla elementów programistycznych, takich jak zmienne lub procedury. Możesz pominąć to ograniczenie, wpisując nazwę w nawiasy kwadratowe ( `[]` ). Aby uzyskać więcej informacji, zobacz "nazwy ucieczki" w [zadeklarowanych nazwach elementów](../../programming-guide/language-features/declared-elements/declared-element-names.md).

> [!NOTE]
> Nie zaleca się używania nazw unikowych, ponieważ może to utrudniać odczytywanie kodu i może prowadzić do podzielenia błędów, które mogą być trudne do znalezienia.
