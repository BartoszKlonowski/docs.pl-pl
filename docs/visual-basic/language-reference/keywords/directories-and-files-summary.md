# <a name="directories-and-files-summary-visual-basic"></a>Katalogi i pliki — Podsumowanie (Visual Basic)
Słowa kluczowe języka Visual Basic i elementy członkowskie biblioteki wykonawczej są uszeregowane według celu i użycia.  
  
 Za pomocą `My` funkcji zapewnia większą produktywność i wydajność operacji We/Wy pliku niż przy użyciu tych elementów członkowskich. Aby uzyskać więcej informacji, zobacz <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
|**Akcja**|**Element języka**|  
|----------------|--------------------------|  
|Zmiana katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|  
|Zmień dysk.|<xref:Microsoft.VisualBasic.FileSystem.ChDrive%2A>|  
|Kopiowanie pliku.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Upewnij się, katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Usuwanie katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|  
|Zmień nazwę pliku, katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.Rename%2A>|  
|Zwraca bieżącą ścieżkę.|<xref:Microsoft.VisualBasic.FileSystem.CurDir%2A>|  
|Zwraca sygnaturę daty i godziny pliku.|<xref:Microsoft.VisualBasic.FileSystem.FileDateTime%2A>|  
|Zwróć pliku, katalogu lub atrybuty etykiety.|<xref:Microsoft.VisualBasic.FileSystem.GetAttr%2A>|  
|Zwraca długość pliku.|<xref:Microsoft.VisualBasic.FileSystem.FileLen%2A>|  
|Zwraca etykietę nazwy lub wolumin pliku.|<xref:Microsoft.VisualBasic.FileSystem.Dir%2A>|  
|Ustaw informacje o atrybutach pliku.|<xref:Microsoft.VisualBasic.FileSystem.SetAttr%2A>|
