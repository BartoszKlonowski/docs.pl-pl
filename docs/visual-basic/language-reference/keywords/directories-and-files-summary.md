# <a name="directories-and-files-summary-visual-basic"></a>Katalogi i pliki — Podsumowanie (Visual Basic)
Słowa kluczowe języka Visual Basic i elementy członkowskie biblioteki wykonawczej są zorganizowane według celu i użycia.  
  
 Przy użyciu `My` funkcja zapewnia większą wydajność i wydajności w ramach operacji We/Wy pliku niż przy użyciu tych elementów członkowskich. Aby uzyskać więcej informacji, zobacz <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
|**Akcja**|**Element języka**|  
|----------------|--------------------------|  
|Zmienianie katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|  
|Zmień dysk.|<xref:Microsoft.VisualBasic.FileSystem.ChDrive%2A>|  
|Skopiuj plik.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Upewnij się, katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Usuwanie katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|  
|Zmień nazwę pliku, katalogu lub folderu.|<xref:Microsoft.VisualBasic.FileSystem.Rename%2A>|  
|Zwracanie bieżącej ścieżki.|<xref:Microsoft.VisualBasic.FileSystem.CurDir%2A>|  
|Zwróć sygnatury czasowej pliku.|<xref:Microsoft.VisualBasic.FileSystem.FileDateTime%2A>|  
|Zwraca atrybuty etykiety plików lub katalogu.|<xref:Microsoft.VisualBasic.FileSystem.GetAttr%2A>|  
|Zwraca długość pliku.|<xref:Microsoft.VisualBasic.FileSystem.FileLen%2A>|
