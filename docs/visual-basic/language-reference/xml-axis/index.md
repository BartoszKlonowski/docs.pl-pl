# <a name="xml-axis-properties-visual-basic"></a>Właściwości osi XML (Visual Basic)
W tematach w tej sekcji przedstawiono składnię właściwości osi XML w Visual Basic. Właściwości osi XML ułatwiają dostęp do pliku XML bezpośrednio w kodzie.  
  
## <a name="in-this-section"></a>W tej sekcji  
  
|Temat|Opis|  
|-----------|-----------------|  
|[Właściwości osi atrybutu XML](xml-attribute-axis-property.md)|Opisuje sposób uzyskiwania dostępu do atrybutów <xref:System.Xml.Linq.XElement> obiektu.|  
|[Właściwości osi elementu podrzędnego XML](xml-child-axis-property.md)|Opisuje sposób uzyskiwania dostępu do elementów podrzędnych <xref:System.Xml.Linq.XElement> obiektu.|  
|[Właściwości osi obiektu podrzędnego XML](xml-descendant-axis-property.md)|Opisuje sposób uzyskiwania dostępu do elementów podrzędnych <xref:System.Xml.Linq.XElement> obiektu.|  
|[Właściwość indeksatora rozszerzenia](extension-indexer-property.md)|Opisuje sposób uzyskiwania dostępu do poszczególnych elementów w kolekcji <xref:System.Xml.Linq.XElement> lub <xref:System.Xml.Linq.XAttribute> obiektów.|  
|[Właściwość wartości XML](xml-value-property.md)|Opisuje, jak uzyskać dostęp do wartości pierwszego elementu kolekcji <xref:System.Xml.Linq.XElement> lub <xref:System.Xml.Linq.XAttribute> obiektów.|
