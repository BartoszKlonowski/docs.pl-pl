# <a name="properties-visual-basic"></a>Właściwości (Visual Basic)

Ta strona zawiera listę właściwości, które są członkami modułów Visual Basic. Inne właściwości, które są członkami określonych obiektów Visual Basic, są wymienione w [obiekcie](objects/index.md).  
  
## <a name="visual-basic-properties"></a>Właściwości Visual Basic  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.DateAndTime.DateString%2A>|Zwraca lub ustawia `String` wartość reprezentującą bieżącą datę ustawioną w systemie.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Now%2A>|Zwraca `Date` wartość zawierającą aktualną datę i godzinę ustawioną w systemie.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngine%2A>|Zwraca wartość `String` reprezentującą aktualnie używany środowisko uruchomieniowe.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineBuildVersion%2A>|Zwraca `Integer` zawierający numer wersji kompilacji aktualnie używanego środowiska uruchomieniowego.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMajorVersion%2A>|Zwraca `Integer` zawierający numer wersji głównej obecnie używanego środowiska uruchomieniowego.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMinorVersion%2A>|Zwraca wartość `Integer` zawierającą pomocniczy numer wersji aktualnie używanego środowiska uruchomieniowego.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeOfDay%2A>|Zwraca lub ustawia `Date` wartość zawierającą bieżącą godzinę, która jest zgodna z systemem.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Timer%2A>|Zwraca `Double` wartość reprezentującą liczbę sekund, jaka upłynęła od północy.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeString%2A>|Zwraca lub ustawia `String` wartość reprezentującą bieżącą godzinę, która jest zgodna z systemem.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Today%2A>|Zwraca lub ustawia `Date` wartość zawierającą bieżącą datę ustawioną w systemie.|
