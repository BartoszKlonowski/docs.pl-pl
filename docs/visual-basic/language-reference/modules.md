# <a name="modules-visual-basic"></a>Moduły (Visual Basic)
Visual Basic oferuje kilka modułów, które pozwalają uprościć typowe zadania w kodzie, w tym manipulowanie ciągami, wykonywanie obliczeń matematycznych, uzyskiwanie informacji o systemie, wykonywanie operacji na plikach i katalogach i tak dalej. Poniższa tabela zawiera listę modułów języka Visual Basic.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants>|Zawiera różne stałe. Te stałych można używać w dowolnym miejscu w kodzie.|  
|<xref:Microsoft.VisualBasic.ControlChars>|Zawiera znaki kontrolne stałej drukowania i wyświetlania tekstu.|  
|<xref:Microsoft.VisualBasic.Conversion>|Zawiera elementy członkowskie, które Konwertuj liczby dziesiętne na inne podstawy liczb do ciągów, ciągów na liczby i danych jednego typu na inny.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|Zawiera elementy członkowskie, które pobieranie bieżącej daty lub godziny, wykonywać obliczenia daty, zwracają daty lub godziny, Ustaw daty lub godziny lub czas trwania procesu.|  
|<xref:Microsoft.VisualBasic.ErrObject>|Zawiera informacje dotyczące błędów czasu wykonywania i metod, aby podnieść lub wyczyścić błąd.|  
|<xref:Microsoft.VisualBasic.FileSystem>|Zawiera elementy członkowskie, które wykonują operacje na pliku, katalogu lub folderu i systemu.|  
|<xref:Microsoft.VisualBasic.Financial>|Zawiera procedury, które są używane do wykonywania obliczeń finansowych.|  
|<xref:Microsoft.VisualBasic.Globals>|Zawiera informacje o bieżącej wersji aparatu obsługi skryptów.|  
|<xref:Microsoft.VisualBasic.Information>|Zawiera elementy członkowskie, które zwracają, testowania lub sprawdź informacje, takie jak rozmiar tablicy, nazwy typów i tak dalej.|  
|<xref:Microsoft.VisualBasic.Interaction>|Zawiera elementy członkowskie wchodzić w interakcje z obiektami, aplikacji i systemów.|  
|<xref:Microsoft.VisualBasic.Strings>|Zawiera elementy członkowskie, które wykonywać operacje na ciągach takie jak ponowne formatowanie ciągów, wyszukiwanie ciągu, wprowadzenie długość ciągu i tak dalej.|  
|<xref:Microsoft.VisualBasic.VBMath>|Zawiera elementy członkowskie wykonywać operacje matematyczne.|
