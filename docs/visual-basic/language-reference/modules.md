# <a name="modules-visual-basic"></a>Moduły (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]zawiera kilka modułów, które pozwalają uprościć typowe zadania w kodzie, w tym manipulowanie ciągami, wykonywania działań matematycznych, uzyskiwanie informacji o systemie, wykonywania działań plików i katalogów i tak dalej. Poniższa tabela zawiera listę modułów podał [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants>|Zawiera różne stałe. Te stałych można używać w dowolnym miejscu w kodzie.|  
|<xref:Microsoft.VisualBasic.ControlChars>|Zawiera znaki kontrolne stałej drukowania i wyświetlanie tekstu.|  
|<xref:Microsoft.VisualBasic.Conversion>|Zawiera elementy członkowskie, które konwertuje dziesiętne innych baz wpisz numery ciągi ciągów na liczby i jednego danych do innego.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|Zawiera elementy członkowskie, które pobrać bieżącą datę lub godzinę, obliczeń daty, zwróć daty lub godziny, ustawić daty lub godziny lub czas trwania procesu.|  
|<xref:Microsoft.VisualBasic.ErrObject>|Zawiera informacje dotyczące błędów czasu wykonywania i metody, aby podnieść lub wyczyścić błąd.|  
|<xref:Microsoft.VisualBasic.FileSystem>|Zawiera elementy członkowskie, które wykonują operacje pliku, katalogu lub folderu i systemu.|  
|<xref:Microsoft.VisualBasic.Financial>|Zawiera procedury, które są używane do obliczeń finansowych.|  
|<xref:Microsoft.VisualBasic.Globals>|Zawiera informacje o bieżącej wersji aparatu skryptów.|  
|<xref:Microsoft.VisualBasic.Information>|Zawiera elementy członkowskie, które zwracają, testowania lub sprawdź informacje, takie jak rozmiar tablicy, wpisz nazwy i tak dalej.|  
|<xref:Microsoft.VisualBasic.Interaction>|Zawiera elementy członkowskie interakcji z obiektów, aplikacji i systemów.|  
|<xref:Microsoft.VisualBasic.Strings>|Zawiera elementy członkowskie, które wykonywać operacje na ciągach takie jak ponowne formatowanie ciągów wyszukiwania ciąg, długość ciągu, pobieranie i tak dalej.|  
|<xref:Microsoft.VisualBasic.VBMath>|Zawiera elementy członkowskie wykonywanie operacji matematycznych.|
