# <a name="queries-visual-basic"></a>Zapytania (Visual Basic)
Visual Basic umożliwia tworzenie wyrażeń programu Query-Integrated Language (LINQ) w kodzie.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Aggregate, klauzula](aggregate-clause.md)  
 Opisuje `Aggregate` klauzulę, która stosuje co najmniej jedną funkcję agregującą do kolekcji.  
  
 [Distinct, klauzula](distinct-clause.md)  
 Opisuje `Distinct` klauzulę ograniczającą wartości bieżącej zmiennej zakresu w celu wyeliminowania zduplikowanych wartości w wynikach zapytania.  
  
 [Klauzula from](from-clause.md)  
 Opisuje `From` klauzulę, która określa kolekcję i zmienną zakresu dla zapytania.  
  
 [Group By, klauzula](group-by-clause.md)  
 Opisuje `Group By` klauzulę, która grupuje elementy wyniku zapytania i może służyć do zastosowania funkcji agregujących do każdej grupy.  
  
 [Group Join. klauzula](group-join-clause.md)  
 Opisuje `Group Join` klauzulę, która łączy dwie kolekcje w jedną hierarchiczną kolekcję.  
  
 [Klauzula join](join-clause.md)  
 Opisuje `Join` klauzulę, która łączy dwie kolekcje w jedną kolekcję.  
  
 [Klauzula Let](let-clause.md)  
 Opisuje `Let` klauzulę, która oblicza wartość i przypisuje ją do nowej zmiennej w zapytaniu.  
  
 [Order By, klauzula](order-by-clause.md)  
 Opisuje `Order By` klauzulę, która określa kolejność sortowania kolumn w zapytaniu.  
  
 [SELECT — klauzula](select-clause.md)  
 Opisuje `Select` klauzulę, która deklaruje zestaw zmiennych zakresu dla zapytania.  
  
 [Skip, klauzula](skip-clause.md)  
 Opisuje `Skip` klauzulę, która pomija określoną liczbę elementów w kolekcji, a następnie zwraca pozostałe elementy.  
  
 [Skip While, klauzula](skip-while-clause.md)  
 Opisuje `Skip While` klauzulę, która pomija elementy w kolekcji, tak długo, jak określony warunek jest, `true` a następnie zwraca pozostałe elementy.  
  
 [Take, klauzula](take-clause.md)  
 Opisuje `Take` klauzulę, która zwraca określoną liczbę elementów sąsiadujących od początku kolekcji.  
  
 [Take While, klauzula](take-while-clause.md)  
 Opisuje `Take While` klauzulę, która zawiera elementy w kolekcji, tak długo, jak określony warunek `true` , i pomija pozostałe elementy.  
  
 [Klauzula WHERE](where-clause.md)  
 Opisuje `Where` klauzulę, która określa warunek filtrowania dla zapytania.
