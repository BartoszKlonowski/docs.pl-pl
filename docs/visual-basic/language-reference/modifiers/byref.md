# <a name="byref-visual-basic"></a>ByRef (Visual Basic)
Określa, że argument jest przekazywany w taki sposób, że wywoływana procedura może zmienić wartość zmiennej bazowy argumentu w wywoływanym kodzie.  
  
## <a name="remarks"></a>Uwagi  
 `ByRef` Modyfikatora można używać w tych sytuacjach:  
  
 [DECLARE — instrukcja](../../../visual-basic/language-reference/statements/declare-statement.md)  
  
 [Function — instrukcja](../../../visual-basic/language-reference/statements/function-statement.md)  
  
 [Sub — instrukcja](../../../visual-basic/language-reference/statements/sub-statement.md)
