# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Określa, że klasa nie może służyć jako klasę podstawową.  
  
## <a name="remarks"></a>Uwagi  
 **Alternatywny warunki**. Klasa, która nie może być dziedziczona, jest czasem nazywany *zapieczętowanego* klasy.
