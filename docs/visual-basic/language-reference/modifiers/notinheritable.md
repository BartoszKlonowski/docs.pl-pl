# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Określa, że klasa nie można użyć jako klasę bazową.  
  
## <a name="remarks"></a>Uwagi  
 **Alternatywne warunki**. Klasa, która nie może być dziedziczona, jest czasami nazywane *zapieczętowanego* klasy.
