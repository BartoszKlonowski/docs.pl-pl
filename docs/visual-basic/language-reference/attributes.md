# <a name="attributes-visual-basic"></a>Atrybuty (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]udostępnia kilka atrybutów, które zezwalają obiektów współdziałanie z kodem niezarządzanym i jeden atrybut, który umożliwia członkom modułu można uzyskać dostęp bez nazwy modułu. Poniższa tabela zawiera listę atrybutów używanych przez [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Instruuje kompilator, aby dodać metadanych, które umożliwia klasy mają być uwidaczniane jako obiekt COM.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Umożliwia członkom modułu uzyskać dostęp za pomocą kwalifikacji potrzebne dla modułu.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Wskazuje, że tablicy w strukturze lub zmiennej lokalnej nie powinny być traktowane jako tablicę o stałej długości.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Wskazuje, że ciąg powinien być traktowany tak, jakby były stałej długości.|
