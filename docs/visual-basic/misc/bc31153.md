# <a name="required-attribute-version-missing-from-xml-declaration"></a>Brak wymaganego atrybutu "Version" w deklaracji XML

`version`Brak atrybutu dla literału dokumentu XML.  
  
 **Identyfikator błędu:** BC31153  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `version` atrybut do deklaracji dokumentu XML, jak w poniższym przykładzie: `<?xml version="1.0"?>`
