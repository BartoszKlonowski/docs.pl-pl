# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>Wymagany atrybut &#39; wersja &#39; brakuje deklaracji XML
`version` Atrybutu literał dokumentu XML nie istnieje.  
  
 **Identyfikator błędu:** BC31153  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `version` atrybutu deklaracji dokumentu XML, jak w poniższym przykładzie:`<?xml version="1.0"?>`
