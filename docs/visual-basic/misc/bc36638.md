# <a name="instance-members-and-39me39-cannot-be-used-within-a-lambda-expression-in-structures"></a><span data-ttu-id="a908e-102">Wystąpienia elementów członkowskich i &#39;mnie&#39; nie można użyć w wyrażeniu lambda w strukturach</span><span class="sxs-lookup"><span data-stu-id="a908e-102">Instance members and &#39;Me&#39; cannot be used within a lambda expression in structures</span></span>
<span data-ttu-id="a908e-103">Z wewnątrz struktury, zdefiniowane wyrażenie lambda, która odwołuje się do elementu członkowskiego wystąpienia struktury lub używa `Me`.</span><span class="sxs-lookup"><span data-stu-id="a908e-103">From within a structure, you have defined a lambda expression that refers to an instance member of the structure or uses `Me`.</span></span> <span data-ttu-id="a908e-104">Poniższy kod ilustruje tych dwóch nieprawidłowe odwołania.</span><span class="sxs-lookup"><span data-stu-id="a908e-104">The following code illustrates these two invalid references.</span></span>  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 <span data-ttu-id="a908e-105">**Identyfikator błędu:** BC36638</span><span class="sxs-lookup"><span data-stu-id="a908e-105">**Error ID:** BC36638</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a908e-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a908e-106">To correct this error</span></span>  
  
-   <span data-ttu-id="a908e-107">Assign — członek wystąpienie do zmiennej lokalnej, a następnie użyć zmiennej lokalnej w instrukcji.</span><span class="sxs-lookup"><span data-stu-id="a908e-107">Assign the instance member to a local variable, and use the local variable in your statement.</span></span>  
  
```vb  
Public Function ExampleFunFix() As Integer  
    Dim temp = InstanceMember  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
