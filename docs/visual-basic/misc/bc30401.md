# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>element " \<interfacename1> " nie może implementować " \<methodname> ", ponieważ nie ma pasującego \<method> interfejsu " \<interfacename2> "

Podjęto próbę zaimplementowania metody w interfejsie, jeśli nie istnieje zgodna Metoda w interfejsie, z którego pochodzi pierwszy interfejs.  
  
 **Identyfikator błędu:** BC30401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nie podano niewpisanej nazwy metody lub nie podawaj brakującej metody.
