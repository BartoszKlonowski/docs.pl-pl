# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>"&lt;interfacename1&gt;nie może implementować&lt;methodname&gt;", ponieważ nie ma pasującego &lt;metoda&gt; w interfejsie "&lt;interfacename2&gt;"
Chcesz zaimplementować metodę w interfejsie, gdy ma odpowiadającej metody interfejsu, z którego zostały uzyskane pierwszy interfejs.  
  
 **Identyfikator błędu:** BC30401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że nie ma źle wpisano nazwę metody lub podać Brak metody.
