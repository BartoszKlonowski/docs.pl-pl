# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>"\<interfacename1 >' nie może implementować"\<methodname > ", ponieważ nie ma pasującego \<metody > w interfejsie"\<interfacename2 > "
Chcesz zaimplementować metodę w interfejsie, gdy ma odpowiadającej metody interfejsu, z którego zostały uzyskane pierwszy interfejs.  
  
 **Identyfikator błędu:** BC30401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że nie ma źle wpisano nazwę metody lub podać Brak metody.
