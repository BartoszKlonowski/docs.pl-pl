# <a name="file-already-exists"></a>Plik już istnieje

Ten błąd występuje w wyniku zmiany nazwy nowego pliku na nazwę istniejącego pliku lub przy użyciu polecenia **Zapisz jako** , aby zapisać aktualnie załadowany projekt o nazwie, która już istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź nazwę i zmień ją na taką, która nie powoduje konfliktu, lub usuń istniejący plik lub projekt, jeśli nie jest już wymagany.
