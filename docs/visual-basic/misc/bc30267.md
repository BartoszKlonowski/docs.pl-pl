# <a name="declaration1-cannot-override-declaration2-because-it-is-declared-notoverridable"></a>'\<declaration1 >' nie może przesłonić'\<declaration2 > ', ponieważ jest on zadeklarowany jako'NotOverridable'
Procedura lub Deklaracja właściwości próbuje przesłonić Dziedziczony element o tej samej nazwie, ale Dziedziczony element jest określony jako `NotOverridable`.  
  
 **Identyfikator błędu:** BC30267  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `NotOverridable` Usuń słowo kluczowe z deklaracji dziedziczonego elementu lub Usuń deklarację zastępującą.
