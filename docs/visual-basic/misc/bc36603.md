# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="d5434-102">Wielowymiarowe tablice nie można przekonwertować na drzewo wyrażenia</span><span class="sxs-lookup"><span data-stu-id="d5434-102">Multi-dimensional array cannot be converted to an expression tree</span></span>
<span data-ttu-id="d5434-103">Większość wyrażenia można konwertować na drzewa wyrażeń, ale nie może być Wielowymiarowe tablice.</span><span class="sxs-lookup"><span data-stu-id="d5434-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="d5434-104">Na przykład poniższy kod powoduje błąd:</span><span class="sxs-lookup"><span data-stu-id="d5434-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.
```  
