# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="108e7-102">Nie można przekonwertować tablicy wielowymiarowej na drzewo wyrażenia</span><span class="sxs-lookup"><span data-stu-id="108e7-102">Multi-dimensional array cannot be converted to an expression tree</span></span>

<span data-ttu-id="108e7-103">Większość wyrażeń można przekonwertować na drzewa wyrażeń, ale tablice wielowymiarowe nie mogą być.</span><span class="sxs-lookup"><span data-stu-id="108e7-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="108e7-104">Na przykład poniższy kod powoduje wystąpienie tego błędu:</span><span class="sxs-lookup"><span data-stu-id="108e7-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
