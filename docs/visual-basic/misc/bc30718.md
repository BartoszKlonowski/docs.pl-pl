# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>Instrukcji "Select" są nieprawidłowe w oknie bezpośrednim
`Select` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30718  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Select` instrukcji debugowania kodu.
