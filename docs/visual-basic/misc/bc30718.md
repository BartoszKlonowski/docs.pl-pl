# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Wybierz&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Select` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30718  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Select` oświadczenia debugowania kodu.
