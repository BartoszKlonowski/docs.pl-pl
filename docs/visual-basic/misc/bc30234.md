# <a name="specifier-is-not-valid-on-a-withevents-declaration"></a>element " \<specifier> " nie jest prawidłowy w deklaracji WithEvents

`Dim`Instrukcja zawiera słowo kluczowe, takie jak `ReadOnly` , które jest nieprawidłowe w połączeniu ze `WithEvents` słowem kluczowym.  
  
 **Identyfikator błędu:** BC30234  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń nieprawidłowe słowo kluczowe z `Dim` instrukcji.
