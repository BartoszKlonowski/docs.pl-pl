# <a name="ltspecifiergt-is-not-valid-on-a-withevents-declaration"></a>"&lt;specyfikator&gt;" nie jest prawidłowy w deklaracji WithEvents
A `Dim` instrukcja zawiera słowo kluczowe `ReadOnly`, która jest nieprawidłowa w połączeniu z `WithEvents` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC30234  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nieprawidłowe słowo kluczowe z `Dim` instrukcji.
