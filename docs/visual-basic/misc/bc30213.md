# <a name="comma-or-3939-expected-procedure-parameter-list"></a>Przecinek lub &#39;) &#39; Oczekiwano (lista parametrów procedury)
Lista parametrów zawiera nieoczekiwany element. Użyj przecinków (`,`) do rozdzielenia parametrów takich na liście, a następnie użyć nawiasu zamykającego (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC30213  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodawanie przecinkami i kontynuować listy parametrów typu lub dodawanie zamknięcia znak nawiasu zakończenie na liście.
