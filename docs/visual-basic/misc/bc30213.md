# <a name="comma-or--expected-procedure-parameter-list"></a>Oczekiwano przecinka lub znaku ")" (lista parametrów procedury)
Lista parametrów zawiera nieoczekiwany element. Użyj przecinków ( `,` ) do oddzielania parametrów na liście, a następnie użyj zamykającego nawiasu `)` (), aby zakończyć listę.  
  
 **Identyfikator błędu:** BC30213  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj przecinek i Kontynuuj listę parametrów typu lub Dodaj znak nawias zamykający, aby zakończyć listę.
