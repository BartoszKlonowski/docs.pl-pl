# <a name="comma-or--expected-procedure-parameter-list"></a>Przecinek lub ')' oczekiwany (lista parametrów procedury)
Lista parametrów zawiera nieoczekiwany element. Używaj przecinków (`,`) do oddzielania parametrów tych na liście, a następnie użyj nawias zamykający (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC30213  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj przecinek i kontynuować lista parametrów typu lub Dodaj zamykający nawias znaków do zakończenia na liście.
