# <a name="39gt39-expected"></a>&#39; &gt;&#39; oczekiwano
Składnia dla określonego kodu jest niepoprawna. Często ten błąd oznacza, że brakuje zamykającego nawiasu ostrego dla atrybutu lub literał elementu XML.  
  
 **Identyfikator błędu:** BC30636  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź składnię kod używany.
