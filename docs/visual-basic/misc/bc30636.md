# <a name="-expected"></a>'>' Oczekiwano
Składnia określonego kodu jest niepoprawna. Ten błąd często oznacza, że brakuje zamykającego nawiasu ostrego dla atrybutu lub literału elementu XML.  
  
 **Identyfikator błędu:** BC30636  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź składnię używanego kodu.
