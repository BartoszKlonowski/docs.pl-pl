# <a name="gt-expected"></a>"&gt;" Oczekiwano
Składnia dla określonego kodu jest niepoprawna. Często ten błąd wskazuje, że brakuje nawiasu ostrego zamykającego dla atrybutu lub literał elementu XML.  
  
 **Identyfikator błędu:** BC30636  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź składnię kod używany.
