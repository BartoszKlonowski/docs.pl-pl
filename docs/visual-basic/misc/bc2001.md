# <a name="file-ltfilenamegt-could-not-be-found"></a>Plik &lt;filename&gt; nie można odnaleźć
Nie można zlokalizować określonego pliku. Może to być wynikiem podawania Niepoprawna ścieżka.  
  
 **Identyfikator błędu:** BC2001  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy nazwa pliku i ścieżka są poprawne.
