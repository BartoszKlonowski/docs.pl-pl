# <a name="file-filename-could-not-be-found"></a>\<filename>Nie można znaleźć pliku

Nie można zlokalizować określonego pliku. Może to być spowodowane podawaniem nieprawidłowej ścieżki pliku.  
  
 **Identyfikator błędu:** BC2001  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwa pliku i ścieżka są poprawne.
