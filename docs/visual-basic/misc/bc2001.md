# <a name="file-filename-could-not-be-found"></a>Plik \<nazwa pliku > nie można odnaleźć
Nie można zlokalizować określonego pliku. Przyczyną może być podanie niepoprawnej ścieżki pliku.  
  
 **Identyfikator błędu:** BC2001  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwa pliku i ścieżka są poprawne.
