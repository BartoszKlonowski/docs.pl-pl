# <a name="delegate-in-an-interface-cannot-be-declared-specifier"></a>Delegat w interfejsie nie może być zadeklarowany jako " \<specifier> "
Określono nieprawidłowy modyfikator z delegatem zadeklarowanym w interfejsie.  
  
 **Identyfikator błędu:** BC31068  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń modyfikatory, takie jak,,,,, `Public` `Private` `Shared` `Friend` `Protected` `Protected Friend` i `Overridable` .
