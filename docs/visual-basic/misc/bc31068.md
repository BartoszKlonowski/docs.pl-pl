# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować delegatu w interfejsie &#39; &lt;Specyfikator&gt;&#39;
Nieprawidłowy modyfikator określono z delegata zadeklarowany w interfejsie.  
  
 **Identyfikator błędu:** BC31068  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń modyfikatory, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, i `Overridable`.
