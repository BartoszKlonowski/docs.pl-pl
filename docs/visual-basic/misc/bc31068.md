# <a name="delegate-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować delegatu w interfejsie jako '\<specyfikator >'
Nieprawidłowy modyfikator została określona za pomocą delegata zadeklarowanych w interfejsie.  
  
 **Identyfikator błędu:** BC31068  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń modyfikatorów, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, i `Overridable`.
