# <a name="databits-must-be-greater-than-0"></a>DataBits musi być większa niż 0
`DataBits` Określono argumentu dla `My.Computer.Ports.OpenSerialPort` metoda musi być większa od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość właściwości `DataBits` argument na liczbę dodatnią.
