# <a name="databits-must-be-greater-than-0"></a>Usługa databitowy musi być większa niż 0

`DataBits`Argument określony dla `My.Computer.Ports.OpenSerialPort` metody musi być większy od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość `DataBits` argumentu na liczbę dodatnią.
