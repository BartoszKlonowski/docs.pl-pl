# <a name="root-folder-cannot-be-renamed"></a>Nie można zmienić nazwy folderu głównego

Podjęto próbę zmiany nazwy katalogu głównego. Może to wynikać z nieprawidłowo przeanalizowanej ścieżki pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź ścieżkę pliku, aby upewnić się, że jest ona poprawna.
