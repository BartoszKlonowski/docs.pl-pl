# <a name="root-folder-cannot-be-renamed"></a>Nie można zmienić nazwy folderu głównego
Nastąpiła próba można zmienić nazwy katalogu głównego. Może to być wynikiem ścieżka pliku nieprawidłowo przeanalizowane.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź ścieżkę pliku, aby upewnić się, że jest poprawny.
