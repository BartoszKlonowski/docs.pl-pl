# <a name="argument-period-must-be-less-than-or-equal-to-argument-life"></a>Argument "period" musi być mniejszy lub równy argumentowi "Life"
Wartość `Period` argumentu określająca okres, dla którego jest obliczana amortyzacja zasobów, jest większa niż wartość `Life` argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, `Life` że `Period` argumenty i są wyrażone w tych samych jednostkach. Na przykład, jeśli `Life` jest mierzona w miesiącach, `Period` powinna być również.
