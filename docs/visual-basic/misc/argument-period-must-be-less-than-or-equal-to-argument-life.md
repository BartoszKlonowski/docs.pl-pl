# <a name="argument-period-must-be-less-than-or-equal-to-argument-life"></a>Argument "okres" musi być mniejsza lub równa argumentowi "Życia"
Wartość `Period` argument, który określa okres, dla którego jest obliczana amortyzacja środka trwałego, jest większa niż wartość `Life` argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `Life` i `Period` argumenty są wyrażone w tych samych jednostkach. Na przykład jeśli `Life` jest mierzony w miesiącach `Period` powinny być również.
