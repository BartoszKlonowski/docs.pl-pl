# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Wartość '\<valuename1 >' nie można przekonwertować na '\<valuename2 >'
Konwersja typu podjęto próbę między dwoma typami niezgodne.  
  
 **Identyfikator błędu:** BC30742  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy, które są rzutowany.
