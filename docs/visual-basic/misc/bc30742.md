# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>Wartość "&lt;valuename1&gt;"nie można przekonwertować na"&lt;valuename2&gt;"
Konwersja typu podjęto próbę między dwoma typami niezgodne.  
  
 **Identyfikator błędu:** BC30742  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź typy, które są rzutowany.
