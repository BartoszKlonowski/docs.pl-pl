# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>\<valuename1>Nie można przekonwertować wartości "" na " \<valuename2> "

Podjęto próbę konwersji typu między dwoma niezgodnymi typami.  
  
 **Identyfikator błędu:** BC30742  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy, które są rzutowane.
