# <a name="cannot-convert-start-value-of-type-type1-limit-value-of-type2-and-step-value-of-type3-to-a-common-type"></a>Nie można przekonwertować wartości początkowej typu " \<type1> ", wartości limitu " \<type2> " i wartości kroku " \<type3> " na typ wspólny
Określone `For...Next` wartości kontrolne mają różne typy i w związku z tym są nieprawidłowe, ponieważ nie mogą być konwertowane na typ wspólny.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień typ jednej lub więcej wartości, aby wszystkie można było skonwertować do typu wspólnego.
