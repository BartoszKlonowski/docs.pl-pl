# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Nie można odczytać strumienia przekazanego do parametru TextFieldParser
[Obiektu TextFieldParser](../../visual-basic/language-reference/objects/textfieldparser-object.md) nie można odczytać strumienia, który został przekazany do niego. Może to być wynikiem próby odczytania pliku, który nie jest plikiem tekstowym.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zaznacz, aby upewnić się, że plik jest plikiem tekstowym.
