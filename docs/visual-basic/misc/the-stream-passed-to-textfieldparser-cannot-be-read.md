# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Nie można odczytać strumienia przesłanego do TextFieldParser
[Obiekt TextFieldParser](../language-reference/objects/textfieldparser-object.md) nie może odczytać strumienia, który został do niego przekazano. Może to wynikać z próby odczytania pliku, który nie jest plikiem tekstowym.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że plik jest plikiem tekstowym.
