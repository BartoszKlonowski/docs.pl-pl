# <a name="overload-resolution-failed-because-no-accessible-method-accepts-this-number-of-arguments"></a>Przeciążenia nie powiodło się, ponieważ nie jest dostępny '\<metody >' nie akceptuje tej liczby argumentów
Wprowadzono wywołanie przeciążonej metody, ale kompilator nie można odnaleźć metody o dopasowania listy argumentów.  
  
 **Identyfikator błędu:** BC30516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź listę argumentów, podane.
