# <a name="overload-resolution-failed-because-no-accessible-method-accepts-this-number-of-arguments"></a>Rozpoznanie przeciążenia nie powiodło się, ponieważ żaden z dostępnych \<method> elementów "" nie akceptuje tej liczby argumentów

Wykonano wywołanie przeciążonej metody, ale kompilator nie może odnaleźć metody ze zgodną listą argumentów.  
  
 **Identyfikator błędu:** BC30516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podaną listę argumentów.
