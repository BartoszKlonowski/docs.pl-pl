# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>Przeciążenia nie powiodło się, ponieważ żaden z dostępnych &#39; &lt;metody&gt;&#39; akceptuje tej liczby argumentów
Wywołanie przeciążonej metody zostały wykonane, ale kompilator nie można odnaleźć metody o dopasowania listy argumentów.  
  
 **Identyfikator błędu:** BC30516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź podane listy argumentów.
