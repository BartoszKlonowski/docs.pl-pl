# <a name="functionname-is-not-declared-visual-basic-error"></a>element " \<functionname> " nie jest zadeklarowany (błąd Visual Basic)

element " \<functionname> " nie jest zadeklarowany. Funkcja we/wy plików jest dostępna w przestrzeni nazw Microsoft. VisualBasic.  
  
 **Identyfikator błędu:** BC30815  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj funkcji we/wy plików zdefiniowanych w `My.Computer.FileSystem` , `System.IO` przestrzeni nazw, Visual Basic funkcji czasu wykonywania lub `Microsoft.VisualBasic.FileSystem` obiektu.
