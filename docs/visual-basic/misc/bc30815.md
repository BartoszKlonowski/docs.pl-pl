# <a name="39ltfunctionnamegt39-is-not-declared-visual-basic-error"></a>&#39; &lt;functionname&gt;&#39; nie jest zadeklarowany jako (Błąd Visual Basic)
"\<functionname >' nie został zadeklarowany. Funkcje We/Wy plików są dostępne w przestrzeni nazw Microsoft.VisualBasic.  
  
 **Identyfikator błędu:** BC30815  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zdefiniowany w funkcji we/wy pliku `My.Computer.FileSystem`, `System.IO` przestrzeni nazw, funkcje wykonawcze języka Visual Basic, lub `Microsoft.VisualBasic.FileSystem` obiektu.
