<span data-ttu-id="28bdc-103">Struktura dziedziczenie powoduje zależność cykliczną między zagnieżdżonych klas, oznacza to, dwie klasy dziedziczy od siebie.</span><span class="sxs-lookup"><span data-stu-id="28bdc-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>

    End Class
