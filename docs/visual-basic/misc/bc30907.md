# <a name="this-inheritance-causes-circular-dependencies-between-type1-typename1-and-its-nested-type2-typename2"></a><span data-ttu-id="1e6b0-102">To dziedziczenie powoduje powstanie zależności cyklicznych między \<type1 > '\<typename1 >' i jego zagnieżdżonych \<type2 > '\<typename2 >'</span><span class="sxs-lookup"><span data-stu-id="1e6b0-102">This inheritance causes circular dependencies between \<type1> '\<typename1>' and its nested \<type2> '\<typename2>'</span></span>
<span data-ttu-id="1e6b0-103">Struktura dziedziczenie powoduje wykryto zależność cykliczną między klas zagnieżdżonych, oznacza to, dwie klasy dziedziczące od siebie nawzajem.</span><span class="sxs-lookup"><span data-stu-id="1e6b0-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="1e6b0-104">Poniższy kod można wygenerować ten komunikat o błędzie.</span><span class="sxs-lookup"><span data-stu-id="1e6b0-104">The following code can generate this error message.</span></span>  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="1e6b0-105">W poprzednim kodzie klasy `c1` dziedziczy z klasy `c4`, ale `c4` jest zagnieżdżona `c3`, który dziedziczy z `c2`, zagnieżdżone wewnątrz `c1`.</span><span class="sxs-lookup"><span data-stu-id="1e6b0-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>
