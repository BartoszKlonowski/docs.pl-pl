# <a name="39withevents39-variables-can-only-be-typed-as-classes-interfaces-or-type-parameters-with-class-constraints"></a>&#39; WithEvents &#39; Zmienne można określać tylko jako klasy, interfejsy lub parametry typu z ograniczeniami klasy
Zadeklarowana zmienna, która zostanie wpisany struktury w połączeniu z `WithEvents`, która nie jest prawidłowe użycie `WithEvents` modyfikator.  
  
 **Identyfikator błędu:** BC30413  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Użyj `AddHandler` do obsługi zdarzeń zdefiniowanych w strukturze.
