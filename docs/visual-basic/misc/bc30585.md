# <a name="event-lteventnamegt-cannot-be-handled-because-it-is-not-accessible-from-ltnamegt"></a>Zdarzenie "&lt;eventname&gt;nie mogą być obsługiwane", ponieważ nie jest dostępny z"&lt;nazwa&gt;"
Podjęto próbę obsłużyć zdarzenie, które jest niedostępne. Na przykład jeśli `Handles` zmienna jest udostępniany, również muszą zostać udostępnione metodzie obsługi zdarzenia.  
  
 **Identyfikator błędu:** BC30585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że zdarzenie jest dostępny.
