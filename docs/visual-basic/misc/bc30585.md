# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>Nie można \<eventname> obsłużyć zdarzenia "", ponieważ nie jest ono dostępne z " \<name> "

Podjęto próbę obsługi zdarzenia, które jest niedostępne. Na przykład jeśli `Handles` zmienna jest udostępniona, metoda obsługi zdarzeń musi również być udostępniona.  
  
 **Identyfikator błędu:** BC30585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że zdarzenie jest dostępne.
