# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>Zdarzenia &#39; &lt;eventname&gt; &#39; nie mogą być obsługiwane, ponieważ nie jest dostępny z &#39; &lt;nazwy&gt;&#39;
Próbowano obsługi zdarzeń, które jest niedostępne. Na przykład jeśli `Handles` zmiennej jest udostępniony, również muszą zostać udostępnione metoda obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC30585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że zdarzenie jest dostępny.
