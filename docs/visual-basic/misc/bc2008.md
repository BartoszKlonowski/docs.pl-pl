# <a name="no-input-sources-specified"></a>Nie źródeł danych wejściowych, które zostały określone
Podczas kompilowania, określono nie źródeł danych wejściowych.  
  
 **Identyfikator błędu:** BC2008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wprowadzono plik kodu źródłowego.
