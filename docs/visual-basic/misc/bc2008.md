# <a name="no-input-sources-specified"></a>Nie określono źródeł wejściowych

Podczas kompilowania nie określono żadnych źródeł wejściowych.  
  
 **Identyfikator błędu:** BC2008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że został dołączony plik kodu źródłowego.
