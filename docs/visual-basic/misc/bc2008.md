# <a name="no-input-sources-specified"></a>Nie określono źródeł dla wejścia
W przypadku kompilowania kodu, określono nie źródeł danych wejściowych.  
  
 **Identyfikator błędu:** BC2008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wprowadzono pliku kodu źródłowego.
