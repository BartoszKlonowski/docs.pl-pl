# <a name="either-the-parameter-type-or-the-return-type-of-this-conversion-operator-must-be-the-containing-type"></a>Typ parametru lub typ zwracany tego operatora konwersji musi być typem zawierającym
Definicję operatora konwersji określa parametr i typ zwracany z typów innych niż klasa lub struktura, w którym zdefiniowano operator.  
  
 Podczas definiowania operatora konwersji w klasie lub strukturze, przekonwertuj na lub z typu tej klasy lub struktury.  
  
 **Identyfikator błędu:** BC33022  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień typ parametru lub typ zwracany typ klasy lub struktury, w którym zdefiniowano operator.
