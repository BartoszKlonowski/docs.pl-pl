# <a name="either-the-parameter-type-or-the-return-type-of-this-conversion-operator-must-be-the-containing-type"></a>Typ parametru lub typ zwracany tego operatora konwersji muszą być typem zawierającym
Definicja operatora konwersji określa parametr i typ zwracany z typami inną niż klasy lub struktury, w którym jest zdefiniowany operator.  
  
 Po zdefiniowaniu operatora konwersji w klasie lub strukturze go przekonwertować na lub z typu tej klasy lub struktury.  
  
 **Identyfikator błędu:** BC33022  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ parametru lub typ zwracany typ klasy lub struktury, w którym jest zdefiniowany operator.
