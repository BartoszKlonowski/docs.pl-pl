# <a name="raiseevent-is-already-declared"></a>Element "RaiseEvent" jest już zadeklarowany

`RaiseEvent`W deklaracji zdarzenia niestandardowego pojawił się więcej niż jedna deklaracja. `RaiseEvent`Deklaracja deklaruje procedurę służącą do podniesienia zdarzenia.  
  
 **Identyfikator błędu:** BC31129  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `RaiseEvent` .
