# <a name="39raiseevent39-is-already-declared"></a>&#39; RaiseEvent &#39; jest już zadeklarowana
Więcej niż jeden `RaiseEvent` deklaracji znajdowały się w deklaracji zdarzenia niestandardowego. A `RaiseEvent` deklaracji deklaruje procedurę wywoływała zdarzenie.  
  
 **Identyfikator błędu:** BC31129  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `RaiseEvent` instrukcji.
