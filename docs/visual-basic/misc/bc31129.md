# <a name="raiseevent-is-already-declared"></a>"RaiseEvent" jest już zadeklarowany.
Więcej niż jeden `RaiseEvent` deklaracji pojawiły się w deklaracji zdarzenia niestandardowego. A `RaiseEvent` deklaracji deklaruje procedurę używane w celu podniesienia zdarzenia.  
  
 **Identyfikator błędu:** BC31129  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `RaiseEvent` instrukcji.
