# <a name="lttypenamegt-is-an-unsupported-type"></a>"&lt;typename&gt;" ma nieobsługiwany typ
Określony typ nie może być reprezentowany przez Visual Studio.  
  
 **Identyfikator błędu:** BC30649  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj obsługiwanego typu.  
  
-   Tworzenie nowego typu.
