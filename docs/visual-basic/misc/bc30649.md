# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;Właściwość TypeName&gt; &#39; ma nieobsługiwany typ
Określony typ nie może być reprezentowany przez Visual Studio.  
  
 **Identyfikator błędu:** BC30649  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj obsługiwanego typu.  
  
-   Tworzenie nowego typu.
