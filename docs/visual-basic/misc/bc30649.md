# <a name="typename-is-an-unsupported-type"></a>" \<typename> " jest nieobsługiwanym typem
Określony typ nie może być reprezentowany przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30649  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj obsługiwanego typu.  
  
- Utwórz nowy typ.
