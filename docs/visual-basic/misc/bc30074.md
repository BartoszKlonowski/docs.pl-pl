# <a name="constant-cannot-be-the-target-of-an-assignment"></a>Stała nie może być elementem docelowym przypisania
Stała występuje w kontekście, który przypisuje do niego wartość. Tylko zapisywalne zmienne, właściwości i elementy tablicy mogą mieć przypisane wartości podczas wykonywania.  
  
 **Identyfikator błędu:** BC30074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamień stałą na jedną zapisywalną zmienną, właściwość lub element tablicy.
