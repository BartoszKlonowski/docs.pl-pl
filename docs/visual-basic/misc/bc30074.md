# <a name="constant-cannot-be-the-target-of-an-assignment"></a>Stała nie może być elementem docelowym przypisania
Stała występuje w kontekście, który przypisuje wartość do niego. Zapisywalny zmiennych, właściwości i elementy tablicy może mieć wartości przypisane do nich podczas wykonywania.  
  
 **Identyfikator błędu:** BC30074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamień na stałą pojedynczej zmiennej zapisywalny, właściwości lub elementu tablicy.
