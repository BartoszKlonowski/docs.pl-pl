# <a name="constant-cannot-be-the-target-of-an-assignment"></a>Stała nie może być elementem docelowym przypisania
Stała występuje w kontekście, który przypisuje wartość do niego. Zapisywalny zmiennych, właściwości i elementów tablicy może mieć wartości przypisanego podczas wykonywania.  
  
 **Identyfikator błędu:** BC30074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Stała Zamień pojedynczą zmienną zapisywalny, właściwość lub element tablicy.
