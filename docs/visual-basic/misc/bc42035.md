# <a name="bad-guid-format"></a>Nieprawidłowy format identyfikatora GUID

Identyfikator GUID został dostarczony z nieprawidłowym formatem.  
  
 **Identyfikator błędu:** BC42035  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź format identyfikatora GUID.
