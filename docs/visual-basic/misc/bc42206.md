# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Przekroczono maksymalną liczbę ostrzeżeń
Maksymalna liczba ostrzeżeń kompilacji to 100. Kompilator nakłada ten limit, aby zmniejszyć rozmiar błąd i ostrzeżenie raportów.  
  
 Domyślnie ten komunikat jest ostrzeżenie. Aby uzyskać więcej informacji na ukrywanie ostrzeżenia lub traktowanie ostrzeżeń jako błędy, zobacz [Konfigurowanie ostrzeżeń w Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identyfikator błędu:** BC42206  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przejrzyj inne ostrzeżenia wygenerowane przez tej kompilacji, aby rozwiązać jak najwięcej. Jeśli kod źródłowy zawiera więcej niż 100 ostrzeżenia, mogą występować więcej ostrzeżeń w następnej kompilacji.
