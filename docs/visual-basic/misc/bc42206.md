# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Przekroczono maksymalną liczbę ostrzeżeń
Maksymalna liczba ostrzeżeń kompilacji wynosi 100. Kompilator nakłada ten limit, aby zmniejszyć rozmiar błąd i ostrzeżenie raportów.  
  
 Domyślnie ta wiadomość jest ostrzeżenie. Aby uzyskać więcej informacji na temat ukrywania ostrzeżenia lub traktowanie ostrzeżeń jako błędy, zobacz [Konfigurowanie ostrzeżeń w języku Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identyfikator błędu:** BC42206  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przejrzyj ostrzeżenia, generowane przez tej kompilacji i rozwiązać jak najwięcej. Jeśli kod źródłowy zawiera więcej niż 100 ostrzeżenia, mogą występować więcej ostrzeżeń w następnej kompilacji.
