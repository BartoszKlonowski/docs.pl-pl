# <a name="attribute-member-39ltmembernamegt39-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-39public39"></a>Element członkowski atrybutu &#39; &lt;membername&gt;&#39; nie może być elementem docelowym przypisania, ponieważ nie jest zadeklarowany jako &#39; Publicznego &#39;
Nastąpiła próba można przypisać wartości do prywatnego elementu członkowskiego atrybutu.  
  
 **Identyfikator błędu:** BC31511  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń przypisanie.  
  
2.  Jeśli przy użyciu atrybutów niestandardowych, które utworzono, zmień modyfikator dostępu elementu członkowskiego atrybut do `Public`.
