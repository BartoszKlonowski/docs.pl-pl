<span data-ttu-id="a34a2-103">Aby zadeklarować wystąpienia typu nazwane i anonimowe za pomocą inicjatora obiektów, lista pól lub właściwości i ich wartości początkowe należy ująć w nawiasy klamrowe ({i}).</span><span class="sxs-lookup"><span data-stu-id="a34a2-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>
