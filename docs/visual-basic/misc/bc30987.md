<span data-ttu-id="89353-103">Aby zadeklarować wystąpienie typu nazwanego lub anonimowego za pomocą inicjatora obiektów, należy ująć listę pól lub właściwości oraz ich wartości początkowe w nawiasach klamrowych ({i}).</span><span class="sxs-lookup"><span data-stu-id="89353-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```vb

Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}
```  
