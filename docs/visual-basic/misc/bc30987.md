# <a name="-expected"></a><span data-ttu-id="529ba-102">"{" Oczekiwano</span><span class="sxs-lookup"><span data-stu-id="529ba-102">'{' expected</span></span>
<span data-ttu-id="529ba-103">Aby zadeklarować wystąpienia typu nazwane i anonimowe za pomocą inicjatora obiektów, lista pól lub właściwości i ich wartości początkowe należy ująć w nawiasy klamrowe ({i}).</span><span class="sxs-lookup"><span data-stu-id="529ba-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```

Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}
```  
