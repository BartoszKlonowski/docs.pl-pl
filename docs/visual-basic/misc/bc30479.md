# <a name="shared-sub-new-cannot-have-any-parameters"></a>Udostępnione "Sub New" nie może mieć żadnych parametrów
Parametr lub parametry zostały uwzględnione w udostępnionej `Sub New` instrukcję, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30479  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń parametr lub parametry.
