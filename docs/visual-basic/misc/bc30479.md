# <a name="shared-sub-new-cannot-have-any-parameters"></a>Udostępniony element "Sub New" nie może mieć żadnych parametrów
W instrukcji udostępnionej dołączono parametr lub parametry `Sub New` , które są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30479  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń parametr lub parametry.
