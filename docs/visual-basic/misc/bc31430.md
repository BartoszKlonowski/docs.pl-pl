# <a name="expression-of-type-lttypename1gt-can-never-be-of-type-lttypename2gt"></a><span data-ttu-id="01718-102">Wyrażenie typu "&lt;typename1&gt;"nigdy nie może być typu"&lt;typename2&gt;"</span><span class="sxs-lookup"><span data-stu-id="01718-102">Expression of type '&lt;typename1&gt;' can never be of type '&lt;typename2&gt;'</span></span>
<span data-ttu-id="01718-103">A `TypeOf`... `Is` wyrażenie sprawdza zmienną obiektu odwołania do typu danych, który nie może on przechowywać.</span><span class="sxs-lookup"><span data-stu-id="01718-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="01718-104">W niektórych przypadkach kompilator można określić, że `TypeOf`... `Is` testu może wyłącznie zakończyć się niepowodzeniem, na przykład, jeśli nie ma żadnych relacji dziedziczenia między dwoma klasami.</span><span class="sxs-lookup"><span data-stu-id="01718-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="01718-105">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="01718-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="01718-106">Ponieważ <xref:System.Windows.Forms.Form> i <xref:System.Array> są całkowicie niepowiązane typów, mogą kompilatora ustalić, który `TypeOf`... `Is` zwraca wyrażenie `False` dla dowolnej wartości `refVar`.</span><span class="sxs-lookup"><span data-stu-id="01718-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="01718-107">**Identyfikator błędu:** BC31430</span><span class="sxs-lookup"><span data-stu-id="01718-107">**Error ID:** BC31430</span></span>
