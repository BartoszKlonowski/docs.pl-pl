# <a name="expression-of-type-typename1-can-never-be-of-type-typename2"></a><span data-ttu-id="10fe0-102">Wyrażenie typu '\<typename1 >' nigdy nie może być typu '\<typename2 >'</span><span class="sxs-lookup"><span data-stu-id="10fe0-102">Expression of type '\<typename1>' can never be of type '\<typename2>'</span></span>
<span data-ttu-id="10fe0-103">A `TypeOf`... `Is` wyrażenie sprawdza zmienną obiektu odwołania do typu danych, który nie może on przechowywać.</span><span class="sxs-lookup"><span data-stu-id="10fe0-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="10fe0-104">W niektórych przypadkach kompilator można określić, że `TypeOf`... `Is` testu może wyłącznie zakończyć się niepowodzeniem, na przykład, jeśli nie ma żadnych relacji dziedziczenia między dwoma klasami.</span><span class="sxs-lookup"><span data-stu-id="10fe0-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="10fe0-105">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="10fe0-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="10fe0-106">Ponieważ <xref:System.Windows.Forms.Form> i <xref:System.Array> są całkowicie niepowiązane typów, mogą kompilatora ustalić, który `TypeOf`... `Is` zwraca wyrażenie `False` dla dowolnej wartości `refVar`.</span><span class="sxs-lookup"><span data-stu-id="10fe0-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="10fe0-107">**Identyfikator błędu:** BC31430</span><span class="sxs-lookup"><span data-stu-id="10fe0-107">**Error ID:** BC31430</span></span>
