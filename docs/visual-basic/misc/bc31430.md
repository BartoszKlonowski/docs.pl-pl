<span data-ttu-id="2ff7a-103">A `TypeOf`... `Is` wyrażenie testów zmiennej odwołania do obiektu z typem danych, które nie może on zawierać.</span><span class="sxs-lookup"><span data-stu-id="2ff7a-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="2ff7a-104">W niektórych przypadkach kompilator można określić, który `TypeOf`... `Is` testu może tylko zakończyć się niepowodzeniem, na przykład, jeśli nie ma żadnych zależności dziedziczenia między dwiema klasami.</span><span class="sxs-lookup"><span data-stu-id="2ff7a-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>

 `End If`  
  
 <span data-ttu-id="2ff7a-106">Ponieważ <xref:System.Windows.Forms.Form> i <xref:System.Array> są całkowicie niepowiązanych typów, kompilator może określić, że `TypeOf`... `Is` zwraca wyrażenie `False` dla dowolnej wartości `refVar`.</span><span class="sxs-lookup"><span data-stu-id="2ff7a-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>
