# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39; &lt;nazwa_zmiennej&gt;&#39; nie jest zmienną lokalną ani parametrem i dlatego nie można użyć jako &#39; CATCH &#39; Zmienna
Zmienne w `Try...Catch...Finally` instrukcji musi być zadeklarowany jako lokalnie lub być parametrami bieżącej procedury.  
  
 **Identyfikator błędu:** BC31082  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Deklarowanie zmiennych lokalnych lub parametrów `Try...Catch...Finally` instrukcje.
