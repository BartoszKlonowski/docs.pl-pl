# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>'\<nazwa_zmiennej >' nie jest zmienną lokalną ani parametrem i dlatego nie można użyć jako zmiennej instrukcji' Catch'
Zmienne w `Try...Catch...Finally` instrukcji musi być zadeklarowany lokalnie lub być parametrów dla bieżącej procedurze.  
  
 **Identyfikator błędu:** BC31082  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zadeklaruj zmienne lokalne lub parametrów `Try...Catch...Finally` instrukcji.
