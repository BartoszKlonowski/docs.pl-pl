# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Składowe w strukturze nie mogą być zadeklarowane jako " \<specifier> "

Użyto specyfikatora, który nie jest prawidłowy w elemencie członkowskim w `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30435  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
