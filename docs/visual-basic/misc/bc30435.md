# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Członkowie w strukturze nie można zadeklarować &#39; &lt;Specyfikator&gt;&#39;
Zostały użyte specyfikator, który jest nieprawidłowy dla elementu członkowskiego w `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30435  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
