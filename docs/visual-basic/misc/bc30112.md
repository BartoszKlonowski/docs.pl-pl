# <a name="39lttypenamegt39-is-a-namespace-and-cannot-be-used-as-an-expression"></a>&#39; &lt;typename&gt;&#39; jest przestrzenią nazw i nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wymagane jest wyrażenie. Wyrażenie musi składać się z zmienne, stałe, literały, właściwości, oraz `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30112  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłową.
