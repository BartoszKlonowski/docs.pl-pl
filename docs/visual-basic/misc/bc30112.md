# <a name="typename-is-a-namespace-and-cannot-be-used-as-an-expression"></a>element " \<typename> " jest przestrzenią nazw i nie można go używać jako wyrażenia
Nazwa typu występuje, gdy wyrażenie jest wymagane. Wyrażenie musi zawierać zmienne, stałe, literały, właściwości i `Function` wywołania procedur.  
  
 **Identyfikator błędu:** BC30112  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nazwę typu i Skonstruuj wyrażenie przy użyciu prawidłowych elementów.
