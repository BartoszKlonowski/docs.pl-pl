# <a name="39ltkeywordgt39-is-not-valid-within-a-structure"></a>&#39;&lt;słowo kluczowe&gt; &#39; nie jest prawidłowy wewnątrz struktury
Struktury są typy wartości, nie typy referencyjne. Struktura nie jest wystąpieniem utworzone z klasy, więc `Me`, `MyClass`, i `MyBase` słowa kluczowe są bezużyteczne w strukturze.  
  
 **Identyfikator błędu:** BC30044  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień do klasy, struktury lub Usuń słowo kluczowe z procedury.
