# <a name="keyword-is-not-valid-within-a-structure"></a>'\<— słowo kluczowe >' jest nieprawidłowa w ramach struktury
Struktury są typami wartości, nie typy odwołań. Struktura nie jest wystąpieniem utworzone na podstawie klasy, więc `Me`, `MyClass`, i `MyBase` słowa kluczowe są bezużyteczne w strukturze.  
  
 **Identyfikator błędu:** BC30044  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień strukturę do klasy, lub Usuń słowo kluczowe z procedury.
