# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Klasy zarządzane pochodną klasę modelu COM nie można wywołać z późnym wiązaniem.
Podjęto próbę wywołania późnego wiązania zarządzanej klasy pochodzącej od klasy COM; takie połączenia nie są obsługiwane.  
  
### <a name="to-correct-the-problem"></a>Aby rozwiązać ten problem  
  
-   Utwórz wywołanie z wczesnym wiązaniem.
