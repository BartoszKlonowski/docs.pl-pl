# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Klasy zarządzane pochodzące od klasy COM, nie można wywołać późnym wiązaniem.
Nastąpiła próba wykonania wywołania późnym wiązaniem zarządzanej klasy pochodzącej od klasy COM; takie połączenia nie są obsługiwane.  
  
### <a name="to-correct-the-problem"></a>Aby rozwiązać ten problem  
  
-   Utwórz wywołania z wczesnym wiązaniem.
