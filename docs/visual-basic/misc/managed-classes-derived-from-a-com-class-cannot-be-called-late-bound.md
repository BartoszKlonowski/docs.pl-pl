# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Klasy zarządzane pochodne od klasy COM nie mogą być wywoływane z późnym wiązaniem.

Podjęto próbę wykonania wywołania z późnym wiązaniem do klasy zarządzanej pochodzącej od klasy COM; takie wywołania nie są obsługiwane.

## <a name="to-correct-the-problem"></a>Aby rozwiązać ten problem

Utwórz wczesne powiązanie wywołania.
