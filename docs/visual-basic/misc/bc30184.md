# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>Element "End Enum" musi być poprzedzony odpowiadającym mu elementem "enum"
`End Enum`Instrukcja występuje bez odpowiedniej `Enum` instrukcji. `End Enum`musi być poprzedzona odpowiadającą jej `Enum` instrukcją.  
  
 **Identyfikator błędu:** BC30184  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że poprzednia `Enum` instrukcja nie jest błędnie wpisana lub jest nieprawidłowa.  
  
2. Upewnij się, że `Enum` elementy członkowskie są poprawnie sformatowane.
