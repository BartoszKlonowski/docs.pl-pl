# <a name="39end-enum39-must-be-preceded-by-a-matching-39enum39"></a>&#39; End Enum &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; Wyliczenia &#39;
`End Enum` Występuje instrukcja bez odpowiadającego `Enum` instrukcji. `End Enum`musi być poprzedzona odpowiadającą `Enum` instrukcji.  
  
 **Identyfikator błędu:** BC30184  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że występuje przed `Enum` instrukcja nie jest błędnie lub są one nieprawidłowe.  
  
2.  Upewnij się, że `Enum` elementy członkowskie są prawidłowo sformatowane.
