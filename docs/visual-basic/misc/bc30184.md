# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>"End Enum" musi być poprzedzona pasującego "Enum"
`End Enum` Instrukcji odbywa się bez odpowiadającego `Enum` instrukcji. `End Enum` musi być poprzedzona odpowiadającą `Enum` instrukcji.  
  
 **Identyfikator błędu:** BC30184  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że występuje przed `Enum` instrukcja nie jest błędnie lub są one nieprawidłowe.  
  
2. Upewnij się, że `Enum` elementy członkowskie są poprawnie sformatowane.
