# <a name="3939-unexpected"></a>&#39;(&#39; nieoczekiwany
'(' Unexpected. Tablice typów ogólnych bez wystąpień nie są dozwolone.  
  
 Visual Basic nie można skompilować tablicy, chyba że jest określonego typu danych. Nie można użyć parametru typu danych typu ogólnego jako typ danych elementu tablicy.  
  
 **Identyfikator błędu:** BC32095  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli musisz użyć tablicy, należy zadeklarować jego typu danych. Nie można użyć parametru typu danych.  
  
-   Jeśli potrzebujesz grupę elementów typu danych, który ma być dostarczona dla parametru typu danych muszą używać kolekcji zamiast tablicy.
