# <a name="-unexpected"></a>'(' Unexpected
'(' Unexpected. Tablice bez wystąpień typów ogólnych są niedozwolone.  
  
 Visual Basic nie można skompilować tablicę, chyba że jest on na określony typ danych. Parametr typu ogólnego typu danych nie można użyć jako typ danych tablicy.  
  
 **Identyfikator błędu:** BC32095  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli zachodzi potrzeba skorzystaj z tablicy, należy zadeklarować ją będzie na określony typ danych. Nie można używać parametrów typu danych.  
  
-   Jeśli potrzebujesz grupy elementów typu danych, które mają zostać dostarczone dla parametru typu danych, muszą używać kolekcji, zamiast tablicy.
