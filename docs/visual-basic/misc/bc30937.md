<span data-ttu-id="15d52-103">Procedura lub właściwość próbuje zaimplementować procedura lub właściwość zdefiniowana w zaimplementowany interfejs, ale kompilator znajdzie więcej niż jedna wersja procedury interfejsu lub właściwość o tej samej nazwie i sygnaturze.</span><span class="sxs-lookup"><span data-stu-id="15d52-103">A procedure or property attempts to implement a procedure or property defined in an implemented interface, but the compiler finds more than one version of the interface procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="15d52-104">Ten błąd może wystąpić w sytuacji, za pomocą typów ogólnych skonstruowany, jak pokazują następujące deklaracje szkielet.</span><span class="sxs-lookup"><span data-stu-id="15d52-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```

    Sub doSomething(ByVal inputValue As String)  
    Sub doSomething(ByVal inputValue As t)  
End Class  
Public Class implementingClass  
    Implements baseInterface(Of String)  
    Sub doSomething(ByVal inputValue As String) _  
        Implements baseInterface(Of String).doSomething  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="15d52-105">Ponieważ `implementingClass` implementuje `baseInterface` dostarczenie `String` na jego parametr typu `t`, dwie wersje `doSomething` w `baseInterface` przyjmą identycznych podpisach w miarę `implementingClass` dotyczy.</span><span class="sxs-lookup"><span data-stu-id="15d52-105">Because `implementingClass` implements `baseInterface` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseInterface` take on identical signatures as far as `implementingClass` is concerned.</span></span> <span data-ttu-id="15d52-106">W rezultacie kompilator nie może określić wersji do zaimplementowania.</span><span class="sxs-lookup"><span data-stu-id="15d52-106">As a result, the compiler cannot determine which version to implement.</span></span>
