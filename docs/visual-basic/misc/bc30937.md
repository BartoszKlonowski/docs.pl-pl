 <span data-ttu-id="8ae68-104">Ten błąd może wystąpić w sytuacji, za pomocą typów ogólnych skonstruowany, jak pokazują następujące deklaracje szkielet.</span><span class="sxs-lookup"><span data-stu-id="8ae68-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>
