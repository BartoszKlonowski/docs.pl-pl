# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "return" nie są prawidłowe w oknie bezpośrednim

`Return`Instrukcja wykonuje rozgałęzianie i nie jest dozwolona w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30147  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie wystawiaj `Return` instrukcji w oknie **bezpośrednim** .
