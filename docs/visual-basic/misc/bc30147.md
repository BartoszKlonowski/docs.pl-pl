# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Return" nie są prawidłowe w oknie bezpośrednim
`Return` Instrukcji wykonuje rozgałęzianie i nie jest dozwolona w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30147  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie wydają `Return` instrukcji w **bezpośrednie** okna.
