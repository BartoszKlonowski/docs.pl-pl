# <a name="disk-not-ready"></a>Dysk nie jest gotowy
Określonej stacji dysków lub stację jest otwarty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź określony dysk i włożenie dysku, jeśli to konieczne.  
  
2.  Zamknij stację.
