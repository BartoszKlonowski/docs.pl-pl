# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>"&lt;nazwa&gt;"jest niejednoznaczny między deklaracjami w modułach"&lt;modulename1&gt;"i"&lt;modulename2&gt;"
Podano nazwę, która jest niejednoznaczny i dlatego jest w konflikcie z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; odróżnić nazwy musi samodzielnie.  
  
 **Identyfikator błędu:** BC30562  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
