# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>'\<name >' jest niejednoznaczny między deklaracjami w modułach\<modulename1 > 'i'\<modulename2 > '
Podano nazwę, która jest niejednoznaczny i dlatego jest w konflikcie z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; odróżnić nazwy musi samodzielnie.  
  
 **Identyfikator błędu:** BC30562  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Pełnej nazwy.
