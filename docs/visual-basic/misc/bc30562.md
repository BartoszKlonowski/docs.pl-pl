# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>element " \<name> " jest niejednoznaczny między deklaracjami w modułach " \<modulename1> " i " \<modulename2> "

Podana nazwa jest niejednoznaczna i dlatego powoduje konflikt z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; należy samodzielnie odróżnić nazwy.  
  
 **Identyfikator błędu:** BC30562  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- W pełni Zakwalifikuj nazwę.
