# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39; &lt;nazwa&gt;&#39; jest niejednoznaczny między deklaracjami w modułach &#39;&lt; modulename1&gt;&#39; i &#39;&lt; modulename2&gt;&#39;
Podana nazwa jest niejednoznaczny i dlatego powoduje konflikt z inną nazwą. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilatora nie ma żadnych reguł rozwiązywania konfliktów; nazwy musi odróżniania samodzielnie.  
  
 **Identyfikator błędu:** BC30562  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
