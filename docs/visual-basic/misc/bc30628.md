# <a name="structures-cannot-have-inherits-statements"></a>Struktury nie mogą zawierać instrukcji "Inherits"
W odróżnieniu od klas struktury nie obsługują dziedziczenia.  
  
 **Identyfikator błędu:** BC30628  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji ze struktury.  
  
-   Ponownie zaprojektować aplikację, aby zamiast tego użyj klas.
