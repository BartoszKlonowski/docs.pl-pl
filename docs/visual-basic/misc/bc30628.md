# <a name="structures-cannot-have-39inherits39-statements"></a>Struktury nie mogą zawierać &#39;Inherits&#39; — instrukcje
W przeciwieństwie do klasy struktury nie obsługują dziedziczenia.  
  
 **Identyfikator błędu:** BC30628  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji ze struktury.  
  
-   Zmodyfikowanie aplikacji zamiast tego użyć klasy.
