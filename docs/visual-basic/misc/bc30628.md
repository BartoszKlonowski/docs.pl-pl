# <a name="structures-cannot-have-inherits-statements"></a>Struktury nie mogą zawierać instrukcji "Inherits"

W przeciwieństwie do klas, struktury nie obsługują dziedziczenia.  
  
 **Identyfikator błędu:** BC30628  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Inherits` instrukcję ze struktury.  
  
- Zaprojektuj aplikację w celu użycia klas zamiast tego.
