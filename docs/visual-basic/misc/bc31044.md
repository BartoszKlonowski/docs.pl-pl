# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Zdarzenia zadeklarowane z klauzulą "As" muszą mieć typ delegata
Określono typ, który nie jest obiektem delegowanym.  
  
 **Identyfikator błędu:** BC31044  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `As` klauzulę.  
  
2. Określ delegata z `As` klauzulą.
