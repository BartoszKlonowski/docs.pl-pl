# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Zdarzenia zadeklarowane z &#39; Jako &#39; Klauzula musi mieć typ delegowany
Określono typ nie jest obiektem delegowanym.  
  
 **Identyfikator błędu:** BC31044  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `As` klauzuli.  
  
2.  Określ delegata z `As` klauzuli.
