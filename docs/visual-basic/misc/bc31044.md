# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Zdarzenia zadeklarowane za pomocą klauzuli "As" musi mieć typ delegata
Określono typ nie jest obiektem delegowanym.  
  
 **Identyfikator błędu:** BC31044  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `As` klauzuli.  
  
2. Określ pełnomocnika z `As` klauzuli.
