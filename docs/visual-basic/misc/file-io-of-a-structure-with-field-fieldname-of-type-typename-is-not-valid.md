# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>Plik we/wy dla struktury zawierającej pole &#39; &lt;fieldname&gt; &#39; typu &#39; &lt;typename&gt; &#39; jest nieprawidłowy
Struktury, które próbują zapisać do pliku zawiera nieobsługiwanego typu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź strukturę, aby określić, gdzie występuje nieobsługiwany typ, a następnie zmień go do prawidłowego typu.
