# <a name="file-io-of-a-structure-with-field-ltfieldnamegt-of-type-lttypenamegt-is-not-valid"></a>Plik we/wy struktury z polem "&lt;fieldname&gt;"typu"&lt;typename&gt;" jest nieprawidłowa
Struktura, którą próbujesz zapisać do pliku zawiera nieobsługiwanego typu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź strukturę, aby określić, gdzie występuje nieobsługiwany typ, a następnie zmienić je do prawidłowego typu.
