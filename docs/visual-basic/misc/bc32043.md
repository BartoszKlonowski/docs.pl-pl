# <a name="too-many-type-arguments-to-genericmethodname"></a>Za dużo argumentów typu dla '\<genericMethodName >'
Metody ogólnej zostało wywołane więcej argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody rodzajowej, należy podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC32043  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń z listy argumentów typu argumentów typu, tak, że jest on dostępny dla każdego parametru typu dla metody rodzajowe, wywoływane.
