# <a name="too-many-type-arguments-to-genericmethodname"></a>Za dużo argumentów typu dla " \<genericMethodName> "

Metoda ogólna została wywołana z więcej argumentów typu niż parametry typu.  
  
 Po wywołaniu metody ogólnej należy podać jeden argument typu dla każdego parametru typu zdefiniowanego przez tę metodę.  
  
 **Identyfikator błędu:** BC32043  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń argumenty typu z listy argumentów typu, tak aby istniała jeden dla każdego parametru typu w wywołaniu metody ogólnej.
