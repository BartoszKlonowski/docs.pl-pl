# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="1c666-102">Ustawienie Option Strict on wymaga zadeklarowania każdego parametru wyrażenia lambda z klauzulą "As", jeśli nie można wywnioskować jej typu</span><span class="sxs-lookup"><span data-stu-id="1c666-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>

<span data-ttu-id="1c666-103">Zadeklarowano parametr w wyrażeniu lambda bez użycia `As` klauzuli with `Option Strict` on.</span><span class="sxs-lookup"><span data-stu-id="1c666-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```vb  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="1c666-104">Poprzednia deklaracja jest prawidłowa, jeśli typ `n` można wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="1c666-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="1c666-105">Na przykład, Jeśli przypiszesz poprzednie wyrażenie lambda do delegata funkcji, `Del` :</span><span class="sxs-lookup"><span data-stu-id="1c666-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="1c666-106">Teraz `n` można wywnioskować typ z parametru `p` :</span><span class="sxs-lookup"><span data-stu-id="1c666-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```vb  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="1c666-107">**Identyfikator błędu:** BC36642</span><span class="sxs-lookup"><span data-stu-id="1c666-107">**Error ID:** BC36642</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1c666-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="1c666-108">To correct this error</span></span>  
  
- <span data-ttu-id="1c666-109">Dodaj `As` klauzulę do deklaracji parametru:</span><span class="sxs-lookup"><span data-stu-id="1c666-109">Add an `As` clause to the parameter declaration:</span></span>

    ```vb  
    Dim increment3 = Function (n As Integer) n + 1  
    ```
