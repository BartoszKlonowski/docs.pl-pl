<span data-ttu-id="684c5-103">Zadeklarowano parametru w wyrażeniu lambda bez użycia `As` klauzuli z `Option Strict` na.</span><span class="sxs-lookup"><span data-stu-id="684c5-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>

' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="684c5-104">Obowiązuje w poprzedniej deklaracji Jeśli typ `n` można wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="684c5-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="684c5-105">Na przykład, jeśli przypisujesz poprzedniego wyrażenia lambda do delegata funkcji `Del`:</span><span class="sxs-lookup"><span data-stu-id="684c5-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="684c5-106">Teraz typ `n` można wywnioskować z parametru `p`:</span><span class="sxs-lookup"><span data-stu-id="684c5-106">Now the type of `n` can be inferred from parameter `p`:</span></span>

```
