# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="fbe9a-102">Ustawienie Option Strict On wymaga, aby każdy parametr wyrażenia lambda zadeklarowany z użyciem klauzuli "As", jeśli nie można wywnioskować jej typu</span><span class="sxs-lookup"><span data-stu-id="fbe9a-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>
<span data-ttu-id="fbe9a-103">Zadeklarowano parametru w wyrażeniu lambda bez użycia `As` klauzuli z `Option Strict` na.</span><span class="sxs-lookup"><span data-stu-id="fbe9a-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="fbe9a-104">Obowiązuje w poprzedniej deklaracji Jeśli typ `n` można wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="fbe9a-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="fbe9a-105">Na przykład, jeśli przypisujesz poprzedniego wyrażenia lambda do delegata funkcji `Del`:</span><span class="sxs-lookup"><span data-stu-id="fbe9a-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="fbe9a-106">Teraz typ `n` można wywnioskować z parametru `p`:</span><span class="sxs-lookup"><span data-stu-id="fbe9a-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="fbe9a-107">**Identyfikator błędu:** BC36642</span><span class="sxs-lookup"><span data-stu-id="fbe9a-107">**Error ID:** BC36642</span></span>

