# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>Instrukcja "End Sub" musi być pierwszą instrukcją w wierszu
`End Sub`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End Sub`musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
