# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>"End Sub" musi być pierwszą instrukcją w wierszu
`End Sub` Instrukcji następuje separatorem instrukcji dwukropek (:). `End Sub` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
