# <a name="39end-sub39-must-be-the-first-statement-on-a-line"></a>&#39; End Sub &#39; musi być pierwszą instrukcją w wierszu
`End Sub` Instrukcji następuje separatora instrukcji dwukropka (:). `End Sub`musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
