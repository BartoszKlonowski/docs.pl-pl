# <a name="inheriting-from-systemltclassnamegt-is-not-valid"></a>Dziedziczenie z "System. &lt;classname&gt;"jest nieprawidłowa
Klasa nie może dziedziczyć z `System.Array`, `System.Delegate`, `System.Enum`, lub `System.ValueType`.  
  
 **Identyfikator błędu:** BC30015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji lub zmienić je, aby były dziedziczone z prawidłową klasę podstawową.
