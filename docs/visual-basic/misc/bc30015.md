# <a name="inheriting-from-39systemltclassnamegt39-is-not-valid"></a>Dziedziczenie z &#39;systemu. &lt;classname&gt; &#39; jest nieprawidłowy
Nie można dziedziczyć po klasie `System.Array`, `System.Delegate`, `System.Enum`, lub `System.ValueType`.  
  
 **Identyfikator błędu:** BC30015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji lub zmień, aby dziedziczyć z prawidłową klasę podstawową.
