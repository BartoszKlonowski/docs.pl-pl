# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Podczas obliczania tego wyrażenia wystąpiło dzielenie przez zero
W ocenie tego wyrażenia podjęto próbę podzielenia liczby przez zero.  
  
 **Identyfikator błędu:** BC30542  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź stałą i wartości zmiennych, aby określić, jak zero występuje w mianowniku wyrażenia.
