# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Podczas szacowania tego wyrażenia wystąpiło dzielenie przez zero
Podczas obliczania tego wyrażenia, nastąpiła próba dzielenia przez zero liczbą.  
  
 **Identyfikator błędu:** BC30542  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź wartości zmiennych i stałych ustalenie, jak zero występuje mianownik w wyrażeniu.
