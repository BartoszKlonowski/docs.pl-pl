# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39; NotOverridable &#39; Nie można określić dla metod, które nie zastępują innej metody
Właściwości i metody są `NotOverridable` domyślnie. `NotOverridable` Modyfikatora można używać tylko metod, które zastępują innej właściwości lub metody.  
  
 **Identyfikator błędu:** BC31088  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `NotOverridable` modyfikator z właściwości i metody, które nie zastępują innego elementu członkowskiego.
