# <a name="notoverridable-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>"NotOverridable" nie można określić dla metody, które nie zastępują innej metody
Właściwości i metody są `NotOverridable` domyślnie. `NotOverridable` Modyfikator należy używać tylko na metody, które zastępują innej właściwości lub metody.  
  
 **Identyfikator błędu:** BC31088  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `NotOverridable` modyfikator z właściwości i metody, które nie zastępują innego członka.
