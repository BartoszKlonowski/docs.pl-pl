# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-39shared39"></a>Metody lub zdarzenia, które implementują interfejs, który nie można zadeklarować elementów członkowskich &#39; Udostępnione &#39;
Podjęto próbę zadeklarować jako `Shared` metody lub zdarzenia, które implementuje element członkowski. Nie może zostać wyznaczony metod i zdarzeń implementowana w klasie `Shared` lub `Private`, z wyjątkiem klasy dziedziczonych.  
  
 **Identyfikator błędu:** BC30505  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Shared` — słowo kluczowe.
