# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>"Variant" nie jest już obsługiwanym typem; Zamiast tego użyj typ "Object"
`Object` Zastępuje typ danych `Variant`.  
  
 **Identyfikator błędu:** BC30804  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj element jako `Object`.
