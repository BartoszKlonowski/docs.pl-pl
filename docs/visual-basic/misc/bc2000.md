# <a name="initialization-error"></a>Błąd inicjowania
Wystąpił błąd podczas inicjowania programu. Może to wynikać z anulowania instalacji, gdy nadal trwa.  
  
 **Identyfikator błędu:** BC2000  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ponowna próba instalacji.
