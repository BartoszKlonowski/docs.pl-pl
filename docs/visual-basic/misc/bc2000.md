# <a name="initialization-error"></a>Błąd podczas inicjowania
Wystąpił błąd podczas inicjowania programu. Może to być wynikiem anulowania instalacji jest nadal w toku.  
  
 **Identyfikator błędu:** BC2000  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Następnie ponów próbę instalacji.
