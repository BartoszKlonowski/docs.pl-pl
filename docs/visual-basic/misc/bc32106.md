# <a name="type-argument-39lttypeargumentnamegt39-does-not-satisfy-the-39class39-constraint-for-type-parameter-39lttypeparameternamegt39"></a>Argument typu &#39; &lt;typeargumentname&gt; &#39; nie spełnia &#39;klasy&#39; ograniczenia dla parametru typu &#39; &lt;typeparametername&gt;&#39;
Argument typu do typu ogólnego nie spełnia ograniczenia typu odwołania na jego odpowiedniego parametru typu.  
  
 Lista ograniczeń nakłada się na typ argumentu przekazanego do parametru typu wymagania. Jeśli w lista ograniczeń nie zawiera żadnych konkretną klasę lub interfejs, można skonfigurować ogólne wymagania, określając jedną z następujących czynności:  
  
-   Argument typu musi być typem wartości (obejmują [struktury (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) ograniczenia)  
  
-   Argument typu musi być typem referencyjnym (obejmują [klasy (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) ograniczenia)  
  
 Nie można określić zarówno `Structure` i `Class` dla tego samego typu parametru, a nie można określić jeden więcej niż jeden raz.  
  
 **Identyfikator błędu:** BC32106  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz w nim argumentu typu dowolnego typu referencyjnego.
