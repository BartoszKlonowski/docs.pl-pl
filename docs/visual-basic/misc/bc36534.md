# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Nie można przekonwertować wyrażenia na drzewo wyrażenia

Większość wyrażeń można przekonwertować na drzewa wyrażeń, ale niektóre z nich nie mogą. Ten błąd wskazuje, że podjęto próbę przekonwertowania wyrażenia, którego nie można przekonwertować.  
  
 **Identyfikator błędu:** BC36534
