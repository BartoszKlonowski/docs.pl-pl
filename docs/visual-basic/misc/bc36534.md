# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Drzewo wyrażenia nie można przekonwertować wyrażenia
Większość wyrażenia można przekonwertować na drzew wyrażeń, ale niektóre nie. Ten błąd wskazuje, podjęta próba do skonwertowania wyrażenia, na których nie można przekonwertować.  
  
 **Identyfikator błędu:** BC36534
