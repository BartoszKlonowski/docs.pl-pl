# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Nie można przekonwertować wyrażenia na drzewo wyrażenia
Większość wyrażenia mogą być konwertowane na drzewa wyrażeń, ale niektóre nie. Ten błąd wskazuje podejmowana próba można przekonwertować wyrażenia nie może zostać przekonwertowany.  
  
 **Identyfikator błędu:** BC36534
