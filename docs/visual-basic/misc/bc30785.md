# <a name="parameter-specifier-is-duplicated"></a>Specyfikator parametru jest zduplikowany

Specyfikator parametru jest stosowany więcej niż raz do parametru.  
  
 **Identyfikator błędu:** BC30785  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowane specyfikatory.
