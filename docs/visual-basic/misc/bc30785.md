# <a name="parameter-specifier-is-duplicated"></a>Specyfikator parametru jest zduplikowana
Specyfikator parametru zastosowano więcej niż jeden raz z parametrem.  
  
 **Identyfikator błędu:** BC30785  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowane specyfikatorów.
