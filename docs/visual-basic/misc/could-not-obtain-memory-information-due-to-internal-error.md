# <a name="could-not-obtain-memory-information-due-to-internal-error"></a>Nie można uzyskać informacji o pamięci z powodu błędu wewnętrznego
Wywołanie do jednej z właściwości informacji o pamięci `My.Computer.Info` obiektu nie powiodło się.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Try...Catch` bloku wokół wywołanie do właściwości informacji o pamięci `My.Computer.Info` obiektu.
