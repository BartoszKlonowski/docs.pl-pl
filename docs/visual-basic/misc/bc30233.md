# <a name="specifier-is-not-valid-on-a-constant-declaration"></a>element " \<specifier> " nie jest prawidłowy w deklaracji stałej

`Const`Instrukcja zawiera nieprawidłowe słowo kluczowe. `Const`Instrukcja może zawierać tylko `Friend` `Private` słowa kluczowe,, `Protected` , `Public` lub `Shadows` .  
  
 **Identyfikator błędu:** BC30233  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że słowo kluczowe jest poprawnie napisane.  
  
2. Usuń nieprawidłowe słowo kluczowe z `Const` instrukcji.
