# <a name="specifier-is-not-valid-on-a-constant-declaration"></a>'\<specyfikator >' nie jest prawidłowy w deklaracji stałej
A `Const` instrukcja zawiera nieprawidłowe słowo kluczowe. A `Const` instrukcji może zawierać tylko `Friend`, `Private`, `Protected`, `Public`, lub `Shadows` słów kluczowych.  
  
 **Identyfikator błędu:** BC30233  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że słowo kluczowe jest poprawna.  
  
2.  Usuń nieprawidłowe słowo kluczowe z `Const` instrukcji.
