# <a name="39ltspecifiergt39-is-not-valid-on-a-constant-declaration"></a>&#39;&lt;Specyfikator&gt; &#39; nie jest prawidłowy w deklaracji stałej
A `Const` instrukcja zawiera nieprawidłowe słowo kluczowe. A `Const` instrukcja może zawierać tylko `Friend`, `Private`, `Protected`, `Public`, lub `Shadows` słów kluczowych.  
  
 **Identyfikator błędu:** BC30233  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że słowo kluczowe jest poprawna.  
  
2.  Usuń nieprawidłowe słowo kluczowe z `Const` instrukcji.
