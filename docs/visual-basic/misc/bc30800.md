# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumenty metody muszą być ujęte w nawiasy

Zasady rządzące wywołaniami procedur są prostsze w nowszych wersjach Visual Basic. Wszystkie argumenty muszą być ujęte w nawiasy.

**Identyfikator błędu:** BC30800

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Ujmij listę argumentów w nawiasy; na przykład:

  ```vb
  MySub(ArrayIndex, NewValue)
  ```
