# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumenty metody muszą być ujęte w nawiasy
Zasady wywołania procedur są łatwiejsze w nowszych wersjach [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]. Wszystkie argumenty muszą być ujęte w nawiasy.  
  
 **Identyfikator błędu:** BC30800  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Umieść w nawiasach; listy argumentów na przykład:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
