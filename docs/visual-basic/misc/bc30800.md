# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumenty metody muszą być ujęte w nawiasy
Zasady dotyczące wywołania procedur są prostsze w nowszej wersji programu Visual Basic. Wszystkie argumenty muszą być ujęte w nawiasy.  
  
 **Identyfikator błędu:** BC30800  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy umieścić na liście argumentów w nawiasach; na przykład:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
