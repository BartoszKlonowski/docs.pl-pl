# <a name="expected-closing--for-xml-tag"></a>Oczekiwano zamykającego '>' dla tagu XML
Znacznik XML literał XML brakuje zamykającego wymagane '>' znaków.  
  
 **Identyfikator błędu:** BC31166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj zamykający '>' znak na końcu tagu XML dla literał XML.
