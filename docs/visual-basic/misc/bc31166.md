# <a name="expected-closing-39gt39-for-xml-tag"></a>Oczekiwano zamykających znaków &#39; &gt; &#39; dla — znacznik XML
Tag XML dla literał XML nie ma nawiasu zamykającego wymagane ' >' znaków.  
  
 **Identyfikator błędu:** BC31166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj zamykający ">" znak na końcu tagu XML dla literału XML.
