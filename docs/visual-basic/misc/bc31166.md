# <a name="expected-closing--for-xml-tag"></a>Oczekiwano zamykającego ">" dla tagu XML

Tag XML dla literału XML nie zawiera wymaganego zamykającego znaku ">".  
  
 **Identyfikator błędu:** BC31166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj zamykający znak ">" na końcu tagu XML dla literału XML.
