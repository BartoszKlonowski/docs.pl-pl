# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>\<assemblyname>Nie można utworzyć zestawu "", ponieważ jego ścieżka jest dłuższa niż 259 znaków

Ścieżka dla tego zestawu przekracza maksymalną długość wynoszącą 259 znaków.  
  
 **Identyfikator błędu:** BC31518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Utwórz krótszą ścieżkę przy użyciu krótszych nazw lub wybierz skróconą hierarchię plików.
