# <a name="assembly-39ltassemblynamegt39-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Zestaw &#39; &lt;assemblyname&gt; &#39; nie można utworzyć, ponieważ jego ścieżka jest dłuższa niż 259 znaków
Ścieżka dla tego zestawu przekracza maksymalną długość 259 znaków.  
  
 **Identyfikator błędu:** BC31518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Utwórz krótszą ścieżkę przy użyciu krótszej nazwy lub wybierając musiał hierarchii plików.
