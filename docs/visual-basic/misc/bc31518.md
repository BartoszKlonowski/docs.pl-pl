# <a name="assembly-ltassemblynamegt-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Zestaw "&lt;assemblyname&gt;" nie można utworzyć, ponieważ jego ścieżka jest więcej niż 259 znaków
Ścieżka do tego zestawu przekracza maksymalną długość 259 znaków.  
  
 **Identyfikator błędu:** BC31518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Utwórz krótszą ścieżkę przy użyciu nazw krótszy lub wybierając shallower hierarchii plików.
