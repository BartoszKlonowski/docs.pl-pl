# <a name="event-eventname-cannot-be-found"></a>Nie można \<eventname> znaleźć zdarzenia ""

Próbowano obsłużyć zdarzenie, którego kompilator nie może zlokalizować.  
  
 **Identyfikator błędu:** BC30590  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nazwa zdarzenia jest prawidłowa.
