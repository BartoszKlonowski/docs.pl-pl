# <a name="event-eventname-cannot-be-found"></a>Zdarzenie '\<eventname >' nie można odnaleźć
Podjęto próbę obsłużyć zdarzenie, które kompilator nie może zlokalizować.  
  
 **Identyfikator błędu:** BC30590  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nazwa zdarzenia jest poprawna.
