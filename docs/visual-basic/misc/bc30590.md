# <a name="event-39lteventnamegt39-cannot-be-found"></a>Zdarzenie &#39; &lt;eventname&gt;&#39; nie można odnaleźć
Próbowano obsługi zdarzeń, które kompilator nie można zlokalizować.  
  
 **Identyfikator błędu:** BC30590  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że nazwa zdarzenia jest poprawna.
