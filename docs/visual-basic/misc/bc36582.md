# <a name="too-many-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Za dużo argumentów dla metody rozszerzenia "&lt;methodname&gt;"zdefiniowane w"&lt;typename&gt;"
Liczby argumentów dostarczone w wywołaniu metody rozszerzenia przekracza liczbę argumentów zdefiniowanych dla metody.  
  
 **Identyfikator błędu:** BC36582  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zapoznaj się z definicji metody rozszerzenie, a następnie usuń dodatkowych argumentów w wywołaniu.
