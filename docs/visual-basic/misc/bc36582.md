# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Za dużo argumentów dla metody rozszerzającej &#39; &lt;methodname&gt;&#39; zdefiniowanego w &#39;&lt; Właściwość TypeName&gt;&#39;
Liczba argumentów w wywołaniu metody rozszerzenia przekracza liczbę argumentów zdefiniowany dla metody.  
  
 **Identyfikator błędu:** BC36582  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zapoznaj się definicja metody rozszerzenia, a następnie usuń dodatkowe argumenty z wywołania.
