# <a name="too-many-arguments-to-extension-method-methodname-defined-in-typename"></a>Za dużo argumentów dla metody rozszerzenia " \<methodname> " zdefiniowanej w elemencie " \<typename> "
Liczba argumentów podana w wywołaniu metody rozszerzenia przekracza liczbę argumentów zdefiniowanych dla metody.  
  
 **Identyfikator błędu:** BC36582  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zapoznaj się z definicją metody rozszerzenia i Usuń dodatkowe argumenty z wywołania.
