# <a name="default-member-ltmembernamegt-is-not-a-property"></a>Domyślny element członkowski "&lt;membername&gt;" nie jest właściwością
Chcesz zadeklarować członka `Default`. Jednak `Default` — słowo kluczowe jest używany do identyfikowania właściwości domyślnych.  
  
 **Identyfikator błędu:** BC30555  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że są identyfikująca właściwość z `Default`, lub Usuń inne `Default` — słowo kluczowe.
