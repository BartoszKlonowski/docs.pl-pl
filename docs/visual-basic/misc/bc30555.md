# <a name="default-member-membername-is-not-a-property"></a>Domyślny element członkowski " \<membername> " nie jest właściwością

Podjęto próbę zadeklarować członka `Default` . Jednak `Default` słowo kluczowe jest używane do identyfikowania właściwości domyślnych.  
  
 **Identyfikator błędu:** BC30555  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że identyfikujesz Właściwość za pomocą elementu `Default` , lub w przeciwnym razie Usuń `Default` słowo kluczowe.
