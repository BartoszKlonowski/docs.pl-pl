# <a name="default-member-39ltmembernamegt39-is-not-a-property"></a>Domyślny element członkowski &#39; &lt;membername&gt;&#39; nie jest właściwością
Nastąpiła próba deklaruje elementu członkowskiego `Default`. Jednak `Default` — słowo kluczowe jest używany do identyfikowania właściwości domyślnych.  
  
 **Identyfikator błędu:** BC30555  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, są identyfikujący właściwość z `Default`, lub usuń else `Default` — słowo kluczowe.
