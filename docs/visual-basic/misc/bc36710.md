# <a name="extension-method-methodname-defined-in-modulename-does-not-have-a-signature-compatible-with-delegate-delegatename"></a>Metoda rozszerzająca " \<methodname> " zdefiniowana w elemencie " \<modulename> " nie ma sygnatury zgodnej z delegatem " \<delegatename> "
Występuje niezgodność między sygnaturami metody rozszerzenia i delegatem, którego próbujesz użyć. `Delegate`Instrukcja definiuje typy parametrów i zwracane typy klasy delegatów. Każda procedura z zgodnymi parametrami, typami parametrów i typem zwracanym może zostać użyta do utworzenia wystąpienia tego typu delegata.  
  
 **Identyfikator błędu:** BC36710
