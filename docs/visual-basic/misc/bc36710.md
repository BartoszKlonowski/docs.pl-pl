# <a name="extension-method-ltmethodnamegt-defined-in-ltmodulenamegt-does-not-have-a-signature-compatible-with-delegate-ltdelegatenamegt"></a>Metoda rozszerzenia "&lt;methodname&gt;"zdefiniowane w"&lt;modulename&gt;"nie ma sygnatury zgodnej z delegatem"&lt;element delegatename&gt;"
Występuje niezgodność między sygnatury metody rozszerzenia a delegat, który próbujesz użyć. `Delegate` Instrukcja definiuje typy parametrów i zwracane typy klasy delegata. Każda procedura zgodnych parametrów, typów parametrów i zwracany typ może służyć do tworzenia wystąpienia tego typu delegata.  
  
 **Identyfikator błędu:** BC36710
