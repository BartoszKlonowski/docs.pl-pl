# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>"\<name >" nie może ujawnić podstawowego typu delegowanego "\<delegatetype >" dla zdarzenia, które implementuje poza projektem za pomocą \<specyfikator > "\<typu >"
Kod przedstawia prywatny typ spoza klasy publicznej.  
  
 **Identyfikator błędu:** BC30915  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Deklarowanie typu `Public`.  
  
     —lub—  
  
- Użyj innego typu.
