# <a name="cannot-create-an-instance-of-module-39ltmodulenamegt39"></a>Nie można utworzyć wystąpienia modułu &#39; &lt;modulename&gt;&#39;
Moduł istnieje tylko jako pojedynczego, współużytkowanego wystąpienia i nie można utworzyć dodatkowe wystąpienia.  
  
 **Identyfikator błędu:** BC30166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiany modułu do klasy lub zastąpienie go w `New` klauzuli z nazwą klasy.
