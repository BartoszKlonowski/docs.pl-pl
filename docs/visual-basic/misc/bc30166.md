# <a name="cannot-create-an-instance-of-module-modulename"></a>Nie można utworzyć wystąpienia modułu " \<modulename> "

Moduł istnieje tylko jako pojedyncze wystąpienie udostępnione i nie można utworzyć dodatkowych wystąpień.  
  
 **Identyfikator błędu:** BC30166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień moduł na klasę lub zastąp go w `New` klauzuli nazwą klasy.
