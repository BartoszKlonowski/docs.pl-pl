# <a name="cannot-create-an-instance-of-module-ltmodulenamegt"></a>Nie można utworzyć wystąpienia modułu "&lt;modulename&gt;"
Moduł istnieje tylko jako pojedynczego, współdzielonego wystąpienia i nie można utworzyć dodatkowe wystąpienia.  
  
 **Identyfikator błędu:** BC30166  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiany modułu do klasy lub zastąp ją w `New` klauzuli nazwą klasy.
