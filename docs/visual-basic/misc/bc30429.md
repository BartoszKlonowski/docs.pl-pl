# <a name="39end-sub39-must-be-preceded-by-a-matching-39sub39"></a>&#39;End Sub&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;Sub&#39;
`End Sub` Występuje instrukcja w kodzie z pasujących `Sub` Definicja procedury przed nim.  
  
 **Identyfikator błędu:** BC30429  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `End Sub` instrukcję, jeśli jest nadmiarowy.  
  
-   Podaj brakujący `Sub` procedura w przypadku braku takiego.  
  
-   Przenieś `End Sub` odpowiednie miejsce w kodzie.
