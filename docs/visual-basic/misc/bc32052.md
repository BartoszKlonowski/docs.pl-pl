 <span data-ttu-id="ab6ee-104">`Is` Porównuje dwa typy referencyjne, aby ustalić, czy wskazują na tym samym wystąpieniu obiektu w pamięci.</span><span class="sxs-lookup"><span data-stu-id="ab6ee-104">`Is` compares two reference types to determine whether they point to the same object instance in memory.</span></span> <span data-ttu-id="ab6ee-105">Nie może korzystać z argumentu, który nie jest typem odwołania, chyba że jest to drugi operand [nic](../../visual-basic/language-reference/nothing.md).</span><span class="sxs-lookup"><span data-stu-id="ab6ee-105">It cannot take an operand that is not a reference type unless the other operand is [Nothing](../../visual-basic/language-reference/nothing.md).</span></span>
