# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>Operator &#39;operator&#39; nie jest zdefiniowany dla typów &#39;type1&#39; i &#39;type2&#39;
Operator "operator" nie jest zdefiniowany dla typów "type1" i "type2". Można użyć właściwości 'Value' można uzyskać wartość ciągu pierwszego elementu obiektu "\<parentElement >".  
  
 Literał XML został użyty w pliku binarnym lub `Boolean` porównania.  
  
 **Identyfikator błędu:** BC31195  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Value` właściwości XML literału dla pliku binarnego lub `Boolean` porównania.
