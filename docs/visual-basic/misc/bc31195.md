# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>Operator 'operator' nie jest zdefiniowany dla typów 'Typ1' i 'type2'
Operator 'operator' nie jest zdefiniowany dla typu "type1" i "type2". Właściwość "Value" umożliwia uzyskać wartość ciągu pierwszego elementu obiektu "\<parentElement >".  
  
 Literał XML został użyty w pliku binarnym lub `Boolean` porównania.  
  
 **Identyfikator błędu:** BC31195  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Value` właściwości XML literału dla pliku binarnego lub `Boolean` porównania.
