# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>Operator "operator" nie jest zdefiniowany dla typów "type1" i "type2"
Operator "operator" nie jest zdefiniowany dla typów "type1" i "type2". Za pomocą właściwości "value" można uzyskać wartość ciągu pierwszego elementu " \<parentElement> ".  
  
 Literał XML został użyty w danych binarnych lub w `Boolean` porównaniu.  
  
 **Identyfikator błędu:** BC31195  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Value` Właściwości literału XML dla danych binarnych lub `Boolean` porównania.
