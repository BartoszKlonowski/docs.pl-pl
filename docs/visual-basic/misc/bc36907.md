<span data-ttu-id="498c9-103">Określono argument typu w wywołaniu metody rozszerzenia, która nie ma ogólnych parametrów lub ma nie parametrów ogólnych, dla których typ nie jest jeszcze określony.</span><span class="sxs-lookup"><span data-stu-id="498c9-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="498c9-104">Na przykład poniższy kod powoduje błąd.</span><span class="sxs-lookup"><span data-stu-id="498c9-104">For example, the following code causes this error.</span></span>  
  
```vb

```

Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()
