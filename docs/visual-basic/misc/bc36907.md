# <a name="extension-method-methodname-defined-in-modulename-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="17fb1-102">Metody rozszerzenia '\<methodname >' zdefiniowane w '\<nazwa_modułu >' nie jest ogólna (lub nie ma parametrów typu bezpłatnej) i dlatego nie może mieć argumentów typu</span><span class="sxs-lookup"><span data-stu-id="17fb1-102">Extension method '\<methodname>' defined in '\<modulename>' is not generic (or has no free type parameters) and so cannot have type arguments</span></span>
<span data-ttu-id="17fb1-103">Określono argument typu w wywołaniu metody rozszerzenia, która nie ma ogólnych parametrów lub ma nie parametrów ogólnych, dla których typ nie jest jeszcze określony.</span><span class="sxs-lookup"><span data-stu-id="17fb1-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="17fb1-104">Na przykład poniższy kod powoduje błąd.</span><span class="sxs-lookup"><span data-stu-id="17fb1-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
