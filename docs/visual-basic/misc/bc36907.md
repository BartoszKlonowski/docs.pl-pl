# <a name="extension-method-methodname-defined-in-modulename-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="28b58-102">Metoda rozszerzająca " \<methodname> " zdefiniowana w elemencie " \<modulename> " nie jest ogólna (lub nie ma parametrów typu dowolnego) i dlatego nie może mieć argumentów typu</span><span class="sxs-lookup"><span data-stu-id="28b58-102">Extension method '\<methodname>' defined in '\<modulename>' is not generic (or has no free type parameters) and so cannot have type arguments</span></span>
<span data-ttu-id="28b58-103">Argument typu został określony w wywołaniu metody rozszerzenia, która nie ma parametrów ogólnych lub nie ma parametrów ogólnych, dla których typ nie został jeszcze określony.</span><span class="sxs-lookup"><span data-stu-id="28b58-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="28b58-104">Na przykład poniższy kod powoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="28b58-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
