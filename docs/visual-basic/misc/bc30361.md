# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>"&lt;właściwość1&gt;"i"&lt;property2&gt;" nie mogą przeciążać siebie nawzajem, ponieważ tylko jeden jest zadeklarowany jako "Default"
Jeśli właściwość określa `Default`, wszystkie właściwości przeciążone na tej nazwy, należy także określić `Default`.  
  
 **Identyfikator błędu:** BC30361  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie przeciążone właściwości są zadeklarowane jako `Default`.
