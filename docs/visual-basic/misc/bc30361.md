# <a name="property1-and-property2-cannot-overload-each-other-because-only-one-is-declared-default"></a>"\<właściwość1 >" i "\<property2 >" nie mogą przeciążać siebie nawzajem, ponieważ tylko jeden jest zadeklarowany jako "Default"
Jeśli właściwość określa `Default`, wszystkie właściwości przeciążone na tej nazwy, należy także określić `Default`.  
  
 **Identyfikator błędu:** BC30361  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie przeciążone właściwości są zadeklarowane jako `Default`.
