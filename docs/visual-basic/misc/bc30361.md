# <a name="39ltproperty1gt39-and-39ltproperty2gt39-cannot-overload-each-other-because-only-one-is-declared-39default39"></a>&#39;&lt;właściwości property1&gt; &#39; i &#39; &lt;property2&gt; &#39; nie mogą przeciążać siebie nawzajem, ponieważ tylko jeden z nich jest zadeklarowany jako &#39;domyślne&#39;
Jeśli właściwość określa `Default`, wszystkie właściwości przeciążony na tej nazwy musi również określać `Default`.  
  
 **Identyfikator błędu:** BC30361  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie właściwości przeciążenia zadeklarowany jako `Default`.
