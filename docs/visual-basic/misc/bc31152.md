# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefiks przestrzeni nazw XML "prefiks" jest zarezerwowana do użytku przez kod XML i nie można zmienić identyfikatora URI przestrzeni nazw
A zastrzeżone prefiks przestrzeni nazw XML, takie jak `xmlns`, została użyta w literał XML.  
  
 **Identyfikator błędu:** BC31152  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień prefiks przestrzeni nazw XML na wartość, która nie jest zarezerwowana.
