# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefiks przestrzeni nazw XML &#39;prefiks&#39; jest zarezerwowany do użytku przez kod XML i nie można zmienić identyfikatora URI przestrzeni nazw
A zastrzeżony prefiks przestrzeni nazw XML, takich jak `xmlns`, użyto literał XML.  
  
 **Identyfikator błędu:** BC31152  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień prefiks przestrzeni nazw XML na wartość, która nie jest zarezerwowana.
