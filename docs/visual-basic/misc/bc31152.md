# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefiks przestrzeni nazw XML "prefix" jest zarezerwowany do użytku przez kod XML i nie można zmienić identyfikatora URI przestrzeni nazw

Zastrzeżony prefiks przestrzeni nazw XML, taki jak `xmlns` , został użyty w literale XML.  
  
 **Identyfikator błędu:** BC31152  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień prefiks przestrzeni nazw XML na wartość, która nie jest zarezerwowana.
