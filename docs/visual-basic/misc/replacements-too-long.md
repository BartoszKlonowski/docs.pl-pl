# <a name="replacements-too-long"></a>Zbyt długie elementy zastępujące
Wybrano zastąpienie, który przekracza dozwolone długości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Ograniczenie długości można znaleźć w dokumentacji składnika.
