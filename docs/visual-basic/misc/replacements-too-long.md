# <a name="replacements-too-long"></a>Wymiana za długa
Określono zastępczy, który przekracza dozwolone długości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Ograniczenia długości można znaleźć w dokumentacji składnika.
