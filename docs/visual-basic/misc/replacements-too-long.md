# <a name="replacements-too-long"></a>Zamienniki są za długie

Określono zastąpienie, które przekracza dozwoloną długość.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją dotyczącą składnika, aby uzyskać ograniczenie długości.
