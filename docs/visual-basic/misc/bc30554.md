# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39; &lt;namespacename&gt;&#39; jest niejednoznaczny
Podana nazwa jest niejednoznaczny i dlatego powoduje konflikt z inną nazwą. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilatora nie ma żadnych reguł rozwiązywania konfliktów; nazwy musi odróżniania samodzielnie.  
  
 **Identyfikator błędu:** BC30554  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
