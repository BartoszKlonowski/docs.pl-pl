# <a name="you-must-specify-a-name"></a>Należy określić nazwę
Podczas wykonywania operacji na plikach takich jak zmiana nazwy pliku, należy określić nazwę.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ nazwę.
