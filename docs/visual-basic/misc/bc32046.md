 <span data-ttu-id="db4a1-104">*Ograniczenie* dla parametru typu nakłada wymóg dla dowolnego argumentu typu przekazaną do tego parametru typu, gdy tworzony jest typ ogólny.</span><span class="sxs-lookup"><span data-stu-id="db4a1-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="db4a1-105">`New`Ograniczenie określa, że argument typu musi ujawniać Konstruktor bez parametrów, który może uzyskać dostęp do kodu.</span><span class="sxs-lookup"><span data-stu-id="db4a1-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="db4a1-106">Jest to, co umożliwia `New` klauzulę w instrukcji deklaracji, aby utworzyć wystąpienie tego typu.</span><span class="sxs-lookup"><span data-stu-id="db4a1-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>
