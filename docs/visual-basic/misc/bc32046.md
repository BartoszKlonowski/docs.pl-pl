 <span data-ttu-id="3f9f3-104">A *ograniczenie* w danym typie parametru nakłada wymaganie na wszystkich argumentów typu do tego parametru typu po utworzeniu typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="3f9f3-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="3f9f3-105">`New` Ograniczenie Określa, że argument typu musi ujawniać konstruktor bez parametrów, które mogą uzyskiwać dostęp do tworzenia kodu.</span><span class="sxs-lookup"><span data-stu-id="3f9f3-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="3f9f3-106">Jest to, co umożliwia `New` klauzuli w instrukcji deklaracji, aby utworzyć wystąpienie tego typu.</span><span class="sxs-lookup"><span data-stu-id="3f9f3-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>
