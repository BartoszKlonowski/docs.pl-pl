# <a name="attribute-39ltattributenamegt39-cannot-be-applied-to-a-method-with-optional-parameters"></a>Atrybut &#39; &lt;attributename&gt;&#39; nie można zastosować do metody z opcjonalnymi parametrami
Ten atrybut można tylko za pomocą metod korzystających z liczbą wymaganych argumentów lub bez argumentów.  
  
 **Identyfikator błędu:** BC30645  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zdefiniuj metody bez parametrów opcjonalnych.  
  
2.  Użyj atrybutu, który może być używany z metod, które mają następujące parametry opcjonalne.  
  
3.  Definiowanie niestandardowego atrybutu, który może być używany w tym kontekście.
