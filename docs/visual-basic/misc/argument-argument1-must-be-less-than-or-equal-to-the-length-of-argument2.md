# <a name="argument-argument1-must-be-less-than-or-equal-to-the-length-of-argument-argument2"></a>Argument " \<argument1> " musi być mniejszy lub równy długości argumentu " \<argument2> "

Długość argumentu przekracza dozwolony limit, który jest określany przez drugi argument długości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że długość wartości argumentu mieści się w prawidłowym zakresie.  
  
2. Jeśli argument jest dostarczany przez obliczenie, sprawdź wartości w procesach obliczania.
