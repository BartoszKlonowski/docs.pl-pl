# <a name="argument-39ltargument1gt39-must-be-less-than-or-equal-to-the-length-of-argument-39ltargument2gt39"></a>Argument &#39; &lt;argument1&gt;&#39; musi być mniejsza niż długość argumentu &#39;&lt; argument2&gt;&#39;
Długość argumentu przekracza dozwolony limit, który jest określany przez drugi argument długość.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że długość wartości argumentu znajduje się w dozwolonym zakresie.  
  
2.  Jeśli argument jest dostarczana przez obliczeń, sprawdź wartości procesów obliczeń.
