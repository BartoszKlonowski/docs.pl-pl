# <a name="argument-ltargument1gt-must-be-less-than-or-equal-to-the-length-of-argument-ltargument2gt"></a>Argument "&lt;moj_skrypt_argument1&gt;'musi być mniejsza niż długość argumentu'&lt;argument2&gt;"
Długość argumentu przekracza dozwolony limit jest określany przez długość drugiego argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że długość wartości argumentu mieści się w prawidłowym zakresem.  
  
2.  Jeśli argument jest dostarczana przez obliczenie, sprawdź wartości procesów obliczeń.
