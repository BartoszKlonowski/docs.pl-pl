# <a name="goto-to-label-ltlabelnamegt-cannot-be-used-within-lambda-expressions"></a>Przejdź do etykiety "&lt;nazwa_etykiety&gt;" nie można używać w wyrażeniach lambda
Treść wyrażenia lambda musi być wyrażeniem, które oblicza i zwraca wartość typu single. Nie są dozwolone, w tym `GoTo` instrukcji.  
  
 **Identyfikator błędu:** BC36597
