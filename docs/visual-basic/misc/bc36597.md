# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>Przejdź do etykiety &#39; &lt;wartość labelname&gt; &#39; nie można używać wewnątrz wyrażenia lambda
Treść wyrażenia lambda musi być wyrażeniem, które wylicza i zwraca pojedynczą wartość. Brak nie są dozwolone, łącznie z `GoTo` instrukcje.  
  
 **Identyfikator błędu:** BC36597
