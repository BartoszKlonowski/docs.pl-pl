# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>Instrukcji GoTo to Label " \<labelname> " nie można używać w wyrażeniach lambda

Treść wyrażenia lambda musi być wyrażeniem, które oblicza i zwraca pojedynczą wartość. Nie są dozwolone żadne instrukcje, w tym `GoTo` instrukcje.  
  
 **Identyfikator błędu:** BC36597
