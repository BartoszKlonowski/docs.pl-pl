# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Typem argumentu " \<argumentname> " jest " \<typename> ", który nie jest wartością numeryczną

Argument niebędący liczbą występuje w sytuacji, w której Oczekiwano argumentu numerycznego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że argument jest wartością numeryczną.
