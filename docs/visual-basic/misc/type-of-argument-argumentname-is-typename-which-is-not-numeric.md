# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Typ argumentu "&lt;argumentname&gt;"is"&lt;typename&gt;", który nie jest liczbą
Argument nienumeryczny występuje w sytuacji, w którym oczekiwano liczbowego argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że argument jest wartością liczbową.
