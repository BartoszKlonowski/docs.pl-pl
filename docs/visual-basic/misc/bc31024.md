# <a name="unable-to-create-a-net-runtime-interface-error"></a>Nie można utworzyć interfejsu środowiska uruchomieniowego .NET: \<błąd >
Wystąpił wewnętrzny błąd kompilatora tworzenie interfejs lub wywołanie metody w interfejsie nie powiodło się.  
  
 **Identyfikator błędu:** BC31024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zapisz swoją pracę i uruchom ponownie program Visual Studio.  
  
2.  Jeśli ten błąd wystąpi, zainstaluj ponownie program Visual Studio.  
  
3.  Jeśli ten błąd będzie występował po ponownej instalacji, powiadom pomoc techniczna firmy Microsoft.
