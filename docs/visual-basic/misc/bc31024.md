# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>Nie można utworzyć interfejsu środowiska wykonawczego platformy .NET: &lt;błąd&gt;
Wystąpił błąd wewnętrzny kompilatora podczas tworzenia interfejsu lub wywołanie metody w interfejsie nie powiodło się.  
  
 **Identyfikator błędu:** BC31024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zapisz swoją pracę i ponownie uruchom program Visual Studio.  
  
2.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie program Visual Studio.  
  
3.  Jeśli błąd będzie nadal występować po ponownej instalacji, powiadamia pomocy technicznej firmy Microsoft.
