# <a name="unable-to-create-a-net-runtime-interface-error"></a>Nie można utworzyć interfejsu środowiska uruchomieniowego platformy .NET: \<error>

Wystąpił wewnętrzny błąd kompilatora podczas tworzenia interfejsu lub wywołanie metody w interfejsie zakończyło się niepowodzeniem.  
  
 **Identyfikator błędu:** BC31024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapisz swoją służbę i ponownie uruchom program Visual Studio.  
  
2. Jeśli błąd się powtarza, zainstaluj ponownie program Visual Studio.  
  
3. Jeśli błąd będzie się powtarzać po ponownej instalacji, powiadom firmę Microsoft o pomocy technicznej.
