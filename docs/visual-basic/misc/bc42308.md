# <a name="xml-comment-parameter-must-have-a-39name39-attribute"></a>Parametr komentarza XML musi mieć &#39;nazwa&#39; atrybutu
Parametr komentarza XML musi mieć atrybut "name". Komentarz XML zostanie zignorowany.  
  
 Brak wymaganego tagu XML `name` atrybutu.  
  
 **Identyfikator błędu:** BC42308  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj wymagane `name` atrybutu dla tagu.
