# <a name="redim-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>"ReDim" wymaga listę nowych granic dla każdego wymiaru tablicy w nawiasach
Należy określić nowy rozmiar tablicy jako część `ReDim` instrukcji.  
  
 **Identyfikator błędu:** BC30670  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj rozmiar każdej rangi tablicy; na przykład:  
  
    ```  
    ReDim arr(5, 6)  
    ```
