# <a name="too-many-arguments"></a>Za dużo argumentów

Liczba argumentów podana w wywołaniu procedury przekracza liczbę argumentów zdefiniowanych dla procedury.  
  
 **Identyfikator błędu:** BC36565  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zapoznaj się z definicją procedury i Usuń dodatkowe argumenty z wywołania.
