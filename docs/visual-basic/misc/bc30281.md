# <a name="structure-ltstructurenamegt-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura "&lt;structurename&gt;" musi zawierać co najmniej jedną zmienną elementu członkowskiego wystąpienia lub deklaracja zdarzenia
Występuje, funkcjonalnie pustą strukturą. Co najmniej jedną zmienną elementu członkowskiego musi być zadeklarowany między `Structure` instrukcji i `End Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30281  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Dim` instrukcji lub `Event` poufności informacji w strukturze.
