# <a name="structure-structurename-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura " \<structurename> " musi zawierać co najmniej jedną zmienną elementu członkowskiego wystąpienia lub deklarację zdarzenia
Występuje funkcjonalnie pusta struktura. Należy zadeklarować co najmniej jeden element członkowski zmiennej między `Structure` instrukcją a `End Structure` instrukcją.  
  
 **Identyfikator błędu:** BC30281  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Dim` instrukcję lub `Event` instrukcję do struktury.
