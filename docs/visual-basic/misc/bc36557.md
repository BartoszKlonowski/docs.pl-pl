# <a name="membername-is-not-a-member-of-contextname-it-does-not-exist-in-the-current-context"></a>'\<nazwa>" nie jest członkiem "contextname\<>"; nie istnieje w obecnym kontekście
Nieistniejąca nazwa elementu członkowskiego została przypisana do właściwości w deklaracji typu anonimowego. W poniższym `.Prop1` przykładzie i `.Prop2` są właściwości typu anonimowego. Próba przypisania `.Prop3` `.Prop2` do powoduje błąd.  
  
```vb  
' Not valid.  
Dim anon1 = New With {.Prop1 = 27, .Prop2 = .Prop3}  
  
' The assignment is valid if the assigned property has been declared
' and initialized.  
Dim anon2 = New With {.Prop1 = 27, .Prop2 = .Prop1}  
```  
  
 **Identyfikator błędu:** Bc36657  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź kod, aby ustalić, co chcesz przypisać. Nazwa zmiennej może być błędnie wątek lub może wymagać kwalifikacji, jeśli jest to właściwość innego obiektu.
