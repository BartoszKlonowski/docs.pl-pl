# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Ogranicznik nie może być Nothing ani ciągiem pustym
`TextFieldParser` Nie może odczytać z pliku, ponieważ `Delimiters` właściwość jest ustawiona na `Nothing` lub jest pusta `String` ("").  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłową wartość dla `Delimiters`.
