# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Ogranicznik nie może mieć wartości Nothing lub być pustym ciągiem

`TextFieldParser`Nie można odczytać z pliku, ponieważ `Delimiters` Właściwość ma wartość `Nothing` lub jest wartością pustą `String` ("").  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłową wartość dla `Delimiters` .
