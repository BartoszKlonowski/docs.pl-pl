# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>Typ '\<typename >' została zaimportowana z różnymi wersjami zestawu '\<nazwa_zestawu > '
Typ '\<typename >' została zaimportowana z różnymi wersjami zestawu '\<nazwa_zestawu > '. Nie można użyć różnych wersji tego samego zestawu, w tym samym projekcie.  
  
 Typ została zaimportowana z różnymi wersjami tego samego zestawu, tworzenie niejednoznaczności.  
  
 **Identyfikator błędu:** BC31525  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń odwołania do wielu wersji tego samego zestawu.
