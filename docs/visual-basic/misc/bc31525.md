# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>Typ " \<typename> " został zaimportowany z różnych wersji zestawu " \<assemblyname> "

Typ " \<typename> " został zaimportowany z różnych wersji zestawu " \<assemblyname> ". Różne wersje tego samego zestawu nie mogą być używane w tym samym projekcie.  
  
 Typ został zaimportowany z różnych wersji tego samego zestawu, tworząc niejednoznaczność.  
  
 **Identyfikator błędu:** BC31525  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń odwołania do wielu wersji tego samego zestawu.
