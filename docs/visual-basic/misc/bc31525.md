# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Wpisz &#39; &lt;typename&gt;&#39; jest importowany z różnych wersji zestawu &#39;&lt; AssemblyName&gt;&#39;
Typ "\<typename >" jest importowany z różnych wersji zestawu "\<assemblyname >". Różne wersje tego samego zestawu nie można użyć w tym samym projekcie.  
  
 Typ została zaimportowana z różnymi wersjami tego samego zestawu, tworzenie niejednoznaczności.  
  
 **Identyfikator błędu:** BC31525  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń odwołania do wielu wersji tego samego zestawu.
