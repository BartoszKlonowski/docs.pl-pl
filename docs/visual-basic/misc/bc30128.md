# <a name="security-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Atrybut zabezpieczeń "&lt;attributename&gt;" jest nieprawidłowa: &lt;błąd&gt;
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w kontekście zabezpieczeń.

**Identyfikator błędu:** BC30128

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zamień go atrybutem, który jest prawidłowy dla kontekstu.

2. Sprawdź komunikat o błędzie w cudzysłowach i podjąć odpowiednie działania.
