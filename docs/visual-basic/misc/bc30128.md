# <a name="security-attribute-attributename-is-not-valid-error"></a>Atrybut zabezpieczeń " \<attributename> " jest nieprawidłowy: \<error>

Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w kontekście zabezpieczeń.

**Identyfikator błędu:** BC30128

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zastąp go atrybutem, który jest prawidłowy dla kontekstu.

2. Przejrzyj cytowany komunikat o błędzie i podejmij odpowiednie działanie.
