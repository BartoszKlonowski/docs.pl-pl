# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>Ustalanie przeciążenia nie powiodło się, ponieważ nie "\<metody >" jest dostępny
Wprowadzono wywołanie przeciążonej metody, ale kompilator nie może odnaleźć dostępnej metody z listą parametrów, który odpowiada pod względem liczby parametrów.  
  
 **Identyfikator błędu:** BC30517  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź listę argumentów, podane.  
  
2.  Sprawdź metodę, którą próbujesz wywołać, aby ustalić, czy jest dostępny.
