# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>Rozpoznanie przeciążenia nie powiodło się, ponieważ nie jest dostępny żaden element " \<method> "
Wykonano wywołanie przeciążonej metody, ale kompilator nie może odnaleźć dostępnej metody z listą parametrów, która jest zgodna z liczbą parametrów.  
  
 **Identyfikator błędu:** BC30517  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź podaną listę argumentów.  
  
2. Sprawdź metodę, którą próbujesz wywołać, aby określić, czy jest dostępna.
