# <a name="overload-resolution-failed-because-no-39ltmethodgt39-is-accessible"></a>Przeciążenia nie powiodło się, ponieważ nie &#39; &lt;metody&gt; &#39; jest dostępny
Wywołanie przeciążonej metody zostały wykonane, ale kompilator nie można odnaleźć dostępnej metody z listą parametrów zgodny pod względem liczby parametrów.  
  
 **Identyfikator błędu:** BC30517  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź podane listy argumentów.  
  
2.  Sprawdź metodę, którą próbujesz wywołania w celu określenia, czy jest dostępny.
