# <a name="partial-methods-must-be-declared-private-instead-of-accessmodifier"></a><span data-ttu-id="7e483-102">Metody częściowe muszą być zadeklarowane jako "Private" zamiast " \<accessModifier> "</span><span class="sxs-lookup"><span data-stu-id="7e483-102">Partial methods must be declared 'Private' instead of '\<accessModifier>'</span></span>

<span data-ttu-id="7e483-103">Modyfikator dostępu `Private` jest wymagany w deklaracji metody częściowej.</span><span class="sxs-lookup"><span data-stu-id="7e483-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="7e483-104">W poniższym przykładzie pokazano użycie `Private` w sygnaturze metody i jej implementacji.</span><span class="sxs-lookup"><span data-stu-id="7e483-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb  
' Definition of the partial method signature.  
Partial Private Sub OnNameChanged()  
    ' The body of the signature is empty.  
End Sub  
```  
  
```vb  
' Implementation of the partial method.  
Private Sub OnNameChanged()  
    MsgBox("Name was changed to " & Me.Name)  
End Sub  
```
