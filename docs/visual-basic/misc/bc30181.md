# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Specyfikatory prawidłowy tylko na początku deklaracji
Słowa kluczowe, określając ułatwień dostępu i innych nazw musi występować przed deklaracją — słowo kluczowe, takie jak `Dim` lub `Function`.  
  
 **Identyfikator błędu:** BC30181  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wszystkie specyfikatory przenoszenie listy atrybutów (jeśli istnieje) i słowo kluczowe deklaracji.
