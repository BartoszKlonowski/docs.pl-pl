# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Specyfikatory są prawidłowe tylko na początku deklaracji

Słowa kluczowe określające dostępność i inne oznaczenia muszą występować przed słowami kluczowymi deklaracji, takimi jak `Dim` lub `Function` .  
  
 **Identyfikator błędu:** BC30181  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś wszystkie specyfikatory między listą atrybutów (jeśli istnieje) i słowem kluczowym deklaracji.
