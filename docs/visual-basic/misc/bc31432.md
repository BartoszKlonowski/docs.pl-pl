<span data-ttu-id="b89bc-103">Modyfikator dostępu `Private` jest wymagany w deklaracji metody częściowej.</span><span class="sxs-lookup"><span data-stu-id="b89bc-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="b89bc-104">W poniższym przykładzie pokazano użycie `Private` w sygnaturze metody i jej implementacji.</span><span class="sxs-lookup"><span data-stu-id="b89bc-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb  
' Definition of the partial method signature.  
Partial Private Sub OnNameChanged()  
    ' The body of the signature is empty.  
End Sub  
```  
  
```vb

Private Sub OnNameChanged()

End Sub  
```
