<span data-ttu-id="c3208-103">Modyfikator dostępu `Private` jest wymagany w deklaracji metody częściowej.</span><span class="sxs-lookup"><span data-stu-id="c3208-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="c3208-104">Poniższy przykład pokazuje użycie `Private` w podpisie metody i jego wdrożenie.</span><span class="sxs-lookup"><span data-stu-id="c3208-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb  
' Definition of the partial method signature.

    ' The body of the signature is empty.  
End Sub  
```  
  
```vb

End Sub
```  
