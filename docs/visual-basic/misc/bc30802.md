# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39; Wpisz &#39; instrukcje nie są już obsługiwane; Użyj &#39; Struktura &#39; instrukcje zamiast niego
`Type` — Słowo kluczowe nie może służyć do definiowania typów zdefiniowanych przez użytkownika.  
  
 **Identyfikator błędu:** BC30802  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Structure` instrukcji do definiowania nowego typu.
