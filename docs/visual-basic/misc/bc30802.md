# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>Instrukcje "Type" nie są już obsługiwane; Zamiast tego użyj instrukcji "Structure"
`Type`Nie można już używać słowa kluczowego do definiowania typów zdefiniowanych przez użytkownika.  
  
 **Identyfikator błędu:** BC30802  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Structure` instrukcji, aby zdefiniować nowy typ.
