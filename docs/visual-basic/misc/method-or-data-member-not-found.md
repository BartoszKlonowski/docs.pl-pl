# <a name="method-or-data-member-not-found"></a>Nie znaleziono metody lub elementu członkowskiego danych
Nie znaleziono metody lub elementu członkowskiego danych, do którego próbowano uzyskać dostęp.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nazwa metody lub elementu członkowskiego danych nie została błędnie.
