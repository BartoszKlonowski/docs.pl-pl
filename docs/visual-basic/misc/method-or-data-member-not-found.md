# <a name="method-or-data-member-not-found"></a>Nie można odnaleźć metody lub dane składowej.
Nie znaleziono członka metody lub danych, z którego próbowano uzyskać dostęp.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nie błędnie nazwę metody lub składowej danych.
