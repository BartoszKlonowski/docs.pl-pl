# <a name="enum-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować wyliczenia w interfejsie jako "\<specyfikator >"
Nieprawidłowy modyfikator została określona za pomocą wyliczania zadeklarowanych w interfejsie.  
  
 **Identyfikator błędu:** BC31069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikatorów, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, lub `Overridable`.
