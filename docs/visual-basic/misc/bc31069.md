# <a name="enum-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować wyliczenia w interfejsie jako " \<specifier> "

Określono nieprawidłowy modyfikator z wyliczeniem zadeklarowanym w interfejsie.  
  
 **Identyfikator błędu:** BC31069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikatory, takie jak,,,,, `Public` `Private` `Shared` `Friend` `Protected` `Protected Friend` lub `Overridable` .
