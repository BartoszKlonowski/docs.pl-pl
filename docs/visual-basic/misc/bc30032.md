# <a name="events-cannot-have-a-return-type"></a>Zdarzenia nie może posiadać typu zwracanego
Zdarzenia można zaakceptować argumentów, ale bezpośrednio nie mogą zwracać wartości.  
  
 **Identyfikator błędu:** BC30032  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń typ zwracany z `Event` instrukcji.
