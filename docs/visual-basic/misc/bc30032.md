# <a name="events-cannot-have-a-return-type"></a>Zdarzenia nie mogą mieć zwracanego typu
Zdarzenia mogą akceptować argumenty, ale bezpośrednio nie może zwracać wartości.  
  
 **Identyfikator błędu:** BC30032  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń typ zwracany z `Event` instrukcji.
