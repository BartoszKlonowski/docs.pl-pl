# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace lub typ &#39; &lt;nazwa&gt;&#39; został już zaimportowany
Próbowano zaimportować więcej niż raz przestrzeni nazw.  
  
 **Identyfikator błędu:** BC31051  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe `Imports` instrukcje.
