# <a name="namespace-or-type-ltnamegt-has-already-been-imported"></a>Namespace lub typu "&lt;nazwa&gt;" został już zaimportowany
Nastąpiła próba do importowania przestrzeni nazw w więcej niż jeden raz.  
  
 **Identyfikator błędu:** BC31051  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe `Imports` instrukcji.
