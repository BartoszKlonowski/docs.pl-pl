# <a name="namespace-or-type-name-has-already-been-imported"></a>Przestrzeń nazw lub typ " \<name> " został już zaimportowany

Podjęto próbę zaimportowania przestrzeni nazw więcej niż raz.  
  
 **Identyfikator błędu:** BC31051  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nadmiarowe `Imports` instrukcje.
