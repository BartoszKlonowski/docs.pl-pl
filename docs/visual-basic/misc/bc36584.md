# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-argument"></a>Parametr "&lt;parametername&gt;"w metodzie rozszerzenia"&lt;methodname&gt;"zdefiniowane w"&lt;typename&gt;" ma już pasujący argument
Argument jest przekazywany przez nazwę więcej niż jeden raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC36584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy jeden z nazwy argumentów jest błędna.  
  
-   Usuń zduplikowane argumentu z listy argumentów.
