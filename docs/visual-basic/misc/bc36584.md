# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parametr &#39; &lt;parametername&gt; &#39; w metodzie rozszerzenia &#39; &lt;methodname&gt; &#39; zdefiniowane w &#39; &lt;typename&gt; &#39; już ma pasujący argument
Argument jest przekazywany przez nazwę więcej niż raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC36584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy nazwy argumentu jest błędna.  
  
-   Usuń zduplikowane argumentu z listy argumentów.
