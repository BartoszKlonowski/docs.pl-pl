# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parametr " \<parametername> " w metodzie rozszerzenia " \<methodname> " zdefiniowanej w elemencie " \<typename> " ma już pasujący argument

Argument jest przesyłany przez nazwę więcej niż raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC36584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwy argumentów są błędne.  
  
- Usuń zduplikowany argument z listy argumentów.
