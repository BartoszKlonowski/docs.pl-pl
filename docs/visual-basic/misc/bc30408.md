# <a name="method-ltmethodnamegt-does-not-have-the-same-signature-as-delegate-ltdelegatenamegt"></a>Metoda "&lt;methodname&gt;"nie ma tej samej sygnatury jak obiekt delegowany"&lt;element delegatename&gt;"
Występuje niezgodność między sygnatury metody i delegata, którą próbujesz użyć. `Delegate` Instrukcja definiuje typy parametrów i zwracane typy klasy delegata. Każda procedura z pasującymi parametrami, typy i zwracany typ może służyć do tworzenia wystąpienia tego typu delegata. Każda klasa obiektu delegowanego definiuje konstruktora, który jest przekazywany do specyfikacji metody obiektu.  
  
 **Identyfikator błędu:** BC30408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdzanie podpisów, aby znaleźć niezgodność i popraw go.
