# <a name="method-39ltmethodnamegt39-does-not-have-the-same-signature-as-delegate-39ltdelegatenamegt39"></a>Metoda &#39; &lt;methodname&gt;&#39; nie ma takiego samego podpisu jak delegata &#39;&lt; Element delegateName&gt;&#39;
Wystąpiła niezgodność między sygnatury metody i delegata, którego próbujesz użyć. `Delegate` Instrukcji definiuje typy parametrów i zwracanych typów klasa delegata. Postępowanie z pasującymi parametrów, typy i typ zwracany może służyć do tworzenia wystąpienia tego typu obiektu delegowanego. Każda klasa delegata definiuje konstruktora, który jest przekazywany specyfikację metody obiektu.  
  
 **Identyfikator błędu:** BC30408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdzanie podpisów można znaleźć niezgodność i popraw go.
