# <a name="method-methodname-does-not-have-the-same-signature-as-delegate-delegatename"></a>Metoda " \<methodname> " nie ma takiego samego podpisu jak obiekt delegowany " \<delegatename> "
Występuje niezgodność między sygnaturami metody i delegatem, którego próbujesz użyć. `Delegate`Instrukcja definiuje typy parametrów i zwracane typy klasy delegatów. Do utworzenia wystąpienia tego typu delegata można użyć dowolnej procedury z pasującymi parametrami, typami i typem zwracanym. Każda Klasa delegatów definiuje konstruktora, który przekazał specyfikację metody obiektu.  
  
 **Identyfikator błędu:** BC30408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podpisy, aby znaleźć niezgodność i popraw ją.
