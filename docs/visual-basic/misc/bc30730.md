# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Metody zadeklarowane jako "Overrides" nie można zadeklarować jako "Overridable", ponieważ są one niejawnie możliwym do zastąpienia
W przeciwieństwie do większości metod metody oznaczone atrybutami `Overrides` modyfikator są możliwym do zastąpienia domyślnie.  
  
 **Identyfikator błędu:** BC30730  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pomiń `Overridable` modyfikator metody oznaczone `Overrides` modyfikator.
