# <a name="statement-cannot-appear-within-a-property-body"></a>Instrukcja nie może występować w treści właściwości

Instrukcja nie może występować w treści właściwości. Przyjęto koniec właściwości.  
  
 Napotkano nieoczekiwaną instrukcję wewnątrz właściwości.  
  
 **Identyfikator błędu:** BC30634  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy właściwość używa poprawnej składni i kończy się od `End Property` konstrukcji.
