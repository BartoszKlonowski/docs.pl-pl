# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Przywoływany obiekt " \<objectname> " ma wartość "Nothing".

Obiekt, do którego się odwoływano, nie został ustawiony lub został ustawiony na wartość `Nothing` .  
  
 **Identyfikator błędu:** BC30705  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwa obiektu jest poprawna i czy obiekt został ustawiony.
