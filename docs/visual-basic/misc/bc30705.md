# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Odwołuje się do obiektu &#39; &lt;objectname&gt;&#39; ma wartość &#39; Nie &#39;
Obiekt, do którego nastąpiło odwołanie, nie została ustawiona lub została ustawiona na wartość `Nothing`.  
  
 **Identyfikator błędu:** BC30705  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, że nazwa obiektu jest poprawna i czy obiekt został ustawiony.
