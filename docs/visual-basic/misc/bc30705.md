# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Odwołuje się do obiektu '\<objectname >' ma wartość 'Nothing'
Obiekt, do którego nastąpiło odwołanie, nie została ustawiona lub została ustawiona na wartość `Nothing`.  
  
 **Identyfikator błędu:** BC30705  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy nazwa obiektu została wpisana poprawnie i czy obiekt został ustawiony.
