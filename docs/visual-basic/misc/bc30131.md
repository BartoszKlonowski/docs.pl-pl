# <a name="security-attribute-ltattributenamegt-cannot-be-applied-to-a-module"></a>Atrybut zabezpieczeń "&lt;attributename&gt;" nie można zastosować do modułu
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu zabezpieczeń w module.

**Identyfikator błędu:** BC30131

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Usuń atrybut z deklaracji.
