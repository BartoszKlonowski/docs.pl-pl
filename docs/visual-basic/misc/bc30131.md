# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>\<attributename>Nie można zastosować atrybutu zabezpieczeń "" do modułu

Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu zabezpieczeń modułu.

**Identyfikator błędu:** BC30131

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Usuń atrybut z deklaracji.
