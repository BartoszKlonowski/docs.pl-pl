# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;Dziedziczy&#39; instrukcji musi występować przed wszelkimi deklaracjami w klasie
`Inherits` Instrukcje musi znajdować się na początku definicji klas.  
  
 **Identyfikator błędu:** BC30683  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `Inherits` instrukcji u góry klasy.
