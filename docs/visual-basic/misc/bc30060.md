# <a name="conversion-from-type1-to-type2-cannot-occur-in-a-constant-expression"></a>Konwersja z '\<type1 >' do '\<type2 >' nie może wystąpić w wyrażeniu stałym
Wyrażenie inicjowania w `Const` instrukcja oblicza wartość do typu danych, który nie może zostać przekonwertowany na deklarowany typ stałej.  
  
 **Identyfikator błędu:** BC30060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zainicjuj stała za pomocą wyrażenia typu danych, który może zostać przekonwertowany do typu zadeklarowany dla stałej.
