# <a name="conversion-from-39lttype1gt39-to-39lttype2gt39-cannot-occur-in-a-constant-expression"></a>Konwersja z &#39; &lt;type1&gt;&#39; do &#39;&lt; type2&gt;&#39; nie może wystąpić w wyrażeniu stałym
Wyrażenia inicjowania w `Const` instrukcji obliczane do typu danych, której nie można przekonwertować na zadeklarowany typ stałej.  
  
 **Identyfikator błędu:** BC30060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Inicjowanie stała z wyrażeniem typu danych, który może zostać przekonwertowany na typ zadeklarowany dla stałej.
