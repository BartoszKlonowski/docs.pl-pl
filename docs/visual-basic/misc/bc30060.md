# <a name="conversion-from-type1-to-type2-cannot-occur-in-a-constant-expression"></a>Konwersja z " \<type1> " na " \<type2> " nie może wystąpić w wyrażeniu stałym
Wyrażenie inicjacji w `Const` instrukcji zwraca typ danych, którego nie można przekonwertować na zadeklarowany typ stałej.  
  
 **Identyfikator błędu:** BC30060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zainicjuj stałą z wyrażeniem typu danych, które można przekonwertować na typ zadeklarowany dla stałej.
