# <a name="unrecognized-option-optionname-ignored"></a>Nierozpoznana opcja \<optionname >; zignorowano
Określono opcję, który nie jest prawidłowy w wierszu polecenia.  
  
 **Identyfikator błędu:** BC2007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podane opcje, aby znaleźć ten, który jest nieprawidłowy.
