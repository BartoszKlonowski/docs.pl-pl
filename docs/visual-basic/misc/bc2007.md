# <a name="unrecognized-option-optionname-ignored"></a>Nierozpoznana opcja \<optionname> ; zignorowano

Określono opcję, która nie jest prawidłowa w wierszu polecenia.  
  
 **Identyfikator błędu:** BC2007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź określone opcje, aby znaleźć te, które nie są prawidłowe.
