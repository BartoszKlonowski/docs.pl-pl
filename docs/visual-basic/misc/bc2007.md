# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>Nierozpoznana opcja &lt;optionname&gt;; zignorowano
Określono opcję, który nie jest prawidłowy w wierszu polecenia.  
  
 **Identyfikator błędu:** BC2007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź określonych opcji, aby znaleźć ten, który jest nieprawidłowy.
