# <a name="you-must-specify-a-path-that-is-under-the-current-folder-or-one-of-its-sub-folders"></a>Należy określić ścieżkę znajdującą się pod bieżącym folderem lub jednym z jej podfolderów
Określona ścieżka nie znajduje się w bieżącym katalogu. Może to być spowodowane błędnym analizowaniem ścieżek plików podczas dostarczania ścieżki względnej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że dostarczono poprawną ścieżkę.
