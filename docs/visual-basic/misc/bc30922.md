# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-outside-the-assembly"></a><span data-ttu-id="a6bfa-102">"&lt;derivedtypename&gt;" nie może dziedziczyć z &lt;typu&gt; "&lt;constructedbasetypename&gt;"ponieważ rozszerza on dostęp do typu"&lt;internaltypename&gt;"poza zestaw</span><span class="sxs-lookup"><span data-stu-id="a6bfa-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' outside the assembly</span></span>
<span data-ttu-id="a6bfa-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu ograniczonego typu przy użyciu go jako argument typu klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="a6bfa-104">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="a6bfa-105">Aby uzyskać dostęp do kodu spoza zestawu, jest niedozwolone. `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="a6bfa-106">Jednak `derivedClass` jest możliwy z kodu, który można odwoływać się do niego.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="a6bfa-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` używa `restrictedStructure` jako argument typu, ponieważ, może spowodować `restrictedStructure` do kodu w dowolnym zestawie.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="a6bfa-108">**Identyfikator błędu:** BC30922</span><span class="sxs-lookup"><span data-stu-id="a6bfa-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a6bfa-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a6bfa-109">To correct this error</span></span>  
  
-   <span data-ttu-id="a6bfa-110">Dostosuj poziomy dostępu dla klas lub interfejsów, więc typu pochodnego nie zwiększa poziom dostępu ograniczonego typu.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="a6bfa-111">—lub—</span><span class="sxs-lookup"><span data-stu-id="a6bfa-111">-or-</span></span>  
  
-   <span data-ttu-id="a6bfa-112">Jeśli nie można dostosować poziomów dostępu, nie należy używać w typie ograniczonym jako argument typu podczas tworzenia klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="a6bfa-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
