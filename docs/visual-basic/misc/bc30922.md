# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="1cfaa-102">element " \<derivedtypename> " nie może dziedziczyć po elemencie \<type> " \<constructedbasetypename> ", ponieważ rozszerza on dostęp typu " \<internaltypename> " poza zestaw</span><span class="sxs-lookup"><span data-stu-id="1cfaa-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="1cfaa-103">Klasa pochodna lub interfejs próbuje rozszerzyć poziom dostępu ograniczonego typu przy użyciu go jako argumentu typu dla klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="1cfaa-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="1cfaa-104">Poniższy kod może wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="1cfaa-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="1cfaa-105">Nie można uzyskać dostępu do kodu poza zestawem `restrictedStructure` .</span><span class="sxs-lookup"><span data-stu-id="1cfaa-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="1cfaa-106">Można jednak `derivedClass` uzyskać do niego dostęp z dowolnego kodu, który może odwoływać się do niego.</span><span class="sxs-lookup"><span data-stu-id="1cfaa-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="1cfaa-107">W związku z tym, `derivedClass` nie można dziedziczyć, `baseClass` Jeśli używa `restrictedStructure` jako argumentu typu, ponieważ może uwidaczniać `restrictedStructure` kod w dowolnym zestawie.</span><span class="sxs-lookup"><span data-stu-id="1cfaa-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="1cfaa-108">**Identyfikator błędu:** BC30922</span><span class="sxs-lookup"><span data-stu-id="1cfaa-108">**Error ID:** BC30922</span></span>

- <span data-ttu-id="1cfaa-110">Dostosuj poziomy dostępu klas lub interfejsów, aby typ pochodny nie rozszerzał poziomu dostępu ograniczonego typu.</span><span class="sxs-lookup"><span data-stu-id="1cfaa-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="1cfaa-111">-lub-</span><span class="sxs-lookup"><span data-stu-id="1cfaa-111">-or-</span></span>
