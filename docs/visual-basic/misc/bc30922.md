# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="8da94-102">"\<derivedtypename >" nie może dziedziczyć z \<typu > "\<constructedbasetypename >", ponieważ rozszerza on dostęp do typu "\<internaltypename >" poza zestaw</span><span class="sxs-lookup"><span data-stu-id="8da94-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="8da94-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu ograniczonego typu przy użyciu go jako argument typu klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8da94-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="8da94-104">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="8da94-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="8da94-105">Aby uzyskać dostęp do kodu spoza zestawu, jest niedozwolone. `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="8da94-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="8da94-106">Jednak `derivedClass` jest możliwy z kodu, który można odwoływać się do niego.</span><span class="sxs-lookup"><span data-stu-id="8da94-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="8da94-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` używa `restrictedStructure` jako argument typu, ponieważ, może spowodować `restrictedStructure` do kodu w dowolnym zestawie.</span><span class="sxs-lookup"><span data-stu-id="8da94-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="8da94-108">**Identyfikator błędu:** BC30922</span><span class="sxs-lookup"><span data-stu-id="8da94-108">**Error ID:** BC30922</span></span>

-   <span data-ttu-id="8da94-112">Jeśli nie można dostosować poziomów dostępu, nie należy używać w typie ograniczonym jako argument typu podczas tworzenia klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8da94-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
