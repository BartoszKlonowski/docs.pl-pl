# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-outside-the-assembly"></a><span data-ttu-id="27c04-102">&#39;&lt;derivedtypename&gt; &#39; nie może dziedziczyć po &lt;typu&gt; &#39; &lt;constructedbasetypename&gt; &#39; ponieważ rozszerza on dostęp do typu &#39; &lt; internaltypename&gt; &#39; poza zestaw</span><span class="sxs-lookup"><span data-stu-id="27c04-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; outside the assembly</span></span>
<span data-ttu-id="27c04-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu typu ograniczonego wykorzystując ją jako argument typu klasy podstawowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="27c04-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="27c04-104">Poniższy kod może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="27c04-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="27c04-105">Kod poza zestaw nie jest dozwolony dostęp do `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="27c04-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="27c04-106">Jednak `derivedClass` są dostępne z kodu, który odwoływać się do niej.</span><span class="sxs-lookup"><span data-stu-id="27c04-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="27c04-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` , gdy jest używana `restrictedStructure` jako argument typu, ponieważ, która może spowodować narażenie `restrictedStructure` kod w dowolnym zestawu.</span><span class="sxs-lookup"><span data-stu-id="27c04-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="27c04-108">**Identyfikator błędu:** BC30922</span><span class="sxs-lookup"><span data-stu-id="27c04-108">**Error ID:** BC30922</span></span>

-   <span data-ttu-id="27c04-110">Dostosuj poziomy dostępu do interfejsów lub klas, więc typu pochodnego nie zwiększa poziom dostępu do ograniczonego typu.</span><span class="sxs-lookup"><span data-stu-id="27c04-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="27c04-111">—lub—</span><span class="sxs-lookup"><span data-stu-id="27c04-111">-or-</span></span>  
  
-   <span data-ttu-id="27c04-112">Jeśli nie można dostosować poziomów dostępu, nie należy używać w typie ograniczonym jako argument typu podczas tworzenia klasy podstawowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="27c04-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
