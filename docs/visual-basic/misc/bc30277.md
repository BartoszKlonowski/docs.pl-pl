# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>Znak typu " \<charactername> " nie jest zgodny z zadeklarowanym typem danych " \<type> "
Zmienna jest zadeklarowana przy użyciu jednego typu danych, ale odwołuje się do znaku typu reprezentującego niezgodny typ danych, na przykład `K$ = 10` gdy `K` jest zadeklarowany `Integer` .  
  
 **Identyfikator błędu:** BC30277  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień zadeklarowany typ danych zmiennej lub Zmień lub Usuń znak typu w odwołaniu.
