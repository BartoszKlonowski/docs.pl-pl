# <a name="type-character-ltcharacternamegt-does-not-match-declared-data-type-lttypegt"></a>Wpisz znak "&lt;charactername&gt;"nie odpowiada zadeklarowanemu typowi danych"&lt;typu&gt;"
Zmienna jest zadeklarowana za pomocą jednego typu danych, ale określonych znakiem typ reprezentujący niezgodny typ danych, na przykład `K$ = 10` podczas `K` zadeklarowano `Integer`.  
  
 **Identyfikator błędu:** BC30277  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ danych zadeklarowany zmiennej, lub zmień lub usuń znak typu odwołania.
