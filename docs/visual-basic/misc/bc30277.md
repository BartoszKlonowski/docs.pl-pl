# <a name="type-character-39ltcharacternamegt39-does-not-match-declared-data-type-39lttypegt39"></a>Wpisz znak &#39; &lt;charactername&gt; &#39; nie odpowiada zadeklarowanemu typowi danych &#39; &lt;typu&gt;&#39;
Zmienna została zadeklarowana z jednego typu danych, lecz określonych ze znakiem typu reprezentujący niezgodny typ danych, na przykład `K$ = 10` podczas `K` zadeklarowano `Integer`.  
  
 **Identyfikator błędu:** BC30277  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ danych zadeklarowany zmiennej, lub zmień lub usuń znak typu odwołania.
