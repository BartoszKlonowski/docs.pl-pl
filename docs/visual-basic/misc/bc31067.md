# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Nie można zadeklarować metody w strukturze, "Protected" lub "Protected Friend"
Metody w strukturze zadeklarowano jako `Protected` lub `Protected Friend`.  
  
 **Identyfikator błędu:** BC31067  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Metody wewnątrz struktury jako `Public` lub `Private`.
