# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Nie można zadeklarować metody w strukturze jako "Protected" lub "Protected Friend"
Metoda w strukturze została zadeklarowana jako `Protected` lub `Protected Friend` .  
  
 **Identyfikator błędu:** BC31067  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Deklaruj metody wewnątrz struktur jako `Public` lub `Private` .
