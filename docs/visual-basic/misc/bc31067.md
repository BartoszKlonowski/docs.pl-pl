# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Nie można zadeklarować metody w strukturze &#39;chronione&#39; lub &#39;Protected Friend&#39;
Metody w strukturze zadeklarowano jako `Protected` lub `Protected Friend`.  
  
 **Identyfikator błędu:** BC31067  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie metody wewnątrz struktury jako `Public` lub `Private`.
