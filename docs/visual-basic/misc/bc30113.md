# <a name="39ltnamespacegt39-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>&#39;&lt;przestrzeń nazw&gt; &#39; nie jest prawidłową nazwą i nie można użyć jako nazwy głównej przestrzeni nazw
Kwalifikujące nazwę przestrzeni nazw nie jest prawidłową nazwę elementu. Nazwa może zawierać tylko znaki alfabetyczne, cyfry dziesiętne i podkreślenia i musi zaczynać się literą lub znakiem podkreślenia (`_`).  
  
 **Identyfikator błędu:** BC30113  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli przestrzeń nazw jest częścią zestawu zagnieżdżonych obszarów nazw, sprawdź, czy jakieś zewnętrzna nazw zawierają nieprawidłowe znaki.
