# <a name="ltnamespacegt-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>"&lt;przestrzeni nazw&gt;" nie jest prawidłową nazwą i nie można użyć jako główna nazwa przestrzeni nazw
Kwalifikującym się nazwą przestrzeni nazw nie jest prawidłową nazwę elementu. Nazwa musi zawierać tylko znaki alfabetu, cyfry dziesiętne i podkreślenia i musi zaczynać się od litery lub znaku podkreślenia (`_`).  
  
 **Identyfikator błędu:** BC30113  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli przestrzeń nazw jest częścią zestawu zagnieżdżone przestrzenie nazw, należy sprawdzić, jeśli jakieś zewnętrznej nazwy zawierają nieprawidłowe znaki.
