# <a name="namespace-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>" \<namespace> " nie jest prawidłową nazwą i nie można jej użyć jako nazwy głównej przestrzeni nazw
Nazwa kwalifikującej przestrzeni nazw nie jest prawidłową nazwą elementu. Nazwa może zawierać tylko litery, cyfry dziesiętne i znaki podkreślenia, a także musi zaczynać się od litery lub znaku podkreślenia ( `_` ).  
  
 **Identyfikator błędu:** BC30113  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli przestrzeń nazw jest częścią zestawu zagnieżdżonych obszarów nazw, sprawdź, czy dowolna z nazw zewnętrznych zawiera nieprawidłowe znaki.
