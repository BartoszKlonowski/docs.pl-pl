# <a name="39ltmembernamegt39-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>&#39;&lt;membername&gt; &#39; nie jest zadeklarowany lub zawierający go moduł nie jest ładowany w sesji debugowania
Nie można odnaleźć określonego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30699  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy element członkowski jest zdefiniowany i wpisana poprawnie.  
  
2.  Uzyskanie dostępu do elementów członkowskich zadeklarowany w module. W niektórych przypadkach debugowania środowiska nie można zlokalizować elementów członkowskich, ponieważ nie zostały jeszcze załadowane moduły, w którym je zadeklarowano.
