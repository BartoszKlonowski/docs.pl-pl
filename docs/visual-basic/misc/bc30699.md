# <a name="ltmembernamegt-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>"&lt;membername&gt;" nie został zadeklarowany lub moduł zawierający ją nie jest załadowana w sesji debugowania
Nie można odnaleźć określonego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30699  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy element członkowski jest zdefiniowany i wpisana poprawnie.  
  
2.  Uzyskanie dostępu do elementów członkowskich zadeklarowanych w module. W niektórych przypadkach środowisko debugowania nie można zlokalizować elementów członkowskich, ponieważ nie zostały jeszcze załadowane moduły, w której są deklarowane.
