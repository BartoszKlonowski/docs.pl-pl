# <a name="definition-of-method-method-is-not-accessible-in-this-context"></a>Definicja metody \<method> nie jest dostępna w tym kontekście

Funkcja agregująca, która nie istnieje, została dołączona do `Into` operatora.  
  
 **Identyfikator błędu:** BC36594  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź pisownię nazwy funkcji agregującej i wprowadź wszelkie niezbędne poprawki.  
  
- Dodaj metodę jako niestandardową funkcję agregującą (Metoda rozszerzenia). Aby uzyskać więcej informacji na temat tworzenia niestandardowej funkcji agregującej, zobacz [klauzula Aggregate](../language-reference/queries/aggregate-clause.md).
