# <a name="definition-of-method-ltmethodgt-is-not-accessible-in-this-context"></a>Definicja metody &lt;metody&gt; nie jest dostępny w tym kontekście
Funkcji agregującej, która nie istnieje została uwzględniona w `Into` operatora.  
  
 **Identyfikator błędu:** BC36594  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź pisownię nazwy funkcji agregowania i wprowadź konieczne poprawki.  
  
-   Dodaj metodę jako niestandardowej funkcji agregującej (— metoda rozszerzenia). Aby uzyskać więcej informacji na temat tworzenia niestandardowych funkcji agregującej, zobacz [Aggregate — klauzula](../../visual-basic/language-reference/queries/aggregate-clause.md).
