# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="f371d-102">Literał XML nie może występować w tym miejscu, o ile nie jest ujęta w nawiasy</span><span class="sxs-lookup"><span data-stu-id="f371d-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="f371d-103">Literał deklaracja XML jest używany w wyrażeniu w lokalizacji, która jest niejednoznaczny dla kompilatora języka Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="f371d-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="f371d-104">Oznacza to, że kompilator Visual Basic nie można ustalić czy mniej-niż znak (<) ma pełnić rolę operatora porównawczych lub początku literał XML.</span><span class="sxs-lookup"><span data-stu-id="f371d-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="f371d-105">Poniższy kod stanowi przykład:</span><span class="sxs-lookup"><span data-stu-id="f371d-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="f371d-106">**Identyfikator błędu:** BC31198</span><span class="sxs-lookup"><span data-stu-id="f371d-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f371d-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="f371d-107">To correct this error</span></span>

```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   <span data-ttu-id="f371d-109">Zmień swój kod, aby odwołać się do istniejącego literał XML, jak pokazano w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="f371d-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>

```vb  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
