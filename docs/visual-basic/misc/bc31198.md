# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="2210d-102">Literał XML nie może pojawić się w tym miejscu, chyba że jest ujęty w nawiasy</span><span class="sxs-lookup"><span data-stu-id="2210d-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="2210d-103">Deklaracja literału XML jest używana w wyrażeniu w lokalizacji, która jest niejednoznaczna dla kompilatora Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="2210d-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="2210d-104">Oznacza to, że kompilator Visual Basic nie może określić, czy znak mniejszości (<) jest przeznaczony jako operator porównawczy, czy początek literału XML.</span><span class="sxs-lookup"><span data-stu-id="2210d-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="2210d-105">Poniższy kod zawiera przykład:</span><span class="sxs-lookup"><span data-stu-id="2210d-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="2210d-106">**Identyfikator błędu:** BC31198</span><span class="sxs-lookup"><span data-stu-id="2210d-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2210d-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="2210d-107">To correct this error</span></span>  
  
- <span data-ttu-id="2210d-108">Ujmij deklarację literału XML w nawiasy, jak pokazano w następującym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="2210d-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
- <span data-ttu-id="2210d-109">Zmodyfikuj swój kod, aby odwołać się do istniejącego literału XML, jak pokazano w następującym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="2210d-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
