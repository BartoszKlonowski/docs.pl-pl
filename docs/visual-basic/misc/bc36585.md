# <a name="ltelementnamegt-is-not-a-parameter-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>"&lt;elementname&gt;"nie jest parametrem metody rozszerzenia"&lt;methodname&gt;"zdefiniowane w"&lt;typename&gt;"
Wywołanie procedury dostarcza dodatkowy argument, który nie jest zdefiniowany w metodzie rozszerzenia.  
  
 **Identyfikator błędu:** BC36585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędne argumentu z listy argumentów.
