# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<elementname >' nie jest parametrem metody rozszerzenia'\<methodname > 'zdefiniowane w'\<typename > '
Wywołanie procedury dostarcza dodatkowy argument, który nie jest zdefiniowany w metodzie rozszerzenia.  
  
 **Identyfikator błędu:** BC36585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędne argumentu z listy argumentów.
