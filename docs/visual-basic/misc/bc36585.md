# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>element " \<elementname> " nie jest parametrem metody rozszerzenia " \<methodname> " zdefiniowanej w elemencie " \<typename> "

Wywołanie procedury dostarcza dodatkowy argument, który nie jest zdefiniowany przez metodę rozszerzenia.  
  
 **Identyfikator błędu:** BC36585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędny argument z listy argumentów.
