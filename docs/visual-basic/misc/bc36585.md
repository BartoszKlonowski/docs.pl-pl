# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39; &lt;elementname&gt;&#39; nie jest parametrem metody rozszerzającej &#39;&lt; methodname&gt;&#39; zdefiniowanego w &#39;&lt; Właściwość TypeName&gt;&#39;
Wywołanie procedury udostępnia dodatkowy argument, który nie jest zdefiniowany przez metodę rozszerzenie.  
  
 **Identyfikator błędu:** BC36585  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędny argumentu z listy argumentów.
