# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>&#39;&lt;Nazwa&gt; &#39; nie jest elementem członkowskim &#39; &lt;classname&gt;&#39;
Element członkowski, który podano nie jest elementem członkowskim klasy.  
  
 **Identyfikator błędu:** BC30456  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź nazwę elementu członkowskiego, aby upewnić się, że są dokładne.  
  
2.  Użyj rzeczywistego elementu członkowskiego klasy.
