# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-lttypegt"></a>"!" wymaga, aby lewy argument operacji zawierał parametr typu, klasę lub typ interfejsu, ale ten argument operacji ma typ "&lt;typu&gt;"
Dostęp do elementu członkowskiego słownik, który używa znaku wykrzyknika (`!`) zamiast okres (`.`), jest prawidłowa tylko na klasę lub interfejs.  
  
 **Identyfikator błędu:** BC30103  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamień na wyrażenie po lewej stronie wykrzyknika taki, który ocenia do zdefiniowanej klasy lub typ interfejsu.
