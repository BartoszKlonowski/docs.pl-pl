# <a name="property-propertyname-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-type-typename-that-is-not-default"></a>Właściwość '\<propertyname >' nie można zadeklarować 'Default', ponieważ zastępuje właściwość na podstawie \<typu > '\<typename >' to znaczy nie domyślne
Podjęto próbę deklarowanie właściwości `Default`, który nie jest prawidłową deklaracji ponieważ zastępuje właściwość, która nie została zadeklarowana `Default` w klasie bazowej.  
  
 **Identyfikator błędu:** BC30504  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Default` deklaracji.
