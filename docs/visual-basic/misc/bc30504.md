# <a name="property-ltpropertynamegt-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-lttypegt-lttypenamegt-that-is-not-default"></a>Właściwość "&lt;propertyname&gt;" nie można zadeklarować "Default", ponieważ zastępuje właściwość na podstawie &lt;typu&gt; "&lt;typename&gt;" to znaczy nie domyślne
Podjęto próbę deklarowanie właściwości `Default`, który nie jest prawidłową deklaracji ponieważ zastępuje właściwość, która nie została zadeklarowana `Default` w klasie bazowej.  
  
 **Identyfikator błędu:** BC30504  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Default` deklaracji.
