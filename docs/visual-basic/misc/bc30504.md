# <a name="property-39ltpropertynamegt39-cannot-be-declared-39default39-because-it-overrides-a-property-on-the-base-lttypegt-39lttypenamegt39-that-is-not-default"></a>Właściwość &#39; &lt;propertyname&gt; &#39; nie można zadeklarować &#39;domyślne&#39; ponieważ zastępuje właściwość w podstawowym &lt;typu&gt; &#39; &lt;typename&gt; &#39; czyli nie domyślne
Podjęto próbę deklarowanie właściwości `Default`, która nie jest prawidłową deklaracji ponieważ zastępuje właściwość, która nie została zadeklarowana `Default` dla klasy podstawowej.  
  
 **Identyfikator błędu:** BC30504  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Default` deklaracji.
