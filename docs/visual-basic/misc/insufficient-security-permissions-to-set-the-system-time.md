# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Niewystarczające uprawnienia zabezpieczeń, aby ustawić czas systemowy

Uprawnienia zabezpieczeń są niewystarczające, aby ustawić czas systemowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień uprawnienia zabezpieczeń.
