# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Niewystarczającymi uprawnieniami, aby ustawić czas systemowy
Uprawnienia zabezpieczeń są za mało, aby ustawić czas systemowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmienić uprawnienia zabezpieczeń.
