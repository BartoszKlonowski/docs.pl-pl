# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Uprawnienia zabezpieczeń nie wystarczają do ustawienia godziny systemowej
Uprawnienia zabezpieczeń są niewystarczające do ustawienia godziny systemowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmienianie uprawnień zabezpieczeń.
