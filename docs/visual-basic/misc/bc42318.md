# <a name="xml-comment-type-parameter-must-have-a-name-attribute"></a>Parametr typu komentarza XML musi mieć atrybut "name"
Parametr typu komentarza XML musi mieć atrybut "name". Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML wymaga `name` atrybut, który jest brakujące lub niekompletne.  
  
 **Identyfikator błędu:** BC42318  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodawanie lub wykonaj wymagane `name` atrybut do tagu.
