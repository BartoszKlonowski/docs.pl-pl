# <a name="xml-comment-type-parameter-must-have-a-39name39-attribute"></a>Parametr typu komentarza XML musi mieć &#39;nazwa&#39; atrybutu
Parametr typu komentarza XML musi mieć atrybut "name". Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML wymaga `name` atrybut, który jest brakujące lub niekompletne.  
  
 **Identyfikator błędu:** BC42318  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj lub wykonaj wymagane `name` atrybut do tagu.
