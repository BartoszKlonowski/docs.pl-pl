# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>Zadeklarowano już parametr o nazwie &#39; &lt;argumentname&gt;&#39;
Deklaracja procedury definiuje dwa argumenty o takiej samej nazwie. Wszystkie nazwy argumentu w deklaracji procedury muszą być unikatowe.  
  
 **Identyfikator błędu:** BC30237  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę jednego z argumentów.
