# <a name="parameter-already-declared-with-name-argumentname"></a>Zadeklarowano już parametr o nazwie " \<argumentname> "
Deklaracja procedury definiuje dwa argumenty o tej samej nazwie. Wszystkie nazwy argumentów w deklaracji procedury muszą być unikatowe.  
  
 **Identyfikator błędu:** BC30237  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę jednego z argumentów.
