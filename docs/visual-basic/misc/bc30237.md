# <a name="parameter-already-declared-with-name-argumentname"></a>Parametr zadeklarowano już o nazwie '\<argumentname >'
Deklaracja procedury definiuje dwa argumenty o takiej samej nazwie. Wszystkie nazwy argumentu, w deklaracji procedury muszą być unikatowe.  
  
 **Identyfikator błędu:** BC30237  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę jednej z podanych argumentów.
