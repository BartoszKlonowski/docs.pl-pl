# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-zero"></a>Argument &#39; &lt;argumentname&gt;&#39; musi być większa lub równa zero
Argument jest nieprawidłowy, ponieważ jest ona mniejsza od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię argumentów. Nieprawidłowo zapisana nazwa zmiennej niejawnie można utworzyć liczbowa zmiennej, która jest ustawiana na wartość zero.  
  
2.  Sprawdź poprzednich operacji na zmiennych w wyrażeniu, zwłaszcza tych, które przekazany do procedury jako argumenty z innych procedur.
