# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument '\<argumentname >' musi być większa lub równa zero
Argument jest nieprawidłowy, ponieważ ona jest mniejsza niż zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię argumentów. Nieprawidłowo zapisana nazwa zmiennej może niejawnie utworzyć zmienną liczbową, której wartość jest ustawiana na zero.  
  
2.  Sprawdź poprzednie operacje na zmiennych w wyrażeniu, zwłaszcza tych, które przekazano do procedury jako argumenty z innych procedur.
