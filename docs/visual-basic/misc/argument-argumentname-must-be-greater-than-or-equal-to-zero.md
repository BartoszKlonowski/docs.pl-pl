# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument " \<argumentname> " musi być większy lub równy zero
Argument jest nieprawidłowy, ponieważ jest mniejszy od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię argumentów. Błędnie wpisana nazwa zmiennej może niejawnie utworzyć zmienną numeryczną, która została zainicjowana do zera.  
  
2. Sprawdź poprzednie operacje na zmiennych w wyrażeniu, szczególnie te przekazane do procedury jako argumenty z innych procedur.
