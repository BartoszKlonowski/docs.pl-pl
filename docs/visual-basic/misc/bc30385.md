# <a name="39ltmodifiergt39-is-not-valid-on-a-delegate-declaration"></a>&#39; &lt;modyfikator&gt;&#39; nie jest prawidłowy w deklaracji obiektu delegowanego
Podjęto próbę użycia modyfikator, takich jak `Shared`, w deklaracji obiektu delegowanego. Obiekty delegowane są obiekty używane do wywoływania metody innych obiektów, definiując konstruktora, który jest przekazywany specyfikację metody obiektu. Nie jest prawidłowy, aby określić modyfikatora w deklaracji obiektu delegowanego.  
  
 **Identyfikator błędu:** BC30385  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń modyfikator.
