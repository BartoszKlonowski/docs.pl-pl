# <a name="xml-processing-instruction-name-name-is-not-valid"></a>Nazwa instrukcji przetwarzania XML "name" jest nieprawidłowa
Nazwa parametru w instrukcji przetwarzania XML jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC31174  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszelkie błędy nazwę instrukcji przetwarzania XML lub usuń go.
