# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>Nazwa instrukcji przetwarzania XML &#39;nazwa&#39; jest nieprawidłowy
Podana nazwa dla instrukcji przetwarzania XML jest nieprawidłowy.  
  
 **Identyfikator błędu:** BC31174  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Popraw błędy w nazwach instrukcji przetwarzania XML, lub usuń go.
