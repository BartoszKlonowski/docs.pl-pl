# <a name="xml-processing-instruction-name-name-is-not-valid"></a>Nazwa instrukcji przetwarzania XML "name" jest nieprawidłowa
Nazwa podana dla instrukcji przetwarzania XML jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC31174  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Popraw wszystkie błędy w nazwie instrukcji przetwarzania XML lub usuń ją.
