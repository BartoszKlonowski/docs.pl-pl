# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-it-could-be-identical-to-interface-39ltinterfacename2gt39-from-which-the-interface-39ltinterfacename3gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="1a286-102">Nie można dziedziczyć interfejsu &#39; &lt;interfacename1&gt; &#39; ponieważ może on być taki sam jak interfejs &#39; &lt;interfacename2&gt; &#39; z którego interfejs &#39; &lt;interfacename3&gt; &#39; dziedziczy dla niektórych argumentów typu</span><span class="sxs-lookup"><span data-stu-id="1a286-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because it could be identical to interface &#39;&lt;interfacename2&gt;&#39; from which the interface &#39;&lt;interfacename3&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="1a286-103">Interfejs ogólny dziedziczy z co najmniej dwa interfejsy ogólne i dwa spadków może powodować konflikt niektórych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="1a286-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="1a286-104">Poniższe instrukcje może spowodować wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="1a286-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface

    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="1a286-105">Jeśli `derivedInterface` jest tworzony lub zaimplementowana podanie tego samego typu zarówno `t1` i `t2`, musi on dziedziczyć dwóch wersji `interfaceA` z argumentami typu identyczne.</span><span class="sxs-lookup"><span data-stu-id="1a286-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="1a286-106">Dzięki temu dałby w efekcie niejednoznaczności wersji do uzyskania dostępu.</span><span class="sxs-lookup"><span data-stu-id="1a286-106">Doing so would produce an ambiguity about which version to access.</span></span>
