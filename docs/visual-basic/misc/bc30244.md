# <a name="ltmodifiergt-is-not-valid-on-a-declare"></a>"&lt;modyfikator&gt;" jest nieprawidłowy w elemencie Declare
A `Declare` instrukcja zawiera nieprawidłowe słowo kluczowe `ReadOnly`.  
  
 **Identyfikator błędu:** BC30244  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowe słowo kluczowe z `Declare` instrukcji.
