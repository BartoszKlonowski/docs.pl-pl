# <a name="modifier-is-not-valid-on-a-declare"></a>element " \<modifier> " nie jest prawidłowy w deklaracji

`Declare`Instrukcja zawiera nieprawidłowe słowo kluczowe, np `ReadOnly` ..  
  
 **Identyfikator błędu:** BC30244  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe słowo kluczowe z `Declare` instrukcji.
