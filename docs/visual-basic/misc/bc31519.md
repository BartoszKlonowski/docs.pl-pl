# <a name="ltfilenamegt-cannot-be-referenced-because-it-is-not-an-assembly"></a>"&lt;filename&gt;" nie może być przywoływany, ponieważ nie jest zestawem
Odwołano się do pliku, który nie jest zestawem .NET.  
  
 **Identyfikator błędu:** BC31519  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź nazwę i lokalizację zestawu.  
  
2.  Sprawdź, czy zestaw nie jest uszkodzony.
