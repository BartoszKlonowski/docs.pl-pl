# <a name="unable-to-get-type-information-for-typename"></a>Nie można pobrać informacji o typie dla " \<typename> "

Odwołuje się do typu, który nie został załadowany przez środowisko uruchomieniowe języka wspólnego.  
  
 **Identyfikator błędu:** BC30751  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Restrukturyzacja kodu, tak aby typ został zdefiniowany w bieżącym zakresie.
