# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>Nie można pobrać informacji o typie &#39; &lt;typename&gt;&#39;
Odwoływano typu, który nie został załadowany przez środowisko uruchomieniowe języka wspólnego.  
  
 **Identyfikator błędu:** BC30751  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja kodu, więc typ jest zdefiniowany w bieżącym zakresie.
