# <a name="unable-to-get-type-information-for-lttypenamegt"></a>Nie można pobrać informacji o typie dla '&lt;typename&gt;"
Typ odwoływano, który nie został załadowany przez środowisko uruchomieniowe języka wspólnego.  
  
 **Identyfikator błędu:** BC30751  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja kodu, więc typ jest zdefiniowany w bieżącym zakresie.
