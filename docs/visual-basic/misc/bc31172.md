# <a name="an-embedded-expression-may-not-be-used-here"></a>Wyrażenie osadzone nie można użyć w tym miejscu
Wyrażenia osadzonego została uwzględniona w nieobsługiwanej lokalizacji w postaci literału ciągu XML. Należy pamiętać, że wyrażenia osadzone nie może służyć do określenia wartości przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC31172  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usunięcie wyrażenia osadzonego z nieobsługiwanej lokalizacji.
