# <a name="an-embedded-expression-may-not-be-used-here"></a>Wyrażenia osadzone nie może być używany tutaj
Wyrażenia osadzone została uwzględniona w miejscu nieobsługiwany literał XML. Należy pamiętać, że wyrażenia osadzone nie może służyć do określania wartości przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC31172  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń osadzony wyrażenia z nieobsługiwanej lokalizacji.
