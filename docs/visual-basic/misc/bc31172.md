# <a name="an-embedded-expression-may-not-be-used-here"></a>Nie można użyć wyrażenia osadzonego w tym miejscu

Wyrażenie osadzone zostało dołączone w nieobsługiwanej lokalizacji w literale XML. Należy zauważyć, że wyrażenia osadzone nie mogą być używane do określania wartości przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC31172  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wyrażenie osadzone z nieobsługiwanej lokalizacji.
