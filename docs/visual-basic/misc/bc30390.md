# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' nie jest dostępny w tym kontekście, ponieważ jest on'\<modyfikator > '
Podjęto próbę uzyskania dostępu do metody, która nie jest dostępna w tym kontekście ze względu na modyfikator, który został do niego zastosowany.  
  
 **Identyfikator błędu:** BC30390  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
