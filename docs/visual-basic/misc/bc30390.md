# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39; &lt;qualifiedmethod&gt;&#39; nie jest dostępny w tym kontekście, ponieważ jest on &#39;&lt; Modyfikator&gt;&#39;
Próbowano uzyskać dostęp do metody, który nie jest dostępny w tym kontekście z powodu modyfikator, która została zastosowana do niego.  
  
 **Identyfikator błędu:** BC30390  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikator.
