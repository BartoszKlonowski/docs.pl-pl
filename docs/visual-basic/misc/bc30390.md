# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' nie jest dostępny w tym kontekście, ponieważ jest on'\<modyfikator > '
Próbowano uzyskać dostęp do metody, która nie jest dostępna w tym kontekście, ponieważ modyfikator, która została zastosowana do niego.  
  
 **Identyfikator błędu:** BC30390  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
