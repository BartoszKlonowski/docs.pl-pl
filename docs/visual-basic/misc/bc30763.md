# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Docelowa wersja platformy .NET Compact Framework nie obsługuje przy użyciu modyfikator Ansi, Auto lub Unicode
Nie obsługuje wersji platformy .NET Compact Framework pracujesz z `Ansi`, `Auto`, i `Unicode` modyfikatorów.  
  
 **Identyfikator błędu:** BC30763  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
