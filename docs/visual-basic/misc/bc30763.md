# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Doprzeznaczona wersja .NET Compact Framework nie obsługuje użycia modyfikatora ANSI, autolub Unicode.
Wersja .NET Compact Framework, z którą pracujesz, nie obsługuje `Ansi` `Auto` `Unicode` modyfikatorów, i.  
  
 **Identyfikator błędu:** BC30763  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
