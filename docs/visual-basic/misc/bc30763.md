# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Docelowa wersja programu .NET Compact Framework nie obsługuje przy użyciu modyfikatora Ansi, Auto ani Unicode
Wersja programu .NET Compact Framework pracujesz nie obsługuje `Ansi`, `Auto`, i `Unicode` modyfikatorów.  
  
 **Identyfikator błędu:** BC30763  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikator.
