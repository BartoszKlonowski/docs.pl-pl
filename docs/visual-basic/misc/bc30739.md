# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39; RaiseEvent &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`RaiseEvent`Nie można używać w **Immediate** okna.  
  
 **Identyfikator błędu:** BC30739  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Test `RaiseEvent` instrukcji z kodu w **edytora kodu**.
