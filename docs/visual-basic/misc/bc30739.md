# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "RaiseEvent" nie są prawidłowe w oknie bezpośrednim
instrukcji `RaiseEvent` nie można używać w oknie **bezpośrednim** .  
  
 **Identyfikator błędu:** BC30739  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przetestuj `RaiseEvent` instrukcji z kodu w **edytorze kodu**.
