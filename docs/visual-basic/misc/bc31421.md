# <a name="ltmembergt-is-already-declared-in-this-structure"></a>"&lt;elementu członkowskiego&gt;" jest już zadeklarowany w tej strukturze
Element członkowski wyliczenia został zadeklarowany więcej niż jeden raz w tej strukturze.  
  
 **Identyfikator błędu:** BC31421  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń deklarację nadmiarowe.
