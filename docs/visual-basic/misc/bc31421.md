# <a name="member-is-already-declared-in-this-structure"></a>element "\<member >" jest już zadeklarowany w tej strukturze
Składowa wyliczenia została zadeklarowana więcej niż raz w tej strukturze.  
  
 **Identyfikator błędu:** BC31421  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń niezbędną deklarację.
