# <a name="39ltmembergt39-is-already-declared-in-this-structure"></a>&#39;&lt;element członkowski&gt; &#39; jest już zadeklarowany w tej strukturze
Element członkowski wyliczenia zadeklarowano więcej niż jeden raz w tej struktury.  
  
 **Identyfikator błędu:** BC31421  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe deklarację.
