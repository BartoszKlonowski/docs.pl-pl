 <span data-ttu-id="f7997-110">Ten błąd może wystąpić, jeśli dwie klasy mają taką samą nazwę i znajdują się w zagnieżdżone przestrzenie nazw za pomocą podkreślenia w ich nazwy.</span><span class="sxs-lookup"><span data-stu-id="f7997-110">This error can occur if two classes have the same name and are in nested namespaces with underscores in their names.</span></span> <span data-ttu-id="f7997-111">Gdy kompilator tworzy nowe nazwy właściwości klasy, jest nadal kolizji nazw.</span><span class="sxs-lookup"><span data-stu-id="f7997-111">When the compiler constructs the new property names for the classes, there is still a name collision.</span></span>
