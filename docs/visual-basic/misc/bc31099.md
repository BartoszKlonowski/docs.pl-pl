# <a name="property-accessors-cannot-be-declared-keyword"></a>Metod dostępu do właściwości nie można zadeklarować jako '\<— słowo kluczowe >'
A `Get` lub `Set` procedury deklaracja zawiera słowo kluczowe, które nie jest prawidłowy w procedury właściwości.  
  
 [Instrukcja Get](../../visual-basic/language-reference/statements/get-statement.md) i [instrukcji Set](../../visual-basic/language-reference/statements/set-statement.md) Zezwalaj modyfikatora dostępu (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **Identyfikator błędu:** BC31099  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe słowo kluczowe z `Get` lub `Set` instrukcji.
