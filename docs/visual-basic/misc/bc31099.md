# <a name="property-accessors-cannot-be-declared-39ltkeywordgt39"></a>Nie można zadeklarować metody dostępu właściwości &#39; &lt;— słowo kluczowe&gt;&#39;
A `Get` lub `Set` procedury deklaracja zawiera słowo kluczowe, które nie jest prawidłowy w procedury właściwości.  
  
 [Instrukcja Get](../../visual-basic/language-reference/statements/get-statement.md) i [Instrukcja Set](../../visual-basic/language-reference/statements/set-statement.md) Zezwalaj modyfikator dostępu (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **Identyfikator błędu:** BC31099  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowe słowo kluczowe z `Get` lub `Set` instrukcji.
