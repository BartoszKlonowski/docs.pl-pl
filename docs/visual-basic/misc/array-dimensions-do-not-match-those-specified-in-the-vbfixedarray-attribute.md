# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Wymiary tablicy nie pasują do określonych w atrybucie "VBFixedArray"
Wymiary tablicy nie są zgodne z wymiarami określonymi w `VBFixedArray` atrybucie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Utwórz Wymiary tablicy w instrukcji Array `Dim` zgodnej z wymiarami określonymi `VBFixedArray` przez atrybut.
