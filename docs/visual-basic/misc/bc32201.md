# <a name="property-parameters-cannot-have-the-name-value"></a>Parametry właściwości nie mogą mieć nazwy "value"

Parametr w `Property` instrukcji ma nazwę `Value` . Ta nazwa jest zarezerwowana w ramach definicji właściwości dla nowej wartości przypisanej do właściwości w `Set` procedurze.  
  
 **Identyfikator błędu:** BC32201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę parametru w `Property` instrukcji.
