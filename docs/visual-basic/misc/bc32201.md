# <a name="property-parameters-cannot-have-the-name-39value39"></a>Parametry właściwości nie mogą mieć nazwę &#39;wartości&#39;
Parametr w `Property` nosi nazwę instrukcji `Value`. Ta nazwa jest zastrzeżony w obrębie nowej wartości, które są przypisywane do właściwości w definicji właściwości `Set` procedury.  
  
 **Identyfikator błędu:** BC32201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę parametru w `Property` instrukcji.
