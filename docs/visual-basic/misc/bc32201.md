# <a name="property-parameters-cannot-have-the-name-value"></a>Parametry właściwości nie może mieć nazwy "Value"
Parametr w `Property` nosi nazwę instrukcji `Value`. Ta nazwa jest zastrzeżona w obrębie definicji właściwości nowej wartości, które są przypisywane do właściwości w `Set` procedury.  
  
 **Identyfikator błędu:** BC32201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę parametru w `Property` instrukcji.
