# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>Właściwości w module nie mogą być zadeklarowane jako " \<specifier> "

Podjęto próbę użycia specyfikatora, takiego jak `MustOverride` , `NotOverridable` , lub `Overridable` , we właściwości w module.  
  
 **Identyfikator błędu:** BC30503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
