# <a name="properties-in-a-module-cannot-be-declared-ltspecifiergt"></a>Właściwości w Module nie można zadeklarować jako "&lt;specyfikator&gt;"
Próbujesz użyć specyfikatora, takich jak `MustOverride`, `NotOverridable`, lub `Overridable`, we właściwości w module.  
  
 **Identyfikator błędu:** BC30503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
