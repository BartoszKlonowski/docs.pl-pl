# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować jako właściwości w Module &#39; &lt;specyfikator&gt;&#39;
Próbowano użyć specyfikatora, takiego jak `MustOverride`, `NotOverridable`, lub `Overridable`, we właściwości w module.  
  
 **Identyfikator błędu:** BC30503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
