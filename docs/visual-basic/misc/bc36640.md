# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-lambda-expression"></a>Nie można użyć wystąpienia typu ograniczonego " \<typename> " w wyrażeniu lambda

Wystąpienia typów z ograniczeniami, takie jak <xref:System.ArgIterator> , <xref:System.RuntimeArgumentHandle> i <xref:System.TypedReference> , nie mogą być używane w wyrażeniu lambda lub wyrażeniu zapytania.  
  
 **Identyfikator błędu:** BC36640
