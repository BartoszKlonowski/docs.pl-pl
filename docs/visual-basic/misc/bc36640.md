# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-lambda-expression"></a>Wystąpienie o typie ograniczonym &#39; &lt;typename&gt;&#39; nie można użyć w wyrażeniu lambda
Wystąpienia ograniczenia typów, takie jak <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, i <xref:System.TypedReference>, nie można użyć w wyrażeniu lambda lub wyrażeniu zapytania.  
  
 **Identyfikator błędu:** BC36640
