# <a name="line-ltnumbergt-cannot-be-parsed-using-the-current-fieldwidths"></a>Wiersz &lt;numer&gt; nie można analizować przy użyciu bieżącego parametru FieldWidths
Nie można analizować określonego wiersza, ponieważ jego pól szerokości innych niż określone.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostosuj `FieldWidths` wiersza może zostać poprawnie zanalizowana lub Wstaw kod obsługi wyjątków, aby umożliwić obsługę wiersza.
