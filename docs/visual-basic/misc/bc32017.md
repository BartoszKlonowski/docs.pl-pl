# <a name="comma-3939-or-a-valid-expression-continuation-expected"></a>Przecinek &#39;) &#39; lub prawidłowej kontynuacji wyrażenia oczekiwano
Listy, na przykład listy argumentów w definicji procedurze lub wywołania procedury nie zostało prawidłowo rozdzielić lub zakończyć jej elementów członkowskich. Jedną z możliwych przyczyn jest dwóch członków listy są oddzielone tylko spacji ani spacji.  
  
 **Identyfikator błędu:** BC32017

-   Użyj przecinka (,) do oddzielania elementów członkowskich listy i nawiasów zamykających ()), aby przerwać na liście.
