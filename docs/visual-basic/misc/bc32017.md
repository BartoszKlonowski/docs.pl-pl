# <a name="comma--or-a-valid-expression-continuation-expected"></a>Oczekiwano przecinka, znaku ")" lub prawidłowej kontynuacji wyrażenia
Lista, taka jak lista argumentów w definicji procedury lub wywołania procedury, nie jest prawidłowo oddzielona ani nie przerywa jej składowych. Jedną z możliwych przyczyn jest to, że dwie składowe listy są oddzielane tylko spacjami lub spacjami.  
  
 **Identyfikator błędu:** BC32017  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj przecinka (,), aby oddzielić elementy członkowskie listy i nawiasy zamykające (), aby zakończyć listę.
