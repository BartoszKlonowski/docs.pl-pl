# <a name="comma--or-a-valid-expression-continuation-expected"></a>Przecinek, ")", lub utrzymania prawidłowe wyrażenie oczekuje
Listy, na przykład listy argumentów w wywołaniu procedury, lub w definicji procedury nie został poprawnie rozdzielić lub zakończenia jej członków. Jedną z możliwych przyczyn jest dwóch członków listy są oddzielone tylko spacją ani spacji.  
  
 **Identyfikator błędu:** BC32017  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zakończenie na liście, należy użyć przecinka (,) do oddzielenia listę elementów członkowskich i zamykającym nawiasem kwadratowym ()).
