# <a name="argument-nper-must-be-greater-than-zero"></a>Argument "NPer" musi być większa niż zero
`NPer` Funkcji, która zwraca `Double` określającą liczbę okresów renty w oparciu o okresowe płatności w stałej kwocie i stałej stopie wymaga argumentu większą niż zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź pisownię argumentów w wyrażeniu. Nieprawidłowo zapisana nazwa zmiennej może niejawnie utworzyć zmienną liczbową, której wartość jest ustawiana na zero.  
  
- Sprawdź poprzednie operacje na zmiennych w wyrażeniu, zwłaszcza tych, które przekazano do procedury jako argumenty z innych procedur.
