# <a name="argument-39nper39-must-be-greater-than-zero"></a>Argument &#39;NPer&#39; musi być większa od zera.
`NPer` Funkcji, która zwraca `Double` określającą liczbę okresów renty wieczystej w oparciu o okresowe płatności w stałej kwocie i przy stałej stopie procentowej wymaga argumentu jest większa niż zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź pisownię argumentów w wyrażeniu. Nieprawidłowo zapisana nazwa zmiennej może niejawnie utworzyć zmienną liczbową, której wartość jest ustawiana na zero.  
  
-   Sprawdź poprzednie operacje na zmiennych w wyrażeniu, zwłaszcza tych, które przekazano do procedury jako argumenty z innych procedur.
