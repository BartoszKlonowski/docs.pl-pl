# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Wartość typu &#39; &lt;type1&gt;&#39; nie można przekonwertować na &#39;&lt; type2&gt;&#39; ponieważ typy tablic mają różną liczbę wymiarów
Nastąpiła próba konwersji do innego z różną liczbę wymiarów tablicy.  
  
 **Identyfikator błędu:** BC30414  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień tablice jeden lub oba, aby odpowiadały liczby wymiarów.
