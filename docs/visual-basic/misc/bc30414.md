# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Wartość typu "&lt;type1&gt;"nie można przekonwertować na"&lt;type2&gt;", ponieważ typy tablic mają różną liczbę wymiarów
Nastąpiła próba konwersji jednej tablicy do innej z różną liczbę wymiarów.  
  
 **Identyfikator błędu:** BC30414  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmienić tablic jeden lub oba, aby odpowiadały liczbę wymiarów.
