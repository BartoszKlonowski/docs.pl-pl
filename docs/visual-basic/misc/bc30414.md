# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>\<type1>Nie można przekonwertować wartości typu "" na " \<type2> ", ponieważ typy tablic mają różną liczbę wymiarów.

Podjęto próbę przekonwertowania jednej tablicy na inną o inną liczbę wymiarów.  
  
 **Identyfikator błędu:** BC30414  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień jedną lub obie tablice tak, aby liczba dopasowań wymiarów.
