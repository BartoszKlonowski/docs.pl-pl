# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Wartość typu "\<type1 >" nie można przekonwertować na "\<type2 >", ponieważ typy tablic mają różną liczbę wymiarów
Nastąpiła próba konwersji jednej tablicy do innej z różną liczbę wymiarów.  
  
 **Identyfikator błędu:** BC30414  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmienić tablic jeden lub oba, aby odpowiadały liczbę wymiarów.
