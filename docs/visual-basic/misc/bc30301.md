# <a name="39ltmethod1gt39-and-39ltmethod2gt39-cannot-overload-each-other-because-they-differ-only-by-return-types"></a>&#39;&lt;metoda1&gt; &#39; i &#39; &lt;metoda2&gt; &#39; nie mogą przeciążać siebie nawzajem, ponieważ różnią się jedynie zwracanymi typami
Podjęto próbę przeciążyć metodę z innej metody, która różni się od pierwszego tylko w jej typu zwracanego. Dotyczące przeciążania, wszystkie wersje dwie metody musi rozróżnianie przez ich listy argumentów; Nie można używać żadnych innych niż listy argumentów do odróżnienia metody.  
  
 **Identyfikator błędu:** BC30301  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że te metody są zróżnicowane przez ich listy argumentów.
