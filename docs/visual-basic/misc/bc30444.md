# <a name="39set39-is-already-declared"></a>&#39; Ustaw &#39; jest już zadeklarowana
A `Set` instrukcji występuje więcej niż raz w bloku kodu. `Set` Deklaruje instrukcji `Set` procedury właściwości używane do przypisania wartości dla właściwości.  
  
 **Identyfikator błędu:** BC30444  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `Set` instrukcji.
