# <a name="set-is-already-declared"></a>Element "Set" jest już zadeklarowany

`Set`Instrukcja pojawia się więcej niż raz w bloku kodu. `Set`Instrukcja deklaruje `Set` procedurę właściwości używaną do przypisywania wartości do właściwości.  
  
 **Identyfikator błędu:** BC30444  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `Set` .
