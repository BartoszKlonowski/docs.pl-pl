# <a name="set-is-already-declared"></a>"Set" jest już zadeklarowana
A `Set` instrukcji występuje więcej niż raz w bloku kodu. `Set` Instrukcja deklaruje `Set` procedury właściwości używane do przypisywania wartości do właściwości.  
  
 **Identyfikator błędu:** BC30444  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `Set` instrukcji.
