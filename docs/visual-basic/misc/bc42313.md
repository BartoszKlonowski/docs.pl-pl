# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>Tag komentarza XML &#39; zwraca &#39; nie jest dozwolony w &#39; WriteOnly &#39; Właściwość
Tag komentarza XML "returns" jest niedozwolona dla właściwości "WriteOnly". Komentarz XML zostanie zignorowany.  
  
 Komentarz XML w deklaracji właściwości tylko do zapisu nie może zawierać \<zwraca > tagu.  
  
 **Identyfikator błędu:** BC42313  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwaną \<zwraca > tagu.
