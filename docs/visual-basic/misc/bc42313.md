# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>Tag komentarza XML "Returns" jest niedozwolony we właściwości "WriteOnly"
Tag komentarza XML "Returns" jest niedozwolony we właściwości "WriteOnly". Komentarz XML zostanie zignorowany.  
  
 Komentarz XML dla deklaracji właściwości tylko do zapisu nie może zawierać \<returns> znacznika.  
  
 **Identyfikator błędu:** BC42313  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwany \<returns> tag.
