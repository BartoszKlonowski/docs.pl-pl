# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>Tag komentarza XML &#39;zwraca&#39; nie jest dozwolona w &#39;WriteOnly&#39; właściwości
Tag komentarza XML "returns" jest niedozwolona dla właściwości "WriteOnly". Komentarz XML zostanie zignorowany.  
  
 Deklaracja właściwości tylko do zapisu w komentarzu XML nie może zawierać \<zwraca > tag.  
  
 **Identyfikator błędu:** BC42313  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane \<zwraca > tag.
