# <a name="warning-number-number-for-the-option-optionname-is-either-not-configurable-or-not-valid"></a>\<number>Nie można skonfigurować numeru ostrzeżenia "" dla opcji " \<optionname> " albo jest on nieprawidłowy

Identyfikator ostrzeżenia jest nieprawidłowy. Identyfikatory muszą być liczbami całkowitymi większymi od 0.  
  
 **Identyfikator błędu:** BC2026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podany identyfikator ostrzeżenia, aby upewnić się, że jest on prawidłowy.
