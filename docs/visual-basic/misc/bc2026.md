# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Ostrzeżenie o numerze "&lt;numer&gt;"dla opcji"&lt;optionname&gt;" to nie można skonfigurować lub jest on nieprawidłowy
Identyfikator ostrzeżenia jest nieprawidłowy. Identyfikatory muszą być wartości całkowite większe niż 0.  
  
 **Identyfikator błędu:** BC2026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź identyfikator ostrzeżenia dostarczane do upewnij się, że jest on prawidłowy.
