# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Numer ostrzeżenia &#39; &lt;numer&gt; &#39; opcji &#39; &lt;optionname&gt; &#39; nie można skonfigurować lub jest on nieprawidłowy
Identyfikator ostrzeżenia jest nieprawidłowy. Identyfikatory musi być większa niż 0. wartości całkowite.  
  
 **Identyfikator błędu:** BC2026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź identyfikator ostrzeżenia dostarczony do upewnij się, że jest ona prawidłowa.
