# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Ocena wyrażeń lambda nie jest prawidłowa w debugerze
Wyrażenia lambda został wprowadzony w **czujki** lub **Immediate** okno podczas sesji debugowania. Wyrażenia lambda nie są dostępne w debugowaniu systemu windows.  
  
 **Identyfikator błędu:** BC36627  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Testowanie wyrażenia lambda w edytorze kodu.
