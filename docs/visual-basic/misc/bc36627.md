# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Obliczenie wyrażenia lambda nie jest prawidłowe w debugerze
Wyrażenie lambda zostało wprowadzone w oknie **czujka** lub **natychmiast** podczas sesji debugowania. Wyrażenia lambda nie są dostępne w oknach debugowania.  
  
 **Identyfikator błędu:** BC36627  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przetestuj wyrażenia lambda w edytorze kodu.
