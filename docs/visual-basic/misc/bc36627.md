# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Obliczanie wyrażeń lambda jest nieprawidłowy w debugerze
Wyrażenie lambda został wprowadzony w **Obejrzyj** lub **bezpośrednie** okna podczas sesji debugowania. Wyrażenia lambda nie są dostępne podczas debugowania systemu windows.  
  
 **Identyfikator błędu:** BC36627  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Testowanie wyrażenia lambda w edytorze kodu.
