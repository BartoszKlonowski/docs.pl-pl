# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Alias instrukcji Imports\<name > 'powoduje konflikt z'\<Nazwa2 > 'zadeklarowana w głównej przestrzeni nazw
Nazwa aliasu, używane w `Imports` instrukcji powoduje konflikt z inną nazwą zadeklarowanych w głównej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC31403  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz unikatową nazwę aliasu posługują się `Imports` instrukcji.
