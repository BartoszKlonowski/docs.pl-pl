# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Alias instrukcji Imports&lt;nazwa&gt;"powoduje konflikt z"&lt;Nazwa2&gt;"zadeklarowana w głównej przestrzeni nazw
Nazwa aliasu, używane w `Imports` instrukcji powoduje konflikt z inną nazwą zadeklarowanych w głównej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC31403  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz unikatową nazwę aliasu posługują się `Imports` instrukcji.
