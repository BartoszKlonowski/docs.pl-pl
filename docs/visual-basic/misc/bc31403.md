# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Alias "" Imports " \<name> " powoduje konflikt z \<name2> elementem "" zadeklarowanym w głównej przestrzeni nazw

Nazwa aliasu używana w `Imports` instrukcji powoduje konflikt z inną nazwą zadeklarowaną w głównej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC31403  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz unikatową nazwę aliasu używanego przez `Imports` instrukcję.
