# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Imports alias &#39; &lt;nazwa&gt;&#39; powoduje konflikt z &#39;&lt; Nazwa2&gt;&#39; zadeklarowany w głównej przestrzeni nazw
Nazwa aliasu używane w `Imports` instrukcji powoduje konflikt z inną nazwą zadeklarowany w głównej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC31403  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz unikatową nazwę aliasu używane przez `Imports` instrukcji.
