# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literały XML i właściwości osi XML są niedostępne
Literały XML i właściwości osi XML nie są dostępne. Dodaj odwołania do zestawów System.Xml, System.Xml.Linq i System.Core.  
  
 Kod jest kompilowane zawiera literały XML i właściwości osi XML, ale nie odwołuje się do zestawów, które są wymagane do skompilowania literały XML i właściwości osi XML.  
  
 **Identyfikator błędu:** BC31190  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołania do zestawów System.Xml.dll, System.Xml.Linq.dll i System.Core.dll.
