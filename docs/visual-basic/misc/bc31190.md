# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literały XML i właściwości osi XML są niedostępne

Literały XML i właściwości osi XML są niedostępne. Dodaj odwołania do System.Xml, System.Xml. LINQ i system. Core.  
  
 Skompilowany kod zawiera literały XML lub właściwości osi XML, ale nie odwołuje się do zestawów, które są niezbędne do kompilowania literałów XML lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31190  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj odwołania do zestawów System.Xml.dll, System.Xml.Linq.dll i System.Core.dll.
