# <a name="event-eventname-event-specified-by-the-defaultevent-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Zdarzenie "\<eventname >" zdarzeń określony przez atrybut "DefaultEvent" nie jest zdarzeniem publicznie dostępnym dla tej klasy
<xref:System.ComponentModel.DefaultEventAttribute> Atrybut należy określić nazwę zdarzeniem publicznie dostępnym w klasie, do którego zastosowano atrybut.  
  
 **Identyfikator błędu:** BC30770  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że klasa definiuje zdarzeniem publicznie dostępnym.  
  
2.  Upewnij się, że nazwa zdarzenia w klasie odpowiada nazwie określonej przez <xref:System.ComponentModel.DefaultEventAttribute> atrybutu.
