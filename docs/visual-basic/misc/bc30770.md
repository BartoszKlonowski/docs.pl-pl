# <a name="event-39lteventnamegt39-event-specified-by-the-39defaultevent39-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Zdarzenia &#39; &lt;eventname&gt; &#39; zdarzeń określony przez &#39;DefaultEvent&#39; atrybut nie jest zdarzeniem publicznie dostępnym dla tej klasy
<xref:System.ComponentModel.DefaultEventAttribute> Atrybut musi określać nazwę publicznie zdarzenia w klasie, do którego zastosowano atrybut.  
  
 **Identyfikator błędu:** BC30770  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że klasa definiuje zdarzeniem publicznie dostępnym.  
  
2.  Upewnij się, że nazwa zdarzenia w klasie odpowiada nazwie określonej przez <xref:System.ComponentModel.DefaultEventAttribute> atrybutu.
