# <a name="field-or-property-39ltmembernamegt39-is-not-found"></a>Pole lub właściwość &#39; &lt;membername&gt; &#39; nie znaleziono
Niezdefiniowane pole lub właściwość odwołuje się do atrybutu.  
  
 **Identyfikator błędu:** BC30661  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię pola lub właściwości.  
  
2.  Sprawdź, czy projektu można uzyskiwanie dostępu do klasy atrybutu poprawne i dodaj go do odwołań projektu, jeśli to konieczne.
