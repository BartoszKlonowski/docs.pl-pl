# <a name="field-or-property-membername-is-not-found"></a>Pole lub właściwość "\<membername >' nie znaleziono
Niezdefiniowane pole lub właściwość odwołuje się do atrybutu.  
  
 **Identyfikator błędu:** BC30661  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię pole lub właściwość.  
  
2.  Sprawdź, czy projekt może dostępu poprawny atrybut klasy i dodać go do odwołań projektu, jeśli to konieczne.
