# <a name="field-or-property-membername-is-not-found"></a>Nie odnaleziono pola lub właściwości " \<membername> "

Odwołanie do niezdefiniowanego pola lub właściwości jest przywoływane dla atrybutu.  
  
 **Identyfikator błędu:** BC30661  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię pola lub właściwości.  
  
2. Sprawdź, czy Twój projekt może uzyskać dostęp do poprawnej klasy atrybutów i dodaj go do odwołań projektu, w razie potrzeby.
