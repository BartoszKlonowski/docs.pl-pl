# <a name="access-denied-to-name"></a>Odmowa dostępu do " \<name> "

Urządzenie, plik lub dysk nie jest dostępny, ponieważ uprawnienia zabezpieczeń komputera są niewystarczające.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że masz wymagane uprawnienia.
