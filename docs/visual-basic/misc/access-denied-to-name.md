# <a name="access-denied-to-name"></a>Odmowa dostępu do '\<name >'
Urządzenia, plik lub dysk nie jest dostępny, ponieważ nie są wystarczające uprawnienia zabezpieczeń komputera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że masz odpowiednie uprawnienia.
