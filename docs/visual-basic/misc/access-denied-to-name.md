# <a name="access-denied-to-39ltnamegt39"></a>Odmowa dostępu do &#39; &lt;nazwa&gt;&#39;
Urządzenia, plik lub dysk nie jest dostępny, ponieważ są niewystarczające uprawnienia zabezpieczeń tego komputera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd
