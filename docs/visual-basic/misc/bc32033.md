# <a name="39end-get39-must-be-the-first-statement-on-a-line"></a>&#39;End Get&#39; musi być pierwszą instrukcją w wierszu
`End Get` Instrukcji następuje separatora instrukcji dwukropka (:). `End Get` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
