# <a name="end-get-must-be-the-first-statement-on-a-line"></a>Instrukcja "End Get" musi być pierwszą instrukcją w wierszu

`End Get`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End Get` musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
