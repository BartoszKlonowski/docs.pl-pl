# <a name="end-get-must-be-the-first-statement-on-a-line"></a>"End Get" musi być pierwszą instrukcją w wierszu
`End Get` Instrukcji następuje separatorem instrukcji dwukropek (:). `End Get` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Oddziel wiele instrukcji na różne wiersze.
