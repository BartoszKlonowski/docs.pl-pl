# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-derived-from-type4"></a>Wartość typu '\<type1 >' nie można przekonwertować na '\<type2 >' ponieważ '\<type3 >' nie dziedziczy od '\<typ 4 >'
Instrukcję stara się przekonwertować na typ tablicy na inny typ tablicy w sytuacji, gdy typy danych poszczególnych elementów tablic nie są oba typy odwołań lub w przypadku, gdy konwersja, albo zwężająca lub poszerzająca, nie jest możliwe między typami element dwa tablice.  
  
 **Identyfikator błędu:** BC30332  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy danych elementów w obu tablicach, aby ustalić przyczynę błędu.
