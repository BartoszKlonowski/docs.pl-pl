# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-derived-from-type4"></a>\<type1>Nie można przekonwertować wartości typu "" na " \<type2> ", ponieważ element " \<type3> " nie pochodzi od elementu " \<type4> "

Instrukcja próbuje skonwertować typ tablicy do innego typu tablicy w sytuacji, w której typy danych elementów tablic nie są zarówno typami referencyjnymi, jak i w przypadku konwersji, rozszerzając lub Narrowing, nie jest możliwe między typami elementów dwóch tablic.  
  
 **Identyfikator błędu:** BC30332  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy danych elementów obu tablic, aby określić źródło błędu.
