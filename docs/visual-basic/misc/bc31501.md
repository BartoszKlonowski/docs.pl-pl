# <a name="readonly-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>'ReadOnly' atrybut właściwości '\<propertyfield >' nie może być elementem docelowym przypisania
Nastąpiła próba do przypisania wartości do `ReadOnly` właściwości w atrybucie.  
  
 **Identyfikator błędu:** BC31501  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń instrukcję przypisania właściwości.  
  
2. Jeśli za pomocą właściwości opracowany Usuń `ReadOnly` lub `Shared` Modyfikatory z właściwości atrybutów.
