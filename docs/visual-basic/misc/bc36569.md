# <a name="argument-not-specified-for-parameter-parametername"></a>Nie określono argumentu dla parametru "\<parametername >"
Nie podano argumentu dla wymaganego parametru.  
  
 **Identyfikator błędu:** BC36569  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy podać argument określony parametr.
