# <a name="cannot-find-net-framework-directory-error"></a>Nie można znaleźć katalogu .NET Framework: \<error>

Nie można zlokalizować katalogu, w którym są przechowywane zestawy .NET Framework.  
  
 **Identyfikator błędu:** BC31508  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Skompiluj ponownie program, aby zobaczyć, czy błąd jest powtarzany.  
  
2. Jeśli błąd będzie nadal występował, Zapisz swoją pracę i ponownie uruchom program Visual Studio.  
  
3. Jeśli błąd się powtarza, zainstaluj ponownie Visual Basic.  
  
4. Jeśli błąd będzie się powtarzać po ponownej instalacji, powiadom firmę Microsoft o pomocy technicznej.
