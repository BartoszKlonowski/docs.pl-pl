# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39; AddHandler &#39; Deklaracja musi kończyć się pasujący &#39; End AddHandler &#39;
`AddHandler` Deklaracji musi kończyć się `End AddHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31115  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `AddHandler` kończy deklaracji `End AddHandler` instrukcji.
