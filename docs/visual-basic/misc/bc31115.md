# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>Deklaracja "AddHandler" musi kończyć się pasującego "End AddHandler"
`AddHandler` Deklaracji musi się kończyć `End AddHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31115  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `AddHandler` deklaracji kończy się `End AddHandler` instrukcji.
