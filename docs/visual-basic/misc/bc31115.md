# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>Deklaracja "AddHandler" musi być zakończona odpowiadającą jej instrukcją "End AddHandler"

`AddHandler`Deklaracja musi kończyć się `End AddHandler` instrukcją.  
  
 **Identyfikator błędu:** BC31115  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `AddHandler` Deklaracja została zakończona `End AddHandler` instrukcją.
