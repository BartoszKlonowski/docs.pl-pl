# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specyfikatory i atrybuty nie są prawidłowe w tej instrukcji

Deklaracja, której nie można modyfikować, taka jak `Inherits` , zawiera co najmniej jeden atrybut lub jedno lub więcej słów kluczowych, takich `Public` jak lub `ReadOnly` . Tylko deklaracje modyfikowalne mogą zawierać takie słowa kluczowe lub atrybuty.  
  
 **Identyfikator błędu:** BC30193  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń atrybut lub słowo kluczowe z deklaracji.
