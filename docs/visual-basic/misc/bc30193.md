# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specyfikatory i atrybuty nie są prawidłowe w tej instrukcji
Deklaracja, której nie można modyfikować, takich jak `Inherits`, zawiera jeden lub więcej atrybutów lub co najmniej jednego słowa kluczowe, takie jak `Public` lub `ReadOnly`. Tylko modyfikowalnych deklaracje może zawierać takie słowa kluczowe lub atrybutów.  
  
 **Identyfikator błędu:** BC30193  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń atrybut lub słowo kluczowe z deklaracji.
