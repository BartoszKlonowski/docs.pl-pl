# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specyfikatory i atrybuty nie są prawidłowe w tej instrukcji
Deklaracja, która nie może być modyfikowany, takich jak `Inherits`, zawiera jeden lub więcej atrybutów lub jedno lub więcej słów kluczowych, takich jak `Public` lub `ReadOnly`. Tylko można modyfikować deklaracje mogą zawierać atrybuty lub słów kluczowych.  
  
 **Identyfikator błędu:** BC30193  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń atrybut lub słowo kluczowe z deklaracji.
