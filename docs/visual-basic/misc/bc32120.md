# <a name="cannot-inherit-interface-interfacename1-because-it-could-be-identical-to-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="92b11-102">Nie można dziedziczyć interfejsu " \<interfacename1> ", ponieważ może on być taki sam jak interfejs " \<interfacename2> " dla niektórych argumentów typu</span><span class="sxs-lookup"><span data-stu-id="92b11-102">Cannot inherit interface '\<interfacename1>' because it could be identical to interface '\<interfacename2>' for some type arguments</span></span>

<span data-ttu-id="92b11-103">Ogólny interfejs dziedziczy więcej niż jeden raz z innego interfejsu ogólnego i dwa dziedziczenia mogą powodować konflikt dla określonych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="92b11-103">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="92b11-104">Poniższe instrukcje mogą generować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="92b11-104">The following statements can generate this error.</span></span>  
  
 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="92b11-105">Jeśli `derivedInterface` jest konstruowany lub zaimplementowany, dostarczając ten sam typ do obu `t1` i `t2` , musi dziedziczyć dwie wersje `interfaceA` z identycznych argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="92b11-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="92b11-106">Wykonanie tej operacji spowodowałoby niejednoznaczność, do której wersji ma dostęp.</span><span class="sxs-lookup"><span data-stu-id="92b11-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="92b11-107">**Identyfikator błędu:** BC32120</span><span class="sxs-lookup"><span data-stu-id="92b11-107">**Error ID:** BC32120</span></span>

- <span data-ttu-id="92b11-109">Zmień jeden z argumentów typu dostarczonych do interfejsu pochodnego, aby nie wystąpił konflikt.</span><span class="sxs-lookup"><span data-stu-id="92b11-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>
