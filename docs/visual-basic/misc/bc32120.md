<span data-ttu-id="c9102-103">Ogólny interfejs dziedziczy więcej niż jeden raz inny interfejs ogólny i dwa spadki mogą powodować konflikt dla niektórych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="c9102-103">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 `End Interface`  
  
 <span data-ttu-id="c9102-105">Jeśli `derivedInterface` jest tworzony lub zaimplementować dostarczenie tego samego typu na wartość oba `t1` i `t2`, musi on dziedziczyć dwóch wersji `interfaceA` z argumentami typu identyczne.</span><span class="sxs-lookup"><span data-stu-id="c9102-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="c9102-106">Ten sposób dałby w efekcie niejednoznaczności dotyczące dostępu do wersji.</span><span class="sxs-lookup"><span data-stu-id="c9102-106">Doing so would produce an ambiguity about which version to access.</span></span>
