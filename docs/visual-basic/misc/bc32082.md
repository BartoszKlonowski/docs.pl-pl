<span data-ttu-id="b4ea4-103">Typ ogólny jest wywoływana z `MustInherit` klasy jako argument typu, podczas gdy odpowiedniego parametru typu jest zadeklarowana za pomocą `New` ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="b4ea4-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>
