# <a name="type-argument-typeargumentname-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-typeparametername"></a><span data-ttu-id="ff576-102">Argument typu " \<typeargumentname> " jest zadeklarowany jako "MustInherit" i nie spełnia ograniczenia "New" dla parametru typu " \<typeparametername> "</span><span class="sxs-lookup"><span data-stu-id="ff576-102">Type argument '\<typeargumentname>' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '\<typeparametername>'</span></span>

<span data-ttu-id="ff576-103">Typ ogólny jest wywoływany z `MustInherit` klasą jako argumentem typu, podczas gdy odpowiedni parametr typu jest zadeklarowany przy użyciu `New` ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="ff576-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="ff576-104">`New`Ograniczenie wymaga, aby typ przesłany przez odpowiedni argument typu musiał obsługiwać Tworzenie obiektów.</span><span class="sxs-lookup"><span data-stu-id="ff576-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="ff576-105">Jednakże Klasa *abstrakcyjna* , która jest klasą zadeklarowaną jako `MustInherit` , nie uwidacznia żadnych konstruktorów, ponieważ nie można utworzyć żadnych obiektów z niej.</span><span class="sxs-lookup"><span data-stu-id="ff576-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="ff576-106">**Identyfikator błędu:** BC32082</span><span class="sxs-lookup"><span data-stu-id="ff576-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ff576-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="ff576-107">To correct this error</span></span>  
  
1. <span data-ttu-id="ff576-108">Jeśli Klasa użyta w argumencie Type nie musi być abstrakcyjna, Usuń `MustInherit` słowo kluczowe z jego deklaracji.</span><span class="sxs-lookup"><span data-stu-id="ff576-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>  
  
2. <span data-ttu-id="ff576-109">Jeśli Klasa użyta w argumencie Type musi być abstrakcyjna, ale nie musi być używana do konstruowania typu ogólnego, a następnie przekazać inną klasę w argumencie typu.</span><span class="sxs-lookup"><span data-stu-id="ff576-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>  
  
3. <span data-ttu-id="ff576-110">Jeśli odpowiedni parametr typu nie musi tworzyć obiektów z tego typu, a następnie usunąć `New` ograniczenie z jego deklaracji.</span><span class="sxs-lookup"><span data-stu-id="ff576-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
