<span data-ttu-id="c90fa-103">Typ ogólny jest wywoływana z `MustInherit` klasy jako argument typu, podczas gdy odpowiedniego parametru typu jest zadeklarowana za pomocą `New` ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="c90fa-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="c90fa-104">`New` Ograniczenie wymaga, że przekazany argument odpowiedni typ typ musi obsługiwać tworzenia obiektów.</span><span class="sxs-lookup"><span data-stu-id="c90fa-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="c90fa-105">Jednak *abstrakcyjne* klasy, oznacza to, że klasy zadeklarowane jako `MustInherit`, nie ujawnia żadnych konstruktorów, ponieważ nie można utworzyć wszystkie obiekty od niego.</span><span class="sxs-lookup"><span data-stu-id="c90fa-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="c90fa-106">**Identyfikator błędu:** BC32082</span><span class="sxs-lookup"><span data-stu-id="c90fa-106">**Error ID:** BC32082</span></span>
