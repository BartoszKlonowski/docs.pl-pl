# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>Nie określono argumentu dla parametru "&lt;parametername&gt;"z"&lt;methodname&gt;"
Nie podano argumentu dla wymaganego parametru.  
  
 **Identyfikator błędu:** BC30455  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy podać argument określony parametr.
