# <a name="39widening39-and-39narrowing39-cannot-be-combined"></a>&#39;Rozszerzanie&#39; i &#39;Narrowing&#39; nie można łączyć
[Operator — instrukcja](../../visual-basic/language-reference/statements/operator-statement.md) Określa [Widening](../../visual-basic/language-reference/modifiers/widening.md) i [Narrowing](../../visual-basic/language-reference/modifiers/narrowing.md).  
  
 Podczas definiowania operatora konwersji muszą deklarować go jako `Widening` lub `Narrowing`. Są właściwości wykluczają się wzajemnie, więc nie można określić jednocześnie.  
  
 **Identyfikator błędu:** BC33001  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń albo `Widening` lub `Narrowing` — słowo kluczowe z `Operator` instrukcji. Należy określić jednego lub drugiego.
