# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>Komentarz XML musi być pierwszą instrukcją w wierszu

Komentarz XML musi być pierwszą instrukcją w wierszu. Komentarz XML zostanie zignorowany.  
  
 Komentarz XML został umieszczony po innym elemencie języka, co jest niedozwolone.  
  
 **Identyfikator błędu:** BC42302  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś komentarz XML do nowego wiersza.
