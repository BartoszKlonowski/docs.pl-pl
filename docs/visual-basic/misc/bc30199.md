# <a name="3939-expected"></a>&#39; (&#39; Oczekiwano
Lista argumentów w wywołaniu procedury nie zaczyna się od nawias otwierający (`(`) znaków.  
  
 **Identyfikator błędu:** BC30199  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj nawias otwierający znak na początku listy.
