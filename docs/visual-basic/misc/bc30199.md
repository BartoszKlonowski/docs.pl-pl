# <a name="-expected"></a>Oczekiwano znaku "("
Lista argumentów w wywołaniu procedury nie zaczyna się od znaku nawiasu otwierającego ( `(` ).  
  
 **Identyfikator błędu:** BC30199  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak nawiasu otwierającego na początku listy.
