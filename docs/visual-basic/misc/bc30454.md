# <a name="expression-is-not-a-method"></a>Wyrażenie nie jest metodą
Obiekt docelowy wywołania metody nie jest metodą.  
  
 **Identyfikator błędu:** BC30454  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że nie błędnie nazwę metody.
