# <a name="xml-namespace-prefix-39prefix39-is-not-defined"></a>Prefiks przestrzeni nazw XML &#39; prefiks &#39; nie zdefiniowano
Określono Niezdefiniowany prefiks przestrzeni nazw XML dla literału XML i właściwości osi XML.  
  
 **Identyfikator błędu:** BC31148  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Definiowanie prefiksu przestrzeni nazw XML przy użyciu `Imports` instrukcji.  
  
 —lub—  
  
-   Zmodyfikuj prefiks przestrzeni nazw XML dla literału XML lub właściwości osi XML do wcześniej zdefiniowanego prefiksu przestrzeni nazw XML.
