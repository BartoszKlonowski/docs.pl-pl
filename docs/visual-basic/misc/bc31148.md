# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>Nie zdefiniowano prefiksu przestrzeni nazw XML "prefiks"
Niezdefiniowany prefiks przestrzeni nazw XML została określona dla literał XML i właściwości osi XML.  
  
 **Identyfikator błędu:** BC31148  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Definiowanie prefiksu przestrzeni nazw XML przy użyciu `Imports` instrukcji.  
  
 —lub—  
  
- Zmodyfikuj prefiks przestrzeni nazw XML dla literał XML i właściwości osi XML do uprzednio zdefiniowany prefiks przestrzeni nazw XML.
