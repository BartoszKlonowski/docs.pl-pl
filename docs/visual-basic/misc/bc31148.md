# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>Nie zdefiniowano prefiksu przestrzeni nazw XML "prefix"

Określono niezdefiniowany prefiks przestrzeni nazw XML dla literału XML lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31148  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj prefiks przestrzeni nazw XML przy użyciu `Imports` instrukcji.  
  
 -lub-  
  
- Zmodyfikuj prefiks przestrzeni nazw XML dla literału XML lub właściwość osi XML na wcześniej zdefiniowany prefiks przestrzeni nazw XML.
