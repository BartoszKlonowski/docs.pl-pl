# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Wznów&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
Wznów instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30714  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Resume` oświadczenia debugowania kodu.
