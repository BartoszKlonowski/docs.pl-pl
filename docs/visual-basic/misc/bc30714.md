# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Resume" nie są prawidłowe w oknie bezpośrednim
Wznów instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30714  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Resume` instrukcji debugowania kodu.
