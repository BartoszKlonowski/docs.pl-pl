# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>Po znaku "Option" musi występować element "Compare", "Explicit", "wnioskowanie" lub "Strict"
Jedyne opcje, które można ustawić za pomocą `Option` instrukcji to `Compare` , `Explicit` , `Infer` , i `Strict` .  
  
 **Identyfikator błędu:** BC30206  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy słowo kluczowe opcji ( `Compare` , `Explicit` , `Infer` lub `Strict` ) jest błędne.  
  
- Sprawdź, czy `Option` instrukcja nie jest kompletna.
