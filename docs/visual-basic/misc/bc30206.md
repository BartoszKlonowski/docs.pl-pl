# <a name="39option39-must-be-followed-by-39compare39-39explicit39-39infer39-or-39strict39"></a>&#39;Opcja&#39; musi następować &#39;porównania&#39;, &#39;Explicit&#39;, &#39;Infer&#39;, lub &#39;Strict&#39;
Tylko opcje, które można ustawić z `Option` instrukcji są `Compare`, `Explicit`, `Infer`, i `Strict`.  
  
 **Identyfikator błędu:** BC30206  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy opcja — słowo kluczowe (`Compare`, `Explicit`, `Infer`, lub `Strict`) jest błędna.  
  
-   Sprawdź, czy `Option` instrukcji jest niekompletna.
