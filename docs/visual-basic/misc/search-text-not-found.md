# <a name="search-text-not-found"></a>Nie można odnaleźć wyszukiwania tekstu
Nie znaleziono określonego tekstu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy określony tekst jest poprawna.
