# <a name="search-text-not-found"></a>Nie można odnaleźć tekstu wyszukiwania
Nie znaleziono tekst, który określono.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy określony tekst jest poprawna.
