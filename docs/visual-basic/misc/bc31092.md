# <a name="paramarray-parameters-must-have-an-array-type"></a>Parametry ParamArray muszą być typem tablicy
Wystąpił problem podczas uzyskiwania dostępu do zestawu MSCorlib, co powoduje `ParamArray` parametrów w nieprawidłowego typu.  
  
 **Identyfikator błędu:** BC31092  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Skompiluj program, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
2.  Jeśli błąd będzie się powtarzał, Zapisz swoją pracę i ponownie uruchom [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Jeśli błąd będzie nadal występować po ponownej instalacji, powiadamia pomocy technicznej firmy Microsoft.
