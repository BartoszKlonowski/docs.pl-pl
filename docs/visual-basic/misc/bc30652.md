# <a name="reference-required-to-assembly-ltassemblynamegt-containing-the-type-ltmembernamegt"></a>Wymagane odwołanie do zestawu "&lt;assemblyname&gt;"zawierającego typ"&lt;membername&gt;"
Wymagane odwołanie do zestawu "\<nazwa_zestawu >" z typem "\<membername >". Dodaj je do projektu.  
  
 Projekt zawiera typ, który jest zależny od innego typu niż zestaw, który nie jest uwzględniony w odwołania do projektu.  
  
 **Identyfikator błędu:** BC30652  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołanie do projektu dla typów zależnych w projekcie.
