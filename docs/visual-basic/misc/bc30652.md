# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Wymagane odwołanie do zestawu &#39; &lt;assemblyname&gt; &#39; zawierający typ &#39; &lt;membername&gt;&#39;
Wymagane odwołanie do zestawu "\<assemblyname >" z typem "\<membername >". Dodaj je do projektu.  
  
 Projekt zawiera typ, który jest zależny od innego typu niż zestawu, który nie znajduje się w odwołaniach projektu.  
  
 **Identyfikator błędu:** BC30652  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołanie do projektu dla typów zależnych w projekcie.
