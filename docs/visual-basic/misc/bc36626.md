# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Wyrażenie lambda nie można przekonwertować na '\<typename >' ponieważ typ '\<typename >' jest zadeklarowana jako 'MustInherit' i nie można utworzyć
Instrukcja próbuje skonwertować wyrażenie lambda na typ, który może być tylko klasą bazową i nie może służyć do tworzenia wystąpienia.  
  
 **Identyfikator błędu:** BC36626
