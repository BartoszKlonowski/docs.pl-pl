# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-type-lttypenamegt-is-declared-mustinherit-and-cannot-be-created"></a>Wyrażenie lambda nie można przekonwertować na "&lt;typename&gt;" ponieważ typ "&lt;typename&gt;" jest zadeklarowana jako "MustInherit" i nie można utworzyć
Instrukcja próbuje przekonwertować wyrażenia lambda do typu, który może być tylko klasy bazowej i nie można użyć do utworzenia wystąpienia.  
  
 **Identyfikator błędu:** BC36626
