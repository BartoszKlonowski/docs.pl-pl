# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="a7aff-102">Wnioskowanie argumentu dla parametru typu nie powiodło się typu '\<typeparametername >' z '\<genericproceduresignature >'</span><span class="sxs-lookup"><span data-stu-id="a7aff-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="a7aff-103">**Identyfikator błędu:** BC32051</span><span class="sxs-lookup"><span data-stu-id="a7aff-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="a7aff-104">Wnioskowanie argumentu dla parametru typu nie powiodło się typu '\<typeparametername >' z '\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="a7aff-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="a7aff-105">Nie można wywnioskować argumentu typu z argumentu przekazanego do parametru '\<parametername >'.</span><span class="sxs-lookup"><span data-stu-id="a7aff-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="a7aff-106">Procedura ogólna jest wywoływana bez dostarczania żadnych argumentów typu, a kompilator nie może wywnioskować typu do przekazania do jednego z tych parametrów.</span><span class="sxs-lookup"><span data-stu-id="a7aff-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="a7aff-107">Zwykle podczas wywoływania procedury ogólnej należy podać argument typu dla każdego parametru typu, który definiuje procedura ogólna.</span><span class="sxs-lookup"><span data-stu-id="a7aff-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="a7aff-108">Jeśli nie podasz żadnych argumentów typu, kompilator próbuje wywnioskować typy do przekazania do parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="a7aff-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="a7aff-109">Jeśli kontekst wywołania zawiera sprzeczne informacje o typie danych dla parametru typu, wnioskowanie o typie kończy się niepowodzeniem.</span><span class="sxs-lookup"><span data-stu-id="a7aff-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="a7aff-110">Następujący kod może generować ten błąd:</span><span class="sxs-lookup"><span data-stu-id="a7aff-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="a7aff-111">W poprzednim przykładzie kompilator wnioskuje typ `Integer` dla `t` w oparciu o wartość 42 przekazaną do `arg2`.</span><span class="sxs-lookup"><span data-stu-id="a7aff-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="a7aff-112">Jednakże wnioskowanie wymagało, aby `arg1` było typu `Integer()`, czyli tablicą `Integer`, a wartość 6 przeniesiona do `arg1` nie jest zgodna z tym typem.</span><span class="sxs-lookup"><span data-stu-id="a7aff-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="a7aff-113">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a7aff-113">To correct this error</span></span>

- <span data-ttu-id="a7aff-114">Podaj argumenty typu do procedury ogólnej, aby kompilator nie musiał ich wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="a7aff-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>

- <span data-ttu-id="a7aff-115">Podaj normalne argumenty o typach, które pasują do tych z argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="a7aff-115">Supply normal arguments with types that match those of the type arguments.</span></span>
