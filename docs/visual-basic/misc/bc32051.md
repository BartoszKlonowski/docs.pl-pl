<span data-ttu-id="a4c96-103">Wnioskowanie argumentu dla parametru typu nie powiodło się typu '\<typeparametername >' z '\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="a4c96-103">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="a4c96-104">Nie można wywnioskować argumentu typu z argumentu przekazanego do parametru '\<parametername >'.</span><span class="sxs-lookup"><span data-stu-id="a4c96-104">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>  
  
 <span data-ttu-id="a4c96-105">Ogólna procedura jest wywoływana bez podawania argumenty typu, a kompilator nie może wywnioskować typ do przekazania do jednego z parametrów.</span><span class="sxs-lookup"><span data-stu-id="a4c96-105">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>  
  
 <span data-ttu-id="a4c96-106">Zwykle po wywołaniu procedury ogólne Podaj argument typu dla każdego parametru typu, który definiuje ogólny procedury.</span><span class="sxs-lookup"><span data-stu-id="a4c96-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="a4c96-107">Jeśli nie podasz żadnych argumentów typu, kompilator podejmuje próbę wnioskowanie typów, które mają być przekazane do parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="a4c96-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="a4c96-108">Jeśli kontekst wywołania udostępnia powodujące konflikt typu danych informacji dla parametru typu, wnioskowanie o typie zakończy się niepowodzeniem.</span><span class="sxs-lookup"><span data-stu-id="a4c96-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="a4c96-109">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="a4c96-109">The following code can generate this error.</span></span>  
  
```  
Public Sub doSomething(Of t)(ByVal arg1 As t(), ByVal arg2 As t)  
End Sub  
Call doSomething(6, 42)  
```  
  
 <span data-ttu-id="a4c96-110">W poprzednim przykładzie, kompilator wnioskuje typ `Integer` dla `t` na podstawie wartości 42 przekazany do `arg2`.</span><span class="sxs-lookup"><span data-stu-id="a4c96-110">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="a4c96-111">Jednak wymaga tego wnioskowania `arg1` typu `Integer()`, oznacza to, że tablica `Integer`, a wartość 6 przekazana do `arg1` jest niezgodna z tego typu.</span><span class="sxs-lookup"><span data-stu-id="a4c96-111">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>  
  
 <span data-ttu-id="a4c96-112">**Identyfikator błędu:** BC32051</span><span class="sxs-lookup"><span data-stu-id="a4c96-112">**Error ID:** BC32051</span></span>
