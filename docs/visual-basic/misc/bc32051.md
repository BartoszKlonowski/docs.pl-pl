 <span data-ttu-id="59e34-105">Procedury ogólne jest wywoływana bez podawania żadnych argumentów typu, a kompilator nie można wnioskować o typie do przekazania do jednego z parametrów.</span><span class="sxs-lookup"><span data-stu-id="59e34-105">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

  
 <span data-ttu-id="59e34-110">W powyższym przykładzie kompilator wnioskuje typ `Integer` dla `t` na podstawie wartości przekazanych do 42 `arg2`.</span><span class="sxs-lookup"><span data-stu-id="59e34-110">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="59e34-111">Jednak wymaga tego wnioskowania `arg1` typu `Integer()`, oznacza to, że tablica `Integer`, a wartość 6 przekazany do `arg1` tego typu nie jest zgodna.</span><span class="sxs-lookup"><span data-stu-id="59e34-111">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>
