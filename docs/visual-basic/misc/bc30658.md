# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>Nie można \<propertyname> znaleźć właściwości "" bez parametrów
Używany atrybut ma właściwość, która wymaga co najmniej jednego parametru.  
  
 **Identyfikator błędu:** BC30658  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Określ parametry dla właściwości.
