# <a name="property-ltpropertynamegt-with-no-parameters-cannot-be-found"></a>Właściwość "&lt;propertyname&gt;" bez parametrów nie można odnaleźć
Atrybut używany ma właściwość, która wymaga co najmniej jeden parametr.  
  
 **Identyfikator błędu:** BC30658  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Określ parametry właściwości.
