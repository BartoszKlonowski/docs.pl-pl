# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Właściwość &#39; &lt;propertyname&gt; &#39; bez parametrów nie można odnaleźć
Atrybut ma właściwość, która wymaga co najmniej jeden parametr.  
  
 **Identyfikator błędu:** BC30658  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Określ parametry właściwości.
