# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>Instrukcji DECLARE w module nie można zadeklarować jako " \<specifier> "
`Declare`Deklaracja zawiera specyfikator, który jest nieprawidłowy w `Module` deklaracji. Nie można tworzyć wystąpień modułów, nie obsługują dziedziczenia ani implementowania interfejsów.  
  
 **Identyfikator błędu:** BC30786  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
