# <a name="declare-statements-in-a-module-cannot-be-declared-ltspecifiergt"></a>Zadeklaruj instrukcji w Module nie można zadeklarować jako "&lt;specyfikator&gt;"
A `Declare` deklaracja zawiera specyfikator, który nie jest prawidłowy w ramach `Module` deklaracji. Moduły nigdy nie mogą zostać utworzone, nie obsługują dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30786  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
