# <a name="declare-statements-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Deklarowanie instrukcji w Module nie można zadeklarować &#39; &lt;specyfikator&gt;&#39;
A `Declare` deklaracja zawiera specyfikator, który nie jest prawidłowy w ramach `Module` deklaracji. Moduły nigdy nie można utworzyć wystąpienia, nie obsługuje dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30786  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
