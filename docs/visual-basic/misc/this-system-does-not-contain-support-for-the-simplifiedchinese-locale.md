# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Ten system nie obsługuje ustawień regionalnych SimplifiedChinese

Próbujesz użyć `VbStrConv.SimplifiedChinese` elementu członkowskiego wyliczenia, ale system nie obsługuje ustawień regionalnych w języku chińskim uproszczonym.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Uaktualnij maszynę do obsługi ustawień regionalnych uproszczonych w języku chińskim lub Uruchom aplikację na komputerze obsługującym ustawienia regionalne.
