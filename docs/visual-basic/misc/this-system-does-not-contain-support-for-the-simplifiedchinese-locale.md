# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Ten system nie obsługuje dla ustawień regionalnych SimplifiedChinese
Podjęto próbę użycia `VbStrConv.SimplifiedChinese` element członkowski wyliczenia, ale system nie obsługuje ustawień regionalnych chiński uproszczony.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Uaktualnienie komputera do obsługi ustawień regionalnych chiński uproszczony, lub uruchomić aplikację na komputerze, który obsługuje ustawień regionalnych.
