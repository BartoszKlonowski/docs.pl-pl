# <a name="class-ltclassnamegt-does-not-implement-the-systemcollectionsicollection-interface"></a>Klasy&lt;classname&gt;"nie implementuje interfejsu System.Collections.ICollection
Klasa nie implementuje `System.Collections.ICollection` interfejs, który definiuje metody rozmiar, wyliczenia i synchronizacji dla wszystkich kolekcji. Może być konieczne ponowne zainstalowanie programu Visual Studio.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zainstaluj ponownie program Visual Studio.
