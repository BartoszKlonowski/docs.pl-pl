# <a name="function-is-not-declared"></a>element " \<function> " nie jest zadeklarowany
element " \<function> " nie jest zadeklarowany. Funkcja została przeniesiona do przestrzeni nazw Microsoft. VisualBasic.  
  
 **Identyfikator błędu:** BC30818  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj odpowiedniej funkcji z `Microsoft.VisualBasic` przestrzeni nazw.
