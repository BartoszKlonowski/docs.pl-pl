# <a name="39ltfunctiongt39-is-not-declared"></a>&#39; &lt;funkcja&gt;&#39; nie jest zadeklarowana
"\<funkcja >' nie został zadeklarowany. Funkcja została przeniesiona do przestrzeni nazw Microsoft.VisualBasic.  
  
 **Identyfikator błędu:** BC30818  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Za pomocą odpowiedniej funkcji z `Microsoft.VisualBasic` przestrzeni nazw.
