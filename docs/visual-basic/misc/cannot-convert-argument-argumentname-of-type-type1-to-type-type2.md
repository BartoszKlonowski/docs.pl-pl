# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>Nie można przekonwertować argumentu "&lt;argumentname&gt;"typu"&lt;type1&gt;" na typ"&lt;type2&gt;"
Operacji konwersji próbuje przekonwertować na niezgodny typ argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że typ, do którego próbujesz przekonwertować argument jest prawidłowy.
