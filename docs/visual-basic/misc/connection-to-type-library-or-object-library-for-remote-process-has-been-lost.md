# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Utracono połączenie z biblioteką typów lub biblioteką obiektów dla procesu zdalnego
Utracono połączenie zdalnego procesu biblioteki lub biblioteki typów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Uruchom ponownie `Application` obiektu.  
  
2.  W polu okna dialogowego błędu, za pomocą których wprowadzono w tym temacie pomocy, kliknij przycisk **OK** do wyświetlenia **odwołania** okno dialogowe. Odwołanie utracone znajduje się od słowa "Brak", po lewej stronie.  
  
3.  Usuń odwołanie do utracone.  
  
4.  W **odwołania** okno dialogowe, zaznacz pole wyboru dla `Application` obiektu.
