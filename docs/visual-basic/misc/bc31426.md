# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Nazwy typu niewewnętrznego nie są dozwolone w wyrażeniach kompilacji warunkowej
Wyrażenie kompilacja warunkowa odwołuje się do typu wewnętrznej, ale dozwolone są tylko typy wewnętrzne.  
  
 Aby uzyskać więcej informacji na temat typu wewnętrznego w języku Visual Basic, zobacz [podsumowanie typów danych](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Identyfikator błędu:** BC31426  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp typem — wewnętrzne wewnętrznego typu.
