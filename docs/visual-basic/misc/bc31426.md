# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Nazwy typów innych niż wewnętrzne są niedozwolone w wyrażeniach kompilacji warunkowej

Wyrażenie kompilacji warunkowej odwołuje się do typu niewewnętrznego, ale dozwolone są tylko typy wewnętrzne.  
  
 Aby uzyskać więcej informacji o typie wewnętrznym w Visual Basic, zobacz [Podsumowanie typów danych](../language-reference/keywords/data-types-summary.md).  
  
 **Identyfikator błędu:** BC31426  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zastąp typ inny niż wewnętrzny typem wewnętrznym.
