# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Nazwy typu niewewnętrznego są niedozwolone w wyrażeniach kompilacji warunkowej
Kompilacja warunkowa wyrażenie odwołuje się do typu — wewnętrzne, ale dozwolone są tylko typy wewnętrzne.  
  
 Aby uzyskać więcej informacji na temat typu wewnętrznego w języku Visual Basic, zobacz [typy danych — Podsumowanie](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Identyfikator błędu:** BC31426  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp — wewnętrzne typu wewnętrznego typu.
