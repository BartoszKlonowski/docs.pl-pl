# <a name="lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>"&lt;typename&gt;" nie można użyć jako atrybutu, ponieważ jest on zadeklarowany jako "MustInherit"
Klasy atrybutów niestandardowych nie mogą być deklarowane jako `MustInherit`.  
  
 **Identyfikator błędu:** BC31506  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `MustInherit` modyfikator z atrybutów niestandardowych.
