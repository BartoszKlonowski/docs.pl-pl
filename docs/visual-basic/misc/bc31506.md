# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39;&lt;Właściwość TypeName&gt; &#39; nie można użyć jako atrybutu, ponieważ jest on zadeklarowany jako &#39;MustInherit&#39;
Niestandardowy atrybut klasy nie można zadeklarować jako `MustInherit`.  
  
 **Identyfikator błędu:** BC31506  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `MustInherit` modyfikator z atrybutów niestandardowych.
