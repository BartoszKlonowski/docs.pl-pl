# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>\<typename>nie można użyć elementu "" jako atrybutu, ponieważ jest on zadeklarowany jako "MustInherit"

Klas atrybutów niestandardowych nie można zadeklarować jako `MustInherit` .  
  
 **Identyfikator błędu:** BC31506  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `MustInherit` modyfikator z atrybutów niestandardowych.
