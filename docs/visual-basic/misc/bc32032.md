# <a name="end-function-must-be-the-first-statement-on-a-line"></a>Instrukcja "End Function" musi być pierwszą instrukcją w wierszu
`End Function`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End Function`musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32032  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
