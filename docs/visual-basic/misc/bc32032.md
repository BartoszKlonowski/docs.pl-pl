# <a name="end-function-must-be-the-first-statement-on-a-line"></a>"End Function" musi być pierwszą instrukcją w wierszu
`End Function` Instrukcji następuje separatorem instrukcji dwukropek (:). `End Function` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32032  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
