# <a name="39end-function39-must-be-the-first-statement-on-a-line"></a>&#39; Funkcja end &#39; musi być pierwszą instrukcją w wierszu
`End Function` Instrukcji następuje separatora instrukcji dwukropka (:). `End Function`musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32032  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
