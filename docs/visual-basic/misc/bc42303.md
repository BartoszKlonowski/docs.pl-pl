# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>Komentarz XML nie może występować w metodzie ani właściwości

Komentarz XML nie może występować w metodzie ani właściwości. Komentarz XML zostanie zignorowany. Bloki komentarza XML nie mogą występować w metodach lub właściwościach.  
  
 **Identyfikator błędu:** BC42303  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń blok komentarza lub przenieś go do bardziej odpowiedniej lokalizacji.
