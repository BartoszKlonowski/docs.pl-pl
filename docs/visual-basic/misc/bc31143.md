 <span data-ttu-id="9a349-105">Poniższy kod ilustruje konwersje nie powiodło się.</span><span class="sxs-lookup"><span data-stu-id="9a349-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="9a349-106">Delegat jest `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="9a349-106">The delegate is `FunDel`.</span></span>

Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="9a349-107">Funkcje następujące różnią się od `FunDel` w sposób, który spowoduje, że ten błąd.</span><span class="sxs-lookup"><span data-stu-id="9a349-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="9a349-108">Każda z następujących instrukcji przypisania powoduje błąd.</span><span class="sxs-lookup"><span data-stu-id="9a349-108">Each of the following assignment statements causes the error.</span></span>

    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.
