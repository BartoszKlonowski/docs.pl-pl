<span data-ttu-id="e7d88-103">Ten błąd występuje, gdy konwersja jest wymagany między metody i delegata, który nie jest możliwe.</span><span class="sxs-lookup"><span data-stu-id="e7d88-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="e7d88-104">Przyczyny tego błędu może być konwersji między parametrami lub, w przypadku metody i delegata funkcji, konwersję, zwracanej wartości.</span><span class="sxs-lookup"><span data-stu-id="e7d88-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="e7d88-105">Poniższy kod ilustruje konwersje nie powiodło się.</span><span class="sxs-lookup"><span data-stu-id="e7d88-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="e7d88-106">Delegat jest `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="e7d88-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="e7d88-107">Funkcje następujące różnią się od `FunDel` w sposób, który spowoduje, że ten błąd.</span><span class="sxs-lookup"><span data-stu-id="e7d88-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

```vb  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="e7d88-108">Każda z następujących instrukcji przypisania powoduje błąd.</span><span class="sxs-lookup"><span data-stu-id="e7d88-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb

    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2

```
