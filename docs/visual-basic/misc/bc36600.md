# <a name="range-variable-ltvariablegt-is-already-declared"></a>Zmienna zakresu &lt;zmiennej&gt; jest już zadeklarowana
Nazwa zmiennej zakresu określonego w `Select` klauzuli lub `Into` częścią `Aggregate`, `Group By`, lub `Group Join` klauzuli duplikaty nazwę zmiennej zakresu już określony w klauzuli zapytania.  
  
 **Identyfikator błędu:** BC36600  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że wszystkie zmienne zakresu w klauzuli zapytania mają unikatowe nazwy.
