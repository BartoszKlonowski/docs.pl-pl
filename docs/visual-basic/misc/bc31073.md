# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>" \<name1> " powoduje konflikt z \<type> nazwą o tej samej nazwie zdefiniowanej w elemencie " \<name2> "
Napotkano nazwę, która powoduje konflikt z typem o tej samej nazwie.  
  
 **Identyfikator błędu:** BC31073  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zadeklaruj unikatową nazwę.  
  
2. Użyj nawiasów kwadratowych wokół nazwy elementu członkowskiego powodującego konflikt.
