# <a name="ltname1gt-conflicts-with-a-lttypegt-by-the-same-name-defined-in-ltname2gt"></a>"&lt;Nazwa1&gt;" powoduje konflikt z &lt;typu&gt; tej samej nazwie, które są zdefiniowane w "&lt;Nazwa2&gt;"
Napotkano nazwę, która powoduje konflikt z typem o takiej samej nazwie.  
  
 **Identyfikator błędu:** BC31073  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zadeklaruj unikatową nazwę.  
  
2.  Użyj nawiasów kwadratowych wokół powodujące konflikt nazwy elementu członkowskiego.
