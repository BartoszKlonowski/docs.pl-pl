# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39;&lt;Nazwa1&gt; &#39; powoduje konflikt z &lt;typu&gt; o takiej samej nazwie, jak określono w &#39; &lt;Nazwa2&gt;&#39;
Napotkano nazwę, która powoduje konflikt z typem o takiej samej nazwie.  
  
 **Identyfikator błędu:** BC31073  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Deklarowanie unikatową nazwę.  
  
2.  Użyj nawiasy kwadratowe nazwę elementu członkowskiego powodujące konflikt.
