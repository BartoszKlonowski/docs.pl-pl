# <a name="end-statement-not-valid"></a>Instrukcja "End" jest nieprawidłowa

`End`Słowo kluczowe jest używane nieprawidłowo lub jest nieodpowiednie w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30678  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj `End` samego siebie, aby spowodować utratę zakresu procedury.  
  
2. Użyj `End` z innymi słowami kluczowymi, aby zakończyć blok kodu, taki jak struktura lub właściwość.
