# <a name="end-statement-not-valid"></a>Instrukcja "End" jest nieprawidłowy
`End` — Słowo kluczowe jest niepoprawnie użyte lub nie jest odpowiednia w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30678  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj `End` przez siebie, aby wywołać procedurę, aby utracić zakresu.  
  
2. Użyj `End` przy użyciu innych słów kluczowych, aby zakończyć blok kodu, takiego jak struktury lub właściwość.
