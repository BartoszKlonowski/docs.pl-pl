# <a name="39end39-statement-not-valid"></a>&#39;Końcowy&#39; instrukcja nie jest prawidłowy
`End` — Słowo kluczowe jest niepoprawnie użyta lub nie jest odpowiednia w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30678  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Użyj `End` samodzielnie spowodować procedury utratę zakresu.  
  
2.  Użyj `End` z innych słów kluczowych, aby zakończyć blok kodu na przykład struktury lub właściwości.
