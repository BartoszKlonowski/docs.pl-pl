# <a name="39catch39-must-end-with-a-matching-39end-try39"></a>&#39; CATCH &#39; musi być zakończona odpowiadającą jej instrukcją &#39; End Try &#39;
A `Catch` występuje instrukcja w kodzie bez odpowiadającego mu `End Try` instrukcji. `Catch`Instrukcje muszą być zawarte z `End Try` instrukcji.  
  
 **Identyfikator błędu:** BC30441  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Catch` instrukcji.  
  
2.  Dodaj `End Try` instrukcji, aby zakończyć blok.
