# <a name="expected-closing-39--gt39-for-xml-comment"></a>Oczekiwano zamykających znaków &#39;—&gt; &#39; dla komentarza XML
Literał komentarza XML nie zawiera znaków wymagane zamykającego "-->".  
  
 **Identyfikator błędu:** BC31161  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamykającego "-->" znaki na końcu literał komentarza XML.
