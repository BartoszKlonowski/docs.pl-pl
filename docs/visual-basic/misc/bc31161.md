# <a name="expected-closing----for-xml-comment"></a>Oczekiwano zamykającego elementu "-->" dla komentarza XML

W literale komentarza XML brakuje wymaganego zamykającego znaku "-->".  
  
 **Identyfikator błędu:** BC31161  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagane zamykające znaki "-->" na końcu literału komentarza XML.
