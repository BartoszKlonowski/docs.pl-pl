# <a name="expected-closing---gt-for-xml-comment"></a>Oczekiwano zamykającego "--&gt;" dla komentarza XML
Literał komentarza XML nie ma znaków wymaganych zamknięcia "-->".  
  
 **Identyfikator błędu:** BC31161  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znaki zamknięcia wymagane "-->" na końcu literał komentarza XML.
