# <a name="option-optionname-can-be-followed-only-by--or--"></a>\<optionname>po opcji może występować tylko znak "+" lub "-"

Tekst, który następuje po opcji nie jest znakiem plus (+) ani znakiem minus (-).  
  
 **Identyfikator błędu:** BC2009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zastąp tekst, który następuje po opcji znakiem plus (+), aby włączyć opcję.  
  
     oraz  
  
- Zastąp tekst, który następuje po opcji znakiem minus (-), aby wyłączyć opcję.
