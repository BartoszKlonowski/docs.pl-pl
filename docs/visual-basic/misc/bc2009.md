# <a name="option-ltoptionnamegt-can-be-followed-only-by-3939-or-39-39"></a>Opcja &lt;optionname&gt; może występować tylko znak &#39; + &#39; lub &#39;-&#39;
Tekst znajdujący się opcja nie jest znak plus (+) lub znak minus (-).  
  
 **Identyfikator błędu:** BC2009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp tekst następujący opcję ze znakiem plus (+), aby włączyć opcję.  
  
     — lub —  
  
-   Zastąp tekst następujący można wyłączyć opcję znakiem minus (-).
