# <a name="option-ltoptionnamegt-can-be-followed-only-by--or--"></a>Opcja &lt;optionname&gt; może występować tylko znak "+" lub "-"
Tekst, który następuje po opcji nie jest znakiem plus (+) lub znak minus (-).  
  
 **Identyfikator błędu:** BC2009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp tekst, który następuje po opcji ze znakiem plus (+), aby włączyć opcję.  
  
     — lub —  
  
-   Zastąp tekst, który następuje możliwość wyłączyć opcję znakiem minus (-).
