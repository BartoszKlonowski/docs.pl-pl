# <a name="type-argument-inference-failed-for-type-parameter-typeparametername1-of-genericproceduresignature"></a><span data-ttu-id="8dc07-102">Wnioskowanie argumentu dla parametru typu nie powiodło się typu '\<typeparametername1 >' z '\<genericproceduresignature >'</span><span class="sxs-lookup"><span data-stu-id="8dc07-102">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'</span></span>
<span data-ttu-id="8dc07-103">Wnioskowanie argumentu dla parametru typu nie powiodło się typu '\<typeparametername1 >' z '\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="8dc07-103">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="8dc07-104">Argument typu wywnioskowany z argumentu przekazanego do parametru '\<nazwaparametru1 >' powoduje konflikt z argumentem typu wywnioskowane z argumentu przekazanego do parametru '\<parametername2 >'.</span><span class="sxs-lookup"><span data-stu-id="8dc07-104">Type argument inferred from the argument passed to parameter '\<parametername1>' conflicts with the type argument inferred from the argument passed to parameter '\<parametername2>'.</span></span>  
  
 <span data-ttu-id="8dc07-105">Ogólna procedura jest wywoływana bez żadnych argumentów typu, a próba wnioskowanie tworzył konflikt typu danych między parametrami typu.</span><span class="sxs-lookup"><span data-stu-id="8dc07-105">A generic procedure is called without any type arguments, and the attempted type inference has produced a data type conflict among the type parameters.</span></span>  
  
 <span data-ttu-id="8dc07-106">Zwykle po wywołaniu procedury ogólne Podaj argument typu dla każdego parametru typu, który definiuje ogólny procedury.</span><span class="sxs-lookup"><span data-stu-id="8dc07-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="8dc07-107">Jeśli nie podasz żadnych argumentów typu, kompilator podejmuje próbę wnioskowanie typów, które mają być przekazane do parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="8dc07-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="8dc07-108">Jeśli kontekst wywołania udostępnia powodujące konflikt typu danych informacji dla parametru typu, wnioskowanie o typie zakończy się niepowodzeniem.</span><span class="sxs-lookup"><span data-stu-id="8dc07-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="8dc07-109">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="8dc07-109">The following code can generate this error.</span></span>  
  
```  
Public Sub takeTwoValues(Of t)(ByVal x As t, ByVal y As t)  
End Sub  
Call takeTwoValues(4, 2.5)  
```  
  
 <span data-ttu-id="8dc07-110">Ponieważ pierwszy argument powoduje, że kompilator wywnioskuje `Integer` dla parametru typu `t`, podczas gdy drugi argument funkcji powoduje, że wywnioskowania `Double` dla tego samego parametru typu występuje konflikt, dotyczące typu danych do przekazania do `t`.</span><span class="sxs-lookup"><span data-stu-id="8dc07-110">Because the first argument causes the compiler to infer `Integer` for type parameter `t`, while the second argument causes it to infer `Double` for the same type parameter, there is a conflict regarding which data type to pass to `t`.</span></span>  
  
 <span data-ttu-id="8dc07-111">**Identyfikator błędu:** BC32116</span><span class="sxs-lookup"><span data-stu-id="8dc07-111">**Error ID:** BC32116</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8dc07-112">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="8dc07-112">To correct this error</span></span>  
  
-   <span data-ttu-id="8dc07-113">Podać argumentów typu dla typu ogólnego, kompilator nie musi wywnioskować ich.</span><span class="sxs-lookup"><span data-stu-id="8dc07-113">Supply type arguments to the generic type, so that the compiler does not have to infer them.</span></span>  
  
    ```  
    Call takeTwoValues(Of Double)(4, 2.5)  
    ```  
  
     <span data-ttu-id="8dc07-114">Należy zauważyć, że w tym przypadku gdy dwa argumenty normalne są różnych typów danych, kod wywołujący musi pomyślnie przejść argumentem typu, który może obsłużyć obu tych typów danych.</span><span class="sxs-lookup"><span data-stu-id="8dc07-114">Notice that in this case, where the two normal arguments are of different data types, the calling code must pass a type argument that can accommodate both of those data types.</span></span> <span data-ttu-id="8dc07-115">W tym przypadku `Integer` rozszerza się na `Double`.</span><span class="sxs-lookup"><span data-stu-id="8dc07-115">In this case, `Integer` widens to `Double`.</span></span>  
  
     <span data-ttu-id="8dc07-116">—lub—</span><span class="sxs-lookup"><span data-stu-id="8dc07-116">-or-</span></span>  
  
-   <span data-ttu-id="8dc07-117">Ponownie zdefiniować ogólnego procedurę, aby określić inny typ parametry normalnej parametrów, tak, aby nie występuje konflikt wnioskowanie typów.</span><span class="sxs-lookup"><span data-stu-id="8dc07-117">Redefine the generic procedure to specify different type parameters for the normal parameters, so that there is no conflict inferring the types.</span></span>  
  
    ```  
    Public Sub takeTwoValues(Of t1, t2)(ByVal x As t1, ByVal y As t2)  
    ```
