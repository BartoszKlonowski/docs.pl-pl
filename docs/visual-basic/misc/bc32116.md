<span data-ttu-id="812fc-103">Wnioskowanie argumentu dla parametru typu nie powiodło się "\<typeparametername1 >" z "\<genericproceduresignature >".</span><span class="sxs-lookup"><span data-stu-id="812fc-103">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="812fc-104">Argument wywnioskować z argumentu przekazanego do parametru typu "\<parametername1 >" powoduje konflikt z argumentem typu wywnioskowane z argumentu przekazanego do parametru "\<parametername2 >".</span><span class="sxs-lookup"><span data-stu-id="812fc-104">Type argument inferred from the argument passed to parameter '\<parametername1>' conflicts with the type argument inferred from the argument passed to parameter '\<parametername2>'.</span></span>  
  
 <span data-ttu-id="812fc-105">Ogólny procedura jest wywoływana bez żadnych argumentów typu, a próba wnioskowanie opracowała konflikt typu danych między parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="812fc-105">A generic procedure is called without any type arguments, and the attempted type inference has produced a data type conflict among the type parameters.</span></span>  
  
 <span data-ttu-id="812fc-106">Zwykle po wywołaniu procedury ogólne Podaj typ argumentu dla każdego parametru typu, który definiuje procedurę ogólnego.</span><span class="sxs-lookup"><span data-stu-id="812fc-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="812fc-107">Jeśli wszystkie argumenty typu nie zostanie podana, kompilator podejmuje próbę wnioskowanie typów, które mają być przekazane do parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="812fc-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="812fc-108">Jeśli kontekst wywołania udostępnia powodujące konflikt typu danych informacji dla parametru typu wnioskowanie o typie zakończy się niepowodzeniem.</span><span class="sxs-lookup"><span data-stu-id="812fc-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="812fc-109">Poniższy kod może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="812fc-109">The following code can generate this error.</span></span>  
  
```  
Public Sub takeTwoValues(Of t)(ByVal x As t, ByVal y As t)  
End Sub  
Call takeTwoValues(4, 2.5)  
```  
  
 <span data-ttu-id="812fc-110">Ponieważ pierwszy argument powoduje, że kompilator wnioskować `Integer` dla parametru typu `t`, a drugi argument powoduje ona rozpoznać `Double` dla tego samego parametru typu występuje konflikt dotyczące typu danych do przekazania do `t`.</span><span class="sxs-lookup"><span data-stu-id="812fc-110">Because the first argument causes the compiler to infer `Integer` for type parameter `t`, while the second argument causes it to infer `Double` for the same type parameter, there is a conflict regarding which data type to pass to `t`.</span></span>  
  
 <span data-ttu-id="812fc-111">**Identyfikator błędu:** BC32116</span><span class="sxs-lookup"><span data-stu-id="812fc-111">**Error ID:** BC32116</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="812fc-112">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="812fc-112">To correct this error</span></span>  
  
-   <span data-ttu-id="812fc-113">Tak, że kompilator nie trzeba ich wnioskować, należy podać argumentów typu dla typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="812fc-113">Supply type arguments to the generic type, so that the compiler does not have to infer them.</span></span>  
  
    ```  
    Call takeTwoValues(Of Double)(4, 2.5)  
    ```  
  
     <span data-ttu-id="812fc-114">Zwróć uwagę, że w takim przypadku w przypadku, gdy dwa argumenty normalne mają różne typy danych, kod wywołujący musi przejść pomyślnie argumentu typu, który może obsłużyć obu tych typów danych.</span><span class="sxs-lookup"><span data-stu-id="812fc-114">Notice that in this case, where the two normal arguments are of different data types, the calling code must pass a type argument that can accommodate both of those data types.</span></span> <span data-ttu-id="812fc-115">W takim przypadku `Integer` rozszerzenie do `Double`.</span><span class="sxs-lookup"><span data-stu-id="812fc-115">In this case, `Integer` widens to `Double`.</span></span>  
  
     <span data-ttu-id="812fc-116">—lub—</span><span class="sxs-lookup"><span data-stu-id="812fc-116">-or-</span></span>  
  
-   <span data-ttu-id="812fc-117">Ponowna definicja ogólnego procedurę, aby określić inny typ parametry normalnej parametrów, aby nie było konfliktu wnioskowanie typów.</span><span class="sxs-lookup"><span data-stu-id="812fc-117">Redefine the generic procedure to specify different type parameters for the normal parameters, so that there is no conflict inferring the types.</span></span>  
  
    ```  
    Public Sub takeTwoValues(Of t1, t2)(ByVal x As t1, ByVal y As t2)  
    ```
