# <a name="type-argument-inference-failed-for-type-parameter-typeparametername1-of-genericproceduresignature"></a><span data-ttu-id="80db6-102">Wnioskowanie argumentu typu nie powiodło się dla parametru typu " \<typeparametername1> " z " \<genericproceduresignature> "</span><span class="sxs-lookup"><span data-stu-id="80db6-102">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="80db6-103">Wnioskowanie argumentu typu nie powiodło się dla parametru typu " \<typeparametername1> " z " \<genericproceduresignature> ".</span><span class="sxs-lookup"><span data-stu-id="80db6-103">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="80db6-104">Argument typu wywnioskowany na podstawie argumentu przekazanego do parametru " \<parametername1> " powoduje konflikt z argumentem typu wywnioskowanym z argumentu przekazanego do parametru " \<parametername2> ".</span><span class="sxs-lookup"><span data-stu-id="80db6-104">Type argument inferred from the argument passed to parameter '\<parametername1>' conflicts with the type argument inferred from the argument passed to parameter '\<parametername2>'.</span></span>  
  
 <span data-ttu-id="80db6-105">Procedura ogólna jest wywoływana bez żadnych argumentów typu, a próba wnioskowania typu wygenerowała konflikt typu danych między parametrami typu.</span><span class="sxs-lookup"><span data-stu-id="80db6-105">A generic procedure is called without any type arguments, and the attempted type inference has produced a data type conflict among the type parameters.</span></span>  
  
 <span data-ttu-id="80db6-106">Zwykle podczas wywoływania procedury ogólnej należy podać argument typu dla każdego parametru typu, który definiuje procedura ogólna.</span><span class="sxs-lookup"><span data-stu-id="80db6-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="80db6-107">Jeśli nie podasz żadnych argumentów typu, kompilator próbuje wywnioskować typy do przekazania do parametrów typu.</span><span class="sxs-lookup"><span data-stu-id="80db6-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="80db6-108">Jeśli kontekst wywołania zawiera sprzeczne informacje o typie danych dla parametru typu, wnioskowanie o typie kończy się niepowodzeniem.</span><span class="sxs-lookup"><span data-stu-id="80db6-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="80db6-109">Poniższy kod może wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="80db6-109">The following code can generate this error.</span></span>  
  
```vb  
Public Sub takeTwoValues(Of t)(ByVal x As t, ByVal y As t)  
End Sub  
Call takeTwoValues(4, 2.5)  
```  
  
 <span data-ttu-id="80db6-110">Ponieważ pierwszy argument powoduje, że kompilator może wywnioskować `Integer` dla parametru typu `t` , podczas gdy drugi argument powoduje wywnioskowanie `Double` dla tego samego parametru typu, występuje konflikt dotyczący typu danych do przekazania `t` .</span><span class="sxs-lookup"><span data-stu-id="80db6-110">Because the first argument causes the compiler to infer `Integer` for type parameter `t`, while the second argument causes it to infer `Double` for the same type parameter, there is a conflict regarding which data type to pass to `t`.</span></span>  
  
 <span data-ttu-id="80db6-111">**Identyfikator błędu:** BC32116</span><span class="sxs-lookup"><span data-stu-id="80db6-111">**Error ID:** BC32116</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="80db6-112">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="80db6-112">To correct this error</span></span>  
  
- <span data-ttu-id="80db6-113">Podaj argumenty typu dla typu ogólnego, aby kompilator nie musiał go wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="80db6-113">Supply type arguments to the generic type, so that the compiler does not have to infer them.</span></span>  
  
    ```vb  
    Call takeTwoValues(Of Double)(4, 2.5)  
    ```  
  
     <span data-ttu-id="80db6-114">Należy zauważyć, że w tym przypadku, gdy dwa normalne argumenty mają różne typy danych, kod wywołujący musi przekazać argument typu, który może obsługiwać oba te typy danych.</span><span class="sxs-lookup"><span data-stu-id="80db6-114">Notice that in this case, where the two normal arguments are of different data types, the calling code must pass a type argument that can accommodate both of those data types.</span></span> <span data-ttu-id="80db6-115">W tym przypadku `Integer` poszerza do `Double` .</span><span class="sxs-lookup"><span data-stu-id="80db6-115">In this case, `Integer` widens to `Double`.</span></span>  
  
     <span data-ttu-id="80db6-116">-lub-</span><span class="sxs-lookup"><span data-stu-id="80db6-116">-or-</span></span>  
  
- <span data-ttu-id="80db6-117">Przedefiniuj procedurę ogólną, aby określić inne parametry typu dla normalnych parametrów, tak aby nie wystąpiły żadne konflikty w odniesieniu do typów.</span><span class="sxs-lookup"><span data-stu-id="80db6-117">Redefine the generic procedure to specify different type parameters for the normal parameters, so that there is no conflict inferring the types.</span></span>  
  
    ```vb  
    Public Sub takeTwoValues(Of t1, t2)(ByVal x As t1, ByVal y As t2)  
    ```
