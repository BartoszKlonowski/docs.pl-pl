# <a name="39end-class39-must-be-preceded-by-a-matching-39class39"></a>&#39; Klasa celu &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; Klasa &#39;
`End Class`Służy do przeprowadzenia `Class` zablokować; dlatego tylko może występować na końcu bloku. Możesz mieć nadmiarowe `End Class`, lub `End Class` instrukcji znajduje się poza granicami odpowiednie `Class` bloku.  
  
 **Identyfikator błędu:** BC30460  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zlokalizuj i usunąć nadmiarowych `End Class` instrukcje.  
  
-   Przenieś `End Class` instrukcji w odpowiedniej lokalizacji w kodzie.
