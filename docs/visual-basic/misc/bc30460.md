# <a name="end-class-must-be-preceded-by-a-matching-class"></a>"End Class" musi być poprzedzona instrukcją "Class"
`End Class` Służy do przeprowadzenia `Class` Blokuj; dlatego tylko może występować na końcu bloku. Możesz mieć nadmiarowe `End Class`, lub `End Class` instrukcji pojawia się poza granicami odpowiadającymi mu dostawcami `Class` bloku.  
  
 **Identyfikator błędu:** BC30460  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zlokalizuj i Usuń nadmiarowych `End Class` instrukcji.  
  
- Przenieś `End Class` instrukcji do odpowiedniej lokalizacji w kodzie.
