# <a name="paramarray-parameter-must-be-an-array"></a>Parametr ParamArray musi być tablicą
A `ParamArray` argument musi być zadeklarowany jako typ jednowymiarową. Deklaracja nie zawiera nawiasy po `ParamArray` Nazwa argumentu.  
  
 **Identyfikator błędu:** BC30050  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj parę nawiasów po `ParamArray` Nazwa argumentu.
