# <a name="paramarray-parameter-must-be-an-array"></a>Parametr ParamArray musi być tablicą
`ParamArray`Argument musi być zadeklarowany jako jednowymiarowy typ tablicy. Deklaracja nie zawiera nawiasów po `ParamArray` nazwie argumentu.  
  
 **Identyfikator błędu:** BC30050  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj parę nawiasów po `ParamArray` nazwie argumentu.
