# <a name="paramarray-parameter-must-be-an-array"></a>Parametrem ParamArray musi być tablicą
A `ParamArray` argument musi być zadeklarowany jako typ Jednowymiarowa tablica. Deklaracja nie obejmuje nawiasów po `ParamArray` nazwę argumentu.  
  
 **Identyfikator błędu:** BC30050  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj parę nawiasów po `ParamArray` nazwę argumentu.
