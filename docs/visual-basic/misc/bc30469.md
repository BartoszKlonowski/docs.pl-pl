# <a name="reference-to-a-non-shared-member-requires-an-object-reference"></a>Odwołanie do elementu członkowskiego nieudostępnione wymaga odwołania do obiektu
Odwołanie do elementu członkowskiego nieudostępniane w obrębie kodu i nie można podać odwołanie do obiektu. Sama nazwa klasy nie można użyć do kwalifikowania element członkowski, który nie jest udostępniony. Wystąpienie najpierw musi być zadeklarowana jako zmienna obiektu, a następnie odwołuje się nazwa zmiennej.  
  
 **Identyfikator błędu:** BC30469  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zadeklaruj wystąpienie jako zmienną obiektu.  
  
2. Odwołanie do wystąpienia według nazwy zmiennej.
