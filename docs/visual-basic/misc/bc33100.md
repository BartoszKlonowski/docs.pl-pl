# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modyfikator dopuszczający wartość null nie może być określony jednocześnie w zmiennej i jej typie
Modyfikator typu dopuszczającego wartość null (?) został uwzględniony dla zadeklarowanej zmiennej i jej zadeklarowanego typu.  
  
 **Identyfikator błędu:** BC33100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń modyfikator typu dopuszczającego wartość null (?) ze zmiennej lub jej zadeklarowanego typu.
