# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modyfikator dopuszczającego wartość null nie można określić zmienną i jej typu
Modyfikator typu dopuszczającego wartość zerową (?) została dołączona do zadeklarowanej zmiennej i deklarowanym typem.  
  
 **Identyfikator błędu:** BC33100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Modyfikator typu dopuszczającego wartość zerową (?) usuwanie zmiennej lub deklarowanym typem.
