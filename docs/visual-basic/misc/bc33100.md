# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modyfikator zerowalny nie można określić jednocześnie w zmiennej i jej typu
Modyfikator typu dopuszczającego wartości null (?) został uwzględniony w zarówno zadeklarowanej zmiennej i deklarowanym typem.  
  
 **Identyfikator błędu:** BC33100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń modyfikator typu dopuszczającego wartości null (?) z zmiennej lub deklarowanym typem.
