# <a name="cannot-rename-with-different-drive"></a>Nie można zmienić nazwy na inny dysk
`Rename` Instrukcji nie można określić litery dysku w ścieżce.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń literę dysku ze ścieżki plików.
