# <a name="cannot-rename-with-different-drive"></a>Nie można zmienić nazwy na inny dysk

`Rename`Instrukcja nie może określać litery dysku w ścieżce pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń literę dysku ze ścieżki pliku.
