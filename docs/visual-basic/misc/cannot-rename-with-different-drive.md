# <a name="cannot-rename-with-different-drive"></a>Nie można zmienić nazwy z innego dysku
`Rename` Instrukcji nie można określić litera dysku w ścieżce pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń literę dysku ze ścieżki plików.
