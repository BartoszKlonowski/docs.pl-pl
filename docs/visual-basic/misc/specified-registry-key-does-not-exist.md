# <a name="specified-registry-key-does-not-exist"></a>Określony klucz rejestru nie istnieje.
Klucz rejestru, który został określony nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź klucz rejestru, które są dostarczanie, aby upewnić się, że poprawnie określona ścieżka.  
  
-   Utwórz klucz rejestru przed ponowieniem próby zapisu do niego.
