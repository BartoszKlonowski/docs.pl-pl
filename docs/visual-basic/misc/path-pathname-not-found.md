# <a name="path-pathname-not-found"></a>\<pathname>Nie znaleziono ścieżki ""

System operacyjny nie może znaleźć ścieżki określonej w trakcie operacji dostępu do pliku lub dysku. Ścieżka do pliku musi zawierać specyfikację dysku oraz katalogi i podkatalogi, które muszą zostać przesunięte, aby zlokalizować plik. Ścieżka może być względna lub bezwzględna.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że ścieżka zawiera wymagane elementy.
