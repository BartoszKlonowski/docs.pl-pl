# <a name="path-39ltpathnamegt39-not-found"></a>Ścieżka &#39; &lt;pathname&gt; &#39; nie znaleziono
System operacyjny nie może znaleźć określonej ścieżki ścieżkę podczas operacji dostępu do plików lub dostępu do dysku. Ścieżka do pliku musi zawierać specyfikację dysku i katalogów oraz podkatalogów, które muszą przejść, aby zlokalizować plik. Ścieżka może być względna lub bezwzględna.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że ścieżka zawiera wymagane elementy.
