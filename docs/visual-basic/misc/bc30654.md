# <a name="return-statement-in-a-function-get-or-operator-must-return-a-value"></a>Instrukcja "Return" w funkcji, Get lub Operator musi zwracać wartość
`Return` instrukcji należy używać w celu zwrócenia wartości do procedury wywołującej. Nie można użyć `Return` instrukcji samodzielnie kontroli programu przepływu.  
  
 **Identyfikator błędu:** BC30654  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Określ wartość, która może zwracać funkcji lub procedury.  
  
2. Użyj `End` instrukcję, aby spowodować, że program zamknąć bieżącą procedurę.
