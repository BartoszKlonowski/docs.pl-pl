# <a name="39return39-statement-in-a-function-get-or-operator-must-return-a-value"></a>&#39; Powrót &#39; Instrukcja w funkcję, Get lub Operator musi zwracać wartość
`Return`instrukcje należy użyć w celu zwrócenia wartości do wywołującego procedurę. Nie można użyć `Return` instrukcje samodzielnie w celu sterowania programu przepływu.  
  
 **Identyfikator błędu:** BC30654  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Określ wartość, która może zwracać funkcji lub procedury.  
  
2.  Użyj `End` instrukcji, aby spowodować, że program zamknąć bieżącą procedurę.
