# <a name="argument-access-is-not-valid-valid-values-for-output-mode-are-openaccesswrite-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowy. Prawidłowe wartości w trybie wyjściowym są "OpenAccess. Write" i "OpenAccess. default"

Wartość dostępu jest nieprawidłowa w trybie wyjściowym.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień wartość na `OpenAccess.Write` lub `OpenAccess.Default` .
