# <a name="argument-access-is-not-valid-valid-values-for-output-mode-are-openaccesswrite-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowa. Prawidłowe wartości dla trybu Output to "OpenAcess.Write" i "OpenAccess.Default"
Wartość dostępu jest nieprawidłowe w trybie danych wyjściowych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień wartość na jedną `OpenAccess.Write` lub `OpenAccess.Default`.
