# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;Kończyć się przy użyciu&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;Using&#39;
`End Using` Występuje instrukcja z pasujących `Using` deklaracji przed nim.  
  
 **Identyfikator błędu:** BC36007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `End Using` instrukcję, jeśli jest nadmiarowy.  
  
-   Podaj brakujący [instrukcji Using](../../visual-basic/language-reference/statements/using-statement.md) Jeśli brakuje jednego.  
  
-   Przenieś `End Using` instrukcji w odpowiednie miejsce w kodzie.
