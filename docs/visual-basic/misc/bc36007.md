# <a name="end-using-must-be-preceded-by-a-matching-using"></a>"End Using" musi być poprzedzona pasującego "Using"
`End Using` Występuje instrukcja nie słów `Using` deklaracji przed nim.  
  
 **Identyfikator błędu:** BC36007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `End Using` instrukcji, jeśli jest nadmiarowy.  
  
-   Podaj brakujące [instrukcji Using](../../visual-basic/language-reference/statements/using-statement.md) Jeśli brakuje jednego.  
  
-   Przenieś `End Using` instrukcji w odpowiednie miejsce w kodzie.
