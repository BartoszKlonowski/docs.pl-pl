# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>Element "End Operator" musi być pierwszą instrukcją w wierszu

`End Operator`Instrukcja jest zgodna z separatorem instrukcji dwukropka ( `:` ). `End Operator` musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC33006  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
