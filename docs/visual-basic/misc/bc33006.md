# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>"End Operator" musi być pierwszą instrukcją w wierszu
`End Operator` Instrukcji następuje dwukropek (`:`) separator instrukcji. `End Operator` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC33006  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Oddziel wiele instrukcji na różne wiersze.
