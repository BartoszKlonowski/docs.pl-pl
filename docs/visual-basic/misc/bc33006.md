# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39; End — Operator &#39; musi być pierwszą instrukcją w wierszu
`End Operator` Instrukcji następuje dwukropkiem (`:`) separatora instrukcji. `End Operator`musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC33006  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
