# <a name="new-cannot-be-used-on-class-ltclassnamegt-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>"New" nie można używać w klasie&lt;classname&gt;", ponieważ zawiera on składowa"MustOverride", który nie został zastąpiony
Podjęto próbę użycia `New` na klasę, która zawiera `MustOverride` elementu członkowskiego, który nie został zastąpiony.  
  
 **Identyfikator błędu:** BC30376  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `New` instrukcji.
