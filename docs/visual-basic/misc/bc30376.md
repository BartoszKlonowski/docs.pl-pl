# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>Nie można użyć elementu "New" w klasie " \<classname> ", ponieważ zawiera on element członkowski "MustOverride", który nie został zastąpiony
Podjęto próbę użycia `New` na klasie, która zawiera `MustOverride` element członkowski, który nie został zastąpiony.  
  
 **Identyfikator błędu:** BC30376  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `New` instrukcję.
