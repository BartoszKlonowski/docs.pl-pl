# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Nowy &#39; Nie można używać w klasie &#39; &lt;classname&gt;&#39; ponieważ zawiera on &#39; MustOverride &#39; element członkowski, który nie został zastąpiony
Podjęto próbę użycia `New` dla klasy, która zawiera `MustOverride` elementu członkowskiego, który nie został zastąpiony.  
  
 **Identyfikator błędu:** BC30376  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `New` instrukcji.
