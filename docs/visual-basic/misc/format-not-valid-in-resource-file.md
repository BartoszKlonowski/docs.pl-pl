# <a name="format-not-valid-in-resource-file"></a>Nieprawidłowy format w pliku zasobów
Plik zasobów w projekcie nie jest zgodny z formatem standardowym pliku zasobów systemu Windows lub występuje niezgodność między plikiem zasobów i projektem Visual Basic.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Utwórz ponownie źródło zasobów i pliki zasobów i Zastosuj się do składni udokumentowanej w zestawie Windows Software Development Kit.  
  
2. Usuń nieprawidłowy plik zasobu z projektu.
