# <a name="format-not-valid-in-resource-file"></a>Format jest nieprawidłowy w pliku zasobów
Plik zasobów w projekcie nie jest zgodna z formatem pliku zasobów systemu Windows lub występuje niezgodność między plikami zasobów i [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] projektu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Utwórz ponownie źródło zasobów i plików zasobów i oparte na składni udokumentowane w systemie Windows Software Development Kit.  
  
2.  Usuń plik nieprawidłowy zasób z projektu.
