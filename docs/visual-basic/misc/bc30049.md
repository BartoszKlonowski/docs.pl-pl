# <a name="ltstatementnamegt-statement-requires-an-array"></a>"&lt;statementname&gt;" instrukcji wymaga tablicy
`Erase` i `ReDim` instrukcje wymagają typów danych tablicy. Podany typ nie jest tablicą.  
  
 **Identyfikator błędu:** BC30049  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana typu danych, typ tablicy.
