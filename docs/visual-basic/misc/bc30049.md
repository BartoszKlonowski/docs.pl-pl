# <a name="statementname-statement-requires-an-array"></a>"\<statementname >" instrukcji wymaga tablicy
`Erase` i `ReDim` instrukcje wymagają typów danych tablicy. Podany typ nie jest tablicą.  
  
 **Identyfikator błędu:** BC30049  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana typu danych, typ tablicy.
