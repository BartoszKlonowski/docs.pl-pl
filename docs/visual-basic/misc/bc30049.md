# <a name="39ltstatementnamegt39-statement-requires-an-array"></a>&#39; &lt;statementname&gt;&#39; instrukcji wymaga tablicy
`Erase` i `ReDim` instrukcje wymaga tablicy typów danych. Podany typ nie jest tablicą.  
  
 **Identyfikator błędu:** BC30049  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ danych na typ tablicy.
