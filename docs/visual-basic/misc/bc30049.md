# <a name="statementname-statement-requires-an-array"></a>\<statementname>instrukcja "" wymaga tablicy

`Erase`Instrukcje i `ReDim` wymagają typów danych tablicowych. Dostarczony typ nie jest tablicą.  
  
 **Identyfikator błędu:** BC30049  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień typ danych na typ tablicy.
