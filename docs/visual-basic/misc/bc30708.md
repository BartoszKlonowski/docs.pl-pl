# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje Loop nie są prawidłowe w oknie bezpośrednim

`Loop` instrukcji nie można używać w oknie **bezpośrednim** .  
  
 **Identyfikator błędu:** BC30708  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Loop` instrukcje z kodu debugowania.
