# <a name="syntax-error-in-conditional-compilation-expression"></a>Błąd składni w wyrażeniu kompilacji warunkowej
Wyrażenie kompilacji warunkowej generuje nieokreślony błąd.  
  
 **Identyfikator błędu:** BC31427  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zbadanie dokumentacji dla kompilacji warunkowej i słowa kluczowe używane w wyrażeniu. Porównanie przykładowego kodu w dokumentacji względem wiersza źródłowego, wygenerowanie tego błędu.  
  
2.  Jeśli nie można zidentyfikować przyczyny tego błędu, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
