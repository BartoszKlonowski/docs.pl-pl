# <a name="syntax-error-in-conditional-compilation-expression"></a>Błąd składni w wyrażeniu kompilacji warunkowej
W wyrażeniu kompilacji warunkowej generuje błąd składniowy nieokreślony.  
  
 **Identyfikator błędu:** BC31427  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź dokumentację dla kompilacji warunkowej i słów kluczowych, używany w wyrażeniu. Porównaj przykładowy kod w dokumentacji względem wiersza źródłowego, generowanie tego błędu.  
  
2.  Jeśli nie można zidentyfikować przyczynę błędu, zebrać informacje dotyczące okoliczności i powiadomić pomocy technicznej firmy Microsoft.
