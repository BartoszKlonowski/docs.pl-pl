# <a name="syntax-error-in-conditional-compilation-expression"></a>Błąd składniowy w wyrażeniu kompilacji warunkowej

Wyrażenie kompilacji warunkowej generuje błąd składniowy nieokreślony.  
  
 **Identyfikator błędu:** BC31427  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją dla kompilacji warunkowej i słów kluczowych używanych w wyrażeniu. Porównaj przykładowy kod w dokumentacji z wierszem źródłowym generującym ten błąd.  
  
2. Jeśli nie możesz zidentyfikować przyczyny błędu, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
