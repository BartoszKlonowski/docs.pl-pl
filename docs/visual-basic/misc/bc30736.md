# <a name="expression-cannot-be-evaluated-at-this-time"></a>W tej chwili nie można obliczyć wyrażenia
Nie można obliczyć wyrażenia określonego w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30736  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień składni wyrażeń.
