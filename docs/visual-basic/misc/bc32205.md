# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>Długość nazwy zmiennej WithEvents nie może przekraczać 1019 znaków
Nazwa lub identyfikator każdej zmiennej zadeklarowanej przy użyciu `WithEvents` jest ograniczone do 1019 znaków.  
  
 **Identyfikator błędu:** BC32205  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmniejsz długość identyfikatora.
