# <a name="assembly-attribute-attributename-is-not-valid-error"></a>Atrybut zestawu " \<attributename> " jest nieprawidłowy: \<error>

Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w zestawie.

**Identyfikator błędu:** BC30129

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zastąp go atrybutem, który jest prawidłowy dla kontekstu.

2. Przejrzyj cytowany komunikat o błędzie i podejmij odpowiednie działanie.
