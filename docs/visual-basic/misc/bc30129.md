# <a name="assembly-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Atrybutu zestawu "&lt;attributename&gt;" jest nieprawidłowa: &lt;błąd&gt;
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w zestawie.

**Identyfikator błędu:** BC30129

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zamień go atrybutem, który jest prawidłowy dla kontekstu.

2. Sprawdź komunikat o błędzie w cudzysłowach i podejmij odpowiednie działanie.
