# <a name="assembly-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Atrybut zestawu &#39; &lt;attributename&gt; &#39; jest nieprawidłowa: &lt;błąd&gt;
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybut w zestawie.

**Identyfikator błędu:** BC30129

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub Zamień atrybut, który jest prawidłowy dla kontekstu.

2. Sprawdź komunikat błędu w cudzysłowie i podejmij odpowiednie działanie.
