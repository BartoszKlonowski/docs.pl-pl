# <a name="target-folder-is-a-file"></a>Folder docelowy to plik

Określasz plik podczas próby wykonania akcji związanej z folderem.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podaną ścieżkę, aby upewnić się, że określasz folder, a nie plik.
