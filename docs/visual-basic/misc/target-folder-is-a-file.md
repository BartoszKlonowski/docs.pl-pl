# <a name="target-folder-is-a-file"></a>Folder docelowy jest plikiem
Podczas próby wykonania akcji związanych z folderu są określania pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź ścieżkę są dostarczanie, aby upewnić się, że określasz folder zamiast pliku.
