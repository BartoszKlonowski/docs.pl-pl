# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Error" nie są prawidłowe w oknie bezpośrednim
instrukcji `Error` można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30724  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcje `Error` z kodu debugowania.
