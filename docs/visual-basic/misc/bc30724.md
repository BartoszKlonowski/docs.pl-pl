# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Błąd&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Error` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30724  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Error` oświadczenia debugowania kodu.
