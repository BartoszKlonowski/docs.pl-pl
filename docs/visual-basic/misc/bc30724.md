# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>"Błąd" nie są prawidłowe w oknie bezpośrednim
`Error` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30724  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Error` instrukcji debugowania kodu.
