# <a name="typename-is-an-interface-type-and-cannot-be-used-as-an-expression"></a>\<typename>Typ "" jest typem interfejsu i nie można go używać jako wyrażenia
Nazwa typu występuje, gdy wyrażenie jest wymagane. Wyrażenie musi zawierać kombinację zmiennych, stałych, literałów, właściwości i `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30111  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nazwę typu i Skonstruuj wyrażenie przy użyciu prawidłowych elementów.
