# <a name="possible-problem-detected-while-building-assembly-lterrorgt"></a>Wykryto potencjalny problem podczas kompilowania zestawu: &lt;błąd&gt;
Narzędzia ALink wywoływane przez [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilator, zgłasza błąd podczas kompilowania zestawu. Jedną z możliwych przyczyn jest to zestaw podpisem utworzenie odwołania do niepodpisanego zestawu.  
  
 Ten komunikat jest ostrzeżenie. Kompilator kontynuuje Generowanie zestawu. Aby uzyskać więcej informacji dotyczących ukrywanie ostrzeżenia lub traktowanie ostrzeżeń jako błędy, zobacz [Konfigurowanie ostrzeżeń w Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identyfikator błędu:** BC40009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź komunikat błędu w cudzysłowie i podejmij odpowiednią akcję.  
  
2.  Skompiluj program, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
3.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilatora.  
  
4.  Jeśli błąd będzie nadal występować po ponownej instalacji, zebrać informacje dotyczące okoliczności i powiadomić pomocy technicznej firmy Microsoft.
