# <a name="argument-39ltargumentnamegt39-is-not-valid-for-the-array"></a>Argument &#39; &lt;argumentname&gt; &#39; nie jest prawidłowy dla tablicy
Tablica zawiera argument, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź pisownię argumentów w wyrażeniu. Nieprawidłowo zapisana nazwa zmiennej może niejawnie utworzyć zmienną liczbową, której wartość jest ustawiana na zero.
