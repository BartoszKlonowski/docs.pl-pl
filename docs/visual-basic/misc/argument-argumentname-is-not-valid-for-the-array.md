# <a name="argument-argumentname-is-not-valid-for-the-array"></a>Argument " \<argumentname> " jest nieprawidłowy dla tablicy
Tablica zawiera argument, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź pisownię argumentów w wyrażeniu. Błędnie wpisana nazwa zmiennej może niejawnie utworzyć zmienną numeryczną, która została zainicjowana do zera.
