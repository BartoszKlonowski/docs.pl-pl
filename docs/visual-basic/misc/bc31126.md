# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39;End RaiseEvent&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;RaiseEvent&#39; deklaracji
`End RaiseEvent` Instrukcji wystąpił bez odpowiadającego `RaiseEvent` instrukcji. `End RaiseEvent` musi być poprzedzona odpowiadającą `RaiseEvent` instrukcji.  
  
 **Identyfikator błędu:** BC31126  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy poprzednie `RaiseEvent` instrukcji jest prawidłowa i poprawna.
