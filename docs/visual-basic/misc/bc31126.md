# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>Element "End RaiseEvent" musi być poprzedzony odpowiadającą mu deklaracją "RaiseEvent"

`End RaiseEvent`Instrukcja wystąpiła bez odpowiedniej `RaiseEvent` instrukcji. `End RaiseEvent` musi być poprzedzona odpowiadającą jej `RaiseEvent` instrukcją.  
  
 **Identyfikator błędu:** BC31126  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy Poprzednia `RaiseEvent` instrukcja jest prawidłowa i poprawna.
