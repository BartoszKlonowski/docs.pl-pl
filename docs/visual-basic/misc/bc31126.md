# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>"End RaiseEvent" musi być poprzedzona zgodna deklaracja "RaiseEvent"
`End RaiseEvent` Instrukcji wystąpił bez odpowiadającego `RaiseEvent` instrukcji. `End RaiseEvent` musi być poprzedzona odpowiadającą `RaiseEvent` instrukcji.  
  
 **Identyfikator błędu:** BC31126  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy poprzednie `RaiseEvent` instrukcja jest prawidłowa i poprawna.
