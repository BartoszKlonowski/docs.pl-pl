# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>"\<name >" jest już zadeklarowany jako parametr tej metody
Określona nazwa parametru została już zadeklarowana dla tej metody.  
  
 **Identyfikator błędu:** BC30734  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę parametru, aby była unikatowa.
