# <a name="range-variable-ltvariablegt-hides-a-variable-in-an-enclosing-block-or-a-range-variable-previously-defined-in-the-query-expression"></a>Zmienna zakresu &lt;zmiennej&gt; ukrywa zmienną w otaczającym bloku lub zmiennej zakresu wcześniej zdefiniowanej w wyrażeniu zapytania.
Zmienną zakresu w zapytaniu ma taką samą nazwę jak zmienna uprzednio zdefiniowany w obrębie tej samej lub zmiennej zakresu wcześniej zdefiniowany w ramach zapytania.  
  
 **Identyfikator błędu:** BC30978  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie zmienne zakresu w zapytaniu unikatowych nazw, które nie duplikatem istniejącej nazwy zmiennych w tym samym zakresie.  
  
-   Zapytania zagnieżdżone za pomocą kontrolki zduplikowane nazwy zmiennych należy ująć w nawiasy, oddzielając zakres dla każdej zmiennej zakresu.
