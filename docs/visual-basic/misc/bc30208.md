# <a name="39compare39-expected"></a>&#39; Porównaj wersję &#39; Oczekiwano
`Option` Określa instrukcji `Text` lub `Binary`, ale nie `Compare`.  
  
 **Identyfikator błędu:** BC30208  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Compare` słowa kluczowego `Option` instrukcji, na przykład `Option Compare Binary`.
