# <a name="compare-expected"></a>Oczekiwano instrukcji "Compare"
`Option`Instrukcja określa `Text` lub `Binary` , ale nie `Compare` .  
  
 **Identyfikator błędu:** BC30208  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Compare` słowo kluczowe do `Option` instrukcji, na przykład `Option Compare Binary` .
