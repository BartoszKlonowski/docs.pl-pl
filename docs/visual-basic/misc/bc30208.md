# <a name="compare-expected"></a>"Compare" Oczekiwano
`Option` Instrukcja Określa `Text` lub `Binary`, ale nie `Compare`.  
  
 **Identyfikator błędu:** BC30208  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Compare` słowa kluczowego `Option` instrukcji, na przykład `Option Compare Binary`.
