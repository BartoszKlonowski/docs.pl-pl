 <span data-ttu-id="aa256-108">Co najmniej jedna zmienna z każdej kolekcji, która jest sprzężona, musi być przywoływana po każdej stronie `Equals` operatora.</span><span class="sxs-lookup"><span data-stu-id="aa256-108">At least one variable from each collection being joined must be referenced on each side of the `Equals` operator.</span></span>
