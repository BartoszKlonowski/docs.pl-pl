# <a name="error-embedding-win32-manifest-manifest"></a>Błąd osadzanie manifestu Win32: \<manifestu >
Plik określony dla `/win32manifest` — opcja kompilatora nie mogą być osadzone. Może to być wynikiem niewystarczające uprawnienia zabezpieczeń do pliku manifestu.  
  
 **Identyfikator błędu:** BC31191  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że tożsamość, uruchamiając polecenie kompilator języka Visual Basic ma dostęp do pliku manifestu Win32.
