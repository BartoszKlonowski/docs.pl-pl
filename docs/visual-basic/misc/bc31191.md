# <a name="error-embedding-win32-manifest-manifest"></a>Błąd podczas osadzania manifestu Win32: > manifestu \<
Nie można osadzić pliku określonego dla opcji kompilatora `/win32manifest`. Może to wynikać z niewystarczających uprawnień zabezpieczeń do pliku manifestu.  
  
 **Identyfikator błędu:** BC31191  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że tożsamość uruchamiana za pomocą polecenia kompilatora Visual Basic ma dostęp do pliku manifestu Win32.
