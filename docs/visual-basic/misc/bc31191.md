# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Błąd podczas osadzania manifestu Win32: &lt;manifestu&gt;
Plik określony dla `/win32manifest` — opcja kompilatora nie mogą być osadzone. Może to być wynikiem uprawnienia zabezpieczeń nie wystarczają do pliku manifestu.  
  
 **Identyfikator błędu:** BC31191  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że tożsamość, uruchamiając polecenie kompilatora Visual Basic ma dostęp do pliku manifestu Win32.
