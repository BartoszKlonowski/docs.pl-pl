# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to--1"></a>Argument &#39; &lt;argumentname&gt;&#39; musi być większa niż lub równy -1
Wartość argumentu jest mniejsza niż -1.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że wartość argumentu znajduje się do prawidłowego zakresu.  
  
2.  Jeśli argument jest dostarczana przez obliczeń, sprawdź wartości procesów obliczeń.
