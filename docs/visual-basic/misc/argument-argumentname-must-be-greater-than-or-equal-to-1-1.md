# <a name="argument-argumentname-must-be-greater-than-or-equal-to--1"></a>Argument " \<argumentname> " musi być większy lub równy-1
Wartość argumentu jest mniejsza niż-1.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że wartość argumentu mieści się w prawidłowym zakresie.  
  
2. Jeśli argument jest dostarczany przez obliczenie, sprawdź wartości w procesach obliczania.
