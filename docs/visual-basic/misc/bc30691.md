# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;Właściwość TypeName&gt; &#39; jest typem w &#39; &lt;containername&gt; &#39; i nie można go używać jako wyrażenia
Jedną z nazw używany jest już zadeklarowany jako nazwy typu i nie można użyć w wyrażeniu.  
  
 **Identyfikator błędu:** BC30691  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Unikatowej nazwy elementu członkowskiego.  
  
-   Użyj nawiasy kwadratowe w nazwie elementu członkowskiego.
