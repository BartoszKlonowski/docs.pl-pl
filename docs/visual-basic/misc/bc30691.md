# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>"&lt;typename&gt;"jest typu na"&lt;containername&gt;" i nie można używać jako wyrażenia
Jedną z nazw, używana jest już zadeklarowany jako nazwy typu, a nie można używać w wyrażeniach.  
  
 **Identyfikator błędu:** BC30691  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Unikatowej nazwy elementu członkowskiego.  
  
-   Użyj nawiasów kwadratowych wokół nazwę elementu członkowskiego.
