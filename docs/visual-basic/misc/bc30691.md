# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>" \<typename> " jest typem w " \<containername> " i nie można go używać jako wyrażenia

Jedna z używanych nazw została już zadeklarowana jako nazwa typu i nie można jej użyć w wyrażeniu.  
  
 **Identyfikator błędu:** BC30691  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Unikatowa nazwa elementu członkowskiego.  
  
- Użyj nawiasów kwadratowych wokół nazwy elementu członkowskiego.
