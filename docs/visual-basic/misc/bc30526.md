# <a name="property-39ltpropertynamegt39-is-39readonly39"></a>Właściwości &#39; &lt;propertyname&gt;&#39; jest &#39; Tylko do odczytu &#39;
Próbowano przypisać wartości do właściwości, która jest zadeklarowana jako `ReadOnly`.  
  
 **Identyfikator błędu:** BC30526  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `ReadOnly` specyfikator w deklaracji właściwości.
