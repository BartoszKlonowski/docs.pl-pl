# <a name="property-propertyname-is-readonly"></a>Właściwość " \<propertyname> " jest tylko do odczytu

Podjęto próbę przypisania wartości do właściwości, która jest zadeklarowana `ReadOnly` .  
  
 **Identyfikator błędu:** BC30526  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `ReadOnly` specyfikator z deklaracji właściwości.
