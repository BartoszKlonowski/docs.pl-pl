# <a name="property-propertyname-is-readonly"></a>Property '\<propertyname>' is 'ReadOnly'
Nastąpiła próba przypisania wartości do właściwości, która jest zadeklarowana `ReadOnly`.  
  
 **Identyfikator błędu:** BC30526  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `ReadOnly` specyfikator w deklaracji właściwości.
