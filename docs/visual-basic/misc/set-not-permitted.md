# <a name="set-not-permitted"></a>Zestaw nie jest dozwolone
Próbowano zmienić właściwości, których ustawienia albo nie można ustawić w czasie wykonywania w przeciwnym razie można ustawić tylko w niektórych warunkach. Na przykład może próbowano zmienić `Appearance`, `ControlBox`,`MinButton`, lub `MaxButton` ustawienia właściwości formularza w czasie wykonywania, lub może próbowano ustawić `Visible` właściwości `False` dla ostatniego pozostałego widoczne podmenu w menu nadrzędnego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź właściwość, aby ustalić, jakie warunkach można ustawić.
