# <a name="set-not-permitted"></a>Zestaw nie jest dozwolone
Próbowano zmienić właściwość, której ustawienia albo nie można ustawić w czasie wykonywania lub — w przeciwnym razie można ustawić tylko w określonych warunkach. Na przykład może próbowano zmienić `Appearance`, `ControlBox`,`MinButton`, lub `MaxButton` ustawienia właściwości dla formularza w czasie wykonywania, lub może próbowano ustawić `Visible` właściwość `False` za ostatnim pozostałe podmenu widoczne menu nadrzędnego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź właściwości, a następnie określić pod jakimi warunkami może zostać ustawiony.
