# <a name="set-not-permitted"></a>Zestaw nie jest dozwolony

Podjęto próbę zmiany właściwości, której ustawienia nie mogą być ustawione w czasie wykonywania lub w przeciwnym razie można ustawić tylko w określonych warunkach. Na przykład może zostać podjęta próba zmiany `Appearance` `ControlBox` ustawień właściwości,, `MinButton` , lub `MaxButton` dla formularza w czasie wykonywania lub być może podjęto próbę ustawienia `Visible` właściwości `False` dla ostatniego widocznego podmenu w menu nadrzędnym.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź Właściwość i Określ warunki, które mogą być ustawione.
