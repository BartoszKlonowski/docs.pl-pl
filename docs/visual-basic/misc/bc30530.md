# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr nie może mieć takiej samej nazwy jak definiująca go funkcja
Użyto tej samej nazwie w parametr jak definiująca go funkcja.  
  
 **Identyfikator błędu:** BC30530  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nazwy parametrów w funkcji i Zmień nazwę parametru powodujące konflikt.
