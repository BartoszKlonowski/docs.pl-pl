# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr nie może mieć taką samą nazwę jak definiująca go funkcja
Użyto tę samą nazwę parametru, jak i definiująca go funkcja.  
  
 **Identyfikator błędu:** BC30530  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź nazwy parametrów w obrębie funkcji, a następnie zmień nazwę parametru powodujące konflikt.
