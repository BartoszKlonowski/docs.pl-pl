# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr nie może mieć takiej samej nazwy jak definiująca go funkcja

Ta sama nazwa została użyta w parametrze i jego funkcji definiującej.  
  
 **Identyfikator błędu:** BC30530  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź nazwy parametrów w funkcji i Zmień nazwę parametru powodującego konflikt.
