# <a name="division-by-zero-visual-basic-error"></a>Dzielenie przez zero (Visual Basic)
Wartość wyrażenia dzielnikiem wynosi zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię zmiennych w wyrażeniu. Błędnie zmiennej niejawnie można utworzyć zmienną numeryczną zainicjować od zera.  
  
2.  Sprawdź poprzednich operacji zmiennej w wyrażeniu, zwłaszcza tych, które przekazany do procedury jako argumenty z innych procedur.
