<span data-ttu-id="01641-103">Operator konwersji jest zadeklarowana za pomocą typu parametru, z którego dziedziczy typ zwracany.</span><span class="sxs-lookup"><span data-stu-id="01641-103">A conversion operator is declared with a parameter type from which the return type derives.</span></span>  
  
 <span data-ttu-id="01641-104">W czasie kompilacji Visual Basic traktuje wstępnie zdefiniowanych konwersję istnieje z dowolnego typu odwołania do dowolnego typu w hierarchii dziedziczenia, czyli dowolnego typu, z której pochodzi, lub pochodzącą od niego.</span><span class="sxs-lookup"><span data-stu-id="01641-104">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="01641-105">Taka Konwersja może się nie powieść w czasie wykonywania, ale kompilator nie można przewidzieć wyniki wykonawczej pozwala takich konwersji skompilować.</span><span class="sxs-lookup"><span data-stu-id="01641-105">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
