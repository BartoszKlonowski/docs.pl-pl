 <span data-ttu-id="9b31d-104">W czasie kompilacji, Visual Basic uznaje, że wstępnie zdefiniowana konwersja ma istnieć z dowolnego typu odwołania do dowolnego typu w hierarchii dziedziczenia, czyli dowolnego typu, z którego pochodzi lub który pochodzi od tego elementu.</span><span class="sxs-lookup"><span data-stu-id="9b31d-104">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="9b31d-105">Taka Konwersja może się nie powieść w czasie wykonywania, ale kompilator nie może przewidzieć wyników w czasie wykonywania, dzięki czemu możliwe jest dokonanie takiej konwersji.</span><span class="sxs-lookup"><span data-stu-id="9b31d-105">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
