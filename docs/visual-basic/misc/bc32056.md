# <a name="type-parameter-not-allowed-in-implements-clause"></a>Parametr typu nie jest dozwolony w klauzuli "Implements"
Klauzula `Implements` w typie ogólnym określa parametr typu jako element członkowski do zaimplementowania.  
  
 Klauzula `Implements` musi określać interfejs i element członkowski. Może przekazać parametr typu do interfejsu, ale nie może przekazać go do elementu członkowskiego ani używać go jako nazwy elementu członkowskiego.  
  
 Poniższe instrukcje mogą generować ten błąd.  
  
```vb  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 **Identyfikator błędu:** BC32056  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ nazwę interfejsu i oryginalny element członkowski interfejsu, postępując zgodnie ze słowem kluczowym `Implements`. W razie potrzeby można przekazać parametr typu do interfejsu.  
  
    ```vb  
    Public Sub doSomething() Implements i1(Of t).doSomething  
    ```
