# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="daa10-102">Parametr typu nie jest dozwolony w klauzuli "Implements"</span><span class="sxs-lookup"><span data-stu-id="daa10-102">Type parameter not allowed in 'Implements' clause</span></span>

<span data-ttu-id="daa10-103">`Implements`Klauzula w typie ogólnym określa parametr typu jako składową, która ma być zaimplementowana.</span><span class="sxs-lookup"><span data-stu-id="daa10-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="daa10-104">`Implements`Klauzula musi określać interfejs i element członkowski.</span><span class="sxs-lookup"><span data-stu-id="daa10-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="daa10-105">Może przekazać parametr typu do interfejsu, ale nie może przekazać go do elementu członkowskiego ani używać go jako nazwy elementu członkowskiego.</span><span class="sxs-lookup"><span data-stu-id="daa10-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="daa10-106">Poniższe instrukcje mogą generować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="daa10-106">The following statements can generate this error.</span></span>  
  
```vb  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="daa10-107">**Identyfikator błędu:** BC32056</span><span class="sxs-lookup"><span data-stu-id="daa10-107">**Error ID:** BC32056</span></span>
