# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>Po "Option Explicit" może występować tylko znak "on" lub "off"

`Option Explicit` instrukcje muszą określać `On` lub `Off` . Wszystkie inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30640  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ albo `Option Explicit On` `Option Explicit Off` .
