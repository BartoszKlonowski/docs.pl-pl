# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>"Option Explicit" może następować tylko "On" lub "Off"
`Option Explicit` należy określić instrukcji `On` lub `Off`. Wszystkie inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30640  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz opcję `Option Explicit On` lub `Option Explicit Off`.
