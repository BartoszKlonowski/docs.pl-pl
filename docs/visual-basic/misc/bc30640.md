# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Explicit&#39; może występować tylko znak &#39;na&#39; lub &#39;wyłączone&#39;
`Option Explicit` instrukcje należy określić albo `On` lub `Off`. Wszystkie inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30640  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `Option Explicit On` lub `Option Explicit Off`.
