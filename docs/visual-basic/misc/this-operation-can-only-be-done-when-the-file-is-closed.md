# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>Ta operacja jest możliwe tylko, gdy plik jest zamknięty.
Po otwarciu pliku nastąpiła próba do wykonywania określonej operacji, które można wykonać tylko, gdy plik jest zamknięty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamknij plik i spróbuj ponownie wykonać operację.
