# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>Tę operację można wykonać tylko wtedy, gdy plik jest zamknięty

Gdy plik jest otwarty, podjęto próbę wykonania operacji, którą można wykonać tylko wtedy, gdy plik jest zamknięty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik i spróbuj ponownie wykonać operację.
