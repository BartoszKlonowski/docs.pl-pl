# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Przy użyciu &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Using`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30186  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Using` instrukcji z **Immediate** okna.
