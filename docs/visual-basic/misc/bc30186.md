# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Using" nie są prawidłowe w oknie bezpośrednim
`Using`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30186  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Using` instrukcję z okna **bezpośredniego** .
