<span data-ttu-id="6b5c0-103">Jeśli wymiary użyte do zainicjowania tablicy nie są zgodne z wymiarami w deklaracji, kompilator nie może wywnioskować typu danych dla tablicy.</span><span class="sxs-lookup"><span data-stu-id="6b5c0-103">If the dimensions used to initialize an array do not match the dimensions in the declaration, the compiler cannot infer a data type for the array.</span></span> <span data-ttu-id="6b5c0-104">Na przykład poniższy kod powoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="6b5c0-104">For example, the following code causes this error.</span></span>
