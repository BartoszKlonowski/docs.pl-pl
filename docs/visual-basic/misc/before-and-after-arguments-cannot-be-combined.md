# <a name="before-and-after-arguments-cannot-be-combined"></a>"Before" i "After" nie można łączyć argumentów
Lista argumentów obejmuje zarówno `Before` i `After` argumentów, które wzajemnie się wykluczają.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Albo usuń `Before` lub `After` argumentu.
