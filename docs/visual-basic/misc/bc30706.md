# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>Wyrażenie lub instrukcja nie jest prawidłowe w oknach debugowania
`Try...Catch...Finally` Nie można używać w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30706  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Try...Catch...Finally` oświadczenia debugowania kodu.
