# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>Wyrażenie lub instrukcja nie jest prawidłowa w oknach debugowania
instrukcji `Try...Catch...Finally` nie można używać w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30706  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń instrukcje `Try...Catch...Finally` z kodu debugowania.
