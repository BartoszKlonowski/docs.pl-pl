# <a name="labels-are-not-valid-in-the-immediate-window"></a>Etykiety nie są prawidłowe w oknie bezpośrednim
Etykiety wiersza włączyć rozgałęzianie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30134  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie wpisuj etykiecie wiersza w **Immediate** okna.
