# <a name="labels-are-not-valid-in-the-immediate-window"></a>Etykiety nie są prawidłowe w oknie bezpośrednim
Etykiety wierszy Włącz rozgałęzianie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30134  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie należy wprowadzać w etykiecie wiersza w **bezpośrednie** okna.
