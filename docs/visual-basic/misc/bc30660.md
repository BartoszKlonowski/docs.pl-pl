# <a name="attributes-cannot-be-applied-to-local-variables"></a>Nie można zastosować atrybutów do zmiennych lokalnych
Nie można zastosować atrybutów do zmiennych lokalnych.  
  
 **Identyfikator błędu:** BC30660  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zadeklaruj ta zmienna w następnym wyższym poziomie zakresu, takich jak klasy lub na poziomie modułu.
