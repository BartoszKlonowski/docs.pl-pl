# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>Etykieta '\<nazwa_etykiety >' jest już zdefiniowana w bieżącej metody/wielowierszowego wyrażenia lambda
Nazwa etykiety jest zduplikowana w ramach procedury. Wszystkie nazwy w obrębie procedury muszą być unikatowe.  
  
 **Identyfikator błędu:** BC30094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmień nazwę zduplikowanych etykiety.
