# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>Etykieta " \<labelname> " jest już zdefiniowana w bieżącej metodzie/wielowierszowym wyrażeniu lambda
Nazwa etykiety jest zduplikowana w ramach procedury. Wszystkie nazwy w ramach procedury muszą być unikatowe.  
  
 **Identyfikator błędu:** BC30094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień nazwę zduplikowanej etykiety.
