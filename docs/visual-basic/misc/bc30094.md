# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Etykieta &#39; &lt;wartość labelname&gt; &#39; jest już zdefiniowana w bieżącej metody/wielowierszowego wyrażenia lambda
Nazwa etykiety jest zduplikowany w ramach procedury. Wszystkie nazwy w procedurze musi być unikatowa.  
  
 **Identyfikator błędu:** BC30094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmień nazwę zduplikowanych etykiety.
