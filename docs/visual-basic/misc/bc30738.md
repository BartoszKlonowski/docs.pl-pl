# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Element Sub Main&#39; zadeklarowano więcej niż raz w &#39; &lt;przestrzeni nazw&gt;&#39;: &lt;wiadomości&gt;
`Sub Main` mogą być deklarowane tylko raz w przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30738  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że istnieje tylko jeden `Sub Main` procedury w projekcie.
