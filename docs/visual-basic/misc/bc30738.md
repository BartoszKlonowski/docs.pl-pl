# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>"Sub Main" zadeklarowano więcej niż jeden raz w "\<przestrzeni nazw >": \<wiadomości >
`Sub Main` mogą być deklarowane tylko raz w przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30738  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że istnieje tylko jeden `Sub Main` procedury w projekcie.
