# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>Element "Sub Main" jest zadeklarowany więcej niż raz w elemencie " \<namespace> ": \<message>

`Sub Main` mogą być deklarowane tylko raz wewnątrz przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30738  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że w projekcie jest tylko pojedyncza `Sub Main` procedura.
