# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Tablice używane jako argumenty atrybutów są wymagane do jawnego określenia wartości dla wszystkich elementów
Tablica użyta jako argument atrybutu ma element z nieokreśloną wartością.  
  
 **Identyfikator błędu:** BC31110  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ wszystkie wartości elementów.
