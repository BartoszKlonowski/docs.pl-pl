# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Tablice używane jako argumentów atrybutów są wymagane do jawnego określenia wartości dla wszystkich elementów
Tablica używana jako argument atrybutu zawiera element z wartością nieokreślony.  
  
 **Identyfikator błędu:** BC31110  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy określić wszystkie wartości elementu.
