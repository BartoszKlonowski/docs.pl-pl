# <a name="39get39-statements-are-no-longer-supported-visual-basic"></a>&#39;Pobierz&#39; instrukcji nie są już obsługiwane (Visual Basic)
`Get` instrukcje nie są już obsługiwane. Funkcje We/Wy plików są zwykle dostępne w `Microsoft.VisualBasic` przestrzeni nazw, ale docelowa wersja programu .NET Compact Framework nie jest ona obsługiwana.  
  
 **Identyfikator błędu:** BC30767  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wykonaj operacje na plikach z funkcji zdefiniowanych w <xref:System.IO> przestrzeni nazw.
