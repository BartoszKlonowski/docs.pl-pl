# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Instrukcje "Get" nie są już obsługiwane (Visual Basic)
`Get` instrukcje nie są już obsługiwane. Funkcje We/Wy pliku jest zwykle dostępna w `Microsoft.VisualBasic` przestrzeni nazw, ale wersji docelowej programu .NET Compact Framework nie jest ona obsługiwana.  
  
 **Identyfikator błędu:** BC30767  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wykonywania operacji na plikach za pomocą funkcji zdefiniowanych w <xref:System.IO> przestrzeni nazw.
