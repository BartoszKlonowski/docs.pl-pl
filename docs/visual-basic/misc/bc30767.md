# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Instrukcje "Get" nie są już obsługiwane (Visual Basic)
instrukcje `Get` nie są już obsługiwane. Funkcja we/wy plików jest zwykle dostępna w przestrzeni nazw `Microsoft.VisualBasic`, ale doużywana wersja .NET Compact Framework nie obsługuje tej funkcji.  
  
 **Identyfikator błędu:** BC30767  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wykonywanie operacji na plikach z funkcjami zdefiniowanymi w przestrzeni nazw <xref:System.IO>.
