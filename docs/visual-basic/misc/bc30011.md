# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Wewnętrzny błąd kompilatora: Generator kodu odebrał nieprawidłowo sformułowane dane wejściowe

Kompilator Visual Basic nie może pomyślnie zinterpretować jednego z zapisanych wewnętrznie plików.  
  
 **Identyfikator błędu:** BC30011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Skompiluj ponownie program, aby zobaczyć, czy błąd jest powtarzany.  
  
2. Jeśli błąd powtarza się, zainstaluj ponownie kompilator Visual Basic.  
  
3. Jeśli błąd będzie się powtarzać po ponownej instalacji, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
