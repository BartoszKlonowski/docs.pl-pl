# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Błąd wewnętrzny kompilatora: generator kodu odebrał źle sformułowane dane wejściowe
Kompilator Visual Basic nie można zinterpretować pomyślnie jeden z jego plików wewnętrznie napisane.  
  
 **Identyfikator błędu:** BC30011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Skompiluj program ponownie, aby zobaczyć, jeśli błąd będzie się powtarzać.  
  
2. Jeśli ten błąd wystąpi, zainstaluj ponownie kompilator Visual Basic.  
  
3. Jeśli ten błąd będzie występował po ponownej instalacji, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
