# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="7ad96-102">Ustawienie Option Strict On nie zezwala na Zawężanie w niejawnych konwersjach typów między wyrażeniem lambda i delegatem " \<delegatename> "</span><span class="sxs-lookup"><span data-stu-id="7ad96-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>

<span data-ttu-id="7ad96-103">W programie `Option Strict` on nie można mieć zawężanej konwersji między typem danych parametru w delegatze i odpowiadającym mu parametrem wyrażenia lambda przypisanego do zmiennej tego typu delegata.</span><span class="sxs-lookup"><span data-stu-id="7ad96-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="7ad96-104">Na przykład, w poniższym kodzie delegat `Del` ma jeden parametr typu `Integer` .</span><span class="sxs-lookup"><span data-stu-id="7ad96-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="7ad96-105">W związku z tym, odpowiedni parametr dowolnego wyrażenia lambda przypisanego do zmiennej typu `Del` może być `Integer` lub dowolnym typem danych, dla którego istnieje konwersja rozszerzająca z `Integer` .</span><span class="sxs-lookup"><span data-stu-id="7ad96-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="7ad96-106">**Identyfikator błędu:** BC36662</span><span class="sxs-lookup"><span data-stu-id="7ad96-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7ad96-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="7ad96-107">To correct this error</span></span>  
  
- <span data-ttu-id="7ad96-108">Zmień typ danych parametru w delegatze lub wyrażeniu lambda tak, aby istniała wymagana poszerzenie relacji.</span><span class="sxs-lookup"><span data-stu-id="7ad96-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
- <span data-ttu-id="7ad96-109">Nie określaj typów danych parametrów w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="7ad96-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="7ad96-110">Typy zostaną wywnioskowane z odpowiednich parametrów w delegacie.</span><span class="sxs-lookup"><span data-stu-id="7ad96-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"  
```
