# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="73fc5-102">Ustawienie Option Strict On nie zezwala na zawężanie w niejawnych konwersjach typów między wyrażenia lambda i delegata '\<element delegatename >'</span><span class="sxs-lookup"><span data-stu-id="73fc5-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>
<span data-ttu-id="73fc5-103">Za pomocą `Option Strict` , nie może mieć konwersji zawężającej między typem danych parametru w obiekt delegowany i odpowiedniego parametru Wyrażenie lambda, przypisany do zmiennej tego typu delegata.</span><span class="sxs-lookup"><span data-stu-id="73fc5-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="73fc5-104">Na przykład w poniższym kodzie delegować `Del` ma jeden parametr typu `Integer`.</span><span class="sxs-lookup"><span data-stu-id="73fc5-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="73fc5-105">W związku z tym, odpowiedniego parametru dowolne Wyrażenie lambda, przypisany do zmiennej typu `Del` może być `Integer` lub dowolny typ danych, dla którego jest konwersją rozszerzającą z `Integer`.</span><span class="sxs-lookup"><span data-stu-id="73fc5-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="73fc5-106">**Identyfikator błędu:** BC36662</span><span class="sxs-lookup"><span data-stu-id="73fc5-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="73fc5-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="73fc5-107">To correct this error</span></span>  
  
- <span data-ttu-id="73fc5-108">Zmień typ danych parametru w delegat lub wyrażenie lambda, tak, że istnieje wymagana relacja rozszerzającej.</span><span class="sxs-lookup"><span data-stu-id="73fc5-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
- <span data-ttu-id="73fc5-109">Nie określaj typów danych parametrów w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="73fc5-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="73fc5-110">Typy zostanie wywnioskowany z odpowiednich parametrów w delegacie.</span><span class="sxs-lookup"><span data-stu-id="73fc5-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"
```  
