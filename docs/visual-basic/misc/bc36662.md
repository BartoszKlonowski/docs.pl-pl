# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-39ltdelegatenamegt39"></a><span data-ttu-id="bfa86-102">Ustawienie Option Strict On nie zezwala na zawężanie w niejawnych konwersjach typów między wyrażeniem lambda a obiektem delegowanym &#39; &lt;element delegatename&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="bfa86-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate &#39;&lt;delegatename&gt;&#39;</span></span>
<span data-ttu-id="bfa86-103">Z `Option Strict` , nie może mieć konwersji zawężającej między typ danych parametru delegat i odpowiadającego mu parametru wyrażenia lambda przypisany do zmiennej typu delegowanego.</span><span class="sxs-lookup"><span data-stu-id="bfa86-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="bfa86-104">Na przykład w poniższym kodzie delegować `Del` ma jeden parametr typu `Integer`.</span><span class="sxs-lookup"><span data-stu-id="bfa86-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="bfa86-105">W związku z tym odpowiadającego mu parametru dowolne Wyrażenie lambda przypisaną do zmiennej typu `Del` może być `Integer` lub dowolny typ danych, dla którego jest konwersję rozszerzającą z `Integer`.</span><span class="sxs-lookup"><span data-stu-id="bfa86-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="bfa86-106">**Identyfikator błędu:** BC36662</span><span class="sxs-lookup"><span data-stu-id="bfa86-106">**Error ID:** BC36662</span></span>
