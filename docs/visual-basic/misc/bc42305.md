# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Tag komentarza XML " \<tag> " występuje z identycznymi atrybutami więcej niż raz w tym samym bloku komentarza XML
Tag komentarza XML \<tag> występuje z identycznymi atrybutami więcej niż raz w tym samym bloku komentarza XML. Komentarz XML zostanie zignorowany.  
  
 Komentarz XML zawiera więcej niż jeden tag z identycznymi atrybutami.  
  
 **Identyfikator błędu:** BC42305  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowany tag.
