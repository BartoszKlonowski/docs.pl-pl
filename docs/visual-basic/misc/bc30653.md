# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>Wymagane odwołanie do modułu " \<modulename> " zawierającego typ " \<membername> "

Wymagane odwołanie do modułu " \<modulename> " zawierającego typ " \<membername> ". Dodaj je do projektu.  
  
 Projekt zawiera typ, który zależy od innego typu z zestawu, który nie jest uwzględniony w odwołaniach projektu.  
  
 **Identyfikator błędu:** BC30653  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj odwołanie do projektu dla typów zależnych w projekcie.
