# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>Wymagane odwołanie do modułu "\<nazwa_modułu >" z typem "\<membername >"
Wymagane odwołanie do modułu "\<nazwa_modułu >" z typem "\<membername >". Dodaj je do projektu.  
  
 Projekt zawiera typ, który jest zależny od innego typu niż zestaw, który nie jest uwzględniony w odwołania do projektu.  
  
 **Identyfikator błędu:** BC30653  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołanie do projektu dla typów zależnych w projekcie.
