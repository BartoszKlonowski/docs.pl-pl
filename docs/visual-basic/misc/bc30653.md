# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Wymagane odwołanie do modułu &#39; &lt;modulename&gt;&#39; zawierający typ &#39;&lt; membername&gt;&#39;
Wymagane odwołanie do modułu "\<nazwa_modułu >" z typem "\<membername >". Dodaj je do projektu.  
  
 Projekt zawiera typ, który jest zależny od innego typu niż zestawu, który nie znajduje się w odwołaniach projektu.  
  
 **Identyfikator błędu:** BC30653  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołanie do projektu dla typów zależnych w projekcie.
