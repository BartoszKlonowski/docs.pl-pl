# <a name="39nothing39-cannot-be-evaluated"></a>&#39;Nic&#39; nie można obliczyć
`Nothing` Użyto słowa kluczowego w oknie debugowania.  
  
 **Identyfikator błędu:** BC30701  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie używaj `Nothing` podczas debugowania.
