# <a name="nothing-cannot-be-evaluated"></a>Nie można obliczyć wartości "Nothing"
`Nothing` Użyto słowa kluczowego w oknie debugowania.  
  
 **Identyfikator błędu:** BC30701  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie używaj `Nothing` podczas debugowania.
