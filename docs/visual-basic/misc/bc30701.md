# <a name="nothing-cannot-be-evaluated"></a>Nie można obliczyć elementu "Nothing"
`Nothing`Słowo kluczowe zostało użyte w oknie debugowania.  
  
 **Identyfikator błędu:** BC30701  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie należy używać `Nothing` podczas debugowania.
