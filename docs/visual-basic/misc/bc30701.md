# <a name="39nothing39-cannot-be-evaluated"></a>&#39; Nie &#39; Nie można obliczyć
`Nothing` Użyto słowa kluczowego w oknie debugowania.  
  
 **Identyfikator błędu:** BC30701  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie używaj `Nothing` podczas debugowania.
