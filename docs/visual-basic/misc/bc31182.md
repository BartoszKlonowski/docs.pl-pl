# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Wartość atrybutu jest nieprawidłowa; Oczekiwano elementu "wartość1" lub "wartość2"

Określono nieprawidłową wartość atrybutu w literale dokumentu XML. Prawidłowe wartości atrybutów dla literału dokumentu XML są oparte na specyfikacji XML 1,0.  
  
 **Identyfikator błędu:** BC31182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość atrybutu na wartość, która jest prawidłowa dla specyfikacji XML 1,0.
