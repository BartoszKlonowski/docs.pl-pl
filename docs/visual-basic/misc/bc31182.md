# <a name="attribute-value-is-not-valid-expecting-39value139-or-39value239"></a>Wartość atrybutu jest nieprawidłowy; Oczekiwano &#39; wartość1 &#39; i &#39; wartość2 &#39;
Określono nieprawidłową wartość atrybutu literał dokumentu XML. Wartości atrybutów Nieprawidłowy literał dokumentu XML są oparte na Specyfikacja XML 1.0.  
  
 **Identyfikator błędu:** BC31182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość atrybutu na wartość, która jest nieprawidłowy według specyfikacji XML 1.0.
