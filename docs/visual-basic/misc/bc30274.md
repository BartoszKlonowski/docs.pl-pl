# <a name="parameter-ltparameternamegt-of-ltmethodnamegt-already-has-a-matching-argument"></a>Parametr "&lt;parametername&gt;"z"&lt;methodname&gt;" ma już pasujący argument
Argument jest przekazywany przez nazwę więcej niż jeden raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC30274  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwy argumentu jest błędnie wpisana.  
  
-   Usuń zduplikowane argumentu z listy argumentów.
