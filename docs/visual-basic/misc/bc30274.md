# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>Parametr " \<parametername> " elementu " \<methodname> " ma już pasujący argument

Argument jest przesyłany przez nazwę więcej niż raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC30274  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwy argumentów są błędne.  
  
- Usuń zduplikowany argument z listy argumentów.
