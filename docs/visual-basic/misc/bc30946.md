# <a name="stop-request-is-pending"></a>Istnieje oczekujące żądanie zatrzymania
W debugerze programu Visual Studio wyrażenie określa wywołania procedury, ale ma żądania zatrzymania wątku.  
  
 Debuger nie próbuje wywołać procedurę w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30946  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli to możliwe należy określić źródło żądania zatrzymania wątku, tak, aby można go zapobiec występowaniu.  
  
- Zakończ i uruchom ponownie debugowanie.
