# <a name="stop-request-is-pending"></a>Żądanie zatrzymania oczekuje
W debugerze programu Visual Studio wyrażenie określa wywołanie procedury, ale jest to żądanie zatrzymania wątku.  
  
 Debuger nie próbuje wywoływanie procedury w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30946  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli to możliwe należy określić źródło żądania zatrzymania przez wątek, tak, aby zapobiec jego ponowne wykonanie.  
  
-   Zakończenie, a następnie ponownie uruchom debugowanie.
