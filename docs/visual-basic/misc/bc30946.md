# <a name="stop-request-is-pending"></a>Żądanie zatrzymania oczekuje

W debugerze programu Visual Studio wyrażenie określa wywołanie procedury, ale istnieje żądanie zatrzymania wątku.  
  
 Debuger nie próbuje wywołać procedury w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30946  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli to możliwe, Określ źródło żądania zatrzymania wątku, aby uniemożliwić jego ponowne uruchomienie.  
  
- Przerwij i ponownie uruchom debugowanie.
