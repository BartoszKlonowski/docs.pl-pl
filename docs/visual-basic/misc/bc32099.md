# <a name="comma-or--expected-type-parameter-list"></a>Przecinek lub ')' oczekiwany (lista parametrów typu)
Lista parametrów typu zawiera nieoczekiwany element. Używaj przecinków (`,`) do oddzielenia parametrów typu w takiej listy i użyj nawiasu zamykającego (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC32099  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj przecinek i kontynuować lista parametrów typu lub Dodaj zamykający nawias znaków do zakończenia na liście.
