# <a name="comma-or--expected-type-parameter-list"></a>Oczekiwano przecinka lub znaku ")" (lista parametrów typu)
Lista parametrów typu zawiera nieoczekiwany element. Użyj przecinków ( `,` ) do oddzielania parametrów typu na liście, a następnie użyj zamykającego nawiasu ( `)` ), aby zakończyć listę.  
  
 **Identyfikator błędu:** BC32099  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj przecinek i Kontynuuj listę parametrów typu lub Dodaj znak nawias zamykający, aby zakończyć listę.
