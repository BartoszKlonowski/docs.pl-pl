# <a name="comma-or-3939-expected-type-parameter-list"></a>Przecinek lub &#39;) &#39; Oczekiwano (lista parametrów typu)
Lista parametrów typu zawiera nieoczekiwany element. Użyj przecinków (`,`) do rozdzielenia parametrów typu w takiej listy i użyj nawiasu zamykającego (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC32099  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodawanie przecinkami i kontynuować listy parametrów typu lub dodawanie zamknięcia znak nawiasu zakończenie na liście.
