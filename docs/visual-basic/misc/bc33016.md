# <a name="operator-39ltoperatorgt39-must-have-either-one-or-two-parameters"></a>Operator &#39; &lt;operator&gt; &#39; musi mieć jeden lub dwa parametry
Operator jednoargumentowy lub dane binarne zdefiniowano bez parametrów lub z więcej niż dwa parametry.  
  
 Operator jedno/dwuargumentowy musi mieć jeden lub dwa parametry.  
  
 **Identyfikator błędu:** BC33016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostosuj definicji, aby określić jeden lub dwa parametry.  
  
-   Jeśli potrzebujesz bez parametrów lub większa niż dwa, należy użyć [instrukcji Function](../../visual-basic/language-reference/statements/function-statement.md) do definiowania `Function` procedury zamiast operatora.
