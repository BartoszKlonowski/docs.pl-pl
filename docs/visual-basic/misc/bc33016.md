# <a name="operator-ltoperatorgt-must-have-either-one-or-two-parameters"></a>Operator "&lt;operator&gt;" musi mieć jeden lub dwa parametry
Operator, który może być jednoargumentowy lub binarny jest zdefiniowany, bez parametrów lub z więcej niż dwoma parametrami.  
  
 Jedno/dwuargumentowy operator musi mieć jeden lub dwa parametry.  
  
 **Identyfikator błędu:** BC33016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostosowanie definicji, aby określić jeden lub dwa parametry.  
  
-   Jeśli potrzebujesz bez parametrów lub większa niż dwa, należy użyć [Function — instrukcja](../../visual-basic/language-reference/statements/function-statement.md) do definiowania `Function` procedury zamiast operatora.
