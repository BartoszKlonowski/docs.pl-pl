# <a name="type-characters-cannot-be-used-in-anonymous-type-declarations"></a><span data-ttu-id="05cbf-102">Nie można używać znaków typu w deklaracjach typu anonimowego</span><span class="sxs-lookup"><span data-stu-id="05cbf-102">Type characters cannot be used in anonymous type declarations</span></span>
<span data-ttu-id="05cbf-103">Nie można użyć znaku typu w nazwie właściwości w przypadku deklarowania wystąpienia typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="05cbf-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="05cbf-104">Typ danych właściwości jest wywnioskowany na podstawie przypisanej do niej wartości.</span><span class="sxs-lookup"><span data-stu-id="05cbf-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="05cbf-105">Na przykład następujące deklaracje są nieprawidłowe.</span><span class="sxs-lookup"><span data-stu-id="05cbf-105">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.  
'Dim anon1 = New With {.ID$ = "abc"}  
'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="05cbf-106">**Identyfikator błędu:** BC36560</span><span class="sxs-lookup"><span data-stu-id="05cbf-106">**Error ID:** BC36560</span></span>
