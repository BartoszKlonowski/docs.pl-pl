<span data-ttu-id="a4364-103">Nie można użyć znaku typu na nazwę właściwości, kiedy Deklarujesz wystąpienie typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="a4364-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="a4364-104">Typ danych właściwości jest wnioskowany z wartości przypisane do niego.</span><span class="sxs-lookup"><span data-stu-id="a4364-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="a4364-105">Na przykład następujące deklaracje są nieprawidłowe.</span><span class="sxs-lookup"><span data-stu-id="a4364-105">For example, the following declarations are not valid.</span></span>

