# <a name="type-characters-cannot-be-used-in-anonymous-type-declarations"></a><span data-ttu-id="74854-102">Nie można używać znaków typu w deklaracjach typu anonimowego</span><span class="sxs-lookup"><span data-stu-id="74854-102">Type characters cannot be used in anonymous type declarations</span></span>

<span data-ttu-id="74854-103">Nie można użyć znaku typu w nazwie właściwości w przypadku deklarowania wystąpienia typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="74854-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="74854-104">Typ danych właściwości jest wywnioskowany na podstawie przypisanej do niej wartości.</span><span class="sxs-lookup"><span data-stu-id="74854-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="74854-105">Na przykład następujące deklaracje są nieprawidłowe.</span><span class="sxs-lookup"><span data-stu-id="74854-105">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.  
'Dim anon1 = New With {.ID$ = "abc"}  
'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="74854-106">**Identyfikator błędu:** BC36560</span><span class="sxs-lookup"><span data-stu-id="74854-106">**Error ID:** BC36560</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="74854-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="74854-107">To correct this error</span></span>  
  
- <span data-ttu-id="74854-108">Usuń znak typu z listy inicjatorów.</span><span class="sxs-lookup"><span data-stu-id="74854-108">Remove the type character from the initializer list.</span></span> <span data-ttu-id="74854-109">Można jawnie skonwertować przypisaną wartość, jeśli jest to konieczne do ustanowienia typu danych dla właściwości.</span><span class="sxs-lookup"><span data-stu-id="74854-109">You can explicitly convert the assigned value if this is necessary to establish the data type you want for the property.</span></span>  
  
```vb

Dim anon2 = New With {.ID = CStr(42)}  
```
