# <a name="in-expected"></a>Oczekiwano "w"
A `From` `Aggregate` lub klauzula została `In` określona bez operatora. Operator służy `In` do identyfikowania kolekcji do kwerendy.  
  
 **Identyfikator błędu:** Bc36607  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  

1. Dodaj `In` operator i pola `From` klucza do lub `Aggregate` klauzuli. Poniżej przedstawiono przykład:  

```vb  
Dim names = From pers In people
            Select pers.FirstName, pers.LastName  
```
