# <a name="named-arguments-cannot-match-paramarray-parameters"></a>Nazwane argumenty nie pasują do parametrów ParamArray

Tablice parametrów muszą być przesyłane przez wartość.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Przekazanie argumentów `ByVal` zamiast nazw.
