# <a name="unable-to-open-resource-file-filename-error"></a>Nie można otworzyć pliku zasobów "\<filename >": \<error >
Wystąpił błąd podczas otwierania określonego pliku zasobów.  
  
 **Identyfikator błędu:** BC31509  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy określona nazwa i lokalizacja są poprawne.  
  
2. Sprawdź, czy plik nie jest uszkodzony lub uszkodzony.
