# <a name="unable-to-open-resource-file-ltfilenamegt-lterrorgt"></a>Nie można otworzyć pliku zasobów "&lt;filename&gt;": &lt;błąd&gt;
Wystąpił błąd podczas otwierania plików określonego zasobu.  
  
 **Identyfikator błędu:** BC31509  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy określona nazwa i lokalizacja są poprawne.  
  
2.  Sprawdź, czy plik nie jest uszkodzony.
