# <a name="unable-to-open-resource-file-39ltfilenamegt39-lterrorgt"></a>Nie można otworzyć pliku zasobów &#39; &lt;filename&gt;&#39;: &lt;błąd&gt;
Wystąpił błąd podczas otwierania pliku określonego zasobu.  
  
 **Identyfikator błędu:** BC31509  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź poprawność określonej nazwy i lokalizacji.  
  
2.  Sprawdź, czy plik nie jest uszkodzony.
