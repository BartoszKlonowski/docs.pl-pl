# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>Docelowa wersja platformy .NET Compact Framework nie obsługuje instrukcji "End"
Nie obsługuje wersji platformy .NET Compact Framework pracujesz z `End` instrukcji.  
  
 **Identyfikator błędu:** BC30769  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń instrukcję.
