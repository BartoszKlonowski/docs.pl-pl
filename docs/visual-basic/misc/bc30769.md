# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>Docelowa wersja programu .NET Compact Framework nie obsługuje &#39;zakończenia&#39; — instrukcja
Wersja programu .NET Compact Framework pracujesz nie obsługuje `End` instrukcji.  
  
 **Identyfikator błędu:** BC30769  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń instrukcji.
