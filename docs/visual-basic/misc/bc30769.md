# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>Docelowa wersja programu .NET Compact Framework nie obsługuje &#39; Końcowy &#39; — Instrukcja
Wersja programu .NET Compact Framework pracujesz nie obsługuje `End` instrukcji.  
  
 **Identyfikator błędu:** BC30769  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń instrukcji.
