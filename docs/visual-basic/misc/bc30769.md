# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>Doprzeznaczona wersja .NET Compact Framework nie obsługuje instrukcji "End"

Wersja .NET Compact Framework, z którą pracujesz, nie obsługuje `End` instrukcji.  
  
 **Identyfikator błędu:** BC30769  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję.
