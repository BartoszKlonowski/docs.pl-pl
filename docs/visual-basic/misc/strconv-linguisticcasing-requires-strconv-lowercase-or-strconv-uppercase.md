# <a name="strconvlinguisticcasing-requires-strconvlowercase-or-strconvuppercase"></a>Element "StrConv. LinguisticCasing" wymaga "StrConv. małe" lub "StrConv. wielkie litery"

Podjęto próbę użycia `StrConv.LinguisticCasing` , która jest prawidłowa tylko w połączeniu z `StrConv.LowerCase` lub `StrConv.UpperCase` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj `StrConv.LowerCase` lub `StrConv.UpperCase` w połączeniu z `StrConv.LinguisticCasing` .
