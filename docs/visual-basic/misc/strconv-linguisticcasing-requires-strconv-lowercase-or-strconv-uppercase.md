# <a name="39strconvlinguisticcasing39-requires-39strconvlowercase39-or-39strconvuppercase39"></a>&#39; StrConv.LinguisticCasing &#39; wymaga &#39; StrConv.LowerCase &#39; i &#39; StrConv.UpperCase &#39;
Podjęto próbę użycia `StrConv.LinguisticCasing`, który jest prawidłowy tylko w połączeniu z `StrConv.LowerCase` lub `StrConv.UpperCase`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Użyj `StrConv.LowerCase` lub `StrConv.UpperCase` w połączeniu z `StrConv.LinguisticCasing`.
