# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-39object39-class"></a>Nazwa zmiennej zakresu nie może być zgodna z nazwą elementu członkowskiego &#39; obiekt &#39; klasy
Zmienną zakresu w zapytaniu składnika LINQ jest zgodna z nazwą elementu członkowskiego <xref:System.Object> klasy. Spowoduje to utworzenie konflikt z obiekty utworzone przez program Visual Basic dla zapytania LINQ.  
  
 **Identyfikator błędu:** BC36606  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Wybierz nową nazwę zmiennej zakresu, który nie jest zgodna z nazwą każdego członka <xref:System.Object> klasy.
