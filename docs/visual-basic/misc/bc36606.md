# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-object-class"></a>Nazwę zmiennej zakresu nie może odpowiadać nazwie elementu członkowskiego klasy "Object"
Zmienną zakresu w zapytaniu programu LINQ jest zgodna z nazwą członkiem <xref:System.Object> klasy. Spowoduje to utworzenie konflikt z obiekty utworzone przez program Visual Basic dla zapytania LINQ.  
  
 **Identyfikator błędu:** BC36606  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Wybierz nową nazwę zmiennej zakresu, który nie jest zgodna z nazwą każdego członka <xref:System.Object> klasy.
