# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-object-class"></a>Nazwa zmiennej zakresu nie może być zgodna z nazwą składowej klasy "Object"

Zmienna zakresu w zapytaniu LINQ jest zgodna z nazwą składowej <xref:System.Object> klasy. Spowoduje to utworzenie konfliktu z obiektami utworzonymi przez Visual Basic dla zapytania LINQ.  
  
 **Identyfikator błędu:** BC36606  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Wybierz nową nazwę zmiennej zakresu, która nie pasuje do nazwy żadnego elementu członkowskiego <xref:System.Object> klasy.
