# <a name="unable-to-load-referenced-library-ltfilenamegt-lterrorgt"></a>Nie można załadować przywoływana Biblioteka "&lt;filename&gt;": &lt;błąd&gt;
Nie można załadować biblioteki z przywoływanego przez ten projekt.  
  
 **Identyfikator błędu:** BC31011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy plik znajduje się w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
