# <a name="unable-to-load-referenced-library-filename-error"></a>Nie można załadować przywoływanej biblioteki " \<filename> ": \<error>

Nie można załadować biblioteki, do której odwołuje się ten projekt.  
  
 **Identyfikator błędu:** BC31011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy plik znajduje się w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
