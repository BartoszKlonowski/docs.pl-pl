# <a name="the-file-is-currently-closed"></a>Plik jest już zamknięty.
Została podjęta próba wykonania operacji, które mogą być wykonywane tylko przy otwartym pliku, ale plik jest już zamknięty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Otwórz plik i spróbuj ponownie wykonać operację.
