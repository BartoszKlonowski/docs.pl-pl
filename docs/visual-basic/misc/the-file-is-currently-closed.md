# <a name="the-file-is-currently-closed"></a>Plik jest teraz niemożliwe.
Został podejmowana próba wykonania operacji, które mogą być wykonywane tylko na otwartego pliku, ale plik jest teraz niemożliwe.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Otwórz plik i spróbuj ponownie wykonać operację.
