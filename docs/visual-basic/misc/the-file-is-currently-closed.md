# <a name="the-file-is-currently-closed"></a>Plik jest obecnie zamknięty
Podjęto próbę wykonania operacji, którą można wykonać tylko w otwartym pliku, ale plik jest obecnie zamknięty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Otwórz plik i spróbuj ponownie wykonać operację.
