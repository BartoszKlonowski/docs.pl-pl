# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>Importowanie typu '\<typename >' z zestawu lub modułu '\<name >' nie powiodło się
Upewnij się, że wszystkie odwołania biblioteki są ze sobą zgodne.  
  
 **Identyfikator błędu:** BC31091  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy pliki zawierające odwołanie typu nie są uszkodzone.
