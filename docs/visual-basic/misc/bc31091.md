# <a name="import-of-type-lttypenamegt-from-assembly-or-module-ltnamegt-failed"></a>Importowanie typu "&lt;typename&gt;"z zestawu lub modułu"&lt;nazwa&gt;" nie powiodło się
Upewnij się, że wszystkie odwołania biblioteki są ze sobą zgodne.  
  
 **Identyfikator błędu:** BC31091  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy pliki zawierające odwołanie typu nie są uszkodzone.
