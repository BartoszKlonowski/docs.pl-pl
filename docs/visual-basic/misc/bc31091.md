# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>Importowanie typu " \<typename> " z zestawu lub modułu " \<name> " nie powiodło się

Upewnij się, że wszystkie biblioteki, do których istnieją odwołania, są zgodne ze sobą.  
  
 **Identyfikator błędu:** BC31091  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy pliki zawierające przywoływany typ nie są uszkodzone.
