# <a name="import-of-type-39lttypenamegt39-from-assembly-or-module-39ltnamegt39-failed"></a>Importowanie typu &#39; &lt;typename&gt; &#39; z zestawu lub modułu &#39; &lt;nazwa&gt; &#39; nie powiodło się
Upewnij się, że wszystkie biblioteki, w którym związane są odwołania są zgodne ze sobą.  
  
 **Identyfikator błędu:** BC31091  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy pliki zawierające typu występującego w odwołaniu nie są uszkodzone.
