# <a name="end-module-must-be-preceded-by-a-matching-module"></a>"End Module" musi być poprzedzona instrukcją "Module"
Moduł bloki zaczynają się od `Module` instrukcji i kończyć się `End Module` konstruowania.  
  
 **Identyfikator błędu:** BC30622  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że blok modułu zaczyna się od `Module` instrukcji.
