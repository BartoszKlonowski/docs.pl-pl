# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>&#39; Moduł zakończenia &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; Moduł &#39;
Moduł bloków zaczyna się od `Module` instrukcji i kończyć się `End Module` utworzenia.  
  
 **Identyfikator błędu:** BC30622  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że blok modułu zaczyna się od `Module` instrukcji.
