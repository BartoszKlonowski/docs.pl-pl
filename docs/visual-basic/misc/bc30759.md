# <a name="referenced-membername-has-a-value-of-nothing"></a>Przywoływany element " \<membername> " ma wartość "Nothing".
Do elementu, do którego się odwoływano, nie została przypisana wartość lub została do niej przypisana wartość `Nothing` .  
  
 **Identyfikator błędu:** BC30759  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy nazwa elementu jest wpisana i ustawiona poprawnie.
