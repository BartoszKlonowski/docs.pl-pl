# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>Odwołanie do &#39; &lt;membername&gt; &#39; ma wartość &#39;nic&#39;
Element, do którego nastąpiło odwołanie nie zostanie przypisana wartość lub została przypisana wartość `Nothing`.  
  
 **Identyfikator błędu:** BC30759  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwę tego elementu jest wpisana i poprawnie ustawiony.
