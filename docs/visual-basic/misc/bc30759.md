# <a name="referenced-ltmembernamegt-has-a-value-of-nothing"></a>Odwołanie do '&lt;membername&gt;"ma wartość"Nothing"
Element, do którego nastąpiło odwołanie nie przypisano wartości lub została ona przypisana wartość `Nothing`.  
  
 **Identyfikator błędu:** BC30759  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwę tego elementu jest wpisana i prawidłowo ustawione.
