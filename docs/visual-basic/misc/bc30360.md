# <a name="property-ltpropertynamegt-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-ltnamegt"></a>Właściwość "&lt;propertyname&gt;"musi być zadeklarowany jako "Default" ponieważ zastępuje ona domyślnej właściwości na podstawie"&lt;nazwa&gt;"
A `Property` zastępuje instrukcji w kodzie `Default` właściwości w klasie bazowej lub struktury, interfejsu i w związku z tym musi być zadeklarowany `Default`.  
  
 **Identyfikator błędu:** BC30360  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zadeklaruj właściwość `Default`.
