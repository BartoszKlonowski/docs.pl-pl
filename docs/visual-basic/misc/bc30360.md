# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Właściwość &#39; &lt;propertyname&gt; &#39; musi być zadeklarowany jako &#39;domyślne&#39; ponieważ zastępuje ona domyślnej właściwości na podstawie &#39; &lt;nazwy&gt;&#39;
A `Property` zastępuje instrukcji w kodzie `Default` właściwości dla klasy podstawowej struktury, interfejsu i dlatego musi być zadeklarowany jako `Default`.  
  
 **Identyfikator błędu:** BC30360  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie właściwości `Default`.
