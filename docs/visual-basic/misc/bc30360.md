# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>Właściwość " \<propertyname> " musi być zadeklarowana jako "default", ponieważ zastępuje domyślną właściwość w podstawowym " \<name> "

`Property`Instrukcja w kodzie zastępuje `Default` Właściwość w klasie bazowej, strukturze lub interfejsie i dlatego musi być zadeklarowana `Default` .  
  
 **Identyfikator błędu:** BC30360  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj Właściwość `Default` .
