# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>Nazwane argumenty nie są prawidłowe jako indeksy tablicy
Indeks tablicy jest określony, może być przekazywany argumentu procedury według nazwy, na przykład przy użyciu składni `Array(Index := 10)`. Ta składnia jest nieprawidłowa dla indeksy tablicy.  
  
 **Identyfikator błędu:** BC30075  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Na przykład użyć zwykłej wyrażenia stałej lub zmienna jako indeks tablicy `Array(10)`.
