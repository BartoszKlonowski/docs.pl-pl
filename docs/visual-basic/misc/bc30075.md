# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>Nazwane argumenty są nieprawidłowe w przypadku indeksy dolne tablicy
Indeks tablicy jest określony, przy użyciu składni do przekazywania argumentu procedury przy użyciu nazwy, na przykład `Array(Index := 10)`. Ta składnia jest nieprawidłowa dla indeksy dolne tablicy.  
  
 **Identyfikator błędu:** BC30075  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Na przykład użyć zwykłej wyrażenia zmiennej czy stałej jako indeks tablicy `Array(10)`.
