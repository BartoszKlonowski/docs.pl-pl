# <a name="event-name-length-cannot-exceed-1011-characters"></a>Długość nazwy zdarzenia nie może przekraczać 1011 znaków
Nazwa lub identyfikator każdego zdarzenia zadeklarowanego za pomocą `Event` instrukcji jest ograniczony do 1011 znaków.  
  
 **Identyfikator błędu:** BC32204  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmniejsz długość identyfikatora.
