# <a name="the-folder-cannot-be-created-since-a-file-already-exists-with-the-same-path"></a>Nie można utworzyć folderu, ponieważ istnieje już plik o tej samej ścieżce

Podjęto próbę utworzenia katalogu, ale plik już istnieje w tej ścieżce. Może to być spowodowane niepoprawnym analizowaniem ścieżek, szczególnie podczas pracy ze ścieżkami względnymi.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy podano poprawną ścieżkę.  
  
- Usuń nadmiarowy plik, jeśli nie jest już wymagany.
