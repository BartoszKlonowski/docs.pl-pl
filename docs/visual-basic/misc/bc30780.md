# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Kontynuować&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Continue` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30780  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Continue` instrukcji z debugowania kodu.
