# <a name="label-labelname-is-not-defined"></a>Etykieta " \<labelname> " nie jest zdefiniowana

Etykieta linii lub numer wiersza jest określany jako, ale nie zdefiniowany w zakresie odwołania. Etykieta musi znajdować się w procedurze zawierającej odwołanie.  
  
 **Identyfikator błędu:** BC30132  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy zmienić strukturę kodu, aby etykieta wiersza i odwołanie były w tej samej procedurze.
