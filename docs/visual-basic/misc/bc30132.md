# <a name="label-39ltlabelnamegt39-is-not-defined"></a>Etykieta &#39; &lt;wartość labelname&gt;&#39; nie jest zdefiniowany
Etykiety linii lub numer wiersza jest określony, ale nie jest zdefiniowana w zakresie odwołania. Etykieta musi być w ramach tej procedury, która zawiera odwołanie.  
  
 **Identyfikator błędu:** BC30132  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja kod, więc etykiety linii i odwołania znajdują się w tej samej procedury.
