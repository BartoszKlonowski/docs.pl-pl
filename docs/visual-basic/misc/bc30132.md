# <a name="label-ltlabelnamegt-is-not-defined"></a>Etykieta "&lt;nazwa_etykiety&gt;" nie jest zdefiniowana
Etykieta wiersza lub numer wiersza jest określony, ale nie jest zdefiniowany w zakresie odwołania. Etykieta musi mieć w ramach procedury, która zawiera odwołanie do.  
  
 **Identyfikator błędu:** BC30132  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja kod, aby etykieta wiersza i odwołania znajdują się w tej samej procedury.
