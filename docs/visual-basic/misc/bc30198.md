# <a name="-expected"></a>Oczekiwano znaku ")"
Lista, taka jak lista argumentów w definicji procedury lub wywołania procedury, nie została zakończona przez nawias zamykający ( `)` ).  
  
 **Identyfikator błędu:** BC30198  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak nawiasu zamykającego na końcu listy.
