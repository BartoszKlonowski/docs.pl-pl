# <a name="3939-expected"></a>&#39;) &#39; Oczekiwano
Listy, na przykład listy argumentów w definicji procedurze lub wywołania procedury nie zostanie zakończony przez nawiasu zamykającego (`)`) znaków.  
  
 **Identyfikator błędu:** BC30198  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj zamykający nawias okrągły znak na końcu listy.
