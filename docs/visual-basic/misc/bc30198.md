# <a name="-expected"></a>')' oczekiwany
Lista, na przykład listy argumentów w wywołaniu procedury, lub w definicji procedury nie jest zakończony przez nawias zamykający (`)`) znaków.  
  
 **Identyfikator błędu:** BC30198  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj nawias zamykający znak na końcu listy.
