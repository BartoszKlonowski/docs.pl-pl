# <a name="type-parameter-typeparametername-has-the-same-name-as-a-type-parameter-of-an-enclosing-type"></a>Parametr typu '\<typeparametername >' ma taką samą nazwę jak parametr typu otaczającego
Parametr typu ogólnego jest zadeklarowany z taką samą nazwę jak parametr typu ogólnego typu zawierającego.  
  
 Na liście typów parametrów typu ogólnego każdy parametr typu musi mieć nazwę różne od wszystkich następujących nazw:  
  
- Każdy parametr typu w tym samym lista parametrów typu  
  
- Każdy parametr typu w liście parametrów typu zawierającego typ ogólny, i  
  
- Nazwa typu ogólnego, sam.  
  
 Domyślnie ta wiadomość jest ostrzeżenie. Uzyskać informacje o ukrywaniu ostrzeżenia lub traktowanie ostrzeżeń jako błędy, zobacz [Konfigurowanie ostrzeżeń w języku Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identyfikator błędu:** BC40048  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę parametru typu być odmienna od każda nazwa wymienionego na liście na tej stronie pomocy.
