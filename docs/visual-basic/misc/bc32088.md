# <a name="type-arguments-unexpected"></a><span data-ttu-id="8e814-102">Nieoczekiwane argumenty typu</span><span class="sxs-lookup"><span data-stu-id="8e814-102">Type arguments unexpected</span></span>
<span data-ttu-id="8e814-103">Klauzula `Implements` dostarcza argumentów typu dla elementu członkowskiego interfejsu, który implementuje.</span><span class="sxs-lookup"><span data-stu-id="8e814-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="8e814-104">Klauzula `Implements` powinna identyfikować tylko interfejs i element członkowski, który implementuje.</span><span class="sxs-lookup"><span data-stu-id="8e814-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="8e814-105">Oznacza to, że w przypadku deklarowania procedury ogólnej, klauzula `Of` i argumenty typu powinny pojawić się w głównej części deklaracji, tak jak gdyby nie zaimplementowano procedury interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8e814-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="8e814-106">Poniższy kod może wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="8e814-106">The following code can generate this error.</span></span>  
  
```vb  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="8e814-107">Deklaracja wcześniejsza niż klauzula `Implements` powinna wyglądać podobnie do definicji interfejsu, z wyjątkiem możliwego dodania modyfikatorów dostępu lub procedur.</span><span class="sxs-lookup"><span data-stu-id="8e814-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="8e814-108">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="8e814-108">The following code avoids the error.</span></span>  
  
```vb  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="8e814-109">**Identyfikator błędu:** BC32088</span><span class="sxs-lookup"><span data-stu-id="8e814-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8e814-110">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="8e814-110">To correct this error</span></span>  
  
- <span data-ttu-id="8e814-111">Usuń listę argumentów typu z klauzuli `Implements`.</span><span class="sxs-lookup"><span data-stu-id="8e814-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
- <span data-ttu-id="8e814-112">W przypadku implementowania ogólnego elementu członkowskiego interfejsu, umieść listę argumentów typu w głównej części deklaracji, poprzedzając słowo kluczowe `Implements`.</span><span class="sxs-lookup"><span data-stu-id="8e814-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
