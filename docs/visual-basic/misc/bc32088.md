# <a name="type-arguments-unexpected"></a><span data-ttu-id="90dd6-102">Nieoczekiwane argumenty typu</span><span class="sxs-lookup"><span data-stu-id="90dd6-102">Type arguments unexpected</span></span>
<span data-ttu-id="90dd6-103">`Implements` Klauzuli dostarcza argumentów typu dla członka interfejsu implementuje.</span><span class="sxs-lookup"><span data-stu-id="90dd6-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="90dd6-104">`Implements` Klauzuli tylko należy zidentyfikować interfejsu i implementuje element członkowski.</span><span class="sxs-lookup"><span data-stu-id="90dd6-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="90dd6-105">Oznacza to, że jeśli są deklarowanie procedury ogólne, `Of` klauzul i argumentów typu powinna pojawić się w głównej części deklaracji, tak samo, jak gdyby zostały nie implementuje interfejsu procedury.</span><span class="sxs-lookup"><span data-stu-id="90dd6-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="90dd6-106">Poniższy kod może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="90dd6-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="90dd6-107">Poprzedniej deklaracji `Implements` klauzuli powinno wyglądać podobnie do definicji interfejsu, z wyjątkiem możliwe dodanie modyfikatorów dostępu lub procedury.</span><span class="sxs-lookup"><span data-stu-id="90dd6-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="90dd6-108">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="90dd6-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="90dd6-109">**Identyfikator błędu:** BC32088</span><span class="sxs-lookup"><span data-stu-id="90dd6-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="90dd6-110">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="90dd6-110">To correct this error</span></span>  
  
-   <span data-ttu-id="90dd6-111">Usuń z listy argumentów typu `Implements` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="90dd6-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="90dd6-112">Jeśli użytkownik wdrażania ogólnego elementu członkowskiego interfejsu, a następnie umieść listy argumentów typu w głównej części deklaracji, przed `Implements` — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="90dd6-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
