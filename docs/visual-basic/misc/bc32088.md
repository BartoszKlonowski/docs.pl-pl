# <a name="type-arguments-unexpected"></a><span data-ttu-id="4a2ad-102">Nieoczekiwane argumenty typu</span><span class="sxs-lookup"><span data-stu-id="4a2ad-102">Type arguments unexpected</span></span>
<span data-ttu-id="4a2ad-103">`Implements` Klauzuli dostarcza argumentów typu dla elementu członkowskiego interfejsu implementuje.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="4a2ad-104">`Implements` Klauzuli tylko należy zidentyfikować interfejsu i członka implementuje.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="4a2ad-105">Oznacza to, że jeśli użytkownik deklaruje procedurę ogólny, `Of` klauzuli oraz argumentów typu powinna być wyświetlana w głównej części deklaracji, tak, jak gdyby były nie implementuje interfejsu procedury.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="4a2ad-106">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="4a2ad-107">Poprzedniej deklaracji `Implements` klauzuli powinno wyglądać podobnie do definicji interfejsu, z wyjątkiem możliwości dodawania modyfikatory dostępu lub procedury.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="4a2ad-108">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="4a2ad-109">**Identyfikator błędu:** BC32088</span><span class="sxs-lookup"><span data-stu-id="4a2ad-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4a2ad-110">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4a2ad-110">To correct this error</span></span>  
  
-   <span data-ttu-id="4a2ad-111">Usuń z listy argumentów typu z `Implements` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="4a2ad-112">Jeśli użytkownik wdrażania ogólnej składowej interfejsu, a następnie umieść lista argumentów typu w głównej części deklaracji, przed `Implements` — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="4a2ad-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
