## <a name="to-correct-this-error"></a><span data-ttu-id="cfc78-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="cfc78-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="cfc78-108">Użyj <xref:System.DBNull> zamiast `Null`.</span><span class="sxs-lookup"><span data-stu-id="cfc78-108">Use <xref:System.DBNull> instead of `Null`.</span></span> <span data-ttu-id="cfc78-109">Poniższy przykład przedstawia to.</span><span class="sxs-lookup"><span data-stu-id="cfc78-109">The following example demonstrates this.</span></span>  
  
    ```  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2.  <span data-ttu-id="cfc78-110">Użyj [nic](../../visual-basic/language-reference/nothing.md) — słowo kluczowe dla przydziałów i porównywanie, gdy w przypadku używania zmiennych obiektu.</span><span class="sxs-lookup"><span data-stu-id="cfc78-110">Use the [Nothing](../../visual-basic/language-reference/nothing.md) keyword for assignments and comparisons when you use object variables.</span></span> <span data-ttu-id="cfc78-111">Poniższy przykład przedstawia to.</span><span class="sxs-lookup"><span data-stu-id="cfc78-111">The following example demonstrates this.</span></span>

    ```  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
