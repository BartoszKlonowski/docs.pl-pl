# <a name="byval-and-byref-cannot-be-combined"></a>Nie można połączyć "ByVal" i "ByRef"
`ByVal` i `ByRef` słowa kluczowe są wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30641  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz opcję `ByVal` lub `ByRef`, ale nie oba jednocześnie.
