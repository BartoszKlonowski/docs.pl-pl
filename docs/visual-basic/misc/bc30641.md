# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39; ByVal &#39; i &#39; ByRef &#39; Nie można łączyć
`ByVal` i `ByRef` słowa kluczowe wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30641  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `ByVal` lub `ByRef`, ale nie oba.
