# <a name="byval-and-byref-cannot-be-combined"></a>Nie można połączyć elementu "ByVal" i "ByRef"

`ByVal` `ByRef` Słowa kluczowe i wykluczają się wzajemnie.  
  
 **Identyfikator błędu:** BC30641  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ jeden `ByVal` lub `ByRef` , ale nie oba.
