# <a name="paramarray-parameters-must-be-declared-byval"></a>Parametry ParamArray musi być zadeklarowany jako "ByVal"
`ParamArray` Nie można używać parametrów `ByRef` modyfikator.  
  
 **Identyfikator błędu:** BC30667  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj `ParamArray` parametrów przy użyciu `ByVal` modyfikator.
