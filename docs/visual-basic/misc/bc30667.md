# <a name="paramarray-parameters-must-be-declared-39byval39"></a>Parametry ParamArray muszą być zadeklarowane &#39;ByVal&#39;
`ParamArray` Nie można używać parametrów `ByRef` modyfikator.  
  
 **Identyfikator błędu:** BC30667  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie `ParamArray` parametry, używając `ByVal` modyfikator.
