# <a name="39group39-not-allowed-in-this-context-identifier-expected"></a>&#39; Grupy &#39; nie są dozwolone w tym kontekście; Oczekiwano identyfikatora
`Group` — Słowo kluczowe jest uwzględniona w `Into` sekcji `Aggregate` klauzuli. `Group` — Słowo kluczowe jest prawidłowy tylko w `Group By` lub `Group Join` klauzul.  
  
 **Identyfikator błędu:** BC36618  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Group` — słowo kluczowe z `Aggregate` klauzuli. Możesz dodać `Group By` klauzuli zapytania do grupowania wyników.
