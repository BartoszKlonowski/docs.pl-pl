# <a name="no-default-member-found-for-type-lttypenamegt"></a>Nie domyślnego elementu członkowskiego znaleziono dla typu "&lt;typename&gt;"
Indeks z późnym wiązaniem `Get` lub `Set` podjęto próbę wywołania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że obiekt ma właściwość domyślną z `Get` lub `Set`, w zależności od tego, który chcesz utworzyć.
