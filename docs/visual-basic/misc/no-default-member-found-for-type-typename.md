# <a name="no-default-member-found-for-type-typename"></a>Nie znaleziono domyślnego elementu członkowskiego dla typu " \<typename> "
`Get`Podjęto próbę wykonania indeksu lub wywołania z późnym wiązaniem `Set` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że obiekt ma właściwość domyślną z `Get` lub `Set` , w zależności od tego, którą próbujesz wykonać.
