# <a name="no-default-member-found-for-type-39lttypenamegt39"></a>Nie można znaleźć typu &#39; członka domyślnego &lt;typename&gt;&#39;
Indeks późnym wiązaniem `Get` lub `Set` podjęto próbę wywołania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że obiekt ma właściwości domyślnej z `Get` lub `Set`, w zależności od tego, który chcesz utworzyć.
