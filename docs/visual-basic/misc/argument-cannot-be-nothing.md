# <a name="argument-cannot-be-nothing"></a>Argument nie może mieć wartości Nothing
Wartość null została podana, musi mieć wartość argumentu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Może mieć próbowano użyć obiektu bez podawania wystąpienia obiektu. W takim przypadku należy użyć `New` — słowo kluczowe, aby utworzyć wystąpienie.  
  
-   Sprawdź, czy wartość jest obliczany poprawnie.
