# <a name="argument-cannot-be-nothing"></a>Argument nie może mieć wartości Nothing

Podano wartość null dla argumentu, który musi mieć wartość.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Być może podjęto próbę użycia obiektu bez podawania wystąpienia obiektu. W takim przypadku należy użyć `New` słowa kluczowego, aby utworzyć wystąpienie.  
  
- Sprawdź, czy wartość jest obliczana prawidłowo.
