<span data-ttu-id="861be-103">Modyfikator typu dopuszczającego wartość null (?) jest zawarty w zmiennej w deklaracji zmiennej, w której Modyfikatory tablic (nawiasy) są zawarte w określonym typie zmiennej.</span><span class="sxs-lookup"><span data-stu-id="861be-103">The nullable type modifier (?) is included on the variable in a variable declaration in which the array modifiers (parentheses) are included on the specified variable type.</span></span> <span data-ttu-id="861be-104">Lub modyfikator typu dopuszczającego wartość null jest uwzględniany w określonym typie zmiennej w deklaracji zmiennej, w której Modyfikatory tablic są uwzględnione w zmiennej.</span><span class="sxs-lookup"><span data-stu-id="861be-104">Or, the nullable type modifier is included on the specified variable type in a variable declaration in which the array modifiers are included on the variable.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="861be-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="861be-106">To correct this error</span></span>  
  
1. <span data-ttu-id="861be-107">Określ modyfikator typu dopuszczającego wartość null (?) i modyfikatory tablicy (nawiasy) dla deklarowanej zmiennej lub określonego typu zmiennej, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="861be-107">Specify both the nullable type modifier (?) and the array modifiers (parentheses) on either the declared variable or the specified variable type, as shown in the following example.</span></span>  
  
```vb

' Dim numbers? As Integer()  
' Dim values() As Integer?  
  
'These are correct.

Dim values As Integer?()  
```
