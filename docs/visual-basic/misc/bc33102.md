<span data-ttu-id="82c3a-103">Modyfikator typu dopuszczającego wartość zerową (?) znajduje się na zmiennej w deklaracji zmiennej, w którym Modyfikatory tablic (nawiasów) znajdują się na określony typ zmiennej.</span><span class="sxs-lookup"><span data-stu-id="82c3a-103">The nullable type modifier (?) is included on the variable in a variable declaration in which the array modifiers (parentheses) are included on the specified variable type.</span></span> <span data-ttu-id="82c3a-104">Lub modyfikator typu dopuszczającego wartość null znajduje się na określony typ zmiennej w deklaracji zmiennej, w którym Modyfikatory tablicy znajdują się w zmiennej.</span><span class="sxs-lookup"><span data-stu-id="82c3a-104">Or, the nullable type modifier is included on the specified variable type in a variable declaration in which the array modifiers are included on the variable.</span></span>
