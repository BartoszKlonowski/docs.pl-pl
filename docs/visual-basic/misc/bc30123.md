# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "End" nie są prawidłowe w oknie bezpośrednim

`Stop`Instrukcje and `End` zawieszają wykonywanie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie należy wystawiać `End` `Stop` instrukcji lub w oknie **bezpośrednim** .
