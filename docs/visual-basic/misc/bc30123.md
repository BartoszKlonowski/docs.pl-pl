# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Końcowy&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Stop` i `End` instrukcje wstrzymać wykonywania i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie wydają `End` lub `Stop` instrukcji w **Immediate** okna.
