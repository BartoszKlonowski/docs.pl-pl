# <a name="sub-new-cannot-be-declared-overrides"></a>"Sub New" nie można zadeklarować jako "Overrides"
Konstruktor wskazuje, że zastępuje ona dziedziczonego konstruktora. Nie można zastąpić konstruktorów.  
  
 **Identyfikator błędu:** BC30283  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Overrides` słowo kluczowe z `Sub` deklaracji.
