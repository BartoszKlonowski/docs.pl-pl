# <a name="character-constant-must-contain-exactly-one-character"></a>Stała znaku musi zawierać dokładnie jeden znak
Stała znaku jest nieprawidłowa, ponieważ zawiera więcej niż jeden znak lub jest znakiem null (wszystkie bity są równe zero).  
  
 **Identyfikator błędu:** BC30004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień znak stałej na pojedynczy znak lub Zmień znak null na prawidłowy znak.
