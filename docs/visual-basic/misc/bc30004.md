# <a name="character-constant-must-contain-exactly-one-character"></a>Stała znakowa musi zawierać dokładnie jeden znak
Stała znakowa jest nieprawidłowy, ponieważ albo zawiera więcej niż jednego znaku lub znak null (wszystkie bity są zero).  
  
 **Identyfikator błędu:** BC30004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień stała znakowa pojedynczy znak lub zmień znakiem pustym nieprawidłowy znak.
