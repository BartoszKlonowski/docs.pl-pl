# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;mathfunction1&gt; &#39; nie jest zadeklarowana
"\<mathfunction1 >' nie został zadeklarowany. Funkcja została przeniesiona do klasy System.Math i nosi teraz nazwę "\<mathfunction2 >".  
  
 Kilka funkcji, które były wewnętrznej na język Visual Basic w poprzednich wersjach zostały przeniesione do <xref:System.Math?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcji ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30819  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj metody zadeklarowane w <xref:System.Math?displayProperty=nameWithType>.
