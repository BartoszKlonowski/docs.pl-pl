# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;mathfunction1&gt; &#39; nie została zadeklarowana
"\<mathfunction1 >' nie został zadeklarowany. Funkcja została przeniesiona do klasy System.Math i jest teraz nazywana "\<mathfunction2 >".  
  
 Kilka funkcji, które były wewnętrzne w języku Visual Basic w poprzednich wersjach zostały przeniesione do ikony <xref:System.Math?displayProperty=nameWithType> przestrzeni nazw. To sprawia, że ich funkcjonalność bardziej ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30819  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Używanie metod zadeklarowanych w <xref:System.Math?displayProperty=nameWithType>.
