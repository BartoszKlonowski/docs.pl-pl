# <a name="mathfunction1-is-not-declared"></a>element " \<mathfunction1> " nie jest zadeklarowany

element " \<mathfunction1> " nie jest zadeklarowany. Funkcja została przeniesiona do klasy System. Math i ma teraz nazwę " \<mathfunction2> ".  
  
 Kilka funkcji, które były wewnętrznie Visual Basic w poprzednich wersjach, zostały przeniesione do <xref:System.Math?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcjonalność jest bardziej ogólnie dostępna dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30819  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj metod zadeklarowanych w <xref:System.Math?displayProperty=nameWithType> .
