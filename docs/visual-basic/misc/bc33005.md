# <a name="end-operator-expected"></a>Oczekiwano instrukcji "End Operator"
`End`Instrukcja musi być zgodna z zamykanym blokiem. `Operator`Definicja procedury nie została prawidłowo zakończona.  
  
 **Identyfikator błędu:** BC33005  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj `End Operator` instrukcję do końca procedury.
