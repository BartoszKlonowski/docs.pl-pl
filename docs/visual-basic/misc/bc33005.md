# <a name="end-operator-expected"></a>Oczekiwano "End Operator"
`End` Instrukcja musi odpowiadać blok zostaje zakończone. `Operator` Definicji procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC33005  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `End Operator` instrukcji na końcu procedury.
