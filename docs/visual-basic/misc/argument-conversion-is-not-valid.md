# <a name="argument-conversion-is-not-valid"></a>Argument "Conversion" jest nieprawidłowy

Wartość `VbStrConv` wyliczenia jest nieprawidłowa, prawdopodobnie dlatego, że jest mniejsza od zera. Wartość wyliczenia musi być większa lub równa zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że wartość jest większa lub równa zero.
