# <a name="argument-argumentname-must-be-within-the-range-1-to-255"></a>Argument " \<argumentname> " musi należeć do zakresu od 1 do 255

Argument jest nieprawidłowy, ponieważ wykracza poza zakres od 0 do 255.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię argumentów w wyrażeniu. Błędnie wpisana nazwa zmiennej może niejawnie utworzyć zmienną numeryczną, która została zainicjowana do zera.  
  
2. Sprawdź poprzednie operacje na zmiennych w wyrażeniu, szczególnie te przekazane do procedury jako argumenty z innych procedur.
