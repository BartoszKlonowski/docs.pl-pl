# <a name="a-nullable-type-cannot-be-inferred-for-variable-variablename"></a>Nie można wywnioskować typu\<nullable dla zmiennej ' variablename>'
Typu wartości możliwej do wartości null nie można wywnioskować z `String`typu odwołania, takiego jak tablica, klasa lub . Wartość, z której wywnioskowany jest typ danych, musi być typem wartości. Poniższy kod ilustruje ten błąd.  
  
```vb  
'' Not valid.
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **Identyfikator błędu:** Bc36628  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń oznaczenie możliwe do wyzernia do wartości.
