# <a name="39redim39-cannot-change-the-number-of-dimensions"></a>&#39; ReDim &#39; Nie można zmienić liczby wymiarów
Operacja próbuje użyć `ReDim` instrukcji, aby zmienić rangę tablicy (liczba wymiarów). `ReDim`można zmienić rozmiar co najmniej jeden wymiar tablicy, która została już formalnie zadeklarowana, ale nie można zmienić rangę tablicy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że chcesz zmienić rangę tablicy i nie rozmiary jej wymiarów i, jeśli to możliwe, użyj `Dim` Aby zadeklarować nowej tablicy z odpowiednią pozycję.
