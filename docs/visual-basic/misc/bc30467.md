# <a name="ltname1gt-for-the-imports-ltname2gt-does-not-refer-to-a-namespace-class-structure-enum-or-module"></a>"&lt;Nazwa1&gt;"dla operacji importu"&lt;Nazwa2&gt;" nie odwołuje się do Namespace, klasy, struktury, wyliczenia lub modułu
Podjęto próbę użycia `Imports` instrukcji element, który nie jest `Namespace`, `Class`, `Structure`, `Enum`, lub `Module`. `Imports` Instrukcji imports nazwy przestrzeni nazw z przywoływane projekty i zestawy lub importuje zdefiniowane w tym samym projekcie jako moduł, w której występuje instrukcja nazwy przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30467  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zaznacz jednostki, że próbujesz zaimportować, i upewnij się, że jest prawidłowa do stosowania z `Imports` instrukcji.
