# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Nie można skojarzyć bloku komentarza XML z dowolnym elementem języka, który obsługuje stosowanie komentarzy dokumentacji XML

Bloku komentarza XML nie można skojarzyć z żadnym prawidłowym elementem języka. Komentarz XML zostanie zignorowany. Ten błąd występuje również, gdy tag XML w bloku komentarza nie ma pasującego elementu języka.  
  
 **Identyfikator błędu:** BC42312  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Znajdź i Usuń nadmiarowy tag XML lub podaj element języka, który opisuje.
