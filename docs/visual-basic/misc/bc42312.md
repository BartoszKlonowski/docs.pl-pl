# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Blok komentarza XML nie może być skojarzony z wszystkich elementów języka, który obsługuje aplikację komentarze dokumentacji XML
Blok komentarza XML nie może być skojarzony z dowolnym prawidłowym elementem języka. Komentarz XML zostanie zignorowany. Ten błąd występuje także, jeśli nie pasującego elementu języka tagu XML w blok komentarza.  
  
 **Identyfikator błędu:** BC42312  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Znajdź i Usuń nadmiarowe tagu XML lub podać elementu języka, które opisano w nim.
