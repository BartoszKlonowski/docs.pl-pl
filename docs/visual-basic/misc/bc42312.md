# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Blok komentarza XML nie może być skojarzony z żadnym elementem języka, który obsługuje aplikację komentarze dokumentacji XML
Blok komentarza XML nie może być skojarzony z żadnym elementem odpowiedni język. Komentarz XML zostanie zignorowany. Ten błąd występuje także jeśli brak pasującego elementu języka ma tagu XML w bloku komentarza.  
  
 **Identyfikator błędu:** BC42312  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Znajdź i Usuń nadmiarowe tagu XML lub podaj element języka, który opisuje.
