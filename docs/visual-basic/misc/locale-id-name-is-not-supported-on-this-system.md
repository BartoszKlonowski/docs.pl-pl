# <a name="locale-id-39ltnamegt39-is-not-supported-on-this-system"></a>Identyfikator ustawień regionalnych &#39; &lt;nazwa&gt; &#39; nie jest obsługiwany w tym systemie
Próbujesz użyć `VbStrConv`, ale system nie obsługuje określonych ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Uaktualnienie komputera do obsługi ustawień regionalnych, lub uruchomić aplikację na komputerze, który obsługuje określone ustawienia regionalne.
