# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Parametrów metod "AddHandler" i "RemoveHandler" nie można zadeklarować jako "ByRef"
Parametry `AddHandler` i `RemoveHandler` metody nie może być zadeklarowana z `ByRef` modyfikator.  
  
 **Identyfikator błędu:** BC31134  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `ByRef` — słowo kluczowe z parametru.
