# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Parametrów metody "AddHandler" i "RemoveHandler" nie można zadeklarować jako "ByRef"

Parametrów `AddHandler` i `RemoveHandler` metod nie można zadeklarować za pomocą `ByRef` modyfikatora.  
  
 **Identyfikator błędu:** BC31134  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `ByRef` słowo kluczowe z parametru.
