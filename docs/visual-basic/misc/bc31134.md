# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39; AddHandler &#39; i &#39; RemoveHandler &#39; Nie można zadeklarować jako parametry metody &#39; ByRef &#39;
Parametry `AddHandler` i `RemoveHandler` metody nie można zadeklarować ze `ByRef` modyfikator.  
  
 **Identyfikator błędu:** BC31134  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `ByRef` — słowo kluczowe z parametru.
