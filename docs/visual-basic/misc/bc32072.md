# <a name="cannot-implement-interface-ltinterfacename1gt-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-ltinterfacename2gt-for-some-type-arguments"></a><span data-ttu-id="b9578-102">Nie można zaimplementować interfejsu "&lt;interfacename1&gt;"ponieważ jego implementacja może powodować konflikt z implementacją innego zaimplementowanego interfejsu"&lt;interfacename2&gt;" dla niektórych argumentów typu</span><span class="sxs-lookup"><span data-stu-id="b9578-102">Cannot implement interface '&lt;interfacename1&gt;' because its implementation could conflict with the implementation of another implemented interface '&lt;interfacename2&gt;' for some type arguments</span></span>
<span data-ttu-id="b9578-103">Deklaracja klasy zawiera `Implements` instrukcję, która określa dwie lub więcej interfejsów, ale co najmniej jednego z interfejsów ogólnych i dwie implementacje mogą powodować konflikt dla niektórych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="b9578-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="b9578-104">Poniższe instrukcje może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="b9578-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="b9578-105">Ponieważ `iFace2` jest tworzony przy użyciu `String`, `testClass` musi implementować dwie wersje `testSub` o identycznych podpisach.</span><span class="sxs-lookup"><span data-stu-id="b9578-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="b9578-106">Ten sposób dałby w efekcie niejednoznaczności dotyczące dostępu do wersji.</span><span class="sxs-lookup"><span data-stu-id="b9578-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="b9578-107">**Identyfikator błędu:** BC32072</span><span class="sxs-lookup"><span data-stu-id="b9578-107">**Error ID:** BC32072</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b9578-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="b9578-108">To correct this error</span></span>

     <span data-ttu-id="b9578-110">—lub—</span><span class="sxs-lookup"><span data-stu-id="b9578-110">-or-</span></span>
