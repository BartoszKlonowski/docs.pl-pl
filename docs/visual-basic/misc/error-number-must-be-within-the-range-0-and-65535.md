# <a name="error-number-must-be-within-the-range-0-and-65535"></a>Numer błędu musi należeć do zakresu od 0 do 65535

Numer błędu wykracza poza dozwolony zakres od 0 do 65535.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że liczba mieści się w prawidłowym zakresie.
