# <a name="all-field-widths-except-the-last-element-must-be-greater-than-zero"></a>Wszystkie szerokości pól z wyjątkiem ostatniego elementu musi być większa od zera.
Wszystkie szerokości pól z wyjątkiem ostatniego elementu musi być większa niż zero. Szerokość pola mniejsza lub równa zero w ostatnim elemencie wskazuje, że jest ostatnie pole zmiennej długości.  
  
 Operacja nie powiodła się, ponieważ szerokość pola innego niż ostatni element jest ustawiony na zero lub mniej. Szerokość pola mniejsza lub równa zero mogą być używane jako ostatni element do wskazywania ostatnie pole zmiennej długości, ale nie można użyć jako innego elementu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ustaw szerokość pola poprawne długość.
