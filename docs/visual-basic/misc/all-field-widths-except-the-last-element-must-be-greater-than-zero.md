# <a name="all-field-widths-except-the-last-element-must-be-greater-than-zero"></a>Wszystkie szerokości pól, z wyjątkiem ostatniego elementu, muszą być większe od zera.

Wszystkie szerokości pól, z wyjątkiem ostatniego elementu, muszą być większe od zera. Szerokość pola mniejsza lub równa zero w ostatnim elemencie wskazuje, że ostatnie pole ma zmienną długość.  
  
 Operacja nie powiodła się, ponieważ szerokość pola inna niż ostatni element ma wartość zero lub mniejszą. Szerokość pola mniejsza lub równa zero może być używana jako ostatni element wskazujący, że ostatnie pole ma zmienną długość, ale nie może być używane jako jakikolwiek inny element.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw szerokość pola na poprawną długość.
