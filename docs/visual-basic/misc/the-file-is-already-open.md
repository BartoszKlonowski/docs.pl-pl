# <a name="the-file-is-already-open"></a>Plik jest już otwarty
Podjęta próba otwarcia pliku, który jest już otwarty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik przed podjęciem próby otwarcia go.  
  
     —lub—  
  
- Usuń zbędną `FileOpen` instrukcji w kodzie.
