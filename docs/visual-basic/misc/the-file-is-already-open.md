# <a name="the-file-is-already-open"></a>Plik jest już otwarty

Podjęto próbę otwarcia pliku, który jest już otwarty.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik przed próbą jego otwarcia.  
  
     -lub-  
  
- Usuń instrukcję nadmiarową `FileOpen` z kodu.
