# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowy. Prawidłowe wartości w trybie wprowadzania to "OpenAccess. Read" i "OpenAccess. default"

Wartość dostępu jest nieprawidłowa w `Input` trybie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość na `OpenAccess.Read` lub `OpenAccess.Default` .
