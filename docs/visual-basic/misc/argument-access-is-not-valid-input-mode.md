# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowa. Prawidłowe wartości dla trybu dane wejściowe to "OpenAccess.Read" i "OpenAccess.Default"
Wartość dostępu jest nieprawidłowy w `Input` trybu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość na jedną `OpenAccess.Read` lub `OpenAccess.Default`.
