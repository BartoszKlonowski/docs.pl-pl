# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;Nazwa&gt; &#39; jest zadeklarowany w projekcie &#39; &lt;projectname1&gt;&#39;, która nie odwołuje się projekt &#39; &lt;projectname2&gt;&#39;
Element programowania jest zdefiniowany w projekcie, do którego bieżący projekt nie ma odwołania.  
  
 **Identyfikator błędu:** BC32004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   W bieżącym projekcie Dodaj odwołanie do projektu, który zawiera określony element.
