# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>"&lt;nazwa&gt;"jest zadeklarowany w projekcie"&lt;projectname1&gt;", który nie odwołuje się projekt"&lt;projectname2&gt;"
Element programowania jest zdefiniowana w projekcie, do którego bieżący projekt nie ma odwołania.  
  
 **Identyfikator błędu:** BC32004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   W bieżącym projekcie Dodaj odwołanie do projektu, który zawiera określony element.
