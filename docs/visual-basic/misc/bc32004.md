# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>'\<name >' jest zadeklarowany w projekcie '\<projectname1 >', który nie odwołuje się projekt'\<projectname2 >'
Element programowania jest zdefiniowana w projekcie, do którego bieżący projekt nie ma odwołania.  
  
 **Identyfikator błędu:** BC32004  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- W bieżącym projekcie Dodaj odwołanie do projektu, który zawiera określony element.
