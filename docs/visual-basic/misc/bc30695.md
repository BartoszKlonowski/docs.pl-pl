# <a name="lttypenamegt-ltmembernamegt-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>&lt;Element TypeName&gt; "&lt;membername&gt;" musi być zadeklarowany jako "Shadows" ponieważ inna składowa o tej nazwie jest zadeklarowana jako "Shadows"
Ta właściwość lub procedura ma taką samą nazwę jak inna właściwość lub procedura jest oznaczona za pomocą `Shadows` modyfikator.  
  
 **Identyfikator błędu:** BC30695  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Shadows` modyfikator.
