# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;Właściwość TypeName&gt; &#39; &lt;membername&gt; &#39; musi być zadeklarowany jako &#39;Shadows&#39; ponieważ inny członek o tej nazwie jest zadeklarowany jako &#39;cieni&#39;
Ta właściwość lub procedura ma taką samą nazwę co inna właściwość lub procedura oznaczonych `Shadows` modyfikator.  
  
 **Identyfikator błędu:** BC30695  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Shadows` modyfikator.
