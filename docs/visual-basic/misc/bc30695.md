# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<typename>element " \<membername> " musi być zadeklarowany jako "Shadows", ponieważ inny element członkowski o tej nazwie jest zadeklarowany jako "Shadows"
Ta właściwość lub procedura ma taką samą nazwę jak inna właściwość lub procedura oznaczona `Shadows` modyfikatorem.  
  
 **Identyfikator błędu:** BC30695  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Shadows` modyfikatora.
