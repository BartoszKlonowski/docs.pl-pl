# <a name="expected-closing--for-xml-entity"></a>Oczekiwano zamykającego ";" dla jednostki XML
Odwołania do jednostki XML (na przykład `>`) brakuje zamykającego średnika (;).  
  
 **Identyfikator błędu:** BC31178  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Obejmują średnikami, na końcu odwołania do jednostki XML.
