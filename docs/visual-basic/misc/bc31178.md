# <a name="expected-closing--for-xml-entity"></a>Oczekiwano zamykającego znaku ";" dla jednostki XML
Odwołanie do jednostki XML (na przykład `>` ) nie zawiera zamykającego średnika (;).  
  
 **Identyfikator błędu:** BC31178  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dołącz średnik na końcu odwołania do jednostki XML.
