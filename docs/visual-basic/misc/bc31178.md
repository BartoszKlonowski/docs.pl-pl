# <a name="expected-closing-3939-for-xml-entity"></a>Oczekiwano zamykających znaków &#39;; &#39; dla jednostki XML
Odwołania do jednostek XML (na przykład `>`) brakuje zamykającego średnikami (;).  
  
 **Identyfikator błędu:** BC31178  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Obejmują średnik na końcu odwołanie do jednostki XML.
