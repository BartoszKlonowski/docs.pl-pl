# <a name="option-39ltoptiongt39-cannot-be-followed-by-3939-or-39-39-ignored"></a>Opcja &#39; &lt;opcji&gt; &#39; nie może następować &#39;+&#39; lub &#39;-&#39;; zignorowano
Określono opcję kompilatora z wartością `+` lub `-`; te wartości są nieobsługiwane.  
  
 Aby uzyskać więcej informacji na temat opcji kompilatora i wartości, które obsługują, zobacz [Visual Basic kompilatora opcje wymienione alfabetycznie](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **Identyfikator błędu:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Aby rozwiązać ten problem, a następnie użyć opcji  
  
-   Określ opcję kompilatora bez wartości.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Aby rozwiązać ten problem i nie używać opcji  
  
-   Usuń opcję kompilatora.
