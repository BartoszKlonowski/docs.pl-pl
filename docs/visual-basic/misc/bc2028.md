# <a name="option-option-cannot-be-followed-by--or---ignored"></a>Opcja '\<opcja >' nie może następować'+'lub'-'; zignorowano
Określono opcję kompilatora z wartością `+` lub `-`; te wartości nie są obsługiwane.  
  
 Aby uzyskać więcej informacji na temat opcji kompilatora i wartości, które obsługują zobacz [Visual Basic kompilatora opcji na liście alfabetycznie](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **Identyfikator błędu:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Aby rozwiązać ten problem i użyj opcji  
  
- Należy określić opcję kompilatora, bez wartości.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Aby rozwiązać ten problem i nie używa opcji  
  
- Remove — opcja kompilatora.
