# <a name="option-option-cannot-be-followed-by--or---ignored"></a>Po opcji " \<option> " nie może następować znak "+" lub "-"; zignorowano
Dla opcji kompilatora określono wartość `+` lub `-` ; te wartości nie są obsługiwane.  
  
 Aby uzyskać więcej informacji na temat opcji kompilatora i obsługiwanych przez nie wartości, zobacz [Opcje kompilatora Visual Basic w porządku alfabetycznym](../reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **Identyfikator błędu:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Aby naprawić ten błąd i użyć opcji  
  
- Określ opcję kompilatora bez wartości.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Aby naprawić ten błąd i nie używać opcji  
  
- Usuń opcję kompilatora.
