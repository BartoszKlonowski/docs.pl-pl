# <a name="statement-cannot-appear-within-a-method-body"></a>Instrukcja nie może występować w treści metody

Procedura zawiera nieprawidłową instrukcję, taką jak inna deklaracja procedury. Deklaracje procedur nie mogą być zagnieżdżane.  
  
 **Identyfikator błędu:** BC30289  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń deklarację procedury lub przenieś ją poza procedurę otaczającą.
