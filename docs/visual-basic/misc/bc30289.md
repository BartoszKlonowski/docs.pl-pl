# <a name="statement-cannot-appear-within-a-method-body"></a>Instrukcja nie może występować w treści metody
Procedura zawiera nieprawidłowy instrukcji, takie jak inna deklaracja procedury. Nie można zagnieżdżać deklaracje procedur.  
  
 **Identyfikator błędu:** BC30289  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń deklarację procedury lub przenieść je poza otaczającej procedury.
