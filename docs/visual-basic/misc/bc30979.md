# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>"Option Infer" może następować tylko "On" lub "Off"
`Option Infer` należy określić instrukcji `On` lub `Off`. Inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30979  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz opcję `Option Infer On` lub `Option Infer Off`.
