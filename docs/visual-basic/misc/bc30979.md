# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>Po "wnioskach" może występować tylko znak "on" lub "off"

`Option Infer` instrukcje muszą określać `On` lub `Off` . Inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30979  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ albo `Option Infer On` `Option Infer Off` .
