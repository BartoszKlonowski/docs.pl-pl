# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; opcja Infer &#39; może występować tylko znak &#39; W &#39; i &#39; Wyłącz &#39;
`Option Infer`instrukcje należy określić albo `On` lub `Off`. Inne wartości nie są prawidłowe.  
  
 **Identyfikator błędu:** BC30979  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `Option Infer On` lub `Option Infer Off`.
