# <a name="enums-must-be-declared-as-an-integral-type"></a>Wyliczenia musi być zadeklarowany jako typ całkowity
Tylko prawidłowe typy, można użyć w wyliczenia to `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, i `ULong`. Można używać nie typów danych.  
  
 **Identyfikator błędu:** BC30650  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ typ danych `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, lub `ULong`.
