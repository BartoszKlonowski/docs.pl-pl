# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Nie można obliczyć liczbę okresów przy użyciu podanych argumentów
Wywołanie `NPer` funkcja nie zawiera wszystkich wymaganych argumentów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że `Rate`, `Prnt` i `PV` wartości znajdują się w wywołaniu funkcji.
