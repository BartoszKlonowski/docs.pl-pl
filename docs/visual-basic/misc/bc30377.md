Można zadeklarować metody interfejsu po raz drugi, jego duplikowania.  
  
 **Identyfikator błędu:** BC30377  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zduplikowane deklarację.
