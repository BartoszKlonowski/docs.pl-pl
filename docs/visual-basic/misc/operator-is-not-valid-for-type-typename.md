# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>Operator jest nieprawidłowy dla typu "&lt;typename&gt;"
Operator nie jest odpowiedni dla typów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że operator jest odpowiedni dla typów, w której są wykonywane działania na.
