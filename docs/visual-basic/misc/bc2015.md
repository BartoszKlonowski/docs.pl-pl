# <a name="the-file-filename-is-not-a-text-file"></a>Plik " \<filename> " nie jest plikiem tekstowym

Nie można zapisać określonego pliku, ponieważ jest to plik binarny.  
  
 **Identyfikator błędu:** BC2015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy podano poprawną nazwę pliku i ścieżkę.
