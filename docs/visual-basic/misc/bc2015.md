# <a name="the-file-ltfilenamegt-is-not-a-text-file"></a>Plik "&lt;filename&gt;" nie jest plikiem tekstowym
Nie można zapisać określonego pliku do, ponieważ jest plikiem binarnym.  
  
 **Identyfikator błędu:** BC2015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy podając poprawną nazwę pliku i ścieżkę.
