# <a name="arguments-are-not-valid"></a>Argumenty są nieprawidłowe
Argumenty, które nie są dozwolone jest przekazywany do funkcji lub procedury.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy argumenty do ustalenia, dlaczego są nieprawidłowe.  
  
2.  Jeśli argumenty są dostarczane przez obliczeń, sprawdź wartości, które przetwarzają obliczeń.
