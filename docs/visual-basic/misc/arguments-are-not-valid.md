# <a name="arguments-are-not-valid"></a>Argumenty są nieprawidłowe
Nieprawidłowe argumenty są przekazywane do funkcji lub procedury podrzędnej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź argumenty, aby określić, dlaczego są one nieprawidłowe.  
  
2. Jeśli argumenty są dostarczane przez obliczenia, sprawdź wartości, które proces obliczeń.
