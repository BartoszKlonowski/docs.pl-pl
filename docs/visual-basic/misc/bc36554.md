## <a name="to-correct-this-error"></a><span data-ttu-id="399cb-111">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="399cb-111">To correct this error</span></span>

- <span data-ttu-id="399cb-112">Jeśli tablica parametrów nie określa typu danych, który ma zostać rozbudowany, Dodaj nowy pierwszy parametr, który określa ten typ.</span><span class="sxs-lookup"><span data-stu-id="399cb-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb
  <Extension()>
  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="399cb-113">Jeśli tablica parametrów określa typ danych, który chcesz zwiększyć, Rozważ zmianę jej na tablicę regularną, wymagając argumentu zamiast tablicy parametrów.</span><span class="sxs-lookup"><span data-stu-id="399cb-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="399cb-114">Regularne tablice można rozszerzyć.</span><span class="sxs-lookup"><span data-stu-id="399cb-114">Regular arrays can be extended.</span></span>

  ```vb
  <Extension()>
  Public Function Sum(ByVal ints() As Integer) As Integer
      Dim total As Integer = 0
      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
