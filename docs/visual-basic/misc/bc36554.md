 <span data-ttu-id="4ada9-105">Pierwszy parametr metody rozszerzenia Określa typ danych, która metoda jest rozszerzeniem.</span><span class="sxs-lookup"><span data-stu-id="4ada9-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="4ada9-106">Dlatego pierwszy parametr jest wymagany i nie może być opcjonalny.</span><span class="sxs-lookup"><span data-stu-id="4ada9-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="4ada9-107">Ponieważ tablicy parametrów jest automatycznie opcjonalne, nie jest prawidłowy jako pierwszego argumentu metody rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="4ada9-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>

 <span data-ttu-id="4ada9-110">**Identyfikator błędu:** BC36554</span><span class="sxs-lookup"><span data-stu-id="4ada9-110">**Error ID:** BC36554</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4ada9-111">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4ada9-111">To correct this error</span></span>  
  
-   <span data-ttu-id="4ada9-112">Jeśli tablica parametrów nie określa typ danych, który ma zostać rozszerzony, Dodaj nowe pierwszy parametr określający tego typu.</span><span class="sxs-lookup"><span data-stu-id="4ada9-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="4ada9-113">Jeśli tablica parametru określony typ danych, który ma zostać rozszerzony, należy rozważyć zmianę go z tablicą regularne wymagające argumentu, zamiast tablicy parametrów.</span><span class="sxs-lookup"><span data-stu-id="4ada9-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="4ada9-114">Regularne tablice mogą być rozszerzane.</span><span class="sxs-lookup"><span data-stu-id="4ada9-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
