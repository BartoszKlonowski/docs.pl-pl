## <a name="to-correct-this-error"></a><span data-ttu-id="b8e61-111">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="b8e61-111">To correct this error</span></span>  
  
-   <span data-ttu-id="b8e61-112">Jeśli tablica parametrów nie określa typ danych, które chcesz rozszerzyć, dodać nowe pierwszy parametr, który określa ten typ.</span><span class="sxs-lookup"><span data-stu-id="b8e61-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="b8e61-113">Jeśli tablica parametrów określony typ danych, które chcesz rozszerzyć, należy rozważyć zmianę jej regularne tablicy wymaga argumentu, zamiast tablicy parametrów.</span><span class="sxs-lookup"><span data-stu-id="b8e61-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="b8e61-114">Można rozszerzyć regularnych tablic.</span><span class="sxs-lookup"><span data-stu-id="b8e61-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
