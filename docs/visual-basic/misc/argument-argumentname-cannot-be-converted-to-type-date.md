# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>\<argumentname>Nie można przekonwertować argumentu "" na typ "date"

Podczas konwersji podjęto próbę zmiany zmiennej na typ `Date` , ale wartość nie jest rozpoznawana jako prawidłowa data.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Znajdź konwersję mającą znaczenie.
