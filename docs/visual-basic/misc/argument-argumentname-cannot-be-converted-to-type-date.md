# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-date"></a>Argument "&lt;argumentname&gt;" nie można przekonwertować na typ, "Date"
Konwersja podjęła próbę zmiany na zmienną typu `Date`, ale wartość nie został rozpoznany jako prawidłowa data.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Znajdź konwersji, która ma znaczenie.
