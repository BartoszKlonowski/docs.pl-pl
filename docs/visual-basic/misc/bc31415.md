# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>element " \<implementsclause> " nie może implementować elementu " \<typename> ", ponieważ element " \<typename> " jest nazwą zastrzeżoną
`Implements`Instrukcja określiła nazwę zastrzeżoną.  
  
 **Identyfikator błędu:** BC31415  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę elementu członkowskiego interfejsu, który nie mógł zostać zaimplementowany.
