# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>'\<implementsclause >' nie może implementować'\<typename > 'ponieważ'\<typename > 'jest zastrzeżoną nazwą
`Implements` Określona żadna instrukcja nazwą zastrzeżoną.  
  
 **Identyfikator błędu:** BC31415  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę składowej interfejsu, który nie jest zaimplementowana.
