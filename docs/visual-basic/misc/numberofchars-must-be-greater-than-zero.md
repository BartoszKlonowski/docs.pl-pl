# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars musi być większy od zera

Korzystając z `PeekChars` metody `TextFieldParser` obiektu, należy podać `NumberOfChars` wartość większą niż `0` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień `NumberOfChars` na wartość, która jest większa niż `0` .
