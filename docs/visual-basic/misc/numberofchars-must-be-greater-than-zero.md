# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars musi być większa niż zero
Korzystając z `PeekChars` metody `TextFieldParser` obiektu, należy podać `NumberOfChars` wartość, która jest większa niż `0`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana `NumberOfChars` wartość, która jest większa niż `0`.
