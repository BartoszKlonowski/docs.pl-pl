# <a name="numberofchars-must-be-greater-than-zero"></a>Parametr NumberOfChars musi być większa od zera.
Korzystając z `PeekChars` metody `TextFieldParser` obiektu, należy podać `NumberOfChars` wartość, która jest większa niż `0`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień `NumberOfChars` jest większa niż wartość `0`.
