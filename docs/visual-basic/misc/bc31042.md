# <a name="sub-new-cannot-implement-interface-members"></a>"Sub New" nie może implementować składowych interfejsu
`Sub New` jest konstruktorem, i nie może implementować, elementy członkowskie.  
  
 **Identyfikator błędu:** BC31042  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Implements` instrukcji `Sub New` procedur.
