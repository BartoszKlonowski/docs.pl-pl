# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39; Sub nowy &#39; nie może implementować członków interfejsu
`Sub New`jest konstruktorem i nie może implementować członków.  
  
 **Identyfikator błędu:** BC31042  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Implements` oświadczenia `Sub New` procedury.
