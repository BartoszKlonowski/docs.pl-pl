# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>VbStrConv. Wide i VbStrConv. Narrow nie mają zastosowania do określonych ustawień regionalnych
Aplikacja próbuje użyć elementów członkowskich wyliczenia `VbStrConv` `Wide` lub `Narrow`, które nie mają zastosowania do określonych ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `VbStrConv.Wide` lub `VbStrConv.Narrow`.
