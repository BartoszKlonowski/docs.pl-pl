# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>Argumenty VbStrConv.Wide a VbStrConv.Narrow nie mają zastosowania do ustawień regionalnych określonych
Aplikacja próbuje użyć `VbStrConv` elementów członkowskich wyliczenia `Wide` lub `Narrow`, które nie mają zastosowania do określonych ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Albo usuń `VbStrConv.Wide` lub `VbStrConv.Narrow`.
