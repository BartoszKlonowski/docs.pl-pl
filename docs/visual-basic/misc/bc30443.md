# <a name="get-is-already-declared"></a>Element "Get" jest już zadeklarowany
`Get`Instrukcja pojawia się więcej niż raz w bloku kodu. `Get`Instrukcja deklaruje `Get` procedurę właściwości używaną do pobrania wartości właściwości.  
  
 **Identyfikator błędu:** BC30443  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `Get` .
