# <a name="39get39-is-already-declared"></a>&#39;Pobierz&#39; jest już zadeklarowana
A `Get` instrukcji występuje więcej niż raz w bloku kodu. `Get` Deklaruje instrukcji `Get` procedury właściwości używane do pobierania wartości właściwości.  
  
 **Identyfikator błędu:** BC30443  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `Get` instrukcji.
