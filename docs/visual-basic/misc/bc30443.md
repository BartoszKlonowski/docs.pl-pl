# <a name="get-is-already-declared"></a>"Get" jest już zadeklarowana
A `Get` instrukcji występuje więcej niż raz w bloku kodu. `Get` Instrukcja deklaruje `Get` procedury właściwości używane do pobierania wartości właściwości.  
  
 **Identyfikator błędu:** BC30443  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędną `Get` instrukcji.
