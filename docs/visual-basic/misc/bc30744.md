# <a name="referenced-array-element-has-a-value-of-nothing"></a>Element tablicy do którego istnieje odwołanie, ma wartość "Nothing"
Nie ustawiono elementu tablicy, do którego nastąpiło odwołanie, lub było wcześniej ustawione na `Nothing`.  
  
 **Identyfikator błędu:** BC30744  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy wartości w tablicy są są prawidłowo ustawione.
