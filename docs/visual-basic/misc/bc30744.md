# <a name="referenced-array-element-has-a-value-of-nothing"></a>Element tablicy, do którego istnieje odwołanie, ma wartość "Nothing".
Element tablicy, do którego się odwoływano, nie został ustawiony lub poprzednio został ustawiony na `Nothing` .  
  
 **Identyfikator błędu:** BC30744  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy wartości w tablicy są ustawiane poprawnie.
