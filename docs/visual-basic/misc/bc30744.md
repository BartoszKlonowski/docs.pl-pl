# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>Tablica przywoływany element ma wartość &#39;nic&#39;
Nie ustawiono elementu tablicy, do którego nastąpiło odwołanie, lub było wcześniej ustawione na `Nothing`.  
  
 **Identyfikator błędu:** BC30744  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy wartości w tablicy są są poprawnie ustawione.
