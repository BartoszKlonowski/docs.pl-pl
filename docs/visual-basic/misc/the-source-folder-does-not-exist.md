# <a name="the-source-folder-does-not-exist"></a>Folder źródłowy nie istnieje

Podjęto próbę wykonania operacji odwołującej się do folderu źródłowego, który nie istnieje. Może to być spowodowane podawaniem nieprawidłowej ścieżki pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy używasz odpowiedniej ścieżki, szczególnie jeśli używasz ścieżek względnych.
