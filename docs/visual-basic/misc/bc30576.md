# <a name="statement-does-not-declare-a-39get39-or-39set39-method"></a>Instrukcja nie deklaruje &#39; Pobierz &#39; i &#39; Ustaw &#39; — Metoda
Instrukcji nie powiodło się dostarcza `Get` lub `Set` instrukcji deklaracji wokół `Property` procedury. Właściwość jest zdefiniowana jako blok kodu ujętą w nawiasy klamrowe `Property` i `End Property` instrukcje. W tym bloku każdego `Property` procedury pojawia się jako wewnętrzne blok ujętą w nawiasy klamrowe instrukcji deklaracji i instrukcji end.  
  
 **Identyfikator błędu:** BC30576  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj `Get` lub `Set` instrukcji deklaracji.
