# <a name="--expected"></a>Oczekiwano znaku "-"

 **Identyfikator błędu:** BC30601  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź składnię kodu, który spowodował błąd.
