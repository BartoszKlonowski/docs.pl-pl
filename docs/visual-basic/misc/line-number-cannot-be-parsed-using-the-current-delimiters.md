# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>\<number>Nie można przeanalizować wiersza przy użyciu bieżących ograniczników

Nie można przeanalizować określonego wiersza, ponieważ używa on ograniczników innych niż określone.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dostosuj `Delimiters` , aby wiersz można było analizować prawidłowo, lub wstawić kod obsługi wyjątków w celu obsługi wiersza.
