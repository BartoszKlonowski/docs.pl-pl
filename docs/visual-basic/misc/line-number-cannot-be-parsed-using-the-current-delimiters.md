# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>Wiersz \<numer > nie można przeanalizować za pomocą bieżącego ograniczników
Nie można przeanalizować określonego wiersza, ponieważ używa ona ograniczniki innych niż określone.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dostosuj `Delimiters` co wiersz może zostać poprawnie zanalizowana, albo Wstaw kod obsługi wyjątków, aby umożliwić obsługę wiersza.
