# <a name="lterrorgt-ltstructurename1gt-contains-ltstructurename2gt"></a>&lt;Błąd&gt;: "&lt;structurename1&gt;"zawiera"&lt;structurename2&gt;"
Definicja struktury cykliczne występuje, gdy struktury wyznacza jako jeden z jego własnej członków.  
  
 **Identyfikator błędu:** BC30293  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę elementu członkowskiego struktury.
