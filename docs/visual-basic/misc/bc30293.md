# <a name="error-structurename1-contains-structurename2"></a>\<Błąd >: '\<structurename1 >' zawiera '\<structurename2 >'
Definicja struktury cykliczne występuje, gdy struktury wyznacza jako jeden z jego własnej członków.  
  
 **Identyfikator błędu:** BC30293  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę elementu członkowskiego struktury.
