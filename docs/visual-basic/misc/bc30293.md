# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;Błąd&gt;: &#39; &lt;structurename1&gt; &#39; zawiera &#39; &lt;structurename2&gt;&#39;
Definicja cykliczna struktury występuje, gdy struktury wyznacza jako jeden z własnych elementów członkowskich.  
  
 **Identyfikator błędu:** BC30293  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę elementu członkowskiego struktury.
