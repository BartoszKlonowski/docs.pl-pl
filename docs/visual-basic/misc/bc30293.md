# <a name="error-structurename1-contains-structurename2"></a>\<error>: element " \<structurename1> " zawiera element \<structurename2> ""
Definicja struktury cyklicznej występuje, gdy struktura wyznacza siebie jako jeden z jej elementów członkowskich.  
  
 **Identyfikator błędu:** BC30293  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę elementu członkowskiego struktury.
