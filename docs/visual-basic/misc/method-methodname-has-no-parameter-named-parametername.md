# <a name="method-methodname-has-no-parameter-named-parametername"></a>Metoda " \<methodname> " nie ma parametru o nazwie " \<parametername> "

Podany nazwany argument nie jest zgodny z żadnym z argumentów metody docelowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że nazwany argument został podany poprawnie.
