# <a name="method-ltmethodnamegt-has-no-parameter-named-ltparameternamegt"></a>Metoda "&lt;methodname&gt;"nie ma parametru o nazwie"&lt;parametername&gt;"
Podany argument nazwany nie pasuje do żadnego z argumentów metody docelowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że argument nazwany został podany poprawnie.
