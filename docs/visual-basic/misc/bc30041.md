# <a name="maximum-number-of-errors-has-been-exceeded"></a>Przekroczono maksymalną liczbę błędów
Maksymalna liczba błędów kompilacji to 100. Kompilator nakłada ten limit, aby ograniczyć liczbę raportów o błędach.  
  
 **Identyfikator błędu:** BC30041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Inne błędy generowane przez tej kompilacji i wyjaśnij, jak najwięcej. Jeśli kod źródłowy zawiera więcej niż 100 błędy, mogą występować błędy więcej w następnej kompilacji.
