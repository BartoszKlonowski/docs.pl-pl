# <a name="maximum-number-of-errors-has-been-exceeded"></a>Przekroczono maksymalną liczbę błędów
Maksymalna liczba błędów kompilacji wynosi 100. Kompilator nakłada limit w ten sposób, aby zmniejszyć rozmiar raporty o błędach.  
  
 **Identyfikator błędu:** BC30041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Popraw błędy, generowane przez tej kompilacji i rozwiązać jak najwięcej. Jeśli kod źródłowy zawiera więcej niż 100 błędy, mogą występować błędy w następnej kompilacji.
