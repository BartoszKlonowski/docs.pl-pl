# <a name="maximum-number-of-errors-has-been-exceeded"></a>Przekroczono maksymalną liczbę błędów

Maksymalna liczba błędów kompilacji to 100. Kompilator nakłada ten limit, aby zmniejszyć rozmiar raportów o błędach.  
  
 **Identyfikator błędu:** BC30041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przejrzyj inne błędy wygenerowane przez tę kompilację i Rozwiąż tyle, ile to możliwe. Jeśli kod źródłowy zawiera więcej niż 100 błędów, może pojawić się więcej błędów w następnej kompilacji.
