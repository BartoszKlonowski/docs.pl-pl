# <a name="automation-object-does-not-have-a-default-value"></a>Obiekt automatyzacji nie ma wartości domyślnej

Visual Basic nie może określić domyślnego elementu członkowskiego dla określonego obiektu.  
  
### <a name="to-correct-the-error"></a>Aby poprawić błąd  
  
- Sprawdź dokumentację obiektu i podaj jawną specyfikację właściwości lub metody.
