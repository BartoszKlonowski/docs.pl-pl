# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>"\<membername1 >" i "\<membername2 >" nie mogą przeciążać siebie nawzajem, ponieważ różnią się tylko typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jeden z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30696  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jako innego typu danych, należy zdefiniować co najmniej jeden parametr wymagany.
