# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>metody " \<membername1> " i " \<membername2> " nie mogą przeciążać siebie nawzajem, ponieważ różnią się jedynie typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jednego wymaganego parametru.  
  
 **Identyfikator błędu:** BC30696  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj co najmniej jeden wymagany parametr jako inny typ danych.
