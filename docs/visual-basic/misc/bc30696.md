# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39; &lt;membername1&gt;&#39; i &#39;&lt; membername2&gt;&#39; nie mogą przeciążać siebie nawzajem, ponieważ różnią się jedynie typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jednego z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30696  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy zdefiniować co najmniej jeden wymagany parametr jako inny typ danych.
