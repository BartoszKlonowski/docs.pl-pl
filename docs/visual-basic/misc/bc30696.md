# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;"i"&lt;membername2&gt;" nie mogą przeciążać siebie nawzajem, ponieważ różnią się tylko typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jeden z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30696  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jako innego typu danych, należy zdefiniować co najmniej jeden parametr wymagany.
