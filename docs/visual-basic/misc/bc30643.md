# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>Właściwość &#39; &lt;propertyname&gt; &#39; ma nieobsługiwany typ.
Właściwość w przywoływanym zestawie zawiera typ, który nie może być przedstawiony.  
  
 **Identyfikator błędu:** BC30643  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ typ, który może być używany z aplikacji.
