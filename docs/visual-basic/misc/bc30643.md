# <a name="property-propertyname-is-of-an-unsupported-type"></a>Właściwość " \<propertyname> " ma nieobsługiwany typ
Właściwość w zestawie, do którego istnieje odwołanie, zawiera typ, który nie może być reprezentowany.  
  
 **Identyfikator błędu:** BC30643  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ typ, który może być używany w aplikacji.
