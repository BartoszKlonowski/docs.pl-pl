# <a name="property-ltpropertynamegt-is-of-an-unsupported-type"></a>Właściwość "&lt;propertyname&gt;" ma nieobsługiwany typ.
Właściwość w przywoływanym zestawie zawiera typ, który nie może być reprezentowana.  
  
 **Identyfikator błędu:** BC30643  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ typ, który może być używany z aplikacji.
