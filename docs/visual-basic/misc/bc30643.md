# <a name="property-propertyname-is-of-an-unsupported-type"></a>Właściwość '\<propertyname >' jest nieobsługiwanego typu
Właściwość w przywoływanym zestawie zawiera typ, który nie może być reprezentowana.  
  
 **Identyfikator błędu:** BC30643  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ typ, który może być używany z aplikacji.
