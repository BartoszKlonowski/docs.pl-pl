# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>Instrukcja "namespace" musi być zakończona odpowiadającą jej instrukcją "End namespace"
`Namespace`bloki muszą kończyć się `End Namespace` instrukcją.  
  
 **Identyfikator błędu:** BC30626  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `End Namespace` instrukcję do końca `Namespace` bloku.
