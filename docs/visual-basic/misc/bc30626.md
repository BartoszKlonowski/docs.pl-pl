# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39; instrukcja musi być zakończona odpowiadającą jej instrukcją &#39;End Namespace&#39;
`Namespace` bloki musi kończyć się `End Namespace` instrukcji.  
  
 **Identyfikator błędu:** BC30626  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Namespace` instrukcji na końcu `Namespace` bloku.
