# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>Instrukcja "Namespace" musi kończyć się pasującego "End Namespace"
`Namespace` musi kończyć się bloki `End Namespace` instrukcji.  
  
 **Identyfikator błędu:** BC30626  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Namespace` instrukcji na końcu `Namespace` bloku.
