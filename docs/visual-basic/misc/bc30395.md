# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39;&lt;Specyfikator&gt; &#39; nie jest prawidłowy w deklaracji struktury
Użyto specyfikatora, takich jak `Dim` lub `Override`, która jest nieprawidłowa na `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30395  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
