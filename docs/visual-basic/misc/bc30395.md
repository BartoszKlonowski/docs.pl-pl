# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>element " \<specifier> " nie jest prawidłowy w deklaracji struktury
Użyto specyfikatora, takiego jak `Dim` lub `Override` , który nie jest prawidłowy w `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30395  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
