# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>'\<specyfikator >' nie jest prawidłowy w deklaracji struktury
Wykorzystano specyfikatora, takich jak `Dim` lub `Override`, to znaczy nie jest prawidłowa w `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30395  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
