# <a name="lttypenamegt-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>"&lt;typename&gt;" musi być zadeklarowany jako "MustInherit" ponieważ zawiera metody zadeklarowane jako "MustOverride"
Typy, które zawierają `MustOverride` musi być zadeklarowany jako elementy członkowskie `MustInherit`.  
  
 **Identyfikator błędu:** BC31411  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `MustInherit` modyfikatora typu.
