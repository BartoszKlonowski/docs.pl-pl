# <a name="construct-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-typename"></a>Konstrukcja wykonuje pośrednie odwołanie do zestawu " \<assemblyname> ", który zawiera element " \<typename> "

Konstrukcja wykonuje pośrednie odwołanie do zestawu " \<assemblyname> ", który zawiera \<typename> . Dodaj odwołanie do pliku do \<filename> projektu.  
  
 Wyrażenie używa typu, takiego jak Klasa, struktura, interfejs, Wyliczenie lub delegat, ale zestaw nie ma odwołania do projektu do zestawu, który definiuje typ.  
  
 **Identyfikator błędu:** BC31528  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- We właściwościach projektu Dodaj odwołanie do pliku zawierającego zestaw, który definiuje używany typ.
