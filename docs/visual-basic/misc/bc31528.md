# <a name="construct-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-lttypenamegt"></a>Konstrukcja wykonuje pośrednie odwołanie do zestawu "&lt;assemblyname&gt;", który zawiera"&lt;typename&gt;"
Konstrukcja wykonuje pośrednie odwołanie do zestawu "\<nazwa_zestawu >", który zawiera \<typename >. Dodaj odwołanie do pliku \<nazwa pliku > do projektu.  
  
 Wyrażenie korzysta z typu, takich jak klasy, struktury, interfejsu, wyliczenie lub delegata, ale zestaw nie ma odwołanie do zestawu, który definiuje typ.  
  
 **Identyfikator błędu:** BC31528  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   We właściwościach projektu Dodaj odwołanie do pliku, który zawiera zestaw, który definiuje typ, którego używasz.
