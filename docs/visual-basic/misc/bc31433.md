# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>Metoda "&lt;methodname&gt;" nie można zadeklarować jako "Partial" ponieważ tylko jedna metoda "&lt;methodname&gt;" mogą zostać oznaczone jako "Partial"
Nie może mieć dwie deklaracje metody częściowej, które mają taką samą nazwę i podpis.  
  
 **Identyfikator błędu:** BC31433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmiana nazwy lub podpis deklaracje metody częściowej.
