# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Metody " \<methodname> " nie można zadeklarować jako "częściowe", ponieważ tylko jedna metoda " \<methodname> " może być oznaczona jako "częściowa"

Nie można mieć dwóch deklaracji metody częściowej, które mają taką samą nazwę i podpis.  
  
 **Identyfikator błędu:** BC31433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień nazwę lub podpis jednej ze deklaracji metody częściowej.
