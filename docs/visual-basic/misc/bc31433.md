# <a name="method-39ltmethodnamegt39-cannot-be-declared-39partial39-because-only-one-method-39ltmethodnamegt39-can-be-marked-39partial39"></a>Metody &#39; &lt;methodname&gt; &#39; nie można zadeklarować &#39;częściowe&#39; ponieważ tylko jedna metoda &#39; &lt;methodname&gt; &#39; może być oznaczony &#39;częściowe&#39;
Nie może mieć dwóch deklaracje metody częściowej, które mają taką samą nazwę i sygnaturę.  
  
 **Identyfikator błędu:** BC31433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmiana nazwy lub podpisu deklaracje metody częściowej.
