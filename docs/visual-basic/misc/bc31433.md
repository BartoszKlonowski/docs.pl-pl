# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Metoda '\<methodname >' nie może być zadeklarowany jako'Partial'ponieważ tylko jedna metoda'\<methodname > 'mogą zostać oznaczone jako'Partial'
Nie może mieć dwie deklaracje metody częściowej, które mają taką samą nazwę i podpis.  
  
 **Identyfikator błędu:** BC31433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmiana nazwy lub podpis deklaracje metody częściowej.
