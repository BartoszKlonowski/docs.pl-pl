# <a name="membername-is-not-declared"></a>element " \<membername> " nie jest zadeklarowany

element " `<membername>` " nie jest zadeklarowany. `Debug` Funkcja obiektu jest dostępna w `System.Diagnostics.Debug` `System` zestawie.  
  
 Nie można odnaleźć określonej nazwy elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30816  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię elementu członkowskiego.  
  
2. Użyj `Imports` instrukcji lub w pełni kwalifikujących się elementów członkowskich zdefiniowanych w innych przestrzeniach nazw. Na przykład `WriteLine` Funkcja jest zadeklarowana w `System.Diagnostics.Debug` przestrzeni nazw.
