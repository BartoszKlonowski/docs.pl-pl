# <a name="membername-is-not-declared"></a>nie zadeklarowano elementu "\<MemberName >"
element "`<membername>`" nie jest zadeklarowany. Funkcja obiektu `Debug` jest dostępna w `System.Diagnostics.Debug` zestawu `System`.  
  
 Nie można odnaleźć określonej nazwy elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30816  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię elementu członkowskiego.  
  
2. Użyj instrukcji `Imports` lub w pełni kwalifikujących się elementów członkowskich zdefiniowanych w innych przestrzeniach nazw. Na przykład funkcja `WriteLine` jest zadeklarowana w przestrzeni nazw `System.Diagnostics.Debug`.
