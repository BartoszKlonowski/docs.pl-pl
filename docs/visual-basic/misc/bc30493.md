# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>Konstruktor musi być zadeklarowany jako procedura, a nie jako funkcja
Podjęto próbę zadeklarować `Function New`. Konstruktory musi być zadeklarowany jako `Sub New`.  
  
 **Identyfikator błędu:** BC30493  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Sub` zamiast `Function`.
