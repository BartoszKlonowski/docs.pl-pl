# <a name="object-doesnt-support-this-action-visual-basic"></a>Obiekt nie obsługuje tej akcji (Visual Basic)

Odwołuje się do metody lub właściwości, która nie jest obsługiwana przez ten obiekt.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją obiektu, aby uzyskać więcej informacji na temat obiektu.  
  
2. Upewnij się, że nazwy właściwości i metod są poprawnie napisane.
