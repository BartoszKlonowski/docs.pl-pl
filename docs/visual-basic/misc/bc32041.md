# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>Nie można użyć znaku typu w deklaracji parametru typu
Deklaracja parametru typu zawiera co najmniej jeden znak typu identyfikator.  
  
 Parametr typu ogólnego typu musi być prawidłową nazwą języka Visual Basic. Dozwolone znaki nie zawierać znaki typu identyfikator (`%`, `&`, `@`, `!`, `#`, i `$`). Zobacz [Zadeklarowane nazwy elementów](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identyfikator błędu:** BC32041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń typ identyfikatora znak lub znaki z deklaracji parametru typu.
