# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>Nie można użyć znaku typu w deklaracji parametru typu

Deklaracja parametru typu zawiera co najmniej jeden znak typu identyfikator.  
  
 Parametr typu generycznego musi być prawidłową nazwą Visual Basic. Dozwolone znaki nie zawierają żadnego ze znaków typu identyfikatora ( `%` ,,,, `&` `@` `!` `#` i `$` ). Zobacz [zadeklarowane nazwy elementów](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identyfikator błędu:** BC32041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń znak identyfikatora typu lub znaki z deklaracji parametru typu.
