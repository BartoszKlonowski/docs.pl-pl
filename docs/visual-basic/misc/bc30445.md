# <a name="const-declaration-cannot-have-an-array-initializer"></a>Deklaracja const nie może mieć inicjatora tablicy

Podjęto próbę użycia inicjatora tablicy w `Const` deklaracji.  
  
 **Identyfikator błędu:** BC30445  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień inicjator na prawidłowe wyrażenie, usuwając otaczające nawiasy klamrowe.
