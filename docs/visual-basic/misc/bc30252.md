# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>Element "End Interface" musi być poprzedzony odpowiadającym mu elementem "Interface"
`End Interface`Instrukcja występuje bez odpowiedniej `Interface` instrukcji. `End Interface`musi być poprzedzona odpowiadającą jej `Interface` instrukcją.  
  
 **Identyfikator błędu:** BC30252  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy Poprzednia `Interface` instrukcja jest błędnie wpisana lub nieprawidłowa.  
  
2. Upewnij się, że `Interface` elementy członkowskie są poprawnie sformatowane.
