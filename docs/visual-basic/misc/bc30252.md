# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>"End Interface" musi być poprzedzona instrukcją "Interface"
`End Interface` Instrukcji odbywa się bez odpowiadającego `Interface` instrukcji. `End Interface` musi być poprzedzona odpowiadającą `Interface` instrukcji.  
  
 **Identyfikator błędu:** BC30252  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zaznacz, aby sprawdzić, czy występuje przed `Interface` instrukcja jest błędnie lub są one nieprawidłowe.  
  
2. Upewnij się, że `Interface` elementy członkowskie są poprawnie sformatowane.
