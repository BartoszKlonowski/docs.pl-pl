# <a name="structure-statement-must-end-with-a-matching-end-structure"></a>Instrukcja "Structure" musi być zakończona odpowiadającą jej instrukcją "End Structure"

Struktury zaczynają się od `Structure` instrukcji i kończą się na `End Structure` konstrukcję.  
  
 **Identyfikator błędu:** BC30624  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że blok struktury jest zakończony `End Structure` instrukcją.
