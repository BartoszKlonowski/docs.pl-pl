# <a name="structure-statement-must-end-with-a-matching-end-structure"></a>Instrukcja "Structure" musi być zakończona z instrukcją "End Structure"
Struktury zaczynają się od `Structure` instrukcji i kończyć się `End Structure` konstruowania.  
  
 **Identyfikator błędu:** BC30624  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że blok struktury kończy się `End Structure` instrukcji.
