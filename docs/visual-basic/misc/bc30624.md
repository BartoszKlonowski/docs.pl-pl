# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39; Struktura &#39; Instrukcja musi kończyć się pasujący &#39; End Structure &#39;
Struktury rozpoczynać się od `Structure` instrukcji i kończyć się `End Structure` utworzenia.  
  
 **Identyfikator błędu:** BC30624  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że blok struktury kończy się `End Structure` instrukcji.
