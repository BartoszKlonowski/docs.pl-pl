# <a name="end-region-must-be-preceded-by-a-matching-region"></a>"#End Region" musi być poprzedzona instrukcją "#Region"
Za pomocą `#Region` można określić bloku kodu, aby rozwinąć lub zwinąć podczas korzystania z funkcji konspektu Edytor kodu programu Visual Studio. Początek i koniec `#Region` instrukcji musi znajdować się w tym samym bloku kodu.  
  
 **Identyfikator błędu:** BC30680  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wstaw `#Region` w odpowiednim miejscu przed odpowiednimi `#End Region` instrukcji.
