# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>& #39; #End region & #39; musi być poprzedzona odpowiadającą jej instrukcją & #39; #Region & #39;
Z `#Region` można określić bloku kodu, aby rozwinąć lub zwinąć, korzystając z funkcji zwijania [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] edytora kodu. Początek i koniec `#Region` instrukcje musi być w tym samym bloku kodu.  
  
 **Identyfikator błędu:** BC30680  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wstaw `#Region` w odpowiednim miejscu przed odpowiadającego `#End``Region` instrukcji.
