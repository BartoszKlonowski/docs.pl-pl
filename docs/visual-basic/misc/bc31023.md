# <a name="unable-to-parse-xml-error"></a>Nie można przeanalizować kodu XML: \<błąd >
Istnieje błąd w pliku XML, używany.  
  
 **Identyfikator błędu:** BC31023  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy kod XML jest dobrze sformułowany i prawidłowy i że plik nie jest uszkodzony.
