# <a name="unable-to-parse-xml-error"></a>Nie można przeanalizować kodu XML: \<error>

Wystąpił błąd w używanym pliku XML.  
  
 **Identyfikator błędu:** BC31023  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że kod XML jest poprawnie sformułowany, prawidłowy i że plik nie jest uszkodzony.
