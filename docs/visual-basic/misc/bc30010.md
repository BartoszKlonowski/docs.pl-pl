# <a name="reference-required-to-module-ltmodulenamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>Wymagane odwołanie do modułu "&lt;modulename&gt;"zawierający zaimplementowany interfejs"&lt;interfacename&gt;"
Wymagane odwołanie do modułu "\<nazwa_modułu >" zawierający zaimplementowany interfejs "\<interfacename >". Dodaj je do projektu.  
  
 Interfejs jest zdefiniowany w module, który nie jest bezpośrednio wywoływany w projekcie. Kompilator Visual Basic wymaga odwołania, aby uniknąć niejednoznaczności, w przypadku, gdy interfejs jest zdefiniowany w więcej niż jeden moduł.  
  
 **Identyfikator błędu:** BC30010  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   W odwołania projektu, należy uwzględnić nazwę bez odwołań modułu.
