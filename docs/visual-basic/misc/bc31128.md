# <a name="removehandler-is-already-declared"></a>Element "RemoveHandler" jest już zadeklarowany

`RemoveHandler`W deklaracji zdarzenia niestandardowego pojawił się więcej niż jedna deklaracja. `RemoveHandler`Deklaracja deklaruje procedurę służącą do usuwania programu obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31128  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `RemoveHandler` .
