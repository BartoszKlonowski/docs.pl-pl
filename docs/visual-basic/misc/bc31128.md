# <a name="39removehandler39-is-already-declared"></a>&#39;RemoveHandler&#39; jest już zadeklarowana
Więcej niż jeden `RemoveHandler` deklaracji znajdowały się w deklaracji zdarzenia niestandardowego. `RemoveHandler` Deklaracji deklaruje procedurę służy do usuwania programu obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31128  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `RemoveHandler` instrukcji.
