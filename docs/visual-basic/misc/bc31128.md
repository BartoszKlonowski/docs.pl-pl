# <a name="removehandler-is-already-declared"></a>"RemoveHandler" jest już zadeklarowany.
Więcej niż jeden `RemoveHandler` deklaracji pojawiły się w deklaracji zdarzenia niestandardowego. `RemoveHandler` Deklaracji deklaruje procedurę, aby usunąć program obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31128  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `RemoveHandler` instrukcji.
