# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Zmienna lokalna nie może mieć takiej samej nazwy jak funkcja zawierająca ją
`Dim`Instrukcja określa zmienną o tej samej nazwie, co `Function` procedura zawierająca ją.  
  
 **Identyfikator błędu:** BC30290  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń deklarację zmiennej lub Zmień nazwę zmiennej.
