A `Dim` instrukcji Określa zmienną z taką samą nazwę jak `Function` procedury zawierającego go.  
  
 **Identyfikator błędu:** BC30290  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklaracja zmiennej Usuń lub zmień nazwę zmiennej.
