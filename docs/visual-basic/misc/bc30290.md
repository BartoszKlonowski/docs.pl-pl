# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Zmienna lokalna nie może mieć taką samą nazwę jak funkcja zawierające je
A `Dim` instrukcja Określa zmienną, taką samą nazwę jak `Function` procedury zawierające je.  
  
 **Identyfikator błędu:** BC30290  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń deklarację zmiennej, lub zmień nazwę zmiennej.
