# <a name="argument-cannot-be-less-than-zero"></a>Argument nie może być mniejsza od zera
Wartość która jest mniejsza niż podano zero, ale argument nie może być mniejsza od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź sposób obliczania wartości w celu upewnij się, że jest poprawny.
