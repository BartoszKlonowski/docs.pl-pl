# <a name="unable-to-create-temp-file-in-path-filename-error-message"></a>Nie można utworzyć pliku tymczasowego w ścieżce " \<filename> ": \<error message>

Kompilator Visual Basic wywołuje konsolidator zestawu (Al.exe, znany również jako ALink) w celu wygenerowania zestawu z manifestem. Konsolidator zgłosił błąd podczas tworzenia pliku lub zapisywania do niego zasobu znajdującego się w pamięci. Może to być problem z konfiguracją.  
  
 **Identyfikator błędu:** BC30138  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź cytowany komunikat o błędzie i zapoznaj się z tematem  [Al.exe (Konsolidator zestawu)](../../framework/tools/al-exe-assembly-linker.md) , aby uzyskać dalsze wyjaśnienia i porady.  
  
2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
