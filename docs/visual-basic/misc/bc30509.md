# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;typu&gt; nie może dziedziczyć z &lt;specifier2&gt; &lt;typu&gt; ponieważ rozszerza on dostęp podstawowego &lt;typu&gt;
Podjęto próbę ma klasę publiczną, która dziedziczy z klasy bazowej, która jest oznaczona jako `Private` lub `Friend`.  
  
 **Identyfikator błędu:** BC30509  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie klasy bazowej `Public` lub zadeklarować klasy dziedziczącej `Private` lub `Friend`.
