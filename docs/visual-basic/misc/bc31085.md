# <a name="date-constant-is-not-valid"></a>Stała data jest nieprawidłowa
Stała daty jest zadeklarowana za pomocą nieprawidłową datę.  
  
 **Identyfikator błędu:** BC31085  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Popraw datę używaną w stałej.
