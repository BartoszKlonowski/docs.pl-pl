# <a name="date-constant-is-not-valid"></a>Stała daty jest nieprawidłowa
Stała daty jest zadeklarowana z nieprawidłową datą.  
  
 **Identyfikator błędu:** BC31085  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Popraw datę użytą w stałej.
