# <a name="the-file-is-currently-open-for-reading"></a>Plik jest obecnie otwarty do odczytu

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik i spróbuj ponownie wykonać operację.
