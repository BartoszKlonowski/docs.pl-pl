# <a name="drive-39ltdrivenamegt39-not-found"></a>Dysk &#39; &lt;drivename&gt;&#39; nie znaleziono
Nie można odnaleźć dysku. Utracono połączenie z siecią albo dysk, którego próbujesz uzyskać dostęp, jest w trybie offline lub nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź zasilanie dysku.  
  
2.  Upewnij się, czy kable połączenie komputera z sieci są poprawnie podłączone.  
  
3.  Uzyskania dostępu do dysku przez sieć, sprawdź, czy połączenie logiczne istnieje między komputerem i dysku.  
  
4.  Ponowne łączenie się z siecią i spróbuj ponownie.
