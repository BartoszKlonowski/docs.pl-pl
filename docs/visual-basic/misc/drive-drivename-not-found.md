# <a name="drive-ltdrivenamegt-not-found"></a>Dysk "&lt;drivename&gt;" nie znaleziono
Nie można odnaleźć dysku. Utracono połączenie z siecią albo dysk, który próbujesz uzyskać dostęp, jest w trybie offline lub nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź zasilanie dysku.  
  
2.  Upewnij się, czy kable Łączenie komputera z sieci są poprawnie podłączone.  
  
3.  Uzyskania dostępu do dysku za pośrednictwem sieci, upewnij się, czy połączenie logiczne istnieje między Twoim komputerem a dysku.  
  
4.  Ponowne nawiązywanie połączenia z siecią i spróbuj ponownie.
