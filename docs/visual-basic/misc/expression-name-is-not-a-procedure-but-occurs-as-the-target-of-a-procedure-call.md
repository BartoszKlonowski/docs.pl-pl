# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Wyrażenie " \<name> " nie jest procedurą, ale występuje jako element docelowy wywołania procedury

Wykonano wywołanie procedury do wyrażenia, które nie jest procedurą.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że nazwa procedury nie została wystawiona.  
  
2. Podaj procedurę, która <`name`> wywołania.
