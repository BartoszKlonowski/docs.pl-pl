# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>Konteksty i instrukcje "with" nie są prawidłowe w oknach debugowania
instrukcji `With` można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30726  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcje `With` z kodu debugowania.
