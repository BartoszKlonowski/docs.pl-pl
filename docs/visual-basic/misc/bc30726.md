# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39; Z &#39; konteksty i instrukcje nie są prawidłowe w oknach debugowania
`With`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30726  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `With` oświadczenia debugowania kodu.
