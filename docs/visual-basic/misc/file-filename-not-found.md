# <a name="file-39ltfilenamegt39-not-found"></a>Plik &#39; &lt;filename&gt;&#39; nie znaleziono

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że nazwa pliku jest poprawna i czy istnieje.
