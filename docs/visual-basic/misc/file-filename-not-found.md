# <a name="file-ltfilenamegt-not-found"></a>Plik "&lt;filename&gt;" nie znaleziono
Nazwa pliku jest nieprawidłowa lub plik nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że nazwa pliku jest poprawna i czy istnieje.
