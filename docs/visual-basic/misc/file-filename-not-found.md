# <a name="file-filename-not-found"></a>\<filename>Nie znaleziono pliku ""

Nazwa pliku jest nieprawidłowa lub plik nie istnieje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że nazwa pliku jest poprawna i że istnieje.
