# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-lttypegt"></a>Komentarz XML nie można zastosować więcej niż raz w częściowym elemencie &lt;typu&gt;
Komentarz XML nie można zastosować więcej niż raz w częściowym elemencie \<typ >. Komentarze XML dla tego \<typ > zostaną zignorowane.  
  
 Blok komentarza XML może znajdować się powyżej tylko jedną część typu częściowego.  
  
 Jeśli blok komentarza XML pojawia się powyżej więcej niż jedną część typu częściowego, to ostrzeżenie jest tworzony dla każdego bloku komentarza i najwyższego poziomu komentarz XML zostanie zignorowany.  
  
 **Identyfikator błędu:** BC42314  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń bloki nadmiarowe komentarz.
