# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-lttypegt"></a>Komentarz XML nie można zastosować więcej niż raz w częściowym &lt;typu&gt;
Komentarz XML nie można zastosować więcej niż raz w częściowym \<typu >. Komentarze XML dla tego \<typu > zostanie zignorowany.  
  
 Blok komentarza XML może pojawić się powyżej tylko jedną część typu częściowego.  
  
 Jeśli blok komentarza XML pojawia się powyżej więcej niż jedną część typu częściowego, to ostrzeżenie jest tworzona dla każdej blok komentarza i najwyższego poziomu komentarz XML zostanie zignorowany.  
  
 **Identyfikator błędu:** BC42314  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe komentarz bloków.
