# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-type"></a>Nie można zastosować komentarza XML więcej niż raz w częściowej \<type>

Nie można zastosować komentarza XML więcej niż raz w częściowej \<type> . Komentarze XML dla tego elementu \<type> zostaną zignorowane.  
  
 Blok komentarza XML może występować powyżej tylko jednej części typu częściowego.  
  
 Jeśli blok komentarza XML pojawia się powyżej więcej niż jednej części typu częściowego, to ostrzeżenie jest tworzone dla każdego bloku komentarza, a komentarz XML najwyższego poziomu zostanie zignorowany.  
  
 **Identyfikator błędu:** BC42314  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nadmiarowe bloki komentarzy.
