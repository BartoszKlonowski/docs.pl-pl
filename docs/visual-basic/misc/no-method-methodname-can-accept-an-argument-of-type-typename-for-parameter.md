# <a name="no-method-methodname-can-accept-an-argument-of-type-typename-for-parameter-parametername"></a>Żadna metoda " \<methodname> " nie może akceptować argumentu typu " \<typename> " dla parametru " \<parametername> "
Wywołanie metody zawiera argument typu, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź typy argumentów.  
  
2. Dopasowuje typy argumentów względem typów parametrów metody, aby upewnić się, że są one zawarte w umowie. Zmień typy w zależności od potrzeb.
