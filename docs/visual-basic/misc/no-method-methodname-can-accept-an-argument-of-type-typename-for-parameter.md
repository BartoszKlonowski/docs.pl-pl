# <a name="no-method-ltmethodnamegt-can-accept-an-argument-of-type-lttypenamegt-for-parameter-ltparameternamegt"></a>Brak metody "&lt;methodname&gt;"może akceptować argumentu typu"&lt;typename&gt;"dla parametru"&lt;parametername&gt;"
Wywołanie metody zawiera argument typu, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź typy argumentów.  
  
2.  Zgodne typy argumentów do typów parametru metody, aby upewnić się, że są one w umowie. Zmienić typy zgodnie z potrzebami.
