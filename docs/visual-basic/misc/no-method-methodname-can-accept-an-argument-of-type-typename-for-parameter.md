# <a name="no-method-39ltmethodnamegt39-can-accept-an-argument-of-type-39lttypenamegt39-for-parameter-39ltparameternamegt39"></a>Brak metody &#39; &lt;methodname&gt;&#39; może akceptować argument typu &#39;&lt; Właściwość TypeName&gt;&#39; dla parametru &#39;&lt; Nazwa parametru&gt;&#39;
Wywołanie metody zawiera argument typu, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź typy argumentów.  
  
2.  Zgodne typy argumentów do typów parametru metody, aby upewnić się, że są one zgodne. Typy należy zmienić w razie potrzeby.
