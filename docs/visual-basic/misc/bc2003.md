# <a name="response-file-filename-included-multiple-times"></a>Plik odpowiedzi został \<filename> uwzględniony wiele razy

Plik odpowiedzi został określony więcej niż raz. Pliki odpowiedzi zawierają opcje kompilatora i pliki kodu źródłowego do skompilowania.  
  
 **Identyfikator błędu:** BC2003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nadmiarowe specyfikatory plików odpowiedzi.
