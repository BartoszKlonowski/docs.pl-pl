# <a name="response-file-ltfilenamegt-included-multiple-times"></a>Plik odpowiedzi &lt;filename&gt; został dołączony wielokrotnie
Plik odpowiedzi została podana więcej niż raz. Pliki odpowiedzi zawiera opcje kompilatora i pliki kodu źródłowego, aby skompilować.  
  
 **Identyfikator błędu:** BC2003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikatory pliku odpowiedzi nadmiarowe.
