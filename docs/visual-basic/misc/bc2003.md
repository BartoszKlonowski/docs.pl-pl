# <a name="response-file-ltfilenamegt-included-multiple-times"></a>Plik odpowiedzi &lt;filename&gt; dołączony wielokrotnie
Plik odpowiedzi określono więcej niż raz. Pliki odpowiedzi zawiera opcje kompilatora i pliki kodu źródłowego, aby skompilować.  
  
 **Identyfikator błędu:** BC2003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe odpowiedzi specyfikatory pliku.
