# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-query-expression"></a>Wystąpienia typu ograniczonego &#39; &lt;typename&gt; &#39; nie można używać w wyrażeniach kwerend
Wystąpienia ograniczenia typów, takie jak <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, i <xref:System.TypedReference>, nie można użyć w wyrażeniu lambda lub wyrażeniu zapytania.  
  
 **Identyfikator błędu:** BC36598
