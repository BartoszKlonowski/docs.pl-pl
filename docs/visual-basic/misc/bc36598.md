# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-query-expression"></a>Wystąpienia typu ograniczonego "\<typename >" nie można używać w wyrażeniu zapytania
Wystąpienia elementu ograniczenia typów, takich jak <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, i <xref:System.TypedReference>, nie można używać w wyrażeniu lambda lub wyrażeniu zapytania.  
  
 **Identyfikator błędu:** BC36598
