# <a name="modifier-is-not-valid-on-an-event-declaration"></a>element " \<modifier> " nie jest prawidłowy w deklaracji zdarzenia
`Event`Instrukcja zawiera nieprawidłowe słowo kluczowe, np `ReadOnly` ..  
  
 **Identyfikator błędu:** BC30243  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń nieprawidłowe słowo kluczowe z `Event` instrukcji.
