# <a name="ltmodifiergt-is-not-valid-on-an-event-declaration"></a>"&lt;modyfikator&gt;" nie jest prawidłowy w deklaracji zdarzenia
`Event` Instrukcja zawiera nieprawidłowe słowo kluczowe `ReadOnly`.  
  
 **Identyfikator błędu:** BC30243  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nieprawidłowe słowo kluczowe z `Event` instrukcji.
