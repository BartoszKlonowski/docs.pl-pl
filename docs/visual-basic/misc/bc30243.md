# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;Modyfikator&gt; &#39; nie jest prawidłowy w deklaracji zdarzenia
`Event` Instrukcja zawiera nieprawidłowe słowo kluczowe `ReadOnly`.  
  
 **Identyfikator błędu:** BC30243  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nieprawidłowe słowo kluczowe z `Event` instrukcji.
