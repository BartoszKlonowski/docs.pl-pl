# <a name="argument-39per39-is-not-valid"></a>Argument &#39;na&#39; jest nieprawidłowy
Argument dostarczony dla `Per`, które muszą być `Double` określający okres płatności w zakresie od 1 do `NPer`, jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że argument należy do prawidłowego zakresu.
