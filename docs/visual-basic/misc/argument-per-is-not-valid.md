# <a name="argument-per-is-not-valid"></a>Argument "na" jest nieprawidłowy

Argument dostarczony dla `Per` , który musi być `Double` określeniem okresu płatności z zakresu od 1 do `NPer` , jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że argument znajduje się w prawidłowym zakresie.
