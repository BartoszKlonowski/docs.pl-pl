# <a name="argument-per-is-not-valid"></a>Argument "Na" jest nieprawidłowy
Argument podany dla `Per`, które muszą być `Double` określenie okresu płatności w zakresie od 1 do `NPer`, jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że argument znajduje się w dozwolonym zakresie.
