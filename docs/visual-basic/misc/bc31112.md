# <a name="statement-cannot-appear-within-an-event-body"></a>Instrukcja nie może wystąpić w treści zdarzenia
Instrukcja nie może wystąpić w treści zdarzenia. Założono koniec zdarzenia.  
  
 Instrukcja, która nie jest prawidłowa wewnątrz treści zdarzenia pojawia się wewnątrz jednej.  
  
 **Identyfikator błędu:** BC31112  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Event` przed instrukcją.
