# <a name="statement-cannot-appear-within-an-event-body"></a>Instrukcja nie może występować w treści zdarzenia
Instrukcja nie może występować w treści zdarzenia. Koniec zakłada, że zdarzenia.  
  
 Instrukcję, która nie jest prawidłowa wewnątrz treści zdarzenia pojawia się w jednym.  
  
 **Identyfikator błędu:** BC31112  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Event` przed instrukcją.
