# <a name="protected-types-can-only-be-declared-inside-of-a-class"></a>Typy chronione mogą być deklarowane tylko wewnątrz klasy
Typ w module został `Protected`zadeklarowany jako .

Najczęściej ten błąd kompilatora powoduje `Protected` po zastosowaniu modyfikatora dostępu do klasy niezagnieżdżonej. Przykład:

```vb
Public Class OuterClass     ' Generates compiler error BC31047.
End Class
```

Ponieważ `Protected` jest modyfikator dostępu do elementu członkowskiego, można go zastosować tylko do elementu członkowskiego klasy, takich jak właściwość, metoda lub klasa zagnieżdżona.

 **Identyfikator błędu:** Bc31047  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zadeklaruj typ w klasie.  
  
2. Usuń `Protected` modyfikator.
