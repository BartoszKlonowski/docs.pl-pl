# <a name="types-are-not-available-in-this-context"></a>Typy nie są dostępne w tym kontekście
Podjęto próbę odwołania do typu w ramach `#If` dyrektywy.  
  
 **Identyfikator błędu:** BC30458  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń odwołanie.
