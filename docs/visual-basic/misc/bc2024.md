# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>Domyślne odwołanie do zestawu " \<reference> " zostało już dodane automatycznie; zignorowano

Podjęto próbę dodania domyślnego odwołania do zestawu, ale zestaw został już dodany, więc próba jest ignorowana.  
  
 **Identyfikator błędu:** BC2024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń kod zbędny.
