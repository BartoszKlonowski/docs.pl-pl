# <a name="the-default-assembly-reference-ltreferencegt-has-already-been-automatically-added-ignored"></a>Domyślne odwołanie do zestawu "&lt;odwołania&gt;" ma już zostały automatycznie dodane; zignorowano
Podjęto próbę Dodaj odwołanie do zestawu domyślnego, ale zestaw został już dodany, dlatego próba jest ignorowana.  
  
 **Identyfikator błędu:** BC2024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuwanie zbędnym kodu.
