# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>Domyślne odwołanie do zestawu &#39; &lt;odwołania&gt; &#39; ma już zostały dodane automatycznie; zignorowano
Została podjęta próba dodania domyślne odwołanie do zestawu, ale zestaw został już dodany, dlatego próba jest ignorowana.  
  
 **Identyfikator błędu:** BC2024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędny kod.
