# <a name="registry-key-keyname-could-not-be-created"></a>Nie można utworzyć klucza rejestru " \<keyname> "

Nie można utworzyć określonego klucza rejestru, prawdopodobnie z powodu niewystarczających uprawnień zabezpieczeń.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że masz wystarczające uprawnienia zabezpieczeń, aby uzyskać dostęp do klucza.
