6. <span data-ttu-id="b12c2-119">We wszystkich przypadkach i jako ogólne wytyczne należy unikać stosowania konwersji zawężania, chyba że można zalewkować błędy w `Catch` bloku i efektywnie zajmować się nimi.</span><span class="sxs-lookup"><span data-stu-id="b12c2-119">In all cases and as a general guideline, you should avoid using narrowing conversions unless you can trap failures in a `Catch` block and deal with them effectively.</span></span>
