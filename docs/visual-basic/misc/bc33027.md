<span data-ttu-id="eddcd-103">Operator konwersji jest zadeklarowany z typem zwracanym, która pochodzi od typu parametru.</span><span class="sxs-lookup"><span data-stu-id="eddcd-103">A conversion operator is declared with a return type that derives from the parameter type.</span></span>  
  
 <span data-ttu-id="eddcd-104">W czasie kompilacji Visual Basic traktuje wstępnie zdefiniowanych konwersję istnieje z dowolnego typu odwołania do dowolnego typu w hierarchii dziedziczenia, czyli dowolnego typu, z której pochodzi, lub pochodzącą od niego.</span><span class="sxs-lookup"><span data-stu-id="eddcd-104">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="eddcd-105">Taka Konwersja może się nie powieść w czasie wykonywania, ale kompilator nie można przewidzieć wyniki wykonawczej pozwala takich konwersji skompilować.</span><span class="sxs-lookup"><span data-stu-id="eddcd-105">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
