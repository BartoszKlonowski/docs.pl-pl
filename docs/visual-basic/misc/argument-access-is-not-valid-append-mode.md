# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowy. Prawidłowe wartości trybu Append to "OpenAccess. Write" i "OpenAccess. default"

Wartość dostępu jest nieprawidłowa w trybie dołączania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień wartość na `OpenAccess.Write` lub `OpenAccess.Default` .
