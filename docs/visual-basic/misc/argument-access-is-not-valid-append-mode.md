# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argument &#39; Dostęp do &#39; nie jest prawidłowy. Prawidłowe wartości są w trybie Append &#39;OpenAccess.Write &#39; i &#39;OpenAccess.Default &#39;
Wartość dostępu jest nieprawidłowa w trybie dołączania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zmień wartość na jedną `OpenAccess.Write` lub `OpenAccess.Default`.
