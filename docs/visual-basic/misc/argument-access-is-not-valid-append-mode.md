# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>Argument "Access" jest nieprawidłowa. Prawidłowe wartości dla trybu Append to "OpenAcess.Write" i "OpenAccess.Default"
Wartość dostępu jest nieprawidłowe w trybie Append.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień wartość na jedną `OpenAccess.Write` lub `OpenAccess.Default`.
