# <a name="array-initializer-has-too-few-dimensions"></a>Inicjator tablicy ma za mało wymiarów

Nie dostarczono wystarczającej liczby wymiarów w inicjatorze tablicy.  
  
 **Identyfikator błędu:** BC30565  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź inicjator tablicy, aby określić, ile wymiarów jest koniecznych.  
  
- Podaj brakujący argument lub argumenty.
