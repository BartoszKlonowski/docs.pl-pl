# <a name="array-initializer-has-too-few-dimensions"></a>Inicjator tablicy ma za mało wymiarów.
Nie podano za mało wymiarów w inicjatorze dla macierzy.  
  
 **Identyfikator błędu:** BC30565  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź z inicjatora tablicy. Aby ustalić liczby wymiarów są niezbędne.  
  
-   Należy podać argument brakujące lub argumentów.
