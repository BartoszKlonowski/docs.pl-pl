# <a name="array-initializer-has-too-few-dimensions"></a>Inicjator tablicy ma za mało wymiarów
Nie podano wystarczających wymiarów w inicjatorze dla tablicy.  
  
 **Identyfikator błędu:** BC30565  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź swoje inicjatora tablicy, aby ustalić, ile wymiary są niezbędne.  
  
-   Podaj brakujące argument lub argumentów.
