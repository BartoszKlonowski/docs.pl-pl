# <a name="argument-not-specified-for-parameter-parametername-of-extension-method-methodname-defined-in-typename"></a>Nie określono argumentu dla parametru '\<parametername >' metody rozszerzenia '\<methodname >' zdefiniowane w '\<typename >'
Nie podano argumentu dla wymaganego parametru w metodzie rozszerzenia.  
  
 **Identyfikator błędu:** BC36586  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy podać argument określony parametr.
