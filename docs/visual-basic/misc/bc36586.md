# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Nie określono argumentu dla parametru "&lt;parametername&gt;"metody rozszerzenia"&lt;methodname&gt;"zdefiniowane w"&lt;typename&gt;"
Nie podano argumentu dla wymaganego parametru w metodzie rozszerzenia.  
  
 **Identyfikator błędu:** BC36586  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy podać argument określony parametr.
