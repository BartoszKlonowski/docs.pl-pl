# <a name="ltelementnamegt-is-not-a-method-parameter"></a>"&lt;elementname&gt;" nie jest parametrem metody
Wywołanie procedury dostarcza dodatkowy argument, który nie jest zdefiniowany przez metodę.  
  
 **Identyfikator błędu:** BC36568  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędne argumentu z listy argumentów.
