# <a name="39ltelementnamegt39-is-not-a-method-parameter"></a>&#39; &lt;elementname&gt;&#39; nie jest parametrem — metoda
Wywołanie procedury udostępnia dodatkowy argument, który nie jest zdefiniowany przez metodę.  
  
 **Identyfikator błędu:** BC36568  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędny argumentu z listy argumentów.
