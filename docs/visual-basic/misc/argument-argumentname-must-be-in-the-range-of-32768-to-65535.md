# <a name="argument-ltargumentnamegt-must-be-in-the-range-of--32768-to-65535"></a>Argument "&lt;argumentname&gt;" musi należeć do zakresu od -32768 do 65535
Wartość argumentu znajduje się poza dozwolonym zakresem-32 768 do 65 535.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że wartość argumentu mieści się w prawidłowym zakresem.  
  
2.  Jeśli argument jest dostarczana przez obliczenie, sprawdź wartości procesów obliczeń.
