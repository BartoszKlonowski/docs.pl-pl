# <a name="argument-argumentname-must-be-in-the-range-of--32768-to-65535"></a>Argument " \<argumentname> " musi należeć do zakresu od-32768 do 65535

Wartość argumentu jest poza dozwolonym zakresem od-32 768 do 65 535.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że wartość argumentu mieści się w prawidłowym zakresie.  
  
2. Jeśli argument jest dostarczany przez obliczenie, sprawdź wartości w procesach obliczania.
