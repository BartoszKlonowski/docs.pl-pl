# <a name="project-level-conditional-compilation-constant-ltconstantnamegt-is-not-valid-lterrorgt"></a>Stała kompilacji warunkowej na poziomie projektu "&lt;constantname&gt;" jest nieprawidłowa: &lt;błąd&gt;
Opcja nieprawidłowy kompilacja warunkowa jest określona dla projektu.  
  
 **Identyfikator błędu:** BC31030  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź stałe kompilacji warunkowej, używane z tym projektem.
