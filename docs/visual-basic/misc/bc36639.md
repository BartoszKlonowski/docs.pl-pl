# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="bd57d-102">Parametr "ByRef"\<' parametrname>' nie może być używany w wyrażeniu lambda</span><span class="sxs-lookup"><span data-stu-id="bd57d-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="bd57d-103">Wyrażenie lambda zadeklarowane `Sub` w obrębie `ByRef` lub funkcji `Sub` nie może używać żadnych parametrów tego lub funkcji.</span><span class="sxs-lookup"><span data-stu-id="bd57d-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="bd57d-104">Na przykład poniższy kod spowoduje ten `ByRef` `n` błąd, ponieważ parametr jest używany w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="bd57d-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="bd57d-105">**Identyfikator błędu:** Bc36639</span><span class="sxs-lookup"><span data-stu-id="bd57d-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bd57d-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="bd57d-106">To correct this error</span></span>  
  
- <span data-ttu-id="bd57d-107">Przypisz `ByRef` parametr do zmiennej lokalnej i użyj zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="bd57d-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
