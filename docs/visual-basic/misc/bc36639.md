# <a name="39byref39-parameter-39ltparameternamegt39-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="02a2a-102">&#39;ByRef&#39; parametru &#39; &lt;parametername&gt; &#39; nie można użyć w wyrażeniu lambda</span><span class="sxs-lookup"><span data-stu-id="02a2a-102">&#39;ByRef&#39; parameter &#39;&lt;parametername&gt;&#39; cannot be used in a lambda expression</span></span>
<span data-ttu-id="02a2a-103">Wyrażenia lambda zadeklarowana wewnątrz `Sub` lub funkcja nie może używać żadnych `ByRef` parametry tego `Sub` lub funkcji.</span><span class="sxs-lookup"><span data-stu-id="02a2a-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="02a2a-104">Na przykład następujący kod będzie powodować tego błędu, ponieważ `ByRef` parametru `n` jest używany w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="02a2a-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="02a2a-105">**Identyfikator błędu:** BC36639</span><span class="sxs-lookup"><span data-stu-id="02a2a-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="02a2a-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="02a2a-106">To correct this error</span></span>  
  
-   <span data-ttu-id="02a2a-107">Przypisz `ByRef` parametr do zmiennej lokalnej i użycie zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="02a2a-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
