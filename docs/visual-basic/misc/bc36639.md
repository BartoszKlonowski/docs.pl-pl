# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="4822f-102">Parametr "ByRef" "\<parametername >" nie można używać w wyrażeniu lambda</span><span class="sxs-lookup"><span data-stu-id="4822f-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="4822f-103">Wyrażenie lambda zadeklarowane wewnątrz `Sub` lub funkcja nie może używać żadnych `ByRef` parametry tego `Sub` lub funkcji.</span><span class="sxs-lookup"><span data-stu-id="4822f-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="4822f-104">Na przykład, poniższy kod będzie powodować błąd, ponieważ `ByRef` parametru `n` jest używana w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="4822f-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="4822f-105">**Identyfikator błędu:** BC36639</span><span class="sxs-lookup"><span data-stu-id="4822f-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4822f-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4822f-106">To correct this error</span></span>  
  
-   <span data-ttu-id="4822f-107">Przypisz `ByRef` parametr do zmiennej lokalnej i użyj zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="4822f-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
