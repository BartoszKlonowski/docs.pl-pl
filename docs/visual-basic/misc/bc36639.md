# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="25aae-102">Parametr 'ByRef' '\<parametername >' nie można używać w wyrażeniu lambda</span><span class="sxs-lookup"><span data-stu-id="25aae-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="25aae-103">Wyrażenie lambda zadeklarowane w `Sub` lub funkcja nie może używać żadnych parametrów `ByRef` tego `Sub` lub funkcji.</span><span class="sxs-lookup"><span data-stu-id="25aae-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="25aae-104">Na przykład poniższy kod spowoduje wystąpienie tego błędu, ponieważ parametr `ByRef` `n` jest używany w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="25aae-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="25aae-105">**Identyfikator błędu:** BC36639</span><span class="sxs-lookup"><span data-stu-id="25aae-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="25aae-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="25aae-106">To correct this error</span></span>  
  
- <span data-ttu-id="25aae-107">Przypisz parametr `ByRef` do zmiennej lokalnej i Użyj zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="25aae-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
