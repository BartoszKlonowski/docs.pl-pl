# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a>Parametr "ByRef"\<' parametrname>' nie może być używany w wyrażeniu lambda
Wyrażenie lambda zadeklarowane `Sub` w obrębie `ByRef` lub funkcji `Sub` nie może używać żadnych parametrów tego lub funkcji. Na przykład poniższy kod spowoduje ten `ByRef` `n` błąd, ponieważ parametr jest używany w wyrażeniu lambda.  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 **Identyfikator błędu:** Bc36639  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przypisz `ByRef` parametr do zmiennej lokalnej i użyj zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
