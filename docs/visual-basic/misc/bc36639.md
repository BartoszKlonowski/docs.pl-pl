# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="83746-102">Nie można użyć parametru "ByRef" " \<parametername> " w wyrażeniu lambda</span><span class="sxs-lookup"><span data-stu-id="83746-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>

<span data-ttu-id="83746-103">Wyrażenie lambda zadeklarowane wewnątrz `Sub` funkcji or nie może używać żadnych `ByRef` parametrów tej `Sub` lub funkcji.</span><span class="sxs-lookup"><span data-stu-id="83746-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="83746-104">Na przykład poniższy kod spowoduje wystąpienie tego błędu, ponieważ `ByRef` parametr `n` jest używany w wyrażeniu lambda.</span><span class="sxs-lookup"><span data-stu-id="83746-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="83746-105">**Identyfikator błędu:** BC36639</span><span class="sxs-lookup"><span data-stu-id="83746-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="83746-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="83746-106">To correct this error</span></span>  
  
- <span data-ttu-id="83746-107">Przypisz `ByRef` parametr do zmiennej lokalnej i Użyj zmiennej lokalnej w wyrażeniu lambda, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="83746-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
