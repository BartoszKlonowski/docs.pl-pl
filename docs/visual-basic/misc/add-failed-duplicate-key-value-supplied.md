# <a name="add-failed-duplicate-key-value-supplied"></a>Dodanie nie powiodło się. Podano zduplikowaną wartość klucza
`Add`Operacja nie powiodła się, ponieważ wartość klucza jest taka sama jak inna wartość klucza. Wartości kluczy muszą być unikatowe.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że wartość klucza jest unikatowa.
