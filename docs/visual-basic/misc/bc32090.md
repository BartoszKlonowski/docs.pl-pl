# <a name="type-parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr typu nie może mieć takiej samej nazwy jak definiująca go funkcja
Parametr typu ogólnego jest zadeklarowana z taką samą nazwę jak typ ogólny.  
  
 Na liście parametrów typu ogólnego typu każdego parametru typu musi mieć nazwę różne od wszystkie z następujących nazw:  
  
-   Co parametr typu w tej samej listy parametrów typu  
  
-   Co parametr typu na liście parametrów typu dowolnego zawierającego typu ogólnego i  
  
-   Nazwa samego typu ogólnego.  
  
 **Identyfikator błędu:** BC32090  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę parametru typu się różnić od każdej nazwy wymienione na liście na tej stronie pomocy.
