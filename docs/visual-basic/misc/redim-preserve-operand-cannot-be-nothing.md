# <a name="redim-preserve-operand-cannot-be-nothing"></a>Argument operacji zachowywania "ReDim" nie może mieć wartości Nothing
Instrukcja próbuje użyć słowa kluczowego, aby zmienić wymiar tablicy, która nie jest ostatnim wymiarem, ale nie dostarcza prawidłowej wartości dla tego operandu. `Preserve` `ReDim`  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `Preserve` Zmień operand na prawidłową wartość.
