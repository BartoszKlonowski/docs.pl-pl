# <a name="39redim39-preserve-operand-cannot-be-nothing"></a>&#39; ReDim &#39; Zachowaj operand nie może być nic
A `ReDim` próbuje użyć instrukcji `Preserve` — słowo kluczowe, aby zmienić wymiaru tablicy, która nie jest ostatni wymiar, ale nie dostarcza prawidłową wartością dla jego argumentu operacji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień `Preserve` operandu na prawidłową wartość.
