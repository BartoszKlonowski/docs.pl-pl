# <a name="object-created-by-edit-and-continue-is-not-yet-available-for-evaluation"></a>Obiekt utworzony przez elementy Edit i Continue nie jest jeszcze dostępna w wersji ewaluacyjnej
W debugerze programu Visual Studio jest tworzony nowy obiekt, ale go nie ma jeszcze dostępu.  
  
 Możliwe przyczyny dla obiekt będzie dostępny: pamięci stanu błędu i obiekt alokacji poza synchronizacji. Są to tymczasowych warunków wewnętrzny i zazwyczaj nie wynik danych wejściowych do debugera.  
  
 **Identyfikator błędu:** BC30958  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zakończenie, a następnie ponownie uruchom debugowanie.
