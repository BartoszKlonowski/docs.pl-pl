# <a name="object-created-by-edit-and-continue-is-not-yet-available-for-evaluation"></a>Obiekt utworzony za pomocą Edytuj i Kontynuuj nie jest jeszcze dostępny do oceny

W debugerze programu Visual Studio tworzony jest nowy obiekt, ale nie można go jeszcze uzyskać do niego dostępu.  
  
 Możliwe przyczyny niedostępności obiektu obejmują niepowodzenie alokacji pamięci i stan obiektu poza synchronizacją. Są to tymczasowe warunki wewnętrzne i zwykle nie są wynikiem danych wejściowych debugera.  
  
 **Identyfikator błędu:** BC30958  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przerwij i ponownie uruchom debugowanie.
