# <a name="object-created-by-edit-and-continue-is-not-yet-available-for-evaluation"></a>Obiekt utworzony przez Edytuj i Kontynuuj nie jest jeszcze dostępna w wersji ewaluacyjnej
W debugerze programu Visual Studio tworzony jest nowy obiekt, ale go nie są jeszcze dostępne.  
  
 Możliwe przyczyny, dla obiektu, niedostępność: alokacji awarii i obiektu stanu pamięci zsynchronizowane. Są tymczasowe warunki wewnętrzny i zazwyczaj nie wynikiem wejścia do debugera.  
  
 **Identyfikator błędu:** BC30958  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zakończ i uruchom ponownie debugowanie.
