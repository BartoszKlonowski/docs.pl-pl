# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="15038-102">Typ elementu " \<variablename> " jest niejednoznaczny, ponieważ granice pętli i zmienna kroku nie są rozszerzane do tego samego typu</span><span class="sxs-lookup"><span data-stu-id="15038-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>

<span data-ttu-id="15038-103">Kod zawiera `For...Next` pętlę, w której kompilator nie może wywnioskować typu danych dla zmiennej kontroli pętli, ponieważ spełnione są następujące warunki:</span><span class="sxs-lookup"><span data-stu-id="15038-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="15038-104">Typ danych zmiennej sterującej pętli nie jest określony z `As` klauzulą.</span><span class="sxs-lookup"><span data-stu-id="15038-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="15038-105">Granice pętli i zmienna kroku zawierają co najmniej dwa typy danych.</span><span class="sxs-lookup"><span data-stu-id="15038-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="15038-106">Istnieje więcej niż jedna możliwa konwersja między typami danych.</span><span class="sxs-lookup"><span data-stu-id="15038-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="15038-107">Nie ma najlepszego typu wśród kandydatów, dlatego wybór typu dla zmiennej sterującej pętli jest niejednoznaczny.</span><span class="sxs-lookup"><span data-stu-id="15038-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="15038-108">Na przykład następująca pętla zawiera jedną pętlę powiązaną z typem `Integer` i jedną pętlą powiązaną z typem `UInteger` :</span><span class="sxs-lookup"><span data-stu-id="15038-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="15038-109">Istnieje możliwość konwersji między i i `Integer` `UInteger` możliwa konwersja między `UInteger` i `Integer` , ale obie te konwersje są zawężane, więc żaden z nich nie jest najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="15038-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="15038-110">W następnym przykładzie zostanie dodana trzecia zmienna typu `Double` .</span><span class="sxs-lookup"><span data-stu-id="15038-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="15038-111">Zarówno `Integer` , jak i `UInteger` mają standardowe konwersje rozszerzające do `Double` , które umożliwiają konwersję na `Double` najlepszy wybór.</span><span class="sxs-lookup"><span data-stu-id="15038-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="15038-112">Przypisuje typ wnioskowania do zmiennej kontroli pętli `i` Typ danych `Double` .</span><span class="sxs-lookup"><span data-stu-id="15038-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="15038-113">**Identyfikator błędu:** BC30983</span><span class="sxs-lookup"><span data-stu-id="15038-113">**Error ID:** BC30983</span></span>
