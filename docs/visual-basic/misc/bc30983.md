 <span data-ttu-id="c1aea-109">Istnieje możliwość konwersji `Integer` `UInteger`między i , `UInteger` i `Integer`możliwe konwersji między i , ale oba są zawężenia konwersji, więc nie jest najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="c1aea-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>
