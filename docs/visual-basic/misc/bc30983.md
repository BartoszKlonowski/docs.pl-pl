# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="b2457-102">Typ '\<nazwa_zmiennej >' jest niejednoznaczny, ponieważ granice pętli i zmienna Step nie mogą zostać poszerzone do tego samego typu</span><span class="sxs-lookup"><span data-stu-id="b2457-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="b2457-103">Kod zawiera pętlę `For...Next`, w której kompilator nie może wywnioskować typu danych dla zmiennej kontroli pętli, ponieważ spełnione są następujące warunki:</span><span class="sxs-lookup"><span data-stu-id="b2457-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="b2457-104">Typ danych zmiennej sterującej pętli nie jest określony z klauzulą `As`.</span><span class="sxs-lookup"><span data-stu-id="b2457-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="b2457-105">Granice pętli i zmienna kroku zawierają co najmniej dwa typy danych.</span><span class="sxs-lookup"><span data-stu-id="b2457-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="b2457-106">Istnieje więcej niż jedna możliwa konwersja między typami danych.</span><span class="sxs-lookup"><span data-stu-id="b2457-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="b2457-107">Nie ma najlepszego typu wśród kandydatów, dlatego wybór typu dla zmiennej sterującej pętli jest niejednoznaczny.</span><span class="sxs-lookup"><span data-stu-id="b2457-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="b2457-108">Na przykład następująca pętla ma jedną pętlę powiązaną z typem `Integer` i jedną pętlą powiązaną z typem `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="b2457-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="b2457-109">Istnieje możliwość konwersji między `Integer` i `UInteger` i możliwą konwersją między `UInteger` i `Integer`, ale obie te konwersje zawężają, więc żaden z nich nie jest najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="b2457-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="b2457-110">W następnym przykładzie zostanie dodana trzecia zmienna typu `Double`.</span><span class="sxs-lookup"><span data-stu-id="b2457-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="b2457-111">Zarówno `Integer` i `UInteger` mają standardowe konwersje rozszerzające na `Double`, co sprawia, że konwersja na `Double` najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="b2457-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="b2457-112">Typ danych przypisuje do wartości zmiennej sterującej pętli `i` typu `Double`.</span><span class="sxs-lookup"><span data-stu-id="b2457-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="b2457-113">**Identyfikator błędu:** BC30983</span><span class="sxs-lookup"><span data-stu-id="b2457-113">**Error ID:** BC30983</span></span>

    ```vb  
    For i = m To CLng(n)
    ```  

    ```vb  
    For i As Double = m To n
    ```  
