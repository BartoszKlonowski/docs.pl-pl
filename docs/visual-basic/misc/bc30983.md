<span data-ttu-id="7d61b-103">W kodzie `For...Next` pętli, w którym kompilator nie można wywnioskować typu danych dla zmienna sterująca pętli, ponieważ dotyczy następujących warunków:</span><span class="sxs-lookup"><span data-stu-id="7d61b-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
-   <span data-ttu-id="7d61b-104">Typ danych zmienna sterująca pętli nie zostanie określony z `As` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="7d61b-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
-   <span data-ttu-id="7d61b-105">Granice pętli i zmienna krok zawiera co najmniej dwóch typów.</span><span class="sxs-lookup"><span data-stu-id="7d61b-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
-   <span data-ttu-id="7d61b-106">Istnieje kilka możliwych konwersji typów danych.</span><span class="sxs-lookup"><span data-stu-id="7d61b-106">More than one possible conversion exists between the data types.</span></span>  
  
-   <span data-ttu-id="7d61b-107">Nie ma najlepsze typu między obiektami, umożliwiając wybór typu zmienna sterująca pętli for jest niejednoznaczna.</span><span class="sxs-lookup"><span data-stu-id="7d61b-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="7d61b-108">Na przykład następującą pętlę ma jeden granica pętli typu `Integer` i powiązane pętli typu `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="7d61b-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="7d61b-109">Brak możliwości konwersji między `Integer` i `UInteger`i możliwa konwersja między `UInteger` i `Integer`, ale są oba zawężanie konwersji więc nie będzie najlepszym rozwiązaniem.</span><span class="sxs-lookup"><span data-stu-id="7d61b-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="7d61b-110">W następnym przykładzie trzeciej zmiennej typu `Double` został dodany.</span><span class="sxs-lookup"><span data-stu-id="7d61b-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="7d61b-111">Zarówno `Integer` i `UInteger` ma standardowych konwersji rozszerzającej na `Double`, co czyni konwersji na `Double` najlepszym rozwiązaniem.</span><span class="sxs-lookup"><span data-stu-id="7d61b-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="7d61b-112">Wnioskowanie o typie przypisuje zmienna sterująca pętli for `i` typu danych `Double`.</span><span class="sxs-lookup"><span data-stu-id="7d61b-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```
