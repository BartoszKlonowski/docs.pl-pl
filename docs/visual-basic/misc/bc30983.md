# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="a9ddc-102">Typ '\<nazwa_zmiennej >' jest niejednoznaczny, ponieważ granice pętli i zmienna Step nie mogą zostać poszerzone do tego samego typu</span><span class="sxs-lookup"><span data-stu-id="a9ddc-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="a9ddc-103">W kodzie `For...Next` pętli, w którym kompilator nie można wywnioskować typu danych dla zmienna sterująca pętli, ponieważ dotyczy następujących warunków:</span><span class="sxs-lookup"><span data-stu-id="a9ddc-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="a9ddc-104">Typ danych zmienna sterująca pętli nie zostanie określony z `As` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="a9ddc-105">Granice pętli i zmienna zawiera co najmniej dwa typy danych.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="a9ddc-106">Istnieje kilka możliwych konwersji między typami danych.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="a9ddc-107">Istnieje nie najlepszy typ między obiektami, tak, aby wybór typu zmienna sterująca pętli jest niejednoznaczna.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="a9ddc-108">Na przykład następującą pętlę ma jedną granicę pętli typu `Integer` pętli one później powiązane typu `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="a9ddc-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="a9ddc-109">Brak możliwości konwersji między `Integer` i `UInteger`i możliwe konwersji między `UInteger` i `Integer`, oba są zawężających, ale tak nie jest najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="a9ddc-110">W następnym przykładzie trzeci zmienną typu `Double` zostanie dodany.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="a9ddc-111">Zarówno `Integer` i `UInteger` ma standardowych konwersji rozszerzającej na `Double`, co sprawia, że konwersja `Double` najlepszym wyborem.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="a9ddc-112">Wnioskowanie o typie przypisuje zmienna sterująca pętli `i` typu danych `Double`.</span><span class="sxs-lookup"><span data-stu-id="a9ddc-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="a9ddc-113">**Identyfikator błędu:** BC30983</span><span class="sxs-lookup"><span data-stu-id="a9ddc-113">**Error ID:** BC30983</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a9ddc-114">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a9ddc-114">To correct this error</span></span>  
  
- <span data-ttu-id="a9ddc-115">Jawnie przekonwertować na typ, czy inne na przykład umożliwiać konwersję rozszerzającą, jedna ze zmiennych:</span><span class="sxs-lookup"><span data-stu-id="a9ddc-115">Explicitly convert one of the variables to a type that the others have a widening conversion to, for example:</span></span>  
  
    ```  
    For i = m To CLng(n)  
    ```

    ```  
    For i As Double = m To n   
    ```
