# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Instrukcji "Throw" nie są prawidłowe w oknie bezpośrednim
`Throw` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30725  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Throw` instrukcji debugowania kodu.
