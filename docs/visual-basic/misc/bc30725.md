# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Throw &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Throw`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30725  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Throw` oświadczenia debugowania kodu.
