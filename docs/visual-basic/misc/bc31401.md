# <a name="static-local-variable-39ltvariablenamegt39-is-already-declared"></a>Statyczna zmienna lokalna &#39; &lt;nazwa_zmiennej&gt;&#39; jest już zadeklarowana
Inny statyczna zmienna lokalna o takiej samej nazwie został już zadeklarowany.  
  
 **Identyfikator błędu:** BC31401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nadmiarowe statycznych lokalne deklaracje.  
  
2.  Nadaj każdej statyczna zmienna lokalna unikatową nazwę.
