# <a name="static-local-variable-variablename-is-already-declared"></a>Statyczna zmienna lokalna " \<variablename> " jest już zadeklarowana
Inna statyczna zmienna lokalna o tej samej nazwie została już zadeklarowana.  
  
 **Identyfikator błędu:** BC31401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń nadmiarowe statyczne deklaracje lokalne.  
  
2. Nadaj każdej statycznej zmiennej lokalnej unikatową nazwę.
