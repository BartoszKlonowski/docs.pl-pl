# <a name="static-local-variable-ltvariablenamegt-is-already-declared"></a>Statyczna zmienna lokalna "&lt;nazwa_zmiennej&gt;" jest już zadeklarowana
Inny statyczna zmienna lokalna o takiej samej nazwie został już zadeklarowany.  
  
 **Identyfikator błędu:** BC31401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nadmiarowym deklaracjom lokalnej statycznej.  
  
2.  Nadaj każdej zmiennej lokalnej statycznej unikatową nazwę.
