# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;Udostępnione&#39; nie można połączyć z &#39; &lt;specyfikator&gt; &#39; w deklaracji metody
Próbujesz połączyć `Shared` modyfikator ze specyfikatorem, takich jak `Overridable`, `NotOverridable`, lub `MustOverride`, która jest nieprawidłowa w kombinacji w deklaracji metody.  
  
 **Identyfikator błędu:** BC30501  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
