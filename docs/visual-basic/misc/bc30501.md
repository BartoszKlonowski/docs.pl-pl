# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-method-declaration"></a>"Shared" nie można połączyć z "&lt;specyfikator&gt;" w deklaracji metody
Próbujesz połączyć `Shared` modyfikator ze specyfikatorem, takich jak `Overridable`, `NotOverridable`, lub `MustOverride`, to znaczy nie są prawidłowe w kombinacji w deklaracji metody.  
  
 **Identyfikator błędu:** BC30501  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
