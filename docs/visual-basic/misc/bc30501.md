# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>Nie można łączyć instrukcji "Shared" z elementem " \<specifier> " w deklaracji metody
Podjęto próbę połączenia `Shared` modyfikatora ze specyfikatorem, takim jak `Overridable` , `NotOverridable` , lub `MustOverride` , która nie jest prawidłowa w takiej kombinacji w deklaracji metody.  
  
 **Identyfikator błędu:** BC30501  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
