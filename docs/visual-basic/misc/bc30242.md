# <a name="ltmodifiergt-is-not-valid-on-a-method-declaration"></a>"&lt;modyfikator&gt;" nie jest prawidłowy w deklaracji metody
A `Function` lub `Sub` instrukcja zawiera nieprawidłowe słowo kluczowe.  
  
 **Identyfikator błędu:** BC30242  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowe słowo kluczowe z `Function` lub `Sub` instrukcji.
