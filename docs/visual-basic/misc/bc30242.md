# <a name="modifier-is-not-valid-on-a-method-declaration"></a>element " \<modifier> " nie jest prawidłowy w deklaracji metody
`Function`Instrukcja or `Sub` zawiera nieprawidłowe słowo kluczowe.  
  
 **Identyfikator błędu:** BC30242  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe słowo kluczowe z `Function` instrukcji or `Sub` .
