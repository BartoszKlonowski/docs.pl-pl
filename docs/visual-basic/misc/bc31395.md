# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Znaki typu są niedozwolone w identyfikatorach etykiet

Etykieta instrukcji zawiera co najmniej jeden znak typu identyfikator.  
  
 Etykieta instrukcji musi być prawidłową nazwą Visual Basic. Dozwolone znaki nie zawierają żadnego ze znaków typu identyfikatora ( `%` ,,,, `&` `@` `!` `#` i `$` ). Zobacz [zadeklarowane nazwy elementów](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identyfikator błędu:** BC31395  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń znak identyfikatora typu lub znaki z etykiety instrukcji.
