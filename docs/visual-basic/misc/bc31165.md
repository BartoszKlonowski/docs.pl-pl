# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>Oczekiwano początku &#39; &lt; &#39; dla tagu XML
Tag XML dla literał XML brakuje wymaganego początku "<" znaków.  
  
 **Identyfikator błędu:** BC31165  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj "<" znak na początku tagu XML dla literału XML.
