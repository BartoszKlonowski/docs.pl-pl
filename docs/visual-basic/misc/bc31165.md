# <a name="expected-beginning--for-an-xml-tag"></a>Oczekiwano początku "\<" dla tagu XML
Znacznik XML literał XML brakuje wymaganego początku "<" znaków.  
  
 **Identyfikator błędu:** BC31165  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodawanie ' <' znak na początku tagu XML dla literał XML.
