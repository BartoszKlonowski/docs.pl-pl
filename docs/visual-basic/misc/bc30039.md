# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Zmienna sterująca pętli nie może być właściwością ani indeksowaną tablicą z późnym wiązaniem

Zmienna użyta do iteracji w `For` pętli musi być typu danych liczbowych.  
  
 **Identyfikator błędu:** BC30039  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień deklarację zmiennej sterującej pętli, aby określić `Integer` , `Short` , `Long` ,,, `Byte` `Single` `Double` lub `Decimal` .
