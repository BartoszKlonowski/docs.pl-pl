# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Zmienna sterująca pętli nie może mieć właściwości ani tablicy indeksowanej późnego wiązania
Zmienna, używany do iterowania `For` pętli musi być typu liczbowego.  
  
 **Identyfikator błędu:** BC30039  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień deklarację zmienna sterująca pętli, aby określić `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, lub `Decimal`.
