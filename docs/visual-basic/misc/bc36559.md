# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="dfc2b-102">Nie można użyć właściwości składowej typu anonimowego " \<propertyname> " do wywnioskowania typu innej właściwości składowej, ponieważ typ " \<propertyname> " nie został jeszcze ustanowiony</span><span class="sxs-lookup"><span data-stu-id="dfc2b-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="dfc2b-103">Dopóki nie zostanie ustanowiony typ właściwości typu anonimowego, nie można jej użyć do ustalenia typu innej właściwości.</span><span class="sxs-lookup"><span data-stu-id="dfc2b-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="dfc2b-104">Na przykład, w następującej deklaracji `.IDName = .LastName` jest nieprawidłowa, ponieważ `.LastName` nie została jeszcze zainicjowana.</span><span class="sxs-lookup"><span data-stu-id="dfc2b-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="dfc2b-105">**Identyfikator błędu:** BC36559</span><span class="sxs-lookup"><span data-stu-id="dfc2b-105">**Error ID:** BC36559</span></span>

