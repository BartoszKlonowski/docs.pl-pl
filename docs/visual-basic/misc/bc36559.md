<span data-ttu-id="69cce-103">Dopóki nie zostanie nawiązane z typem właściwości typu anonimowego, go nie można ustanowić typu innej właściwości.</span><span class="sxs-lookup"><span data-stu-id="69cce-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="69cce-104">Na przykład w następującej deklaracji `.IDName = .LastName` jest nieprawidłowy ponieważ `.LastName` nie została jeszcze zainicjowana.</span><span class="sxs-lookup"><span data-stu-id="69cce-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>
