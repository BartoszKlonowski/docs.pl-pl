# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>Nazwa zasobu &#39; &lt;resourcename&gt; &#39; nie można użyć więcej niż raz
Nazwa określonego zasobu w bibliotece łączone dynamicznie lub pliku wykonywalnego użyto więcej niż raz.  
  
 **Identyfikator błędu:** BC31502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Unikatowej nazwy zasobów.
