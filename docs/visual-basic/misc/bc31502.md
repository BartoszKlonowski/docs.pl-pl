# <a name="resource-name-ltresourcenamegt-cannot-be-used-more-than-once"></a>Nazwa zasobu "&lt;resourcename&gt;" nie można użyć więcej niż raz
Nazwa określonego zasobu biblioteki łączone dynamicznie lub plik wykonywalny został użyty więcej niż jeden raz.  
  
 **Identyfikator błędu:** BC31502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Unikatowej nazwy zasobów.
