# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>Nie można użyć nazwy zasobu "\<resourcename >" więcej niż raz
Nazwa określonego zasobu w bibliotece połączonej dynamicznie lub pliku wykonywalnego została użyta więcej niż raz.  
  
 **Identyfikator błędu:** BC31502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zasoby z unikatowymi nazwami.
