# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion-ltlistgt"></a>Żaden dostępny przeciążony element &#39; &lt;methodname&gt;&#39; może być wywołany z tymi argumentami bez użycia konwersji zawężającej: &lt;listy&gt;
Przeciążona metoda została wywołana, ale metoda nie można dopasować lista podanych argumentów bez użycia konwersji zawężającej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Określ `Option``Strict` `Off`.  
  
2.  Zmień argumentów pasują podpisu przeciążonej metody.
