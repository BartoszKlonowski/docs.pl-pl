# <a name="set-not-supported-at-run-time"></a>Zestaw nie jest obsługiwany w czasie wykonywania

Podjęto próbę ustawienia lub zmiany właściwości, której wartość można ustawić tylko w czasie projektowania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń odwołanie do właściwości z kodu.  
  
2. Zmień odwołanie, aby zwrócić wartość właściwości w czasie wykonywania.
