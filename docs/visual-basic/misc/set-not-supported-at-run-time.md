# <a name="set-not-supported-at-run-time"></a>Zestaw nie jest obsługiwane w czasie wykonywania
Próbowano ustawić lub zmienić właściwości, których wartość można ustawić tylko w czasie projektowania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń odwołanie do właściwości w kodzie.  
  
2.  Zmień odwołania, aby zwracać tylko wartości właściwości w czasie wykonywania.
