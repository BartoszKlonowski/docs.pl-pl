# <a name="events-in-a-module-cannot-be-declared-specifier"></a>Zdarzenia w Module nie można zadeklarować jako '\<specyfikator >'
Wykorzystano specyfikator, który nie jest prawidłowa na zdarzenia w obrębie `Module` instrukcji. Moduły nigdy nie mogą zostać utworzone, nie obsługują dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30434  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
