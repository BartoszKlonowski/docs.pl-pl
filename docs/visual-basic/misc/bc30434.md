# <a name="events-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Zdarzenia w Module nie można zadeklarować &#39; &lt;specyfikator&gt;&#39;
Zostały użyte specyfikator, który nie jest prawidłowa na zdarzenia w `Module` instrukcji. Moduły nigdy nie można utworzyć wystąpienia, nie obsługuje dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30434  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
