# <a name="events-in-a-module-cannot-be-declared-specifier"></a>Zdarzenia w module nie mogą być zadeklarowane jako " \<specifier> "

Użyto specyfikatora, który nie jest prawidłowy w zdarzeniu w `Module` instrukcji. Nie można tworzyć wystąpień modułów, nie obsługują dziedziczenia ani implementowania interfejsów.  
  
 **Identyfikator błędu:** BC30434  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
