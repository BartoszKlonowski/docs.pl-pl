<span data-ttu-id="e7222-103">Odwołanie do elementu podrzędnego XML odwołuje się do obiektu, który nie jest typu <xref:System.Xml.Linq.XElement> , <xref:System.Xml.Linq.XDocument> lub `IEnumerable(Of XElement)` .</span><span class="sxs-lookup"><span data-stu-id="e7222-103">An XML descendant has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="e7222-104">Aby uzyskać więcej informacji, zobacz [Właściwość osi elementu podrzędnego XML](../language-reference/xml-axis/xml-descendant-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="e7222-104">For more information, see [XML Descendant Axis Property](../language-reference/xml-axis/xml-descendant-axis-property.md).</span></span>  
  
```vb

Dim var = "sample text"...<childElement>  
```  
  
 <span data-ttu-id="e7222-105">**Identyfikator błędu:** BC36809</span><span class="sxs-lookup"><span data-stu-id="e7222-105">**Error ID:** BC36809</span></span>
