<span data-ttu-id="460d2-103">Obiekt podrzędny XML został wskazany dla obiektu, który nie jest typu <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, lub `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="460d2-103">An XML descendant has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="460d2-104">Aby uzyskać więcej informacji, zobacz [właściwości osi Descendant XML](../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="460d2-104">For more information, see [XML Descendant Axis Property](../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md).</span></span>  
  

 <span data-ttu-id="460d2-105">**Identyfikator błędu:** BC36809</span><span class="sxs-lookup"><span data-stu-id="460d2-105">**Error ID:** BC36809</span></span>
