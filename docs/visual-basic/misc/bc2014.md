# <a name="the-value-value-is-invalid-for-option-optionname"></a>wartość "\<wartość >" jest nieprawidłowa dla opcji "\<optionname >"
Podano wartości dla opcji wiersza polecenia, która nie jest prawidłowa.  
  
 **Identyfikator błędu:** BC2014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź wartości, aby znaleźć ten, który jest nieprawidłowy.
