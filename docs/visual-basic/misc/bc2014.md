# <a name="the-value-value-is-invalid-for-option-optionname"></a>wartość " \<value> " jest nieprawidłowa dla opcji " \<optionname> "

Podano wartość dla opcji wiersza polecenia, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC2014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź wartości, aby znaleźć te, które nie są prawidłowe.
