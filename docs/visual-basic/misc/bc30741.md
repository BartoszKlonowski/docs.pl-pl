# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>"&lt;typename1&gt;"nie można przekonwertować na"&lt;typename2&gt;"
Konwersja typu podjęto próbę między dwoma typami niezgodne.  
  
 **Identyfikator błędu:** BC30741  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź typy, które są rzutowany.
