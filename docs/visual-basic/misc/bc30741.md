# <a name="typename1-cannot-be-converted-to-typename2"></a>'\<typename1 >' nie można przekonwertować na '\<typename2 >'
Konwersja typu podjęto próbę między dwoma typami niezgodne.  
  
 **Identyfikator błędu:** BC30741  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy, które są rzutowany.
