# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39;&lt;membername1&gt; &#39; nie może zastąpić &#39; &lt;membername2&gt; &#39; , ponieważ różnią się one typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jednego z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30697  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy zdefiniować co najmniej jeden wymagany parametr jako inny typ danych.
