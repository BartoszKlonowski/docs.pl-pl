# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>element " \<membername1> " nie może przesłonić elementu " \<membername2> ", ponieważ różnią się one typami parametrów opcjonalnych.
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jednego wymaganego parametru.  
  
 **Identyfikator błędu:** BC30697  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj co najmniej jeden wymagany parametr jako inny typ danych.
