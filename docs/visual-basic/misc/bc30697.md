# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;nie może przesłonić&lt;membername2&gt;", ponieważ różnią się typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jeden z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30697  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jako innego typu danych, należy zdefiniować co najmniej jeden parametr wymagany.
