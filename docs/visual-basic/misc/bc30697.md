# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>'\<membername1 >' nie może przesłonić'\<membername2 > ', ponieważ różnią się typami parametrów opcjonalnych
Przeciążone elementy członkowskie muszą mieć różne typy danych dla co najmniej jeden z wymaganych parametrów.  
  
 **Identyfikator błędu:** BC30697  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jako innego typu danych, należy zdefiniować co najmniej jeden parametr wymagany.
