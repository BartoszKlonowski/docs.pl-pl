<span data-ttu-id="fd2ae-103">Nie można zainicjować wystąpienia programu `Object` przy użyciu składni inicjatora obiektów.</span><span class="sxs-lookup"><span data-stu-id="fd2ae-103">You cannot initialize an instance of `Object` by using object initializer syntax.</span></span> <span data-ttu-id="fd2ae-104">Wystąpienie nie `Object` ma właściwości ani pól, do których można przypisać wartość, a składnia inicjatora obiektów wymaga co najmniej jednej takiej właściwości lub pola.</span><span class="sxs-lookup"><span data-stu-id="fd2ae-104">An instance of `Object` has no properties or fields to assign a value to, and object initializer syntax requires at least one such property or field.</span></span>

```vb  
' Dim obj1 = New Object With {}  
' Dim obj2 = New Object With {.ToString = <some value>}  
```
