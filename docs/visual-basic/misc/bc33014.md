# <a name="operator-39ltoperatorgt39-must-have-one-parameter"></a>Operator &#39; &lt;operator&gt;&#39; musi mieć jeden parametr
Operator jednoargumentowy zdefiniowano bez parametrów lub z więcej niż jeden parametr.  
  
 Operator jednoargumentowy musi mieć dokładnie jeden parametr.  
  
 **Identyfikator błędu:** BC33014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostosuj definicji, aby określić dokładnie jeden parametr.  
  
-   Jeśli potrzebne są dwa parametry, należy zdefiniować operator binarny.  
  
-   Jeśli potrzebujesz bez parametrów lub większa niż dwa, należy użyć [instrukcji Function](../../visual-basic/language-reference/statements/function-statement.md) do definiowania `Function` procedury zamiast operatora.
