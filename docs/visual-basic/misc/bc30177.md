# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Można określić tylko jedną z elementów "NotOverridable", "MustOverride" lub "Zastąp"

Sprzeczne specyfikacje przesłaniania występują w deklaracji elementu. Nadmiarowe słowa kluczowe wykluczają się wzajemnie.  
  
 **Identyfikator błędu:** BC30177  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz jeden z zastępujący specyfikator i Usuń inne słowa kluczowe z deklaracji.
