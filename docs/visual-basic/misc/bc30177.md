# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Można określić tylko jedną "NotOverridable", "MustOverride" lub "Overridable"
Konflikt pomijania specyfikacji występować w deklaracji elementu. Przesłanianie słowa kluczowe są wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30177  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz jeden specyfikator nadrzędne i Usuń innych słów kluczowych z deklaracji.
