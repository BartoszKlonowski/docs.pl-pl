# <a name="39lteventnamegt39-is-not-an-event-of-39ltcontainernamegt39"></a>&#39; &lt;eventname&gt;&#39; nie jest zdarzeniem elementu &#39;&lt; Właściwość ContainerName&gt;&#39;
Nie zadeklarowano zdarzeń określony dla obiektu.  
  
 **Identyfikator błędu:** BC30676  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię nazwy zdarzenia.  
  
2.  Sprawdź, czy uzyskują dostęp do odpowiedni obiekt. Użyj w pełni kwalifikowanego odwołania do obiektu lub `Imports` instrukcji, aby zaimportować odpowiedni obszar nazw, w razie potrzeby.
