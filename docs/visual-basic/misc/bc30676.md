# <a name="eventname-is-not-an-event-of-containername"></a>" \<eventname> " nie jest zdarzeniem elementu " \<containername> "

Określone zdarzenie nie jest zadeklarowane dla obiektu.  
  
 **Identyfikator błędu:** BC30676  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię nazwy zdarzenia.  
  
2. Upewnij się, że uzyskujesz dostęp do poprawnego obiektu. Użyj w pełni kwalifikowanych odwołań do obiektu lub Użyj `Imports` instrukcji w celu zaimportowania odpowiedniej przestrzeni nazw w razie potrzeby.
