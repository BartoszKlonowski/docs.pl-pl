# <a name="lteventnamegt-is-not-an-event-of-ltcontainernamegt"></a>"&lt;eventname&gt;"nie jest zdarzeniem elementu"&lt;containername&gt;"
Nie zadeklarowano zdarzeń określony dla obiektu.  
  
 **Identyfikator błędu:** BC30676  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię nazwy zdarzenia.  
  
2.  Sprawdź, czy uzyskują dostęp do odpowiedni obiekt. Użyj w pełni kwalifikowanego odwołania do obiektu lub `Imports` instrukcję, aby zaimportować odpowiedniego obszaru nazw, jeśli to konieczne.
