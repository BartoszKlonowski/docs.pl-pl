# <a name="anonymous-type-member-or-property-39ltpropertynamegt39-is-already-declared"></a><span data-ttu-id="7700a-102">Właściwość lub członek typu anonimowego &#39; &lt;propertyname&gt; &#39; jest już zadeklarowana</span><span class="sxs-lookup"><span data-stu-id="7700a-102">Anonymous type member or property &#39;&lt;propertyname&gt;&#39; is already declared</span></span>
<span data-ttu-id="7700a-103">Nazwa właściwości można ustanowić tylko raz w deklaracji typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="7700a-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="7700a-104">Na przykład następujące deklaracje nie są prawidłowe:</span><span class="sxs-lookup"><span data-stu-id="7700a-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="7700a-105">**Identyfikator błędu:** BC36547</span><span class="sxs-lookup"><span data-stu-id="7700a-105">**Error ID:** BC36547</span></span>
