# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="4f281-102">Składowa lub właściwość typu anonimowego '\<propertyname >' jest już zadeklarowana</span><span class="sxs-lookup"><span data-stu-id="4f281-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="4f281-103">Nazwa właściwości może zostać nawiązana tylko raz w deklaracji typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="4f281-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="4f281-104">Na przykład następujące deklaracje nie są prawidłowe:</span><span class="sxs-lookup"><span data-stu-id="4f281-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="4f281-105">**Identyfikator błędu:** BC36547</span><span class="sxs-lookup"><span data-stu-id="4f281-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4f281-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="4f281-106">To correct this error</span></span>  
  
-   <span data-ttu-id="4f281-107">Wybierz inną nazwę dla jednej właściwości.</span><span class="sxs-lookup"><span data-stu-id="4f281-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.

```  
  
-   <span data-ttu-id="4f281-108">Podaj nowe nazwy zmiennych lub nazw właściwości, z których są wnioskowanie nazw i wartości.</span><span class="sxs-lookup"><span data-stu-id="4f281-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
