# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="f5fcd-102">Składowa lub właściwość typu anonimowego "&lt;propertyname&gt;" jest już zadeklarowana</span><span class="sxs-lookup"><span data-stu-id="f5fcd-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="f5fcd-103">Nazwa właściwości może zostać nawiązana tylko raz w deklaracji typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="f5fcd-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="f5fcd-104">Na przykład następujące deklaracje nie są prawidłowe:</span><span class="sxs-lookup"><span data-stu-id="f5fcd-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="f5fcd-105">**Identyfikator błędu:** BC36547</span><span class="sxs-lookup"><span data-stu-id="f5fcd-105">**Error ID:** BC36547</span></span>

```vb

```

```vb  
' Valid.

```
