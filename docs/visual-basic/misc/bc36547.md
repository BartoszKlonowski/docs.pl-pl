# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="a92ea-102">Składowa lub właściwość typu anonimowego " \<propertyname> " jest już zadeklarowana</span><span class="sxs-lookup"><span data-stu-id="a92ea-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="a92ea-103">Nazwę właściwości można ustalić tylko raz w deklaracji typu anonimowego.</span><span class="sxs-lookup"><span data-stu-id="a92ea-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="a92ea-104">Na przykład następujące deklaracje są nieprawidłowe:</span><span class="sxs-lookup"><span data-stu-id="a92ea-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="a92ea-105">**Identyfikator błędu:** BC36547</span><span class="sxs-lookup"><span data-stu-id="a92ea-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a92ea-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a92ea-106">To correct this error</span></span>  
  
- <span data-ttu-id="a92ea-107">Wybierz inną nazwę dla jednej z właściwości.</span><span class="sxs-lookup"><span data-stu-id="a92ea-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="a92ea-108">Podaj nowe nazwy zmiennych lub nazw właściwości, z których mają zostać wywnioskowane nazwy i wartości.</span><span class="sxs-lookup"><span data-stu-id="a92ea-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
