# <a name="relational-operator-expected"></a>Oczekiwano operatora relacyjnego
`Case`Instrukcja zawiera `Is` klauzulę, ale nie ma operatora porównania, takiego jak `=` lub `>` . Jeśli `Case` instrukcja nie zawiera operatora, `=` przyjmuje się i `Is` nie jest używana.  
  
 **Identyfikator błędu:** BC30239  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Is` słowo kluczowe lub postępuj zgodnie z operatorem porównania.
