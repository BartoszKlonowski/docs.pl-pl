# <a name="class-classname-could-not-be-created-error"></a>Nie można \<classname> utworzyć klasy "": \<error>

Inicjowanie klasy nie powiodło się. Jedną z możliwych przyczyn jest uszkodzona instalacja .NET Framework.  
  
 **Identyfikator błędu:** BC32400  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Przejrzyj cytowany komunikat o błędzie i podejmij odpowiednie działanie.  
  
2. Zainstaluj ponownie program .NET Framework, a następnie wykonaj ponowną kompilację programu.  
  
3. Jeśli błąd powtarza się, zainstaluj ponownie kompilator Visual Basic.  
  
4. Jeśli błąd będzie się powtarzać po ponownej instalacji, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
