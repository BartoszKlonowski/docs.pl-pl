# <a name="class-ltclassnamegt-could-not-be-created-lterrorgt"></a>Klasy&lt;classname&gt;"nie można utworzyć: &lt;błąd&gt;
Inicjowanie klasy nie powiodła się. Jedną z możliwych przyczyn jest uszkodzony [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] instalacji.  
  
 **Identyfikator błędu:** BC32400  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź komunikat o błędzie w cudzysłowach i podjąć odpowiednie działania.  
  
2.  Zainstaluj ponownie [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] i ponownie skompilować program.  
  
3.  Jeśli ten błąd wystąpi, zainstaluj ponownie kompilator Visual Basic.  
  
4.  Jeśli ten błąd będzie występował po ponownej instalacji, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
