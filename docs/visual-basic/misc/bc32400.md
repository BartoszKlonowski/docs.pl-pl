# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>Klasa &#39; &lt;classname&gt;&#39; nie można utworzyć: &lt;błąd&gt;
Inicjowanie klasy nie powiodła się. Jedną z możliwych przyczyn jest to uszkodzony [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] instalacji.  
  
 **Identyfikator błędu:** BC32400  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź komunikat błędu w cudzysłowie i podejmij odpowiednią akcję.  
  
2.  Zainstaluj ponownie [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] i skompiluj ponownie ten program.  
  
3.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilatora.  
  
4.  Jeśli błąd będzie nadal występować po ponownej instalacji, zebrać informacje dotyczące okoliczności i powiadomić pomocy technicznej firmy Microsoft.
