# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>Właściwość &lt;propertyname&gt; nie można ustawić na wartość Nothing
Właściwość wymaga wartości innej niż `Nothing`. Może to być wynikiem niepoprawnie obliczania wartości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłową wartość.
