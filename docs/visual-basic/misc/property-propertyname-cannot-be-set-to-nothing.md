# <a name="property-propertyname-cannot-be-set-to-nothing"></a>Właściwość \<propertyname> nie może być ustawiona na wartość Nothing
Właściwość wymaga wartości innej niż `Nothing` . Może to być spowodowane błędnym obliczaniem wartości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłową wartość.
