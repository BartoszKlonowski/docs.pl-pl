# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Case" nie są prawidłowe w oknie bezpośrednim
instrukcji `Case` można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30719  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcje `Case` z okna **bezpośredniego** .
