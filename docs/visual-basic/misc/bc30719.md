# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Przypadek&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Case` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30719  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Case` oświadczenia **Immediate** okna.
