# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Literały XML i właściwości osi XML nie są dostępne podczas tej sesji debugowania, ponieważ nie są używane w tym zestawie
Literał XML i właściwości osi XML został przywołany w **czujki** lub **Immediate** okno podczas sesji debugowania, w których XML w [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] funkcje są niedostępne. Jest to w przypadku zestawu, który nie używa albo kod XML w [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] funkcji lub jest kompilacji wydania.  
  
 **Identyfikator błędu:** BC31196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Rozpocznij nową sesję debugowania korzystanie z kompilacji debugowania zestawu.
