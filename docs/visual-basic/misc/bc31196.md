# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Literały XML i właściwości osi XML nie są dostępne podczas tej sesji debugowania, ponieważ nie są używane w tym zestawie
Literał XML i właściwości osi XML został przywołany w **czujki** lub **Immediate** okno podczas sesji debugowania, w których XML w Visual Basic funkcje nie są dostępne. Jest to w przypadku zestawu, która nie używa pliku XML w funkcji języka Visual Basic lub kompilacji wydania.  
  
 **Identyfikator błędu:** BC31196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Rozpocznij nową sesję debugowania korzystanie z kompilacji debugowania zestawu.
