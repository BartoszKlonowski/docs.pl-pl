# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Literały XML i właściwości osi XML nie są dostępne podczas tej sesji debugowania, ponieważ nie są one używane w tym zestawie.
Literał XML i właściwości osi XML został wskazany w **Obejrzyj** lub **bezpośrednie** okna podczas sesji debugowania, w których XML w Visual Basic nie są dostępne funkcje. Jest to w przypadku zestawu, która nie używa pliku XML w funkcji języka Visual Basic lub kompilację wydania.  
  
 **Identyfikator błędu:** BC31196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozpocznij nową sesję debugowania, korzystanie z kompilacji debugowania zestawu.
