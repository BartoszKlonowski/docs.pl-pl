# <a name="class-classname-must-declare-a-sub-new-because-its-base-class-classname2-does-not-have-an-accessible-sub-new-that-can-be-called-with-no-arguments"></a><span data-ttu-id="241b7-102">Klasa " \<classname> " musi deklarować element "Sub New", ponieważ jej Klasa bazowa " \<classname2> " nie ma dostępnego elementu "Sub New", który można wywołać bez argumentów</span><span class="sxs-lookup"><span data-stu-id="241b7-102">Class '\<classname>' must declare a 'Sub New' because its base class '\<classname2>' does not have an accessible 'Sub New' that can be called with no arguments</span></span>

<span data-ttu-id="241b7-103">Klasa pochodna nie deklaruje konstruktora, a Visual Basic nie może wygenerować tego elementu, ponieważ nie istnieje Konstruktor klasy bazowej, który może zostać wywołany.</span><span class="sxs-lookup"><span data-stu-id="241b7-103">A derived class does not declare a constructor, and Visual Basic cannot generate one because there is no base class constructor it can call.</span></span>  
  
 <span data-ttu-id="241b7-104">Gdy Klasa pochodna nie deklaruje konstruktora, Visual Basic próbuje wygenerować niejawnego konstruktora bez parametrów, który wywołuje `MyBase.New()` .</span><span class="sxs-lookup"><span data-stu-id="241b7-104">When a derived class does not declare a constructor, Visual Basic attempts to generate an implicit parameterless constructor that calls `MyBase.New()`.</span></span> <span data-ttu-id="241b7-105">Jeśli w klasie bazowej nie ma dostępnego konstruktora, który można wywołać bez argumentów lub jeśli istnieje więcej niż jeden, Visual Basic nie może wygenerować niejawnego konstruktora.</span><span class="sxs-lookup"><span data-stu-id="241b7-105">If there is no accessible constructor in the base class that can be called without arguments, or if there is more than one, Visual Basic cannot generate an implicit constructor.</span></span>  
  
 <span data-ttu-id="241b7-106">**Identyfikator błędu:** BC30387</span><span class="sxs-lookup"><span data-stu-id="241b7-106">**Error ID:** BC30387</span></span>
