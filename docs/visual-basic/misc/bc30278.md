# <a name="sub-or-function-expected-after-delegate"></a>"Sub" lub "Function" po "Delegowanie"
A `Delegate` instrukcji nie określa `Sub` lub `Function` procedury. `Sub` Lub `Function` — słowo kluczowe należy natychmiast wykonać `Delegate` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC30278  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `Sub` lub `Function` — słowo kluczowe natychmiast po `Delegate`.  
  
2.  Określ nazwę procedury, lista argumentów, a zwracany typ zgodnie z potrzebami.
