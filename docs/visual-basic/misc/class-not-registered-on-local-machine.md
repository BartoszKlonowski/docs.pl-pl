# <a name="class-not-registered-on-local-machine"></a>Klasa nie jest zarejestrowana na komputerze lokalnym
Ten błąd jest zwykle spowodowany uszkodzonym plikiem. Może być konieczne ponowne zainstalowanie programu Visual Studio.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zainstaluj ponownie program Visual Studio.
