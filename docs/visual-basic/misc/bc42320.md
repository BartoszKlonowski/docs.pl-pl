# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>Nie można dołączyć fragmentu XML " \<elementpath> " pliku " \<commentfile> "

`<include>`Tag w komentarzu XML zawiera wyrażenie XPath dla nieistniejącego elementu lub plik zawiera nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42320  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy kod XML w pliku jest prawidłowy.  
  
2. Sprawdź, czy fragment kodu XML istnieje w pliku.
