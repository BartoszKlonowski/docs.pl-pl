# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>Nie można dołączyć fragmentu XML &#39; &lt;elementpath&gt;&#39; plik &#39;&lt; commentfile&gt;&#39;
`<include>` Tag w komentarzu XML zawiera wyrażenie XPath dla elementu, który nie istnieje, lub plik ma nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42320  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź poprawność kodu XML w pliku.  
  
2.  Sprawdź, czy w pliku znajduje się XML fragment.
