# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>Nie można dołączyć fragmentu XML '\<tagu elementpath >' pliku '\<commentfile >'
`<include>` Tagu w komentarz XML zawiera wyrażenie XPath dla elementu, który nie istnieje lub plik zawiera nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42320  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy kod XML w pliku jest prawidłowa.  
  
2. Sprawdź, czy w pliku znajduje się XML fragment.
