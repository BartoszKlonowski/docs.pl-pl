# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>Element "End Structure" musi być poprzedzony odpowiadającą jej instrukcją "Structure"

`End Structure`Instrukcja może być używana tylko na końcu bloku struktury, który rozpoczyna się od `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30621  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że bieżący blok struktury zaczyna się od `Structure` instrukcji.
