# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>"End struktury" musi być poprzedzona pasującego "Structure"
`End Structure` Instrukcji należy używać tylko na końcu bloku struktury, który rozpoczyna się od `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30621  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że bieżący blok struktury zaczyna się od `Structure` instrukcji.
