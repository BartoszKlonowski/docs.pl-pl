# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39; End Structure &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; Struktura &#39;
`End Structure` Instrukcji można użyć tylko na końcu bloku struktury zaczyna się od `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30621  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że bieżący blok struktury rozpoczyna się od `Structure` instrukcji.
