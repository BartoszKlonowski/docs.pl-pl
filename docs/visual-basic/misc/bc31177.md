# <a name="white-space-may-not-appear-here"></a>Biały znak może nie pojawić się w tym miejscu
Spacja lub spacje są zawarte w nazwie literału elementu XML.  
  
 **Identyfikator błędu:** BC31177  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń spację lub spacje z nazwy literału elementu XML.
