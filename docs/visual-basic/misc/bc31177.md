# <a name="white-space-may-not-appear-here"></a>Biały znak nie może tutaj występować
Spację lub spacje są ujęte w nazwie literał elementu XML.  
  
 **Identyfikator błędu:** BC31177  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń miejsca ani spacji z nazwy literał elementu XML.
