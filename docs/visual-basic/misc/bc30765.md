# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Wszystkie projekty w rozwiązaniu Visual Basic muszą wskazywać tę samą platformę, ale projekt, który próbujesz dodać elementy docelowe platformy inną niż określona w tym rozwiązaniu
Wszystkie projekty w [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] rozwiązanie musi wskazywać na tę samą platformę, ale w przypadku dodawania elementem docelowym projektu jest platformy inną niż określona w tym rozwiązaniu.  
  
 **Identyfikator błędu:** BC30765  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj tylko projekty, które odnoszą się do tej samej platformy określona w tym rozwiązaniu.
