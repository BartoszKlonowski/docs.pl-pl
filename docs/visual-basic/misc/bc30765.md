# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Wszystkie projekty w rozwiązaniu Visual Basic musi być przeznaczony dla tej samej platformy, ale projekt, który próbujesz dodać jest przeznaczony dla platform innych niż zestaw określony przez rozwiązania
Wszystkie projekty w rozwiązaniu Visual Basic musi być przeznaczony dla tej samej platformy, ale projekt dodajesz platform innych niż ten, który określono elementów docelowych w rozwiązaniu.  
  
 **Identyfikator błędu:** BC30765  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj tylko projektów przeznaczonych dla tej samej platformy, określona w tym rozwiązaniu.
