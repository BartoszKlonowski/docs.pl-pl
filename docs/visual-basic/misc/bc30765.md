# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Wszystkie projekty w rozwiązaniu Visual Basic muszą mieć tę samą platformę, ale projekt, który próbujesz dodać, jest obiektem innym niż określony przez rozwiązanie

Wszystkie projekty w rozwiązaniu Visual Basic muszą być zgodne z tą samą platformą, ale dodawany projekt jest przeznaczony dla platformy innej niż określona przez rozwiązanie.  
  
 **Identyfikator błędu:** BC30765  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj tylko projekty, które są przeznaczone dla tej samej platformy określonej przez rozwiązanie.
