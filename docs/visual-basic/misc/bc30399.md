# <a name="mybase-cannot-be-used-with-method-ltmethodnamegt-because-it-is-declared-mustoverride"></a>Instrukcja "MyBase" nie można używać z metodą "&lt;methodname&gt;", ponieważ jest on zadeklarowany jako "MustOverride"
Podjęto próbę użycia `MyBase` z metodą, która została zadeklarowana `MustOverride`.  
  
 **Identyfikator błędu:** BC30399  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `MustOverride` deklaracji.
