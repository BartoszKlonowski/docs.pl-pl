# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>Nie można użyć elementu "MyBase" z metodą " \<methodname> ", ponieważ jest ona zadeklarowana jako "MustOverride"

Podjęto próbę użycia `MyBase` z metodą, która została zadeklarowana `MustOverride` .  
  
 **Identyfikator błędu:** BC30399  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `MustOverride` deklarację.
