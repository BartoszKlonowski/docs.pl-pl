# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39; nie może być użyta z metodą &#39; &lt;methodname&gt; &#39; , ponieważ jest on zadeklarowany jako &#39;MustOverride&#39;
Podjęto próbę użycia `MyBase` za pomocą metody, która została zadeklarowana `MustOverride`.  
  
 **Identyfikator błędu:** BC30399  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `MustOverride` deklaracji.
