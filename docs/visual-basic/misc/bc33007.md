# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39; End — Operator &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; Operator &#39;
`End Operator` Występuje instrukcja z pasujących `Operator` deklaracji przed nim.  
  
 **Identyfikator błędu:** BC33007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `End Operator` instrukcję, jeśli jest nadmiarowy.  
  
-   Podaj brakujący `Operator` procedura w przypadku braku takiego.  
  
-   Przenieś `End Operator` instrukcji w odpowiednie miejsce w kodzie.
