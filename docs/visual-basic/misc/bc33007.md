# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>"End Operator" musi być poprzedzona pasującego 'Operator'
`End Operator` Występuje instrukcja nie słów `Operator` deklaracji przed nim.  
  
 **Identyfikator błędu:** BC33007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `End Operator` instrukcji, jeśli jest nadmiarowy.  
  
-   Podaj brakujące `Operator` procedury, jeśli brakuje jednego.  
  
-   Przenieś `End Operator` instrukcji w odpowiednie miejsce w kodzie.
