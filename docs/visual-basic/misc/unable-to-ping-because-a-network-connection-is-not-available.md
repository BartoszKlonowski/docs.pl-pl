# <a name="unable-to-ping-because-a-network-connection-is-not-available"></a>Nie można wysłać polecenia ping, ponieważ połączenie sieciowe jest niedostępne

Nie można wykonać operacji, ponieważ nie jest dostępne żadne połączenie sieciowe.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nawiąż połączenie z siecią i ponów próbę wykonania operacji.
