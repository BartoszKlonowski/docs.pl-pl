# <a name="39readonly39-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="e6015-102">&#39;Tylko do odczytu&#39; zmiennej nie może być elementem docelowym przypisania w wyrażeniu lambda wewnątrz konstruktora</span><span class="sxs-lookup"><span data-stu-id="e6015-102">&#39;ReadOnly&#39; variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>
<span data-ttu-id="e6015-103">Ma określone `ReadOnly` zmienną z wewnątrz wyrażenia lambda, co jest niedozwolone.</span><span class="sxs-lookup"><span data-stu-id="e6015-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="e6015-104">Poniższy kod powoduje, że ten błąd, wysyłając `ReadOnly` zmiennej `m` jako argument `ByRef` parametru.</span><span class="sxs-lookup"><span data-stu-id="e6015-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="e6015-105">**Identyfikator błędu:** BC36602</span><span class="sxs-lookup"><span data-stu-id="e6015-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e6015-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="e6015-106">To correct this error</span></span>  
  
-   <span data-ttu-id="e6015-107">Jeśli to możliwe, Zmień parametr `n` w funkcji `Test` do `ByVal` parametru, jak pokazano w poniższym kodzie.</span><span class="sxs-lookup"><span data-stu-id="e6015-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e6015-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="e6015-108">To correct this error</span></span>  
  
-   <span data-ttu-id="e6015-109">Przypisz `ReadOnly` zmiennej `m` do zmiennej lokalnej w Konstruktorze i użyć zmiennej lokalnej zamiast `m`, jak pokazano w poniższym kodzie.</span><span class="sxs-lookup"><span data-stu-id="e6015-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
