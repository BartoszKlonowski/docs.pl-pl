# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a>Zmienna "ReadOnly" nie może być elementem docelowym przypisania w wyrażeniu lambda wewnątrz konstruktora
Użytkownik odwołuje się do `ReadOnly` zmiennej z wyrażenia lambda, co jest niedozwolone. Poniższy kod powoduje błąd przez wysłanie `ReadOnly` zmiennej `m` `ByRef` jako argumentu do parametru.  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 **Identyfikator błędu:** BC36602  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli to możliwe, Zmień `n` parametr w `Test` funkcji na `ByVal` parametr, jak pokazano w poniższym kodzie.  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `ReadOnly` Przypisz zmienną `m` do zmiennej lokalnej w konstruktorze i Użyj zmiennej `m`lokalnej zamiast, jak pokazano w poniższym kodzie.  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
