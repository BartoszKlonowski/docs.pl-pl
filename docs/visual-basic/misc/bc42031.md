# <a name="39catch39-block-never-reached-ltexceptiongt-handled-above-in-the-same-39try39-statement"></a>&#39;CATCH&#39; blok nie jest nigdy osiągany; &lt;wyjątek&gt; obsługiwany powyżej w tym samym &#39;spróbuj&#39; — instrukcja
A `Catch` bloku w kodzie nie można nawiązać połączenia, ponieważ odbywa się w poprzednich `Try` bloku.  
  
 Domyślnie ta wiadomość jest ostrzeżenie. Aby uzyskać więcej informacji na temat ukrywania ostrzeżenia lub traktowanie ostrzeżeń jako błędy, zobacz [Konfigurowanie ostrzeżeń w języku Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)  
  
 **Identyfikator błędu:** BC42031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń instrukcję nadmiarową.
