# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent&#39; deklaracja musi być zakończona odpowiadającą jej instrukcją &#39;End RaiseEvent&#39;
A `RaiseEvent` deklaracji musi kończyć się `End RaiseEvent` instrukcji.  
  
 **Identyfikator błędu:** BC31117  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `RaiseEvent` kończy deklaracji `End RaiseEvent` instrukcji.
