# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>Deklaracja "RaiseEvent" musi być zakończona odpowiadającą jej instrukcją "End RaiseEvent"
`RaiseEvent`Deklaracja musi kończyć się `End RaiseEvent` instrukcją.  
  
 **Identyfikator błędu:** BC31117  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `RaiseEvent` Deklaracja została zakończona `End RaiseEvent` instrukcją.
