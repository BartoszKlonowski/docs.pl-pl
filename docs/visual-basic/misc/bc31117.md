# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>Deklaracja "RaiseEvent" musi kończyć się pasującego "End RaiseEvent"
A `RaiseEvent` deklaracji musi się kończyć `End RaiseEvent` instrukcji.  
  
 **Identyfikator błędu:** BC31117  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `RaiseEvent` deklaracji kończy się `End RaiseEvent` instrukcji.
