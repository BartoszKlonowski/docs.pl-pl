# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Błąd osadzanie manifestu Win32: Plik manifestu &lt;pliku&gt; nie można odnaleźć
Plik określony dla `/win32manifest` — opcja kompilatora nie można odnaleźć.  
  
 **Identyfikator błędu:** BC31192  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Podaj prawidłową nazwę pliku i ścieżki dla `/win32manifest` — opcja kompilatora.
