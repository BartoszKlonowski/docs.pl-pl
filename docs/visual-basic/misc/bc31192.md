# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Błąd podczas osadzania manifestu Win32: nie można odnaleźć pliku manifestu \<file >
Nie można znaleźć pliku określonego dla opcji kompilatora `/win32manifest`.  
  
 **Identyfikator błędu:** BC31192  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Podaj prawidłową nazwę pliku i ścieżkę dla opcji kompilatora `/win32manifest`.
