# <a name="loop-control-variable-of-type-39lttypenamegt39-does-not-implement-the-39systemicomparable39-interface"></a>Pętla zmienna sterująca typu &#39; &lt;typename&gt;&#39; nie implementuje &#39; System.IComparable &#39; Interfejs
Zmienna jest typu, który nie implementuje `System.IComparable` interfejsu. `System.IComparable`musi być implementowana przez typ wartości lub klasą.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że zmienna jest typem wartości.
