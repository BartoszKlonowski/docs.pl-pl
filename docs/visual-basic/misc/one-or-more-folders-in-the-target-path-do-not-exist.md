# <a name="one-or-more-folders-in-the-target-path-do-not-exist"></a>Jeden lub więcej folderów w ścieżce docelowej nie istnieje.
Co najmniej jeden katalogi w ścieżce docelowej nie istnieje. Może to być wynikiem ścieżką nieprawidłowo przeanalizowane, szczególnie w przypadku, jeśli dostarczenie ścieżką względną.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, upewnij się, że są podając ścieżkę docelową poprawne.
