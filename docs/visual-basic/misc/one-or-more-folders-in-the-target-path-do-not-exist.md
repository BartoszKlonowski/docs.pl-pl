# <a name="one-or-more-folders-in-the-target-path-do-not-exist"></a>Co najmniej jeden folder w ścieżce docelowej nie istnieje

Co najmniej jeden z katalogów dostarczonych w ścieżce docelowej nie istnieje. Może to wynikać z nieprawidłowo przeanalizowanej ścieżki, szczególnie w przypadku podawania ścieżki względnej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że dostarczono poprawną ścieżkę docelową.
