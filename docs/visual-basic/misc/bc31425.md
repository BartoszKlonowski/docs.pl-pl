<span data-ttu-id="27b14-103">Zestaw używa <xref:System.Runtime.CompilerServices.TypeForwardedToAttribute> do przesyłania dalej jednego z typów do innego zestawu, ale określa ten sam typ w tym samym zestawie.</span><span class="sxs-lookup"><span data-stu-id="27b14-103">An assembly uses the <xref:System.Runtime.CompilerServices.TypeForwardedToAttribute> to forward one of its types to another assembly, but specifies the same type in the same assembly.</span></span>  
  
 <span data-ttu-id="27b14-104">*Przekazywanie typu* oznacza ponowne przypisanie definicji klasy, struktury, interfejsu, delegata lub wyliczenia do zestawu innego niż ten, w którym został pierwotnie zdefiniowany.</span><span class="sxs-lookup"><span data-stu-id="27b14-104">*Type forwarding* means reassigning the definition of a class, structure, interface, delegate, or enumeration to an assembly other than the one in which it was originally defined.</span></span> <span data-ttu-id="27b14-105">Jest on często używany w połączeniu z *refaktoryzacją kodu*, dzięki czemu można podzielić zestaw na dwa lub więcej zestawów lub przenieść kod z jednego zestawu do drugiego.</span><span class="sxs-lookup"><span data-stu-id="27b14-105">It is often used in conjunction with *code refactoring*, by which you split an assembly into two or more assemblies or move code from one assembly to another.</span></span>
