# <a name="unused-local-constant-constant"></a>Nieużywana stała lokalna: '\<stała >'
Stała lokalna został zadeklarowany, który nie jest używany wszędzie w kodzie.  
  
 **Identyfikator błędu:** BC42099  
  
### <a name="to-remove-this-warning"></a>Aby usunąć to ostrzeżenie  
  
1. Ten błąd, ostrzeżenie i można je zignorować. Aby usunąć to ostrzeżenie, należy usunąć z kodu nieużywanych deklaracji stałej.
