# <a name="unused-local-constant-constant"></a>Nieużywana stała lokalna: " \<constant> "

Została zadeklarowana stała lokalna, która nie jest używana w dowolnym miejscu w kodzie.  
  
 **Identyfikator błędu:** BC42099  
  
### <a name="to-remove-this-warning"></a>Aby usunąć to ostrzeżenie  
  
1. Ten błąd jest ostrzeżeniem i można go zignorować. Aby usunąć ostrzeżenie, Usuń nieużywaną deklarację stałą z kodu.
