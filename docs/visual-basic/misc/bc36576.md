<span data-ttu-id="2ab00-103">Wartość, w którym nazwa właściwości nie można wywnioskować została uwzględniona na liście inicjatora w deklaracji typu anonimowego nie są przypisane do nazwy właściwości.</span><span class="sxs-lookup"><span data-stu-id="2ab00-103">A value from which a property name cannot be inferred has been included in the initializer list of an anonymous type declaration without being assigned to a property name.</span></span>
