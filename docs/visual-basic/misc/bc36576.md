<span data-ttu-id="f2f45-103">Wartość, z której nie można wywnioskować nazwy właściwości, została uwzględniona na liście inicjatorów deklaracji typu anonimowego bez przypisywania do nazwy właściwości.</span><span class="sxs-lookup"><span data-stu-id="f2f45-103">A value from which a property name cannot be inferred has been included in the initializer list of an anonymous type declaration without being assigned to a property name.</span></span>
