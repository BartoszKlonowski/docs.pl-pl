# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>"On Error" instrukcje nie są prawidłowe w oknie bezpośrednim
`On Error` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30720  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `On Error` instrukcji debugowania kodu.
