# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;W przypadku błędu&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`On Error` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30720  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `On Error` oświadczenia debugowania kodu.
