# <a name="39lttypenamegt39-is-a-class-type-and-cannot-be-used-as-an-expression"></a>&#39; &lt;typename&gt;&#39; jest typ klasy i nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wymagane jest wyrażenie. Wyrażenie musi składać się z kombinacji zmienne, stałe, literały, właściwości, oraz `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30109  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłową.
