# <a name="expected-closing--for-embedded-expression"></a>Oczekiwano zamykającego "% >" dla wyrażenia osadzonego

W wyrażeniu osadzonym w literale XML brakuje wymaganych znaków zamykających `%>` .  
  
 **Identyfikator błędu:** BC31159  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagane znaki zamykające `%>` na końcu wyrażenia osadzonego.
