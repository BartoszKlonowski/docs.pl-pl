# <a name="expected-closing--for-embedded-expression"></a>Oczekiwano zamykającego "% >" dla wyrażenia osadzone
Wyrażenia osadzone w literał XML brakuje zamykającego wymagane `%>` znaków.  
  
 **Identyfikator błędu:** BC31159  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamknięcia `%>` znaki do końca osadzone wyrażenia.
