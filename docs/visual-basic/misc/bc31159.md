# <a name="expected-closing-39gt39-for-embedded-expression"></a>Oczekiwano zamykającego &#39; %&gt;&#39; dla wyrażenia osadzonego
Wyrażenia osadzone w literał XML nie ma nawiasu zamykającego wymagane `%>` znaków.  
  
 **Identyfikator błędu:** BC31159  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamknięcia `%>` znaki na końcu wyrażenia osadzonego.
