# <a name="division-by-zero-visual-basic-run-time-error"></a>Dzielenie przez zero (Visual Basic środowiska wykonawczego)
Wyrażenie używane jako dzielnik ma wartość zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię zmiennych w wyrażeniu. Nieprawidłowo zapisana nazwa zmiennej może niejawnie utworzyć zmienną liczbową, której wartość jest ustawiana na zero.  
  
2.  Sprawdź poprzednie operacje na zmiennych w wyrażeniu, zwłaszcza tych, które przekazano do procedury jako argumenty z innych procedur.
