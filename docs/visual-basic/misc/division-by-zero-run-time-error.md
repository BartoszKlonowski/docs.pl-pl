# <a name="division-by-zero-visual-basic-run-time-error"></a>Dzielenie przez zero (Visual Basic środowiska wykonawczego)
Wyrażenie używane jako dzielnik ma wartość zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię zmiennych w wyrażeniu. Nieprawidłowo zapisana nazwa zmiennej niejawnie można utworzyć liczbowa zmiennej, która jest ustawiana na wartość zero.  
  
2.  Sprawdź poprzednich operacji na zmiennych w wyrażeniu, zwłaszcza tych, które przekazany do procedury jako argumenty z innych procedur.
