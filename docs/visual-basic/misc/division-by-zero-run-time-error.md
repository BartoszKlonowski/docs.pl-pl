# <a name="division-by-zero-visual-basic-run-time-error"></a>Dzielenie przez zero (błąd czasu wykonywania Visual Basic)
Wyrażenie używane jako dzielnik ma wartość zero.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź pisownię zmiennych w wyrażeniu. Błędnie wpisana nazwa zmiennej może niejawnie utworzyć zmienną numeryczną, która została zainicjowana do zera.  
  
2. Sprawdź poprzednie operacje na zmiennych w wyrażeniu, szczególnie te przekazane do procedury jako argumenty z innych procedur.
