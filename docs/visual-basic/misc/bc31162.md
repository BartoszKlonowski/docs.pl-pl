# <a name="expected-closing--for-xml-cdata-section"></a>Oczekiwano zamykającego "]] >" dla sekcji XML CDATA
Literał XML CDATA brakuje zamykającego wymagane "]] >" znaków.  
  
 **Identyfikator błędu:** BC31162  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamknięcia "]] >" znaki do końca literał XML CDATA.
