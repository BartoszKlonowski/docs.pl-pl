# <a name="expected-closing--for-xml-cdata-section"></a>Oczekiwano zamykającego "]] >" dla sekcji CDATA XML

W literale CDATA XML brakuje wymaganego zamykającego znaku "]] >".  
  
 **Identyfikator błędu:** BC31162  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagane zamykające znaki "]] >" na końcu literału XML CDATA.
