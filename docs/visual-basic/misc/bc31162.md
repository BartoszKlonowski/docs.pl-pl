# <a name="expected-closing-39gt39-for-xml-cdata-section"></a>Oczekiwano zamykających znaków &#39;]]&gt; &#39; dla sekcji XML CDATA
Literał XML CDATA brakuje wymaganego zamykającego "]] >" znaków.  
  
 **Identyfikator błędu:** BC31162  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamykającego "]] >" znaki na końcu literał XML CDATA.
