# <a name="named-argument-not-found-visual-basic"></a>Nazwany argument nie znaleziono (Visual Basic)
Nazwany argument nie może służyć w wywołaniu procedury, chyba że prawdopodobnie w definicji procedury. Określony argument nazwany, ale procedura nie został zdefiniowany do akceptowania argumentu o takiej nazwie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź pisownię nazwy argumentu.
