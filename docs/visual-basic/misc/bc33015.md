# <a name="operator-operator-must-have-two-parameters"></a>Operator '\<operatora >' musi mieć dwa parametry
Operator binarny jest zdefiniowana z mniej niż dwa lub więcej niż dwa parametry.  
  
 Operatora binarnego musi mieć dokładnie dwóch parametrów.  
  
 **Identyfikator błędu:** BC33015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dostosowanie definicji, aby określić dokładnie dwóch parametrów.  
  
- Jeśli potrzebujesz tylko jeden parametr, należy zdefiniować operator jednoargumentowy.  
  
- Jeśli potrzebujesz bez parametrów lub większa niż dwa, należy użyć [Function — instrukcja](../../visual-basic/language-reference/statements/function-statement.md) do definiowania `Function` procedury zamiast operatora.
