# <a name="operator-39ltoperatorgt39-must-have-two-parameters"></a>Operator &#39; &lt;operator&gt;&#39; musi mieć dwa parametry
Operator binarny jest zdefiniowana z mniej niż dwóch lub więcej niż dwa parametry.  
  
 Operatora binarnego musi mieć dokładnie dwa parametry.  
  
 **Identyfikator błędu:** BC33015  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostosuj definicji, aby określić dokładnie dwa parametry.  
  
-   Jeśli potrzebujesz tylko jeden parametr, należy zdefiniować operatora jednoargumentowego.  
  
-   Jeśli potrzebujesz bez parametrów lub większa niż dwa, należy użyć [instrukcji Function](../../visual-basic/language-reference/statements/function-statement.md) do definiowania `Function` procedury zamiast operatora.
