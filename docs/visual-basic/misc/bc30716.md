# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>"Finally" nie są prawidłowe w oknie bezpośrednim
`Finally` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30716  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Finally` instrukcji debugowania kodu.
