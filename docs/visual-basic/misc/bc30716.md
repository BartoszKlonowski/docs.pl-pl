# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Na koniec&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Finally` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30716  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Finally` oświadczenia debugowania kodu.
