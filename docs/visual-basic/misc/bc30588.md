# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Pominięty argument nie jest zgodny z parametrem ParamArray

Pominięto Argument wymagany do dopasowania do `ParamArray` parametru. `ParamArray`, która pozwala przekazać dowolną liczbę argumentów do procedury, może być używana tylko na ostatnim argumencie listy argumentów.  
  
 **Identyfikator błędu:** BC30588  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj pominięty argument.
