# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Pominięty argument nie jest zgodny z parametrem ParamArray
Pominięto argument muszą odpowiadać `ParamArray` parametru. `ParamArray`, co pozwala na przekazywanie z dowolnej liczby argumentów do procedury, można używać tylko na ostatniego argumentu listy argumentów.  
  
 **Identyfikator błędu:** BC30588  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj pominięty argument.
