# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39;End RaiseEvent&#39; musi być pierwszą instrukcją w wierszu
`End RaiseEvent` Instrukcji następuje separatora instrukcji dwukropka (:). `End RaiseEvent` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31120  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
