# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>Element "End RaiseEvent" musi być pierwszą instrukcją w wierszu
`End RaiseEvent`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End RaiseEvent`musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31120  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
