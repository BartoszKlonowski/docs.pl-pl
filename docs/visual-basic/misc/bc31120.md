# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>"End RaiseEvent" musi być pierwszą instrukcją w wierszu
`End RaiseEvent` Instrukcji następuje separatorem instrukcji dwukropek (:). `End RaiseEvent` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31120  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
