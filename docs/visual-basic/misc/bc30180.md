# <a name="keyword-does-not-name-a-type"></a>Słowo kluczowe nie ma nazwy typu

Słowo kluczowe pojawia się, gdy nazwa typu jest wymagana. Kontekst wywołuje dla typu danych podstawowych lub zadeklarowanej nazwy elementu reprezentującej typ.  
  
 **Identyfikator błędu:** BC30180  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zastąp słowo kluczowe prawidłową nazwą typu.
