# <a name="keyword-does-not-name-a-type"></a>Słowo kluczowe nie nadaje nazwy typu
Słowo kluczowe jest wyświetlany, gdy wymagana jest nazwa typu. Kontekst wywołania dla typu danych podstawowych lub nazwa element zadeklarowany, reprezentujący typu.  
  
 **Identyfikator błędu:** BC30180  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zastąp słowo kluczowe z prawidłową nazwą typu.
