# <a name="39exit-property39-is-not-valid-in-a-function-or-sub"></a>&#39;Zamknij właściwości&#39; jest nieprawidłowy w przypadku funkcji lub procedury
`Exit Property` zostanie wyświetlony w `Function` procedury lub `Sub` procedury. `Exit` Instrukcja musi odpowiadać bloku, w którym występuje.  
  
 **Identyfikator błędu:** BC30066  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp `Exit Property` z `Exit Function` lub `Exit Sub` oświadczenie zależnie od potrzeb.
