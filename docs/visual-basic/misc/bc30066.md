# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>"Exit właściwość" jest nieprawidłowy w funkcji lub procedury
`Exit Property` pojawia się w `Function` procedury lub `Sub` procedury. `Exit` Instrukcja musi odpowiadać bloku, w której występuje.  
  
 **Identyfikator błędu:** BC30066  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp `Exit Property` z `Exit Function` lub `Exit Sub` instrukcji zgodnie z potrzebami.
