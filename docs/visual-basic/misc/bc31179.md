# <a name="expected--at-start-of-an-embedded-expression"></a>Oczekiwano "% =" na początku wyrażenia osadzone
Identyfikator rozpoczęcia dla wyrażenia osadzone `<%=`, nie został poprawnie kodowane. Należy pamiętać, że nie można użyć znaku procentu (%) w nazwie literał elementu XML.  
  
 **Identyfikator błędu:** BC31179  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że identyfikator start osadzone wyrażenie jest zakodowane jako `<%=`.
