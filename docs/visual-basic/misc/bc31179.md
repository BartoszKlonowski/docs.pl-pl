# <a name="expected-3939-at-start-of-an-embedded-expression"></a>Oczekiwano &#39;% =&#39; na początku wyrażenia osadzonego
Identyfikator uruchomienia dla wyrażenia osadzonego `<%=`, nie zawiera prawidłowo kodowany. Należy pamiętać, że nie można użyć znaku procentu (%) w nazwie literał elementu XML.  
  
 **Identyfikator błędu:** BC31179  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że identyfikator początku wyrażenia osadzonego jest zakodowane jako `<%=`.
