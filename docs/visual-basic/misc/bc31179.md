# <a name="expected--at-start-of-an-embedded-expression"></a>Oczekiwano znaku "% =" na początku wyrażenia osadzonego
Identyfikator początkowy wyrażeniu osadzonego, `<%=` nie został poprawnie zakodowany. Należy pamiętać, że nie można użyć znaku procentu (%) w nazwie literału elementu XML.  
  
 **Identyfikator błędu:** BC31179  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że Identyfikator początkowy wyrażenia osadzonego jest kodowany jako `<%=` .
