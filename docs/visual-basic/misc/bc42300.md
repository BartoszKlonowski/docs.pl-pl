# <a name="xml-comment-block-must-immediately-precede-the-language-element-to-which-it-applies"></a>Blok komentarza XML musi bezpośrednio poprzedzać element języka, którego dotyczy
Blok komentarza XML musi bezpośrednio poprzedzać element języka, do której jest stosowany. Komentarz XML zostanie zignorowany.  
  
 Ten błąd wystąpi również, jeśli tag jest umieszczony w nieprawidłowym miejscu lub źle sformułowane.  
  
 **Identyfikator błędu:** BC42300  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Przenieś blok komentarza poprzedzać element języka, do której jest stosowany. Upewnij się, że nadmiarowe znaki nie zostały przypadkowo wstawione przed tagu początkowego.  
  
2.  Upewnij się, że znacznika jest prawidłowy.
