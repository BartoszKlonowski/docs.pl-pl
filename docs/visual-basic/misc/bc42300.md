# <a name="xml-comment-block-must-immediately-precede-the-language-element-to-which-it-applies"></a>Blok komentarza XML musi bezpośrednio poprzedzać element języka, do którego jest stosowany
Blok komentarza XML musi bezpośrednio poprzedzać element języka, do którego jest stosowany. Komentarz XML zostanie zignorowany.  
  
 Ten błąd będzie również wystąpić, jeśli znacznik jest niewłaściwie lub źle sformułowane.  
  
 **Identyfikator błędu:** BC42300  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Przenieś do poprzedzać element języka, którego dotyczy blok komentarza. Upewnij się, że nadmiarowe znaki nie przypadkowo wstawiono przed tagu początkowego.  
  
2.  Upewnij się, że znacznika jest prawidłowy.
