# <a name="codepage-name-is-invalid-or-not-installed"></a>Strona kodowa "\<Name >" jest nieprawidłowa lub nie została zainstalowana
Określona strona kodowa jest nieprawidłowa lub nie można jej znaleźć.  
  
 **Identyfikator błędu:** BC2016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź kod i określ rozwiązanie, które jest odpowiednie dla konkretnego przypadku
