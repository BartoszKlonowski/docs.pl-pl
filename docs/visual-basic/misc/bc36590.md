# <a name="too-few-type-arguments-to-extension-method-methodname-defined-in-typename"></a>Za mało argumentów typu dla metody rozszerzenia "\<methodname >" zdefiniowane w "\<typename >"
Metoda rozszerzenia ogólnego zostało wywołane z mniejszą liczbą argumentów typu niż liczba parametrów typu w definicji metody.  
  
 Po wywołaniu metody rodzajowej, należy podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36590  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu metody rozszerzenia, które są wywoływane, Dodaj argumenty typu do listy argumentów typu.
