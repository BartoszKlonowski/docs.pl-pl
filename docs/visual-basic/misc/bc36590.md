# <a name="too-few-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Za mało argumentów typu dla metody rozszerzającej &#39; &lt;methodname&gt;&#39; zdefiniowanego w &#39;&lt; Właściwość TypeName&gt;&#39;
Metody ogólne rozszerzenie zostało wywołane z mniejszą liczbę argumentów typu niż w definicji metoda ma parametry typu.  
  
 Po wywołaniu metody ogólnej, musisz podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36590  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu metody rozszerzenia, która jest należy dodać argumentów typu do listy argumentów typu.
