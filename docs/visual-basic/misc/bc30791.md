# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>Instrukcji "DECLARE" w strukturze nie można zadeklarować jako " \<specifier> "
`Declare`Deklaracja zawiera specyfikator, który jest nieprawidłowy w `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30791  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
