# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;Deklarowanie&#39; instrukcje w strukturze nie można zadeklarować &#39; &lt;Specyfikator&gt;&#39;
A `Declare` deklaracja zawiera specyfikator, który nie jest prawidłowy w ramach `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30791  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
