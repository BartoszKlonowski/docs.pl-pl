# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>"Declare" instrukcji w strukturze nie można zadeklarować jako "\<specyfikator >"
A `Declare` deklaracja zawiera specyfikator, który nie jest prawidłowy w ramach `Structure` deklaracji.  
  
 **Identyfikator błędu:** BC30791  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
