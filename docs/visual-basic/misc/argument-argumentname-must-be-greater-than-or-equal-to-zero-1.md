# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument \<argumentname> musi być większy lub równy zero

Wartość argumentu jest mniejsza od zera. Wartości argumentów muszą być większe lub równe zeru.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że wartość argumentu jest większa lub równa zero.  
  
2. Jeśli argument jest dostarczany przez obliczenie, sprawdź wartości przetwarzane przez obliczenie.
