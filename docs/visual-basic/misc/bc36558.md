## <a name="example"></a><span data-ttu-id="c4403-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="c4403-111">Example</span></span>

<span data-ttu-id="c4403-112">Poniższy przykład używa atrybutu, `AttributeUsage` aby określić, które elementy programu `ExtensionAttribute` mogą dotyczyć nowej wersji.</span><span class="sxs-lookup"><span data-stu-id="c4403-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="c4403-113">W przykładzie określono `AttributeTargets` trzy elementy członkowskie wyliczenia: `Assembly`, `Class`, i `Method`.</span><span class="sxs-lookup"><span data-stu-id="c4403-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="c4403-114">Pominięcie jednego z tych elementów spowoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="c4403-114">The omission of any one of these elements will cause this error.</span></span>

```vb
Namespace System.Runtime.CompilerServices
    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="c4403-115">Alternatywnie można zezwolić `ExtensionAttribute` na stosowanie do wszystkich elementów programu przy `All` użyciu elementu członkowskiego `AttributeTargets`.</span><span class="sxs-lookup"><span data-stu-id="c4403-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

<span data-ttu-id="c4403-116">`AttributeUsage` Usunięcie wiersza, jak pokazano w poniższym kodzie, daje ten sam wynik.</span><span class="sxs-lookup"><span data-stu-id="c4403-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>
