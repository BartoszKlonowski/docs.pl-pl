## <a name="example"></a><span data-ttu-id="c53a0-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="c53a0-111">Example</span></span>

<span data-ttu-id="c53a0-112">Poniższy przykład używa atrybutu, `AttributeUsage` Aby określić, które elementy programu `ExtensionAttribute` mogą dotyczyć nowej wersji.</span><span class="sxs-lookup"><span data-stu-id="c53a0-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="c53a0-113">W przykładzie określono trzy elementy członkowskie `AttributeTargets` wyliczenia: `Assembly` , `Class` , i `Method` .</span><span class="sxs-lookup"><span data-stu-id="c53a0-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="c53a0-114">Pominięcie jednego z tych elementów spowoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="c53a0-114">The omission of any one of these elements will cause this error.</span></span>

```vb

    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```
