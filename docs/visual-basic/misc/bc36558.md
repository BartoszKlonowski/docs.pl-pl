## <a name="example"></a><span data-ttu-id="9f234-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="9f234-111">Example</span></span>  
 <span data-ttu-id="9f234-112">W poniższym przykładzie użyto `AttributeUsage` atrybutu, aby określić, który program elementy do nowej wersji `ExtensionAttribute` może dotyczyć.</span><span class="sxs-lookup"><span data-stu-id="9f234-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="9f234-113">W przykładzie określono trzech członków `AttributeTargets` wyliczenia: `Assembly`, `Class`, i `Method`.</span><span class="sxs-lookup"><span data-stu-id="9f234-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="9f234-114">Pominięcie któregokolwiek z tych elementów spowoduje, że ten błąd.</span><span class="sxs-lookup"><span data-stu-id="9f234-114">The omission of any one of these elements will cause this error.</span></span>  
  
```  
Namespace System.Runtime.CompilerServices  
    <AttributeUsage(AttributeTargets.Assembly Or _  
        AttributeTargets.Class Or AttributeTargets.Method)>  
    Class ExtensionAttribute  
        Inherits System.Attribute  
        ' Definitions of methods, fields, and properties.  
    End Class  
End Namespace  
```  
  
 <span data-ttu-id="9f234-115">Alternatywnie, można zezwolić `ExtensionAttribute` można zastosować do wszystkich elementów programu za pomocą `All` członkiem `AttributeTargets`.</span><span class="sxs-lookup"><span data-stu-id="9f234-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

  
 <span data-ttu-id="9f234-116">Usuwanie `AttributeUsage` wiersza, jak pokazano w poniższym kodzie daje ten sam wynik.</span><span class="sxs-lookup"><span data-stu-id="9f234-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>
