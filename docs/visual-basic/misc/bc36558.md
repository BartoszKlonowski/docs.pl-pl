## <a name="example"></a><span data-ttu-id="ac312-111">Przykład</span><span class="sxs-lookup"><span data-stu-id="ac312-111">Example</span></span>  
 <span data-ttu-id="ac312-112">W poniższym przykładzie użyto `AttributeUsage` atrybutu, aby określić nową wersję programu elementy `ExtensionAttribute` może dotyczyć.</span><span class="sxs-lookup"><span data-stu-id="ac312-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="ac312-113">W przykładzie określono trzech elementów członkowskich `AttributeTargets` wyliczenie: `Assembly`, `Class`, i `Method`.</span><span class="sxs-lookup"><span data-stu-id="ac312-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="ac312-114">Pominięcie któregokolwiek z tych elementów spowoduje, że ten błąd.</span><span class="sxs-lookup"><span data-stu-id="ac312-114">The omission of any one of these elements will cause this error.</span></span>  
  
```

    <AttributeUsage(AttributeTargets.Assembly Or _  
        AttributeTargets.Class Or AttributeTargets.Method)>  
    Class ExtensionAttribute  
        Inherits System.Attribute  
        ' Definitions of methods, fields, and properties.  
    End Class  
End Namespace  
```  
  
 <span data-ttu-id="ac312-115">Alternatywnie można zezwolić `ExtensionAttribute` do zastosowania do wszystkich elementów programu za pomocą `All` członkiem `AttributeTargets`.</span><span class="sxs-lookup"><span data-stu-id="ac312-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>
