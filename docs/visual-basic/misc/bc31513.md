# <a name="systemstathreadattribute-and-systemmtathreadattribute-cannot-both-be-applied-to-1"></a>'System.STAThreadAttribute"i"System.MTAThreadAttribute"nie można zastosować do ' | 1'
`System.STAThreadAttribute` i `System.MTAThreadAttribute` atrybuty wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC31513  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zastosuj opcję `System.MTAThreadAttribute` lub `System.STAThreadAttribute`, ale nie oba jednocześnie.
