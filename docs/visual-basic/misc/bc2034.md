# <a name="option-win32manifest-ignored"></a>Opcja/win32manifest została zignorowana
Opcja/win32manifest została zignorowana. Można określić tylko wtedy, gdy element docelowy jest zestawem.  
  
 Kompilator Visual Basic został przekazany `/win32manifest` — opcja kompilatora podczas `/target` ustawiono opcję `module`.  
  
 **Identyfikator błędu:** BC2034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `/win32manifest` — opcja kompilatora lub zestawu `/target` opcji w celu `exe`, `winexe`, lub `library`.
