# <a name="comma-expected"></a>Oczekiwano przecinka
Brak przecinka (,) między dwoma argumentami listy argumentów.  
  
 **Identyfikator błędu:** BC30196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie argumenty są oddzielone przecinkami.
