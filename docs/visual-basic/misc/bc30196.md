# <a name="comma-expected"></a>Oczekiwany przecinek
Lista argumentów jest Brak przecinka (,) między dwoma argumentami.  
  
 **Identyfikator błędu:** BC30196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie argumenty są oddzielone przecinkami.
