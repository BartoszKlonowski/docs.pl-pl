# <a name="comma-expected"></a>Oczekiwano przecinka
Lista argumentów nie zawiera przecinka (,) między dwoma argumentami.  
  
 **Identyfikator błędu:** BC30196  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że wszystkie argumenty są rozdzielone przecinkami.
