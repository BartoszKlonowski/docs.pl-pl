# <a name="too-few-type-arguments-to-genericmethodname"></a>Za mało argumentów typu dla " \<genericMethodName> "
Metoda ogólna została wywołana z mniejszą liczbą argumentów niż liczba parametrów typu.  
  
 Po wywołaniu metody ogólnej należy podać jeden argument typu dla każdego parametru typu zdefiniowanego przez tę metodę.  
  
 **Identyfikator błędu:** BC32042  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj argumenty typu do listy argumentów typu, tak aby istniała jeden dla każdego parametru typu w wywołaniu metody ogólnej.
