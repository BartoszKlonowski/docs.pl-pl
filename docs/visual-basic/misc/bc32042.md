# <a name="too-few-type-arguments-to-ltgenericmethodnamegt"></a>Za mało argumentów typu dla "&lt;genericMethodName&gt;"
Metody ogólnej zostało wywołane z mniejszą liczbą argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody rodzajowej, należy podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC32042  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu dla metody rodzajowe, wywoływane, Dodaj argumenty typu do listy argumentów typu.
