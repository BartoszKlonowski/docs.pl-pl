# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Specyfikatory i atrybuty nie są prawidłowe w instrukcjach "Namespace"
Przestrzenie nazw są używane do hierarchicznej struktury bloków program i elementów programowania. Nie przyjmują żadnych modyfikatorów ani atrybutów.  
  
 **Identyfikator błędu:** BC32026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszystkie słowa kluczowe, a atrybut bloków `Namespace` instrukcji.
