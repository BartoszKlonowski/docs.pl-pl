# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Specyfikatory i atrybuty nie są prawidłowe w instrukcjach "namespace"

Przestrzenie nazw są używane dla hierarchicznej organizacji bloków programu i elementów programistycznych. Nie przyjmują żadnych modyfikatorów ani atrybutów.  
  
 **Identyfikator błędu:** BC32026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wszystkie słowa kluczowe i bloki atrybutów z `Namespace` instrukcji.
