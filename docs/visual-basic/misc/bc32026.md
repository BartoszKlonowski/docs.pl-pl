# <a name="specifiers-and-attributes-are-not-valid-on-39namespace39-statements"></a>Specyfikatory i atrybuty, są one nieprawidłowe na &#39; Namespace &#39; instrukcje
Przestrzenie nazw są używane do hierarchiczna organizacja program bloków i elementami programowania. Ich nie mają żadnych modyfikatorów ani atrybutów.  
  
 **Identyfikator błędu:** BC32026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszystkie słowa kluczowe i atrybutu bloków z `Namespace` instrukcji.
