# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>"Opcja &lt;specyfikator&gt;" instrukcji może wystąpić tylko raz na plik
`Option Compare`, `Option Explicit`, I `Option Strict` instrukcji można używać tylko jeden raz w tym samym pliku źródłowym.  
  
 **Identyfikator błędu:** BC30225  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zduplikowane `Option` instrukcji.
