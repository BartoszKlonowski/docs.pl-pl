# <a name="option-specifier-statement-can-only-appear-once-per-file"></a>Instrukcja "Option \<specifier> " może wystąpić tylko raz dla pliku
`Option Compare`Instrukcji, `Option Explicit` i `Option Strict` można używać tylko jeden raz w tym samym pliku źródłowym.  
  
 **Identyfikator błędu:** BC30225  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowaną `Option` instrukcję.
