# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39;Opcja &lt;specyfikator&gt; &#39; instrukcja może wystąpić tylko raz dla każdego pliku
`Option Compare`, `Option Explicit`, I `Option Strict` instrukcji można używać tylko jeden raz w tym samym pliku źródłowego.  
  
 **Identyfikator błędu:** BC30225  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zduplikowane `Option` instrukcji.
