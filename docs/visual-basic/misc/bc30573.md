# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>Prefiks przestrzeni nazw XML "prefiks" jest już zadeklarowany.
Prefiks przestrzeni nazw XML zdefiniowane przez `Imports` instrukcji odpowiada uprzednio zdefiniowany prefiks przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC30573  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj unikatowej nazwy prefiksu przestrzeni nazw XML.
