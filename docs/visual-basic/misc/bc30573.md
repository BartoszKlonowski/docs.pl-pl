# <a name="xml-namespace-prefix-39prefix39-is-already-declared"></a>Prefiks przestrzeni nazw XML &#39;prefiks&#39; jest już zadeklarowana
Prefiks przestrzeni nazw XML, zdefiniowane przez `Imports` instrukcji zgodny z wcześniej zdefiniowanego prefiksu przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC30573  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj unikatowej nazwy dla prefiksu przestrzeni nazw XML.
