# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>Prefiks przestrzeni nazw XML "prefix" jest już zadeklarowany

Prefiks przestrzeni nazw XML zdefiniowany przez `Imports` instrukcję pasuje do wcześniej zdefiniowanego prefiksu przestrzeni nazw XML.  
  
 **Identyfikator błędu:** BC30573  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj unikatowej nazwy dla prefiksu przestrzeni nazw XML.
