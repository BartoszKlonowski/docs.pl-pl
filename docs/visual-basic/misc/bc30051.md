# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>Parametr ParamArray musi być tablicą jednowymiarową

`ParamArray`Argument musi być zadeklarowany jako jednowymiarowy typ tablicy. Deklaracja określa więcej niż jeden wymiar dla `ParamArray` argumentu.  
  
 **Identyfikator błędu:** BC30051  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wszystkie przecinki wewnątrz nawiasów po `ParamArray` nazwie argumentu.
