# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>Parametr ParamArray musi być tablicą jednowymiarową
A `ParamArray` argument musi być zadeklarowany jako typ jednowymiarową. Deklaracja określa więcej niż jednym wymiarze dla `ParamArray` argumentu.  
  
 **Identyfikator błędu:** BC30051  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszystkie przecinków wewnątrz nawiasów po `ParamArray` Nazwa argumentu.
