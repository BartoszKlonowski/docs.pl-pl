# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Dozwolone jest tylko jeden blok komentarza XML na element języka
Więcej niż jeden blok komentarza XML zostały zastosowane do elementu języka.  
  
 **Identyfikator błędu:** BC42301  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędny blok komentarza XML.
