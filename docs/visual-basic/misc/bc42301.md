# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Dozwolony jest tylko jeden blok komentarza XML na element języka

Do elementu języka zastosowano więcej niż jeden blok komentarza XML.  
  
 **Identyfikator błędu:** BC42301  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędny blok komentarza XML.
