# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Tylko jeden blok komentarza XML jest dozwolony na element języka
Więcej niż jeden blok komentarza XML zostały doliczone do elementu języka.  
  
 **Identyfikator błędu:** BC42301  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuwanie zbędnym blok komentarza XML.
