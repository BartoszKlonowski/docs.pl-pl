# <a name="set-not-supported-read-only-property"></a>Zestaw nie jest obsługiwany (właściwość tylko do odczytu)

Podjęto próbę ustawienia lub zmiany właściwości, która jest tylko do odczytu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń odwołanie do właściwości z kodu.  
  
2. Zmień odwołanie, aby zwrócić wartość właściwości w czasie wykonywania.
