# <a name="set-not-supported-read-only-property"></a>Nieobsługiwany zestaw (właściwość tylko do odczytu)
Próbowano ustawić lub zmienić właściwość, która jest tylko do odczytu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń odwołanie do właściwości w kodzie.  
  
2.  Zmień odwołania, aby zwracać tylko wartości właściwości w czasie wykonywania.
