# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>Zmienne "WithEvents" nie może być typu tablicowego
Podjęto próbę zadeklarować tablicę przy użyciu `WithEvents`. Można zadeklarować dowolną liczbę zmiennych poszczególnych jak takimi jak wymaganie użycia `WithEvents`, ale nie można zadeklarować tablic w ten sposób.  
  
 **Identyfikator błędu:** BC30476  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zadeklaruj zmienne indywidualnie.
