# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>Zmiennych "WithEvents" nie można wpisywać jako tablic
Podjęto próbę zadeklarować tablicę przy użyciu `WithEvents` . Można zadeklarować dowolną liczbę indywidualnych zmiennych, które są używane `WithEvents` , ale nie można zadeklarować tablic w taki sposób.  
  
 **Identyfikator błędu:** BC30476  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zadeklaruj zmienne pojedynczo.
