# <a name="39withevents39-variables-cannot-be-typed-as-arrays"></a>&#39; WithEvents &#39; zmienne nie może być typu tablicowego
Podjęto próbę zadeklarować tablicy przy użyciu `WithEvents`. Można zadeklarować dowolną liczbę zmiennych poszczególnych jako użytkownik na przykład za pomocą `WithEvents`, ale nie można deklarować tablic w ten sposób.  
  
 **Identyfikator błędu:** BC30476  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zadeklaruj zmienne pojedynczo.
