# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>Przypisanie z późnym wiązaniem do pola typu wartości "&lt;typename&gt;"nie jest dozwolone podczas"&lt;nazwa&gt;" jest wynikiem wyrażenia z późnym wiązaniem
Próbowano przypisanie z późnym wiązaniem, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Utwórz przypisanie wcześnie powiązana.
