# <a name="the-path-has-not-been-set"></a>Ścieżka nie została ustawiona.
Podjęto próbę operacji na pliku, np. przenoszenie lub kopiowanie pliku, bez podawania niezbędne ścieżki. Może to wynikać z niepoprawnie analizowaniu ścieżki.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy ścieżka do pliku zostało dostarczone i jest poprawna.
