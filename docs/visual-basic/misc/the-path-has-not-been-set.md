# <a name="the-path-has-not-been-set"></a>Ścieżka nie została ustawiona
Podjęto próbę wykonania operacji na pliku, na przykład przeniesienia lub skopiowania pliku, bez podawania niezbędnej ścieżki. Może to wynikać z nieprawidłowej analizy ścieżki.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy ścieżka do pliku została podana i czy jest poprawna.
