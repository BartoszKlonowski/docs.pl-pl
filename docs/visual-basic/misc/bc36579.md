# <a name="too-many-type-arguments"></a>Za dużo argumentów typu
Metoda ogólna została wywołana z liczbą argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody ogólnej, musisz podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36579  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń argumentów typu z listy argumentów typu, dzięki czemu jest on dostępny dla każdego parametru typu metody ogólnej, który jest.
