# <a name="too-many-type-arguments"></a>Za dużo argumentów typu
Metoda ogólna została wywołana z więcej argumentów typu niż parametry typu.  
  
 Po wywołaniu metody ogólnej należy podać jeden argument typu dla każdego parametru typu zdefiniowanego przez tę metodę.  
  
 **Identyfikator błędu:** BC36579  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń argumenty typu z listy argumentów typu, tak aby istniała jeden dla każdego parametru typu metody ogólnej, którą wywołujesz.
