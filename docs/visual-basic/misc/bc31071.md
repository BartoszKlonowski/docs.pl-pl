# <a name="structure-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować struktury w interfejsie jako " \<specifier> "
Określono nieprawidłowy modyfikator ze strukturą zadeklarowaną w interfejsie.  
  
 **Identyfikator błędu:** BC31071  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikatory, takie jak,,,,, `Public` `Private` `Shared` `Friend` `Protected` `Protected Friend` lub `Overridable` .
