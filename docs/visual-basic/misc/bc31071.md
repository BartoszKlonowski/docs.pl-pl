# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować struktury w interfejsie &#39; &lt;Specyfikator&gt;&#39;
Nieprawidłowy modyfikator określono ze strukturą zadeklarowany w interfejsie.  
  
 **Identyfikator błędu:** BC31071  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikatory, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, lub `Overridable`.
