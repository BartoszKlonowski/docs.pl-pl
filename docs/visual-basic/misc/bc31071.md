# <a name="structure-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować struktury w interfejsie jako "\<specyfikator >"
Nieprawidłowy modyfikator określono ze strukturą zadeklarowanych w interfejsie.  
  
 **Identyfikator błędu:** BC31071  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikatorów, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, lub `Overridable`.
