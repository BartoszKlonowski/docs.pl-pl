# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="d1186-102">Nie można zainicjować rozwiniętych właściwości</span><span class="sxs-lookup"><span data-stu-id="d1186-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="d1186-103">Właściwość, która jest implementowana przez funkcję, może zostać zainicjowana w ramach swojej deklaracji, ale rozwinięta właściwość nie może być.</span><span class="sxs-lookup"><span data-stu-id="d1186-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="d1186-104">**Identyfikator błędu:** BC36714</span><span class="sxs-lookup"><span data-stu-id="d1186-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d1186-105">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="d1186-105">To correct this error</span></span>  
  
- <span data-ttu-id="d1186-106">Użyj zaimplementowanej właściwości lub Usuń inicjalizację z deklaracji właściwości.</span><span class="sxs-lookup"><span data-stu-id="d1186-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d1186-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="d1186-107">Example</span></span>  
 <span data-ttu-id="d1186-108">W poniższych przykładach pokazano zarówno zaimplementowane, jak i rozwinięte właściwości.</span><span class="sxs-lookup"><span data-stu-id="d1186-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="d1186-109">Właściwości zaimplementowane przez funkcję można zainicjować za pomocą przypisania lub `New` klauzuli, ale rozwinięte właściwości nie mogą być.</span><span class="sxs-lookup"><span data-stu-id="d1186-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
