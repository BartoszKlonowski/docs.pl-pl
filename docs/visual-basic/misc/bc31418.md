# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>&#39; Uchwyty &#39; należy określić w modułach &#39; WithEvents &#39; Zmienna kwalifikowana za pomocą pojedynczego identyfikatora
Do określania obsługi zdarzenia `Handles` instrukcje należy określić zmienną obiektu, który został zadeklarowany z `WithEvents` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC31418  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `WithEvents` modyfikator do deklarowania zmiennych, które będą używane z `Handles` instrukcji.
