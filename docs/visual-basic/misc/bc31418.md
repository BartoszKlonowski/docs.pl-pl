# <a name="handles-in-modules-must-specify-a-withevents-variable-qualified-with-a-single-identifier"></a>"Handles w module należy określić zmienną"WithEvents"kwalifikowany za pomocą pojedynczego identyfikatora
Aby określić program obsługi zdarzeń `Handles` instrukcji należy określić zmienną obiektu, który został zadeklarowany za pomocą `WithEvents` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC31418  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `WithEvents` modyfikator do deklarowania zmiennych, które będą używane z `Handles` instrukcji.
