# <a name="set-statement-must-end-with-a-matching-end-set"></a>Instrukcja "Set" musi być zakończona odpowiadającą jej instrukcją "End Set"

`Set` procedury właściwości muszą kończyć się `End Set` instrukcją.  
  
 **Identyfikator błędu:** BC30633  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że `Set` procedura właściwości jest zakończona w `End Set` konstrukcji.
