# <a name="set-statement-must-end-with-a-matching-end-set"></a>Instrukcja "Set" musi kończyć się instrukcją "End Set"
`Set` procedury własności musi się kończyć `End Set` instrukcji.  
  
 **Identyfikator błędu:** BC30633  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `Set` kończy procedurę właściwości `End Set` konstruowania.
