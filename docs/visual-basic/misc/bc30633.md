# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39; Ustaw &#39; Instrukcja musi kończyć się pasujący &#39; Ustaw zakończenia &#39;
`Set`procedury własności musi kończyć się `End Set` instrukcji.  
  
 **Identyfikator błędu:** BC30633  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `Set` kończy procedurę właściwości `End Set` utworzenia.
