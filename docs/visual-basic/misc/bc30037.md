# <a name="character-is-not-valid"></a>Znak jest nieprawidłowy
Bieżący kontekst nie zezwala na tego znaku. Na przykład tyldy (`~`) lub pionowej kreski (`|`) nie może być częścią nazwy deklarowanych elementów.  
  
 **Identyfikator błędu:** BC30037  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy dodatkowy znak został przypadkowo wstawiony do elementu programistycznego.  
  
2.  Sprawdź, czy elementu programistycznego została wpisana lub tworzony przy użyciu tylko znaki, które są prawidłowe dla tego rodzaju elementu.
