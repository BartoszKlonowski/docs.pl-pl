# <a name="character-is-not-valid"></a>Nieprawidłowy znak

Bieżący kontekst nie zezwala na ten znak. Na przykład, znak tyldy ( `~` ) lub pionowy pasek ( `|` ) nie może być częścią zadeklarowanej nazwy elementu.  
  
 **Identyfikator błędu:** BC30037  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy dodatkowy znak został przypadkowo wstawiony do elementu programowania.  
  
2. Sprawdź, czy element programistyczny jest wpisany lub skonstruowany tylko przy użyciu znaków, które są prawidłowe dla tego rodzaju elementu.
