# <a name="character-is-not-valid"></a>Znak jest nieprawidłowy
Bieżący kontekst nie zezwala na tego znaku. Na przykład tyldy (`~`) lub pionowy pasek (`|`) nie może być częścią nazwy zadeklarowanych elementów.  
  
 **Identyfikator błędu:** BC30037  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, jeśli nadmiarowy znak został przypadkowo wstawiony do elementu programistycznego.  
  
2.  Sprawdź, czy element programowania jest wpisany, lub tworzona przy użyciu tylko te znaki, które są prawidłowe dla tego rodzaju elementu.
