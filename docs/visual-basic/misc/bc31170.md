# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>&#39;znaków; znaku &#39; (0 x&lt;szesnastkową&gt;) nie jest dozwolona w nazwie XML
Nieobsługiwany znak jest zawarty w nazwie literał XML i właściwości osi XML.  
  
 **Identyfikator błędu:** BC31170  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwany znak z nazw literał XML lub właściwości osi XML.
