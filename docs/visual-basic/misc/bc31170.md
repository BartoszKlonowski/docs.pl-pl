# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>Znak "Character" (0x \<hexadecimal> ) jest niedozwolony w nazwie XML
Nieobsługiwany znak jest zawarty w nazwie literału XML lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31170  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwany znak z nazwy literału XML lub właściwości osi XML.
