# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>Znak "character" (0 x\<szesnastkowe >) jest niedozwolone w nazwie XML
Nieobsługiwany znak jest zawarty w nazwie literał XML i właściwości osi XML.  
  
 **Identyfikator błędu:** BC31170  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane znaki z nazwą literał XML i właściwości osi XML.
