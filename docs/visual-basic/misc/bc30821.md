# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39; nie została zadeklarowana
"\<rsetstmt >' nie został zadeklarowany. Instrukcje RSet nie są już obsługiwane; Zamiast tego użyj Microsoft.VisualBasic.RSet.  
  
 Kilka funkcji, które były wewnętrzne w języku Visual Basic w poprzednich wersjach zostały przeniesione do ikony <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. To sprawia, że ich funkcjonalność bardziej ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30821  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `RSet` działa w programach `Microsoft.VisualBasic` przestrzeni nazw zamiast tego.
