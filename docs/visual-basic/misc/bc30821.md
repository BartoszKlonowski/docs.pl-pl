# <a name="rsetstmt-is-not-declared"></a>element " \<rsetstmt> " nie jest zadeklarowany

element " \<rsetstmt> " nie jest zadeklarowany. Instrukcje RSet nie są już obsługiwane; Zamiast tego należy użyć Microsoft. VisualBasic. RSet.  
  
 Kilka funkcji, które były wewnętrznie Visual Basic w poprzednich wersjach, zostały przeniesione do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcjonalność jest bardziej ogólnie dostępna dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30821  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `RSet`Zamiast tego użyj funkcji w `Microsoft.VisualBasic` przestrzeni nazw.
