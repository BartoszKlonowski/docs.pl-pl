# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39; nie jest zadeklarowana
"\<rsetstmt >' nie został zadeklarowany. Instrukcje RSet nie są już obsługiwane; Zamiast tego użyj Microsoft.VisualBasic.RSet.  
  
 Kilka funkcji, które były wewnętrznej na język Visual Basic w poprzednich wersjach zostały przeniesione do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcji ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30821  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `RSet` działać w `Microsoft.VisualBasic` przestrzeni nazw zamiast tego.
