# <a name="too-few-type-arguments"></a>Za mało argumentów typu
Metody ogólnej zostało wywołane z mniejszą liczbą argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody rodzajowej, należy podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36578  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu metody ogólnej, które są wywoływane, Dodaj argumenty typu do listy argumentów typu.
