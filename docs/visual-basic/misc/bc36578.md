# <a name="too-few-type-arguments"></a>Za mało argumentów typu

Metoda ogólna została wywołana z mniejszą liczbą argumentów niż liczba parametrów typu.  
  
 Po wywołaniu metody ogólnej należy podać jeden argument typu dla każdego parametru typu zdefiniowanego przez tę metodę.  
  
 **Identyfikator błędu:** BC36578  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj argumenty typu do listy argumentów typu, tak że istnieje jeden dla każdego parametru typu metody generycznej, która jest wywoływana.
