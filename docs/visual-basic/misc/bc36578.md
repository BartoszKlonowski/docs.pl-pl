# <a name="too-few-type-arguments"></a>Za mało argumentów typu
Metoda ogólna została wywołana z mniejszą liczbę argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody ogólnej, musisz podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36578  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu ogólnego metody, która jest należy dodać argumentów typu do listy argumentów typu.
