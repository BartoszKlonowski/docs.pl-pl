# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>Operator " \<operatorname> " nie jest zdefiniowany dla typów " \<type1> " i " \<type2> "
Podjęto próbę użycia operatora binarnego w celu wykonania operacji, która nie jest prawidłowa dla tych elementów kodu zwracających wartość.  
  
 **Identyfikator błędu:** BC30452  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź dwa elementy i upewnij się, że są one zgodne.
