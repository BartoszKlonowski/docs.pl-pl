# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>Operator &#39; &lt;operatorname&gt; &#39; nie jest zdefiniowany dla typów &#39; &lt;type1&gt; &#39; i &#39; &lt;type2&gt;&#39;
Podjęto próbę użycia operatora binarnego do wykonywania określonej operacji, która jest nieprawidłowa dla dwóch zwracającego wartość elementy kodu związanego.  
  
 **Identyfikator błędu:** BC30452  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź dwa elementy i upewnij się, że są one zgodne.
