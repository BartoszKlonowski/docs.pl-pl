# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Przeciążenia nie powiodło się, ponieważ nie jest dostępny '\<metody >' można wywołać z tymi argumentami:\<błąd >
Wprowadzono wywołanie przeciążonej metody, ale kompilator nie można odnaleźć metody o dopasowania listy argumentów.  
  
 **Identyfikator błędu:** BC30518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź listę argumentów, podane.
