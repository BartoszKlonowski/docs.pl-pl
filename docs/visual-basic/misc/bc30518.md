# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Rozpoznanie przeciążenia nie powiodło się, ponieważ żaden z dostępnych " \<method> " nie może być wywołany z tymi argumentami:\<error>
Wykonano wywołanie przeciążonej metody, ale kompilator nie może odnaleźć metody ze zgodną listą argumentów.  
  
 **Identyfikator błędu:** BC30518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podaną listę argumentów.
