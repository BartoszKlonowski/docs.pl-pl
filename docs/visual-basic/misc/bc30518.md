# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>Przeciążenia nie powiodło się, ponieważ żaden z dostępnych &#39; &lt;metody&gt; &#39; może być wywołany z tymi argumentami:&lt;błąd&gt;
Wywołanie przeciążonej metody zostały wykonane, ale kompilator nie można odnaleźć metody o dopasowania listy argumentów.  
  
 **Identyfikator błędu:** BC30518

-   Sprawdź podane listy argumentów.
