# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>Przeciążenia nie powiodło się, ponieważ nie jest dostępny "&lt;metoda&gt;" można wywołać z tymi argumentami:&lt;błąd&gt;
Wprowadzono wywołanie przeciążonej metody, ale kompilator nie można odnaleźć metody o dopasowania listy argumentów.  
  
 **Identyfikator błędu:** BC30518  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź listę argumentów, podane.
