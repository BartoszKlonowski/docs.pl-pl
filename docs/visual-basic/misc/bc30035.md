# <a name="syntax-error-visual-basic-error"></a>Błąd składniowy (błąd w Visual Basic)
Wiersz kodu źródłowego generuje nieokreślony błąd.  
  
 **Identyfikator błędu:** BC30035  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zbadanie dokumentacji dla każdego słowa kluczowe używane w wierszu kodu źródłowego. Porównanie przykładowego kodu w dokumentacji względem wiersza źródłowego, wygenerowanie tego błędu.  
  
2. Jeśli nie można zidentyfikować przyczyny tego błędu, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
