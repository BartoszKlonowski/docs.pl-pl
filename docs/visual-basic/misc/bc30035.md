# <a name="syntax-error-visual-basic-error"></a>Błąd składniowy (Visual Basic błąd)

Wiersz kodu źródłowego generuje błąd składniowy nieokreślony.  
  
 **Identyfikator błędu:** BC30035  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją dla każdego słowa kluczowego używanego w wierszu kodu źródłowego. Porównaj przykładowy kod w dokumentacji z wierszem źródłowym generującym ten błąd.  
  
2. Jeśli nie możesz zidentyfikować przyczyny błędu, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
