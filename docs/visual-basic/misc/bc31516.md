# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>\<typename>Nie można użyć typu "" jako atrybutu, ponieważ nie jest on zadeklarowany jako "Public"
Atrybuty muszą być publiczne.  
  
 **Identyfikator błędu:** BC31516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj `Public` modyfikator do klasy Attribute.
