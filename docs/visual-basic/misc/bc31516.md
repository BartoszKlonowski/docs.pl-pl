# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Typ "&lt;typename&gt;" nie można użyć jako atrybutu, ponieważ nie jest zadeklarowany jako "Public"
Atrybuty muszą być publiczne.  
  
 **Identyfikator błędu:** BC31516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `Public` modyfikatora klasy atrybutu.
