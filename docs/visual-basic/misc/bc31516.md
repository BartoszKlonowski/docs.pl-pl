# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Wpisz &#39; &lt;typename&gt;&#39; nie można użyć jako atrybutu, ponieważ nie jest zadeklarowany jako &#39; Publicznego &#39;
Atrybuty muszą być publiczne.  
  
 **Identyfikator błędu:** BC31516  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `Public` modyfikator klasy atrybutu.
