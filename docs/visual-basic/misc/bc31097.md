# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Element członkowski &#39; &lt;membername&gt; &#39; nie można znaleźć w klasie &#39; &lt;classname&gt;&#39;
Element członkowski "\<membername >' nie można odnaleźć w klasie\<classname >". Ta sytuacja jest zazwyczaj wynikiem niedopasowania biblioteki "Microsoft.VisualBasic.dll".  
  
 Nie można odnaleźć zdefiniowanego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC31097  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Skompiluj program, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
2.  Jeśli błąd będzie się powtarzał, Zapisz swoją pracę i ponownie uruchom [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie Visual Basic.  
  
4.  Jeśli błąd będzie nadal występować po ponownej instalacji, powiadamia pomocy technicznej firmy Microsoft.
