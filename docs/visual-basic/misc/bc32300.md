# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>Istnieje już obiekt z powodu błędu kompilacji lub usunięcia
Wystąpił błąd dostępu do elementu kodu wewnątrz kompilator Visual Basic.  
  
 **Identyfikator błędu:** BC32300  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Skompiluj program, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
2.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie kompilator Visual Basic.  
  
3.  Jeśli błąd będzie nadal występować po ponownej instalacji, zebrać informacje dotyczące okoliczności i powiadomić pomocy technicznej firmy Microsoft.
