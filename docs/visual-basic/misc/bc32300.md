# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>Obiekt nie istnieje z powodu błędu kompilacji lub usunięcia
Błąd dostępu do elementu kodu wystąpił wewnątrz kompilator Visual Basic.  
  
 **Identyfikator błędu:** BC32300  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Skompiluj program ponownie, aby zobaczyć, jeśli błąd będzie się powtarzać.  
  
2. Jeśli ten błąd wystąpi, zainstaluj ponownie kompilator Visual Basic.  
  
3. Jeśli ten błąd będzie występował po ponownej instalacji, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
