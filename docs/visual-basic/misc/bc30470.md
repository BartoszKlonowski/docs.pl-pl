# <a name="myclass-cannot-be-used-outside-of-a-class"></a>Nie można użyć elementu "MyClass" poza klasą
Wykorzystano `MyClass` — słowo kluczowe poza strukturą klasy.  
  
 **Identyfikator błędu:** BC30470  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś `MyClass` deklaracji w strukturze klas.
