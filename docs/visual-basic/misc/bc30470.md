# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39; MyClass &#39; Nie można użyć poza klasą
Zostały użyte `MyClass` — słowo kluczowe poza struktury klasy.  
  
 **Identyfikator błędu:** BC30470  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `MyClass` deklaracji w strukturze klas.
