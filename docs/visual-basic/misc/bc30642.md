# <a name="optional-and-paramarray-cannot-be-combined"></a>Nie można połączyć "Optional" i "ParamArray"
`Optional` i `ParamArray` Modyfikatory wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30642  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz opcję `ParamArray` lub `Optional`, ale nie oba jednocześnie.
