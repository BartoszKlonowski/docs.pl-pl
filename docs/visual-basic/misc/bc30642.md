# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39; opcjonalne &#39; i &#39; ParamArray &#39; Nie można łączyć
`Optional` i `ParamArray` Modyfikatory wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30642  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `ParamArray` lub `Optional`, ale nie oba.
