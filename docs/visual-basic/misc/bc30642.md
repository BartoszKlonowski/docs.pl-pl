# <a name="optional-and-paramarray-cannot-be-combined"></a>Nie można łączyć instrukcji "optional" i "ParamArray"
`Optional` `ParamArray` Modyfikatory i wykluczają się wzajemnie.  
  
 **Identyfikator błędu:** BC30642  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ jeden `ParamArray` lub `Optional` , ale nie oba.
