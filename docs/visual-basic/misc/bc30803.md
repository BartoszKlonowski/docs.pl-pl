# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>"Waluta" nie jest już obsługiwanym typem; Zamiast tego użyj typu "Decimal"
`Currency` — Typ danych został zastąpiony przez `Decimal` typu danych.  
  
 **Identyfikator błędu:** BC30803  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Decimal` typu danych zamiast `Currency`.
