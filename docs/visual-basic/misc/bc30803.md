# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>Typ "Currency" nie jest już obsługiwany. Zamiast tego użyj typu "Decimal"

`Currency`Typ danych został zastąpiony przez `Decimal` Typ danych.  
  
 **Identyfikator błędu:** BC30803  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Decimal` typu danych zamiast `Currency` .
