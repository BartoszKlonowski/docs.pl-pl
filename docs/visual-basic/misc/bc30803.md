# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39; Currency &#39; nie jest już obsługiwanym typem; Użyj &#39; dziesiętnych &#39; Zamiast tego wpisz
`Currency` — Typ danych został zastąpiony przez `Decimal` — typ danych.  
  
 **Identyfikator błędu:** BC30803  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Decimal` zamiast typów danych `Currency`.
