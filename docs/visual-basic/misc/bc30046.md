# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nie może mieć zarówno ParamArray, jak i opcjonalne parametry
A `ParamArray` argument jest opcjonalny automatycznie i musi być tylko opcjonalny argument w deklaracji procedury. Wszystkie argumenty poprzedniego muszą być wymagane.  
  
 **Identyfikator błędu:** BC30046  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usunąć opcjonalne argumenty w deklaracji.
