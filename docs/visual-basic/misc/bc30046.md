# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nie może mieć zarówno ParamArray, jak i Optional
A `ParamArray` argument jest automatycznie opcjonalny i musi być tylko opcjonalny argument w deklaracji procedury. Wszystkie argumenty poprzedniego musi być wymagane.  
  
 **Identyfikator błędu:** BC30046  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń argumentów opcjonalnych w deklaracji.
