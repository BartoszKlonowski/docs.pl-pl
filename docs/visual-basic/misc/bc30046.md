# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nie może mieć zarówno parametru ParamArray, jak i parametrów opcjonalnych
`ParamArray`Argument jest automatycznie opcjonalny i musi być jedynym opcjonalnym argumentem w deklaracji procedury. Wszystkie poprzednie argumenty muszą być wymagane.  
  
 **Identyfikator błędu:** BC30046  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń opcjonalne argumenty w deklaracji.
