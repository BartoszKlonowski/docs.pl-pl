# <a name="field-fieldname-of-type-typename-is-readonly"></a>Pole " \<fieldname> " typu " \<typename> " ma typ "ReadOnly"
Podjęto próbę zapisu do pola, które zostało zadeklarowane `ReadOnly` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień pole, aby nie było już zadeklarowane `ReadOnly` .  
  
- —lub—  
  
- Nie należy próbować zapisywać do tego pola.
