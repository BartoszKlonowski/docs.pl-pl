# <a name="field-ltfieldnamegt-of-type-lttypenamegt-is-readonly"></a>Pole "&lt;fieldname&gt;"typu"&lt;typename&gt;" to "ReadOnly"
Próba zapisu do pola, która jest zadeklarowana `ReadOnly`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość pola, więc nie jest już zadeklarowany jako `ReadOnly`.  
  
-   —lub—  
  
-   Nie należy próbować zapisu do tego pola.
