# <a name="attribute-value-is-not-valid-expecting-value"></a>Wartość atrybutu jest nieprawidłowy; Oczekiwano 'value'
Określono nieprawidłową wartość dla atrybutu literał dokumentu XML. Wartości prawidłowego atrybutu literał dokumentu XML są określane w specyfikacji XML 1.0.  
  
 **Identyfikator błędu:** BC31181  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zaktualizuj wartość atrybutu na prawidłową wartość, jak to określono w specyfikacji XML 1.0.
