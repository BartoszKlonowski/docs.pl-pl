# <a name="attribute-value-is-not-valid-expecting-value"></a>Wartość atrybutu jest nieprawidłowa; Oczekiwano wartości "value"
Określono nieprawidłową wartość atrybutu w literale dokumentu XML. Prawidłowe wartości atrybutów dla literału dokumentu XML są określone w specyfikacji XML 1,0.  
  
 **Identyfikator błędu:** BC31181  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zaktualizuj wartość atrybutu do prawidłowej wartości, jak określono w specyfikacji XML 1,0.
