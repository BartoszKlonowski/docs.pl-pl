# <a name="attribute-value-is-not-valid-expecting-39value39"></a>Wartość atrybutu jest nieprawidłowy; Oczekiwano &#39; wartość &#39;
Określono nieprawidłową wartość atrybutu literał dokumentu XML. Wartości atrybutów Nieprawidłowy literał dokumentu XML są określone w specyfikacji XML 1.0.  
  
 **Identyfikator błędu:** BC31181  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zaktualizuj wartość atrybutu na prawidłową wartość, jak określono w specyfikacji XML 1.0.
