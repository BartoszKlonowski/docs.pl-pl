# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicjatory typu anonimowego nie są prawidłowe w oknach debugowania
Inicjator typu anonimowego został wprowadzony w **czujki** lub **Immediate** okno podczas sesji debugowania. Inicjatory typu anonimowego nie są dostępne w debugowaniu systemu windows.  
  
 **Identyfikator błędu:** BC36577  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przetestuj inicjatory typu anonimowego w edytorze kodu.
