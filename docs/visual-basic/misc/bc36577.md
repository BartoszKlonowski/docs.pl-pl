# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicjatory typu anonimowego są nieprawidłowe w oknach debugowania

W oknie **czujka** lub **natychmiastowe** w trakcie sesji debugowania został wprowadzony inicjator typu anonimowego. Inicjatory typu anonimowego nie są dostępne w oknach debugowania.  
  
 **Identyfikator błędu:** BC36577  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przetestuj inicjatory typu anonimowego w edytorze kodu.
