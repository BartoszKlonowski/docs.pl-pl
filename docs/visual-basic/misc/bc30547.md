# <a name="interface-interfacename-cannot-be-indexed-because-it-has-no-default-property"></a>\<interfacename>Nie można indeksować interfejsu "", ponieważ nie ma on domyślnej właściwości
Wyrażenia indeksu muszą powodować wartość, której typem jest tablica, w wartości, której typ ma zestaw przeciążonych właściwości domyślnych lub w zestawie przeciążonych właściwości. Interfejs może mieć tylko jedną domyślną metodę lub właściwość. Oznacza to, że może dziedziczyć interfejs zawierający domyślną metodę lub właściwość lub jego blok definicji może zawierać jedną metodę lub właściwość, która jest oznaczona jako domyślna.  
  
 **Identyfikator błędu:** BC30547  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj domyślną właściwość w interfejsie.
