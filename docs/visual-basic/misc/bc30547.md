# <a name="interface-interfacename-cannot-be-indexed-because-it-has-no-default-property"></a>Interfejs '\<interfacename >' nie można indeksować, ponieważ nie ma on domyślnej właściwości
Wyrażenia indeksu musi powodować wartość, której typem jest tablicą, wartości, którego typ zawiera zestaw domyślnych przeciążone właściwości lub zbiór przeciążone właściwości. Interfejs może mieć tylko jedną domyślną metodę lub właściwość. Oznacza to, że interfejs zawierający domyślne metody lub właściwości albo może dziedziczyć lub jego definicja blok może zawierać jedną metodę lub właściwość, która jest oznaczona jako domyślna.  
  
 **Identyfikator błędu:** BC30547  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj właściwości domyślnej w interfejsie.
