# <a name="interface-39ltinterfacenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Interfejs &#39; &lt;interfacename&gt;&#39; nie można indeksować, ponieważ nie ma ona domyślnej właściwości
Wyrażenia indeksu musi powodować wartości, którego typ jest tablicą, wartości, którego typ zawiera zestaw domyślnych przeciążone właściwości lub zbiór przeciążone właściwości. Interfejs może mieć tylko jeden domyślny metody lub właściwości. To oznacza, że interfejs zawierający domyślne metody lub właściwości albo może dziedziczyć lub jego definicji bloku może zawierać jeden metody lub właściwości, który jest oznaczony jako domyślny.  
  
 **Identyfikator błędu:** BC30547  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj właściwości domyślnej w interfejsie.
