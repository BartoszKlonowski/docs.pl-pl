# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Nie można używać znaków typu w deklaracjach zmiennych zakresu
Znak typu został uwzględniony na końcu nazwy deklaracji zmiennej zakresu w zapytaniu LINQ.  
  
 **Identyfikator błędu:** BC36601  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń znak typu z deklaracji zmiennej zakresu.
