# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Nie można używać typu znaków w deklaracji zmiennej zakresu
Znak typu została uwzględniona na końcu nazwy deklaracji zmiennej zakresu w zapytaniu programu LINQ.  
  
 **Identyfikator błędu:** BC36601  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń znak typu w deklaracji zmiennej zakresu.
