# <a name="type-cannot-be-inherited-more-than-once"></a>'\<typu >' nie może być dziedziczona więcej niż raz
Zduplikowano `Inherits` instrukcji.  
  
 **Identyfikator błędu:** BC30584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędną `Inherits` instrukcji.
