# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;Typ&gt; &#39; nie może być dziedziczona więcej niż raz
Mają zduplikowane `Inherits` instrukcji.  
  
 **Identyfikator błędu:** BC30584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `Inherits` instrukcji.
