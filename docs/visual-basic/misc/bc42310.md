# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>Tag komentarza XML 'include' musi mieć '\<atrybut >' atrybut
Tag komentarza XML 'include' musi mieć \<atrybut > atrybutu. Komentarz XML zostanie zignorowany.  
  
 A `file` lub `path` atrybutu, przekazana do `<`obejmują`>` tag, został brakujące lub niekompletne.  
  
 **Identyfikator błędu:** BC42310  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodawanie lub wymaganego atrybutu, aby ukończyć \<obejmują > tag.
