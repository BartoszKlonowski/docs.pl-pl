# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>Tag komentarza XML &#39;obejmują&#39; musi mieć &#39; &lt;atrybutu&gt; &#39; atrybutu
Tag komentarza XML "include" musi mieć \<atrybut > atrybutu. Komentarz XML zostanie zignorowany.  
  
 A `file` lub `path` atrybutu przekazany do `<`obejmują`>` tagów, został brakujące lub niekompletne.  
  
 **Identyfikator błędu:** BC42310  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj lub Zakończ wymaganego atrybutu do \<obejmują > tagu.
