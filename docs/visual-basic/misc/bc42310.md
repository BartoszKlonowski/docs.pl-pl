# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>Tag komentarza XML "include" musi mieć \<attribute> atrybut ""

Tag komentarza XML "include" musi mieć \<attribute> atrybut. Komentarz XML zostanie zignorowany.  
  
 `file`Atrybut lub `path` , który został przekazano `<` do `>` znacznika include, nie został odnaleziony lub jest niekompletny.  
  
 **Identyfikator błędu:** BC42310  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj lub Uzupełnij wymagany atrybut do \<include> znacznika.
