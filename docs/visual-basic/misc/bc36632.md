# <a name="nullable-parameters-must-specify-a-type"></a>Dopuszcza wartości null parametrów, należy określić typ
Parametr zawiera modyfikator typu dopuszczającego wartość zerową (?), ale nie określa typu.  
  
 **Identyfikator błędu:** BC36632  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ typ podczas deklarowania parametru dopuszczającego wartość null.
