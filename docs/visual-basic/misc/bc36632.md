# <a name="nullable-parameters-must-specify-a-type"></a>Parametry zerowalne muszą określać typ
Parametr zawiera modyfikator typu dopuszczającego wartości null (?), ale nie ma określonego typu.  
  
 **Identyfikator błędu:** BC36632  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ typ przy deklarowaniu parametru dopuszczającego wartość null.
