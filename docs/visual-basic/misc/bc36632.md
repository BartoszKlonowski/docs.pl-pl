# <a name="nullable-parameters-must-specify-a-type"></a>Parametry dopuszczające wartość null muszą określać typ

Parametr zawiera modyfikator typu dopuszczającego wartość null (?), ale nie określa typu.  
  
 **Identyfikator błędu:** BC36632  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ typ podczas deklarowania parametru typu wartości null.
