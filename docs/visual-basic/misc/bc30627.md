# <a name="option-statements-must-precede-any-declarations-or-imports-statements"></a>Instrukcje "Option" musi poprzedzać wszystkie deklaracji lub instrukcji "Imports"
`Option` Instrukcje powinny być pierwszej instrukcji bez komentarzy w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30627  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś `Option` instrukcji na górze źródła kodu, bezpośrednio przed `Imports` i `Namespace` instrukcji.
