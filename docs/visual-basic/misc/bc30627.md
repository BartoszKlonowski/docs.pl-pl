# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39; opcję &#39; instrukcje musi występować przed wszelkimi deklaracjami lub &#39; importów &#39; instrukcje
`Option`Instrukcje powinny być pierwszej instrukcji komentarza-w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30627  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `Option` instrukcje źródło na początku kodu bezpośrednio przed `Imports` i `Namespace` instrukcje.
