# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Ten system nie obsługuje określonych ustawień regionalnych
Podjęto próbę użycia wyliczenia `VbStrConv`, ale system nie obsługuje określonych ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Uaktualnij maszynę, aby obsługiwała ustawienia regionalne, lub Uruchom aplikację na komputerze obsługującym tę funkcję.
