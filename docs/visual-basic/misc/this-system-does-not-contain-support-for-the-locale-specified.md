# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Ten system nie obsługuje podanych ustawień regionalnych
Podjęto próbę użycia `VbStrConv` wyliczenie, ale system nie obsługuje określonych ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Uaktualnienie komputera do obsługi ustawień regionalnych lub uruchom aplikację na komputerze, który ją obsługuje.
