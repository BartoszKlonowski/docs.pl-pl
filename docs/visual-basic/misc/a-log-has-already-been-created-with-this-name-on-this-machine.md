# <a name="a-log-has-already-been-created-with-this-name-on-this-machine"></a>Dziennik został już utworzony przy użyciu tej nazwy na tym komputerze

Podjęto próbę utworzenia dziennika o nazwie, która jest już używana.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń dziennik powodujący konflikt.  
  
- Użyj innej nazwy dla dziennika.
