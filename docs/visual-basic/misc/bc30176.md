# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>Można określić tylko jedną "Public", "Private", "Protected", "Friend" lub "Protected Friend"
Możliwości dostępu do powodujące konflikt występować w deklaracji elementu. Można określić tylko jeden ułatwień dostępu.  
  
 **Identyfikator błędu:** BC30176  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz jeden ułatwień dostępu i usuwanie innych słów kluczowych z deklaracji.
