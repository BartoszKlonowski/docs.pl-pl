# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>Można określić tylko jedną z elementów "Public", "Private", "Protected", "Friend" lub "Protected Friend"

W deklaracji elementu wystąpił konflikt podano. Można określić tylko jedną dostępność.  
  
 **Identyfikator błędu:** BC30176  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz jedną dostępność i Usuń inne słowa kluczowe z deklaracji.
