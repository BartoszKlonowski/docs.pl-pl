# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Tylko jeden z &#39;publicznego&#39;, &#39;prywatnej&#39;, &#39;chronionym&#39;, &#39;Friend&#39;, lub &#39;Protected Friend&#39; można określić
W deklaracji elementu występować dostępności powodujące konflikt. Można określić tylko jeden ułatwień dostępu.  
  
 **Identyfikator błędu:** BC30176  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz jeden ułatwień dostępu i usunąć innych słów kluczowych z deklaracji.
