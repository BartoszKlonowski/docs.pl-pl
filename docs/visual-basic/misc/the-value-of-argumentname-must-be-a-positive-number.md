# <a name="the-value-of-ltargumentnamegt-must-be-a-positive-number"></a>Wartość &lt;argumentname&gt; musi być liczbą dodatnią
Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> właściwość musi być większa od zera.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> Właściwość określa ilość wolnego miejsca na dysku, w bajtach, niezbędne przed komunikaty mogą być zapisywane w pliku dziennika  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> właściwości na wartość dodatnią.
