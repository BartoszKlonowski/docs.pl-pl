# <a name="the-value-of-argumentname-must-be-a-positive-number"></a>Wartość \<argumentname> musi być liczbą dodatnią.

Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> właściwości musi być większa od zera.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A>Właściwość określa ilość wolnego miejsca na dysku (w bajtach), które są niezbędne przed zapisaniem komunikatów w pliku dziennika  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> Właściwość na liczbę dodatnią.
