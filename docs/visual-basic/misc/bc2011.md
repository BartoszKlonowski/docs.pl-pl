# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>Nie można otworzyć pliku odpowiedzi &#39; &lt;filename&gt;&#39;
Plik, który próbujesz uzyskać dostęp, jest niedostępna.  
  
 **Identyfikator błędu:** BC2011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że połączenie sieciowe jest nadal aktywne.  
  
-   Sprawdź swoje uprawnienia do pliku.
