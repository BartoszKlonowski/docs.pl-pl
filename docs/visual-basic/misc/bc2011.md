# <a name="unable-to-open-response-file-filename"></a>Nie można otworzyć pliku odpowiedzi '\<nazwa pliku >'
Plik, który próbujesz uzyskać dostęp, jest niedostępny.  
  
 **Identyfikator błędu:** BC2011  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że połączenie sieciowe jest nadal aktywne.  
  
- Sprawdź swoje uprawnienia do pliku.
