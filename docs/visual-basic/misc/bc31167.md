# <a name="expected--for-xml-end-tag"></a>Oczekiwano słowa kluczowego ' /' w tagu końcowego dla formatu XML
Tag końcowy XML, aby uzyskać literał XML brakuje wymaganego "/" znak.  
  
 **Identyfikator błędu:** BC31167  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak "/" na początku tagu końcowego XML literał XML.
