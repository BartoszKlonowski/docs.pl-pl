# <a name="expected--for-xml-end-tag"></a>Oczekiwano znaku "/" dla tagu końcowego XML
Tag końcowy XML dla literału XML nie zawiera wymaganego znaku "/".  
  
 **Identyfikator błędu:** BC31167  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak "/" na początku tagu końcowego XML dla literału XML.
