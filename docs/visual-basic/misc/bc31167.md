# <a name="expected-3939-for-xml-end-tag"></a>Oczekiwano &#39; / &#39; dla tagu końcowego XML
Brak tagu końcowego XML dla literał XML wymagany "/" znak.  
  
 **Identyfikator błędu:** BC31167  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znak '/', początek tagu końcowego XML dla literału XML.
