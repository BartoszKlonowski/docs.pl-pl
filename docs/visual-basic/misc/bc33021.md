# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-typename"></a>Co najmniej jeden typ parametru tego operatora binarnego musi być typem zawierającym " \<typename> "

Definicja operatora binarnego określa oba parametry o typach innych niż Klasa lub struktura, w której jest zdefiniowany operator.  
  
 Podczas definiowania operatora w klasie lub strukturze, co najmniej jeden z parametrów musi być typu tej klasy lub struktury.  
  
 **Identyfikator błędu:** BC33021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień typ jednego lub obu parametrów na typ klasy lub struktury, w której jest zdefiniowany operator.
