# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-39lttypenamegt39"></a>Co najmniej jeden typ parametr tego operatora binarnego musi być typem zawierającym &#39; &lt;typename&gt;&#39;
Definicja operatora binarnego określa oba parametry o typach inną niż klasy lub struktury, w którym jest zdefiniowany operator.  
  
 Po zdefiniowaniu operatora w klasie lub strukturze co najmniej jeden z parametrów musi być typu tej klasy lub struktury.  
  
 **Identyfikator błędu:** BC33021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ co najmniej jeden z parametrów na typ klasy lub struktury, w którym jest zdefiniowany operator.
