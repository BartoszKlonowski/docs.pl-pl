# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-typename"></a>Co najmniej jeden typ parametr tego operatora binarnego musi być typem zawierającym "\<typename >"
Definicję operatora binarnego określa oba parametry, z typów innych niż klasa lub struktura, w którym zdefiniowano operator.  
  
 Po zdefiniowaniu operatora w klasie lub strukturze co najmniej jeden z parametrów musi być typu tej klasy lub struktury.  
  
 **Identyfikator błędu:** BC33021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ co najmniej jeden z parametrów na typ klasy lub struktury, w którym zdefiniowano operator.
