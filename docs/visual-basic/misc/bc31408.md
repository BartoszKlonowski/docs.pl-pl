# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>"&lt;specifier1&gt;"i"&lt;specifier2&gt;" nie może być łączone
Dwa specyfikatory niezgodne, takie jak `Shadows` i `Overrides` są używane razem.  
  
 **Identyfikator błędu:** BC31408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń co najmniej innych specyfikator.
