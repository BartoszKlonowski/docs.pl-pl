# <a name="specifier1-and-specifier2-cannot-be-combined"></a>'\<specifier1 >' i '\<specifier2 >' nie może być łączone
Dwa specyfikatory niezgodne, takie jak `Shadows` i `Overrides` są używane razem.  
  
 **Identyfikator błędu:** BC31408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń co najmniej innych specyfikator.
