# <a name="specifier1-and-specifier2-cannot-be-combined"></a>\<specifier1>nie można łączyć "" i " \<specifier2> "
Dwa niezgodne specyfikatory, takie jak `Shadows` i `Overrides` są używane razem.  
  
 **Identyfikator błędu:** BC31408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń jeden lub inny specyfikator.
