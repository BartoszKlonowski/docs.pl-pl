# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39;&lt;specifier1&gt; &#39; i &#39; &lt;specifier2&gt; &#39; nie można łączyć
Dwa niezgodne Specyfikatory, takich jak `Shadows` i `Overrides` są używane razem.  
  
 **Identyfikator błędu:** BC31408  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usunięcie jednego lub innym specyfikatorem.
