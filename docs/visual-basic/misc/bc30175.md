# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; "&lt;typename&gt;" i &lt;type2&gt; "&lt;typename&gt;", zadeklarowanych w"&lt;nazwa&gt;", powoduje konflikt w elemencie &lt;przestrzeni nazw &gt; "&lt;namespacename&gt;"
Dwa zadeklarowane elementy, takie jak klasy i moduł, mają taką samą nazwę w tej samej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30175  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś deklarację jeden z elementów do innej przestrzeni nazw.  
  
-   Zmień nazwę jednego z elementów.
