# <a name="type1-typename-and-type2-typename-declared-in-name-conflict-in-namespace-namespacename"></a>\<type1> " \<typename> " i \<type2> " \<typename> " zadeklarowane w elemencie " \<name> " powodują konflikt \<namespace> w \<namespacename> elemencie ""

Dwa zadeklarowane elementy, takie jak Klasa i moduł, mają taką samą nazwę w tej samej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30175  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś deklarację jednego z elementów do innej przestrzeni nazw.  
  
- Zmień nazwę jednego z elementów.
