# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>Nie można znaleźć punktu wejścia " \<name> " w bibliotece dll " \<filename> ":\<error>
Nie można odnaleźć nazwy procedury w podanej bibliotece DLL.  
  
 **Identyfikator błędu:** BC31014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy nazwa procedury jest wpisana prawidłowo i czy wielkość liter pasuje do funkcji w bibliotece DLL.
