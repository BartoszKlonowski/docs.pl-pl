# <a name="unable-to-find-entry-point-39ltnamegt39-in-dll-39ltfilenamegt39-lterrorgt"></a>Nie można odnaleźć punktu wejścia &#39; &lt;nazwa&gt;&#39; w pliku DLL &#39;&lt; Nazwa pliku&gt;&#39;: &lt;błąd&gt;
Nie można odnaleźć nazwy procedury w określonej bibliotece DLL.  
  
 **Identyfikator błędu:** BC31014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy nazwa procedury została wpisana poprawnie i że wielkość liter jest zgodna z wersją funkcji w bibliotece DLL.
