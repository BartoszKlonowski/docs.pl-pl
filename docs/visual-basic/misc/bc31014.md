# <a name="unable-to-find-entry-point-ltnamegt-in-dll-ltfilenamegt-lterrorgt"></a>Nie można odnaleźć punktu wejścia "&lt;nazwa&gt;"w pliku DLL"&lt;filename&gt;": &lt;błąd&gt;
Nie można odnaleźć nazwy procedury w określonej bibliotece DLL.  
  
 **Identyfikator błędu:** BC31014  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź nazwę procedury została wpisana poprawnie i czy małych i wielkich liter jest zgodny z typem funkcji w DLL.
