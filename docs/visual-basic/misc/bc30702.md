# <a name="index-39ltindexnumbergt39-for-dimension-39ltdimensionnumbergt39-is-out-of-range"></a>Indeks &#39; &lt;numer_indeksu&gt; &#39; wymiaru &#39; &lt;dimensionnumber&gt; &#39; jest poza zakresem
Jednym z indeksów przekracza zakres zdefiniowany.  
  
 **Identyfikator błędu:** BC30702  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień logikę procedury, aby granice tablicy nie zostaną przekroczone.  
  
-   Ponowna definicja tablicy, tak aby umożliwić większej liczbie wartości.
