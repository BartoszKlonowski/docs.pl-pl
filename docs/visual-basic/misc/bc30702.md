# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>Indeks " \<indexnumber> " dla wymiaru " \<dimensionnumber> " jest poza zakresem

Jeden z indeksów przekracza zdefiniowany zakres.  
  
 **Identyfikator błędu:** BC30702  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień logikę procedury tak, aby granice tablicy nie zostały przekroczone.  
  
- Ponownie zdefiniuj tablicę, aby zezwalała na szerszy zakres wartości.
