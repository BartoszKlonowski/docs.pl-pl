# <a name="index-ltindexnumbergt-for-dimension-ltdimensionnumbergt-is-out-of-range"></a>Indeks "&lt;numer_indeksu&gt;"dla wymiaru"&lt;dimensionnumber&gt;" jest poza zakresem
Jednym z indeksów przekracza zakres zdefiniowany.  
  
 **Identyfikator błędu:** BC30702  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień logikę procedury, aby nie przekroczono granice tablicy.  
  
-   Ponownie zdefiniować tablicę, aby zezwolić na szerszy zakres wartości.
