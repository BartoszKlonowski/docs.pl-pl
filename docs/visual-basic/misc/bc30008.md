# <a name="reference-required-to-module-modulename-containing-the-base-class-classname"></a>Wymagane odwołanie do modułu " \<modulename> " zawierające klasę bazową " \<classname> "

Wymagane odwołanie do modułu " \<modulename> " zawierające klasę bazową " \<classname> ". Dodaj je do projektu.  
  
 Klasa jest zdefiniowana w module, który nie jest bezpośrednio przywoływany w projekcie. Kompilator Visual Basic wymaga odwołania, aby uniknąć niejednoznaczności na wypadek, gdyby Klasa została zdefiniowana w więcej niż jednym module.  
  
 **Identyfikator błędu:** BC30008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dołącz nazwę modułu, do którego nie ma odwołania, w odwołaniach do projektu.
