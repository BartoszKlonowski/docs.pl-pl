# <a name="reference-required-to-module-modulename-containing-the-base-class-classname"></a>Wymagane odwołanie do modułu '\<nazwa_modułu >' z klasą bazową\<nazwa_klasy > '
Wymagane odwołanie do modułu '\<nazwa_modułu >' z klasą bazową\<nazwa_klasy > '. Dodaj je do projektu.  
  
 Klasa jest zdefiniowana w module, który nie jest bezpośrednio wywoływany w projekcie. Kompilator Visual Basic wymaga odwołania, aby uniknąć niejednoznaczności, w przypadku, gdy klasa jest zdefiniowana w więcej niż jeden moduł.  
  
 **Identyfikator błędu:** BC30008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- W odwołania projektu, należy uwzględnić nazwę bez odwołań modułu.
