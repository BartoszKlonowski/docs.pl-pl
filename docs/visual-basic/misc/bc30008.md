# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Wymagane odwołanie do modułu &#39; &lt;modulename&gt; &#39; z klasą podstawową &#39; &lt;classname&gt;&#39;
Wymagane odwołanie do modułu "\<nazwa_modułu >" zawierającego klasę podstawową\<classname > ". Dodaj je do projektu.  
  
 Klasa jest zdefiniowana w module, który nie jest bezpośrednio wywoływany w projekcie. Kompilator Visual Basic wymaga odwołania, aby uniknąć niejednoznaczności w przypadku, gdy klasa jest zdefiniowana w więcej niż jeden moduł.  
  
 **Identyfikator błędu:** BC30008  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Uwzględnij nazwę modułu bez odwołań w odwołaniach do projektu.
