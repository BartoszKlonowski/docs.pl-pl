# <a name="baselogname-cannot-be-nothing-or-an-empty-string"></a>BaseLogName nie może być Nothing lub pusty ciąg
Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> właściwość nie może być `Nothing` ani być pustym ciągiem.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> Właściwość określa nazwę podstawową dla plików dziennika.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> właściwości na ciąg, który zawiera co najmniej jeden znak.
