# <a name="baselogname-cannot-be-nothing-or-an-empty-string"></a>BaseLogName nie może mieć wartości Nothing ani być ciągiem pustym

Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> właściwości nie może być `Nothing` ciągiem pustym.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A>Właściwość określa podstawową nazwę plików dziennika.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> Właściwość na ciąg, który zawiera co najmniej jeden znak.
