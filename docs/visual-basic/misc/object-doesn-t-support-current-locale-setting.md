# <a name="object-doesnt-support-current-locale-setting-visual-basic"></a>Obiekt nie obsługuje bieżących ustawień regionalnych (Visual Basic)
Próbowano uzyskać dostęp do obiektu, który nie obsługuje ustawień regionalnych dla bieżącego projektu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź ustawienia regionalne, który obsługuje obiektu.
