# <a name="file-specified-by-filename-does-not-use-the-encoding-specified-by-fileencoding"></a>Plik określony przez FileName nie używa kodowania określonego przez FileEncoding
Podjęto próbę odczytu z pliku, ale określone kodowanie nie jest używane przez plik.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że podasz poprawne kodowanie pliku.
