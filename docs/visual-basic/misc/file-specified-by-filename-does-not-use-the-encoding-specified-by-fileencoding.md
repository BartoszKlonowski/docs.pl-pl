# <a name="file-specified-by-filename-does-not-use-the-encoding-specified-by-fileencoding"></a>Plik określony przez nazwę pliku nie korzysta z kodowaniem określonym przez FileEncoding
Podjęto próbę można odczytać z pliku, ale określone kodowanie nie jest używane przez plik.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, upewnij się, że są podawania kodowanie prawidłowego pliku.
