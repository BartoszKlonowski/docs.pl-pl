# <a name="property-missing-end-property"></a>Właściwość "End Property" Brak
`End` Instrukcja musi odpowiadać blok zostaje zakończone. A `Property` definicja została nieprawidłowo zakończona.  
  
 **Identyfikator błędu:** BC30025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Property` instrukcji na końcu definicji właściwości.
