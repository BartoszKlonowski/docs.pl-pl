# <a name="property-missing-39end-property39"></a>Brak właściwości &#39;właściwość End&#39;
`End` Instrukcja musi odpowiadać bloku zostaje zakończone. A `Property` definicji nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC30025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Property` instrukcji na końcu definicji właściwości.
