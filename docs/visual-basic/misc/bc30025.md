# <a name="property-missing-end-property"></a>Brak właściwości "End Property"
`End`Instrukcja musi być zgodna z zamykanym blokiem. `Property`Definicja nie została prawidłowo zakończona.  
  
 **Identyfikator błędu:** BC30025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `End Property` instrukcję do końca definicji właściwości.
