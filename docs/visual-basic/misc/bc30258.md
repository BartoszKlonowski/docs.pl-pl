# <a name="classes-can-inherit-only-from-other-classes"></a>Klasy mogą dziedziczyć tylko z innych klas
`Inherits` Instrukcja określa, jak jego typ podstawowy typu innego niż klasa.  
  
 **Identyfikator błędu:** BC30258  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ podstawowy do klasy, która jest już zdefiniowany.
