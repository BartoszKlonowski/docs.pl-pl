# <a name="classes-can-inherit-only-from-other-classes"></a>Klasy mogą dziedziczyć tylko z innych klas

`Inherits`Instrukcja określa typ inny niż Klasa jako typ podstawowy.  
  
 **Identyfikator błędu:** BC30258  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień typ podstawowy na klasę, która jest już zdefiniowana.
