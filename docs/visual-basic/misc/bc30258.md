# <a name="classes-can-inherit-only-from-other-classes"></a>Klasy mogą dziedziczyć tylko innych klas
`Inherits` Instrukcji określa typ innych niż klasa jako jego typ podstawowy.  
  
 **Identyfikator błędu:** BC30258  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień typ podstawowy do klasy, która jest już zdefiniowany.
