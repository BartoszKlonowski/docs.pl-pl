# <a name="readonly-and-writeonly-cannot-be-combined"></a>Nie można łączyć typu "ReadOnly" i "WriteOnly"
`ReadOnly` `WriteOnly` Modyfikatory i wykluczają się wzajemnie.  
  
 **Identyfikator błędu:** BC30635  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ jeden `ReadOnly` lub `WriteOnly` , ale nie oba.
