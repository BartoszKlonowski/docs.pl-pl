# <a name="readonly-and-writeonly-cannot-be-combined"></a>Nie można połączyć "ReadOnly" i "WriteOnly"
`ReadOnly` i `WriteOnly` Modyfikatory wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30635  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz opcję `ReadOnly` lub `WriteOnly`, ale nie oba jednocześnie.
