# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39; Tylko do odczytu &#39; i &#39; WriteOnly &#39; Nie można łączyć
`ReadOnly` i `WriteOnly` Modyfikatory wzajemnie się wykluczają.  
  
 **Identyfikator błędu:** BC30635  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `ReadOnly` lub `WriteOnly`, ale nie oba.
