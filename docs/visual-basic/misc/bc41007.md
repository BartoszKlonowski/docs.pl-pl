# <a name="attribute-conditional-is-only-valid-on-sub-declarations"></a>Atrybut "Conditional" jest prawidłowy tylko w deklaracjach "Sub"
Ten <xref:System.Diagnostics.ConditionalAttribute> atrybut został zastosowany do deklaracji metody innej niż `Sub` procedura.  
  
 **Identyfikator błędu:** BC41007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń <xref:System.Diagnostics.ConditionalAttribute> z nieprawidłowej deklaracji.
