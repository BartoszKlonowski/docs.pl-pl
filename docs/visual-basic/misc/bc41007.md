# <a name="attribute-conditional-is-only-valid-on-sub-declarations"></a>Atrybut "Conditional" jest prawidłowy tylko w deklaracjach "Sub"
<xref:System.Diagnostics.ConditionalAttribute> Zastosowano atrybut do deklaracji metody innej niż `Sub` procedury.  
  
 **Identyfikator błędu:** BC41007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń <xref:System.Diagnostics.ConditionalAttribute> z nieprawidłową deklaracji.
