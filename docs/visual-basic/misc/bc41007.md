# <a name="attribute-39conditional39-is-only-valid-on-39sub39-declarations"></a>Atrybut &#39;warunkowego&#39; jest prawidłowy tylko w &#39;Sub&#39; deklaracji
<xref:System.Diagnostics.ConditionalAttribute> Atrybutu zastosowaniu deklaracji metody inne niż `Sub` procedury.  
  
 **Identyfikator błędu:** BC41007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń <xref:System.Diagnostics.ConditionalAttribute> z nieprawidłową deklaracją.
