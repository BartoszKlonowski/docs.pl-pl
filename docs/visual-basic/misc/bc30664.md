# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Konwersje z &#39; &lt;typename1&gt; &#39; do &#39; &lt;typename2&gt; &#39; muszą być określone jawnie
Nie można niejawnie dokonać konwersji.  
  
 **Identyfikator błędu:** BC30664  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `CType`, lub jeden z określonych konwersji funkcji, takich jak `CInt` lub `CDec`.
