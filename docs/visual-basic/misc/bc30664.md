# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>Konwersje z " \<typename1> " na " \<typename2> " muszą być jawne
Tej konwersji nie można wykonać niejawnie.  
  
 **Identyfikator błędu:** BC30664  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `CType` lub jednej z określonych funkcji konwersji, takich jak `CInt` lub `CDec` .
