# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>Konwersje z elementu "\<typename1 >" do "\<typename2 >" musi być jawne
Ta konwersja nie może wykonać niejawnie.  
  
 **Identyfikator błędu:** BC30664  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `CType`, lub jeden z określonych konwersji funkcje, takie jak `CInt` lub `CDec`.
