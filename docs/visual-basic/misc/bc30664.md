# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>Konwersje z elementu "&lt;typename1&gt;"to"&lt;typename2&gt;" musi być jawne
Ta konwersja nie może wykonać niejawnie.  
  
 **Identyfikator błędu:** BC30664  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `CType`, lub jeden z określonych konwersji funkcje, takie jak `CInt` lub `CDec`.
