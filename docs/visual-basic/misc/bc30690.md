# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Struktura &#39; &lt;structurename&gt; &#39; nie można indeksować, ponieważ nie ma ona domyślnej właściwości
Próbowano użyć składni właściwości domyślnej ze strukturą, która nie ma domyślnej właściwości.  
  
 **Identyfikator błędu:** BC30690  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Aby uzyskać dostęp do właściwości struktury, należy użyć składni standardowe właściwości.
