# <a name="structure-structurename-cannot-be-indexed-because-it-has-no-default-property"></a>Struktura '\<structurename >' nie można indeksować, ponieważ nie ma on domyślnej właściwości
Nastąpiła próba Składnia właściwości domyślne za pomocą strukturę, która nie ma domyślnej właściwości.  
  
 **Identyfikator błędu:** BC30690  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy użyć składni właściwości standardowe, uzyskiwania dostępu do właściwości struktury.
