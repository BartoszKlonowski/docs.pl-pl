# <a name="structure-structurename-cannot-be-indexed-because-it-has-no-default-property"></a>\<structurename>Nie można indeksować struktury "", ponieważ nie ma ona domyślnej właściwości.
Podjęto próbę użycia domyślnej składni właściwości ze strukturą, która nie ma domyślnej właściwości.  
  
 **Identyfikator błędu:** BC30690  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj standardowej składni właściwości, aby uzyskać dostęp do właściwości struktury.
