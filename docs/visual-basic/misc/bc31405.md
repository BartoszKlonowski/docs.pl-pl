# <a name="optional-parameters-cannot-have-structure-types"></a>Parametry opcjonalne nie mogą mieć typów struktury
Opcjonalne parametry nie mogą być typami struktury.  
  
 **Identyfikator błędu:** BC31405  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zadeklaruj parametry typu struktury zgodnie z wymaganiami.  
  
2. Zadeklaruj statyczny element członkowski klasy, aby przyjąć strukturę.
