# <a name="optional-parameters-cannot-have-structure-types"></a>Parametry opcjonalne nie mogą mieć typów struktury
Parametry opcjonalne nie mogą być typy struktur.  
  
 **Identyfikator błędu:** BC31405  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Deklarowanie parametrów typu struktury, zgodnie z wymaganiami.  
  
2.  Deklaruje elementu członkowskiego klasy statycznej została wykonana struktury.
