# <a name="optional-parameters-cannot-have-structure-types"></a>Parametry opcjonalne nie może mieć typy struktur
Parametry opcjonalne nie może być typy struktur.  
  
 **Identyfikator błędu:** BC31405  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Deklarowanie struktury parametrów typu zgodnie z potrzebami.  
  
2.  Deklaruje składowej klasy statycznej wyznacz struktury.
