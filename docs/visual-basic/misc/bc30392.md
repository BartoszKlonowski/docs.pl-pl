# <a name="catch-cannot-catch-type-typename-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>Instrukcji 'catch' nie może przechwycić typu '\<typename >', ponieważ nie jest to 'System.Exception' ani klasa, która dziedziczy po 'System.Exception'
`Catch` można tylko przechwycić wyjątki, a następnie podjęto próbę catch coś, co nie jest tworzony na podstawie wyjątek.  
  
 **Identyfikator błędu:** BC30392  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Catch` instrukcji, lub zmień obiekt docelowy `Catch` do faktyczny wyjątek.
