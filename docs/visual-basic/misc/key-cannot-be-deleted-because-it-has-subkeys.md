# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>Nie można usunąć klucza, ponieważ ma on podklucze
Nie można usunąć klucza, ponieważ ma on podklucze.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przed podjęciem próby wykonania tej operacji Ustal i Usuń podklucze określonego klucza.
