# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>Typy dopuszczające wartości null nie są obsługiwane, ponieważ bieżąca wersja elementu "mscorlib. dll" nie definiuje typu "Nullable (Of T)"
Używana wersja platformy nie obsługuje typów wartości dopuszczających wartość null.  
  
 **Identyfikator błędu:** BC32130
