# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>Typy nullable nie są obsługiwane, ponieważ bieżąca wersja 'mscorlib.dll' nie definiuje typu "Nullable(of T)"
Wersja struktury, której używasz, nie obsługuje typów wartości z dopuszczalnym do wartości null.  
  
 **Identyfikator błędu:** Bc32130
