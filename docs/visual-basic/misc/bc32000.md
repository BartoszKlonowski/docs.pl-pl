# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Zmienna lokalna &#39; &lt;nazwa_zmiennej&gt; &#39; nie może odwoływać się do przed jej zadeklarowaniem
Zmienna w procedurze jest używana przed jej deklaracją.  
  
 **Identyfikator błędu:** BC32000  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś deklarację poprzedzającą pierwsze odwołanie do zmiennej.
