# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>\<variablename>Nie można odwołać się do zmiennej lokalnej "" przed jej zadeklarowaniem
Zmienna wewnątrz procedury jest używana przed jej deklaracją.  
  
 **Identyfikator błędu:** BC32000  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś deklarację, aby poprzedzać pierwsze odwołanie do zmiennej.
