# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>Zmienna lokalna '\<nazwa_zmiennej >' nie mogą odwoływać się przed jej zadeklarowaniem
Zmienna wewnątrz procedury, jest używana przed zadeklarowaniem.  
  
 **Identyfikator błędu:** BC32000  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś deklarację poprzedzać pierwszym odwołaniu do zmiennej.
