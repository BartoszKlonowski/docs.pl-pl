# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>"End AddHandler" musi być poprzedzona zgodna deklaracja "AddHandler"
`End AddHandler` Instrukcji wystąpił bez odpowiadającego `AddHandler` instrukcji. `End AddHandler` musi być poprzedzona odpowiadającą `AddHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31124  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy występuje przed `AddHandler` instrukcja jest prawidłowa i poprawna.
