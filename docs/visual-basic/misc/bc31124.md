# <a name="39end-addhandler39-must-be-preceded-by-a-matching-39addhandler39-declaration"></a>&#39; End AddHandler &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; AddHandler &#39; Deklaracja
`End AddHandler` Instrukcji wystąpił bez odpowiadającego `AddHandler` instrukcji. `End AddHandler`musi być poprzedzona odpowiadającą `AddHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31124  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy występuje przed `AddHandler` instrukcji jest prawidłowa i poprawna.
