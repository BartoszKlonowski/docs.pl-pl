# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>Element "End AddHandler" musi być poprzedzony odpowiadającą mu deklaracją "AddHandler"
`End AddHandler`Instrukcja wystąpiła bez odpowiedniej `AddHandler` instrukcji. `End AddHandler`musi być poprzedzona odpowiadającą jej `AddHandler` instrukcją.  
  
 **Identyfikator błędu:** BC31124  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy Poprzednia `AddHandler` instrukcja jest prawidłowa i poprawna.
