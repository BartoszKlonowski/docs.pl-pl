# <a name="resume-or-goto-expected"></a>Oczekiwano instrukcji "resume" lub "GoTo"

`On Error`Instrukcja nie zawiera `Resume` klauzuli ani `GoTo` klauzuli. Musi ona używać jednej klauzuli lub innej.  
  
 **Identyfikator błędu:** BC32019  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Resume` klauzulę lub `GoTo` klauzulę do `On Error` instrukcji.
