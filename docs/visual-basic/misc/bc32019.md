# <a name="resume-or-goto-expected"></a>"Resume" lub "GoTo", oczekiwano
`On Error` Nie zawiera instrukcji `Resume` klauzuli lub `GoTo` klauzuli. Nazwa może zawierać jedną klauzulę lub innych.  
  
 **Identyfikator błędu:** BC32019  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj opcję `Resume` klauzuli lub `GoTo` klauzuli, która `On Error` instrukcji.
