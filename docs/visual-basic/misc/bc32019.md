# <a name="39resume39-or-39goto39-expected"></a>&#39; Wznów &#39; i &#39; Przejdź do &#39; Oczekiwano
`On Error` Instrukcji nie zawiera albo `Resume` klauzuli lub `GoTo` klauzuli. Należy użyć jedną klauzulę lub innych.  
  
 **Identyfikator błędu:** BC32019  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj albo `Resume` klauzuli lub `GoTo` klauzulę `On Error` instrukcji.
