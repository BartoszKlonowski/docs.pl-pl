# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Wartość parametru ConnectionTimeout musi być większa od 0
Podczas przekazywania i pobierania plików z opcją [My. Computer. Network](xref:Microsoft.VisualBasic.Devices.Network)należy określić wartość `connectionTimeout` większą niż `0` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj wartość `connectionTimeout` większą niż `0` .
