# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Wartość parametru ConnectionTimeout musi być większa niż 0
Podczas przekazywania i pobierania plików z [My.Computer.Network — obiekt](../../visual-basic/language-reference/objects/my-computer-network-object.md), należy określić `connectionTimeout` większa niż `0`.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj `connectionTimeout` większą `0`.
