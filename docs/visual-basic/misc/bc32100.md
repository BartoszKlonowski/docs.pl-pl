# <a name="as-comma-or--expected"></a>"As", przecinkiem lub ")" Oczekiwano
Brak parametru typu, który został dostarczony `As`, przecinek (`,`), lub nawiasu zamykającego ( `)` ).  
  
 **Identyfikator błędu:** BC32100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj brakujące `As`, przecinkiem lub nawias zamykający.
