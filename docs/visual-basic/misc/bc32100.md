# <a name="39as39-comma-or-3939-expected"></a>&#39;Jak&#39;, przecinkiem lub &#39;)&#39; oczekiwano
Parametr typu, który został dostarczony jest Brak `As`, przecinka (`,`), lub nawiasu zamykającego ( `)` ).  
  
 **Identyfikator błędu:** BC32100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj brakujący `As`, przecinkiem lub nawiasu zamykającego.
