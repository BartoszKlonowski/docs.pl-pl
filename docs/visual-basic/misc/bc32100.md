# <a name="as-comma-or--expected"></a>Oczekiwano operatora "As", przecinka lub znaku ")"
Parametr typu, który został dostarczony, nie ma `As` `,` zamykającego nawiasu klamrowego () ani klamrowego ( `)` ).  
  
 **Identyfikator błędu:** BC32100  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj brakujący `As` , przecinek lub nawias zamykający.
