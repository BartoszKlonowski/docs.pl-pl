# <a name="xml-name-expected"></a>Oczekiwano nazwy XML
Literał XML i właściwości osi XML została określona bez nazwy.  
  
 **Identyfikator błędu:** BC31146  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj nazwę literał XML i właściwości osi XML.
