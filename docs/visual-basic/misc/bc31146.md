# <a name="xml-name-expected"></a>Oczekiwano nazwy XML

Wartość literału XML lub właściwość osi XML została określona bez nazwy.  
  
 **Identyfikator błędu:** BC31146  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj nazwę dla literału XML lub właściwości osi XML.
