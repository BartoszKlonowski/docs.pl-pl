# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>Klasy\<nazwa_klasy > "nie można indeksować, ponieważ nie ma on domyślnej właściwości
Nie możesz podać właściwość domyślną dla klasy.  
  
 **Identyfikator błędu:** BC30367  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ właściwość jako domyślną właściwość przez uruchomienie jej deklaracji z `Default` — słowo kluczowe.
