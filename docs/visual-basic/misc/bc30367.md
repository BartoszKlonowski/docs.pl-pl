# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>\<classname>Nie można zaindeksować klasy "", ponieważ nie ma ona domyślnej właściwości.

Nie powiodło się dostarczenie domyślnej właściwości klasy.  
  
 **Identyfikator błędu:** BC30367  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ właściwość jako właściwość domyślną, rozpoczynając jej deklarację za pomocą `Default` słowa kluczowego.
