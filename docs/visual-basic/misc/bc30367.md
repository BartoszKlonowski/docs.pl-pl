# <a name="class-39ltclassnamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Klasa &#39; &lt;classname&gt; &#39; nie można indeksować, ponieważ nie ma ona domyślnej właściwości
Nie można podać właściwości domyślnej dla klasy.  
  
 **Identyfikator błędu:** BC30367  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ właściwość jako domyślną właściwość przez uruchomienie jej deklaracji z `Default` — słowo kluczowe.
