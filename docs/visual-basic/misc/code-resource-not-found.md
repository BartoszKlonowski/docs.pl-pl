# <a name="code-resource-not-found"></a>Nie znaleziono zasobu kodu
Wykonano wywołanie procedury w zasobie kodu, ale nie można odnaleźć zasobu kodu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy zasób jest dostępny i czy jest poprawnie przywoływany.
