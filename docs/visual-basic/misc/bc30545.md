# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Dostęp do właściwości musi przypisywać wartość właściwości lub używać jej wartości
Nastąpiła próba dostępu do właściwości bez przypisywania do niego lub przy użyciu jego wartości.
  
 **Identyfikator błędu:** BC30545  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przypisywanie wartości do właściwości.  
  
     \-lub -  
  
-   Użyj wartości właściwości.
