# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Dostęp do właściwości musi przypisywać do właściwości lub użyć jej wartości

Podjęto próbę uzyskania dostępu do właściwości bez przypisywania do niej ani używania jej wartości.
  
 **Identyfikator błędu:** BC30545  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przypisz wartość do właściwości.  
  
     \- oraz  
  
- Użyj wartości właściwości.
