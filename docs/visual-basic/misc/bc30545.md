# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Dostęp do właściwości, należy przypisać do właściwości lub użyć jego wartość
Nastąpiła próba dostępu do właściwości bez przypisywania do niego lub przy użyciu jego wartości.
  
 **Identyfikator błędu:** BC30545  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przypisz wartości do właściwości.  
  
     \- lub —  
  
- Użyj wartości właściwości.
