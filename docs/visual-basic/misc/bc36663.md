# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-method-methodname-and-delegate-delegatename"></a><span data-ttu-id="0d499-102">Ustawienie Option Strict On nie zezwala na Zawężanie w niejawnych konwersjach typów między metodą " \<methodname> " i delegatem " \<delegatename> "</span><span class="sxs-lookup"><span data-stu-id="0d499-102">Option Strict On does not allow narrowing in implicit type conversions between method '\<methodname>' and delegate '\<delegatename>'</span></span>
<span data-ttu-id="0d499-103">W programie `Option Strict` on nie można mieć zawężanej konwersji między typem danych parametru w delegatze i odpowiadającym mu parametrem funkcji lub `Sub` przypisanym do zmiennej tego typu delegata.</span><span class="sxs-lookup"><span data-stu-id="0d499-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a function or `Sub` assigned to a variable of that delegate type.</span></span> <span data-ttu-id="0d499-104">Na przykład delegat funkcji `Del` ma jeden parametr typu `Integer` , a funkcje `Conversion1` , `Conversion2` i `Conversion3` mają jeden parametr różnych typów liczbowych.</span><span class="sxs-lookup"><span data-stu-id="0d499-104">For example, function delegate `Del` has one parameter of type `Integer`, and functions `Conversion1`, `Conversion2`, and `Conversion3` have one parameter of different numeric types.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
  
Function Conversion1(ByVal n As Integer) As String  
    Return "Valid"  
End Function  
  
Function Conversion2(ByVal n As Long) As String  
    Return "Valid"  
End Function  
  
Function Conversion3(ByVal n As Short) As String  
    Return "Not valid"  
End Function  
```  
  
 <span data-ttu-id="0d499-105">Ponieważ istnieje konwersja rozszerzająca z `Integer` do `Integer` i do `Long` , następujące przypisania są prawidłowe.</span><span class="sxs-lookup"><span data-stu-id="0d499-105">Because there is a widening conversion from `Integer` to `Integer` and to `Long`, the following assignments are valid.</span></span>  
  
```vb  
' Valid.  
Dim funDel1 As Del = AddressOf Conversion1  
Dim funDel2 As Del = AddressOf Conversion2  
```  
  
 <span data-ttu-id="0d499-106">Konwersja z `Integer` do na `Short` to zawężana konwersja.</span><span class="sxs-lookup"><span data-stu-id="0d499-106">The conversion from `Integer` to `Short` is a narrowing conversion.</span></span> <span data-ttu-id="0d499-107">W związku z tym następujące przypisanie jest nieprawidłowe.</span><span class="sxs-lookup"><span data-stu-id="0d499-107">Therefore, the following assignment is not valid.</span></span>  
  
```vb  
' Not valid.  
Dim funDel3 As Del = AddressOf Conversion3  
```  
  
 <span data-ttu-id="0d499-108">Identyfikator błędu: BC36663</span><span class="sxs-lookup"><span data-stu-id="0d499-108">Error ID: BC36663</span></span>
