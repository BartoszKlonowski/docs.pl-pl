# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-method-methodname-and-delegate-delegatename"></a><span data-ttu-id="7f717-102">Ustawienie Option Strict On nie zezwala na zawężanie w niejawnych konwersjach typów między metodą '\<methodname >' i delegowanie '\<element delegatename >'</span><span class="sxs-lookup"><span data-stu-id="7f717-102">Option Strict On does not allow narrowing in implicit type conversions between method '\<methodname>' and delegate '\<delegatename>'</span></span>
<span data-ttu-id="7f717-103">Za pomocą `Option Strict` , nie może mieć konwersji zawężającej między typem danych parametru w obiekt delegowany i odpowiedniego parametru funkcji lub `Sub` przypisany do zmiennej tego typu delegata.</span><span class="sxs-lookup"><span data-stu-id="7f717-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a function or `Sub` assigned to a variable of that delegate type.</span></span> <span data-ttu-id="7f717-104">Na przykład delegat funkcji `Del` ma jeden parametr typu `Integer`, funkcje i `Conversion1`, `Conversion2`, i `Conversion3` mieć jeden parametr różne typy liczbowe.</span><span class="sxs-lookup"><span data-stu-id="7f717-104">For example, function delegate `Del` has one parameter of type `Integer`, and functions `Conversion1`, `Conversion2`, and `Conversion3` have one parameter of different numeric types.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
  
Function Conversion1(ByVal n As Integer) As String  
    Return "Valid"  
End Function  
  
Function Conversion2(ByVal n As Long) As String  
    Return "Valid"  
End Function  
  
Function Conversion3(ByVal n As Short) As String  
    Return "Not valid"  
End Function  
```  
  
 <span data-ttu-id="7f717-105">Ponieważ nie istnieje konwersja rozszerzająca `Integer` do `Integer` i `Long`, obowiązują następujące przypisania.</span><span class="sxs-lookup"><span data-stu-id="7f717-105">Because there is a widening conversion from `Integer` to `Integer` and to `Long`, the following assignments are valid.</span></span>  
  
```vb  
' Valid.  
Dim funDel1 As Del = AddressOf Conversion1  
Dim funDel2 As Del = AddressOf Conversion2  
```  
  
 <span data-ttu-id="7f717-106">Konwersja z `Integer` do `Short` jest konwersją zawężającą.</span><span class="sxs-lookup"><span data-stu-id="7f717-106">The conversion from `Integer` to `Short` is a narrowing conversion.</span></span> <span data-ttu-id="7f717-107">W związku z tym następujące przypisanie jest nieprawidłowy.</span><span class="sxs-lookup"><span data-stu-id="7f717-107">Therefore, the following assignment is not valid.</span></span>  
  
```vb  
' Not valid.  
Dim funDel3 As Del = AddressOf Conversion3  
```  
  
 <span data-ttu-id="7f717-108">Identyfikator błędu: BC36663</span><span class="sxs-lookup"><span data-stu-id="7f717-108">Error ID: BC36663</span></span>
