# <a name="end-set-must-be-the-first-statement-on-a-line"></a>Element "End Set" musi być pierwszą instrukcją w wierszu

`End Set`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End Set` musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
