# <a name="39end-set39-must-be-the-first-statement-on-a-line"></a>&#39;End Set&#39; musi być pierwszą instrukcją w wierszu
`End Set` Instrukcji następuje separatora instrukcji dwukropka (:). `End Set` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
