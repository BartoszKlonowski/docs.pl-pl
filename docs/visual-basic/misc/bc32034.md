# <a name="end-set-must-be-the-first-statement-on-a-line"></a>"End Set" musi być pierwszą instrukcją w wierszu
`End Set` Instrukcji następuje separatorem instrukcji dwukropek (:). `End Set` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
