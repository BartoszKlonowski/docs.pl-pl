# <a name="-expected-object-initializer"></a>Oczekiwano znaku "=" (inicjator obiektów)
Aby ustalić początkową wartość pola lub właściwości w deklaracji inicjatora obiektów, należy użyć znaku równości. Na przykład Poniższa deklaracja przypisuje "Microsoft" jako wartość początkową właściwości `Name` `client`.  
  
```vb  
Dim client As New Customer { .Name = "Microsoft" }  
```  
  
 **Identyfikator błędu:** BC30984  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak równości w pozycji pokazanej w poprzednim przykładzie.
