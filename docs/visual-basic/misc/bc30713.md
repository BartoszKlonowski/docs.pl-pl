# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Exit" są nieprawidłowe w oknie bezpośrednim
`Exit` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30713  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Exit` instrukcji debugowania kodu.
