# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Zakończ&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Exit` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30713  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Exit` oświadczenia debugowania kodu.
