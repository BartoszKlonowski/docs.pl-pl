# <a name="type-arguments-cannot-be-applied-to-the-expression-39ltexpressiongt39"></a>Argumentów typu nie można zastosować do wyrażenia &#39; &lt;wyrażenia&gt;&#39;
Alias importu jest zdefiniowana z [z](../../visual-basic/language-reference/statements/of-clause.md) klauzuli tej przekazuje wpisz argumenty do aliasu importu.  
  
 Argumenty typu są używane dla typów ogólnych i tylko klasy, struktury, interfejsy, procedury i delegatów mogą być ogólne. Aliasów przestrzeni nazw ani importu nie może być rodzajowy.  
  
 **Identyfikator błędu:** BC32058  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Of` klauzul i jego argumentów typu z aliasu importu.
