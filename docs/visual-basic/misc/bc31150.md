# <a name="end-tag-tagname-expected"></a>Tagu końcowego \</tagName > oczekiwano
Tag końcowy w literał elementu XML jest niezgodny z odpowiedniego tagu początkowego.  
  
 **Identyfikator błędu:** BC31150  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmodyfikuj końcowy — tag XML do pasuje do tagu początkowego XML, aby uzyskać literał elementu XML.
