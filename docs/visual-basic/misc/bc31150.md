# <a name="end-tag-tagname-expected"></a>\</tagName>Oczekiwano tagu końcowego

Tag końcowy w literale elementu XML nie jest zgodny z odpowiadającym mu tagiem początkowym.  
  
 **Identyfikator błędu:** BC31150  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmodyfikuj końcowy tag XML, aby pasował do początkowego tagu XML dla literału elementu XML.
