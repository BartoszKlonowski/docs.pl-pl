<span data-ttu-id="e4868-103">Interfejs ogólny dziedziczy z co najmniej dwa interfejsy ogólne i dwa spadków może powodować konflikt niektórych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="e4868-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="e4868-104">Poniższe instrukcje może spowodować wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="e4868-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="e4868-105">Jeśli `derivedInterface` jest tworzony lub zaimplementowana podanie tego samego typu zarówno `t1` i `t2`, musi on dziedziczyć dwóch wersji `interfaceX` z argumentami typu identyczne.</span><span class="sxs-lookup"><span data-stu-id="e4868-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="e4868-106">Dzięki temu dałby w efekcie niejednoznaczności wersji do uzyskania dostępu.</span><span class="sxs-lookup"><span data-stu-id="e4868-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="e4868-107">**Identyfikator błędu:** BC32121</span><span class="sxs-lookup"><span data-stu-id="e4868-107">**Error ID:** BC32121</span></span>
