# <a name="cannot-inherit-interface-ltinterfacename1gt-because-the-interface-ltinterfacename2gt-from-which-it-inherits-could-be-identical-to-interface-ltinterfacename3gt-for-some-type-arguments"></a><span data-ttu-id="35251-102">Nie można dziedziczyć interfejsu "&lt;interfacename1&gt;" ponieważ interfejs "&lt;interfacename2&gt;", z której dziedziczy może być taki sam jak interfejs"&lt;interfacename3&gt;" dla niektórych argumentów typu</span><span class="sxs-lookup"><span data-stu-id="35251-102">Cannot inherit interface '&lt;interfacename1&gt;' because the interface '&lt;interfacename2&gt;' from which it inherits could be identical to interface '&lt;interfacename3&gt;' for some type arguments</span></span>
<span data-ttu-id="35251-103">Ogólny interfejs dziedziczy z co najmniej dwóch interfejsach ogólnych i dwa spadki mogą powodować konflikt dla niektórych wartości argumentów typu.</span><span class="sxs-lookup"><span data-stu-id="35251-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="35251-104">Poniższe instrukcje może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="35251-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="35251-105">Jeśli `derivedInterface` jest tworzony lub zaimplementować dostarczenie tego samego typu na wartość oba `t1` i `t2`, musi on dziedziczyć dwóch wersji `interfaceX` z argumentami typu identyczne.</span><span class="sxs-lookup"><span data-stu-id="35251-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="35251-106">Ten sposób dałby w efekcie niejednoznaczności dotyczące dostępu do wersji.</span><span class="sxs-lookup"><span data-stu-id="35251-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="35251-107">**Identyfikator błędu:** BC32121</span><span class="sxs-lookup"><span data-stu-id="35251-107">**Error ID:** BC32121</span></span>
