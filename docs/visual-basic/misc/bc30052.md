# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Tablica przekracza limit 32 wymiarów
Tablice nie mogą mieć więcej niż 32 wymiarów.  
  
 **Identyfikator błędu:** BC30052  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmniejsz liczbę wymiarów w deklaracji tablicy.
