# <a name="unable-to-create-assembly-linker-object-error-message"></a>Nie można utworzyć obiektu konsolidatora zestawu: \<error message>

Kompilator Visual Basic wywołuje konsolidator zestawu (Al.exe, znany również jako ALink) w celu wygenerowania zestawu z manifestem. Konsolidator zgłosił błąd podczas tworzenia obiektu roboczego. Może to być problem z generowaniem odwołania do pliku metadanych lub Niepowodzenie wywołania pliku importu.  
  
 **Identyfikator błędu:** BC30141  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź cytowany komunikat o błędzie i zapoznaj się z tematem [Al.exe](../../framework/tools/al-exe-assembly-linker.md) w celu uzyskania dalszych wyjaśnień i porad.  
  
2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
