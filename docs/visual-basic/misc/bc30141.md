# <a name="unable-to-create-assembly-linker-object-error-message"></a>Nie można utworzyć obiektu Assembly Linker: \<komunikat o błędzie >
Kompilator Visual Basic wywołuje Assembly Linker (Al.exe, znany także jako Alink) do generowania manifestu zestawu. Konsolidator zgłosiła błąd podczas tworzenia obiektu swojej pracy. Może to być problem podczas generowania odwołanie do pliku metadanych lub błędu, aby wywołać plik importu.  
  
 **Identyfikator błędu:** BC30141  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź komunikat o błędzie w cudzysłowach i zapoznaj się z tematem [Al.exe](../../framework/tools/al-exe-assembly-linker.md) uzyskać dokładniejsze objaśnienie i porady.  
  
2. Jeśli błąd będzie się powtarzać, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
