# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>Moduły nie mogą być deklarowane jako &#39; &lt;specyfikator&gt;&#39;
Moduł została zadeklarowana z modyfikatorem nieprawidłowy dostęp.  
  
 **Identyfikator błędu:** BC31052  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamień modyfikator dostępu modyfikator prawomocny dostęp, takich jak `Public` lub `Friend`.
