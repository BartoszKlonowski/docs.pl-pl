# <a name="modules-cannot-be-declared-specifier"></a>Moduły nie można zadeklarować jako '\<specyfikator >'
Moduł został uznany za pomocą modyfikatora nieprawidłowego dostępu.  
  
 **Identyfikator błędu:** BC31052  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamień modyfikator dostępu modyfikator dostępu nie jest ważna, takich jak `Public` lub `Friend`.
