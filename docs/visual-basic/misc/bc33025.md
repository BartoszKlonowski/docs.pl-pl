# <a name="conversion-operators-cannot-convert-to-an-interface-type"></a>Operatory konwersji nie mogą konwertować na typ interfejsu
Operator konwersji jest zadeklarowany za pomocą typu interfejsu jako typu zwracanego.  
  
 W czasie kompilacji Visual Basic uznaje, że wstępnie zdefiniowana konwersja ma istnieć z dowolnego typu odwołania do dowolnego interfejsu. Taka Konwersja może się nie powieść w czasie wykonywania, ale kompilator nie może przewidzieć wyników w czasie wykonywania, dzięki czemu możliwe jest dokonanie takiej konwersji.  
  
 Ponieważ kompilator uważa, że ta konwersja jest już zdefiniowana, nie pozwala na jej ponowne zdefiniowanie.  
  
 **Identyfikator błędu:** BC33025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń całkowicie tę definicję operatora. Jest już wstępnie zdefiniowany.
