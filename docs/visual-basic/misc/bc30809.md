# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Instrukcje "Wend" nie są już obsługiwane; Użyj zamiast tego instrukcji "End while"

`End While`Konstrukcja zastępuje `Wend` słowo kluczowe.  
  
 **Identyfikator błędu:** BC30809  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `End While` , aby zakończyć `While` blok.
