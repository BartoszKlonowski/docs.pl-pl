# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Instrukcje "Wend" nie są już obsługiwane; Zamiast tego użyj instrukcji "End While"
`End While` Konstruowania zastępuje `Wend` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC30809  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `End While` do końca `While` bloku.
