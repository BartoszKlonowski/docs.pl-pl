# <a name="type-parameter-lttypeparameternamegt-can-only-have-one-constraint-that-is-a-class"></a>Parametr typu "&lt;typeparametername&gt;" może mieć tylko jedno ograniczenie, które jest klasą
Lista ograniczeń zawiera więcej niż jednej klasy.  
  
 Lista ograniczeń dla parametru typu można zaakceptować dowolną liczbę interfejsów, ale co najwyżej jedną klasę. Argument typu podana dla, że parametr typu musi dziedziczyć z klasy i Visual Basic nie obsługuje wielokrotne dziedziczenie.  
  
 **Identyfikator błędu:** BC32047  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz jedną klasę, a następnie dołącz tylko tej klasy do listy ograniczeń.  
  
-   Można zdefiniować parametry dodatkowy typ klasy lub klas, które nie należą na tej liście ograniczeń.
