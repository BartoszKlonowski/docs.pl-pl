# <a name="types-declared-private-must-be-inside-another-type"></a>Typy zadeklarowane jako "Private" musi znajdować się wewnątrz innego typu
`Private` Modyfikator został użyty na typie nie wewnątrz innego typu.  
  
 **Identyfikator błędu:** BC31089  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj mniej restrykcyjny modyfikator dostępu, takich jak `Friend`.  
  
2. Deklarowanie typu w ramach innego typu.
