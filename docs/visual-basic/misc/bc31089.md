# <a name="types-declared-39private39-must-be-inside-another-type"></a>Typy zadeklarowane &#39;prywatnej&#39; musi znajdować się wewnątrz innego typu
`Private` Modyfikator został użyty na typie nie wewnątrz innego typu.  
  
 **Identyfikator błędu:** BC31089  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Użyj mniej restrykcyjnych modyfikator dostępu, takich jak `Friend`.  
  
2.  Zadeklarowana wewnątrz innego typu.
