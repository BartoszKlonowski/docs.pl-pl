# <a name="named-argument-cannot-match-a-paramarray-parameter"></a>Nazwany argument nie pasuje do parametru ParamArray
Wprowadzonych w argumencie nazwanym (określona zadeklarowana Nazwa argumentu z dwukropkiem i znakiem równości, a następnie wartość argumentu); jednak nie można przekazać tablicy parametrów według nazwy. Po wywołaniu procedury, podaj nieokreśloną liczbę rozdzielone przecinkami argumenty dla tablicy parametrów, a kompilator nie można skojarzyć więcej niż jeden argument pod jedną nazwą.  
  
 **Identyfikator błędu:** BC30587  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przekaż argument pozycji, a nie nazwę.
