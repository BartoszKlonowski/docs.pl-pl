# <a name="named-argument-cannot-match-a-paramarray-parameter"></a>Nazwany argument nie jest zgodny z parametrem ParamArray
Podano argument nazwany (określony za pomocą zadeklarowana Nazwa argumentu następuje dwukropek i znaku równości, a następnie wartość argumentu); jednak nie można przekazać tablicy parametrów według nazwy. Po wywołaniu procedury, podaj nieskończona liczba rozdzielonych przecinkami argumenty tablicą parametrów, a kompilator nie można skojarzyć więcej niż jeden argument nazwą jednego.  
  
 **Identyfikator błędu:** BC30587  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przekazać argument pozycji, a nie nazwy.
