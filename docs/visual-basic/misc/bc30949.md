# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="8a1fa-102">Nie można określić inicjatora tablicy dla niestałego wymiaru. Użyj pustego inicjatora{}""</span><span class="sxs-lookup"><span data-stu-id="8a1fa-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="8a1fa-103">Tablica inicjuje wymiar, który nie jest znany w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="8a1fa-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="8a1fa-104">Poniższy kod generuje ten błąd.</span><span class="sxs-lookup"><span data-stu-id="8a1fa-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="8a1fa-105">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="8a1fa-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="8a1fa-106">**Identyfikator błędu:** BC30949</span><span class="sxs-lookup"><span data-stu-id="8a1fa-106">**Error ID:** BC30949</span></span>

- <span data-ttu-id="8a1fa-108">Jeśli to możliwe, określ stały wymiar w deklaracji tablicy.</span><span class="sxs-lookup"><span data-stu-id="8a1fa-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="8a1fa-109">Jeśli nie można określić stałego wymiaru, należy zainicjować tablicę przy użyciu pętli, gdy jest znany wymiar niestały.</span><span class="sxs-lookup"><span data-stu-id="8a1fa-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
