# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="7aecf-102">Inicjatora tablicy nie można określić dla niestałego wymiaru; Użyj pustego inicjatora&#39;{}&#39;</span><span class="sxs-lookup"><span data-stu-id="7aecf-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="7aecf-103">Tablica inicjuje wymiaru, który nie jest znany w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="7aecf-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="7aecf-104">Poniższy kod generuje ten błąd.</span><span class="sxs-lookup"><span data-stu-id="7aecf-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="7aecf-105">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="7aecf-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="7aecf-106">**Identyfikator błędu:** BC30949</span><span class="sxs-lookup"><span data-stu-id="7aecf-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7aecf-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="7aecf-107">To correct this error</span></span>  
  
-   <span data-ttu-id="7aecf-108">Jeśli to możliwe należy określić stałej wymiaru w deklaracji tablicy.</span><span class="sxs-lookup"><span data-stu-id="7aecf-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="7aecf-109">Nie można określić stałej wymiaru, należy zainicjować tablicy za pomocą pętli, jeśli wymiar nieunikatowego staje się znane.</span><span class="sxs-lookup"><span data-stu-id="7aecf-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
