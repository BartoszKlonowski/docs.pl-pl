# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="ead7e-102">Nie można określić inicjatora tablicy dla niestałego wymiaru. Użyj pustego inicjatora "{}"</span><span class="sxs-lookup"><span data-stu-id="ead7e-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="ead7e-103">Tablica inicjuje wymiar, który nie jest znany w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="ead7e-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="ead7e-104">Poniższy kod generuje ten błąd.</span><span class="sxs-lookup"><span data-stu-id="ead7e-104">The following code generates this error.</span></span>  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="ead7e-105">Poniższy kod pozwala uniknąć błędu.</span><span class="sxs-lookup"><span data-stu-id="ead7e-105">The following code avoids the error.</span></span>  
  
```vb  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="ead7e-106">**Identyfikator błędu:** BC30949</span><span class="sxs-lookup"><span data-stu-id="ead7e-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ead7e-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="ead7e-107">To correct this error</span></span>  
  
- <span data-ttu-id="ead7e-108">Jeśli to możliwe, określ stały wymiar w deklaracji tablicy.</span><span class="sxs-lookup"><span data-stu-id="ead7e-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="ead7e-109">Jeśli nie można określić stałego wymiaru, należy zainicjować tablicę przy użyciu pętli, gdy jest znany wymiar niestały.</span><span class="sxs-lookup"><span data-stu-id="ead7e-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
