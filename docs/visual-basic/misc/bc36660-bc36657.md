# <a name="data-types-of-the-type-parameters-in-method-methodname-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="60475-102">Na podstawie tych argumentów nie można wywnioskować typów danych parametrów typu w metodzie " \<methodname> ", ponieważ nie są one konwertowane na ten sam typ</span><span class="sxs-lookup"><span data-stu-id="60475-102">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="60475-103">Na podstawie tych argumentów nie można wywnioskować typów danych parametrów typu w metodzie " \<methodname> ", ponieważ nie są one konwertowane na ten sam typ.</span><span class="sxs-lookup"><span data-stu-id="60475-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="60475-104">Jawne określenie typów danych może naprawić ten błąd.</span><span class="sxs-lookup"><span data-stu-id="60475-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="60475-105">Podjęto próbę użycia wnioskowania o typie w celu określenia typu danych lub typów parametru lub parametrów typu podczas oceny wywołania do procedury ogólnej.</span><span class="sxs-lookup"><span data-stu-id="60475-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="60475-106">Kompilator nie może znaleźć typu danych, który spełnia ograniczenia wszystkich argumentów.</span><span class="sxs-lookup"><span data-stu-id="60475-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="60475-107">W związku z tym ten błąd został zgłoszony.</span><span class="sxs-lookup"><span data-stu-id="60475-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="60475-108">Gdy określenie argumentów nie jest opcją (na przykład w przypadku operatorów zapytań w wyrażeniach zapytań), komunikat o błędzie jest wyświetlany bez drugiego zdania.</span><span class="sxs-lookup"><span data-stu-id="60475-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="60475-109">Poniższy kod ilustruje błąd.</span><span class="sxs-lookup"><span data-stu-id="60475-109">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module1
    Sub Main()

        '' Not valid. Integer and Date do not convert to the same type.
        'targetMethod(19, #3/4/2007#)

    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

End Module
```

<span data-ttu-id="60475-110">**Identyfikator błędu:** BC36660 i BC36657</span><span class="sxs-lookup"><span data-stu-id="60475-110">**Error ID:** BC36660 and BC36657</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="60475-111">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="60475-111">To correct this error</span></span>

  ```vb

  ```

- <span data-ttu-id="60475-113">Możesz określić typ danych dla parametru typu lub parametrów, do których argumenty są konwertowane, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="60475-113">You may be able to specify a data type for the type parameter or parameters to which the arguments convert, as shown in the following code:</span></span>

