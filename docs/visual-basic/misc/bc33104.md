# <a name="if-operator-requires-either-two-or-three-operands"></a>Operatora "If" wymaga argumentów operacji, dwóch lub trzech
`If` Operator akceptuje tylko dwa lub trzy argumenty.  
  
 **Identyfikator błędu:** BC33104  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź `If` wyrażenia, aby zobaczyć, czy liczba operatorów można zredukować lub rozszerzone do dwóch lub trzech argumentów.  
  
- Należy wziąć pod uwagę przy użyciu innej konstrukcji warunkowych, takich jak `If...Then...Else` instrukcji lub zagnieżdżoną `If...Then...Else` instrukcję, aby zorganizować argumenty.
