# <a name="39if39-operator-requires-either-two-or-three-operands"></a>&#39;Jeśli&#39; operator wymaga dwóch lub trzech argumentów operacji
`If` Operator akceptuje tylko dwóch lub trzech argumentów.  
  
 **Identyfikator błędu:** BC33104  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź `If` wyrażenia, aby sprawdzić, czy liczba operatorów można zredukować lub rozszerzony do dwóch lub trzech argumentów.  
  
-   Należy wziąć pod uwagę przy użyciu innego warunkowego konstrukcji, takich jak `If...Then...Else` instrukcji lub zagnieżdżoną `If...Then...Else` instrukcji do organizowania argumentów.
