# <a name="if-operator-requires-either-two-or-three-operands"></a>Operator "If" wymaga dwóch lub trzech operandów
`If`Operator akceptuje tylko dwa lub trzy argumenty.  
  
 **Identyfikator błędu:** BC33104  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź `If` wyrażenie, aby sprawdzić, czy liczba operatorów może być zredukowana lub rozszerzona do dwóch lub trzech argumentów.  
  
- Rozważ użycie innej konstrukcji warunkowej, takiej jak `If...Then...Else` instrukcja lub zagnieżdżona `If...Then...Else` instrukcja do organizowania argumentów.
