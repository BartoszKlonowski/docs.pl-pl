# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Znak "Character" (0x \<hexadecimal> ) jest niedozwolony w literale XML
W literale XML użyto nieobsługiwanego znaku.  
  
 **Identyfikator błędu:** BC31176  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwany znak.
