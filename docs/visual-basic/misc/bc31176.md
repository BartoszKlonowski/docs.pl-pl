# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Znak 'character' (0 x\<szesnastkowe >) jest niedozwolony w języku XML literału
Nieobsługiwany znak został użyty w literał XML.  
  
 **Identyfikator błędu:** BC31176  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwane znaki.
