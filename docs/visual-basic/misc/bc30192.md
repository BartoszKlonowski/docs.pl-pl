`ParamArray`W deklaracji procedury występuje co najmniej jedna Specyfikacja argumentu.  
  
 **Identyfikator błędu:** BC30192  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś `ParamArray` na koniec listy argumentów.
