# <a name="end-of-parameter-list-expected"></a>Koniec Oczekiwano listy parametrów
A `ParamArray` specyfikacji następuje co najmniej jeden innych argumentu specyfikacji w deklaracji procedury.  
  
 **Identyfikator błędu:** BC30192  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `ParamArray` na końcu listy argumentów.
