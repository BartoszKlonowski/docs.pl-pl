# <a name="end-of-parameter-list-expected"></a>Koniec listy parametrów oczekuje
A `ParamArray` specyfikacji następuje jeden lub więcej innych specyfikacje argumentu w deklaracji procedury.  
  
 **Identyfikator błędu:** BC30192  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `ParamArray` na końcu listy argumentów.
