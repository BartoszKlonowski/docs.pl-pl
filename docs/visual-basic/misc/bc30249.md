# <a name="-expected-declaration"></a>Oczekiwano znaku "=" (Deklaracja)
`Mid` Instrukcja lubdyrektywakompilacji`#Const` nie zawiera przypisania wartości.  
  
 **Identyfikator błędu:** BC30249  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak równości (`=`) i wartość, która ma zostać przypisana do ciągu lub stałej kompilacji.
