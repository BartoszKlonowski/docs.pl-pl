# <a name="3939-expected-declaration"></a>&#39;=&#39; oczekiwano (deklaracji)
A `Mid` instrukcji lub `#Const` dyrektywie kompilacji nie ma przypisanie wartości.  
  
 **Identyfikator błędu:** BC30249  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znak równości (`=`) i wartość do przypisania do ciągu lub Stała kompilacji.
