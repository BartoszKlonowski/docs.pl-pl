# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Stała kompilacji warunkowej na poziomie projektu nie jest prawidłowa: \<błąd >
Opcja nieprawidłowy kompilacja warunkowa jest określona dla projektu.  
  
 **Identyfikator błędu:** BC31031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź stałe kompilacji warunkowej, używane z tym projektem.
