# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Stała kompilacji warunkowej na poziomie projektu jest nieprawidłowa: \<> błędu

 **Identyfikator błędu:** BC31031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd
