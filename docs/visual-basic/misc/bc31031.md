# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Stała kompilacji warunkowej na poziomie projektu jest nieprawidłowa: \<error>

Określono nieprawidłową opcję kompilacji warunkowej dla projektu.  
  
 **Identyfikator błędu:** BC31031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź stałe kompilacji warunkowej używane z tym projektem.
