# <a name="an-invalid-name-was-specified-for-the-event-log"></a>Określono nieprawidłową nazwę dziennika zdarzeń
Określono nieprawidłową nazwę dziennika zdarzeń. Zazwyczaj jest to wynik nieprawidłowych znaków w nazwa, Nazwa pustego pliku lub nazwę pliku, który jest zbyt długi.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli określona nazwa jest więcej niż osiem znaków, upewnij się, że nie występuje konflikt z nazwami innych dzienników zdarzeń. Tylko pierwszych osiem znaków są oceniane podczas ustalania, czy nazwa jest unikatowa.  
  
-   Jeśli podając ścieżkę, upewnij się, że jest poprawnie przeanalizować.  
  
-   Sprawdź, czy nie są nieprawidłowe znaki w nazwie. Znaki, których nie można użyć w nazwie pliku to `<`, `>`, `:`, `"`, `/`, `\`, i `|`.
