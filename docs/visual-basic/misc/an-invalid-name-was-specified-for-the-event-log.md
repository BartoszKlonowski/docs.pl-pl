# <a name="an-invalid-name-was-specified-for-the-event-log"></a>Określono nieprawidłową nazwę dziennika zdarzeń
Określono nieprawidłową nazwę dziennika zdarzeń. Zwykle jest to wynik nieprawidłowych znaków w nazwie, pustej nazwie pliku lub nazwy pliku, która jest zbyt długa.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli określona nazwa jest dłuższa niż osiem znaków, upewnij się, że nie występują konflikty z nazwami innych dzienników zdarzeń. Podczas ustalania, czy nazwa jest unikatowa, oceniane są tylko pierwsze osiem znaków.  
  
- W przypadku podawania ścieżki upewnij się, że została ona prawidłowo przeanalizowana.  
  
- Sprawdź, czy nazwa nie zawiera nieprawidłowych znaków. Znaki, których nie można użyć w nazwie pliku, obejmują,,,,, `<` `>` `:` `"` `/` `\` i `|` .
