# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>Parametr TextFieldParser nie obsługuje ograniczników zawierających znaki endline
Ogranicznik, który zawiera znak endline zostały podane, ale `TextFieldParser` nie obsługuje ograniczników zawierających znaki endline.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłowy ogranicznik.
