# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>TextFieldParser nie obsługuje ograniczników zawierających znaki EndLine
Podano ogranicznik zawierający znak EndLine, ale nie `TextFieldParser` obsługuje ograniczników zawierających znaki EndLine.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłowy ogranicznik.
