# <a name="integer-constant-expected"></a>Oczekiwano stałej całkowitej
`#ExternalSource` Dyrektywy występuje, w którym drugi argument nie jest literał liczby całkowitej. Tylko literał liczby całkowitej jest prawidłowa w tym kontekście. Nazwanego elementu członkowskiego stałej lub wyliczenia jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30204  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj nazwanego elementu członkowskiego stałej lub wyliczenia zamiast literału.  
  
2. Podaj literału jako drugi argument typu integer `#ExternalSource` dyrektywy.
