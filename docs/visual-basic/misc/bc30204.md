# <a name="integer-constant-expected"></a>Oczekiwano stałej całkowitej

`#ExternalSource`Występuje dyrektywa, w której drugi argument nie jest literałem liczby całkowitej. Tylko literał liczby całkowitej jest prawidłowy w tym kontekście. Nazwana stała lub składowa wyliczenia są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30204  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj nazwanej stałej lub składowej wyliczenia zamiast literału.  
  
2. Podaj literał liczby całkowitej jako drugi argument `#ExternalSource` dyrektywy.
