# <a name="integer-constant-expected"></a>Oczekiwano stałej całkowitej
`#ExternalSource` Dyrektywy występuje, w którym drugi argument nie jest liczbą całkowitą literału. Tylko całkowitą literału jest nieprawidłowe w tym kontekście. Nazwane członek stałej lub wyliczenia jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30204  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zamiast literału, należy użyć członka o nazwie stałą lub wyliczenia.  
  
2.  Dodatnią liczbę całkowitą literałów jako drugi argument `#ExternalSource` dyrektywy.
