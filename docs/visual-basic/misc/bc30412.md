# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39;WithEvents&#39; zmienne muszą mieć &#39;jako&#39; — klauzula
Nie podano `As` klauzula ze słowem kluczowym `WithEvents`. Deklarowanie zmiennej jako określonej klasy, które może wywoływać zdarzenia.  
  
 **Identyfikator błędu:** BC30412  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `As` klauzuli, która określa klasę, które może wywoływać zdarzenia.
