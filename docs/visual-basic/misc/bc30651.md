# <a name="property-parameters-cannot-be-declared-byref"></a>\<Property>parametrów nie można zadeklarować jako "ByRef"
Parametry właściwości muszą być zadeklarowane za pomocą `ByVal` modyfikatora.  
  
 **Identyfikator błędu:** BC30651  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj parametry właściwości jako `ByVal` .
