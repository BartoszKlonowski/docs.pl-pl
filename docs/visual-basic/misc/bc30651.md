# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Właściwość&gt; parametry nie mogą być zadeklarowane &#39;ByRef&#39;
Parametry właściwości musi być zadeklarowany za pomocą `ByVal` modyfikator.  
  
 **Identyfikator błędu:** BC30651  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zdefiniuj parametry właściwości jako `ByVal`.
