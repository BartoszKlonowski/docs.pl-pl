# <a name="ltpropertygt-parameters-cannot-be-declared-byref"></a>&lt;Właściwość&gt; parametrów nie można zadeklarować jako "ByRef"
Parametry dla właściwości musi być zadeklarowany za pomocą `ByVal` modyfikator.  
  
 **Identyfikator błędu:** BC30651  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zdefiniuj parametry właściwości jako `ByVal`.
