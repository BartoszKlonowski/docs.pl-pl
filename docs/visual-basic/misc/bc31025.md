# <a name="unable-to-open-key-file-ltfilenamegt-lterrorgt"></a>Nie można otworzyć pliku klucza "&lt;filename&gt;": &lt;błąd&gt;
Nie można otworzyć pliku kluczowego określonego dla tego projektu.  
  
 **Identyfikator błędu:** BC31025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy plik klucza, który istnieje i nie jest zablokowany przez inną aplikację.
