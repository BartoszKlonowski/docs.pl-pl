# <a name="unable-to-open-key-file-filename-error"></a>Nie można otworzyć pliku klucza " \<filename> ": \<error>

Nie można otworzyć pliku klucza określonego dla tego projektu.  
  
 **Identyfikator błędu:** BC31025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy plik klucza istnieje i nie jest zablokowany przez inną aplikację.
