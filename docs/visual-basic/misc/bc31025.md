# <a name="unable-to-open-key-file-39ltfilenamegt39-lterrorgt"></a>Nie można otworzyć pliku klucza &#39; &lt;filename&gt;&#39;: &lt;błąd&gt;
Nie można otworzyć pliku kluczowego, określonego dla tego projektu.  
  
 **Identyfikator błędu:** BC31025  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy istnieje plik klucza i nie jest zablokowany przez inną aplikację.
