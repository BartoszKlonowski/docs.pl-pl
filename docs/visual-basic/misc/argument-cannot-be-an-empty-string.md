# <a name="argument-cannot-be-an-empty-string"></a>Argument nie może być pustym ciągiem
Argument została zdefiniowana jako ciąg pusty ("").  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłową wartość dla argumentu.
