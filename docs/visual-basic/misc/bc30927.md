<span data-ttu-id="45106-103">Procedura lub właściwość jest zadeklarowana jako `MustOverride` należące do klasy, która jest zdefiniowana w wielu deklaracjach częściowych, ale jedna z definicji częściowych określa `NotInheritable` dla klasy.</span><span class="sxs-lookup"><span data-stu-id="45106-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="45106-104">Podczas dzielenia definicji klasy między kilka deklaracji częściowych kompilator traktuje klasę jako Unię wszystkich jej częściowych deklaracji.</span><span class="sxs-lookup"><span data-stu-id="45106-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="45106-105">Dotyczy to nie tylko elementów członkowskich, ale również do implementacji, dziedziczenia i poziomu dostępu.</span><span class="sxs-lookup"><span data-stu-id="45106-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>
