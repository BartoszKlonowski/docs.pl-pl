# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Skutki uboczne są nieprawidłowe podczas obliczania wyrażenia w tym kontekście

Instrukcje, które zmieniają stan obiektu, nie są dozwolone w tym kontekście.  
  
 **Identyfikator błędu:** BC30700  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień kod, aby instrukcje były oceniane bez zmiany stanu innych elementów.
