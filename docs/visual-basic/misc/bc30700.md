# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Efekty uboczne nie jest prawidłowy podczas obliczania wyrażenia w tym kontekście
Instrukcje, które spowodują zmianę stanu obiektu nie są dozwolone w tym kontekście.  
  
 **Identyfikator błędu:** BC30700  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień swój kod, aby instrukcje są oceniane bez zmiany stanu innych elementów.
