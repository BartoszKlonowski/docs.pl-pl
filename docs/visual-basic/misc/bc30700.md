# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Efekty uboczne, które nie jest prawidłowy podczas oceny wyrażenia w tym kontekście
Instrukcje, które zmieniają stan obiektu nie są dozwolone w tym kontekście.  
  
 **Identyfikator błędu:** BC30700  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień swój kod, dzięki czemu instrukcje są oceniane bez zmiany stanu innych elementów.
