# <a name="method-39ltmethodnamegt39-cannot-be-called-with-ltnumbergt-arguments"></a>Metoda &#39; &lt;methodname&gt;&#39; nie można wywołać z &lt;numer&gt; argumentów
Wywołanie metody zawiera niewłaściwą liczbę argumentów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Liczba argumentów, które zostały wprowadzone.  
  
2.  Zgodna z liczbą przed z liczbą parametrów metody, którą chcesz się połączyć.  
  
3.  Dostosuj liczbę argumentów, dwóch liczb podanych zgody.
