# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="3cc1e-102">"&lt;derivedtypename&gt;" nie może dziedziczyć z &lt;typu&gt; "&lt;constructedbasetypename&gt;"ponieważ rozszerza on dostęp do typu"&lt;internaltypename&gt;"Aby &lt;region&gt; "&lt;regionname&gt;"</span><span class="sxs-lookup"><span data-stu-id="3cc1e-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="3cc1e-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu typu wewnętrznego przy użyciu go jako argument typu klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="3cc1e-104">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="3cc1e-105">Kod poza `containingClass` jest niedozwolona w celu uzyskania dostępu do `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="3cc1e-106">Jednak `derivedClass` jest możliwy z kodu z tego samego zestawu.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="3cc1e-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` używa `internalStructure` jako argument typu, ponieważ, może spowodować `internalStructure` w całym regionie definiujące kodu.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="3cc1e-108">**Identyfikator błędu:** BC30921</span><span class="sxs-lookup"><span data-stu-id="3cc1e-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3cc1e-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="3cc1e-109">To correct this error</span></span>  
  
-   <span data-ttu-id="3cc1e-110">Dostosuj poziomy dostępu dla klas lub interfejsów, tak, aby typ pochodny nie zwiększa poziom dostępu typu wewnętrznego.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="3cc1e-111">—lub—</span><span class="sxs-lookup"><span data-stu-id="3cc1e-111">-or-</span></span>  
  
-   <span data-ttu-id="3cc1e-112">Jeśli nie można dostosować poziomów dostępu, nie należy używać wewnętrznego typu jako argumentu typu podczas tworzenia klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="3cc1e-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
