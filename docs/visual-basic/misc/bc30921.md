# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="05e2d-102">'\<derivedtypename >' nie może dziedziczyć z \<typu >'\<constructedbasetypename > ', ponieważ rozszerza on dostęp do typu'\<internaltypename > 'do \<region >'\<regionname > '</span><span class="sxs-lookup"><span data-stu-id="05e2d-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="05e2d-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu typu wewnętrznego przy użyciu go jako argument typu klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="05e2d-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="05e2d-104">Poniższy kod można wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="05e2d-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="05e2d-105">Kod poza `containingClass` jest niedozwolona w celu uzyskania dostępu do `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="05e2d-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="05e2d-106">Jednak `derivedClass` jest możliwy z kodu z tego samego zestawu.</span><span class="sxs-lookup"><span data-stu-id="05e2d-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="05e2d-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` używa `internalStructure` jako argument typu, ponieważ, może spowodować `internalStructure` w całym regionie definiujące kodu.</span><span class="sxs-lookup"><span data-stu-id="05e2d-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="05e2d-108">**Identyfikator błędu:** BC30921</span><span class="sxs-lookup"><span data-stu-id="05e2d-108">**Error ID:** BC30921</span></span>
