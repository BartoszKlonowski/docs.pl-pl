# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="e7d40-102">element " \<derivedtypename> " nie może dziedziczyć po elemencie \<type> " \<constructedbasetypename> ", ponieważ rozszerza on dostęp typu " \<internaltypename> " do \<region> elementu " \<regionname> "</span><span class="sxs-lookup"><span data-stu-id="e7d40-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="e7d40-103">Klasa pochodna lub interfejs próbuje rozszerzyć poziom dostępu typu wewnętrznego przy użyciu go jako argumentu typu dla klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="e7d40-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="e7d40-104">Poniższy kod może wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="e7d40-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="e7d40-105">Nie można `containingClass` uzyskać dostępu do kodu poza `internalStructure` .</span><span class="sxs-lookup"><span data-stu-id="e7d40-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="e7d40-106">Można jednak `derivedClass` uzyskać dostęp z dowolnego kodu w tym samym zestawie.</span><span class="sxs-lookup"><span data-stu-id="e7d40-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="e7d40-107">W związku z tym, `derivedClass` nie można dziedziczyć, `baseClass` Jeśli używa `internalStructure` jako argumentu typu, ponieważ może uwidaczniać w `internalStructure` całym regionie kodu.</span><span class="sxs-lookup"><span data-stu-id="e7d40-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="e7d40-108">**Identyfikator błędu:** BC30921</span><span class="sxs-lookup"><span data-stu-id="e7d40-108">**Error ID:** BC30921</span></span>
