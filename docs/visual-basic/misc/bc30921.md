# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="6ce63-102">&#39;&lt;derivedtypename&gt; &#39; nie może dziedziczyć po &lt;typu&gt; &#39; &lt;constructedbasetypename&gt; &#39; ponieważ rozszerza on dostęp do typu &#39; &lt; internaltypename&gt; &#39; do &lt;region&gt; &#39; &lt;regionname&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="6ce63-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="6ce63-103">Pochodne klasy lub interfejsu próbuje rozwiń poziom dostępu typu wewnętrznej przy użyciu go jako argument typu klasy podstawowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="6ce63-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="6ce63-104">Poniższy kod może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="6ce63-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="6ce63-105">Kod poza `containingClass` nie może uzyskać dostępu do `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="6ce63-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="6ce63-106">Jednak `derivedClass` można uzyskać z dowolnego kodu w tym samym zestawie.</span><span class="sxs-lookup"><span data-stu-id="6ce63-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="6ce63-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass` , gdy jest używana `internalStructure` jako argument typu, ponieważ, która może spowodować narażenie `internalStructure` w całym definiującego regionie kodu.</span><span class="sxs-lookup"><span data-stu-id="6ce63-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="6ce63-108">**Identyfikator błędu:** BC30921</span><span class="sxs-lookup"><span data-stu-id="6ce63-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6ce63-109">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="6ce63-109">To correct this error</span></span>  
  
-   <span data-ttu-id="6ce63-110">Dostosuj poziomy dostępu do interfejsów lub klas, aby typ pochodny nie zwiększa poziom dostępu typu wewnętrznego.</span><span class="sxs-lookup"><span data-stu-id="6ce63-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="6ce63-111">—lub—</span><span class="sxs-lookup"><span data-stu-id="6ce63-111">-or-</span></span>  
  
-   <span data-ttu-id="6ce63-112">Jeśli nie można dostosować poziomów dostępu, nie należy używać wewnętrznego typu jako argumentu typu podczas tworzenia klasy podstawowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="6ce63-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
