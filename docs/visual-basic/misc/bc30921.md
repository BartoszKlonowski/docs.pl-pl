# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="8afba-102">'\<derivedtypename >' nie może dziedziczyć z \<typu >'\<constructedbasetypename > ', ponieważ rozszerza on dostęp do typu'\<internaltypename > 'do \<region >'\<regionname > '</span><span class="sxs-lookup"><span data-stu-id="8afba-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="8afba-103">Klasa pochodna lub interfejs próbuje rozszerzyć poziom dostępu typu wewnętrznego przy użyciu go jako argumentu typu dla klasy bazowej lub interfejsu.</span><span class="sxs-lookup"><span data-stu-id="8afba-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="8afba-104">Poniższy kod może wygenerować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="8afba-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="8afba-105">Kod spoza `containingClass` nie może uzyskać dostępu do `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="8afba-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="8afba-106">Można jednak uzyskać dostęp do `derivedClass` z dowolnego kodu w tym samym zestawie.</span><span class="sxs-lookup"><span data-stu-id="8afba-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="8afba-107">W związku z tym `derivedClass` nie może dziedziczyć `baseClass`, jeśli używa `internalStructure` jako argumentu typu, ponieważ może uwidaczniać `internalStructure` w całym regionie definiowania kodu.</span><span class="sxs-lookup"><span data-stu-id="8afba-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="8afba-108">**Identyfikator błędu:** BC30921</span><span class="sxs-lookup"><span data-stu-id="8afba-108">**Error ID:** BC30921</span></span>
