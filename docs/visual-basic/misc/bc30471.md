# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Wyrażenie nie jest tablicą ani metodą i nie może mieć listy argumentów
Podana lista argumentów dla wyrażenia, który nie jest tablicą lub metoda. Tylko tablice lub metod może mieć listy argumentów.  
  
 **Identyfikator błędu:** BC30471  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń z listą argumentów.
