# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Wyrażenie nie jest tablicą lub metoda i nie może mieć listy argumentów
Wyrażenie, które nie jest tablicą lub metoda ma przewidziane listy argumentów. Tylko tablic lub metod może mieć listy argumentów.  
  
 **Identyfikator błędu:** BC30471  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń z listy argumentów.
