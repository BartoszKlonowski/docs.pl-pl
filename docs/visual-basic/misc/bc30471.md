# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Wyrażenie nie jest tablicą ani metodą i nie może mieć listy argumentów

Podano listę argumentów dla wyrażenia, które nie jest tablicą ani metodą. Tylko tablice lub metody mogą mieć listy argumentów.  
  
 **Identyfikator błędu:** BC30471  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń listę argumentów.
