# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Kompilator Visual Basic nie jest w stanie odzyskać z powodu następującego błędu: \<błąd >
Kompilator Visual Basic nie jest w stanie odzyskać z powodu następującego błędu: <`error`>.  
  
 Zostały wyłączone niektóre funkcje środowiska IDE.  
  
 Wystąpił błąd wewnętrzny w kompilatorze języka Visual Basic.  
  
 **Identyfikator błędu:** BC31021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zapisz swoją pracę i uruchom ponownie program Visual Studio.  
  
2.  Jeśli ten błąd wystąpi, zainstaluj ponownie program Visual Studio.  
  
3.  Jeśli ten błąd będzie występował po ponownej instalacji, powiadom pomoc techniczna firmy Microsoft.
