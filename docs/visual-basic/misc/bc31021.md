# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Kompilator Visual Basic nie jest w stanie odzyskać z powodu następującego błędu: &lt;błąd&gt;
Kompilator Visual Basic nie jest w stanie odzyskać z powodu następującego błędu: <`error`>.  
  
 Niektóre funkcje IDE zostały wyłączone.  
  
 Wystąpił wewnętrzny błąd kompilatora Visual Basic.  
  
 **Identyfikator błędu:** BC31021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zapisz swoją pracę i ponownie uruchom program Visual Studio.  
  
2.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie program Visual Studio.  
  
3.  Jeśli błąd będzie nadal występować po ponownej instalacji, powiadamia pomocy technicznej firmy Microsoft.
