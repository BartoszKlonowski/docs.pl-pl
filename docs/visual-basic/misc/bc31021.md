# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Kompilator Visual Basic nie może odzyskać sprawności po następującym błędzie: \<error>

Kompilator Visual Basic nie może odzyskać sprawności po następującym błędzie: <`error`>.  
  
 Niektóre funkcje IDE zostały wyłączone.  
  
 Wystąpił błąd wewnętrzny w kompilatorze Visual Basic.  
  
 **Identyfikator błędu:** BC31021  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapisz swoją służbę i ponownie uruchom program Visual Studio.  
  
2. Jeśli błąd się powtarza, zainstaluj ponownie program Visual Studio.  
  
3. Jeśli błąd będzie się powtarzać po ponownej instalacji, powiadom firmę Microsoft o pomocy technicznej.
