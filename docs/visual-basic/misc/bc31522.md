# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>&#39; Elementu System.Runtime.InteropServices.DllImportAttribute &#39; Nie można zastosować do Sub, Function lub Operator z niepustym tekstem
`DllImportAttribute` Zastosowano atrybut `Sub`, `Function`, lub `Operator` to nie jest pusty.  
  
 **Identyfikator błędu:** BC31522  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń cały kod z `Sub`, `Function`, lub `Operator` można użyć tego atrybutu.
