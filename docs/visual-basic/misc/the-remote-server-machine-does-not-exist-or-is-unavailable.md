# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>Serwer zdalny nie istnieje lub jest niedostępna (Visual Basic)
Próbujesz nawiązać połączenie z serwerem zdalnym, który jest niedostępny lub nie istnieje. Ten błąd może być spowodowany przez utratę połączenia sieciowego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ponowne nawiązywanie połączenia z siecią i spróbuj ponownie.
