# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Nie można obliczyć metody sieci Web w oknach debugowania
Nie można przetestować metod usługi XML sieci Web w oknie debugowania.  
  
 **Identyfikator błędu:** BC30732  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Testowanie metody usług XML sieci Web w edytorze kodu.
