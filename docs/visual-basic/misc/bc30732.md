# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Nie można ocenić metod sieci Web w oknach debugowania

W oknie debugowania nie można testować metod usługi sieci Web XML.  
  
 **Identyfikator błędu:** BC30732  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przetestuj metody usługi sieci Web XML z edytora kodu.
