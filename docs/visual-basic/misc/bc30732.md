# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Nie można obliczyć metody sieci Web w oknach debugowania
Nie można przetestować metody usługi sieci Web XML w obrębie okna debugowania.  
  
 **Identyfikator błędu:** BC30732  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Testowanie metody usługi sieci Web XML w edytorze kodu.
