# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować zmienne w modułach &#39; &lt;Specyfikator&gt;&#39;
Użyto specyfikatora, takich jak `MustInherit`, zmiennej w `Module` instrukcji. Moduły nigdy nie można utworzyć wystąpienia, nie obsługuje dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30593  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
