# <a name="address-is-not-a-valid-remote-file-address"></a>\<address>nie jest prawidłowym adresem pliku zdalnego
\<address>nie jest prawidłowym adresem pliku zdalnego. Prawidłowy adres powinien zawierać protokół, ścieżkę i nazwę pliku.  
  
 Określono nieprawidłowy adres dla operacji przekazywania lub pobierania. Adresy muszą zawierać protokół, taki jak `ftp` lub `http` , prawidłową ścieżkę i nazwę pliku. Przyczyną tego błędu może być niepoprawna analiza ścieżek plików.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź podany adres, aby określić, gdzie nie działa, i podaj niezbędne informacje.
