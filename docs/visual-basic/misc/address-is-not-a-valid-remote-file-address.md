# <a name="ltaddressgt-is-not-a-valid-remote-file-address"></a>&lt;adres&gt; nie jest prawidłowym adresem pliku zdalnego
\<adres > nie jest prawidłowym adresem pliku zdalnego. Prawidłowy adres powinien zawierać protokół, ścieżkę i nazwę pliku.  
  
 Określono nieprawidłowy adres dla operacji przekazywania lub pobierania. Adresy musi zawierać nazwę protokołu, takich jak `ftp` lub `http`, prawidłową ścieżkę i nazwę pliku. Ten błąd może być wynikiem niepoprawnie podczas analizowania ścieżki do pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź adres określone ustalenie, gdzie zakończy się niepowodzeniem i podaj potrzebne informacje.
