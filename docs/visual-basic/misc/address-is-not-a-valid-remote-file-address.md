# <a name="address-is-not-a-valid-remote-file-address"></a>\<adres > nie jest prawidłowym adresem pliku zdalnego
\<adres > nie jest prawidłowym adresem pliku zdalnego. Prawidłowy adres powinien zawierać, protokół, ścieżkę i nazwę pliku.  
  
 Określono nieprawidłowy adres dla operacji przekazywania lub pobierania. Adresy muszą zawierać nazwę protokołu, takich jak `ftp` lub `http`, prawidłową ścieżkę i nazwę pliku. Ten błąd może być wynikiem niepoprawnie podczas analizowania ścieżki pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź adres, który określiłeś w celu określenia, gdzie kończy się niepowodzeniem, a następnie podaj wymagane informacje.
