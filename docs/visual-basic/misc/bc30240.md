# <a name="exit-must-be-followed-by-sub-function-property-do-for-while-select-or-try"></a>"Wyjście" musi następować "Sub", "Function", "Property", "Do", "For", "While", "Select" lub "Try"
`Exit` Instrukcja zawiera nieprawidłowe słowo kluczowe. `Exit` należy określić bloku, z którego ma zostać przeniesiony do instrukcji następującej bloku; kontroli na przykład `End While`.  
  
 **Identyfikator błędu:** BC30240  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj następujące prawidłowe — słowo kluczowe `Exit`, lub usuń `Exit` instrukcji.
