# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39; Zakończ &#39; musi następować &#39; Sub &#39; &#39; Funkcja &#39; &#39; &#39; właściwości, &#39; &#39; nie, &#39; Dla &#39; &#39; Podczas &#39; &#39; Wybierz opcję &#39; lub &#39; Spróbuj &#39;
`Exit` Instrukcja zawiera nieprawidłowe słowo kluczowe. `Exit`należy określić bloku, z którego ma zostać przeniesiony do instrukcji następującej bloku; formantu na przykład `End While`.  
  
 **Identyfikator błędu:** BC30240  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj następujące prawidłową — słowo kluczowe `Exit`, lub usuń `Exit` instrukcji.
