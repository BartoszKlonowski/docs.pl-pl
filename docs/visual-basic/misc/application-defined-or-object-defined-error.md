# <a name="application-defined-or-object-defined-error"></a>Błąd zdefiniowanych przez aplikację lub zdefiniowane przez obiekt
Wystąpił błąd zdefiniowanych przez aplikację lub zdefiniowane przez użytkownika. Sprawdź komunikat o błędzie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zapoznaj się z błędu, aby zobaczyć, gdzie są zdefiniowane i jego rozwiązania.
