# <a name="application-defined-or-object-defined-error"></a>Zdefiniowany przez aplikację lub błąd zdefiniowany przez obiekt

Wystąpił błąd zdefiniowany przez aplikację lub zdefiniowany przez użytkownika. Sprawdź błąd.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zapoznaj się z błędem, aby zobaczyć, gdzie jest zdefiniowany i jak rozwiązać ten problem.
