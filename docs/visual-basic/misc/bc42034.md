# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Nazwa pliku została już zadeklarowana z innym identyfikatorem GUID i wartością sumy kontrolnej
Podjęto próbę zadeklarować plik, ale istnieje już plik z innym identyfikatorem GUID i wartością sumy kontrolnej.  
  
 **Identyfikator błędu:** BC42034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź nazwę pliku i ścieżkę, aby upewnić się, że każda z nich jest poprawna.  
  
- Usuń nadmiarowy plik, jeśli nie jest to konieczne.
