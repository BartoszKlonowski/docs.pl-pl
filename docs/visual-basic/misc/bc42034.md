# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Nazwa pliku już zadeklarowany pod inny identyfikator GUID i wartością sumy kontrolnej
Została podjęta próba zadeklarować pliku, ale plik już istnieje, podając inną wartość identyfikatora GUID i sumy kontrolnej.  
  
 **Identyfikator błędu:** BC42034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwę pliku i ścieżkę do upewnij się, że każdy jest poprawna.  
  
-   Usuń nadmiarowe plik, jeśli nie jest wymagana.
