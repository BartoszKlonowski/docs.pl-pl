# <a name="method-call-did-not-return-a-value"></a>Wywołanie metody nie zwrócił wartość
Metoda wywoływana nie zwrócił wartość zgodnie z oczekiwaniami.  
  
 **Identyfikator błędu:** BC30723  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy metodę o nazwie jest funkcją, która zwraca wartość.
