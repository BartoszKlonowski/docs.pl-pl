# <a name="method-call-did-not-return-a-value"></a>Wywołanie metody nie zwróciło wartości

Wywołana metoda nie zwróciła wartości zgodnie z oczekiwaniami.  
  
 **Identyfikator błędu:** BC30723  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy wywoływana metoda jest funkcją, która zwraca wartość.
