# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>Deklaracja XML nie zezwala na użycie atrybutu "attributeName"

Określono nieprawidłowy atrybut dla literału dokumentu XML. Prawidłowe wartości atrybutów dla literału dokumentu XML są oparte na specyfikacji XML 1,0.  
  
 **Identyfikator błędu:** BC31154  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowy atrybut z literału dokumentu XML.
