# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>Deklaracja XML nie zezwala na atrybut "attributeName"
Określono nieprawidłowy atrybut literał dokumentu XML. Wartości prawidłowego atrybutu literał dokumentu XML są oparte na specyfikacji XML 1.0.  
  
 **Identyfikator błędu:** BC31154  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowy atrybut z literał dokumentu XML.
