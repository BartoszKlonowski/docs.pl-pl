# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>Deklaracja XML nie zezwala na atrybut &#39;attributeName&#39;
Określono nieprawidłowy atrybut literał dokumentu XML. Wartości atrybutów Nieprawidłowy literał dokumentu XML są oparte na Specyfikacja XML 1.0.  
  
 **Identyfikator błędu:** BC31154  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowy atrybut z literał dokumentu XML.
