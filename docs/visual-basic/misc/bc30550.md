# <a name="constructors-cannot-implement-interface-methods"></a>Konstruktory nie mogą implementować metod interfejsu

Próbowano użyć `New` słowa kluczowego, aby zaimplementować metodę interfejsu, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30550  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Implementuj interfejsy za pomocą `Implements` słowa kluczowego.
