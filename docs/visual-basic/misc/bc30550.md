# <a name="constructors-cannot-implement-interface-methods"></a>Konstruktory nie może implementować metody interfejsu
Podjęto próbę użycia `New` słowo kluczowe, aby zaimplementować metodę interfejsu, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30550  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Implementowanie interfejsów z `Implements` — słowo kluczowe.
