# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Metody w Module nie mogą implementować członków interfejsu
Elementy członkowskie interfejsu musi być implementowana przez klasy lub struktury.  
  
 **Identyfikator błędu:** BC31083  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Implementować członków interfejsu klasy lub struktury.
