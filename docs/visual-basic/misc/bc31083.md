# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Metody w module nie mogą implementować składowych interfejsu

Elementy członkowskie interfejsu muszą być implementowane przez klasy lub struktury.  
  
 **Identyfikator błędu:** BC31083  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Implementowanie członków interfejsu z klas lub struktur.
