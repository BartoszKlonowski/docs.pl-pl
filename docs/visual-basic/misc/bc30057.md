# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>Za dużo argumentów dla &#39; &lt;nazwaprocedury&gt;&#39;
Liczba argumentów w wywołaniu procedury przekracza liczbę argumentów zdefiniowane dla procedury.  
  
 **Identyfikator błędu:** BC30057  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zapoznaj się z definicji procedury i Usuń dodatkowe argumenty z wywołania.
