# <a name="is-operand-of-type-typename-can-only-be-compared-to-nothing-because-typename-is-a-nullable-type"></a>'Is' operand typu 'typename' można porównać tylko z "Nothing", ponieważ "typename" jest typem nullable
Zmienna zadeklarowana jako nullable została porównana z wyrażeniem innym niż `Nothing` przy użyciu `Is` operatora.  
  
 **Identyfikator błędu:** Bc32127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd
  
1. Aby porównać typ nullable do `Nothing` wyrażenia `Is` innego niż `GetType` przy użyciu operatora, wywołać metodę na typ nullable i porównać wynik do wyrażenia, jak pokazano w poniższym przykładzie.  
  
```vb  
Dim number? As Integer = 5  

If number IsNot Nothing Then  
  If number.GetType() Is Type.GetType("System.Int32") Then

  End If  
End If  
```
