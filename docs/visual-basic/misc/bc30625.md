# <a name="module-statement-must-end-with-a-matching-end-module"></a>Instrukcja "Module" musi być zakończona z instrukcją "End Module"
Blokuje moduł musi się kończyć `End Module` instrukcji.  
  
 **Identyfikator błędu:** BC30625  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Module` instrukcji na końcu `Module` bloku.
