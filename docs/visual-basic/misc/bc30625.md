# <a name="module-statement-must-end-with-a-matching-end-module"></a>Instrukcja "module" musi być zakończona odpowiadającą jej instrukcją "End Module"

Bloki modułów muszą kończyć się `End Module` instrukcją.  
  
 **Identyfikator błędu:** BC30625  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `End Module` instrukcję do końca `Module` bloku.
