# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39; Moduł &#39; Instrukcja musi kończyć się pasujący &#39; Moduł zakończenia &#39;
Blokuje moduł musi być zakończona odpowiadającą `End``Module` instrukcji.  
  
 **Identyfikator błędu:** BC30625  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Module` instrukcji na końcu `Module` bloku.
