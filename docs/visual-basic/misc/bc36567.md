# <a name="parameter-parametername-already-has-a-matching-argument"></a>Parametr '\<parametername >' ma już pasujący argument
Argument jest przekazywany przez nazwę więcej niż jeden raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC36567  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy jeden z nazwy argumentów jest błędna.  
  
- Usuń zduplikowane argumentu z listy argumentów.
