# <a name="parameter-39ltparameternamegt39-already-has-a-matching-argument"></a>Parametr &#39; &lt;parametername&gt;&#39; ma już pasujący argument
Argument jest przekazywany przez nazwę więcej niż jeden raz w tym samym wywołaniu procedury.  
  
 **Identyfikator błędu:** BC36567  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy nazwy argumentu jest błędna.  
  
-   Usuń zduplikowane argumentu z listy argumentów.
