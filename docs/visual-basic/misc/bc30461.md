# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Nie można zadeklarować klasy "&lt;specyfikator&gt;"
Podjęto próbę zadeklarować `Class` lub `Module` z Nieprawidłowy specyfikator.  
  
 **Identyfikator błędu:** BC30461  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
