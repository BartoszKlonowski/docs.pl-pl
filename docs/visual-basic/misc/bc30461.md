# <a name="classes-cannot-be-declared-specifier"></a>Nie można zadeklarować klas jako " \<specifier> "
Podjęto próbę zadeklarować `Class` lub `Module` z nieprawidłowym specyfikatorem.  
  
 **Identyfikator błędu:** BC30461  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
