# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować klasy &#39; &lt;Specyfikator&gt;&#39;
Podjęto próbę zadeklarować `Class` lub `Module` z Nieprawidłowy specyfikator.  
  
 **Identyfikator błędu:** BC30461  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
