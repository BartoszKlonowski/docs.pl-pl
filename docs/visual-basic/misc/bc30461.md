# <a name="classes-cannot-be-declared-specifier"></a>Nie można zadeklarować klasy '\<specyfikator >'
Podjęto próbę zadeklarować `Class` lub `Module` z Nieprawidłowy specyfikator.  
  
 **Identyfikator błędu:** BC30461  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
