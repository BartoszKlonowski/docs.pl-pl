# <a name="the-file-is-currently-open-for-writing"></a>Plik jest obecnie otwarty do zapisu

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik i spróbuj ponownie wykonać operację.
