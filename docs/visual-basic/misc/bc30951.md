# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Niepoprawna liczba argumentów typu przeniesiona do metody " \<genericprocedurename> ".

Procedura ogólna jest wywoływana z liczbą argumentów typu, które nie pasują do liczby parametrów typu, dla których jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30951  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj argument typu dla każdego parametru typu zdefiniowanego dla procedury ogólnej.  
  
     -lub-  
  
- Wywołaj procedurę generyczną bez argumentów typu w ogóle i pozwól kompilatorowi próbować wywnioskowania argumentów typu.
