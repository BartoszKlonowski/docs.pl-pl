# <a name="wrong-number-of-type-arguments-passed-to-method-39ltgenericprocedurenamegt39"></a>Niewłaściwa liczba argumentów typu przekazany do metody &#39; &lt;genericprocedurename&gt;&#39;
Procedury ogólne jest wywoływana z liczby argumentów typu, która jest niezgodna z liczbą parametrów typu, z którymi jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30951  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj typ argumentu dla każdego parametru typu zdefiniowanego dla ogólnego procedury.  
  
     —lub—  
  
-   Wywołanie procedury ogólnego bez argumentów typu na wszystkich i pozwól kompilatora próba wywnioskować argumentów typu.
