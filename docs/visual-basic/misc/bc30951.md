# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Niewłaściwa liczba argumentów typu przekazywany do metody "\<genericprocedurename >"
Ogólna procedura jest wywoływana z liczby argumentów typu nie jest zgodna z liczbą parametrów typu, z którymi jest zdefiniowana.  
  
 **Identyfikator błędu:** BC30951  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy podać argument typu dla każdego parametru typu zdefiniowane dla ogólnego procedury.  
  
     —lub—  
  
-   Wywołanie procedury ogólny bez argumentów typu na wszystkich i pozwolić kompilatorowi próba wywnioskować argumentów typu.
