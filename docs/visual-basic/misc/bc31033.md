# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>Interfejs '\<interfacename >' można zaimplementować tylko raz według tego typu
Wybrany interfejs został już zaimplementowany przez tego typu i nie można zaimplementować ponownie.  
  
 **Identyfikator błędu:** BC31033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Implementuje ten interfejs tylko raz przy użyciu tego typu.
