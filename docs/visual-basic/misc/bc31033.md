# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>Interfejs " \<interfacename> " można zaimplementować tylko raz według tego typu
Określony interfejs jest już zaimplementowany przez ten typ i nie można go zaimplementować ponownie.  
  
 **Identyfikator błędu:** BC31033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zaimplementuj ten interfejs przy użyciu tego typu tylko raz.
