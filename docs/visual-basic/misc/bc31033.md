# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Interfejs &#39; &lt;interfacename&gt; &#39; może być zaimplementowany tylko raz przez tego typu
Wybrany interfejs jest już zaimplementowany przez ten typ, a nie można zaimplementować ponownie.  
  
 **Identyfikator błędu:** BC31033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Implementuje ten interfejs tylko raz przy użyciu tego typu.
