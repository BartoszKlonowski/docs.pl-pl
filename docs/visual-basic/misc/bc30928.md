<span data-ttu-id="99ceb-103">Klasa jest zdefiniowana w co najmniej dwóch deklaracjach częściowych, które zawierają więcej niż jedną [instrukcję Inherits](../language-reference/statements/inherits-statement.md) określającą więcej niż jedną klasę bazową.</span><span class="sxs-lookup"><span data-stu-id="99ceb-103">A class is defined in two or more partial declarations, which contain more than one [Inherits Statement](../language-reference/statements/inherits-statement.md) specifying more than one base class.</span></span>  
  
 <span data-ttu-id="99ceb-104">Podczas dzielenia definicji klasy między kilka deklaracji częściowych kompilator traktuje klasę jako Unię wszystkich jej częściowych deklaracji.</span><span class="sxs-lookup"><span data-stu-id="99ceb-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="99ceb-105">Dotyczy to nie tylko elementów członkowskich, ale również do implementacji, dziedziczenia i poziomu dostępu.</span><span class="sxs-lookup"><span data-stu-id="99ceb-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="99ceb-106">Klasa może zaimplementować więcej niż jeden interfejs, ale nie może dziedziczyć z więcej niż jednej klasy bazowej.</span><span class="sxs-lookup"><span data-stu-id="99ceb-106">A class can implement more than one interface, but it cannot inherit from more than one base class.</span></span> <span data-ttu-id="99ceb-107">W związku z tym wszystkie `Inherits` instrukcje muszą określać tę samą klasę bazową.</span><span class="sxs-lookup"><span data-stu-id="99ceb-107">Therefore, all `Inherits` statements must specify the same base class.</span></span>
