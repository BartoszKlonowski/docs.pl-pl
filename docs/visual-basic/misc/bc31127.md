# <a name="39addhandler39-is-already-declared"></a>&#39; AddHandler &#39; jest już zadeklarowana
Więcej niż jeden `AddHandler` deklaracji pojawia się w deklaracji zdarzenia niestandardowego. `AddHandler` Deklaracji deklaruje procedurę umożliwiają dodawanie obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `AddHandler` instrukcji.
