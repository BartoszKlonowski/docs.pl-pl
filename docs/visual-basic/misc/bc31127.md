# <a name="addhandler-is-already-declared"></a>Element "AddHandler" jest już zadeklarowany
`AddHandler`W deklaracji zdarzenia niestandardowego występuje więcej niż jedna deklaracja. `AddHandler`Deklaracja deklaruje procedurę użytą do dodania procedury obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `AddHandler` .
