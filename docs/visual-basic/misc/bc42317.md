# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parametr typu komentarza XML &lt;parametru&gt; jest niezgodny z parametrem typu w odpowiedniej &lt;— słowo kluczowe&gt; — instrukcja
Parametr komentarza XML \<parametr > jest niezgodny z parametrem typu w odpowiedniej \<— słowo kluczowe > instrukcji. Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML został określony dla niektórych, ale nie wszystkich parametrów typu w metodzie.  
  
 **Identyfikator błędu:** BC42317  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj tagi komentarza XML dla wszystkich parametrów typu w metodzie.
