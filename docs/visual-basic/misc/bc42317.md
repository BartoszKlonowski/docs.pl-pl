# <a name="xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement"></a>Parametr typu komentarza XML nie \<parameter> jest zgodny z parametrem typu w odpowiedniej \<keyword> instrukcji
Parametr komentarza XML \<parameter> jest niezgodny z parametrem typu w odpowiedniej \<keyword> instrukcji. Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML został określony dla niektórych, ale nie wszystkich, parametrów typu w metodzie.  
  
 **Identyfikator błędu:** BC42317  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj Tagi komentarza XML dla wszystkich parametrów typu w metodzie.
