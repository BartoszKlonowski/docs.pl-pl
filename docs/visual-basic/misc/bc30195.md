# <a name="expected-one-of-dim-const-public-private-protected-friend-shadows-readonly-or-shared"></a>Oczekiwano typu "Ciemny", "Const", "Public", "Private", "Protected", "Friend", "Shadows", "ReadOnly" lub "Shared"
Brak słowa kluczowego deklaracji instrukcji deklaracji. Jedną z możliwych przyczyn jest to, że deklaracji atrybutu wywołuje metodę.  
  
 **Identyfikator błędu:** BC30195  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, jeśli metoda jest zadeklarowana wewnątrz deklaracji atrybutu.  
  
2.  Uruchom instrukcję za pomocą słowa kluczowego odpowiednie deklaracji.
