# <a name="expected-one-of-39dim39-39const39-39public39-39private39-39protected39-39friend39-39shadows39-39readonly39-or-39shared39"></a>Oczekiwano jednego z &#39; Dim &#39; &#39; Const &#39; &#39; Publicznego &#39; &#39; Prywatne &#39; &#39; Chronione &#39; &#39; Friend &#39; &#39; Shadows &#39; &#39; Tylko do odczytu &#39; i &#39; Udostępnione &#39;
Brak słowa kluczowego deklaracji instrukcji deklaracji. Jedną z możliwych przyczyn jest to, że deklaracji atrybutu wywołuje metodę.  
  
 **Identyfikator błędu:** BC30195  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy metoda jest zadeklarowana wewnątrz deklaracji atrybutu.  
  
2.  Uruchom instrukcję ze słowem kluczowym odpowiedniej deklaracji.
