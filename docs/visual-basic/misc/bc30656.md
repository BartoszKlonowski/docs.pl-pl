# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Pole &#39; &lt;fieldname&gt; &#39; ma nieobsługiwany typ.
Pole jest zadeklarowany jako typ, który nie może być reprezentowany przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30656  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zdefiniuj pole jako obsługiwanym typem.  
  
2.  Tworzenie nowego typu, która opisuje dane w polu.
