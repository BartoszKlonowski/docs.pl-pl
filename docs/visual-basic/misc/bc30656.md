# <a name="field-fieldname-is-of-an-unsupported-type"></a>Pole " \<fieldname> " ma nieobsługiwany typ
Pole jest zadeklarowane jako typ, który nie może być reprezentowany przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30656  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zdefiniuj pole jako obsługiwany typ.  
  
2. Utwórz nowy typ, który opisuje dane w polu.
