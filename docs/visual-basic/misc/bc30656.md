# <a name="field-ltfieldnamegt-is-of-an-unsupported-type"></a>Pole "&lt;fieldname&gt;" ma nieobsługiwany typ.
Pole jest zadeklarowany jako typ, który nie może być reprezentowana przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30656  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zdefiniuj pola jako obsługiwanego typu.  
  
2.  Tworzenie nowego typu, która opisuje dane w polu.
