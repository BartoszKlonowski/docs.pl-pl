# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Parametr komentarza XML " \<parameter> " nie jest zgodny z parametrem w odpowiedniej \<keyword> instrukcji

Parametr komentarza XML nie jest \<parameter> zgodny z parametrem w odpowiedniej \<keyword> instrukcji. Komentarz XML zostanie zignorowany.  
  
 Określono Tag komentarza XML dla niektórych, ale nie wszystkich parametrów w metodzie.  
  
 **Identyfikator błędu:** BC42307  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj Tag komentarza XML dla wszystkich parametrów w metodzie.
