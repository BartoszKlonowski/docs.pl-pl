# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parametr komentarza XML &#39; &lt;parametru&gt; &#39; jest niezgodny z parametrem w odpowiedniej &lt;— słowo kluczowe&gt; — instrukcja
Parametr komentarza XML \<parametr > jest niezgodny z parametrem w odpowiedniej \<— słowo kluczowe > instrukcji. Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML został określony dla niektórych, ale nie wszystkie parametry w metodzie.  
  
 **Identyfikator błędu:** BC42307  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj tag komentarza XML dla wszystkich parametrów metody.
