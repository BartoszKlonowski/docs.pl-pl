# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt; &#39; nie może zastąpić &#39; &lt;NazwaWłaściwości2&gt; &#39; , ponieważ różnią się one &#39;tylko do odczytu&#39; lub &#39;WriteOnly&#39;
Próbowano zastąpić właściwość z drugą właściwością, która różni się od pierwszego tylko przez `ReadOnly` lub `WriteOnly`.  
  
 **Identyfikator błędu:** BC30362  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że właściwości są zróżnicowane przez więcej niż `ReadOnly` i `WriteOnly`.
