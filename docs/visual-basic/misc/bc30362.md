# <a name="ltpropertyname1gt-cannot-override-ltpropertyname2gt-because-they-differ-by-readonly-or-writeonly"></a>"&lt;propertyname1&gt;nie może przesłonić&lt;NazwaWłaściwości2&gt;", ponieważ różnią się ustawieniem "ReadOnly" lub "WriteOnly"
Próbowano zastąpić właściwość z drugą właściwością, która różni się od pierwszego tylko przez `ReadOnly` lub `WriteOnly`.  
  
 **Identyfikator błędu:** BC30362  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że właściwości są zróżnicowane według więcej niż `ReadOnly` i `WriteOnly`.
