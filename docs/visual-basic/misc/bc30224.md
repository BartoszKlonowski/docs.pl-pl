# <a name="39is39-expected"></a>&#39;Jest&#39; oczekiwano
A `TypeOf` operatora odbywa się bez `Is` klauzuli.  
  
 **Identyfikator błędu:** BC30224  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Is` następujące klauzuli `TypeOf` operatora, na przykład `TypeOf MyControl Is ComboBox`.
