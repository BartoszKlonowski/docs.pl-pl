<span data-ttu-id="a35cf-103">Procedura lub właściwość próbuje zastąpić dziedziczonej procedury lub właściwości, ale kompilator odnajdzie więcej niż jedną wersję podstawowej procedury lub właściwości o tej samej nazwie i podpisie.</span><span class="sxs-lookup"><span data-stu-id="a35cf-103">A procedure or property attempts to override an inherited procedure or property, but the compiler finds more than one version of the base procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="a35cf-104">Ten błąd może wystąpić w sytuacji z konstruowanymi typami ogólnymi, ponieważ ilustruje to następujące deklaracje szkieletu.</span><span class="sxs-lookup"><span data-stu-id="a35cf-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```vb

    Public Overridable Sub doSomething(ByVal inputValue As String)  
    End Sub  
    Public Overridable Sub doSomething(ByVal inputValue As t)  
    End Sub  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of String)  
    Overrides Sub doSomething(ByVal inputValue As String)  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="a35cf-105">Ponieważ `derivedClass` dziedziczy `baseClass` dostarczenie `String` do jego parametru typu `t` , dwie wersje `doSomething` w `baseClass` systemie pobierają identyczne sygnatury, o ile `derivedClass` jest to konieczne.</span><span class="sxs-lookup"><span data-stu-id="a35cf-105">Because `derivedClass` inherits `baseClass` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseClass` take on identical signatures as far as `derivedClass` is concerned.</span></span> <span data-ttu-id="a35cf-106">W związku z tym kompilator nie może określić wersji, która ma zostać przesłonięta.</span><span class="sxs-lookup"><span data-stu-id="a35cf-106">As a result, the compiler cannot determine which version to override.</span></span>
