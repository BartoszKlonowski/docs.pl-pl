<span data-ttu-id="43577-103">Procedura lub właściwość próbuje przesłonić dziedziczonej procedura lub właściwość, ale kompilator znajdzie więcej niż jedna wersja bazowej procedury lub właściwość o tej samej nazwie i sygnaturze.</span><span class="sxs-lookup"><span data-stu-id="43577-103">A procedure or property attempts to override an inherited procedure or property, but the compiler finds more than one version of the base procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="43577-104">Ten błąd może wystąpić w sytuacji, za pomocą typów ogólnych skonstruowany, jak pokazują następujące deklaracje szkielet.</span><span class="sxs-lookup"><span data-stu-id="43577-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```

    Public Overridable Sub doSomething(ByVal inputValue As String)  
    End Sub  
    Public Overridable Sub doSomething(ByVal inputValue As t)  
    End Sub  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of String)  
    Overrides Sub doSomething(ByVal inputValue As String)  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="43577-105">Ponieważ `derivedClass` dziedziczy `baseClass` dostarczenie `String` na jego parametr typu `t`, dwie wersje `doSomething` w `baseClass` przyjmą identycznych podpisach w miarę `derivedClass` dotyczy.</span><span class="sxs-lookup"><span data-stu-id="43577-105">Because `derivedClass` inherits `baseClass` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseClass` take on identical signatures as far as `derivedClass` is concerned.</span></span> <span data-ttu-id="43577-106">W rezultacie kompilator nie może określić wersji do zastąpienia.</span><span class="sxs-lookup"><span data-stu-id="43577-106">As a result, the compiler cannot determine which version to override.</span></span>
