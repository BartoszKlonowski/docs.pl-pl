# <a name="members-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować elementów członkowskich w Module &#39; &lt;Specyfikator&gt;&#39;
Zostały użyte specyfikator, który jest nieprawidłowy dla elementu członkowskiego w `Module` instrukcji. Moduły nigdy nie można utworzyć wystąpienia, nie obsługuje dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30436  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
