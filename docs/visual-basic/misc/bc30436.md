# <a name="members-in-a-module-cannot-be-declared-specifier"></a>Elementy członkowskie w Module nie można zadeklarować jako '\<specyfikator >'
Wykorzystano specyfikator, który jest nieprawidłowy w elemencie członkowskim w ramach `Module` instrukcji. Moduły nigdy nie mogą zostać utworzone, nie obsługują dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30436  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
