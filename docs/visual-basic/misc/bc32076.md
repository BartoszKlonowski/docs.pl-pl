# <a name="39lttypenamegt39-is-a-generic-type-and-requires-type-arguments"></a>&#39;&lt;Właściwość TypeName&gt; &#39; jest typem ogólnym i wymaga argumentów typu
Zmienna, parametr procedury lub funkcji, zwracany jest zadeklarowany jako typ ogólny klasy lub struktury, ale deklaracji nie dostarcza żadnych argumentów typu.  
  
 Z natury co ogólnej klasy i struktury jest zdefiniowana z co najmniej jeden typ parametru. Użycie typu ogólnego Aby zadeklarować skonstruowane klasy lub struktury, należy podać typ argumentu dla każdego parametru typu określone przez typ ogólny.  
  
 **Identyfikator błędu:** BC32076  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodawanie listy typów do deklaracji ujęte w nawiasy, a począwszy od `Of` — słowo kluczowe.
