# <a name="for-loop-not-initialized-visual-basic"></a>Nie zainicjowano pętli (Visual Basic)
Wyniósł do środka `For...Next` pętli.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń skok do pętli.
