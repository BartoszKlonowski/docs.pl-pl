# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>Nieprawidłowo sformułowany kod XML. \<commentfile>Nie można uwzględnić pliku ""
`<include>`Tag w jednym z komentarzy XML odwołuje się do nieistniejącego pliku lub pliku zawierającego nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42321  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy plik istnieje.  
  
2. Sprawdź, czy kod XML w pliku jest prawidłowy.
