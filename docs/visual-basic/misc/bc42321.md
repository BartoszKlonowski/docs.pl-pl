# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>Nieprawidłowo sformułowany kod XML. Plik '\<commentfile >' nie może być dołączony
`<include>` Tagu w jednym z komentarzy XML odwołuje się do pliku, który nie istnieje lub plik, który zawiera nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42321  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy plik istnieje.  
  
2. Sprawdź, czy kod XML w pliku jest prawidłowa.
