# <a name="badly-formed-xml-file-39ltcommentfilegt39-cannot-be-included"></a>Nieprawidłowo sformułowany kod XML. Plik &#39; &lt;commentfile&gt; &#39; nie można dołączyć
`<include>` Tag w jednym z komentarze XML odwołuje się do pliku, który nie istnieje lub plik, który zawiera nieprawidłowo sformułowany kod XML.  
  
 **Identyfikator błędu:** BC42321  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy plik istnieje.  
  
2.  Sprawdź poprawność kodu XML w pliku.
