# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'\<nazwa_zestawu >' jest zestawem i nie może być przywoływany jako modułu
Próbowano użyć zestawu jako moduł.  
  
 **Identyfikator błędu:** BC31077  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień odwołania do modułu.
