# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>element " \<assemblyname> " jest zestawem i nie można się do niego odwoływać jako modułu

Podjęto próbę użycia zestawu jako modułu.  
  
 **Identyfikator błędu:** BC31077  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień odwołanie do modułu.
