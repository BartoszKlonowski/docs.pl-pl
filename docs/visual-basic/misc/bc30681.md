# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39; #Region &#39; Instrukcja musi być zakończona z odpowiadającego mu &#39; #End Region &#39;
Użyj `#Region` dyrektywy bloku kodu, który można rozwinąć lub zwinąć, korzystając z funkcji zwijania [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] edytora kodu. Początek i koniec `#Region` instrukcje musi być w tym samym bloku kodu.  
  
 **Identyfikator błędu:** BC30681  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Wstaw `#End Region` w odpowiednim miejscu w kodzie po `#Region` instrukcji.
