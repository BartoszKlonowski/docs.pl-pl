# <a name="array-dimensions-cannot-have-a-negative-size"></a>Wymiary tablicy nie może mieć rozmiaru ujemnego
Co najmniej jeden granic tablicy określona jest wartość ujemną. Ujemna wartość indeksu można określić tylko wtedy, gdy użycie górna granica -1 w celu zadeklarowania pustą tablicę. Takie tablicy nie ma żadnego elementu, ale nie jest `Nothing`.  
  
 **Identyfikator błędu:** BC30611  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp ujemna tablicy związane z liczbą dodatnią.
