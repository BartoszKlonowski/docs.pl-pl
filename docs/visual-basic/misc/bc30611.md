# <a name="array-dimensions-cannot-have-a-negative-size"></a>Wymiary tablicy nie może mieć ujemnym rozmiarem
Co najmniej jeden określony granice tablicy jest liczbą ujemną. Ujemny indeks dolny można określić tylko wtedy, gdy używasz górna granica wartości -1 do deklarowania pustą tablicę. Takie tablica nie ma żadnego elementu, ale nie jest `Nothing`.  
  
 **Identyfikator błędu:** BC30611  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp tablicy ujemna, powiązane z liczbą dodatnią.
