# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>\<typename>nie można użyć elementu "" jako atrybutu, ponieważ nie jest on klasą

Podjęto próbę użycia typu, który nie jest klasą jako atrybutem.  
  
 **Identyfikator błędu:** BC31503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zdefiniuj atrybuty niestandardowe jako klasy pochodne od `System.Attribute` .
