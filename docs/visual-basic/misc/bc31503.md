# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>"\<typename >" nie można użyć jako atrybutu, ponieważ nie jest klasą
Nastąpiła próba typu, który nie jest klasą jako atrybut.  
  
 **Identyfikator błędu:** BC31503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Definiowanie atrybutów niestandardowych, które jako klasy, które wynikają z `System.Attribute`.
