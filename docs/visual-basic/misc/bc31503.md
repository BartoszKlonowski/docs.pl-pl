# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39; &lt;typename&gt;&#39; nie można użyć jako atrybutu, ponieważ nie jest klasą
Próbowano użyć typu, który nie jest klasą jako atrybut.  
  
 **Identyfikator błędu:** BC31503  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Definiowanie atrybutów niestandardowych jako klasy, które pochodzą z `System.Attribute`.
