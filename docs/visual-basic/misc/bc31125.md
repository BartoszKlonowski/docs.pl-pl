# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;End RemoveHandler&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;RemoveHandler&#39; deklaracji
`End RemoveHandler` Instrukcji wystąpił bez odpowiadającego `RemoveHandler` instrukcji. `End RemoveHandler` musi być poprzedzona odpowiadającą `RemoveHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31125  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy poprzednie `RemoveHandler` instrukcji jest prawidłowa i poprawna.
