# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>"End RemoveHandler" musi być poprzedzona zgodna deklaracja "RemoveHandler"
`End RemoveHandler` Instrukcji wystąpił bez odpowiadającego `RemoveHandler` instrukcji. `End RemoveHandler` musi być poprzedzona odpowiadającą `RemoveHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31125  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy poprzednie `RemoveHandler` instrukcja jest prawidłowa i poprawna.
