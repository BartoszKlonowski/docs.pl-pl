# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>Element "End RemoveHandler" musi być poprzedzony odpowiadającą mu deklaracją "RemoveHandler"
`End RemoveHandler`Instrukcja wystąpiła bez odpowiedniej `RemoveHandler` instrukcji. `End RemoveHandler`musi być poprzedzona odpowiadającą jej `RemoveHandler` instrukcją.  
  
 **Identyfikator błędu:** BC31125  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy Poprzednia `RemoveHandler` instrukcja jest prawidłowa i poprawna.
