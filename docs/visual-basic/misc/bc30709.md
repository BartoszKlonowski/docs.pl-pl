# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje deklaracji zmiennych nie są prawidłowe w oknie bezpośrednim
Deklaracje zmiennych nie można używać w **bezpośrednie** okna.  
  
 **Identyfikator błędu:** BC30709  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie deklaruj zmiennych w **bezpośrednie** okna.
