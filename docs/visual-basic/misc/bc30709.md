# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje deklaracji zmiennej nie są prawidłowe w oknie bezpośrednim

Deklaracji zmiennych nie można używać w oknie **bezpośrednim** .  
  
 **Identyfikator błędu:** BC30709  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie deklaruj zmiennych w oknie **bezpośrednim** .
