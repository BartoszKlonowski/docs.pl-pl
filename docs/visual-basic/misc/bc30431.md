# <a name="end-property-must-be-preceded-by-a-matching-property"></a>"End Property" musi być poprzedzona instrukcją "Property"
`End Property` Występuje instrukcja w kodzie nie słów `Property` deklaracji przed nim.  
  
 **Identyfikator błędu:** BC30431  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `End Property` instrukcji, jeśli jest nadmiarowy.  
  
- Podaj brakujące `Property` procedury, jeśli brakuje jednego.  
  
- Przenieś `End Property` w odpowiednie miejsce w kodzie.
