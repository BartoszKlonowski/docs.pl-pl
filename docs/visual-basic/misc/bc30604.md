# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>Instrukcja nie może występować w treści interfejsu (błąd w Visual Basic)
Napotkano nieoczekiwany język konstrukcji. Zakłada się, że `End Interface` konstrukcja nie istnieje, i dowolnego kodu źródłowego po tym punkcie znajduje się poza interfejsu.  
  
 **Identyfikator błędu:** BC30604  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź składnię kodu używana wewnątrz definicji interfejsu.  
  
2.  Upewnij się, że definicja interfejsu kończy się `End Interface` konstruowania.
