# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>Instrukcja nie może występować w treści interfejsu (Visual Basic błąd)

Napotkano nieoczekiwaną konstrukcję języka. Przyjęto założenie, że `End Interface` konstrukcja nie istnieje i że każdy kod źródłowy po tym punkcie znajduje się poza interfejsem.  
  
 **Identyfikator błędu:** BC30604  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź składnię kodu używanego wewnątrz definicji interfejsu.  
  
2. Upewnij się, że definicja interfejsu jest zakończona z `End Interface` konstrukcja.
