# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>Instrukcja nie może wystąpić w treści interfejsu (Błąd Visual Basic)
Napotkano konstrukcję nieoczekiwany języka. Zakłada ona, że `End Interface` brakuje konstrukcję i kod źródłowy po tym punkcie znajduje się poza interfejsu.  
  
 **Identyfikator błędu:** BC30604  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź składnię kod używany wewnątrz definicji interfejsu.  
  
2.  Upewnij się, że definicja interfejsu kończy się `End Interface` utworzenia.
