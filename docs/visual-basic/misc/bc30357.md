# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>Instrukcje "Inherits" muszą poprzedzać wszystkie deklaracje w interfejsie

`Inherits`Instrukcja musi być pierwszą niepustą linią niebędącą komentarzem w definicji klasy.  
  
 **Identyfikator błędu:** BC30357  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś `Inherits` instrukcję, aby poprzedzał wszystkie deklaracje.
