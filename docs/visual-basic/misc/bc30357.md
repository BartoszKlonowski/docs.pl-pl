# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>Instrukcje "Inherits" musi poprzedzać wszystkie deklaracje w interfejsie
`Inherits` Instrukcja musi być pierwszy wiersz nie jest pusty, bez komentarza w definicji klasy.  
  
 **Identyfikator błędu:** BC30357  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś swoje `Inherits` instrukcji, tak że poprzedza wszystkie deklaracje.
