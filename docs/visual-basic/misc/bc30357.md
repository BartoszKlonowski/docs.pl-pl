# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;Dziedziczy&#39; instrukcje musi występować przed wszelkimi deklaracjami w interfejsie
`Inherits` Instrukcja musi być pierwszym wierszem niepustą, komentarza-w definicji klasy.  
  
 **Identyfikator błędu:** BC30357  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieść użytkownika `Inherits` instrukcję, tak że poprzedza wszystkie deklaracje.
