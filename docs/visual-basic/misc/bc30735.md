# <a name="type-in-a-module-cannot-be-declared-ltspecifiergt"></a>Typ w Module nie można zadeklarować jako "&lt;specyfikator&gt;"
Typ o nieprawidłowe modyfikatory został zadeklarowany w module.  
  
 **Identyfikator błędu:** BC30735  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie używaj `Protected` lub `Protected Friend` modyfikatorów na typach zadeklarowany w module.
