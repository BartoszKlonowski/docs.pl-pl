# <a name="type-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować typu w Module &#39; &lt;Specyfikator&gt;&#39;
Typ z nieprawidłową Modyfikatory został zadeklarowany w module.  
  
 **Identyfikator błędu:** BC30735  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie używaj `Protected` lub `Protected Friend` modyfikatorów dla typów zadeklarowany w module.
