# <a name="type-in-a-module-cannot-be-declared-specifier"></a>Nie można zadeklarować typu w module jako " \<specifier> "
Typ z nieprawidłowymi modyfikatorami został zadeklarowany w module.  
  
 **Identyfikator błędu:** BC30735  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie używaj `Protected` ani `Protected Friend` modyfikatorów dla typów zadeklarowanych w module.
