# <a name="typename-is-an-enum-type-and-cannot-be-used-as-an-expression"></a>"\<typename >" jest typu wyliczeniowego, a nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wyrażenie jest wymagana. Wyrażenie musi składać się z kombinacji zmienne, stałe, literały, właściwości i `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30107  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłowe.
