# <a name="get-statement-must-end-with-a-matching-end-get"></a>Instrukcja "Get" musi być zakończona odpowiadającą jej instrukcją "End Get"
`Get`procedury właściwości muszą kończyć się `End Get` instrukcją.  
  
 **Identyfikator błędu:** BC30631  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że `Get` procedura właściwości jest zakończona w `End Get` konstrukcji.
