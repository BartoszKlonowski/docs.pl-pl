# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39; Pobierz &#39; Instrukcja musi kończyć się pasujący &#39; Pobierz koniec &#39;
`Get`procedury własności musi kończyć się `End Get` instrukcji.  
  
 **Identyfikator błędu:** BC30631  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `Get` kończy procedurę właściwości `End Get` utworzenia.
