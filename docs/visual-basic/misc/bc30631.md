# <a name="get-statement-must-end-with-a-matching-end-get"></a>Instrukcja "Get" musi kończyć się instrukcją "End Get"
`Get` procedury własności musi się kończyć `End Get` instrukcji.  
  
 **Identyfikator błędu:** BC30631  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `Get` kończy procedurę właściwości `End Get` konstruowania.
