# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39; GoSub &#39; instrukcje są już obsługiwane.
`GoSub`Nie można wywołać procedury.  
  
 **Identyfikator błędu:** BC30814  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wywołanie procedury bezpośrednio, bez użycia `GoSub`, np.:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
