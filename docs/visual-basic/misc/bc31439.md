# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Metoda częściowa &#39; &lt;methodname&gt;&#39; nie można użyć &#39; implementuje &#39;— słowo kluczowe
Definicja metoda częściowa nie może implementować interfejsu lub elementu członkowskiego interfejsu.  
  
 **Identyfikator błędu:** BC31439  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Implements` instrukcji z definicji metody częściowej.
