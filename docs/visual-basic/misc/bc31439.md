# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>Metoda częściowa "\<methodname >" nie można użyć słowa kluczowego "Implements"
Definicji metody częściowej nie może implementować interfejs lub składowej interfejsu.  
  
 **Identyfikator błędu:** BC31439  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `Implements` instrukcji z definicji metody częściowej.
