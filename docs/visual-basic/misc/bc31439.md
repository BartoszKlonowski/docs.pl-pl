# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>Metoda częściowa " \<methodname> " nie może używać słowa kluczowego "Implements"
Częściowa definicja metody nie może implementować elementu członkowskiego interfejsu lub interfejsu.  
  
 **Identyfikator błędu:** BC31439  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `Implements` instrukcję ze częściowej definicji metody.
