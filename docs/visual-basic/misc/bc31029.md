# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Metoda &#39; &lt;methodname&gt;&#39; nie może obsłużyć zdarzenia &#39;&lt; eventName&gt;&#39; ponieważ nie mają tę samą sygnaturę
Parametry lub typy parametrów programu obsługi zdarzeń określony nie pasują zdefiniowane dla określonego zdarzenia.  
  
 **Identyfikator błędu:** BC31029  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ponownie zdefiniować zdarzenia lub program obsługi zdarzeń, aby parametr wymieniono dopasowania.
