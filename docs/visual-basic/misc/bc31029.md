# <a name="method-methodname-cannot-handle-event-eventname-because-they-do-not-have-the-same-signature"></a>Metoda '\<methodname >' nie może obsłużyć zdarzenia'\<eventname > ', ponieważ nie mają taki sam podpis
Parametrów ani typów parametrów dla procedury obsługi określonego zdarzenia nie są zgodne identyczne ze zdefiniowanymi dla określonego zdarzenia.  
  
 **Identyfikator błędu:** BC31029  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ponownie zdefiniować zdarzeń lub procedura obsługi zdarzeń, aby parametr zawiera dopasowania.
