# <a name="method-methodname-cannot-handle-event-eventname-because-they-do-not-have-the-same-signature"></a>Metoda " \<methodname> " nie może obsłużyć zdarzenia " \<eventname> ", ponieważ nie mają tej samej sygnatury
Parametry lub typy parametrów dla określonego programu obsługi zdarzeń nie są zgodne z określonymi dla określonego zdarzenia.  
  
 **Identyfikator błędu:** BC31029  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj ponownie zdarzenie lub procedurę obsługi zdarzeń, tak aby lista parametrów była zgodna.
