# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>'\<argumentname >' nie jest parametrem'\<methoddeclaration > '
Wywołanie procedury dostarcza dodatkowy argument, który nie jest zdefiniowany przez procedurę.  
  
 **Identyfikator błędu:** BC30272  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zbędne argumentu z listy argumentów.
