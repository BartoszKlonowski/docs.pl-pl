# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39; &lt;argumentname&gt;&#39; nie jest parametrem &#39;&lt; MethodDeclaration&gt;&#39;
Wywołanie procedury udostępnia dodatkowy argument, który nie jest zdefiniowany przez procedurę.  
  
 **Identyfikator błędu:** BC30272  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędny argumentu z listy argumentów.
