# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>" \<modulename> " jest modułem i nie można się do niego odwoływać jako zestawu

Podjęto próbę użycia modułu jako zestawu.  
  
 **Identyfikator błędu:** BC31076  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień odwołanie do zestawu.
