# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<nazwa_modułu >' jest modułem i nie może być przywoływany jako zestawu
Nastąpiła próba używania modułu jako zestawu.  
  
 **Identyfikator błędu:** BC31076  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień odwołania do zestawu.
