# <a name="module-attribute-attributename-is-not-valid-error"></a>Atrybut modułu '\<attributename >' jest nieprawidłowa: \<błąd >
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w module.

**Identyfikator błędu:** BC30130

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zastąp go atrybutem, który jest prawidłowy dla kontekstu.

2. Sprawdź komunikat o błędzie w cudzysłowach i podjąć odpowiednie działania.
