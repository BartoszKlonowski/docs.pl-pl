# <a name="module-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Atrybut modułu "&lt;attributename&gt;" jest nieprawidłowa: &lt;błąd&gt;
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w module.

**Identyfikator błędu:** BC30130

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Usuń atrybut z deklaracji lub zamień go atrybutem, który jest prawidłowy dla kontekstu.

2. Sprawdź komunikat o błędzie w cudzysłowach i podjąć odpowiednie działania.
