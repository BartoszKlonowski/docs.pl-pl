# <a name="unable-to-access-member"></a>Nie można uzyskać dostępu do członka
Nie można uzyskać dostępu do określonego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30749  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, że składowa jest zdefiniowana i prawidłowo wpisana nazwa elementu członkowskiego.  
  
2. Wypróbuj, uzyskiwanie dostępu do jednego z elementów członkowskich zadeklarowanych w module. W niektórych przypadkach środowisko debugowania nie można zlokalizować elementów członkowskich, ponieważ moduły, w której są deklarowane nie został jeszcze załadowany.
