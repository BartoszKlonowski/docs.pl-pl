# <a name="unable-to-access-member"></a>Nie można uzyskać dostępu do członka
Nie można uzyskać dostępu do określonego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC30749  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy element członkowski jest zdefiniowany i prawidłowo wpisana nazwa elementu członkowskiego.  
  
2.  Spróbuj, uzyskiwanie dostępu do jednego z elementów członkowskich zadeklarowany w module. W niektórych przypadkach debugowania środowiska nie można zlokalizować elementów członkowskich, ponieważ moduły, w którym je zadeklarowano nie zostały jeszcze załadowane.
