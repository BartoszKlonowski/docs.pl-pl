# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<type1 > "\<typename >" i \<type2 > "\<typename >" powoduje konflikt w elemencie \<przestrzeni nazw > "\<namespacename >"
Dwa zadeklarowane elementy, takie jak klasy i moduł, mają taką samą nazwę w tej samej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30179  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś deklarację jeden z elementów do innej przestrzeni nazw.  
  
- Zmień nazwę jednego z elementów.
