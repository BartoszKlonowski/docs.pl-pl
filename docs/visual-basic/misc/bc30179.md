# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39;&lt; Właściwość TypeName&gt;&#39; i &lt;type2&gt; &#39;&lt; Właściwość TypeName&gt;&#39; konflikt w &lt;przestrzeni nazw&gt; &#39;&lt; Nazwa przestrzeni nazw&gt;&#39;
Dwa elementy zadeklarowane, takich jak klasy i moduł, mieć takiej samej nazwie w tej samej przestrzeni nazw.  
  
 **Identyfikator błędu:** BC30179  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś deklarację jednego z elementów na inną przestrzeń nazw.  
  
-   Zmień nazwę jednego z elementów.
