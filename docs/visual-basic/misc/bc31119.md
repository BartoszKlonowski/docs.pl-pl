# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>Element "End RemoveHandler" musi być pierwszą instrukcją w wierszu

`End RemoveHandler`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End RemoveHandler` musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31119  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
