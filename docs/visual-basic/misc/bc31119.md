# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>"End RemoveHandler" musi być pierwszą instrukcją w wierszu
`End RemoveHandler` Instrukcji następuje separatorem instrukcji dwukropek (:). `End RemoveHandler` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31119  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Oddziel wiele instrukcji na różne wiersze.
