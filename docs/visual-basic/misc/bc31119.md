# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39; End RemoveHandler &#39; musi być pierwszą instrukcją w wierszu
`End RemoveHandler` Instrukcji następuje separatora instrukcji dwukropka (:). `End RemoveHandler`musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31119  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
