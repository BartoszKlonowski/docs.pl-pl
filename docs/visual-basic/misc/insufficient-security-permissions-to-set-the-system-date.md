# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Uprawnienia zabezpieczeń nie wystarczają do ustawienia daty systemowej
Uprawnienia zabezpieczeń nie wystarczają do ustawienia daty systemowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień uprawnienia zabezpieczeń umożliwiające ustawienie daty systemowej.
