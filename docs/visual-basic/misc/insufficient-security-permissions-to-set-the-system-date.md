# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Niewystarczającymi uprawnieniami, aby ustawić daty systemowej
Uprawnienia zabezpieczeń są niewystarczające do ustawiania daty systemowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmienić uprawnienia zabezpieczeń umożliwiające ustawienie daty systemowej.
