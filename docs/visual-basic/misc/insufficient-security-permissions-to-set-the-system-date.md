# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Niewystarczające uprawnienia zabezpieczeń do ustawienia daty systemowej

Uprawnienia zabezpieczeń są niewystarczające do ustawienia daty systemowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień uprawnienia zabezpieczeń, aby zezwolić na ustawienie daty systemowej.
