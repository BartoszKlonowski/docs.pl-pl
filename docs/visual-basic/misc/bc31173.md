# <a name="missing-required-white-space"></a>Brak wymaganego białego znaku
Literał XML został określony bez spacji rozdzielające. Przykład przestrzeni rozdzielające to obszar między kolejnych atrybutów w literał elementu XML.  
  
 **Identyfikator błędu:** BC31173  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodawanie separatora wymaganego miejsca.
