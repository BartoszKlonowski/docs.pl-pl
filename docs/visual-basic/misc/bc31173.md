# <a name="missing-required-white-space"></a>Brak wymaganego odstępu
Literał XML został określony bez wymaganego odstępu. Przykładem wymaganego miejsca separatora jest spacja między kolejnymi atrybutami w literale elementu XML.  
  
 **Identyfikator błędu:** BC31173  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagany separator spacji.
