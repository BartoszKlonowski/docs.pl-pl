# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-lttype3gt-is-not-a-reference-type"></a>Wartość typu "&lt;type1&gt;"nie można przekonwertować na"&lt;type2&gt;"ponieważ"&lt;type3&gt;" nie jest typem referencyjnym
Instrukcję stara się przekonwertować typ tablicy na inny typ tablicy, w których typ danych elementów tablicy docelowej nie jest typem referencyjnym.  
  
 **Identyfikator błędu:** BC30333  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź typy danych elementów w obu tablicach, aby ustalić przyczynę błędu.
