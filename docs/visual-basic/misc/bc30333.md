# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-a-reference-type"></a>\<type1>Nie można przekonwertować wartości typu "" na " \<type2> ", ponieważ element " \<type3> " nie jest typem referencyjnym
Instrukcja próbuje skonwertować typ tablicy do innego typu tablicy, gdzie typ danych elementów tablicy docelowej nie jest typem referencyjnym.  
  
 **Identyfikator błędu:** BC30333  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź typy danych elementów obu tablic, aby określić źródło błędu.
