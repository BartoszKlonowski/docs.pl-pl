<span data-ttu-id="c4cb5-103">Zadeklarowano typ ogólny z ograniczeniami powodującymi konflikt w parametrze typu.</span><span class="sxs-lookup"><span data-stu-id="c4cb5-103">A generic type is declared with conflicting constraints on a type parameter.</span></span>  
  
 <span data-ttu-id="c4cb5-104">Poniższe instrukcje mogą generować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="c4cb5-104">The following statement can generate this error.</span></span>  
  
 `Public Class testClass(Of t As {Structure, Class })`  
  
 <span data-ttu-id="c4cb5-105">Ograniczenia `Structure` i `Class` powodują konflikt dla parametru typu `t` , ponieważ `Structure` ograniczenie wymaga, aby odpowiedni argument typu był typem wartości, a jednocześnie wymaga, aby był `Class` typem referencyjnym.</span><span class="sxs-lookup"><span data-stu-id="c4cb5-105">The constraints `Structure` and `Class` cause a conflict for type parameter `t`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
