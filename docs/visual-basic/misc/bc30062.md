# <a name="39me39-cannot-be-the-target-of-an-assignment"></a>&#39;ME&#39; nie może być elementem docelowym przypisania
`Me` — Słowo kluczowe zawsze odwołuje się do określonego wystąpienia klasy, gdy kod jest aktualnie wykonywany. Nie można zmienić jej wartość.  
  
 **Identyfikator błędu:** BC30062  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń instrukcji, która przypisuje wartość do `Me`.
