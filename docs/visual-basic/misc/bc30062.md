# <a name="me-cannot-be-the-target-of-an-assignment"></a>"Me" nie może być elementem docelowym przypisania
`Me` — Słowo kluczowe zawsze odwołuje się do określonego wystąpienia klasy, w którym aktualnie wykonuje kod. Nie można zmienić jego wartość.  
  
 **Identyfikator błędu:** BC30062  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję, która przypisuje wartość do `Me`.
