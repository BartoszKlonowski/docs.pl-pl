# <a name="finally-can-only-appear-once-in-a-try-statement"></a>Instrukcja "finally" może wystąpić tylko raz w instrukcji "try"
`Finally`służy do ukończenia `Try...Catch...Finally` bloku; w związku z tym może wystąpić tylko raz na końcu bloku.  
  
 **Identyfikator błędu:** BC30381  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Znajdź i Usuń nadmiarowy `Finally` .
