# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39; Na koniec &#39; może wystąpić tylko raz w &#39; Spróbuj &#39; — Instrukcja
`Finally`Służy do przeprowadzenia `Try...Catch...Finally` zablokować; dlatego go może wystąpić tylko raz na końcu bloku.  
  
 **Identyfikator błędu:** BC30381  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zlokalizuj i Usuń zbędną `Finally`.
