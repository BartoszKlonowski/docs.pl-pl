# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39; &lt;modyfikator&gt;&#39; nie jest prawidłowy w deklaracji stałej lokalnej
A `Const` instrukcji wewnątrz procedury zawiera nieprawidłowe słowo kluczowe, takie jak `Overloads`.  
  
 **Identyfikator błędu:** BC30246  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń nieprawidłowe słowo kluczowe z `Const` instrukcji.
