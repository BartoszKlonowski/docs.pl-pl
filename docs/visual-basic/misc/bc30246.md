# <a name="modifier-is-not-valid-on-a-local-constant-declaration"></a>'\<modyfikator >' nie jest prawidłowy w deklaracji stałej lokalnej
A `Const` instrukcji wewnątrz procedury zawiera nieprawidłowe słowo kluczowe, takie jak `Overloads`.  
  
 **Identyfikator błędu:** BC30246  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń nieprawidłowe słowo kluczowe z `Const` instrukcji.
