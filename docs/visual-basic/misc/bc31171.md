# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Znak sekwencji &#39;--&#39; nie jest dozwolona w komentarzu XML
Literał komentarza XML zawiera sekwencję znaków "--", który nie jest obsługiwany.  
  
 **Identyfikator błędu:** BC31171  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń sekwencja znaków "--" z literał komentarza XML.
