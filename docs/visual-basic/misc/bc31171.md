# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>Sekwencja znaków "--" nie jest dozwolona w komentarzu XML
Literał komentarza XML zawiera sekwencję znaków "--", która nie jest obsługiwana.  
  
 **Identyfikator błędu:** BC31171  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń sekwencję znaków "--" z literał komentarza XML.
