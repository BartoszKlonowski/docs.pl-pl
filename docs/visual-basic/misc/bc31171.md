# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>Sekwencja znaków "--" jest niedozwolona w komentarzu XML

Literał komentarza XML zawiera sekwencję znaków "--", która nie jest obsługiwana.  
  
 **Identyfikator błędu:** BC31171  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń sekwencję znaków "--" z literału komentarza XML.
