# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>nie można znaleźć elementu " \<name> ", który jest niezbędny do skompilowania tej konstrukcji.

Nie można znaleźć elementu wymaganego do kompilacji.  
  
 **Identyfikator błędu:** BC30930  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Podaj wymagany element pomocnika.  
  
2. Upewnij się, że kod jest prawidłowym kodem zarządzanym.
