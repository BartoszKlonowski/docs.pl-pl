# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39; &lt;nazwa&gt;&#39; niezbędne do skompilowania tej konstrukcji, nie można odnaleźć
Nie można odnaleźć elementu niezbędne do kompilacji.  
  
 **Identyfikator błędu:** BC30930  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Należy podać element niezbędne pomocnika.  
  
2.  Upewnij się, że kod jest prawidłowy kod zarządzany.
