# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>"\<name >", jest to niezbędne do kompilowania ta konstrukcja nie można odnaleźć
Nie można odnaleźć elementu niezbędne do kompilacji.  
  
 **Identyfikator błędu:** BC30930  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Podaj element pomocnika niezbędne.  
  
2.  Upewnij się, że kod jest prawidłowy kod zarządzany.
