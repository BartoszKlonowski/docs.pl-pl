# <a name="unable-to-load-dll-39ltfilenamegt39-lterrorgt"></a>Nie można załadować biblioteki DLL &#39; &lt;filename&gt;&#39;: &lt;błąd&gt;
Nie można załadować biblioteki DLL przywoływane przez ten projekt.  
  
 **Identyfikator błędu:** BC31013  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy plik znajduje się w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
