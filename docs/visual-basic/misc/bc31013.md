# <a name="unable-to-load-dll-filename-error"></a>Nie można załadować biblioteki DLL " \<filename> ": \<error>

Nie można załadować biblioteki DLL, do której odwołuje się ten projekt.  
  
 **Identyfikator błędu:** BC31013  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy plik znajduje się w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
