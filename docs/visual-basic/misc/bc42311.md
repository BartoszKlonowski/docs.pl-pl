# <a name="unable-to-create-xml-documentation-file-39ltnamegt39-ltmessagegt"></a>Nie można utworzyć pliku dokumentacji XML &#39; &lt;nazwa&gt;&#39;: &lt;wiadomości&gt;
Nie można utworzyć pliku dokumentacji XML z powodu błędu.  
  
 **Identyfikator błędu:** BC42311  
  
 Następujące komunikaty może pojawić się z powodu następującego błędu:

-   [Tag komentarza XML "returns" jest niedozwolony we właściwości "WriteOnly"](../../visual-basic/misc/bc42313.md).  
  
-   [Komentarz XML nie można zastosować więcej niż raz w częściowym \<typu >](../../visual-basic/misc/bc42314.md).  
  
-   [Tag komentarza XML "returns" jest niedozwolony w elemencie języka "declare sub"](../../visual-basic/misc/bc42315.md).  
  
-   [Błąd analizowania dokumentacji XML: tag początkowy "\<tag >" nie ma pasującego tagu końcowego](../../visual-basic/misc/bc42316.md).  
  
-   [Parametr typu komentarza XML \<parametru > jest niezgodny z parametrem typu w odpowiedniej \<— słowo kluczowe > instrukcji](../../visual-basic/misc/bc42317.md).
