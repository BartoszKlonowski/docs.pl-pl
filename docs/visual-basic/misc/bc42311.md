# <a name="unable-to-create-xml-documentation-file-name-message"></a>Nie można utworzyć pliku dokumentacji XML " \<name> ":\<message>
Nie można utworzyć pliku dokumentacji XML z powodu błędu.  
  
 **Identyfikator błędu:** BC42311  
  
 Następujące komunikaty mogą pojawić się z powodu tego błędu:  
  
- [Nie można skojarzyć bloku komentarza XML z dowolnym elementem języka, który obsługuje stosowanie komentarzy dokumentacji XML](bc42312.md).  
  
- [Tag komentarza XML "Returns" jest niedozwolony we właściwości "WriteOnly"](bc42313.md).  
  
- [Nie można zastosować komentarza XML więcej niż raz w częściowej \<type> ](bc42314.md).  
  
- [Tag komentarza XML "Returns" jest niedozwolony w elemencie języka "DECLARE Sub"](bc42315.md).  
  
- [Błąd analizowania dokumentacji XML: tag początkowy " \<tag> " nie ma zgodnego tagu końcowego](bc42316.md).  
  
- [Parametr typu komentarza XML nie \<parameter> jest zgodny z parametrem typu w odpowiedniej \<keyword> instrukcji](bc42317.md).  
  
- [Parametr typu komentarza XML musi mieć atrybut "name"](bc42318.md).  
  
- [Wyjątek komentarza XML musi mieć atrybut "cref"](../language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zapoznaj się z podanym komunikatem, aby uzyskać zalecenia.
