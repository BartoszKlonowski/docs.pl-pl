# <a name="unable-to-create-xml-documentation-file-ltnamegt-ltmessagegt"></a>Nie można utworzyć pliku XML dokumentacji "&lt;nazwa&gt;": &lt;wiadomości&gt;
Nie można utworzyć pliku dokumentacji XML z powodu błędu.  
  
 **Identyfikator błędu:** BC42311  
  
 Następujące komunikaty mogą być wyświetlane z powodu następującego błędu:  
  
-   [Blok komentarza XML nie może być skojarzony z wszystkich elementów języka, który obsługuje aplikację komentarze dokumentacji XML](../../visual-basic/misc/bc42312.md).  
  
-   [Tag komentarza XML "returns" jest niedozwolony we właściwości "WriteOnly"](../../visual-basic/misc/bc42313.md).  
  
-   [Komentarz XML nie można zastosować więcej niż raz w częściowym elemencie \<typ >](../../visual-basic/misc/bc42314.md).  
  
-   [Tag komentarza XML "returns" jest niedozwolony w elemencie języka "declare sub"](../../visual-basic/misc/bc42315.md).  
  
-   [Błąd analizowania dokumentacji XML: Tag początkowy "\<tagu >" nie ma zgodnego tagu końcowego](../../visual-basic/misc/bc42316.md).  
  
-   [Parametr typu komentarza XML \<parametr > jest niezgodny z parametrem typu w odpowiedniej \<— słowo kluczowe > instrukcji](../../visual-basic/misc/bc42317.md).  
  
-   [Parametr typu komentarza XML musi mieć atrybut "name"](../../visual-basic/misc/bc42318.md).  
  
-   [Komentarz XML wyjątek musi mieć atrybut "cref"](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zapoznaj się z podanym komunikatem zaleceń.
