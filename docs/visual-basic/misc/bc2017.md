# <a name="could-not-find-library-libraryname"></a>Nie można odnaleźć biblioteki "\<libraryname >"
Nie można odnaleźć określonej biblioteki.  
  
 Ten problem może wystąpić podczas kompilowania z wiersza polecenia oraz przy użyciu odpowiedzi (rsp) plik, który odwołuje się do wielu zestawów  
  
 **Identyfikator błędu:** BC2017  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwę biblioteki, aby upewnić się, że jest on poprawny.
