# <a name="could-not-find-library-39ltlibrarynamegt39"></a>Nie można odnaleźć biblioteki &#39; &lt;libraryname&gt;&#39;
Nie można odnaleźć określonej biblioteki.  
  
 Ten problem może wystąpić podczas kompilowania z wiersza polecenia i za pomocą odpowiedzi (.rsp —) pliku, który odwołuje się do wielu zestawów  
  
 **Identyfikator błędu:** BC2017  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź nazwę biblioteki, aby upewnij się, że jest poprawny.
