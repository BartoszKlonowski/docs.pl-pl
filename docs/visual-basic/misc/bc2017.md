# <a name="could-not-find-library-libraryname"></a>Nie można znaleźć biblioteki "\<LibraryName >"
Nie można znaleźć określonej biblioteki.  
  
 Ten problem może wystąpić podczas kompilowania z wiersza polecenia i przy użyciu pliku odpowiedzi (. RSP), który odwołuje się do wielu zestawów  
  
 **Identyfikator błędu:** BC2017  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź nazwę biblioteki, aby upewnić się, że jest ona poprawna.
