# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>Doprzeznaczona wersja .NET Compact Framework nie obsługuje rozpoznawania przeciążenia ustalanie

Doprzeznaczona wersja .NET Compact Framework nie obsługuje rozwiązywania przeciążenia z późnym wiązaniem.  
  
 **Identyfikator błędu:** BC30764  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Parametry rzutowania typu Object na odpowiedni typ.
