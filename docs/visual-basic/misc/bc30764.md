# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>Docelowa wersja programu .NET Compact Framework nie obsługuje ustalania przeładowań z późnym wiązaniem
Docelowa wersja programu .NET Compact Framework nie obsługuje Rozpoznanie przeciążenia z późnym wiązaniem.  
  
 **Identyfikator błędu:** BC30764  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Rzutowanie parametrów typu obiektu do odpowiedniego typu.
