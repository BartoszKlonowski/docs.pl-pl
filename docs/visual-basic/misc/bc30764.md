# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>Docelowa wersja platformy .NET Compact Framework nie obsługuje przeciążenia późnego
Docelowa wersja platformy .NET Compact Framework nie obsługuje funkcja rozpoznawania przeciążeń z późnym wiązaniem.  
  
 **Identyfikator błędu:** BC30764  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rzutowanie parametrów typu obiektu do odpowiedniego typu.
