# <a name="as-expected"></a>Oczekiwano znaku "As"

Występuje deklaracja, która nie dostarcza `As` klauzuli dla co najmniej jednego zadeklarowanego elementu.  
  
 **Identyfikator błędu:** BC30197  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dostarcz `As` klauzulę jako wymaganą lub Włącz `Option Strict Off` .
