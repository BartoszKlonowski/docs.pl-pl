# <a name="as-expected"></a>Oczekiwano "as"
Występuje deklaracji, która nie dostarcza `As` klauzula dla co najmniej jeden z elementów został zadeklarowany.  
  
 **Identyfikator błędu:** BC30197  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dostarcza `As` klauzuli zgodnie z wymaganiami lub wyłącz `Option Strict Off`.
