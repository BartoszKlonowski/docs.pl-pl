# <a name="39as39-expected"></a>&#39; Jako &#39; Oczekiwano
Występuje deklaracji, który nie dostarcza `As` klauzula dla co najmniej jednego z elementów został zadeklarowany.  
  
 **Identyfikator błędu:** BC30197  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dostarcza `As` klauzuli zgodnie z wymaganiami lub Włącz `Option Strict Off`.
