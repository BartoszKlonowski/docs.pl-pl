# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Elementy członkowskie interfejsu muszą być metodami, właściwościami, zdarzeniami lub definicjami typów
Tylko zagnieżdżone typy, metody, właściwości i zdarzenia mogą być członków interfejsu.  
  
 **Identyfikator błędu:** BC30602  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że element jest metody, właściwości, zdarzenia lub typu zagnieżdżonego, takich jak wyliczenia.
