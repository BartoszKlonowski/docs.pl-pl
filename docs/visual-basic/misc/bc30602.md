# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Elementy członkowskie interfejsu muszą być metodami, właściwościami, zdarzeniami lub definicjami typów

Tylko typy zagnieżdżone, metody, właściwości i zdarzenia mogą być elementami członkowskimi interfejsu.  
  
 **Identyfikator błędu:** BC30602  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że element jest metodą, właściwością, zdarzeniem lub typem zagnieżdżonym, takim jak Wyliczenie.
