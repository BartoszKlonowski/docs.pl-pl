# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Członkowie interfejsu muszą być metody, właściwości, zdarzenia lub definicje typów
Tylko zagnieżdżone typy, metody, właściwości i zdarzenia mogą być członkami interfejsu.  
  
 **Identyfikator błędu:** BC30602  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że element jest metody, właściwości, zdarzenia lub typu zagnieżdżonego, takich jak wyliczenia.
