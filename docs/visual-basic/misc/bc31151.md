# <a name="element-is-missing-an-end-tag"></a>Brak tagu końcowego elementu
Brak tagu końcowego XML lub identyfikator tagu pusty literał elementu XML "/" (na przykład `<tagName />`).  
  
 **Identyfikator błędu:** BC31151  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj tag końcowy dla tagu XML lub dodać tag pusty identyfikator ' /' do pliku XML tagu.
