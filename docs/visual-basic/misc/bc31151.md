# <a name="element-is-missing-an-end-tag"></a>Brak tagu końcowego w elemencie

W literale elementu XML brakuje tagu końcowego XML lub pustego identyfikatora tagu "/" (na przykład `<tagName />` ).  
  
 **Identyfikator błędu:** BC31151  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj tag końcowy do tagu XML lub Dodaj pusty identyfikator tagu "/" do tagu XML.
