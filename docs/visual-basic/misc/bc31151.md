# <a name="element-is-missing-an-end-tag"></a>Element nie ma tagu końcowego
Literał elementu XML nie ma tagu końcowego XML lub identyfikator pustą etykietę "/" (na przykład `<tagName />`).  
  
 **Identyfikator błędu:** BC31151  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj tag końcowy dla tagu XML lub dodać identyfikator pustą etykietę "/" do pliku XML tag.
