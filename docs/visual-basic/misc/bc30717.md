# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje try nie są prawidłowe w oknie bezpośrednim

`Try` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30717  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Try` instrukcje z kodu debugowania.
