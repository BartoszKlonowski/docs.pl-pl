# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Spróbuj &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Try`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30717  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Try` oświadczenia debugowania kodu.
