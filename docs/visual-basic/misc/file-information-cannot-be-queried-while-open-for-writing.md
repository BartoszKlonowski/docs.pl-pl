# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>Nie można zbadać informacji o pliku podczas otwierania do zapisu
Podjęto próbę uzyskania informacji o pliku, który jest aktualnie otwarty do zapisu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamknij plik i spróbuj ponownie wykonać operację.
