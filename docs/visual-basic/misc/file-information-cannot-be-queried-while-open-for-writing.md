# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>Nie można żądać informacji plik otwarty do zapisu
Nastąpiła próba uzyskanie informacji o pliku, który jest aktualnie otwarty do zapisu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zamknij plik i spróbuj ponownie wykonać operację.
