# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>Nie można zastosować elementu "System. Runtime. InteropServices. DllImportAttribute" do elementu "Get" lub "Set"
Ten `DllImportAttribute` atrybut został zastosowany do `Get` `Set` procedury lub właściwości.  
  
 **Identyfikator błędu:** BC31524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `DllImportAttribute` z `Get` `Set` procedur i.
