# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39;Elementu System.Runtime.InteropServices.DllImportAttribute&#39; nie można zastosować do &#39;uzyskać&#39; lub &#39;ustawić&#39;
`DllImportAttribute` Zastosowano atrybut `Get` lub `Set` procedury właściwości.  
  
 **Identyfikator błędu:** BC31524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `DllImportAttribute` z `Get` i `Set` procedury właściwości.
