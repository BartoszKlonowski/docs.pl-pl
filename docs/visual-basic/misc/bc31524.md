# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>Nie można zastosować elementu "System.Runtime.InteropServices.DllImportAttribute" "Get" lub "Set"
`DllImportAttribute` Atrybut została zastosowana do `Get` lub `Set` procedury właściwości.  
  
 **Identyfikator błędu:** BC31524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `DllImportAttribute` z `Get` i `Set` procedury właściwości.
