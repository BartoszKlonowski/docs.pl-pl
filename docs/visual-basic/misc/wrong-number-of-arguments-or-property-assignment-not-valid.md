# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>Nieprawidłową liczbę argumentów lub nieprawidłowe przypisanie właściwości
Podjęto przypisania, który jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy liczba argumentów, które podano zgodna z liczbą wymagane przez element docelowy.  
  
2.  Sprawdź przypisanie właściwości.
