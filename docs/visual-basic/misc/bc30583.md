# <a name="39ltmethodnamegt39-cannot-be-implemented-more-than-once"></a>&#39;&lt;methodname&gt; &#39; nie może być więcej niż raz
Podjęto próbę wykonania metody, która została już dokonana.  
  
 **Identyfikator błędu:** BC30583  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź nazwę metody, aby upewnić się, że ma ona wpisana poprawnie.  
  
2.  Jeśli metoda jest nadmiarowy, Usuń deklarację dodatkowe.
