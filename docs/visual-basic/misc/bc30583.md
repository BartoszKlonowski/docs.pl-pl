# <a name="methodname-cannot-be-implemented-more-than-once"></a>\<methodname>nie można zaimplementować "" więcej niż raz

Podjęto próbę zaimplementowania metody, która została już zaimplementowana.  
  
 **Identyfikator błędu:** BC30583  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź nazwę metody, aby upewnić się, że została wpisana poprawnie.  
  
2. Jeśli metoda jest nadmiarowa, Usuń dodatkową deklarację.
