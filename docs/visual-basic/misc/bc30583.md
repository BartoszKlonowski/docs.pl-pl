# <a name="methodname-cannot-be-implemented-more-than-once"></a>'\<methodname >' nie może mieć więcej niż raz
Podjęto próbę wykonania metody, która została już dokonana.  
  
 **Identyfikator błędu:** BC30583  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź nazwę metody, aby upewnić się, że możesz mieć wpisana poprawnie.  
  
2. Jeśli metoda jest nadmiarowy, Usuń deklarację dodatkowe.
