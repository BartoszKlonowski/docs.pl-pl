# <a name="new-is-not-valid-in-this-context"></a>"New" jest nieprawidłowy w tym kontekście
`New` — Słowo kluczowe występuje, gdy wymagana jest nazwa typu. Kontekst wywołania dla typu wartości, której nie można używać w połączeniu z `New`.  
  
 **Identyfikator błędu:** BC30200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `New` — słowo kluczowe.
