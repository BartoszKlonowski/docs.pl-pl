# <a name="39new39-is-not-valid-in-this-context"></a>&#39;Nowy&#39; jest nieprawidłowy w tym kontekście
`New` — Słowo kluczowe występuje, gdy wymagana jest nazwa typu. Kontekst odwołuje się do typu wartości, które nie mogą być łączone z `New`.  
  
 **Identyfikator błędu:** BC30200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `New` — słowo kluczowe.
