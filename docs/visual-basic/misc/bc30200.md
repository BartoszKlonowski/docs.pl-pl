# <a name="new-is-not-valid-in-this-context"></a>Element "New" jest nieprawidłowy w tym kontekście
`New`Słowo kluczowe występuje, gdy nazwa typu jest wymagana. Kontekst wywołuje dla typu wartości, którego nie można łączyć z `New` .  
  
 **Identyfikator błędu:** BC30200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `New` słowo kluczowe.
