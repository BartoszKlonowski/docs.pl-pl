# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>Instrukcji "Catch" nie są prawidłowe w oknie bezpośrednim
`Catch` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30715  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Catch` instrukcji debugowania kodu.
