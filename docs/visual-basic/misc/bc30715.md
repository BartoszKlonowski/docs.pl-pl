# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Catch" nie są prawidłowe w oknie bezpośrednim
instrukcji `Catch` można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30715  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcje `Catch` z kodu debugowania.
