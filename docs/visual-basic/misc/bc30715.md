# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;CATCH&#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Catch` instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30715  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Catch` oświadczenia debugowania kodu.
