# <a name="unable-to-apply-security-attributes-to-ltnamegt-lterrorgt"></a>Nie można zastosować atrybutów zabezpieczeń do "&lt;nazwa&gt;": &lt;błąd&gt;
Wystąpił błąd podczas stosowania atrybutów zabezpieczeń. Może to być wynikiem niepoprawnie przy użyciu atrybutów zabezpieczeń lub może wskazywać na problem z kompilatora.  
  
 **Identyfikator błędu:** BC31520  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że składnia i argumenty dla atrybutu.  
  
2.  Skompiluj program ponownie, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
3.  Jeśli błąd będzie się powtarzał, Zapisz swoją pracę i uruchom ponownie program Visual Studio.  
  
4.  Jeśli błąd nie ustąpi, zainstaluj ponownie Visual Basic.  
  
5.  Jeśli ten błąd będzie występował po ponownej instalacji, powiadom pomoc techniczna firmy Microsoft.
