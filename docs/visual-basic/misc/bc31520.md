# <a name="unable-to-apply-security-attributes-to-name-error"></a>Nie można zastosować atrybutów zabezpieczeń do " \<name> ": \<error>

Wystąpił błąd podczas stosowania atrybutów zabezpieczeń. Może to być spowodowane nieprawidłowym użyciem atrybutów zabezpieczeń lub może wskazywać na problem z kompilatorem.  
  
 **Identyfikator błędu:** BC31520  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy składnia i argumenty atrybutu.  
  
2. Skompiluj ponownie program, aby zobaczyć, czy błąd wystąpił.  
  
3. Jeśli błąd będzie nadal występował, Zapisz swoją pracę i ponownie uruchom program Visual Studio.  
  
4. Jeśli błąd się powtarza, zainstaluj ponownie Visual Basic.  
  
5. Jeśli błąd będzie się powtarzać po ponownej instalacji, powiadom firmę Microsoft o pomocy technicznej.
