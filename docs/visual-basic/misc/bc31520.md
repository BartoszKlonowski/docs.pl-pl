# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>Nie można zastosować atrybutów zabezpieczeń do &#39; &lt;nazwa&gt;&#39;: &lt;błąd&gt;
Wystąpił błąd podczas stosowania atrybutów zabezpieczeń. Może to być wynikiem niepoprawnie przy użyciu atrybutów zabezpieczeń lub może to wskazywać na problem z kompilatora.  
  
 **Identyfikator błędu:** BC31520  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że składnia i argumentów dla atrybutu.  
  
2.  Skompiluj program, aby zobaczyć, jeśli błąd będzie się powtarzał, ponownie.  
  
3.  Jeśli błąd będzie się powtarzał, należy zapisać pracę i ponownie uruchom program Visual Studio.  
  
4.  Jeśli błąd będzie się powtarzał, zainstaluj ponownie Visual Basic.  
  
5.  Jeśli błąd będzie nadal występować po ponownej instalacji, powiadamia pomocy technicznej firmy Microsoft.
