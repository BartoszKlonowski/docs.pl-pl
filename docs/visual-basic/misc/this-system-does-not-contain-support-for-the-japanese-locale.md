# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Ten system nie obsługuje dla japońskiego ustawień regionalnych
Próbujesz użyć `VbStrConv.Japanese` wyliczeniowego, ale system nie obsługuje japońskich ustawień regionalnych.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Uaktualnij maszynę do obsługi japońskich ustawień regionalnych, lub uruchomić aplikację na komputerze, który obsługuje ustawień regionalnych.
