# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Aby uzyskać&#39; zmienna sterująca pętli już używani przez otaczającą &#39;dla&#39; pętli
Zagnieżdżone `For` wszystkie pętli musi używać zmiennych unikatowy formantu.  
  
 **Identyfikator błędu:** BC30975  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana nazwy zmiennej formantu.
