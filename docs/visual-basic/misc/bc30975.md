# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Zmienna sterująca pętli już w użyciu przez otaczającą pętlę "For" "for"
Zagnieżdżone `For` wszystkich pętli musi używać zmiennych unikatową kontroli.  
  
 **Identyfikator błędu:** BC30975  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana nazwy zmiennej kontrolki.
