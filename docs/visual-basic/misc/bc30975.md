# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Zmienna sterująca pętli "for" jest już używana przez otaczającą pętlę "for"
`For`Pętle zagnieżdżone muszą używać unikatowych zmiennych kontroli.  
  
 **Identyfikator błędu:** BC30975  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień jedną z nazw zmiennych sterujących.
