# <a name="argument-argumentname-cannot-be-an-empty-string-or-nothing"></a>Argument \<argumentname> nie może być pustym ciągiem ani niczego
Argument wymaga wartości innej niż pusty ciąg ("") lub `Nothing` . Może to być spowodowane błędnym obliczaniem wartości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłową wartość dla tego argumentu.
