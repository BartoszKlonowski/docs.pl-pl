# <a name="unable-to-read-fixed-width-fields-because-fieldwidths-is-nothing-or-empty"></a>Nie można odczytać pól o stałej szerokości, ponieważ FieldWidths ma wartość Nothing lub jest pusta
`TextFieldParser`Nie można odczytać pól o stałej szerokości, ponieważ `FieldWidths` Właściwość jest ustawiona na `Nothing` lub jest pusta.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw `FieldWidths` prawidłową wartość.
