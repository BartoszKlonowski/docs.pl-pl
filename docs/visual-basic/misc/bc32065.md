# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>W tej deklaracji nie można określić parametrów typu

Element programowania jest zadeklarowany z listą parametrów typu, ale element programowania nie kwalifikuje się do typu ogólnego.  
  
 Elementy programistyczne, które nie kwalifikują się do ogólnego uwzględnienia właściwości, operatorów, zdarzeń i konstruktorów. Deklarowanie któregokolwiek z tych elementów za pomocą listy parametrów typu powoduje błąd.  
  
 **Identyfikator błędu:** BC32065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Of` słowo kluczowe i parametry typu z deklaracji.
