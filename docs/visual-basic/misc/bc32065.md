# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>Nie można określić parametrów typu w tej deklaracji
Element programowania zadeklarowano z listą parametrów typu, ale elementu programistycznego nie może być typem ogólnym.  
  
 Elementy programowania, które nie są objęte jest ogólny zawierają właściwości, operatory, zdarzeń i konstruktorów. Deklarowanie te elementy z typem parametru listy wyników w tym błędzie.  
  
 **Identyfikator błędu:** BC32065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Of` — słowo kluczowe i parametrów typu w deklaracji.
