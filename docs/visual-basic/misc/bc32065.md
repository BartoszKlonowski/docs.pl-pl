# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>Nie można określić parametrów typu dla tej deklaracji
Element programowania jest zadeklarowana za pomocą lista parametrów typu, ale elementu programistycznego nie kwalifikuje się do typu ogólnego.  
  
 Elementy programowania, które nie są uprawnione do być ogólna zawierają właściwości, operatory, zdarzenia i konstruktorów. Deklarowanie jednego z tych elementów z typem parametru listy wyników, w wyniku tego błędu.  
  
 **Identyfikator błędu:** BC32065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Of` słów kluczowych i parametrów typu w deklaracji.
