# <a name="module-39ltmodulenamegt39-cannot-be-used-as-a-type"></a>Moduł &#39; &lt;modulename&gt;&#39; nie może być używany jako typ
Podjęto próbę użycia jako typ modułu. Moduł nie może pełnić z wyjątkiem `Get` typu.  
  
 **Identyfikator błędu:** BC30371  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp element, który może służyć jako prawidłowego typu modułu.
