# <a name="module-ltmodulenamegt-cannot-be-used-as-a-type"></a>Moduł "&lt;modulename&gt;" nie może być używany jako typ
Podjęto próbę używaj modułu TPM jako typu. Moduł nie może pełnić z wyjątkiem `Get` typu.  
  
 **Identyfikator błędu:** BC30371  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp moduł coś, co może służyć jako prawidłowego typu.
