# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>element "\<MemberName >" jest już zadeklarowany przez element "\<procedurname >", który został wygenerowany dla tego \<ContainerName >

Zadeklarowano składową, która ma taką samą nazwę jak element członkowski, który jest zadeklarowany przez środowisko uruchomieniowe języka wspólnego.

**Identyfikator błędu:** BC30733

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Zmień nazwę elementu członkowskiego, aby usunąć konflikt.
