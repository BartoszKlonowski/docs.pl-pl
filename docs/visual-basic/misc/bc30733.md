# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39;&lt;membername&gt; &#39; jest już zadeklarowany przez &#39; &lt;nazwaprocedury&gt;&#39;, który został wygenerowany dla tego &lt;containername&gt;
Element członkowski został zadeklarowany, który ma taką samą nazwę jak element członkowski, który jest deklarowany przez środowisko uruchomieniowe języka comom.  
  
 **Identyfikator błędu:** BC30733  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę elementu członkowskiego, aby usunąć konflikt.
