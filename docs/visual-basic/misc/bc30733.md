# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>"&lt;membername&gt;"jest już zadeklarowana przez"&lt;nazwaprocedury&gt;", który został wygenerowany dla tego &lt;containername&gt;
Członek został zadeklarowany, który ma taką samą nazwę jak element członkowski, który jest zadeklarowana przez środowisko uruchomieniowe języka comom.  
  
 **Identyfikator błędu:** BC30733  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę członka do usunięcia konfliktu.
