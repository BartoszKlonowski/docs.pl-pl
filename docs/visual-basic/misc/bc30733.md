# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<membername >' jest już zadeklarowana przez '\<nazwaprocedury >', który został wygenerowany dla tego \<containername >

Członek został zadeklarowany, który ma taką samą nazwę jak element członkowski, który jest deklarowana przez środowisko uruchomieniowe języka wspólnego.

**Identyfikator błędu:** BC30733

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

- Zmień nazwę członka do usunięcia konfliktu.
