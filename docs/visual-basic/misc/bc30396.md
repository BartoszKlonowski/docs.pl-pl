# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>element " \<specifier> " nie jest prawidłowy w deklaracji wyliczenia
Użyto specyfikatora, który nie jest prawidłowy w `Enum` deklaracji.  
  
 **Identyfikator błędu:** BC30396  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń specyfikator.
