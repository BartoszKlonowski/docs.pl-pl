# <a name="ltspecifiergt-is-not-valid-on-an-enum-declaration"></a>"&lt;specyfikator&gt;" nie jest prawidłowy w deklaracji wyliczenia
Wykorzystano specyfikator, który nie jest prawidłowy w `Enum` deklaracji.  
  
 **Identyfikator błędu:** BC30396  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń specyfikator.
