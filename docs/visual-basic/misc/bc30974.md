# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Instrukcje komentarzy nie można obliczyć wartości (nie w spisie treści)
Instrukcja komentarz został wprowadzony w oknie bezpośrednim podczas sesji debugowania. Instrukcje komentarzy w oknie bezpośrednim nie są sprawdzane.  
  
 **Identyfikator błędu:** BC30974
