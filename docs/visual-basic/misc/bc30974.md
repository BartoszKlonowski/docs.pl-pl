# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Nie można obliczyć instrukcji komentarzy (nie w spisie treści)

Instrukcja Comment została wprowadzona w oknie bezpośrednim podczas sesji debugowania. Instrukcje komentarza w oknie bezpośrednim nie są oceniane.  
  
 **Identyfikator błędu:** BC30974
