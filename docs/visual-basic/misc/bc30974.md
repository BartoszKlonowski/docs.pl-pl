# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Nie można obliczyć instrukcje komentarzy (nie w spisie treści)
Instrukcja komentarz został wprowadzony w oknie bezpośrednim podczas sesji debugowania. Instrukcje komentarzy w oknie bezpośrednim nie są uwzględniane.  
  
 **Identyfikator błędu:** BC30974
