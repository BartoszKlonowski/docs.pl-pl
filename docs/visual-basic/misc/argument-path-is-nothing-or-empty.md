# <a name="argument-path-is-nothing-or-empty"></a>Argument "Path" nic nie lub pusty
Nazwa ścieżki jest nieprawidłowy, ponieważ jest on inicjowany `Nothing` lub jest pusty. Ścieżka musi mieć na określony typ danych i prawidłową nazwę.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj nazwę prawidłową ścieżkę.
