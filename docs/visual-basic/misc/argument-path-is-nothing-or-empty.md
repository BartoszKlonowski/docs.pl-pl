# <a name="argument-path-is-nothing-or-empty"></a>Argument "Path" ma wartość Nothing lub jest pusty
Nazwa ścieżki jest nieprawidłowa, ponieważ została zainicjowana `Nothing` lub jest pusta. Ścieżka musi mieć określony typ danych i prawidłową nazwę.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj prawidłową nazwę ścieżki.
