# <a name="named-argument-argumentname-specified-multiple-times"></a>Nazwany argument " \<argumentname> " został określony wiele razy

Określono nazwany argument więcej niż raz.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wszystkie oprócz jednej specyfikacji.
