# <a name="cannot-determine-array-type-because-it-is-nothing"></a>Nie można określić typu tablicy, ponieważ nie ma żadnego elementu

Nie można ustalić typu tablicy, ponieważ została ona ustawiona na wartość `Nothing` .  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Podaj wartość inną niż `Nothing` Tablica.
