# <a name="no-mouse-wheel-is-present"></a>Brak kółka myszy

`My.Computer.Mouse.WheelScrollLines`Właściwość została wywołana, ale mysz nie ma kółka przewijania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź `My.Computer.Mouse.WheelExists` Właściwość, aby sprawdzić, czy mysz ma kółko przewijania przed wywołaniem `My.Computer.Mouse.WheelScrollLines` właściwości.  
  
     -lub-  
  
- Zainstaluj mysz z kółkiem przewijania na komputerze.
