# <a name="no-mouse-wheel-is-present"></a>Nie kółka myszy
`My.Computer.Mouse.WheelScrollLines` Właściwość została wywołana, ale myszą ma nie kółka przewijania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź `My.Computer.Mouse.WheelExists` właściwość, aby sprawdzić, czy przycisk myszy kółka przewijania, przed wywołaniem `My.Computer.Mouse.WheelScrollLines` właściwości.  
  
     —lub—  
  
-   Instalowanie myszy przy użyciu kółka przewijania na komputerze.
