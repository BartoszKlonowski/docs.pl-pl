# <a name="no-mouse-wheel-is-present"></a>Nie ma kółka myszy obecne
`My.Computer.Mouse.WheelScrollLines` Właściwość została wywołana, ale nie kółko przewijania myszy ma myszy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź `My.Computer.Mouse.WheelExists` właściwość, aby sprawdzić, czy wskaźnik myszy ma kółka przewijania przed wywołaniem `My.Computer.Mouse.WheelScrollLines` właściwości.  
  
     —lub—  
  
-   Zainstaluj myszy w kółko przewijania na komputerze.
