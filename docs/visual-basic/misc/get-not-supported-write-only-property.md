# <a name="get-not-supported-write-only-property"></a>Pobieranie nie jest obsługiwane (właściwość tylko do zapisu)

Nie można odczytać właściwości, ponieważ jest tylko do zapisu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź Właściwość i Określ warunki, które mogą być ustawione.  
  
2. Usuń odwołanie do właściwości.
