# <a name="get-not-supported-write-only-property"></a>Pobierz nie jest obsługiwany (właściwość tylko do zapisu)
Nie można odczytać właściwości, ponieważ jest tylko do zapisu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź właściwość, aby ustalić, jakie warunkach można ustawić.  
  
2.  Usuń odwołanie do właściwości.
