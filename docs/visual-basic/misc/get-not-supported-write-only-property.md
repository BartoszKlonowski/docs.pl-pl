# <a name="get-not-supported-write-only-property"></a>Pobieranie nie jest obsługiwany (tylko do zapisu właściwości)
Nie można odczytać właściwości, ponieważ jest tylko do zapisu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź właściwości, a następnie określić pod jakimi warunkami może zostać ustawiony.  
  
2.  Usuń odwołanie do właściwości.
