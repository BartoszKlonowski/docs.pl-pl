# <a name="no-mouse-is-present"></a>Nie ma myszy
Jedna z właściwości obiektu `My.Computer.Mouse` obiektu została wywołana, ale komputer nie ma myszy ani port myszy zainstalowany.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Try...Catch` bloku wokół wywołanie do właściwości `My.Computer.Mouse` obiektu.  
  
     — lub —  
  
-   Zainstaluj myszy na komputerze.
