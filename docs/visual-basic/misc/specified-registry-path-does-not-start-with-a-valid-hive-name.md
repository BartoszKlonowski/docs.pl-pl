# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Określona ścieżka rejestru nie rozpoczyna się od prawidłowej nazwy gałęzi
Określona ścieżka rejestru nie zaczyna się od prawidłowej nazwy gałęzi. Prawidłowe nazwy gałęzi to:  
  
- `HKEY_CLASSES_ROOT`  
  
- `HKEY_CURRENT_CONFIG`  
  
- `HKEY_CURRENT_USER`  
  
- `HKEY_DYNDATA`  
  
- `HKEY_LOCAL_MACHINE`  
  
- `HKEY_PERFORMANCE_DATA`  
  
- `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy podajesz prawidłową nazwę gałęzi.
