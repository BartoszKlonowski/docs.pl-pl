# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Ścieżka rejestru określonego nie rozpoczyna się od nazwy prawidłową gałąź
Ścieżka rejestru określonego nie zaczyna się od nazwy gałęzi prawidłowe. Nieprawidłowa gałąź nazwy są:  
  
- `HKEY_CLASSES_ROOT`  
  
- `HKEY_CURRENT_CONFIG`  
  
- `HKEY_CURRENT_USER`  
  
- `HKEY_DYNDATA`  
  
- `HKEY_LOCAL_MACHINE`  
  
- `HKEY_PERFORMANCE_DATA`  
  
- `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy jest podanie nazwy gałęzi prawidłowe.
