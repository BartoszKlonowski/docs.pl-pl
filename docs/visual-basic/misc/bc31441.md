# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="710fa-102">Metoda " \<methodname1> " musi być zadeklarowana jako "Private", aby można było zaimplementować metodę częściową " \<methodname2> "</span><span class="sxs-lookup"><span data-stu-id="710fa-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>

<span data-ttu-id="710fa-103">Implementacja metody częściowej musi być zadeklarowana `Private` .</span><span class="sxs-lookup"><span data-stu-id="710fa-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="710fa-104">Na przykład poniższy kod powoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="710fa-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="710fa-105">**Identyfikator błędu:** BC31441</span><span class="sxs-lookup"><span data-stu-id="710fa-105">**Error ID:** BC31441</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="710fa-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="710fa-106">To correct this error</span></span>

```vb  
End Sub  
```
