# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a>Metoda\<'methodname1>' musi być zadeklarowana jako "Prywatna"\<w celu wdrożenia metody częściowej ' methodname2>'
Należy zadeklarować `Private`wdrożenie metody częściowej. Na przykład następujący kod powoduje ten błąd.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **Identyfikator błędu:** Bc31441  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Użyj modyfikatora `Private` dostępu w implementacji metody częściowej, jak pokazano w poniższym przykładzie.  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
