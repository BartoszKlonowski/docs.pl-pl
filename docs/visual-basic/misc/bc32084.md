 <span data-ttu-id="db069-104">`New` Ograniczenia oznacza, że argumentu typu dostarczonego do parametru typu musi ujawniać konstruktor bez parametrów dostępnych do kodu, który tworzy obiekty od niego.</span><span class="sxs-lookup"><span data-stu-id="db069-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="db069-105">Wszystkie typy wartości mieć konstruktory bez parametrów, ale nie wszystkie typy referencyjne.</span><span class="sxs-lookup"><span data-stu-id="db069-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="db069-106">W związku z tym `Structure` spełnia ograniczenia `New` ograniczenia, ale `Class` ograniczenia lub nazwę klasy lub interfejsu, nie ma.</span><span class="sxs-lookup"><span data-stu-id="db069-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>
