# <a name="type-parameter-typeparametername1-must-have-either-a-new-constraint-or-a-structure-constraint-to-satisfy-the-new-constraint-for-type-parameter-typeparametername2"></a><span data-ttu-id="e5b4d-102">Parametr typu " \<typeparametername1> " musi mieć ograniczenie "New" lub "Structure", aby spełniać ograniczenie "New" dla parametru typu " \<typeparametername2> "</span><span class="sxs-lookup"><span data-stu-id="e5b4d-102">Type parameter '\<typeparametername1>' must have either a 'New' constraint or a 'Structure' constraint to satisfy the 'New' constraint for type parameter '\<typeparametername2>'</span></span>

<span data-ttu-id="e5b4d-103">Instrukcja konstruuje typ ogólny, przekazując parametr typu, który nie jest ograniczony do spełnienia `New` ograniczenia.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>

<span data-ttu-id="e5b4d-104">`New`Ograniczenie oznacza, że argument typu dostarczony do tego parametru typu musi ujawniać konstruktora bez parametrów dostępnego dla kodu, który tworzy obiekty z.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="e5b4d-105">Wszystkie typy wartości mają konstruktory bez parametrów, ale nie wszystkie typy odwołań.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="e5b4d-106">W związku z tym `Structure` ograniczenie spełnia `New` warunki ograniczające, ale `Class` ograniczenie lub nazwa klasy lub interfejsu, nie.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>

<span data-ttu-id="e5b4d-107">Poniższe instrukcje mogą generować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-107">The following statements can generate this error.</span></span>

```vb
Public Class c1(Of t As New)
End Class
Public Class c2(Of u)
    Public testObject As New c1(Of u)
End Class
```

<span data-ttu-id="e5b4d-108">Gdy Klasa `c2` tworzy obiekt z `c1` , parametr typu nie `u` spełnia `New` ograniczenia parametru typu `t` .</span><span class="sxs-lookup"><span data-stu-id="e5b4d-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>

<span data-ttu-id="e5b4d-109">**Identyfikator błędu:** BC32084</span><span class="sxs-lookup"><span data-stu-id="e5b4d-109">**Error ID:** BC32084</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="e5b4d-110">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="e5b4d-110">To correct this error</span></span>

- <span data-ttu-id="e5b4d-111">Jeśli parametr typu, który ma zostać przesłany do typu ogólnego, może `Structure` spełnić `New` ograniczenie lub, a następnie dodać odpowiednie ograniczenie do jego definicji.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-111">If the type parameter to be passed to the generic type can satisfy the `Structure` or `New` constraint, then add the appropriate constraint to its definition.</span></span>

  ```vb
  Public Class c2(Of u As Structure)
  ```

- <span data-ttu-id="e5b4d-112">Jeśli parametr typu nie może spełniać `Structure` ograniczenia lub `New` , nie przekazuj go do typu ogólnego.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-112">If the type parameter cannot satisfy the `Structure` or `New` constraint, then do not pass it to the generic type.</span></span> <span data-ttu-id="e5b4d-113">Należy przekazać coś innego jako argumentu typu.</span><span class="sxs-lookup"><span data-stu-id="e5b4d-113">You must pass something else as the type argument.</span></span>
