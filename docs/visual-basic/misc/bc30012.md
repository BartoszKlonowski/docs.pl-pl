# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39; bloku musi być zakończona odpowiadającą jej instrukcją &#39;#End If&#39;
`#If` jest warunkowej dyrektywie kompilacji. `#If` Blok nie zostanie zakończony przez `#End If` dyrektywy.  
  
 **Identyfikator błędu:** BC30012  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `#End If` dyrektywy do końca bloku kompilacji warunkowej.
