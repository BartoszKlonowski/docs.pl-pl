# <a name="if-block-must-end-with-a-matching-end-if"></a>Blok "#If" musi być zakończony odpowiadającym mu elementem "#End If"
`#If`jest dyrektywy kompilacji warunkowej. `#If`Blok nie został zakończony przez `#End If` dyrektywę.  
  
 **Identyfikator błędu:** BC30012  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `#End If` dyrektywę na końcu bloku kompilacji warunkowej.
