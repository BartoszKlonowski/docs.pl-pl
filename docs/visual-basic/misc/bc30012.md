# <a name="if-block-must-end-with-a-matching-end-if"></a>Blok "#If" musi kończyć się z instrukcją "#End If"
`#If` jest dyrektywą warunkowa. `#If` Przez nie został przerwany blok `#End If` dyrektywy.  
  
 **Identyfikator błędu:** BC30012  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `#End If` dyrektywy do końca bloku kompilacji warunkowej.
