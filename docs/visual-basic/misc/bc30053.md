# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="9dbaa-102">Nie można zadeklarować tablic przy użyciu "New"</span><span class="sxs-lookup"><span data-stu-id="9dbaa-102">Arrays cannot be declared with 'New'</span></span>

<span data-ttu-id="9dbaa-103">`New` Słowo kluczowe może występować tylko w części inicjalizacji deklaracji tablicy.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="9dbaa-104">Ta metoda `New` musi znajdować się po prawej stronie znaku równości (`=`), aby można było utworzyć nowy typ tablicy, który ma zostać przypisany do zmiennej tablicowej.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>

<span data-ttu-id="9dbaa-105">Skrót do inicjowania klasy nie jest dostępny dla tablic.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="9dbaa-106">Poniższe dwa wiersze kodu są prawidłowe i są równoważne, ponieważ inicjują obiekt z klasy.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>

```vb
Dim formA as Form = New Form
Dim formA as New Form
```

<span data-ttu-id="9dbaa-107">Jednak inicjowanie tablicy nie może używać tego samego skrótu co Inicjalizacja klasy.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>

<span data-ttu-id="9dbaa-108">Należy zauważyć, `New` że klauzula tablicy musi zawierać zarówno nawiasy, `()`, jak i nawiasy klamrowe, `{}`.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="9dbaa-109">Nawiasy określają, że nowy typ jest tablicą, a nawiasy te dostarczają wartości inicjalizacji.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="9dbaa-110">Kompilator wymaga nawiasów klamrowych, nawet jeśli są puste, czyli nawet jeśli nie zainicjowano żadnej wartości tablicy.</span><span class="sxs-lookup"><span data-stu-id="9dbaa-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>

<span data-ttu-id="9dbaa-111">**Identyfikator błędu:** BC30053</span><span class="sxs-lookup"><span data-stu-id="9dbaa-111">**Error ID:** BC30053</span></span>
