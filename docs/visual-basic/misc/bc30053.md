# <a name="arrays-cannot-be-declared-with-39new39"></a><span data-ttu-id="74f69-102">Nie można deklarować tablic przy użyciu &#39;nowy&#39;</span><span class="sxs-lookup"><span data-stu-id="74f69-102">Arrays cannot be declared with &#39;New&#39;</span></span>
<span data-ttu-id="74f69-103">`New` — Słowo kluczowe może występować tylko w części inicjowania deklarację tablicy.</span><span class="sxs-lookup"><span data-stu-id="74f69-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="74f69-104">Oznacza to, że `New` musi być po prawej stronie znaku równości (`=`), można utworzyć typu tablicy ma być przypisany do zmiennej tablicy.</span><span class="sxs-lookup"><span data-stu-id="74f69-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="74f69-105">Skrót do inicjowania klasy nie jest dostępna dla tablic.</span><span class="sxs-lookup"><span data-stu-id="74f69-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="74f69-106">Następujące wiersze kodu dwóch są prawidłowe i są równoważne, ponieważ inicjowania obiekt z klasy.</span><span class="sxs-lookup"><span data-stu-id="74f69-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="74f69-107">Jednak inicjowanie tablicy nie może być ten sam skrót inicjowania klasy.</span><span class="sxs-lookup"><span data-stu-id="74f69-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="74f69-108">Należy pamiętać, że `New` klauzula dla tablicy musi zawierać zarówno nawiasów `()`oraz nawiasy klamrowe, `{}`.</span><span class="sxs-lookup"><span data-stu-id="74f69-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="74f69-109">Nawiasy określić, że nowy typ jest tablicą, i podaj wartości inicjowania w nawiasy klamrowe.</span><span class="sxs-lookup"><span data-stu-id="74f69-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="74f69-110">Kompilator wymaga nawiasy klamrowe, nawet jeśli są puste, oznacza to, nawet jeśli nie są inicjowanie tablicy wartości.</span><span class="sxs-lookup"><span data-stu-id="74f69-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="74f69-111">**Identyfikator błędu:** BC30053</span><span class="sxs-lookup"><span data-stu-id="74f69-111">**Error ID:** BC30053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="74f69-112">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="74f69-112">To correct this error</span></span>  
  
-   <span data-ttu-id="74f69-113">Zamień instrukcję, takich jak `Dim myDates() As New Date` z instrukcją, takich jak `Dim myDates() As Date = New Date() {}`.</span><span class="sxs-lookup"><span data-stu-id="74f69-113">Replace a statement such as `Dim myDates() As New Date` with a statement such as `Dim myDates() As Date = New Date() {}`.</span></span>
