# <a name="interface-can-inherit-only-from-another-interface"></a>Interfejs może dziedziczyć tylko z innego interfejsu
Podjęto próbę dziedziczyć coś, co nie jest interfejsem.  
  
 **Identyfikator błędu:** BC30354  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że do interfejsu podstawowego lub interfejsów, z których próbujesz dziedziczą są faktycznie interfejsów.
