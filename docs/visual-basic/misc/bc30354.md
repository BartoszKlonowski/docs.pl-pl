# <a name="interface-can-inherit-only-from-another-interface"></a>Interfejs może dziedziczyć tylko z innego interfejsu
Podjęto próbę dziedziczyć z elementu, który nie jest interfejsem.  
  
 **Identyfikator błędu:** BC30354  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że interfejs podstawowy lub interfejsów, spośród których próbujesz dziedziczą są faktycznie interfejsów.
