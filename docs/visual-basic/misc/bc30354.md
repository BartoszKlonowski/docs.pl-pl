# <a name="interface-can-inherit-only-from-another-interface"></a>Interfejs może dziedziczyć tylko po innym interfejsie

Podjęto próbę dziedziczenia z elementu, który nie jest interfejsem.  
  
 **Identyfikator błędu:** BC30354  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że interfejs podstawowy lub interfejsy, z których próbujesz dziedziczyć, są w rzeczywistości interfejsy.
