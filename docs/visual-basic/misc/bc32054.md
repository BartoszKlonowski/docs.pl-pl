# <a name="39ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>&#39; &lt;membername&gt; ma taką samą nazwę jak parametr typu
Element członkowski ogólnej klasy lub struktury ma taką samą nazwę jak parametr typu tej klasy lub struktury.  
  
 Co zmienna, stała wyliczenia, procedury, właściwości, zdarzeń i delegata musi mieć nazwę różne od co parametr typu.  
  
 **Identyfikator błędu:** BC32054  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę elementu członkowskiego klasy lub struktury, aby była inna niż nazwa każdego parametru typu.
