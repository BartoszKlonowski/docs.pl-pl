# <a name="membername-has-the-same-name-as-a-type-parameter"></a>" \<membername> ma taką samą nazwę jak parametr typu

Składowa klasy generycznej lub struktury ma taką samą nazwę jak parametr typu tej klasy lub struktury.  
  
 Każda zmienna, stała, Wyliczenie, procedura, właściwość, zdarzenie i delegat muszą mieć nazwy różne od każdego parametru typu.  
  
 **Identyfikator błędu:** BC32054  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę elementu członkowskiego klasy lub struktury, tak aby była inna niż każda nazwa parametru typu.
