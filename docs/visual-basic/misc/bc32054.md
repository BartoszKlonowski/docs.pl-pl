# <a name="membername-has-the-same-name-as-a-type-parameter"></a>"\<membername > ma taką samą nazwę jak parametr typu
Element członkowski ogólnej klasy lub struktury ma taką samą nazwę jak parametr typu tej klasy lub struktury.  
  
 Każdej zmiennej, — stała, wyliczenia, procedury, właściwości, zdarzeń i delegata musi mieć nazwę różne od co parametr typu.  
  
 **Identyfikator błędu:** BC32054  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę elementu członkowskiego klasy lub struktury, tak, aby jest inny niż nazwa każdego parametru typu.
