# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Długość argumentu &#39; &lt;argumentname&gt;&#39; musi być większa od zera.
Argument ma wartość, która jest równa lub mniejsza od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj argument, który jest większy niż zero.
