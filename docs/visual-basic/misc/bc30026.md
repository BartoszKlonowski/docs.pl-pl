# <a name="end-sub-expected"></a>Oczekiwano "End Sub"
`End` Instrukcja musi odpowiadać blok zostaje zakończone. A `Sub` definicji procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC30026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Sub` instrukcji na końcu procedury.
