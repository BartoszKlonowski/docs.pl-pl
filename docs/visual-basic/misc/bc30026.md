# <a name="end-sub-expected"></a>Oczekiwano instrukcji "End Sub"
`End`Instrukcja musi być zgodna z zamykanym blokiem. `Sub`Definicja procedury nie została prawidłowo zakończona.  
  
 **Identyfikator błędu:** BC30026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `End Sub` instrukcję do końca procedury.
