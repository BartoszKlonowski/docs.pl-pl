# <a name="39end-sub39-expected"></a>&#39; End Sub &#39; Oczekiwano
`End` Instrukcja musi odpowiadać bloku zostaje zakończone. A `Sub` Definicja procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC30026  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Sub` instrukcji na końcu procedury.
