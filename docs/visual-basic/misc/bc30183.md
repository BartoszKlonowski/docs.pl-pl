# <a name="keyword-is-not-valid-as-an-identifier"></a>Słowo kluczowe nie jest prawidłowe jako identyfikator
Słowo kluczowe występuje, gdy wymagana jest nazwa elementu. Kontekst odwołuje się do elementu, takiego jak zmienna lub procedury.  
  
 **Identyfikator błędu:** BC30183  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp kluczowego prawidłową nazwę elementu.
