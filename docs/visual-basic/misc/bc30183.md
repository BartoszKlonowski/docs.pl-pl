# <a name="keyword-is-not-valid-as-an-identifier"></a>Słowo kluczowe nie jest prawidłowe jako identyfikator
Słowo kluczowe występuje, gdy wymagana jest nazwa elementu. Kontekst wywołuje element, taki jak zmienna lub procedura.  
  
 **Identyfikator błędu:** BC30183  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zastąp słowo kluczowe prawidłową nazwą elementu.
