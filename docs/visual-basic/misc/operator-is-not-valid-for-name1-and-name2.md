# <a name="operator-is-not-valid-for-name1-and-name2"></a>Operator jest nieprawidłowy dla elementu " \<name1> " i " \<name2> "
Operator nie jest odpowiedni dla typów, w których są obsługiwane.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że operator jest odpowiedni dla typów, w których są obsługiwane.
