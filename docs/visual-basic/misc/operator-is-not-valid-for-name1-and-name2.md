# <a name="operator-is-not-valid-for-ltname1gt-and-ltname2gt"></a>Operator nie jest prawidłowy dla "&lt;Nazwa1&gt;"i"&lt;Nazwa2&gt;"
Operator nie jest odpowiedni dla typów, w której są wykonywane działania na.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że operator jest odpowiedni dla typów, w której są wykonywane działania na.
