# <a name="internal-error-visual-basic"></a>Błąd wewnętrzny (Visual Basic)
Usterka wewnętrzny wystąpił w języku Visual Basic.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że ten błąd nie został wygenerowany przez `Error` instrukcji lub `Raise` metody. Jeśli tak jest, skontaktuj się z pomocą techniczną firmy Microsoft, aby zgłosić warunków, w których pojawił się komunikat.
