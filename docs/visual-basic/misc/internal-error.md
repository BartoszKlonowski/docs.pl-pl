# <a name="internal-error-visual-basic"></a>Błąd wewnętrzny (Visual Basic)
Wystąpił wewnętrzny błąd w Visual Basic.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że ten błąd nie został wygenerowany przez instrukcję `Error` lub metodę `Raise`. O ile nie była to możliwe, skontaktuj się z pomocą techniczną firmy Microsoft, aby zgłosić warunki, w których pojawił się komunikat.
