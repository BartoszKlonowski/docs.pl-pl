# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>Argument &#39; &lt;argumentname&gt;&#39; musi być większa niż 0 lub równy -1
Wartość argumentu jest mniejsza od 0, ale nie ma wartości -1.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że wartość argumentu znajduje się do prawidłowego zakresu.  
  
2.  Jeśli argument jest dostarczana przez obliczeń, sprawdź wartości procesów obliczeń.
