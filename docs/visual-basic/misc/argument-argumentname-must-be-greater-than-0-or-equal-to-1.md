# <a name="argument-ltargumentnamegt-must-be-greater-than-0-or-equal-to--1"></a>Argument "&lt;argumentname&gt;" musi być większa niż 0 lub równy-1
Wartość argumentu jest mniejsza niż 0, ale nie ma wartości -1.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że wartość argumentu mieści się w prawidłowym zakresem.  
  
2.  Jeśli argument jest dostarczana przez obliczenie, sprawdź wartości procesów obliczeń.
