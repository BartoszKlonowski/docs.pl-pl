# <a name="statement-is-not-valid-in-an-interface"></a>Instrukcja nie jest prawidłowa w interfejsie
Napotkano nieprawidłową instrukcję w interfejsie.  
  
 **Identyfikator błędu:** BC31041  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Należy sprawdzić, czy instrukcje w interfejsie definiują elementy, ale nie zapewniają implementacji.
