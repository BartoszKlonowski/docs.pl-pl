# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>We/Wy pliku z typu &#39; &lt;typename&gt;&#39; jest nieprawidłowa
Podjęto próbę zapisu w pliku nieobsługiwanego typu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź strukturę, aby określić, gdzie występuje nieobsługiwany typ, a następnie zmień go do prawidłowego typu.
