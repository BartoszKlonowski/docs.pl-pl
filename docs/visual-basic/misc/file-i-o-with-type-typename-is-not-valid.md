# <a name="file-io-with-type-lttypenamegt-is-not-valid"></a>Plik we/wy o typie "&lt;typename&gt;" jest nieprawidłowa
Podjęto próbę zapisu nieobsługiwany typ pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź strukturę, aby określić, gdzie występuje nieobsługiwany typ, a następnie zmienić je do prawidłowego typu.
