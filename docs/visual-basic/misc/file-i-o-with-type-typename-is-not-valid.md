# <a name="file-io-with-type-typename-is-not-valid"></a>We/wy pliku z typem " \<typename> " jest nieprawidłowy

Podjęto próbę zapisania nieobsługiwanego typu w pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź strukturę, aby określić, gdzie wystąpi nieobsługiwany typ, a następnie zmień go na prawidłowy typ.
