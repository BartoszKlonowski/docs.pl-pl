# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>Plik we/wy o typie &#39; &lt;typename&gt; &#39; jest nieprawidłowy
Podjęto próbę zapisu w pliku nieobsługiwanego typu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź strukturę, aby określić, gdzie występuje nieobsługiwany typ, a następnie zmień go do prawidłowego typu.
