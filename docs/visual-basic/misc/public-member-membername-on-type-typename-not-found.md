# <a name="public-member-membername-on-type-typename-not-found"></a>Nie znaleziono publicznej składowej " \<membername> " w typie " \<typename> "

Nie można znaleźć elementu członkowskiego, do którego próbujesz uzyskać dostęp. Ten błąd może wystąpić podczas uzyskiwania dostępu do niepublicznych członków obiektów z późnym wiązaniem.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Jeśli element członkowski istnieje, upewnij się, że jest zadeklarowany jako `Public` .
