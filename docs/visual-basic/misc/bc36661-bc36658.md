# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="b6f88-102">Typów danych parametrów typu w metodzie rozszerzenia '\<methodname >' zdefiniowane w 'typename' nie można wywnioskować na podstawie tych argumentów, ponieważ nie są konwertowane na ten sam typ.</span><span class="sxs-lookup"><span data-stu-id="b6f88-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="b6f88-103">Typów danych parametrów typu w metodzie rozszerzenia '\<methodname >' zdefiniowane w 'typename' nie można wywnioskować na podstawie tych argumentów, ponieważ nie są konwertowane do tego samego typu.</span><span class="sxs-lookup"><span data-stu-id="b6f88-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="b6f88-104">Jawne określenie typów danych może naprawić ten błąd.</span><span class="sxs-lookup"><span data-stu-id="b6f88-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="b6f88-105">Podjęto próbę użycia wnioskowania o typie w celu określenia typu danych (lub typów) parametru typu (lub parametrów) podczas oceny wywołania do ogólnej metody rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="b6f88-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="b6f88-106">Kompilator nie może znaleźć typu danych, który spełnia ograniczenia wszystkich argumentów.</span><span class="sxs-lookup"><span data-stu-id="b6f88-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="b6f88-107">W związku z tym ten błąd został zgłoszony.</span><span class="sxs-lookup"><span data-stu-id="b6f88-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="b6f88-108">Gdy określenie argumentów nie jest opcją (na przykład w przypadku operatorów zapytań w wyrażeniach zapytań), komunikat o błędzie jest wyświetlany bez drugiego zdania.</span><span class="sxs-lookup"><span data-stu-id="b6f88-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="b6f88-109">Poniższy kod ilustruje błąd.</span><span class="sxs-lookup"><span data-stu-id="b6f88-109">The following code demonstrates the error.</span></span>

```vb

    Sub Main()

        Dim c1 As New Class1

        '' Not valid. Integer and Date do not convert to the same type.
        'c1.targetMethod(19, #3/4/2007#)

    End Sub

    <System.Runtime.CompilerServices.Extension()> _
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```
