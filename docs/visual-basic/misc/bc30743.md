# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Wpisz &#39; &lt;typename&gt;&#39; nie jest zdefiniowany lub zawierający go moduł nie jest ładowany w sesji debugowania
Typ odwoływano, jest niedostępny lub nie jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30743  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy nazwa typu jest poprawna.  
  
2.  Definiowanie typu w ramach bieżącej klasy lub module.
