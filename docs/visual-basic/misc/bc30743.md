# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Typ "\<TypeName >" nie jest zdefiniowany lub moduł zawierający go nie jest ładowany w sesji debugowania
Odwołuje się do typu, który jest niedostępny lub nie jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30743  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy nazwa typu jest wpisana poprawnie.  
  
2. Zdefiniuj typ w bieżącej klasie lub module.
