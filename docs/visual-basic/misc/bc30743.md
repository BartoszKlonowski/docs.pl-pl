# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Typ '\<typename >' nie został zdefiniowany lub moduł zawierający ją nie jest załadowana w sesji debugowania
Typ odwoływano, jest niedostępny lub nie został zdefiniowany.  
  
 **Identyfikator błędu:** BC30743  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy nazwa typu została wpisana poprawnie.  
  
2. Definiowanie typu w ramach bieżącej klasy lub modułu.
