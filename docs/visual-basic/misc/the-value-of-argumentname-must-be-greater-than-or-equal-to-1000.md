# <a name="the-value-of-ltargumentnamegt-must-be-greater-than-or-equal-to-1000"></a>Wartość &lt;argumentname&gt; musi być większa lub równa 1000
Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> właściwość musi być większa lub równa 1000.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Właściwość określa maksymalną liczbę bajtów dozwoloną w pliku dziennika, zanim zostanie utworzony nowy plik dziennika.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> właściwości liczba większa niż lub równa 1000.
