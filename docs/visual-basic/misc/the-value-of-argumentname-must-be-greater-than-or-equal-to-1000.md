# <a name="the-value-of-argumentname-must-be-greater-than-or-equal-to-1000"></a>Wartość \<argumentname> musi być większa lub równa 1000

Wartość <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> właściwości musi być większa lub równa 1000.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A>Właściwość określa maksymalną liczbę bajtów dozwoloną w pliku dziennika, zanim zostanie utworzony nowy plik dziennika.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Ustaw <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Właściwość na wartość większą lub równą 1000.
