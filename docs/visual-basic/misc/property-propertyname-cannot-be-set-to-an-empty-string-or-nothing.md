# <a name="property-ltpropertynamegt-cannot-be-set-to-an-empty-string-or-nothing"></a>Właściwość &lt;propertyname&gt; nie można ustawić na ciąg pusty lub nie rób
Właściwość wymaga wartości innej niż ciąg pusty ("") lub `Nothing`. Może to być wynikiem niepoprawnie obliczania wartości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłową wartość.
