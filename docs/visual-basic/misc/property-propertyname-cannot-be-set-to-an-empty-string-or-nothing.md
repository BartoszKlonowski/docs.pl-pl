# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>\<propertyname>Dla właściwości nie można ustawić pustego ciągu lub niczego
Właściwość wymaga wartości innej niż pusty ciąg ("") lub `Nothing` . Może to być spowodowane błędnym obliczaniem wartości.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłową wartość.
