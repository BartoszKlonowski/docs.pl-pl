# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39; AddressOf &#39; wyrażenia nie są prawidłowe w oknach debugowania
`AddressOf`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30731  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `AddressOf` oświadczenia debugowania kodu.
