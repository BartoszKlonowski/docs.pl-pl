# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>Wyrażenia "AddressOf" nie są prawidłowe w oknach debugowania
`AddressOf`instrukcji można używać tylko w kodzie źródłowym.  
  
 **Identyfikator błędu:** BC30731  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `AddressOf` instrukcje z kodu debugowania.
