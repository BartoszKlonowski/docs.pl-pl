# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>"Let" i "Set" przypisanie instrukcje nie są już obsługiwane
`Let` i `Set` nie można używać w przypisaniach.  
  
 **Identyfikator błędu:** BC30807  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pomiń `Let` i `Set` z instrukcji przypisania; na przykład:  
  
     `X = 2`
