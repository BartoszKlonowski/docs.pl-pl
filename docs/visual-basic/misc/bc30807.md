# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>Instrukcje przypisania "let" i "Set" nie są już obsługiwane
`Let`i `Set` nie mogą być używane w przypisaniach.  
  
 **Identyfikator błędu:** BC30807  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Instrukcje pomijania `Let` i `Set` przypisywania; na przykład:  
  
     `X = 2`
