# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39; Let &#39; i &#39; Ustaw &#39; instrukcje przypisania są już obsługiwane.
`Let`i `Set` nie można używać w przypisania.  
  
 **Identyfikator błędu:** BC30807  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pomiń `Let` i `Set` z instrukcji przypisania; na przykład:  
  
     `X = 2`
