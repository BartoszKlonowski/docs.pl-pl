# <a name="global-not-allowed-in-handles-local-name-expected"></a>"Global" nie jest dozwolone w uchwyty; Oczekiwano nazwy lokalnego
A `Handles` klauzuli musi odwoływać się do lokalnego zdarzenia. `Global` — Słowo kluczowe zapewnia dostęp do elementów programowania, globalne.  
  
 **Identyfikator błędu:** BC36002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana `Handles` klauzuli do odwoływania się do lokalnego wystąpienia zdarzenia, zamiast globalnego wystąpienia.
