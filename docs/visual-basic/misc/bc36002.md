# <a name="39global39-not-allowed-in-handles-local-name-expected"></a>&#39;Globalne&#39; niedozwolony w uchwytach; Oczekiwano nazwy lokalnej
A `Handles` klauzuli musi odwoływać się do lokalnym zdarzeniem. `Global` — Słowo kluczowe zapewnia dostęp do elementów programowania globalnego.  
  
 **Identyfikator błędu:** BC36002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień `Handles` klauzuli do odwoływania się do lokalnego wystąpienia zdarzenia, zamiast globalnego wystąpienia.
