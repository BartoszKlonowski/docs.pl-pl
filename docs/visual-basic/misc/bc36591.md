# <a name="too-many-type-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Za dużo argumentów typu dla metody rozszerzenia "&lt;methodName&gt;"zdefiniowane w"&lt;typeName&gt;"
Metoda rozszerzenia ogólnego zostało wywołane więcej argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody rodzajowej, należy podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36591  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tak, że jest on dostępny dla każdego parametru typu definicją metody rodzajowej, które są wywoływane, Usuń argumentów typu na liście argumentów typu.
