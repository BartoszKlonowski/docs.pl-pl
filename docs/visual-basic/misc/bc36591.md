# <a name="too-many-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Za dużo argumentów typu dla metody rozszerzającej &#39; &lt;methodName&gt; &#39; zdefiniowane w &#39; &lt;typeName&gt;&#39;
Metody ogólne rozszerzenie zostało wywołane więcej argumentów typu niż liczba parametrów typu.  
  
 Po wywołaniu metody ogólnej, musisz podać jeden argument typu dla każdego parametru typu określone przez tę metodę.  
  
 **Identyfikator błędu:** BC36591  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń argumentów typu z listy argumentów typu, dzięki czemu jest on dostępny dla każdego parametru typ zdefiniowany przez metodę ogólnego, który jest.
