# <a name="evaluation-of-expression-or-statement-stopped"></a>Obliczenie wyrażenia lub instrukcji zostało zatrzymane
Wybrano element menu **Zatrzymaj ocenę** , co spowoduje zatrzymanie szacowania.  
  
 **Identyfikator błędu:** BC30721  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zezwól na kontynuację bez wybierania **stop oceny**.
