# <a name="type-or-with-expected"></a>Oczekiwano typu lub "with"
W przypadku deklarowania instancji klasy słowo kluczowe `New` musi następować po nazwie typu lub przez `With`. Na przykład następujące instrukcje deklarują `client` jako wystąpienie klasy `Customer`. Nazwa typu `Customer` następuje `New`.  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 Począwszy od Visual Basic 2008, można zadeklarować obiekt jako wystąpienie typu anonimowego, w tym przypadku nie określasz typu danych. W deklaracjach typu anonimowego słowo kluczowe `With` następuje `New`.  
  
```vb  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 **Identyfikator błędu:** BC30988  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień deklarację tak, aby `With` lub nazwa typu następuje `New`.
