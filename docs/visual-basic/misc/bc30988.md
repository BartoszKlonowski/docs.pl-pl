# <a name="type-or-with-expected"></a>Oczekiwano typu lub "with"
Gdy deklarujesz wystąpienie klasy, `New` słowo kluczowe musi następować po nazwie typu `With`lub. Na przykład następujące instrukcje deklarują `client` jako wystąpienie `Customer` klasy. Nazwa `Customer` typu jest następująca `New`.  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 Począwszy od Visual Basic 2008, można zadeklarować obiekt jako wystąpienie typu anonimowego, w tym przypadku nie określasz typu danych. W deklaracjach typu anonimowego słowo `With` kluczowe `New`jest następujące.  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 **Identyfikator błędu:** BC30988  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień deklarację tak, `With` aby lub nazwa typu była `New`następująca.
