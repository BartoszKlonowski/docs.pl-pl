<span data-ttu-id="02c19-103">W deklaracji wystąpienia klasy, `New` — słowo kluczowe musi występować według nazwy typu lub `With`.</span><span class="sxs-lookup"><span data-stu-id="02c19-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="02c19-104">Na przykład deklarować instrukcje następujące `client` można instancji `Customer` klasy.</span><span class="sxs-lookup"><span data-stu-id="02c19-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="02c19-105">Nazwa typu `Customer` następuje `New`.</span><span class="sxs-lookup"><span data-stu-id="02c19-105">The type name `Customer` follows `New`.</span></span>

```  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="02c19-106">Począwszy od [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], można zadeklarować obiektu być wystąpieniem typu anonimowego, w którym to przypadku należy określać typu danych.</span><span class="sxs-lookup"><span data-stu-id="02c19-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="02c19-107">W deklaracjach typu anonimowego, słowo kluczowe `With` następuje `New`.</span><span class="sxs-lookup"><span data-stu-id="02c19-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>
