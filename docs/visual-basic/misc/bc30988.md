# <a name="type-or-with-expected"></a><span data-ttu-id="c0b34-102">Wpisz lub "With" Oczekiwano</span><span class="sxs-lookup"><span data-stu-id="c0b34-102">Type or 'With' expected</span></span>
<span data-ttu-id="c0b34-103">Kiedy Deklarujesz wystąpienia klasy `New` — słowo kluczowe musi występować przez nazwę typu lub przez `With`.</span><span class="sxs-lookup"><span data-stu-id="c0b34-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="c0b34-104">Na przykład poniższe instrukcje każdego deklarują `client` za wystąpienie `Customer` klasy.</span><span class="sxs-lookup"><span data-stu-id="c0b34-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="c0b34-105">Nazwa typu `Customer` następuje `New`.</span><span class="sxs-lookup"><span data-stu-id="c0b34-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="c0b34-106">Począwszy od [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], można zadeklarować obiektu wystąpienie typu anonimowego, w którym to przypadku nie określisz typ danych.</span><span class="sxs-lookup"><span data-stu-id="c0b34-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="c0b34-107">W deklaracjach typu anonimowego, słowo kluczowe `With` następuje `New`.</span><span class="sxs-lookup"><span data-stu-id="c0b34-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
