# <a name="argument-ltargumentnamegt-cannot-be-a-multidimensional-array"></a>Argument "&lt;argumentname&gt;" nie może być tablicy wielowymiarowej
Ranga tablicy argumentu jest większa niż 1, ale może być tylko Jednowymiarowa tablica.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień pozycja argumentu na 1.
