# <a name="argument-argumentname-cannot-be-a-multidimensional-array"></a>Argument " \<argumentname> " nie może być tablicą wielowymiarową

Ranga argumentu tablicy jest większa niż 1, ale może być tylko tablicą jednowymiarową.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość argumentu ranga na 1.
