# <a name="argument-ltargumentnamegt-is-not-a-valid-value"></a>Argument "&lt;argumentname&gt;" nie jest prawidłową wartością
Wartość argumentu jest nieprawidłowa.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź wartość argumentu.  
  
2.  Jeśli argument jest dostarczany przez obliczanie, sprawdź wartości przetwarzane podczas obliczania.
