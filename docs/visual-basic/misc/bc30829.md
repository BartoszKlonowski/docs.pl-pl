# <a name="39get39-statements-are-no-longer-supported"></a>&#39;Pobierz&#39; instrukcje są już obsługiwane.
`Get` instrukcje nie są już obsługiwane. Funkcje We/Wy plików są dostępne w `Microsoft.VisualBasic` przestrzeni nazw.  
  
 `Get` nie jest obsługiwany dla operacji na plikach i można używać tylko w składni procedury właściwości.  
  
 **Identyfikator błędu:** BC30829  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Wykonaj operacje na plikach za pomocą elementów członkowskich `System.IO`, `FileSystemObject`i funkcje wykonawcze języka Visual Basic.
