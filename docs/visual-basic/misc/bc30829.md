# <a name="get-statements-are-no-longer-supported"></a>Instrukcje "Get" nie są już obsługiwane
`Get`instrukcje nie są już obsługiwane. Funkcja we/wy plików jest dostępna w `Microsoft.VisualBasic` przestrzeni nazw.  
  
 `Get`nie jest obsługiwany w przypadku operacji na plikach i może być używany tylko w składni procedury właściwości.  
  
 **Identyfikator błędu:** BC30829  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Wykonywanie operacji na plikach przy użyciu elementów członkowskich `System.IO` , `FileSystemObject` i Visual Basic funkcje czasu wykonywania.
