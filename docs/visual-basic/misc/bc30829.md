# <a name="get-statements-are-no-longer-supported"></a>Instrukcje "Get" nie są już obsługiwane
`Get` instrukcje nie są już obsługiwane. Dostępne są funkcje We/Wy pliku `Microsoft.VisualBasic` przestrzeni nazw.  
  
 `Get` nie jest obsługiwana dla operacji na plikach i można używać tylko w składni procedury właściwości.  
  
 **Identyfikator błędu:** BC30829  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Wykonywania operacji na plikach za pomocą elementów członkowskich `System.IO`, `FileSystemObject`i funkcje środowiska wykonawczego języka Visual Basic.
