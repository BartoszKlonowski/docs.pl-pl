<span data-ttu-id="60c76-103">Elementy członkowskie zainicjowana na liście inicjatora obiektu musi być pól lub właściwości.</span><span class="sxs-lookup"><span data-stu-id="60c76-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="60c76-104">Ponadto właściwości na liście inicjatora nie mogą mieć parametrów.</span><span class="sxs-lookup"><span data-stu-id="60c76-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="60c76-105">Właściwość przyczyną tego błędu jest przeciążony i każdy jego wersje wymaga argumentów.</span><span class="sxs-lookup"><span data-stu-id="60c76-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="60c76-106">W związku z tym nie można zainicjować właściwości na liście inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="60c76-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="60c76-107">**Identyfikator błędu:** BC30993</span><span class="sxs-lookup"><span data-stu-id="60c76-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="60c76-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="60c76-108">To correct this error</span></span>  
  
-   <span data-ttu-id="60c76-109">Usuń właściwość, która wymaga argumentów, na liście inicjatora.</span><span class="sxs-lookup"><span data-stu-id="60c76-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="60c76-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="60c76-110">Example</span></span>  
 <span data-ttu-id="60c76-111">Następujące klasy zawiera trzy definicji właściwości: jeden dla `TotalItems` i dwa dla `Item`, który jest przeciążona.</span><span class="sxs-lookup"><span data-stu-id="60c76-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="60c76-112">`TotalItems` Właściwość wymaga argumentów i mogą być inicjowane w listy inicjowania obiektu, jak pokazano w poniższych deklaracji.</span><span class="sxs-lookup"><span data-stu-id="60c76-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="60c76-113">`Item` Właściwości jest przeciążony i każdego przeciążenia wymaga argumentu.</span><span class="sxs-lookup"><span data-stu-id="60c76-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="60c76-114">W związku z tym `Item` nie może występować na liście inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="60c76-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="60c76-115">Aby uniknąć tego błędu, należy zainicjować `Item` właściwości poza inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="60c76-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
