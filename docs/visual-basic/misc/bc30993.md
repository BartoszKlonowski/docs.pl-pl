## <a name="example"></a><span data-ttu-id="d7e96-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="d7e96-110">Example</span></span>  
 <span data-ttu-id="d7e96-111">Następująca Klasa zawiera trzy definicje właściwości: jeden dla `TotalItems` i dwa dla `Item` , który jest przeciążony.</span><span class="sxs-lookup"><span data-stu-id="d7e96-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="d7e96-112">`TotalItems`Właściwość nie wymaga żadnych argumentów i można ją zainicjować na liście inicjalizacji obiektów, jak pokazano w poniższej deklaracji.</span><span class="sxs-lookup"><span data-stu-id="d7e96-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="d7e96-113">`Item`Właściwość jest przeciążona, a każde Przeciążenie wymaga argumentu.</span><span class="sxs-lookup"><span data-stu-id="d7e96-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="d7e96-114">W związku z tym `Item` nie może pojawić się na liście inicjatora obiektów.</span><span class="sxs-lookup"><span data-stu-id="d7e96-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="d7e96-115">Aby uniknąć tego błędu, zainicjuj `Item` Właściwość poza inicjatorem obiektu.</span><span class="sxs-lookup"><span data-stu-id="d7e96-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb

```
