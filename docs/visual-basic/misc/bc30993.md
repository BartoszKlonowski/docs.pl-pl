# <a name="property-ltpropertynamegt-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="5f94b-102">Właściwość "&lt;propertyname&gt;" nie można zainicjować w wyrażeniu inicjatora obiektów, ponieważ wszystkie dostępne przeciążenia wymagają argumentów</span><span class="sxs-lookup"><span data-stu-id="5f94b-102">Property '&lt;propertyname&gt;' cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="5f94b-103">Elementy członkowskie zainicjowana na liście inicjatora obiektu musi być pól lub właściwości.</span><span class="sxs-lookup"><span data-stu-id="5f94b-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="5f94b-104">Ponadto właściwości listy inicjalizatora nie może mieć parametrów.</span><span class="sxs-lookup"><span data-stu-id="5f94b-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="5f94b-105">Właściwość przyczyną tego błędu jest przeciążona i wszystkich jego wersji wymaga argumentów.</span><span class="sxs-lookup"><span data-stu-id="5f94b-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="5f94b-106">W związku z tym właściwości nie można zainicjować na liście inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="5f94b-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="5f94b-107">**Identyfikator błędu:** BC30993</span><span class="sxs-lookup"><span data-stu-id="5f94b-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5f94b-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="5f94b-108">To correct this error</span></span>  
  
-   <span data-ttu-id="5f94b-109">Usuń właściwość, która wymaga argumentów, na liście inicjatora.</span><span class="sxs-lookup"><span data-stu-id="5f94b-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5f94b-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="5f94b-110">Example</span></span>  
 <span data-ttu-id="5f94b-111">Następujące klasy zawiera trzy definicje, właściwości: jeden dla `TotalItems` , a dwa `Item`, która jest przeciążona.</span><span class="sxs-lookup"><span data-stu-id="5f94b-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="5f94b-112">`TotalItems` Właściwości wymaga bez argumentów i mogą być inicjowane z listy inicjowania obiektu, jak pokazano w poniższej deklaracji.</span><span class="sxs-lookup"><span data-stu-id="5f94b-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="5f94b-113">`Item` Właściwość jest przeciążona, a każde przeciążenie wymaga argumentu.</span><span class="sxs-lookup"><span data-stu-id="5f94b-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="5f94b-114">W związku z tym `Item` nie może występować na liście inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="5f94b-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="5f94b-115">Aby uniknąć tego błędu, należy zainicjować `Item` właściwość poza inicjatora obiektu.</span><span class="sxs-lookup"><span data-stu-id="5f94b-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
