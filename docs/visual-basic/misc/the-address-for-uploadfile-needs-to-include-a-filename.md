# <a name="the-address-for-uploadfile-needs-to-include-a-filename"></a>Adres UploadFile musi zawierać nazwę pliku

Podczas przekazywania plików przy użyciu `UploadFile` metody należy podać nazwę pliku w ścieżce docelowej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj nazwę pliku.
