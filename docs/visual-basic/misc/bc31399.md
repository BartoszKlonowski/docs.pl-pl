<span data-ttu-id="d9b1c-103">Klasa używa [instrukcji Inherits](../../visual-basic/language-reference/statements/inherits-statement.md) , aby określić klasę bazową, ale nie może uzyskać dostępu do żadnego konstruktora dla zamierzonej klasy podstawowej.</span><span class="sxs-lookup"><span data-stu-id="d9b1c-103">A class uses an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) to specify a base class, but it cannot access any constructor on the intended base class.</span></span>  
  
 <span data-ttu-id="d9b1c-104">Taka sytuacja może wystąpić, jeśli zamierzona Klasa bazowa nie ma konstruktorów lub jeśli ma konstruktory z poziomami dostępu, które uniemożliwiają dostęp z innej klasy.</span><span class="sxs-lookup"><span data-stu-id="d9b1c-104">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>  
  
 <span data-ttu-id="d9b1c-105">Podczas dziedziczenia klasy, Konstruktor powinien wywołać konstruktora klasy bazowej przy użyciu elementu [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span><span class="sxs-lookup"><span data-stu-id="d9b1c-105">When you inherit a class, your constructor should call the base class constructor using [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span></span> <span data-ttu-id="d9b1c-106">Jeśli to wywołanie nie zostanie wywołane lub jeśli nie napisze jawnego konstruktora, Visual Basic generuje niejawnego konstruktora, który wywołuje `MyBase.New()`.</span><span class="sxs-lookup"><span data-stu-id="d9b1c-106">If you do not make this call, or if you do not even write an explicit constructor, Visual Basic generates an implicit constructor that calls `MyBase.New()`.</span></span>  
  
 <span data-ttu-id="d9b1c-107">**Identyfikator błędu:** BC31399</span><span class="sxs-lookup"><span data-stu-id="d9b1c-107">**Error ID:** BC31399</span></span>

1. <span data-ttu-id="d9b1c-109">Jeśli masz kontrolę źródła nad zamierzoną klasą bazową, Zmień poziom dostępu co najmniej jednego z jego konstruktorów, tak aby inna Klasa mogła uzyskać do nich dostęp.</span><span class="sxs-lookup"><span data-stu-id="d9b1c-109">If you have source control over the intended base class, then change the access level of at least one of its constructors so that another class can access them.</span></span>
