# <a name="case-cannot-follow-a-case-else-in-the-same-select-statement"></a>"Case" nie mogą następować po "Case Else" w tej samej instrukcji "Select"
A `Case Else` instrukcji wprowadza instrukcje do wykonania, jeśli nie zostanie znalezione dopasowanie dla początkowego `Case`. A `Case` znaleziono instrukcji po `Case Else` w tym samym `Select` bloku.  
  
 **Identyfikator błędu:** BC30321  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `Case Else` do odpowiedniej lokalizacji po `Case` instrukcji.
