# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39; Przypadek &#39; Nie można wykonać &#39; Case Else &#39; w taki sam &#39; Wybierz opcję &#39; — Instrukcja
A `Case Else` instrukcji wprowadza instrukcje do wykonania, jeśli nie znaleziono dla początkowej `Case`. A `Case` znaleziono po instrukcji `Case Else` w tym samym `Select` bloku.  
  
 **Identyfikator błędu:** BC30321  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `Case Else` w odpowiedniej lokalizacji po `Case` instrukcji.
