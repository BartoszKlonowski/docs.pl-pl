  
 <span data-ttu-id="ff296-105">Ograniczenia pośrednie `Structure` i `Class` powodują konflikt dla parametru typu `t1` , ponieważ `Structure` ograniczenie wymaga, aby odpowiedni argument typu był typem wartości, a jednocześnie wymaga, aby był `Class` typem referencyjnym.</span><span class="sxs-lookup"><span data-stu-id="ff296-105">The indirect constraints `Structure` and `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
