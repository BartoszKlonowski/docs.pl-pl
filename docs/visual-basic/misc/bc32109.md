 <span data-ttu-id="44f20-105">Ograniczenia pośrednie `Structure` i `Class` powodować konfliktu dla parametru typu `t1`, ponieważ `Structure` ograniczenie wymaga, że odpowiedni argument typu jako typów wartości podczas `Class` wymaga, aby były Typ odwołania.</span><span class="sxs-lookup"><span data-stu-id="44f20-105">The indirect constraints `Structure` and `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
