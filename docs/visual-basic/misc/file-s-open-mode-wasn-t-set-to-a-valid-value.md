# <a name="file39s-open-mode-wasn39t-set-to-a-valid-value"></a>Plik &#39;był trybu otwartego s &#39; ustaw wartość na prawidłową t
Wartość podana dla trybu otwartego pliku jest nieprawidłowa. W poniższej tabeli przedstawiono prawidłowe wartości <xref:Microsoft.VisualBasic.OpenMode> wyliczenia.  
  
|Wartość|Tryb|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź wartość jest podana dla trybu otwartego pliku.
