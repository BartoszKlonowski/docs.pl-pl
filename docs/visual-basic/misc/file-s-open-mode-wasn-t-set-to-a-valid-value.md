# <a name="files-open-mode-wasnt-set-to-a-valid-value"></a>Tryb otwierania pliku nie został ustawiony na prawidłową wartość
Wartość podana dla trybu otwarcia pliku jest nieprawidłowa. W poniższej tabeli przedstawiono prawidłowe wartości <xref:Microsoft.VisualBasic.OpenMode> wyliczania.  
  
|Wartość|Tryb|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź wartość podaną dla trybu otwierania pliku.
