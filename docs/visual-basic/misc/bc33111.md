# <a name="nullable-types-are-not-allowed-in-conditional-compilation-expressions"></a>Typy nullable nie są dozwolone w wyrażeniach kompilacji warunkowej
Typ nullable nie może być używany w wyrażeniu dyrektywy kompilacji warunkowej. Na przykład następujący kod powoduje ten błąd.  
  
```vb  
'#Const triggerPoint = 0  
  
'' Not valid.  
'#If CType(triggerpoint, Boolean?) = True Then  
'        ' Body of the conditional directive.  
'#End If  
```  
  
 **Identyfikator błędu:** Bc33111  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń oznaczenie typu wartości możliwej do wartości null.
