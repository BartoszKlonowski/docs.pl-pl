# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>Projekt &#39; &lt;projectname&gt; &#39; nie może odwoływać się projektu &#39; &lt;projectname&gt; &#39; ponieważ &#39; &lt;projectname&gt; &#39; bezpośrednio lub pośrednio odwołuje się do &#39; &lt;projectname&gt;&#39;
Projekt zawiera odwołanie do projektu drugiego. Drugi projekt z kolei zawiera odwołanie do projektu, w którym odwołuje się do.  
  
 **Identyfikator błędu:** BC30761  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń jedno z odwołań.
