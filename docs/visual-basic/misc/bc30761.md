# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Projekt " \<projectname> " nie może odwoływać się do projektu " \<projectname> ", ponieważ " \<projectname> " bezpośrednio lub pośrednio odwołuje się do " \<projectname> "

Projekt zawiera odwołanie do drugiego projektu. Drugi projekt z kolei zawiera odwołanie do projektu, do którego się odwołuje.  
  
 **Identyfikator błędu:** BC30761  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń jedno z odwołań.
