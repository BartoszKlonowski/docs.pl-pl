# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Projekt "&lt;projectname&gt;"nie może odwoływać się do projektu"&lt;projectname&gt;"ponieważ"&lt;projectname&gt;"bezpośrednio lub pośrednio odwołuje się do"&lt;projectname &gt;'
Projekt zawiera odwołanie do drugiego projektu. Drugi projekt z kolei zawiera odwołanie do projektu, w którym jest przywoływany.  
  
 **Identyfikator błędu:** BC30761  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń jedno z odwołań.
