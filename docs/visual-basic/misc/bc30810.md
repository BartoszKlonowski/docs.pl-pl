# <a name="methods-cannot-be-declared-static"></a>Metody nie mogą być zadeklarowane jako "Static"
`Static` Modyfikator nie można zastosować do całej procedury.  
  
 **Identyfikator błędu:** BC30810  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Static` modyfikator do deklarowania poszczególnych elementów w obrębie procedury jako statyczny.
