# <a name="methods-cannot-be-declared-39static39"></a>Nie można zadeklarować metody &#39;statyczne&#39;
`Static` Modyfikator nie można zastosować do całej procedury.  
  
 **Identyfikator błędu:** BC30810  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Static` modyfikator Aby zadeklarować poszczególnych elementów w obrębie procedury jako statyczny.
