# <a name="methods-cannot-be-declared-static"></a>Metody nie mogą być deklarowane jako "static"

`Static`Modyfikator nie może zostać zastosowany do całych procedur.  
  
 **Identyfikator błędu:** BC30810  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Static` modyfikatora, aby zadeklarować poszczególne elementy w ramach procedury jako static.
