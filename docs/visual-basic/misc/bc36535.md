# <a name="instance-members-and-me-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="59cb8-102">Nie można używać składowych wystąpień i elementu "Me" w wyrażeniu zapytania</span><span class="sxs-lookup"><span data-stu-id="59cb8-102">Instance members and 'Me' cannot be used in a query expression</span></span>

<span data-ttu-id="59cb8-103">Zapytanie LINQ w tabeli `Structure` zawiera odwołanie do `Me` wystąpienia lub do elementu członkowskiego struktury.</span><span class="sxs-lookup"><span data-stu-id="59cb8-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="59cb8-104">Odwołania do `Me` lub składowe wystąpienia są niedozwolone w wyrażeniach zapytań w elemencie `Structure` .</span><span class="sxs-lookup"><span data-stu-id="59cb8-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="59cb8-105">**Identyfikator błędu:** BC36535</span><span class="sxs-lookup"><span data-stu-id="59cb8-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="59cb8-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="59cb8-106">To correct this error</span></span>  
  
1. <span data-ttu-id="59cb8-107">Utwórz kopię elementu członkowskiego wystąpienia lub wartość zwracaną przez odwołanie do `Me` i Użyj kopii w wyrażeniu zapytania, jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="59cb8-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
