## <a name="example"></a><span data-ttu-id="f67d6-109">Przykład</span><span class="sxs-lookup"><span data-stu-id="f67d6-109">Example</span></span>  
 <span data-ttu-id="f67d6-110">Następujące klasy zawiera właściwości domyślnej `defaultProp`, który wymaga argumentu liczby całkowitej.</span><span class="sxs-lookup"><span data-stu-id="f67d6-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

```

```
