# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Instrukcje deklaracji metody muszą być pierwszym elementem w linii logicznej
`Function`Instrukcja or jest `Sub` zgodna z dwukropkiem (:) Separator instrukcji. `Function`lub `Sub` musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
