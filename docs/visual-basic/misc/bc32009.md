# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Instrukcje deklaracji metody musi mieć pierwszy w wierszu logicznym
A `Function` lub `Sub` instrukcji następuje separatorem instrukcji dwukropek (:). `Function` lub `Sub` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC32009  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
