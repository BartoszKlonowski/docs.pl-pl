# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese a VbStrConv.TraditionalChinese nelze kombinovat
Aplikacja próbuje połączyć `VbStrConv` elementów członkowskich wyliczenia `SimplifiedChinese` i `TraditionalChinese`, które wzajemnie się wykluczają.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Albo usuń `VbStrConv.SimplifiedChinese` lub `VbStrConv.TraditionalChinese`.
