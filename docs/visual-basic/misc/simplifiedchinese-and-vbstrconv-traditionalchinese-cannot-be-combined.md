# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>Nie można łączyć SimplifiedChinese i VbStrConv. TraditionalChinese
Aplikacja próbuje połączyć `VbStrConv` członków wyliczenia `SimplifiedChinese` i `TraditionalChinese`, które wzajemnie się wykluczają.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `VbStrConv.SimplifiedChinese` lub `VbStrConv.TraditionalChinese`.
