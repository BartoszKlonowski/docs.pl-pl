# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath określa istniejący folder

Parametr `TargetFilePath` określa istniejący katalog, ale musi określać nazwę nowego katalogu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień `TargetFilePath` , aby określić nowy, a nie istniejący katalog.
