# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath określa istniejącego folderu
Parametr `TargetFilePath` określa istniejącym katalogiem, ale należy określić nazwę nowego katalogu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień `TargetFilePath` określić nowe, zamiast istniejącego katalogu.
