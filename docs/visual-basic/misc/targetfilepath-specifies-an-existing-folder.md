# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath Określa istniejący folder
Parametr `TargetFilePath` Określa istniejący katalog, ale należy określić nazwę nowego katalogu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana `TargetFilePath` określić katalog nowe, zamiast istniejącej.
