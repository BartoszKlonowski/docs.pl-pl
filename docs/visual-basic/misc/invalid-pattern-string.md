# <a name="invalid-pattern-string"></a>Nieprawidłowy ciąg znaków wzorca
Wzorzec ciągu określonego w `Like` operacji wyszukiwania jest nieprawidłowy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Przejrzyj prawidłowe znaki listy wyrażeń.  
  
2.  W zakresie wzorzec, upewnij się, że początkowy znaku zakresu jest mniejsza niż końcowy znak zakresu, podobnie jak w `[a-z]`.  
  
3.  W zakresie wzorzec, upewnij się, że nie ma wiele łączników obok siebie, podobnie jak w `[a--z]`.  
  
4.  Zakresów wzorzec końcowego nawiasu zamykającego.
