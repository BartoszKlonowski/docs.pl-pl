# <a name="no-accessible-overloaded-ltmethodnamegt-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>Przeciążone nie dostępne "&lt;methodname&gt;" można wywołać z tymi argumentami bez konwersji rozszerzającej: &lt;listy&gt;
Przeciążona metoda została wywołana, ale żadna metoda może zostać dopasowane do listy podanych argumentów bez konwersji rozszerzającej.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `Option Strict Off`.  
  
-   Zmień argumentów na odpowiadającą jednemu z nich podpisów przeciążonej metody.
