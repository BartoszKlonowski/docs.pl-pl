# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>Żaden dostępny przeciążony element &#39; &lt;methodname&gt;&#39; może być wywołany z tymi argumentami bez użycia konwersji rozszerzającej: &lt;listy&gt;
Przeciążona metoda została wywołana, ale brak metody mogłyby zostać dopasowane do podanych argumentów bez użycia konwersji rozszerzającej z listy.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ `Option Strict Off`.  
  
-   Zmień argumenty, które mają być zgodny z jednym podpisów przeciążonej metody.
