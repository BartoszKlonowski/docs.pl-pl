# <a name="array-initializer-is-missing-number-elements"></a>Inicjator tablicy nie ma \<liczby elementów >
Nie dostarczono wystarczającej liczby elementów w inicjatorze dla tablicy.  
  
 **Identyfikator błędu:** BC30567  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj brakujący argument lub argumenty.  
  
- Użyj `ReDim` , aby zmienić rozmiar tablicy.
