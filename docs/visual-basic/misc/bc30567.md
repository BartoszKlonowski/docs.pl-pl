# <a name="array-initializer-is-missing-number-elements"></a>Brak inicjatora tablicy \<numer > elementy
Nie podano za mało elementów w inicjatorze dla tablicy.  
  
 **Identyfikator błędu:** BC30567  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj brakujące argument lub argumentów.  
  
-   Użyj `ReDim` można zmienić rozmiar tablicy.
