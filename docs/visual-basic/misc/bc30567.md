# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>Inicjatorze tablicy Brak &lt;numer&gt; elementów
Nie podano za mało elementów w inicjatorze dla macierzy.  
  
 **Identyfikator błędu:** BC30567  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy podać argument brakujące lub argumentów.  
  
-   Użyj `ReDim` Aby zmienić rozmiar tablicy.
