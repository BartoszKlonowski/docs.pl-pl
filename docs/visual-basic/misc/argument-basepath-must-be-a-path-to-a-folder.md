# <a name="argument-basepath-must-be-a-path-to-a-folder"></a>Argument BasePath musi być ścieżką do folderu
Argument `BasePath` musi zawierać ścieżkę do folderu. Może być niepoprawnie analizowania parametrów i podanie wartości, który nie został rozpoznany jako prawidłowa ścieżka.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź wartość są podawania dla `BasePath` aby upewnić się, jest prawidłową ścieżkę do folderu.
