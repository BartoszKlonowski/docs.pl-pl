# <a name="argument-basepath-must-be-a-path-to-a-folder"></a>Argument BasePath musi być ścieżką do folderu

Argument `BasePath` musi składać się ze ścieżki do folderu. Możliwe jest nieprawidłowe analizowanie ciągu i podanie wartości, która nie jest rozpoznawana jako prawidłowa ścieżka.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź wartość, która jest dostarczana, `BasePath` Aby upewnić się, że jest to prawidłowa ścieżka do folderu.
