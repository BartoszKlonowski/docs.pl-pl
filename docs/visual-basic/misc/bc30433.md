# <a name="methods-in-a-module-cannot-be-declared-ltspecifiergt"></a>Nie można zadeklarować metody w Module "&lt;specyfikator&gt;"
Wykorzystano specyfikator, który jest nieprawidłowy dla metody w ramach `Module` instrukcji. Moduły nigdy nie mogą zostać utworzone, nie obsługują dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
