# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>Metody w module nie mogą być zadeklarowane jako " \<specifier> "

Użyto specyfikatora, który nie jest prawidłowy w metodzie w `Module` instrukcji. Nie można tworzyć wystąpień modułów, nie obsługują dziedziczenia ani implementowania interfejsów.  
  
 **Identyfikator błędu:** BC30433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
