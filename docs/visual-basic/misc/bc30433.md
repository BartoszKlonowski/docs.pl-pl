# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Nie można zadeklarować metody w Module &#39; &lt;specyfikator&gt;&#39;
Zostały użyte specyfikator, który jest nieprawidłowy dla metody w ramach `Module` instrukcji. Moduły nigdy nie można utworzyć wystąpienia, nie obsługuje dziedziczenia i nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30433  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
