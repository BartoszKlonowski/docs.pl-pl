# <a name="array-initializer-has-too-many-dimensions"></a>Inicjator tablicy ma za dużo wymiarów

Podano zbyt wiele wymiarów w inicjatorze dla tablicy.  
  
 **Identyfikator błędu:** BC30566  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź inicjator tablicy, aby określić, ile wymiarów jest koniecznych.  
  
- Usuń dodatkowy argument lub argumenty.
