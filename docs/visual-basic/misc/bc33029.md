# <a name="conversion-operators-cannot-convert-from-an-interface-type"></a>Operatory konwersji nie mogą konwertować z typu interfejsu
Operator konwersji jest zadeklarowana z typu interfejsu dla parametru.  
  
 W czasie kompilacji Visual Basic uwzględnia wstępnie zdefiniowanych konwersji istnieje z dowolnego interfejsu do dowolnego typu odwołania. Takie Konwersja może się nie powieść w czasie wykonywania, ale kompilator nie można przewidzieć wyniki środowiska wykonawczego, więc umożliwia takie konwersji skompilować.  
  
 Ponieważ kompilator uwzględnia konwersji do zdefiniowania już, jego pozwalają zdefiniować ją ponownie.  
  
 **Identyfikator błędu:** BC33029  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Całkowicie usunąć definicję tego operatora. Już jest wstępnie zdefiniowane.
