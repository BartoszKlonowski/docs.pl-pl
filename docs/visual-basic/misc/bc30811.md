# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>&#39;ReDim&#39; instrukcje już służy do deklarowania zmiennych tablicowych
`ReDim` można użyć tylko aby zmienić rozmiar istniejącej tablicy.  
  
 **Identyfikator błędu:** BC30811  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ rozmiar tablic, jeśli są deklarowane jako; na przykład:  
  
    ```  
    Dim X(20) As Integer  
    ```
