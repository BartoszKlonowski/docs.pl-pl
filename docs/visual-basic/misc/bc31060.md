# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>element " \<propertyname> " niejawnie definiuje element " \<membername> ", który powoduje konflikt z elementem członkowskim o tej samej nazwie w elemencie \<type> " \<typename> "
Nazwa elementu członkowskiego typu powoduje konflikt z nazwą elementu członkowskiego niejawnie utworzonego dla właściwości. Właściwości niejawnie tworzą kilka niejawnych zmiennych, które mogą powodować konflikt ze zdefiniowanymi elementami członkowskimi.  
  
 **Identyfikator błędu:** BC31060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę jawnie zadeklarowanego elementu członkowskiego, aby usunąć konflikt nazw.
