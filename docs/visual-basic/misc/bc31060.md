# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39; &lt;propertyname&gt;&#39; niejawnie definiuje &#39;&lt; membername&gt;&#39; który powoduje konflikt z członkiem o tej samej nazwie w &lt;typu&gt; &#39;&lt; Właściwość TypeName&gt;&#39;
Nazwa wystąpił konflikt elementu członkowskiego typu o nazwie elementu członkowskiego niejawnie utworzone dla właściwości. Właściwości niejawnie Utwórz kilka niejawne zmienne, które mogą powodować konflikt z określonych elementów członkowskich.  
  
 **Identyfikator błędu:** BC31060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę jawnie zadeklarowanego elementu członkowskiego, aby usunąć konflikt nazw.
