# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>'\<propertyname >' niejawnie definiuje '\<membername >', który powoduje konflikt ze składową o tej samej nazwie w \<typu > '\<typename >'
Nazwa konflikt składowych typu o nazwie elementu członkowskiego tworzone niejawnie dla właściwości. Właściwości niejawnie Utwórz kilka niejawne zmienne, które mogą powodować konflikt z określonych elementów członkowskich.  
  
 **Identyfikator błędu:** BC31060  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę zadeklarowany w sposób jawny element członkowski, aby usunąć konflikt nazw.
