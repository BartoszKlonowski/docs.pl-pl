# <a name="argument-argumentname-is-nothing-or-empty"></a>Argument " \<argumentname> " ma wartość Nothing lub jest pusty
Nazwa argumentu jest nieprawidłowa, ponieważ jest ona zainicjowana `Nothing` lub jest pusta. Argument musi mieć określony typ danych i prawidłową nazwę.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj prawidłową nazwę argumentu.
