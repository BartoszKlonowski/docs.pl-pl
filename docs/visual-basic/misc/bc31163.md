# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Oczekiwano dopasowania zamykanie pojedynczy cudzysłów wartości atrybutu XML
Wartość atrybutu, który sterujący jest ujęty w znaki pojedynczego cudzysłowu brakuje zamykającego znaku cudzysłowu.  
  
 **Identyfikator błędu:** BC31163  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak pojedynczego cudzysłowu zamykającego (') do ograniczania wartości atrybutu.
