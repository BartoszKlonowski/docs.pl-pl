# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Oczekiwano pasującego pojedynczego cudzysłowu zamykającego dla wartości atrybutu XML
Wartość atrybutu, który jest rozdzielone pojedynczy cudzysłów brakuje zamykającego znaku cudzysłowu.  
  
 **Identyfikator błędu:** BC31163  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znak pojedynczego cudzysłowu zamykającego (') do ograniczania wartości atrybutu.
