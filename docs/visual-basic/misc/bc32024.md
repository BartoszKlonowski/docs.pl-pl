# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>Nie można podać wartości domyślne dla parametrów, które nie są zadeklarowane jako "Optional"
Procedura jest wywoływana, gdy nie przyjmują wartości domyślne w deklaracji, należy podać wymagane argumenty.  
  
 **Identyfikator błędu:** BC32024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj argument był `Optional` lub usuń wartość domyślną.
