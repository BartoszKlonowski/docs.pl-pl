# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>Nie można podać wartości domyślne dla parametrów, które nie są zadeklarowane &#39; opcjonalne &#39;
Argument musi być podany procedura jest wywoływana, a nie przyjmują wartości domyślne w deklaracji.  
  
 **Identyfikator błędu:** BC32024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Argument można zadeklarować `Optional` lub usuń wartość domyślną.
