# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>Nie można podać wartości domyślnych dla parametrów, które nie są zadeklarowane jako opcjonalne.

Wymagane argumenty muszą być dostarczone, gdy procedura jest wywoływana i nie przyjmuje wartości domyślnych w deklaracji.  
  
 **Identyfikator błędu:** BC32024  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj argument `Optional` lub usuń wartość domyślną.
