# <a name="member-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>" \<member> " powoduje konflikt z zastrzeżoną składową o tej nazwie, która jest niejawnie zadeklarowana we wszystkich wyliczeniach
Nazwa elementu członkowskiego typu powoduje konflikt z nazwą elementu członkowskiego niejawnie zadeklarowaną we wszystkich wyliczeniach.  
  
 Kompilator Visual Basic tworzy niejawnych członków odpowiadających określonym elementom programistycznym, które deklarujesz. Wyliczenia niejawnie deklarują element członkowski `value__member` .  
  
 **Identyfikator błędu:** BC31420  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmodyfikuj nazwę elementu członkowskiego.
