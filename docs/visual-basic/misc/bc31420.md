# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>&#39;&lt;element członkowski&gt; &#39; powoduje konflikt z zastrzeżonym członkiem o tej nazwie, niejawnie zadeklarowanym we wszystkich wyliczeniach
Nazwa powoduje konflikt typu elementu członkowskiego o nazwie elementu członkowskiego niejawnie zadeklarowany we wszystkich wyliczeniach.  
  
 Kompilator Visual Basic tworzy niejawne elementy członkowskie odpowiadający niektórych programistyczny zadeklarowanego. Wyliczenia niejawnie deklaruje element członkowski `value__member`.  
  
 **Identyfikator błędu:** BC31420  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmodyfikuj nazwę elementu członkowskiego.
