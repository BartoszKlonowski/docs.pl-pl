# <a name="39inherits39-not-valid-in-modules"></a>&#39; dziedziczy &#39; nie jest prawidłowa w modułach
`Inherits` Występuje instrukcja wewnątrz modułu. Moduły nie może dziedziczyć z klasy.  
  
 **Identyfikator błędu:** BC30230  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji lub wpisz ponownie moduł jako klasa.
