# <a name="inherits-not-valid-in-modules"></a>"Inherits" nie jest prawidłowy w modułach
`Inherits`Instrukcja występuje wewnątrz modułu. Moduły nie mogą dziedziczyć z klas.  
  
 **Identyfikator błędu:** BC30230  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Inherits` instrukcję lub ponownie wpisz moduł jako klasę.
