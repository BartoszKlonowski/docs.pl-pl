# <a name="inherits-not-valid-in-modules"></a>"Inherits" nie jest prawidłowa w modułach
`Inherits` Występuje instrukcja wewnątrz modułu. Moduły nie może dziedziczyć z klas.  
  
 **Identyfikator błędu:** BC30230  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Inherits` instrukcji lub wpisz ponownie moduł jako klasa.
