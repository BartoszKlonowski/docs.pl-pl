# <a name="parametername-is-already-declared-as-a-type-parameter-of-this-method"></a>"\<parametername >" jest już zadeklarowany jako parametr typu tej metody
Ogólna procedura definiuje normalne parametr lub zmienna lokalna o tej samej nazwy co parametr typu.  
  
 Każdy parametr procedury, w tym co parametr typu ogólnego procedurę, musi mieć nazwę różne od innych parametrów. Parametry procedury są używane jako zmienne lokalne, wszelkie zmienna lokalna zadeklarowana wewnątrz procedury musi również mieć różne od wszystkich parametrów nazwę i parametry typu.  
  
 **Identyfikator błędu:** BC32089  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę normalne parametr lub zmienna lokalna.
