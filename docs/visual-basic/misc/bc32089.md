# <a name="39ltparameternamegt39-is-already-declared-as-a-type-parameter-of-this-method"></a>&#39; &lt;parametername&gt;&#39; jest już zadeklarowany jako parametr typu tej metody
Procedury ogólne definiuje normalne parametr lub zmienna lokalna o takiej samej nazwie jak parametr typu.  
  
 Każdy parametr procedury, w tym co parametr typu ogólnego procedury, musi mieć nazwę różne od innych parametrów. Ponieważ parametry procedury są używane jako zmienne lokalne, wszelkie zmienna lokalna zadeklarowana w ramach procedury musi również mieć nazwę różne od wszystkie parametry i parametry typu.  
  
 **Identyfikator błędu:** BC32089  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę normalne parametr lub zmienna lokalna.
