# <a name="duplicate-xml-attribute-attributename"></a>Zduplikowany atrybut XML "attributeName"
Atrybut określony dla literału elementu XML jest zgodny z atrybutem, który został wcześniej zdefiniowany dla literału elementu XML. Nazwy atrybutów w literałach elementu XML muszą być unikatowe.  
  
 **Identyfikator błędu:** BC31149  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę atrybutu na nazwę, która jest unikatowa dla literału elementu XML.
