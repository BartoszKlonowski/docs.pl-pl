# <a name="duplicate-xml-attribute-attributename"></a>Zduplikowany atrybut XML "attributeName"
Atrybut, który został określony literał elementu XML jest zgodny z atrybutu, który został wcześniej zdefiniowany dla literał elementu XML. Nazwy atrybutów w literałach — element XML musi być unikatowa.  
  
 **Identyfikator błędu:** BC31149  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę atrybutu na nazwę, która jest unikatowa dla literał elementu XML.
