# <a name="duplicate-xml-attribute-39attributename39"></a>Zduplikowany atrybut XML &#39; attributeName &#39;
Określony literał elementu XML atrybut jest zgodny ze atrybut, który został wcześniej zdefiniowany dla literał elementu XML. Atrybut nazwy w literałach XML elementu muszą być unikatowe.  
  
 **Identyfikator błędu:** BC31149  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę atrybutu na nazwę, która jest unikatowa dla literał elementu XML.
