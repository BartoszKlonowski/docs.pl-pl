# <a name="expression-expected"></a>Oczekiwano wyrażenia
Operator, takich jak `+` lub `=` wymaga wyrażenia uzyskać jego prawej strony, ale nie rozpoznać wyrażenia.  
  
 **Identyfikator błędu:** BC30201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłowe wyrażenie dla prawy argument operacji operatora.
