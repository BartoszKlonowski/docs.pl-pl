# <a name="expression-expected"></a>Oczekiwano wyrażenia
Operator, takie jak `+` lub `=` wymaga wyrażenia prawy argument operacji, ale żadne wyrażenie rozpoznawalna pojawia się.  
  
 **Identyfikator błędu:** BC30201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj prawidłowe wyrażenie prawy operand operatora.
