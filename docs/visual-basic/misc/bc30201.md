# <a name="expression-expected"></a>Oczekiwano wyrażenia

Operator, taki jak `+` lub `=` wymaga wyrażenia dla swojego operandu, ale nie pojawia się wyrażenie rozpoznawalne.  
  
 **Identyfikator błędu:** BC30201  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Podaj prawidłowe wyrażenie dla prawego operandu operatora.
