# <a name="39end-function39-must-be-preceded-by-a-matching-39function39"></a>&#39;End Function&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;— funkcja&#39;
`End Function` Występuje instrukcja w kodzie z pasujących `Function` Definicja procedury przed nim.  
  
 **Identyfikator błędu:** BC30430  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `End Function` instrukcję, jeśli jest nadmiarowy.  
  
2.  Podaj brakujący `Function` procedura w przypadku braku takiego.  
  
3.  Przenieś `End Function` odpowiednie miejsce w kodzie.
