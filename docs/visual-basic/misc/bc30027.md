# <a name="end-function-expected"></a>Oczekiwano "End Function"
`End` Instrukcja musi odpowiadać blok zostaje zakończone. A `Function` definicji procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC30027  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Function` instrukcji na końcu procedury.
