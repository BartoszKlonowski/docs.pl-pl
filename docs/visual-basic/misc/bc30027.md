# <a name="39end-function39-expected"></a>&#39; Funkcja end &#39; Oczekiwano
`End` Instrukcja musi odpowiadać bloku zostaje zakończone. A `Function` Definicja procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC30027  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `End Function` instrukcji na końcu procedury.
