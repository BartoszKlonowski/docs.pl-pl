# <a name="end-function-expected"></a>Oczekiwano instrukcji "End Function"

`End`Instrukcja musi być zgodna z zamykanym blokiem. `Function`Definicja procedury nie została prawidłowo zakończona.  
  
 **Identyfikator błędu:** BC30027  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `End Function` instrukcję do końca procedury.
