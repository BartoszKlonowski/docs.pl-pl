# <a name="ltmembergt-implicitly-defined-for-lteventnamegt-cannot-shadow-a-mustoverride-method-in-the-base-ltclassgt-ltclassnamegt"></a>"&lt;elementu członkowskiego&gt;", niejawnie zdefiniowane dla"&lt;eventname&gt;", nie może zasłonić metody "MustOverride" w podstawowym &lt;klasy&gt; "&lt;classname&gt;"
Określone zdarzenie niejawnie deklaruje element członkowski o takiej samej nazwie, zgodnie z metodą zadeklarowana za pomocą `MustOverride` modyfikator.  
  
 **Identyfikator błędu:** BC31413  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `MustOverride` modyfikator metody w klasie bazowej lub podać właściwość lub metoda unikatową nazwę.
