# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>Powoduje konflikt z "&lt;Nazwa1&gt;", która niejawnie jest tworzona dla"&lt;Nazwa2&gt;" w &lt;typu&gt; "&lt;nazwa3&gt;"
Nazwa wystąpił konflikt składowej typu innemu członkowi stworzonego.  
  
 **Identyfikator błędu:** BC31058  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę zadeklarowany w sposób jawny element członkowski, aby usunąć konflikt nazw.
