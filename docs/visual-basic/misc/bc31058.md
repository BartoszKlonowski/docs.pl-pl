# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>Powoduje konflikt z &#39; &lt;Nazwa1&gt;&#39;, która niejawnie jest tworzona dla &#39; &lt;Nazwa2&gt; &#39; w &lt;typu&gt; &#39; &lt;nazwa3&gt;&#39;
Nazwa wystąpił konflikt elementu członkowskiego typu z innym elementem członkowskim niejawnie tworzonych.  
  
 **Identyfikator błędu:** BC31058  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę jawnie zadeklarowanego elementu członkowskiego, aby usunąć konflikt nazw.
