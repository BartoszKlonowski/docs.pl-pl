# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Powoduje konflikt z '\<name1 >', która niejawnie jest tworzona dla'\<Nazwa2 >' w \<typu >'\<nazwa3 > '
Nazwa elementu członkowskiego typu powoduje konflikt z innym niejawnie utworzonym członkiem.  
  
 **Identyfikator błędu:** BC31058  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień nazwę jawnie zadeklarowanego elementu członkowskiego, aby usunąć konflikt nazw.
