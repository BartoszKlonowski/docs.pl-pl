# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Powoduje konflikt z "\<name1 >", która niejawnie jest tworzona dla"\<Nazwa2 >' w \<typu >"\<nazwa3 > "
Nazwa wystąpił konflikt składowej typu innemu członkowi stworzonego.  
  
 **Identyfikator błędu:** BC31058  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień nazwę zadeklarowany w sposób jawny element członkowski, aby usunąć konflikt nazw.
