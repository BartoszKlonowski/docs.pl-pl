# <a name="modifier-is-not-valid-on-an-interface-declaration"></a>'\<modyfikator >' nie jest prawidłowy w deklaracji interfejsu
Użyto modyfikatora, który nie jest prawidłowy w `Interface` deklaracji. Jedyne prawidłowe Modyfikatory dla `Sub`, `Function`, `Interface` lub `Property` instrukcji zadeklarowanych w deklaracji są `Overloads` słowami `Default` kluczowymi i. Inne modyfikatory, takie jak `Public` `Friend`, `Private` `Protected` ,,`Shared`,,, ,`MustOverride` i ,`Overridable`sąnieprawidłowe. `Overrides` `Static`  
  
 **Identyfikator błędu:** BC30397  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
