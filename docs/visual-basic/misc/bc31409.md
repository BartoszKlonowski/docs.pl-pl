# <a name="method-methodname1-must-be-declared-overloads-because-another-methodname2-is-declared-overloads"></a>\<Metoda > '\<methodname1 >' musi być zadeklarowany jako 'Overloads' ponieważ inny '\<methodname2 >' jest zadeklarowany jako 'Overloads'
Inną właściwość lub metoda o takiej samej nazwie jest zadeklarowana za pomocą `Overloads` modyfikator.  
  
 **Identyfikator błędu:** BC31409  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj `Overloads` modyfikatora właściwości lub metody.  
  
2. Usuń `Overloads` modyfikator z wszystkich właściwości i metody o tej samej nazwie.
