# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;Metoda&gt; "&lt;methodname1&gt;" musi być zadeklarowany jako "Overloads" ponieważ inny "&lt;methodname2&gt;" jest zadeklarowany jako "Overloads"
Inną właściwość lub metoda o takiej samej nazwie jest zadeklarowana za pomocą `Overloads` modyfikator.  
  
 **Identyfikator błędu:** BC31409  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `Overloads` modyfikatora właściwości lub metody.  
  
2.  Usuń `Overloads` modyfikator z wszystkich właściwości i metody o tej samej nazwie.
