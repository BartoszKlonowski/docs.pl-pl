# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;Metoda&gt; &#39; &lt;methodname1&gt; &#39; musi być zadeklarowany jako &#39;Overloads&#39; ponieważ innego &#39; &lt;methodname2&gt; &#39; zadeklarowano &#39; Przeciążenia&#39;
Innej właściwości lub metody o tej samej nazwie jest zadeklarowany za pomocą `Overloads` modyfikator.  
  
 **Identyfikator błędu:** BC31409  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `Overloads` modyfikator do właściwości lub metody.  
  
2.  Usuń `Overloads` modyfikator z wszystkich właściwości i metody o tej samej nazwie.
