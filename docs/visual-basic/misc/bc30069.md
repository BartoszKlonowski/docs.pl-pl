# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Aby uzyskać zmienna sterująca pętli "\<nazwa_zmiennej >' już w użyciu przez otaczającą pętlę For
Zagnieżdżone `For` pętli wszystkich musi używać zmiennych unikatową kontroli.  
  
 **Identyfikator błędu:** BC30069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana nazwy zmiennej kontrolki.
