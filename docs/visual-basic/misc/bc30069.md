# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Zmienna sterująca pętli for \<variablename> jest już używana przez otaczającą pętlę for

Zagnieżdżone `For` pętle All muszą używać unikatowych zmiennych kontrolnych.  
  
 **Identyfikator błędu:** BC30069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień jedną z nazw zmiennych sterujących.
