# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>Zmienna sterująca pętli for &#39; &lt;nazwa_zmiennej&gt;&#39; już używana przez otaczającą pętlę for
Zagnieżdżone `For` pętli wszystkich musi używać zmiennych unikatowy formantu.  
  
 **Identyfikator błędu:** BC30069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana nazwy zmiennej formantu.
