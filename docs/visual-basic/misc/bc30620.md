# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>Po znaku "Option Strict" może występować tylko polecenie "on" lub "off"

`Option Strict` instrukcje muszą określać `On` lub `Off` . Wszystkie inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30620  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ albo `On` `Off` dla `Option Strict` instrukcji.
