# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>"Option Strict" może następować tylko "On" lub "Off"
`Option Strict` należy określić instrukcji `On` lub `Off`. Wszystkie inne wartości są nieprawidłowe.  
  
 **Identyfikator błędu:** BC30620  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wybierz opcję `On` lub `Off` dla `Option Strict` instrukcji.
