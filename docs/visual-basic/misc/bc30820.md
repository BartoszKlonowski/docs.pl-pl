# <a name="lsetstmt-is-not-declared"></a>element " \<lsetstmt> " nie jest zadeklarowany

element " \<lsetstmt> " nie jest zadeklarowany. Instrukcje LSet nie są już obsługiwane; Zamiast tego należy użyć Microsoft. VisualBasic. LSet.  
  
 Kilka funkcji, które były wewnętrznie Visual Basic w poprzednich wersjach, zostały przeniesione do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcjonalność jest bardziej ogólnie dostępna dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30820  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `LSet`Zamiast tego użyj funkcji w `Microsoft.VisualBasic` przestrzeni nazw.
