# <a name="lsetstmt-is-not-declared"></a>'\<lsetstmt >' nie jest zadeklarowana
'\<lsetstmt >' nie został zadeklarowany. Instrukcje LSet nie są już obsługiwane; Zamiast tego użyj Microsoft.VisualBasic.LSet.  
  
 Kilka funkcji, które były wewnętrzne w języku Visual Basic w poprzednich wersjach zostały przeniesione do ikony <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. To sprawia, że ich funkcjonalność bardziej ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30820  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `LSet` działa w programach `Microsoft.VisualBasic` przestrzeni nazw zamiast tego.
