# <a name="39ltlsetstmtgt39-is-not-declared"></a>&#39; &lt;lsetstmt&gt;&#39; nie jest zadeklarowana
"\<lsetstmt >' nie został zadeklarowany. Instrukcje LSet nie są już obsługiwane; Zamiast tego użyj Microsoft.VisualBasic.LSet.  
  
 Kilka funkcji, które były wewnętrznej do [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] w poprzednich wersjach zostały przeniesione do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> przestrzeni nazw. Dzięki temu ich funkcji ogólnie dostępne dla wszystkich języków programowania.  
  
 **Identyfikator błędu:** BC30820  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `LSet` działać w `Microsoft.VisualBasic` przestrzeni nazw zamiast tego.
