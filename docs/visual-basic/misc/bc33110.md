# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="ca192-102">Nie można wywnioskować wspólnego typu dla pierwszego i drugiego operandu binarnego operatora "If"</span><span class="sxs-lookup"><span data-stu-id="ca192-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="ca192-103">Nie można wywnioskować wspólnego typu dla pierwszego i drugiego operandu binarnego operatora "If".</span><span class="sxs-lookup"><span data-stu-id="ca192-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="ca192-104">Jeden z nich musi mieć konwersję rozszerzającą na drugą.</span><span class="sxs-lookup"><span data-stu-id="ca192-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="ca192-105">Operator binarny `If` wymaga, aby istnieje konwersja rozszerzająca między jednym z argumentów a innym argumentem.</span><span class="sxs-lookup"><span data-stu-id="ca192-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="ca192-106">Na przykład, ponieważ nie istnieje konwersja rozszerzająca w obu kierunkach między `Integer` i `String` , poniższy kod powoduje wystąpienie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="ca192-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="ca192-107">**Identyfikator błędu:** BC33110</span><span class="sxs-lookup"><span data-stu-id="ca192-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="ca192-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="ca192-108">To correct this error</span></span>

- <span data-ttu-id="ca192-109">Zapewnianie jawnej konwersji jednego z operandów, jeśli jest to możliwe w kodzie:</span><span class="sxs-lookup"><span data-stu-id="ca192-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="ca192-110">Napisz ponownie kod przy użyciu innej konstrukcji warunkowej.</span><span class="sxs-lookup"><span data-stu-id="ca192-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
