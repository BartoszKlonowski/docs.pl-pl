# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-39if39-operator"></a><span data-ttu-id="a80b6-102">Nie można wywnioskować wspólnego typu dla pierwszego i drugiego argumentu operacji binarnego &#39;Jeśli&#39; — operator</span><span class="sxs-lookup"><span data-stu-id="a80b6-102">Cannot infer a common type for the first and second operands of the binary &#39;If&#39; operator</span></span>
<span data-ttu-id="a80b6-103">Nie można wywnioskować wspólnego typu dla pierwszego i drugiego argumentu operacji operatora binarnego "If".</span><span class="sxs-lookup"><span data-stu-id="a80b6-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="a80b6-104">Jeden musi mieć konwersję rozszerzającą do drugiego.</span><span class="sxs-lookup"><span data-stu-id="a80b6-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="a80b6-105">Plik binarny `If` operator wymaga istnienia konwersję rozszerzającą między jeden z argumentów i innych argumentu.</span><span class="sxs-lookup"><span data-stu-id="a80b6-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="a80b6-106">Na przykład ponieważ nie jest konwersję rozszerzającą w obu kierunkach między `Integer` i `String`, poniższy kod przyczyny tego błędu.</span><span class="sxs-lookup"><span data-stu-id="a80b6-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>  
  
```vb  
Dim first? As Integer  
Dim second As String = "First is Nothing"  
'' Not valid.  
' Console.WriteLine(If(first, second))  
```  
  
 <span data-ttu-id="a80b6-107">**Identyfikator błędu:** BC33110</span><span class="sxs-lookup"><span data-stu-id="a80b6-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a80b6-108">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="a80b6-108">To correct this error</span></span>  
  
-   <span data-ttu-id="a80b6-109">Przewidują jawnej konwersji jeden z argumentów, jeżeli jest to możliwe w kodzie:</span><span class="sxs-lookup"><span data-stu-id="a80b6-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>  
  
    ```  
    Console.WriteLine(If(first, CInt(second)))   
    ```  
  
-   <span data-ttu-id="a80b6-110">Należy zmodyfikować kod przy użyciu różnych konstrukcji warunkowego.</span><span class="sxs-lookup"><span data-stu-id="a80b6-110">Rewrite the code by using a different conditional construction.</span></span>  
  
    ```  
    If first IsNot Nothing Then  
        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```
