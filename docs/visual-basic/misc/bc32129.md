# <a name="enum-types-cannot-be-nullable"></a>Typów wyliczanych nie można anulować
Typ podstawowy, który jest używany do deklarowania wyliczenia nie może być typem wartości nullable. Na przykład następujący kod powoduje ten błąd:  
  
```vb  
'' Not valid.  
' Enum exampleEnum As Integer?  
'     Member declarations.  
' End Enum  
```  
  
 **Identyfikator błędu:** Bc32129  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie należy używać typu wartości możliwej do `Enum` wartości null jako typu bazowego w deklaracji.
