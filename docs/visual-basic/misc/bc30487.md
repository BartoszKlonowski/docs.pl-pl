# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>Operator &#39; &lt;operatorname&gt; &#39; nie jest zdefiniowany dla typu &#39; &lt;typename&gt;&#39;
Podjęto próbę użycia operatora jednoargumentowego (`+`, `-`, lub `Not`) na typ, takich jak `String`, dla której nie jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30487  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp operator Jednoargumentowy operator odpowiednie dla wyrażenia.
