# <a name="operator-operatorname-is-not-defined-for-type-typename"></a>Operator " \<operatorname> " nie jest zdefiniowany dla typu " \<typename> "
Podjęto próbę użycia operatora jednoargumentowego ( `+` , `-` , lub `Not` ) dla typu, takiego jak `String` , dla którego nie jest zdefiniowany.  
  
 **Identyfikator błędu:** BC30487  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zamień operator jednoargumentowy na odpowiedni operator dla wyrażenia.
