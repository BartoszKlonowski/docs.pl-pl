# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Zagnieżdżanie nawiasów na liście parametrów wyrażenia lambda przekracza maksymalną dozwoloną Głębokość
Zagnieżdżone nawiasy na liście parametrów wyrażenia lambda nie mogą przekroczyć głębokości 128.  
  
 **Identyfikator błędu:** BC36664
