# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Zagnieżdżanie nawiasów listy parametrów wyrażenia lambda przekracza maksymalną dozwoloną głębokość
Zagnieżdżone nawiasy listy parametrów wyrażenia lambda nie może przekraczać głębokość 128.  
  
 **Identyfikator błędu:** BC36664
