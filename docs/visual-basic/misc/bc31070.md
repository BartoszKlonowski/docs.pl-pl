# <a name="class-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Nie można zadeklarować klasy w interfejsie jako "&lt;specyfikator&gt;"
Nieprawidłowy modyfikator została określona za pomocą klasy zadeklarowanych w interfejsie.  
  
 **Identyfikator błędu:** BC31070  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikatorów, takich jak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, lub `Overridable`.
