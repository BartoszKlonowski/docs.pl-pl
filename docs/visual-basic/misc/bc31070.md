# <a name="class-in-an-interface-cannot-be-declared-specifier"></a>Nie można zadeklarować klasy w interfejsie jako " \<specifier> "
Określono nieprawidłowy modyfikator z klasą zadeklarowaną w interfejsie.  
  
 **Identyfikator błędu:** BC31070  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikatory, takie jak,,,,, `Public` `Private` `Shared` `Friend` `Protected` `Protected Friend` lub `Overridable` .
