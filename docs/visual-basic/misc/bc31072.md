# <a name="warning-treated-as-error-lterrorgt"></a>Ostrzeżenie potraktowano jako błąd: &lt;błąd&gt;
Ostrzeżenie podczas uruchamiania zestawu skompilowanego z **Traktuj ostrzeżenia jako błędy** włączona opcja.  
  
 **Identyfikator błędu:** BC31072  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Popraw ostrzeżenia.  
  
2.  Kompilowanie zestawu z **Traktuj ostrzeżenia jako błędy** opcja jest wyłączona.
