# <a name="warning-treated-as-error-error"></a>Ostrzeżenie potraktowano jako błąd: \<error>
Napotkano ostrzeżenie podczas wykonywania zestawu skompilowanego z włączoną opcją **Traktuj ostrzeżenia jako błędy** .  
  
 **Identyfikator błędu:** BC31072  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Popraw ostrzeżenie.  
  
2. Kompiluj zestaw z wyłączoną opcją **Traktuj ostrzeżenia jako błędy** .
