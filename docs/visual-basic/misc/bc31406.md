# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>element " \<methodname> " nie może przesłonić metody, która została obsłonięta

Metoda podjęła próbę zastąpienia metody, która zasłania inny element członkowski.  
  
 **Identyfikator błędu:** BC31406  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `Shadows` modyfikatora, aby zastąpić elementy członkowskie w tle.
