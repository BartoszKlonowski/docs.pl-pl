# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39; &lt;methodname&gt;&#39; nie można zastąpić metodę, która została przesłonięta
Metoda próbował zastąpić metodę, któremu innego elementu członkowskiego.  
  
 **Identyfikator błędu:** BC31406  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Shadows` modyfikator do zastąpienia członków zasłonięty.
