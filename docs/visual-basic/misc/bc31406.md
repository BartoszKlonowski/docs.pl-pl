# <a name="ltmethodnamegt-cannot-override-a-method-that-has-been-shadowed"></a>"&lt;methodname&gt;" nie można przesłonić metody, która została przesłonięta
Metoda próbował zastąpić metodę, która zasłania innego członka.  
  
 **Identyfikator błędu:** BC31406  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Użyj `Shadows` modyfikator zastąpić tekst z cieniem elementów członkowskich.
