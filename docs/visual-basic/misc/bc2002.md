# <a name="source-file-filename-specified-multiple-times"></a>Plik źródłowy \<nazwa pliku > określony wiele razy
Plik została określona wiele razy.  
  
 **Identyfikator błędu:** BC2002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nadmiarowe pliku specyfikatorów.
