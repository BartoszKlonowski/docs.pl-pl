# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Plik źródłowy &lt;filename&gt; określony wiele razy

 **Identyfikator błędu:** BC2002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowe pliku specyfikatory.
