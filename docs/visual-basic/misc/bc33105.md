# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="f6354-102">Operandy "If" nie mogą być argumentami nazwanymi</span><span class="sxs-lookup"><span data-stu-id="f6354-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="f6354-103">Używanie nazwanych argumentów w argumentach operacji operatora `If` jest nieprawidłowe.</span><span class="sxs-lookup"><span data-stu-id="f6354-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="f6354-104">Poniższy przykład powoduje wystąpienie tego błędu:</span><span class="sxs-lookup"><span data-stu-id="f6354-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="f6354-105">Różni się to od funkcji `IIf`, która zezwala na nazwane argumenty, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="f6354-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="f6354-106">**Identyfikator błędu:** BC33105</span><span class="sxs-lookup"><span data-stu-id="f6354-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f6354-107">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="f6354-107">To correct this error</span></span>

    ```vb  
    result = If(i > 0, "positive", "not positive")  
    ```
