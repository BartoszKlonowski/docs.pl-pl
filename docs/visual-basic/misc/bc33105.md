# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="3a69e-102">Operandy "If" nie może mieć nazwy argumentów</span><span class="sxs-lookup"><span data-stu-id="3a69e-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="3a69e-103">Za pomocą nazwanych argumentów argumenty operacji `If` operator jest nieprawidłowy.</span><span class="sxs-lookup"><span data-stu-id="3a69e-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="3a69e-104">Poniższy przykład powoduje, że ten błąd:</span><span class="sxs-lookup"><span data-stu-id="3a69e-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="3a69e-105">To różni się od `IIf` funkcji, która zezwala na o nazwie argumentów, jak pokazano w poniższym kodzie:</span><span class="sxs-lookup"><span data-stu-id="3a69e-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="3a69e-106">**Identyfikator błędu:** BC33105</span><span class="sxs-lookup"><span data-stu-id="3a69e-106">**Error ID:** BC33105</span></span>

    ```  
    result = If(i > 0, "positive", "not positive")  
    ```
