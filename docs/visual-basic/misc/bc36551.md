## <a name="example"></a><span data-ttu-id="5a711-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="5a711-108">Example</span></span>

```  
Imports System.Runtime.CompilerServices  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _  
        Public Sub Print (ByVal str As String)  
            Console.WriteLine(str)  
        End Sub  
    End Module  
End Namespace  
```
