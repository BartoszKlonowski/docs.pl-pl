## <a name="example"></a><span data-ttu-id="02256-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="02256-108">Example</span></span>

```vb  
Imports System.Runtime.CompilerServices  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _  
        Public Sub Print (ByVal str As String)

        End Sub  
    End Module  
End Namespace  
```
