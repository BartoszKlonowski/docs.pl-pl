# <a name="set-parameter-cannot-be-declared-specifier"></a>Nie można zadeklarować parametru "Set" jako " \<specifier> "

Określono nieprawidłowy modyfikator z parametrem w `Set` procedurze właściwości.  
  
 **Identyfikator błędu:** BC31065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuwanie nieprawidłowych modyfikatorów, takich jak `ByRef` , `ParamArray` , i `Optional` .
