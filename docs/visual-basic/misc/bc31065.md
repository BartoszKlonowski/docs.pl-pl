# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39; Ustaw &#39; Nie można zadeklarować parametru &#39; &lt;specyfikator&gt;&#39;
Określono nieprawidłowy modyfikator z parametrem w `Set` procedury właściwości.  
  
 **Identyfikator błędu:** BC31065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowe modyfikatory, takich jak `ByRef`, `ParamArray`, i `Optional`.
