# <a name="set-parameter-cannot-be-declared-specifier"></a>Parametr 'Set' nie można zadeklarować jako '\<specyfikator >'
Określono nieprawidłowy modyfikator z parametrem `Set` procedury właściwości.  
  
 **Identyfikator błędu:** BC31065  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe modyfikatory, takich jak `ByRef`, `ParamArray`, i `Optional`.
