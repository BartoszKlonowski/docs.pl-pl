# <a name="shared-attribute-property-ltpropertyfieldgt-cannot-be-the-target-of-an-assignment"></a>"Shared" atrybut właściwości "&lt;propertyfield&gt;" nie może być elementem docelowym przypisania
Nastąpiła próba do przypisania wartości do `ReadOnly` lub `Shared` właściwości w atrybucie.  
  
 **Identyfikator błędu:** BC31500  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń instrukcję przypisania właściwości.  
  
2.  Jeśli za pomocą właściwości opracowany Usuń `ReadOnly` lub `Shared` Modyfikatory z właściwości atrybutów.
