# <a name="shared-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>'Shared' atrybut właściwości '\<propertyfield >' nie może być elementem docelowym przypisania
Podjęto próbę przypisania wartości do `ReadOnly` właściwości lub `Shared` w atrybucie.  
  
 **Identyfikator błędu:** BC31500  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń instrukcję przypisania właściwości.  
  
2. W przypadku używania właściwości, które zostały opracowane `ReadOnly` , `Shared` Usuń Modyfikatory or z właściwości atrybutu.
