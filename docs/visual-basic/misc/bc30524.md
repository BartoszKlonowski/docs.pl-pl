# <a name="property-propertyname-is-writeonly"></a>Property '\<propertyname>' is 'WriteOnly'
Nastąpiła próba pobrania wartości z właściwością, która jest zadeklarowana `WriteOnly`.  
  
 **Identyfikator błędu:** BC30524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `WriteOnly` specyfikator w deklaracji właściwości.
