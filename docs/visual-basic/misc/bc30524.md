# <a name="property-39ltpropertynamegt39-is-39writeonly39"></a>Właściwości &#39; &lt;propertyname&gt;&#39; jest &#39; WriteOnly &#39;
Nastąpiła próba pobrania wartości z właściwości, która jest zadeklarowana jako `WriteOnly`.  
  
 **Identyfikator błędu:** BC30524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `WriteOnly` specyfikator w deklaracji właściwości.
