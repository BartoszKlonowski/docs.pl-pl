# <a name="property-propertyname-is-writeonly"></a>Właściwość " \<propertyname> " jest typu "WriteOnly"
Podjęto próbę pobrania wartości z właściwości, która została zadeklarowana `WriteOnly` .  
  
 **Identyfikator błędu:** BC30524  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `WriteOnly` specyfikator z deklaracji właściwości.
