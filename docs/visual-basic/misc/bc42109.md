# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Zmienna " \<variablename> " jest używana, zanim zostanie do niej przypisana wartość (Visual Basic błąd)
Zmienna jest używana, zanim zostanie do niej przypisana wartość, a wynikiem będzie <xref:System.NullReferenceException> .  
  
 **Identyfikator błędu:** BC42109  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że zmienna ma przypisaną wartość.
