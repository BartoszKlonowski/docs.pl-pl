# <a name="variable-39ltvariablenamegt39-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Zmienna &#39; &lt;nazwa_zmiennej&gt; &#39; jest używana, zanim została do niej przypisana wartość (Błąd Visual Basic)
Zmienna jest używana, zanim została do niej przypisana wartość wynikową w <xref:System.NullReferenceException>.  
  
 **Identyfikator błędu:** BC42109  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że zmienna zostanie przypisana wartość.
