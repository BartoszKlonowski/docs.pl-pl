# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Zmienna "\<nazwa_zmiennej >" jest używana, zanim została do niej przypisana wartość (błąd w Visual Basic)
Zmienna jest używany, zanim została do niej przypisana wartość, co powodowało <xref:System.NullReferenceException>.  
  
 **Identyfikator błędu:** BC42109  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że zmienna zostanie przypisana wartość.
