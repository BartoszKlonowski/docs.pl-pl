# <a name="39structure39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39; Struktura &#39; ograniczenia nie można określić wiele razy dla tego samego parametru typu
Lista ograniczeń zawiera [struktury (Visual Basic)](http://msdn.microsoft.com/en-us/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) ograniczenia więcej niż raz.  
  
 Listę ograniczenia dla parametru typu można określić, że typ argumentu przekazanego do parametru typu musi być typem wartości (z `Structure` ograniczenie) lub musi być typem referencyjnym (z [klasy (Visual Basic)](http://msdn.microsoft.com/en-us/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) ograniczenia). Nie można określić zarówno ograniczenia dotyczące tego samego parametru typu i nie można określić jedną więcej niż raz.  
  
 Identyfikator błędu: BC32102  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowych `Structure` słów kluczowych. Powinien mieć tylko jeden na liście ograniczeń.
