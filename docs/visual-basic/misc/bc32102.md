# <a name="39structure39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;Struktura&#39; ograniczenia nie można określić wiele razy dla tego samego parametru typu
Lista ograniczeń zawiera [struktury (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) ograniczenie więcej niż jeden raz.  
  
 Lista ograniczeń dla parametru typu można określić, że argument typu przekazany do tego parametru typu musi być typem wartości (przy użyciu `Structure` ograniczenie) lub musi być typem referencyjnym (przy użyciu [klasy (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) ograniczenia). Nie można określić zarówno ograniczeń dla tego samego parametru typu, a nie można określić jedną więcej niż jeden raz.  
  
 Identyfikator błędu: BC32102  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nadmiarowych `Structure` słów kluczowych. Należy mieć tylko jeden na liście ograniczeń.
