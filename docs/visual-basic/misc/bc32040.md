# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>'\<— Słowo kluczowe >' słowo kluczowe jest używane do przeciążania dziedziczonych składowych; nie należy używać '\<— słowo kluczowe >' — słowo kluczowe przeciążania elementu 'Sub New'
Konstruktor jest zadeklarowany za pomocą `Overloads` słowa kluczowego.  
  
 Visual Basic nie obsługuje konstruktorów dziedziczenia ani przeciążenia.  
  
 **Identyfikator błędu:** BC32040  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `Overloads` Usuń słowo kluczowe ze wszystkich deklaracji konstruktora.
