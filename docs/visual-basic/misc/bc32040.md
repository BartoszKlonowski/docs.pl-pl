# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39; &lt;— Słowo kluczowe&gt; &#39; — słowo kluczowe służy do przeciążania dziedziczonych członków; nie używaj &#39; &lt;— słowo kluczowe&gt; &#39; — słowo kluczowe przeciążania &#39;Sub New&#39;
Konstruktor jest zadeklarowany za pomocą `Overloads` — słowo kluczowe.  
  
 Visual Basic nie obsługuje dziedziczenia lub przeładowania konstruktorów.  
  
 **Identyfikator błędu:** BC32040  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Overloads` — słowo kluczowe z wszystkie deklaracje konstruktora.
