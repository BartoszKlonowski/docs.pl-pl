# <a name="-is-not-allowed"></a>":" jest niedozwolone
":" jest niedozwolone. W tym kontekście nie można używać kwalifikowanych nazw XML.  
  
 Literał XML został określony z kwalifikowaną nazwą w nieprawidłowej lokalizacji, na przykład obiektem docelowym w literale instrukcji przetwarzania.  
  
 **Identyfikator błędu:** BC31186  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń literał XML z kwalifikowaną nazwą lub przenieś go do prawidłowej lokalizacji.  
  
     Aby uzyskać informacje o tym, które znaki są prawidłowe w nazwach XML, zobacz [XML (XML) 1,0](https://www.w3.org/TR/xml).
