# <a name="3939-is-not-allowed"></a>&#39;: &#39; nie jest dozwolona
":" nie jest dozwolone. XML kwalifikowanej nazwy nie mogą być używane w tym kontekście.  
  
 Literał XML został określony z nazwą kwalifikowaną w nieprawidłowej lokalizacji, na przykład obiektem docelowym instrukcji przetwarzania literału.  
  
 **Identyfikator błędu:** BC31186  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń literał o nazwie kwalifikowanej XML, lub przenieś go do prawidłowej lokalizacji.  
  
     Aby uzyskać informacje o tym, które są dozwolone w nazwach XML znaków, zobacz [języka XML (Extensible Markup) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927).
