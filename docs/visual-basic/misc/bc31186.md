# <a name="-is-not-allowed"></a>":" nie jest dozwolona
":" nie jest dozwolone. XML kwalifikowana nazwy nie mogą być używane w tym kontekście.  
  
 Literał XML został określony z nazwą kwalifikowaną w nieprawidłowej lokalizacji, na przykład obiektem docelowym instrukcji przetwarzania literału.  
  
 **Identyfikator błędu:** BC31186  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń literału o nazwie kwalifikowanej XML lub przenieś go do prawidłowej lokalizacji.  
  
     Aby uzyskać informacje o tym, jakie znaki są dozwolone w nazwach XML, zobacz [języka XML (Extensible Markup) 1.0](https://www.w3.org/TR/xml).
