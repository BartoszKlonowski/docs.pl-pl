# <a name="unable-to-open-module-file-39ltfilenamegt39-lterrorgt"></a>Nie można otworzyć pliku modułu &#39; &lt;filename&gt;&#39;: &lt;błąd&gt;
Nie można otworzyć pliku zawierającego modułu dla tego projektu.  
  
 **Identyfikator błędu:** BC31007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy plik jest w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
