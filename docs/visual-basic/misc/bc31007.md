# <a name="unable-to-open-module-file-filename-error"></a>Nie można otworzyć pliku modułu "\<nazwa pliku >": \<błąd >
Nie można otworzyć pliku zawierającego modułu dla tego projektu.  
  
 **Identyfikator błędu:** BC31007  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy plik znajduje się w określonej lokalizacji i nie jest zablokowany przez inną aplikację.
