# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'\<nazwaprocedury >' ma zwracany typ, który nie jest obsługiwany lub typami parametrów, które nie są obsługiwane
Procedura jest zadeklarowana za pomocą parametrów lub typ zwracany nie może być reprezentowana przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30657  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zdefiniuj typy zwracane lub parametry jako obsługiwanego typu.  
  
-   Tworzenie nowego typu, który zawiera opis danych używane przez tę procedurę.
