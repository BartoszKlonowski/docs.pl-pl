# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;nazwaprocedury&gt; &#39; ma zwracany typ, który nie jest obsługiwany lub typy parametrów, które nie są obsługiwane
Procedura jest zadeklarowana przy użyciu parametrów lub typ zwracany nie może być reprezentowany przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30657  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zdefiniuj typy zwracane lub parametry jako obsługiwanym typem.  
  
-   Tworzenie nowego typu, który opisuje dane używane przez tę procedurę.
