# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>element " \<procedurename> " ma typ zwracany, który nie jest obsługiwany lub typy parametrów, które nie są obsługiwane.
Procedura jest zadeklarowana z parametrami lub typem zwracanym, który nie może być reprezentowany przez program Visual Studio.  
  
 **Identyfikator błędu:** BC30657  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zdefiniuj typy zwracane lub parametry jako obsługiwane.  
  
- Utwórz nowy typ, który opisuje dane używane w tej procedurze.
