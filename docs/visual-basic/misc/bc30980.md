# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="dc1f8-102">Typ '\<nazwa_zmiennej >' nie można wywnioskować z wyrażenia zawierającego '\<nazwa_zmiennej >'</span><span class="sxs-lookup"><span data-stu-id="dc1f8-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>
<span data-ttu-id="dc1f8-103">Kompilator nie może wywnioskować typu danych zmiennej, jeśli zmienna jest używana w celu ustalenia jej początkowej wartości w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="dc1f8-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="dc1f8-104">Na przykład przy `Option Infer` ustawionym na `On` następujące przykłady nie są kompilowane:</span><span class="sxs-lookup"><span data-stu-id="dc1f8-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="dc1f8-105">Deklaracje</span><span class="sxs-lookup"><span data-stu-id="dc1f8-105">Declarations</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="dc1f8-106">Pętla `For`</span><span class="sxs-lookup"><span data-stu-id="dc1f8-106">`For` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="dc1f8-107">Pętla `For Each`</span><span class="sxs-lookup"><span data-stu-id="dc1f8-107">`For Each` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="dc1f8-108">**Identyfikator błędu:** BC30980</span><span class="sxs-lookup"><span data-stu-id="dc1f8-108">**Error ID:** BC30980</span></span>
