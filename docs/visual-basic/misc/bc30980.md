<span data-ttu-id="a1e1c-103">Kompilator nie można wywnioskować typu danych zmiennej, jeśli zmienna zostanie użyta w ustalaniu jej wartość początkową w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="a1e1c-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="a1e1c-104">Na przykład za pomocą `Option Infer` równa `On`, nie można skompilować następujące przykłady:</span><span class="sxs-lookup"><span data-stu-id="a1e1c-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="a1e1c-105">Deklaracje</span><span class="sxs-lookup"><span data-stu-id="a1e1c-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="a1e1c-106">`For` pętli</span><span class="sxs-lookup"><span data-stu-id="a1e1c-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="a1e1c-107">`For Each` pętli</span><span class="sxs-lookup"><span data-stu-id="a1e1c-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="a1e1c-108">**Identyfikator błędu:** BC30980</span><span class="sxs-lookup"><span data-stu-id="a1e1c-108">**Error ID:** BC30980</span></span>
