<span data-ttu-id="8b004-103">Kompilator nie można wywnioskować typu danych zmiennej, jeśli zmienna zostanie użyta w ustalaniu jej wartość początkową w deklaracji.</span><span class="sxs-lookup"><span data-stu-id="8b004-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="8b004-104">Na przykład za pomocą `Option Infer` równa `On`, nie można skompilować następujące przykłady:</span><span class="sxs-lookup"><span data-stu-id="8b004-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>

    ```  
    ' Does not compile with Option Infer on.

    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="8b004-106">`For` pętli</span><span class="sxs-lookup"><span data-stu-id="8b004-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="8b004-107">`For Each` pętli</span><span class="sxs-lookup"><span data-stu-id="8b004-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.

    Next  
    ```  
  
 <span data-ttu-id="8b004-108">**Identyfikator błędu:** BC30980</span><span class="sxs-lookup"><span data-stu-id="8b004-108">**Error ID:** BC30980</span></span>
