# <a name="ltmembernamegt-cannot-be-declared-shadows-outside-of-a-class-structure-or-interface"></a>"&lt;membername&gt;" nie można zadeklarować jako "Shadows" poza klasy, struktury lub interfejsu
`Shadows` — Słowo kluczowe pojawia się w deklaracji w przestrzeni nazw, modułu lub na poziomie pliku. Przesłanianie ma znaczenie tylko w obrębie elementu programistycznego, który może dziedziczyć z elementu podstawowego.  
  
 **Identyfikator błędu:** BC32200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Shadows` — słowo kluczowe lub Przenieś deklaracja wewnątrz klasy, struktury lub interfejsu.
