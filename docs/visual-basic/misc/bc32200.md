# <a name="39ltmembernamegt39-cannot-be-declared-39shadows39-outside-of-a-class-structure-or-interface"></a>&#39;&lt;membername&gt; &#39; nie można zadeklarować &#39;Shadows&#39; poza klasy, struktury lub interfejsu
`Shadows` — Słowo kluczowe jest wyświetlana w deklaracji na poziomie pliku, modułu lub przestrzeni nazw. Przesłanianie jest znaczący tylko w obrębie elementu programistycznego, który może dziedziczyć z elementu podstawowego.  
  
 **Identyfikator błędu:** BC32200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Shadows` — słowo kluczowe lub Przenieś deklarację wewnątrz klasy, struktury lub interfejsu.
