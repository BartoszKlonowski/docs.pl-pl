# <a name="membername-cannot-be-declared-shadows-outside-of-a-class-structure-or-interface"></a>\<membername>nie można zadeklarować elementu "" jako "Shadows" poza klasą, strukturą lub interfejsem

`Shadows`Słowo kluczowe pojawia się w deklaracji na poziomie przestrzeni nazw, modułu lub pliku. Cieniowanie ma znaczenie tylko w obrębie elementu programistycznego, który może dziedziczyć po elemencie podstawowym.  
  
 **Identyfikator błędu:** BC32200  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Shadows` słowo kluczowe lub Przenieś deklarację wewnątrz klasy, struktury lub interfejsu.
