# <a name="custom-event-must-end-with-a-matching-end-event"></a>Element "Custom Event" musi być zakończony odpowiadającym mu elementem "End Event"
`Custom Event`Deklaracja musi kończyć się `End Event` instrukcją.  
  
 **Identyfikator błędu:** BC31114  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `Custom Event` Deklaracja została zakończona `End Event` instrukcją.
