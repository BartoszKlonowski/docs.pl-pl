# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Zdarzenie niestandardowe&#39; musi być zakończona odpowiadającą jej instrukcją &#39;zakończenia zdarzenia&#39;
A `Custom Event` deklaracji musi kończyć się `End Event` instrukcji.  
  
 **Identyfikator błędu:** BC31114  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `Custom Event` kończy deklaracji `End Event` instrukcji.
