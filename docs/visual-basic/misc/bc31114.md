# <a name="custom-event-must-end-with-a-matching-end-event"></a>"Zdarzenie niestandardowe" musi kończyć się pasującego "End Event"
A `Custom Event` deklaracji musi się kończyć `End Event` instrukcji.  
  
 **Identyfikator błędu:** BC31114  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `Custom Event` deklaracji kończy się `End Event` instrukcji.
