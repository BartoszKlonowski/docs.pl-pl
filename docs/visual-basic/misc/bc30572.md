# <a name="alias-namespacename-is-already-declared"></a>Alias\<namespacename > "jest już zadeklarowana
Podjęto próbę użycia `Imports` do importowania przestrzeni nazw, która jest już zadeklarowany.  
  
 **Identyfikator błędu:** BC30572  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź obszar nazw, którą próbujesz zaimportować.
