# <a name="alias-namespacename-is-already-declared"></a>Alias " \<namespacename> " jest już zadeklarowany
Podjęto próbę użycia `Imports` do zaimportowania przestrzeni nazw, która jest już zadeklarowana.  
  
 **Identyfikator błędu:** BC30572  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź przestrzeń nazw, którą próbujesz zaimportować.
