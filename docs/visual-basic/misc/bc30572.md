# <a name="alias-39ltnamespacenamegt39-is-already-declared"></a>Alias &#39; &lt;namespacename&gt; &#39; jest już zadeklarowana
Podjęto próbę użycia `Imports` do importowania przestrzeni nazw, która jest już zadeklarowany.  
  
 **Identyfikator błędu:** BC30572  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź obszar nazw, który próbujesz zaimportować.
