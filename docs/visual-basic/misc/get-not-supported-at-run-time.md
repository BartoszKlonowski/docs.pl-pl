# <a name="get-not-supported-at-run-time"></a>Pobierz nieobsługiwane w czasie wykonywania
Próbowano odczytać właściwości w czasie wykonywania, który jest dostępny tylko w czasie projektowania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź właściwości, a następnie określić pod jakimi warunkami może zostać ustawiony.  
  
2.  Usuń odwołanie do właściwości.
