# <a name="get-not-supported-at-run-time"></a>Pobieranie nie jest obsługiwane w czasie wykonywania

Podjęto próbę odczytu właściwości w czasie wykonywania, która jest dostępna tylko w czasie projektowania.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź Właściwość i Określ warunki, które mogą być ustawione.  
  
2. Usuń odwołanie do właściwości.
