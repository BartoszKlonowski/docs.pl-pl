# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39; przejdź do &#39; i &#39; na GoSub &#39; instrukcje są już obsługiwane.
Nie jest już można użyć wartości zmiennej lub wyrażenie z `GoTo` i `GoSub` instrukcje do kontroli programu przepływu.  
  
 **Identyfikator błędu:** BC30817  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja aplikacji umożliwiająca użycie `If...Then...Else` lub `Case` instrukcje.
