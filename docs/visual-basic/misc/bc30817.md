# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39;W instrukcji GoTo&#39; i &#39;na GoSub&#39; instrukcje nie są już obsługiwane
Nie jest już prawidłowe użycie wartości zmiennej lub wyrażenia `GoTo` i `GoSub` instrukcji sterowania programu przepływu.  
  
 **Identyfikator błędu:** BC30817  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Restrukturyzacja aplikacji umożliwiająca użycie `If...Then...Else` lub `Case` instrukcji.
