# <a name="set-method-cannot-have-more-than-one-parameter"></a>Metoda "Set" nie może mieć więcej niż jednego parametru

Zdefiniowano więcej niż jeden parametr dla procedur właściwości `Set` .  
  
 **Identyfikator błędu:** BC31063  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wszystkie dodatkowe parametry z `Set` procedur właściwości.
