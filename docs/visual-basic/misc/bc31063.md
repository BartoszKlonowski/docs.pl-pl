# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39;Ustaw&#39; metoda nie może mieć więcej niż jeden parametr
Więcej niż jeden parametr jest zdefiniowany dla właściwości `Set` procedury.  
  
 **Identyfikator błędu:** BC31063  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszelkie dodatkowe parametry z właściwości `Set` procedury.
