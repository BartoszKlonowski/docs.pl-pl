# <a name="sub-new-cannot-be-declared-modifier"></a>'Sub New' nie można zadeklarować jako '\<modyfikator >'
Podjęto próbę zadeklarować `Sub New` za pomocą modyfikatora, takich jak `Overrides`, która jest nieprawidłowy.  
  
 **Identyfikator błędu:** BC30364  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
