# <a name="sub-new-cannot-be-declared-modifier"></a>Nie można zadeklarować element "Sub New" jako " \<modifier> "

Podjęto próbę zadeklarować `Sub New` z modyfikatorem, np `Overrides` ., który jest nieprawidłowy.  
  
 **Identyfikator błędu:** BC30364  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator.
