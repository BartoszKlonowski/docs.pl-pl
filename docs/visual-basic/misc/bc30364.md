# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub New&#39; nie można zadeklarować &#39; &lt;modyfikator&gt;&#39;
Podjęto próbę zadeklarować `Sub New` modyfikatorem, takich jak `Overrides`, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30364  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikator.
