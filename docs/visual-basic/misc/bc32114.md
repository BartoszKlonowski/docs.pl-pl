# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="6797a-102">Typ parametru "Structure" ograniczenie nie może być używane jako ograniczenia</span><span class="sxs-lookup"><span data-stu-id="6797a-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="6797a-103">Parametrem typu `Structure` warunek ograniczający jest używany jako ograniczenie dla innego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="6797a-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="6797a-104">`Structure` Ograniczenie wymaga argumentów typu jako parametr typu typu wartości.</span><span class="sxs-lookup"><span data-stu-id="6797a-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="6797a-105">Jednak typ wartości nie można zaimplementować lub dziedziczone, więc nie ma sensu będzie używany jako ograniczenie, które wymagałyby inne parametry typu do jej wdrożenia lub dziedziczyć po nim.</span><span class="sxs-lookup"><span data-stu-id="6797a-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="6797a-106">Dotyczy tylko interpretacja tej sytuacji jest, że oba argumenty typu musi być dokładnie tego samego typu.</span><span class="sxs-lookup"><span data-stu-id="6797a-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="6797a-107">Jeśli tak jest rzeczywiście typu rodzajowego wymaga tylko jeden parametr typu.</span><span class="sxs-lookup"><span data-stu-id="6797a-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="6797a-108">Poniższa instrukcja może wygenerować tego błędu.</span><span class="sxs-lookup"><span data-stu-id="6797a-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="6797a-109">Typ przekazywany do `t2` nie może implementować lub dziedziczyć z typu przekazany do `t1`, ponieważ typ przekazywany do `t1` musi być typem wartości.</span><span class="sxs-lookup"><span data-stu-id="6797a-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="6797a-110">**Identyfikator błędu:** BC32114</span><span class="sxs-lookup"><span data-stu-id="6797a-110">**Error ID:** BC32114</span></span>

- <span data-ttu-id="6797a-113">Jeśli oba parametry typu wymaga ten sam typ wartości, należy zdefiniować typu ogólnego przy użyciu tylko jednego typu parametru.</span><span class="sxs-lookup"><span data-stu-id="6797a-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
