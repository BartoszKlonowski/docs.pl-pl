# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="31fee-102">Parametru typu z &#39;struktury&#39; ograniczenia nie można użyć jako ograniczenia</span><span class="sxs-lookup"><span data-stu-id="31fee-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="31fee-103">Parametru typu z `Structure` warunek ograniczający jest używany jako ograniczenie dla parametru innego typu.</span><span class="sxs-lookup"><span data-stu-id="31fee-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="31fee-104">`Structure` Ograniczenia wymaga, aby typ argumentu przekazywanego do jego typ parametru typu wartości.</span><span class="sxs-lookup"><span data-stu-id="31fee-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="31fee-105">Jednak typu wartości nie zaimplementowano lub dziedziczone, więc nie ma sensu, aby używać go jako ograniczenie, która wymaga parametru typu ją wdrożyć lub dziedziczyć po nim.</span><span class="sxs-lookup"><span data-stu-id="31fee-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="31fee-106">Tylko istotne interpretacji tej sytuacji jest, że oba argumenty typu musi być dokładnie tego samego typu.</span><span class="sxs-lookup"><span data-stu-id="31fee-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="31fee-107">Jeśli tak jest, Twoje typu ogólnego wymaga tylko jednego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="31fee-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="31fee-108">Następująca instrukcja może spowodować wygenerowanie tego błędu.</span><span class="sxs-lookup"><span data-stu-id="31fee-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="31fee-109">Typ przekazany do `t2` nie może implementować lub dziedziczyć typ przekazany do `t1`, ponieważ typ przekazany do `t1` musi być typem wartości.</span><span class="sxs-lookup"><span data-stu-id="31fee-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="31fee-110">**Identyfikator błędu:** BC32114</span><span class="sxs-lookup"><span data-stu-id="31fee-110">**Error ID:** BC32114</span></span>
