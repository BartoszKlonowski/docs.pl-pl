# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="eb3c2-102">Parametru typu z ograniczeniem "Structure" nie można użyć jako ograniczenia</span><span class="sxs-lookup"><span data-stu-id="eb3c2-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>

<span data-ttu-id="eb3c2-103">Parametr typu z `Structure` ograniczeniem jest używany jako ograniczenie dla innego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="eb3c2-104">`Structure`Ograniczenie wymaga, aby argument typu przesłany do jego parametru typu był typem wartości.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="eb3c2-105">Jednak typ wartości nie może być zaimplementowany ani Dziedziczony, dlatego nie ma znaczenia, aby użyć go jako ograniczenia, co wymagałoby, aby inny parametr typu mógł go zaimplementować lub dziedziczyć po nim.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="eb3c2-106">Jedyną istotną interpretacją tej sytuacji jest to, że oba argumenty typu muszą być dokładnie tego samego typu.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="eb3c2-107">W takim przypadku typ generyczny wymaga tylko jednego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="eb3c2-108">Poniższe instrukcje mogą generować ten błąd.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="eb3c2-109">Typ przesłany do `t2` nie może implementować lub dziedziczyć typu przesłanego do `t1` , ponieważ typ, który został przesłany do `t1` musi być typem wartości.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="eb3c2-110">**Identyfikator błędu:** BC32114</span><span class="sxs-lookup"><span data-stu-id="eb3c2-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="eb3c2-111">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="eb3c2-111">To correct this error</span></span>  
  
- <span data-ttu-id="eb3c2-112">Usuń parametr typu ograniczony do `Structure` z listy ograniczeń w innym parametrze typu.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="eb3c2-113">Jeśli oba parametry typu wymagają tego samego typu wartości, zdefiniuj typ ogólny tylko z jednym parametrem typu.</span><span class="sxs-lookup"><span data-stu-id="eb3c2-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
