# <a name="code-resource-lock-error"></a>Błąd blokowania zasobu kodu
Wykonano wywołanie do procedury w zasobie kodu. Odnaleziono zasobu kodu, ale wystąpił błąd, gdy nastąpiła próba może zablokować zasobu, ponieważ jest on używany.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź zasobu, aby ustalić przyczynę wystąpienia błędu.
