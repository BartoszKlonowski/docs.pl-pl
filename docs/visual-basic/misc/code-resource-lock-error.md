# <a name="code-resource-lock-error"></a>Błąd blokady zasobu kodu
Wykonano wywołanie procedury w zasobie kodu. Znaleziono zasobu kodu, ale wystąpił błąd podczas próbowano zablokować zasobu, być może, ponieważ jest on używany.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź zasobu, aby ustalić przyczynę wystąpienia błędu.
