# <a name="code-resource-lock-error"></a>Błąd blokady zasobu kodu
Wykonano wywołanie procedury w zasobie kodu. Znaleziono zasób kodu, ale wystąpił błąd podczas próby zablokowania zasobu, prawdopodobnie dlatego, że jest w użyciu.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź zasób, aby określić, dlaczego wystąpił błąd.
