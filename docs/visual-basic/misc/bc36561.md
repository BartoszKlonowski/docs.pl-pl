# <a name="extension-method-ltmethodnamegt-has-type-constraints-that-can-never-be-satisfied"></a><span data-ttu-id="cdd9f-102">Metoda rozszerzenia "&lt;methodname&gt;" ma ograniczenia typu, które nigdy nie może zostać wykonane</span><span class="sxs-lookup"><span data-stu-id="cdd9f-102">Extension method '&lt;methodname&gt;' has type constraints that can never be satisfied</span></span>
<span data-ttu-id="cdd9f-103">Parametry typu tej metody wchodzić w interakcje w sposób, który uniemożliwia nigdy nie są spełnione.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-103">The type parameters of this method interact in a way that prevents them from ever being satisfied.</span></span> <span data-ttu-id="cdd9f-104">Następujące metody rozszerzenia znajduje się przykład.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-104">The following extension method is an example.</span></span>  
  
```  
'' Not valid.  
'<Extension()> _  
'Sub extensionExample(Of T As U, U)(ByVal para1 As T, ByVal para2 As U)  
'End Sub  
```  
  
 <span data-ttu-id="cdd9f-105">Ponieważ metoda jest metodą rozszerzenia, kompilator musi być w stanie określić typ danych lub typy, które rozszerza metoda oparte tylko na pierwszy parametr w deklaracji metody `para1`, a argument wysyłane dla tego parametru.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-105">Because the method is an extension method, the compiler must be able to determine the data type or types that the method extends based only on the first parameter in the method declaration, `para1`, and the argument sent in for that parameter.</span></span> <span data-ttu-id="cdd9f-106">Pierwszy parametr, który odwołuje się do parametrów typu genetycznego `para1 as T`, warunki ograniczające w parametrach rodzajowych ograniczyć zestaw typów, do których zostanie zastosowana metoda.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-106">When the first parameter refers to generic type parameters, `para1 as T`, the constraints on the generic parameters restrict the set of types to which the method applies.</span></span>  
  
 <span data-ttu-id="cdd9f-107">Możliwość zastosowania metody rozszerzenia jest określana na podstawie argument podany jako pierwszy parametr, który jest `arg1` w poniższym kodzie.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-107">Applicability of an extension method is determined from the argument provided for the first parameter, which is `arg1` in the following code.</span></span>  
  
 `'' Not valid.`  
  
 `'arg1.extensionExample(arg2)`  
  
 <span data-ttu-id="cdd9f-108">Musi być możliwość sprawdzenia ograniczenia dotyczące wszystkich parametrów typu genetycznego określone za pomocą pierwszego parametru `para1`, analizując tylko pierwszy argument `arg1`.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-108">It must be possible to verify the constraints on all generic type parameters referred to by the first parameter, `para1`, by looking at only the first argument, `arg1`.</span></span> <span data-ttu-id="cdd9f-109">W `extensionExample`, zestaw typów, który jest rozszerzany nie może zostać określona tylko pierwszy parametr.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-109">In `extensionExample`, the set of types that is being extended cannot be determined from the first parameter alone.</span></span> <span data-ttu-id="cdd9f-110">Parametr typu `T` jest ograniczony przez parametr typu `U`, który nie odwołuje się `para1` i nie można wywnioskować z `arg1`.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-110">Type parameter `T` is constrained by type parameter `U`, which is not referenced by `para1` and cannot be inferred from `arg1`.</span></span> <span data-ttu-id="cdd9f-111">Dlatego nie można zweryfikować zastosowania metody do dowolnego typu możliwe i nigdy nie można wywołać metody.</span><span class="sxs-lookup"><span data-stu-id="cdd9f-111">Therefore, the applicability of the method to any possible type cannot be verified, and the method can never be called.</span></span>
