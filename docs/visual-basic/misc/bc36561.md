# <a name="extension-method-methodname-has-type-constraints-that-can-never-be-satisfied"></a><span data-ttu-id="bb3b4-102">Metoda rozszerzenia '\<methodname >' ma ograniczenia typu, które nigdy nie może zostać wykonane</span><span class="sxs-lookup"><span data-stu-id="bb3b4-102">Extension method '\<methodname>' has type constraints that can never be satisfied</span></span>
<span data-ttu-id="bb3b4-103">Parametry typu tej metody współpracują w sposób, który uniemożliwia ich spełnienie.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-103">The type parameters of this method interact in a way that prevents them from ever being satisfied.</span></span> <span data-ttu-id="bb3b4-104">Poniżej przedstawiono metodę rozszerzenia.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-104">The following extension method is an example.</span></span>  
  
```vb  
'' Not valid.  
'<Extension()> _  
'Sub extensionExample(Of T As U, U)(ByVal para1 As T, ByVal para2 As U)  
'End Sub  
```  
  
 <span data-ttu-id="bb3b4-105">Ponieważ metoda jest metodą rozszerzenia, kompilator musi być w stanie określić typ danych lub typy, które metoda rozszerza, na podstawie pierwszego parametru w deklaracji metody, `para1` i argumentu wysłanego dla tego parametru.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-105">Because the method is an extension method, the compiler must be able to determine the data type or types that the method extends based only on the first parameter in the method declaration, `para1`, and the argument sent in for that parameter.</span></span> <span data-ttu-id="bb3b4-106">Gdy pierwszy parametr odwołuje się do parametrów typu ogólnego, `para1 as T`, ograniczenia dotyczące parametrów ogólnych ograniczają zestaw typów, do których stosuje się ta metoda.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-106">When the first parameter refers to generic type parameters, `para1 as T`, the constraints on the generic parameters restrict the set of types to which the method applies.</span></span>  
  
 <span data-ttu-id="bb3b4-107">Możliwość zastosowania metody rozszerzenia jest określana na podstawie argumentu podanego dla pierwszego parametru, który jest `arg1` w poniższym kodzie.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-107">Applicability of an extension method is determined from the argument provided for the first parameter, which is `arg1` in the following code.</span></span>  
  
 `'' Not valid.`  
  
 `'arg1.extensionExample(arg2)`  
  
 <span data-ttu-id="bb3b4-108">Musi być możliwe sprawdzenie ograniczeń dla wszystkich parametrów typu ogólnego, do których odwołuje się pierwszy parametr, `para1`, patrząc tylko na pierwszy argument, `arg1`.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-108">It must be possible to verify the constraints on all generic type parameters referred to by the first parameter, `para1`, by looking at only the first argument, `arg1`.</span></span> <span data-ttu-id="bb3b4-109">W `extensionExample` zestaw typów, które są rozszerzane nie może być określony tylko na podstawie pierwszego parametru.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-109">In `extensionExample`, the set of types that is being extended cannot be determined from the first parameter alone.</span></span> <span data-ttu-id="bb3b4-110">Parametr typu `T` jest ograniczony przez parametr typu `U`, do którego nie odwołuje się `para1` i nie można go wywnioskować z `arg1`.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-110">Type parameter `T` is constrained by type parameter `U`, which is not referenced by `para1` and cannot be inferred from `arg1`.</span></span> <span data-ttu-id="bb3b4-111">W związku z tym nie można zweryfikować zastosowania metody do dowolnego możliwego typu, a metoda nie może być wywoływana.</span><span class="sxs-lookup"><span data-stu-id="bb3b4-111">Therefore, the applicability of the method to any possible type cannot be verified, and the method can never be called.</span></span>  
  
 <span data-ttu-id="bb3b4-112">**Identyfikator błędu:** BC36561</span><span class="sxs-lookup"><span data-stu-id="bb3b4-112">**Error ID:** BC36561</span></span>
