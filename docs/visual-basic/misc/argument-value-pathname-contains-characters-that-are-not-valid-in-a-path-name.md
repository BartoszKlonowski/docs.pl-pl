# <a name="argument-value-pathname-contains-characters-that-are-not-valid-in-a-path-name"></a>Wartość argumentu " \<pathname> " zawiera znaki, które nie są prawidłowe w nazwie ścieżki

Nazwa ścieżki zawiera jeden lub więcej nieprawidłowych znaków.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń znak z nazwy ścieżki.
