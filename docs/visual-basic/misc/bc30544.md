# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Metoda nie może zawierać jednocześnie instrukcji "try" i instrukcji "On Error" lub "resume"
Instrukcja została połączona `Try` z `On Error` lub `Resume` .  
  
 **Identyfikator błędu:** BC30544  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `On Error` lub `Resume` .
