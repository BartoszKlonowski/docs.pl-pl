# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>Metoda nie może zawierać zarówno &#39; Spróbuj &#39; Instrukcja i &#39; On Error &#39; i &#39; Wznów &#39; — Instrukcja
Połączono `Try` instrukcji z `On Error` lub `Resume`.  
  
 **Identyfikator błędu:** BC30544  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń `On Error` lub `Resume`.
