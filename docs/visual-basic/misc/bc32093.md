# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>&#39; &#39; wymagany podczas określania argumentów typu dla typu ogólnego lub metody
Instrukcję podejmie próbę utworzenia typu z typu ogólnego lub wywołanie metody rodzajowej bez użycia [z](../../visual-basic/language-reference/statements/of-clause.md) klauzuli.  
  
 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Wymaga składni dla typów ogólnych parametrów typu i argumentów typu dla przez `Of` — słowo kluczowe. Ponadto listy parametrów typu lub listy argumentów typu muszą być ujęte w nawiasy.  
  
 **Identyfikator błędu:** BC32093  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Obejmują `Of` — słowo kluczowe na początku argument typu listy i ujmij całą listę wewnątrz nawiasów.
