# <a name="of-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>"" Wymaganych podczas określania argumentów typu dla typu ogólnego lub metody
Instrukcję podejmuje próbę utworzenia typu z typu ogólnego lub wywołanie metody rodzajowej, bez korzystania z [z](../../visual-basic/language-reference/statements/of-clause.md) klauzuli.  
  
 Składnia języka Visual Basic dla typów ogólnych wymaga parametrów typu i argumentów typu dla wprowadzony przez `Of` — słowo kluczowe. Ponadto lista parametrów typu lub lista argumentów typu muszą być ujęte w nawiasach.  
  
 **Identyfikator błędu:** BC32093  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Obejmują `Of` — słowo kluczowe na początku argument typu listy i ujmij całą listę wewnątrz nawiasów.
