# <a name="printer-error"></a>Błąd drukarki

Na drukarce Wystąpił błąd, ale do komputera, który wysłał plik, nie zostały zwrócone żadne inne informacje.  
  
### <a name="to-correct-the-error"></a>Aby poprawić błąd  
  
- Fizycznie Badaj drukarkę. Upewnij się, że wszystkie połączenia między komputerem a drukarką są trwałe. Większość drukarek zapewnia wyświetlanie informacji o błędach, takich jak "offline" lub "poza papierem".
