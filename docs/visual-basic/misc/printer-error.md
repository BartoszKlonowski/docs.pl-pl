# <a name="printer-error"></a>Błąd drukarki
Wystąpił błąd w drukarce, ale żadne inne informacje została zwrócona do komputera, który wysłał pliku.  
  
### <a name="to-correct-the-error"></a>Aby poprawić błąd  
  
-   Fizycznie zbadać drukarki. Upewnij się, że wszystkie połączenia między komputerem a drukarki są wypełnione. Większość drukarek Podaj wyświetlenie informacje o błędzie, takie jak "Offline" lub "poza papier."
