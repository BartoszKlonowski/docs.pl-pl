# <a name="printer-error"></a>Błąd drukarki
Wystąpił błąd w drukarce, ale żadne inne informacje zwrócił się do komputera, który wysłał pliku.  
  
### <a name="to-correct-the-error"></a>Aby poprawić ten błąd  
  
- Sprawdź fizycznie drukarki. Upewnij się, że wszystkie połączenia między komputerem a drukarki są wypełnione. Większość drukarek, określ właściwość informacje o błędzie, np. "Offline" lub "poza papier."
