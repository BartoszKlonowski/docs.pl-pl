# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>Tylko osiem pierwszych znaków nazwy dziennika niestandardowego jest znaczące
Podczas sprawdzania nazw dzienników zdarzeń pod kątem unikatowości są brane pod uwagę tylko pierwsze osiem znaków. Konflikt może wynikać z dzienników zdarzeń, które dzielą swoje pierwsze osiem znaków.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nadaj dziennikowi zdarzeń nazwę, w której pierwsze osiem znaków jest unikatowe.
