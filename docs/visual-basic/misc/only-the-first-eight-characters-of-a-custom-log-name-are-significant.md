# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>Tylko pierwszych osiem znaków niestandardowej nazwy dziennika są istotne
Podczas sprawdzania dostępności nazwy dziennika zdarzeń unikatowość, uwzględniane są tylko pierwszych osiem znaków. Konflikt mogą wynikać z dzienników zdarzeń, które mają ich pierwszych osiem znaków.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nadaj nazwę, w której pierwszych osiem znaków są unikatowe dziennika zdarzeń.
