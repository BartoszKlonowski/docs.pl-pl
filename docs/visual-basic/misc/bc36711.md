# <a name="argument-matching-parameter-ltparameternamegt-narrows-to-lttypenamegt"></a>Parametr pasujący do argumentu "&lt;parametername&gt;"zawęża element do"&lt;typename&gt;"
Podjęto wywołanie przeciążonej metody, ale kompilator nie może odnaleźć metody, które można wywołać bez konwersji zawężającej. A *konwersja zawężająca* zmienia wartość na typ danych, który może nie posiadać pewnych możliwych wartości.  
  
 **Identyfikator błędu:** BC36711
