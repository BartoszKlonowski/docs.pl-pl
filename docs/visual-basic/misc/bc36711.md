# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>Argument odpowiadający parametrowi &#39; &lt;parametername&gt; &#39; zawęża element do &#39; &lt;typename&gt;&#39;
Wprowadzono wywołanie do metody przeciążonej, ale kompilator nie można znaleźć metody, który można wywołać bez konwersji zawężającej. A *konwersja zawężająca* zmiany wartości na typ danych, który może nie posiadać pewnych możliwych wartości.  
  
 **Identyfikator błędu:** BC36711
