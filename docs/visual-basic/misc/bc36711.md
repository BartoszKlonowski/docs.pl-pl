# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Parametr pasujący do argumentu " \<parametername> " jest wąski do " \<typename> "

Nastąpiło wywołanie do przeciążonej metody, ale kompilator nie może znaleźć metody, która może zostać wywołana bez zawężania konwersji. *Konwersja zawężania* zmienia wartość na typ danych, który może nie być w stanie przechowywać niektórych możliwych wartości.  
  
 **Identyfikator błędu:** BC36711
