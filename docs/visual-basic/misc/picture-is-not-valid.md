# <a name="picture-is-not-valid"></a>Obraz nie jest prawidłowy
Próbowano przypisać format graficzny niż mapa bitowa, ikona lub Windows metafile do `Picture` właściwości formularza lub kontrolki.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że próbujesz załadować do pliku `Picture` właściwość jest prawidłowy obsługiwany przez plik [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].
