# <a name="enum-39ltenumnamegt39-must-contain-at-least-one-member"></a>Wyliczenia &#39; &lt;enumname&gt;&#39; musi zawierać co najmniej jednego członka
Występuje puste wyliczenie. Co najmniej jeden element członkowski musi być zadeklarowany jako między `Enum` instrukcji i `End Enum` instrukcji.  
  
 **Identyfikator błędu:** BC30280  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodawanie deklaracji elementu członkowskiego do wyliczenia.
