# <a name="enum-enumname-must-contain-at-least-one-member"></a>Wyliczenie " \<enumname> " musi zawierać co najmniej jeden element członkowski
Występuje puste Wyliczenie. Należy zadeklarować co najmniej jeden element członkowski między `Enum` instrukcją a `End Enum` instrukcją.  
  
 **Identyfikator błędu:** BC30280  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Dodaj deklarację elementu członkowskiego do wyliczenia.
