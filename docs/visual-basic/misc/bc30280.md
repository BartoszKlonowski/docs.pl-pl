# <a name="enum-ltenumnamegt-must-contain-at-least-one-member"></a>Enum '&lt;enumname&gt;"musi zawierać co najmniej jednego członka
Występuje, puste wyliczenie. Co najmniej jeden element członkowski musi być zadeklarowany między `Enum` instrukcji i `End Enum` instrukcji.  
  
 **Identyfikator błędu:** BC30280  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodawanie deklaracji elementu członkowskiego do wyliczenia.
