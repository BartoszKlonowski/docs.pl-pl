# <a name="39option-compare39-must-be-followed-by-39text39-or-39binary39"></a>&#39;Option Compare&#39; musi następować &#39;tekst&#39; lub &#39;binarne&#39;
`Option Compare` Instrukcja zawiera nieprawidłowe ustawienie lub brak ustawienia. Dozwolone tylko ustawienia w `Option Compare` są `Text` i `Binary`.  
  
 **Identyfikator błędu:** BC30207  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy specyfikator ustawienie jest błędna.  
  
2.  Dodaj albo `Text` lub `Binary` do `Option Compare` instrukcji, na przykład `Option Compare Text`.
