# <a name="attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Atrybut &#39; &lt;attributename&gt;&#39; jest nieprawidłowa: &lt;błąd&gt;
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń atrybut z deklaracji lub Zamień atrybut, który jest prawidłowy dla kontekstu.  
  
2.  Sprawdź komunikat o błędzie i podejmij odpowiednią akcję.
