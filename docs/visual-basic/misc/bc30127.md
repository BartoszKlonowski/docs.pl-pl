# <a name="attribute-attributename-is-not-valid-error"></a>Atrybut '\<attributename >' jest nieprawidłowa: \<błąd >
Środowisko uruchomieniowe języka wspólnego nie akceptuje atrybutu w bieżącym kontekście.  
  
 **Identyfikator błędu:** BC30127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń atrybut z deklaracji lub zamień go atrybutem, który jest prawidłowy dla kontekstu.  
  
2. Sprawdź komunikat o błędzie i podejmij odpowiednią akcję.
