# <a name="procedurename-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>element " \<procedurename> " nie ma parametrów, a jego typ zwracany nie może być indeksowany
Wywołanie `Function` `Sub` procedury lub dostarcza jeden lub więcej argumentów, ale procedura nie przyjmuje żadnych argumentów, a jej typ zwracany (jeśli jest `Function` ) nie jest typem tablicy.  
  
 **Identyfikator błędu:** BC32016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń argument lub argumenty z wywołania procedury.
