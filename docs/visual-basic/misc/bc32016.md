# <a name="ltprocedurenamegt-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>"&lt;nazwaprocedury&gt;" nie ma parametrów, a jego typem zwracanym nie mogą być indeksowane.
Wywołanie `Function` lub `Sub` procedury dostarcza jeden lub więcej argumentów, ale procedura nie przyjmuje żadnych argumentów i zwracanego przez nią typu (jeśli jest `Function`) nie jest typem tablicy.  
  
 **Identyfikator błędu:** BC32016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuwanie argumentów lub argument wywołania procedury.
