# <a name="39ltprocedurenamegt39-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>&#39; &lt;nazwaprocedury&gt;&#39; nie ma parametrów, a jego typ zwracany nie może być indeksowany.
Wywołanie `Function` lub `Sub` procedura zawiera jeden lub więcej argumentów, ale procedura nie przyjmuje żadnych argumentów i jej typu zwracanego (jeśli jest `Function`) nie jest typem tablicy.  
  
 **Identyfikator błędu:** BC32016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń argument lub argumentów w wywołaniu procedury.
