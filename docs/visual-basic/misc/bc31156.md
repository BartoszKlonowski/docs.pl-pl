# <a name="xml-attribute-version-must-be-the-first-attribute-in-xml-declaration"></a>Atrybut XML "Version" musi być pierwszym atrybutem w deklaracji XML
`version`Atrybut literału dokumentu XML nie jest pierwszym określonym atrybutem. `version`Atrybut musi być pierwszym atrybutem określonym dla literału dokumentu XML, zgodnie ze specyfikacją XML 1,0.  
  
 **Identyfikator błędu:** BC31156  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień położenie `version` atrybutu literału dokumentu XML, tak aby był to pierwszy atrybut.
