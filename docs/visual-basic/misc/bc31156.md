# <a name="xml-attribute-39version39-must-be-the-first-attribute-in-xml-declaration"></a>Atrybut XML &#39;wersji&#39; musi być pierwszym atrybutem w deklaracji XML
`version` Atrybut literał dokumentu XML nie jest określony atrybut pierwszy. `version` Atrybut musi być pierwszy atrybut określony dla literału ciągu Specyfikacja XML 1.0 dokumentu XML.  
  
 **Identyfikator błędu:** BC31156  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmienia położenie `version` atrybutu literału tak, aby pierwszy atrybut dokumentu XML.
