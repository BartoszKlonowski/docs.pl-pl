# <a name="xml-attribute-version-must-be-the-first-attribute-in-xml-declaration"></a>Atrybut XML "version" musi być pierwszy atrybut w deklaracji XML
`version` Atrybutu literał dokumentu XML nie jest pierwszy atrybut określony. `version` Atrybut musi być pierwszy atrybut określona dla literału w poszczególnych Specyfikacja XML 1.0 dokumentu XML.  
  
 **Identyfikator błędu:** BC31156  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana położenia `version` atrybutu literał, tak, aby pierwszy atrybut dokumentu XML.
