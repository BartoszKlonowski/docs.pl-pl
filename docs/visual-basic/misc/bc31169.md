# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak "Character" (0x \<hexadecimal> ) jest niedozwolony na początku nazwy XML
Nieobsługiwany znak jest uwzględniony na początku nazwy literału XML lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31169  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwany znak z początku nazwy literału XML lub właściwości osi XML.
