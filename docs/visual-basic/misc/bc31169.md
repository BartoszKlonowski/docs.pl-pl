# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak &#39;znak&#39; (0 x&lt;szesnastkową&gt;) jest niedozwolony na początku nazwy XML
Nieobsługiwany znak jest dołączony na początku nazwy XML literału lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31169  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane znaki od początku nazwy XML literału lub właściwości osi XML.
