# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak "character" (0 x\<szesnastkowe >) nie jest dozwolona na początku nazwy XML
Nieobsługiwany znak jest dołączony na początku nazwy literał XML lub właściwości osi XML.  
  
 **Identyfikator błędu:** BC31169  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane znaki od początku nazwy literał XML i właściwości osi XML.
