# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>"&lt;nazwa&gt;" nie może ujawnić typu &lt;type1&gt; w &lt;specyfikator&gt; &lt;type2&gt; "&lt;typename&gt;" za pośrednictwem &lt;nazwy&gt;
Podjęto próbę udostępnienia prywatny typ poza klasę publiczną `MyClass`.  
  
 **Identyfikator błędu:** BC30508  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie typu `Public`.
