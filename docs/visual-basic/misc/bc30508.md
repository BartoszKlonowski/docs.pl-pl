# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>element " \<name> " nie może ujawniać typu \<type1> w \<specifier> \<type2> " \<typename> " za poorednictwem \<name>

Podjęto próbę uwidocznienia typu prywatnego poza klasą publiczną `MyClass` .  
  
 **Identyfikator błędu:** BC30508  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zadeklaruj typ `Public` .
