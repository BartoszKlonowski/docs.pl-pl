# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>"\<nazwa >" nie może ujawnić typu \<type1 > w \<specyfikator > \<type2 > "\<typename >" za pośrednictwem \<name >
Podjęto próbę udostępnienia prywatny typ poza klasę publiczną `MyClass`.  
  
 **Identyfikator błędu:** BC30508  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie typu `Public`.
