# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39;&lt;Nazwa&gt; &#39; nie może ujawnić typu &lt;type1&gt; w &lt;specyfikator&gt; &lt;type2&gt; &#39; &lt;typename&gt; &#39; za pomocą &lt;nazwy&gt;
Podjęto próbę ujawnić prywatnej typu poza Klasa publiczna `MyClass`.  
  
 **Identyfikator błędu:** BC30508  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklaruje typ `Public`.
