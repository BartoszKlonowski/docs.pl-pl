# <a name="unable-to-open-file-filename-error"></a>Nie można otworzyć pliku '\<nazwa pliku >': \<błąd >
Wystąpił błąd podczas otwierania pliku w tym projekcie.  
  
 **Identyfikator błędu:** BC31027  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy plik istnieje i jest nie uszkodzony lub zablokowany przez inną aplikację.
