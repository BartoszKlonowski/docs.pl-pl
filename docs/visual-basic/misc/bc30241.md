# <a name="named-argument-expected"></a>Oczekiwano argumentu nazwanego
Lista argumentów w wywołaniu procedury przekazuje argument według nazwy, a następnie argumentu według położenia. Wszystkie argumenty pozycyjne musi poprzedzać wszystkie argument przekazany przez nazwę w wywołaniu procedury.  
  
 **Identyfikator błędu:** BC30241  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy zmodyfikować listy argumentów można umieścić Argumenty przekazane za pomocą pozycji wcześniejsze Argumenty przekazane według nazwy.
