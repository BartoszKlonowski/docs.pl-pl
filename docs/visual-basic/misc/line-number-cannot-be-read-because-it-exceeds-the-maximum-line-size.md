# <a name="line-number-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>\<number>Nie można odczytać wiersza, ponieważ przekracza on maksymalny rozmiar wiersza.
Nie można odczytać wiersza, ponieważ przekracza on maksymalny rozmiar wiersza. Zazwyczaj oznacza to, że w pliku występuje cudzysłów otwierający, ale bez cudzysłowu zamykającego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wstaw znak cudzysłowu zamykającego w pliku.
