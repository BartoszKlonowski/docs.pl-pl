# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>Tag komentarza XML &#39; &lt;tag&gt;&#39; nie jest dozwolony w &#39;&lt; element&gt;&#39; element języka
Tag komentarza XML \<tag > nie jest dozwolony w "\<elementu >" elementu. Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML nie jest obsługiwany przez element języka, do którego jest stosowana.  
  
 **Identyfikator błędu:** BC42306  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane tag komentarza XML.
