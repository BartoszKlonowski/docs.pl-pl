# <a name="xml-comment-tag-lttaggt-is-not-permitted-on-a-ltelementgt-language-element"></a>Tag komentarza XML "&lt;tag&gt;" nie jest dozwolony w "&lt;elementu&gt;" element języka
Tag komentarza XML \<tagu > nie jest dozwolony w "\<element >' element. Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML nie jest obsługiwany przez element języka, do którego jest stosowany.  
  
 **Identyfikator błędu:** BC42306  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieobsługiwane tag komentarza XML.
