# <a name="withevents-variable-does-not-raise-any-events"></a>Zmienna "WithEvents" nie wywołuje żadnych zdarzeń
Zmienna obiektu zadeklarowana ze `WithEvents` słowem kluczowym nie odwołuje się do wystąpienia klasy, które może zgłaszać zdarzenia.  
  
 **Identyfikator błędu:** BC30591  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że zmienna obiektu odwołuje się do wystąpienia klasy, które może zgłaszać zdarzenia.
