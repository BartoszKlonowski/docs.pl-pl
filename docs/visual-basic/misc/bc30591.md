# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39; WithEvents &#39; Zmienna nie wygenerował żadnych zdarzeń
Zmienna obiektu zadeklarowana z `WithEvents` — słowo kluczowe nie odwołuje się do wystąpienia klasy, które może wywoływać zdarzenia.  
  
 **Identyfikator błędu:** BC30591  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że zmienna obiektu odwołuje się do wystąpienia klasy, które może wywoływać zdarzenia.
