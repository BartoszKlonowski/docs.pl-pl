# <a name="withevents-variable-does-not-raise-any-events"></a>Zmienna "WithEvents" nie powoduje żadnych zdarzeń
Zmienna obiektu zadeklarowane za pomocą `WithEvents` — słowo kluczowe nie odwołuje się do wystąpienia klasy, które może wywoływać zdarzenia.  
  
 **Identyfikator błędu:** BC30591  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że zmienna obiektu odwołuje się do wystąpienia klasy, które może wywoływać zdarzenia.
