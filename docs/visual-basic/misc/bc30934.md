 <span data-ttu-id="6d03f-104">Atrybut udostępnia metadane dla elementu, który jest stosowany do, a kompilator musi mieć możliwość utworzenia wszystkich metadanych w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="6d03f-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="6d03f-105">Z tego powodu każdego atrybutu musi używać wartości, które są stałe w czasie kompilacji, a w związku z tym każdy argument atrybutu musi obliczać wartość stałą czasu kompilacji.</span><span class="sxs-lookup"><span data-stu-id="6d03f-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>
