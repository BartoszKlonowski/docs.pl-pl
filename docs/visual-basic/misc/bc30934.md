<span data-ttu-id="21ec3-103">Wynikiem obliczenia wyrażenia używane do argument atrybutu jest innym niż z odpowiadającym mu parametrem atrybutu typu danych i Visual Basic nie zezwala na konwersji typu wymagane dla argumentów atrybutów.</span><span class="sxs-lookup"><span data-stu-id="21ec3-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and Visual Basic does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="21ec3-104">Atrybut zawiera metadanych dla elementu, który jest stosowany do, a kompilator musi mieć możliwość utworzenia wszystkich metadanych w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="21ec3-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="21ec3-105">Z tego powodu każdego atrybutu, należy użyć wartości, które są stałe w czasie kompilacji, a w związku z tym każdy argument atrybutu musi być wartością stałą czasu kompilacji.</span><span class="sxs-lookup"><span data-stu-id="21ec3-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="21ec3-106">Niektóre konwersje typów nie może tworzyć wartości, które są stałe w czasie kompilacji.</span><span class="sxs-lookup"><span data-stu-id="21ec3-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="21ec3-107">Na przykład konwertowanie `String` do `Double` lub `Date` zależy od ustawień regionalnych w czasie wykonywania.</span><span class="sxs-lookup"><span data-stu-id="21ec3-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="21ec3-108">Inne konwersji, takich jak tablica typu pochodnego tablicę `Object`, istnieje szereg problemów, które nie zezwalają na kompilator, aby zezwolić im na argumentów atrybutu.</span><span class="sxs-lookup"><span data-stu-id="21ec3-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>
