# <a name="xml-documentation-parse-error-start-tag-39lttaggt39-doesn39t-have-a-matching-end-tag"></a>Błąd analizowania dokumentacji XML: tag początkowy &#39; &lt;tag&gt; &#39; &#39;ma zgodnego tagu końcowego
Błąd analizowania dokumentacji XML: tag początkowy \<tagu > nie ma zgodnego tagu końcowego. Komentarz XML zostanie zignorowany.  
  
 Komentarz XML zawiera tag początkowy, ale nie ma zgodnego tagu końcowego.  
  
 **Identyfikator błędu:** BC42316  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj tag końcowy, który pasuje do tagu początkowego.  
  
     — lub —  
  
-   Jeśli znacznik nie zawiera wewnętrzny tekstu, takie jak [ \<SeeAlso — >](../../visual-basic/language-reference/xmldoc/seealso.md), określ ukośnik przed nawias zamykający.
