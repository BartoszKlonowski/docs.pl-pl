# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "GoTo" nie są prawidłowe w oknie bezpośrednim

`GoTo`Instrukcja wykonuje rozgałęzianie i nie jest dozwolona w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30133  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie wystawiaj `GoTo` instrukcji w oknie **bezpośrednim** .
