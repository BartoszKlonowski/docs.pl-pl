# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>Instrukcja "GoTo" nie są prawidłowe w oknie bezpośrednim
`GoTo` Instrukcji wykonuje rozgałęzianie i nie jest dozwolona w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30133  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie wydają `GoTo` instrukcji w **bezpośrednie** okna.
