# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Przejdź do &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`GoTo` Instrukcji wykonuje rozgałęzianie i nie jest dozwolona w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30133  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie wydają `GoTo` instrukcji w **Immediate** okna.
