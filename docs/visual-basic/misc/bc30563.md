# <a name="ltnamegt-is-ambiguous-in-the-application-objects-ltlistgt"></a>"&lt;nazwa&gt;"jest niejednoznaczny w obiektach aplikacji"&lt;listy&gt;"
Podano nazwę, która jest niejednoznaczny i dlatego jest w konflikcie z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; odróżnić nazwy musi samodzielnie. Lista dostarczana w komunikacie o błędzie obiekty aplikacji, gdzie nazwa jest niejednoznaczna.  
  
 **Identyfikator błędu:** BC30563  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
