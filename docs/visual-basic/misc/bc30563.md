# <a name="name-is-ambiguous-in-the-application-objects-list"></a>element " \<name> " jest niejednoznaczny w obiektach aplikacji " \<list> "

Podana nazwa jest niejednoznaczna i dlatego powoduje konflikt z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; należy samodzielnie odróżnić nazwy. Lista podana w komunikacie o błędzie zawiera szczegółowe informacje o obiektach aplikacji, w których nazwa jest niejednoznaczna.  
  
 **Identyfikator błędu:** BC30563  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- W pełni Zakwalifikuj nazwę.
