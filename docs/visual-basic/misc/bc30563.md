# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;Nazwa&gt; &#39; jest niejednoznaczny w obiektach aplikacji &#39; &lt;listy&gt;&#39;
Podana nazwa jest niejednoznaczny i dlatego powoduje konflikt z inną nazwą. Kompilator Visual Basic nie ma żadnych reguł rozwiązywania konfliktów; odróżniania nazw musi samodzielnie. Lista dostępnych w szczegóły komunikatu o obiektów aplikacji, której nazwa jest niejednoznaczna.  
  
 **Identyfikator błędu:** BC30563  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Pełnej nazwy.
