# <a name="type-expected"></a>Oczekiwano typu
Znaki kodu źródłowego wystąpienia, które nie są rozpoznawalne jako nazwy typu. Kontekst wywołania dla typu danych podstawowych lub nazwa element zadeklarowany, reprezentujący typu.  
  
 **Identyfikator błędu:** BC30182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy typ został zdefiniowany.  
  
- Upewnij się, że nazwa typu została wpisana poprawnie.
