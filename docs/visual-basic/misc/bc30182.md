# <a name="type-expected"></a>Oczekiwano typu
Występują znaki kodu źródłowego, które nie są rozpoznawane jako nazwa typu. Kontekst wywołuje dla typu danych podstawowych lub zadeklarowanej nazwy elementu reprezentującej typ.  
  
 **Identyfikator błędu:** BC30182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy typ został zdefiniowany.  
  
- Sprawdź, czy nazwa typu jest wpisana poprawnie.
