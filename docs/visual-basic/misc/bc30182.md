# <a name="type-expected"></a>Oczekiwano typu
Znaki kodu źródłowego wystąpić, które nie jest rozpoznawalna jako nazwę typu. Kontekst odwołuje się do typu danych podstawowych lub nazwę zadeklarowany element reprezentujący typu.  
  
 **Identyfikator błędu:** BC30182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, że typ został zdefiniowany.  
  
-   Sprawdź, czy nazwa typu jest poprawna.
