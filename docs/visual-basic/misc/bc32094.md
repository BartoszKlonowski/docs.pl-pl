# <a name="comma-or--expected-type-argument-list"></a>Przecinek lub ')' oczekiwany (lista argumentów typu)
Lista argumentów typu zawiera nieoczekiwany element. Używaj przecinków (`,`) do oddzielenia argumentów typu w takiej listy i użyj nawias zamykający (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC32094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj przecinek i kontynuować lista argumentów typu lub Dodaj zamykający nawias znaków do zakończenia na liście.
