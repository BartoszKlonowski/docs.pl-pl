# <a name="comma-or-3939-expected-type-argument-list"></a>Przecinek lub &#39;) &#39; Oczekiwano (lista argumentów typu)
Lista argumentów typu zawiera nieoczekiwany element. Użyj przecinków (`,`) do oddzielne argumentów typu na takiej liście, a następnie użyć nawiasu zamykającego (`)`) do końca listy.  
  
 **Identyfikator błędu:** BC32094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodawanie przecinkami i kontynuować listy argumentów typu lub dodawanie zamknięcia znak nawiasu zakończenie na liście.
