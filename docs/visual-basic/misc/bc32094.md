# <a name="comma-or--expected-type-argument-list"></a>Oczekiwano przecinka lub znaku ")" (lista argumentów typu)
Lista argumentów typu zawiera nieoczekiwany element. Użyj przecinków ( `,` ) do oddzielania argumentów typu na tej liście i użyj nawiasu zamykającego ( `)` ), aby zakończyć listę.  
  
 **Identyfikator błędu:** BC32094  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj przecinek i Kontynuuj listę argumentów typu lub Dodaj znak nawias zamykający, aby zakończyć listę.
