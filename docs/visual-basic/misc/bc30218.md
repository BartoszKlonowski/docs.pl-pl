# <a name="39lib39-expected"></a>&#39;Lib&#39; oczekiwano
A `Declare` instrukcji Określa nazwę biblioteki bez użycia `Lib` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC30218  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Lib` — słowo kluczowe przed nazwę biblioteki.
