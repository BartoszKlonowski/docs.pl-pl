# <a name="lib-expected"></a>Oczekiwano elementu "lib"
`Declare`Instrukcja określa nazwę biblioteki bez użycia `Lib` słowa kluczowego.  
  
 **Identyfikator błędu:** BC30218  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Lib` słowo kluczowe przed nazwą biblioteki.
