# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>Przekroczono maksymalną liczbę błędów

Osiągnięto maksymalną liczbę błędów kompilatora 100.  
  
 **Identyfikator błędu:** BC2020  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmniejsz liczbę wygenerowanych błędów przez debugowanie i rozadresowanie każdego z nich.
