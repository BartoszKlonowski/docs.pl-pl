# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>Przekroczono maksymalną liczbę błędów
Osiągnięto maksymalną kompilatora 100 błędów.  
  
 **Identyfikator błędu:** BC2020  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmniejsz liczbę wygenerowane błędy debugowania, a każdy z kolei adresowania.
