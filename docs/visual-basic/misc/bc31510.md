# <a name="attribute-constant-constantname-cannot-be-the-target-of-an-assignment"></a>Stała atrybutu " \<constantname> " nie może być elementem docelowym przypisania
Podjęto próbę przypisania wartości do stałej zadeklarowanej w atrybucie.  
  
 **Identyfikator błędu:** BC31510  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń błędne przypisanie.  
  
- Jeśli używasz niestandardowych atrybutów, należy ponownie zdefiniować element członkowski w atrybucie jako pole.
