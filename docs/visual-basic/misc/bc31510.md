# <a name="attribute-constant-ltconstantnamegt-cannot-be-the-target-of-an-assignment"></a>Stała atrybutu "&lt;constantname&gt;" nie może być elementem docelowym przypisania
Próbowano przypisać wartość stałą, zadeklarowany w atrybucie.  
  
 **Identyfikator błędu:** BC31510  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń przypisanie błędne.  
  
-   Jeśli używasz atrybutów niestandardowych, które opracowany ponownie zdefiniować element członkowski w atrybucie jako pole.
