# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Zatrzymaj &#39; instrukcje nie są prawidłowe w oknie bezpośrednim
`Stop` i `End` instrukcje wstrzymać wykonywania i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30122  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Nie wydają `Stop` lub `End` instrukcji w **Immediate** okna.
