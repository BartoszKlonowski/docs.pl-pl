# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Stop" są nieprawidłowe w oknie bezpośrednim
`Stop` i `End` instrukcje wstrzymał wykonywanie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30122  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie wydają `Stop` lub `End` instrukcji w **bezpośrednie** okna.
