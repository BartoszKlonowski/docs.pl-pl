# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "Stop" nie są prawidłowe w oknie bezpośrednim

`Stop`Instrukcje and `End` zawieszają wykonywanie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30122  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie należy wystawiać `Stop` `End` instrukcji lub w oknie **bezpośrednim** .
