# <a name="implicit-variable-ltvariablenamegt-is-invalid-because-of-ltmessagegt"></a>Zmienna niejawna "&lt;nazwa_zmiennej&gt;"jest nieprawidłowa z powodu"&lt;komunikat&gt;"
Nieprawidłowy zmienną niejawną został wygenerowany podczas debugowania.  
  
 **Identyfikator błędu:** BC36016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź wspominane komunikat dla przyczynę i możliwe rozwiązanie.  
  
2.  Jeśli błąd będzie się powtarzać, zebrać informacje dotyczące okoliczności i powiadom techniczna firmy Microsoft.
