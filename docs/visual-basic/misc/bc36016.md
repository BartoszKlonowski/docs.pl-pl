# <a name="implicit-variable-variablename-is-invalid-because-of-message"></a>Zmienna niejawna "\<VariableName >" jest nieprawidłowa z powodu "\<komunikatu >"
Podczas debugowania Wygenerowano nieprawidłową zmienną niejawną.  
  
 **Identyfikator błędu:** BC36016  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Przejrzyj cytowany komunikat z przyczyn i ewentualnych środków zaradczych.  
  
2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
