# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>Liczba indeksów jest mniejsza niż liczba wymiarów tablicy indeksowanej
Liczba indeksów, które umożliwiają dostęp do elementu tablicy muszą być dokładnie takie same rangą tablicy, czyli liczby wymiarów zadeklarowany dla niego.  
  
 **Identyfikator błędu:** BC30105  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj indeksy dolne odwołania do tablicy, dopóki liczba indeksów dolnych equals rangą tablicy.
