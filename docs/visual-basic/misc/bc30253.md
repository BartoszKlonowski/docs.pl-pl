# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39; interfejs &#39; musi być zakończona odpowiadającą jej instrukcją &#39; Interfejs &#39;
`Interface` Występuje instrukcja bez odpowiadającego `End Interface` instrukcji. Należy użyć `End Interface` instrukcji, aby zakończyć definicji interfejsu.  
  
 **Identyfikator błędu:** BC30253  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy `Interface` elementy członkowskie są prawidłowo sformatowane.  
  
2.  Dodaj `End Interface` instrukcji tot koniec definicji interfejsu.
