# <a name="interface-must-end-with-a-matching-end-interface"></a>"Interface" musi kończyć się z instrukcją "End Interface"
`Interface` Instrukcji odbywa się bez odpowiadającego `End Interface` instrukcji. Należy użyć `End Interface` instrukcję, aby zakończyć definicji interfejsu.  
  
 **Identyfikator błędu:** BC30253  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy `Interface` elementy członkowskie są poprawnie sformatowane.  
  
2. Dodaj `End Interface` instrukcji tot koniec definicji interfejsu.
