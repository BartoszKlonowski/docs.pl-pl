# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>Wymagane odwołanie do modułu " \<modulename> " zawierające definicję dla zdarzenia " \<eventname> "

Wymagane odwołanie do modułu "<`modulename`>" zawierające definicję zdarzenia "<`eventname`>". Dodaj je do projektu.  
  
 Zdarzenie jest zdefiniowane w module, który nie jest bezpośrednio przywoływany w projekcie. Kompilator Visual Basic wymaga odwołania, aby uniknąć niejednoznaczności na wypadek, gdyby zdarzenie zostało zdefiniowane w więcej niż jednym module.  
  
 **Identyfikator błędu:** BC30006  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dołącz nazwę modułu, do którego nie ma odwołania, w odwołaniach do projektu.
