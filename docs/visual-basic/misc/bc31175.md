# <a name="xml-dtds-are-not-supported"></a>Definicje DTD XML nie są obsługiwane.
Definicja typu dokumentu (DTD) został określony jako literał XML.  
  
 **Identyfikator błędu:** BC31175  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń element DTD. Definicja schematu XML (XSD) umożliwia określenie schemat dokumentu XML.
