# <a name="xml-dtds-are-not-supported"></a>Definicje DTD XML nie są obsługiwane

Definicja typu dokumentu (DTD) została określona jako literał XML.  
  
 **Identyfikator błędu:** BC31175  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń DTD. Użyj definicji schematu XML (XSD), aby określić schemat dla dokumentu XML.
