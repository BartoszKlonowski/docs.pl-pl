# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-ltlistgt"></a>Żaden dostępny przeciążony element &#39; &lt;methodname&gt;&#39; może być wywołany z tymi argumentami: &lt;listy&gt;
Przeciążona metoda jest wywoływana, ale metoda nie można dopasować z listą argumentów.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zgodne z listą argumentów przeciążeń, przeznaczonych do wywołania.  
  
-   Dostosuj listy argumentów, aby sygnatury przeciążenia.
