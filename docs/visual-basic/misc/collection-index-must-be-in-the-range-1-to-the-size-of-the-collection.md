# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>Indeks kolekcji musi zawierać się w zakresie od 1 do rozmiaru kolekcji

Podana wartość indeksu znajduje się poza prawidłowym zakresem, czyli 1 do rozmiaru kolekcji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmień podaną wartość na prawidłową.
