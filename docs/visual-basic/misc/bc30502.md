# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>Nie można łączyć instrukcji "Shared" z elementem " \<specifier> " w deklaracji właściwości
Podjęto próbę połączenia `Shared` modyfikatora ze specyfikatorem, który nie jest prawidłowy w takiej kombinacji w deklaracji właściwości.  
  
 **Identyfikator błędu:** BC30502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
