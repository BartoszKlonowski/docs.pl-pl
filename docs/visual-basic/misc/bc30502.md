# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;Udostępnione&#39; nie można połączyć z &#39; &lt;specyfikator&gt; &#39; w deklaracji właściwości
Próbujesz połączyć `Shared` modyfikator ze specyfikatorem, który nie jest prawidłowy w kombinacji w deklaracji właściwości.  
  
 **Identyfikator błędu:** BC30502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
