# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>"Shared" nie można połączyć z "&lt;specyfikator&gt;" w deklaracji właściwości
Próbujesz połączyć `Shared` modyfikator ze specyfikatorem, który nie jest prawidłowy w kombinacji w deklaracji właściwości.  
  
 **Identyfikator błędu:** BC30502  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
