# <a name="baudrate-must-be-greater-than-0"></a>Szybkość transmisji musi być większa niż 0
`BaudRate` Określono argumentu dla `My.Computer.Ports.OpenSerialPort` metoda musi być większa od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość `BaudRate` argument liczbą dodatnią.
