# <a name="baudrate-must-be-greater-than-0"></a>Szybkość transmisji musi być większa niż 0

`BaudRate`Argument określony dla `My.Computer.Ports.OpenSerialPort` metody musi być większy od zera.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zmień wartość `BaudRate` argumentu na liczbę dodatnią.
