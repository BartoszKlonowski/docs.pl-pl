# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>Właściwość lub pole '\<name >' nie ma prawidłowego typu atrybutu
Może być używany Konstruktor atrybutu, który nie akceptuje prawidłowego typu atrybutu, mimo że atrybut może mieć inne odpowiednie konstruktory.  
  
 **Identyfikator błędu:** BC30659  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź użycie tego atrybutu.
