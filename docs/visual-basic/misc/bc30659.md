# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>Właściwość lub pole "\<name >" nie ma prawidłowego typu atrybutu
Być może używasz atrybut konstruktora, który akceptuje to prawidłowy typ atrybutu, nawet jeśli ten atrybut może mieć inne odpowiednie konstruktorów.  
  
 **Identyfikator błędu:** BC30659  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź użycie tego atrybutu.
