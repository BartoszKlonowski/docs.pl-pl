<span data-ttu-id="ab0a7-103">A [instrukcja Get](../../visual-basic/language-reference/statements/get-statement.md) lub [instrukcji Set](../../visual-basic/language-reference/statements/set-statement.md) w `NotOverridable` zawiera właściwość `Private` — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-103">A [Get Statement](../../visual-basic/language-reference/statements/get-statement.md) or [Set Statement](../../visual-basic/language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="ab0a7-104">Następujący wiersz logikę wyjaśnia, dlaczego `NotOverridable` i `Private` nie można łączyć w [Property — instrukcja](../../visual-basic/language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="ab0a7-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../../visual-basic/language-reference/statements/property-statement.md):</span></span>  
  
1.  <span data-ttu-id="ab0a7-105">Właściwość lub procedurę, która nie powoduje zastąpienia właściwości klasy bazowej lub procedura ma domyślne ustawienie [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="ab0a7-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span></span>  
  
2.  <span data-ttu-id="ab0a7-106">Jednakże, właściwość lub procedura w klasie pochodnej, która zastępuje właściwość klasy bazowej lub procedura ma domyślne ustawienie [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span><span class="sxs-lookup"><span data-stu-id="ab0a7-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="ab0a7-107">Zakończenie hierarchii zastępowanie, trzeba je zadeklarować `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="ab0a7-108">To jest tylko kontekst, w którym można użyć `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="ab0a7-109">Oznacza to, że używasz `NotOverridable` tylko w połączeniu z [zastępuje](../../visual-basic/language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="ab0a7-109">That is, you can use `NotOverridable` only in combination with [Overrides](../../visual-basic/language-reference/modifiers/overrides.md).</span></span>  
  
3.  <span data-ttu-id="ab0a7-110">Jeśli zadeklarowano klasę bazową właściwość lub procedura [prywatnej](../../visual-basic/language-reference/modifiers/private.md), klasa pochodna nie może zastąpić tego właściwość lub procedura, ponieważ on nie można uzyskać do niego dostęp.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-110">If a base class property or procedure is declared [Private](../../visual-basic/language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="ab0a7-111">W związku z tym nie można użyć `Private` w połączeniu z `Overridable`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4.  <span data-ttu-id="ab0a7-112">Aby zastąpić właściwość lub procedura, nadrzędnych właściwość lub procedura musi mieć nie tylko podpisu identyczne, ale także ten sam poziom dostępu.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="ab0a7-113">Oznacza to, że nadrzędnych właściwość lub procedura nie można określić `Private`, ponieważ nie można określić możliwym do zastąpienia właściwość lub procedura `Private`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5.  <span data-ttu-id="ab0a7-114">Ponieważ można określić `NotOverridable` tylko na przesłanianie właściwość lub procedura, nie można połączyć ją z `Private`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="ab0a7-115">Za samą logikę, procedur pojedynczej właściwości (`Get` i `Set`) przez nadrzędne właściwość nie może być `Private`.</span><span class="sxs-lookup"><span data-stu-id="ab0a7-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
