<span data-ttu-id="eaa6d-103">Instrukcja [Get](../language-reference/statements/get-statement.md) lub [Set](../language-reference/statements/set-statement.md) w `NotOverridable` właściwości zawiera `Private` słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="eaa6d-103">A [Get Statement](../language-reference/statements/get-statement.md) or [Set Statement](../language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="eaa6d-104">Poniższy wiersz przyczyny wyjaśnia dlaczego `NotOverridable` i `Private` nie można go łączyć w [instrukcji Property](../language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="eaa6d-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../language-reference/statements/property-statement.md):</span></span>  
  
1. <span data-ttu-id="eaa6d-105">Właściwość lub procedura, która nie przesłania właściwości klasy bazowej lub procedury, ma domyślne ustawienie [NotOverridable](../language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="eaa6d-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../language-reference/modifiers/notoverridable.md).</span></span>  
  
2. <span data-ttu-id="eaa6d-106">Jednak właściwość lub procedura w klasie pochodnej, która zastępuje właściwość klasy bazowej lub procedurę, ma domyślne [ustawienie wartości](../language-reference/modifiers/overridable.md)zastępczej.</span><span class="sxs-lookup"><span data-stu-id="eaa6d-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="eaa6d-107">Aby zakończyć hierarchię przesłaniania, możesz ją zadeklarować `NotOverridable` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="eaa6d-108">Jest to jedyny kontekst, w którym można użyć programu `NotOverridable` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="eaa6d-109">Oznacza to, że można użyć `NotOverridable` tylko w połączeniu z [zastępowaniem](../language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="eaa6d-109">That is, you can use `NotOverridable` only in combination with [Overrides](../language-reference/modifiers/overrides.md).</span></span>  
  
3. <span data-ttu-id="eaa6d-110">Jeśli właściwość lub procedura klasy bazowej jest zadeklarowana jako [prywatna](../language-reference/modifiers/private.md), Klasa pochodna nie może przesłonić tej właściwości lub procedury, ponieważ nie może uzyskać do niej dostępu.</span><span class="sxs-lookup"><span data-stu-id="eaa6d-110">If a base class property or procedure is declared [Private](../language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="eaa6d-111">Z tego powodu nie można używać `Private` programu w połączeniu z `Overridable` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4. <span data-ttu-id="eaa6d-112">Aby przesłonić właściwość lub procedurę, zastępowanie właściwości lub procedury musi mieć nie tylko identyczny podpis, ale również ten sam poziom dostępu.</span><span class="sxs-lookup"><span data-stu-id="eaa6d-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="eaa6d-113">Oznacza to, że zastępowanie właściwości lub procedury nie może określać `Private` , ponieważ właściwość lub procedura, której nie można określić `Private` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5. <span data-ttu-id="eaa6d-114">Ponieważ można określić `NotOverridable` tylko dla zastępujące właściwość lub procedurę, nie można połączyć jej z `Private` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="eaa6d-115">Z tego samego powodu procedury poszczególnych właściwości ( `Get` i `Set` ) właściwości zastępującej nie mogą być `Private` .</span><span class="sxs-lookup"><span data-stu-id="eaa6d-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
