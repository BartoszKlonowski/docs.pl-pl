 <span data-ttu-id="afa29-104">Celem `Global` jest umożliwienie kodowi dostępu do przestrzeni nazw na poziomie głównym z wnętrza struktury przestrzeni nazw, która blokuje przestrzeń nazw na poziomie głównym.</span><span class="sxs-lookup"><span data-stu-id="afa29-104">The purpose of `Global` is to allow your code to access a root-level namespace from inside a namespace structure that has blocked the root-level namespace.</span></span>
