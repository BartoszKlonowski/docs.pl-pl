<span data-ttu-id="5f324-103">[Globalne](../programming-guide/program-structure/namespaces.md#global-keyword-in-fully-qualified-names) słowo kluczowe pojawia się w kontekście innym niż dostęp do przestrzeni nazw.</span><span class="sxs-lookup"><span data-stu-id="5f324-103">The [Global](../programming-guide/program-structure/namespaces.md#global-keyword-in-fully-qualified-names) keyword appears in a context other than accessing a namespace.</span></span>  
  
 <span data-ttu-id="5f324-104">Celem `Global` jest umożliwienie kodowi dostępu do przestrzeni nazw na poziomie głównym z wnętrza struktury przestrzeni nazw, która blokuje przestrzeń nazw na poziomie głównym.</span><span class="sxs-lookup"><span data-stu-id="5f324-104">The purpose of `Global` is to allow your code to access a root-level namespace from inside a namespace structure that has blocked the root-level namespace.</span></span>
