# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>element " \<modifier> " nie jest prawidłowy w deklaracji zmiennej lokalnej

`Dim`Instrukcja wewnątrz procedury zawiera nieprawidłowe słowo kluczowe, np `Overloads` ..  
  
 **Identyfikator błędu:** BC30247  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieprawidłowe słowo kluczowe z `Dim` instrukcji.
