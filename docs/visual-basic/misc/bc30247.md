# <a name="ltmodifiergt-is-not-valid-on-a-local-variable-declaration"></a>"&lt;modyfikator&gt;" nie jest prawidłowy w deklaracji zmiennej lokalnej
A `Dim` instrukcji wewnątrz procedury zawiera nieprawidłowe słowo kluczowe, takie jak `Overloads`.  
  
 **Identyfikator błędu:** BC30247  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nieprawidłowe słowo kluczowe z `Dim` instrukcji.
