# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>&#39; domyślne &#39; można zastosować do tylko jednej nazwy właściwości w &lt;typu&gt;
Próbowano zastosować `Default` do więcej niż jednej nazwy właściwości w obrębie klasy, interfejsu lub struktury.  
  
 **Identyfikator błędu:** BC30359  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszystkie oprócz jednego z `Default` deklaracji.
