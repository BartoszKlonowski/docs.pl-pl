# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>"Default" można zastosować do tylko jednej nazwy właściwości w \<typ >
Podjęto próbę zastosowania `Default` do więcej niż jednej nazwy właściwości w obrębie klasy, interfejsu lub struktury.  
  
 **Identyfikator błędu:** BC30359  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszystkie oprócz jednego z `Default` deklaracji.
