# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>Element "default" można zastosować tylko do jednej nazwy właściwości w elemencie \<type>

Podjęto próbę zastosowania `Default` do więcej niż jednej nazwy właściwości w ramach klasy, interfejsu lub struktury.  
  
 **Identyfikator błędu:** BC30359  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń wszystkie oprócz jednej `Default` deklaracji.
