# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>Udostępnione &#39; Sub nowy &#39; Nie można zadeklarować jako &#39; &lt;specyfikator&gt;&#39;
Podjęto próbę użycia specyfikator w `Shared Sub New` deklaracji, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30480  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
