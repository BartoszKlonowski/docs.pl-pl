# <a name="shared-sub-new-cannot-be-declared-specifier"></a>Nie można zadeklarować udostępnianego "Sub New" " \<specifier> "

Próbowano użyć specyfikatora w `Shared Sub New` deklaracji, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30480  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
