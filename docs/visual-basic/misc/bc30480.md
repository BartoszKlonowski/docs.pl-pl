# <a name="shared-sub-new-cannot-be-declared-specifier"></a>Udostępnione 'Sub New' nie można zadeklarować jako '\<specyfikator >'
Podjęto próbę użycia specyfikatora w `Shared Sub New` deklaracji, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30480  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
