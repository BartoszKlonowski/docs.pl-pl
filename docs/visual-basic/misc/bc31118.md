# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>"End AddHandler" musi być pierwszą instrukcją w wierszu
`End AddHandler` Instrukcji następuje separatorem instrukcji dwukropek (:). `End AddHandler` musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31118  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
