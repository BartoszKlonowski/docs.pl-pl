# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>Element "End AddHandler" musi być pierwszą instrukcją w wierszu
`End AddHandler`Instrukcja następuje po dwukropku (:) Separator instrukcji. `End AddHandler`musi być jedyną instrukcją w wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31118  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Rozdziel instrukcje wielokrotne w różnych wierszach.
