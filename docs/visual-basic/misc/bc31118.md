# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39; End AddHandler &#39; musi być pierwszą instrukcją w wierszu
`End AddHandler` Instrukcji następuje separatora instrukcji dwukropka (:). `End AddHandler`musi być jedyną instrukcją w jego wierszu źródłowym.  
  
 **Identyfikator błędu:** BC31118  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Oddziel wiele instrukcji na różne wiersze.
