# <a name="bracketed-identifier-is-missing-closing-"></a>Identyfikator w nawiasach kwadratowych brakuje zamykającego "]"
Nawiasy kwadratowe w nazwie ucieczki musi występować w dopasowane pary.  
  
 **Identyfikator błędu:** BC30034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Na końcu nazwy o zmienionym znaczeniu, należy umieścić nawias zamykający.
