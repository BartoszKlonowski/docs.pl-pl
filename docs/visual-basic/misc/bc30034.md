# <a name="bracketed-identifier-is-missing-closing-"></a>Brak zamykającego identyfikatora "]" w nawiasie klamrowym
Nawiasy w nazwie o zmienionym znaczeniu muszą wystąpić w dopasowanych parach.  
  
 **Identyfikator błędu:** BC30034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Umieść nawias zamykający na końcu zmienionej nazwy.
