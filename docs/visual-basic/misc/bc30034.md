# <a name="bracketed-identifier-is-missing-closing-3939"></a>Brak identyfikatora w nawiasach kwadratowych zamknięcia &#39;] &#39;
Nawiasy kwadratowe w nazwie zmienionym musi występować w par.  
  
 **Identyfikator błędu:** BC30034  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Umieść zamykający nawias kwadratowy na końcu nazwy zmienionym znaczeniu.
