# <a name="try-must-have-at-least-one-catch-or-a-finally"></a>Blok try musi zawierać co najmniej jeden element "Catch" lub "finally"

`Try`Blok musi zawierać blok `Finally` lub co najmniej jeden `Catch` blok przed `End Try` instrukcją.  
  
 **Identyfikator błędu:** BC30030  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Catch` blok or `Finally` w `Try` bloku.
