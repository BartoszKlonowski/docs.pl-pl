# <a name="try-must-have-at-least-one-39catch39-or-a-39finally39"></a>Blok TRY musi zawierać co najmniej jeden &#39; CATCH &#39; i &#39; Na koniec &#39;
A `Try` bloku musi zawierać albo `Finally` blok lub co najmniej jeden `Catch` blokować przed `End Try` instrukcji.  
  
 **Identyfikator błędu:** BC30030  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Catch` lub `Finally` zablokować w `Try` bloku.
