# <a name="try-must-have-at-least-one-catch-or-a-finally"></a>Spróbuj musi mieć co najmniej jedną instrukcję "Catch" lub "Finally"
A `Try` bloku musi zawierać albo `Finally` bloku lub co najmniej jeden `Catch` blokować przed `End Try` instrukcji.  
  
 **Identyfikator błędu:** BC30030  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Catch` lub `Finally` blokowania w ramach `Try` bloku.
