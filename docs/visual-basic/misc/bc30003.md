# <a name="39next39-expected"></a>&#39; Dalej &#39; Oczekiwano
`On Error Resume` Konstrukcja odbywa się bez zawierania `Next`.  
  
 **Identyfikator błędu:** BC30003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Next` — słowo kluczowe na końcu `On Error` instrukcji.
