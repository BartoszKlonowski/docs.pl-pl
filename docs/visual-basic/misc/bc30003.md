# <a name="next-expected"></a>Oczekiwano instrukcji "Next"

`On Error Resume`Konstrukcja występuje bez końcowej `Next` .  
  
 **Identyfikator błędu:** BC30003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Next` słowo kluczowe do końca `On Error` instrukcji.
