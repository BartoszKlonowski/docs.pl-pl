# <a name="next-expected"></a>Oczekiwano "Dalej"
`On Error Resume` Konstrukcji odbywa się bez zawierania `Next`.  
  
 **Identyfikator błędu:** BC30003  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Next` — słowo kluczowe do końca `On Error` instrukcji.
