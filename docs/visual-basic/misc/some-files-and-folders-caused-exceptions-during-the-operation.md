# <a name="some-files-and-folders-caused-exceptions-during-the-operation"></a>Niektóre pliki i foldery spowodowały wyjątki podczas operacji

Katalogi i pliki mogą spowodować wyjątki w przypadku próby wykonania operacji, ale użytkownik nie ma wymaganych uprawnień.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj uprawnienia niezbędne do uzyskiwania dostępu do wszystkich plików i folderów.
