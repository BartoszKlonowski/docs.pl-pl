# <a name="file-format-not-valid"></a>Nieprawidłowy format pliku
Podjęto próbę załadowania pliku do składnika, ale format danych w pliku był niezgodny ze składnikiem lub próbowano zapisać dane składnika do pliku, ale format danych jest niezgodny z formatem pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją składnika, aby określić właściwy format danych pliku dyskowego.  
  
2. Zapoznaj się z dokumentacją składnika, aby określić, czy zapewnia on obsługę konwersji z jednego formatu na inny.
