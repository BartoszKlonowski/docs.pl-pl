# <a name="file-format-not-valid"></a>Nieprawidłowy format pliku
Albo próbował załadować plik do składnika, ale format danych w pliku nie są zgodne ze składnikiem, lub próbowano zapisać dane składnika w pliku, ale format danych nie są zgodne z formatem pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zapoznaj się z dokumentacją dla składnika do określenia prawidłowego formatu dla dysku danych plików.  
  
2. Zapoznaj się z dokumentacją w celu określenia, czy zapewnia obsługę dla konwersji z jednego formatu do innego składnika.
