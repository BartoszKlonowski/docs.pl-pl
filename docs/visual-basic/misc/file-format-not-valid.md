# <a name="file-format-not-valid"></a>Nieprawidłowy format pliku
Albo próbowano załadować plik do składnika, ale formatu danych w pliku nie są zgodne ze składnikiem, lub próbowano zapisać danych do pliku, ale format danych jest niezgodny z formatem pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zajrzyj do dokumentacji w celu określenia prawidłowego formatu danych plików dysku składnika.  
  
2.  Zajrzyj do dokumentacji w celu ustalenia, czy zapewnia obsługę dla konwersji z jednego formatu do innego składnika.
