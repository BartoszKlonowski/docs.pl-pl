# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Wszystkie parametry muszą mieć jawnie określone, jeśli istnieją
Ma łączyć różnego typu deklaracji w funkcji.  
  
 **Identyfikator błędu:** BC30529  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie parametry są jawnie określone typy.
