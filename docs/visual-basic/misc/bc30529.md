# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Wszystkie parametry muszą zostać jawnie wpisane, jeśli dowolny
Ma łączyć różnego rodzaju deklaracje typu funkcji.  
  
 **Identyfikator błędu:** BC30529  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że wszystkie parametry są jawnie określone typy.
