# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Wszystkie parametry muszą być jawnie wpisane, jeśli istnieją

W funkcji są mieszane różne typy deklaracji typu.  
  
 **Identyfikator błędu:** BC30529  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że wszystkie parametry zostały jawnie wpisane.
