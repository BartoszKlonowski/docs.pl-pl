# <a name="stop-statement-encountered"></a>Napotkano instrukcję Stop

`Stop`Napotkano instrukcję, która wstrzymuje wykonywanie.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `Stop` instrukcję.
