# <a name="stop-statement-encountered"></a>Stop — instrukcja napotkano
A `Stop` instrukcji, co wstrzymuje wykonywanie, napotkano.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń `Stop` instrukcji.
