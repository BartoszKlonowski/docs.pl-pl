# <a name="local-variable-ltvariablenamegt-is-already-declared-in-the-current-block"></a>Zmienna lokalna "&lt;nazwa_zmiennej&gt;" jest już zadeklarowana w bieżącym bloku
Dwie zmienne o takiej samej nazwie są zadeklarowane w tym samym bloku (która może być procedurą lub blokiem kontroli taką jak `If...Then`).  
  
 **Identyfikator błędu:** BC30288  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś jeden deklaracje do innego bloku lub zmienić nazwę jednej ze zmiennych.
