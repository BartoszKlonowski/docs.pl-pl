# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>Zmienna lokalna " \<variablename> " jest już zadeklarowana w bieżącym bloku

Dwie zmienne o tej samej nazwie są zadeklarowane w tym samym bloku (co może być procedurą lub blokiem sterującym, takim jak `If...Then` ).  
  
 **Identyfikator błędu:** BC30288  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś jedną z deklaracji do innego bloku lub Zmień nazwę jednej ze zmiennych.
