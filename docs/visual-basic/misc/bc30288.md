# <a name="local-variable-39ltvariablenamegt39-is-already-declared-in-the-current-block"></a>Zmienna lokalna &#39; &lt;nazwa_zmiennej&gt; &#39; jest już zadeklarowana w bieżącym bloku
Dwie zmienne o takiej samej nazwie jest zadeklarowany w tym samym bloku (może to procedurę lub blok kontroli takich jak `If...Then`).  
  
 **Identyfikator błędu:** BC30288  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeden z deklaracji przeniesiony na inny blok, lub zmień nazwę jednego z zmienne.
