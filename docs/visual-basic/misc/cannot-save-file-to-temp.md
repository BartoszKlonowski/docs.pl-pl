# <a name="cannot-save-file-to-temp"></a>Nie można zapisać pliku do katalogu TEMP

Składnik nie może znaleźć katalogu o nazwie TEMP lub dysk lub partycja zawierająca katalog TEMP brak wystarczającej ilości miejsca do zapisania informacji.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Utwórz katalog o nazwie "TEMP" i Ustaw zmienną środowiskową TEMP równą jej ścieżce.  
  
2. Zwolnij miejsce na dysku, usuwając zbędne pliki, lub Utwórz katalog TEMP na innej partycji i Ustaw zmienną środowiskową TEMP równą jej ścieżce.
