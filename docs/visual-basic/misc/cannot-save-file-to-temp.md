# <a name="cannot-save-file-to-temp"></a>Nie można zapisać pliku Temp
Składnik nie można znaleźć katalogu o nazwie TEMP albo dysku lub partycji zawierającej katalogu TEMP nie ma wystarczającej ilości miejsca, aby zapisać informacje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Utwórz katalog o nazwie "TEMP" i ustawić tymczasowej zmiennej środowiskowej równa jego ścieżki.  
  
2.  Zwiększyć ilość miejsca na dysku przez usuwanie niepotrzebnych plików, lub Utwórz katalogu TEMP na innej partycji, a następnie ustawić tymczasowej zmiennej środowiskowej równa jego ścieżki.
