# <a name="cannot-save-file-to-temp"></a>Nie można zapisać pliku do katalogu TEMP
Składnik nie może znaleźć katalogu o nazwie TEMP albo dysk lub partycję zawierającą katalogu TEMP nie ma wystarczającą ilość miejsca, aby zapisać informacje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Utwórz katalog o nazwie "TEMP" i ustaw TEMP zmiennej środowiskowej ścieżki.  
  
2.  Uczynić miejsca na dysku przez wymazywanie niepotrzebne pliki lub utworzyć katalogu TEMP na innej partycji i równa TEMP zmiennej środowiskowej ścieżki.
