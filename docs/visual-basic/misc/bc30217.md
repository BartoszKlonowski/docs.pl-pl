# <a name="string-constant-expected"></a>Oczekiwano ciągu
Znak inny niż znak podwójnego cudzysłowu (") występuje, gdy od kontekstu stałą typu string. Jedną z możliwych przyczyn to Brak z ciągu znaków cudzysłowu.  
  
 **Identyfikator błędu:** BC30217  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, czy ciąg jest poprawnie ujęty w znaki podwójnego cudzysłowu.  
  
2.  Wprowadź stałą typu string.
