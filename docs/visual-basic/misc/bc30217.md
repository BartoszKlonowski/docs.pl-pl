# <a name="string-constant-expected"></a>Oczekiwano stałej ciągu

Znak inny niż znak podwójnego cudzysłowu (") występuje, gdy kontekst wymaga stałej ciągu. Jedną z możliwych przyczyn jest to, że w ciągu brakuje cudzysłowu.  
  
 **Identyfikator błędu:** BC30217  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że ciąg jest prawidłowo ujęty w znaki podwójnego cudzysłowu.  
  
2. Podaj stałą typu String.
