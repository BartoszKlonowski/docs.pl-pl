# <a name="string-constant-expected"></a>Oczekiwano ciągu
Znak inny niż znak podwójnego cudzysłowu (") występuje, gdy kontekstu stałą typu string. Jedną z możliwych przyczyn jest Brak z ciągu znaków cudzysłowu.  
  
 **Identyfikator błędu:** BC30217  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że ciąg prawidłowo jest ujęta w znaki podwójnego cudzysłowu.  
  
2.  Wprowadź stałą typu string.
