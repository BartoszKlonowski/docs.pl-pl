# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>Element "End Event" musi być poprzedzony odpowiadającym mu elementem "Custom Event"
`End Event`Instrukcja występuje bez odpowiedniej `Custom Event` instrukcji. `End Event`musi być poprzedzona odpowiadającą jej `Custom Event` instrukcją.  
  
 **Identyfikator błędu:** BC31123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że poprzednia `Custom Event` instrukcja jest prawidłowa i poprawna.
