# <a name="39end-event39-must-be-preceded-by-a-matching-39custom-event39"></a>&#39;End Event&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;zdarzenie niestandardowe&#39;
`End Event` Występuje instrukcja bez odpowiadającego `Custom Event` instrukcji. `End Event` musi być poprzedzona odpowiadającą `Custom Event` instrukcji.  
  
 **Identyfikator błędu:** BC31123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że występuje przed `Custom Event` instrukcji jest prawidłowa i poprawna.
