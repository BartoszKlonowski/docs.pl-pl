# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>"End Event" musi być poprzedzona przez pasującego "zdarzenie niestandardowe"
`End Event` Instrukcji odbywa się bez odpowiadającego `Custom Event` instrukcji. `End Event` musi być poprzedzona odpowiadającą `Custom Event` instrukcji.  
  
 **Identyfikator błędu:** BC31123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że występuje przed `Custom Event` instrukcja jest prawidłowa i poprawna.
