# <a name="exponent-is-not-valid"></a>Wykładnik nie jest prawidłowy
Podjęto próbę użycia wykładnik nie jest prawidłowy z literałem liczb zmiennoprzecinkowych. Ten błąd zazwyczaj powoduje po typu zmiennoprzecinkowego literał kłóci rozmiar do przeprowadzenia literał zmiennoprzecinkowych.  
  
 **Identyfikator błędu:** BC30495  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Kontrola: typ literału zmiennoprzecinkowych i zmień go do prawidłowego typu.
