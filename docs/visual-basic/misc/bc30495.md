# <a name="exponent-is-not-valid"></a>Wykładnik jest nieprawidłowy
Podjęto próbę użycia wykładnik jest nieprawidłowy w przypadku zmiennoprzecinkowych literału. Ten błąd zazwyczaj powoduje po kłóci rozmiar wymagany do przechowania zmiennoprzecinkowych literał typu zmiennoprzecinkowego literału.  
  
 **Identyfikator błędu:** BC30495  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź typ zmiennoprzecinkowych literału i zmienić ją na prawidłowym typem.
