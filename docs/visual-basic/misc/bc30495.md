# <a name="exponent-is-not-valid"></a>Wykładnik jest nieprawidłowy
Podjęto próbę użycia wykładnika, która nie jest prawidłowa w przypadku literału zmiennoprzecinkowego. Ten błąd zazwyczaj powstaje, gdy typ literału zmiennoprzecinkowego ma wartość szanse o rozmiarze wymaganym do przechowywania literału zmiennoprzecinkowego.  
  
 **Identyfikator błędu:** BC30495  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź typ literału zmiennoprzecinkowego i zmień go na prawidłowy typ.
