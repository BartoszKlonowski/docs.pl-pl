# <a name="implements-not-valid-in-modules"></a>"Implements" nie jest prawidłowy w modułach
`Implements` Instrukcji lub klauzuli występuje wewnątrz modułu. Moduły nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30231  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Implements` instrukcji lub klauzuli lub wpisz ponownie moduł jako klasa.
