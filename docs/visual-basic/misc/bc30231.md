# <a name="implements-not-valid-in-modules"></a>Element "Implements" nie jest prawidłowy w modułach

`Implements`Instrukcja lub klauzula występuje wewnątrz modułu. Moduły nie mogą implementować interfejsów.  
  
 **Identyfikator błędu:** BC30231  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Implements` instrukcję lub klauzulę lub ponownie wpisz moduł jako klasę.
