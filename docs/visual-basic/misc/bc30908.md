# <a name="type-typename-cannot-inherit-from-a-type-nested-within-it"></a><span data-ttu-id="204f2-102">Typ '\<typename >' nie może dziedziczyć z typu w nim zagnieżdżony</span><span class="sxs-lookup"><span data-stu-id="204f2-102">Type '\<typename>' cannot inherit from a type nested within it</span></span>
<span data-ttu-id="204f2-103">Definicja klasy lub interfejsu zawiera [instrukcję Inherits](../../visual-basic/language-reference/statements/inherits-statement.md) , która określa typ, który jest w nim zagnieżdżony.</span><span class="sxs-lookup"><span data-stu-id="204f2-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="204f2-104">Dziedziczenie musi być liniowe, niecykliczne.</span><span class="sxs-lookup"><span data-stu-id="204f2-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="204f2-105">Typ nie może dziedziczyć po typie, który od niego dziedziczy.</span><span class="sxs-lookup"><span data-stu-id="204f2-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="204f2-106">Ograniczenie powiązane polega na tym, że typ nie może dziedziczyć z typu, który jeszcze nie został zdefiniowany.</span><span class="sxs-lookup"><span data-stu-id="204f2-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="204f2-107">Dziedziczenie obejmuje możliwość ponownego wykorzystania elementów członkowskich klasy podstawowej, co z kolei wymaga zdefiniowania tych elementów członkowskich.</span><span class="sxs-lookup"><span data-stu-id="204f2-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="204f2-108">W związku z tym Visual Basic nie może kompilować kodu, takiego jak Poniższy przykład.</span><span class="sxs-lookup"><span data-stu-id="204f2-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>  
  
```vb  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```  
  
 <span data-ttu-id="204f2-109">**Identyfikator błędu:** BC30908</span><span class="sxs-lookup"><span data-stu-id="204f2-109">**Error ID:** BC30908</span></span>
