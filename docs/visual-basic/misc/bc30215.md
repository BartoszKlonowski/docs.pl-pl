# <a name="39sub39-or-39function39-expected"></a>&#39; Sub &#39; i &#39; Funkcja &#39; Oczekiwano
A `Declare` instrukcji nie został określony żaden `Sub` lub `Function`. `Declare`identyfikuje zewnętrznej procedury i musi określać rodzaj procedury.  
  
 **Identyfikator błędu:** BC30215  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj albo `Sub` lub `Function` słowa kluczowego `Declare` instrukcji.
