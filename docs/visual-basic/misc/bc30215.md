# <a name="sub-or-function-expected"></a>Oczekiwano instrukcji "sub" lub "Function"
A `Declare` instrukcji nie został określony żaden `Sub` lub `Function`. `Declare` identyfikuje zewnętrznej procedury i należy określić rodzaj procedury.  
  
 **Identyfikator błędu:** BC30215  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj opcję `Sub` lub `Function` słowa kluczowego `Declare` instrukcji.
