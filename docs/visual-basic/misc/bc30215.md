# <a name="sub-or-function-expected"></a>Oczekiwano instrukcji "Sub" lub "Function"

`Declare`Instrukcja nie określa żadnej `Sub` lub `Function` . `Declare` Identyfikuje zewnętrzną procedurę i musi określić, jakiego rodzaju procedurę.  
  
 **Identyfikator błędu:** BC30215  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Sub` `Function` słowo kluczowe or do `Declare` instrukcji.
