# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Nie można użyć nazwy funkcji agregującej ze znakiem typu
Znak typu został uwzględniony na końcu nazwy funkcji agregującej.  
  
 **Identyfikator błędu:** BC36617  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń znak typu z nazwy funkcji agregującej.
