# <a name="suspend-request-is-pending"></a>Wstrzymanie żądania jest w stanie oczekiwania
W debugerze programu Visual Studio wyrażenie określa wywołanie procedury, ale jest to żądanie wstrzymania wątku.  
  
 Debuger nie próbuje wywoływanie procedury w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30947  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli to możliwe należy określić źródło żądania wstrzymania wątku, aby zapobiec jego ponowne wykonanie.  
  
-   Zakończenie, a następnie ponownie uruchom debugowanie.
