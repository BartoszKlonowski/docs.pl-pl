# <a name="suspend-request-is-pending"></a>Oczekujące żądanie wstrzymania
W debugerze programu Visual Studio wyrażenie określa wywołanie procedury, ale istnieje żądanie wstrzymania wątku.  
  
 Debuger nie próbuje wywołać procedury w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30947  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli to możliwe, Określ źródło żądania zawieszenia wątku, aby można było zapobiec jego ponownemu występowaniu.  
  
- Przerwij i ponownie uruchom debugowanie.
