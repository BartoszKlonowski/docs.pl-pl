# <a name="suspend-request-is-pending"></a>Wstrzymywanie istnieje oczekujące żądanie
W debugerze programu Visual Studio wyrażenie określa wywołania procedury, ale ma żądania wstrzymania wątku.  
  
 Debuger nie próbuje wywołać procedurę w wątku, który nie jest aktywny.  
  
 **Identyfikator błędu:** BC30947  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli to możliwe Ustal źródło żądania wstrzymania wątku, dzięki czemu użytkownik może uniemożliwić ich ponowne wykonanie.  
  
- Zakończ i uruchom ponownie debugowanie.
