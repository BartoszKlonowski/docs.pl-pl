# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Wyrażenia zapytań nie są prawidłowe w oknach debugowania
Zapytania LINQ został wprowadzony w **czujki** lub **Immediate** okno podczas sesji debugowania. Zapytania nie są dostępne w oknach debugowania.  
  
 **Identyfikator błędu:** BC36609
