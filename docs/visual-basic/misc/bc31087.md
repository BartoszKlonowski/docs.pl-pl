# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Nie można określić modyfikatorów tablicy zmienna i jej typ
Modyfikatory tablic są obecne w zmiennej i jego typu, wskazujący tablicy tablic.  
  
 **Identyfikator błędu:** BC31087  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń modyfikator tablicy ze specyfikatora typu.
