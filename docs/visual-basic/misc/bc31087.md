# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modyfikatory tablicy nie mogą być określone jednocześnie w zmiennej i jej typie

Modyfikatory tablic są obecne zarówno dla zmiennej, jak i jej typu, co wskazuje na tablicę tablic.  
  
 **Identyfikator błędu:** BC31087  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń Modyfikator tablicy ze specyfikatora typu.
