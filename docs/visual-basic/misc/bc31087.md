# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modyfikatory tablic nie można określić jednocześnie w zmiennej i jej typu
Modyfikatory tablic znajdują się na zmienną i jej typ, wskazuje tablicy tablic.  
  
 **Identyfikator błędu:** BC31087  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń modyfikator tablicy z specyfikatora typu.
