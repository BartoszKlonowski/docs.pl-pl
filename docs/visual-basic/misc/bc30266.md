# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-they-have-different-access-levels"></a>&#39; &lt;declaration1&gt;&#39; nie zastąpienie &#39;&lt; declaration2&gt;&#39; ponieważ mają one różne poziomy dostępu
Deklaracja procedura lub właściwość próbuje przesłonić odziedziczonego elementu o tej samej nazwie, ale określa różne ułatwień dostępu niż odziedziczonego elementu. Dziedziczone z elementu ułatwień dostępu, takich jak `Public` lub `Private`, muszą zostać zachowane w zastępowanie.  
  
 **Identyfikator błędu:** BC30266  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień dostępność elementu zastępowanie odpowiadać odziedziczonego elementu.
