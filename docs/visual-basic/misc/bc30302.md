# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Wpisz znak &#39; &lt;znak&gt;&#39; nie można użyć w deklaracji z typem jawnym
Użyto znaku typu, takich jak `$`, gdy jawne określenie typu.  
  
 **Identyfikator błędu:** BC30302  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń znak typ lub usuń `As <Type>` klauzuli z deklaracji.
