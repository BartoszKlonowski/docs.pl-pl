# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>\<character>Nie można użyć znaku typu "" w deklaracji z typem jawnym
Użyto znaku typu, takiego jak `$` , gdy jawnie określono typ.  
  
 **Identyfikator błędu:** BC30302  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń znak typu lub Usuń `As <Type>` klauzulę z deklaracji.
