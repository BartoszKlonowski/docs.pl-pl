# <a name="type-character-ltcharactergt-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Wpisz znak "&lt;znak&gt;" nie można używać w deklaracji z typem jawnym
Użyto znaku typu, takie jak `$`, gdy jawne określenie typu.  
  
 **Identyfikator błędu:** BC30302  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń znak typu lub usuń `As <Type>` klauzuli po deklaracji.
