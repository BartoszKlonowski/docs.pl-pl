# <a name="properties-cannot-be-declared-39ltmodifiernamegt39"></a>Nie można zadeklarować właściwości jako &#39; &lt;modifiername&gt;&#39;
Istnieje błąd w składni stosowanej w `Property` deklaracji. Błąd ten zwykle wskazuje, że określone słowa kluczowe, takie jak `WithEvents`, nie można używać w `Property` deklaracji.  
  
 **Identyfikator błędu:** BC30639  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź składnię `Property` deklaracji.
