# <a name="properties-cannot-be-declared-modifiername"></a>Właściwości nie mogą być zadeklarowane jako " \<modifiername> "
Wystąpił błąd w składni używanej w `Property` deklaracji. Ten błąd zazwyczaj wskazuje, że `WithEvents` w deklaracji nie można użyć określonego słowa kluczowego, takiego jak `Property` .  
  
 **Identyfikator błędu:** BC30639  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź składnię dla `Property` deklaracji.
