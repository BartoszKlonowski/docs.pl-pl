# <a name="properties-cannot-be-declared-ltmodifiernamegt"></a>Nie można zadeklarować właściwości jako "&lt;modifiername&gt;"
Występuje błąd w składnią używaną w `Property` deklaracji. Ten błąd zazwyczaj oznacza, że określone słowa kluczowe, takie jak `WithEvents`, nie można używać w `Property` deklaracji.  
  
 **Identyfikator błędu:** BC30639  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź składnię `Property` deklaracji.
