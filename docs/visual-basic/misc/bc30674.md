# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;SyncLock&#39;
`SyncLock` bloków zaczyna się od `SyncLock` — słowo kluczowe i kończyć się `End``SyncLock` utworzenia.  
  
 **Identyfikator błędu:** BC30674  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `SyncLock` bloku rozpoczyna się od `SyncLock` instrukcji.
