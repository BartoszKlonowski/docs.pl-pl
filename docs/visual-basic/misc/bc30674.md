# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>Element "End SyncLock" musi być poprzedzony odpowiadającym mu elementem "SyncLock"
`SyncLock`bloki zaczynają się od `SyncLock` słowa kluczowego i kończą się na `End SyncLock` konstrukcję.  
  
 **Identyfikator błędu:** BC30674  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że `SyncLock` blok rozpoczyna się od `SyncLock` instrukcji.
