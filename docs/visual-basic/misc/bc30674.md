# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>"End SyncLock" musi być poprzedzona pasującego "SyncLock"
`SyncLock` bloki zaczynają się od `SyncLock` — słowo kluczowe oraz kończyć się `End SyncLock` konstruowania.  
  
 **Identyfikator błędu:** BC30674  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `SyncLock` bloku zaczyna się od `SyncLock` instrukcji.
