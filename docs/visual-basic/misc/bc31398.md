# <a name="type-characters-are-not-allowed-on-imports-aliases"></a>Znaki typu nie są dozwolone w aliasy Importy
Import aliasu w `Imports` instrukcja zawiera co najmniej jeden znak typu identyfikator.  
  
 Alias importu musi być prawidłową nazwą języka Visual Basic. Dozwolone znaki nie zawierać znaki typu identyfikator (`%`, `&`, `@`, `!`, `#`, i `$`). Zobacz [Zadeklarowane nazwy elementów](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identyfikator błędu:** BC31398  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń typ identyfikatora znak lub znaki z aliasu importu.
