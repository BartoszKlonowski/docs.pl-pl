# <a name="constants-must-have-a-value"></a>Stałe muszą mieć wartość
Stała jest zadeklarowana bez przypisywania go wartości.  
  
 **Identyfikator błędu:** BC30438  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przypisz wartości do stałej.
