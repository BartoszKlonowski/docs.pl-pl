# <a name="cannot-delete-a-registry-hive"></a>Nie można usunąć gałęzi rejestru

Podjęto próbę usunięcia gałęzi rejestru. Gałęzie to klucze rejestru najwyższego poziomu, takie jak `HKEY_CURRENT_USER` i i `HKEY_LOCAL_MACHINE` nie można ich usunąć.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź klucz rejestru, który chcesz usunąć, i upewnij się, że podano poprawną ścieżkę.
