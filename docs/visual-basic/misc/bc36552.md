## <a name="example"></a><span data-ttu-id="ad17e-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="ad17e-110">Example</span></span>  
 <span data-ttu-id="ad17e-111">Pierwszy parametr w następującym przykładzie oznacza to, że `Print` metoda jest rozszerzeniem `String` — typ danych.</span><span class="sxs-lookup"><span data-stu-id="ad17e-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="ad17e-112">Gdy rozszerzenia ma zostać wywołana metoda następujące parametru `str` w metodzie jest powiązany z `greeting`, wystąpienie `String` wywołującym `Print`.</span><span class="sxs-lookup"><span data-stu-id="ad17e-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="ad17e-113">Kompilator użyje `greeting` jako argument do metody rozszerzenia `Print`.</span><span class="sxs-lookup"><span data-stu-id="ad17e-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```
