## <a name="example"></a><span data-ttu-id="3eeca-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="3eeca-110">Example</span></span>  
 <span data-ttu-id="3eeca-111">Pierwszy parametr w poniższym przykładzie wskazuje, że metoda `Print` rozszerza typ danych `String`.</span><span class="sxs-lookup"><span data-stu-id="3eeca-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="3eeca-112">Gdy Metoda rozszerzenia jest wywoływana w następujący sposób, parametr `str` w metodzie jest powiązany z `greeting`, wystąpieniem `String`, który wywołuje `Print`.</span><span class="sxs-lookup"><span data-stu-id="3eeca-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="3eeca-113">Kompilator użyje `greeting` jako argumentu metody rozszerzającej `Print`.</span><span class="sxs-lookup"><span data-stu-id="3eeca-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```vb  
greeting.Print()  
```
