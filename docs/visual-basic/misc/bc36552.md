## <a name="example"></a><span data-ttu-id="b51bf-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="b51bf-110">Example</span></span>  
 <span data-ttu-id="b51bf-111">Pierwszy parametr w poniższym przykładzie wskazuje, że `Print` metoda rozszerza `String` Typ danych.</span><span class="sxs-lookup"><span data-stu-id="b51bf-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

```vb  
End Sub  
```
