# <a name="ltkeywordgt-is-valid-only-within-a-class"></a>"&lt;— słowo kluczowe&gt;" jest prawidłowy tylko w obrębie klasy
Słowo kluczowe powiązane z klasami, takich jak `Me` lub `MyClass`, jest używany poza definicją klasy.  
  
 **Identyfikator błędu:** BC32002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli kod przy użyciu słowa kluczowego obejmuje wystąpień klas, przenieś go do implementacji klasy.  
  
-   Jeśli kod przy użyciu słowa kluczowego nie ma zastosowania do klasy, Usuń nieprawidłowe słowo kluczowe.
