# <a name="39ltkeywordgt39-is-valid-only-within-a-class"></a>&#39;&lt;słowo kluczowe&gt; &#39; jest prawidłowy tylko wewnątrz klasy
Słowo kluczowe powiązane z klasami, takich jak `Me` lub `MyClass`, jest używana poza definicją klasy.  
  
 **Identyfikator błędu:** BC32002  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeśli kod za pomocą słowa kluczowego obejmuje wystąpień klasy, przenieś go do implementacji klasy.  
  
-   Jeśli kod za pomocą słowa kluczowego nie ma zastosowania do klasy, Usuń nieprawidłowe słowo kluczowe.
