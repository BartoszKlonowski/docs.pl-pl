# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>"\<name >" jest już zadeklarowany jako\<deklaracji > "w tym \<declarationspace >
Dwa elementy programowania są deklarowane przy użyciu tej samej nazwie w tej samej przestrzeni deklaracji (klasy, modułu, interfejsu lub struktury).  
  
 **Identyfikator błędu:** BC30260  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieść jeden z deklaracji przestrzeni różnych deklaracji lub zmień nazwę jednego z elementów zadeklarowane.
