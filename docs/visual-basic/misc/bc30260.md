# <a name="39ltnamegt39-is-already-declared-as-39ltdeclarationgt39-in-this-ltdeclarationspacegt"></a>&#39; &lt;nazwa&gt;&#39; jest już zadeklarowany jako &#39;&lt; Deklaracja&gt;&#39; w tym &lt;declarationspace&gt;
Dwa elementy programowania są deklarowane jako o takiej samej nazwie w tej samej przestrzeni deklaracji (klasy, modułu, interfejsu lub struktury).  
  
 **Identyfikator błędu:** BC30260  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Jeden z deklaracji przeniesiony do innej deklaracji przestrzeni lub zmień nazwę jednego z elementów zadeklarowane.
