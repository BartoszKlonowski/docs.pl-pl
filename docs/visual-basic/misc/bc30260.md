# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>element " \<name> " jest już zadeklarowany jako " \<declaration> " w tym elemencie \<declarationspace>

Dwa elementy programowania są zadeklarowane z tą samą nazwą w tym samym obszarze deklaracji (klasy, modułu, interfejsu lub struktury).  
  
 **Identyfikator błędu:** BC30260  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś jedną z deklaracji do innej przestrzeni deklaracji lub Zmień nazwę jednego z zadeklarowanych elementów.
