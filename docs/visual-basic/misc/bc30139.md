# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Błąd podczas ustawiania opcji manifestu zestawu: &lt;komunikat o błędzie&gt;
Kompilator Visual Basic wywołuje Assembly Linker (Al.exe, znany także jako Alink) do generowania manifestu zestawu. Konsolidator zgłosił błąd wyszukiwania pliku klucza lub nazwa klucza do podpisywania zestawu.  
  
 **Identyfikator błędu:** BC30139  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź komunikat o błędzie w cudzysłowach i konsultować się błędów wymienionych na [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) uzyskać dokładniejsze objaśnienie i porady.  
  
2.  Jeśli błąd będzie się powtarzać, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
