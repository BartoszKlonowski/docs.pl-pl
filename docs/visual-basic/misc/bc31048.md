# <a name="properties-with-no-required-parameters-cannot-be-declared-39default39"></a>Nie można zadeklarować jako właściwości bez żadnych wymaganych parametrów &#39; domyślne &#39;
Właściwość została określona bez wymaganych parametrów, ale z `Default` modyfikator.  
  
 **Identyfikator błędu:** BC31048  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Deklarowanie wymaganych parametrów dla właściwości domyślnych.  
  
-   Usuń `Default` modyfikator z definicji właściwości.
