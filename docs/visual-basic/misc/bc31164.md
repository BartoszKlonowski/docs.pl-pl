# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Oczekiwane dopasowanie podwójnego cudzysłowu zamykającego dla wartości atrybutu XML
Wartość atrybutu, która jest rozdzielana podwójnym cudzysłowem, nie zawiera znaku cudzysłowu zamykającego.  
  
 **Identyfikator błędu:** BC31164  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak podwójnego cudzysłowu zamykającego, aby rozdzielić wartość atrybutu.
