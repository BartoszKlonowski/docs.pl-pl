# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Oczekiwano pasującego podwójnego cudzysłowu zamykającego dla wartości atrybutu XML
Wartość atrybutu, który jest rozdzielone podwójnych cudzysłowów prostych brakuje zamykającego znaku cudzysłowu.  
  
 **Identyfikator błędu:** BC31164  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znak cudzysłowu zamykającego do ograniczania wartości atrybutu.
