# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Oczekiwano dopasowania zamykanie podwójny cudzysłów wartości atrybutu XML
Wartość atrybutu, który sterujący jest ujęty w znaki podwójnego cudzysłowu brakuje zamykającego znaku cudzysłowu.  
  
 **Identyfikator błędu:** BC31164  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak cudzysłowu zamykającego do ograniczania wartości atrybutu.
