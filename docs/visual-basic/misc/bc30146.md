# <a name="unable-to-sign-assembly-error-message"></a>Nie można podpisać zestawu: \<komunikat o błędzie >
Kompilator Visual Basic wywołuje Assembly Linker (Al.exe, znany także jako Alink) do generowania manifestu zestawu. Konsolidator zgłosił błąd podpisywania zestawu po jej utworzeniu. Może to być z powodu nieprawidłowego klucza.  
  
 **Identyfikator błędu:** BC30146  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź komunikat o błędzie w cudzysłowach i zapoznaj się z tematem [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) uzyskać dokładniejsze objaśnienie i porady.  
  
2. Jeśli błąd będzie się powtarzać, należy zebrać informacje dotyczące okoliczności i powiadom pomoc techniczna firmy Microsoft.
