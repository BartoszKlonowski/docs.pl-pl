# <a name="unable-to-sign-assembly-error-message"></a>Nie można podpisać zestawu: \<error message>

Kompilator Visual Basic wywołuje konsolidator zestawu (Al.exe, znany również jako ALink) w celu wygenerowania zestawu z manifestem. Konsolidator zgłosił błąd podczas podpisywania zestawu po jego utworzeniu. Może to być spowodowane nieprawidłowym kluczem.  
  
 **Identyfikator błędu:** BC30146  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź cytowany komunikat o błędzie i zapoznaj się z tematem  [Al.exe (Konsolidator zestawu)](../../framework/tools/al-exe-assembly-linker.md) , aby uzyskać dalsze wyjaśnienia i porady.  
  
2. Jeśli błąd będzie się powtarzać, Zbierz informacje o okolicznościach i powiadom usługi pomocy technicznej firmy Microsoft.
