# <a name="expected-closing-39gt39-for-xml-processor-instruction"></a>Oczekiwano zamykających znaków &#39;? &gt; &#39; dla instrukcji procesora XML
Literał brakuje zamykającego wymagane instrukcji przetwarzania XML `?>` znaków.  
  
 **Identyfikator błędu:** BC31160  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamknięcia `?>` znaki na końcu instrukcji przetwarzania XML literału.
