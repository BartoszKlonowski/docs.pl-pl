# <a name="expected-closing--for-xml-processor-instruction"></a>Oczekiwano zamykającego "? >" dla instrukcji procesora XML
W literale instrukcji przetwarzania XML brakuje wymaganych znaków zamykających `?>` .  
  
 **Identyfikator błędu:** BC31160  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagane znaki zamykające `?>` na końcu literału instrukcji przetwarzania XML.
