# <a name="expected-closing-gt-for-xml-processor-instruction"></a>Oczekiwano zamykającego "? &gt;"dla instrukcji procesor XML
Literał brakuje zamykającego wymagane instrukcji przetwarzania XML `?>` znaków.  
  
 **Identyfikator błędu:** BC31160  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj wymagane zamknięcia `?>` znaki do końca instrukcji przetwarzania XML literału.
