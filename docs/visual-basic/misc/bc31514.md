# <a name="project-projectname-cannot-generate-a-reference-to-file-filename"></a>Projekt '\<nazwa_projektu >' nie można wygenerować odwołania do pliku '\<nazwa pliku >'
Projekt ' <`projectname`> 'nie można wygenerować odwołania do pliku ' <`filename`>'. Może być konieczne, Usuń i ponownie dodać to odwołanie, a następnie wykonaj ponownej pełnej kompilacji rozwiązania.  
  
 Wystąpił błąd podczas tworzenia odwołania do pliku.  
  
 **Identyfikator błędu:** BC31514  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń odwołanie, klikając prawym przyciskiem myszy element z **odwołania do projektu** w **Eksploratora rozwiązań** i klikając **Usuń**.  
  
2. Przywróć odwołania przez kliknięcie prawym przyciskiem myszy **odwołania** w **Eksploratora rozwiązań** , a następnie klikając polecenie **Dodaj odwołanie**.
