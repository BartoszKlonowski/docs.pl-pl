# <a name="project-39ltprojectnamegt39-cannot-generate-a-reference-to-file-39ltfilenamegt39"></a>Projekt &#39; &lt;projectname&gt;&#39; nie można wygenerować odwołania do pliku &#39;&lt; Nazwa pliku&gt;&#39;
Projekt ' <`projectname`>' nie można wygenerować odwołania do pliku "<`filename`>". Należy usunąć i ponownie dodać to odwołanie, a następnie wykonaj ponownej pełnej kompilacji rozwiązania.  
  
 Wystąpił błąd podczas tworzenia odwołania do pliku.  
  
 **Identyfikator błędu:** BC31514  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Usuń odwołanie, klikając prawym przyciskiem myszy element z **odwołania do projektu** w **Eksploratora rozwiązań** i klikając **Usuń**.  
  
2.  Przywróć odwołanie przez kliknięcie prawym przyciskiem myszy **odwołania** w **Eksploratora rozwiązań** , a następnie klikając polecenie **Dodaj odwołanie**.
