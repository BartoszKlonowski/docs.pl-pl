# <a name="project-projectname-cannot-generate-a-reference-to-file-filename"></a>Projekt " \<projectname> " nie może wygenerować odwołania do pliku " \<filename> "

Projekt "<`projectname`>" nie może wygenerować odwołania do pliku "<`filename`>". Może być konieczne usunięcie i ponowne dodanie tego odwołania, a następnie przeprowadzenie pełnej odbudowy rozwiązania.  
  
 Wystąpił błąd podczas tworzenia odwołania do pliku.  
  
 **Identyfikator błędu:** BC31514  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń odwołanie, klikając prawym przyciskiem myszy element z **odwołań projektu** w **Eksplorator rozwiązań** i klikając polecenie **Usuń**.  
  
2. Przywróć odwołanie, klikając prawym przyciskiem myszy pozycję **odwołania** w **Eksplorator rozwiązań** a następnie klikając pozycję **Dodaj odwołanie**.
