# <a name="registry-key-39ltkeynamegt39-could-not-be-found"></a>Klucz rejestru &#39; &lt;keyname&gt;&#39; nie można odnaleźć
Nie można odnaleźć klucza rejestru.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź pisownię nazwy klucza.  
  
2.  Upewnij się, że masz wystarczające uprawnienia dostępu do klucza.
