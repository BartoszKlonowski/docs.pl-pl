# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>Atrybut XML "Attribute1" musi występować przed atrybutem XML "attribute2"
Atrybuty w literale dokumentu XML są określone w nieprawidłowej kolejności. Prawidłowa kolejność atrybutów dla literału dokumentu XML jest oparta na specyfikacji XML 1,0. Na przykład `encoding` atrybut literału dokumentu XML musi występować bezpośrednio po `version` atrybucie.  
  
 **Identyfikator błędu:** BC31157  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zaktualizuj kolejność atrybutów dla literału dokumentu XML przy użyciu wytycznych określonych w specyfikacji XML 1,0.
