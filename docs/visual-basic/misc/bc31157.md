# <a name="xml-attribute-39attribute139-must-appear-before-xml-attribute-39attribute239"></a>Atrybut XML &#39; attribute1 &#39; musi występować przed atrybutem XML &#39; attribute2 &#39;
Atrybuty w literał dokumentu XML są określone w nieprawidłowa kolejność. Nieprawidłowy atrybut kolejności literał dokumentu XML jest oparta na Specyfikacja XML 1.0. Na przykład `encoding` atrybutu literał dokumentu XML musi występować zaraz po `version` atrybutu.  
  
 **Identyfikator błędu:** BC31157  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zaktualizować atrybut kolejności literał dokumentu XML przy użyciu wskazówek określonych w specyfikacji XML 1.0.
