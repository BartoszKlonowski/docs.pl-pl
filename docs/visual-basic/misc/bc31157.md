# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>Atrybut XML "attribute1" musi występować przed atrybutem XML "attribute2"
Atrybuty literał dokumentu XML są określone w nieprawidłowa kolejność. Nieprawidłowy atrybut kolejności literał dokumentu XML jest oparta na Specyfikacja XML 1.0. Na przykład `encoding` atrybutu literał dokumentu XML musi bezpośrednio po `version` atrybutu.  
  
 **Identyfikator błędu:** BC31157  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Aktualizuj porządek atrybutu literał dokumentu XML za pomocą wytycznych określonych w specyfikacji XML 1.0.
