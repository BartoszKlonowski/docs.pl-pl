# <a name="39redim39-cannot-change-the-number-of-dimensions-of-an-array"></a>&#39;ReDim&#39; nie można zmienić liczby wymiarów tablicy
Podjęto próbę użycia `ReDim` zmienić rangę tablicy (liczba wymiarów). `ReDim` Instrukcji można używać, aby zmienić rozmiar co najmniej jeden wymiar tablicy, która została już formalnie zadeklarowana, ale nie można zmienić rangę tablicy.  
  
 **Identyfikator błędu:** BC30415  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że mają rangę zamiast rozmiary tablicy i jeśli jest to możliwe, `Dim` Aby zadeklarować nowej tablicy z odpowiednią pozycję.
