# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-lttypenamegt-is-not-a-delegate-type"></a><span data-ttu-id="74077-102">Wyrażenie lambda nie można przekonwertować na "&lt;typename&gt;"ponieważ"&lt;typename&gt;" nie jest typem delegowanym</span><span class="sxs-lookup"><span data-stu-id="74077-102">Lambda expression cannot be converted to '&lt;typename&gt;' because '&lt;typename&gt;' is not a delegate type</span></span>
<span data-ttu-id="74077-103">Wyrażenia lambda może służyć gdzie delegaci są prawidłowe.</span><span class="sxs-lookup"><span data-stu-id="74077-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="74077-104">Mogą być konwertowane na typy delegatów zgodne, ale nie do jakichkolwiek innych typów.</span><span class="sxs-lookup"><span data-stu-id="74077-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="74077-105">Na przykład można zdefiniować typ obiektu delegowanego i przypisać wyrażenia lambda lub Wyślij do wyrażenia lambda jako argument <xref:System.Func%601> parametru.</span><span class="sxs-lookup"><span data-stu-id="74077-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="74077-106">Te przykłady przedstawiono w poniższym kodzie.</span><span class="sxs-lookup"><span data-stu-id="74077-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```
