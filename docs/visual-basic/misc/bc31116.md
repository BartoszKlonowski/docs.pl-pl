# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>Deklaracja "RemoveHandler" musi być zakończona odpowiadającą jej instrukcją "End RemoveHandler"

`RemoveHandler`Deklaracja musi kończyć się `End RemoveHandler` instrukcją.  
  
 **Identyfikator błędu:** BC31116  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź, czy `RemoveHandler` Deklaracja została zakończona `End RemoveHandler` instrukcją.
