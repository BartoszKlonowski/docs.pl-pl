# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>Deklaracja "RemoveHandler" musi kończyć się pasującego "End RemoveHandler"
A `RemoveHandler` deklaracji musi się kończyć `End RemoveHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31116  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `RemoveHandler` deklaracji kończy się `End RemoveHandler` instrukcji.
