# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39; RemoveHandler &#39; Deklaracja musi kończyć się pasujący &#39; End RemoveHandler &#39;
A `RemoveHandler` deklaracji musi kończyć się `End RemoveHandler` instrukcji.  
  
 **Identyfikator błędu:** BC31116  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy `RemoveHandler` kończy deklaracji `End RemoveHandler` instrukcji.
