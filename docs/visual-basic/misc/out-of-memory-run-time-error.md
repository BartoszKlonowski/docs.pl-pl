# <a name="out-of-memory-visual-basic-run-time-error"></a>Za mało pamięci (błąd czasu wykonywania Visual Basic)
Większa ilość pamięci jest wymagana, niż jest dostępne.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zamknij zbędnych aplikacji, dokumentów lub plików źródłowych, które są otwarte.  
  
2.  Podział modułów dużych lub procedury na mniejsze.  
  
3.  Wyeliminowanie przerwanie i pozostań rezydentne programy.  
  
4.  Usuń niepotrzebne sterowniki urządzeń.  
  
5.  Zmniejsz liczbę zmiennych publicznych.
