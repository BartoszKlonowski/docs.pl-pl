# <a name="object-doesn39t-support-named-arguments"></a>Obiekt&#39;t obsługę argumenty nazwane
Próbowano uzyskać dostęp do metody, których nie obsługuje obiektu argumentami nazwanymi.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Określ argumenty różnych pozycjach podczas uzyskiwania dostępu do metody dla tego obiektu. Zobacz dokumentację obiektu Aby uzyskać więcej informacji na pozycji argumentu i typów.
