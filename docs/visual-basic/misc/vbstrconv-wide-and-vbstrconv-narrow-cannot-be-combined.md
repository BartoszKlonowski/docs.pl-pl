# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>Nie można łączyć elementów VbStrConv. Wide i VbStrConv. Narrow

Aplikacja próbuje połączyć `VbStrConv` elementy członkowskie wyliczenia `Wide` i `Narrow` , które wzajemnie się wykluczają.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Usuń albo `VbStrConv.Wide` `VbStrConv.Narrow` .
