# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>Argumenty VbStrConv.Wide a VbStrConv.Narrow nelze kombinovat
Twoja aplikacja próbuje połączyć `VbStrConv` elementów członkowskich wyliczenia `Wide` i `Narrow`, które wzajemnie się wykluczają.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Albo usuń `VbStrConv.Wide` lub `VbStrConv.Narrow`.
