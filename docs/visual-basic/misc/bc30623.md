# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>"End Namespace" musi być poprzedzona pasującego "Namespace"
Namespace bloki zaczynają się od `Namespace` instrukcji i kończyć się `End Namespace` konstruowania.  
  
 **Identyfikator błędu:** BC30623  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że w bloku przestrzeni nazw rozpoczyna się od `Namespace` instrukcji.
