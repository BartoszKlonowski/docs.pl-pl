# <a name="39end-namespace39-must-be-preceded-by-a-matching-39namespace39"></a>&#39;End Namespace&#39; musi być poprzedzona odpowiadającą jej instrukcją &#39;Namespace&#39;
Namespace bloków zaczyna się od `Namespace` instrukcji i kończyć się `End Namespace` utworzenia.  
  
 **Identyfikator błędu:** BC30623  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że blok przestrzeń nazw rozpoczyna się od `Namespace` instrukcji.
