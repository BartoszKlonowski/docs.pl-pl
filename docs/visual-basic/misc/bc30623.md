# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>Element "End namespace" musi być poprzedzony odpowiadającym mu elementem "namespace"
Bloki przestrzeni nazw zaczynają się od `Namespace` instrukcji i kończą się na `End Namespace` konstrukcję.  
  
 **Identyfikator błędu:** BC30623  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że blok przestrzeni nazw zaczyna się od `Namespace` instrukcji.
