# <a name="default-cannot-be-combined-with-specifier"></a>Nie można łączyć instrukcji "default" z elementem " \<specifier> "
Podjęto próbę połączenia `Default` słowa kluczowego ze specyfikatorem, takim jak `Shared` lub `Private` , który jest nieprawidłowy w tym kontekście.  
  
 **Identyfikator błędu:** BC30490  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń `Default` .  
  
     -lub-  
  
- Usuń specyfikator powodujący konflikt.
