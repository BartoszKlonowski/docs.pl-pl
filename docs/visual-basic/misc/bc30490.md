# <a name="default-cannot-be-combined-with-ltspecifiergt"></a>"Default" nie można połączyć z "&lt;specyfikator&gt;"
Próbujesz połączyć `Default` — słowo kluczowe ze specyfikatorem, takich jak `Shared` lub `Private`, to znaczy nie są prawidłowe w tym kontekście.  
  
 **Identyfikator błędu:** BC30490  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Default`.  
  
     —lub—  
  
-   Usuń specyfikator powodujące konflikt.
