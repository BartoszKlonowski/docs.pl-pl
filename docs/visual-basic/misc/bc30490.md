# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39; domyślne &#39; Nie można połączyć z &#39; &lt;specyfikator&gt;&#39;
Próbujesz połączyć `Default` — słowo kluczowe ze specyfikatorem, takich jak `Shared` lub `Private`, która jest nieprawidłowy w tym kontekście.  
  
 **Identyfikator błędu:** BC30490  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Default`.  
  
     —lub—  
  
-   Usuń specyfikator powodujące konflikt.
