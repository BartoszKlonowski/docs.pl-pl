<span data-ttu-id="2cc04-103">Pętle zagnieżdżone kończą się za pomocą jednego `Next` instrukcję, która określa więcej zmiennych pętli nie istnieją pętle w zagnieżdżonych, jak w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="2cc04-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```  
   For J = 1 To 5  
      ' ...

```
