# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="7ac76-102">Instrukcja "Next" zawiera więcej zmiennych, niż istnieją pasujące instrukcje "for"</span><span class="sxs-lookup"><span data-stu-id="7ac76-102">'Next' statement names more variables than there are matching 'For' statements</span></span>

<span data-ttu-id="7ac76-103">Pętle zagnieżdżone są kończone z pojedynczą `Next` instrukcją, która określa więcej zmiennych pętli niż pętle w gnieździe, tak jak w poniższym przykładzie:</span><span class="sxs-lookup"><span data-stu-id="7ac76-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```vb  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```  
  
 <span data-ttu-id="7ac76-104">**Identyfikator błędu:** BC32037</span><span class="sxs-lookup"><span data-stu-id="7ac76-104">**Error ID:** BC32037</span></span>
