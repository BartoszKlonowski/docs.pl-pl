# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Projekt &#39; &lt;projectname&gt; &#39; wykonuje pośrednie odwołanie do zestawu &#39; &lt;assemblyname&gt; &#39; zawierającą &#39; &lt;classname&gt;&#39;
Projekt "\<projectname >" wykonuje pośrednie odwołanie do zestawu "\<assemblyname >" zawierającego "\<classname >". Dodaj odwołanie do '\<assemblyname > "do projektu.  
  
 Ten projekt zawiera elementy członkowskie, które są zależne od zestawu, który nie zawiera odwołania do projektu.  
  
 **Identyfikator błędu:** BC31515  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj odwołanie do zestawu, klikając prawym przyciskiem myszy **odwołania** w **Eksploratora rozwiązań** , a następnie klikając polecenie **Dodaj odwołanie**.
