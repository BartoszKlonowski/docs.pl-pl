# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Projekt " \<projectname> " wykonuje pośrednie odwołanie do zestawu " \<assemblyname> ", który zawiera element " \<classname> "

Projekt " \<projectname> " wykonuje pośrednie odwołanie do zestawu " \<assemblyname> ", który zawiera element " \<classname> ". Dodaj odwołanie do elementu " \<assemblyname> " do projektu.  
  
 Ten projekt zawiera elementy członkowskie, które są zależne od zestawu, który nie znajduje się w odwołaniach projektu.  
  
 **Identyfikator błędu:** BC31515  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Aby dodać odwołanie do zestawu, kliknij prawym przyciskiem myszy pozycję **odwołania** w **Eksplorator rozwiązań** a następnie kliknij pozycję **Dodaj odwołanie**.
