# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Projekt '\<nazwa_projektu >' wykonuje pośrednie odwołanie do zestawu '\<assemblyname >' zawierającą '\<nazwa_klasy >'
Projekt '\<nazwa_projektu >' wykonuje pośrednie odwołanie do zestawu '\<assemblyname >' zawierającą '\<nazwa_klasy >'. Dodaj odwołanie do '\<nazwa_zestawu > 'do projektu.  
  
 Ten projekt zawiera elementy członkowskie, które są zależne od zestawu, który nie jest zawarta w odwołaniach projektu.  
  
 **Identyfikator błędu:** BC31515  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj odwołanie do zestawu, klikając prawym przyciskiem myszy **odwołania** w **Eksploratora rozwiązań** , a następnie klikając polecenie **Dodaj odwołanie**.
