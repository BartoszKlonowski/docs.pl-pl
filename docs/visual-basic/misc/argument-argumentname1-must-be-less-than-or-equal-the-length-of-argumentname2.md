# <a name="argument-39ltargumentname1gt39-must-be-less-than-or-equal-the-length-of-argument-39ltargumentname2gt39"></a>Argument &#39; &lt;argumentname1&gt;&#39; musi być mniejsza niż lub równy długości argumentu &#39;&lt; argumentname2&gt;&#39;
Długość argumentu dostarczonego przekracza długość argumentu, który zastępuje go.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Mniejsze niż lub równa długości drugi argument należy długość pierwszego argumentu.
