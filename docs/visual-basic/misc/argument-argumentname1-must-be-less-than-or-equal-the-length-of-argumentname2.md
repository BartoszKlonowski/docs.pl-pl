# <a name="argument-argumentname1-must-be-less-than-or-equal-the-length-of-argument-argumentname2"></a>Argument " \<argumentname1> " musi być mniejszy lub równy długości argumentu " \<argumentname2> "

Długość podanego argumentu przekracza długość argumentu, który zastępuje.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wprowadź długość pierwszego argumentu mniejszą lub równą długości drugiego argumentu.
