# <a name="argument-ltargumentname1gt-must-be-less-than-or-equal-the-length-of-argument-ltargumentname2gt"></a>Argument "&lt;argumentname1&gt;" musi być mniejsza lub równa długości argumentu "&lt;argumentname2&gt;"
Długość podanej argumentu przekracza długość argumentu, który zastępuje go.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Długość pierwszego argumentu należy mniejsza niż długość drugiego argumentu.
