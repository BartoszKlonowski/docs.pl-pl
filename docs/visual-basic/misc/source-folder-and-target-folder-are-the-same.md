# <a name="source-folder-and-target-folder-are-the-same"></a>Folder źródłowy i folder docelowy są takie same

Podjęto próbę skopiowania lub przeniesienia pliku, ale określony katalog źródłowy i katalog docelowy są takie same. Może to być spowodowane błędnym analizowaniem ścieżki pliku.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że podano poprawne Katalogi źródłowe i docelowe.
