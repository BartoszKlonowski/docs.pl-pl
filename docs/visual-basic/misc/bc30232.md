# <a name="implemented-type-must-be-an-interface"></a>Implementowany typ musi być interfejsem
`Implements` Instrukcji nie określa interfejs. Klasę można zaimplementować tylko interfejs.  
  
 **Identyfikator błędu:** BC30232  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Upewnij się, że nazwa interfejsu jest poprawna.  
  
2.  Jeśli instrukcja Określa klasę, należy rozważyć użycie `Inherits` instrukcji.
