# <a name="implemented-type-must-be-an-interface"></a>Zaimplementowany typ musi być interfejsem
`Implements`Instrukcja nie określa interfejsu. Klasa może implementować tylko interfejs.  
  
 **Identyfikator błędu:** BC30232  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że nazwa interfejsu jest wpisana poprawnie.  
  
2. Jeśli instrukcja Określa klasę, należy rozważyć użycie `Inherits` instrukcji.
