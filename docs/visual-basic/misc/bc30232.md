# <a name="implemented-type-must-be-an-interface"></a>Wdrożony typ musi być interfejsem
`Implements` Instrukcji nie określa interfejs. Klasa może implementować tylko interfejs.  
  
 **Identyfikator błędu:** BC30232  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Upewnij się, że poprawna nazwa interfejsu.  
  
2. Jeśli instrukcja Określa klasę, należy wziąć pod uwagę przy użyciu `Inherits` instrukcji.
