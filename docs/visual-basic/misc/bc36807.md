# <a name="xml-elements-cannot-be-selected-from-type-type"></a><span data-ttu-id="157a7-102">Nie można wybierać elementów XML z typu "Type"</span><span class="sxs-lookup"><span data-stu-id="157a7-102">XML elements cannot be selected from type 'type'</span></span>
<span data-ttu-id="157a7-103">Odwołanie do elementu podrzędnego XML odwołuje się do obiektu, który nie jest typu <xref:System.Xml.Linq.XElement> , <xref:System.Xml.Linq.XDocument> lub `IEnumerable(Of XElement)` .</span><span class="sxs-lookup"><span data-stu-id="157a7-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="157a7-104">Aby uzyskać więcej informacji, zobacz [Właściwość osi elementu podrzędnego XML](../language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="157a7-104">For more information, see [XML Child Axis Property](../language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="157a7-105">**Identyfikator błędu:** BC36807</span><span class="sxs-lookup"><span data-stu-id="157a7-105">**Error ID:** BC36807</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="157a7-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="157a7-106">To correct this error</span></span>
