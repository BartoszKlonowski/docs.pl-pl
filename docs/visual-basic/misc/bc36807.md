<span data-ttu-id="bfe34-103">Element podrzędny XML został wskazany dla obiektu, który nie jest typu <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, lub `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="bfe34-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="bfe34-104">Aby uzyskać więcej informacji, zobacz [właściwości osi elementu podrzędnego XML](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="bfe34-104">For more information, see [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb

Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="bfe34-105">**Identyfikator błędu:** BC36807</span><span class="sxs-lookup"><span data-stu-id="bfe34-105">**Error ID:** BC36807</span></span>
