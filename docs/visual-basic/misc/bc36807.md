# <a name="xml-elements-cannot-be-selected-from-type-type"></a><span data-ttu-id="26758-102">Nie można wybierać elementów XML z typu "type"</span><span class="sxs-lookup"><span data-stu-id="26758-102">XML elements cannot be selected from type 'type'</span></span>
<span data-ttu-id="26758-103">Element podrzędny XML został wskazany dla obiektu, który nie jest typu <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, lub `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="26758-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="26758-104">Aby uzyskać więcej informacji, zobacz [właściwości osi elementu podrzędnego XML](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="26758-104">For more information, see [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="26758-105">**Identyfikator błędu:** BC36807</span><span class="sxs-lookup"><span data-stu-id="26758-105">**Error ID:** BC36807</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="26758-106">Aby poprawić ten błąd</span><span class="sxs-lookup"><span data-stu-id="26758-106">To correct this error</span></span>
