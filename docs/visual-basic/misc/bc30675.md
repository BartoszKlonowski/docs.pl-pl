# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>Instrukcja "SyncLock" musi być zakończona odpowiadającą jej instrukcją "End SyncLock"
`SyncLock`bloki zaczynają się od `SyncLock` słowa kluczowego i kończą się na `End SyncLock` konstrukcję.  
  
 **Identyfikator błędu:** BC30675  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Upewnij się, że `SyncLock` blok jest zakończony przez `End SyncLock` konstrukcję.
