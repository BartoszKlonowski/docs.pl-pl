# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39; SyncLock &#39; Instrukcja musi kończyć się pasujący &#39; End SyncLock &#39;
`SyncLock`bloków zaczyna się od `SyncLock` — słowo kluczowe i kończyć się `End SyncLock` utworzenia.  
  
 **Identyfikator błędu:** BC30675  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `SyncLock` bloku kończy się wyrazem `End SyncLock` utworzenia.
