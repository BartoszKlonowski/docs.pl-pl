# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>Instrukcji "SyncLock" musi kończyć się pasującego "End SyncLock"
`SyncLock` bloki zaczynają się od `SyncLock` — słowo kluczowe oraz kończyć się `End SyncLock` konstruowania.  
  
 **Identyfikator błędu:** BC30675  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że `SyncLock` kończy blok `End SyncLock` konstruowania.
