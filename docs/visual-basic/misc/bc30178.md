# <a name="specifier-is-duplicated"></a>Specyfikator jest zduplikowana
Słowo kluczowe występuje więcej niż raz w tej samej deklaracji.  
  
 **Identyfikator błędu:** BC30178  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowane wystąpienia słowa kluczowego z deklaracji.
