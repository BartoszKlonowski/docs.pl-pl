# <a name="specifier-is-duplicated"></a>Specyfikator jest zduplikowany
Słowo kluczowe występuje więcej niż jeden raz w tej samej deklaracji.  
  
 **Identyfikator błędu:** BC30178  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń zduplikowane wygląd słowa kluczowego z deklaracji.
