# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Nie można znaleźć konstruktora delegata dla typu " \<typename> ", który jest niezbędny dla implementacji delegatów

Zestaw zawierający typ może być uszkodzony lub nieprawidłowy.  
  
 **Identyfikator błędu:** BC31074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy zestaw zawierający ten typ jest prawidłowy.  
  
2. Skompiluj ponownie program, aby zobaczyć, czy błąd jest powtarzany.  
  
3. Jeśli błąd będzie nadal występował, Zapisz swoją pracę i ponownie uruchom program Visual Studio.
