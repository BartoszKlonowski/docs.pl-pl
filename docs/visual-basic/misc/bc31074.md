# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Konstruktora delegowanego dla typu &#39; &lt;typename&gt;&#39;, niezbędne do implementacji obiektów delegowanych nie można odnaleźć.
Zestaw zawierający ten typ może być uszkodzony lub nieprawidłowy.  
  
 **Identyfikator błędu:** BC31074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy zestaw zawierający ten typ jest nieprawidłowy.  
  
2.  Skompiluj program, aby zobaczyć, jeśli ten błąd wystąpi ponownie.  
  
3.  Jeśli błąd będzie się powtarzał, należy zapisać pracę i ponownie uruchom program Visual Studio.
