# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Konstruktor delegata dla typu "\<typename >", jest to niezbędne do wykonania delegatów, nie można odnaleźć
Zestawu zawierającego typ może być uszkodzony lub nie jest prawidłowy.  
  
 **Identyfikator błędu:** BC31074  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Sprawdź, czy zestaw zawierający ten typ jest prawidłowa.  
  
2. Skompiluj program ponownie, aby zobaczyć, jeśli błąd będzie się powtarzać.  
  
3. Jeśli błąd będzie się powtarzał, Zapisz swoją pracę i uruchom ponownie program Visual Studio.
