# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'Zmiennej WithEvents nie zgłaszała wszelkie zdarzenia wystąpienia, które są dostępne dla'\<containername > '
Obiekt, do którego odwołuje się, nie zgłasza zdarzeń dostępnych.  
  
 **Identyfikator błędu:** BC30600  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1. Zmodyfikuj Modyfikatory dostępu dla zdarzenia lub wybierz zdarzenie, które jest dostępne w bieżącym kontekście.
