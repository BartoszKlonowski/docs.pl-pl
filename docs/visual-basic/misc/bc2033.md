# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Błąd osadzanie manifestu Win32: Opcja/win32manifest powoduje konflikt z/nowin32manifest
Kompilator Visual Basic został przekazany zarówno `/win32manifest` — opcja kompilatora i `/nowin32manifest` — opcja kompilatora. Tylko jeden z tych opcji mogą być przekazywane do kompilatora języka Visual Basic.  
  
 **Identyfikator błędu:** BC2033  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Albo usuń `/win32manifest` — opcja kompilatora lub `/nowin32manifest` — opcja kompilatora.
