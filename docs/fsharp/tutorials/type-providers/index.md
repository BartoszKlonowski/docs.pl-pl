# <a name="type-providers"></a><span data-ttu-id="e2c54-103">Dostawcy typów</span><span class="sxs-lookup"><span data-stu-id="e2c54-103">Type Providers</span></span>

<span data-ttu-id="e2c54-104">Dostawca typów języka F# to składnik, który dostarcza typy, właściwości i metody używane w programie.</span><span class="sxs-lookup"><span data-stu-id="e2c54-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="e2c54-105">Generowanie dostawców typów, co to są znane jako **podane typy**, które są generowane przez kompilator F # i opierają się na zewnętrznego źródła danych.</span><span class="sxs-lookup"><span data-stu-id="e2c54-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="e2c54-106">Na przykład dostawcę typów F # do języka SQL, można wygenerować typy reprezentujące tabele i kolumny w relacyjnej bazie danych.</span><span class="sxs-lookup"><span data-stu-id="e2c54-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="e2c54-107">W rzeczywistości, co jest [SQLProvider](https://fsprojects.github.io/SQLProvider/) jest dostawcy typów.</span><span class="sxs-lookup"><span data-stu-id="e2c54-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="e2c54-108">Pod warunkiem, że typy są zależne od parametrów wejściowych do dostawcy typu.</span><span class="sxs-lookup"><span data-stu-id="e2c54-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="e2c54-109">Takie dane wejściowe mogą być przykładowe źródło danych (na przykład plik schematu JSON) adres URL wskazujący bezpośrednio do usługi zewnętrznej lub parametry połączenia ze źródłem danych.</span><span class="sxs-lookup"><span data-stu-id="e2c54-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="e2c54-110">Dostawca typów może również zagwarantować, że grupy typów są tylko rozszerzone na żądanie; oznacza to, że one rozszerzane, jeśli typy odwołuje się program.</span><span class="sxs-lookup"><span data-stu-id="e2c54-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="e2c54-111">Umożliwia to bezpośrednią integrację na żądanie przestrzeni informacji o wielkiej skali, takich jak sieciowe rynki danych, w sposób silnie typizowany.</span><span class="sxs-lookup"><span data-stu-id="e2c54-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="e2c54-112">Dostawcy typów generatywną i wymazane</span><span class="sxs-lookup"><span data-stu-id="e2c54-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="e2c54-113">Dostawcy typów są dostępne w dwóch formach: Generatywną i wymazane.</span><span class="sxs-lookup"><span data-stu-id="e2c54-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="e2c54-114">Generatywną dostawców typów generuje typy, które mogą być zapisywane jako typów .NET do zestawu, w którym są tworzone.</span><span class="sxs-lookup"><span data-stu-id="e2c54-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="e2c54-115">Uprawnienie pozwoli im do użycia z kodu w innych zestawach.</span><span class="sxs-lookup"><span data-stu-id="e2c54-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="e2c54-116">Oznacza to, że wpisane reprezentujący źródło danych zazwyczaj musi być taki, który jest możliwe do reprezentowania z typami .NET.</span><span class="sxs-lookup"><span data-stu-id="e2c54-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="e2c54-117">Wymazywanie dostawców typów generuje typy, które mogą być używane tylko w zestawie lub projektu, które są generowane na podstawie.</span><span class="sxs-lookup"><span data-stu-id="e2c54-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="e2c54-118">Typy są efemeryczne; oznacza to nie są zapisywane do zestawu i nie mogą być używane przez kod w innych zestawach.</span><span class="sxs-lookup"><span data-stu-id="e2c54-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="e2c54-119">Mogą one zawierać *opóźnione* członków, co umożliwia typy użycia dostarczane z obszaru potencjalnie nieskończonej informacji.</span><span class="sxs-lookup"><span data-stu-id="e2c54-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="e2c54-120">Są one przydatne przy użyciu małego podzbioru źródła danych dużych i połączonych ze sobą.</span><span class="sxs-lookup"><span data-stu-id="e2c54-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
