# <a name="type-providers"></a><span data-ttu-id="c0f9c-103">Dostawcy typów</span><span class="sxs-lookup"><span data-stu-id="c0f9c-103">Type Providers</span></span>

<span data-ttu-id="c0f9c-106">Na przykład F # typ dostawcy dla serwera SQL może wygenerować typy reprezentujące tabel i kolumn w relacyjnej bazie danych.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="c0f9c-107">W rzeczywistości, co jest [SQLProvider](https://fsprojects.github.io/SQLProvider/) jest typ dostawcy.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="c0f9c-108">Pod warunkiem, że typy są zależne od dostawcy typów parametry wejściowe.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="c0f9c-109">Takie dane wejściowe mogą być źródłem danych przykładowych (na przykład plik schematu JSON) adres URL wskazuje bezpośrednio do zewnętrznej usługi lub parametry połączenia ze źródłem danych.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="c0f9c-110">Typ dostawcy można również upewnij się, że grupy typy są tylko rozszerzone na żądanie; oznacza to, że ich rozwinięciu Jeśli typy faktycznie przywoływane przez program.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="c0f9c-111">Umożliwia to bezpośrednią integrację na żądanie przestrzeni informacji o wielkiej skali, takich jak sieciowe rynki danych, w sposób silnie typizowany.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="c0f9c-114">Dostawcy typów generative tworzy typy, które mogą być zapisywane jako typów .NET do zestawu, w którym są produkowane.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="c0f9c-115">Umożliwia im to być używane z kodu w innych zestawów.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="c0f9c-116">Oznacza to, że reprezentacja typu źródła danych zwykle musi być taki, który jest możliwe do reprezentowania z typów .NET.</span><span class="sxs-lookup"><span data-stu-id="c0f9c-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>
