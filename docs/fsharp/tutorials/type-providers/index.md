środowiskach w przedsiębiorstwach.</span><span class="sxs-lookup"><span data-stu-id="75638-109">The key to information-rich programming is to eliminate barriers to working with diverse information sources found on the Internet and in modern enterprise environments.</span></span> <span data-ttu-id="75638-110">Jedną ze znaczących barier w dołączaniu źródła informacji do programu jest potrzeba <span data-ttu-id="75638-113">Typy dostarczanie przez dostawców typów języka F# są zazwyczaj oparte na zewnętrznych źródłach informacji.</span><span class="sxs-lookup"><span data-stu-id="75638-113">The types provided by F# type providers are usually based on external information sources.</span></span> <span data-ttu-id="75638-114">Na przykład dostawca typów języka F# dla języka SQL dostarcza typy, właściwości i metody, które są wymagane do pracy bezpośrednio z tabelami dowolnej bazy danych SQL, do której użytkownik ma dostęp.</span><span class="sxs-lookup"><span data-stu-id="75638-114">For example, an F# type provider for SQL will provide the types, properties, and methods you need to work directly with the tables of any SQL database you have access to.</span></span> <span data-ttu-id="75638-115">Podobnie dostawcy typów dla usług sieci Web WSDL dostarczają typy, właściwości i metody potrzebne do bezpośredniej pracy z dowolnymi usługami sieci Web WSDL.</span><span class="sxs-lookup"><span data-stu-id="75638-115">Similarly, a type provider for WSDL web services will provide the types, properties, and methods you need to work directly with any WSDL web service.</span></span>

<span data-ttu-id="75638-116">Zestaw typów, właściwości i metod dostarczony przez dostawcę typów języka F# może zależeć od parametrów podanych w kodzie programu.</span><span class="sxs-lookup"><span data-stu-id="75638-116">The set of types, properties, and methods provided by an F# type provider can depend on parameters given in program code.</span></span> <span data-ttu-id="75638-117">Na przykład dostawca typów może dostarczyć różne typy, w zależności od parametrów połączenia lub adresu URL usługi.</span><span class="sxs-lookup"><span data-stu-id="75638-117">For example, a type provider can provide different types depending on a connection string or a service URL.</span></span> <span data-ttu-id="75638-118">W ten sposób przestrzeń informacji dostępna w parametrach połączenia lub adresie URL jest bezpośrednio zintegrowana z programem.</span><span class="sxs-lookup"><span data-stu-id="75638-118">In this way, the information space available by means of a connection string or URL is directly integrated into your program.</span></span> <span data-ttu-id="75638-119">Dostawca typów może również zagwarantować, że grupy typów są rozszerzane jedynie na żądanie, co oznacza, że są one rozszerzane, tylko jeśli do danego typu odwołuje się program.</span><span class="sxs-lookup"><span data-stu-id="75638-119">A type provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="75638-120">Umożliwia to bezpośrednią integrację na żądanie przestrzeni informacji o wielkiej skali, takich jak sieciowe rynki danych, w sposób silnie typizowany.</span><span class="sxs-lookup"><span data-stu-id="75638-120">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="75638-121">Język F# zawiera kilku wbudowanych dostawców typów przeznaczonych do obsługi popularnych internetowych i firmowych usług danych.</span><span class="sxs-lookup"><span data-stu-id="75638-121">F# contains several built-in type providers for commonly used Internet and enterprise data services.</span></span> <span data-ttu-id="75638-122">Ci dostawcy typów oferują prosty i regularny dostęp do relacyjnych baz danych SQL oraz sieciowych usług OData i WSDL, a także obsługują używanie zapytań LINQ języka F# dla tych źródeł danych.</span><span class="sxs-lookup"><span data-stu-id="75638-122">These type providers give simple and regular access to SQL relational databases and network-based OData and WSDL services and support the use of F# LINQ queries against these data sources.</span></span>

<span data-ttu-id="75638-123">Gdy jest to konieczne, można tworzyć własnych niestandardowych dostawców typów lub odwoływać się do dostawców typów utworzonych przez innych użytkowników.</span><span class="sxs-lookup"><span data-stu-id="75638-123">Where necessary, you can create your own custom type providers, or reference type providers that have been created by others.</span></span> <span data-ttu-id="75638-124">Na przykład organizacja ma usługę danych dostarczającą dużą i rosnącą liczbę nazwanych zestawów danych, z których każdy ma własny stabilny schemat danych.</span><span class="sxs-lookup"><span data-stu-id="75638-124">For example, assume your organization has a data service providing a large and growing number of named data sets, each with its own stable data schema.</span></span> <span data-ttu-id="75638-125">Można utworzyć dostawcę typów, który odczytuje te schematy i przedstawia programiście najnowsze dostępne zestawy danych w sposób mocno typizowany.</span><span class="sxs-lookup"><span data-stu-id="75638-125">You may choose to create a type provider that reads the schemas and presents the latest available data sets to the programmer in a strongly typed way.</span></span>
