<span data-ttu-id="22c4f-108">Dostarczone typy są zależne od parametrów wejściowych dostawcy typów.</span><span class="sxs-lookup"><span data-stu-id="22c4f-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="22c4f-109">Takie dane wejściowe mogą być przykładowym źródłem danych (takim jak plik schematu JSON), adres URL wskazujący bezpośrednio na usługę zewnętrzną lub parametry połączenia ze źródłem danych.</span><span class="sxs-lookup"><span data-stu-id="22c4f-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="22c4f-110">Dostawca typów może również upewnić się, że grupy typów są rozwinięte na żądanie. oznacza to, że są one rozwinięte, jeśli do tych typów rzeczywiście odwołuje się program.</span><span class="sxs-lookup"><span data-stu-id="22c4f-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="22c4f-111">Umożliwia to bezpośrednią integrację na żądanie przestrzeni informacji o wielkiej skali, takich jak sieciowe rynki danych, w sposób silnie typizowany.</span><span class="sxs-lookup"><span data-stu-id="22c4f-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="22c4f-114">Dostawcy typów pozyskania tworzą typy, które mogą być zapisywane jako typy .NET w zestawie, w którym są produkowane.</span><span class="sxs-lookup"><span data-stu-id="22c4f-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="22c4f-115">Dzięki temu mogą one być używane z kodu w innych zestawach.</span><span class="sxs-lookup"><span data-stu-id="22c4f-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="22c4f-116">Oznacza to, że reprezentacja źródła danych musi być na ogół taka, która jest możliwe do reprezentowania z typami .NET.</span><span class="sxs-lookup"><span data-stu-id="22c4f-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>
