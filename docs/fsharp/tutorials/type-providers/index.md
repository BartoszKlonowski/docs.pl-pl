<span data-ttu-id="45ae0-114">Dostawcy typów pozyskania tworzą typy, które mogą być zapisywane jako typy .NET w zestawie, w którym są produkowane.</span><span class="sxs-lookup"><span data-stu-id="45ae0-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="45ae0-115">Dzięki temu mogą one być używane z kodu w innych zestawach.</span><span class="sxs-lookup"><span data-stu-id="45ae0-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="45ae0-116">Oznacza to, że reprezentacja źródła danych musi być na ogół taka, która jest możliwe do reprezentowania z typami .NET.</span><span class="sxs-lookup"><span data-stu-id="45ae0-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>
