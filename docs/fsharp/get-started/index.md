# <a name="get-started-with-f"></a>Wprowadzenie do języka F\#

Możesz rozpocząć pracę z językiem F # na swojej maszynie lub w trybie online.

## <a name="get-started-on-your-machine"></a>Wprowadzenie do maszyny

Istnieje kilka przewodników dotyczących instalowania i używania F # po raz pierwszy na komputerze.  Poniższa tabela ułatwia podejmowanie decyzji:

| System operacyjny | Preferuj Visual Studio | Preferuj Visual Studio Code | Preferuj wiersz polecenia |
| -- |------------------------|--------------------------|-----------------------------|-------------------------|
| Windows | [Wprowadzenie do programu Visual Studio](get-started-visual-studio.md) | [Wprowadzenie do Visual Studio Code](get-started-vscode.md) | [Wprowadzenie do interfejs wiersza polecenia platformy .NET Core](get-started-command-line.md) |
| macOS | [Wprowadzenie do programu VS for Mac](get-started-with-visual-studio-for-mac.md) | [Wprowadzenie do Visual Studio Code](get-started-vscode.md) | [Wprowadzenie do interfejs wiersza polecenia platformy .NET Core](get-started-command-line.md) |
| Linux | Nie dotyczy | [Wprowadzenie do Visual Studio Code](get-started-vscode.md) | [Wprowadzenie do interfejs wiersza polecenia platformy .NET Core](get-started-command-line.md) |

Ogólnie rzecz biorąc nie ma określonego, co jest lepsze niż reszta. Zalecamy, aby wszystkie metody na maszynie używały języka F #, aby zobaczyć, co Ci się najlepiej!

## <a name="get-started-online"></a>Wprowadzenie do trybu online

Jeśli nie zainstalujesz języka F # i platformy .NET na maszynie, możesz również rozpocząć pracę z językiem F # w przeglądarce:

* [Wprowadzenie do języka F # w programie Binder](https://mybinder.org/v2/gh/dotnet/interactive/main?urlpath=lab) to [Jupyter Notes](https://jupyter.org/) w usłudze hostowanej za pośrednictwem bezpłatnej usługi [spinacza](https://mybinder.org/) . Nie jest wymagana żadna rejestracja.
* [FABLE REPL](https://fable.io/repl/) to interaktywna, w przeglądarce REPL, która używa [Fable](https://fable.io/) do tłumaczenia kodu F # na JavaScript. Zapoznaj się z wieloma przykładami, które są od podstaw języka F # do w pełni dopracowaneej gry wideo wykonywanej w przeglądarce.
