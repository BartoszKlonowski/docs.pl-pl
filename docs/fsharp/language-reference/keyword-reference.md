Następujące tokeny są zastrzeżone w języku F #, ponieważ są one słów kluczowych w języku OCaml:

* `asr`
* `land`
* `lor`
* `lsl`
* `lsr`
* `lxor`
* `mod`
* `sig`

Jeśli używasz `--mlcompatibility` — opcja kompilatora, powyżej słowa kluczowe są dostępne do użycia jako identyfikatorów.
