<span data-ttu-id="d95ac-106">*Elementy członkowskie* są funkcjami, które są częścią definicji typu i są zadeklarowane za pomocą słowa kluczowego `member`.</span><span class="sxs-lookup"><span data-stu-id="d95ac-106">*Members* are features that are part of a type definition and are declared with the `member` keyword.</span></span> <span data-ttu-id="d95ac-107">F#typy obiektów, takie jak rekordy, klasy, Unii rozłącznych, interfejsy i struktury, obsługują członków.</span><span class="sxs-lookup"><span data-stu-id="d95ac-107">F# object types such as records, classes, discriminated unions, interfaces, and structures support members.</span></span> <span data-ttu-id="d95ac-108">Aby uzyskać więcej informacji, zobacz [rekordy](../records.md), [klasy](../classes.md), [związki rozłącznych](../discriminated-Unions.md), [interfejsy](../interfaces.md)i [struktury](../structures.md).</span><span class="sxs-lookup"><span data-stu-id="d95ac-108">For more information, see [Records](../records.md), [Classes](../classes.md), [Discriminated Unions](../discriminated-Unions.md), [Interfaces](../interfaces.md), and [Structures](../structures.md).</span></span>

<span data-ttu-id="d95ac-109">Członkowie zwykle tworzą interfejs publiczny dla typu, dlatego są one publiczne, chyba że określono inaczej.</span><span class="sxs-lookup"><span data-stu-id="d95ac-109">Members typically make up the public interface for a type, which is why they are public unless otherwise specified.</span></span> <span data-ttu-id="d95ac-110">Elementy członkowskie mogą być również deklarowane jako prywatne lub wewnętrzne.</span><span class="sxs-lookup"><span data-stu-id="d95ac-110">Members can also be declared private or internal.</span></span> <span data-ttu-id="d95ac-111">Aby uzyskać więcej informacji, zobacz [Access Control](../access-Control.md).</span><span class="sxs-lookup"><span data-stu-id="d95ac-111">For more information, see [Access Control](../access-Control.md).</span></span> <span data-ttu-id="d95ac-112">Sygnatury typów mogą być również używane do ujawniania lub nieujawniania określonych elementów członkowskich typu.</span><span class="sxs-lookup"><span data-stu-id="d95ac-112">Signatures for types can also be used to expose or not expose certain members of a type.</span></span> <span data-ttu-id="d95ac-113">Aby uzyskać więcej informacji, zobacz [podpisy](../signature-files.md).</span><span class="sxs-lookup"><span data-stu-id="d95ac-113">For more information, see [Signatures](../signature-files.md).</span></span>
