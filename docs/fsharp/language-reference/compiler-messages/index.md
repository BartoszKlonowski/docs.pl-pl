# <a name="f-compiler-messages"></a>F#komunikaty kompilatora

Ta sekcja zawiera szczegółowe informacje o błędach i F# ostrzeżeniach kompilatora, które kompilator emituje dla określonych konstrukcji. Domyślne zestawy błędów można zmienić, wykonując następujące działania:

- Traktowanie określonych ostrzeżeń jako błędów przy użyciu opcji kompilatora `-warnaserror+`,

- Ignorowanie określonych ostrzeżeń przy użyciu opcji kompilatora `-nowarn`

Jeśli określone ostrzeżenie lub błąd nie został jeszcze zarejestrowany w tej sekcji, przejdź na koniec tej strony i Wyślij opinię, która zawiera liczbę lub tekst błędu.
