# <a name="net-for-apache-spark-resources"></a>Platforma .NET dla Apache Spark zasobów

Następujące zasoby dotyczące [platformy .NET dla Apache Spark](../index.yml) mogą być przydatne do uzyskiwania dostępu do Apache Spark w aplikacjach .NET:

* [Przykłady platformy .NET dla Apache Spark C# i F #](https://github.com/dotnet/spark#samples) zawierają przykłady, które obejmują kilka scenariuszy analitycznych, takich jak dataframes i SparkSQL, strukturalne przesyłanie strumieniowe oraz zapytania TPC-H.

* Połącz się ze społecznością deweloperów przy użyciu znacznika **Spark-dotnet** na [Stack Overflow](https://stackoverflow.com/questions/tagged/spark-dotnet).

* Interesuje Cię udział? Odwiedź [repozytorium GitHub](https://github.com/dotnet/spark) , aby dowiedzieć się, jak skompilować ze źródła.

### <a name="how-to-engage-contribute-and-provide-feedback"></a>Jak angażować się, współtworzyć i przekazać Opinie

Platforma .NET dla Apache Spark Team zachęca do [uczestnictwa](https://github.com/dotnet/spark/blob/master/docs/contributing.md), zarówno w formie problemów, jak i żądań ściągnięcia. Pierwszym krokiem jest znalezienie istniejącego [problemu](https://github.com/dotnet/spark/issues) , do którego chcesz współtworzyć. Jeśli problem nie istnieje, Otwórz [nowy problem](https://github.com/dotnet/spark/issues?utf8=%E2%9C%93&q=is%3Aissue+is%3Aopen+).
