# <a name="net-for-apache-spark-resources"></a>Zasoby platformy .NET dla platformy Spark apache

Następujące zasoby [platformy .NET for Apache Spark](../index.yml) mogą być przydatne w przypadku uzyskiwania dostępu do platformy Apache Spark w aplikacjach platformy .NET:

* [.NET dla Apache Spark C# i F# przykłady](https://github.com/dotnet/spark#samples) zawiera przykłady, które obejmują kilka scenariuszy analizy, w tym Dataframes i SparkSQL, grupowe przesyłanie strumieniowe i zapytania TPC-H.

* Połącz się ze społecznością deweloperów, używając tagu **spark-dotnet** w [usłudze Stack Overflow](https://stackoverflow.com/questions/tagged/spark-dotnet).

* Chcesz wnosić swój wkład? Odwiedź [repozytorium GitHub,](https://github.com/dotnet/spark) aby dowiedzieć się, jak tworzyć z źródła.

### <a name="how-to-engage-contribute-and-provide-feedback"></a>Jak angażować się, przyczyniać się i przekazywać informacje zwrotne

Zespół .NET for Apache Spark zachęca do [wpłat](https://github.com/dotnet/spark/blob/master/docs/contributing.md), zarówno w postaci problemów, jak i żądań ściągnięcia. Pierwszym krokiem jest znalezienie istniejącego [problemu,](https://github.com/dotnet/spark/issues) do którego chcesz się przyczynić. Jeśli problem nie istnieje, otwórz [nowy problem](https://github.com/dotnet/spark/issues?utf8=%E2%9C%93&q=is%3Aissue+is%3Aopen+).
