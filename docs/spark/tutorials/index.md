# <a name="net-for-apache-spark-tutorials"></a>Samouczki platformy .NET dla Apache Spark

Poniższe samouczki umożliwiają zrozumienie sposobu korzystania z [platformy .NET dla Apache Spark](../index.yml) w celu tworzenia niestandardowych rozwiązań danych Big Data za pomocą Apache Spark i integrowania ich z aplikacjami .NET:

* [Wprowadzenie](get-started.md): demonstruje kompilację i uruchamianie prostej aplikacji platformy .net dla Apache Spark.
* [Wdrażanie aplikacji .NET dla Apache Spark w usłudze Azure HDInsight](hdinsight-deployment.md): pokazuje, jak wdrożyć platformę .net dla Apache Spark aplikacji do usługi Azure HDInsight.
* [Wdrażanie aplikacji .net for Apache Spark w usłudze Amazon EMR Spark](amazon-emr-spark-deployment.md): pokazuje, jak wdrożyć platformę .net dla Apache Spark aplikacji w systemie Amazon EMR Spark.
* [Wdrażanie aplikacji platformy .NET dla Apache Spark w kostkach](databricks-deployment.md)danych: pokazuje, jak wdrożyć platformę .net dla Apache Spark aplikacji w kostkach danych.

## <a name="next-steps"></a>Następne kroki

Aby uzyskać więcej przykładów, które korzystają z platformy .NET dla Apache Spark, zapoznaj się z [tematem Apache Spark z platformy .NET w witrynie GitHub](https://github.com/dotnet/spark#samples).
