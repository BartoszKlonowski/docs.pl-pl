# <a name="mlnet-tutorials"></a>ML.NET samouczki

Poniższe samouczki umożliwiają zrozumienie, jak używać [ML.NET](../index.yml) do tworzenia niestandardowych rozwiązań uczenia maszynowego i integrowania ich z aplikacjami .NET:

- [Analiza tonacji:](sentiment-analysis.md)pokazuje, jak zastosować zadanie **klasyfikacji binarnej** przy użyciu ML.NET.
- [Klasyfikacja problemów GitHub](github-issue-classification.md): pokazuje, jak zastosować **zadanie klasyfikacji wieloklasowej** przy użyciu ML.NET.
- [Predyktor cen](predict-prices.md): pokazuje, jak zastosować zadanie **regresji** przy użyciu ML.NET.
- [Klastrowanie przysłony:](iris-clustering.md)pokazuje, jak zastosować zadanie **klastrowania** przy użyciu ML.NET.
- [Zalecenie](movie-recommendation.md): generowanie **rekomendacji filmowych** na podstawie poprzednich ocen użytkowników
- [Klasyfikacja obrazu](image-classification.md): pokazuje, jak ponownie trenować istniejący model TensorFlow, aby utworzyć niestandardowy klasyfikator obrazu przy użyciu ML.NET.
- [Wykrywanie anomalii](sales-anomaly-detection.md): pokazuje, jak utworzyć aplikację do wykrywania anomalii do analizy danych sprzedaży produktów.
- [Wykrywanie obiektów na obrazach](object-detection-onnx.md): pokazuje, jak wykrywać obiekty na obrazach przy użyciu wstępnie przeszkolonego modelu ONNX.
- [Sklasyfikuj sentyment recenzji filmów](text-classification-tf.md): naucz się ładować wstępnie przeszkolony model TensorFlow, aby sklasyfikować sentyment recenzji filmów.
