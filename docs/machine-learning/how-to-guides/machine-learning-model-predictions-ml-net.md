```csharp
{
    Size = 900f,
    HistoricalPrices = new float[] { 155000f, 190000f, 220000f }
};

// Get Prediction
HousingPrediction prediction = predictionEngine.Predict(inputData);
```

<span data-ttu-id="9471b-122">Jeśli uzyskujesz dostęp do `Score` właściwość `prediction` obiektu, należy uzyskać podobny do wartości `150079`.</span><span class="sxs-lookup"><span data-stu-id="9471b-122">If you access the `Score` property of the `prediction` object, you should get a value similar to `150079`.</span></span>
