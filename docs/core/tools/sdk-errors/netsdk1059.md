# <a name="netsdk1059-project-contains-obsolete-net-cli-tool"></a>NETSDK1059: projekt zawiera przestarzałe narzędzie interfejsu wiersza polecenia platformy .NET

**Ten artykuł ma zastosowanie do:** ✔️ .NET 2.1.100 SDK i nowszych wersji

Gdy zestaw SDK programu .NET wystawia ostrzeżenie NETSDK1059, projekt zawiera przestarzałe narzędzie interfejsu wiersza polecenia platformy .NET. Począwszy od programu .NET 2,1, narzędzia te są dołączone do zestawu .NET SDK i nie muszą być jawnie przywoływane przez projekt. Więcej informacji na temat migracji do programu .NET 2,1 można znaleźć [tutaj](https://aka.ms/dotnetclitools-in-box).
