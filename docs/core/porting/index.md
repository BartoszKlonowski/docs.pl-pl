4. <span data-ttu-id="9482a-118">Przekonwertuj wszystkie zależności `packages.config` na format [PackageReference](/nuget/consume-packages/package-references-in-project-files) za pomocą [narzędzia konwersji w programie Visual Studio](/nuget/consume-packages/migrate-packages-config-to-package-reference).</span><span class="sxs-lookup"><span data-stu-id="9482a-118">Convert all of your `packages.config` dependencies to the [PackageReference](/nuget/consume-packages/package-references-in-project-files) format with the [conversion tool in Visual Studio](/nuget/consume-packages/migrate-packages-config-to-package-reference).</span></span>

   <span data-ttu-id="9482a-119">Ten krok obejmuje przekonwertowanie zależności ze starszego formatu `packages.config`.</span><span class="sxs-lookup"><span data-stu-id="9482a-119">This step involves converting your dependencies from the legacy `packages.config` format.</span></span> <span data-ttu-id="9482a-120">`packages.config` nie działa w oprogramowaniu .NET Core, dlatego ta konwersja jest wymagana, jeśli istnieją zależności pakietów.</span><span class="sxs-lookup"><span data-stu-id="9482a-120">`packages.config` doesn't work on .NET Core, so this conversion is required if you have package dependencies.</span></span>

5. <span data-ttu-id="9482a-121">Utwórz nowe projekty dla platformy .NET Core i skopiuj pliki źródłowe lub spróbuj skonwertować istniejący plik projektu za pomocą narzędzia.</span><span class="sxs-lookup"><span data-stu-id="9482a-121">Create new projects for .NET Core and copy over source files, or attempt to convert your existing project file with a tool.</span></span>

   <span data-ttu-id="9482a-122">W przypadku programu .NET Core jest stosowany uproszczony (i różny) [Format pliku projektu](../tools/csproj.md) niż .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="9482a-122">.NET Core uses a simplified (and different) [project file format](../tools/csproj.md) than .NET Framework.</span></span> <span data-ttu-id="9482a-123">Aby kontynuować, musisz przekonwertować pliki projektu w ten format.</span><span class="sxs-lookup"><span data-stu-id="9482a-123">You'll need to convert your project files into this format to continue.</span></span>

6. <span data-ttu-id="9482a-124">Port kodu testu.</span><span class="sxs-lookup"><span data-stu-id="9482a-124">Port your test code.</span></span>

   <span data-ttu-id="9482a-125">Ponieważ przenoszenie do programu .NET Core jest taka znacząca zmiana w bazie kodu, zdecydowanie zaleca się przeprowadzenie testów do portu, dzięki czemu można uruchamiać testy podczas przenoszenia kodu do programu.</span><span class="sxs-lookup"><span data-stu-id="9482a-125">Because porting to .NET Core is such a significant change to your codebase, it's highly recommended to get your tests ported, so that you can run tests as you port your code over.</span></span> <span data-ttu-id="9482a-126">MSTest, xUnit i NUnit działają na platformie .NET Core.</span><span class="sxs-lookup"><span data-stu-id="9482a-126">MSTest, xUnit, and NUnit all work on .NET Core.</span></span>
