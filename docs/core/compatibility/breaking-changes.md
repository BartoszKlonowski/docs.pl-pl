# <a name="net-core-breaking-changes"></a>Podstawowe zmiany w programie .NET Core

Poniższy selektor wersji zawiera przefiltrowaną listę istotnych zmian dla platformy .NET Core, ASP.NET Core i EF Core. Wersje docelowe zawierają nie tylko główne wersje, takie jak .NET Core 2,2 i .NET Core 3,0, ale również wersje zapoznawcze, takie jak .NET Core 3,0 Preview 7.

Możesz również wyświetlić poszczególne problemy, które opisują istotne zmiany w następujących repozytoriach usługi GitHub:

- W przypadku platformy .NET Core — repozytorium [dotnet/docs](https://github.com/dotnet/docs/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
- W przypadku ASP.NET Core repozytorium [ASPNET/AspNetCore](https://github.com/aspnet/AspNetCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
- W przypadku Entity Framework Core repozytorium [ASPNET/EntityFrameworkCore](https://github.com/aspnet/EntityFrameworkCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) .
