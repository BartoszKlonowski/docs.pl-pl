# <a name="breaking-changes-in-windows-forms-net-core-to-net-core"></a>Istotne zmiany w Windows Forms (.NET Core do .NET Core)

Obsługa Windows Forms została dodana do programu .NET Core w wersji 3,0. W tym artykule wymieniono istotne zmiany Windows Forms przez wersję programu .NET Core, w której zostały wprowadzone. W przypadku uaktualniania aplikacji Windows Forms z poprzedniej wersji platformy .NET Core (3,0 lub nowszej) ten artykuł dotyczy Ciebie. Jeśli przeprowadzasz migrację aplikacji Windows Forms z .NET Framework do programu .NET Core, zobacz temat istotne [zmiany w Windows Forms (.NET Framework do programu .NET Core)](../porting/winforms-breaking-changes.md).
