W przypadku zarządzania instalacją programu .NET przy użyciu Menedżera pakietów, jeśli wcześniej zainstalowano wersję zapoznawczą, może wystąpić konflikt. Menedżer pakietów może interpretować wersję bez wersji zapoznawczej jako wcześniejszą wersję programu .NET. Aby zainstalować wersję bez wersji zapoznawczej, najpierw Odinstaluj wersje w wersji zapoznawczej. Aby uzyskać więcej informacji na temat odinstalowywania programu .NET, zobacz [Jak usunąć środowisko uruchomieniowe .NET i zestaw SDK](../remove-runtime-sdk-versions.md?pivots=os-linux#uninstall-net).
