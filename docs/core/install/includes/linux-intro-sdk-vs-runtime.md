Zainstaluj zestaw SDK (zawierający środowisko uruchomieniowe), jeśli chcesz opracowywać aplikacje platformy .NET. Lub, jeśli musisz tylko uruchamiać aplikacje, zainstaluj środowisko uruchomieniowe. Jeśli instalujesz środowisko uruchomieniowe, zalecamy zainstalowanie **ASP.NET Core środowiska uruchomieniowego** , ponieważ obejmuje środowisko uruchomieniowe programu .net i ASP.NET Core.

Jeśli już zainstalowano zestaw SDK lub środowisko uruchomieniowe, `dotnet --list-sdks` Użyj `dotnet --list-runtimes` poleceń i, aby sprawdzić, które wersje są zainstalowane. Aby uzyskać więcej informacji, zobacz [Jak sprawdzić, czy program .NET jest już zainstalowany](../how-to-detect-installed-versions.md).
