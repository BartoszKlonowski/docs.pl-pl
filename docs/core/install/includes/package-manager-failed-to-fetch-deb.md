Podczas instalowania pakietu .NET może zostać wyświetlony komunikat o błędzie podobny do `Failed to fetch ... File has unexpected size ... Mirror sync in progress?` . Ten błąd może oznaczać, że źródło danych pakietu dla platformy .NET jest uaktualniane z nowszymi wersjami pakietu i należy spróbować ponownie później. Podczas uaktualniania kanał informacyjny pakietu nie powinien być dostępny przez ponad 30 minut. Jeśli ten błąd będzie ciągle wyświetlany przez ponad 30 minut, należy rozwiązać problem o <https://github.com/dotnet/core/issues> .
