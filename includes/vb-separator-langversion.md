Aby użyć znaku podkreślenia jako separatora wiodącego, należy dodać następujący element do pliku projektu Visual Basic (\*.vbproj):

```xml
<PropertyGroup>
  <LangVersion>15.5</LangVersion>
</PropertyGroup>
```

Aby uzyskać więcej informacji, zobacz [ustawienie wersji języka Visual Basic](../docs/visual-basic/language-reference/configure-language-version.md).
