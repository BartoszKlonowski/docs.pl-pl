> [!NOTE]
> .NET Framework 4,8 to Ostatnia wersja .NET Framework i nie zostaną wydane żadne dalsze wersje. Jednak .NET Framework będą [nadal obsługiwane](https://devblogs.microsoft.com/dotnet/?s=%22.net+framework%22) z miesięcznymi poprawkami zabezpieczeń i niezawodności. Ponadto będzie on nadal dołączany do systemu Windows, bez planu, aby go usunąć. Nie musisz migrować aplikacji .NET Framework, ale w celu tworzenia nowych rozwiązań Użyj [programu .net 5,0 lub nowszego](~/docs/core/introduction.md).
