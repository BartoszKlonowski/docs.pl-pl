### <a name="parameter-names-changed-in-reference-assemblies"></a>Nazwy parametrów zostały zmienione w zestawach odwołań

Niektóre nazwy parametrów zestawu odwołania zostały zmienione w celu dopasowania do nazw parametrów w zestawach implementacji.

#### <a name="change-description"></a>Zmień opis

W poprzednich wersjach programu .NET niektóre nazwy parametrów [zestawu odwołania](../../../../docs/standard/assembly/reference-assemblies.md) są inne niż odpowiadające im parametry w zestawie implementacji. Może to spowodować problemy podczas korzystania z nazwanych argumentów i odbicie.

W programie .NET 5,0 te niezgodne nazwy parametrów zostały zaktualizowane w zestawach referencyjnych, aby dokładnie dopasować odpowiednie nazwy parametrów w zestawach implementacji.

#### <a name="reason-for-change"></a>Przyczyna zmiany

Nazwy parametrów zostały zmienione pod kątem spójności i w celu uniknięcia błędów przy użyciu nazwanych argumentów i odbicia.

#### <a name="version-introduced"></a>Wprowadzona wersja

5.0

#### <a name="recommended-action"></a>Zalecana akcja

Jeśli wystąpi błąd kompilatora z powodu zmiany nazwy parametru, należy odpowiednio zaktualizować nazwę parametru.
