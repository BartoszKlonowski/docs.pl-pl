<span data-ttu-id="c6ae2-103">Ze względu na wewnętrzne zmiany typu <xref:System.Collections.Concurrent.ConcurrentDictionary%602> obiekty, które są serializowane z .NET Framework 4,5 przy użyciu <xref:System.Runtime.Serialization.NetDataContractSerializer?displayProperty=fullName> nie można deserializować w .NET Framework 4.5.1 lub w .NET Framework 4.5.2.
