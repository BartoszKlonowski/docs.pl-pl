### <a name="objectdisposedexception-thrown-by-wpf-spellchecker"></a>Objectdisposedexception — generowane przez sprawdzanie pisowni w WPF

|   |   |
|---|---|
|Szczegóły|Aplikacje WPF od czasu do czasu awarii podczas zamykania aplikacji za pomocą <xref:System.ObjectDisposedException?displayProperty=name> zgłoszony przez sprawdzanie pisowni. Problem ten został rozwiązany w .NET Framework 4.7 WPF Obsługa wyjątku bezpiecznie i zapewnić w ten sposób, że aplikacje są już naruszone. Należy zauważyć, że okazjonalne wyjątki pierwszej szansy będzie w dalszym ciągu należy przestrzegać w aplikacjach działających w debugerze.|
|Sugestia|Uaktualnianie do programu .NET Framework 4.7|
|Zakres|Krawędź|
|Wersja|4.6.1|
|Typ|Środowisko uruchomieniowe|
