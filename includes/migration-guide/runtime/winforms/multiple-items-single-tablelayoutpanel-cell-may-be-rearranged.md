### <a name="multiple-items-in-a-single-tablelayoutpanel-cell-may-be-rearranged"></a>Można przestawiać wielu elementów w komórce formantu TableLayoutPanel

|   |   |
|---|---|
|Szczegóły|W programie .NET Framework 4.5, jeśli wiele elementów są umieszczane w tym samym <xref:System.Windows.Forms.TableLayoutPanel?displayProperty=name> komórki, mogą być wyświetlane w innej kolejności, niż w .NET Framework 4.0.|
|Sugestia|To zachowanie została przywrócona do obsługi aktualizacji dla programu .NET Framework 4.5. Zaktualizuj .NET Framework 4.5, lub uaktualnienia programu .NET Framework 4.5.1 lub nowszej, aby rozwiązać ten problem. Alternatywnie uniknąć niejednoznaczne przypadku wielu elementów w tym samym <xref:System.Windows.Forms.TableLayoutPanel?displayProperty=name> komórki.|
|Zakres|Pomocnicza|
