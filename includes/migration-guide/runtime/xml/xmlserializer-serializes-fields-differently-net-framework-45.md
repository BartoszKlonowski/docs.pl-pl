### <a name="xmlserializer-serializes-fields-differently-in-net-framework-45"></a>Element XmlSerializer serializuje pola inaczej w programie .NET Framework 4.5

|   |   |
|---|---|
|Szczegóły|Zmiany w <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name> w programie .NET Framework 4.5 spowodował pola mają być inaczej sformatowane serializacji XML.|
|Sugestia|Ten problem został rozwiązany w aktualizacja obsługi programu .NET Framework 4.5. Zaktualizuj .NET Framework 4.5, lub uaktualnienia programu .NET Framework 4.5.1 lub nowszej, aby rozwiązać ten problem.
