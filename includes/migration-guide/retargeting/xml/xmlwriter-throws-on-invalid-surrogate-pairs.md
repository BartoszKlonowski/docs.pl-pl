### <a name="xmlwriter-throws-on-invalid-surrogate-pairs"></a>Element XmlWriter zgłoszenie nieprawidłowe znaki dwuskładnikowe

|   |   |
|---|---|
|Szczegóły|W przypadku aplikacji przeznaczonych dla platformy .NET Framework 4.5.2 lub starszych wersji napisanie para zastępcza nieprawidłowy przy użyciu rezerwowy obsługi wyjątków nie zawsze zgłasza wyjątek. Dla aplikacji przeznaczonych dla platformy .NET Framework 4.6, próbuje zapisać nieprawidłowy Surogat generuje parę <xref:System.ArgumentException?displayProperty=name>.|
|Sugestia|W razie potrzeby tego podziału można zaoszczędzone przez przeznaczonych dla platformy .NET Framework 4.5.2 lub starszym. Alternatywnie można wstępnie przetworzonych w prawidłowym kodem xml przed ich zapisaniem nieprawidłowe znaki dwuskładnikowe.|
