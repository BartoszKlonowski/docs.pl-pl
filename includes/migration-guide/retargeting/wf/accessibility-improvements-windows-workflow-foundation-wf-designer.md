#### <a name="suggestion"></a><span data-ttu-id="b51c2-135">Sugestia</span><span class="sxs-lookup"><span data-stu-id="b51c2-135">Suggestion</span></span>

<span data-ttu-id="b51c2-136">Jeśli masz aplikację z nieobsługiwanym projektantem przepływu pracy, Twoja aplikacja może korzystać z tych zmian, wykonując jedną z następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="b51c2-136">If you have an application with a re-hosted workflow designer, your application can benefit from these changes by performing either of these actions:</span></span>

- <span data-ttu-id="b51c2-137">Skompiluj ponownie aplikację pod kątem .NET Framework 4.7.1.</span><span class="sxs-lookup"><span data-stu-id="b51c2-137">Recompile your application to target the .NET Framework 4.7.1.</span></span> <span data-ttu-id="b51c2-138">Te zmiany ułatwień dostępu są domyślnie włączone.</span><span class="sxs-lookup"><span data-stu-id="b51c2-138">These accessibility changes are enabled by default.</span></span>
- <span data-ttu-id="b51c2-139">Jeśli aplikacja jest przeznaczona dla .NET Framework 4,7 lub wcześniejszych, ale jest uruchomiona na .NET Framework 4.7.1, można zrezygnować z tych starszych zachowań dostępności, dodając następujący [przełącznik AppContext](~/docs/framework/configure-apps/file-schema/runtime/appcontextswitchoverrides-element.md) do `<runtime>` sekcji pliku app.config i ustawiając go na `false` , tak jak pokazano w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="b51c2-139">If your application targets the .NET Framework 4.7 or earlier but is running on the .NET Framework 4.7.1, you can opt out of these legacy accessibility behaviors by adding the following [AppContext switch](~/docs/framework/configure-apps/file-schema/runtime/appcontextswitchoverrides-element.md) to the `<runtime>` section of the app.config file and set it to `false`, as the following example shows.</span></span>

```xml
<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <startup>
    <supportedRuntime version="v4.0" sku=".NETFramework,Version=v4.7"/>
  </startup>
  <runtime>
    <!-- AppContextSwitchOverrides value attribute is in the form of 'key1=true/false;key2=true/false  -->
    <AppContextSwitchOverrides value="Switch.UseLegacyAccessibilityFeatures=false" />
  </runtime>
</configuration>
```

<span data-ttu-id="b51c2-140">Aplikacje, które są przeznaczone dla .NET Framework 4.7.1 lub nowszych i chcą zachować starsze zachowanie ułatwień dostępu, mogą zrezygnować z używania starszych funkcji ułatwień dostępu przez jawne ustawienie tego przełącznika AppContext na `true` .</span><span class="sxs-lookup"><span data-stu-id="b51c2-140">Applications that target the .NET Framework 4.7.1 or later and want to preserve the legacy accessibility behavior can opt in to the use of legacy accessibility features by explicitly setting this AppContext switch to `true`.</span></span>
