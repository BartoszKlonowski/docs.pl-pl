### <a name="workflow-30-types-are-obsolete"></a>Typy przepływów pracy 3.0 są nieaktualne

|   |   |
|---|---|
|Szczegóły|Interfejsy API Windows Workflow Foundation (WWF) 3.0, (te z przestrzeni nazw System.Workflow) są obecnie przestarzałe.|
|Sugestia|Zamiast tego należy używać nowych interfejsów API 4.0 WWF (w System.Activities). Przykładem korzystania z nowych interfejsów API można znaleźć [tutaj](~/docs/framework/windows-workflow-foundation/how-to-update-the-definition-of-a-running-workflow-instance.md) i dalsze wskazówki są dostępne [tutaj](https://blogs.msdn.com/b/workflowteam/archive/2012/02/08/deprecatingwf3.aspx). Alternatywnie ponieważ interfejsy API 3.0 WWF są nadal obsługiwane, mogą być używane i ostrzeżenie czas kompilacji unika również pomijaniem go lub za pomocą starszego kompilatora.|
|Zakres|Główne|
|Wersja|4.5|
|Typ|Trwa przekierowywanie|
