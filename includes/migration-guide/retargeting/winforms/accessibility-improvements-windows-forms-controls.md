### <a name="accessibility-improvements-in-windows-forms-controls"></a>Ulepszenia ułatwień dostępu w kontrolkach formularzy Windows Forms

|   |   |
|---|---|
|Szczegóły|Windows Forms w celu ulepszania jak to działa przy użyciu technologii ułatwień dostępu w celu skuteczniejszej obsługi klientów Windows Forms. Obejmują one następujące zmiany, począwszy od programu .NET Framework 4.7.1:<ul><li>Zmiany w celu zwiększenia wyświetlanych podczas trybu wysokiego kontrastu.</li><li>Zmiany w ulepszaniu środowiska przeglądarkę właściwości.
