### <a name="ensure-systemuri-uses-a-consistent-reserved-character-set"></a>Upewnij się, że System.Uri korzysta z zestawu spójne zastrzeżonego znaku

|   |   |
|---|---|
|Szczegóły|W <xref:System.Uri?displayProperty=fullName>, niektóre teraz spójnie lewej kodowania znaków procent, kodowane, które zostały czasami zdekodowane. Dzieje się tak we właściwości i metody, które uzyskują dostęp do składników ścieżki, zapytania, fragment lub informacje o użytkowniku identyfikatora URI. Spowoduje to zmianę działania tylko wtedy, gdy oba poniższe są spełnione:<ul><li>Identyfikator URI zawiera formie zakodowanej któregokolwiek z następujących znaków zarezerwowanych: <code>:</code>, <code>'</code>, <code>(</code>, <code>)</code>, <code>!</code> lub <code>*</code>.</li><li>Identyfikator URI zawiera Unicode lub zakodować, niezastrzeżonych znaków. Jeśli oba powyższe warunki są spełnione, lewej są zakodowane zakodowanego zarezerwowanych znaków. W poprzednich wersjach programu .NET Framework są zdekodowane.</li></ul>|
|Sugestia|Dla aplikacji przeznaczonych dla wersji programu .NET Framework, począwszy od 4.7.2 nowe zachowanie dekodowania jest domyślnie włączona. Jeśli ta zmiana jest niepożądane, można ją wyłączyć, dodając następujące [AppContextSwitchOverrides](~/docs/framework/configure-apps/file-schema/runtime/appcontextswitchoverrides-element.md) przełączyć się do <code>&lt;runtime&gt;</code> sekcji pliku konfiguracji aplikacji:<pre><code class="lang-xml">&lt;runtime&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.System.Uri.DontEnableStrictRFC3986ReservedCharacterSets=true&quot; /&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;</code></pre>W przypadku aplikacji docelowych wcześniejszych wersji programu .NET Framework, które działają w ramach wersji w programie .NET Framework 4.7.2 nowe zachowanie dekodowania jest domyślnie wyłączona. Możesz je włączyć, dodając następujące [AppContextSwitchOverrides](~/docs/framework/configure-apps/file-schema/runtime/appcontextswitchoverrides-element.md) przełączyć się do <code>&lt;runtime&gt;</code> sekcji pliku konfiguracji aplikacji::<pre><code class="lang-xml">&lt;runtime&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.System.Uri.DontEnableStrictRFC3986ReservedCharacterSets=false&quot; /&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;</code></pre>|
|Zakres|Pomocnicza|
|Wersja|4.7.2|
|Typ|Przekierowania|
|Dotyczy interfejsów API|<ul><li><xref:System.Uri?displayProperty=nameWithType></li></ul>|
