### <a name="iasyncresultcompletedsynchronously-property-must-be-correct-for-the-resulting-task-to-complete"></a>Właściwość IAsyncResult.CompletedSynchronously musi być poprawna, aby wynikowe zadanie zostało wykonane

|   |   |
|---|---|
|Szczegóły|Podczas wywoływania TaskFactory.FromAsync, implementacja <xref:System.IAsyncResult.CompletedSynchronously> właściwość musi być poprawna, aby wynikowe zadanie zostało wykonane. Oznacza to że właściwość musi zwracać wartość true przypadku i tylko wtedy, gdy implementacja została zakończona synchronicznie. Uprzednio ta właściwość nie była sprawdzana.|
|Sugestia|Jeśli <xref:System.IAsyncResult?displayProperty=name> implementacje poprawnie zwracają wartość true dla <xref:System.IAsyncResult.CompletedSynchronously?displayProperty=name> właściwości tylko wtedy, gdy zadanie zostało ukończone synchronicznie, a następnie będzie można zauważyć bez przerwy. Użytkownicy powinni sprawdzić <xref:System.IAsyncResult?displayProperty=name> implementacje posiadanych (jeśli istnieje), aby upewnić się, że są poprawnie ocenić, czy zadanie zostało ukończone, synchroniczny, czy nie.|
